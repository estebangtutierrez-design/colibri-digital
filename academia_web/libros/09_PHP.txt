CAPITULO 1: INTRODUCCIÓN A PHP
================================================================================

¿Qué es PHP?
------------
PHP (Hypertext Preprocessor) es un lenguaje de scripting de propósito general
especialmente diseñado para el desarrollo web del lado del servidor. Fue creado
por Rasmus Lerdorf en 1995 como un conjunto de herramientas CGI para su página
personal, y ha evolucionado hasta convertirse en el lenguaje de programación
del lado del servidor más popular de la web.

Características principales:
- Dinámico y de tipado débil (no necesita declarar tipos).
- Amplio soporte para bases de datos (MySQL, PostgreSQL, SQLite, etc.).
- Gran ecosistema de frameworks (Laravel, Symfony, CodeIgniter, CakePHP).
- Integración nativa con HTML (se puede intercalar con código HTML).
- Open Source y gran comunidad.
- Ideal para aplicaciones web, APIs, sistemas de gestión de contenido (CMS).
- Interpretado (no necesita compilación).

Filosofía: PHP está diseñado para ser práctico y resolver problemas rápidamente.

Versiones actuales: PHP 8.2, PHP 8.3 (2024). PHP 8.x trae mejoras de rendimiento,
tipos de unión, atributos, match expression, y más.

CAPITULO 4: ESTRUCTURAS DE CONTROL Y BUCLES

Condicional if/else/elseif:
---------------------------
if ($edad >= 18) {
    echo "Mayor de edad";
} elseif ($edad > 12) {
    echo "Adolescente";
} else {
    echo "Niño";
}

// Sintaxis alternativa (para templates):
if ($edad >= 18) :
    echo "Mayor";
else :
    echo "Menor";
endif;

Operador ternario:
------------------
$mensaje = ($edad >= 18) ? "Mayor" : "Menor";

// Operador null coalesce (PHP 7+):
$nombre = $_GET['nombre'] ?? "Invitado";

// Operador nave espacial (PHP 7+):
echo 2 <=> 1; // 1 (mayor)
echo 1 <=> 1; // 0 (igual)
echo 1 <=> 2; // -1 (menor)

Switch:
-------
switch ($dia) {
    case 1:
        echo "Lunes";
        break;
    case 2:
        echo "Martes";
        break;
    default:
        echo "Otro día";
}

// Match expression (PHP 8+ - más moderno):
$resultado = match ($dia) {
    1 => "Lunes",
    2 => "Martes",
    default => "Otro día",
};

Bucles:
-------
for ($i = 0; $i < 10; $i++) {
    echo $i;
}

$i = 0;
while ($i < 10) {
    echo $i;
    $i++;
}

do {
    echo $i;
    $i++;
} while ($i < 10);

foreach ($array as $valor) {
    echo $valor;
}

foreach ($array as $clave => $valor) {
    echo "$clave: $valor";
}

Break y continue: funcionan como en otros lenguajes.

CAPITULO 7: SUPERGLOBALS ($_GET, $_POST, $_SESSION)
================================================================================

Las superglobals son variables predefinidas accesibles en cualquier ámbito.

$_GET: Recibir parámetros de la URL (query string).
--------------------------------------------------
// URL: http://localhost?nombre=Esteban&edad=30
$nombre = $_GET['nombre'] ?? '';
$edad = $_GET['edad'] ?? 0;

$_POST: Recibir datos de formularios con method="POST".
-------------------------------------------------------
$nombre = $_POST['nombre'] ?? '';
$email = $_POST['email'] ?? '';

$_REQUEST: Combina $_GET, $_POST, $_COOKIE.
-------------------------------------------
$dato = $_REQUEST['dato'] ?? '';

$_SESSION: Variables de sesión (persisten entre peticiones).
------------------------------------------------------------
session_start(); // Siempre al inicio
$_SESSION['usuario'] = "Esteban";
$_SESSION['id'] = 123;

// En otra página:
session_start();
echo $_SESSION['usuario']; // "Esteban"
session_destroy(); // eliminar sesión

$_COOKIE: Cookies del navegador.
---------------------------------
setcookie("nombre", "Esteban", time() + 3600); // 1 hora
$nombre = $_COOKIE['nombre'] ?? '';

$_SERVER: Información del servidor y petición.
-----------------------------------------------
echo $_SERVER['REQUEST_METHOD']; // GET, POST, etc.
echo $_SERVER['HTTP_USER_AGENT']; // navegador
echo $_SERVER['REMOTE_ADDR']; // IP del usuario
echo $_SERVER['SERVER_NAME']; // dominio
echo $_SERVER['DOCUMENT_ROOT']; // ruta raíz

$_FILES: Archivos subidos (con enctype="multipart/form-data").
---------------------------------------------------------------
$archivo = $_FILES['mi_archivo'];
$nombre_original = $archivo['name'];
$temporal = $archivo['tmp_name'];
move_uploaded_file($temporal, "uploads/" . $nombre_original);

CAPITULO 8: PROGRAMACIÓN ORIENTADA A OBJETOS (POO)
================================================================================

Clases y objetos:
-----------------
class Persona {
    // Propiedades (atributos)
    public $nombre;
    private $edad;      // solo dentro de la clase
    protected $id;      // dentro de la clase y herencia

    // Constructor
    public function __construct($nombre, $edad) {
        $this->nombre = $nombre;
        $this->edad = $edad;
        $this->id = rand(1, 1000);
    }

    // Métodos
    public function saludar() {
        echo "Hola, soy $this->nombre";
    }

    public function getEdad() {
        return $this->edad;
    }

    public function setEdad($edad) {
        if ($edad > 0) {
            $this->edad = $edad;
        }
    }

    // Destructor (se ejecuta al destruir el objeto)
    public function __destruct() {
        echo "Objeto destruido";
    }
}

// Uso:
$p = new Persona("Esteban", 30);
$p->saludar();
echo $p->getEdad();
$p->setEdad(31);

Herencia:
---------
class Empleado extends Persona {
    private $salario;

    public function __construct($nombre, $edad, $salario) {
        parent::__construct($nombre, $edad);
        $this->salario = $salario;
    }

    public function mostrarSalario() {
        echo "Salario: $this->salario";
    }

    // Sobrescribir método:
    public function saludar() {
        parent::saludar();
        echo " y soy empleado";
    }
}

Interfaces (contrato):
----------------------
interface Animal {
    public function hacerSonido();

}

class Perro implements Animal {
    public function hacerSonido() {
        echo "Guau";
    }
    public function moverse() {
        echo "Corriendo";
    }
}

Clases abstractas (no se pueden instanciar):
--------------------------------------------
abstract class Figura {
    protected $color;

class Circulo extends Figura {

    public function __construct($radio) {

    public function area() {
        return pi() * $this->radio * $this->radio;

trait Utilidades {

CAPITULO 10: TRABAJO CON ARCHIVOS Y DIRECTORIOS
================================================================================

Leer archivo:
-------------
// Todo el contenido:
$contenido = file_get_contents("archivo.txt");
echo $contenido;

// Array de líneas:
$lineas = file("archivo.txt");
foreach ($lineas as $linea) {
    echo $linea;
}

// Con fopen (para mayor control):
$fp = fopen("archivo.txt", "r");
if ($fp) {
    while (!feof($fp)) {
        $linea = fgets($fp);
        echo $linea;
    }
    fclose($fp);
}

Escribir archivo:
-----------------
// Todo el contenido (sobrescribe):
file_put_contents("salida.txt", "Contenido\nNueva línea");

// Añadir (append):
file_put_contents("log.txt", "Nueva entrada\n", FILE_APPEND);

// Con fopen:
$fp = fopen("salida.txt", "w");
fwrite($fp, "Línea 1\n");
fwrite($fp, "Línea 2\n");
fclose($fp);

Modos de apertura (fopen):
--------------------------
"r"  -> solo lectura
"r+" -> lectura y escritura
"w"  -> solo escritura (sobrescribe)
"w+" -> lectura y escritura (sobrescribe)
"a"  -> solo escritura (añade al final)
"a+" -> lectura y escritura (añade al final)

Directorios:
------------
// Crear:
mkdir("carpeta", 0755);

// Eliminar:
rmdir("carpeta");

// Listar contenido:
$archivos = scandir(".");
foreach ($archivos as $archivo) {
    if ($archivo != "." && $archivo != "..") {
        echo $archivo;
    }
}

// Verificar existencia:
if (file_exists("archivo.txt")) {
    echo "Existe";
}
if (is_dir("carpeta")) { echo "Es directorio"; }
if (is_file("archivo.txt")) { echo "Es archivo"; }

CAPITULO 11: CONEXIÓN A BASES DE DATOS (PDO)
================================================================================

PDO (PHP Data Objects) es la forma segura y moderna de conectar con bases de datos.

Conexión:
---------
$dsn = "mysql:host=localhost;dbname=mi_bd;charset=utf8mb4";
$usuario = "root";
$password = "1234";
try {
    $pdo = new PDO($dsn, $usuario, $password);
    $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
    $pdo->setAttribute(PDO::ATTR_DEFAULT_FETCH_MODE, PDO::FETCH_ASSOC);
    echo "Conectado";
} catch (PDOException $e) {
    die("Error: " . $e->getMessage());
}

SELECT (consulta):
------------------
$stmt = $pdo->query("SELECT * FROM usuarios");
while ($row = $stmt->fetch()) {
    echo $row['nombre'] . " - " . $row['email'];
}

// Obtener todos los resultados:
$usuarios = $pdo->query("SELECT * FROM usuarios")->fetchAll();

SELECT con prepared statement (seguro contra inyección SQL):
------------------------------------------------------------
$id = $_GET['id'] ?? 0;
$stmt = $pdo->prepare("SELECT * FROM usuarios WHERE id = ?");
$stmt->execute([$id]);
$usuario = $stmt->fetch();

// Con parámetros nombrados:
$stmt = $pdo->prepare("SELECT * FROM usuarios WHERE email = :email");
$stmt->execute(['email' => $email]);
$usuario = $stmt->fetch();

INSERT:
-------
$stmt = $pdo->prepare("INSERT INTO usuarios (nombre, email, edad) VALUES (?, ?, ?)");
$stmt->execute(["Esteban", "esteban@mail.com", 30]);
$id = $pdo->lastInsertId(); // obtener ID generado

UPDATE:
-------
$stmt = $pdo->prepare("UPDATE usuarios SET edad = :edad WHERE id = :id");
$stmt->execute(['edad' => 31, 'id' => $id]);

DELETE:
-------
$stmt = $pdo->prepare("DELETE FROM usuarios WHERE id = ?");
$stmt->execute([$id]);

Transacciones:
--------------
try {
    $pdo->beginTransaction();
    // múltiples operaciones
    $pdo->commit();
} catch (Exception $e) {
    $pdo->rollBack();
    throw $e;
}

HTML del formulario:

Procesar (procesar.php):
------------------------
if ($_SERVER['REQUEST_METHOD'] === 'POST') {
    // Validar nombre
    $nombre = trim($_POST['nombre'] ?? '');
    if (empty($nombre)) {
        $errores[] = "El nombre es obligatorio";
    } elseif (strlen($nombre) < 3) {
        $errores[] = "El nombre debe tener al menos 3 caracteres";
    }

    // Validar email
    $email = trim($_POST['email'] ?? '');
    if (!filter_var($email, FILTER_VALIDATE_EMAIL)) {
        $errores[] = "Email inválido";
    }

    // Validar edad
    $edad = (int)($_POST['edad'] ?? 0);
    if ($edad < 18 || $edad > 100) {
        $errores[] = "Edad debe estar entre 18 y 100";
    }

    // Validar archivo
    if (isset($_FILES['archivo']) && $_FILES['archivo']['error'] === UPLOAD_ERR_OK) {
        $extension = pathinfo($_FILES['archivo']['name'], PATHINFO_EXTENSION);
        if (!in_array($extension, ['jpg', 'png', 'pdf'])) {
            $errores[] = "Tipo de archivo no permitido";
        }
    }

    if (empty($errores)) {
        // Procesar datos (guardar en BD)
        echo "Formulario válido";
    } else {
        // Mostrar errores
        foreach ($errores as $error) {
            echo "<p style='color:red'>$error</p>";
        }
    }
}

CAPITULO 14: BUENAS PRÁCTICAS EN PHP
================================================================================

1. Usa PDO o MySQLi para bases de datos (NO uses mysql_* obsoleto).
2. Valida y filtra toda entrada de usuario:

6. Mantén la lógica separada de la presentación (patrón MVC).
7. Usa variables de entorno para configuración (getenv()).
8. Maneja errores con try/catch.
9. Mantén PHP actualizado.
10. Usa sesiones seguras (regenerar ID después de login).
11. Evita exponer información sensible en mensajes de error (display_errors off en producción).
12. Usa namespaces para organizar código.
13. Escribe tests (PHPUnit).

class Usuario {
    private $pdo;
    public function __construct($pdo) {
        $this->pdo = $pdo;
    }
    public function getAll() {
        $stmt = $this->pdo->query("SELECT * FROM usuarios ORDER BY id DESC");
        return $stmt->fetchAll(PDO::FETCH_ASSOC);
    }
    public function getById($id) {
        $stmt = $this->pdo->prepare("SELECT * FROM usuarios WHERE id = ?");
        $stmt->execute([$id]);
        return $stmt->fetch(PDO::FETCH_ASSOC);
    }
    public function create($nombre, $email, $password) {
        $hash = password_hash($password, PASSWORD_DEFAULT);
        $stmt = $this->pdo->prepare("INSERT INTO usuarios (nombre, email, password) VALUES (?, ?, ?)");
        return $stmt->execute([$nombre, $email, $hash]);
    }
    public function update($id, $nombre, $email, $password = null) {
        if ($password) {
            $hash = password_hash($password, PASSWORD_DEFAULT);
            $stmt = $this->pdo->prepare("UPDATE usuarios SET nombre=?, email=?, password=? WHERE id=?");
            return $stmt->execute([$nombre, $email, $hash, $id]);
        } else {
            $stmt = $this->pdo->prepare("UPDATE usuarios SET nombre=?, email=? WHERE id=?");
            return $stmt->execute([$nombre, $email, $id]);
        }
    }
    public function delete($id) {
        $stmt = $this->pdo->prepare("DELETE FROM usuarios WHERE id = ?");
        return $stmt->execute([$id]);
    }
    public function authenticate($email, $password) {
        $stmt = $this->pdo->prepare("SELECT * FROM usuarios WHERE email = ?");
        $stmt->execute([$email]);
        $usuario = $stmt->fetch(PDO::FETCH_ASSOC);
        if ($usuario && password_verify($password, $usuario['password'])) {
            return $usuario;
        }
        return false;
    }
}
