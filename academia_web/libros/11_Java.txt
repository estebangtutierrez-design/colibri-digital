CAPITULO 1: INTRODUCCIÓN A JAVA
================================================================================

¿Qué es Java?
-------------
Java es un lenguaje de programación de propósito general, orientado a objetos,
diseñado para ser escrito una vez y ejecutarse en cualquier plataforma (WORA -
Write Once, Run Anywhere). Fue creado por James Gosling en Sun Microsystems
(ahora Oracle) y lanzado en 1995.

Características principales:
- Orientado a objetos (POO puro, todo es objeto excepto tipos primitivos).
- Independiente de plataforma gracias a la JVM (Java Virtual Machine).
- Sintaxis similar a C/C++.
- Gestión automática de memoria (recolector de basura - Garbage Collector).
- Amplio ecosistema de bibliotecas y frameworks (Spring, Hibernate, etc.).
- Seguro y robusto (manejo de excepciones, tipado fuerte).
- Multihilo nativo.

Ediciones de Java:
------------------
- Java SE (Standard Edition): para aplicaciones de escritorio y servidores.
- Java EE / Jakarta EE: para aplicaciones empresariales.
- Java ME (Micro Edition): para dispositivos móviles embebidos.

Versiones LTS (Long-Term Support): Java 8, Java 11, Java 17, Java 21.
La versión más reciente LTS es Java 21 (septiembre 2023).

CAPITULO 2: INSTALACIÓN Y CONFIGURACIÓN (JDK, JVM, JRE)
================================================================================

Conceptos clave:
----------------
- JVM (Java Virtual Machine): Ejecuta el bytecode de Java.
- JRE (Java Runtime Environment): JVM + bibliotecas para ejecutar programas.
- JDK (Java Development Kit): JRE + herramientas de desarrollo (compilador, debugger).

Instalación en Linux Mint:
--------------------------
sudo apt update
sudo apt install openjdk-21-jdk

Verificar instalación:
java -version
javac -version

Configurar JAVA_HOME (opcional):
--------------------------------
export JAVA_HOME=/usr/lib/jvm/java-21-openjdk-amd64
export PATH=$PATH:$JAVA_HOME/bin

Primer programa (HolaMundo.java):
---------------------------------
public class HolaMundo {
    public static void main(String[] args) {
        System.out.println("¡Hola, Mundo!");
    }
}

Compilar y ejecutar:
--------------------
javac HolaMundo.java   # Genera HolaMundo.class (bytecode)
java HolaMundo         # Ejecuta en la JVM

Comentarios:
------------

Tipos de datos primitivos:
--------------------------
byte     -> 8 bits, -128 a 127
short    -> 16 bits, -32,768 a 32,767
int      -> 32 bits, -2^31 a 2^31-1
long     -> 64 bits, sufijo L (ej: 100L)
float    -> 32 bits, sufijo f (ej: 3.14f)
double   -> 64 bits (ej: 3.14)
char     -> 16 bits, caracteres Unicode (comillas simples: 'A')
boolean  -> true o false

Declaración de variables:
-------------------------
int edad = 30;
double altura = 1.75;
char inicial = 'E';

-------------------

-----------------------------------

Conversión de tipos (casting):
------------------------------

double d = 10;           // conversión implícita (widening)
int y = (int)d;          // conversión explícita (narrowing)

Operadores:

CAPITULO 4: ESTRUCTURAS DE CONTROL Y BUCLES
================================================================================

Condicional if/else:
--------------------
if (edad >= 18) {
    System.out.println("Mayor de edad");
} else if (edad > 12) {
    System.out.println("Adolescente");
} else {
    System.out.println("Niño");
}

Operador ternario:
------------------
String mensaje = (edad >= 18) ? "Mayor" : "Menor";

Switch (con expresión - Java 14+):
----------------------------------
String dia = switch (numero) {
    case 1 -> "Lunes";
    case 2 -> "Martes";
    default -> "Otro día";
};

Switch tradicional:
-------------------
switch (dia) {
    case 1:
        System.out.println("Lunes");
        break;
    case 2:
        System.out.println("Martes");
        break;
    default:
        System.out.println("Otro día");
}

Bucles:
-------
for (int i = 0; i < 10; i++) {
    System.out.println(i);
}

int i = 0;
while (i < 10) {
    System.out.println(i);
    i++;
}

do {
    System.out.println(i);
    i++;
} while (i < 10);

// For-each (para colecciones):
int[] numeros = {1,2,3,4,5};
for (int n : numeros) {
    System.out.println(n);
}

Break y continue: funcionan como en otros lenguajes.

CAPITULO 6: PROGRAMACIÓN ORIENTADA A OBJETOS (POO)
================================================================================

Clases y Objetos:
-----------------
public class Persona {
    // Atributos (variables de instancia)
    private String nombre;
    private int edad;

    // Constructor
    public Persona(String nombre, int edad) {
        this.nombre = nombre;
        this.edad = edad;
    }

    // Métodos (getters y setters)
    public String getNombre() {
        return nombre;
    }
    public void setNombre(String nombre) {
        this.nombre = nombre;
    }
    public int getEdad() {
        return edad;
    }
    public void setEdad(int edad) {
        if (edad > 0) {
            this.edad = edad;
        }
    }

    // Método de instancia
    public void saludar() {
        System.out.println("Hola, soy " + nombre);
    }

    // Método estático (pertenece a la clase)
    public static void especie() {
        System.out.println("Humano");
    }
}

Uso:
----
Persona p = new Persona("Esteban", 30);
p.saludar();
p.setEdad(31);
System.out.println(p.getNombre());

Persona.especie();  // Llamada a método estático

Modificadores de acceso:
------------------------
public      -> Accesible desde cualquier parte
protected   -> Accesible en el mismo paquete y subclases
(default)   -> Accesible en el mismo paquete
private     -> Solo dentro de la clase

CAPITULO 7: HERENCIA Y POLIMORFISMO
================================================================================

Herencia (extends):
-------------------
public class Empleado extends Persona {
    private double salario;

    public Empleado(String nombre, int edad, double salario) {
        super(nombre, edad);   // Llama al constructor de Persona
        this.salario = salario;
    }

    public double getSalario() {
        return salario;
    }

    public void setSalario(double salario) {
        this.salario = salario;
    }

    @Override   // Sobrescritura de método
    public void saludar() {
        super.saludar();   // Llama al método de Persona
        System.out.println("Soy empleado y gano " + salario);
    }

    // Método adicional
    public void trabajar() {
        System.out.println("Trabajando...");
    }
}

Polimorfismo:
-------------
Persona p = new Empleado("Luis", 28, 3000);
p.saludar();   // Llama al método sobrescrito de Empleado

// Instanceof (verificar el tipo real):
if (p instanceof Empleado) {
    Empleado e = (Empleado) p;   // Casting
    System.out.println(e.getSalario());
}

Interfaz (contrato):
--------------------
public interface Animal {
    void hacerSonido();   // Método abstracto (sin cuerpo)
    default void dormir() {   // Método con implementación por defecto (Java 8)
        System.out.println("Durmiendo...");
    }
}

public class Perro implements Animal {
    @Override
    public void hacerSonido() {
        System.out.println("Guau");
    }
}

Clase abstracta (no se puede instanciar):
-----------------------------------------
public abstract class Figura {
    protected String color;

    public abstract double area();   // Método abstracto

    public String getColor() {
        return color;
    }
}

public class Circulo extends Figura {
    private double radio;

    public Circulo(double radio, String color) {
        this.radio = radio;
        this.color = color;
    }

    @Override
    public double area() {
        return Math.PI * radio * radio;
    }
}

CAPITULO 9: MANEJO DE EXCEPCIONES (TRY-CATCH)
================================================================================

Try-catch:
----------
try {
    int resultado = 10 / 0;   // ArithmeticException
} catch (ArithmeticException e) {
    System.out.println("División por cero: " + e.getMessage());
} catch (Exception e) {       // Captura cualquier otra excepción
    System.out.println("Error general: " + e);
} finally {
    System.out.println("Siempre se ejecuta");
}

Lanzar excepciones (throw):
---------------------------
public void validarEdad(int edad) throws IllegalArgumentException {
    if (edad < 0) {
        throw new IllegalArgumentException("Edad negativa no permitida");
    }
}

Excepciones personalizadas:
---------------------------
public class MiExcepcion extends Exception {
    public MiExcepcion(String mensaje) {
        super(mensaje);
    }
}

// Uso:
throw new MiExcepcion("Error personalizado");

Try-with-resources (cierre automático de recursos):
---------------------------------------------------
try (BufferedReader br = new BufferedReader(new FileReader("archivo.txt"))) {
    String linea;
    while ((linea = br.readLine()) != null) {
        System.out.println(linea);
    }
} catch (IOException e) {
    System.out.println("Error: " + e.getMessage());
}

CAPITULO 10: ENTRADA/SALIDA DE ARCHIVOS

Leer archivo de texto:
----------------------
import java.io.*;

try (BufferedReader br = new BufferedReader(new FileReader("archivo.txt"))) {
    String linea;
    while ((linea = br.readLine()) != null) {
        System.out.println(linea);
    }

    e.printStackTrace();
}

// Con Scanner:
try (Scanner sc = new Scanner(new File("archivo.txt"))) {
    while (sc.hasNextLine()) {
        System.out.println(sc.nextLine());
    }
} catch (FileNotFoundException e) {
    e.printStackTrace();
}

Escribir archivo:
-----------------
try (BufferedWriter bw = new BufferedWriter(new FileWriter("salida.txt"))) {
    bw.write("Línea 1");
    bw.newLine();
    bw.write("Línea 2");
} catch (IOException e) {
    e.printStackTrace();
}

// Añadir al final (append):
try (BufferedWriter bw = new BufferedWriter(new FileWriter("log.txt", true))) {
    bw.write("Nueva entrada");
    bw.newLine();
}

Leer/Escribir objetos (serialización):
--------------------------------------
// Guardar objeto:
try (ObjectOutputStream oos = new ObjectOutputStream(new FileOutputStream("obj.dat"))) {

// Leer objeto:

CAPITULO 11: HILOS Y CONCURRENCIA (THREADS)
================================================================================

Crear un hilo (heredando Thread):
--------------------------------
class MiHilo extends Thread {
    @Override
    public void run() {
        for (int i = 0; i < 5; i++) {
            System.out.println("Hilo: " + i);
            try {
                Thread.sleep(1000);  // Pausa 1 segundo
            } catch (InterruptedException e) {
                System.out.println("Interrumpido");
            }
        }
    }
}

// Uso:
MiHilo hilo = new MiHilo();
hilo.start();   // Inicia el hilo

Crear un hilo (implementando Runnable):
---------------------------------------
class MiRunnable implements Runnable {
    @Override
    public void run() {
        // código
    }
}

Thread t = new Thread(new MiRunnable());
t.start();

// Con lambda (Java 8+):
Thread t = new Thread(() -> {
    System.out.println("Hilo con lambda");
});
t.start();

Sincronización (evitar condiciones de carrera):
-----------------------------------------------
public synchronized void metodoSeguro() {
    // Solo un hilo puede ejecutar esto a la vez
}

// Bloque sincronizado:
synchronized (objeto) {
    // código
}

ExecutorService (mejor práctica para manejar hilos):
----------------------------------------------------
import java.util.concurrent.*;
ExecutorService executor = Executors.newFixedThreadPool(5);
executor.submit(() -> System.out.println("Tarea 1"));
executor.submit(() -> System.out.println("Tarea 2"));
executor.shutdown();   // Terminar cuando terminen las tareas

JDBC (Java Database Connectivity) es la API para conectar con bases de datos.

Agregar dependencia (Maven - MySQL):
------------------------------------
<dependency>
    <groupId>com.mysql</groupId>
    <artifactId>mysql-connector-j</artifactId>
    <version>8.2.0</version>
</dependency>

Conectar y consultar:
---------------------
import java.sql.*;

String url = "jdbc:mysql://localhost:3306/mi_bd";
String user = "root";
String password = "1234";

try (Connection conn = DriverManager.getConnection(url, user, password);
     Statement stmt = conn.createStatement();
     ResultSet rs = stmt.executeQuery("SELECT * FROM usuarios")) {

    while (rs.next()) {
        int id = rs.getInt("id");
        String nombre = rs.getString("nombre");
        int edad = rs.getInt("edad");
        System.out.println(id + " - " + nombre + " - " + edad);
    }
} catch (SQLException e) {
    e.printStackTrace();
}

PreparedStatement (prevención de inyección SQL):
------------------------------------------------
String sql = "INSERT INTO usuarios (nombre, email, edad) VALUES (?, ?, ?)";
try (PreparedStatement pstmt = conn.prepareStatement(sql)) {
    pstmt.setString(1, "Carlos");
    pstmt.setString(2, "carlos@mail.com");
    pstmt.setInt(3, 30);
    int filas = pstmt.executeUpdate();
    System.out.println("Filas insertadas: " + filas);
}

// UPDATE:
String sqlUpdate = "UPDATE usuarios SET edad = ? WHERE id = ?";
// DELETE:
String sqlDelete = "DELETE FROM usuarios WHERE id = ?";

JavaFX (moderno, reemplaza a Swing):

Ejemplo básico JavaFX:
----------------------
import javafx.application.Application;
import javafx.scene.Scene;
import javafx.scene.control.Button;
import javafx.scene.layout.StackPane;
import javafx.stage.Stage;

public class HolaJavaFX extends Application {
    @Override
    public void start(Stage primaryStage) {
        Button btn = new Button("Haz clic");
        btn.setOnAction(e -> System.out.println("¡Clic!"));

        StackPane root = new StackPane();
        root.getChildren().add(btn);

        Scene scene = new Scene(root, 300, 250);
        primaryStage.setTitle("Mi App");
        primaryStage.setScene(scene);
        primaryStage.show();
    }

    public static void main(String[] args) {
        launch(args);
    }
}

Swing (clásico, más simple para empezar):
-----------------------------------------
import javax.swing.*;
import java.awt.*;

public class MiSwingApp {
    public static void main(String[] args) {
        JFrame frame = new JFrame("Mi Ventana");
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setSize(300, 200);

        JButton button = new JButton("Haz clic");
        button.addActionListener(e -> System.out.println("¡Clic!"));

        frame.add(button, BorderLayout.CENTER);
        frame.setVisible(true);
    }
}
