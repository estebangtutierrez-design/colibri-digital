1. INTRODUCCIÓN AL TIPO STRING
===============================================================================

El tipo string en TypeScript representa secuencias de caracteres Unicode.
Es un tipo primitivo e inmutable: cualquier operación que modifique una
cadena devuelve una nueva, dejando la original intacta.

TypeScript ofrece tres formas de declarar strings:

```typescript
// Comillas simples
let simple: string = 'Hola, mundo';

// Comillas dobles
let doble: string = "Hola, mundo";

// Template literals (backticks)
let template: string = `Hola, mundo`;

console.log(simple);  // Hola, mundo
console.log(doble);   // Hola, mundo
console.log(template); // Hola, mundo
```

Las tres formas son equivalentes en TypeScript. La elección depende del
estilo y las necesidades del proyecto. Las template literals ofrecen
funcionalidades adicionales.

---

📌 **Las cadenas son inmutables en TypeScript**

Cada operación sobre un string devuelve un nuevo string. El original
nunca se modifica.

```typescript
let saludo = "Hola";
saludo.toUpperCase();       // Devuelve "HOLA", pero no modifica saludo
console.log(saludo);        // "Hola"

saludo = saludo.toUpperCase(); // Reasignación
console.log(saludo);        // "HOLA"
```

---

🔵 **Ejemplo 1: Declaración y tipo inferido**

```typescript
let mensaje = "Esto es un string";    // type: string
let vacio = "";                       // type: string (cadena vacía)
let caracter = "a";                   // type: string (no hay tipo char)

function saludar(nombre: string): string {
  return "Hola, " + nombre;
}

console.log(saludar("Lucas"));  // "Hola, Lucas"
```

TypeScript infiere el tipo string automáticamente al asignar un valor
de texto a una variable.

🔵 **Ejemplo 2: Escape de caracteres**

```typescript
let conComillaSimple = "Dijo: 'Hola'";
let conComillaDoble = 'Ella dijo: "Hola"';
let conEscape = "Línea 1\nLínea 2";
let conTab = "Col1\tCol2";
let conBackslash = "C:\\carpeta\\archivo.txt";
let conUnicode = "\u00A9";  // ©
let conEmoji = "\u{1F989}"; // 🦉

console.log(conComillaSimple);   // Dijo: 'Hola'
console.log(conComillaDoble);    // Ella dijo: "Hola"
console.log(conEscape);          // Línea 1 (newline) Línea 2
console.log(conUnicode);         // ©
console.log(conEmoji);           // 🦉
```

🔵 **Ejemplo 3: Longitud de una cadena**

```typescript
let texto = "TypeScript Master Book";
console.log(texto.length);           // 22
console.log("".length);              // 0
console.log(" ".length);             // 1
console.log("🦉".length);            // 2 (emoji usa 2 unidades de código)
console.log(Array.from("🦉").length); // 1 (forma correcta para emojis)
```

===============================================================================
2. TEMPLATE LITERALS
===============================================================================

Los template literals (o plantillas literales) son strings delimitados por
backticks (`) que ofrecen características avanzadas como interpolación de
expresiones y strings multilínea.

---

📌 **Interpolación de expresiones con ${ }**

Puedes incrustar cualquier expresión de JavaScript dentro de ${ } en un
template literal. La expresión se evalúa y se convierte a string.

```typescript
let nombre = "Ana";
let edad = 28;
let mensaje = `Me llamo ${nombre} y tengo ${edad} años.`;
console.log(mensaje); // "Me llamo Ana y tengo 28 años."

// Expresiones complejas
let total = `Total: $${(19.99 * 1.16).toFixed(2)}`;
console.log(total);   // "Total: $23.19"
```

---

🔵 **Ejemplo 4: Strings multilínea**

```typescript
// Con comillas normales (requiere \n)
let textoNormal = "Línea 1\nLínea 2\nLínea 3";
console.log(textoNormal);

// Con template literals (multilínea natural)
let textoMultilinea = `Línea 1
Línea 2
Línea 3`;
console.log(textoMultilinea);

// Ejemplo práctico: generación de HTML
let nombre = "María";
let html = `
<div class="tarjeta">
  <h2>${nombre}</h2>
  <p>Bienvenido a TypeScript</p>
</div>
`;
console.log(html);
```

🔵 **Ejemplo 5: Templates anidados y funciones**

```typescript
function formatearUsuario(nombre: string, rol: string): string {
  return `Usuario: ${nombre} (${rol.toUpperCase()})`;
}

function generarLista(items: string[]): string {
  return `
<ul>
  ${items.map(item => `  <li>${item}</li>`).join("\n")}
</ul>`;
}

console.log(formatearUsuario("Carlos", "admin"));
// "Usuario: Carlos (ADMIN)"

console.log(generarLista(["Manzana", "Banana", "Cereza"]));
// <ul>
//   <li>Manzana</li>
//   <li>Banana</li>
//   <li>Cereza</li>
// </ul>
```

🔵 **Ejemplo 6: Tagged templates**

Los tagged templates permiten procesar un template literal con una función.

```typescript
function resaltar(strings: TemplateStringsArray, ...valores: string[]): string {
  let resultado = "";
  strings.forEach((str, i) => {
    resultado += str;
    if (i < valores.length) {
      resultado += `**${valores[i].toUpperCase()}**`;
    }
  });
  return resultado;
}

let nombre = "Lucas";
let accion = "aprender";
let mensaje = resaltar`${nombre} está ${accion} TypeScript`;
console.log(mensaje);
// "**LUCAS** está **APRENDER** TypeScript"
```

3. MÉTODOS PRINCIPALES DE STRING
===============================================================================

TypeScript hereda todos los métodos de String de JavaScript. Aquí están
los más importantes organizados por categoría.

---

📌 **Métodos de extracción y búsqueda**

```typescript
let texto = "TypeScript Master Book";

console.log(texto.charAt(0));            // "T"
console.log(texto.charCodeAt(0));        // 84 (código Unicode)
console.log(texto.indexOf("Script"));    // 4
console.log(texto.lastIndexOf("a"));     // 14
console.log(texto.includes("Master"));   // true
console.log(texto.startsWith("Type"));   // true
console.log(texto.endsWith("Book"));     // true
console.log(texto.slice(0, 4));          // "Type"
console.log(texto.substring(4, 10));     // "Script"
```

---

🔵 **Ejemplo 7: Métodos de transformación**

```typescript
let mensaje = "  Hola, TypeScript!  ";

console.log(mensaje.toLowerCase());      // "  hola, typescript!  "
console.log(mensaje.toUpperCase());      // "  HOLA, TYPESCRIPT!  "
console.log(mensaje.trim());             // "Hola, TypeScript!"
console.log(mensaje.trimStart());        // "Hola, TypeScript!  "
console.log(mensaje.trimEnd());          // "  Hola, TypeScript!"

let csv = "manzana,banana,cereza";
console.log(csv.split(","));             // ["manzana", "banana", "cereza"]

let repetido = "JS ".repeat(3);
console.log(repetido);                   // "JS JS JS "

let reemplazo = "Hola Juan".replace("Juan", "María");
console.log(reemplazo);                  // "Hola María"
```

🔵 **Ejemplo 8: Búsqueda avanzada con expresiones regulares**

```typescript
let texto = "El correo es usuario@ejemplo.com y teléfono 555-1234";

// Buscar patrón
let emailRegex = /[\w.-]+@[\w.-]+\.\w+/;
let email = texto.match(emailRegex);
console.log(email?.[0]);  // "usuario@ejemplo.com"

// Probar si existe patrón
let tieneTelefono = /\(\d{3}\)\s?\d{3}-\d{4}/.test(texto);
console.log(tieneTelefono);  // false (formato diferente)

// Reemplazar con regex
let textoLimpio = texto.replace(/\d{3}-\d{4}/g, "[OCULTO]");
console.log(textoLimpio);  // "El correo es usuario@ejemplo.com y teléfono [OCULTO]"

// Extraer todas las coincidencias
let palabras = texto.match(/\b\w+\b/g);
console.log(palabras);  // ["El", "correo", "es", "usuario", "ejemplo", "com", ...]
```

🔵 **Ejemplo 9: Padding y truncado**

```typescript
function formatearNumero(valor: number, longitud: number): string {
  return valor.toString().padStart(longitud, "0");
}

function alinearDerecha(texto: string, ancho: number): string {
  return texto.padStart(ancho);
}

console.log(formatearNumero(42, 5));       // "00042"
console.log(formatearNumero(7, 3));        // "007"
console.log(alinearDerecha("Total:", 20)); // "            Total:"
console.log("Hola".padEnd(10, "-"));       // "Hola------"
```

4. CONVERSIÓN ENTRE STRINGS Y OTROS TIPOS
===============================================================================

Es muy común necesitar convertir strings a números, booleanos u otros tipos,
y viceversa.

---

📌 **Conversión a string**

Existen varias formas de convertir cualquier valor a string en TypeScript:

```typescript
let numero = 42;
let booleano = true;
let objeto = { nombre: "Lucas" };

// Método toString()
console.log(numero.toString());      // "42"
console.log(booleano.toString());    // "true"

// String() function
console.log(String(numero));         // "42"
console.log(String(booleano));       // "true"
console.log(String(objeto));         // "[object Object]"

// Concatenación
console.log(numero + "");            // "42"

// Template literal
console.log(`${numero}`);            // "42"
```

---

🔵 **Ejemplo 10: Conversión desde string a número**

```typescript
let entrada = "42";

let conNumber = Number(entrada);       // 42
let conParseInt = parseInt(entrada);   // 42
let conParseFloat = parseFloat(entrada); // 42

// parseInt con base (radix)
let hex = parseInt("FF", 16);          // 255
let bin = parseInt("1010", 2);         // 10

// Conversión segura
function convertirANumero(texto: string): number | null {
  let resultado = Number(texto);
  if (texto.trim() === "" || Number.isNaN(resultado)) {
    return null;
  }
  return resultado;
}

console.log(convertirANumero("3.14"));  // 3.14
console.log(convertirANumero("abc"));   // null
```

🔵 **Ejemplo 11: Conversión a booleano y otros tipos**

```typescript
// String a booleano
function stringABoolean(texto: string): boolean {
  return texto.toLowerCase() === "true";
}

console.log(stringABoolean("true"));   // true
console.log(stringABoolean("false"));  // false

// Boolean a string
console.log(true.toString());          // "true"
console.log(String(false));            // "false"

// String a array de caracteres
let palabra = "Hola";
let caracteres = Array.from(palabra);
console.log(caracteres);               // ["H", "o", "l", "a"]

// String a array por delimitador
let frutas = "manzana,pera,uva";
console.log(frutas.split(","));        // ["manzana", "pera", "uva"]
```

5. UNICODE Y CARACTERES ESPECIALES
===============================================================================

TypeScript maneja Unicode de forma nativa, lo que permite trabajar con
texto en cualquier idioma y emojis.

🔵 **Ejemplo 12: Trabajando con Unicode**

```typescript
// Diferentes formas de representar caracteres Unicode
let e1 = "\u00E9";         // é (código hexadecimal 16-bit)
let e2 = "\u0065\u0301";   // e + acento agudo (combinación)
let e3 = "é";              // literal

console.log(e1);            // é
console.log(e2);            // é
console.log(e1 === e2);    // false (diferente representación)
console.log(e1 === e3);    // true
console.log(e1.length);    // 1
console.log(e2.length);    // 2

// Normalización Unicode
console.log(e1.normalize() === e2.normalize()); // true
```

🔵 **Ejemplo 13: Emojis y caracteres de 4 bytes**

```typescript
let emoji = "🦉";  // Unicode: U+1F989
console.log(emoji.length);                    // 2 (surrogate pair)
console.log(emoji.charCodeAt(0));             // 55357 (high surrogate)
console.log(emoji.charCodeAt(1));             // 56457 (low surrogate)
console.log(Array.from(emoji).length);        // 1 (forma correcta)

// Iteración correcta con for...of
let mensaje = "Hola 🦉 Mundo";
for (let char of mensaje) {
  console.log(char);  // H, o, l, a, espacio, 🦉, espacio, M, u, n, d, o
}

// Spread operator
let caracteres = [...mensaje];
console.log(caracteres); // ["H", "o", "l", "a", " ", "🦉", " ", "M", ...]
```

🔵 **Ejemplo 14: Validación de strings con Unicode**

```typescript
function esPalindromo(texto: string): boolean {
  let limpio = texto.toLowerCase().replace(/[\W_]/g, "");
  let invertido = [...limpio].reverse().join("");
  return limpio === invertido;
}

function contarPalabras(texto: string): Map<string, number> {
  let palabras = texto.toLowerCase().match(/\b\w+\b/g) || [];
  let conteo = new Map<string, number>();
  for (let palabra of palabras) {
    conteo.set(palabra, (conteo.get(palabra) || 0) + 1);
  }
  return conteo;
}

console.log(esPalindromo("Anita lava la tina"));  // true
console.log(contarPalabras("Hola mundo hola"));
// Map { "hola" => 2, "mundo" => 1 }
```

🔵 **Ejemplo 15: Comparación de strings**

```typescript
// Comparación básica (case-sensitive)
console.log("Hola" === "Hola");  // true
console.log("Hola" === "hola");  // false

// Comparación case-insensitive
function igualesIgnoreCase(a: string, b: string): boolean {
  return a.toLowerCase() === b.toLowerCase();
}

console.log(igualesIgnoreCase("TypeScript", "typescript"));  // true

// Comparación lexicográfica
console.log("a" < "b");    // true
console.log("Z" < "a");    // true (Z tiene código menor que a)
console.log("100" < "20"); // true (comparación lexicográfica, no numérica)
```

🔵 **Ejemplo 16: Construcción eficiente de strings**

```typescript
// Para concatenaciones múltiples, usa array + join
let partes: string[] = [];
for (let i = 0; i < 1000; i++) {
  partes.push(`Elemento ${i}`);
}
let resultado = partes.join("\n");

// Template literals vs concatenación
let nombre = "Lucas";
let edad = 30;

// Menos legible
let msg1 = "Me llamo " + nombre + " y tengo " + edad + " años.";

// Más legible (preferido)
let msg2 = `Me llamo ${nombre} y tengo ${edad} años.`;
```

🔵 **Ejemplo 17: Type guards con strings**

```typescript
function procesarEntrada(entrada: string | number): string {
  if (typeof entrada === "string") {
    return entrada.toUpperCase();
  }
  return entrada.toFixed(2);
}

// String literal types
type Direccion = "norte" | "sur" | "este" | "oeste";

function mover(direccion: Direccion, pasos: number): void {
  console.log(`Moviendo ${pasos} pasos al ${direccion}`);
}

mover("norte", 5);   // OK
// mover("noroeste", 3); // Error: Type '"noroeste"' is not assignable
```

RESUMEN DEL CAPÍTULO
===============================================================================

- El tipo string representa secuencias de caracteres Unicode y es inmutable.
- Existen tres formas de declarar strings: comillas simples, dobles y
  template literals (backticks).
- Los template literales permiten interpolación de expresiones con ${ },
  strings multilínea y tagged templates.
- TypeScript hereda todos los métodos de String de JavaScript: slice,
  split, trim, replace, includes, entre muchos otros.
- La conversión entre strings y otros tipos debe hacerse explícitamente
  usando String(), Number(), toString(), etc.
- Unicode se maneja de forma nativa; ten cuidado con surrogate pairs
  al medir longitud de strings con emojis.
- Para construir strings largos, prefiere arrays con join() en lugar de
  concatenación repetitiva.

PREGUNTAS FRECUENTES
===============================================================================

❓ **¿Qué tipo de comillas debería usar?**
TypeScript no impone una preferencia. La comunidad suele recomendar usar
comillas simples (') por convención, aunque muchos proyectos usan dobles.
Los template literales (backticks) son ideales cuando necesitas interpolación
o multilínea. Lo importante es ser consistente. Configura ESLint si deseas
uniformidad automática.

❓ **¿Cómo puedo saber si un string está vacío o es solo espacios?**
Usa `texto.trim() === ""` para verificar si un string es vacío o contiene
solo espacios en blanco. Para verificar si es null o undefined, combine
con una comprobación de tipo: `texto == null || texto.trim() === ""`.

❓ **¿Por qué "🦉".length es 2 en lugar de 1?**
JavaScript (y TypeScript) usa UTF-16 internamente. Los emojis y otros
caracteres fuera del plano básico Unicode (BMP) requieren dos unidades
de código (surrogate pairs) para representarse. Por eso length devuelve 2.
Usa `Array.from(str).length` o el operador spread `[...str].length` para
contar correctamente los caracteres visibles.

❓ **¿Cuál es la diferencia entre slice, substring y substr?**
slice(inicio, fin) y substring(inicio, fin) son similares, pero slice
acepta índices negativos (cuenta desde el final) mientras que substring
los trata como 0. substr (obsoleto) toma inicio y longitud. Se recomienda
usar slice por su flexibilidad y consistencia.
