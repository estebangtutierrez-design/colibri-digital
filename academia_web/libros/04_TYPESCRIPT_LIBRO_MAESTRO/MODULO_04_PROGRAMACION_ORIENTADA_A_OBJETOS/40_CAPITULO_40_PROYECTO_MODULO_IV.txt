1. VISIÓN GENERAL DEL PROYECTO
===============================================================================

Vamos a construir un **Sistema de Gestión de Biblioteca** (Library
Management System) que permita:

- Gestionar diferentes tipos de materiales (libros, revistas, DVDs).
- Administrar usuarios (estudiantes, profesores).
- Controlar préstamos y devoluciones.
- Aplicar multas por retrasos.
- Generar reportes.

La arquitectura estará dividida en capas:
1. **Modelos/Dominio**: Interfaces y clases que representan entidades.
2. **Repositorios**: Almacenamiento y recuperación de datos.
3. **Servicios**: Lógica de negocio.
4. **Utilidades**: Funciones auxiliares.

```typescript
// clases/materiales/Material.ts
export abstract class Material implements InfoMaterial, Prestable {
  public estado: EstadoMaterial = "disponible";
  constructor(
    public readonly id: string, public readonly titulo: string,
    public readonly autor: string, public readonly anioPublicacion: number,
    public readonly tipo: string
  ) {}
  abstract obtenerInfoCompleta(): string;
  prestar(): ResultadoOperacion {
    if (this.estado !== "disponible") return { exito: false, error: `"${this.titulo}" no disponible` };
    this.estado = "prestado"; return { exito: true };
  }
  devolver(): ResultadoOperacion {
    if (this.estado !== "prestado") return { exito: false, error: "No está prestado" };
    this.estado = "disponible"; return { exito: true };
  }
  renovar(): ResultadoOperacion {
    if (this.estado !== "prestado") return { exito: false, error: "Solo renovar prestados" };
    return { exito: true };
  }
  estaDisponible(): boolean { return this.estado === "disponible"; }
}

// clases/materiales/Libro.ts
export class Libro extends Material {
  constructor(id: string, titulo: string, autor: string, anio: number,
    public readonly isbn: string, public readonly genero: GeneroLibro,
    public readonly editorial: string, public readonly paginas: number) {
    super(id, titulo, autor, anio, "Libro");
  }
  obtenerInfoCompleta(): string {
    return `📖 ${this.titulo} - ${this.autor} (${this.anioPublicacion}) ISBN: ${this.isbn} [${this.estado}]`;
  }
}

// clases/materiales/Revista.ts
export class Revista extends Material {
  constructor(id: string, titulo: string, autor: string, anio: number,
    public readonly edicion: number, public readonly issn: string) {
    super(id, titulo, autor, anio, "Revista");
  }
  obtenerInfoCompleta(): string {
    return `📰 ${this.titulo} #${this.edicion} - ${this.autor} (${this.anioPublicacion}) [${this.estado}]`;
  }
}

// clases/materiales/DVD.ts
export class DVD extends Material {
  constructor(id: string, titulo: string, autor: string, anio: number,
    public readonly duracion: number, public readonly estudio: string) {
    super(id, titulo, autor, anio, "DVD");
  }
  obtenerInfoCompleta(): string {
    return `🎬 ${this.titulo} - ${this.autor} (${this.duracion}min) [${this.estado}]`;
```

```typescript
// clases/usuarios/Usuario.ts
export abstract class Usuario implements InfoUsuario, Multable {
  private _multasPendientes: number = 0;
  private _prestamosActuales: number = 0;
  public activo: boolean = true;

  constructor(
    public readonly id: string, public readonly nombre: string,
    private _email: string, public readonly tipo: TipoUsuario,
    public readonly maxPrestamosSimultaneos: number,
    public readonly diasPrestamo: number
  ) {}

  get email(): string { return this._email; }
  set email(valor: string) {
    if (!/^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(valor)) throw new Error("Email inválido");
    this._email = valor;
  }
  get prestamosActuales(): number { return this._prestamosActuales; }
  set prestamosActuales(valor: number) { this._prestamosActuales = Math.max(0, valor); }
  get multasPendientes(): number { return this._multasPendientes; }
  set multasPendientes(valor: number) { this._multasPendientes = Math.max(0, valor); }

  puedePrestar(): boolean {
    return this.activo && this._prestamosActuales < this.maxPrestamosSimultaneos && this._multasPendientes <= 0;
  }
  abstract calcularMulta(diasRetraso: number): number;
  pagarMulta(monto: number): ResultadoOperacion {
    if (monto <= 0) return { exito: false, error: "Monto debe ser positivo" };
    if (monto > this._multasPendientes) return { exito: false, error: "Excede multa pendiente" };
    this._multasPendientes -= monto;
    return { exito: true, datos: `Pagado: $${monto}` };
  }
  obtenerTotalMultas(): number { return this._multasPendientes; }
}

// clases/usuarios/Estudiante.ts
export class Estudiante extends Usuario {
  constructor(id: string, nombre: string, email: string,
    public readonly carrera: string, public readonly semestre: number) {
    super(id, nombre, email, "estudiante", 3, 7);
  }
  calcularMulta(diasRetraso: number): number { return diasRetraso * 1.0; }
}

// clases/usuarios/Profesor.ts
export class Profesor extends Usuario {
  constructor(id: string, nombre: string, email: string,
    public readonly departamento: string, public readonly especialidad: string) {
    super(id, nombre, email, "profesor", 5, 14);
  }
  calcularMulta(diasRetraso: number): number { return diasRetraso * 0.5; }
```

6. CLASE PRÉSTAMO

La clase `Prestamo` gestiona la relación entre un material y un usuario,
demostrando composición (tiene un material y un usuario).

```typescript
// clases/prestamos/Prestamo.ts
export class Prestamo implements EstadoPrestamo {
  public readonly id: string;
  public readonly fechaPrestamo: Date;
  public readonly fechaDevolucionEsperada: Date;
  public fechaDevolucionReal?: Date;
  public estaActivo: boolean = true;

  constructor(public readonly materialId: string, public readonly usuarioId: string, diasPrestamo: number) {
    this.id = crypto.randomUUID();
    this.fechaPrestamo = new Date();
    this.fechaDevolucionEsperada = new Date(this.fechaPrestamo.getTime() + diasPrestamo * 86400000);
  }

  estaVencido(): boolean { return this.estaActivo && new Date() > this.fechaDevolucionEsperada; }

  diasRetraso(): number {
    if (!this.estaActivo && this.fechaDevolucionReal) {
      const diff = this.fechaDevolucionReal.getTime() - this.fechaDevolucionEsperada.getTime();
      return Math.max(0, Math.ceil(diff / 86400000));
    }
    if (!this.estaVencido()) return 0;
    return Math.ceil((Date.now() - this.fechaDevolucionEsperada.getTime()) / 86400000);
  }

  calcularMulta(tasaPorDia: number): number { return this.diasRetraso() * tasaPorDia; }

  cerrar(): ResultadoOperacion {
    if (!this.estaActivo) return { exito: false, error: "Préstamo ya cerrado" };
    this.fechaDevolucionReal = new Date();
    this.estaActivo = false;
    return { exito: true };
  }
}
```

```typescript
// ==========================================
// repositorios/RepositorioBiblioteca.ts
// ==========================================

import { ID, ResultadoOperacion } from "../interfaces/tipos.js";

export interface Repositorio<T extends { id: ID }> {
  obtenerPorId(id: ID): T | undefined;
  obtenerTodos(): T[];
  guardar(item: T): ResultadoOperacion;
  eliminar(id: ID): ResultadoOperacion;
  contar(): number;
  buscar(criterio: (item: T) => boolean): T[];
}

export class RepositorioBiblioteca<T extends { id: ID }>
  implements Repositorio<T> {

  private readonly _items: Map<ID, T> = new Map();

  obtenerPorId(id: ID): T | undefined {
    return this._items.get(id);
  }

  obtenerTodos(): T[] {
    return Array.from(this._items.values());
  }

  guardar(item: T): ResultadoOperacion {
    if (this._items.has(item.id)) {
      return { exito: false, error: `Item ${item.id} ya existe` };
    }
    this._items.set(item.id, item);
    return { exito: true, datos: item.id };
  }

  actualizar(item: T): ResultadoOperacion {
    if (!this._items.has(item.id)) {
      return { exito: false, error: `Item ${item.id} no encontrado` };
    }
    this._items.set(item.id, item);
    return { exito: true };
  }

  eliminar(id: ID): ResultadoOperacion {
    if (!this._items.has(id)) {
      return { exito: false, error: `Item ${id} no encontrado` };
    }
    this._items.delete(id);
    return { exito: true };
  }

  contar(): number {
    return this._items.size;
  }

  buscar(criterio: (item: T) => boolean): T[] {
    return Array.from(this._items.values()).filter(criterio);
  }

  obtenerTodosOrdenados(
    comparador: (a: T, b: T) => number
  ): T[] {
    return Array.from(this._items.values()).sort(comparador);
  }
}
```

```typescript
// ==========================================

import { Prestamo } from "../clases/prestamos/Prestamo.js";
import { Usuario } from "../clases/usuarios/Usuario.js";
import { ResultadoOperacion } from "../interfaces/tipos.js";

export class ServicioMultas {
  calcularMultaPrestamo(
    prestamo: Prestamo,
    usuario: Usuario
  ): number {
    if (!prestamo.estaVencido()) return 0;
    return usuario.calcularMulta(prestamo.diasRetraso());
  }

  aplicarMulta(
    prestamo: Prestamo,
    usuario: Usuario
  ): ResultadoOperacion {
    const multa = this.calcularMultaPrestamo(prestamo, usuario);
    if (multa <= 0) {
      return { exito: true, datos: "Sin multa" };
    }
    usuario.multasPendientes += multa;
    return {
      exito: true,
      datos: `Multa de $${multa.toFixed(2)} aplicada a ${usuario.nombre}`
    };
  }

  procesarPagoMulta(
    usuario: Usuario,
    monto: number
  ): ResultadoOperacion {
    return usuario.pagarMulta(monto);
  }
}
```

```typescript
// ==========================================
// servicios/ServicioBiblioteca.ts
// ==========================================

import { RepositorioBiblioteca } from "../repositorios/RepositorioBiblioteca.js";
import { Material } from "../clases/materiales/Material.js";
import { Usuario } from "../clases/usuarios/Usuario.js";
import { Prestamo } from "../clases/prestamos/Prestamo.js";
import { ServicioMultas } from "./ServicioMultas.js";
import { ResultadoOperacion } from "../interfaces/tipos.js";

export class ServicioBiblioteca {
  private readonly materialesRepo: RepositorioBiblioteca<Material>;
  private readonly usuariosRepo: RepositorioBiblioteca<Usuario>;
  private readonly prestamosRepo: RepositorioBiblioteca<Prestamo>;
  private readonly servicioMultas: ServicioMultas;

  constructor() {
    this.materialesRepo = new RepositorioBiblioteca<Material>();
    this.usuariosRepo = new RepositorioBiblioteca<Usuario>();
    this.prestamosRepo = new RepositorioBiblioteca<Prestamo>();
    this.servicioMultas = new ServicioMultas();
  }

  // --- Gestión de materiales ---

  registrarMaterial(material: Material): ResultadoOperacion {
    return this.materialesRepo.guardar(material);
  }

  buscarMaterial(id: string): Material | undefined {
    return this.materialesRepo.obtenerPorId(id);
  }

  listarMateriales(): Material[] {
    return this.materialesRepo.obtenerTodos();
  }

  buscarMaterialesPorTitulo(texto: string): Material[] {
    const busqueda = texto.toLowerCase();
    return this.materialesRepo.buscar(
      m => m.titulo.toLowerCase().includes(busqueda)
    );
  }

  buscarDisponibles(): Material[] {
    return this.materialesRepo.buscar(m => m.estaDisponible());
  }

  // --- Gestión de usuarios ---

  registrarUsuario(usuario: Usuario): ResultadoOperacion {
    return this.usuariosRepo.guardar(usuario);
  }

  buscarUsuario(id: string): Usuario | undefined {
    return this.usuariosRepo.obtenerPorId(id);
  }

  listarUsuarios(): Usuario[] {
    return this.usuariosRepo.obtenerTodos();
  }

  // --- Gestión de préstamos ---

  prestarMaterial(
    materialId: string,
    usuarioId: string
  ): ResultadoOperacion<Prestamo> {
    const material = this.materialesRepo.obtenerPorId(materialId);
    if (!material) {
      return { exito: false, error: "Material no encontrado" };
    }

    const usuario = this.usuariosRepo.obtenerPorId(usuarioId);
    if (!usuario) {
      return { exito: false, error: "Usuario no encontrado" };
    }

    if (!usuario.puedePrestar()) {
      if (!usuario.activo) {
        return { exito: false, error: "Usuario inactivo" };
      }
      if (usuario.prestamosActuales >= usuario.maxPrestamosSimultaneos) {
        return {
          exito: false,
          error: `Límite de préstamos alcanzado (${usuario.maxPrestamosSimultaneos})`
        };
      }
      return { exito: false, error: "Usuario tiene multas pendientes" };
    }

    const prestarResult = material.prestar();
    if (!prestarResult.exito) {
      return prestarResult as ResultadoOperacion<Prestamo>;
    }

    const prestamo = new Prestamo(
      materialId, usuarioId, usuario.diasPrestamo
    );

    const guardarResult = this.prestamosRepo.guardar(prestamo);
    if (!guardarResult.exito) {
      material.devolver(); // Rollback
      return guardarResult as ResultadoOperacion<Prestamo>;
    }

    usuario.prestamosActuales++;
    return { exito: true, datos: prestamo };
  }

  devolverMaterial(
    prestamoId: string
  ): ResultadoOperacion {
    const prestamo = this.prestamosRepo.obtenerPorId(prestamoId);
    if (!prestamo) {
      return { exito: false, error: "Préstamo no encontrado" };
    }

    if (!prestamo.estaActivo) {
      return { exito: false, error: "Préstamo ya está cerrado" };
    }

    const material = this.materialesRepo.obtenerPorId(prestamo.materialId);
    const usuario = this.usuariosRepo.obtenerPorId(prestamo.usuarioId);

    if (material) {
      material.devolver();
    }

    prestamo.cerrar();

    if (usuario) {
      usuario.prestamosActuales--;
      if (prestamo.estaVencido()) {
        this.servicioMultas.aplicarMulta(prestamo, usuario);
        return {
          exito: true,
          datos: `Material devuelto con retraso. Multa aplicada.`
        };
      }
    }

    return { exito: true, datos: "Material devuelto exitosamente" };
  }

  listarPrestamosActivos(): Prestamo[] {
    return this.prestamosRepo.buscar(p => p.estaActivo);
  }

  listarPrestamosVencidos(): Prestamo[] {
    return this.prestamosRepo.buscar(p => p.estaActivo && p.estaVencido());
  }

  // --- Reportes ---

  generarReporteGeneral(): string {
    const materiales = this.materialesRepo.contar();
    const usuarios = this.usuariosRepo.contar();
    const prestamosActivos = this.listarPrestamosActivos().length;
    const prestamosVencidos = this.listarPrestamosVencidos().length;
    const disponibles = this.buscarDisponibles().length;

    return [
      "═══════════════════════════════════════",
      "    REPORTE GENERAL DE BIBLIOTECA",
      "═══════════════════════════════════════",
      `Materiales registrados: ${materiales}`,
      `Materiales disponibles: ${disponibles}`,
      `Usuarios registrados: ${usuarios}`,
      `Préstamos activos: ${prestamosActivos}`,
      `Préstamos vencidos: ${prestamosVencidos}`,
      "═══════════════════════════════════════"
    ].join("\n");
  }

  generarReporteMateriales(): string {
    const materiales = this.materialesRepo.obtenerTodos();
    const lineas = materiales.map(m => m.obtenerInfoCompleta());
    return [
      "═══════════════════════════════════════",
      "    CATÁLOGO DE MATERIALES",
      "═══════════════════════════════════════",
      ...lineas,
      `Total: ${materiales.length} materiales`
    ].join("\n\n");
  }
}
```

```typescript
  );

  biblioteca.registrarUsuario(estudiante);
  biblioteca.registrarUsuario(profesor);

  // 3. Realizar préstamos
  console.log("\n--- REALIZANDO PRÉSTAMOS ---");

  const prestamo1 = biblioteca.prestarMaterial("LIB-001", "USR-001");
  console.log("Préstamo 1:", prestamo1.exito ? "✅" : "❌", prestamo1.datos ?? prestamo1.error);

  const prestamo2 = biblioteca.prestarMaterial("DVD-001", "USR-002");
  console.log("Préstamo 2:", prestamo2.exito ? "✅" : "❌", prestamo2.datos ?? prestamo2.error);

  // Intentar prestar un material no disponible
  const prestamo3 = biblioteca.prestarMaterial("LIB-001", "USR-002");
  console.log("Préstamo 3 (debería fallar):", prestamo3.exito ? "✅" : "❌", prestamo3.error);

  // 4. Consultar disponibilidad
  console.log("\n--- MATERIALES DISPONIBLES ---");
  const disponibles = biblioteca.buscarDisponibles();
  disponibles.forEach(m =>
    console.log(`   ${m.titulo} [${m.estado}]`)
  );

  // 5. Devolver material
  console.log("\n--- DEVOLVIENDO MATERIAL ---");
  if (prestamo1.exito && prestamo1.datos) {
    const resultado = biblioteca.devolverMaterial(prestamo1.datos.id);
    console.log("Devolución:", resultado.exito ? "✅" : "❌", resultado.datos ?? resultado.error);
  }

  // 6. Generar reportes
  console.log("\n");
  console.log(biblioteca.generarReporteGeneral());

  console.log("\n");
  console.log(biblioteca.generarReporteMateriales());

  // 7. Buscar materiales
  console.log("\n--- BÚSQUEDA: 'Cien' ---");
  const resultados = biblioteca.buscarMaterialesPorTitulo("Cien");
  resultados.forEach(m => console.log(`   ${m.obtenerInfoCompleta()}`));

  // 8. Verificar préstamos vencidos
  console.log("\n--- PRÉSTAMOS VENCIDOS ---");
  const vencidos = biblioteca.listarPrestamosVencidos();
  console.log(`   ${vencidos.length} préstamos vencidos`);
}

main();
```

RESUMEN DEL CAPÍTULO
===============================================================================

- El proyecto integra todos los conceptos del Módulo IV en un sistema
  funcional de gestión de biblioteca.
- Las **interfaces** definen contratos claros entre componentes.
- La **clase abstracta Material** comparte lógica de préstamos mientras
  delega la información específica a cada subclase.
- La **herencia** crea jerarquías naturales (Libro/Revista/DVD extienden
  Material; Estudiante/Profesor extienden Usuario).
- La **encapsulación** protege el estado interno mediante propiedades
  privadas y getters/setters con validación.
- El **polimorfismo** permite tratar todos los materiales y usuarios
  de forma uniforme mediante sus interfaces.
- Los **modificadores de acceso** (public, private, protected, readonly)
  controlan la visibilidad en cada nivel.
- La **composición** se usa en el ServicioBiblioteca (tiene repositorios
  y servicio de multas) y en Prestamo (tiene referencias a material y
  usuario).
- El **repositorio genérico** demuestra el poder de los genéricos.
- El sistema es **extensible**: nuevos tipos de materiales o usuarios
  se añaden sin modificar el código existente (OCP).

PREGUNTAS FRECUENTES
===============================================================================

❓ **¿Por qué usaste composición en lugar de herencia para el ServicioBiblioteca?**
El servicio "tiene" repositorios y servicios de multas, no "es" un
repositorio. La composición es más flexible y permite cambiar
implementaciones sin afectar al servicio.

❓ **¿Por qué el repositorio es genérico?**
Para reutilizar la misma lógica de almacenamiento para materiales,
usuarios y préstamos, manteniendo el tipado fuerte en cada caso.

❓ **¿Cómo manejarías la concurrencia en un entorno real?**
Usarías bases de datos con transacciones y bloqueos. En TypeScript,
podrías usar async/await y colas de operaciones.

❓ **¿El sistema soporta múltiples bibliotecas?**
Actualmente no, pero podrías crear una clase `Biblioteca` que contenga
sus propios repositorios y servicios. Luego una clase `SistemaRed` que
gestione múltiples bibliotecas.

❓ **¿Cómo pruebo el sistema sin la implementación real?**
Las interfaces permiten crear mocks fácilmente. Por ejemplo, un
`RepositorioMock` que devuelva datos predecibles para pruebas.
