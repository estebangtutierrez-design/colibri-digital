Has dominado los bucles: while, do...while, for y foreach. Pero hay dos
palabras que pueden alterar el comportamiento de cualquier bucle: break y

sentido continuar. Continue te permite saltarte el resto de la iteración
actual y pasar a la siguiente. Son herramientas de precisión que, usadas

salir o continuar en múltiples niveles de bucles anidados. Y en PHP 8.3,
continue también acepta argumentos en match.

Prepárate para dominar el control fino de la ejecución de bucles.

===============================================================================
1. INTRODUCCIÓN A BREAK
===============================================================================

break es una instrucción que termina la ejecución del bucle o switch
actual. Cuando PHP encuentra break, sale inmediatamente de la estructura
de control más interna.

Sintaxis:

```php
<?php
break;     // Sale del bucle/switch actual
break N;   // Sale de N niveles de bucles/switch anidados
?>
```

Ejemplo básico con while:

```php
<?php
$i = 1;
while (true) {  // Bucle infinito intencional
    echo "$i ";
    if ($i >= 5) {
        echo "\n¡Break!\n";
        break;
    }
    $i++;
}
// Salida: 1 2 3 4 5
//         ¡Break!
?>
```

Ejemplo con for:

```php
<?php
for ($i = 1; $i <= 10; $i++) {
    if ($i === 7) {
        echo "Encontramos el 7. Deteniendo.\n";
        break;
    }
    echo "$i ";
}
// Salida: 1 2 3 4 5 6 Encontramos el 7. Deteniendo.
?>
```

Ejemplo con foreach:

```php
<?php
$usuarios = ["Ana", "Carlos", "María", "Juan", "Elena"];

foreach ($usuarios as $usuario) {
    echo "Buscando a Juan... ";
    if ($usuario === "Juan") {
        echo "¡Encontrado!\n";
        break;
    }
    echo "No es $usuario.\n";
}
?>
```

Break en switch:

```php
<?php
$opcion = 2;

switch ($opcion) {
    case 1:
        echo "Opción 1\n";
        break; // Sin break, continuaría al caso 2
    case 2:
        echo "Opción 2\n";
        break;
    case 3:
        echo "Opción 3\n";
        break;
    default:
        echo "Otra opción\n";
}
?>
```

Break es esencial en switch para evitar fall-through no deseado.

===============================================================================
2. BREAK CON NIVELES (BREAK N)
===============================================================================

Cuando tienes bucles anidados, break solo sale del bucle más interno. Para
salir de múltiples niveles, usa break seguido de un número entero.

Sintaxis:

```php
<?php
break 2; // Sale de 2 niveles
break 3; // Sale de 3 niveles
?>
```

Ejemplo con bucles anidados:

```php
<?php
for ($i = 1; $i <= 5; $i++) {
    echo "Fila $i: ";
    for ($j = 1; $j <= 5; $j++) {
        echo "($i,$j) ";
        if ($i === 3 && $j === 3) {
            echo "[BREAK 2]";
            break 2; // Sale de ambos bucles
        }
    }
    echo "\n"; // Esto no se ejecuta después de break 2
}
echo "\nBúsqueda completada.\n";
?>
```

Salida:
```
Fila 1: (1,1) (1,2) (1,3) (1,4) (1,5)
Fila 2: (2,1) (2,2) (2,3) (2,4) (2,5)
Fila 3: (3,1) (3,2) (3,3) [BREAK 2]
Búsqueda completada.
```

Break 2 en un switch anidado dentro de un bucle:

```php
<?php
for ($i = 1; $i <= 5; $i++) {
    echo "Iteración $i: ";

    switch ($i) {
        case 3:
            echo "Encontramos el 3. Saliendo del bucle.\n";
            break 2; // Sale del switch Y del for
        case 1:
            echo "Uno\n";
            break; // Solo sale del switch
        case 2:
            echo "Dos\n";
            break;
        default:
            echo "Otro\n";
            break;
    }
}
echo "Bucle terminado después de break 2.\n";
?>
```

Break N en estructuras complejas:

```php
<?php
$matriz = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9],
];

$objetivo = 5;
$encontrado = false;

for ($fila = 0; $fila < count($matriz); $fila++) {
    for ($col = 0; $col < count($matriz[$fila]); $col++) {
        if ($matriz[$fila][$col] === $objetivo) {
            echo "Encontrado $objetivo en posición [$fila][$col]\n";
            $encontrado = true;
            break 2; // Sale de ambos for
        }
    }
}

if (!$encontrado) {
    echo "Elemento $objetivo no encontrado.\n";
}
?>
```

El argumento de break debe ser un número entero literal mayor o igual a 1.
No puedes usar una variable.

```php
<?php
$niveles = 2;
// break $niveles; // Error de sintaxis
break 2; // Correcto
?>
```

===============================================================================
3. INTRODUCCIÓN A CONTINUE
===============================================================================

continue omite el resto de la iteración actual y salta a la siguiente
iteración del bucle. No termina el bucle, solo salta lo que queda de la
iteración actual.

Sintaxis:

```php
<?php
continue;     // Salta al final del bucle actual
continue N;   // Salta al final del N-ésimo bucle contenedor
?>
```

Ejemplo básico: mostrar solo pares

```php
<?php
for ($i = 1; $i <= 10; $i++) {
    if ($i % 2 !== 0) {
        continue; // Salta impares
    }
    echo "$i ";
}
// Salida: 2 4 6 8 10
?>
```

Continue con while:

```php
<?php
$i = 0;
while ($i < 10) {
    $i++;
    if ($i % 3 === 0) {
        continue; // Salta múltiplos de 3
    }
    echo "$i ";
}
// Salida: 1 2 4 5 7 8 10
?>
```

IMPORTANTE: En while, continue vuelve a la condición. Si la variable
de condición se actualiza DESPUÉS de continue, el bucle puede volverse
infinito. Por eso, en while, actualiza la variable ANTES de continue.

```php
<?php
// CORRECTO: actualizar antes de continue
$i = 0;
while ($i < 10) {
    $i++;
    if ($i % 2 === 0) {
        continue; // $i ya se incrementó
    }
    echo "$i ";
}

// INCORRECTO: bucle infinito
$i = 0;
while ($i < 10) {
    if ($i % 2 === 0) {
        continue; // ¡$i nunca se incrementa!
    }
    echo "$i ";
    $i++;
}
?>
```

En for, continue ejecuta automáticamente la expresión de incremento, lo
que lo hace más seguro:

```php
<?php
for ($i = 1; $i <= 10; $i++) {
    if ($i % 2 === 0) {
        continue; // $i++ se ejecuta automáticamente
    }
    echo "$i ";
}
?>
```

Continue en foreach:

```php
<?php
$usuarios = [
    ["nombre" => "Ana", "activo" => true],
    ["nombre" => "Carlos", "activo" => false],
    ["nombre" => "María", "activo" => true],
    ["nombre" => "Juan", "activo" => false],
];

foreach ($usuarios as $usuario) {
    if (!$usuario["activo"]) {
        continue; // Salta usuarios inactivos
    }
    echo "{$usuario['nombre']} está activo.\n";
}
?>
```

Continue también puede usarse en switch dentro de bucles (aunque break
es más común):

```php
<?php
for ($i = 1; $i <= 10; $i++) {
    switch ($i) {
        case 3:
        case 6:
        case 9:
            continue 2; // Salta al incremento del for
        default:
            echo "$i ";
    }
}
// Salida: 1 2 4 5 7 8 10
?>
```

===============================================================================
4. CONTINUE CON NIVELES (CONTINUE N)
===============================================================================

Al igual que break, continue puede ir seguido de un número para indicar
cuántos niveles de bucles anidados debe saltar.

Ejemplo: matriz de números, saltar filas que contienen un cero

```php
<?php
$matriz = [
    [1, 2, 3],
    [4, 0, 6],
    [7, 8, 9],
    [0, 2, 4],
];

echo "Filas sin ceros:\n";
for ($fila = 0; $fila < count($matriz); $fila++) {
    for ($col = 0; $col < count($matriz[$fila]); $col++) {
        if ($matriz[$fila][$col] === 0) {
            echo "  Fila $fila contiene un cero. Saltando.\n";
            continue 2; // Salta al siguiente for exterior
        }
    }
    echo "  Fila $fila: " . implode(", ", $matriz[$fila]) . "\n";
}
?>
```

Continue 2 en procesamiento de datos:

```php
<?php
$pedidos = [
    ["id" => 1, "items" => ["laptop", "mouse"], "estado" => "completado"],
    ["id" => 2, "items" => ["teclado"], "estado" => "cancelado"],
    ["id" => 3, "items" => ["monitor", "cable"], "estado" => "completado"],
    ["id" => 4, "items" => [], "estado" => "pendiente"],
];

foreach ($pedidos as $pedido) {
    if ($pedido["estado"] !== "completado") {
        continue; // Salta pedidos no completados
    }

    if (empty($pedido["items"])) {
        echo "Pedido #{$pedido['id']} no tiene items. Saltando.\n";
        continue;
    }

    echo "Pedido #{$pedido['id']}:\n";
    foreach ($pedido["items"] as $item) {
        if ($item === "cable") {
            echo "  Item '$item' no disponible. Saltando resto del pedido.\n";
            continue 2; // Salta al siguiente pedido
        }
        echo "  - $item\n";
    }
}
?>
```

Al igual que break, continue N debe usar un número literal, no una variable.

===============================================================================
5. BREAK Y CONTINUE EN DIFERENTES BUCLES
===============================================================================

Cada tipo de bucle maneja break y continue con matices diferentes.

En while:

```php
<?php
$i = 0;
while ($i < 10) {
    $i++;
    if ($i === 5) {
        continue; // Vuelve a evaluar $i < 10
    }
    if ($i === 8) {
        break; // Sale del bucle
    }
    echo "$i ";
}
// Salida: 1 2 3 4 6 7
// (5 se salta con continue, 8 termina con break)
?>
```

En do...while:

```php
<?php
$i = 0;
do {
    $i++;
    if ($i === 3) continue;
    if ($i === 6) break;
    echo "$i ";
} while ($i < 10);
// Salida: 1 2 4 5
?>
```

En for:

```php
<?php
for ($i = 1; $i <= 10; $i++) {
    if ($i % 2 === 0) continue; // Incremento automático
    if ($i === 7) break;
    echo "$i ";
}
// Salida: 1 3 5
?>
```

En foreach:

```php
<?php
$datos = [1, 2, 3, 4, 5, 6];
foreach ($datos as $valor) {
    if ($valor === 3) continue;
    if ($valor === 5) break;
    echo "$valor ";
}
// Salida: 1 2 4
?>
```

Diferencia clave: en while, continue vuelve a la condición. En for,
continue ejecuta el incremento y luego evalúa la condición. En foreach,
continue pasa al siguiente elemento.

6. PATRONES COMUNES CON BREAK Y CONTINUE
===============================================================================

Búsqueda con break:

```php
<?php
function encontrarPrimerPar(array $numeros): ?int {
    foreach ($numeros as $numero) {
        if ($numero % 2 === 0) {
            return $numero; // return también sale del bucle
        }
    }
    return null;
}

$nums = [1, 3, 5, 8, 9, 10];
$resultado = encontrarPrimerPar($nums);
echo "Primer par: " . ($resultado ?? "ninguno") . "\n";
?>
```

Validación con continue:

```php
<?php
function validarUsuarios(array $usuarios): array {
    $validos = [];
    foreach ($usuarios as $usuario) {
        if (empty($usuario["nombre"])) {
            echo "  Error: nombre vacío. Saltando.\n";
            continue;
        }
        if (!filter_var($usuario["email"], FILTER_VALIDATE_EMAIL)) {
            echo "  Error: email inválido para {$usuario['nombre']}. Saltando.\n";
            continue;
        }
        if ($usuario["edad"] < 18) {
            echo "  Error: {$usuario['nombre']} es menor de edad. Saltando.\n";
            continue;
        }
        $validos[] = $usuario;
    }
    return $validos;
}

$usuarios = [
    ["nombre" => "Ana", "email" => "ana@example.com", "edad" => 25],
    ["nombre" => "", "email" => "vacio", "edad" => 30],
    ["nombre" => "Carlos", "email" => "carlos@test.com", "edad" => 17],
    ["nombre" => "María", "email" => "maria@example.com", "edad" => 22],
];

echo "Validando usuarios:\n";
$validos = validarUsuarios($usuarios);
echo "\nUsuarios válidos: " . count($validos) . "\n";
?>
```

Procesamiento por lotes con break por límite:

```php
<?php
function procesarLotes(array $datos, int $tamanoLote = 3): void {
    $lote = 1;
    $contador = 0;

    foreach ($datos as $item) {
        if ($contador >= $tamanoLote) {
            echo "--- Fin del lote $lote ---\n\n";
            $lote++;
            $contador = 0;
        }
        echo "Procesando: $item\n";
        $contador++;
    }

    if ($contador > 0) {
        echo "--- Fin del lote $lote ---\n";
    }
}

$items = ["A", "B", "C", "D", "E", "F", "G", "H"];
procesarLotes($items, 3);
?>
```

Búsqueda en matriz con break 2:

```php
<?php
function buscarEnMatriz(array $matriz, mixed $objetivo): ?string {
    foreach ($matriz as $fila => $valores) {
        foreach ($valores as $col => $valor) {
            if ($valor === $objetivo) {
                return "[$fila][$col] = $valor";
            }
        }
    }
    return null;
}

$tablero = [
    ["X", "O", "X"],
    ["O", "X", "O"],
    ["O", "X", "X"],
];

echo buscarEnMatriz($tablero, "O") . "\n";
?>
```

===============================================================================
7. BREAK Y CONTINUE EN SWITCH/MATCH
===============================================================================

Break en switch es obligatorio para evitar fall-through:

```php
<?php
switch ($codigo) {
    case 200:
        echo "OK\n";
        break;
    case 404:
        echo "Not Found\n";
        break;
    case 500:
        echo "Server Error\n";
        break;
}
?>
```

Continue en switch dentro de un bucle:

```php
<?php
for ($i = 1; $i <= 10; $i++) {
    switch ($i) {
        case 2:
        case 4:
        case 6:
        case 8:
        case 10:
            echo "Par: $i\n";
            break;
        default:
            continue 2; // Salta al siguiente $i en el for
    }
    echo " (procesado)\n";
}
?>
```

Match no necesita break (es inherente):

```php
<?php
$valor = 42;
$resultado = match ($valor) {
    1 => "uno",
    2 => "dos",
    default => "otro",
};
// No hay break porque match es una expresión
?>
```

===============================================================================
8. BUENAS PRÁCTICAS Y ERRORES COMUNES
===============================================================================

Buenas prácticas:

1. Usa break/continue con moderación. Demasiados pueden hacer el código
   difícil de seguir.

2. Prefiere condiciones claras en el bucle en lugar de break infinito:

```php
<?php
// Menos claro
while (true) {
    $dato = obtenerDato();
    if ($dato === null) break;
    procesar($dato);
}

// Más claro
$dato = obtenerDato();
while ($dato !== null) {
    procesar($dato);
    $dato = obtenerDato();
}
?>
```

3. Para continue en while, actualiza la variable ANTES del continue.

4. Siempre usa unset después de foreach con referencia.

5. Para break/continue con niveles, prefiere break 2 sobre banderas:

```php
<?php
// Con banderas (menos claro)
$encontrado = false;
for ($i = 0; $i < 10 && !$encontrado; $i++) {
    for ($j = 0; $j < 10; $j++) {
        if ($matriz[$i][$j] === $objetivo) {
            $encontrado = true;
            break;
        }
    }
}

// Con break 2 (más claro)
for ($i = 0; $i < 10; $i++) {
    for ($j = 0; $j < 10; $j++) {
        if ($matriz[$i][$j] === $objetivo) {
            break 2;
        }
    }
}
?>
```

6. Documenta el uso de break/continue con niveles si no es obvio.

7. En funciones, prefiere return sobre break.

Errores comunes:

1. break en match (error de sintaxis).

2. Olvidar break en switch (fall-through no deseado).

3. Bucle infinito por continue mal ubicado en while:

```php
<?php
while ($i < 10) {
    if (condicion()) {
        continue; // ¡$i nunca se incrementa!
    }
    $i++;
}
?>
```

4. break sin llaves (break solo la primera instrucción):

```php
<?php
while ($cond) break echo "adiós"; // Error de sintaxis
?>
```

5. continue N mayor que el número de bucles anidados (error fatal).

Ejemplo 1: Procesador de archivo CSV con validación

```php
<?php
$csv = [
    ["nombre", "email", "edad"],
    ["Ana López", "ana@example.com", "28"],
    ["", "carlos@test.com", "30"],
    ["María García", "maria@example.com", "17"],
    ["Juan Pérez", "juan@example.com", "35"],
    ["Elena Ruiz", "email-invalido", "22"],
];

$encabezados = array_shift($csv);
$validos = 0;
$invalidos = 0;

foreach ($csv as $numero => $fila) {
    $nombre = $fila[0] ?? "";
    $email = $fila[1] ?? "";
    $edad = (int)($fila[2] ?? 0);

    if (empty($nombre)) {
        echo "Línea " . ($numero + 2) . ": nombre vacío. Saltando.\n";
        $invalidos++;
        continue;
    }

    if (!filter_var($email, FILTER_VALIDATE_EMAIL)) {
        echo "Línea " . ($numero + 2) . ": email inválido ('$email'). Saltando.\n";
        $invalidos++;
        continue;
    }

    if ($edad < 18) {
        echo "Línea " . ($numero + 2) . ": $nombre es menor ($edad). Saltando.\n";
        $invalidos++;
        continue;
    }

    $validos++;
    echo "✓ Línea " . ($numero + 2) . ": $nombre registrado.\n";
}

echo "\nResumen: $validos válidos, $invalidos inválidos.\n";
?>
```

Ejemplo 2: Búsqueda en estructura de árbol

```php
<?php
$arbol = [
    "nombre" => "root",
    "hijos" => [
        [
            "nombre" => "admin",
            "hijos" => [
                ["nombre" => "usuarios.php"],
                ["nombre" => "config.php"],
            ],
        ],
        [
            "nombre" => "api",
            "hijos" => [
                ["nombre" => "v1"],
                ["nombre" => "v2"],
            ],
        ],
        ["nombre" => "index.php"],
    ],
];

function buscarArchivo(array $nodo, string $objetivo, string $ruta = ""): ?string {
    $rutaActual = $ruta ? "$ruta/{$nodo['nombre']}" : $nodo['nombre'];

    if ($nodo["nombre"] === $objetivo) {
        return $rutaActual;
    }

    if (!isset($nodo["hijos"])) {
        return null;
    }

    foreach ($nodo["hijos"] as $hijo) {
        $resultado = buscarArchivo($hijo, $objetivo, $rutaActual);
        if ($resultado !== null) {
            return $resultado; // break implícito
        }
    }

    return null;
}

$buscado = "config.php";
$ruta = buscarArchivo($arbol, $buscado);
echo "Archivo '$buscado' encontrado en: " . ($ruta ?? "NO ENCONTRADO") . "\n";
?>
```

Ejemplo 3: Generación de números primos con continue

```php
<?php
function generarPrimos(int $limite): array {
    $primos = [];

    for ($num = 2; $num <= $limite; $num++) {
        if ($num > 2 && $num % 2 === 0) {
            continue; // Saltar pares mayores que 2
        }

        $esPrimo = true;
        $maxDivisor = (int)sqrt($num);

        for ($divisor = 3; $divisor <= $maxDivisor; $divisor += 2) {
            if ($num % $divisor === 0) {
                $esPrimo = false;
                break; // No es primo, salir del bucle interno
            }
        }

        if ($esPrimo) {
            $primos[] = $num;
        }
    }

    return $primos;
}

$primos = generarPrimos(100);
echo "Primos hasta 100: " . implode(", ", $primos) . "\n";
echo "Total: " . count($primos) . " primos.\n";
?>
```

Ejemplo 4: Simulador de torneo con eliminación

```php
<?php
$equipos = ["Águilas", "Búhos", "Cóndores", "Delfines", "Élites", "Fénix"];

echo "=== TORNEO ELIMINATORIO ===\n\n";
$ronda = 1;

while (count($equipos) > 1) {
    echo "--- Ronda $ronda ---\n";
    $siguienteRonda = [];

    for ($i = 0; $i < count($equipos); $i += 2) {
        if (!isset($equipos[$i + 1])) {
            // Equipo avanza sin jugar (bye)
            echo "{$equipos[$i]} avanza (bye).\n";
            $siguienteRonda[] = $equipos[$i];
            continue; // Saltar al siguiente par
        }

        $equipo1 = $equipos[$i];
        $equipo2 = $equipos[$i + 1];
        $ganador = rand(0, 1) ? $equipo1 : $equipo2;
        $perdedor = $ganador === $equipo1 ? $equipo2 : $equipo1;

        echo "$equipo1 vs $equipo2 => Ganador: $ganador\n";
        $siguienteRonda[] = $ganador;

        if ($ganador === "Fénix" || $ganador === "Élites") {
            echo "  ¡$ganador está imparable!\n";
        }
    }

    $equipos = $siguienteRonda;
    $ronda++;

    if ($ronda > 10) {
        echo "¡Límite de rondas alcanzado!\n";
        break;
    }
}

echo "\n=== CAMPEÓN: {$equipos[0]} ===\n";
?>
```

RESUMEN DEL CAPÍTULO
===============================================================================

- break termina el bucle o switch actual inmediatamente.
- break N sale de N niveles de bucles/switch anidados.
- continue omite el resto de la iteración y salta a la siguiente.
- continue N salta a la siguiente iteración del N-ésimo bucle exterior.
- En while, continue vuelve a la condición; en for, ejecuta el incremento.
- break es obligatorio en switch para evitar fall-through.
- match no necesita break (es una expresión).
- Prefiere return sobre break en funciones.
- Usa break/continue con moderación para mantener la legibilidad.

===============================================================================
PREGUNTAS FRECUENTES
===============================================================================

❓ **¿Puedo usar break fuera de un bucle o switch?** No. break solo tiene
sentido dentro de bucles (while, do...while, for, foreach) y switch. Usarlo
fuera causa un error fatal.

❓ **¿Cuál es la diferencia entre break y continue?** break termina el bucle
por completo; continue solo salta la iteración actual y continúa con la
siguiente.

❓ **¿Puedo usar continue en switch?** Sí, pero solo tiene sentido si el
switch está dentro de un bucle. continue 2 salta a la siguiente iteración
del bucle exterior.

❓ **¿Qué pasa si uso break 0 o break negativo?** Error de sintaxis. El
argumento debe ser un entero positivo.

❓ **¿Puedo usar una variable con break, como break $n?** No en versiones
anteriores a PHP 8.0. En PHP 8+ tampoco se recomienda; usa un número
literal.

❓ **¿Cuándo debería evitar continue?** Cuando hace el código más difícil de
seguir. Si tienes muchos continue en un bucle, considera refactorizar.

❓ **¿Return es mejor que break?** En funciones, return es mejor porque sale
del bucle Y de la función simultáneamente, con un valor de retorno explícito.
