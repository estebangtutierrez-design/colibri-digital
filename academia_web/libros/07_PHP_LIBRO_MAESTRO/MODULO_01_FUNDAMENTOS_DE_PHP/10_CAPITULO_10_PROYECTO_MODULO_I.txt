1. DESCRIPCIÓN DEL PROYECTO
===============================================================================

Construiremos un **Task Manager** (Gestor de Tareas) con las siguientes
características:

- **Autenticación simulada**: Login/logout con usuarios predefinidos.
- **Dashboard**: Resumen de tareas (total, completadas, pendientes).
- **CRUD de tareas**: Crear, listar, editar, eliminar y marcar como
  completada.
- **Categorías**: Las tareas pueden tener categorías (Trabajo, Personal,
  Estudio).
- **Búsqueda y filtros**: Filtrar por categoría, estado y búsqueda por
  texto.
- **Persistencia**: Los datos se almacenan en un archivo JSON.
- **Interfaz limpia**: HTML + CSS básico (sin frameworks).

**Estructura del proyecto:**

```
task-manager/
├── public/
│   └── index.php          # Punto de entrada único (front controller)
├── src/
│   ├── auth.php           # Funciones de autenticación
│   ├── tasks.php          # Funciones de gestión de tareas
│   ├── storage.php        # Funciones de almacenamiento JSON
│   └── helpers.php        # Funciones de utilería
├── views/
│   ├── login.view.php     # Pantalla de login
│   ├── dashboard.view.php # Panel principal
│   └── layout.view.php    # Plantilla base (header/footer)
├── data/
│   └── tasks.json         # Archivo de almacenamiento
└── assets/
    └── style.css          # Estilos CSS
```

declare(strict_types=1);

/**
 * Redirige a una URL y termina la ejecución.
 *
 * @param string $url URL de destino
 */
function redirect(string $url): never
{
    header("Location: $url");
    exit;
}

/**
 * Obtiene un valor seguro de $_GET o $_POST.
 *
 * @param string $key     Clave a buscar
 * @param mixed  $default Valor por defecto si no existe
 *
 * @return mixed Valor sanitizado o default
 */
function input(string $key, mixed $default = null): mixed
{
    return $_GET[$key] ?? $_POST[$key] ?? $default;
}

/**
 * Escapa HTML para prevenir XSS.
 *
 * @param string $value Texto a escapar
 *
 * @return string Texto escapado
 */
function esc(string $value): string
{
    return htmlspecialchars($value, ENT_QUOTES | ENT_SUBSTITUTE, 'UTF-8');
}

3. PASO 2: SISTEMA DE ALMACENAMIENTO (JSON)
===============================================================================

**3.1. Funciones de almacenamiento**

```php
<?php
// src/storage.php
declare(strict_types=1);

/**
 * Lee el archivo JSON y retorna su contenido como array.
 *
 * @return array<string, mixed> Datos del archivo
 */
function leerDatos(): array
{
    $archivo = __DIR__ . '/../data/tasks.json';
    if (!file_exists($archivo)) {
        return ['users' => [], 'tasks' => []];
    }
    $contenido = file_get_contents($archivo);
    if ($contenido === false) {
        throw new RuntimeException("No se pudo leer el archivo de datos");
    }
    return json_decode($contenido, true) ?? ['users' => [], 'tasks' => []];
}

/**
 * Guarda los datos en el archivo JSON.
 *
 * @param array<string, mixed> $datos Datos a guardar
 */
function guardarDatos(array $datos): void
{
    $archivo = __DIR__ . '/../data/tasks.json';
    $json = json_encode(
        $datos,
        JSON_PRETTY_PRINT | JSON_UNESCAPED_UNICODE | JSON_THROW_ON_ERROR
    );
    $bytes = file_put_contents($archivo, $json, LOCK_EX);
    if ($bytes === false) {
        throw new RuntimeException("No se pudo guardar el archivo de datos");
    }
}
```

4. PASO 3: SISTEMA DE AUTENTICACIÓN
===============================================================================

**4.1. Funciones de autenticación**

```php
<?php
// src/auth.php
declare(strict_types=1);

require_once __DIR__ . '/storage.php';

/**
 * Inicia la sesión si no está activa.
 */
function iniciarSesion(): void
{
    if (session_status() === PHP_SESSION_NONE) {
        session_start();
    }
}

/**
 * Verifica las credenciales del usuario.
 *
 * @param string $username Nombre de usuario
 * @param string $password Contraseña sin encriptar
 *
 * @return array|null Datos del usuario o null si falla
 */
function login(string $username, string $password): ?array
{
    $datos = leerDatos();

    foreach ($datos['users'] as $user) {
        if (
            $user['username'] === $username
            && password_verify($password, $user['password'])
        ) {
            $_SESSION['user_id'] = $user['id'];
            $_SESSION['user_nombre'] = $user['nombre'];
            return $user;
        }
    }
    return null;
}

/**
 * Cierra la sesión del usuario.
 */
function logout(): void
{
    iniciarSesion();
    session_destroy();
    setcookie(session_name(), '', time() - 3600, '/');
}

/**
 * Verifica si el usuario está autenticado.
 *
 * @return bool True si está autenticado
 */
function estaAutenticado(): bool
{
    iniciarSesion();
    return isset($_SESSION['user_id']);
}

/**
 * Obtiene el ID del usuario autenticado.
 *
 * @return int|null ID del usuario o null si no está autenticado
 */
function usuarioId(): ?int
{
    iniciarSesion();
    return $_SESSION['user_id'] ?? null;
}
```

5. PASO 4: GESTIÓN DE TAREAS (CRUD)
===============================================================================

**5.1. Funciones del CRUD**

```php
<?php
// src/tasks.php
declare(strict_types=1);

require_once __DIR__ . '/storage.php';

/**
 * Obtiene todas las tareas de un usuario.
 *
 * @param int      $userId    ID del usuario
 * @param ?string  $categoria Filtro por categoría (opcional)
 * @param ?bool    $completada Filtro por estado (opcional)
 * @param ?string  $busqueda  Búsqueda por texto (opcional)
 *
 * @return array<int, array<string, mixed>> Lista de tareas
 */
function obtenerTareas(
    int $userId,
    ?string $categoria = null,
    ?bool $completada = null,
    ?string $busqueda = null
): array {
    $datos = leerDatos();
    $tareas = array_filter(
        $datos['tasks'],
        fn(array $t): bool => $t['user_id'] === $userId
    );

    // Aplicar filtros
    if ($categoria !== null && $categoria !== '') {
        $tareas = array_filter(
            $tareas,
            fn(array $t): bool => $t['categoria'] === $categoria
        );
    }

    if ($completada !== null) {
        $tareas = array_filter(
            $tareas,
            fn(array $t): bool => $t['completada'] === $completada
        );
    }

    if ($busqueda !== null && $busqueda !== '') {
        $busqueda = mb_strtolower($busqueda);
        $tareas = array_filter(
            $tareas,
            fn(array $t): bool =>
                str_contains(mb_strtolower($t['titulo']), $busqueda)
                || str_contains(mb_strtolower($t['descripcion']), $busqueda)
        );
    }

    // Ordenar por fecha (más reciente primero)
    usort(
        $tareas,
        fn(array $a, array $b): int => strtotime($b['created_at'])
            <=> strtotime($a['created_at'])
    );

    return $tareas;
}

/**
 * Crea una nueva tarea.
 *
 * @param int    $userId      ID del usuario
 * @param string $titulo      Título de la tarea
 * @param string $descripcion Descripción
 * @param string $categoria   Categoría
 *
 * @return array<string, mixed> Tarea creada
 */
function crearTarea(
    int $userId,
    string $titulo,
    string $descripcion,
    string $categoria
): array {
    $datos = leerDatos();

    $maxId = 0;
    foreach ($datos['tasks'] as $tarea) {
        if ($tarea['id'] > $maxId) {
            $maxId = $tarea['id'];
        }
    }

    $tarea = [
        'id' => $maxId + 1,
        'user_id' => $userId,
        'titulo' => trim($titulo),
        'descripcion' => trim($descripcion),
        'categoria' => $categoria,
        'completada' => false,
        'created_at' => date('Y-m-d H:i:s'),
    ];

    $datos['tasks'][] = $tarea;
    guardarDatos($datos);

    return $tarea;
}

/**
 * Actualiza una tarea existente.
 *
 * @param int    $tareaId     ID de la tarea
 * @param int    $userId      ID del usuario (propietario)
 * @param string $titulo      Nuevo título
 * @param string $descripcion Nueva descripción
 * @param string $categoria   Nueva categoría
 *
 * @return bool True si se actualizó correctamente
 */
function actualizarTarea(
    int $tareaId,
    int $userId,
    string $titulo,
    string $descripcion,
    string $categoria
): bool {
    $datos = leerDatos();

    foreach ($datos['tasks'] as &$tarea) {
        if ($tarea['id'] === $tareaId && $tarea['user_id'] === $userId) {
            $tarea['titulo'] = trim($titulo);
            $tarea['descripcion'] = trim($descripcion);
            $tarea['categoria'] = $categoria;
            guardarDatos($datos);
            return true;
        }
    }
    return false;
}

/**
 * Marca una tarea como completada o pendiente.
 *
 * @param int  $tareaId    ID de la tarea
 * @param int  $userId     ID del usuario
 * @param bool $completada Nuevo estado
 *
 * @return bool True si se actualizó correctamente
 */
function toggleTarea(int $tareaId, int $userId, bool $completada): bool
{
    $datos = leerDatos();

    foreach ($datos['tasks'] as &$tarea) {
        if ($tarea['id'] === $tareaId && $tarea['user_id'] === $userId) {
            $tarea['completada'] = $completada;
            guardarDatos($datos);
            return true;
        }
    }
    return false;
}

/**
 * Elimina una tarea.
 *
 * @param int $tareaId ID de la tarea
 * @param int $userId  ID del usuario
 *
 * @return bool True si se eliminó correctamente
 */
function eliminarTarea(int $tareaId, int $userId): bool
{
    $datos = leerDatos();

    $indice = -1;
    foreach ($datos['tasks'] as $i => $tarea) {
        if ($tarea['id'] === $tareaId && $tarea['user_id'] === $userId) {
            $indice = $i;
            break;
        }
    }

    if ($indice >= 0) {
        array_splice($datos['tasks'], $indice, 1);
        guardarDatos($datos);
        return true;
    }
    return false;
}
```

6. PASO 5: PUNTO DE ENTRADA ÚNICO (FRONT CONTROLLER)
===============================================================================

**6.1. index.php — El cerebro de la aplicación**

```php
<?php
// public/index.php — Front Controller
declare(strict_types=1);

// Cargar archivos fuente
require_once __DIR__ . '/../src/helpers.php';
require_once __DIR__ . '/../src/auth.php';
require_once __DIR__ . '/../src/storage.php';
require_once __DIR__ . '/../src/tasks.php';

// Iniciar sesión y datos
iniciarSesion();
inicializarDatos();

// Determinar la acción solicitada
$accion = input('accion', 'dashboard');

// Enrutamiento básico
match ($accion) {
    'login' => manejarLogin(),
    'logout' => manejarLogout(),
    'crear' => manejarCrear(),
    'editar' => manejarEditar(),
    'toggle' => manejarToggle(),
    'eliminar' => manejarEliminar(),
    default => manejarDashboard(),
};

// ──────────────────────────────────────────────────────────────────────
// MANEJADORES DE ACCIONES
// ──────────────────────────────────────────────────────────────────────

/**
 * Muestra el formulario de login o procesa el inicio de sesión.
 */
function manejarLogin(): void
{
    if ($_SERVER['REQUEST_METHOD'] === 'POST') {
        $username = trim(input('username', ''));
        $password = input('password', '');

        if ($username === '' || $password === '') {
            $error = 'Todos los campos son obligatorios';
            renderizarVista('login', ['error' => $error]);
            return;
        }

        $usuario = login($username, $password);
        if ($usuario !== null) {
            redirect('index.php?accion=dashboard');
        }

        renderizarVista('login', [
            'error' => 'Usuario o contraseña incorrectos',
            'username' => $username,
        ]);
        return;
    }

    renderizarVista('login');
}

/**
 * Cierra la sesión y redirige al login.
 */
function manejarLogout(): void
{
    logout();
    redirect('index.php?accion=login');
}

/**
 * Muestra el dashboard con las tareas del usuario.
 */
function manejarDashboard(): void
{
    if (!estaAutenticado()) {
        redirect('index.php?accion=login');
    }

    $userId = usuarioId();
    $categoria = input('categoria');
    $estado = input('estado');
    $busqueda = input('busqueda');

    $completada = match ($estado) {
        'completadas' => true,
        'pendientes' => false,
        default => null,
    };

    $tareas = obtenerTareas($userId, $categoria, $completada, $busqueda);
    $todasTareas = obtenerTareas($userId);

    $total = count($todasTareas);
    $completadas = count(array_filter(
        $todasTareas,
        fn(array $t): bool => $t['completada']
    ));
    $pendientes = $total - $completadas;

    // Obtener categorías únicas
    $categorias = array_unique(array_map(
        fn(array $t): string => $t['categoria'],
        $todasTareas
    ));
    sort($categorias);

    renderizarVista('dashboard', [
        'tareas' => $tareas,
        'total' => $total,
        'completadas' => $completadas,
        'pendientes' => $pendientes,
        'categorias' => $categorias,
        'filtroCategoria' => $categoria,
        'filtroEstado' => $estado,
        'busqueda' => $busqueda,
    ]);
}

/**
 * Procesa la creación de una nueva tarea.
 */
function manejarCrear(): void
{
    if (!estaAutenticado()) {
        redirect('index.php?accion=login');
    }

    if ($_SERVER['REQUEST_METHOD'] !== 'POST') {
        redirect('index.php?accion=dashboard');
    }

    $userId = usuarioId();
    $titulo = trim(input('titulo', ''));
    $descripcion = trim(input('descripcion', ''));
    $categoria = input('categoria', 'Personal');

    if ($titulo === '') {
        $_SESSION['error'] = 'El título es obligatorio';
        redirect('index.php?accion=dashboard');
    }

    crearTarea($userId, $titulo, $descripcion, $categoria);
    $_SESSION['exito'] = 'Tarea creada exitosamente';
    redirect('index.php?accion=dashboard');
}

/**
 * Procesa la edición de una tarea.
 */
function manejarEditar(): void
{
    if (!estaAutenticado()) {
        redirect('index.php?accion=login');
    }

    $userId = usuarioId();
    $tareaId = (int) input('id', 0);

    if ($_SERVER['REQUEST_METHOD'] === 'POST') {
        $titulo = trim(input('titulo', ''));
        $descripcion = trim(input('descripcion', ''));
        $categoria = input('categoria', 'Personal');

        if ($titulo === '') {
            $_SESSION['error'] = 'El título es obligatorio';
            redirect("index.php?accion=editar&id=$tareaId");
        }

        if (actualizarTarea($tareaId, $userId, $titulo, $descripcion, $categoria)) {
            $_SESSION['exito'] = 'Tarea actualizada';
        } else {
            $_SESSION['error'] = 'No se pudo actualizar la tarea';
        }
        redirect('index.php?accion=dashboard');
    }

    // Mostrar formulario de edición
    $tareas = obtenerTareas($userId);
    $tarea = null;
    foreach ($tareas as $t) {
        if ($t['id'] === $tareaId) {
            $tarea = $t;
            break;
        }
    }

    if ($tarea === null) {
        $_SESSION['error'] = 'Tarea no encontrada';
        redirect('index.php?accion=dashboard');
    }

    renderizarVista('editar', ['tarea' => $tarea]);
}

/**
 * Marca/desmarca una tarea como completada.
 */
function manejarToggle(): void
{
    if (!estaAutenticado()) {
        redirect('index.php?accion=login');
    }

    $userId = usuarioId();
    $tareaId = (int) input('id', 0);
    $completada = input('estado') === '1';

    toggleTarea($tareaId, $userId, $completada);
    redirect('index.php?accion=dashboard');
}

/**
 * Elimina una tarea.
 */
function manejarEliminar(): void
{
    if (!estaAutenticado()) {
        redirect('index.php?accion=login');
    }

    $userId = usuarioId();
    $tareaId = (int) input('id', 0);

    if (eliminarTarea($tareaId, $userId)) {
        $_SESSION['exito'] = 'Tarea eliminada';
    } else {
        $_SESSION['error'] = 'No se pudo eliminar la tarea';
    }
    redirect('index.php?accion=dashboard');
}

/**
 * Renderiza una vista dentro del layout base.
 *
 * @param string $nombre Nombre del archivo de vista
 * @param array  $datos  Datos para pasar a la vista
 */
function renderizarVista(string $nombre, array $datos = []): void
{
    extract($datos);
    include __DIR__ . '/../views/layout.view.php';
}
```

===============================================================================
PREGUNTAS FRECUENTES
===============================================================================

❓ **¿Por qué usamos JSON en lugar de MySQL?**
Para mantener el proyecto autónomo y no requerir instalación de base de
datos. En el Módulo III migraremos a MySQL.

❓ **¿Es seguro almacenar contraseñas con password_hash()?**
Sí, bcrypt (el algoritmo por defecto) es seguro y resistente a ataques
de fuerza bruta.

❓ **¿Por qué el Front Controller redirige siempre a index.php?**
Es el patrón de diseño que centraliza todas las peticiones, facilitando
el mantenimiento y la seguridad.

❓ **¿Puedo agregar más funcionalidades al proyecto?**
¡Claro! Este proyecto es tu base. Puedes agregar usuarios registrados,
comentarios, fechas límite, etc.

❓ **¿Debo borrar el proyecto después de la evaluación?**
No. Este proyecto crecerá contigo a lo largo del libro. En módulos
posteriores lo migraremos a Laravel y le agregaremos funcionalidades.
