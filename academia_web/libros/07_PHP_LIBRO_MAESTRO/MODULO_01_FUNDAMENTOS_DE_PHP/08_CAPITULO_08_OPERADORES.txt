===============================================================================
1. OPERADORES ARITMÉTICOS
===============================================================================

Los operadores aritméticos realizan operaciones matemáticas básicas.

```php
<?php
$a = 10;
$b = 3;

echo $a + $b;   // 13 (suma)
echo $a - $b;   // 7 (resta)
echo $a * $b;   // 30 (multiplicación)
echo $a / $b;   // 3.3333333333333 (división float)
echo $a % $b;   // 1 (módulo: resto de la división)
echo $a ** $b;  // 1000 (potencia: 10^3, PHP 5.6+)

// Módulo con negativos
echo -10 % 3;   // -1
echo 10 % -3;   // 1

// División entera (PHP 8+)
echo intdiv(10, 3);  // 3 (división entera)
?>
```

**1.1. Precisión y redondeo en operaciones**

```php
<?php
$resultado = 10 / 3;
echo number_format($resultado, 2);          // 3.33
echo round($resultado, 2);                 // 3.33
echo ceil($resultado);                     // 4 (redondea hacia arriba)
echo floor($resultado);                    // 3 (redondea hacia abajo)
?>
```

===============================================================================
2. OPERADORES DE ASIGNACIÓN
===============================================================================

El operador básico es `=`. Existen operadores compuestos que combinan
asignación con otra operación.

**2.1. Asignación simple**

```php
<?php
$nombre = "Lucas";
$edad = 25;
$precio = 100.50;
?>
```

**2.2. Operadores de asignación compuesta**

```php
<?php
$total = 10;

$total += 5;    // $total = $total + 5;   → 15
$total -= 3;    // $total = $total - 3;   → 12
$total *= 2;    // $total = $total * 2;   → 24
$total /= 4;    // $total = $total / 4;   → 6
$total %= 4;    // $total = $total % 4;   → 2
$total **= 3;   // $total = $total ** 3;  → 8 (PHP 5.6+)

// Con cadenas
$saludo = "Hola";
$saludo .= " ";       // "Hola "
$saludo .= "Mundo";   // "Hola Mundo"

// Con operadores de bit (menos comunes)
$bits = 0b1010;
$bits &= 0b1100;      // AND bit a bit
$bits |= 0b0011;      // OR bit a bit
$bits ^= 0b1111;      // XOR bit a bit
$bits <<= 2;          // Desplazamiento izquierda
$bits >>= 1;          // Desplazamiento derecha
?>
```

===============================================================================
3. OPERADORES DE COMPARACIÓN
===============================================================================

Comparan dos valores y devuelven un booleano. PHP tiene dos niveles:
comparación laxa (==) y estricta (===).

**3.1. Tabla completa de comparación**

```php
<?php
$a = 5;
$b = "5";
$c = 10;

var_dump($a == $b);     // true  (igualdad laxa: convierte tipos)
var_dump($a === $b);    // false (identidad estricta: mismo tipo y valor)
var_dump($a != $b);     // false (diferente laxo)
var_dump($a !== $b);    // true  (diferente estricto)
var_dump($a <> $c);     // true  (diferente, igual que !=)
var_dump($a < $c);      // true  (menor que)
var_dump($a > $c);      // false (mayor que)
var_dump($a <= $c);     // true  (menor o igual)
var_dump($a >= $c);     // false (mayor o igual)
var_dump($a <=> $b);    // 0     (spaceship: -1, 0, 1)
?>
```

**3.2. Operador spaceship (<=>)** — PHP 7+

```php
<?php
// Retorna -1 si izquierda < derecha
// Retorna 0 si son iguales
// Retorna 1 si izquierda > derecha
echo 1 <=> 2;   // -1
echo 2 <=> 2;   // 0
echo 3 <=> 2;   // 1

// Útil para ordenamiento
$numeros = [3, 1, 4, 1, 5];
usort($numeros, fn($a, $b) => $a <=> $b);
print_r($numeros);  // [1, 1, 3, 4, 5]
?>
```

**3.3. Comparación de cadenas**

```php
<?php
// Comparación lexicográfica
var_dump("manzana" < "pera");   // true (m antes que p)
var_dump("PHP" === "php");      // false (case-sensitive)
var_dump(strcasecmp("PHP", "php")); // 0 (case-insensitive)

// strcmp: 0 si iguales, <0 si str1 < str2, >0 si str1 > str2
echo strcmp("a", "b");  // -1
echo strcmp("c", "a");  // 1
?>
```

===============================================================================
4. OPERADORES LÓGICOS
===============================================================================

Operan con valores booleanos y son la base de las condiciones
compuestas.

```php
<?php
$activo = true;
$admin = false;
$edad = 25;

// AND (&&) — ambas deben ser true
var_dump($activo && $admin);           // false
var_dump($activo && $edad >= 18);      // true

// OR (||) — al menos una debe ser true
var_dump($activo || $admin);           // true
var_dump(!$activo || $admin);          // false

// NOT (!) — invierte el valor
var_dump(!$activo);                    // false
var_dump(!$admin);                     // true

// XOR — una true y la otra false (excluyente)
var_dump($activo xor $admin);          // true
var_dump($activo xor true);            // false

// Versiones textuales (menor precedencia)
var_dump($activo and $admin);          // false
var_dump($activo or $admin);           // true
?>
```

**4.1. Short-circuit evaluation (evaluación de cortocircuito)**

PHP evalúa condiciones lógicas solo hasta que puede determinar el
resultado:

```php
<?php
function peligrosa(): bool {
    echo "¡Nunca me llamarán!\n";
    return true;
}

// &&: si la primera es false, no evalúa la segunda
$resultado = false && peligrosa();  // No imprime nada

// ||: si la primera es true, no evalúa la segunda
$resultado2 = true || peligrosa();  // No imprime nada

// Útil para evitar errores
$archivo = null;
if ($archivo !== null && file_exists($archivo)) {
    echo "El archivo existe";
}
// Si $archivo es null, nunca llama a file_exists
?>
```

===============================================================================
5. OPERADORES DE INCREMENTO Y DECREMENTO
===============================================================================

Son atajos para aumentar o disminuir en 1 el valor de una variable.

```php
<?php
$contador = 0;

// Pre-incremento (++$var): primero incrementa, luego usa
echo ++$contador;   // 1 (incrementa a 1, luego imprime)
echo $contador;     // 1

// Post-incremento ($var++): primero usa, luego incrementa
echo $contador++;   // 1 (imprime 1, luego incrementa a 2)
echo $contador;     // 2

// Pre-decremento
echo --$contador;   // 1 (decrementa a 1, luego imprime)

// Post-decremento
echo $contador--;   // 1 (imprime 1, luego decrementa a 0)
echo $contador;     // 0
?>
```

**5.1. Aplicación práctica en bucles**

```php
<?php
// Bucle for clásico con post-incremento
for ($i = 0; $i < 5; $i++) {
    echo $i . " ";  // 0 1 2 3 4
}

// while con pre-incremento
$j = 0;
while ($j < 5) {
    echo ++$j . " ";  // 1 2 3 4 5 (empieza en 1)
}
?>
```

===============================================================================
6. OPERADOR TERNARIO Y NULL COALESCING
===============================================================================

Son operadores condicionales que escriben condiciones en una línea.

**6.1. Operador ternario (? : )**

```php
<?php
$edad = 20;

// Sintaxis: condición ? valor_si_true : valor_si_false
$mensaje = $edad >= 18 ? "Mayor de edad" : "Menor de edad";
echo $mensaje;  // Mayor de edad

// Ternario anidado (usar con moderación)
$nota = 85;
$calificacion = $nota >= 90 ? "A" : ($nota >= 80 ? "B" : "C");
echo $calificacion;  // B
?>
```

**6.2. Null coalescing (??)** — PHP 7+

```php
<?php
// Sintaxis: $variable ?? $default
// Retorna $variable si existe y no es null, sino $default

$nombre = $_GET['nombre'] ?? 'Invitado';
echo $nombre;

// Equivalente a:
$nombre = isset($_GET['nombre']) ? $_GET['nombre'] : 'Invitado';

// Encadenamiento
$usuario = $_GET['usuario'] ?? $_POST['usuario'] ?? 'Anónimo';

// Null coalescing assignment (??=) — PHP 7.4+
$cache = [];
$cache['clave'] ??= 'valor';
// Equivalente a: if (!isset($cache['clave'])) { $cache['clave'] = 'valor'; }
?>
```

**6.3. Nullsafe operator (?->)** — PHP 8+

```php
<?php
// Sin nullsafe
$pais = null;
if ($usuario !== null) {
    if ($usuario->direccion !== null) {
        $pais = $usuario->direccion->pais;
    }
}

// Con nullsafe
$pais = $usuario?->direccion?->pais;
// Si algo es null en la cadena, $pais será null sin errores
?>
```

7. OPERADORES DE CADENA
===============================================================================

PHP tiene dos operadores específicos para cadenas:

```php
<?php
// Concatenación (.)
$nombre = "Lucas";
$apellido = "EGA";
$nombreCompleto = $nombre . " " . $apellido;
echo $nombreCompleto;  // Lucas EGA

// Asignación con concatenación (.=)
$mensaje = "Hola";
$mensaje .= ", ";
$mensaje .= "Mundo";
echo $mensaje;  // Hola, Mundo

// Concatenación en asignación compuesta
$texto = "PHP ";
$texto .= "es ";
$texto .= "increíble!";
echo $texto;  // PHP es increíble!
?>
```

===============================================================================
8. OPERADORES DE ARRAY
===============================================================================

PHP ofrece operadores específicos para trabajar con arrays:

```php
<?php
$a = ['a', 'b', 'c'];
$b = ['d', 'e', 'f'];
$c = ['a' => 1, 'b' => 2];
$d = ['b' => 3, 'c' => 4];

// Unión (+): combina, manteniendo las claves del primero
$union = $a + $b;
print_r($union);  // ['a', 'b', 'c'] (ignora $b porque $a tiene índices 0,1,2)

$union2 = $c + $d;
print_r($union2); // ['a' => 1, 'b' => 2, 'c' => 4] (b se mantiene de $c)

// Igualdad (==): mismo par clave/valor (sin importar orden)
var_dump($c == ['b' => 2, 'a' => 1]);  // true

// Identidad (===): mismo par clave/valor y mismo orden
var_dump($c === ['b' => 2, 'a' => 1]); // false (orden diferente)

// Diferencia (!=, <>)
var_dump($a != $b);  // true

// Spread operator en arrays (PHP 8.1+)
$partes1 = [1, 2, 3];
$partes2 = [4, 5, 6];
$completo = [...$partes1, ...$partes2];  // [1, 2, 3, 4, 5, 6]
?>
```

9. PRECEDENCIA DE OPERADORES
===============================================================================

La precedencia determina el orden en que se evalúan las operaciones.
Es crucial para escribir expresiones correctas.

**9.1. Tabla de precedencia (de mayor a menor)**

| Nivel | Operadores | Asociatividad |
|-------|------------|---------------|
| 1 | **clone, new, ** | Derecha |
| 2 | ++, --, ~, (int), (float), (string), (bool), (array), (object), @ | Derecha |
| 3 | instanceof | Ninguna |
| 4 | ! | Derecha |
| 5 | *, /, % | Izquierda |
| 6 | +, -, . | Izquierda |
| 7 | <<, >> | Izquierda |
| 8 | <, <=, >, >= | Ninguna |
| 9 | ==, !=, ===, !==, <>, <=> | Ninguna |
| 10 | & | Izquierda |
| 11 | ^ | Izquierda |
| 12 | \| | Izquierda |
| 13 | && | Izquierda |
| 14 | \|\| | Izquierda |
| 15 | ?? | Derecha |
| 16 | ? : | Izquierda |
| 17 | =, +=, -=, *=, etc. | Derecha |
| 18 | and, xor, or | Izquierda |

```php
<?php
// Ejemplos de precedencia
$a = 5 + 3 * 2;      // 11 (* tiene mayor precedencia que +)
$b = (5 + 3) * 2;    // 16 (paréntesis cambia el orden)

// Asociatividad izquierda
$c = 10 - 3 - 2;     // ((10 - 3) - 2) = 5

// Asociatividad derecha (asignación)
$x = $y = $z = 5;    // $x = ($y = ($z = 5))

// REGLA DE ORO: usa paréntesis para hacer explícita la precedencia
$total = ($precio * $cantidad) + ($impuesto * $subtotal);
// Aunque sepas la precedencia, los paréntesis hacen el código legible
?>
```

10. OPERADORES DE BIT
===============================================================================

Los operadores de bit trabajan a nivel de bits. Son menos usados en
PHP web, pero importantes para ciertas optimizaciones.

**10.1. Operadores de bit disponibles**

```php
<?php
$a = 0b1100;  // 12 en decimal
$b = 0b1010;  // 10 en decimal

echo "a & b  = " . ($a & $b) . "  // AND:  " . decbin($a & $b) . "\n";  // 1000 (8)
echo "a | b  = " . ($a | $b) . "  // OR:   " . decbin($a | $b) . "\n";  // 1110 (14)
echo "a ^ b  = " . ($a ^ $b) . "  // XOR:  " . decbin($a ^ $b) . "\n";  // 0110 (6)
echo "~a     = " . (~$a & 0xF) . "  // NOT:  " . decbin(~$a & 0xF) . "\n";  // 0011 (3)
echo "a << 1 = " . ($a << 1) . " // << 1: " . decbin($a << 1) . "\n";  // 11000 (24)
echo "a >> 1 = " . ($a >> 1) . " // >> 1: " . decbin($a >> 1) . "\n";  // 0110 (6)
?>
```

**10.2. Uso práctico: permisos**

Los operadores de bit son ideales para sistemas de permisos:

```php
<?php
// Definir permisos como bits
define('PERMISO_LECTURA', 0b001);   // 1
define('PERMISO_ESCRITURA', 0b010); // 2
define('PERMISO_EJECUCION', 0b100); // 4

// Asignar permisos combinados
$permisos = PERMISO_LECTURA | PERMISO_ESCRITURA;  // 0b011 (3)

// Verificar permisos
function tienePermiso(int $permisos, int $permiso): bool {
    return ($permisos & $permiso) === $permiso;
}

echo tienePermiso($permisos, PERMISO_LECTURA) ? '✅' : '❌';  // ✅
echo tienePermiso($permisos, PERMISO_EJECUCION) ? '✅' : '❌'; // ❌
?>
```

RESUMEN DEL CAPÍTULO
===============================================================================

- Operadores aritméticos: +, -, *, /, %, **, intdiv().
- Asignación compuesta: +=, -=, *=, /=, %=, **=, .=.
- Comparación: == (laxo), === (estricto), !=, !==, <>, <, >, <=, >=, <=>.
- Spaceship (<=>) retorna -1, 0, 1: ideal para ordenamiento.
- Lógicos: &&, ||, !, xor (cortocircuito), and, or (menor precedencia).
- Incremento/decremento: ++$var (pre), $var++ (post).
- Ternario: `$cond ? $si : $no` y null coalescing `$var ?? $default`.
- Nullsafe: `$obj?->prop` evita errores en cadenas de acceso.
- Concatenación: `.` para unir cadenas, `.=` para asignar concatenando.
- Arrays: + (unión), ==, ===, !=, !==.

===============================================================================
PREGUNTAS FRECUENTES
===============================================================================

❓ **¿Cuándo debo usar == vs ===?**
Siempre que puedas, usa ===. La comparación estricta evita conversiones
sorpresivas y bugs difíciles de encontrar.

❓ **¿Por qué "and" y "or" tienen menor precedencia que && y ||?**
Por razones históricas. && y || tienen precedencia alta (como en C/C++).
and y or tienen precedencia baja, útil en asignaciones.

❓ **¿Qué hace exactamente el operador @?**
El operador de control de errores (@) suprime mensajes de error de la
expresión que le sigue. No se recomienda su uso.

❓ **¿Puedo concatenar cadenas con otros tipos?**
Sí, PHP convierte automáticamente a string. `"Número: " . 42` funciona
sin problemas.

❓ **¿El operador ** es para potencias? ¿Y si quiero raíz cuadrada?**
** es exponenciación. Para raíz cuadrada usa sqrt(). Para raíz n-ésima,
usa $numero ** (1/$n).
