1. DESCRIPCIÓN DEL PROYECTO
===============================================================================

Vas a construir un **Sistema de Gestión de Tareas y Proyectos** (TaskFlow)
desde cero, usando exclusivamente funciones (sin clases, sin POO).

El sistema debe permitir:

  • Gestionar proyectos (CRUD básico).
  • Gestionar tareas dentro de proyectos.
  • Asignar prioridades, estados y fechas límite.
  • Filtrar y buscar tareas.
  • Generar reportes y estadísticas.
  • Persistir datos en archivos JSON.
  • Interfaz de línea de comandos (CLI) o web básica.

Requisitos técnicos:

  • `declare(strict_types=1)` en todos los archivos.
  • Tipos de retorno y parámetros en TODAS las funciones.
  • PHPDoc completo en todas las funciones públicas.
  • Organización modular con include/require.
  • Funciones anónimas y arrow functions para callbacks.
  • Variables estáticas para caché y persistencia temporal.
  • Sin superglobales modificables (solo lectura de $_GET, $_POST).
  • Sin clases (reservado para Módulo IV).

**archivos.php**: funciones para leer y escribir JSON:

```php
<?php

declare(strict_types=1);

/**
 * Lee un archivo JSON y devuelve un array.
 * Usa caché estática para evitar lecturas repetidas.
 */
function leer_json(string $archivo): array
{
    static $cache = [];
    
    if (isset($cache[$archivo])) {
        return $cache[$archivo];
    }
    
    $ruta = DATA_DIR . '/' . $archivo;
    
    if (!file_exists($ruta)) {
        return [];
    }
    
    $contenido = file_get_contents($ruta);
    $datos = json_decode($contenido, true);
    
    if (json_last_error() !== JSON_ERROR_NONE) {
        throw new RuntimeException("Error al leer JSON: " . json_last_error_msg());
    }
    
    $cache[$archivo] = $datos;
    return $datos;
}

/**
 * Escribe un array en un archivo JSON.
 */
function escribir_json(string $archivo, array $datos): void
{
    $ruta = DATA_DIR . '/' . $archivo;
    $json = json_encode($datos, JSON_PRETTY_PRINT | JSON_UNESCAPED_UNICODE);
    
    if ($json === false) {
        throw new RuntimeException("Error al codificar JSON");
    }
    
    if (file_put_contents($ruta, $json) === false) {
        throw new RuntimeException("Error al escribir archivo: $archivo");
    }
}

/**
 * Limpia la caché de lectura de archivos.
 */
function limpiar_cache_archivos(): void
{
    // No podemos limpiar una static desde fuera fácilmente
    // pero podemos reinicializarla... En la práctica, el cache se limpia
    // al finalizar la petición.
}
```

**validacion.php**: funciones de validación reutilizables:

```php
<?php

declare(strict_types=1);

function validar_requerido(string $campo, mixed $valor): ?string
{
    if (empty($valor) && $valor !== '0' && $valor !== 0) {
        return "El campo '$campo' es obligatorio";
    }
    return null;
}

function validar_longitud_minima(string $campo, string $valor, int $min): ?string
{
    if (strlen(trim($valor)) < $min) {
        return "'$campo' debe tener al menos $min caracteres";
    }
    return null;
}

function validar_longitud_maxima(string $campo, string $valor, int $max): ?string
{
    if (strlen(trim($valor)) > $max) {
        return "'$campo' no debe exceder $max caracteres";
    }
    return null;
}

function validar_entero_positivo(string $campo, mixed $valor): ?string
{
    if (!is_numeric($valor) || (int) $valor <= 0) {
        return "'$campo' debe ser un número positivo";
    }
    return null;
}

function validar_fecha(string $campo, string $valor): ?string
{
    if (strtotime($valor) === false) {
        return "'$campo' no es una fecha válida";
    }
    return null;
}

function validar_opciones(string $campo, string $valor, array $opciones): ?string
{
    if (!in_array($valor, $opciones, true)) {
        $opciones_str = implode(', ', $opciones);
        return "'$campo' debe ser uno de: $opciones_str";
    }
    return null;
}

function validar_con_reglas(array $datos, array $reglas): array
{
    $errores = [];
    
    foreach ($reglas as $campo => $validadores) {
        $valor = $datos[$campo] ?? '';
        foreach ($validadores as $validador) {
            $error = $validador($valor);
            if ($error !== null) {
                if (!isset($errores[$campo])) {
                    $errores[$campo] = [];
                }
                $errores[$campo][] = $error;
            }
        }
    }
    
    return $errores;
}
```

**formateo.php**: funciones de formateo:

```php
<?php

declare(strict_types=1);

function formatear_fecha(?string $fecha): string
{
    if ($fecha === null || $fecha === '') {
        return 'Sin fecha';
    }
    
    $timestamp = strtotime($fecha);
    if ($timestamp === false) {
        return 'Fecha inválida';
    }
    
    return date('d/m/Y H:i', $timestamp);
}

function formatear_estado(string $estado): string
{
    return match ($estado) {
        'pendiente' => '⏳ Pendiente',
        'en_progreso' => '🔄 En Progreso',
        'completada' => '✅ Completada',
        'cancelada' => '❌ Cancelada',
        default => $estado,
    };
}

function formatear_prioridad(string $prioridad): string
{
    return match ($prioridad) {
        'baja' => '🟢 Baja',
        'media' => '🟡 Media',
        'alta' => '🟠 Alta',
        'urgente' => '🔴 Urgente',
        default => $prioridad,
    };
}

function formatear_fecha_limite(?string $fecha): string
{
    if ($fecha === null || $fecha === '') {
        return 'Sin fecha límite';
    }
    
    $timestamp = strtotime($fecha);
    $hoy = strtotime('today');
    $diferencia = (int) (($timestamp - $hoy) / 86400);
    
    $fecha_formateada = date('d/m/Y', $timestamp);
    
    return match (true) {
        $diferencia < 0 => "⚠️ VENCIDA ($fecha_formateada)",
        $diferencia === 0 => "🚨 HOY ($fecha_formateada)",
        $diferencia === 1 => "⏰ Mañana ($fecha_formateada)",
        $diferencia <= 7 => "📅 En $diferencia días ($fecha_formateada)",
        default => "📅 $fecha_formateada",
    };
}

function resumir_texto(string $texto, int $max = 100): string
{
    if (strlen($texto) <= $max) {
        return $texto;
    }
    
    return substr($texto, 0, $max) . '...';
}

function generar_id_unico(): string
{
    static $contador = 0;
    $contador++;
    return date('Ymd') . '-' . str_pad((string) $contador, 6, '0', STR_PAD_LEFT) . '-' . substr(uniqid(), -5);
}

function truncar_palabras(string $texto, int $max_palabras = 20): string
{
    $palabras = str_word_count($texto, 1);
    if (count($palabras) <= $max_palabras) {
        return $texto;
    }
    return implode(' ', array_slice($palabras, 0, $max_palabras)) . '...';
}
```

**respuesta.php**: funciones para generar respuesta:

```php
<?php

declare(strict_types=1);

function responder_vista(string $vista, array $datos = []): void
{
    extract($datos);
    
    // Si existe el módulo en la ruta, se incluye desde modules/
    $archivo_vista = MODULES_DIR . '/' . $vista . '.php';
    
    if (!file_exists($archivo_vista)) {
        $archivo_vista = MODULES_DIR . '/error/vistas/error.php';
        $mensaje = "Vista no encontrada: $vista";
    }
    
    include $archivo_vista;
}

function redirigir(string $url): never
{
    header("Location: $url");
    exit;
}

function responder_json(mixed $datos, int $codigo = 200): never
{
    http_response_code($codigo);
    header('Content-Type: application/json; charset=utf-8');
    echo json_encode($datos, JSON_UNESCAPED_UNICODE | JSON_PRETTY_PRINT);
    exit;
}

function obtener_parametro(string $clave, mixed $default = null): mixed
{
    return $_GET[$clave] ?? $_POST[$clave] ?? $default;
}

function es_peticion_post(): bool
{
    return ($_SERVER['REQUEST_METHOD'] ?? 'GET') === 'POST';
}

function es_peticion_ajax(): bool
{
    return isset($_SERVER['HTTP_X_REQUESTED_WITH']) &&
           strtolower($_SERVER['HTTP_X_REQUESTED_WITH']) === 'xmlhttprequest';
}
```

3.3 MÓDULO PROYECTOS

**modelo.php** — funciones CRUD para proyectos:

```php
<?php

declare(strict_types=1);

/**
 * Obtiene todos los proyectos.
 */
function obtener_proyectos(): array
{
    return leer_json('proyectos.json');
}

/**
 * Obtiene un proyecto por su ID.
 */
function obtener_proyecto_por_id(string $id): ?array
{
    $proyectos = obtener_proyectos();
    
    $filtrado = array_filter(
        $proyectos,
        fn(array $p): bool => $p['id'] === $id
    );
    
    $valores = array_values($filtrado);
    return $valores[0] ?? null;
}

/**
 * Crea un nuevo proyecto.
 */
function crear_proyecto(array $datos): array
{
    $proyectos = obtener_proyectos();
    
    $nuevo = [
        'id' => generar_id_unico(),
        'nombre' => $datos['nombre'],
        'descripcion' => $datos['descripcion'] ?? '',
        'fecha_creacion' => date('Y-m-d H:i:s'),
        'fecha_actualizacion' => date('Y-m-d H:i:s'),
        'estado' => 'activo',
    ];
    
    $proyectos[] = $nuevo;
    escribir_json('proyectos.json', $proyectos);
    
    limpiar_cache_archivos(); // Indicamos que se limpiará
    
    return $nuevo;
}

/**
 * Actualiza un proyecto existente.
 */
function actualizar_proyecto(string $id, array $datos): ?array
{
    $proyectos = obtener_proyectos();
    
    $encontrado = false;
    $proyectos = array_map(
        function(array $proyecto) use ($id, $datos, &$encontrado): array {
            if ($proyecto['id'] === $id) {
                $encontrado = true;
                return array_merge($proyecto, $datos, [
                    'fecha_actualizacion' => date('Y-m-d H:i:s'),
                ]);
            }
            return $proyecto;
        },
        $proyectos
    );
    
    if (!$encontrado) {
        return null;
    }
    
    escribir_json('proyectos.json', $proyectos);
    return obtener_proyecto_por_id($id);
}

/**
 * Elimina un proyecto y sus tareas asociadas.
 */
function eliminar_proyecto(string $id): bool
{
    $proyectos = obtener_proyectos();
    $tareas = leer_json('tareas.json');
    
    $proyecto_existia = count($proyectos) !== count(array_filter(
        $proyectos,
        fn(array $p): bool => $p['id'] !== $id
    ));
    
    $proyectos = array_values(array_filter(
        $proyectos,
        fn(array $p): bool => $p['id'] !== $id
    ));
    
    // Eliminar tareas asociadas al proyecto
    $tareas = array_values(array_filter(
        $tareas,
        fn(array $t): bool => $t['proyecto_id'] !== $id
    ));
    
    escribir_json('proyectos.json', $proyectos);
    escribir_json('tareas.json', $tareas);
    
    return $proyecto_existia;
}

/**
 * Obtiene proyectos paginados.
 */
function obtener_proyectos_paginados(int $pagina = 1, int $por_pagina = 10): array
{
    $proyectos = obtener_proyectos();
    $total = count($proyectos);
    $offset = ($pagina - 1) * $por_pagina;
    
    $paginados = array_slice($proyectos, $offset, $por_pagina);
    
    return [
        'datos' => $paginados,
        'total' => $total,
        'pagina' => $pagina,
        'por_pagina' => $por_pagina,
        'total_paginas' => (int) ceil($total / $por_pagina),
    ];
}

/**
 * Busca proyectos por término.
 */
function buscar_proyectos(string $termino): array
{
    $termino = mb_strtolower($termino);
    $proyectos = obtener_proyectos();
    
    return array_values(array_filter(
        $proyectos,
        fn(array $p): bool =>
            mb_strpos(mb_strtolower($p['nombre']), $termino) !== false ||
            mb_strpos(mb_strtolower($p['descripcion'] ?? ''), $termino) !== false
    ));
}
```

**controlador.php** — acciones del módulo proyectos:

```php
<?php

declare(strict_types=1);

function ejecutar_proyectos_listar(): void
{
    $pagina = max(1, (int) (obtener_parametro('pagina', 1)));
    $termino = obtener_parametro('buscar', '');
    
    if ($termino) {
        $proyectos = buscar_proyectos($termino);
        $resultado = [
            'datos' => $proyectos,
            'total' => count($proyectos),
            'pagina' => 1,
            'por_pagina' => count($proyectos),
            'total_paginas' => 1,
        ];
    } else {
        $resultado = obtener_proyectos_paginados($pagina, ITEMS_POR_PAGINA);
    }
    
    responder_vista('proyectos/vistas/listar', [
        'resultado' => $resultado,
        'termino' => $termino,
    ]);
}

function ejecutar_proyectos_nuevo(): void
{
    if (es_peticion_post()) {
        $datos = [
            'nombre' => obtener_parametro('nombre', ''),
            'descripcion' => obtener_parametro('descripcion', ''),
        ];
        
        $reglas = [
            'nombre' => [
                fn($v) => validar_requerido('Nombre', $v),
                fn($v) => validar_longitud_minima('Nombre', $v, 3),
                fn($v) => validar_longitud_maxima('Nombre', $v, 100),
            ],
        ];
        
        $errores = validar_con_reglas($datos, $reglas);
        
        if (empty($errores)) {
            $proyecto = crear_proyecto($datos);
            redirigir('index.php?modulo=proyectos&accion=detalle&id=' . $proyecto['id']);
            return;
        }
        
        responder_vista('proyectos/vistas/formulario', [
            'proyecto' => $datos,
            'errores' => $errores,
            'modo' => 'nuevo',
        ]);
    } else {
        responder_vista('proyectos/vistas/formulario', [
            'proyecto' => ['nombre' => '', 'descripcion' => ''],
            'errores' => [],
            'modo' => 'nuevo',
        ]);
    }
}

function ejecutar_proyectos_detalle(): void
{
    $id = obtener_parametro('id', '');
    
    if (empty($id)) {
        redirigir('index.php?modulo=error&accion=no_encontrado');
        return;
    }
    
    $proyecto = obtener_proyecto_por_id($id);
    
    if ($proyecto === null) {
        responder_vista('error/vistas/404', ['mensaje' => 'Proyecto no encontrado']);
        return;
    }
    
    // Obtener tareas del proyecto
    $tareas = obtener_tareas_por_proyecto($id);
    
    responder_vista('proyectos/vistas/detalle', [
        'proyecto' => $proyecto,
        'tareas' => $tareas,
    ]);
}

function ejecutar_proyectos_editar(): void
{
    $id = obtener_parametro('id', '');
    
    if (es_peticion_post()) {
        $datos = [
            'nombre' => obtener_parametro('nombre', ''),
            'descripcion' => obtener_parametro('descripcion', ''),
        ];
        
        $reglas = [
            'nombre' => [
                fn($v) => validar_requerido('Nombre', $v),
                fn($v) => validar_longitud_minima('Nombre', $v, 3),
                fn($v) => validar_longitud_maxima('Nombre', $v, 100),
            ],
        ];
        
        $errores = validar_con_reglas($datos, $reglas);
        
        if (empty($errores)) {
            $proyecto = actualizar_proyecto($id, $datos);
            if ($proyecto === null) {
                responder_vista('error/vistas/404', ['mensaje' => 'Proyecto no encontrado']);
                return;
            }
            redirigir('index.php?modulo=proyectos&accion=detalle&id=' . $id);
            return;
        }
        
        $proyecto = array_merge(['id' => $id], $datos);
        responder_vista('proyectos/vistas/formulario', [
            'proyecto' => $proyecto,
            'errores' => $errores,
            'modo' => 'editar',
        ]);
    } else {
        $proyecto = obtener_proyecto_por_id($id);
        if ($proyecto === null) {
            responder_vista('error/vistas/404', ['mensaje' => 'Proyecto no encontrado']);
            return;
        }
        responder_vista('proyectos/vistas/formulario', [
            'proyecto' => $proyecto,
            'errores' => [],
            'modo' => 'editar',
        ]);
    }
}

function ejecutar_proyectos_eliminar(): void
{
    $id = obtener_parametro('id', '');
    
    if (es_peticion_post()) {
        $confirmacion = obtener_parametro('confirmar', 'no');
        
        if ($confirmacion === 'si') {
            eliminar_proyecto($id);
        }
        
        redirigir('index.php?modulo=proyectos&accion=listar');
        return;
    }
    
    $proyecto = obtener_proyecto_por_id($id);
    if ($proyecto === null) {
        responder_vista('error/vistas/404', ['mensaje' => 'Proyecto no encontrado']);
        return;
    }
    
    responder_vista('proyectos/vistas/eliminar', [
        'proyecto' => $proyecto,
    ]);
}
```

```php
<?php

declare(strict_types=1);

function obtener_tareas(): array
{
    return leer_json('tareas.json');
}

function obtener_tarea_por_id(string $id): ?array
{
    $tareas = obtener_tareas();
    
    $filtradas = array_values(array_filter(
        $tareas,
        fn(array $t): bool => $t['id'] === $id
    ));
    
    return $filtradas[0] ?? null;
}

function obtener_tareas_por_proyecto(string $proyecto_id): array
{
    $tareas = obtener_tareas();
    
    return array_values(array_filter(
        $tareas,
        fn(array $t): bool => ($t['proyecto_id'] ?? '') === $proyecto_id
    ));
}

function crear_tarea(array $datos): array
{
    $tareas = obtener_tareas();
    
    $nueva = [
        'id' => generar_id_unico(),
        'proyecto_id' => $datos['proyecto_id'],
        'titulo' => $datos['titulo'],
        'descripcion' => $datos['descripcion'] ?? '',
        'estado' => $datos['estado'] ?? 'pendiente',
        'prioridad' => $datos['prioridad'] ?? 'media',
        'fecha_limite' => $datos['fecha_limite'] ?? null,
        'fecha_creacion' => date('Y-m-d H:i:s'),
        'fecha_actualizacion' => date('Y-m-d H:i:s'),
    ];
    
    $tareas[] = $nueva;
    escribir_json('tareas.json', $tareas);
    
    return $nueva;
}

function actualizar_tarea(string $id, array $datos): ?array
{
    $tareas = obtener_tareas();
    $encontrada = false;
    
    $tareas = array_map(
        function(array $tarea) use ($id, $datos, &$encontrada): array {
            if ($tarea['id'] === $id) {
                $encontrada = true;
                return array_merge($tarea, $datos, [
                    'fecha_actualizacion' => date('Y-m-d H:i:s'),
                ]);
            }
            return $tarea;
        },
        $tareas
    );
    
    if (!$encontrada) {
        return null;
    }
    
    escribir_json('tareas.json', $tareas);
    return obtener_tarea_por_id($id);
}

function eliminar_tarea(string $id): bool
{
    $tareas = obtener_tareas();
    $count_antes = count($tareas);
    
    $tareas = array_values(array_filter(
        $tareas,
        fn(array $t): bool => $t['id'] !== $id
    ));
    
    if (count($tareas) === $count_antes) {
        return false;
    }
    
    escribir_json('tareas.json', $tareas);
    return true;
}

function obtener_tareas_filtradas(array $filtros = []): array
{
    $tareas = obtener_tareas();
    
    $filtros_aplicados = array_filter($filtros, fn($v) => $v !== '' && $v !== null);
    
    if (empty($filtros_aplicados)) {
        return $tareas;
    }
    
    return array_values(array_filter(
        $tareas,
        function(array $tarea) use ($filtros_aplicados): bool {
            foreach ($filtros_aplicados as $campo => $valor) {
                if (isset($tarea[$campo]) && $tarea[$campo] !== $valor) {
                    return false;
                }
            }
            return true;
        }
    ));
}

function buscar_tareas(string $termino): array
{
    $termino = mb_strtolower($termino);
    $tareas = obtener_tareas();
    
    return array_values(array_filter(
        $tareas,
        fn(array $t): bool =>
            mb_strpos(mb_strtolower($t['titulo']), $termino) !== false ||
            mb_strpos(mb_strtolower($t['descripcion'] ?? ''), $termino) !== false
    ));
}
```

```php
<?php

declare(strict_types=1);

function obtener_estadisticas_globales(): array
{
    $proyectos = obtener_proyectos();
    $tareas = obtener_tareas();
    
    $total_proyectos = count($proyectos);
    $total_tareas = count($tareas);
    
    // Tareas por estado
    $tareas_por_estado = array_reduce(
        $tareas,
        function(array $acum, array $t) use ($tareas): array {
            $estado = $t['estado'];
            $acum[$estado] = ($acum[$estado] ?? 0) + 1;
            return $acum;
        },
        array_combine(ESTADOS_TAREA, array_fill(0, count(ESTADOS_TAREA), 0))
    );
    
    // Tareas por prioridad
    $tareas_por_prioridad = array_reduce(
        $tareas,
        function(array $acum, array $t): array {
            $p = $t['prioridad'];
            $acum[$p] = ($acum[$p] ?? 0) + 1;
            return $acum;
        },
        array_combine(PRIORIDADES_TAREA, array_fill(0, count(PRIORIDADES_TAREA), 0))
    );
    
    // Tareas vencidas
    $hoy = date('Y-m-d');
    $tareas_vencidas = count(array_filter(
        $tareas,
        fn(array $t): bool =>
            $t['fecha_limite'] !== null &&
            $t['fecha_limite'] < $hoy &&
            $t['estado'] !== 'completada' &&
            $t['estado'] !== 'cancelada'
    ));
    
    // Proyectos con más tareas (top 5)
    $tareas_por_proyecto = array_reduce(
        $tareas,
        function(array $acum, array $t): array {
            $pid = $t['proyecto_id'];
            $acum[$pid] = ($acum[$pid] ?? 0) + 1;
            return $acum;
        },
        []
    );
    
    arsort($tareas_por_proyecto);
    $top_proyectos = array_slice($tareas_por_proyecto, 0, 5, true);
    
    // Porcentaje de completitud general
    $porcentaje_completado = $total_tareas > 0
        ? round(($tareas_por_estado['completada'] / $total_tareas) * 100, 1)
        : 0;
    
    return [
        'total_proyectos' => $total_proyectos,
        'total_tareas' => $total_tareas,
        'tareas_por_estado' => $tareas_por_estado,
        'tareas_por_prioridad' => $tareas_por_prioridad,
        'tareas_vencidas' => $tareas_vencidas,
        'porcentaje_completado' => $porcentaje_completado,
        'top_proyectos' => $top_proyectos,
        'fecha_generacion' => date('Y-m-d H:i:s'),
    ];
}

function exportar_reporte_json(): string
{
    $estadisticas = obtener_estadisticas_globales();
    $proyectos = obtener_proyectos();
    $tareas = obtener_tareas();
    
    $reporte = [
        'aplicacion' => APP_NAME,
        'version' => APP_VERSION,
        'fecha' => date('Y-m-d H:i:s'),
        'resumen' => $estadisticas,
        'proyectos' => $proyectos,
        'tareas' => $tareas,
    ];
    
    return json_encode($reporte, JSON_UNESCAPED_UNICODE | JSON_PRETTY_PRINT);
}
```

9. CRITERIOS DE EVALUACIÓN
===============================================================================

Tu proyecto será evaluado en base a:

  • **Funcionalidad (30%)**: el sistema funciona correctamente, CRUD
    completo, filtros, búsqueda, paginación, reportes.

  • **Organización (25%)**: estructura de directorios clara, separación
    de responsabilidades, uso correcto de include/require, bootstrap.

  • **Calidad del código (20%)**: strict_types, tipos de retorno,
    PHPDoc, nombres descriptivos, funciones pequeñas, DRY.

  • **Uso de características del módulo (15%)**: closures, arrow
    functions, variables estáticas, funciones de orden superior.

  • **Seguridad (10%)**: lista blanca de módulos, validación de entrada,
    no hay LFI, no hay vars globales mutables, escape de salida.

DESAFÍO DEL ARQUITECTO
===============================================================================

🏗️ **Refactorización a POO + Extensiones del Sistema**

Una vez que completes TaskFlow con funciones, te desafiamos a:

  1. **Refactorización a POO**: Convierte los módulos de proyectos
     y tareas en clases. Los modelos serán clases con métodos,
     los controladores serán clases con métodos de acción.

  2. **Autenticación de usuarios**: Implementa registro, login,
     logout con sesiones y contraseñas hasheadas (password_hash).

  3. **API REST**: Agrega un punto de entrada `api.php` que permita
     acceso mediante JSON a todas las operaciones CRUD, usando
     responder_json() y validación con tokens de API.

  4. **Comentarios en tareas**: Cada tarea puede tener comentarios.
     Almacénalos en un archivo `comentarios.json` y muéstralos en
     el detalle de la tarea.

  5. **Historial de cambios**: Registra cada cambio en proyectos y
     tareas en un archivo `historial.json`. Muestra el historial
     en el detalle de cada elemento.

  6. **Estadísticas avanzadas**: Agrega al dashboard:
     - Gráfica de tendencia (tareas creadas por día/semana/mes).
     - Tiempo promedio de completitud de tareas.
     - Productividad por día de la semana.
     - Nube de etiquetas (si implementas etiquetas).

  7. **Interfaz CLI**: Crea un archivo `taskflow.php` para operar
     desde terminal:

     ```
     php taskflow.php proyecto:listar
     php taskflow.php proyecto:crear "Nombre" "Descripción"
     php taskflow.php tarea:listar --proyecto=ID
     php taskflow.php reporte:exportar --formato=json
     ```

  8. **Tests unitarios**: Escribe pruebas para las funciones más
     críticas (validación, modelo de proyectos, estadísticas)
     usando assert() o una librería simple.

Este desafío lleva TaskFlow de una aplicación funcional a un sistema
completo listo para producción, integrando TODO el conocimiento
de los Módulos I, II y III.
