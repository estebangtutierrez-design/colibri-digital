Saludos, arquitecto backend. Has construido APIs REST, manejado JSON,
implementado JWT y dominado middlewares. Ahora es momento de unir todas
esas piezas en un sistema completo de autenticación.

La autenticación es el proceso de verificar la identidad de un usuario.
En el contexto de una API REST, esto significa: recibir credenciales
(email y contraseña), validarlas contra la base de datos, generar un
token JWT, y devolverlo al cliente para que lo use en peticiones
subsiguientes.

Pero la autenticación va más allá del login básico. Un sistema empresarial
necesita: registro de usuarios, verificación de email, recuperación de
contraseña, autenticación de dos factores (2FA), detección de intrusiones,
y políticas de bloqueo por intentos fallidos.

En este capítulo construirás un sistema de autenticación completo,
profesional y seguro, listo para producción empresarial.

1. REGISTRO DE USUARIOS
===============================================================================

El registro es el primer contacto del usuario con tu sistema. Debe ser
seguro, validado y confirmado.

```php
<?php
// src/Controllers/AuthController.php
namespace App\Controllers;

use App\Auth\JwtHandler;
use App\Services\EmailService;

class AuthController
{
    private PDO $db;
    private JwtHandler $jwtHandler;
    private EmailService $emailService;

    public function __construct(PDO $db)
    {
        $this->db = $db;
        $this->jwtHandler = new JwtHandler();
        $this->emailService = new EmailService();
    }

    public function register(): void
    {
        $datos = json_decode(file_get_contents('php://input'), true);

        // Validar campos requeridos
        $reglas = [
            'nombre'   => ['required', 'string', 'min' => 2, 'max' => 100],
            'email'    => ['required', 'email', 'max' => 255],
            'password' => ['required', 'string', 'min' => 8, 'max' => 72],
            'password_confirm' => ['required']
        ];

        $errores = $this->validar($datos, $reglas);

        // Verificar que passwords coincidan
        if ($datos['password'] !== $datos['password_confirm']) {
            $errores['password_confirm'][] = 'Las contraseñas no coinciden';
        }

        if (!empty($errores)) {
            Response::error('Error de validación', 422, $errores);
            return;
        }

        // Verificar email único
        $stmt = $this->db->prepare(
            "SELECT id FROM usuarios WHERE email = ?"
        );
        $stmt->execute([$datos['email']]);

        if ($stmt->fetch()) {
            Response::error(
                'El email ya está registrado',
                422,
                ['email' => ['Este email ya está en uso']]
            );
            return;
        }

        // Generar hash de contraseña
        $passwordHash = password_hash(
            $datos['password'],
            PASSWORD_BCRYPT,
            ['cost' => 12]
        );

        // Generar token de verificación de email
        $tokenVerificacion = bin2hex(random_bytes(32));

        // Insertar usuario
        $stmt = $this->db->prepare(
            "INSERT INTO usuarios (nombre, email, password, rol, token_verificacion, created_at)
             VALUES (?, ?, ?, 'usuario', ?, NOW())"
        );
        $stmt->execute([
            $datos['nombre'],
            $datos['email'],
            $passwordHash,
            $tokenVerificacion
        ]);

        $usuarioId = (int)$this->db->lastInsertId();

        // Enviar email de verificación
        $this->emailService->enviarVerificacionEmail(
            $datos['email'],
            $datos['nombre'],
            $tokenVerificacion
        );

        Response::success(
            ['id' => $usuarioId],
            'Registro exitoso. Por favor verifica tu email.',
            201
        );
    }

    private function validar(array $datos, array $reglas): array
    {
        $errores = [];

        foreach ($reglas as $campo => $reglasCampo) {
            $valor = $datos[$campo] ?? '';

            foreach ($reglasCampo as $regla) {
                switch (true) {
                    case $regla === 'required' && empty($valor):
                        $errores[$campo][] = "El campo {$campo} es requerido";
                        break;

                    case $regla === 'email' && !empty($valor) &&
                         !filter_var($valor, FILTER_VALIDATE_EMAIL):
                        $errores[$campo][] = 'El email no es válido';
                        break;

                    case str_starts_with($regla, 'min:') && !empty($valor):
                        $min = (int)explode(':', $regla)[1];
                        if (strlen($valor) < $min) {
                            $errores[$campo][] = "Mínimo {$min} caracteres";
                        }
                        break;

                    case str_starts_with($regla, 'max:') && !empty($valor):
                        $max = (int)explode(':', $regla)[1];
                        if (strlen($valor) > $max) {
                            $errores[$campo][] = "Máximo {$max} caracteres";
                        }
                        break;
                }
            }
        }

        return $errores;
    }
}
?>
```

```php
<?php
// src/Controllers/AuthController.php (continuación)

    public function login(): void
    {
        $datos = json_decode(file_get_contents('php://input'), true);
        $email = trim($datos['email'] ?? '');
        $password = $datos['password'] ?? '';

        if (empty($email) || empty($password)) {
            Response::error('Email y contraseña son requeridos', 422);
            return;
        }

        // Buscar usuario por email
        $stmt = $this->db->prepare(
            "SELECT id, nombre, email, password, rol, email_verified_at,
                    bloqueado_hasta, intentos_fallidos
             FROM usuarios WHERE email = ?"
        );
        $stmt->execute([$email]);
        $usuario = $stmt->fetch(PDO::FETCH_ASSOC);

        // Verificar si el usuario existe
        if (!$usuario) {
            $this->registrarIntentoFallido($email);
            Response::error('Credenciales inválidas', 401);
            return;
        }

        // Verificar si la cuenta está bloqueada
        if ($usuario['bloqueado_hasta'] !== null) {
            $bloqueadoHasta = strtotime($usuario['bloqueado_hasta']);

            if (time() < $bloqueadoHasta) {
                $minutosRestantes = ceil(($bloqueadoHasta - time()) / 60);
                Response::error(
                    "Cuenta bloqueada. Intenta de nuevo en {$minutosRestantes} minutos.",
                    423
                );
                return;
            }

            // Desbloquear si ya pasó el tiempo
            $this->desbloquearCuenta($usuario['id']);
        }

        // Verificar contraseña
        if (!password_verify($password, $usuario['password'])) {
            $this->incrementarIntentosFallidos($usuario['id']);
            Response::error('Credenciales inválidas', 401);
            return;
        }

        // Verificar email verificado
        if ($usuario['email_verified_at'] === null) {
            Response::error(
                'Por favor verifica tu email antes de iniciar sesión',
                403
            );
            return;
        }

        // Reiniciar intentos fallidos
        $this->reiniciarIntentosFallidos($usuario['id']);

        // Generar tokens
        $tokenAcceso = $this->jwtHandler->generarToken([
            'id'     => $usuario['id'],
            'nombre' => $usuario['nombre'],
            'email'  => $usuario['email'],
            'rol'    => $usuario['rol']
        ]);

        $tokenRefresco = $this->jwtHandler->generarTokenRefresco(
            $usuario['id']
        );

        // Almacenar refresh token
        $stmt = $this->db->prepare(
            "INSERT INTO refresh_tokens (usuario_id, token, expires_at, created_at)
             VALUES (?, ?, DATE_ADD(NOW(), INTERVAL 30 DAY), NOW())"
        );
        $stmt->execute([$usuario['id'], $tokenRefresco]);

        // Registrar login exitoso
        $this->registrarLoginExitoso($usuario['id']);

        Response::success([
            'token_acceso'   => $tokenAcceso,
            'token_refresco'  => $tokenRefresco,
            'token_type'     => 'Bearer',
            'expires_in'     => 3600,
            'usuario'        => [
                'id'     => $usuario['id'],
                'nombre' => $usuario['nombre'],
                'email'  => $usuario['email'],
                'rol'    => $usuario['rol']
            ]
        ], 'Inicio de sesión exitoso');
    }

    private function registrarIntentoFallido(string $email): void
    {
        $stmt = $this->db->prepare(
            "UPDATE usuarios SET intentos_fallidos = intentos_fallidos + 1
             WHERE email = ?"
        );
        $stmt->execute([$email]);
    }

    private function incrementarIntentosFallidos(int $usuarioId): void
    {
        $maxIntentos = 5;

        $stmt = $this->db->prepare(
            "UPDATE usuarios
             SET intentos_fallidos = intentos_fallidos + 1
             WHERE id = ?"
        );
        $stmt->execute([$usuarioId]);

        // Bloquear si excede el máximo
        $stmt = $this->db->prepare(
            "SELECT intentos_fallidos FROM usuarios WHERE id = ?"
        );
        $stmt->execute([$usuarioId]);
        $intentos = (int)$stmt->fetchColumn();

        if ($intentos >= $maxIntentos) {
            $stmt = $this->db->prepare(
                "UPDATE usuarios
                 SET bloqueado_hasta = DATE_ADD(NOW(), INTERVAL 15 MINUTE)
                 WHERE id = ?"
            );
            $stmt->execute([$usuarioId]);
        }
    }

    private function reiniciarIntentosFallidos(int $usuarioId): void
    {
        $stmt = $this->db->prepare(
            "UPDATE usuarios SET intentos_fallidos = 0,
             bloqueado_hasta = NULL WHERE id = ?"
        );
        $stmt->execute([$usuarioId]);
    }

    private function desbloquearCuenta(int $usuarioId): void
    {
        $stmt = $this->db->prepare(
            "UPDATE usuarios SET intentos_fallidos = 0,
             bloqueado_hasta = NULL WHERE id = ?"
        );
        $stmt->execute([$usuarioId]);
    }

    private function registrarLoginExitoso(int $usuarioId): void
    {
        $stmt = $this->db->prepare(
            "UPDATE usuarios SET ultimo_login = NOW() WHERE id = ?"
        );
        $stmt->execute([$usuarioId]);
    }
?>
```

3. PROTECCIÓN DE RUTAS CON MIDDLEWARE
===============================================================================

El middleware de autenticación protege los endpoints que requieren
usuario autenticado.

```php
<?php
// src/Middleware/AuthenticationMiddleware.php
namespace App\Middleware;

use App\Auth\JwtHandler;

class AuthenticationMiddleware implements MiddlewareInterface
{
    private JwtHandler $jwtHandler;
    private array $rutasPublicas;

    public function __construct(array $rutasPublicas = [])
    {
        $this->jwtHandler = new JwtHandler();
        $this->rutasPublicas = $rutasPublicas;
    }

    public function handle(array $request, callable $next): array
    {
        // Rutas públicas no requieren autenticación
        $uri = parse_url($request['uri'], PHP_URL_PATH);

        foreach ($this->rutasPublicas as $publica) {
            if (str_starts_with($uri, $publica)) {
                return $next($request);
            }
        }

        // Extraer token del header Authorization
        $authHeader = $request['headers']['Authorization'] ?? '';

        if (!preg_match('/^Bearer\s+(.+)$/i', $authHeader, $matches)) {
            return [
                'status' => 401,
                'headers' => ['WWW-Authenticate' => 'Bearer'],
                'body'   => [
                    'success' => false,
                    'error'   => [
                        'codigo'  => 401,
                        'mensaje' => 'Token de autenticación requerido. Usa el header: Authorization: Bearer <token>'
                    ]
                ]
            ];
        }

        $token = $matches[1];

        try {
            // Verificar token en blacklist
            if ($this->estaEnBlacklist($token)) {
                throw new \RuntimeException('Token ha sido invalidado');
            }

            // Decodificar y verificar
            $usuario = $this->jwtHandler->extraerUsuarioDeToken($token);

            // Inyectar usuario en la request
            $request['usuario'] = $usuario;

            return $next($request);

        } catch (\RuntimeException $e) {
            return [
                'status' => 401,
                'headers' => ['WWW-Authenticate' => 'Bearer'],
                'body'   => [
                    'success' => false,
                    'error'   => [
                        'codigo'  => 401,
                        'mensaje' => $e->getMessage()
                    ]
                ]
            ];
        }
    }

    private function estaEnBlacklist(string $token): bool
    {
        $archivo = __DIR__ . '/../../storage/blacklist.txt';

        if (!file_exists($archivo)) {
            return false;
        }

        $handle = fopen($archivo, 'r');
        if ($handle) {
            while (($linea = fgets($handle)) !== false) {
                if (trim($linea) === $token) {
                    fclose($handle);
                    return true;
                }
            }
            fclose($handle);
        }

        return false;
    }
}
?>
```

```php
    public function verificarEmail(string $token): void
    {
        if (empty($token)) {
            Response::error('Token de verificación requerido', 422);
            return;
        }

        $stmt = $this->db->prepare(
            "SELECT id, email_verified_at
             FROM usuarios

            return;
        }

        $stmt = $this->db->prepare(
            "UPDATE usuarios
             SET email_verified_at = NOW(),
                 token_verificacion = NULL
             WHERE id = ?"
        );
        $stmt->execute([$usuario['id']]);

        Response::success(null, 'Email verificado exitosamente');
    }

    public function reenviarVerificacion(): void
    {
        $datos = json_decode(file_get_contents('php://input'), true);
        $email = trim($datos['email'] ?? '');

        if (empty($email)) {
            Response::error('Email requerido', 422);
            return;
        }

        $stmt = $this->db->prepare(
            "SELECT id, nombre, email_verified_at, token_verificacion
             FROM usuarios WHERE email = ?"
        );
        $stmt->execute([$email]);
        $usuario = $stmt->fetch(PDO::FETCH_ASSOC);

        if (!$usuario) {
            // No revelar si el email existe o no
            Response::success(
                null,
                'Si el email existe, recibirás un enlace de verificación'
            );
            return;
        }

        if ($usuario['email_verified_at'] !== null) {
            Response::success(null, 'El email ya está verificado');
            return;
        }

        // Regenerar token si es necesario
        if (empty($usuario['token_verificacion'])) {
            $nuevoToken = bin2hex(random_bytes(32));
            $stmt = $this->db->prepare(
                "UPDATE usuarios SET token_verificacion = ? WHERE id = ?"
            );
            $stmt->execute([$nuevoToken, $usuario['id']]);
        } else {
            $nuevoToken = $usuario['token_verificacion'];
        }

        // Enviar email con nuevo token
        $this->emailService->enviarVerificacionEmail(
            $usuario['email'],
            $usuario['nombre'],
            $nuevoToken
        );

        Response::success(
            null,
            'Si el email existe, recibirás un enlace de verificación'
        );
    }
```

```php
// src/Controllers/AuthController.php (continuación)

    public function solicitarRecuperacionPassword(): void
    {
        $datos = json_decode(file_get_contents('php://input'), true);
        $email = trim($datos['email'] ?? '');

        if (empty($email)) {
            Response::error('Email requerido', 422);
            return;
        }

        $stmt = $this->db->prepare(
            "SELECT id, nombre FROM usuarios WHERE email = ?"
        );
        $stmt->execute([$email]);
        $usuario = $stmt->fetch(PDO::FETCH_ASSOC);

        // No revelar si el email existe
        if (!$usuario) {
            Response::success(
                null,
                'Si el email está registrado, recibirás instrucciones'
            );
            return;
        }

        // Generar token de recuperación (válido por 1 hora)
        $token = bin2hex(random_bytes(32));
        $expiracion = date('Y-m-d H:i:s', time() + 3600);

        $stmt = $this->db->prepare(
            "UPDATE usuarios
             SET token_recuperacion = ?,
                 token_recuperacion_expira = ?
             WHERE id = ?"
        );
        $stmt->execute([$token, $expiracion, $usuario['id']]);

        // Enviar email con enlace de recuperación
        $this->emailService->enviarRecuperacionPassword(
            $email,
            $usuario['nombre'],
            $token
        );

        Response::success(
            null,
            'Si el email está registrado, recibirás instrucciones'
        );
    }

    public function restablecerPassword(): void
    {
        $datos = json_decode(file_get_contents('php://input'), true);
        $token = $datos['token'] ?? '';
        $password = $datos['password'] ?? '';
        $passwordConfirm = $datos['password_confirm'] ?? '';

        if (empty($token) || empty($password)) {
            Response::error('Token y nueva contraseña son requeridos', 422);
            return;
        }

        if ($password !== $passwordConfirm) {
            Response::error('Las contraseñas no coinciden', 422);
            return;
        }

        if (strlen($password) < 8) {
            Response::error(
                'La contraseña debe tener al menos 8 caracteres',
                422
            );
            return;
        }

        $stmt = $this->db->prepare(
            "SELECT id FROM usuarios
             WHERE token_recuperacion = ?
             AND token_recuperacion_expira > NOW()"
        );
        $stmt->execute([$token]);
        $usuario = $stmt->fetch(PDO::FETCH_ASSOC);

        if (!$usuario) {
            Response::error(
                'Token inválido o expirado. Solicita un nuevo enlace.',
                400
            );
            return;
        }

        // Actualizar contraseña
        $passwordHash = password_hash($password, PASSWORD_BCRYPT, ['cost' => 12]);

        $stmt = $this->db->prepare(
            "UPDATE usuarios
             SET password = ?,
                 token_recuperacion = NULL,
                 token_recuperacion_expira = NULL,
                 intentos_fallidos = 0,
                 bloqueado_hasta = NULL
             WHERE id = ?"
        );
        $stmt->execute([$passwordHash, $usuario['id']]);

        // Invalidar todos los refresh tokens del usuario
        $stmt = $this->db->prepare(
            "DELETE FROM refresh_tokens WHERE usuario_id = ?"
        );
        $stmt->execute([$usuario['id']]);

        Response::success(
            null,
            'Contraseña restablecida exitosamente. Ya puedes iniciar sesión.'
        );
    }
?>
```

```php
<?php
// src/Controllers/AuthController.php (continuación)

    public function logout(): void
    {
        $authHeader = $_SERVER['HTTP_AUTHORIZATION'] ?? '';

        if (!preg_match('/^Bearer\s+(.+)$/i', $authHeader, $matches)) {
            Response::error('Token requerido para cerrar sesión', 401);
            return;
        }

        $token = $matches[1];
        $usuarioId = $this->obtenerUsuarioIdDeToken($token);

        // Invalidar refresh tokens del usuario
        if ($usuarioId) {
            $stmt = $this->db->prepare(
                "DELETE FROM refresh_tokens WHERE usuario_id = ?"
            );
            $stmt->execute([$usuarioId]);
        }

        // Agregar token a blacklist
        $this->agregarABlacklist($token);

        Response::success(null, 'Sesión cerrada exitosamente');
    }

    private function obtenerUsuarioIdDeToken(string $token): ?int
    {
        try {
            $decoded = $this->jwtHandler->verificarToken($token);
            return (int)$decoded->sub;
        } catch (\Exception $e) {
            return null;
        }
    }

    private function agregarABlacklist(string $token): void
    {
        $archivo = __DIR__ . '/../../storage/blacklist.txt';
        $directorio = dirname($archivo);

        if (!is_dir($directorio)) {
            mkdir($directorio, 0775, true);
        }

        file_put_contents(
            $archivo,
            $token . "\n",
            FILE_APPEND | LOCK_EX
        );
    }
?>
```

7. REFRESCO DE TOKENS
===============================================================================

El endpoint de refresh permite obtener un nuevo token de acceso sin
pedir credenciales nuevamente.

```php
<?php
// src/Controllers/AuthController.php (continuación)

    public function refresh(): void
    {
        $datos = json_decode(file_get_contents('php://input'), true);
        $tokenRefresco = $datos['token_refresco'] ?? '';

        if (empty($tokenRefresco)) {
            Response::error('Token de refresco requerido', 422);
            return;
        }

        try {
            // Verificar el refresh token
            $decoded = $this->jwtHandler->verificarToken($tokenRefresco);

            if (!isset($decoded->type) || $decoded->type !== 'refresh') {
                throw new \RuntimeException('Tipo de token inválido');
            }

            $usuarioId = (int)$decoded->sub;

            // Verificar que el refresh token existe en BD
            $stmt = $this->db->prepare(
                "SELECT id FROM refresh_tokens
                 WHERE token = ? AND usuario_id = ? AND expires_at > NOW()"
            );
            $stmt->execute([$tokenRefresco, $usuarioId]);
            $tokenDb = $stmt->fetch(PDO::FETCH_ASSOC);

            if (!$tokenDb) {
                // Posible robo de token — invalidar todos los tokens
                $stmt = $this->db->prepare(
                    "DELETE FROM refresh_tokens WHERE usuario_id = ?"
                );
                $stmt->execute([$usuarioId]);

                Response::error(
                    'Token de refresco inválido o reusado. Inicia sesión nuevamente.',
                    401
                );
                return;
            }

            // Eliminar el refresh token usado (rotación)
            $stmt = $this->db->prepare(
                "DELETE FROM refresh_tokens WHERE id = ?"
            );
            $stmt->execute([$tokenDb['id']]);

            // Obtener datos actualizados del usuario
            $stmt = $this->db->prepare(
                "SELECT id, nombre, email, rol FROM usuarios WHERE id = ?"
            );
            $stmt->execute([$usuarioId]);
            $usuario = $stmt->fetch(PDO::FETCH_ASSOC);

            if (!$usuario) {
                Response::error('Usuario no encontrado', 404);
                return;
            }

            // Generar nuevos tokens
            $nuevoAcceso = $this->jwtHandler->generarToken($usuario);
            $nuevoRefresco = $this->jwtHandler->generarTokenRefresco($usuarioId);

            // Almacenar el nuevo refresh token
            $stmt = $this->db->prepare(
                "INSERT INTO refresh_tokens (usuario_id, token, expires_at, created_at)
                 VALUES (?, ?, DATE_ADD(NOW(), INTERVAL 30 DAY), NOW())"
            );
            $stmt->execute([$usuarioId, $nuevoRefresco]);

            Response::success([
                'token_acceso'   => $nuevoAcceso,
                'token_refresco'  => $nuevoRefresco,
                'token_type'     => 'Bearer',
                'expires_in'     => 3600
            ], 'Token refrescado exitosamente');

        } catch (\RuntimeException $e) {
            Response::error($e->getMessage(), 401);
        }
    }
?>
```

```php
<?php
// src/Services/PasswordPolicyService.php
namespace App\Services;

class PasswordPolicyService
{
    private int $minLongitud;
    private bool $requerirMayusculas;
    private bool $requerirNumeros;
    private bool $requerirSimbolos;
    private int $historialContrasenas;

    public function __construct()
    {
        $this->minLongitud = (int)(getenv('PASSWORD_MIN_LENGTH') ?: 8);
        $this->requerirMayusculas = (bool)(getenv('PASSWORD_REQUIRE_UPPER') ?: true);
        $this->requerirNumeros = (bool)(getenv('PASSWORD_REQUIRE_NUMBER') ?: true);
        $this->requerirSimbolos = (bool)(getenv('PASSWORD_REQUIRE_SYMBOL') ?: true);
        $this->historialContrasenas = (int)(getenv('PASSWORD_HISTORY') ?: 5);
    }

    public function validar(string $password, ?int $usuarioId = null): array
    {
        $errores = [];

        // Longitud mínima
        if (strlen($password) < $this->minLongitud) {
            $errores[] = "La contraseña debe tener al menos {$this->minLongitud} caracteres";
        }

        // Mayúsculas
        if ($this->requerirMayusculas && !preg_match('/[A-Z]/', $password)) {
            $errores[] = 'La contraseña debe contener al menos una mayúscula';
        }

        // Minúsculas
        if (!preg_match('/[a-z]/', $password)) {
            $errores[] = 'La contraseña debe contener al menos una minúscula';
        }

        // Números
        if ($this->requerirNumeros && !preg_match('/[0-9]/', $password)) {
            $errores[] = 'La contraseña debe contener al menos un número';
        }

        // Símbolos
        if ($this->requerirSimbolos && !preg_match('/[^a-zA-Z0-9]/', $password)) {
            $errores[] = 'La contraseña debe contener al menos un símbolo';
        }

        // No debe contener espacios
        if (str_contains($password, ' ')) {
            $errores[] = 'La contraseña no debe contener espacios';
        }

        // Verificar historial de contraseñas
        if ($usuarioId !== null) {
            $errores = array_merge(
                $errores,
                $this->verificarHistorial($password, $usuarioId)
            );
        }

        return $errores;
    }

    private function verificarHistorial(string $password, int $usuarioId): array
    {

        // para evitar que el usuario reutilice contraseñas anteriores

    }

    public function calcularFortaleza(string $password): array
    {
        $puntaje = 0;
        $maximo = 100;

        // Longitud
        $puntaje += min(40, strlen($password) * 4);

        // Mayúsculas
        if (preg_match('/[A-Z]/', $password)) $puntaje += 10;
        if (preg_match('/[A-Z].*[A-Z]/', $password)) $puntaje += 5;

        // Números
        if (preg_match('/[0-9]/', $password)) $puntaje += 10;
        if (preg_match('/[0-9].*[0-9]/', $password)) $puntaje += 5;

        // Símbolos
        if (preg_match('/[^a-zA-Z0-9]/', $password)) $puntaje += 15;
        if (preg_match('/[^a-zA-Z0-9].*[^a-zA-Z0-9]/', $password)) $puntaje += 5;

        // Variedad
        $tipos = 0;
        if (preg_match('/[a-z]/', $password)) $tipos++;
        if (preg_match('/[A-Z]/', $password)) $tipos++;
        if (preg_match('/[0-9]/', $password)) $tipos++;
        if (preg_match('/[^a-zA-Z0-9]/', $password)) $tipos++;
        $puntaje += $tipos * 5;

        $nivel = match (true) {
            $puntaje >= 80 => 'fuerte',
            $puntaje >= 50 => 'media',
            default        => 'debil'
        };

        return [
            'puntaje' => min($maximo, $puntaje),
            'nivel'   => $nivel
        ];
    }
}
```

RESUMEN DEL CAPÍTULO
===============================================================================

- El registro debe incluir: validación de datos, verificación de email
  único, hash seguro con `password_hash` (cost 12), y token de verificación.
- El login verifica credenciales con `password_verify`, chequea bloqueo
  por intentos fallidos, requiere email verificado, y genera tokens JWT.
- La protección de rutas usa el middleware AuthenticationMiddleware que
  extrae el token del header Authorization y lo verifica.
- La verificación de email usa un token único de 32 bytes almacenado en
  BD y enviado por email al usuario.
- La recuperación de contraseña genera token de 1 hora, no revela si el
  email existe (protección contra enumeración), y permite restablecer.
- El logout invalida el token (blacklist) y elimina refresh tokens.
- Las políticas de seguridad incluyen: longitud mínima, mayúsculas,
  números, símbolos, y bloqueo tras 5 intentos fallidos.

PREGUNTAS FRECUENTES
===============================================================================

❓ **¿Cuánto debe durar un token de acceso?** 15-60 minutos. Suficiente
para sesiones ágiles, pero limitando el daño si el token es robado.

❓ **¿Debo bloquear cuentas tras intentos fallidos?** Sí, bloquea por
15-30 minutos tras 5 intentos fallidos. Esto previene ataques de fuerza
bruta sin afectar la usabilidad.

❓ **¿Qué algoritmo de hash debo usar para contraseñas?** `PASSWORD_BCRYPT`
con cost 12. Es lento (protección contra fuerza bruta) y ampliamente
soportado. Para mayor seguridad, usa `PASSWORD_ARGON2ID`.

❓ **¿Cómo protejo el endpoint de login contra ataques de enumeración?**
No reveles si el email existe. Usa respuestas genéricas como "Credenciales
inválidas" tanto para email incorrecto como para contraseña incorrecta.

❓ **¿Es necesario el refresh token rotation?** Sí. Cada vez que se usa un
refresh token, invalida el anterior y emite uno nuevo. Si un atacante roba
un refresh token y el usuario legítimo lo usa primero, el atacante pierde
acceso.
