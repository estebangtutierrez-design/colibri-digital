### 3.1. Value Object: ValorMonetario

```php
<?php

declare(strict_types=1);

namespace LucasEgaAcademy\SGCE\Modelos;

/**
 * Value Object inmutable para representar cantidades monetarias.
 */
readonly class ValorMonetario
{
    public function __construct(
        public float $cantidad,
        public string $moneda = 'MXN'
    ) {
        if ($cantidad < 0) {
            throw new \InvalidArgumentException(
                'La cantidad no puede ser negativa.'
            );
        }
        if (!in_array($moneda, ['MXN', 'USD', 'EUR'])) {
            throw new \InvalidArgumentException("Moneda no soportada: $moneda");
        }
    }

    public function sumar(ValorMonetario $otro): ValorMonetario
    {
        if ($this->moneda !== $otro->moneda) {
            throw new \InvalidArgumentException(
                'No se pueden sumar monedas diferentes.'
            );
        }
        return new ValorMonetario(
            cantidad: $this->cantidad + $otro->cantidad,
            moneda: $this->moneda
        );
    }

    public function aplicarPorcentaje(float $porcentaje): ValorMonetario
    {
        return new ValorMonetario(
            cantidad: $this->cantidad * (1 - $porcentaje / 100),
            moneda: $this->moneda
        );
    }

    public function formatear(): string
    {
        $simbolo = match ($this->moneda) {
            'MXN' => '$', 'USD' => 'US$', 'EUR' => '€',
        };
        return $simbolo . number_format($this->cantidad, 2);
    }
}
```

### 3.2. Traits: Timestampable, Validable, AuditTrail

```php
<?php

declare(strict_types=1);

namespace LucasEgaAcademy\SGCE\Traits;

trait Timestampable
{
    private \DateTimeImmutable $creadoEn;
    private \DateTimeImmutable $actualizadoEn;

    public function inicializarTimestamps(): void
    {
        $this->creadoEn = new \DateTimeImmutable();
        $this->actualizadoEn = new \DateTimeImmutable();
    }

    public function marcarActualizado(): void
    {
        $this->actualizadoEn = new \DateTimeImmutable();
    }

    public function getCreadoEn(): \DateTimeImmutable
    {
        return $this->creadoEn;
    }

    public function getActualizadoEn(): \DateTimeImmutable
    {
        return $this->actualizadoEn;
    }
}
```

```php
<?php

declare(strict_types=1);

namespace LucasEgaAcademy\SGCE\Traits;

trait Validable
{
    private array $erroresValidacion = [];

    public function esValido(): bool
    {
        $this->erroresValidacion = [];
        $this->validar();
        return empty($this->erroresValidacion);
    }

    public function getErrores(): array
    {
        return $this->erroresValidacion;
    }

    abstract protected function validar(): void;

    protected function agregarError(string $campo, string $mensaje): void
    {
        $this->erroresValidacion[$campo] = $mensaje;
    }
}
```

```php
<?php

declare(strict_types=1);

namespace LucasEgaAcademy\SGCE\Traits;

trait AuditTrail
{
    private array $auditLog = [];

    public function registrarEvento(string $accion, array $detalles = []): void
    {
        $this->auditLog[] = [
            'accion' => $accion,
            'detalles' => $detalles,
            'fecha' => new \DateTimeImmutable(),
            'usuario' => $this->getUsuarioActual() ?? 'sistema',
        ];
    }

    public function getAuditLog(): array
    {
        return $this->auditLog;
    }

    abstract protected function getUsuarioActual(): ?string;
}
```

### 3.4. Clase Abstracta: Persona

```php
<?php

declare(strict_types=1);

namespace LucasEgaAcademy\SGCE\Modelos;

use LucasEgaAcademy\SGCE\Traits\Timestampable;
use LucasEgaAcademy\SGCE\Traits\Validable;
use LucasEgaAcademy\SGCE\Interfaces\Exportable;

abstract class Persona implements Exportable
{
    use Timestampable, Validable {
        Validable::esValido as traitEsValido;
    }

    public function __construct(
        protected string $nombre,
        protected string $email,
        protected string $telefono = '',
        protected ?\DateTimeImmutable $fechaNacimiento = null
    ) {
        $this->inicializarTimestamps();
    }

    public function getNombre(): string { return $this->nombre; }
    public function getEmail(): string { return $this->email; }
    public function getTelefono(): string { return $this->telefono; }
    public function getFechaNacimiento(): ?\DateTimeImmutable
    {
        return $this->fechaNacimiento;
    }

    public function setNombre(string $nombre): void
    {
        $this->nombre = $nombre;
        $this->marcarActualizado();
    }

    public function setEmail(string $email): void
    {
        if (!filter_var($email, FILTER_VALIDATE_EMAIL)) {
            throw new \InvalidArgumentException('Email inválido.');
        }
        $this->email = $email;
        $this->marcarActualizado();
    }

    public function getEdad(): int
    {
        if ($this->fechaNacimiento === null) return 0;
        return $this->fechaNacimiento->diff(new \DateTimeImmutable())->y;
    }

    abstract public function getRol(): string;

    protected function validar(): void
    {
        if (strlen($this->nombre) < 2) {
            $this->agregarError('nombre', 'El nombre debe tener al menos 2 caracteres.');
        }
        if (!filter_var($this->email, FILTER_VALIDATE_EMAIL)) {
            $this->agregarError('email', 'El formato del email no es válido.');
        }
    }

    public function toArray(): array
    {
        return [
            'nombre' => $this->nombre,
            'email' => $this->email,
            'telefono' => $this->telefono,
            'rol' => $this->getRol(),
            'edad' => $this->getEdad(),
        ];
    }

    public function toJSON(): string
    {
        return json_encode($this->toArray(), JSON_PRETTY_PRINT | JSON_UNESCAPED_UNICODE);
    }

    public function toCSV(): string
    {
        return implode(',', [
            $this->nombre,
            $this->email,
            $this->telefono,
            $this->getRol(),
            (string)$this->getEdad(),
        ]);
    }
}
```

### 3.5. Clase: Estudiante

```php
<?php

declare(strict_types=1);

namespace LucasEgaAcademy\SGCE\Modelos;

use LucasEgaAcademy\SGCE\Excepciones\EstudianteNoEncontradoException;

class Estudiante extends Persona
{
    private string $matricula;
    private int $creditosAcumulados = 0;
    private bool $activo = true;

    public function __construct(
        string $nombre,
        string $email,
        string $telefono = '',
        ?\DateTimeImmutable $fechaNacimiento = null
    ) {
        parent::__construct($nombre, $email, $telefono, $fechaNacimiento);
        $this->matricula = $this->generarMatricula();
    }

    private function generarMatricula(): string
    {
        return 'EST-' . strtoupper(substr(md5(uniqid()), 0, 8));
    }

    public function getMatricula(): string { return $this->matricula; }
    public function getCreditosAcumulados(): int { return $this->creditosAcumulados; }
    public function isActivo(): bool { return $this->activo; }

    public function agregarCreditos(int $creditos): void
    {
        if ($creditos <= 0) {
            throw new \InvalidArgumentException(
                'Los créditos a agregar deben ser positivos.'
            );
        }
        $this->creditosAcumulados += $creditos;
        $this->marcarActualizado();
        $this->registrarEvento('creditos_agregados', [
            'creditos' => $creditos,
            'total' => $this->creditosAcumulados,
        ]);
    }

    public function suspender(): void
    {
        $this->activo = false;
        $this->marcarActualizado();
        $this->registrarEvento('estudiante_suspendido', [
            'email' => $this->email,
        ]);
    }

    public function reactivar(): void
    {
        $this->activo = true;
        $this->marcarActualizado();
        $this->registrarEvento('estudiante_reactivado', []);
    }

    public function getRol(): string
    {
        return 'estudiante';
    }

    protected function validar(): void
    {
        parent::validar();
        if (strlen($this->matricula) < 5) {
            $this->agregarError('matricula', 'Matrícula inválida.');
        }
    }

    public function toArray(): array
    {
        return array_merge(parent::toArray(), [
            'matricula' => $this->matricula,
            'creditos' => $this->creditosAcumulados,
            'activo' => $this->activo,
        ]);
    }

    protected function getUsuarioActual(): ?string
    {
        return $this->email;
    }
}
```

### 3.7. Repositorio: RepositorioEstudiantes

```php
<?php

declare(strict_types=1);

namespace LucasEgaAcademy\SGCE\Repositorios;

use LucasEgaAcademy\SGCE\Interfaces\Repositorio;
use LucasEgaAcademy\SGCE\Modelos\Estudiante;
use LucasEgaAcademy\SGCE\Excepciones\EstudianteNoEncontradoException;

class RepositorioEstudiantes implements Repositorio
{
    /** @var array<string, Estudiante> */
    private array $estudiantes = [];

    public function guardar(object $entidad): void
    {
        if (!$entidad instanceof Estudiante) {
            throw new \InvalidArgumentException(
                'Se esperaba una instancia de Estudiante.'
            );
        }
        $this->estudiantes[$entidad->getMatricula()] = $entidad;
    }

    public function eliminar(string $id): bool
    {
        if (!isset($this->estudiantes[$id])) {
            return false;
        }
        unset($this->estudiantes[$id]);
        return true;
    }

    public function obtenerPorId(string $id): ?Estudiante
    {
        return $this->estudiantes[$id] ?? null;
    }

    public function obtenerTodos(): array
    {
        return array_values($this->estudiantes);
    }

    public function contar(): int
    {
        return count($this->estudiantes);
    }

    public function buscarPorEmail(string $email): ?Estudiante
    {
        foreach ($this->estudiantes as $estudiante) {
            if ($estudiante->getEmail() === $email) {
                return $estudiante;
            }
        }
        return null;
    }

    public function obtenerActivos(): array
    {
        return array_values(
            array_filter(
                $this->estudiantes,
                fn(Estudiante $e) => $e->isActivo()
            )
        );
    }

    public function obtenerPorMatricula(string $matricula): Estudiante
    {
        $estudiante = $this->estudiantes[$matricula] ?? null;
        if ($estudiante === null) {
            throw new EstudianteNoEncontradoException(
                "Estudiante con matrícula '$matricula' no encontrado."
            );
        }
        return $estudiante;
    }
}
```

### 3.9. Servicio: GestorInscripciones

```php
<?php

declare(strict_types=1);

namespace LucasEgaAcademy\SGCE\Servicios;

use LucasEgaAcademy\SGCE\Modelos\Estudiante;
use LucasEgaAcademy\SGCE\Modelos\Curso;
use LucasEgaAcademy\SGCE\Modelos\Inscripcion;
use LucasEgaAcademy\SGCE\Repositorios\RepositorioEstudiantes;
use LucasEgaAcademy\SGCE\Repositorios\RepositorioCursos;
use LucasEgaAcademy\SGCE\Repositorios\RepositorioInscripciones;
use LucasEgaAcademy\SGCE\Excepciones\EstudianteNoEncontradoException;
use LucasEgaAcademy\SGCE\Excepciones\CursoNoDisponibleException;
use LucasEgaAcademy\SGCE\Excepciones\CupoExcedidoException;

class GestorInscripciones
{
    public function __construct(
        private readonly RepositorioEstudiantes $repoEstudiantes,
        private readonly RepositorioCursos $repoCursos,
        private readonly RepositorioInscripciones $repoInscripciones
    ) {}

    public function inscribirEstudiante(
        string $matricula,
        string $codigoCurso
    ): Inscripcion {
        $estudiante = $this->repoEstudiantes->obtenerPorMatricula($matricula);

        if (!$estudiante->isActivo()) {
            throw new CursoNoDisponibleException(
                'El estudiante no está activo y no puede inscribirse.'
            );
        }

        $curso = $this->repoCursos->obtenerPorCodigo($codigoCurso);

        if (!$curso->isDisponible()) {
            throw new CursoNoDisponibleException(
                "El curso '{$curso->getNombre()}' no está disponible."
            );
        }

        if ($curso->getInscritosActuales() >= $curso->getCupoMaximo()) {
            throw new CupoExcedidoException(
                "El curso '{$curso->getNombre()}' ha alcanzado su cupo máximo de {$curso->getCupoMaximo()} estudiantes."
            );
        }

        $inscripcion = new Inscripcion($estudiante, $curso);
        $this->repoInscripciones->guardar($inscripcion);
        $curso->incrementarInscritos();

        return $inscripcion;
    }

    public function cancelarInscripcion(string $idInscripcion): bool
    {
        $inscripcion = $this->repoInscripciones->obtenerPorId($idInscripcion);
        if ($inscripcion === null) return false;

        $inscripcion->marcarCancelada();
        $inscripcion->getCurso()->decrementarInscritos();

        return true;
    }

    public function listarInscripcionesPorEstudiante(string $matricula): array
    {
        $estudiante = $this->repoEstudiantes->obtenerPorMatricula($matricula);
        return $this->repoInscripciones->obtenerPorEstudiante(
            $estudiante->getMatricula()
        );
    }

    public function listarInscripcionesPorCurso(string $codigoCurso): array
    {
        $curso = $this->repoCursos->obtenerPorCodigo($codigoCurso);
        return $this->repoInscripciones->obtenerPorCurso(
            $curso->getCodigo()
        );
    }
}
```

### 3.10. Servicio: GeneradorReportes

```php
<?php

declare(strict_types=1);

namespace LucasEgaAcademy\SGCE\Servicios;

use LucasEgaAcademy\SGCE\Modelos\Estudiante;
use LucasEgaAcademy\SGCE\Modelos\Curso;
use LucasEgaAcademy\SGCE\Interfaces\Exportable;
use LucasEgaAcademy\SGCE\Repositorios\RepositorioEstudiantes;
use LucasEgaAcademy\SGCE\Repositorios\RepositorioCursos;
use LucasEgaAcademy\SGCE\Repositorios\RepositorioInscripciones;

class GeneradorReportes implements Exportable
{
    private array $datosReporte = [];

    public function __construct(
        private readonly RepositorioEstudiantes $repoEstudiantes,
        private readonly RepositorioCursos $repoCursos,
        private readonly RepositorioInscripciones $repoInscripciones
    ) {}

    public function generarReporteGeneral(): self
    {
        $totalEstudiantes = $this->repoEstudiantes->contar();
        $totalCursos = $this->repoCursos->contar();
        $totalInscripciones = $this->repoInscripciones->contar();
        $estudiantesActivos = count($this->repoEstudiantes->obtenerActivos());

        $cursosPopulares = $this->repoCursos->obtenerPopulares(5);

        $this->datosReporte = [
            'fecha_generacion' => (new \DateTimeImmutable())->format('Y-m-d H:i:s'),
            'resumen' => [
                'total_estudiantes' => $totalEstudiantes,
                'estudiantes_activos' => $estudiantesActivos,
                'estudiantes_inactivos' => $totalEstudiantes - $estudiantesActivos,
                'total_cursos' => $totalCursos,
                'total_inscripciones' => $totalInscripciones,
            ],
            'cursos_populares' => array_map(
                fn(Curso $c) => [
                    'nombre' => $c->getNombre(),
                    'codigo' => $c->getCodigo(),
                    'inscritos' => $c->getInscritosActuales(),
                    'disponible' => $c->isDisponible(),
                ],
                $cursosPopulares
            ),
        ];

        return $this;
    }

    public function generarReporteEstudiante(string $matricula): self
    {
        $estudiante = $this->repoEstudiantes->obtenerPorMatricula($matricula);
        $inscripciones = $this->repoInscripciones->obtenerPorEstudiante($matricula);

        $this->datosReporte = [
            'estudiante' => $estudiante->toArray(),
            'cursos_inscritos' => count($inscripciones),
            'inscripciones' => array_map(
                fn($i) => [
                    'curso' => $i->getCurso()->getNombre(),
                    'fecha' => $i->getFechaInscripcion()->format('Y-m-d'),
                    'estado' => $i->getEstado(),
                ],
                $inscripciones
            ),
        ];

        return $this;
    }

    public function toArray(): array
    {
        return $this->datosReporte;
    }

    public function toJSON(): string
    {
        return json_encode($this->datosReporte, JSON_PRETTY_PRINT | JSON_UNESCAPED_UNICODE);
    }

    public function toCSV(): string
    {
        $lineas = [];
        if (isset($this->datosReporte['resumen'])) {
            $lineas[] = 'Indicador,Valor';
            foreach ($this->datosReporte['resumen'] as $key => $value) {
                $lineas[] = "$key,$value";
            }
        }
        return implode("\n", $lineas);
```

PREGUNTAS FRECUENTES
===============================================================================

❓ **¿Debo implementar todos los archivos exactamente como están aquí?**
Este es un proyecto de demostración. Puedes simplificarlo o extenderlo.
Lo importante es que demuestres dominio de todos los conceptos de POO.
Siéntete libre de añadir funcionalidades o mejorar el diseño.

❓ **¿Necesito una base de datos real?**
No para esta demostración. Los repositorios en memoria son suficientes
para mostrar la arquitectura. Si quieres ir más allá, puedes implementar
repositorios con PDO y una base de datos SQLite o MySQL.

❓ **¿Debo usar Composer?**
Sí, es parte del proyecto. El autoloading PSR-4 con Composer es un
requisito profesional. Instala Composer, crea el `composer.json`, ejecuta
`composer dump-autoload` y usa el autoloader generado.

❓ **¿Qué hago si recibo errores de autoloading?**
Verifica que: (1) el namespace en el archivo PHP coincide con la ruta
del directorio, (2) el nombre del archivo coincide con el nombre de la
clase, (3) los archivos tienen la extensión `.php`, (4) ejecutaste
`composer dump-autoload`.

❓ **¿Puedo usar PHP 8.0 en lugar de 8.1?**
Sí, pero perderás algunas características como `readonly` class (8.2),
`readonly` properties (8.1), y `mixed` return type. El proyecto se puede
adaptar a PHP 8.0 sin problemas.

❓ **¿Cómo pruebo el sistema?**
Ejecuta `composer start` o `php -S localhost:8000 -t public/` y abre
http://localhost:8000 en tu navegador. También puedes ejecutar
`php public/index.php` desde la línea de comandos.
