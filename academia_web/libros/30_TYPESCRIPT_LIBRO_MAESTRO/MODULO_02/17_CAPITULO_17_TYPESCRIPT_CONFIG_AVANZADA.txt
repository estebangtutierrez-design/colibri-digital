CONTENIDO DEL CAPÍTULO
===============================================================================

1. TSCONFIG PROFUNDO

strict: true (strictNullChecks, strictFunctionTypes, etc).

noUncheckedIndexedAccess, exactOptionalPropertyTypes.

paths y baseUrl para imports absolutos. Project references.



2. PROJECT REFERENCES

Dividir código en proyectos más pequeños. Compilación incremental.

Referencias en tsconfig: references: [{ path: './shared' }].

Build mode: tsc -b para compilar todo. typeRoots y types.



3. OPTIMIZACIONES

skipLibCheck: true (acelera compilación). incremental: true.

composite: true para project references. isolatedModules para

compatibilidad con transpiladores. downlevelIteration.

EJEMPLO DE CÓDIGO
----------------------------------------

{
  "compilerOptions": {
    "target": "ES2022",
    "module": "NodeNext",
    "moduleResolution": "NodeNext",
    "strict": true,
    "noUncheckedIndexedAccess": true,
    "exactOptionalPropertyTypes": true,
    "skipLibCheck": true,
    "incremental": true,
    "outDir": "./dist",
    "rootDir": "./src",
    "paths": {
      "@/*": ["./src/*"]
    }
  },
  "references": [
    { "path": "./packages/shared" }
  ]
}
