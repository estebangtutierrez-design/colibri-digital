1. TEMPLATE LITERAL TYPES

type Event = `on${Change}`. Combinan string literals con uniones.

type Key = `v${1|2|3}.${'x'|'y'|'z'}`. Generan todas las combinaciones.



2. STRING MANIPULATION TYPES

Uppercase<S>, Lowercase<S>, Capitalize<S>, Uncapitalize<S>.

Útiles para normalizar props, eventos, y nombres de API.

Intrinsic: funcionan a nivel de tipo, no runtime.



3. CASOS DE USO

CSS properties tipadas, event handlers, API endpoint builders.

Action types en Redux/Zustand. Database query builders.

ORM column types. Validation rule types.

EJEMPLO DE CÓDIGO
----------------------------------------

type CSSProperty = 'margin' | 'padding'
type CSSDirection = 'top' | 'right' | 'bottom' | 'left'
type CSSShorthand = `${CSSProperty}-${CSSDirection}`
// 'margin-top' | 'margin-right' | ...

type EventName = 'click' | 'focus' | 'blur'
type Handler = `on${Capitalize<EventName>}`
// 'onClick' | 'onFocus' | 'onBlur'

type ExtractValues<T> = T extends `${infer _}:${infer V}` ? V : never
