CONTENIDO DEL CAPÍTULO
===============================================================================

1. MAPPED TYPES

{ [P in K]: T }. Iteran sobre keys. Remapping: as ${P}${Suffix}.

Filter: as T extends Condition ? P : never. Homomórficos vs no.



2. CONDITIONAL TYPES AVANZADOS

T extends U ? X : Y. Nested conditionals. Distributive conditional

types (union se distribuye). Infer para extraer tipos internos.



3. PATRONES COMUNES

Nullable<T>, NullableProperties<T>, DeepPartial<T>, DeepReadonly<T>.

UnionToIntersection, FunctionPropertyNames, AsyncReturnType.

PickByValue, OmitByType. Validator types.

EJEMPLO DE CÓDIGO
----------------------------------------

// Deep Partial
type DeepPartial<T> = T extends object ? {
  [P in keyof T]?: DeepPartial<T[P]>
} : T

// Pick by value
type PickByValue<T, V> = {
  [K in keyof T as T[K] extends V ? K : never]: T[K]
}

// Unión a intersección
type UnionToIntersection<U> = 
  (U extends any ? (k: U) => void : never) extends 
  (k: infer I) => void ? I : never
