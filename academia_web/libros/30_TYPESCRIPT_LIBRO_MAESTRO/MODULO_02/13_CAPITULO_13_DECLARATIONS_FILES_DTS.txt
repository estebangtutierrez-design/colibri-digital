CONTENIDO DEL CAPÍTULO
===============================================================================

1. DECLARATION FILES

Archivos .d.ts describen la API de librerías JS sin implementación.

declare module, declare namespace, declare function, declare class.

Ambient declarations para código JS existente.



2. TYPES PARA LIBRERÍAS

@types/<package> en DefinitelyTyped. Crear types propios cuando

no existen. module augmentation para extender tipos existentes.



3. DECLARATION MERGING

Interfaces del mismo nombre se fusionan. Namespaces se fusionan.

Module augmentation: declare module 'express' { interface Request {} }

Global augmentation: declare global { interface Window {} }.

EJEMPLO DE CÓDIGO
----------------------------------------

// types.d.ts
export interface User {
  id: number
  name: string
}

export function createUser(name: string): User

// Augmentación de módulo
declare module 'express' {
  interface Request {
    user?: User
  }
}

// Ambient declarations para JS
declare module 'old-library' {
  export function doSomething(cb: (err: Error | null) => void): void
}

declare function require(module: string): any
