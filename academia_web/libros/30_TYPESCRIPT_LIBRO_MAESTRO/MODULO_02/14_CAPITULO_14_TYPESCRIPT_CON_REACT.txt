CONTENIDO DEL CAPÍTULO
===============================================================================

1. COMPONENTES TIPADOS

React.FC<Props> (deprecated). Props interface. Children con ReactNode.

useState<Type>, useRef<Type>, useReducer<State, Action>.

Event handlers tipados: ChangeEvent, MouseEvent, FormEvent.



2. HOOKS TIPADOS

useState con inicializador: useState<User | null>(null).

useReducer: action discriminated unions. useRef para DOM elements.

Custom hooks con tipos genéricos. useMemo/useCallback tipados.



3. PATRONES AVANZADOS

Generic components: <T extends {}>. Render props tipados.

Higher-order components con tipos. Context tipado con createContext.

ForwardRef con genéricos. Suspense y ErrorBoundary tipados.

EJEMPLO DE CÓDIGO
----------------------------------------

interface Props {
  name: string
  age?: number
  onChange: (value: string) => void
  children: React.ReactNode
}

const MyComponent: React.FC<Props> = ({ name, age, onChange, children }) => {
  const [count, setCount] = useState<number>(0)
  const inputRef = useRef<HTMLInputElement>(null)
  
  const handleChange = (e: ChangeEvent<HTMLInputElement>) => {
    onChange(e.target.value)
  }
  
  return <div onChange={handleChange}>{children}</div>
}

// Generic component
function List<T>({ items, render }: {
  items: T[]
  render: (item: T) => React.ReactNode
}) {
  return <ul>{items.map(render)}</ul>
}
