CONTENIDO DEL CAPÍTULO
===============================================================================

1. OPTIMIZACIÓN DE TIPOS

Evitar tipos recursivos profundos. Preferir interfaces sobre types

para objetos (más rápido). type vs interface tradeoffs.

Limit conditional types complejos. Mapped types con cuidado.



2. COMPILACIÓN

tsc --noEmit + esbuild para build rápido. project references.

incremental: true. skipLibCheck. isolatedModules.

TS Server performance: disable plugin, exclude grandes.



3. MONITOREO

tsc --generateTrace para tracing. @typescript/analyze-trace.

Medir tiempos de compilación. Diagnosticar tipos lentos.

TypeScript Performance Dashboard. Perf en CI.

EJEMPLO DE CÓDIGO
----------------------------------------

// Perfilamiento
tsc --generateTrace trace
npx @typescript/analyze-trace trace

// Medir tiempos
npm run typecheck -- --diagnostics

// Optimizar
{
  "compilerOptions": {
    "skipLibCheck": true,
    "incremental": true,
    "isolatedModules": true,
    "noEmit": true
  },
  "exclude": ["node_modules", "dist", "**/*.test.ts"]
}

// ts-server perf
// VSCode: TypeScript: Show Build Performance
