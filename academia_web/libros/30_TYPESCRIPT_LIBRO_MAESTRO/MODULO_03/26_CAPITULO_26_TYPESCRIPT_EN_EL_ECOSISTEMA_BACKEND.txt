CONTENIDO DEL CAPÍTULO
===============================================================================

1. FRAMEWORKS BACKEND

NestJS: TS nativo, decoradores, DI. Express + types. Fastify con TS.

AdonisJS: TS completo. Hono: lightweight, multi-runtime, TS.

tRPC: APIs tipadas end-to-end. Prisma + TS para BD.



2. VALIDACIÓN Y SERIALIZACIÓN

class-validator + class-transformer con NestJS. Zod para schemas.

io-ts: tipos funcionales. typia: transformación en compilación.

Superstruct: composable. Ajv + JSON Schema con TS.



3. BASE DE DATOS

Prisma: schema -> types. Drizzle: SQL-like, types inferidos.

TypeORM: decoradores. MikroORM: TS nativo. Kysely: SQL tipado.

Mongoose con TS para MongoDB. Sequelize con TS.

// NestJS Controller
@Controller('users')
export class UsersController {
  constructor(private usersService: UsersService) {}

  @Get()
  async findAll(): Promise<User[]> {
    return this.usersService.findAll()
  }

  @Post()
  @HttpCode(201)
  async create(@Body() createUserDto: CreateUserDto): Promise<User> {
    return this.usersService.create(createUserDto)
  }
}

// tRPC Router
export const appRouter = t.router({
  userList: t.procedure
    .input(z.number().optional())
    .query(({ input }) => {
      return db.user.findMany({ take: input })
    })
})
