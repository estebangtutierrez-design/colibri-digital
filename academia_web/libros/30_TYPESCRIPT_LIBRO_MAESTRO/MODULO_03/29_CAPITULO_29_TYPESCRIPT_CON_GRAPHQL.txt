CONTENIDO DEL CAPÍTULO
===============================================================================

1. CODE GENERATION

GraphQL Code Generator: tipos desde schema .graphql. graphql-codegen

genera hooks tipados. @graphql-typed/nodes para queries tipadas.



2. URQL Y APOLLO

urql: useQuery<UserQuery> con tipos generados. Apollo: useQuery<TData>.

TypedDocumentNode: queries con tipos incluidos. Fragment masking.



3. TYPEGRAPHQL (BACKEND)

@ObjectType, @Field, @Arg, @Query, @Mutation. Decoradores para

crear schema GraphQL desde clases TS. Validación con decoradores.

Resolver tipados. Context tipado. Dataloaders con tipos.

EJEMPLO DE CÓDIGO
----------------------------------------

# graphql-codegen presets
schema: ./src/schema.graphql
generates:
  ./src/generated.ts:
    plugins:
      - typescript
      - typescript-operations
      - typescript-react-query

// TypeGraphQL
@ObjectType()
class User {
  @Field()
  id!: number

  @Field()
  name!: string

  @Field({ nullable: true })
  email?: string
}

@Resolver(User)
class UserResolver {
  @Query(() => [User])
  async users(): Promise<User[]> {
    return db.user.findMany()
  }
}
