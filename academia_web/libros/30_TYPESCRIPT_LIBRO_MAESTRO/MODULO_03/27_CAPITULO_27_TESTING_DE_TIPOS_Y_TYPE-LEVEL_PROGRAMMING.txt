1. TEST DE TIPOS CON VITEST

vitest typecheck: typecheck en tsconfig. expectTypeOf para aserciones.

toEqualTypeOf, toMatchTypeOf. assertType para verificaciones.



2. TYPE-LEVEL PROGRAMMING

Recursive conditional types. Type arithmetic con tuples.

SAT (Satisfiability) solving a nivel de tipos. FizzBuzz en tipos.

Parser combinators en tipos. Estado y máquinas de estado en tipos.



3. HERRAMIENTAS

ts-expect-error para tests negativos. tsd: linter para types.

typepark: librería de utilidades type-level. ts-essentials.

type-fest: tipos esenciales. typelevel-ts: type utilities.

EJEMPLO DE CÓDIGO
----------------------------------------

import { describe, it, expectTypeOf } from 'vitest'

// Type-level test
type MyPick<T, K extends keyof T> = {
  [P in K]: T[P]
}

interface User { id: number; name: string; email: string }
type Picked = MyPick<User, 'id' | 'name'>

it('should pick keys', () => {
  expectTypeOf<Picked>().toEqualTypeOf<{ id: number; name: string }>()
})

// Type-level FizzBuzz
type FizzBuzz<N extends number> =
  N extends 0 ? [] :
  [...FizzBuzz<Subtract<N, 1>>, 
    N extends Fizz<N> ? N extends Buzz<N> ? 'FizzBuzz' : 'Fizz' :
    N extends Buzz<N> ? 'Buzz' : N]
