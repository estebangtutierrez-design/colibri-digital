CONTENIDO DEL CAPÍTULO
===============================================================================

1. FRAMEWORKS TS

Next.js: TS nativo. npx create-next-app --typescript. Nuxt 3: TS

nativo. Remix: TS first. Gatsby: TS soporte oficial.

Astro: TS en frontmatter. SvelteKit: TS integrado.



2. STATE MANAGEMENT

Zustand: stores tipados con genéricos. Redux Toolkit: createSlice

con tipos inferidos. Jotai: atoms atómicos tipados.

Context API con createContext tipado. XState con máquinas tipadas.



3. DATA FETCHING

React Query (TanStack Query): query tipadas. SWR: useSWR<User>.

RTK Query: endpoints tipados. Apollo Client con GraphQL types.

tRPC: type-safe RPC entre frontend y backend.

EJEMPLO DE CÓDIGO
----------------------------------------

// Zustand store
type BearStore = {
  bears: number
  increase: () => void
}

export const useBearStore = create<BearStore>((set) => ({
  bears: 0,
  increase: () => set((state) => ({ bears: state.bears + 1 }))
}))

// React Query
const { data: users } = useQuery<User[]>({
  queryKey: ['users'],
  queryFn: () => fetch('/api/users').then(r => r.json())
})
