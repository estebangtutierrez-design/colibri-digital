CONTENIDO DEL CAPÍTULO
===============================================================================

1. TYPE NARROWING

typeof, instanceof, in, type unions, discriminated unions. TypeScript

reduce el tipo según condiciones. Control flow analysis.



2. TYPE GUARDS

function isString(x: unknown): x is string { return typeof x === 'string' }

El predicado x is T le dice al compilador el tipo. user-defined type guards.

typeof, instanceof como guards nativos. never para exhaustiveness.



3. ASSERTION FUNCTIONS

function assert(condition: any, msg?: string): asserts condition.

asserts x is Type para narrowing específico. throw si falla.

assertNotNull, assertDefined. Patrón común en APIs.

EJEMPLO DE CÓDIGO
----------------------------------------

function isString(value: unknown): value is string {
  return typeof value === 'string'
}

type Fish = { swim: () => void }
type Bird = { fly: () => void }

function isFish(pet: Fish | Bird): pet is Fish {
  return (pet as Fish).swim !== undefined
}

// Usando type guard
function move(pet: Fish | Bird) {
  if (isFish(pet)) {
    pet.swim()  // TypeScript sabe que es Fish
  }
}

// Assertion
function assertNonNull<T>(x: T): asserts x is NonNullable<T> {
  if (x === null || x === undefined) throw new Error()
}
