CONTENIDO DEL CAPÍTULO
===============================================================================

1. TIPOS PRIMITIVOS

string, number, boolean, null, undefined, void, never, any.

TypeScript infiere tipos automáticamente. Anotaciones opcionales.

const name: string = 'TypeScript'; let age: number = 5;



2. ARRAYS Y TUPLAS

number[], string[], Array<number>. Tuplas: [string, number].

ReadonlyArray para inmutabilidad. readonly Tuple.



3. ANY, UNKNOWN, NEVER

any: desactiva type checking (evitar). unknown: tipo seguro, requiere

type narrowing. never: nunca ocurre (funciones que lanzan error).

void: función sin return. undefined/null como tipos.

EJEMPLO DE CÓDIGO
----------------------------------------

let name: string = 'TypeScript'
let age: number = 5
let isDone: boolean = false

let list: number[] = [1, 2, 3]
let tuple: [string, number] = ['hello', 42]

let input: unknown = 'text'
if (typeof input === 'string') {
  console.log(input.toUpperCase())
}

function error(): never {
  throw new Error('Always fails')
}
