1. GENERIC FUNCTIONS

function identity<T>(arg: T): T { return arg }. Inferencia automática.

Constraints: <T extends Lengthwise>. Type parameters en arrow functions.

<T, U> múltiples parámetros. Default type: <T = string>.



2. GENERIC CLASSES

class Stack<T> { private items: T[] = []; push(item: T) {} }.

Generic constraints con keyof: <T, K extends keyof T>. Mapped types

con genéricos: { [K in keyof T]: T[K] }.



3. CONDITIONAL TYPES

type IsString<T> = T extends string ? true : false. Distributive.

infer keyword: type Return<T> = T extends (...args: any[]) => infer R

? R : never. Template literal types con infer.

EJEMPLO DE CÓDIGO
----------------------------------------

function first<T>(arr: T[]): T | undefined {
  return arr[0]
}

interface HasLength {
  length: number
}

function logLength<T extends HasLength>(arg: T): void {
  console.log(arg.length)
}

// Conditional type
type ArrayElement<T> = T extends (infer U)[] ? U : never
type Item = ArrayElement<string[]>  // string

// Mapped type
type Readonly<T> = {
  readonly [K in keyof T]: T[K]
}
