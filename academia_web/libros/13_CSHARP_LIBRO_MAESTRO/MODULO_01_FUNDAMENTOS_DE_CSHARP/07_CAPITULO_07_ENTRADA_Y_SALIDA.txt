1. CONSOLE.WRITELINE — SALIDA DE TEXTO
===============================================================================
El método fundamental para mostrar información en la consola.

```csharp
using System;

// Uso básico
Console.WriteLine("Hola, Mundo!");

// Múltiples llamadas
Console.WriteLine("Primera línea");
Console.WriteLine("Segunda línea");
Console.WriteLine("Tercera línea");
// Output:
// Primera línea
// Segunda línea
// Tercera línea

// Valores de diferentes tipos
Console.WriteLine(42);           // Entero
Console.WriteLine(3.1416);       // Double
Console.WriteLine(true);         // Booleano
Console.WriteLine('A');          // Carácter
Console.WriteLine(3.14f);        // Float
Console.WriteLine(3500.50m);     // Decimal

// Múltiples argumentos (placeholders)
string nombre = "Ana";
int edad = 30;
Console.WriteLine("Nombre: {0}, Edad: {1}", nombre, edad);
Console.WriteLine("{0} tiene {1} años.", nombre, edad);

// Interpolación de cadenas (C# 6+) — Método preferido
Console.WriteLine($"Nombre: {nombre}, Edad: {edad}");

// Caracteres especiales
Console.WriteLine("Línea 1\nLínea 2");     // \n = nueva línea
Console.WriteLine("Col1\tCol2\tCol3");    // \t = tabulador
Console.WriteLine("C:\\Users\\Ana");        // \\ = backslash
Console.WriteLine("Dijo: \"Hola\"");       // \" = comillas

// Verbatim string (@) — ignora escapes
Console.WriteLine(@"C:\Users\Ana\Documentos");
Console.WriteLine(@"Línea 1
Línea 2
Línea 3");

// Raw string literal (C# 11+) — multilínea sin escapes
Console.WriteLine("""
{
    "nombre": "Ana",
    "edad": 30
}
""");
```

3. CONSOLE.READLINE — LECTURA DE ENTRADA
===============================================================================
```csharp
using System;

// Lectura básica
Console.Write("Ingresa tu nombre: ");
string nombre = Console.ReadLine();
Console.WriteLine($"Hola, {nombre}!");

// Lectura y conversión a número
Console.Write("Ingresa tu edad: ");
string entrada = Console.ReadLine();
int edad = int.Parse(entrada);
Console.WriteLine($"El año que viene tendrás {edad + 1} años.");

// Validación con TryParse
Console.Write("Ingresa tu peso (kg): ");
if (double.TryParse(Console.ReadLine(), out double peso))
{
    Console.WriteLine($"Peso registrado: {peso:F1} kg");
}
else
{
    Console.WriteLine("Peso inválido. Usando 70.0 kg por defecto.");
    peso = 70.0;
}

// Múltiples entradas
Console.WriteLine("\n--- Registro de Usuario ---");
Console.Write("Nombre: ");
string? nom = Console.ReadLine()?.Trim();
Console.Write("Email: ");
string? email = Console.ReadLine()?.Trim();
Console.Write("Teléfono: ");
string? telefono = Console.ReadLine()?.Trim();

Console.WriteLine("\nResumen del registro:");
Console.WriteLine($"  Nombre: {nom}");
Console.WriteLine($"  Email: {email}");
Console.WriteLine($"  Teléfono: {telefono}");

// Entrada con valor por defecto
Console.Write("\nCiudad (Madrid): ");
string ciudad = Console.ReadLine();
ciudad = string.IsNullOrWhiteSpace(ciudad) ? "Madrid" : ciudad;
Console.WriteLine($"Ciudad: {ciudad}");

// Bucle de entrada hasta que sea válida
int edadValida;
do
{
    Console.Write("\nEdad (0-150): ");
} while (!int.TryParse(Console.ReadLine(), out edadValida) || edadValida < 0 || edadValida > 150);
Console.WriteLine($"Edad válida: {edadValida}");

// Leer múltiples valores separados por comas
Console.Write("\nIngresa números separados por comas (ej: 1,2,3): ");
string? linea = Console.ReadLine();
if (!string.IsNullOrWhiteSpace(linea))
{
    string[] partes = linea.Split(',');
    int[] numeros = new int[partes.Length];
    for (int i = 0; i < partes.Length; i++)
    {
        if (int.TryParse(partes[i].Trim(), out int num))
        {
            numeros[i] = num;
        }
    }
    Console.WriteLine($"Suma: {numeros.Sum()}, Promedio: {numeros.Average():F2}");
}
```

4. CONSOLE.READKEY — CAPTURA DE TECLAS
===============================================================================
```csharp
using System;

// Esperar cualquier tecla
Console.WriteLine("Presiona cualquier tecla para continuar...");
Console.ReadKey();
Console.WriteLine("\n¡Continuamos!");

// Capturar información de la tecla
Console.WriteLine("\nPresiona una tecla (mostraré info):");
ConsoleKeyInfo tecla = Console.ReadKey();
Console.WriteLine($"\n\nInformación de la tecla:");
Console.WriteLine($"  KeyChar: '{tecla.KeyChar}'");
Console.WriteLine($"  Key: {tecla.Key}");
Console.WriteLine($"  Modifiers: {tecla.Modifiers}");
Console.WriteLine($"  KeyCode: {(int)tecla.Key}");

// No mostrar la tecla presionada (intercept)
Console.WriteLine("\nPresiona una tecla (no se mostrará):");
tecla = Console.ReadKey(true);  // true = intercept
Console.WriteLine($"Presionaste: {tecla.KeyChar} (en secreto)");

// Menú interactivo con teclas
Console.WriteLine("\n=== MENÚ INTERACTIVO ===");
Console.WriteLine("Presiona 1, 2, 3 o ESC para salir");

while (true)
{
    Console.Write("\nOpción: ");
    var key = Console.ReadKey(true);
    
    switch (key.Key)
    {
        case ConsoleKey.D1:
        case ConsoleKey.NumPad1:
            Console.WriteLine("Seleccionaste Opción 1");
            break;
        case ConsoleKey.D2:
        case ConsoleKey.NumPad2:
            Console.WriteLine("Seleccionaste Opción 2");
            break;
        case ConsoleKey.D3:
        case ConsoleKey.NumPad3:
            Console.WriteLine("Seleccionaste Opción 3");
            break;
        case ConsoleKey.Escape:
            Console.WriteLine("Saliendo...");
            return;
        default:
            Console.WriteLine("Opción inválida");
            break;
    }
}

// Juego simple de teclas direccionales
Console.Clear();
Console.WriteLine("Presiona flechas para mover el cursor (ESC para salir):");

int xCursor = 40, yCursor = 12;
Console.SetCursorPosition(xCursor, yCursor);
Console.Write("■");

while (true)
{
    var key = Console.ReadKey(true);
    Console.SetCursorPosition(xCursor, yCursor);
    Console.Write(" ");  // Borra posición anterior
    
    switch (key.Key)
    {
        case ConsoleKey.UpArrow: yCursor = Math.Max(0, yCursor - 1); break;
        case ConsoleKey.DownArrow: yCursor = Math.Min(Console.WindowHeight - 1, yCursor + 1); break;
        case ConsoleKey.LeftArrow: xCursor = Math.Max(0, xCursor - 1); break;
        case ConsoleKey.RightArrow: xCursor = Math.Min(Console.WindowWidth - 1, xCursor + 1); break;
        case ConsoleKey.Escape: return;
    }
    
    Console.SetCursorPosition(xCursor, yCursor);
    Console.ForegroundColor = ConsoleColor.Cyan;
    Console.Write("■");
    Console.ResetColor();
}
```

**5.1 Formatos numéricos**

```csharp
using System;

double numero = 12345.6789;

// Formatos estándar
Console.WriteLine($"N (número):    {numero:N}");       // 12,345.68
Console.WriteLine($"N0 (sin dec):  {numero:N0}");      // 12,346
Console.WriteLine($"N4:            {numero:N4}");       // 12,345.6789
Console.WriteLine($"F (fijo):      {numero:F}");       // 12345.68
Console.WriteLine($"F2 (2 dec):    {numero:F2}");      // 12345.68
Console.WriteLine($"F0 (entero):   {numero:F0}");      // 12346
Console.WriteLine($"C (moneda):    {numero:C}");       // $12,345.68
Console.WriteLine($"C2:            {numero:C2}");      // $12,345.68
Console.WriteLine($"E (científico): {numero:E}");      // 1.234568E+004
Console.WriteLine($"E4:            {numero:E4}");      // 1.2346E+004
Console.WriteLine($"P (porcentaje): {0.15:P}");        // 15.00%
Console.WriteLine($"P0:            {0.15:P0}");       // 15%
Console.WriteLine($"G (general):   {numero:G}");       // 12345.6789
Console.WriteLine($"G4:            {numero:G4}");     // 1.235E+04

// Formato personalizado
Console.WriteLine($"\nFormatos personalizados:");
Console.WriteLine($"#,##0.00:      {numero:#,##0.00}");  // 12,345.68
Console.WriteLine($"000000:        {42:000000}");         // 000042
Console.WriteLine($"##.##:         {numero:##.##}");      // 12345.68
Console.WriteLine($"0.0%:          {0.15:0.0%}");         // 15.0%

// Alineación en campos
Console.WriteLine("\nAlineación:");
Console.WriteLine($"{"Nombre",-10} {"Edad",5} {"Nota",8}");
Console.WriteLine($"{"Ana",-10} {30,5} {95.5,8:F1}");
Console.WriteLine($"{"Luis",-10} {25,5} {87.3,8:F1}");
Console.WriteLine($"{"Carlos",-10} {22,5} {91.0,8:F1}");
```

**5.2 Formato de fechas**

```csharp
using System;

DateTime ahora = DateTime.Now;
DateTime fecha = new DateTime(2023, 12, 25, 10, 30, 45);

Console.WriteLine("Formatos de fecha:");
Console.WriteLine($"d (corta):     {ahora:d}");        // 7/2/2026
Console.WriteLine($"D (larga):     {ahora:D}");        // Thursday, July 2, 2026
Console.WriteLine($"f (completa):  {ahora:f}");        // Thursday, July 2, 2026 10:30 AM
Console.WriteLine($"F (full):      {ahora:F}");        // Thursday, July 2, 2026 10:30:45 AM
Console.WriteLine($"g (gen corta): {ahora:g}");        // 7/2/2026 10:30 AM
Console.WriteLine($"G (gen larga): {ahora:G}");        // 7/2/2026 10:30:45 AM
Console.WriteLine($"t (hora corta):{ahora:t}");        // 10:30 AM
Console.WriteLine($"T (hora larga):{ahora:T}");        // 10:30:45 AM
Console.WriteLine($"M (mes-día):   {ahora:M}");        // July 2
Console.WriteLine($"Y (año-mes):   {ahora:Y}");        // July 2026
Console.WriteLine($"O (roundtrip): {ahora:O}");        // 2026-07-02T10:30:45.0000000
Console.WriteLine($"R (RFC1123):   {ahora:R}");        // Thu, 02 Jul 2026 10:30:45 GMT
Console.WriteLine($"u (universal): {ahora:u}");        // 2026-07-02 10:30:45Z
Console.WriteLine($"U (universal): {ahora:U}");        // Thursday, July 2, 2026 10:30:45 AM

// Formatos personalizados de fecha
Console.WriteLine("\nFormatos personalizados de fecha:");
Console.WriteLine($"yyyy-MM-dd:    {ahora:yyyy-MM-dd}");            // 2026-07-02
Console.WriteLine($"dd/MM/yyyy:    {ahora:dd/MM/yyyy}");            // 02/07/2026
Console.WriteLine($"dddd, MMMM yyyy: {ahora:dddd, MMMM yyyy}");    // Thursday, July 2026
Console.WriteLine($"HH:mm:ss:      {ahora:HH:mm:ss}");              // 10:30:45 (24h)
Console.WriteLine($"hh:mm tt:      {ahora:hh:mm tt}");              // 10:30 AM (12h)
```

```csharp
            break;

            Console.WriteLine($"  Hora: {DateTime.Now:T}");
            break;
        case "3":
            Console.Write("\n  Número 1: ");
            double n1 = double.TryParse(Console.ReadLine(), out double a) ? a : 0;
            Console.Write("  Número 2: ");
            double n2 = double.TryParse(Console.ReadLine(), out double b) ? b : 0;
            Console.WriteLine($"  {n1} + {n2} = {n1 + n2}");
            Console.WriteLine($"  {n1} - {n2} = {n1 - n2}");
            Console.WriteLine($"  {n1} * {n2} = {n1 * n2}");
            Console.WriteLine($"  {n1} / {n2} = {(n2 != 0 ? (n1 / n2).ToString("F2") : "Error")}");
            break;
        case "4":
            Console.WriteLine($"\n  SO: {Environment.OSVersion}");
            Console.WriteLine($"  .NET: {Environment.Version}");
            Console.WriteLine($"  64 bits: {Environment.Is64BitProcess}");
            break;
        case "0":
            salir = true;
            Console.WriteLine("\n  ¡Hasta luego!");
            break;
        default:
            Console.ForegroundColor = ConsoleColor.Red;
            Console.WriteLine("\n  Opción inválida. Intenta de nuevo.");
            Console.ResetColor();
            break;
    }
    
    if (!salir)
    {
        Console.WriteLine("\n  Presiona cualquier tecla para continuar...");
        Console.ReadKey(true);
    }
}
```

**8.3 Tabla dinámica**

```csharp
using System;
using System.Collections.Generic;

var productos = new List<(string Nombre, double Precio, int Stock)>
{
    ("Laptop Gamer", 1200.00, 15),
    ("Mouse Inalámbrico", 45.50, 200),
    ("Teclado Mecánico", 89.99, 75),
    ("Monitor 4K", 350.00, 30),
    ("Webcam HD", 65.25, 100)
};

void MostrarTabla(List<(string Nombre, double Precio, int Stock)> items)
{
    Console.Clear();
    Console.ForegroundColor = ConsoleColor.Cyan;
    Console.WriteLine("╔══════════════════════╦══════════╦═══════╗");
    Console.WriteLine("║ Producto             ║ Precio   ║ Stock ║");
    Console.WriteLine("╠══════════════════════╬══════════╬═══════╣");
    Console.ResetColor();
    
    double total = 0;
    foreach (var item in items)
    {
        string precio = item.Precio.ToString("C2").PadLeft(8);
        string stock = item.Stock.ToString().PadLeft(5);
        string nombre = item.Nombre.PadRight(20);
        Console.WriteLine($"║ {nombre} ║ {precio} ║ {stock} ║");
        total += item.Precio * item.Stock;
    }
    
    Console.ForegroundColor = ConsoleColor.Cyan;
    Console.WriteLine("╠══════════════════════╬══════════╬═══════╣");
    Console.ResetColor();
    Console.WriteLine($"║ {"TOTAL",-20} ║ {"",-8} ║ {"",-5} ║");
    Console.ForegroundColor = ConsoleColor.Green;
    Console.WriteLine($"║ {"Valor inventario:",-20} ║ {total,8:C2} ║ {"",-5} ║");
    Console.ResetColor();
    Console.ForegroundColor = ConsoleColor.Cyan;
    Console.WriteLine("╚══════════════════════╩══════════╩═══════╝");
    Console.ResetColor();
}

MostrarTabla(productos);
```

```csharp
using System;
using System.Text.RegularExpressions;

// Función de entrada con validación genérica
T? LeerEntrada<T>(
    string mensaje,
    Func<string, (bool, T?)> validador,
    string mensajeError = "Entrada inválida.")
{
    while (true)
    {
        Console.Write(mensaje);
        string? entrada = Console.ReadLine()?.Trim();
        
        if (string.IsNullOrWhiteSpace(entrada))
        {
            Console.ForegroundColor = ConsoleColor.Red;
            Console.WriteLine("La entrada no puede estar vacía.");
            Console.ResetColor();
            continue;
        }
        
        var (exito, valor) = validador(entrada);
        if (exito)
        {
            return valor;
        }
        
        Console.ForegroundColor = ConsoleColor.Red;
        Console.WriteLine(mensajeError);
        Console.ResetColor();
    }
}

// Validadores específicos
var validadorEntero = (string s) =>
{
    bool ok = int.TryParse(s, out int v);
    return (ok, (int?)(ok ? v : 0));
};

var validadorDouble = (string s) =>
{
    bool ok = double.TryParse(s, out double v);
    return (ok, (double?)(ok ? v : 0));
};

var validadorEmail = (string s) =>
{
    var regex = new Regex(@"^[^@\s]+@[^@\s]+\.[^@\s]+$");
    bool ok = regex.IsMatch(s);
    return (ok, ok ? s : null);
};

var validadorTelefono = (string s) =>
{
    var regex = new Regex(@"^\+?\d{7,15}$");
    bool ok = regex.IsMatch(s);
    return (ok, ok ? s : null);
};

// Uso de las funciones de validación
Console.WriteLine("=== FORMULARIO CON VALIDACIÓN ===");

int? edad2 = LeerEntrada<int>("Edad: ", validadorEntero, "Edad inválida. Ingresa un número entero.");
double? peso2 = LeerEntrada<double>("Peso (kg): ", validadorDouble, "Peso inválido. Ingresa un número.");
string? email = LeerEntrada<string>("Email: ", validadorEmail, "Email inválido. Usa formato usuario@dominio.com");
string? telefono2 = LeerEntrada<string>("Teléfono: ", validadorTelefono, "Teléfono inválido. Usa solo dígitos.");

Console.WriteLine("\n=== DATOS REGISTRADOS ===");
```

```csharp
using System;
using System.Threading;

// Animación de carga
void MostrarCarga(string texto, int duracionMs = 2000)
{
    string[] frames = { "|", "/", "-", "\\" };
    int totalFrames = duracionMs / 100;
    
    for (int i = 0; i < totalFrames; i++)
    {
        Console.Write($"\r{texto} {frames[i % frames.Length]}");
        Thread.Sleep(100);
    }
    Console.Write($"\r{texto} ✓\n");
}

MostrarCarga("Cargando configuración");

MostrarCarga("Conectando a la base de datos");

// Barra de progreso
void MostrarBarraProgreso(int progreso, int total)
{
    int porcentaje = (int)((double)progreso / total * 100);
    int barrasCompletas = porcentaje / 2;
    int barrasVacias = 50 - barrasCompletas;
    
    Console.Write($"\r[{new string('█', barrasCompletas)}{new string('░', barrasVacias)}] {porcentaje}%");
}

Console.WriteLine("\nProcesando archivos:\n");
for (int i = 0; i <= 100; i++)

    MostrarBarraProgreso(i, 100);
    Thread.Sleep(50);

}
Console.Write("\rTiempo restante:  0 segundos ✓\n");
```

RESUMEN DEL CAPÍTULO

- `Console.WriteLine()` imprime texto con salto de línea; `Console.Write()` sin
  salto de línea.
- `Console.ReadLine()` lee una línea completa de texto ingresada por el
  usuario hasta Enter.
- `Console.ReadKey()` captura una tecla individual; con `true` no la muestra.
- `ConsoleKeyInfo` proporciona información detallada de la tecla presionada

  formatear salida.
- Los formatos numéricos incluyen N (número), C (moneda), F (fijo), P
  (porcentaje), E (científico).
- Los formatos de fecha incluyen d (corta), D (larga), t (hora), T (hora
  larga), y personalizados (yyyy-MM-dd).
- `Console.ForegroundColor` y `Console.BackgroundColor` controlan colores del
  texto y fondo.
- `Console.SetCursorPosition(x, y)` mueve el cursor a coordenadas específicas.
- `Console.Clear()` limpia la pantalla; `Console.Title` cambia el título de
  la ventana.
- `Console.CursorVisible` controla la visibilidad del cursor.
- `Console.Beep(frecuencia, duracion)` produce sonidos.
- `Console.WindowWidth`, `WindowHeight`, `BufferWidth`, `BufferHeight`
  controlan el tamaño de la ventana y buffer.
- La validación de entrada es crucial: usa `TryParse` y bucles para garantizar
  datos correctos.
- `Console.IsOutputRedirected` permite detectar si la salida está redirigida.
- Las animaciones en consola se logran combinando `SetCursorPosition`, `Write`,
  `Thread.Sleep` y caracteres de retorno de carro (\r).
- Se pueden crear interfaces de usuario completas en consola usando menús,
  formularios, tablas y colores.

PREGUNTAS FRECUENTES
===============================================================================
❓ **¿Cuál es la diferencia entre ReadLine y ReadKey?**
ReadLine espera que el usuario presione Enter y devuelve todo el texto
ingresado como string. ReadKey captura una sola tecla inmediatamente (sin
esperar Enter) y devuelve información detallada de la tecla.

❓ **¿Cómo evito que el programa se cierre inmediatamente?**
Agrega `Console.ReadKey();` o `Console.ReadLine();` al final del programa
para que espere una acción del usuario antes de cerrarse.

❓ **¿Puedo cambiar el color de una parte específica del texto?**
Sí. Cambia `Console.ForegroundColor` antes de escribir esa parte, y
restáuralo después con `Console.ResetColor()`.

❓ **¿Cómo leo un número directamente sin tener que convertirlo?**
No puedes. Console.ReadLine() siempre devuelve un string. Debes convertirlo
con int.Parse(), double.TryParse() o Convert.ToInt32().

❓ **¿Qué es el carácter \r y cómo se usa?**
\r es el retorno de carro. Vuelve el cursor al inicio de la línea actual,
permitiendo sobrescribir el texto. Se usa para animaciones y progreso.

❓ **¿Cómo mantengo el cursor en la misma línea después de la entrada?**
Usa Console.Write() en lugar de Console.WriteLine() para la pregunta, y
Console.ReadLine() leerá en la misma línea.

❓ **¿Puedo leer contraseñas sin mostrar los caracteres?**
No hay un método nativo. Pero puedes usar Console.ReadKey(true) en un bucle,
mostrando '*' por cada carácter y construyendo el string manualmente.

❓ **¿Cómo centro texto en la consola?**
Calcula la posición: `int left = (Console.WindowWidth - texto.Length) / 2;`
y usa `Console.SetCursorPosition(left, top);`.

❓ **¿Qué hago si Console.ReadLine() devuelve null?**
Significa que la entrada fue redirigida y se alcanzó el final del stream o
que el usuario presionó Ctrl+Z (EOF). Usa `?? ""` o verifica con
`if (linea is not null)`.

❓ **¿Puedo cambiar la fuente de la consola desde C#?**
No directamente. La fuente se configura en las propiedades de la terminal.
Pero sí puedes cambiar el tamaño con Console.SetWindowSize().
