25.1 — LIST<T> — LISTA DINÁMICA
---------------------------------

List<T> es una lista dinámica que almacena elementos en un array interno
que se redimensiona automáticamente. Es la colección más usada.

```csharp
// Creación
List<int> numeros = new List<int>();
List<string> nombres = new List<string>(100);
List<double> valores = new List<double> { 1.5, 2.3, 4.7 };

// Métodos principales
numeros.Add(10);
numeros.AddRange(new[] { 20, 30 });
numeros.Insert(1, 15);
numeros.Remove(20);
numeros.RemoveAt(0);
numeros.Clear();

int primero = numeros[0];
int count = numeros.Count;
int capacidad = numeros.Capacity;

numeros.Sort();
numeros.Reverse();
int index = numeros.IndexOf(30);
bool existe = numeros.Contains(20);
int[] arr = numeros.ToArray();
```

25.2 — CAPACIDAD Y RENDIMIENTO
--------------------------------

```csharp
List<int> lista = new List<int>(4);
Console.WriteLine(lista.Capacity); // 4

lista.Add(1);
lista.Add(2);
lista.Add(3);
lista.Add(4);
lista.Add(5); // Redimensiona a capacidad 8

lista.Capacity = 100;
Console.WriteLine(lista.Capacity); // 100

lista.TrimExcess();

// Medición de rendimiento
var sw = System.Diagnostics.Stopwatch.StartNew();
var test = new List<int>();
for (int i = 0; i < 1000000; i++)
    test.Add(i);
sw.Stop();
Console.WriteLine($"Sin capacidad inicial: {sw.ElapsedMilliseconds} ms");

sw.Restart();
var test2 = new List<int>(1000000);
for (int i = 0; i < 1000000; i++)
    test2.Add(i);
sw.Stop();
Console.WriteLine($"Con capacidad inicial: {sw.ElapsedMilliseconds} ms");
```

25.3 — ITERACIÓN
------------------

```csharp
List<string> frutas = new() { "Manzana", "Banana", "Naranja", "Uva", "Pera" };

// foreach
foreach (string fruta in frutas)
    Console.WriteLine(fruta);

// for con índice
for (int i = 0; i < frutas.Count; i++)
    Console.WriteLine($"{i}: {frutas[i]}");

// ForEach con Action
frutas.ForEach(f => Console.WriteLine(f.ToUpper()));

// LINQ
frutas.Where(f => f.Length > 4)
      .OrderBy(f => f)
      .ToList()
      .ForEach(f => Console.WriteLine(f));

// Convertir a array
string[] arrFrutas = frutas.ToArray();

// Index from end (C# 8+)
Index idx = ^1;
Console.WriteLine(frutas[idx]); // Pera
```

25.4 — DICTIONARY<K,V> — DICCIONARIO
--------------------------------------

Dictionary<K,V> almacena pares clave-valor con búsqueda O(1).

```csharp
// Creación
Dictionary<string, int> edades = new();
Dictionary<int, string> usuarios = new(100);
var config = new Dictionary<string, string>
{
    ["host"] = "localhost",
    ["port"] = "8080"
};

// Métodos principales
edades["Ana"] = 30;
edades.Add("Luis", 25);
edades.TryAdd("Carlos", 35);

int edadAna = edades["Ana"];
bool existe = edades.ContainsKey("Ana");
bool existeValor = edades.ContainsValue(30);

edades.Remove("Luis");
edades.Clear();

// TryGetValue
if (edades.TryGetValue("Pedro", out int edad))
    Console.WriteLine(edad);
else
    Console.WriteLine("No encontrado");

// Recorrer
foreach (KeyValuePair<string, int> kvp in edades)
    Console.WriteLine($"{kvp.Key}: {kvp.Value}");

foreach (string key in edades.Keys)
    Console.WriteLine(key);

foreach (int value in edades.Values)
    Console.WriteLine(value);
```

```csharp
// Comparador que ignora mayúsculas/minúsculas
var dict = new Dictionary<string, int>(StringComparer.OrdinalIgnoreCase)
{
    ["abc"] = 1

// Comparador personalizado
public class CaseInsensitiveComparer : IEqualityComparer<string>
{
    public bool Equals(string? a, string? b) =>
        string.Equals(a, b, StringComparison.OrdinalIgnoreCase);

    public int GetHashCode(string s) =>
        s.ToUpperInvariant().GetHashCode();
}

var dict2 = new Dictionary<string, int>(new CaseInsensitiveComparer());

public record PersonaKey(string Nombre, string Apellido);

var contactos = new Dictionary<PersonaKey, string>();

```

25.6 — HASHSET<T> — CONJUNTO
------------------------------

HashSet<T> almacena elementos únicos sin orden, con operaciones O(1).

```csharp
// Creación
HashSet<int> numeros = new();
HashSet<string> palabras = new(StringComparer.OrdinalIgnoreCase);
var inicial = new HashSet<int> { 1, 2, 3, 4, 5 };

// Métodos principales
numeros.Add(10);
numeros.Remove(10);
numeros.Contains(10);
numeros.Clear();
int count = numeros.Count;

// Operaciones de conjuntos
var a = new HashSet<int> { 1, 2, 3, 4, 5 };
var b = new HashSet<int> { 4, 5, 6, 7, 8 };

a.UnionWith(b);                // Unión
a.IntersectWith(b);            // Intersección
a.ExceptWith(b);               // Diferencia
a.SymmetricExceptWith(b);      // Diferencia simétrica

// Consultas
a.IsSubsetOf(b);
a.IsSupersetOf(b);
a.Overlaps(b);
a.SetEquals(b);

// Iteración
foreach (int n in numeros)
    Console.WriteLine(n);
```

25.7 — QUEUE<T> — COLA FIFO
-----------------------------

```csharp
Queue<string> cola = new();
Queue<int> numeros = new(100);

cola.Enqueue("Cliente 1");
cola.Enqueue("Cliente 2");
cola.Enqueue("Cliente 3");

string primero = cola.Peek();
Console.WriteLine(primero); // Cliente 1

string atendido = cola.Dequeue();
Console.WriteLine(atendido); // Cliente 1
Console.WriteLine(cola.Count); // 2

// Recorrer sin vaciar
foreach (string c in cola)
    Console.WriteLine(c);

// Vaciar
while (cola.TryDequeue(out string? item))
    Console.WriteLine($"Procesando: {item}");

cola.Clear();
```

25.8 — STACK<T> — PILA LIFO
-----------------------------

```csharp
Stack<string> pila = new();
Stack<int> numeros = new(100);

pila.Push("Página 1");
pila.Push("Página 2");
pila.Push("Página 3");

string tope = pila.Peek();
Console.WriteLine(tope); // Página 3

string pagina = pila.Pop();
Console.WriteLine(pagina); // Página 3
Console.WriteLine(pila.Count); // 2

// Recorrer sin vaciar
foreach (string p in pila)
    Console.WriteLine(p);

// TryPop sin excepción
if (pila.TryPop(out string? item))
    Console.WriteLine($"Desapilado: {item}");

// Vaciar
while (pila.TryPop(out string? restante))
    Console.WriteLine($"Procesando: {restante}");

pila.Clear();
```

25.12 — COMPLEJIDAD ALGORÍTMICA
---------------------------------

```
Colección        | Acceso | Búsqueda | Inserción | Eliminación | Memoria
-----------------|--------|----------|-----------|-------------|----------
List<T>          | O(1)   | O(n)     | O(n)      | O(n)        | Baja
Dictionary<K,V>  | O(1)*  | O(1)*    | O(1)*     | O(1)*       | Alta
HashSet<T>       | O(1)*  | O(1)*    | O(1)*     | O(1)*       | Alta
Queue<T>         | O(1)   | O(n)     | O(1)      | O(1)        | Baja
Stack<T>         | O(1)   | O(n)     | O(1)      | O(1)        | Baja
LinkedList<T>    | O(n)   | O(n)     | O(1)      | O(1)        | Media
SortedList<K,V>  | O(log n)| O(log n)| O(n)      | O(n)        | Baja
SortedSet<T>     | O(log n)| O(log n)| O(log n)  | O(log n)    | Media

* Con buen hash function. Peor caso: O(n).
```

25.13 — EJEMPLO COMPLETO: SISTEMA DE PROCESAMIENTO DE PEDIDOS
---------------------------------------------------------------

```csharp
using System;
using System.Collections.Generic;
using System.Linq;

public enum EstadoPedido { Pendiente, Procesando, Enviado, Entregado, Cancelado }

public class Pedido
{
    public int Id { get; set; }
    public string Cliente { get; set; }
    public decimal Total { get; set; }
    public EstadoPedido Estado { get; set; }
    public DateTime Fecha { get; set; }

    public Pedido(int id, string cliente, decimal total)
    {
        Id = id;
        Cliente = cliente;
        Total = total;
        Estado = EstadoPedido.Pendiente;
        Fecha = DateTime.Now;
    }

    public override string ToString() =>
        $"[{Id}] {Cliente} - {Total:C} ({Estado})";
}

public class SistemaProcesamientoPedidos
{
    private readonly Dictionary<int, Pedido> _pedidos = new();
    private readonly Queue<Pedido> _colaProcesamiento = new();
    private readonly Stack<Pedido> _historial = new();
    private readonly HashSet<int> _idsProcesados = new();
    private int _siguienteId = 1;

    public Pedido CrearPedido(string cliente, decimal total)
    {
        var pedido = new Pedido(_siguienteId++, cliente, total);
        _pedidos[pedido.Id] = pedido;
        _colaProcesamiento.Enqueue(pedido);
        return pedido;
    }

    public List<Pedido> ProcesarLote(int cantidad)
    {
        var procesados = new List<Pedido>();

        for (int i = 0; i < cantidad && _colaProcesamiento.Count > 0; i++)
        {
            if (_colaProcesamiento.TryDequeue(out var pedido))
            {
                pedido.Estado = EstadoPedido.Procesando;
                _idsProcesados.Add(pedido.Id);
                _historial.Push(pedido);
                procesados.Add(pedido);
            }
        }

        return procesados;
    }

    public void EnviarPedido(int id)
    {
        if (_pedidos.TryGetValue(id, out var pedido) && pedido.Estado == EstadoPedido.Procesando)
            pedido.Estado = EstadoPedido.Enviado;
    }

    public void EntregarPedido(int id)
    {
        if (_pedidos.TryGetValue(id, out var pedido) && pedido.Estado == EstadoPedido.Enviado)
            pedido.Estado = EstadoPedido.Entregado;
    }

    public void CancelarPedido(int id)
    {
        if (_pedidos.TryGetValue(id, out var pedido))
        {
            pedido.Estado = EstadoPedido.Cancelado;
            _idsProcesados.Remove(id);
        }
    }

    public Pedido? UltimoProcesado() =>
        _historial.TryPeek(out var p) ? p : null;

    public bool EsProcesado(int id) => _idsProcesados.Contains(id);

    public List<Pedido> PedidosPorEstado(EstadoPedido estado) =>
        _pedidos.Values.Where(p => p.Estado == estado).ToList();

    public void MostrarResumen()
    {
        Console.WriteLine($"Total pedidos: {_pedidos.Count}");
        Console.WriteLine($"En cola: {_colaProcesamiento.Count}");
        Console.WriteLine($"Procesados hoy: {_idsProcesados.Count}");
        Console.WriteLine($"Historial: {_historial.Count}");

        var porEstado = _pedidos.Values.GroupBy(p => p.Estado);
        foreach (var grupo in porEstado)
            Console.WriteLine($"  {grupo.Key}: {grupo.Count()}");
    }
}

public class Programa
{
    public static void Main()
    {
        var sistema = new SistemaProcesamientoPedidos();

        // Crear pedidos
        sistema.CrearPedido("Ana García", 1500.00m);
        sistema.CrearPedido("Carlos López", 2300.50m);
        sistema.CrearPedido("María Rodríguez", 890.00m);
        sistema.CrearPedido("Pedro Martínez", 3200.00m);
        sistema.CrearPedido("Laura Fernández", 1750.75m);

        // Procesar lote
        var procesados = sistema.ProcesarLote(3);
        Console.WriteLine("Procesados:");
        foreach (var p in procesados)
            Console.WriteLine($"  {p}");

        // Enviar y entregar
        sistema.EnviarPedido(1);
        sistema.EntregarPedido(1);
        sistema.EnviarPedido(2);
        sistema.CancelarPedido(4);

        Console.WriteLine("\nÚltimo procesado:");
        Console.WriteLine($"  {sistema.UltimoProcesado()}");

        Console.WriteLine("\nResumen:");
        sistema.MostrarResumen();

        Console.WriteLine("\nPedidos entregados:");
        foreach (var p in sistema.PedidosPorEstado(EstadoPedido.Entregado))
            Console.WriteLine($"  {p}");

        // Usar LINQ con colecciones
        var stats = new Dictionary<string, int>
        {
            ["Procesados"] = procesados.Count,
            ["Pendientes"] = sistema.PedidosPorEstado(EstadoPedido.Pendiente).Count,
            ["Cancelados"] = sistema.PedidosPorEstado(EstadoPedido.Cancelado).Count,
        };

        var top = stats.OrderByDescending(kvp => kvp.Value).First();
        Console.WriteLine($"\nMayor categoría: {top.Key} con {top.Value}");
    }
}
```

================================================================================
  FAQ — PREGUNTAS FRECUENTES
================================================================================

P: ¿Cuándo usar List<T> vs array?
R: Usa List<T> cuando el tamaño sea dinámico. Usa array cuando el tamaño
   sea fijo y necesites máximo rendimiento o interoperabilidad.

P: ¿Dictionary<K,V> permite claves null?
R: Depende del comparador. El comparador por defecto permite null para
   tipos de referencia. Usa TryGetValue para acceso seguro.

P: ¿Qué pasa si modifico una colección durante un foreach?
R: Se lanza InvalidOperationException. Usa for con índice o copia la
   colección con ToList().

P: ¿HashSet<T> mantiene el orden de inserción?
R: No. Para orden de inserción, usa List<T> o LinkedHashSet (no incluido
   en BCL; usa第三方 librerías).

P: ¿Cómo elijo entre Queue<T> y ConcurrentQueue<T>?
R: Siempre ConcurrentQueue<T> si hay múltiples hilos. Queue<T> solo para
   uso en un solo hilo.

P: ¿Stack<T> tiene límite de capacidad?
R: No, crece dinámicamente igual que List<T>.

P: ¿LinkedList<T> es más lenta que List<T>?
R: Para iteración, sí (peor localidad de caché). Para inserciones/
   eliminaciones en medio, LinkedList<T> es O(1) vs O(n) de List<T>.

P: ¿Puedo usar record como clave en Dictionary?
R: Sí, porque los records implementan igualdad estructural y GetHashCode
   automáticamente.

  DESAFÍO DEL ARQUITECTO
================================================================================

Diseña un motor de procesamiento de eventos en tiempo real utilizando
todas las colecciones aprendidas:

- Los eventos llegan a una ConcurrentQueue<Event> desde múltiples fuentes.
- Un Event tiene Timestamp, Tipo, Datos (Dictionary<string, object>),
  Prioridad y Estado.
- Los eventos se clasifican por tipo en un Dictionary<string,
  ConcurrentQueue<Event>>.
- Los eventos procesados se archivan en un Stack<Event> (historial).
- Los eventos fallidos se almacenan en una cola de reintentos con
  límite de 3 intentos (Queue<Event>).
- Los IDs de eventos procesados se guardan en un HashSet<string> para
  evitar duplicados.
- Las estadísticas en tiempo real se mantienen en un
  ConcurrentDictionary<string, long> (tipo de evento -> contador).
- Implementa:
  - Productor(es) que generan eventos aleatorios.
  - Consumidor(es) que procesan eventos con diferentes velocidades.
  - Un monitor que muestre estadísticas cada 5 segundos.
  - Un sistema de prioridades donde eventos críticos se procesan antes.
  - Persistencia periódica a archivo JSON de las estadísticas.
  - Capacidad de pausar/reanudar el procesamiento.

Requisitos adicionales:
- Usar CancellationToken para detener gracefulmente.
- Logging de cada evento procesado.
- Métricas de rendimiento (eventos/segundo).
- Manejo de eventos huérfanos (sin tipo reconocido).

Este desafío integra colecciones, concurrencia y procesamiento en
tiempo real en un sistema completo.
