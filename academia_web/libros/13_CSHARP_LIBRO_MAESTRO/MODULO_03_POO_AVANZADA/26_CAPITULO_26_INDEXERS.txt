26.1 — ¿QUÉ ES UN INDEXADOR?
-------------------------------

Un indexador es un miembro especial que permite acceder a una instancia
de una clase o struct usando la sintaxis de corchetes (this[]), similar
a los arrays.

```csharp
public class ColeccionPalabras
{
    private string[] _palabras = new string[10];

    // Indexador
    public string this[int index]
    {
        get => _palabras[index];
        set => _palabras[index] = value;
    }
}

// Uso
var coleccion = new ColeccionPalabras();
coleccion[0] = "Hola";
coleccion[1] = "Mundo";
Console.WriteLine(coleccion[0]); // Hola
Console.WriteLine(coleccion[1]); // Mundo
```

26.2 — SINTAXIS BÁSICA
------------------------

```csharp
[modificador] tipo this[tipoParametro param]
{
    get { return ...; }
    set { ... = value; }
}

// Equivalentes a propiedades, pero con parámetros
```

Características:
- No tienen nombre propio; se declaran con this.
- Pueden tener get y/o set.
- Pueden tener cualquier tipo de parámetro (int, string, enum, etc.).
- Pueden ser virtuales, abstractos, override, sealed.
- Pueden tener múltiples parámetros.
- No pueden ser static (pero su contenedor sí puede serlo).

26.3 — INDEXADOR DE SOLO LECTURA
----------------------------------

```csharp
public class TemperaturasMensuales
{
    private readonly double[] _temperaturas = new double[12];

    public TemperaturasMensuales()
    {
        // Valores por defecto
        for (int i = 0; i < 12; i++)
            _temperaturas[i] = 15.0 + (i * 2.5);
    }

    // Indexador de solo lectura (sin set)
    public double this[int mes] => _temperaturas[mes];

    // Indexador con string (nombre del mes)
    public double this[string nombreMes]
    {
        get
        {
            int idx = nombreMes.ToLower() switch
            {
                "enero" => 0, "febrero" => 1, "marzo" => 2,
                "abril" => 3, "mayo" => 4, "junio" => 5,
                "julio" => 6, "agosto" => 7, "septiembre" => 8,
                "octubre" => 9, "noviembre" => 10, "diciembre" => 11,
                _ => throw new ArgumentOutOfRangeException(nameof(nombreMes))
            };
            return _temperaturas[idx];
        }
    }
}

// Uso
var temps = new TemperaturasMensuales();
Console.WriteLine(temps[0]);       // 15
Console.WriteLine(temps["julio"]); // 30
```

26.4 — MÚLTIPLES PARÁMETROS
-----------------------------

```csharp
public class Matriz2D<T>
{
    private readonly T[,] _datos;

    public Matriz2D(int filas, int columnas)
    {
        _datos = new T[filas, columnas];
    }

    public int Filas => _datos.GetLength(0);
    public int Columnas => _datos.GetLength(1);

    // Indexador con dos parámetros
    public T this[int fila, int columna]
    {
        get => _datos[fila, columna];
        set => _datos[fila, columna] = value;
    }

    // Indexador con un solo parámetro (fila completa)
    public T[] this[int fila]
    {
        get
        {
            var resultado = new T[Columnas];
            for (int c = 0; c < Columnas; c++)
                resultado[c] = _datos[fila, c];
            return resultado;
        }
    }
}

// Uso
var matriz = new Matriz2D<double>(3, 3);
matriz[0, 0] = 1.0;
matriz[0, 1] = 2.0;
matriz[1, 0] = 3.0;
Console.WriteLine(matriz[0, 0]); // 1
var fila0 = matriz[0]; // { 1, 2, 0 }
```

26.5 — INDEXADOR CON ENUM
---------------------------

```csharp
public enum DiaLaboral { Lunes, Martes, Miercoles, Jueves, Viernes }

public class HorarioSemanal
{
    private readonly Dictionary<DayOfWeek, TimeSpan> _horarios = new();

    // Indexador con DayOfWeek
    public TimeSpan this[DayOfWeek dia]
    {
        get => _horarios.TryGetValue(dia, out var h) ? h : TimeSpan.Zero;
        set => _horarios[dia] = value;
    }

    // Indexador con string (nombre del día)
    public TimeSpan this[string dia]
    {
        get
        {
            if (Enum.TryParse<DayOfWeek>(dia, true, out var d))
                return this[d];
            return TimeSpan.Zero;
        }
    }

    // Indexador con int (0=Lunes, ..., 6=Domingo)
    public TimeSpan this[int dia]
    {
        get => this[(DayOfWeek)dia];
        set => this[(DayOfWeek)dia] = value;
    }
}

// Uso
var horario = new HorarioSemanal();
horario[DayOfWeek.Monday] = new TimeSpan(9, 0, 0);
horario[DayOfWeek.Tuesday] = new TimeSpan(9, 0, 0);
horario["Wednesday"] = new TimeSpan(10, 0, 0);
Console.WriteLine(horario[1]); // 09:00:00
Console.WriteLine(horario["jueves"]); // 00:00:00 (no definido)
```

26.6 — INDEXADOR GENÉRICO
---------------------------

```csharp
public class CacheIndexado<TKey, TValue>
{
    private readonly Dictionary<TKey, TValue> _cache = new();

    // Indexador genérico
    public TValue this[TKey key]
    {
        get => _cache.TryGetValue(key, out var val) ? val : default!;
        set => _cache[key] = value;
    }

    public bool Contiene(TKey key) => _cache.ContainsKey(key);
    public int Count => _cache.Count;
    public void Limpiar() => _cache.Clear();
    public IEnumerable<TKey> Keys => _cache.Keys;
}

// Uso
var cache = new CacheIndexado<string, int>();
cache["contador"] = 42;
Console.WriteLine(cache["contador"]); // 42
Console.WriteLine(cache["noexiste"]); // 0 (default)
```

26.7 — INDEXADOR VIRTUAL Y POLIMORFISMO
-----------------------------------------

```csharp
public abstract class RepositorioBase<T>
{
    protected List<T> _items = new();

    public abstract T this[int index] { get; set; }

    public void Agregar(T item) => _items.Add(item);
    public int Count => _items.Count;
}

public class RepositorioEstricto<T> : RepositorioBase<T>
{
    public override T this[int index]
    {
        get
        {
            if (index < 0 || index >= _items.Count)
                throw new IndexOutOfRangeException();
            return _items[index];
        }
        set
        {
            if (index < 0 || index >= _items.Count)
                throw new IndexOutOfRangeException();
            _items[index] = value;
        }
    }
}

public class RepositorioFlexible<T> : RepositorioBase<T>
{
    public override T this[int index]
    {
        get
        {
            if (index < 0 || index >= _items.Count)
                return default!;
            return _items[index];
        }
        set
        {
            while (_items.Count <= index)
                _items.Add(default!);
            _items[index] = value;
        }
    }
}
```

26.8 — INTERFAZ CON INDEXADOR
-------------------------------

```csharp
public interface IIndexable<T>
{
    T this[int index] { get; set; }
    int Count { get; }
}

public class ListaIndexable<T> : IIndexable<T>
{
    private readonly List<T> _items = new();

    public T this[int index]
    {
        get => _items[index];
        set => _items[index] = value;
    }

    public int Count => _items.Count;

    public void Add(T item) => _items.Add(item);
}

public static class IndexableExtensions
{
    public static void MostrarTodos<T>(this IIndexable<T> indexable)
    {
        for (int i = 0; i < indexable.Count; i++)
            Console.WriteLine($"[{i}]: {indexable[i]}");
    }

    public static T[] ToArray<T>(this IIndexable<T> indexable)
    {
        var arr = new T[indexable.Count];
        for (int i = 0; i < arr.Length; i++)
            arr[i] = indexable[i];
        return arr;
    }
}

// Uso
var lista = new ListaIndexable<string>();
lista.Add("A");
lista.Add("B");
lista.Add("C");
lista.MostrarTodos();
```

26.9 — INDEXADOR EN STRUCT
----------------------------

```csharp
public readonly struct Vector
{
    private readonly double[] _componentes;

    public Vector(params double[] componentes)
    {
        _componentes = new double[componentes.Length];
        componentes.CopyTo(_componentes, 0);
    }

    public int Dimension => _componentes.Length;

    // Indexador en struct (solo get, readonly)
    public double this[int index] => _componentes[index];

    // Indexador con string
    public double this[string nombre]
    {
        get => nombre.ToLower() switch
        {
            "x" when Dimension >= 1 => _componentes[0],
            "y" when Dimension >= 2 => _componentes[1],
            "z" when Dimension >= 3 => _componentes[2],
            "w" when Dimension >= 4 => _componentes[3],
            _ => throw new ArgumentException($"Componente '{nombre}' no disponible")
        };
    }

    public double Magnitud
    {
        get
        {
            double suma = 0;
            for (int i = 0; i < Dimension; i++)
                suma += _componentes[i] * _componentes[i];
            return Math.Sqrt(suma);
        }
    }
}

// Uso
var v = new Vector(3.0, 4.0);
Console.WriteLine(v[0]);    // 3
Console.WriteLine(v["x"]);  // 3
Console.WriteLine(v["y"]);  // 4
Console.WriteLine(v.Magnitud); // 5
```

26.11 — INDEXADOR ESTÁTICO
----------------------------

Los indexadores no pueden ser static. Para simularlo, usa
una propiedad estática que devuelva una instancia singleton.

```csharp
public class Configuracion
{
    private static readonly Dictionary<string, string> _valores = new();

    public static Configuracion Instance { get; } = new();

    // Indexador de instancia
    public string this[string clave]
    {
        get => _valores.TryGetValue(clave, out var val) ? val : "";
        set => _valores[clave] = value;
    }
}

// Uso (simula indexador estático)
Configuracion.Instance["api_url"] = "https://api.example.com";
Console.WriteLine(Configuracion.Instance["api_url"]);
```

26.12 — SOBRECARGA DE INDEXADORES
------------------------------------

Puedes tener múltiples indexadores en la misma clase, siempre que
tengan diferentes firmas de parámetros.

```csharp
public class DatosMultiAcceso
{
    private readonly Dictionary<string, object> _dic = new();
    private readonly List<object> _lista = new();

    // Indexador por string (diccionario)
    public object this[string key]
    {
        get => _dic.TryGetValue(key, out var val) ? val : null!;
        set => _dic[key] = value;
    }

    // Indexador por int (lista)
    public object this[int index]
    {
        get => _lista[index];
        set
        {
            if (index >= _lista.Count)
                _lista.Add(value);
            else
                _lista[index] = value;
        }
    }

    // Indexador por (int, int) — matriz virtual
    public int this[int fila, int columna]
    {
        get => fila * 10 + columna;
    }

    // Indexador por rango (C# 8+)
    public object[] this[Range rango]
    {
        get => _lista.ToArray()[rango];
    }
}

// Uso
var datos = new DatosMultiAcceso();
datos["nombre"] = "Ana";
datos[0] = "Primero";
Console.WriteLine(datos["nombre"]); // Ana
Console.WriteLine(datos[0]);        // Primero
Console.WriteLine(datos[2, 3]);     // 23
```

26.13 — INDEXADOR CON VALIDACIÓN
-----------------------------------

```csharp
public class Inventario
{
    private readonly Dictionary<int, Producto> _productos = new();

    public Producto this[int id]
    {
        get
        {
            if (id <= 0)
                throw new ArgumentException("ID debe ser positivo");
            return _productos.TryGetValue(id, out var p) ? p : null!;
        }
        set
        {
            if (id <= 0)
                throw new ArgumentException("ID debe ser positivo");
            if (value == null)
                throw new ArgumentNullException(nameof(value));
            _productos[id] = value;
        }
    }

    public Producto this[string nombre]
    {
        get => _productos.Values.FirstOrDefault(p =>
            p.Nombre.Equals(nombre, StringComparison.OrdinalIgnoreCase))!;
    }
}

public class Producto
{
    public int Id { get; set; }
    public string Nombre { get; set; } = "";
    public decimal Precio { get; set; }
}
```

26.14 — INDEXADOR EN COLECCIONES PERSONALIZADAS
--------------------------------------------------

```csharp
public class PaginatedList<T>
{
    private readonly List<T> _todosLosItems;
    private readonly int _tamanoPagina;

    public PaginatedList(IEnumerable<T> items, int tamanoPagina = 10)
    {
        _todosLosItems = items.ToList();
        _tamanoPagina = tamanoPagina;
    }

    public int TotalPaginas => (int)Math.Ceiling((double)_todosLosItems.Count / _tamanoPagina);
    public int TotalItems => _todosLosItems.Count;

    // Indexador que devuelve una página
    public List<T> this[int numeroPagina]
    {
        get
        {
            if (numeroPagina < 0 || numeroPagina >= TotalPaginas)
                throw new ArgumentOutOfRangeException(nameof(numeroPagina));

            int inicio = numeroPagina * _tamanoPagina;
            int count = Math.Min(_tamanoPagina, _todosLosItems.Count - inicio);
            return _todosLosItems.GetRange(inicio, count);
        }
    }

    // Indexador con rango
    public IEnumerable<T> this[Range rango]
    {
        get
        {
            var (offset, length) = rango.GetOffsetAndLength(_todosLosItems.Count);
            return _todosLosItems.Skip(offset).Take(length);
        }
    }
}

// Uso
var numeros = Enumerable.Range(1, 100).ToList();
var paginador = new PaginatedList<int>(numeros, 10);
var pagina3 = paginador[2]; // Items 21-30
Console.WriteLine(string.Join(", ", pagina3));
```

26.15 — EJEMPLO COMPLETO: SISTEMA DE ARCHIVOS VIRTUAL
-------------------------------------------------------

```csharp
using System;
using System.Collections.Generic;
using System.Linq;

public class ArchivoVirtual
{
    public string Nombre { get; set; }
    public string Contenido { get; set; }
    public DateTime FechaCreacion { get; set; }
    public bool EsDirectorio { get; set; }

    public ArchivoVirtual(string nombre, string contenido = "", bool esDirectorio = false)
    {
        Nombre = nombre;
        Contenido = contenido;
        FechaCreacion = DateTime.Now;
        EsDirectorio = esDirectorio;
    }

    public int Tamano => EsDirectorio ? 0 : Contenido.Length;

    public override string ToString() =>
        $"{(EsDirectorio ? "[DIR]" : "[FILE]")} {Nombre,-20} {Tamano,8} bytes  {FechaCreacion:g}";
}

public class SistemaArchivos
{
    private readonly List<ArchivoVirtual> _archivos = new();
    private readonly Dictionary<string, int> _indicePorNombre = new(StringComparer.OrdinalIgnoreCase);

    public void Agregar(ArchivoVirtual archivo)
    {
        _archivos.Add(archivo);
        _indicePorNombre[archivo.Nombre] = _archivos.Count - 1;
    }

    public bool Eliminar(string nombre)
    {
        if (_indicePorNombre.TryGetValue(nombre, out int idx))
        {
            _archivos.RemoveAt(idx);
            Reindexar();
            return true;
        }
        return false;
    }

    private void Reindexar()
    {
        _indicePorNombre.Clear();
        for (int i = 0; i < _archivos.Count; i++)
            _indicePorNombre[_archivos[i].Nombre] = i;
    }

    // Indexador por índice
    public ArchivoVirtual this[int index]
    {
        get
        {
            if (index < 0 || index >= _archivos.Count)
                throw new IndexOutOfRangeException();
            return _archivos[index];
        }
    }

    // Indexador por nombre
    public ArchivoVirtual this[string nombre]
    {
        get
        {
            if (_indicePorNombre.TryGetValue(nombre, out int idx))
                return _archivos[idx];
            return null!;
        }
    }

    // Indexador por tipo (archivos o directorios)
    public List<ArchivoVirtual> this[bool soloDirectorios]
    {
        get => _archivos.Where(a => a.EsDirectorio == soloDirectorios).ToList();
    }

    // Indexador por extensión
    public List<ArchivoVirtual> this[string extension, bool porExtension]
    {
        get => _archivos.Where(a =>
            a.Nombre.EndsWith(extension, StringComparison.OrdinalIgnoreCase)).ToList();
    }

    // Indexador por rango de fechas
    public List<ArchivoVirtual> this[DateTime desde, DateTime hasta]
    {
        get => _archivos.Where(a =>
            a.FechaCreacion >= desde && a.FechaCreacion <= hasta).ToList();
    }

    public int Count => _archivos.Count;

    public void Listar()
    {
        foreach (var a in _archivos)
            Console.WriteLine(a);
        Console.WriteLine($"\nTotal: {_archivos.Count} archivos");
    }
}

public class Programa
{
    public static void Main()
    {
        var fs = new SistemaArchivos();

        fs.Agregar(new ArchivoVirtual("documento.txt", "Contenido del documento"));
        fs.Agregar(new ArchivoVirtual("imagen.png", "datos binarios..."));
        fs.Agregar(new ArchivoVirtual("datos.csv", "a,b,c\n1,2,3"));
        fs.Agregar(new ArchivoVirtual("scripts", "", esDirectorio: true));
        fs.Agregar(new ArchivoVirtual("fotos", "", esDirectorio: true));
        fs.Agregar(new ArchivoVirtual("presentacion.pptx", "slides..."));
        fs.Agregar(new ArchivoVirtual("notas.txt", "Notas importantes"));

        Console.WriteLine("=== TODOS LOS ARCHIVOS ===");
        fs.Listar();

        Console.WriteLine("\n=== ACCESO POR ÍNDICE ===");
        Console.WriteLine(fs[0]);

        Console.WriteLine("\n=== ACCESO POR NOMBRE ===");
        Console.WriteLine(fs["datos.csv"]);

        Console.WriteLine("\n=== SOLO DIRECTORIOS ===");
        foreach (var d in fs[true])
            Console.WriteLine(d);

        Console.WriteLine("\n=== ARCHIVOS .TXT ===");
        foreach (var f in fs[".txt", true])
            Console.WriteLine(f);

        Console.WriteLine("\n=== ARCHIVOS CREADOS HOY ===");
        foreach (var f in fs[DateTime.Today, DateTime.Today.AddDays(1)])
            Console.WriteLine(f);

        // Modificar contenido vía indexador
        var doc = fs["documento.txt"];
        if (doc != null)
        {
            doc.Contenido = "Contenido actualizado";
            Console.WriteLine($"\nContenido actualizado: {doc.Contenido}");
        }

        Console.WriteLine($"\nTotal archivos: {fs.Count}");
    }
}
```

================================================================================
  FAQ — PREGUNTAS FRECUENTES
================================================================================

P: ¿Puedo tener indexadores estáticos?
R: No. Los indexadores requieren una instancia. Usa un singleton
   con indexador de instancia para simularlo.

P: ¿Los indexadores pueden ser virtuales?
R: Sí. Puedes declararlos virtual, override, abstract o sealed.

P: ¿Puedo tener varios indexadores en la misma clase?
R: Sí, mientras tengan diferentes firmas (parámetros de distinto tipo
   o número).

P: ¿Los indexadores pueden ser async?
R: No directamente. Los indexadores no soportan el modificador async.
   Puedes devolver Task<T> pero el get/set no serán asíncronos.

P: ¿Puedo usar expression-bodied para indexadores?
R: Sí: public T this[int i] => _items[i];

P: ¿Los indexadores pueden lanzar excepciones?
R: Sí, y es recomendable validar parámetros en el get/set.

P: ¿Cuál es la diferencia entre un indexador y una propiedad?
R: Las propiedades no tienen parámetros; los indexadores sí.
   Las propiedades tienen nombre; los indexadores usan this.

P: ¿Puedo implementar una interfaz con indexador?
R: Sí. Las interfaces pueden declarar indexadores.

  (convierte automáticamente a coordenadas).

- Las celdas pueden tener fórmulas como "=A1+B2" que se evalúan
  automáticamente.
- Recálculo automático en orden topológico cuando cambia una celda.
- Formato condicional: this[int fila, int columna, string formato]
  para establecer formato (color, negrita, número decimales).
- Exportar a CSV mediante un indexador this[bool exportarCSV].
- Historial de cambios con indexador por fecha/hora.
- Detección de referencias circulares.

El sistema debe tener al menos 5 indexadores diferentes. Debe incluir
validación, manejo de errores y un ejemplo funcional que demuestre
todas las capacidades.

Este desafío integra indexadores, parsing, grafos (topological sort),
y diseño de API limpia en un escocio real.
