22.1 — ¿QUÉ ES UN ENUM?
-------------------------

Un enum (enumeración) es un tipo de valor formado por un conjunto de
constantes con nombre llamado lista de enumeradores. Cada enumerador
tiene un valor entero asociado.

```csharp
enum DiaSemana
{
    Lunes,       // 0
    Martes,      // 1
    Miercoles,   // 2
    Jueves,      // 3
    Viernes,     // 4
    Sabado,      // 5
    Domingo      // 6
}
```

Uso básico:

```csharp
DiaSemana hoy = DiaSemana.Miercoles;
Console.WriteLine(hoy); // Miercoles
Console.WriteLine((int)hoy); // 2

if (hoy == DiaSemana.Sabado || hoy == DiaSemana.Domingo)
    Console.WriteLine("¡Fin de semana!");
else
    Console.WriteLine("Día laboral");
```

22.2 — TIPO SUBYACENTE (UNDERLYING TYPE)
------------------------------------------

Por defecto, el tipo subyacente es int. Puedes cambiarlo a byte, sbyte,
short, ushort, uint, long o ulong.

```csharp
enum EstadoPedido : byte
{
    Pendiente,      // 0
    Confirmado,     // 1
    EnPreparacion,  // 2
    Enviado,        // 3
    Entregado,      // 4
    Cancelado       // 5
}

enum Permisos : long
{
    Ninguno = 0,
    Lectura = 1L << 0,
    Escritura = 1L << 1,
    Ejecucion = 1L << 2,
    Admin = 1L << 63
}
```

22.4 — EL ATRIBUTO [FLAGS]
---------------------------

El atributo Flags indica que el enum puede tratarse como una máscara
de bits, permitiendo combinar valores con operadores |, &, ^.

```csharp
[Flags]
enum DiasLaborales
{
    Ninguno = 0,
    Lunes = 1 << 0,      // 1
    Martes = 1 << 1,     // 2
    Miercoles = 1 << 2,  // 4
    Jueves = 1 << 3,     // 8
    Viernes = 1 << 4,    // 16
    Sabado = 1 << 5,     // 32
    Domingo = 1 << 6,    // 64
    EntreSemana = Lunes | Martes | Miercoles | Jueves | Viernes,
    FinDeSemana = Sabado | Domingo,
    TodaLaSemana = EntreSemana | FinDeSemana
}
```

Uso:

```csharp
DiasLaborales trabajo = DiasLaborales.Lunes | DiasLaborales.Martes | DiasLaborales.Miercoles;

// Verificar si contiene un valor
if (trabajo.HasFlag(DiasLaborales.Lunes))
    Console.WriteLine("Trabajas los lunes");

// Añadir un día
trabajo |= DiasLaborales.Jueves;

// Quitar un día
trabajo &= ~DiasLaborales.Martes;

// Verificar combinación
if ((trabajo & DiasLaborales.FinDeSemana) == 0)
    Console.WriteLine("No trabajas fines de semana");
```

22.5 — CONVERSIONES Y PARSING
-------------------------------

```csharp
// De número a enum
EstadoPedido estado = (EstadoPedido)3; // Enviado

// De string a enum
string input = "Confirmado";
EstadoPedido parsed = Enum.Parse<EstadoPedido>(input);

// Enum.Parse con ignoreCase
EstadoPedido parsed2 = Enum.Parse<EstadoPedido>("confirmado", ignoreCase: true);

// TryParse (seguro)
if (Enum.TryParse<EstadoPedido>("Entregado", out var resultado))
    Console.WriteLine(resultado); // Entregado

// Obtener todos los nombres
string[] nombres = Enum.GetNames<EstadoPedido>();

// Obtener todos los valores
int[] valores = Enum.GetValues<EstadoPedido>().Cast<int>().ToArray();

// Obtener el nombre de un valor específico
string nombre = Enum.GetName(typeof(EstadoPedido), 4); // Entregado

// De enum a string
string estadoStr = EstadoPedido.Enviado.ToString(); // "Enviado"

// De enum a entero
int estadoInt = (int)EstadoPedido.Enviado; // 3

// Formato "D" (valor decimal)
string valDecimal = EstadoPedido.Enviado.ToString("D"); // "3"

// Formato "X" (hexadecimal)
string valHex = EstadoPedido.Enviado.ToString("X"); // "00000003"
```

22.7 — ENUMS CON ATRIBUTOS PERSONALIZADOS
-------------------------------------------

```csharp
[AttributeUsage(AttributeTargets.Field)]
public class DescripcionAttribute : Attribute
{
    public string Texto { get; }
    public DescripcionAttribute(string texto) => Texto = texto;
}

public enum Prioridad
{
    [Descripcion("Baja - Sin urgencia")]
    Baja = 1,

    [Descripcion("Media - Urgencia normal")]
    Media = 2,

    [Descripcion("Alta - Urgente")]
    Alta = 3,

    [Descripcion("Crítica - Resolver inmediatamente")]
    Critica = 4
}

// Método helper para obtener la descripción
public static class EnumExtensions
{
    public static string ObtenerDescripcion(this Enum valor)
    {
        var field = valor.GetType().GetField(valor.ToString());
        var attr = field?.GetCustomAttribute<DescripcionAttribute>();
        return attr?.Texto ?? valor.ToString();
    }
}

// Uso
Prioridad p = Prioridad.Critica;
Console.WriteLine(p.ObtenerDescripcion()); // "Crítica - Resolver inmediatamente"
```

22.9 — FLAGS AVANZADO — OPERACIONES BITS
------------------------------------------

```csharp
[Flags]
enum PermisosArchivo
{
    Ninguno = 0,
    Leer = 1,
    Escribir = 2,
    Ejecutar = 4,
    Borrar = 8,
    Todo = Leer | Escribir | Ejecutar | Borrar
}

public class Archivo
{
    public string Nombre { get; set; }
    public PermisosArchivo Permisos { get; set; }

    public bool TienePermiso(PermisosArchivo permiso) =>
        Permisos.HasFlag(permiso);

    public void Conceder(PermisosArchivo permiso) =>
        Permisos |= permiso;

    public void Revocar(PermisosArchivo permiso) =>
        Permisos &= ~permiso;
}

// Uso
var archivo = new Archivo { Nombre = "reporte.pdf", Permisos = PermisosArchivo.Leer };
archivo.Conceder(PermisosArchivo.Escribir);
Console.WriteLine(archivo.TienePermiso(PermisosArchivo.Ejecutar)); // False
```

```csharp
public static class EnumHelper
{
    public static T[] GetValues<T>() where T : Enum
        => Enum.GetValues<T>();

    public static IEnumerable<(T Valor, string Nombre)> Listar<T>() where T : Enum
        => GetValues<T>().Select(v => (v, v.ToString()));

    public static T Parse<T>(string value, bool ignoreCase = true)
        where T : Enum
        => Enum.Parse<T>(value, ignoreCase);
}

// Uso

{
    Console.WriteLine($"{nombre} = {(int)(object)valor}");
```

22.11 — VALIDACIÓN DE ENUMS
-----------------------------

```csharp
public static bool EsValido<T>(T value) where T : Enum
    => Enum.IsDefined(typeof(T), value);

public static bool EsValido<T>(string value) where T : Enum
{
    foreach (var name in Enum.GetNames<T>())
        if (name.Equals(value, StringComparison.OrdinalIgnoreCase))
            return true;
    return false;
}

// Para flags, la validación es diferente
public static bool EsValidoFlags<T>(T value) where T : Enum
{
    int intValue = Convert.ToInt32(value);
    int maxValue = Enum.GetValues<T>().Cast<int>().Aggregate((a, b) => a | b);
    return (intValue & ~maxValue) == 0;
}
```

22.12 — ENUMS EN CLASES DE DOMINIO
------------------------------------

```csharp
public enum TipoCuenta
{
    Ahorro,
    Corriente,
    Inversion,
    Credito
}

public enum EstadoCuenta
{
    Activa,
    Congelada,
    Cerrada,
    Suspendida
}

public class CuentaBancaria
{
    public string Numero { get; set; }
    public string Titular { get; set; }
    public TipoCuenta Tipo { get; set; }
    public EstadoCuenta Estado { get; set; }
    public decimal Saldo { get; private set; }

    public CuentaBancaria(string numero, string titular, TipoCuenta tipo)
    {
        Numero = numero;
        Titular = titular;
        Tipo = tipo;
        Estado = EstadoCuenta.Activa;
        Saldo = 0;
    }

    public string Depositar(decimal monto)
    {
        if (Estado != EstadoCuenta.Activa)
            return "Cuenta no activa";

        Saldo += monto;
        return $"Depósito exitoso. Nuevo saldo: {Saldo:C}";
    }

    public string Retirar(decimal monto)
    {
        if (Estado != EstadoCuenta.Activa)
            return "Cuenta no activa";

        if (monto > Saldo && Tipo != TipoCuenta.Credito)
            return "Saldo insuficiente";

        Saldo -= monto;
        return $"Retiro exitoso. Nuevo saldo: {Saldo:C}";
    }

    public void CambiarEstado(EstadoCuenta nuevoEstado)
    {
        Estado = nuevoEstado;
    }

    public override string ToString() =>
        $"[{Tipo}] {Numero} - {Titular} ({Estado}) - Saldo: {Saldo:C}";
}
```

```csharp
public class Pedido
{
    public int Id { get; set; }
    public string Cliente { get; set; }
    public EstadoPedido Estado { get; set; }

    public enum EstadoPedido
    {
        Nuevo,
        Procesando,
        Enviado,
        Entregado,
        Cancelado,
        Reembolsado
    }

    public Pedido(int id, string cliente)
    {
        Id = id;
        Cliente = cliente;
        Estado = EstadoPedido.Nuevo;
    }

    public void AvanzarEstado()
    {
        Estado = Estado switch
        {
            EstadoPedido.Nuevo => EstadoPedido.Procesando,
            EstadoPedido.Procesando => EstadoPedido.Enviado,
            EstadoPedido.Enviado => EstadoPedido.Entregado,
            _ => Estado
        };
    }
}

// Uso
var pedido = new Pedido(1001, "María García");
pedido.AvanzarEstado();
Console.WriteLine(pedido.Estado); // Procesando
Console.WriteLine(Pedido.EstadoPedido.Entregado); // Acceso: Pedido.EstadoPedido.Entregado
```

```csharp
        NivelLog.Information => ConsoleColor.Green,

        return resultado;
```

22.15 — COMPARACIONES Y ORDENAMIENTO
--------------------------------------

Los enums son comparables porque su tipo subyacente es un valor numérico.

```csharp
Prioridad p1 = Prioridad.Baja;
Prioridad p2 = Prioridad.Critica;

Console.WriteLine(p1 < p2);  // True
Console.WriteLine(p2 > p1);  // True
Console.WriteLine(p1.CompareTo(p2)); // -1

// Ordenar lista de enums
var prioridades = new List<Prioridad>
{
    Prioridad.Critica,
    Prioridad.Baja,
    Prioridad.Media,
    Prioridad.Alta
};

prioridades.Sort();
foreach (var p in prioridades)
    Console.WriteLine(p); // Baja, Media, Alta, Critica

// También puedes ordenar por descripción
prioridades.Sort((a, b) => a.ObtenerDescripcion().CompareTo(b.ObtenerDescripcion()));
```

22.18 — EJEMPLO COMPLETO: SISTEMA DE PERMISOS
-----------------------------------------------

```csharp
using System;
using System.Collections.Generic;
using System.Linq;

[Flags]
enum Permiso
{
    Ninguno = 0,
    Leer = 1,
    Escribir = 2,
    Ejecutar = 4,
    Eliminar = 8,
    Compartir = 16,

    // Combinaciones comunes
    Visor = Leer,
    Editor = Leer | Escribir,
    Desarrollador = Leer | Escribir | Ejecutar,
    Administrador = Leer | Escribir | Ejecutar | Eliminar,
    Propietario = Leer | Escribir | Ejecutar | Eliminar | Compartir,
}

enum RecursoTipo
{
    Documento,
    Carpeta,
    Proyecto,
    Configuracion,
    Usuario
}

class Recurso
{
    public string Nombre { get; set; }
    public RecursoTipo Tipo { get; set; }
    public string Propietario { get; set; }

    public Recurso(string nombre, RecursoTipo tipo, string propietario)
    {
        Nombre = nombre;
        Tipo = tipo;
        Propietario = propietario;
    }
}

class UsuarioSistema
{
    public string Nombre { get; set; }
    public RolUsuario Rol { get; set; }

    public UsuarioSistema(string nombre, RolUsuario rol)
    {
        Nombre = nombre;
        Rol = rol;
    }

    public Permiso ObtenerPermisosBase() => Rol switch
    {
        RolUsuario.Visitante => Permiso.Leer,
        RolUsuario.Usuario => Permiso.Leer | Permiso.Escribir | Permiso.Compartir,
        RolUsuario.Moderador => Permiso.Leer | Permiso.Escribir | Permiso.Ejecutar | Permiso.Eliminar,
        RolUsuario.Admin => Permiso.Administrador,
        RolUsuario.SuperAdmin => Permiso.Propietario,
        _ => Permiso.Ninguno
    };
}

public class SistemaPermisos
{
    private readonly Dictionary<string, List<(UsuarioSistema Usuario, Permiso Permisos)>> _permisosExtra = new();

    public bool TieneAcceso(UsuarioSistema usuario, Recurso recurso, Permiso permisoRequerido)
    {
        Permiso permisos = usuario.ObtenerPermisosBase();

        if (_permisosExtra.TryGetValue(recurso.Nombre, out var extra))
        {
            var extraPermiso = extra
                .Where(e => e.Usuario.Nombre == usuario.Nombre)
                .Select(e => e.Permisos)
                .FirstOrDefault();
            permisos |= extraPermiso;
        }

        return permisos.HasFlag(permisoRequerido);
    }

    public void ConcederPermisoExtra(UsuarioSistema usuario, Recurso recurso, Permiso permiso)
    {
        if (!_permisosExtra.ContainsKey(recurso.Nombre))
            _permisosExtra[recurso.Nombre] = new();
        _permisosExtra[recurso.Nombre].Add((usuario, permiso));
    }

    public string ListarPermisos(UsuarioSistema usuario)
    {
        var basePermisos = usuario.ObtenerPermisosBase();
        var nombres = Enum.GetValues<Permiso>()
            .Where(p => p != Permiso.Ninguno && basePermisos.HasFlag(p))
            .Select(p => p.ToString());
        return string.Join(", ", nombres);
    }
}

public class Programa
{
    public static void Main()
    {
        var admin = new UsuarioSistema("Ana", RolUsuario.Admin);
        var usuario = new UsuarioSistema("Luis", RolUsuario.Usuario);

        var documento = new Recurso("reporte.pdf", RecursoTipo.Documento, "Ana");

        var sistema = new SistemaPermisos();

        Console.WriteLine($"¿Ana puede eliminar? {sistema.TieneAcceso(admin, documento, Permiso.Eliminar)}");
        Console.WriteLine($"¿Luis puede eliminar? {sistema.TieneAcceso(usuario, documento, Permiso.Eliminar)}");
        Console.WriteLine($"Permisos de Ana: {sistema.ListarPermisos(admin)}");
        Console.WriteLine($"Permisos de Luis: {sistema.ListarPermisos(usuario)}");

        sistema.ConcederPermisoExtra(usuario, documento, Permiso.Eliminar);
        Console.WriteLine($"¿Luis puede eliminar ahora? {sistema.TieneAcceso(usuario, documento, Permiso.Eliminar)}");

        // Demostraciones adicionales
        Console.WriteLine($"\nTodos los roles: {string.Join(", ", Enum.GetNames<RolUsuario>())}");
        Console.WriteLine($"Valor de Admin: {(int)RolUsuario.Admin}");

        // Parse desde string
        if (Enum.TryParse<RolUsuario>("Moderador", out var rol))
            Console.WriteLine($"Parseado: {rol}");

        // Iterar flags
        var permisosAdmin = admin.ObtenerPermisosBase();
        Console.Write("Permisos individuales: ");
        foreach (Permiso p in Enum.GetValues<Permiso>())
        {
            if (p != Permiso.Ninguno && permisosAdmin.HasFlag(p))
                Console.Write($"{p} ");
        }
        Console.WriteLine();
    }
}
```

================================================================================
  FAQ — PREGUNTAS FRECUENTES
================================================================================

P: ¿Puedo agregar métodos a un enum?
R: No directamente, pero puedes crear métodos de extensión (extension methods)
   para añadir funcionalidad.

P: ¿Enum es tipo de valor o referencia?
R: Los enums son tipos de valor. Heredan de System.Enum, que a su vez
   hereda de System.ValueType.

P: ¿Qué pasa si asigno un valor no definido a una variable enum?
R: El compilador no lo impide. Usa Enum.IsDefined para validar.

P: ¿Puedo usar enums con genéricos?
R: Sí, con la constraint where T : Enum (C# 7.3+).

P: ¿Cuándo debo usar [Flags]?
R: Cuando necesites combinar múltiples valores. Por ejemplo: permisos,
   opciones, atributos, días de la semana.

P: ¿Enum.ToString() es caro?
R: Usa reflexión la primera vez, pero los resultados se cachean.
   Para alto rendimiento, considera diccionarios precompilados.

P: ¿Puedo tener un enum con valor 0 repetido?
R: Sí, puedes tener múltiples nombres para el mismo valor. Enum.ToString()
   devolverá el primero definido.

P: ¿Los enums soportan herencia?
R: No. Un enum no puede heredar de otro enum, solo de System.Enum.
   Usa clases estáticas con constantes si necesitas "herencia".

empresarial con los siguientes requisitos:

  - Validación de que no se puedan conceder permisos contradictorios.

  - Un Invitado solo puede leer ciertos recursos.
  - Un Administrador puede todo excepto auditar.
  - Un Auditor solo puede leer y exportar.
