¡Bienvenido al Capítulo 27 del C# Master Book! Los operadores de
conversión permiten definir cómo se convierten tus tipos personalizados
de un tipo a otro. Puedes crear conversiones implícitas (sin casting)
y explícitas (con casting). Además, puedes sobrecargar operadores
aritméticos, de comparación y lógicos para que tus tipos se comporten
como tipos primitivos.

Esta característica es especialmente útil para crear tipos numéricos
personalizados (Fraccion, Dinero, Vector), unidades de medida, y
cualquier tipo que se beneficie de una sintaxis natural y expresiva.

27.1 — CONVERSIONES IMPLÍCITAS Y EXPLÍCITAS
-----------------------------------------------

```csharp
public struct Fraccion
{
    public int Numerador { get; }
    public int Denominador { get; }

    public Fraccion(int numerador, int denominador)
    {
        if (denominador == 0)
            throw new ArgumentException("Denominador no puede ser cero");
        Numerador = numerador;
        Denominador = denominador;
    }

    // Conversión implícita de int a Fraccion
    public static implicit operator Fraccion(int numero) =>
        new Fraccion(numero, 1);

    // Conversión explícita de Fraccion a double
    public static explicit operator double(Fraccion f) =>
        (double)f.Numerador / f.Denominador;

    // Conversión explícita de Fraccion a decimal
    public static explicit operator decimal(Fraccion f) =>
        (decimal)f.Numerador / f.Denominador;

    public override string ToString() => $"{Numerador}/{Denominador}";
}

// Uso
Fraccion f = 5;                    // implícita: int -> Fraccion
Console.WriteLine(f);              // 5/1

Fraccion media = new Fraccion(1, 2);
double valor = (double)media;      // explícita: Fraccion -> double
Console.WriteLine(valor);          // 0.5

decimal valorDec = (decimal)media; // explícita
Console.WriteLine(valorDec);       // 0.5
```

27.2 — REGLAS PARA OPERADORES DE CONVERSIÓN
-----------------------------------------------

- Deben ser static y public.
- Deben usar implicit o explicit.
- implicit: conversión que nunca falla ni pierde datos.
- explicit: conversión que puede fallar o perder datos (requiere cast).
- No puedes redefinir conversiones entre tipos que ya tienen una
  conversión integrada (como int a long).
- Puedes definir conversiones entre tus tipos y otros tipos.
- La conversión debe estar definida dentro de uno de los dos tipos
  (origen o destino).

```csharp
public readonly struct GradosCelsius
{
    public double Valor { get; }
    public GradosCelsius(double valor) => Valor = valor;

    public override string ToString() => $"{Valor:F1} °C";

    // Conversión implícita de double a Celsius
    public static implicit operator GradosCelsius(double valor) =>
        new GradosCelsius(valor);

    // Conversión explícita de Celsius a Fahrenheit
    public static explicit operator GradosFahrenheit(GradosCelsius c) =>
        new GradosFahrenheit(c.Valor * 9 / 5 + 32);
}

public readonly struct GradosFahrenheit
{
    public double Valor { get; }
    public GradosFahrenheit(double valor) => Valor = valor;

    public override string ToString() => $"{Valor:F1} °F";

    public static implicit operator GradosFahrenheit(double valor) =>
        new GradosFahrenheit(valor);

    public static explicit operator GradosCelsius(GradosFahrenheit f) =>
        new GradosCelsius((f.Valor - 32) * 5 / 9);
}

// Uso
GradosCelsius tempC = 25.0;                    // implícita double -> Celsius
Console.WriteLine(tempC);                      // 25.0 °C

GradosFahrenheit tempF = (GradosFahrenheit)tempC; // explícita Celsius -> Fahrenheit
Console.WriteLine(tempF);                      // 77.0 °F

GradosCelsius tempC2 = (GradosCelsius)tempF;   // explícita Fahrenheit -> Celsius
Console.WriteLine(tempC2);                     // 25.0 °C
```

27.4 — SOBRECARGA DE OPERADORES ARITMÉTICOS
-----------------------------------------------

```csharp
public readonly struct Vector2D
{
    public double X { get; }
    public double Y { get; }

    public Vector2D(double x, double y) => (X, Y) = (x, y);

    // Sobrecarga de operadores binarios
    public static Vector2D operator +(Vector2D a, Vector2D b) =>
        new Vector2D(a.X + b.X, a.Y + b.Y);

    public static Vector2D operator -(Vector2D a, Vector2D b) =>
        new Vector2D(a.X - b.X, a.Y - b.Y);

    public static Vector2D operator *(Vector2D v, double escalar) =>
        new Vector2D(v.X * escalar, v.Y * escalar);

    public static Vector2D operator *(double escalar, Vector2D v) =>
        v * escalar;

    public static Vector2D operator /(Vector2D v, double escalar)
    {
        if (escalar == 0) throw new DivideByZeroException();
        return new Vector2D(v.X / escalar, v.Y / escalar);
    }

    // Operador unario
    public static Vector2D operator -(Vector2D v) =>
        new Vector2D(-v.X, -v.Y);

    public override string ToString() => $"({X:F2}, {Y:F2})";
}

// Uso
var v1 = new Vector2D(3, 4);
var v2 = new Vector2D(1, 2);
var suma = v1 + v2;     // (4, 6)
var resta = v1 - v2;    // (2, 2)
var escalado = v1 * 2;  // (6, 8)
var invertido = -v1;    // (-3, -4)
Console.WriteLine(suma);
Console.WriteLine(escalado);
```

27.5 — SOBRECARGA DE OPERADORES DE COMPARACIÓN
-------------------------------------------------

Para sobrecargar operadores de comparación, también debes sobrecargar
Equals y GetHashCode.

```csharp
public readonly struct Dinero : IEquatable<Dinero>, IComparable<Dinero>
{
    public decimal Cantidad { get; }
    public string Moneda { get; }

    public Dinero(decimal cantidad, string moneda)
    {
        Cantidad = cantidad;
        Moneda = moneda ?? throw new ArgumentNullException(nameof(moneda));
    }

    // Operadores de comparación
    public static bool operator ==(Dinero a, Dinero b) =>
        a.Cantidad == b.Cantidad && a.Moneda == b.Moneda;

    public static bool operator !=(Dinero a, Dinero b) => !(a == b);

    public static bool operator <(Dinero a, Dinero b) =>
        a.MismaMoneda(b) && a.Cantidad < b.Cantidad;

    public static bool operator >(Dinero a, Dinero b) =>
        a.MismaMoneda(b) && a.Cantidad > b.Cantidad;

    public static bool operator <=(Dinero a, Dinero b) =>
        a.MismaMoneda(b) && a.Cantidad <= b.Cantidad;

    public static bool operator >=(Dinero a, Dinero b) =>
        a.MismaMoneda(b) && a.Cantidad >= b.Cantidad;

    // Operadores aritméticos
    public static Dinero operator +(Dinero a, Dinero b)
    {
        a.ValidarMismaMoneda(b);
        return new Dinero(a.Cantidad + b.Cantidad, a.Moneda);
    }

    public static Dinero operator -(Dinero a, Dinero b)
    {
        a.ValidarMismaMoneda(b);
        return new Dinero(a.Cantidad - b.Cantidad, a.Moneda);
    }

    public static Dinero operator *(Dinero a, decimal factor) =>
        new Dinero(a.Cantidad * factor, a.Moneda);

    public static Dinero operator /(Dinero a, decimal divisor) =>
        new Dinero(a.Cantidad / divisor, a.Moneda);

    private bool MismaMoneda(Dinero otro) => Moneda == otro.Moneda;

    private void ValidarMismaMoneda(Dinero otro)
    {
        if (!MismaMoneda(otro))
            throw new InvalidOperationException(
                $"No se pueden combinar monedas {Moneda} y {otro.Moneda}");
    }

    // IEquatable<Dinero>
    public bool Equals(Dinero other) => this == other;
    public override bool Equals(object? obj) => obj is Dinero d && Equals(d);

    public override int GetHashCode() => HashCode.Combine(Cantidad, Moneda);

    // IComparable<Dinero>
    public int CompareTo(Dinero other)
    {
        if (!MismaMoneda(other))
            throw new InvalidOperationException("No se pueden comparar monedas diferentes");
        return Cantidad.CompareTo(other.Cantidad);
    }

    public override string ToString() => $"{Moneda} {Cantidad:N2}";
}

// Uso
var precio1 = new Dinero(100.50m, "USD");
var precio2 = new Dinero(200.75m, "USD");

Console.WriteLine(precio1 + precio2); // USD 301.25
Console.WriteLine(precio2 - precio1); // USD 100.25
Console.WriteLine(precio1 * 2);       // USD 201.00
Console.WriteLine(precio1 < precio2); // True
Console.WriteLine(precio1 == new Dinero(100.50m, "USD")); // True
```

```csharp
public readonly struct Contador
{
    public int Valor { get; }

    public Contador(int valor) => Valor = valor;

    // Pre-incremento
    public static Contador operator ++(Contador c) =>
        new Contador(c.Valor + 1);

    // Pre-decremento
    public static Contador operator --(Contador c) =>
        new Contador(c.Valor - 1);

    // Operador de negación lógica
    public static bool operator !(Contador c) =>
        c.Valor == 0;

    // Operador de negación bitwise (~)
    public static Contador operator ~(Contador c) =>
        new Contador(~c.Valor);

    // true/false operator
    public static bool operator true(Contador c) => c.Valor != 0;
    public static bool operator false(Contador c) => c.Valor == 0;

    public override string ToString() => Valor.ToString();
}

// Uso
var c = new Contador(5);
c++; // Contador(6)
c--; // Contador(5)
Console.WriteLine(!c); // False (5 != 0)
Console.WriteLine(~c); // -6

if (c)
    Console.WriteLine("Verdadero"); // Se ejecuta
```

27.8 — OPERADORES CON CHECKED/UNCHECKED
------------------------------------------

```csharp
public readonly struct NumeroSeguro
{
    public int Valor { get; }

    public NumeroSeguro(int valor) => Valor = valor;

    // Overflow checked explícitamente
    public static NumeroSeguro operator checked +(NumeroSeguro a, NumeroSeguro b)
    {
        int resultado = checked(a.Valor + b.Valor);
        return new NumeroSeguro(resultado);
    }

    public static NumeroSeguro operator +(NumeroSeguro a, NumeroSeguro b)
    {
        int resultado = unchecked(a.Valor + b.Valor);
        return new NumeroSeguro(resultado);
    }

    // Conversión explícita checked
    public static explicit operator checked byte(NumeroSeguro n) =>
        checked((byte)n.Valor);

    public static explicit operator byte(NumeroSeguro n) =>
        unchecked((byte)n.Valor);

    public override string ToString() => Valor.ToString();
}

// Uso
var a = new NumeroSeguro(int.MaxValue);
var b = new NumeroSeguro(1);

// Sin checked (unchecked por defecto)
var c = a + b;
Console.WriteLine(c); // -2147483648 (wrap around)

// Con checked
try
{
    var d = checked(a + b); // OverflowException
}
catch (OverflowException ex)
{
    Console.WriteLine($"Overflow: {ex.Message}");
}
```

27.9 — OPERADOR DE CONVERSIÓN A TIPO NULLABLE
-----------------------------------------------

```csharp
public readonly struct Temperatura
{
    public double Valor { get; }

    public Temperatura(double valor) => Valor = valor;

    // Conversión implícita a double
    public static implicit operator double(Temperatura t) => t.Valor;

    // Conversión explícita de double? a Temperatura?
    public static explicit operator Temperatura?(double? valor) =>
        valor.HasValue ? new Temperatura(valor.Value) : null;

    // Conversión explícita de Temperatura? a double?
    public static explicit operator double?(Temperatura? t) =>
        t.HasValue ? t.Value.Valor : null;

    public override string ToString() => $"{Valor:F1}°";
}

// Uso
Temperatura? t1 = (Temperatura?)36.5;
double? d1 = (double?)t1;
Console.WriteLine(d1); // 36.5

Temperatura? t2 = (Temperatura?)(double?)null;
Console.WriteLine(t2.HasValue); // False
```

27.10 — OPERADOR DE CONVERSIÓN A BOOLEAN
------------------------------------------

```csharp
public readonly struct Resultado
{
    public bool EsExitoso { get; }
    public string? MensajeError { get; }

    private Resultado(bool exitoso, string? error = null)
    {
        EsExitoso = exitoso;
        MensajeError = error;
    }

    public static Resultado Ok() => new Resultado(true);
    public static Resultado Error(string mensaje) => new Resultado(false, mensaje);

    // Conversión implícita a bool para usar en if
    public static implicit operator bool(Resultado r) => r.EsExitoso;

    // Operador true/false para cortocircuito
    public static bool operator true(Resultado r) => r.EsExitoso;
    public static bool operator false(Resultado r) => !r.EsExitoso;

    // Operador lógico AND
    public static Resultado operator &(Resultado a, Resultado b) =>
        a.EsExitoso && b.EsExitoso ? Ok() :
        Error(a.MensajeError ?? b.MensajeError ?? "Error combinado");

    // Operador lógico OR
    public static Resultado operator |(Resultado a, Resultado b) =>
        a.EsExitoso || b.EsExitoso ? Ok() :
        Error(a.MensajeError ?? b.MensajeError ?? "Error combinado");

    public override string ToString() =>
        EsExitoso ? "OK" : $"ERROR: {MensajeError}";
}

// Uso
var r1 = Resultado.Ok();
var r2 = Resultado.Error("Algo salió mal");

if (r1) Console.WriteLine("r1 es exitoso");
if (!r2) Console.WriteLine("r2 es fallido");

var combinado = r1 && r2; // Usa operador &
Console.WriteLine(combinado); // ERROR: Algo salió mal
```

```csharp
using System;

public readonly struct Metro
{
    public double Valor { get; }
    public Metro(double valor) => Valor = valor;

    public static Metro operator +(Metro a, Metro b) => new Metro(a.Valor + b.Valor);
    public static Metro operator -(Metro a, Metro b) => new Metro(a.Valor - b.Valor);
    public static Metro operator *(Metro a, double factor) => new Metro(a.Valor * factor);
    public static Metro operator /(Metro a, double divisor) => new Metro(a.Valor / divisor);
    public static bool operator ==(Metro a, Metro b) => a.Valor == b.Valor;
    public static bool operator !=(Metro a, Metro b) => !(a == b);

    public override bool Equals(object? obj) => obj is Metro m && m.Valor == Valor;
    public override int GetHashCode() => Valor.GetHashCode();

    public static implicit operator Metro(double valor) => new Metro(valor);

    public override string ToString() => $"{Valor:F2} m";
}

public readonly struct Kilometro
{
    public double Valor { get; }
    public Kilometro(double valor) => Valor = valor;

    public static Kilometro operator +(Kilometro a, Kilometro b) => new Kilometro(a.Valor + b.Valor);
    public static Kilometro operator -(Kilometro a, Kilometro b) => new Kilometro(a.Valor - b.Valor);

    // Conversiones entre unidades
    public static implicit operator Kilometro(Metro m) => new Kilometro(m.Valor / 1000.0);
    public static explicit operator Metro(Kilometro km) => new Metro(km.Valor * 1000.0);

    // Conversión desde double
    public static implicit operator Kilometro(double valor) => new Kilometro(valor);

    public override string ToString() => $"{Valor:F3} km";
}

public readonly struct Milla
{
    public double Valor { get; }
    public Milla(double valor) => Valor = valor;

    public static implicit operator Milla(Kilometro km) => new Milla(km.Valor / 1.609344);
    public static explicit operator Kilometro(Milla m) => new Kilometro(m.Valor * 1.609344);
    public static implicit operator Milla(double valor) => new Milla(valor);

    public override string ToString() => $"{Valor:F2} mi";
}

public class CalculadoraDistancias
{
    public static Kilometro Sumar(params Kilometro[] distancias)
    {
        Kilometro total = 0;
        foreach (var d in distancias)
            total += d;
        return total;
    }
}

public class Programa
{
    public static void Main()
    {
        // Conversiones implícitas
        Metro m1 = 1500;           // 1500 m
        Kilometro km1 = m1;        // implícita: Metro -> Kilometro
        Console.WriteLine(km1);    // 1.500 km

        Kilometro km2 = 3.5;       // 3.5 km
        Metro m2 = (Metro)km2;     // explícita: Kilometro -> Metro
        Console.WriteLine(m2);     // 3500.00 m

        // Conversión a millas
        Milla mi = km2;            // implícita: Kilometro -> Milla
        Console.WriteLine(mi);     // 2.17 mi

        Kilometro kmDeMilla = (Kilometro)mi; // explícita
        Console.WriteLine(kmDeMilla); // 3.500 km

        // Operaciones aritméticas
        var distanciaTotal = CalculadoraDistancias.Sumar(1.5, 2.3, 0.8);
        Console.WriteLine($"Total: {distanciaTotal}");

        var enMillas = (Milla)distanciaTotal;
        Console.WriteLine($"En millas: {enMillas}");

        // Usar en expresiones
        var mediaMaraton = new Kilometro(21.0975);
        var diezKm = new Kilometro(10);
        var diferencia = mediaMaraton - diezKm;
        Console.WriteLine($"Diferencia: {diferencia}");

        // Comparación
        Console.WriteLine(km1 == 1.5);  // True
        Console.WriteLine(km2 > km1);   // True (3.5 > 1.5)
    }
}
```

================================================================================
  FAQ — PREGUNTAS FRECUENTES
================================================================================

P: ¿Puedo sobrecargar el operador =?
R: No. El operador de asignación no se puede sobrecargar.

P: ¿Puedo sobrecargar el operador ?: (ternario)?
R: No es un operador sobrecargable.

P: ¿Puedo sobrecargar operadores para nullable types?
R: Sí, los tipos nullable ya tienen sobrecarga de operadores.

P: ¿Qué operadores no se pueden sobrecargar?
R: =, ?:, ??, &&, ||, ?., ->, new, typeof, default, checked, unchecked,
   is, as, await, sizeof, y.

P: ¿Puedo definir un operador entre dos tipos que no he creado yo?
R: No. Al menos uno de los tipos debe ser el que contiene el operador.

P: ¿Los operadores de conversión pueden ser virtuales?
R: No. Son static y no pueden ser virtuales ni abstractos.

P: ¿Debo siempre sobrecargar Equals y GetHashCode con == y !=?
R: Sí. Es una buena práctica y requerida por el compilador.

P: ¿Puedo tener operadores checked y unchecked en el mismo tipo?
R: Sí, desde C# 11 puedes definir versiones checked de operadores.

conversión y aritméticos. El sistema debe incluir:

  - Kilogramo * (M/S)^2 = Joule (energía).

- Operadores de conversión para mostrar en diferentes sistemas
  (MKS, CGS, Imperial).
- Serialización/deserialización de cantidades con unidad.
- Un método ToString que muestre la unidad correctamente.
- Parseo desde string: "10.5 m/s" -> Velocidad(10.5, "m/s").

- Mínimo 20 tipos de unidades.
- Mínimo 30 operadores sobrecargados.
