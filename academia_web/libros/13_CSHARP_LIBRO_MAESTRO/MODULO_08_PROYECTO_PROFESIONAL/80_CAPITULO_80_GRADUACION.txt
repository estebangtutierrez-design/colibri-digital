  [REVISIÓN DEL VIAJE — TODOS LOS MÓDULOS]
================================================================================

  MÓDULO I: FUNDAMENTOS DE C#
  ────────────────────────────
  • Variables, tipos de datos, operadores.
  • Estructuras de control: if, switch, bucles.
  • Arrays y strings.
  • Programación estructurada.

  MÓDULO II: PROGRAMACIÓN ORIENTADA A OBJETOS
  ─────────────────────────────────────────────
  • Clases, objetos, propiedades, métodos.
  • Herencia, polimorfismo, interfaces.
  • Clases abstractas y selladas.
  • Sobrecarga de operadores.

  MÓDULO III: COLECCIONES, LINQ Y DELEGADOS
  ───────────────────────────────────────────
  • List<T>, Dictionary<T>, HashSet<T>.
  • LINQ: Where, Select, GroupBy, Join.
  • Delegados, eventos, Action/Func.

  MÓDULO IV: PROGRAMACIÓN AVANZADA
  ──────────────────────────────────
  • async/await, Task, Parallel.
  • Manejo de excepciones.
  • Reflexión y atributos.
  • unsafe code, pointers.

  MÓDULO V: PROGRAMACIÓN FUNCIONAL
  ──────────────────────────────────
  • Expresiones lambda y closures.
  • LINQ avanzado, IEnumerable<T>.
  • Inmutabilidad y efectos secundarios.
  • Pattern matching.

  MÓDULO VI: ACCESO A DATOS
  ───────────────────────────
  • ADO.NET, Dapper.
  • Entity Framework Core.
  • Migraciones, relaciones.
  • Transacciones.

  MÓDULO VII: ASP.NET CORE
  ──────────────────────────
  • MVC y Web API.
  • Middleware, DI.
  • Autenticación y autorización.
  • SignalR, Health Checks.

  MÓDULO VIII: PROYECTO PROFESIONAL
  ───────────────────────────────────
  • Arquitectura en capas.
  • Entity Framework Core avanzado.
  • Debugging profesional.
  • Pruebas unitarias (xUnit, Moq).
  • Optimización (Span<T>, StringBuilder).
  • Seguridad (JWT, cifrado, SQL injection).
  • Examen profesional.
  • Proyecto final.
  • Buenas prácticas.
  • GRADUACIÓN.
