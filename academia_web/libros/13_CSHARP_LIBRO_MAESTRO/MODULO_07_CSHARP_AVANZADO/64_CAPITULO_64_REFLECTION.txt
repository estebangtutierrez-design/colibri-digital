--- 1. INTRODUCCIÓN A REFLECTION ---

Reflection es la capacidad de un programa de inspeccionar y manipular su
propia estructura. En .NET, esto es posible gracias a los metadatos que el
compilador incrusta en los assemblies (el IL y los símbolos).

Usos comunes de reflection:
- Inspectores de tipos (IDE, herramientas)
- Serialización/Deserialización
- Inversión de Control (IoC) y Dependency Injection
- ORMs (Entity Framework, Dapper)
- Testing (NUnit, xUnit)
- Plugins y sistemas de extensión
- Mapeo objeto-objeto (AutoMapper)

--- 2. OBTENER UN TYPE ---

Hay tres formas principales de obtener un objeto Type:

```csharp
using System;
using System.Reflection;

class Program
{
    static void Main()
    {
        Console.WriteLine("=== OBTENER UN TYPE ===\n");
        
        // 1. typeof()
        Type tipo1 = typeof(string);
        Console.WriteLine($"typeof(string): {tipo1.FullName}");
        
        // 2. GetType() - método de instancia
        string mensaje = "Hola";
        Type tipo2 = mensaje.GetType();
        Console.WriteLine($"\"Hola\".GetType(): {tipo2.FullName}");
        
        // 3. Type.GetType() - desde nombre string
        Type tipo3 = Type.GetType("System.Int32");
        Console.WriteLine($"Type.GetType(\"System.Int32\"): {tipo3?.FullName}");
        
        // Otras formas
        Type tipo4 = typeof(Program).Assembly.GetType("Program");
        Console.WriteLine($"Assembly.GetType: {tipo4?.FullName}");
        
        // Comparar types
        Console.WriteLine($"\n¿typeof(string) == \"Hola\".GetType()? {tipo1 == tipo2}");
        Console.WriteLine($"typeof(int) == typeof(Int32)? {typeof(int) == typeof(Int32)}");
    }
}
```

--- 4. EXPLORAR MIEMBROS (MÉTODOS, PROPIEDADES, CAMPOS, CONSTRUCTORES) ---

```csharp
using System;
using System.Reflection;

class Program
{
    static void Main()
    {
        Console.WriteLine("=== EXPLORAR MIEMBROS ===\n");
        
        Type tipo = typeof(Calculadora);
        
        Console.WriteLine("=== MÉTODOS ===");
        foreach (var metodo in tipo.GetMethods(
            BindingFlags.Public | BindingFlags.Instance | BindingFlags.Static))
        {
            string parametros = "";
            foreach (var param in metodo.GetParameters())
                parametros += $"{param.ParameterType.Name} {param.Name}, ";
            if (parametros.Length > 2) parametros = parametros[..^2];
            Console.WriteLine($"  {metodo.ReturnType.Name} {metodo.Name}({parametros})");
        }
        
        Console.WriteLine("\n=== PROPIEDADES ===");
        foreach (var prop in tipo.GetProperties(
            BindingFlags.Public | BindingFlags.Instance))
        {
            Console.WriteLine($"  {prop.PropertyType.Name} {prop.Name} " +
                $"(get: {prop.CanRead}, set: {prop.CanWrite})");
        }
        
        Console.WriteLine("\n=== CAMPOS ===");
        foreach (var campo in tipo.GetFields(
            BindingFlags.Public | BindingFlags.Instance | BindingFlags.NonPublic))
        {
            Console.WriteLine($"  {campo.FieldType.Name} {campo.Name} (público: {campo.IsPublic})");
        }
        
        Console.WriteLine("\n=== CONSTRUCTORES ===");
        foreach (var ctor in tipo.GetConstructors())
        {
            string parametros = "";
            foreach (var param in ctor.GetParameters())
                parametros += $"{param.ParameterType.Name} {param.Name}, ";
            if (parametros.Length > 2) parametros = parametros[..^2];
            Console.WriteLine($"  Constructor({parametros})");
        }
    }
}

public class Calculadora
{
    public int Id { get; set; }
    public string Nombre;
    private decimal _resultado;
    
    public Calculadora() { }
    public Calculadora(int id) { Id = id; }
    
    public int Sumar(int a, int b) => a + b;
    public int Restar(int a, int b) => a - b;
    public static int Multiplicar(int a, int b) => a * b;
    private void Resetear() => _resultado = 0;
}
```

--- 5. INVOCAR MIEMBROS ---

```csharp
using System;
using System.Reflection;

class Program
{
    static void Main()
    {
        Console.WriteLine("=== INVOCAR MIEMBROS ===\n");
        
        Type tipo = typeof(Calculadora);
        object calc = Activator.CreateInstance(tipo);
        
        // InvokeMember
        int r1 = (int)tipo.InvokeMember("Sumar",
            BindingFlags.InvokeMethod | BindingFlags.Instance | BindingFlags.Public,
            null, calc, new object[] { 10, 20 });
        Console.WriteLine($"calc.Sumar(10, 20) = {r1}");
        
        // MethodInfo.Invoke
        MethodInfo metodoRestar = tipo.GetMethod("Restar");
        int r2 = (int)metodoRestar.Invoke(calc, new object[] { 50, 15 });
        Console.WriteLine($"calc.Restar(50, 15) = {r2}");
        
        // Método estático
        MethodInfo metodoMul = tipo.GetMethod("Multiplicar",
            BindingFlags.Public | BindingFlags.Static);
        int r3 = (int)metodoMul.Invoke(null, new object[] { 6, 7 });
        Console.WriteLine($"Calculadora.Multiplicar(6, 7) = {r3}");
        
        // PropertyInfo
        PropertyInfo propId = tipo.GetProperty("Id");
        propId.SetValue(calc, 42);
        Console.WriteLine($"calc.Id = {propId.GetValue(calc)}");
        
        // FieldInfo
        FieldInfo campoNombre = tipo.GetField("Nombre",
            BindingFlags.Public | BindingFlags.Instance);
        campoNombre.SetValue(calc, "MiCalc");
        Console.WriteLine($"calc.Nombre = {campoNombre.GetValue(calc)}");
    }
}
```

--- 6. ACTIVATOR.CREATEINSTANCE ---

```csharp
using System;
using System.Reflection;
using System.Collections;

class Program
{
    static void Main()
    {
        Console.WriteLine("=== ACTIVATOR.CREATEINSTANCE ===\n");
        
        // Sin parámetros
        Type tipoPersona = typeof(Persona);
        Persona p1 = (Persona)Activator.CreateInstance(tipoPersona);
        p1.Nombre = "Juan";
        Console.WriteLine($"p1: {p1.Nombre}, {p1.Edad}");
        
        // Con parámetros
        Persona p2 = (Persona)Activator.CreateInstance(
            tipoPersona, new object[] { "María", 30 });
        Console.WriteLine($"p2: {p2.Nombre}, {p2.Edad}");
        
        // Tipo genérico
        Type tipoLista = typeof(System.Collections.Generic.List<>);
        Type tipoListaString = tipoLista.MakeGenericType(typeof(string));
        var lista = (IList)Activator.CreateInstance(tipoListaString);
        lista.Add("Hola");
        lista.Add("Mundo");
        Console.WriteLine($"\nLista<string> creada: {lista.Count} elementos");
        Console.WriteLine($"  Tipo real: {lista.GetType().Name}");
        
        // Assembly.CreateInstance
        var assembly = typeof(Persona).Assembly;
        Persona p3 = (Persona)assembly.CreateInstance(
            typeof(Persona).FullName, false, BindingFlags.Default, null,
            new object[] { "Carlos", 25 }, null, null);
        Console.WriteLine($"p3: {p3.Nombre}, {p3.Edad}");
        
        // Tipo anónimo
        var anonimo = new { Nombre = "Test", Valor = 100 };
        Type tipoAnonimo = anonimo.GetType();
        Console.WriteLine($"\nTipo anónimo: {tipoAnonimo.Name}");
        foreach (var prop in tipoAnonimo.GetProperties())
            Console.WriteLine($"  {prop.Name}: {prop.GetValue(anonimo)}");
    }
}

public class Persona
{
    public string Nombre { get; set; }
    public int Edad { get; set; }
    
    public Persona()
    {
        Nombre = "Sin nombre";
        Edad = 0;
    }
    
    public Persona(string nombre, int edad)
    {
        Nombre = nombre;
        Edad = edad;
    }
}
```

```csharp
using System;
using System.Reflection;

class Program
{
    static void Main()
    {
        Console.WriteLine("=== INVOKEMEMBER AVANZADO ===\n");
        
        object calc = Activator.CreateInstance(typeof(Calculadora));
        Type tipo = typeof(Calculadora);
        var flags = BindingFlags.Public | BindingFlags.Instance | BindingFlags.InvokeMethod;
        
        // Set property
        tipo.InvokeMember("Id",
            BindingFlags.Public | BindingFlags.Instance | BindingFlags.SetProperty,
            null, calc, new object[] { 99 });
        
        // Get property
        object id = tipo.InvokeMember("Id",
            BindingFlags.Public | BindingFlags.Instance | BindingFlags.GetProperty,
            null, calc, null);
        Console.WriteLine($"Id = {id}");
        
        // Get field
        object nombre = tipo.InvokeMember("Nombre",
            BindingFlags.Public | BindingFlags.Instance | BindingFlags.GetField,
            null, calc, null);
        Console.WriteLine($"Nombre = {nombre}");
        
        // Invocar con parámetros out
        // (demostración conceptual)
        Console.WriteLine("\nBindingFlags útiles:");
        Console.WriteLine("  InvokeMethod: invocar método");
        Console.WriteLine("  GetProperty/SetProperty: propiedades");
        Console.WriteLine("  GetField/SetField: campos");
        Console.WriteLine("  CreateInstance: constructor");
        Console.WriteLine("  Static: miembros estáticos");
        Console.WriteLine("  Instance: miembros de instancia");
        Console.WriteLine("  Public/NonPublic: visibilidad");
        Console.WriteLine("  FlattenHierarchy: incluir miembros heredados");
        Console.WriteLine("  DeclaredOnly: solo de la clase actual");
        Console.WriteLine("  IgnoreCase: ignorar mayúsculas/minúsculas");
        Console.WriteLine("  OptionalParamBinding: parámetros opcionales");
    }
}
```

```csharp
using System;
using System.Reflection;
using System.Linq;

class Program
{
    static void Main()
    {
        Console.WriteLine("=== EXPLORAR ASSEMBLIES ===\n");
        
        // Assembly actual
        Assembly asmActual = Assembly.GetExecutingAssembly();
        Console.WriteLine($"Assembly actual: {asmActual.GetName().Name}");
        Console.WriteLine($"  Versión: {asmActual.GetName().Version}");
        Console.WriteLine($"  Ubicación: {asmActual.Location}");

        Assembly asmCore = typeof(string).Assembly;
        Console.WriteLine($"\nAssembly de string: {asmCore.GetName().Name}");
        
        // Assembly de entrada
        Assembly asmEntry = Assembly.GetEntryAssembly();
        Console.WriteLine($"\nAssembly de entrada: {asmEntry?.GetName().Name}");
        
        // Assembly que llamó a este código
        Assembly asmCaller = Assembly.GetCallingAssembly();
        Console.WriteLine($"Assembly llamante: {asmCaller.GetName().Name}");
        
        // Cargar assembly por nombre
        try
        {
            Assembly asmCargado = Assembly.Load("System.Text.Json");
            Console.WriteLine($"\nAssembly cargado: {asmCargado.GetName().Name}");
            Console.WriteLine($"  Tipos públicos: {asmCargado.GetExportedTypes().Length}");
        }
        catch (Exception ex)
        {
            Console.WriteLine($"\nNo se pudo cargar: {ex.Message}");
        }
        
        // Listar tipos públicos
        Console.WriteLine("\nTipos en assembly actual:");
        foreach (var tipo in asmActual.GetExportedTypes().Take(10))
            Console.WriteLine($"  - {tipo.FullName}");
    }
```

--- 9. OBTENER Y USAR ATRIBUTOS POR REFLEXIÓN ---

```csharp
using System;
using System.Reflection;

[AttributeUsage(AttributeTargets.Class | AttributeTargets.Method)]
public class DescripcionAttribute : Attribute
{
    public string Texto { get; }
    public DescripcionAttribute(string texto) => Texto = texto;
}

[Descripcion("Clase que representa un producto")]
class Producto
{
    [Descripcion("Identificador único del producto")]
    public int Id { get; set; }
    
    [Descripcion("Nombre comercial del producto")]
    public string Nombre { get; set; }
    
    [Descripcion("Calcula el precio con IVA incluido")]
    public decimal CalcularPrecioConIVA(decimal precioBase)
    {
        return precioBase * 1.21m;
    }
}

class Program
{
    static void Main()
    {
        Console.WriteLine("=== ATRIBUTOS POR REFLEXIÓN ===\n");
        
        Type tipo = typeof(Producto);
        
        // Atributos de la clase
        var descClase = tipo.GetCustomAttribute<DescripcionAttribute>();
        Console.WriteLine($"Clase '{tipo.Name}': {descClase?.Texto}");
        
        // Atributos de propiedades
        foreach (var prop in tipo.GetProperties())
        {
            var desc = prop.GetCustomAttribute<DescripcionAttribute>();
            if (desc != null)
                Console.WriteLine($"  Propiedad '{prop.Name}': {desc.Texto}");
        }
        
        // Atributos de métodos
        foreach (var metodo in tipo.GetMethods(BindingFlags.Public | BindingFlags.Instance | BindingFlags.DeclaredOnly))
        {
            var desc = metodo.GetCustomAttribute<DescripcionAttribute>();
            if (desc != null)
                Console.WriteLine($"  Método '{metodo.Name}': {desc.Texto}");
        }
        
        // Obtener todas las propiedades con un atributo específico
        var propsConAtributo = tipo.GetProperties()
            .Where(p => p.GetCustomAttribute<DescripcionAttribute>() != null);
        
        Console.WriteLine($"\nPropiedades con [Descripcion]: {propsConAtributo.Count()}");
    }
}
```

```csharp
using System;
using System.Reflection;

class SimpleMapper
{
    public static TDestino Map<TOrigen, TDestino>(TOrigen origen)
        where TDestino : new()
    {
        TDestino destino = new TDestino();
        Type tipoOrigen = typeof(TOrigen);
        Type tipoDestino = typeof(TDestino);
        
        foreach (var propOrigen in tipoOrigen.GetProperties(
            BindingFlags.Public | BindingFlags.Instance))
        {
            var propDestino = tipoDestino.GetProperty(propOrigen.Name);
            if (propDestino != null && propDestino.CanWrite &&
                propDestino.PropertyType == propOrigen.PropertyType)
            {
                object valor = propOrigen.GetValue(origen);
                propDestino.SetValue(destino, valor);
            }
        }
        
        return destino;
    }
}

public class EmpleadoDTO
{
    public string Nombre { get; set; }
    public int Edad { get; set; }
    public decimal Salario { get; set; }
}

public class EmpleadoViewModel
{
    public string Nombre { get; set; }
    public int Edad { get; set; }
    public decimal Salario { get; set; }
    public string Departamento { get; set; }
}

class Program
{
    static void Main()
    {
        Console.WriteLine("=== MAPPER POR REFLEXIÓN ===\n");
        
        var emp = new EmpleadoDTO
        {
            Nombre = "Carlos Ruiz",
            Edad = 35,
            Salario = 55000m
        };
        
        var vm = SimpleMapper.Map<EmpleadoDTO, EmpleadoViewModel>(emp);
```

--- 12. EJEMPLO: INVOCAR MÉTODO PRIVADO ---

```csharp
using System;
using System.Reflection;

class Program
{
    static void Main()
    {
        Console.WriteLine("=== INVOCAR MIEMBROS PRIVADOS ===\n");
        
        var secreto = new CajaFuerte();
        
        // No podemos acceder directamente:
        // secreto._combinacion; // Error: private
        
        // Pero con reflection sí:
        Type tipo = typeof(CajaFuerte);
        
        // Campo privado
        var campoCombinacion = tipo.GetField("_combinacion",
            BindingFlags.NonPublic | BindingFlags.Instance);
        string combinacion = (string)campoCombinacion.GetValue(secreto);
        Console.WriteLine($"Combinación (campo privado): {combinacion}");
        
        // Método privado
        var metodoAbrir = tipo.GetMethod("Abrir",
            BindingFlags.NonPublic | BindingFlags.Instance);
        bool abrio = (bool)metodoAbrir.Invoke(secreto, new object[] { "1234" });
        Console.WriteLine($"¿Abrió con 1234? {abrio}");
        
        bool abrio2 = (bool)metodoAbrir.Invoke(secreto, new object[] { "0000" });
        Console.WriteLine($"¿Abrió con 0000? {abrio2}");
        
        // Propiedad privada
        var propIntentos = tipo.GetProperty("IntentosFallidos",
            BindingFlags.NonPublic | BindingFlags.Instance);
        int intentos = (int)propIntentos.GetValue(secreto);
        Console.WriteLine($"Intentos fallidos: {intentos}");
        
        // Evento privado
        var evento = tipo.GetEvent("Alarma",
            BindingFlags.NonPublic | BindingFlags.Instance);
        Console.WriteLine($"Evento privado encontrado: {evento?.Name}");
    }
}

public class CajaFuerte
{
    private string _combinacion = "1234";
    private int IntentosFallidos { get; set; }
    
    private event Action Alarma;
    
    private bool Abrir(string combinacion)
    {
        if (combinacion == _combinacion)
        {
            Console.WriteLine("  ¡Caja abierta!");
            IntentosFallidos = 0;
            return true;
        }
        else
        {
            IntentosFallidos++;
            Console.WriteLine($"  Combinación incorrecta (intento {IntentosFallidos})");
            if (IntentosFallidos >= 3)
                Alarma?.Invoke();
            return false;
        }
    }
    
    public string Estado()
    {
        return $"Caja con {IntentosFallidos} intentos fallidos";
    }
}
```

```csharp
        Console.WriteLine($"\nTipo creado dinámicamente: {tipoDinamico.Name}");
        Console.WriteLine($"  Argumento: {tipoDinamico.GetGenericArguments()[0]}");
        
        // Crear instancia
        object instancia = Activator.CreateInstance(tipoDinamico);
        Console.WriteLine($"  Instancia creada: {instancia.GetType().Name}");
        
        // Invocar método genérico
        var metodo = typeof(Utils).GetMethod("CrearLista");
        var metodoCerrado = metodo.MakeGenericMethod(typeof(int));

    }

    public T Obtener() => default;

    {

    }
}
```

--- 14. REFLECTION Y RENDIMIENTO: DELEGADOS VS INVOKE ---

```csharp
using System;
using System.Diagnostics;
using System.Reflection;

class Program
{
    static void Main()
    {
        Console.WriteLine("=== RENDIMIENTO: INVOKE VS DELEGADOS ===\n");
        
        var calc = new Calculadora();
        Type tipo = typeof(Calculadora);
        MethodInfo metodoSumar = tipo.GetMethod("Sumar");
        
        const int iteraciones = 1_000_000;
        var sw = new Stopwatch();
        
        // 1. Llamada directa
        sw.Start();
        for (int i = 0; i < iteraciones; i++)
            calc.Sumar(i, i + 1);
        sw.Stop();
        Console.WriteLine($"Llamada directa: {sw.ElapsedMilliseconds}ms");
        
        // 2. MethodInfo.Invoke
        sw.Restart();
        for (int i = 0; i < iteraciones; i++)
            metodoSumar.Invoke(calc, new object[] { i, i + 1 });
        sw.Stop();
        Console.WriteLine($"MethodInfo.Invoke: {sw.ElapsedMilliseconds}ms");
        
        // 3. Delegado creado por reflection
        var delegadoSumar = (Func<int, int, int>)metodoSumar.CreateDelegate(
            typeof(Func<int, int, int>), calc);
        sw.Restart();
        for (int i = 0; i < iteraciones; i++)
            delegadoSumar(i, i + 1);
        sw.Stop();
        Console.WriteLine($"Delegado (CreateDelegate): {sw.ElapsedMilliseconds}ms");
        
        // 4. Delegado directo
        Func<int, int, int> funcionDirecta = calc.Sumar;
        sw.Restart();
        for (int i = 0; i < iteraciones; i++)
            funcionDirecta(i, i + 1);
        sw.Stop();
        Console.WriteLine($"Delegado directo: {sw.ElapsedMilliseconds}ms");
        
        Console.WriteLine("\nConclusión: MethodInfo.Invoke es ~50x más lento.");
        Console.WriteLine("Use CreateDelegate para caching de métodos vía reflection.");
    }
}
```

```csharp
// Interfaz común
public interface INotificador

    void Enviar(string mensaje);

// Implementaciones concretas
public class EmailNotificador : INotificador
{
    public void Enviar(string mensaje)

        Console.WriteLine($"  [EMAIL] {mensaje}");
    }
}

public class SMSNotificador : INotificador
{
    public void Enviar(string mensaje)
    {
        Console.WriteLine($"  [SMS] {mensaje}");
    }
}

public class PushNotificador : INotificador
{
    public void Enviar(string mensaje)
    {
        Console.WriteLine($"  [PUSH] {mensaje}");
    }
}

// Factoría con reflection
public class NotificadorFactory
{
    private static Dictionary<string, Type> _tipos;
    
    static NotificadorFactory()
    {
        _tipos = new Dictionary<string, Type>();
        
        // Registrar todos los tipos que implementan INotificador
        var assembly = Assembly.GetExecutingAssembly();
        foreach (var tipo in assembly.GetTypes())
        {
            if (typeof(INotificador).IsAssignableFrom(tipo) && !tipo.IsInterface)
            {
                _tipos[tipo.Name.Replace("Notificador", "").ToLower()] = tipo;
            }
        }
    }
    
    public static INotificador Crear(string tipo)
    {
        string clave = tipo.ToLower();
        if (_tipos.TryGetValue(clave, out Type tipoNotificador))
        {
            return (INotificador)Activator.CreateInstance(tipoNotificador);
        }
        throw new ArgumentException($"Tipo '{tipo}' no registrado.");
    }
    
    public static IEnumerable<string> TiposDisponibles()
    {
        return _tipos.Keys;
    }
}

class Program
{
    static void Main()
    {
        Console.WriteLine("=== FACTORÍA CON REFLECTION ===\n");
        
        Console.WriteLine("Tipos disponibles:");
        foreach (var t in NotificadorFactory.TiposDisponibles())
            Console.WriteLine($"  - {t}");
        
        Console.WriteLine("\nEnviando notificaciones:");
        var notificador = NotificadorFactory.Crear("email");
        notificador.Enviar("Bienvenido al sistema.");
        
        notificador = NotificadorFactory.Crear("sms");
        notificador.Enviar("Su código de verificación es 1234.");
        
        notificador = NotificadorFactory.Crear("push");
        notificador.Enviar("Nuevo mensaje recibido.");
    }
}
```

```csharp
using System;
using System.Reflection;
using System.Reflection.Emit;

// Este ejemplo muestra cómo crear un proxy simple con DispatchProxy
public interface IServicio
{
    void Ejecutar();
    int Calcular(int x);
}

// Implementación real
public class ServicioReal : IServicio
{
    public void Ejecutar()
    {
        Console.WriteLine("  Ejecutando lógica real...");
    }
    
    public int Calcular(int x)
    {
        Console.WriteLine($"  Calculando con {x}...");
        return x * 2;
    }
}

// Proxy con logging
public class LoggingProxy<T> : DispatchProxy
{
    private T _objetoReal;
    
    protected override object Invoke(MethodInfo targetMethod, object[] args)
    {
        Console.WriteLine($"[LOG] Llamando a {targetMethod.Name}...");
        
        var sw = System.Diagnostics.Stopwatch.StartNew();
        
        try
        {
            object resultado = targetMethod.Invoke(_objetoReal, args);
            sw.Stop();
            Console.WriteLine($"[LOG] {targetMethod.Name} completado en {sw.ElapsedMilliseconds}ms");
            return resultado;
        }
        catch (Exception ex)
        {
            sw.Stop();
            Console.WriteLine($"[LOG] {targetMethod.Name} falló: {ex.InnerException?.Message}");
            throw;
        }
    }
    
    public static T Crear(T objetoReal)
    {
        object proxy = DispatchProxy.Create<T, LoggingProxy<T>>();
        ((LoggingProxy<T>)proxy)._objetoReal = objetoReal;
        return (T)proxy;
    }
}

class Program
{
    static void Main()
    {
        Console.WriteLine("=== PROXY DINÁMICO ===\n");
        
        var real = new ServicioReal();
        var proxy = LoggingProxy<IServicio>.Crear(real);
        
        Console.WriteLine("Llamando a través del proxy:");
        proxy.Ejecutar();
        int resultado = proxy.Calcular(5);
        Console.WriteLine($"Resultado: {resultado}");
    }
}
```

--- 20. BUENAS PRÁCTICAS CON REFLECTION ---

```csharp
using System;
using System.Collections.Generic;
using System.Reflection;

class BuenasPracticasReflection
{
    // ✅ CORRECTO: Cachear MethodInfo, Type, etc.
    private static readonly Dictionary<Type, PropertyInfo[]> _cachePropiedades
        = new Dictionary<Type, PropertyInfo[]>();
    
    public static PropertyInfo[] ObtenerPropiedadesCache(Type tipo)
    {
        if (!_cachePropiedades.TryGetValue(tipo, out var propiedades))
        {
            propiedades = tipo.GetProperties(BindingFlags.Public | BindingFlags.Instance);
            _cachePropiedades[tipo] = propiedades;
        }
        return propiedades;
    }
    
    // ✅ CORRECTO: Usar CreateDelegate en lugar de Invoke en bucles
    // (ver ejemplo de rendimiento arriba)
    
    // ✅ CORRECTO: Usar typeof() siempre que sea posible en vez de Type.GetType(string)
    // typeof es más rápido y evita errores tipográficos
    
    // ✅ CORRECTO: Verificar null después de GetMethod/GetProperty
    public static void InvocarMetodoSeguro(object obj, string nombreMetodo)
    {
        if (obj == null) throw new ArgumentNullException(nameof(obj));
        
        var tipo = obj.GetType();
        var metodo = tipo.GetMethod(nombreMetodo,
            BindingFlags.Public | BindingFlags.Instance);
        
        if (metodo == null)
        {
            Console.WriteLine($"Método '{nombreMetodo}' no encontrado en {tipo.Name}");
            return;
        }
        
        metodo.Invoke(obj, null);
    }
    
    // ✅ CORRECTO: Usar BindingFlags restrictivos
    public static void ExplorarMetodosPublicos(Type tipo)
    {
        // Solo miembros públicos de instancia
        var metodos = tipo.GetMethods(
            BindingFlags.Public | BindingFlags.Instance | BindingFlags.DeclaredOnly);
        
        foreach (var m in metodos)
            Console.WriteLine($"  {m.Name}");
    }
    
    // ✅ CORRECTO: Try-Catch alrededor de Invoke
    public static object InvocarSeguro(MethodInfo metodo, object obj, object[] args)
    {
        try
        {
            return metodo.Invoke(obj, args);
        }
        catch (TargetInvocationException ex)
        {
            // La excepción real está en InnerException
            Console.WriteLine($"Error en {metodo.Name}: {ex.InnerException?.Message}");
            throw ex.InnerException ?? ex;
        }
        catch (ArgumentException ex)
        {
            Console.WriteLine($"Argumentos inválidos para {metodo.Name}: {ex.Message}");
            throw;
        }
    }
    
    // ❌ MAL: Llamar a GetMethod/GetProperty cada vez
    public static void MalEjemplo(object obj)
    {
        var tipo = obj.GetType();
        for (int i = 0; i < 1000; i++)
        {
            // Esto es ineficiente: GetMethod se llama 1000 veces
            var metodo = tipo.GetMethod("Metodo");
            metodo?.Invoke(obj, null);
        }
    }
    
    // ✅ CORRECTO: Cachear el MethodInfo
    public static void BuenEjemplo(object obj, MethodInfo metodoCacheado)
    {
        for (int i = 0; i < 1000; i++)
            metodoCacheado?.Invoke(obj, null);
    }
}
```

================================================================================
  RESUMEN DEL CAPÍTULO
================================================================================

1. Reflection permite inspeccionar tipos, métodos, propiedades y otros
   miembros en tiempo de ejecución.
2. Type es la clase central: se obtiene con typeof(), GetType() o Type.GetType().
3. BindingFlags controla qué miembros se obtienen (visibilidad, tipo, etc.).
4. GetMethods, GetProperties, GetFields, GetConstructors exploran miembros.
5. InvokeMember y MethodInfo.Invoke invocan miembros dinámicamente.
6. Activator.CreateInstance crea instancias sin conocer el tipo en
   compilación.
7. La reflexión es más lenta que la invocación directa; use CreateDelegate
   para caching y rendimiento.
8. Use reflection con moderación: prefiera interfaces, genéricos y patrones
   de diseño cuando sea posible.

================================================================================
  PREGUNTAS FRECUENTES (FAQ)
================================================================================

P: ¿Reflection es lento?
R: Comparado con llamadas directas, sí (~10-50x más lento). Sin embargo,
para operaciones que se ejecutan una vez (carga de plugins, inicialización)
no es problema. Para bucles, cachee los MemberInfo y use CreateDelegate.

P: ¿Cuándo debo usar reflection en lugar de interfaces?
R: Cuando no conoce los tipos en tiempo de compilación (plugins, scripts,
carga dinámica). Si conoce las interfaces, prefiera interfaces y DI.

P: ¿Puedo usar reflection en .NET Core/MAUI/Xamarin?
R: Sí, pero con restricciones. Algunos miembros pueden no estar disponibles
en plataformas restringidas. Use reflection con precaución en entornos
de confianza parcial.

P: ¿Puedo modificar el código IL con reflection?
R: No directamente. Para emitir IL dinámico, use System.Reflection.Emit.

P: ¿GetType() y typeof() son intercambiables?
R: typeof() se evalúa en compilación; GetType() en runtime. typeof(string)
siempre devuelve el mismo Type. "hola".GetType() devuelve el tipo en
tiempo de ejecución (String).
