¡Bienvenido al Capítulo 63! Los atributos (attributes) son una característica

--- 1. ¿QUÉ SON LOS ATRIBUTOS? ---

Los atributos son clases que heredan de System.Attribute y permiten agregar
metadatos a elementos del código (clases, métodos, propiedades, etc.).
Se escriben entre corchetes [ ] antes del elemento al que aplican.

```csharp
using System;

[Serializable]  // Atributo: esta clase puede serializarse
public class Producto
{
    [Obsolete("Usar NuevaPropiedad en su lugar.")]  // Atributo: marcado como obsoleto
    public string PropiedadVieja { get; set; }
    
    public string NuevaPropiedad { get; set; }
}
```

Los atributos no alteran el comportamiento por sí mismos. Otro código
(herramientas, frameworks, el runtime) los lee mediante reflexión y actúa
en consecuencia.

--- 2. ATRIBUTOS BUILT-IN COMUNES ---

.NET incluye docenas de atributos. Estos son los más utilizados:

```csharp
using System;
using System.Diagnostics;
using System.Runtime.InteropServices;

[Serializable]  // Permite serialización binaria
public class EjemploSerializable
{
    [NonSerialized]  // Excluye un campo de la serialización
    private string _cacheTemporal;
    
    public string Datos { get; set; }
}

[Obsolete("Esta clase está obsoleta. Use NuevaClase.")]
public class ClaseVieja { }

public class EjemploObsoleto
{
    [Obsolete("Este método está obsoleto.", false)]  // Warning
    public void MetodoViejo() { }
    
    [Obsolete("Este método causará error de compilación.", true)]  // Error
    public void MetodoRemovido() { }
}

public class EjemploConditional
{
    [Conditional("DEBUG")]
    public void SoloEnDebug()
    {
        Console.WriteLine("Esto solo se ejecuta en configuración DEBUG.");
    }
    
    [Conditional("TRACE")]
    public void SoloConTrace()
    {
        Console.WriteLine("Esto solo se ejecuta cuando TRACE está definido.");
    }
}

[Flags]
public enum Permisos
{
    Ninguno = 0,
    Lectura = 1,
    Escritura = 2,
    Ejecucion = 4,
    Administrador = Lectura | Escritura | Ejecucion
}

public class EjemploCallerInfo
{
    public void Log(
        string mensaje,
        [System.Runtime.CompilerServices.CallerMemberName] string miembro = "",
        [System.Runtime.CompilerServices.CallerFilePath] string archivo = "",
        [System.Runtime.CompilerServices.CallerLineNumber] int linea = 0)
    {
        Console.WriteLine($"[{miembro}] {archivo}:{linea} - {mensaje}");
    }
}
```

--- 3. ATTRIBUTE USAGE ---

AttributeUsage controla dónde puede aplicarse un atributo:

```csharp
using System;

// Restricciones básicas
[AttributeUsage(AttributeTargets.Class | AttributeTargets.Struct)]
public class DatosImportantesAttribute : Attribute { }

// Atributo que puede aplicarse múltiples veces
[AttributeUsage(AttributeTargets.Property, AllowMultiple = true)]
public class ValidacionAttribute : Attribute 
{ 
    public string Regla { get; }
    public ValidacionAttribute(string regla) => Regla = regla;
}

// Atributo que se hereda
[AttributeUsage(AttributeTargets.Class, Inherited = true)]
public class VersionableAttribute : Attribute 
{ 
    public int Version { get; }
    public VersionableAttribute(int version) => Version = version;
}

[AttributeUsage(AttributeTargets.Method, AllowMultiple = false, Inherited = false)]
public class MetodoCriticalAttribute : Attribute { }

// Demo de AttributeUsage
[Versionable(1)]
[DatosImportantes]
public class EntidadBase 
{ 
    [Validacion("Requerido")]
    [Validacion("MaxLength(100)")]
    public string Nombre { get; set; }
}

[Versionable(2)]  // Heredado = true, así que funciona
public class EntidadDerivada : EntidadBase 
{ 
    [MetodoCritical]
    public void Procesar() { }
}
```

--- 4. ATRIBUTO OBSOLETE EN DETALLE ---

```csharp
using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("=== ATRIBUTO OBSOLETE ===\n");
        
        var viejo = new SistemaViejo();
        viejo.MetodoLegacy();
        viejo.MetodoDeprecado();
        viejo.MetodoFuncional();
    }
}

public class SistemaViejo
{
    // Warning de compilación (mensaje opcional)
    [Obsolete("Use MetodoNuevo() en su lugar.")]
    public void MetodoLegacy()
    {
        Console.WriteLine("Implementación legacy.");
    }
    
    // Error de compilación
    [Obsolete("Este método ha sido eliminado. Use Initialize().", true)]
    public void MetodoDeprecado()
    {
        Console.WriteLine("Ya no debe usarse.");
    }
    
    // Sin obsoleto
    public void MetodoFuncional()
    {
        Console.WriteLine("Método actual.");
    }
    
    // Con información de versión
    [Obsolete("Reemplazado por Calcular(). Será eliminado en v3.0.")]
    public int CalcularViejo(int a, int b) => a + b;
    
    public int Calcular(int a, int b) => a + b;
}
```

--- 5. ATRIBUTO CONDITIONAL ---

```csharp
using System;
using System.Diagnostics;

class Program
{
    static void Main()
    {
        Console.WriteLine("=== ATRIBUTO CONDITIONAL ===\n");
        
        var utils = new Utilidades();
        
        // Estos métodos solo se compilan si DEBUG está definido
        utils.LogInfo("Inicio del programa");
        utils.LogWarning("Advertencia de prueba");
        
        // Este método se compila si TRACE está definido
        utils.LogTrace("Traza detallada");
        
        Console.WriteLine("Programa ejecutado.");
        
        utils.LogInfo("Fin del programa");
        
        // Nota: En Release, LogInfo y LogWarning no se compilan
        // Por lo tanto, las llamadas simplemente desaparecen del IL
    }
}

public class Utilidades
{
    [Conditional("DEBUG")]
    public void LogInfo(string mensaje)
    {
        Console.WriteLine($"[INFO] {mensaje}");
    }
    
    [Conditional("DEBUG")]
    public void LogWarning(string mensaje)
    {
        Console.WriteLine($"[WARN] {mensaje}");
    }
    
    [Conditional("TRACE")]
    public void LogTrace(string mensaje)
    {
        Console.WriteLine($"[TRACE] {mensaje}");
    }
    
    // Condicional múltiple (OR lógico)
    [Conditional("DEBUG"), Conditional("TRACE")]
    public void LogDiagnostico(string mensaje)
    {
        Console.WriteLine($"[DIAG] {mensaje}");
    }
}
```

--- 6. ATRIBUTO SERIALIZABLE ---

```csharp
using System;
using System.IO;
using System.Runtime.Serialization.Formatters.Binary;

[Serializable]
public class Empleado
{
    public string Nombre { get; set; }
    public int Edad { get; set; }
    
    [NonSerialized]  // No se serializa
    private string _password;
    
    public string Password 
    { 
        get => _password;
        set => _password = value;
    }
    
    [NonSerialized]  // Campo calculado, no se serializa
    public string InfoTemporal;
    
    public Empleado(string nombre, int edad, string password)
    {
        Nombre = nombre;
        Edad = edad;
        _password = password;
        InfoTemporal = $"Creado: {DateTime.Now}";
    }
    
    public override string ToString()
    {
        return $"Empleado: {Nombre}, Edad: {Edad}, " +
               $"Password: {(_password != null ? "***" : "null")}, " +
               $"InfoTemporal: {InfoTemporal}";
    }
}

class Program
{
    static void Main()
    {
        Console.WriteLine("=== ATRIBUTO SERIALIZABLE ===\n");
        
        var empleado = new Empleado("Juan Pérez", 30, "secreto123");
        empleado.InfoTemporal = "Datos cargados en memoria";
        
        Console.WriteLine("Antes de serializar:");
        Console.WriteLine($"  {empleado}");
        Console.WriteLine($"  InfoTemporal: {empleado.InfoTemporal}");
        
        // Serializar
        var formatter = new BinaryFormatter();
        string archivo = "empleado.bin";
        
        using (var stream = File.Create(archivo))
        {
            formatter.Serialize(stream, empleado);
        }
        Console.WriteLine($"\nEmpleado serializado en {archivo}");
        
        // Deserializar
        empleado.InfoTemporal = "Modificado después de serializar";
        
        using (var stream = File.OpenRead(archivo))
        {
            var empleadoCargado = (Empleado)formatter.Deserialize(stream);
            Console.WriteLine("\nDespués de deserializar:");
            Console.WriteLine($"  {empleadoCargado}");
            Console.WriteLine($"  InfoTemporal: {empleadoCargado.InfoTemporal ?? "null"}");
        }
    }
}
```

--- 7. ATRIBUTOS DE MARSHALING ---

```csharp
using System;
using System.Runtime.InteropServices;

class Program
{
    // DllImport usa atributos para definir P/Invoke
    [DllImport("user32.dll", CharSet = CharSet.Auto)]
    public static extern int MessageBox(
        IntPtr hWnd, 
        string text, 
        string caption, 
        uint type);
    
    [DllImport("kernel32.dll", SetLastError = true)]
    public static extern bool Beep(uint dwFreq, uint dwDuration);
    
    // StructLayout controla la disposición en memoria
    [StructLayout(LayoutKind.Sequential)]
    public struct Punto
    {
        public int X;
        public int Y;
    }
    
    [StructLayout(LayoutKind.Explicit)]
    public struct UnionExample
    {
        [FieldOffset(0)] public int Entero;
        [FieldOffset(0)] public float Flotante;
        [FieldOffset(4)] public long Largo;
    }
    
    static void Main()
    {
        // Ejemplo sencillo
        Beep(1000, 200);
        
        // StructLayout demo
        var p = new Punto { X = 10, Y = 20 };
        Console.WriteLine($"Punto: ({p.X}, {p.Y})");
        Console.WriteLine($"Tamaño: {Marshal.SizeOf(p)} bytes");
        
        var u = new UnionExample { Entero = 12345 };
        Console.WriteLine($"\nUnión - Entero: {u.Entero}, Flotante: {u.Flotante}");
    }
}
```

--- 8. ATRIBUTOS EN PARÁMETROS Y RETURN ---

```csharp
using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("=== ATRIBUTOS EN PARÁMETROS Y RETURN ===\n");
        
        var servicio = new Servicio();
        servicio.ProcesarDatos([Importante] "dato crítico");
        
        var resultado = servicio.ObtenerResultado();
        Console.WriteLine($"Resultado: {resultado}");
    }
}

// Atributo para parámetros
[AttributeUsage(AttributeTargets.Parameter)]
public class ImportanteAttribute : Attribute { }

// Atributo para valor de retorno
[AttributeUsage(AttributeTargets.ReturnValue)]
public class ResultadoCriticoAttribute : Attribute { }

public class Servicio
{
    public void ProcesarDatos([Importante] string datos)
    {
        Console.WriteLine($"Procesando datos importantes: {datos}");
        Console.WriteLine($"  Atributos del parámetro:");
        var parametro = typeof(Servicio)
            .GetMethod(nameof(ProcesarDatos))
            .GetParameters()[0];
        
        foreach (var attr in parametro.GetCustomAttributes(false))
            Console.WriteLine($"    - {attr.GetType().Name}");
    }
    
    [return: ResultadoCritico]
    public int ObtenerResultado()
    {
        Console.WriteLine("Obteniendo resultado crítico...");
        return 42;
    }
}
```

--- 11. ATRIBUTO Y COMPILADOR ---

```csharp
using System;
using System.Runtime.CompilerServices;

class Program
{
    static void Main()
    {
        Console.WriteLine("=== ATRIBUTOS DEL COMPILADOR ===\n");
        
        // MethodImpl controla optimizaciones
        var utils = new Utils();
        Console.WriteLine($"Suma: {utils.Sumar(3, 4)}");
        
        // UnsafeValueType
        Console.WriteLine($"Tamaño de Vector2: {System.Runtime.InteropServices.Marshal.SizeOf<Vector2>()} bytes");
    }
}

public class Utils
{
    // NoInline: el compilador no inlinea este método
    [MethodImpl(MethodImplOptions.NoInlining)]
    public int Sumar(int a, int b)
    {
        return a + b;
    }
    
    // AggressiveInlining: fuerza al compilador a inlinear
    [MethodImpl(MethodImplOptions.AggressiveInlining)]
    public int Duplicar(int x) => x * 2;
    
    // PreserveSig: no convertir HRESULT en excepción
    [MethodImpl(MethodImplOptions.PreserveSig)]
    public int MetodoComInterop() => 0;
}

// Struct con layout secuencial explícito
[System.Runtime.InteropServices.StructLayout(
    System.Runtime.InteropServices.LayoutKind.Sequential)]
public struct Vector2
{
    public float X;
    public float Y;
}
```

--- 13. ATRIBUTOS DE VALIDACIÓN (SIMULACIÓN) ---

```csharp
using System;
using System.Collections.Generic;
using System.Reflection;

// Sistema de validación basado en atributos

[AttributeUsage(AttributeTargets.Property)]
public class RequeridoAttribute : Attribute 
{
    public string MensajeError { get; set; }
    
    public RequeridoAttribute()
    {
        MensajeError = "El campo es requerido.";
    }
    
    public RequeridoAttribute(string mensaje)
    {
        MensajeError = mensaje;
    }
}

[AttributeUsage(AttributeTargets.Property)]
public class RangoAttribute : Attribute
{
    public int Minimo { get; }
    public int Maximo { get; }
    
    public RangoAttribute(int min, int max)
    {
        Minimo = min;
        Maximo = max;
    }
}

[AttributeUsage(AttributeTargets.Property)]
public class ExpresionRegularAttribute : Attribute
{
    public string Patron { get; }
    
    public ExpresionRegularAttribute(string patron)
    {
        Patron = patron;
    }
}

[AttributeUsage(AttributeTargets.Property)]
public class LongitudMaximaAttribute : Attribute
{
    public int Maximo { get; }
    
    public LongitudMaximaAttribute(int max) => Maximo = max;
}

// Validador genérico
public class Validador
{
    public static List<string> Validar<T>(T objeto)
    {
        var errores = new List<string>();
        var tipo = typeof(T);
        
        foreach (var prop in tipo.GetProperties())
        {
            var valor = prop.GetValue(objeto);
            
            // Validar Requerido
            var requerido = prop.GetCustomAttribute<RequeridoAttribute>();
            if (requerido != null)
            {
                if (valor == null || (valor is string s && string.IsNullOrWhiteSpace(s)))
                    errores.Add($"'{prop.Name}': {requerido.MensajeError}");
            }
            
            // Validar Rango
            var rango = prop.GetCustomAttribute<RangoAttribute>();
            if (rango != null && valor is int intVal)
            {
                if (intVal < rango.Minimo || intVal > rango.Maximo)
                    errores.Add($"'{prop.Name}' debe estar entre {rango.Minimo} y {rango.Maximo}.");
            }
            
            // Validar LongitudMaxima
            var longitud = prop.GetCustomAttribute<LongitudMaximaAttribute>();
            if (longitud != null && valor is string strVal)
            {
                if (strVal.Length > longitud.Maximo)
                    errores.Add($"'{prop.Name}' no debe exceder {longitud.Maximo} caracteres.");
            }
        }
        
        return errores;
    }
}

// Modelo con validación
public class UsuarioRegistro
{
    [Requerido(MensajeError = "El nombre de usuario es obligatorio.")]
    [LongitudMaxima(50)]
    public string NombreUsuario { get; set; }
    
    [Requerido]
    [LongitudMaxima(100)]
    public string Email { get; set; }
    
    [Rango(18, 120)]
    public int Edad { get; set; }
    
    [Requerido]
    public string Password { get; set; }
}

class Program
{
    static void Main()
    {
        Console.WriteLine("=== VALIDACIÓN CON ATRIBUTOS ===\n");
        
        var usuario = new UsuarioRegistro
        {
            NombreUsuario = "",
            Email = "test@ejemplo.com",
            Edad = 15,
            Password = null
        };
        
        var errores = Validador.Validar(usuario);
        
        if (errores.Count == 0)
        {
            Console.WriteLine("Usuario válido.");
        }
        else
        {
            Console.WriteLine("Errores de validación:");
            foreach (var error in errores)
                Console.WriteLine($"  - {error}");
        }
    }
}
```

--- 15. COMBINACIÓN DE ATRIBUTOS ---

```csharp
using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("=== COMBINACIÓN DE ATRIBUTOS ===\n");
        
        // Un elemento puede tener múltiples atributos
        var tipo = typeof(Configuracion);
        var atributos = tipo.GetCustomAttributes(false);
        
        Console.WriteLine($"Clase: {tipo.Name}");
        Console.WriteLine("Atributos:");
        
        foreach (var attr in atributos)
        {
            Console.WriteLine($"  - {attr.GetType().Name}");
            foreach (var prop in attr.GetType().GetProperties())
            {
                if (prop.Name != "TypeId")
                    Console.WriteLine($"      {prop.Name}: {prop.GetValue(attr)}");
            }
        }
    }
}

// Múltiples atributos en una clase
[Serializable]
```

```csharp
// Generador SQL basado en atributos
public class GeneradorSQL
{
    public string GenerarCreateTable<T>()
    {
        var tipo = typeof(T);
        var tablas = tipo.GetCustomAttributes<TablaAttribute>();
        
        // En producción se usaría la tabla específica
        return $"CREATE TABLE {tipo.Name} (...)";
    }
    
    public string GenerarInsert<T>(T entidad)
    {
        var tipo = typeof(T);
        var propiedades = tipo.GetProperties();
        var columnas = new List<string>();
        var valores = new List<string>();
        
        foreach (var prop in propiedades)
        {
            var columna = prop.GetCustomAttribute<ColumnaAttribute>();
            if (columna != null && !columna.EsAutonumerico)
            {
                columnas.Add(columna.Nombre);
                var valor = prop.GetValue(entidad);
                valores.Add(valor is string ? $"'{valor}'" : valor?.ToString() ?? "NULL");
            }
        }
        
        string tabla = tipo.GetCustomAttribute<TablaAttribute>()?.Nombre ?? tipo.Name;
        return $"INSERT INTO {tabla} ({string.Join(", ", columnas)}) " +
               $"VALUES ({string.Join(", ", valores)})";
    }
}

{

    {

        var generador = new GeneradorSQL();
        
        var cliente = new Cliente
        {
            Nombre = "Juan Pérez",
            Email = "juan@email.com",
            Telefono = "555-0123"
        };
        
        string sql = generador.GenerarInsert(cliente);
        Console.WriteLine("SQL generado:");
        Console.WriteLine(sql);
    }
}
```

```csharp
using System;
using System.Collections.Generic;
using System.Reflection;

[AttributeUsage(AttributeTargets.Class | AttributeTargets.Method)]
public class RequierePermisoAttribute : Attribute
{
    public string Permiso { get; }
    public string RolesPermitidos { get; set; }
    
    public RequierePermisoAttribute(string permiso)
    {
        Permiso = permiso;
    }
}

[AttributeUsage(AttributeTargets.Method)]
public class AuditoriaAttribute : Attribute
{
    public string Accion { get; }
    
    public AuditoriaAttribute(string accion) => Accion = accion;
}

public class ControlAcceso
{
    private readonly Dictionary<string, List<string>> _usuariosRoles;
    private readonly Dictionary<string, List<string>> _rolesPermisos;
    
    public ControlAcceso()
    {
        _usuariosRoles = new Dictionary<string, List<string>>
        {
            ["admin"] = new() { "Administrador" },
            ["jperez"] = new() { "Usuario", "Supervisor" },
            ["mgarcia"] = new() { "Usuario" }
        };
        
        _rolesPermisos = new Dictionary<string, List<string>>
        {
            ["Administrador"] = new() { "Leer", "Escribir", "Eliminar", "Administrar" },
            ["Supervisor"] = new() { "Leer", "Escribir" },
            ["Usuario"] = new() { "Leer" }
        };
    }
    
    public bool TienePermiso(string usuario, string permiso)
    {
        if (!_usuariosRoles.TryGetValue(usuario, out var roles))
            return false;
        
        foreach (var rol in roles)
        {
            if (_rolesPermisos.TryGetValue(rol, out var permisos))
            {
                if (permisos.Contains(permiso))
                    return true;
            }
        }
        
        return false;
    }
    
    public bool EjecutarSiTienePermiso(
        object objeto, string metodo, string usuario, params object[] args)
    {
        var tipo = objeto.GetType();
        var methodInfo = tipo.GetMethod(metodo);
        
        if (methodInfo == null)
            throw new ArgumentException($"Método '{metodo}' no encontrado.");
        
        var permisoAttr = methodInfo.GetCustomAttribute<RequierePermisoAttribute>();
        
        if (permisoAttr != null)
        {
            if (!TienePermiso(usuario, permisoAttr.Permiso))
            {
                Console.WriteLine($"ACCESO DENEGADO: '{usuario}' no tiene permiso '{permisoAttr.Permiso}'.");
                return false;
            }
        }
        
        // Ejecutar con auditoría
        var auditoria = methodInfo.GetCustomAttribute<AuditoriaAttribute>();
        if (auditoria != null)
        {
            Console.WriteLine($"[AUDITORÍA] {usuario} ejecutó {auditoria.Accion} a las {DateTime.Now:HH:mm:ss}");
        }
        
        methodInfo.Invoke(objeto, args);
        return true;
    }
}

public class ServicioFinanciero
{
    [RequierePermiso("Leer")]
    public void VerReportes()
    {
        Console.WriteLine("  Reportes visibles.");
    }
    
    [RequierePermiso("Escribir")]
    [Auditoria("CrearTransaccion")]
    public void CrearTransaccion(decimal monto)
    {
        Console.WriteLine($"  Transacción de {monto:C} creada.");
    }
    
    [RequierePermiso("Eliminar", RolesPermitidos = "Administrador")]
    public void EliminarTransaccion(int id)
    {
        Console.WriteLine($"  Transacción #{id} eliminada.");
    }
}

class Program
{
    static void Main()
    {
        Console.WriteLine("=== ATRIBUTOS Y SEGURIDAD ===\n");
        
        var servicio = new ServicioFinanciero();
        var control = new ControlAcceso();
        
        string[] usuarios = { "admin", "jperez", "mgarcia" };
        
        foreach (var usuario in usuarios)
        {
            Console.WriteLine($"\n--- Usuario: {usuario} ---");
            control.EjecutarSiTienePermiso(servicio, "VerReportes", usuario);
            control.EjecutarSiTienePermiso(servicio, "CrearTransaccion", usuario, 500m);
            control.EjecutarSiTienePermiso(servicio, "EliminarTransaccion", usuario, 1);
        }
    }
}
```

--- 19. ATRIBUTOS Y COMPORTAMIENTO EN TIEMPO DE EJECUCIÓN ---

```csharp
using System;
using System.Reflection;

class Program
{
    static void Main()
    {
        Console.WriteLine("=== ATRIBUTOS EN TIEMPO DE EJECUCIÓN ===\n");
        
        // Los atributos no afectan el comportamiento por sí mismos
        var ejemplo = new EjemploAtributos();
        ejemplo.MetodoNormal();
        // ejemplo.MetodoObsoleto(); // Esto daría warning/error en compilación
        
        // Pero podemos leerlos en runtime para modificar comportamiento
        var tipo = typeof(EjemploAtributos);
        var metodos = tipo.GetMethods();
        
        foreach (var metodo in metodos)
        {
            Console.WriteLine($"Método: {metodo.Name}");
            
            if (metodo.GetCustomAttribute<ObsoleteAttribute>() is ObsoleteAttribute obs)
            {
                Console.WriteLine($"  [OBSOLETO] {obs.Message} | Error: {obs.IsError}");
            }
            
            if (metodo.GetCustomAttribute<ConditionalAttribute>() is ConditionalAttribute cond)
            {
                Console.WriteLine($"  [CONDITIONAL] Símbolo: {cond.ConditionString}");
            }
        }
    }
}

public class EjemploAtributos
{
    public void MetodoNormal()
    {
        Console.WriteLine("Método normal ejecutado.");
    }
    
    [Obsolete("Usar MetodoNormal en su lugar.")]
    public void MetodoObsoleto()
    {
        Console.WriteLine("Método obsoleto.");
    }
    
    [Conditional("DEBUG")]
    public void MetodoDebug()
    {
        Console.WriteLine("Solo en debug.");
    }
}
```

================================================================================
  RESUMEN DEL CAPÍTULO
================================================================================

1. Los atributos son clases que heredan de Attribute y añaden metadatos.
2. Se usan entre corchetes [ ] antes del elemento.
3. [Serializable]: Permite serialización binaria.
4. [Obsolete]: Marca código como obsoleto (warning o error).
5. [Conditional]: Compila el método solo si el símbolo está definido.
6. [AttributeUsage]: Controla dónde y cómo se puede usar un atributo.
7. Los atributos por sí mismos no ejecutan código; otros componentes
   (reflexión, frameworks, compilador) los interpretan.
8. Atributos comunes: Serializable, Obsolete, Conditional, Flags,
   DllImport, StructLayout, CallerMemberName, etc.

================================================================================
  PREGUNTAS FRECUENTES (FAQ)
================================================================================

P: ¿Los atributos afectan el rendimiento?
R: Los atributos en sí no afectan el rendimiento. La reflexión para leerlos
sí tiene costo, pero puedes mitigarlo cacheando los resultados.

P: ¿Puedo crear un atributo sin heredar de Attribute?
R: No. Todas las clases de atributos deben heredar directa o indirectamente
de System.Attribute.

P: ¿Cuántos atributos puede tener un elemento?
R: Ilimitados, a menos que AllowMultiple = false en AttributeUsage.

P: ¿Los atributos se compilan en el ensamblado?
R: Sí. Los metadatos de los atributos se almacenan en el IL del ensamblado.

P: ¿Puedo cambiar un atributo en tiempo de ejecución?
R: No. Los atributos son metadatos estáticos incrustados en el ensamblado.
No se pueden modificar en runtime.
