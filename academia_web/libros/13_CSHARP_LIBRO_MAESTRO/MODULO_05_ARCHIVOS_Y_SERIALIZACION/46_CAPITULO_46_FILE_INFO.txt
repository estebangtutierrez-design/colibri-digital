1. LA CLASE FILE: MÉTODOS ESTÁTICOS
----------------------------------------------------------------------

File proporciona métodos estáticos para crear, copiar, eliminar, mover y abrir
archivos. Es la clase más conveniente para operaciones únicas.

Métodos de existencia y metadatos:
  ```csharp
  bool existe = File.Exists("archivo.txt");
  DateTime creado = File.GetCreationTime("archivo.txt");
  DateTime modificado = File.GetLastWriteTime("archivo.txt");
  DateTime accedido = File.GetLastAccessTime("archivo.txt");
  FileAttributes atributos = File.GetAttributes("archivo.txt");
  ```

Métodos de manipulación:
  ```csharp
  // Copiar
  File.Copy("origen.txt", "destino.txt");
  File.Copy("origen.txt", "destino.txt", overwrite: true); // Sobrescribir

  // Mover (también renombra)
  File.Move("origen.txt", "nuevo_nombre.txt");
  File.Move("origen.txt", @"subdir\archivo.txt"); // Mover a subdirectorio

  // Eliminar
  File.Delete("temporal.txt");

  // Crear
  using FileStream fs = File.Create("nuevo.txt");
  // Equivalente a new FileStream("nuevo.txt", FileMode.Create)
  ```

Métodos de apertura:
  ```csharp
  using FileStream fs1 = File.Open("archivo.txt", FileMode.Open);
  using FileStream fs2 = File.OpenRead("archivo.txt");  // Solo lectura
  using FileStream fs3 = File.OpenWrite("archivo.txt"); // Solo escritura
  using StreamReader sr = File.OpenText("archivo.txt"); // StreamReader
  ```

2. FILEINFO: LA ALTERNATIVA ORIENTADA A OBJETOS
----------------------------------------------------------------------

FileInfo representa un archivo como objeto. Es más eficiente cuando realizas
múltiples operaciones sobre el mismo archivo, porque no verifica la existencia
en cada llamada.

  ```csharp
  FileInfo fi = new FileInfo("documento.txt");

  // Propiedades
  Console.WriteLine($"Nombre: {fi.Name}");
  Console.WriteLine($"Nombre sin extensión: {fi.Name}");
  Console.WriteLine($"Extensión: {fi.Extension}");
  Console.WriteLine($"Directorio: {fi.DirectoryName}");
  Console.WriteLine($"Ruta completa: {fi.FullName}");
  Console.WriteLine($"Tamaño: {fi.Length} bytes");
  Console.WriteLine($"Existe: {fi.Exists}");
  Console.WriteLine($"Creado: {fi.CreationTime}");
  Console.WriteLine($"Modificado: {fi.LastWriteTime}");
  Console.WriteLine($"Atributos: {fi.Attributes}");
  ```

Métodos de FileInfo (similares a File pero como instancia):
  ```csharp
  FileInfo fi = new FileInfo("datos.txt");

  fi.CopyTo("copia.txt");
  fi.CopyTo("copia_sobrescrita.txt", overwrite: true);

  fi.MoveTo("renombrado.txt");
  fi.MoveTo(@"backup\archivo.txt");

  fi.Delete();

  using StreamReader sr = fi.OpenText();
  using FileStream fs = fi.Open(FileMode.Open, FileAccess.Read);
  using FileStream fs2 = fi.OpenRead();
  using FileStream fs3 = fi.OpenWrite();
  ```

Ventaja clave: FileInfo cachea los valores de las propiedades. Tras crear
el objeto, las propiedades se calculan una vez.

  ```csharp
  FileInfo fi = new FileInfo("log.txt");
  long tamano1 = fi.Length; // Consulta al disco
  long tamano2 = fi.Length; // Usa valor cacheado (más rápido)
  fi.Refresh();             // Refresca los valores desde el disco
  long tamano3 = fi.Length; // Consulta fresca
  ```

3. FILE VS FILEINFO: CUÁNDO USAR CADA UNO
----------------------------------------------------------------------

  Escenario                     | Usar File       | Usar FileInfo
  ------------------------------|-----------------|---------------------
  Una sola operación            | Sí              | No necesario
  Verificar existencia          | File.Exists()   | fi.Exists
  Múltiples operaciones mismo   | No eficiente    | Sí (caché)
  Obtener tamaño varias veces   | No              | Sí (Refresh)
  LINQ sobre colección archivos | No              | Sí (FileInfo[])
  Operación rápida única        | Sí              | Verboso innecesario
  Binding a UI                  | No              | Sí (propiedades)

Regla práctica:
  - Una llamada → métodos estáticos de File.
  - Múltiples llamadas sobre el mismo archivo → FileInfo.
  - Procesar colecciones de archivos → FileInfo[].

  ```csharp
  // Bueno para una sola vez
  if (File.Exists("config.json"))
      string json = File.ReadAllText("config.json");

  // Bueno para múltiples accesos
  FileInfo fi = new FileInfo("grande.log");
  if (fi.Exists)
  {
      Console.WriteLine($"Tamaño: {fi.Length}");
      Console.WriteLine($"Última modificación: {fi.LastWriteTime}");
      // Copia de seguridad
      fi.CopyTo($"backup_{fi.LastWriteTime:yyyyMMdd}.log");
  }
  ```

4. LA CLASE DIRECTORY
----------------------------------------------------------------------

Directory proporciona métodos estáticos para manipular directorios.

  ```csharp
  // Crear directorios
  Directory.CreateDirectory("nueva_carpeta");
  Directory.CreateDirectory(@"a\b\c"); // Crea toda la jerarquía

  // Verificar existencia
  bool existe = Directory.Exists("carpeta");

  // Eliminar
  Directory.Delete("carpeta_vacia");
  Directory.Delete("carpeta", recursive: true); // Elimina contenido

  // Mover
  Directory.Move("origen", "destino");

  // Obtener información
  DateTime creado = Directory.GetCreationTime("carpeta");
  DateTime modificado = Directory.GetLastWriteTime("carpeta");
  DirectoryInfo diPa = Directory.GetParent("carpeta\sub");
  string dirActual = Directory.GetCurrentDirectory();
  ```

Métodos de enumeración:
  ```csharp
  // Archivos
  string[] archivos = Directory.GetFiles("carpeta");
  string[] soloCS = Directory.GetFiles("carpeta", "*.cs");
  string[] todos = Directory.GetFiles("carpeta", "*.*", SearchOption.AllDirectories);

  // Directorios
  string[] dirs = Directory.GetDirectories("carpeta");
  string[] subDirs = Directory.GetDirectories("carpeta", "A*");

  // Archivos y directorios
  string[] entradas = Directory.GetFileSystemEntries("carpeta");
  ```

Diferencia entre GetFiles y EnumerateFiles:
  - GetFiles: devuelve string[], todos los resultados en memoria.
  - EnumerateFiles: devuelve IEnumerable<string>, lazy.

  ```csharp
  // GetFiles: carga todo
  string[] todosArchivos = Directory.GetFiles(@"C:\Windows", "*.dll",
      SearchOption.AllDirectories); // Puede ser mucha memoria

  // EnumerateFiles: lazy, procesa mientras enumera
  foreach (string archivo in Directory.EnumerateFiles(@"C:\Windows", "*.dll",
      SearchOption.AllDirectories))
  {
      Console.WriteLine(archivo); // Solo un string en memoria a la vez
  }
  ```

5. DIRECTORYINFO: DIRECTORIOS COMO OBJETOS
----------------------------------------------------------------------

DirectoryInfo es la contrapartida de instancia para directorios.

  ```csharp
  DirectoryInfo di = new DirectoryInfo("carpeta");

  // Propiedades
  Console.WriteLine($"Nombre: {di.Name}");
  Console.WriteLine($"Ruta completa: {di.FullName}");
  Console.WriteLine($"Padre: {di.Parent}");
  Console.WriteLine($"Raíz: {di.Root}");
  Console.WriteLine($"Existe: {di.Exists}");
  Console.WriteLine($"Creado: {di.CreationTime}");
  Console.WriteLine($"Atributos: {di.Attributes}");
  ```

Métodos de DirectoryInfo:
  ```csharp
  DirectoryInfo di = new DirectoryInfo("mi_carpeta");

  // Crear
  di.Create();
  di.CreateSubdirectory("subcarpeta");
  di.CreateSubdirectory(@"a\b\c");

  // Eliminar
  di.Delete();              // Solo si está vacía
  di.Delete(recursive: true); // Elimina todo el contenido

  // Mover
  di.MoveTo(@"otra_carpeta\mi_carpeta");

  // Enumerar
  FileInfo[] archivos = di.GetFiles();
  FileInfo[] CS = di.GetFiles("*.cs");
  DirectoryInfo[] subdirs = di.GetDirectories();
  FileSystemInfo[] entradas = di.GetFileSystemInfos();

  // Enumeración lazy
  foreach (FileInfo fi in di.EnumerateFiles("*.log"))
  {
      Console.WriteLine($"{fi.Name} - {fi.Length} bytes");
  }
  ```

6. PROPIEDADES Y METADATOS
----------------------------------------------------------------------

Tanto FileInfo como DirectoryInfo heredan de FileSystemInfo, que proporciona
propiedades comunes:

  ```csharp
  FileSystemInfo fsi = new FileInfo("archivo.txt");

  string nombre = fsi.Name;           // "archivo.txt"
  string fullName = fsi.FullName;     // "/home/user/archivo.txt"
  string extension = fsi.Extension;   // ".txt"
  bool exists = fsi.Exists;
  DateTime creationTime = fsi.CreationTime;
  DateTime lastWriteTime = fsi.LastWriteTime;
  DateTime lastAccessTime = fsi.LastAccessTime;
  FileAttributes attributes = fsi.Attributes;
  ```

Propiedades específicas de FileInfo:
  ```csharp
  FileInfo fi = new FileInfo("datos.bin");
  long tamano = fi.Length;          // Tamaño en bytes
  bool isReadOnly = fi.IsReadOnly;  // Atributo de solo lectura
  DirectoryInfo? dir = fi.Directory; // Directorio contenedor
  string dirName = fi.DirectoryName; // Ruta del directorio
  ```

Propiedades específicas de DirectoryInfo:
  ```csharp
  DirectoryInfo di = new DirectoryInfo("carpeta");
  DirectoryInfo? parent = di.Parent;    // Directorio padre
  DirectoryInfo root = di.Root;         // Raíz del volumen
  ```

Fecha en formato UTC:
  ```csharp
  FileInfo fi = new FileInfo("datos.txt");
  DateTime utcCreation = fi.CreationTimeUtc;
  DateTime utcModified = fi.LastWriteTimeUtc;
  DateTime utcAccessed = fi.LastAccessTimeUtc;
  ```

7. ATRIBUTOS DE ARCHIVO
----------------------------------------------------------------------

FileAttributes es una enumeración flags que describe los atributos del
archivo o directorio.

  ```csharp
  FileInfo fi = new FileInfo("documento.txt");
  FileAttributes attrs = fi.Attributes;

  // Verificar atributos individuales
  bool esSoloLectura = attrs.HasFlag(FileAttributes.ReadOnly);
  bool esOculto = attrs.HasFlag(FileAttributes.Hidden);
  bool esSistema = attrs.HasFlag(FileAttributes.System);
  bool esDirectorio = attrs.HasFlag(FileAttributes.Directory);
  bool esArchivo = attrs.HasFlag(FileAttributes.Archive);
  bool esComprimido = attrs.HasFlag(FileAttributes.Compressed);
  bool esEncriptado = attrs.HasFlag(FileAttributes.Encrypted);
  bool esTemporal = attrs.HasFlag(FileAttributes.Temporary);
  ```

Modificar atributos:
  ```csharp
  // Quitar solo lectura
  FileInfo fi = new FileInfo("protegido.txt");
  if (fi.Exists)
  {
      FileAttributes attrs = fi.Attributes;
      if (attrs.HasFlag(FileAttributes.ReadOnly))
      {
          fi.Attributes = attrs & ~FileAttributes.ReadOnly;
          Console.WriteLine("Atributo ReadOnly eliminado.");
      }
  }

  // Hacer oculto
  File.SetAttributes("secreto.txt", FileAttributes.Hidden);

  // Combinar atributos
  File.SetAttributes("importante.txt",
      FileAttributes.Archive | FileAttributes.ReadOnly);
  ```

Ocultar y mostrar archivos:
  ```csharp
  public static void OcultarArchivo(string path)
  {
      FileInfo fi = new FileInfo(path);
      if (fi.Exists && !fi.Attributes.HasFlag(FileAttributes.Hidden))
      {
          fi.Attributes = fi.Attributes | FileAttributes.Hidden;
      }
  }

  public static void MostrarArchivo(string path)
  {
      FileInfo fi = new FileInfo(path);
      if (fi.Exists && fi.Attributes.HasFlag(FileAttributes.Hidden))
      {
          fi.Attributes = fi.Attributes & ~FileAttributes.Hidden;
      }
  }
  ```

Atributos de directorio:
  ```csharp
  DirectoryInfo di = new DirectoryInfo("carpeta");
  if (di.Attributes.HasFlag(FileAttributes.Directory))
      Console.WriteLine("Es un directorio");

  // Los directorios no pueden tener contenido (longitud 0)
  Console.WriteLine($"Bytes del directorio: {di.Attributes.HasFlag(FileAttributes.ReparsePoint) ? "Enlace simbólico" : "Directorio normal"}");
  ```

8. OPERACIONES DE COPIA, MOVIMIENTO Y ELIMINACIÓN
----------------------------------------------------------------------

Copia:
  ```csharp
  // File estático
  File.Copy("origen.txt", "destino.txt");
  File.Copy("origen.txt", "destino.txt", overwrite: true);

  // FileInfo
  FileInfo fiOrigen = new FileInfo("origen.txt");
  fiOrigen.CopyTo("destino.txt");
  fiOrigen.CopyTo("destino.txt", overwrite: true);
  ```

Movimiento (también renombrado):
  ```csharp
  // File estático
  File.Move("viejo.txt", "nuevo.txt");       // Renombrar
  File.Move("archivo.txt", @"subdir\archivo.txt"); // Mover

  // Si el destino existe, lanza IOException
  // No hay overwrite en Move (a diferencia de Copy)

  // FileInfo
  FileInfo fi = new FileInfo("origen.txt");
  fi.MoveTo("destino.txt");

  // Directory
  Directory.Move("carpeta", "nueva_ubicacion");
  ```

Eliminación:
  ```csharp
  // File
  File.Delete("temporal.txt"); // No lanza error si no existe

  // FileInfo
  FileInfo fi = new FileInfo("temporal.txt");
  fi.Delete(); // Lanza excepción si no existe

  // Directory
  Directory.Delete("carpeta_vacia");
  Directory.Delete("carpeta_contenido", recursive: true);

  // DirectoryInfo
  DirectoryInfo di = new DirectoryInfo("carpeta");
  di.Delete(true);
  ```

Diferencia clave: File.Delete no lanza excepción si el archivo no existe.
FileInfo.Delete lanza FileNotFoundException.

9. ENUMERACIÓN DE ARCHIVOS Y DIRECTORIOS
----------------------------------------------------------------------

Enumeración básica:
  ```csharp
  DirectoryInfo di = new DirectoryInfo(".");

  // Todos los archivos
  foreach (FileInfo fi in di.GetFiles())
      Console.WriteLine($"{fi.Name} - {fi.Length} bytes");

  // Con filtro
  foreach (FileInfo fi in di.GetFiles("*.txt"))
      Console.WriteLine(fi.FullName);

  // Recursivo
  foreach (FileInfo fi in di.GetFiles("*.cs", SearchOption.AllDirectories))
      Console.WriteLine(fi.FullName);
  ```

Enumeración lazy (recomendado para directorios grandes):
  ```csharp
  DirectoryInfo di = new DirectoryInfo(@"C:\Users");

  // EnumerateFiles: no carga todo en memoria
  foreach (FileInfo fi in di.EnumerateFiles("*.*", SearchOption.AllDirectories))
  {
      if (fi.Length > 100_000_000) // Archivos > 100 MB
      {
          Console.WriteLine($"Archivo grande: {fi.FullName} ({fi.Length / 1e6:F1} MB)");
      }
  }
  ```

Listar estructura completa:
  ```csharp
  static void ListarArbol(DirectoryInfo dir, string prefix = "")
  {
      foreach (DirectoryInfo subdir in dir.GetDirectories())
      {
          Console.WriteLine($"{prefix}[{subdir.Name}]");
          ListarArbol(subdir, prefix + "  ");
      }

      foreach (FileInfo fi in dir.GetFiles())
      {
          Console.WriteLine($"{prefix}{fi.Name} ({fi.Length} bytes)");
      }
  }
  ```

10. SEGURIDAD Y PERMISOS
----------------------------------------------------------------------

Comprobar permisos antes de operar:
  ```csharp
  public static bool CanRead(string path)
  {
      try
      {
          using FileStream fs = File.Open(path, FileMode.Open,
              FileAccess.Read, FileShare.ReadWrite);
          return true;
      }
      catch (UnauthorizedAccessException) { return false; }
      catch (IOException) { return false; }
  }

  public static bool CanWrite(string path)
  {
      try
      {
          using FileStream fs = File.Open(path, FileMode.OpenOrCreate,
              FileAccess.Write, FileShare.ReadWrite);
          return true;
      }
      catch (UnauthorizedAccessException) { return false; }
      catch (IOException) { return false; }
  }
  ```

Atributos de solo lectura:
  ```csharp
  // Hacer un archivo editable
  FileInfo fi = new FileInfo("protegido.txt");
  if (fi.IsReadOnly)
      fi.IsReadOnly = false;

  // Proteger archivo contra escritura
  File.SetAttributes("importante.txt",
      File.GetAttributes("importante.txt") | FileAttributes.ReadOnly);
  ```

11. BUENAS PRÁCTICAS
----------------------------------------------------------------------

a) No asumas que un archivo existe después de verificar con File.Exists,
   porque otro proceso podría eliminarlo entre la verificación y la operación.

b) Usa Directory.CreateDirectory antes de crear archivos en una ruta nueva.
   No lanza error si el directorio ya existe.

c) Prefiere EnumerateFiles sobre GetFiles para directorios con muchos archivos.

d) FileInfo y DirectoryInfo cachean propiedades. Usa Refresh() si el archivo
   pudo cambiar externamente.

e) Usa Path.Combine para construir rutas, no concatenación de strings.

f) File.Delete no lanza excepción si el archivo no existe. Las demás
   operaciones (Move, Copy) sí lanzan.

g) Para archivos temporales, usa Path.GetTempFileName() o
   Path.GetRandomFileName().

h) Siempre maneja excepciones de I/O en código de producción.

  CÓDIGO DE EJEMPLO COMPLETO
================================================================================

  ```csharp
  using System;
  using System.Collections.Generic;
  using System.IO;
  using System.Linq;

  namespace Capitulo46.FileFileInfoDirectory
  {
      class Program
      {
          static void Main(string[] args)
          {
              string dir = "ejemplos_fileinfo";
              Directory.CreateDirectory(dir);

              Console.WriteLine("=== DEMO 1: FILE ESTÁTICO ===");
              string ruta1 = Path.Combine(dir, "test_static.txt");

              Console.WriteLine("=== DEMO 2: FILEINFO PROPIEDADES ===");
              FileInfo fi = new FileInfo(ruta1);
              Console.WriteLine($"  Name: {fi.Name}");

              Console.WriteLine($"  Extension: {fi.Extension}");
              Console.WriteLine($"  DirectoryName: {fi.DirectoryName}");
              Console.WriteLine($"  Length: {fi.Length} bytes");
              Console.WriteLine($"  IsReadOnly: {fi.IsReadOnly}");
              Console.WriteLine($"  CreationTime: {fi.CreationTime}");
              Console.WriteLine($"  LastWriteTime: {fi.LastWriteTime}");
              Console.WriteLine($"  LastAccessTime: {fi.LastAccessTime}");
              Console.WriteLine($"  Attributes: {fi.Attributes}");
              Console.WriteLine();

              Console.WriteLine("=== DEMO 3: COPY, MOVE, DELETE ===");
              string ruta3a = Path.Combine(dir, "original.txt");
              string ruta3b = Path.Combine(dir, "copia.txt");
              string ruta3c = Path.Combine(dir, "movido.txt");
              File.WriteAllText(ruta3a, "Original");
              File.Copy(ruta3a, ruta3b);
              Console.WriteLine($"  Copia creada: {File.Exists(ruta3b)}");
              File.Move(ruta3b, ruta3c);
              Console.WriteLine($"  Movido existe: {File.Exists(ruta3c)}");
              Console.WriteLine($"  Original ya no en copia: {!File.Exists(ruta3b)}");
              File.Delete(ruta3a);
              Console.WriteLine($"  Original eliminado: {!File.Exists(ruta3a)}");
              Console.WriteLine();

              Console.WriteLine("=== DEMO 4: DIRECTORYINFO ===");
              DirectoryInfo di = new DirectoryInfo(dir);
              Console.WriteLine($"  Name: {di.Name}");
              Console.WriteLine($"  FullName: {di.FullName}");
              Console.WriteLine($"  Parent: {di.Parent?.Name}");
              Console.WriteLine($"  Root: {di.Root}");
              Console.WriteLine($"  Exists: {di.Exists}");
              Console.WriteLine($"  CreationTime: {di.CreationTime}");
              Console.WriteLine();

              Console.WriteLine("=== DEMO 5: GETFILES Y GETDIRECTORIES ===");
              Directory.CreateDirectory(Path.Combine(dir, "sub1"));
              Directory.CreateDirectory(Path.Combine(dir, "sub2"));
              File.WriteAllText(Path.Combine(dir, "sub1", "a.txt"), "A");
              File.WriteAllText(Path.Combine(dir, "sub1", "b.log"), "B");
              File.WriteAllText(Path.Combine(dir, "sub2", "c.txt"), "C");

              DirectoryInfo di5 = new DirectoryInfo(dir);
              Console.WriteLine("  Subdirectorios:");
              foreach (DirectoryInfo sd in di5.GetDirectories())
                  Console.WriteLine($"    {sd.Name}");
              Console.WriteLine("  Archivos .txt:");
              foreach (FileInfo f in di5.GetFiles("*.txt"))
                  Console.WriteLine($"    {f.Name}");
              Console.WriteLine("  Todos recursivo:");
              foreach (FileInfo f in di5.GetFiles("*.*", SearchOption.AllDirectories))
                  Console.WriteLine($"    {f.FullName}");
              Console.WriteLine();

              Console.WriteLine("=== DEMO 6: ENUMERATEFILES (LAZY) ===");
              foreach (FileInfo f in di5.EnumerateFiles("*.*", SearchOption.AllDirectories))
              {
                  Console.WriteLine($"  {f.Name} ({f.Length} bytes)");
              }
              Console.WriteLine();

              Console.WriteLine("=== DEMO 7: ATRIBUTOS ===");
              string ruta7 = Path.Combine(dir, "atributos.txt");
              File.WriteAllText(ruta7, "Test de atributos");
              FileInfo fi7 = new FileInfo(ruta7);
              Console.WriteLine($"  Attributes: {fi7.Attributes}");
              Console.WriteLine($"  IsReadOnly: {fi7.IsReadOnly}");
              Console.WriteLine($"  Hidden: {fi7.Attributes.HasFlag(FileAttributes.Hidden)}");
              Console.WriteLine($"  Archive: {fi7.Attributes.HasFlag(FileAttributes.Archive)}");

              // Marcar como solo lectura
              fi7.Attributes = fi7.Attributes | FileAttributes.ReadOnly;
              Console.WriteLine($"  Ahora ReadOnly: {fi7.IsReadOnly}");
              fi7.Attributes = fi7.Attributes & ~FileAttributes.ReadOnly;
              Console.WriteLine($"  ReadOnly quitado: {!fi7.IsReadOnly}");
              Console.WriteLine();

              Console.WriteLine("=== DEMO 8: DIRECTORY CREATE ===");
              string ruta8 = Path.Combine(dir, "a", "b", "c");
              DirectoryInfo di8 = Directory.CreateDirectory(ruta8);
              Console.WriteLine($"  Creada jerarquía: {di8.FullName}");
              Console.WriteLine($"  Existe: {Directory.Exists(ruta8)}");
              Console.WriteLine();

              Console.WriteLine("=== DEMO 9: MOVER DIRECTORIO ===");
              string dir9a = Path.Combine(dir, "mover_origen");
              string dir9b = Path.Combine(dir, "mover_destino");
              Directory.CreateDirectory(dir9a);
              File.WriteAllText(Path.Combine(dir9a, "interno.txt"), "interno");
              Directory.Move(dir9a, dir9b);
              Console.WriteLine($"  Directorio movido: {Directory.Exists(dir9b)}");
              Console.WriteLine($"  Origen eliminado: {!Directory.Exists(dir9a)}");
              Console.WriteLine();

              Console.WriteLine("=== DEMO 10: FILEINFO VS FILE ===");
              string ruta10 = Path.Combine(dir, "benchmark.txt");
              // Método estático (una llamada)
              var sw = System.Diagnostics.Stopwatch.StartNew();
              for (int i = 0; i < 1000; i++)
                  File.Exists(ruta10);
              sw.Stop();
              Console.WriteLine($"  File.Exists x1000: {sw.ElapsedMilliseconds} ms");

              // FileInfo (múltiples llamadas)
              FileInfo fi10 = new FileInfo(ruta10);
              sw.Restart();
              for (int i = 0; i < 1000; i++)
              {
                  bool ex = fi10.Exists; // Usa caché
                  fi10.Refresh();         // Refresca cada 10
                  if (i % 10 == 0) fi10.Refresh();
              }
              sw.Stop();
              Console.WriteLine($"  FileInfo.Exists x1000: {sw.ElapsedMilliseconds} ms");

              // Limpieza
              Directory.Delete(dir, true);
              Console.WriteLine("\n¡Todas las demostraciones completadas!");
          }
      }
  }
  ```

================================================================================
  PREGUNTAS FRECUENTES (FAQ)
================================================================================

P: ¿File.Exists es costosa?
R: Sí, relativamente. Cada llamada consulta al sistema de archivos.
   Para múltiples verificaciones, usa FileInfo.

P: ¿Cuándo debo usar DirectoryInfo en lugar de Directory?
R: Cuando necesites múltiples operaciones sobre el mismo directorio o
   cuando quieras cachear propiedades.

P: ¿FileInfo.Length se actualiza automáticamente?
R: No. FileInfo cachea los valores. Usa Refresh() para obtener datos
   actualizados si el archivo cambió externamente.

P: ¿Qué SearchOption debo usar?
R: TopDirectoryOnly (default) para no recursivo; AllDirectories para
   búsqueda completa. Ten cuidado con directorios muy profundos.

P: ¿GetFiles vs EnumerateFiles?
R: GetFiles devuelve string[] (todo en memoria); EnumerateFiles devuelve
   IEnumerable<string> (lazy). Usa EnumerateFiles para grandes volúmenes.

  RESUMEN DEL CAPÍTULO

Hemos aprendido:

  - File: métodos estáticos para operaciones rápidas (Exists, Copy, Move, Delete).
  - FileInfo: instancia para múltiples operaciones con caché de propiedades.
  - Directory: métodos estáticos para directorios (CreateDirectory, GetFiles).
  - DirectoryInfo: instancia para directorios con EnumerateFiles.
  - Propiedades: Name, FullName, Extension, Length, CreationTime, Attributes.
  - Atributos: FileAttributes (ReadOnly, Hidden, Archive, etc.).
  - Copia: File.Copy, FileInfo.CopyTo (con overwrite).
  - Movimiento: File.Move, Directory.Move.
  - Eliminación: File.Delete (no lanza si no existe), FileInfo.Delete (sí lanza).
  - Enumeración: GetFiles vs EnumerateFiles (lazy).
