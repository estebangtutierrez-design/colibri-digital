CONTENIDO DEL CAPÍTULO
===============================================================================

1. REFACTORING FUNDAMENTAL

Extraer método/función. Renombrar variable/función/clase. Mover método a

otra clase. Extraer clase. Introducir parámetro. Encapsular campo.



2. EXTRACT METHOD

Fragmento de código que puede agruparse. Crear nueva función con nombre

descriptivo. Reemplazar fragmento con llamada. Mejora legibilidad y

reutilización. Método < 10 líneas.



3. RENAME

Cuando un nombre no revela intención. IDEs modernos (VS Code, IntelliJ)

tienen refactorings automáticos seguros. Renombrar es barato y seguro.



4. REPLACE CONDITIONAL WITH POLYMORPHISM

Switch/if-else según tipo → herencia/polimorfismo. Cada subclase implementa

su comportamiento. Nuevos tipos no modifican código existente (OCP).

# Despues
def process_order(order):
  final = calculate_final_price(order)
  send_confirmation(order.email, final)
  update_inventory(order.items)

def calculate_final_price(order):
  total = sum(item.price for item in order.items)
  tax = calculate_tax(total)
  discount = apply_discount(total, order.coupon)
  shipping = calculate_shipping(order.address)
  return total + tax - discount + shipping
