CONTENIDO DEL CAPÍTULO
===============================================================================

Spring Boot: framework Java para microservicios. Auto-configuración.

Starters: web, jpa, security, data-rest. Embedded Tomcat.

start.spring.io. mvn/gradle wrapper.

EJEMPLO DE CÓDIGO
----------------------------------------

@SpringBootApplication
public class Application {
  public static void main(String[] args) {
    SpringApplication.run(Application.class, args);
  }
}

@RestController
@RequestMapping("/api")
public class UserController {
  @GetMapping("/hello")
  public String hello() { return "Hello"; }
}
