CONTENIDO DEL CAPÍTULO
===============================================================================

1. SEGURIDAD EN SWITCHES

Port Security: límite de MACs por puerto. DHCP Snooping: valida DHCP.

Dynamic ARP Inspection: valida ARP. Storm Control: limita broadcasts.



2. DHCP SNOOPING

Filtra mensajes DHCP no autorizados. Puerto confiable (trunk, hacia DHCP

server) vs no confiable (acceso). Rate limiting de DHCP.



3. DAI (DYNAMIC ARP INSPECTION)

Valida paquetes ARP usando tabla DHCP Snooping. Previene ARP spoofing.

Verifica MAC/IP/interface. ARP ACLs para más control.



4. VLAN SEGURIDAD

VLAN Hopping: evitar saltos entre VLANs. Private VLANs: aislamiento dentro

de VLAN. VACL (VLAN ACLs). PVLAN Edge (protected ports).

EJEMPLO DE CÓDIGO
----------------------------------------

# Port Security
interface gigabitethernet0/1
 switchport port-security
 switchport port-security maximum 2
 switchport port-security violation shutdown
 switchport port-security mac-address sticky

# DHCP Snooping
ip dhcp snooping
ip dhcp snooping vlan 100,200
interface gigabitethernet0/1
 ip dhcp snooping trust
interface gigabitethernet0/2
 ip dhcp snooping limit rate 10

# Dynamic ARP Inspection
ip arp inspection vlan 100
interface gigabitethernet0/1
 ip arp inspection trust
