CONTENIDO DEL CAPÍTULO
===============================================================================

1. HTTP/2

Multiplexación: múltiples streams en una conexión. Server push. Compresión

de headers (HPACK). Binario (vs texto en HTTP/1.1). Prioritización.



2. HTTP/3 (QUIC)

Basado en QUIC (UDP). 0-RTT handshake. Multiplexación sin HOL blocking.

Conexión migrable (cambia IP sin reconectar). Cifrado integrado.



3. gRPC

RPC moderno de Google. HTTP/2 subyacente. Protocol Buffers (protobuf).

Streaming bidireccional. Generación de clientes automática.



4. WEBSOCKETS

Full-duplex sobre TCP. Handshake HTTP, luego actualiza a WebSocket.

Tiempo real: chats, juegos, notificaciones. Socket.io (Node.js).

EJEMPLO DE CÓDIGO
----------------------------------------

# Nginx HTTP/2
server {
  listen 443 ssl http2;
  server_name example.com;
  # ... configuracion SSL
}

# gRPC con Nginx
location / {
  grpc_pass grpc://backend:50051;
}

# WebSocket en Nginx
location /ws/ {
  proxy_pass http://backend:3000;
  proxy_http_version 1.1;
  proxy_set_header Upgrade $http_upgrade;
  proxy_set_header Connection "upgrade";
}

# QUIC en Nginx (HTTP/3)
server {
  listen 443 quic reuseport;
  http3 on;
}
