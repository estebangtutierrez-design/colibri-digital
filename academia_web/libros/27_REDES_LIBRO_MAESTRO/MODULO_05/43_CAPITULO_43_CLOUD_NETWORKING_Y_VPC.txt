CONTENIDO DEL CAPÍTULO
===============================================================================

1. VPC (VIRTUAL PRIVATE CLOUD)

Red virtual aislada en cloud. Subredes públicas y privadas. Internet

Gateway. NAT Gateway. Security Groups (firewall estado). Route tables.



2. AWS VPC

CIDR block (10.0.0.0/16). Subnets en Availability Zones. Internet Gateway

para tráfico público. NAT Gateway para privadas. VPC Peering. VPN.



3. AZURE VNET

Virtual Network. Subnets. Network Security Groups. Service Endpoints.

Azure VPN Gateway. ExpressRoute (conexión dedicada). VNet Peering.



4. GCP VPC

Global VPC (todas regions). Subnets por región. Cloud NAT. Cloud Router.

Shared VPC. VPC Network Peering. Cloud Armor (WAF).

EJEMPLO DE CÓDIGO
----------------------------------------

# AWS VPC con Terraform
resource "aws_vpc" "main" {
  cidr_block = "10.0.0.0/16"
}

resource "aws_subnet" "public" {
  vpc_id     = aws_vpc.main.id
  cidr_block = "10.0.1.0/24"
}

resource "aws_security_group" "web" {
  vpc_id = aws_vpc.main.id
  ingress {
    from_port   = 80
    to_port     = 80
    protocol    = "tcp"
    cidr_blocks = ["0.0.0.0/0"]
  }
}

resource "aws_internet_gateway" "gw" {
  vpc_id = aws_vpc.main.id
}
