CONTENIDO DEL CAPÍTULO
===============================================================================

1. SNMP (SIMPLE NETWORK MANAGEMENT PROTOCOL)

Protocolo para gestionar y monitorear dispositivos de red. Versiones:

v1 (básica), v2c (community string), v3 (autenticación y cifrado).



2. MIB (MANAGEMENT INFORMATION BASE)

Base de datos jerárquica de variables gestionables. .1.3.6.1.2.1 (mib-2).

Interfaces, sistema, IP, TCP, UDP, almacenamiento, procesos.



3. OID (OBJECT IDENTIFIER)

.1.3.6.1.2.1.1.5.0 sysName. .1.3.6.1.2.1.25.3.3.1.2 CPU load.

.1.3.6.1.2.1.25.2.3.1.6 disk usage. snmpwalk lista árbol OID.



4. HERRAMIENTAS

snmpwalk: recorre árbol MIB. snmpget: obtiene valor específico.

snmpd: agente SNMP. snmptrap: receptor de traps (alertas). MRTG/Cacti:

gráficos históricos. Nagios/Zabbix: monitoreo completo.

EJEMPLO DE CÓDIGO
----------------------------------------

# snmpd.conf
rocommunity public 192.168.1.0/24
sysLocation Datacenter
sysContact admin@example.com

# Consultas SNMP
snmpwalk -v2c -c public localhost system
snmpget -v2c -c public localhost sysName.0
snmpwalk -v2c -c public localhost interfaces
snmpwalk -v2c -c public localhost hrProcessorLoad

# Traps
snmptrap -v2c -c public manager.example.com '' NET-SNMP-EXAMPLES-MIB::netSnmpExampleHeartbeat

# MRTG para graficos
target[router]: /public@192.168.1.1
maxbytes[router]: 1250000
Option[_]: growright, bits
