CONTENIDO DEL CAPÍTULO
===============================================================================

1. SIP (SESSION INITIATION PROTOCOL)

Señalización para llamadas VoIP. Establece, modifica y termina sesiones.

Puertos 5060 (UDP/TCP) y 5061 (TLS). Mensajes: INVITE, ACK, BYE, REGISTER.



2. RTP/RTCP

Real-time Transport Protocol: transporte de audio/video. Puertos pares

16384-32786. RTCP: estadísticas de calidad (jitter, pérdida de paquetes).



3. CÓDECS

G.711: sin compresión, 64 kbps. G.729: comprimido, 8 kbps. Opus: libre,

6-510 kbps, calidad variable. H.264: video. VP8/VP9: video libre.



4. PBX IP

FreePBX/Asterisk: centralita telefónica IP. Extensiones, IVR, colas,

grabación. Voicemail, conferencias. Integración con SIP trunk (ISP).

EJEMPLO DE CÓDIGO
----------------------------------------

# Extension Asterisk
# /etc/asterisk/sip.conf
[1001]
type=friend
host=dynamic
secret=secreto
context=internal
qualify=yes

# /etc/asterisk/extensions.conf
[internal]
exten => _XXXX,1,Dial(SIP/${EXTEN},30)
exten => _XXXX,2,Voicemail(${EXTEN}@default)
exten => _XXXX,3,Hangup

# SIP INVITE ejemplo
INVITE sip:1001@pbx.example.com SIP/2.0
Via: SIP/2.0/UDP 192.168.1.10:5060
From: <sip:1000@example.com>
To: <sip:1001@example.com>
Call-ID: abc123@192.168.1.10
CSeq: 1 INVITE
Contact: <sip:1000@192.168.1.10:5060>
