CONTENIDO DEL CAPÍTULO

1. NTP

Network Time Protocol. Puerto 123 UDP. Sincroniza relojes en redes.

Estratificación (stratum): 0 (reloj atómico), 1, 2... Exactitud: ms.



2. SERVIDORES NTP

pool.ntp.org (pool mundial). es.pool.ntp.org (España). time.google.com.

0.pool.ntp.org, 1.pool.ntp.org, 2.pool.ntp.org, 3.pool.ntp.org.



3. CLIENTE NTP

ntpd: demonio de sincronización continua. ntpdate: sincronización única.

chronyd: moderno (reemplaza ntpd en RHEL/CentOS 8+). systemd-timesyncd.



4. CHRONY

chrony.conf: pool, server, makestep. chronyc sources -v ver fuentes.

chronyc tracking estadísticas. Mejor manejo de redes intermitentes.

EJEMPLO DE CÓDIGO
----------------------------------------

# /etc/chrony/chrony.conf
pool 0.pool.ntp.org iburst
pool 1.pool.ntp.org iburst
makestep 1.0 3
rtcsync

# systemd-timesyncd
# /etc/systemd/timesyncd.conf
[NTP]
NTP=0.pool.ntp.org 1.pool.ntp.org
FallbackNTP=ntp.ubuntu.com

# Comandos
$ timedatectl status
$ timedatectl set-ntp true
$ chronyc sources -v
$ chronyc tracking
$ ntpq -p
