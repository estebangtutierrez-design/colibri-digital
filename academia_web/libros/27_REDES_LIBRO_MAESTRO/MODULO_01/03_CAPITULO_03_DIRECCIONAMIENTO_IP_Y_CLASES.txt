CONTENIDO DEL CAPÍTULO
===============================================================================

1. DIRECCIONES IP

IPv4: 32 bits (4 octetos). 192.168.1.1. ~4.3 mil millones de direcciones.

IPv6: 128 bits (8 grupos hex). 2001:db8::1. Soluciona agotamiento IPv4.



2. CLASES DE IPv4

Clase A: 1.0.0.0 a 126.255.255.255 (16M hosts). Clase B: 128.0.0.0 a

191.255.255.255 (65K hosts). Clase C: 192.0.0.0 a 223.255.255.255 (254 hosts).

Clase D: multicast. Clase E: reservada.



3. DIRECCIONES PRIVADAS

10.0.0.0/8 (Clase A). 172.16.0.0/12 (Clase B). 192.168.0.0/16 (Clase C).

127.0.0.0/8 (loopback/localhost). 169.254.0.0/16 (APIPA/link-local).



4. CIDR (CLASSLESS INTER-DOMAIN ROUTING)

Notación: 192.168.1.0/24 (máscara 255.255.255.0). /8, /16, /24, /32.

Permite subdivisión flexible sin clases fijas.

EJEMPLO DE CÓDIGO
----------------------------------------

# Ejemplos CIDR
192.168.1.0/24  # 256 direcciones (254 hosts)
10.0.0.0/8      # 16M direcciones
172.16.0.0/12   # 1M direcciones

# Calcular: 2^(32-mascara) = total direcciones
# /24: 2^8 = 256
# /16: 2^16 = 65536
# /8: 2^24 = 16777216
