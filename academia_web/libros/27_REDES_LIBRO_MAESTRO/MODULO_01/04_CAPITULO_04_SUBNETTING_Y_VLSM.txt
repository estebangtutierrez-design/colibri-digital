1. ¿QUÉ ES SUBNETTING?

Dividir una red grande en subredes más pequeñas. Mejora eficiencia,

seguridad y gestión. Reduce el tráfico de broadcast.



2. CÁLCULO DE SUBNETES

Para /26: 4 subredes de 62 hosts cada una. Primer host: .1, último: .62.

Broadcast: .63. Red: .0. Máscara: 255.255.255.192.



3. VLSM (VARIABLE LENGTH SUBNET MASK)

Máscaras de longitud variable dentro de la misma red. Optimiza uso de

direcciones. Asigna tamaño de subred según necesidad específica.



4. EJEMPLO PRÁCTICO

Red 192.168.1.0/24. Subred A (50 hosts): /26 (62 hosts). Subred B (25

hosts): /27 (30 hosts). Subred C (10 hosts): /28 (14 hosts). Punto a

punto: /30 (2 hosts).

EJEMPLO DE CÓDIGO
----------------------------------------

# 192.168.1.0/24 a subredes
# /26 (255.255.255.192)
# Subred 0: 192.168.1.0 - 192.168.1.63
# Subred 1: 192.168.1.64 - 192.168.1.127
# Subred 2: 192.168.1.128 - 192.168.1.191
# Subred 3: 192.168.1.192 - 192.168.1.255
#
# Cada subred: red, 62 hosts, broadcast
# Ej: Subred 0
# Red: 192.168.1.0
# Primer host: 192.168.1.1
# Ultimo host: 192.168.1.62
# Broadcast: 192.168.1.63
