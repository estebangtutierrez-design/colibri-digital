CONTENIDO DEL CAPÍTULO
===============================================================================

1. ETHERNET

Estándar IEEE 802.3. CSMA/CD (Carrier Sense Multiple Access with Collision

Detection). Velocidades: 10Mbps, 100Mbps, 1Gbps, 10Gbps, 40Gbps, 100Gbps.



2. TRAMA ETHERNET

Preamble (7 bytes), SFD (1), MAC destino (6), MAC origen (6), EtherType

(2), Payload (46-1500), FCS (4). Jumbo frames: hasta 9000 bytes payload.



3. DUPLEX Y VELOCIDAD

Half-duplex: solo una dirección a la vez. Full-duplex: ambas direcciones

simultáneamente (sin colisiones). Auto-negociación de velocidad y duplex.



4. ENLACES AGREGADOS (LACP)

IEEE 802.3ad. Agrupa múltiples enlaces físicos en uno lógico. Balanceo

de carga y redundancia. LACP (Link Aggregation Control Protocol).

EJEMPLO DE CÓDIGO
----------------------------------------

# Agregacion de enlaces (bonding en Linux)
# /etc/netplan/01-bond.yaml
network:
  version: 2
  bonds:
    bond0:
      interfaces: [eth0, eth1]
      parameters:
        mode: 802.3ad
        lacp-rate: fast
        mii-monitor-interval: 100
  vlans:
    bond0.100:
      id: 100
      link: bond0
      addresses: [192.168.100.1/24]

# Ver estado
$ cat /proc/net/bonding/bond0
$ ip link show bond0
