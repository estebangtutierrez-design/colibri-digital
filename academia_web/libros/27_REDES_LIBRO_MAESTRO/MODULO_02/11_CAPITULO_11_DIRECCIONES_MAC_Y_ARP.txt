1. DIRECCIÓN MAC

Media Access Control. Identificador único de 48 bits (6 bytes).

Formato: 00:1A:2B:3C:4D:5E. Primeros 3 bytes: OUI (fabricante).

Últimos 3 bytes: número de serie. Quemada en ROM de NIC.



2. ARP (ADDRESS RESOLUTION PROTOCOL)

Protocolo de capa 2/3. Traduce IP a MAC. Broadcast: ¿quién tiene IP X?

Respuesta: yo tengo IP X, mi MAC es Y. Cache ARP: ip neigh show.



3. TABLA ARP

arp -a lista tabla. ip neigh show equivalente moderno. Entradas

dinámicas (timeout) y estáticas (permanentes). Gratuitous ARP para

actualizar vecinos. ARP spoofing: ataque de seguridad.



4. INTERFAZ DE RED

ip link show interfaces. eth0, wlan0, lo (loopback). Flags: UP, BROADCAST,

RUNNING, MULTICAST. MAC address, MTU (1500 default).

EJEMPLO DE CÓDIGO
----------------------------------------

$ ip link show
$ ip neigh show
$ arp -a
$ ip addr show
# MAC address: 00:1a:2b:3c:4d:5e
# Broadcast: ff:ff:ff:ff:ff:ff
# Tabla ARP:
# 192.168.1.1 dev eth0 lladdr 00:11:22:33:44:55 REACHABLE
# 192.168.1.10 dev eth0 lladdr aa:bb:cc:dd:ee:ff STALE
