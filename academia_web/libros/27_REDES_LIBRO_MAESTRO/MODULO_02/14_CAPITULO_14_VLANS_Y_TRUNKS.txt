CONTENIDO DEL CAPÍTULO
===============================================================================

1. VLAN (VIRTUAL LAN)

Segmenta redes de capa 2 sin cambiar topología física. IEEE 802.1Q.

Múltiples redes lógicas en mismo switch. Reduce broadcast domain.

Mejora seguridad y gestión.



2. TIPOS DE VLAN

VLAN por puerto: asignación estática. VLAN por MAC: dinámica. VLAN por

protocolo (IP/subred). Default VLAN (1). Native VLAN (tráfico sin tag).



3. TRUNKING (802.1Q)

Trunk transporta múltiples VLANs en un enlace. Etiqueta 802.1Q en trama

Ethernet. VLAN ID 12 bits (1-4094). DTP (Dynamic Trunking Protocol).



4. VLAN EN LINUX

ip link add link eth0 name eth0.100 type vlan id 100. ifconfig eth0.100.

/etc/network/interfaces o netplan para configuración persistente.

EJEMPLO DE CÓDIGO
----------------------------------------

# Configuracion VLAN en switch
vlan 100
 name Ventas
vlan 200
 name IT
interface gigabitethernet0/1
 switchport mode access
 switchport access vlan 100
interface gigabitethernet0/24
 switchport mode trunk
 switchport trunk allowed vlan 100,200

# VLAN en Linux
ip link add link eth0 name eth0.100 type vlan id 100
ip addr add 192.168.100.1/24 dev eth0.100
ip link set eth0.100 up

# Ver VLANs
$ ip -d link show eth0.100
$ bridge vlan show
