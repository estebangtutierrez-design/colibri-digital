CONTENIDO DEL CAPÍTULO
===============================================================================

1. IEEE 802.11

Estándar WiFi. Bandas: 2.4GHz (alcance) y 5GHz (velocidad). 6GHz (WiFi 6E).

Versiones: a, b, g, n (WiFi 4), ac (WiFi 5), ax (WiFi 6), be (WiFi 7).



2. MODOS DE OPERACIÓN

Infrastructure: AP + clientes. Ad-hoc: peer-to-peer. Mesh: múltiples APs

interconectados. Monitor: captura de paquetes (sniffing).



3. SEGURIDAD WIFI

WEP: obsoleto, inseguro. WPA: TKIP, mejor. WPA2: AES, estándar actual.

WPA3: SAE, más seguro. 802.1X: autenticación empresarial (RADIUS).



4. CONFIGURACIÓN

SSID (nombre red), canal (1-11 2.4GHz, 36-165 5GHz), potencia, seguridad.

Roaming entre APs. QoS (WMM) para VoIP/video.

EJEMPLO DE CÓDIGO
----------------------------------------

# Escanear redes WiFi
$ iwlist wlan0 scan
$ nmcli dev wifi list

# Conectar con nmcli
$ nmcli dev wifi connect "MiRed" password "clave"

# Configurar AP con hostapd
# /etc/hostapd/hostapd.conf
interface=wlan0
driver=nl80211
ssid=MiAP
hw_mode=g
channel=6
auth_algs=1
wpa=2
wpa_passphrase=MiClaveSegura
wpa_key_mgmt=WPA-PSK
rsn_pairwise=CCMP

# Ver calidad señal
$ iwconfig wlan0
