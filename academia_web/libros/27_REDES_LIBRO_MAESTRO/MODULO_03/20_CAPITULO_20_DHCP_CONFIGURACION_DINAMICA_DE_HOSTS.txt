CONTENIDO DEL CAPÍTULO
===============================================================================

1. DHCP

Dynamic Host Configuration Protocol. Asigna IP automáticamente. Reduce

errores de configuración. Renovación periódica de leases.



2. PROCESO DORA

Discover (cliente busca servidor). Offer (servidor ofrece IP). Request

(cliente solicita IP). Acknowledge (servidor confirma).



3. CONFIGURACIÓN DHCP

Rango de direcciones, máscara, gateway, DNS, lease time (recomendado 24h).

Reservas: MAC → IP fija. Exclusiones: IPs fuera del rango.



4. DHCP EN LINUX

Servidor: isc-dhcp-server. Config /etc/dhcp/dhcpd.conf. Cliente:

dhclient o dhcpcd. Ver lease: /var/lib/dhcp/dhclient.leases.

EJEMPLO DE CÓDIGO
----------------------------------------

# /etc/dhcp/dhcpd.conf
subnet 192.168.1.0 netmask 255.255.255.0 {
  range 192.168.1.100 192.168.1.200;
  option routers 192.168.1.1;
  option domain-name-servers 8.8.8.8, 1.1.1.1;
  option domain-name "local.network";
  default-lease-time 86400;
  max-lease-time 172800;
}

# Reserva por MAC
host servidor {
  hardware ethernet 00:11:22:33:44:55;
  fixed-address 192.168.1.10;
}

# Ver leases
$ cat /var/lib/dhcp/dhcpd.leases
