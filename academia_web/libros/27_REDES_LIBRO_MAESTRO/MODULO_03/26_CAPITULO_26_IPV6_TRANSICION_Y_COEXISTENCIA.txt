CONTENIDO DEL CAPÍTULO
===============================================================================

1. IPv6

Direcciones de 128 bits. 340 undecillones de direcciones. Cabecera

simplificada (40 bytes fijos). No NAT. Autoconfiguración (SLAAC).

IPsec nativo. Multicast en vez de broadcast.



2. NOTACIÓN IPv6

2001:0db8:85a3:0000:0000:8a2e:0370:7334. :: abrevia ceros consecutivos.

2001:db8::1. ::1 loopback. fe80::/10 link-local. fc00::/7 unique-local.



3. TRANSICIÓN

Dual Stack: IPv4 e IPv6 simultáneos. Tunneling: 6in4, 6to4, Teredo.

NAT64: traducir IPv6 a IPv4. DNS64: sintetizar registros AAAA.



4. CONFIGURACIÓN

Network Manager, netplan o /etc/network/interfaces. SLAAC o DHCPv6.

Prefix Delegation para ISPs. Firewall para IPv6 (ip6tables).

EJEMPLO DE CÓDIGO
----------------------------------------

# Verificar IPv6
$ ip -6 addr show
$ ip -6 route show
$ ping6 google.com

# netplan config
network:
  version: 2
  ethernets:
    eth0:
      dhcp4: yes
      dhcp6: yes
      accept-ra: yes

# iptables para IPv6
ip6tables -P INPUT DROP
ip6tables -A INPUT -m state --state ESTABLISHED,RELATED -j ACCEPT
ip6tables -A INPUT -p ipv6-icmp -j ACCEPT
ip6tables -A INPUT -i lo -j ACCEPT

# Test IPv6
$ curl -6 http://ipv6.google.com
