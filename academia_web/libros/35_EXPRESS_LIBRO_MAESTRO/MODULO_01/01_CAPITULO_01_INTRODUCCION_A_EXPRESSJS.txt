CONTENIDO DEL CAPÍTULO
===============================================================================

Express: framework web minimalista para Node.js. Creado por TJ Holowaychuk.

npm install express. Sin opiniones, flexible. Middleware pipeline.

Rutas, middleware, templates, estáticos.

EJEMPLO DE CÓDIGO
----------------------------------------

const express = require("express")
const app = express()
app.get("/", (req, res) => res.send("Hello"))
app.listen(3000)
