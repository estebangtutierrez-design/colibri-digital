CAPITULO 1: INTRODUCCIÓN A C
================================================================================

¿Qué es C?
----------
C es un lenguaje de programación de propósito general, creado por Dennis Ritchie
en 1972 en los laboratorios Bell. Es un lenguaje compilado, de bajo nivel, que
proporciona acceso directo a la memoria y al hardware. Es la base de muchos
sistemas operativos (Linux, Windows) y lenguajes (C++, C#, Java, Python).

Características:
- Eficiente y rápido.
- Punteros y aritmética de punteros.
- Gestión manual de memoria.
- Lenguaje de sistemas.

Versión estándar: C18 (ISO/IEC 9899:2018).

CAPITULO 3: SINTAXIS BÁSICA
================================================================================

Comentarios:
------------
// Comentario de línea (C99 en adelante)
/* Comentario de bloque */

Tipos de datos primitivos:
--------------------------
char      -> 1 byte (-128 a 127)
int       -> 2 o 4 bytes (depende de la plataforma)
float     -> 4 bytes (precisión simple)
double    -> 8 bytes (precisión doble)
void      -> sin valor

Modificadores: signed, unsigned, short, long.

Variables:
----------
int edad = 30;
float altura = 1.75;
char inicial = 'E';
double pi = 3.1415926535;

Constantes (const o #define):
------------------------------
const int DIAS = 7;
#define PI 3.1416

Operadores:
-----------
Aritméticos: + - * / % (módulo)
Asignación: = += -= *= /=
Comparación: == != > < >= <=
Lógicos: && || !

Incremento/Decremento: ++ --

CAPITULO 4: ESTRUCTURAS DE CONTROL
================================================================================

If:
---
if (condicion) {
    // código
} else if (otra) {
    // código
} else {
    // código
}

Switch:
-------
switch (variable) {
    case 1: printf("Uno"); break;
    case 2: printf("Dos"); break;
    default: printf("Otro");
}

Bucles:
-------
for (int i = 0; i < 10; i++) {
    printf("%d ", i);
}

while (condicion) { ... }

do { ... } while (condicion);

Break y continue: igual que en otros lenguajes.

CAPITULO 5: FUNCIONES Y PASO DE PARÁMETROS
================================================================================

Declaración y definición:
-------------------------
int sumar(int a, int b); // Prototipo

int main() { ... }

int sumar(int a, int b) {
    return a + b;
}

Paso por valor (C es por valor siempre):
----------------------------------------
void cambiar(int x) {
    x = 10; // No afecta al original
}

Para modificar el original, usamos punteros (ver capítulo 7).

Parámetros por defecto: C no tiene. Se usan macros o funciones separadas.

Arreglos (arrays):

CAPITULO 7: PUNTEROS Y GESTIÓN DE MEMORIA
================================================================================

Un puntero almacena una dirección de memoria.

Declaración:
------------
int *ptr;        // puntero a int
char *cptr;      // puntero a char

Operadores:
-----------
& (dirección de): int x = 5; int *p = &x;
* (indirección): *p = 10; // cambia x

Punteros y arreglos:
--------------------
int arr[5] = {1,2,3,4,5};
int *p = arr;      // p apunta al primer elemento
*(p + 1) == arr[1]; // verdadero

Aritmética de punteros: p++, p--, p + n.

Puntero a función:
------------------
void (*funcPtr)(int) = &miFuncion;

CAPITULO 8: ESTRUCTURAS (struct) Y UNIONES
================================================================================

Definir estructura:
-------------------
struct Persona {
    char nombre[50];
    int edad;
    float altura;
};

Crear variable y acceder:
-------------------------
struct Persona p1;
strcpy(p1.nombre, "Ana");
p1.edad = 25;

struct Persona p2 = {"Luis", 30, 1.80};

Punteros a estructuras:
-----------------------
struct Persona *ptr = &p1;
ptr->edad = 26;  // Equivalente a (*ptr).edad

Typedef (alias):
----------------
typedef struct Persona Persona;
Persona p3;

Uniones (comparten memoria):
----------------------------
union Dato {
    int i;
    float f;
    char c;
};
union Dato d;
d.i = 10;  // ocupa 4 bytes

CAPITULO 9: ENTRADA/SALIDA ESTÁNDAR
================================================================================

printf (salida formateada):
---------------------------
printf("Hola %s, tienes %d años\n", nombre, edad);
%c: char, %d: int, %f: float, %s: string, %p: puntero, %x: hexadecimal.

scanf (entrada formateada):
---------------------------
int edad;
printf("Edad: ");
scanf("%d", &edad);  // ¡Ojo! usa & para variables

char nombre[50];
scanf("%s", nombre); // sin & para arreglos

getchar() y putchar() para caracteres:
--------------------------------------
char c = getchar();
putchar(c);

CAPITULO 10: MANEJO DE ARCHIVOS (FILE)
================================================================================

Abrir archivo:
--------------
FILE *fp = fopen("archivo.txt", "r"); // r, w, a, r+, w+
if (fp == NULL) { printf("Error al abrir"); }

Leer archivo:
-------------
char buffer[100];
while (fgets(buffer, 100, fp) != NULL) {
    printf("%s", buffer);
}

Escribir archivo:
-----------------
fprintf(fp, "Escribiendo: %d\n", 123);
fputs("Línea de texto", fp);

Cerrar archivo:
---------------
fclose(fp);

Leer/Escribir binario:
----------------------
fread(buffer, sizeof(int), 10, fp);
fwrite(buffer, sizeof(int), 10, fp);

CAPITULO 11: PREPROCESADOR Y MACROS
================================================================================

Directivas comunes:
-------------------
#include <stdio.h>        // incluir biblioteca
#define PI 3.1416          // macro constante
#define CUADRADO(x) ((x)*(x))  // macro con parámetros
#ifdef DEBUG               // compilación condicional
    printf("Debug\n");
#endif
#undef PI                  // eliminar macro

Macros vs funciones: las macros son reemplazo de texto (más rápidas, sin chequeo de tipos).

CAPITULO 13: ASIGNACIÓN DINÁMICA DE MEMORIA
================================================================================

Biblioteca stdlib.h:
--------------------
#include <stdlib.h>

malloc: reserva memoria sin inicializar.
calloc: reserva e inicializa a cero.
realloc: cambia el tamaño de un bloque.
free: libera memoria.

Ejemplo:
--------
int *arr = (int*)malloc(5 * sizeof(int));
if (arr == NULL) { printf("Error de memoria"); }
arr[0] = 10;
arr = (int*)realloc(arr, 10 * sizeof(int));
free(arr);

¡Importante! C no tiene recolector de basura, debes liberar la memoria manualmente.

CAPITULO 14: BUENAS PRÁCTICAS EN C
================================================================================

1. Siempre inicializa variables.
2. Verifica punteros NULL antes de usarlos.
3. Libera memoria dinámica (free).
4. Usa fgets en lugar de scanf para strings.
5. Usa prototipos de funciones antes de main.
6. Comenta el código complejo.
7. Evita variables globales innecesarias.
8. Usa const para parámetros de solo lectura.
9. Siempre cierra archivos con fclose.
10. Compila con advertencias: gcc -Wall -Wextra programa.c.
