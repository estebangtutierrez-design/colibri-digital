--------------------------------------------------------------------------------

  ▸ BIENVENIDA

  En la mayoría de los lenguajes modernos, los strings son un tipo de
  datos nativo con operadores y métodos propios. En C, un string es
  simplemente un array de caracteres (char) que termina con un carácter
  especial: el null terminator ('\0').

  Esta simplicidad tiene ventajas y desventajas. Por un lado, entiendes
  exactamente cómo se almacenan y manipulan los strings en memoria.
  Por otro lado, operaciones como copiar, comparar o concatenar requieren
  funciones específicas (que veremos en el capítulo 35).

  En este capítulo aprenderás a declarar, inicializar y trabajar con
  strings. Entenderás el papel crucial del '\0', las diferencias entre
  strings literales y arrays de caracteres, y las funciones básicas
  para leer y escribir strings.

  Bienvenido al mundo del texto en C.

  ▸ CONTENIDO

  --- 1. ¿QUÉ ES UN STRING? ---

  En C, un string es un array de caracteres (char) que termina con el
  carácter nulo '\0' (cuyo valor ASCII es 0).

  El '\0' actúa como marcador de final: las funciones de la biblioteca
  estándar recorren el string hasta encontrar este carácter.

  Representación gráfica del string "Hola":

    Índice:  0     1     2     3     4
            +-----+-----+-----+-----+------+
            | 'H' | 'o' | 'l' | 'a' | '\0' |
            +-----+-----+-----+-----+------+
    ASCII:  72    111   108   97    0

  El array tiene 5 elementos (4 letras + 1 terminador), aunque el
  string "tiene" 4 caracteres.

  --- 2. DECLARACIÓN E INICIALIZACIÓN DE STRINGS ---

  Hay varias formas de declarar e inicializar strings:

  a) Como array de char con tamaño explícito:

    char saludo[6] = {'H', 'o', 'l', 'a', '\0'};

  b) Como array de char con inicialización por string literal:

    char saludo[6] = "Hola";

  c) Sin tamaño (el compilador cuenta):

    char saludo[] = "Hola";  // tamaño 5 (4 + 1)

  d) Como puntero a char (string literal):

    char *saludo = "Hola";

  e) Inicialización parcial (el resto se rellena con '\0'):

    char saludo[10] = "Hola";
    // saludo = {'H','o','l','a','\0','\0','\0','\0','\0','\0'}

  f) Carácter por carácter:

    char saludo[6];
    saludo[0] = 'H';
    saludo[1] = 'o';
    saludo[2] = 'l';
    saludo[3] = 'a';
    saludo[4] = '\0';

  --- 3. CHAR s[] vs CHAR *s ---

  Esta es una de las diferencias más importantes en C:

  char s[] = "Hola";
    • Crea un array de 5 chars en la pila (stack).
    • El contenido se puede modificar.
    • s es un puntero constante (no se puede reasignar).
    • sizeof(s) = 5.

  char *s = "Hola";
    • Crea un puntero a un literal de string.
    • El literal está en una región de solo lectura.
    • Intentar modificar *s es comportamiento indefinido.
    • s se puede reasignar para apuntar a otra dirección.
    • sizeof(s) = 8 (tamaño de puntero).

  Ejemplo:

    char arr[] = "Hola";
    arr[0] = 'h';  // OK → "hola"

    char *ptr = "Hola";
    ptr[0] = 'h';  // PELIGRO: undefined behavior (segmentación fault)

  Siempre usa const char * para punteros a literales:

    const char *s = "Hola";  // seguro

  --- 4. EL NULL TERMINATOR ---

  El carácter '\0' (null terminator) es fundamental. Sin él, las
  funciones de string no sabrían dónde termina la cadena.

  Importante:
    • '\0' no es lo mismo que '0' (el carácter cero, ASCII 48).
    • '\0' tiene valor ASCII 0.
    • NULL es un puntero (void*)0, diferente de '\0'.
    • '0' es el carácter cero (48), diferente de '\0'.

  Un array de char sin '\0' NO es un string:

    char no_string[3] = {'A', 'B', 'C'};  // no tiene '\0'
    printf("%s", no_string);  // imprime "ABC" + basura hasta encontrar '\0'

  Siempre asegúrate de que haya espacio para el '\0':

    char nombre[5];  // solo 4 caracteres + '\0'
    // scanf("%s", nombre) con "Lucas" (5 letras) desbordaría

  --- 5. LONGITUD DE UN STRING ---

  La longitud de un string es el número de caracteres ANTES del '\0'.

  "Hola" → longitud 4
  "" → longitud 0 (el array tiene solo '\0')

  Calcular longitud manualmente:

    char *s = "Hola";
    int len = 0;
    while (s[len] != '\0') len++;
    // len = 4

  Con punteros:

    const char *p = s;
    while (*p) p++;
    int len = p - s;  // 4

  Pero normalmente usamos strlen() de <string.h>:

    #include <string.h>
    size_t len = strlen("Hola");  // 4

  --- 6. IMPRIMIR STRINGS ---

  Con printf usando %s:

    char nombre[] = "Ana";
    printf("%s\n", nombre);          // "Ana"
    printf("%.3s\n", "Hola Mundo");  // "Hol" (limita a 3 chars)
    printf("%10s\n", "Hola");        // "      Hola" (ancho mínimo 10)

  Con puts (añade salto de línea automático):

    puts("Hola Mundo");  // imprime "Hola Mundo\n"

  Con putchar (carácter por carácter):

    char *s = "Hola";
    while (*s) putchar(*s++);  // imprime "Hola"
    putchar('\n');

  Con fputs (sin salto de línea):

    fputs("Hola", stdout);  // imprime "Hola" sin \n

  --- 7. LEER STRINGS DEL USUARIO ---

  a) scanf con %s (PELIGROSO):

    char nombre[20];
    scanf("%s", nombre);  // no usa & porque nombre ya es puntero
    // Lee hasta espacio, tab o newline.

  Peligro: si el usuario escribe más de 19 caracteres, hay desbordamiento.

  Versión segura con ancho:

    scanf("%19s", nombre);  // máximo 19 caracteres + '\0'

  b) fgets (SEGURO — recomendado):

    char nombre[50];
    fgets(nombre, sizeof(nombre), stdin);
    // Lee hasta newline o hasta sizeof-1 caracteres
    // Incluye el newline en el string

  Para eliminar el newline:

    size_t len = strlen(nombre);
    if (len > 0 && nombre[len-1] == '\n')
        nombre[len-1] = '\0';

  c) getchar (un carácter a la vez):

    char buffer[100];
    int i = 0;
    char c;
    while ((c = getchar()) != '\n' && i < 99)
        buffer[i++] = c;
    buffer[i] = '\0';

  d) gets (NUNCA USAR — eliminado del estándar):

    char buffer[100];
    gets(buffer);  // sin límite de tamaño → buffer overflow seguro

  --- 8. ACCESO Y MODIFICACIÓN DE CARACTERES ---

  Los caracteres de un string se acceden como cualquier array:

    char s[] = "Hola";
    printf("%c\n", s[0]);  // 'H'
    printf("%c\n", s[1]);  // 'o'

    s[0] = 'h';  // ahora s = "hola"
    s[2] = 'L';  // ahora s = "hoLa"

  Recorrer un string:

    for (int i = 0; s[i] != '\0'; i++)
        printf("%c ", s[i]);

  O con puntero:

    for (char *p = s; *p; p++)
        printf("%c ", *p);

  --- 9. STRINGS VACÍOS Y NO INICIALIZADOS ---

  String vacío:

    char vacio[10] = "";    // todos los elementos son '\0'
    char vacio[10] = {0};   // todos a cero, mismo efecto

  String no inicializado (contiene basura):

    char basura[10];
    printf("%s", basura);  // imprime basura hasta encontrar '\0'

  Siempre inicializa los strings:

    char buffer[100] = "";  // seguro, buffer[0] = '\0'

  --- 10. TAMAÑO DEL ARRAY vs LONGITUD DEL STRING ---

  Es crucial distinguir:

  • Tamaño del array (capacity): espacio total reservado.
  • Longitud del string: número de caracteres hasta '\0'.

    char s[20] = "Hola";

    sizeof(s)                // 20 (tamaño del array)
    strlen(s)                // 4 (longitud del string)
    sizeof(s) / sizeof(s[0]) // 20 (elementos del array)

  El string puede ser más corto que el array, pero nunca más largo
  (sin contar el '\0').

  --- 11. EJEMPLOS COMPLETOS ---

  EJEMPLO 1: Contar vocales en un string

    #include <stdio.h>

    int main() {
        char texto[] = "Hola Mundo desde C";
        int vocales = 0;

        for (int i = 0; texto[i] != '\0'; i++) {
            char c = texto[i];
            if (c == 'a' || c == 'e' || c == 'i' ||
                c == 'o' || c == 'u' ||
                c == 'A' || c == 'E' || c == 'I' ||
                c == 'O' || c == 'U')
                vocales++;
        }

        printf("Vocales: %d\n", vocales);
        return 0;
    }

  EJEMPLO 2: Invertir un string

    #include <stdio.h>
    #include <string.h>

    int main() {
        char s[] = "Hola Mundo";
        int len = strlen(s);

        for (int i = 0; i < len / 2; i++) {
            char temp = s[i];
            s[i] = s[len - 1 - i];
            s[len - 1 - i] = temp;
        }

        printf("Invertido: %s\n", s);  // "odnuM aloH"
        return 0;
    }

  EJEMPLO 3: Palíndromo

    #include <stdio.h>
    #include <string.h>
    #include <ctype.h>

    int main() {
        char s[] = "Anita lava la tina";
        int i = 0, j = strlen(s) - 1;
        int es_palindromo = 1;

        while (i < j) {
            if (!isalpha(s[i])) { i++; continue; }
            if (!isalpha(s[j])) { j--; continue; }
            if (tolower(s[i]) != tolower(s[j])) {
                es_palindromo = 0;
                break;
            }
            i++;
            j--;
        }

        printf("%s: %s\n", s,
               es_palindromo ? "es palindromo" : "no es palindromo");
        return 0;
    }

  EJEMPLO 4: Pasar a mayúsculas

    #include <stdio.h>
    #include <ctype.h>

    int main() {
        char s[] = "Hola Mundo 123";

        for (int i = 0; s[i]; i++)
            s[i] = toupper(s[i]);

        printf("%s\n", s);  // "HOLA MUNDO 123"
        return 0;
    }

  EJEMPLO 5: Quitar espacios al inicio y final (trim)

    #include <stdio.h>
    #include <string.h>
    #include <ctype.h>

    void trim(char *s) {
        int inicio = 0, fin = strlen(s) - 1;

        while (isspace(s[inicio])) inicio++;
        while (fin > inicio && isspace(s[fin])) fin--;

        int i;
        for (i = 0; inicio <= fin; i++)
            s[i] = s[inicio++];
        s[i] = '\0';
    }

    int main() {
        char s[] = "   Hola Mundo   ";
        trim(s);
        printf("'%s'\n", s);  // 'Hola Mundo'
        return 0;
    }

  EJEMPLO 6: Contar palabras

    #include <stdio.h>
    #include <ctype.h>

    int main() {
        char s[] = "  Hola   Mundo desde   C  ";
        int palabras = 0;
        int en_palabra = 0;

        for (int i = 0; s[i]; i++) {
            if (isspace(s[i]))
                en_palabra = 0;
            else if (!en_palabra) {
                en_palabra = 1;
                palabras++;
            }
        }

        printf("Palabras: %d\n", palabras);  // 4
        return 0;
    }

  EJEMPLO 7: Reemplazar una letra

    #include <stdio.h>

    int main() {
        char s[] = "Hola Mundo";
        char buscar = 'o';
        char reemplazar = 'x';

        for (int i = 0; s[i]; i++)
            if (s[i] == buscar)
                s[i] = reemplazar;

        printf("%s\n", s);  // "Hxla Mundx"
        return 0;
    }

  EJEMPLO 8: Decimal a binario (string)

    #include <stdio.h>

    int main() {
        int num = 42;
        char binario[33] = "";
        int i = 0;

        if (num == 0) {
            binario[i++] = '0';
        } else {
            while (num > 0) {
                binario[i++] = (num % 2) + '0';
                num /= 2;
            }
        }
        binario[i] = '\0';

        // Invertir
        int len = strlen(binario);
        for (int j = 0; j < len / 2; j++) {
            char t = binario[j];
            binario[j] = binario[len - 1 - j];
            binario[len - 1 - j] = t;
        }

        printf("Binario: %s\n", binario);
        return 0;
    }

  EJEMPLO 9: Leer línea completa con fgets

    #include <stdio.h>
    #include <string.h>

    int main() {
        char linea[100];

        printf("Ingresa una linea: ");
        fgets(linea, sizeof(linea), stdin);

        size_t len = strlen(linea);
        if (len > 0 && linea[len-1] == '\n')
            linea[len-1] = '\0';

        printf("Leido: '%s' (longitud: %zu)\n", linea, strlen(linea));
        return 0;
    }

  EJEMPLO 10: Generar substring

    #include <stdio.h>
    #include <string.h>

    void substring(const char *s, int inicio, int fin, char *out) {
        int j = 0;
        for (int i = inicio; i <= fin && s[i] != '\0'; i++)
            out[j++] = s[i];
        out[j] = '\0';
    }

    int main() {
        char *texto = "Hola Mundo";
        char sub[20];
        substring(texto, 0, 3, sub);
        printf("Substring: '%s'\n", sub);  // "Hola"
        substring(texto, 5, 9, sub);
        printf("Substring: '%s'\n", sub);  // "Mundo"
        return 0;
  }

  --- 12. TABLA DE CARACTERES ESPECIALES ---

  Código  | Secuencia | Significado
  --------|-----------|---------------------
  '\0'    | null      | Fin de string
  '\n'    | newline   | Salto de línea
  '\t'    | tab       | Tabulación horizontal
  '\r'    | carriage  | Retorno de carro
  '\\'    | backslash | Barra invertida
  '\''    | single    | Comilla simple
  '\"'    | double    | Comilla doble
  '\a'    | alert     | Beep/sonido
  '\b'    | backspace | Retroceso

  --- 13. MATRICES DE CARACTERES vs STRINGS ---

  No toda matriz de caracteres es un string:

    char matrix[3][3] = {
        {'A', 'B', 'C'},
        {'D', 'E', 'F'},
        {'G', 'H', 'I'}
    };
    printf("%s", matrix[0]);  // PELIGRO: no hay '\0'

  Para strings en matrices 2D:

    char names[3][10] = {"Ana", "Luis", "Pedro"};
    // Cada fila es un string independiente con su '\0'

  --- 14. ERRORES COMUNES ---

  1. Olvidar el espacio para '\0':

     char s[4] = "Hola";  // ERROR: no cabe '\0'

  2. Confundir sizeof con strlen:

     char s[20] = "Hola";
     for (int i = 0; i < sizeof(s); i++) ...  // recorre 20, no 4

  3. No terminar un string con '\0':

     char s[5] = {'H','o','l','a'};  // falta '\0'
     printf("%s", s);  // undefined behavior

  4. Intentar modificar un string literal:

     char *s = "Hola";
     s[0] = 'h';  // undefined behavior

  5. Usar gets:

     gets(buffer);  // buffer overflow asegurado

  6. Leer con scanf sin especificar ancho:

     scanf("%s", buffer);  // sin límite → overflow

  7. No eliminar el '\n' de fgets:

     fgets(s, 100, stdin);
     // s puede contener '\n' al final

  8. Confundir 'a' con "a":

     'a' → carácter (1 byte, valor 97)
     "a" → string (2 bytes: 'a' + '\0')

  ▸ RESUMEN DEL CAPÍTULO

  • Un string en C es un array de char terminado en '\0'.
  • El null terminator ('\0') tiene valor ASCII 0, diferente de '0'.
  • char s[] = "..." crea un array modificable.
  • char *s = "..." crea un puntero a un literal inmutable.
  • Usa fgets para leer strings de forma segura.
  • Nunca uses gets (es peligroso).
  • Siempre asegura espacio para el '\0'.
  • strlen da la longitud sin contar '\0'.
  • sizeof da el tamaño total del array.
  • Los strings se recorren hasta encontrar '\0'.

  ▸ PREGUNTAS FRECUENTES (FAQ)

  P: ¿Por qué C no tiene un tipo string nativo?
  R: Por razones históricas y de filosofía. C fue diseñado para ser
     minimalista y cercano al hardware. Un string como array de char
     refleja exactamente cómo se almacena en memoria.

  P: ¿Cuál es la diferencia entre '\0' y NULL?
  R: '\0' es un carácter (valor 0), se usa para terminar strings.
     NULL es un puntero nulo ((void*)0), se usa para indicar que un
     puntero no apunta a nada válido.

  P: ¿Qué pasa si olvido el '\0'?
  R: Las funciones como printf("%s"), strlen, etc. seguirán leyendo
     memoria hasta encontrar un '\0' accidental. Esto es undefined
     behavior y puede causar fallos o vulnerabilidades.

  P: ¿fgets incluye el salto de línea?
  R: Sí, fgets almacena el '\n' si hay espacio. Es responsabilidad del
     programador eliminarlo si no se desea.

  P: ¿Puedo usar scanf para leer strings con espacios?
  R: No, %s de scanf lee hasta el primer espacio. Para leer líneas
     completas usa fgets o scanf con %[^\n] (aunque fgets es mejor).

  P: ¿Cuál es el límite de tamaño de un string?
  R: El límite teórico es el tamaño del array. En la práctica, el
     límite es la memoria disponible en la pila para arrays locales,
     o la memoria dinámica si usas malloc.

  P: ¿Cómo se compara si dos strings son iguales?
  R: NO se usa == (eso compara direcciones). Se usa strcmp() de
     <string.h>.

  P: ¿Puedo tener un string con caracteres nulos internos?
  R: Técnicamente no, porque '\0' marca el final. Si necesitas datos
     binarios con ceros, usa arrays de unsigned char y maneja la
     longitud por separado.
