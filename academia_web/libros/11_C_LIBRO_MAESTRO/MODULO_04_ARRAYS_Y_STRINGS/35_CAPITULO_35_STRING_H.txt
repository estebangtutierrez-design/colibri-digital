  ▸ BIENVENIDA

  En el capítulo anterior aprendiste que los strings son arrays de char
  terminados en '\0'. También viste que operaciones como copiar, comparar
  o concatenar requieren trabajo manual si lo haces desde cero.

  Afortunadamente, la biblioteca estándar de C incluye <string.h>, un
  conjunto de funciones que realizan las operaciones más comunes sobre
  strings de forma eficiente y probada.

  En este capítulo exploraremos las funciones principales: strlen, strcpy,
  strcat, strcmp, strchr, strstr, y muchas más. Aprenderás a usarlas
  correctamente, conocerás sus peligros y descubrirás alternativas más
  seguras cuando existan.

  Domina string.h y escribirás código más limpio, seguro y eficiente.

  ▸ CONTENIDO

  --- 1. ¿QUÉ ES string.h? ---

  <string.h> es un header de la biblioteca estándar de C que declara
  funciones para manipular arrays de caracteres (strings) y memoria
  en bruto.

  Para usarla:

    #include <string.h>

  Las funciones se clasifican en:
  • Funciones de longitud: strlen
  • Funciones de copia: strcpy, strncpy, memcpy, memmove
  • Funciones de concatenación: strcat, strncat
  • Funciones de comparación: strcmp, strncmp, memcmp
  • Funciones de búsqueda: strchr, strrchr, strstr, strpbrk, strspn
  • Funciones de tokenización: strtok
  • Funciones de inicialización: memset
  • Funciones de error: strerror

  --- 2. strlen — LONGITUD DE STRING ---

  Prototipo:

    size_t strlen(const char *s);

  Devuelve el número de caracteres en s antes del '\0' (sin incluirlo).

    char *s = "Hola Mundo";
    size_t len = strlen(s);  // 10

  Implementación propia:

    size_t mi_strlen(const char *s) {
        const char *p = s;
        while (*p) p++;
        return p - s;
    }

  Nota: strlen recorre el string completo, así que es O(n). No lo
  llames innecesariamente en bucles.

    // Ineficiente:
    for (size_t i = 0; i < strlen(s); i++)  // O(n²)
        ...

    // Eficiente:
    size_t len = strlen(s);
    for (size_t i = 0; i < len; i++)        // O(n)
        ...

  --- 3. strcpy — COPIAR STRING ---

  Prototipo:

    char *strcpy(char *dest, const char *src);

  Copia el string src en dest (incluyendo '\0'). dest debe tener
  suficiente espacio.

    char src[] = "Hola";
    char dest[10];
    strcpy(dest, src);  // dest ahora contiene "Hola"

  Devuelve dest (útil para encadenamiento).

  PELIGRO: No verifica que dest tenga espacio suficiente.

    char dest[5];
    strcpy(dest, "Hola Mundo");  // ¡desbordamiento!

  Versión propia:

    char *mi_strcpy(char *dest, const char *src) {
        char *p = dest;
        while ((*p++ = *src++));
        return dest;
    }

  Alternativa segura: strncpy.

  --- 4. strncpy — COPIAR CON LÍMITE ---

  Prototipo:

    char *strncpy(char *dest, const char *src, size_t n);

  Copia hasta n caracteres de src a dest. Si src es más corto que n,
  rellena el resto con '\0'. Si src es más largo, NO añade '\0'.

    char dest[10];
    strncpy(dest, "Hola Mundo", sizeof(dest) - 1);
    dest[sizeof(dest) - 1] = '\0';  // siempre asegurar terminación

  IMPORTANTE: strncpy NO garantiza que dest termine en '\0' si src
  tiene n o más caracteres. Siempre debes añadirlo manualmente.

  Mi implementación:

    char *mi_strncpy(char *dest, const char *src, size_t n) {
        size_t i;
        for (i = 0; i < n && src[i]; i++)
            dest[i] = src[i];
        while (i < n)
            dest[i++] = '\0';
        return dest;
    }

  --- 5. strcat — CONCATENAR STRINGS ---

  Prototipo:

    char *strcat(char *dest, const char *src);

  Agrega src al final de dest. El '\0' original de dest se sobrescribe
  y se añade un nuevo '\0' al final del resultado.

    char dest[20] = "Hola ";
    strcat(dest, "Mundo");  // dest = "Hola Mundo"

  PELIGRO: No verifica que dest tenga espacio suficiente.

  Versión propia:

    char *mi_strcat(char *dest, const char *src) {
        char *p = dest;
        while (*p) p++;
        while ((*p++ = *src++));
        return dest;
  }

  --- 6. strncat — CONCATENAR CON LÍMITE ---

  Prototipo:

    char *strncat(char *dest, const char *src, size_t n);

  Agrega hasta n caracteres de src al final de dest. Siempre añade '\0'.

    char dest[20] = "Hola ";
    strncat(dest, "Mundo", 3);  // dest = "Hola Mun"

  A diferencia de strncpy, strncat SÍ garantiza la terminación nula.

  --- 7. strcmp — COMPARAR STRINGS ---

  Prototipo:

    int strcmp(const char *s1, const char *s2);

  Compara s1 y s2 carácter por carácter (lexicográficamente).
  Devuelve:
  • 0 si son iguales.
  • <0 si s1 < s2 (en orden lexicográfico).
  • >0 si s1 > s2.

    strcmp("abc", "abc");  // 0
    strcmp("abc", "abd");  // <0 (porque 'c' < 'd')
    strcmp("abd", "abc");  // >0
    strcmp("a", "ab");     // <0 (porque '\0' < 'b')

  Normalmente se usa así:

    if (strcmp(s1, s2) == 0)  // iguales
    if (strcmp(s1, s2) != 0)  // diferentes

  Versión propia:

    int mi_strcmp(const char *s1, const char *s2) {
        while (*s1 && *s1 == *s2) {
            s1++;
            s2++;
        }
        return *(unsigned char*)s1 - *(unsigned char*)s2;
    }

  --- 8. strncmp — COMPARAR CON LÍMITE ---

  Prototipo:

    int strncmp(const char *s1, const char *s2, size_t n);

  Compara hasta n caracteres de s1 y s2.

    strncmp("abcdef", "abcxyz", 3);  // 0 (los primeros 3 son iguales)
    strncmp("abcdef", "abcxyz", 4);  // <0 ('d' < 'x')

  Útil para comparar prefijos.

  --- 9. strchr — BUSCAR CARÁCTER ---

  Prototipo:

    char *strchr(const char *s, int c);

  Busca la primera ocurrencia del carácter c (convertido a char) en s.
  Devuelve un puntero a esa posición, o NULL si no se encuentra.

    char *s = "Hola Mundo";
    char *p = strchr(s, 'M');
    if (p) printf("Encontrado en posicion %td\n", p - s);  // 5

  Para buscar la última ocurrencia: strrchr.

    char *ultimo = strrchr(s, 'o');  // apunta al 'o' de "Mundo"

  --- 10. strstr — BUSCAR SUBSTRING ---

  Prototipo:

    char *strstr(const char *haystack, const char *needle);

  Busca la primera ocurrencia del substring needle en haystack.
  Devuelve puntero al inicio de la ocurrencia, o NULL.

    char *s = "Hola Mundo desde C";
    char *p = strstr(s, "Mundo");
    if (p) printf("'Mundo' comienza en posicion %td\n", p - s);  // 5

  --- 11. strtok — TOKENIZAR STRING ---

  Prototipo:

    char *strtok(char *s, const char *delim);

  Divide s en tokens (substrings) separados por los caracteres en delim.
  La primera llamada recibe el string; las siguientes reciben NULL.

    char s[] = "Hola,Mundo,desde,C";
    char *token = strtok(s, ",");
    while (token) {
        printf("Token: %s\n", token);
        token = strtok(NULL, ",");
    }
    // Imprime: Hola, Mundo, desde, C

  IMPORTANTE: strtok MODIFICA el string original (reemplaza delimitadores
  por '\0'). No es reentrante (usa estático interno). Usa strtok_r en
  sistemas POSIX para versión reentrante.

  --- 12. OTRAS FUNCIONES ÚTILES ---

  strpbrk:
    Busca el primer carácter en s que coincida con cualquiera de los
    caracteres en accept.

    char *p = strpbrk("Hola Mundo", "aeiou");  // 'o' (primera vocal)

  strspn / strcspn:
    strspn: longitud del segmento inicial de s que consiste solo de
    caracteres en accept.
    strcspn: longitud del segmento inicial que NO contiene caracteres
    de reject.

    size_t n = strspn("abc123", "abc");     // 3
    size_t n = strcspn("abc123", "123");    // 3

  --- 13. FUNCIONES DE MEMORIA ---

  Estas funciones trabajan con memoria en bruto (void*), no solo strings.

  memset: llena un bloque de memoria con un valor.

    int arr[10];
    memset(arr, 0, sizeof(arr));  // pone todo a 0

  memcpy: copia memoria (no debe solaparse).

    int src[5] = {1,2,3,4,5};
    int dest[5];
    memcpy(dest, src, sizeof(src));  // copia 20 bytes

  memmove: copia memoria (permite solapamiento).

    int arr[5] = {1,2,3,4,5};
    memmove(arr + 1, arr, 4 * sizeof(int));  // desplaza a la derecha

  memcmp: compara bloques de memoria.

    int a[3] = {1,2,3};
    int b[3] = {1,2,3};
    if (memcmp(a, b, sizeof(a)) == 0)  // iguales

  Función   | Descripción                    | Segura
  ----------|-------------------------------|--------
  strlen    | Longitud del string           | Sí

  strcat    | Concatenar                    | No
  strncat   | Concatenar con límite         | Sí
  strcmp    | Comparar strings              | Sí
  strncmp   | Comparar con límite           | Sí
  strchr    | Buscar carácter               | Sí
  strrchr   | Buscar último carácter        | Sí

  --- 15. EJEMPLOS COMPLETOS ---

  EJEMPLO 1: Longitudes y copias seguras

    #include <stdio.h>
    #include <string.h>

    int main() {
        char nombre[50];
        char copia[50];

        printf("Ingresa tu nombre: ");
        fgets(nombre, sizeof(nombre), stdin);
        nombre[strcspn(nombre, "\n")] = '\0';

        strncpy(copia, nombre, sizeof(copia) - 1);
        copia[sizeof(copia) - 1] = '\0';

        printf("Original: '%s' (%zu)\n", nombre, strlen(nombre));
        printf("Copia:    '%s' (%zu)\n", copia, strlen(copia));

        return 0;
    }

  EJEMPLO 2: Concatenación segura

    #include <stdio.h>
    #include <string.h>

    int main() {
        char ruta[256] = "/home/usuario/";
        char archivo[] = "documento.txt";
        size_t espacio = sizeof(ruta) - strlen(ruta) - 1;

        strncat(ruta, archivo, espacio);
        printf("Ruta completa: %s\n", ruta);

        return 0;
    }

  EJEMPLO 3: Comparación y ordenamiento

    #include <stdio.h>
    #include <string.h>

    int main() {
        char *palabras[] = {"perro", "gato", "abeja", "delfin", "cabra"};
        int n = 5;

        for (int i = 0; i < n - 1; i++) {
            for (int j = i + 1; j < n; j++) {
                if (strcmp(palabras[i], palabras[j]) > 0) {
                    char *temp = palabras[i];
                    palabras[i] = palabras[j];
                    palabras[j] = temp;
                }
            }
        }

        for (int i = 0; i < n; i++)
            printf("%s\n", palabras[i]);

        return 0;
    }

  EJEMPLO 4: Búsqueda con strchr y strstr

    #include <stdio.h>
    #include <string.h>

    int main() {
        char texto[] = "El gato y el perro juegan en el parque";

        char *p = strchr(texto, 'g');
        while (p) {
            printf("'g' encontrado en posicion %td\n", p - texto);
            p = strchr(p + 1, 'g');
        }

        char *sub = strstr(texto, "perro");
        if (sub)
            printf("'perro' comienza en posicion %td\n", sub - texto);

        return 0;
    }

  EJEMPLO 5: Tokenización con strtok

    #include <stdio.h>
    #include <string.h>

    int main() {
        char datos[] = "Juan,25,Madrid,Ingeniero";
        char *token = strtok(datos, ",");

        while (token) {
            printf("%s\n", token);
            token = strtok(NULL, ",");
        }

        return 0;
    }

  EJEMPLO 6: Contar palabras con strspn/strcspn

    #include <stdio.h>
    #include <string.h>

    int main() {
        char s[] = "  Hola   Mundo desde   C  ";
        char *p = s;
        int palabras = 0;

        while (*p) {
            p += strspn(p, " \t\n");
            if (!*p) break;
            palabras++;
            p += strcspn(p, " \t\n");
        }

        printf("Palabras: %d\n", palabras);
        return 0;
    }

  EJEMPLO 7: memset para inicializar arrays

    #include <stdio.h>
    #include <string.h>

    int main() {
        int arr[10];
        memset(arr, 0xFF, sizeof(arr));

        for (int i = 0; i < 10; i++)
            printf("%08X ", arr[i]);
        printf("\n");

        char buffer[100];
        memset(buffer, 0, sizeof(buffer));

        return 0;
    }

  EJEMPLO 8: memcpy vs memmove

    #include <stdio.h>
    #include <string.h>

    int main() {
        int arr[] = {1, 2, 3, 4, 5, 6, 7, 8};

        // Desplazar elementos a la derecha (sí se solapan)
        memmove(arr + 1, arr, 7 * sizeof(int));
        arr[0] = 0;

        for (int i = 0; i < 8; i++)
            printf("%d ", arr[i]);
        printf("\n");

        return 0;
    }

  EJEMPLO 9: Validar prefijo con strncmp

    #include <stdio.h>
    #include <string.h>

    int main() {
        char *archivos[] = {"foto.jpg", "doc.pdf", "imagen.png", "texto.txt"};
        int n = 4;

        for (int i = 0; i < n; i++) {
            if (strncmp(archivos[i], "foto", 4) == 0)
                printf("'%s' comienza con 'foto'\n", archivos[i]);
        }

        // Validar extensión
        for (int i = 0; i < n; i++) {
            char *ext = strrchr(archivos[i], '.');
            if (ext && strcmp(ext, ".jpg") == 0)
                printf("'%s' es un JPEG\n", archivos[i]);
        }

        return 0;
    }

  EJEMPLO 10: strerror para mensajes de error

    #include <stdio.h>
    #include <string.h>
    #include <errno.h>

    int main() {
        FILE *f = fopen("no_existe.txt", "r");
        if (!f) {
            printf("Error: %s\n", strerror(errno));
        }
        return 0;
    }

  --- 16. ERRORES COMUNES ---

  1. Desbordamiento con strcpy y strcat:

     char s[5];
     strcpy(s, "demasiado largo");  // overflow

  2. Olvidar el '\0' con strncpy:

     char s[5];
     strncpy(s, "hola mundo", 5);  // s NO termina en '\0'
     s[4] = '\0';  // necesario

  3. Confundir strcmp con una comparación booleana:

     if (strcmp(s1, s2))  // cierto si son DIFERENTES → confuso
     if (strcmp(s1, s2) == 0)  // cierto si son IGUALES → claro

  4. Usar strtok en un string constante:

     char *s = "hola,mundo";
     strtok(s, ",");  // segmentation fault (modifica string solo lectura)

  5. No entender que strtok modifica el original:

     char s[] = "a,b,c";
     strtok(s, ",");
     // s ahora es "a\0b\0c"

  6. Pasar NULL a strlen si la función falló:

     char *s = NULL;
     size_t n = strlen(s);  // segmentation fault

  7. Usar memcpy con regiones solapadas:

     memcpy(arr, arr + 1, 4 * sizeof(int));  // undefined behavior

  8. Olvidar #include <string.h> y que el compilador asuma
     prototipos incorrectos.

  • <string.h> proporciona funciones esenciales para manipular strings.
  • strlen: longitud (sin '\0').
  • strcpy/strncpy: copiar strings (strncpy es más segura).
  • strcat/strncat: concatenar (strncat es más segura).
  • strcmp/strncmp: comparar lexicográficamente.
  • strchr/strrchr: buscar caracteres.
  • strstr: buscar substrings.
  • strtok: dividir strings en tokens.
  • memset/memcpy/memmove/memcmp: manipulación de memoria.
  • Siempre verifica los límites de los buffers.
  • strncpy no garantiza terminación nula; strncat sí.

  ▸ PREGUNTAS FRECUENTES (FAQ)

  P: ¿Por qué se dice que strcpy es peligrosa?
  R: Porque no hay verificación de límites. Si el origen es más largo
     que el destino, se produce un buffer overflow, que puede corromper
     memoria y causar vulnerabilidades de seguridad.

  P: ¿Cuándo debo usar strncpy en lugar de strcpy?
  R: Siempre que no estés 100% seguro del tamaño del origen. strncpy
     limita la copia a n caracteres, pero recuerda que si el origen es
     más largo que n, NO añade '\0'.

  P: ¿strncat es segura?
  R: Sí, strncat siempre añade '\0' al final, incluso si se copian
     exactamente n caracteres.

  P: ¿Qué diferencia hay entre memcpy y memmove?
  R: memcpy no garantiza funcionamiento si origen y destino se solapan.
     memmove sí maneja solapamiento correctamente. Si no hay solapamiento,
     ambos funcionan igual.

  P: ¿Por qué strtok modifica el string original?
  R: strtok reemplaza los delimitadores por '\0' para crear tokens sin
     necesidad de copiar memoria. Si necesitas el string original, haz
     una copia antes de tokenizar.

  P: ¿Qué es strtok_r?
  R: Es una versión reentrante de strtok (disponible en POSIX). Usa un
     parámetro adicional char **saveptr para mantener el estado entre
     llamadas, lo que permite usarlo en hilos o bucles anidados.

  P: ¿Cuál es la diferencia entre strcmp y memcmp?
  R: strcmp compara strings (hasta '\0'), mientras que memcmp compara
     bloques de memoria de un tamaño específico. memcmp no se detiene
     en '\0'.

  P: ¿Cómo verifico si un string termina con cierta extensión?
  R: Usa strrchr para encontrar el último '.' y luego strcmp:
     char *ext = strrchr(nombre, '.');
     if (ext && strcmp(ext, ".txt") == 0) ...
