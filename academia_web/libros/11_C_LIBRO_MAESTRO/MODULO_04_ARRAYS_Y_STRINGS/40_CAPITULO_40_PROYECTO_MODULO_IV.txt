  ▸ CONTENIDO

  --- 1. DESCRIPCIÓN GENERAL ---

  El Sistema de Gestión de Inventario (SGI) permite a un usuario
  administrar los productos de una tienda. Las operaciones incluyen:

  • Alta, baja y modificación de productos.
  • Listado de productos ordenados por diferentes criterios.
  • Búsqueda de productos por nombre o código.
  • Control de stock mínimo.
  • Generación de reportes.
  • Carga y guardado de datos en archivos.

  --- 2. ESTRUCTURA DE DATOS ---

  Cada producto se representa con una estructura:

    typedef struct {
        int codigo;          // código único del producto
        char nombre[50];     // nombre del producto
        char categoria[30];  // categoría (Electrónica, Ropa, etc.)
        float precio;        // precio unitario
        int stock;           // cantidad en inventario
        int stock_minimo;    // stock mínimo para alertas
    } Producto;

  El inventario es un array de productos:

    #define MAX_PRODUCTOS 1000
    Producto inventario[MAX_PRODUCTOS];
    int num_productos = 0;

  --- 3. FUNCIONALIDADES DEL SISTEMA ---

  a) Gestión de productos:
    • Agregar producto: solicita todos los datos y lo añade al array.
    • Eliminar producto: por código o nombre.
    • Modificar producto: cambiar precio o stock.
    • Ver producto: mostrar datos de un producto específico.

  b) Listados y ordenamiento:
    • Listar todos los productos.
    • Ordenar por código (ascendente/descendente).
    • Ordenar por nombre (alfabéticamente).
    • Ordenar por precio (menor a mayor / mayor a menor).
    • Ordenar por stock (de mayor a menor cantidad).

  c) Búsqueda:
    • Búsqueda lineal por nombre (búsqueda parcial, sin distinguir
      mayúsculas/minúsculas).
    • Búsqueda binaria por código (el array debe estar ordenado por
      código).

  d) Reportes:
    • Productos con stock por debajo del mínimo (alerta de reorden).
    • Productos más caros (top 5).
    • Productos con mayor stock.
    • Valor total del inventario (suma de precio * stock por producto).
    • Promedio de precios.

  e) Archivos:
    • Cargar inventario desde archivo CSV al iniciar.
    • Guardar inventario en archivo CSV al salir.

  PASO 2: Función para agregar producto

    void agregar_producto(Producto inv[], int *n) {
        if (*n >= MAX_PRODUCTOS) {
            printf("Inventario lleno\n");
            return;
        }

        Producto p;
        printf("Codigo: "); scanf("%d", &p.codigo); getchar();
        printf("Nombre: "); fgets(p.nombre, 50, stdin);
        p.nombre[strcspn(p.nombre, "\n")] = '\0';
        printf("Categoria: "); fgets(p.categoria, 30, stdin);
        p.categoria[strcspn(p.categoria, "\n")] = '\0';
        printf("Precio: "); scanf("%f", &p.precio);
        printf("Stock: "); scanf("%d", &p.stock);
        printf("Stock minimo: "); scanf("%d", &p.stock_minimo);

        inv[*n] = p;
        (*n)++;
        printf("Producto agregado exitosamente\n");
    }

  PASO 3: Ordenamiento con qsort

    int cmp_codigo(const void *a, const void *b) {
        return ((Producto*)a)->codigo - ((Producto*)b)->codigo;
    }

    int cmp_nombre(const void *a, const void *b) {
        return strcmp(((Producto*)a)->nombre, ((Producto*)b)->nombre);
    }

    int cmp_precio(const void *a, const void *b) {
        float diff = ((Producto*)a)->precio - ((Producto*)b)->precio;
        if (diff < 0) return -1;
        if (diff > 0) return 1;
        return 0;
    }

    int cmp_stock_desc(const void *a, const void *b) {
        return ((Producto*)b)->stock - ((Producto*)a)->stock;
    }

    void listar_productos(Producto inv[], int n) {
        int criterio;
        printf("Ordenar por: 1=Codigo 2=Nombre 3=Precio 4=Stock: ");
        scanf("%d", &criterio);

        switch (criterio) {
            case 1: qsort(inv, n, sizeof(Producto), cmp_codigo); break;
            case 2: qsort(inv, n, sizeof(Producto), cmp_nombre); break;
            case 3: qsort(inv, n, sizeof(Producto), cmp_precio); break;
            case 4: qsort(inv, n, sizeof(Producto), cmp_stock_desc); break;
        }

        printf("\n%-6s %-20s %-15s %8s %6s %6s\n",
               "COD", "NOMBRE", "CATEGORIA", "PRECIO", "STOCK", "MIN");
        printf("------------------------------------------------------------\n");
        for (int i = 0; i < n; i++)
            printf("%-6d %-20s %-15s %8.2f %6d %6d\n",
                   inv[i].codigo, inv[i].nombre, inv[i].categoria,
                   inv[i].precio, inv[i].stock, inv[i].stock_minimo);
    }

  PASO 4: Búsqueda

    void buscar_producto(Producto inv[], int n) {
        int tipo;
        printf("Buscar por: 1=Nombre 2=Codigo: ");
        scanf("%d", &tipo);
        getchar();

        if (tipo == 1) {
            char nombre[50];
            printf("Nombre a buscar: ");
            fgets(nombre, 50, stdin);
            nombre[strcspn(nombre, "\n")] = '\0';

            // Búsqueda lineal (parcial, case-insensitive)
            for (int i = 0; i < n; i++) {
                char inv_lower[50], bus_lower[50];
                strcpy(inv_lower, inv[i].nombre);
                strcpy(bus_lower, nombre);
                for (int j = 0; inv_lower[j]; j++)
                    inv_lower[j] = tolower(inv_lower[j]);
                for (int j = 0; bus_lower[j]; j++)
                    bus_lower[j] = tolower(bus_lower[j]);

                if (strstr(inv_lower, bus_lower)) {
                    printf("%d: %s - $%.2f (stock: %d)\n",
                           inv[i].codigo, inv[i].nombre,
                           inv[i].precio, inv[i].stock);
                }
            }
        } else {
            int cod;
            printf("Codigo a buscar: ");
            scanf("%d", &cod);

            // Ordenar por código para búsqueda binaria
            qsort(inv, n, sizeof(Producto), cmp_codigo);

            Producto *res = bsearch(&cod, inv, n, sizeof(Producto),
                                    cmp_codigo);
            if (res)
                printf("Encontrado: %s - $%.2f (stock: %d)\n",
                       res->nombre, res->precio, res->stock);
            else
                printf("Producto con codigo %d no encontrado\n", cod);
        }
    }

  PASO 5: Reportes

    void reportes(Producto inv[], int n) {
        if (n == 0) { printf("Inventario vacio\n"); return; }

        // Productos con stock bajo
        printf("\n--- ALERTA DE STOCK BAJO ---\n");
        for (int i = 0; i < n; i++)
            if (inv[i].stock < inv[i].stock_minimo)
                printf("%s (cod %d): stock %d < minimo %d\n",
                       inv[i].nombre, inv[i].codigo,
                       inv[i].stock, inv[i].stock_minimo);

        // Valor total del inventario
        float total = 0;
        for (int i = 0; i < n; i++)
            total += inv[i].precio * inv[i].stock;
        printf("Valor total del inventario: $%.2f\n", total);
        printf("Promedio de precios: $%.2f\n", total / n);

        // Top 5 más caros
        qsort(inv, n, sizeof(Producto), cmp_precio_desc);
        printf("Top 5 productos mas caros:\n");
        int top = n < 5 ? n : 5;
        for (int i = 0; i < top; i++)
            printf("%d. %s - $%.2f\n", i+1,
                   inv[i].nombre, inv[i].precio);
    }

  PASO 6: Archivos

    void guardar_datos(Producto inv[], int n) {
        FILE *f = fopen("inventario.csv", "w");
        if (!f) { perror("Error"); return; }

        fprintf(f, "codigo,nombre,categoria,precio,stock,stock_minimo\n");
        for (int i = 0; i < n; i++)
            fprintf(f, "%d,%s,%s,%.2f,%d,%d\n",
                    inv[i].codigo, inv[i].nombre, inv[i].categoria,
                    inv[i].precio, inv[i].stock, inv[i].stock_minimo);
        fclose(f);
        printf("Datos guardados en inventario.csv\n");
    }

    void cargar_datos(Producto inv[], int *n) {
        FILE *f = fopen("inventario.csv", "r");
        if (!f) {
            printf("No se encontro archivo previo. Iniciando vacio.\n");
            return;
        }

        char linea[256];
        fgets(linea, sizeof(linea), f);  // saltar cabecera
        while (fgets(linea, sizeof(linea), f) && *n < MAX_PRODUCTOS) {
            linea[strcspn(linea, "\n")] = '\0';
            Producto p;
            sscanf(linea, "%d,%49[^,],%29[^,],%f,%d,%d",
                   &p.codigo, p.nombre, p.categoria,
                   &p.precio, &p.stock, &p.stock_minimo);
            inv[*n] = p;
            (*n)++;
        }
        fclose(f);
        printf("Cargados %d productos\n", *n);
    }

  --- 6. FUNCIONES ADICIONALES ---

  Para completar el proyecto, implementa también:

    void eliminar_producto(Producto inv[], int *n) {
        int cod;
        printf("Codigo del producto a eliminar: ");
        scanf("%d", &cod);

        for (int i = 0; i < *n; i++) {
            if (inv[i].codigo == cod) {
                for (int j = i; j < *n - 1; j++)
                    inv[j] = inv[j + 1];
                (*n)--;
                printf("Producto eliminado\n");
                return;
            }
        }
        printf("Producto no encontrado\n");
    }

    void modificar_producto(Producto inv[], int n) {
        int cod;
        printf("Codigo del producto a modificar: ");
        scanf("%d", &cod);

        for (int i = 0; i < n; i++) {
            if (inv[i].codigo == cod) {
                printf("Nuevo precio (actual: %.2f): ", inv[i].precio);
                scanf("%f", &inv[i].precio);
                printf("Nuevo stock (actual: %d): ", inv[i].stock);
                scanf("%d", &inv[i].stock);
                printf("Producto modificado\n");
                return;
            }
        }
        printf("Producto no encontrado\n");
    }

    int cmp_precio_desc(const void *a, const void *b) {

  ▸ PREGUNTAS FRECUENTES (FAQ)

  P: ¿Puedo usar memoria dinámica en lugar de un array estático?
  R: Sí, sería una mejora. Usa malloc y realloc para que el inventario
     pueda crecer dinámicamente en lugar de tener un límite fijo.

  P: ¿Cómo manejo la búsqueda case-insensitive?
  R: Convierte ambos strings (el del producto y el buscado) a
     minúsculas usando tolower() antes de comparar con strstr().

  P: ¿Qué hago si el archivo CSV tiene errores de formato?
  R: Valida cada línea al leer. Si una línea no tiene el formato
     esperado, salta a la siguiente y muestra un mensaje de error.

  P: ¿Cómo evito duplicados en los códigos?
  R: Antes de agregar, recorre el array para verificar que el código
     no exista ya. Si existe, muestra un error y pide otro código.

  P: ¿Puedo ordenar por categoría y luego por nombre?
  R: Sí, implementa una función de comparación que primero compare
     categoría, y si son iguales, compare nombre.

  P: ¿Cómo manejo la entrada de strings con espacios?
  R: Usa fgets en lugar de scanf("%s"). Con fgets puedes leer líneas
     completas incluyendo espacios. Luego elimina el '\n' final.

  P: ¿Debo usar punteros en lugar de índices?
  R: El proyecto usa índices por claridad, pero puedes implementar
     las funciones con aritmética de punteros como desafío adicional.

  P: ¿Cómo verifico que el programa no tenga memory leaks?
  R: Como usamos arrays estáticos (no malloc), no hay memory leaks
     que gestionar. Si usas memoria dinámica, usa valgrind.

  las estructuras de datos y los algoritmos. Los arrays (y sus
  abstracciones como vectores y listas) y los algoritmos de
  ordenamiento y búsqueda son los pilares sobre los que se construye
  todo lo demás.

  En el Módulo V exploraremos las funciones a un nivel más profundo,
  incluyendo recursividad, punteros a funciones y alcance de variables.
  También nos adentraremos en la memoria dinámica con malloc, calloc,
  realloc y free.

  El viaje continúa. Sigue programando.
