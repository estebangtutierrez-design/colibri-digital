1. INTRODUCCIÓN A LA COMPILACIÓN CONDICIONAL
===============================================================================

La compilación condicional permite incluir o excluir fragmentos de código
del proceso de compilación basándose en condiciones evaluadas por el
preprocesador. Esto ocurre ANTES de que el compilador vea el código.

Ejemplo conceptual:

```c
#include <stdio.h>

#define ACTIVAR_MODO_DEBUG

int main(void) {
    printf("Programa iniciado\n");
    
#ifdef ACTIVAR_MODO_DEBUG
    printf("[DEBUG] Modo depuración activo\n");
    printf("[DEBUG] Realizando diagnósticos...\n");
#endif
    
    printf("Programa finalizado\n");
    return 0;
}
```

Si definimos ACTIVAR_MODO_DEBUG, las líneas entre #ifdef y #endif se
compilan. Si no está definida, esas líneas simplemente desaparecen antes
de la compilación.

Flujo de decisión del preprocesador:

```c
#include <stdio.h>

#define PLATAFORMA "Linux"

int main(void) {
    printf("=== DEMOSTRACIÓN DE COMPILACIÓN CONDICIONAL ===\n\n");
    
    // El preprocesador evalúa estas condiciones en tiempo de compilación
    printf("Este mensaje SIEMPRE se compila\n");
    
#ifdef PLATAFORMA
    printf("PLATAFORMA está definida: %s\n", PLATAFORMA);
#endif
    
#ifndef VERSION
    printf("VERSION NO está definida (se definirá ahora)\n");
    #define VERSION "1.0"
#endif
    
    printf("Versión: %s\n", VERSION);
    
    return 0;
}
```

===============================================================================
2. #ifdef Y #ifndef — VERIFICACIÓN DE DEFINICIÓN
===============================================================================

#ifdef (if defined) evalúa si una macro está definida.
#ifndef (if not defined) evalúa si una macro NO está definida.

Sintaxis básica:

```c
#ifdef NOMBRE_MACRO
    // código si está definida
#endif

#ifndef NOMBRE_MACRO
    // código si NO está definida
#endif
```

Ejemplos completos:

```c
#include <stdio.h>

#define EXPERIMENTAL 1
// #define ESTABLE

int main(void) {
    printf("=== VERIFICACIÓN DE DEFINICIONES ===\n\n");
    
#ifdef EXPERIMENTAL
    printf("Módulo EXPERIMENTAL incluido\n");
#endif
    
#ifdef ESTABLE
    printf("Módulo ESTABLE incluido\n");
#endif
    
#ifndef ESTABLE
    printf("ADVERTENCIA: Módulo ESTABLE no presente\n");
#endif
    
    return 0;
}
```

#ifdef no comprueba el VALOR, solo si está definido:

```c
#include <stdio.h>

#define A 0
#define B
#define C 100

int main(void) {
    printf("=== #ifdef VS VALOR ===\n\n");
    
#ifdef A
    printf("A está definido (valor: %d)\n", A);
#endif
    
    // Aunque A vale 0, #ifdef es verdadero porque A está definido
    
#ifdef B
    printf("B está definido\n");
#endif
    
#ifdef C
    printf("C está definido (valor: %d)\n", C);
#endif
    
#ifndef D
    printf("D NO está definido\n");
#endif
    
    return 0;
}
```

Uso típico: configuración de funcionalidades:

```c
#include <stdio.h>

// Configuración del programa
#define CARACTERISTICA_A
#define CARACTERISTICA_B
// #define CARACTERISTICA_C

void funcionalidad_a(void) {
    printf("Ejecutando funcionalidad A\n");
}

void funcionalidad_b(void) {
    printf("Ejecutando funcionalidad B\n");
}

void funcionalidad_c(void) {
    printf("Ejecutando funcionalidad C\n");
}

int main(void) {
    printf("=== CONFIGURACIÓN DE FUNCIONALIDADES ===\n\n");
    
    printf("Funcionalidades activas:\n");
    
#ifdef CARACTERISTICA_A
    printf("  - Característica A: ACTIVA\n");
    funcionalidad_a();
#endif
    
#ifdef CARACTERISTICA_B
    printf("  - Característica B: ACTIVA\n");
    funcionalidad_b();
#endif
    
#ifdef CARACTERISTICA_C
    printf("  - Característica C: ACTIVA\n");
    funcionalidad_c();
#endif
    
#ifndef CARACTERISTICA_C
    printf("  - Característica C: INACTIVA (versión gratuita)\n");
#endif
    
    return 0;
}
```

#ifdef con #else:

```c
#include <stdio.h>

#define MODO_OSCURO

int main(void) {
    printf("=== TEMA DE INTERFAZ ===\n\n");
    
#ifdef MODO_OSCURO
    printf("\033[30;47m"); // Fondo blanco, texto negro
    printf("Modo OSCURO activado\n");
    printf("\033[0m");     // Reset
#else
    printf("\033[37;40m"); // Fondo negro, texto blanco
    printf("Modo CLARO activado\n");
    printf("\033[0m");
#endif
    
    return 0;
}
```

===============================================================================
3. #if, #else, #elif — EXPRESIONES CONSTANTES
===============================================================================

#if permite evaluar expresiones constantes enteras, no solo verificar si una
macro está definida. Esto es mucho más poderoso que #ifdef porque permite
comparaciones y operaciones aritméticas.

Sintaxis:

```c
#if expresion_constante
    // código si expresión != 0
#elif otra_expresion
    // código si otra_expresión != 0
#else
    // código si ninguna condición se cumple
#endif
```

Ejemplos con expresiones:

```c
#include <stdio.h>

#define NIVEL 3
#define MAX_INTENTOS 5

int main(void) {
    printf("=== EXPRESIONES CONSTANTES ===\n\n");
    
#if NIVEL >= 5
    printf("Nivel avanado: todas las funciones desbloqueadas\n");
#elif NIVEL >= 3
    printf("Nivel intermedio: funciones básicas y medias\n");
#elif NIVEL >= 1
    printf("Nivel básico: solo funciones básicas\n");
#else
    printf("Nivel invitado: funciones limitadas\n");
#endif
    
#if MAX_INTENTOS > 10
    printf("Muchos intentos permitidos\n");
#elif MAX_INTENTOS > 3
    printf("Número moderado de intentos\n");
#else
    printf("Pocos intentos permitidos\n");
#endif
    
    return 0;
}
```

Operadores en #if:

```c
#include <stdio.h>

#define TAMANO 1024
#define BLOQUE 256
#define ERROR -1
#define EXITO 0

int main(void) {
    printf("=== OPERADORES EN #if ===\n\n");
    
#if TAMANO > 0 && TAMANO <= 1024
    printf("Tamaño dentro del rango válido (1-1024)\n");
#endif
    
#if TAMANO % BLOQUE == 0
    printf("El tamaño es múltiplo del bloque (%d / %d = %d)\n",
           TAMANO, BLOQUE, TAMANO / BLOQUE);
#endif
    
#if defined(EXITO) && (EXITO == 0)
    printf("EXITO definido como 0\n");
#endif
    
#if ERROR != 0
    printf("ERROR definido como valor no-cero: %d\n", ERROR);
#endif
    
    return 0;
}
```

Operador defined() con #if:

```c
#include <stdio.h>

#define OPCION_A
#define OPCION_B 42
// #define OPCION_C

int main(void) {
    printf("=== OPERADOR defined() ===\n\n");
    
#if defined(OPCION_A)
    printf("OPCION_A está definida\n");
#endif
    
#if defined(OPCION_B)
    printf("OPCION_B está definida con valor: %d\n", OPCION_B);
#endif
    
#if !defined(OPCION_C)
    printf("OPCION_C NO está definida\n");
#endif
    
#if defined(OPCION_A) && defined(OPCION_B) && !defined(OPCION_C)
    printf("Configuración válida: A y B presentes, C ausente\n");
#endif
    
    return 0;
}
```

Diferencia entre #ifdef y #if defined():

```c
#include <stdio.h>

// Son equivalentes:
#ifdef DEBUG
    // código si DEBUG definido
#endif

#if defined(DEBUG)
    // código si DEBUG definido
#endif

// Pero defined() es más flexible porque permite combinaciones lógicas:
#if defined(DEBUG) && defined(VERBOSE) && !defined(SILENCIOSO)
    // código si DEBUG Y VERBOSE definidos y SILENCIOSO no
#endif

// Esto NO se puede hacer con #ifdef solo:
#if defined(OPCION_A) || defined(OPCION_B)
    printf("Al menos una opción está definida\n");
#endif

int main(void) {
    printf("defined() ofrece más flexibilidad\n");
    return 0;
}
```

#if con números negativos y expresiones complejas:

```c
#include <stdio.h>

#define VALOR -5
#define LIMITE 100
#define FACTOR 2

int main(void) {
    printf("=== EXPRESIONES COMPLEJAS ===\n\n");
    
#if VALOR < 0
    printf("VALOR es negativo: %d\n", VALOR);
#endif
    
#if (LIMITE * FACTOR) > 150
    printf("Límite * Factor = %d > 150\n", LIMITE * FACTOR);
#else
    printf("Límite * Factor = %d <= 150\n", LIMITE * FACTOR);
#endif
    
#if VALOR >= -10 && VALOR <= 10
    printf("VALOR está en el rango [-10, 10]\n");
#endif
    
    return 0;
}
```

4. INCLUDE GUARDS — PROTECCIÓN CONTRA INCLUSIÓN MÚLTIPLE
===============================================================================

Los include guards son la aplicación más común de las directivas condicionales.
Evitan que un archivo de encabezado sea incluido múltiples veces, lo que
causaría errores de redefinición.

Estructura del include guard tradicional:

```c
#ifndef NOMBRE_ARCHIVO_H
#define NOMBRE_ARCHIVO_H

// Contenido del archivo de encabezado

#endif /* NOMBRE_ARCHIVO_H */
```

Ejemplo completo de include guards:

```c
// archivo: config.h
#ifndef CONFIG_H
#define CONFIG_H

#define VERSION "3.0"
#define MAX_USUARIOS 1000
#define TIMEOUT_SEG 30

typedef struct {
    char nombre[50];
    int edad;
    float promedio;
} Estudiante;

int iniciar_sistema(void);
void cerrar_sistema(void);

#endif /* CONFIG_H */
```

```c
// archivo: funciones.h
#ifndef FUNCIONES_H
#define FUNCIONES_H

#include "config.h"  // También tiene include guard

int calcular_promedio(int suma, int count);
void mostrar_estudiante(const Estudiante* e);

#endif /* FUNCIONES_H */
```

```c
// archivo: main.c
#include <stdio.h>
#include "config.h"    // Incluido
#include "funciones.h" // Incluye config.h otra vez, pero el guard lo protege
#include "config.h"    // Inclusión directa repetida, protegida

int main(void) {
    Estudiante est = {"Ana", 22, 9.5};
    mostrar_estudiante(&est);
    printf("Promedio: %d\n", calcular_promedio(95, 10));
    return 0;
}
```

Sin include guards, el error sería:

```c
// archivo: datos.h (SIN include guard)
#define MAX_ELEMENTOS 100
typedef struct { int x, y; } Punto;
```

```c
// archivo: main.c
#include "datos.h"
#include "datos.h"  // ERROR: MAX_ELEMENTOS redefinido, Punto redefinido

int main(void) {
    return 0;
}
```

#pragma once como alternativa moderna:

```c
// archivo: moderno.h
#pragma once

// Equivalente a include guard, pero más limpio
// No es estándar C pero funciona en GCC, Clang, MSVC

int funcion_moderna(void);
```

Include guards avanzados con detección de inclusión previa:

```c
// archivo: types.h
#ifndef TYPES_H
#define TYPES_H

// Este archivo define tipos base

typedef int Entero;
typedef float Real;

#endif
```

```c
// archivo: extended_types.h
#ifndef EXTENDED_TYPES_H
#define EXTENDED_TYPES_H

#include "types.h"

// Verificar que types.h fue incluido
#ifndef TYPES_H
    #error "extended_types.h requiere types.h"
#endif

typedef Entero ID;
typedef Real Porcentaje;

#endif
```

5. MACROS DE DEPURACIÓN CONDICIONAL
===============================================================================

Una de las aplicaciones más útiles de la compilación condicional es la
creación de macros de depuración que se activan solo cuando se define un
flag de depuración.

Sistema básico de depuración:

```c
#include <stdio.h>

// Descomentar para activar depuración:
#define DEBUG

#ifdef DEBUG
    #define DEBUG_PRINT(msg) printf("[DEBUG] %s\n", msg)
    #define DEBUG_VAR(fmt, var) printf("[DEBUG] " #var " = " fmt "\n", var)
#else
    #define DEBUG_PRINT(msg)
    #define DEBUG_VAR(fmt, var)
#endif

int main(void) {
    printf("Sistema iniciado\n");
    
    DEBUG_PRINT("Inicializando módulos...");
    
    int contador = 42;
    DEBUG_VAR("%d", contador);
    
    double precio = 99.99;
    DEBUG_VAR("%.2f", precio);
    
    DEBUG_PRINT("Módulos inicializados correctamente");
    
    printf("Sistema operativo\n");
    return 0;
}
```

Sistema avanzado con niveles de depuración:

```c
#include <stdio.h>

// Niveles de depuración
#define NIVEL_NINGUNO 0
#define NIVEL_ERROR   1
#define NIVEL_WARNING 2
#define NIVEL_INFO    3
#define NIVEL_DEBUG   4

// Configurar nivel (definir antes de incluir, o pasar con -D)
#ifndef NIVEL_DEBUG
    #define NIVEL_DEBUG NIVEL_INFO
#endif

// Macros de depuración por nivel
#if NIVEL_DEBUG >= NIVEL_ERROR
    #define LOG_ERROR(msg, ...) \
        fprintf(stderr, "[ERROR] %s:%d: " msg "\n", __FILE__, __LINE__, ##__VA_ARGS__)
#else
    #define LOG_ERROR(msg, ...)
#endif

#if NIVEL_DEBUG >= NIVEL_WARNING
    #define LOG_WARNING(msg, ...) \
        printf("[WARNING] " msg "\n", ##__VA_ARGS__)
#else
    #define LOG_WARNING(msg, ...)
#endif

#if NIVEL_DEBUG >= NIVEL_INFO
    #define LOG_INFO(msg, ...) \
        printf("[INFO] " msg "\n", ##__VA_ARGS__)
#else
    #define LOG_INFO(msg, ...)
#endif

#if NIVEL_DEBUG >= NIVEL_DEBUG
    #define LOG_DEBUG(msg, ...) \
        printf("[DEBUG] %s:%d: " msg "\n", __FILE__, __LINE__, ##__VA_ARGS__)
#else
    #define LOG_DEBUG(msg, ...)
#endif

int funcion_ejemplo(int x) {
    LOG_INFO("Entrando a funcion_ejemplo con x=%d", x);
    
    if (x < 0) {
        LOG_ERROR("x es negativo: %d", x);
        return -1;
    }
    
    LOG_DEBUG("Procesando x=%d", x);
    
    if (x > 100) {
        LOG_WARNING("x es muy grande: %d, puede haber problemas de rendimiento", x);
    }
    
    LOG_INFO("Saliendo de funcion_ejemplo");
    return x * 2;
}

int main(void) {
    LOG_INFO("Iniciando programa");
    printf("Nivel de depuración: %d\n", NIVEL_DEBUG);
    
    funcion_ejemplo(50);
    funcion_ejemplo(-5);
    funcion_ejemplo(500);
    
    LOG_INFO("Programa finalizado");
    return 0;
}
```

6. DETECCIÓN DE PLATAFORMA CON DIRECTIVAS CONDICIONALES
===============================================================================

La detección multiplataforma es una de las aplicaciones más importantes de
la compilación condicional. Diferentes sistemas operativos y compiladores
definen macros predefinidas que podemos detectar.

Sistema operativo:

```c
#include <stdio.h>

int main(void) {
    printf("=== DETECCIÓN DE SISTEMA OPERATIVO ===\n\n");
    
#if defined(_WIN32) || defined(_WIN64)
    printf("Sistema: Windows\n");
    #define SO_WINDOWS
#elif defined(__linux__)
    printf("Sistema: Linux\n");
    #define SO_LINUX
#elif defined(__APPLE__) && defined(__MACH__)
    printf("Sistema: macOS\n");
    #define SO_MACOS
#elif defined(__FreeBSD__)
    printf("Sistema: FreeBSD\n");
    #define SO_FREEBSD
#elif defined(__ANDROID__)
    printf("Sistema: Android\n");
    #define SO_ANDROID
#else
    printf("Sistema: Desconocido\n");
    #define SO_DESCONOCIDO
#endif
    
    return 0;
}
```

Arquitectura del procesador:

```c
#include <stdio.h>

int main(void) {
    printf("=== DETECCIÓN DE ARQUITECTURA ===\n\n");
    
#if defined(__x86_64__) || defined(_M_X64) || defined(__amd64__)
    printf("Arquitectura: x86_64 (64 bits)\n");
    #define ARCH_X64
#elif defined(__i386__) || defined(_M_IX86) || defined(__i386)
    printf("Arquitectura: x86 (32 bits)\n");
    #define ARCH_X86
#elif defined(__aarch64__) || defined(_M_ARM64)
    printf("Arquitectura: ARM64\n");
    #define ARCH_ARM64
#elif defined(__arm__) || defined(_M_ARM)
    printf("Arquitectura: ARM\n");
    #define ARCH_ARM
#elif defined(__riscv)
    printf("Arquitectura: RISC-V\n");
    #define ARCH_RISCV
#else
    printf("Arquitectura: Desconocida\n");
    #define ARCH_DESCONOCIDA
#endif
    
    return 0;
}
```

Compilador:

```c
#include <stdio.h>

int main(void) {
    printf("=== DETECCIÓN DE COMPILADOR ===\n\n");
    
#if defined(__GNUC__) || defined(__clang__)
    #if defined(__clang__)
        printf("Compilador: Clang\n");
        printf("Versión: %d.%d.%d\n", __clang_major__, __clang_minor__, __clang_patchlevel__);
        #define COMPILADOR_CLANG
    #elif defined(__GNUC__)
        printf("Compilador: GCC\n");
        printf("Versión: %d.%d.%d\n", __GNUC__, __GNUC_MINOR__, __GNUC_PATCHLEVEL__);
        #define COMPILADOR_GCC
    #endif
#elif defined(_MSC_VER)
    printf("Compilador: MSVC\n");
    printf("Versión: %d\n", _MSC_VER);
    #define COMPILADOR_MSVC
#elif defined(__TINYC__)
    printf("Compilador: TCC (Tiny C Compiler)\n");
    #define COMPILADOR_TCC
#else
    printf("Compilador: Desconocido\n");
#endif
    
    return 0;
}
```

7. SISTEMA DE CONFIGURACIÓN MULTIPLATAFORMA
===============================================================================

Construyamos un sistema completo de configuración multiplataforma:

```c
#include <stdio.h>
#include <stdlib.h>

//==============================================================================
// CONFIGURACIÓN AUTOMÁTICA DEL SISTEMA
//==============================================================================

// Detectar sistema operativo
#if defined(_WIN32) || defined(_WIN64)
    #define PLATAFORMA_WINDOWS
    #define NOMBRE_SO "Windows"
    #define LIMPIAR_PANTALLA "cls"
    #define SEPARADOR_RUTA "\\"
    #define SALTO_LINEA "\r\n"
#elif defined(__linux__)
    #define PLATAFORMA_LINUX
    #define NOMBRE_SO "Linux"
    #define LIMPIAR_PANTALLA "clear"
    #define SEPARADOR_RUTA "/"
    #define SALTO_LINEA "\n"
#elif defined(__APPLE__) && defined(__MACH__)
    #define PLATAFORMA_MACOS
    #define NOMBRE_SO "macOS"
    #define LIMPIAR_PANTALLA "clear"
    #define SEPARADOR_RUTA "/"
    #define SALTO_LINEA "\n"
#else
    #define PLATAFORMA_DESCONOCIDA
    #define NOMBRE_SO "Desconocido"
    #define LIMPIAR_PANTALLA "clear"
    #define SEPARADOR_RUTA "/"
    #define SALTO_LINEA "\n"
    #warning "Plataforma no reconocida, usando valores por defecto"
#endif

// Detectar tipo de sistema (Little/Big Endian)
#if defined(__BYTE_ORDER__) && __BYTE_ORDER__ == __ORDER_BIG_ENDIAN__
    #define ES_BIG_ENDIAN
#elif defined(__BYTE_ORDER__) && __BYTE_ORDER__ == __ORDER_LITTLE_ENDIAN__
    #define ES_LITTLE_ENDIAN
#else
    // Detección en tiempo de ejecución
#endif

// Configuración de tipos portables
#ifdef PLATAFORMA_WINDOWS
    #define SLEEP(ms) Sleep(ms)
    #include <windows.h>
#else
    #include <unistd.h>
    #define SLEEP(ms) usleep((ms) * 1000)
#endif

// Configuración de funciones específicas de plataforma
void limpiar_pantalla(void) {
    system(LIMPIAR_PANTALLA);
}

void pausa(void) {
#ifdef PLATAFORMA_WINDOWS
    system("pause");
#else
    printf("Presione Enter para continuar...");
    getchar();
#endif
}

void mostrar_info_sistema(void) {
    printf("====================================\n");
    printf("  INFORMACIÓN DEL SISTEMA\n");
    printf("====================================\n");
    printf("Sistema Operativo: %s\n", NOMBRE_SO);
    printf("Separador de rutas: %s\n", SEPARADOR_RUTA);
    
#ifdef ES_BIG_ENDIAN
    printf("Endianness: Big Endian\n");
#elif defined(ES_LITTLE_ENDIAN)
    printf("Endianness: Little Endian\n");
#else
    printf("Endianness: Desconocido\n");
#endif
    
    printf("Línea de compilación: %s:%d\n", __FILE__, __LINE__);
    printf("Compilado el: %s a las %s\n", __DATE__, __TIME__);
    printf("====================================\n\n");
}

int main(void) {
    mostrar_info_sistema();
    
    printf("Directorio actual: ");
#ifdef PLATAFORMA_WINDOWS
    system("echo %cd%");
#else
    system("pwd");
#endif
    
    printf("\nRuta de ejemplo: /home%susuario%sdocumentos\n",
           SEPARADOR_RUTA, SEPARADOR_RUTA);
    
    printf("\nEsperando 2 segundos...\n");
    SLEEP(2000);
    printf("Continuando...\n");
    
    pausa();
    return 0;
}
```

8. #if vs #ifdef — CUÁNDO USAR CADA UNO
===============================================================================

La diferencia sutil pero importante entre #ifdef MACRO y #if defined(MACRO):

```c
#include <stdio.h>

#define VALOR 0

int main(void) {
    printf("=== #ifdef VS #if CON VALOR 0 ===\n\n");
    
#ifdef VALOR
    printf("#ifdef: VALOR está definido (aunque sea 0)\n");
#endif
    
#if VALOR
    printf("#if: VALOR es verdadero (no-cero)\n");
#else
    printf("#if: VALOR es falso (cero)\n");
#endif
    
#if defined(VALOR) && VALOR != 0
    printf("defined() && no-cero: VALOR está definido y no es 0\n");
#else
    printf("defined() && no-cero: VALOR no está definido o es 0\n");
#endif
    
    return 0;
}
```

Reglas prácticas:

```c
#include <stdio.h>

// Use #ifdef cuando solo importe si la macro existe
// Use #if MACRO cuando importe el valor

#define FEATURE_A     // Habilitar característica A
#define FEATURE_B  1  // Habilitar característica B (valor 1)
#define FEATURE_C  0  // Deshabilitar característica C (valor 0)
// #define FEATURE_D    // Deshabilitar característica D (no definida)

int main(void) {
    printf("=== COMPARACIÓN DE ENFOQUES ===\n\n");
    
    // Enfoque con #ifdef (solo importa la existencia)
    printf("Característica A: ");
#ifdef FEATURE_A
    printf("ACTIVA\n");
#else
    printf("INACTIVA\n");
#endif
    
    // Enfoque con #if (importa el valor)
    printf("Característica B: ");
#if FEATURE_B
    printf("ACTIVA (valor: %d)\n", FEATURE_B);
#else
    printf("INACTIVA\n");
#endif
    
    // Con #if, FEATURE_C está definida pero es 0, entonces es falsa
    printf("Característica C: ");
#if FEATURE_C
    printf("ACTIVA\n");
#else
    printf("INACTIVA (valor: %d)\n", FEATURE_C);
#endif
    
    // Con #if, FEATURE_D no está definida, entonces es falsa
    printf("Característica D: ");
#if defined(FEATURE_D) && FEATURE_D
    printf("ACTIVA\n");
#else
    printf("INACTIVA (no definida)\n");
#endif
    
    return 0;
}
```

```c
#define ASSERT_TAMANO(tipo, tamano) \
    typedef char assert_##tipo##_tamano[(sizeof(tipo) == tamano) ? 1 : -1]

// Uso
STATIC_ASSERT(sizeof(int) >= 4, int_debe_ser_4_bytes);
STATIC_ASSERT(sizeof(char) == 1, char_debe_ser_1_byte);

// ASSERT_TAMANO(int, 4);

int main(void) {
    printf("Compilación exitosa - todas las aserciones pasaron\n");
    
    printf("sizeof(int) = %zu\n", sizeof(int));
    printf("sizeof(char) = %zu\n", sizeof(char));
    
    return 0;
}
```

RESUMEN DEL CAPÍTULO
===============================================================================

- #ifdef/#ifndef verifican si una macro está o no definida (sin importar su valor)
- #if evalúa expresiones constantes completas con operadores aritméticos y lógicos
- #else y #elif permiten bifurcaciones múltiples en compilación condicional
- #endif cierra cualquier bloque condicional abierto
- El operador defined() ofrece más flexibilidad que #ifdef permitiendo
  combinaciones lógicas con && y ||
- Los include guards (#ifndef + #define + #endif) evitan inclusiones múltiples
- Las macros de depuración condicional permiten activar logging según el nivel
- La detección multiplataforma usa macros predefinidas del compilador
- #if y #ifdef tienen semánticas diferentes: #if evalúa valor, #ifdef verifica existencia
- Las directivas se pueden anidar para configuraciones complejas

PREGUNTAS FRECUENTES
===============================================================================

❓ **¿Cuál es la diferencia entre #ifdef y #if defined()?** Ninguna funcional.
Ambos verifican si una macro está definida. La ventaja de #if defined() es
que permite combinaciones como #if defined(A) && !defined(B).

❓ **¿Puedo usar operadores lógicos en #if?** Sí. #if soporta &&, ||, !,
operadores de comparación (==, !=, <, >, <=, >=) y aritméticos (+, -, *, /, %).

❓ **¿Qué pasa si olvido el #endif?** El compilador generará un error
indicando que falta el #endif. Es como olvidar cerrar una llave.

❓ **¿Los include guards tienen que tener el nombre del archivo?** No es
obligatorio, pero es una convención muy recomendada para evitar colisiones.
El nombre debe ser único en todo el proyecto.

❓ **¿Puedo anidar directivas condicionales?** Sí, pero el código se vuelve
difícil de leer. Se recomienda un máximo de 2-3 niveles de anidación.
