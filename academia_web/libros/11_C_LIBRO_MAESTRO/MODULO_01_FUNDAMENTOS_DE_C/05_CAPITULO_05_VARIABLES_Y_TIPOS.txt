1. ¿QUÉ ES UNA VARIABLE?
===============================================================================

**1.1 Concepto fundamental**

Una **variable** es un nombre simbólico que asociamos a una posición
de memoria donde almacenamos un valor. Piensa en una variable como
una caja etiquetada donde puedes guardar algo.

```
Variable: edad
Valor:    25
Dirección: 0x7ffd5a1b4c08

+----------------+
|      25        |   <- Contenido (valor)
|    edad        |   <- Nombre (etiqueta)
|  0x7ffd...     |   <- Dirección en memoria
+----------------+
```

**1.2 Declaración de variables**

En C, debes declarar una variable antes de usarla. La declaración
tiene este formato:

```c
tipo nombre;
tipo nombre = valor_inicial;  /* Declaración con inicialización */
```

Ejemplos:

```c
int edad;              /* Declara una variable entera llamada edad */
float altura;          /* Declara una variable de punto flotante */
char inicial;          /* Declara una variable de carácter */
int contador = 0;      /* Declara e inicializa a 0 */
double pi = 3.1416;    /* Declara e inicializa con un valor */
```

**1.3 Reglas para nombres de variables**

- Pueden contener letras (mayúsculas y minúsculas), dígitos y
  guiones bajos (_).
- Deben comenzar con una letra o guión bajo (no con un dígito).
- Son sensibles a mayúsculas/minúsculas: `edad`, `Edad` y `EDAD`
  son tres variables diferentes.
- No pueden usar palabras reservadas de C (int, return, etc.).

```c
#include <stdio.h>

int main(void) {
    /* Nombres válidos */
    int edad = 25;
    int _contador = 0;
    int numero2 = 10;
    int miVariable = 5;

    printf("Edad: %d\n", edad);
    printf("Contador: %d\n", _contador);
    printf("Numero: %d\n", numero2);
    printf("Mi variable: %d\n", miVariable);

    return 0;
}
```

===============================================================================
2. TIPOS DE DATOS FUNDAMENTALES
===============================================================================

C proporciona cuatro tipos de datos básicos:

| Tipo      | Descripcion                     | Tamano tipico |
|-----------|---------------------------------|---------------|
| int       | Numeros enteros                 | 4 bytes       |
| char      | Caracter (o entero pequeno)     | 1 byte        |
| float     | Numero de punto flotante        | 4 bytes       |
| double    | Numero de punto flotante doble  | 8 bytes       |

Ademas, desde C99 tenemos:

| Tipo    | Descripcion                     | Tamano tipico |
|---------|---------------------------------|---------------|
| _Bool   | Valor booleano (0 o 1)          | 1 byte        |

**2.1 El tipo int (enteros)**

Almacena numeros enteros (sin parte fraccionaria). Rango tipico:
de -2,147,483,648 a 2,147,483,647 (en sistemas de 32/64 bits).

```c
#include <stdio.h>

int main(void) {
    int a = 10;
    int b = -5;
    int c = 0;
    int d = 1000000;

    printf("Enteros:\n");
    printf("a = %d\n", a);
    printf("b = %d\n", b);
    printf("c = %d\n", c);
    printf("d = %d\n", d);

    return 0;
}
```

**Modificadores de enteros:**

- `short int` - Entero mas pequeno (normalmente 2 bytes).
- `long int` - Entero mas grande (normalmente 8 bytes).
- `long long int` - Entero aun mas grande (al menos 8 bytes, C99).
- `unsigned int` - Entero sin signo (solo positivos y cero).

```c
#include <stdio.h>

int main(void) {
    short int s = 100;
    long int l = 100000L;
    long long int ll = 10000000000LL;
    unsigned int u = 4000000000U;

    printf("short:    %hd\n", s);
    printf("long:     %ld\n", l);
    printf("long long: %lld\n", ll);
    printf("unsigned: %u\n", u);

    return 0;
}
```

**2.2 El tipo char (caracteres)**

Almacena un solo caracter o un entero pequeno (0-255). Ocupa 1 byte.

```c
#include <stdio.h>

int main(void) {
    char letra = 'A';
    char numero = 65;        /* Equivalente a 'A' en ASCII */
    char simbolo = '$';

    printf("Caracteres:\n");
    printf("letra = %c (codigo ASCII: %d)\n", letra, letra);
    printf("numero = %c (codigo ASCII: %d)\n", numero, numero);
    printf("simbolo = %c\n", simbolo);

    char siguiente = letra + 1;
    printf("Siguiente de 'A': %c\n", siguiente);

    return 0;
}
```

**2.3 El tipo float (punto flotante)**

Almacena numeros con parte decimal. Precision de aproximadamente
6-7 digitos decimales. Ocupa 4 bytes.

```c
#include <stdio.h>

int main(void) {
    float temperatura = 36.5f;
    float pi = 3.14159f;
    float notacion = 1.5e-4f;

    printf("Punto flotante (float):\n");
    printf("Temperatura: %.1f C\n", temperatura);
    printf("Pi: %.5f\n", pi);
    printf("Notacion cientifica: %e\n", notacion);

    return 0;
}
```

**2.4 El tipo double (doble precision)**

Similar a float pero con el doble de precision (aproximadamente
15-17 digitos decimales). Ocupa 8 bytes.

```c
#include <stdio.h>

int main(void) {
    double pi = 3.141592653589793;
    double gravedad = 9.80665;
    double planck = 6.62607015e-34;

    printf("Doble precision (double):\n");
    printf("Pi: %.15f\n", pi);
    printf("Gravedad: %.5f m/s^2\n", gravedad);
    printf("Constante de Planck: %.15e J*s\n", planck);

    return 0;
}
```

**2.5 El tipo _Bool (booleanos)**

Desde C99, C tiene un tipo booleano. Almacena 0 (falso) o 1
(verdadero). Ocupa 1 byte.

```c
#include <stdio.h>
#include <stdbool.h>

int main(void) {
    _Bool activo = 1;
    bool conectado = true;
    bool terminado = false;

    printf("Booleanos:\n");
    printf("activo = %d\n", activo);
    printf("conectado = %d\n", conectado);
    printf("terminado = %d\n", terminado);

    bool prueba = 42;
    printf("prueba (con 42) = %d (cualquier != 0 es true)\n", prueba);

    return 0;
}
```

===============================================================================
3. EL OPERADOR sizeof
===============================================================================

`sizeof` es un operador (no una funcion) que devuelve el tamano
en bytes de un tipo o variable. Es fundamental para escribir codigo
portatil.

**3.1 Sintaxis**

```c
sizeof(tipo)       /* Tamano de un tipo */
sizeof(variable)   /* Tamano de una variable */
sizeof expresion   /* Tamano de una expresion */
```

**3.2 Ejemplo completo**

```c
#include <stdio.h>

int main(void) {
    printf("=== TAMANOS DE TIPOS EN BYTES ===\n\n");

    printf("char:        %zu byte(s)\n", sizeof(char));
    printf("short:       %zu bytes\n", sizeof(short));
    printf("int:         %zu bytes\n", sizeof(int));
    printf("long:        %zu bytes\n", sizeof(long));
    printf("long long:   %zu bytes\n", sizeof(long long));
    printf("float:       %zu bytes\n", sizeof(float));
    printf("double:      %zu bytes\n", sizeof(double));
    printf("long double: %zu bytes\n", sizeof(long double));
    printf("_Bool:       %zu byte(s)\n", sizeof(_Bool));
    printf("void*:       %zu bytes\n", sizeof(void*));

    printf("\n=== MODIFICADORES ===\n\n");
    printf("unsigned int:       %zu bytes\n", sizeof(unsigned int));
    printf("short int:          %zu bytes\n", sizeof(short int));
    printf("long int:           %zu bytes\n", sizeof(long int));
    printf("unsigned long:      %zu bytes\n", sizeof(unsigned long));
    printf("long long int:      %zu bytes\n", sizeof(long long int));

    return 0;
}
```

4. ESPECIFICADORES DE FORMATO
===============================================================================

Cada tipo de dato tiene uno o mas especificadores de formato
para usarse con printf y scanf.

| Tipo                 | Especificador | Ejemplo                       |
|----------------------|---------------|-------------------------------|
| int                  | %d            | printf("%d", x);              |
| unsigned int         | %u            | printf("%u", x);              |
| short                | %hd           | printf("%hd", x);             |
| long                 | %ld           | printf("%ld", x);             |
| long long            | %lld          | printf("%lld", x);            |
| unsigned long        | %lu           | printf("%lu", x);             |
| unsigned long long   | %llu          | printf("%llu", x);            |
| float                | %f            | printf("%f", x);              |
| double               | %f o %lf      | printf("%lf", x);             |
| long double          | %Lf           | printf("%Lf", x);             |
| char                 | %c            | printf("%c", x);              |
| _Bool                | %d            | printf("%d", x);              |
| char* (cadena)       | %s            | printf("%s", str);            |
| void* (puntero)      | %p            | printf("%p", ptr);            |
| size_t               | %zu           | printf("%zu", sizeof(x));     |

```c
#include <stdio.h>

int main(void) {
    int entero = 42;
    unsigned int positivo = 3000000000U;
    long largo = 123456789L;
    float flotante = 3.14f;
    double doble = 2.718281828459045;
    char letra = 'Z';
    _Bool verdad = 1;

    printf("int:            %d\n", entero);
    printf("unsigned int:   %u\n", positivo);
    printf("long:           %ld\n", largo);
    printf("float:          %.2f\n", flotante);
    printf("double:         %.15f\n", doble);
    printf("char:           %c\n", letra);
    printf("_Bool:          %d\n", verdad);
    printf("sizeof(int):    %zu bytes\n", sizeof(int));

    return 0;
}
```

5. INICIALIZACIÓN DE VARIABLES
===============================================================================

**5.1 Inicialización en la declaración**

Puedes asignar un valor inicial a una variable cuando la declaras.

```c
#include <stdio.h>

int main(void) {
    int a = 10;          /* Inicializacion directa */
    int b = a + 5;       /* Inicializacion con expresion */
    int c = b;           /* Inicializacion con otra variable */

    printf("a = %d, b = %d, c = %d\n", a, b, c);

    return 0;
}
```

**5.2 Sin inicialización = valor basura**

Si declaras una variable sin inicializarla, su valor es indeterminado
(cualquier cosa que hubiera en esa posicion de memoria).

```c
#include <stdio.h>

int main(void) {
    int x;  /* NO inicializada - contiene "basura" */

    printf("x (sin inicializar) = %d\n", x);
    printf("VALOR INDETERMINADO - puede ser cualquier cosa\n");

    return 0;
}
```

**5.3 Inicialización multiple**

```c
#include <stdio.h>

int main(void) {
    int a = 1, b = 2, c = 3;  /* Multiples variables en una linea */
    int x, y, z;               /* Solo declaracion (sin inicializar) */
    x = y = z = 0;             /* Asignacion multiple */

    printf("a=%d b=%d c=%d\n", a, b, c);
    printf("x=%d y=%d z=%d\n", x, y, z);

    return 0;
}
```

6. AMBITO (SCOPE) DE VARIABLES
===============================================================================

**6.1 Ambito local**

Las variables declaradas dentro de una funcion solo existen dentro
de esa funcion.

```c
#include <stdio.h>

void funcion(void) {
    int local = 42;  /* Solo existe dentro de funcion() */
    printf("Dentro de funcion: %d\n", local);
}

int main(void) {
    /* printf("%d\n", local);  ERROR: local no existe aqui */
    funcion();
    return 0;
}
```

**6.2 Ambito de bloque**

Desde C99, puedes declarar variables dentro de cualquier bloque {}.

```c
#include <stdio.h>

int main(void) {
    int x = 10;

    {  /* Inicio de bloque */
        int y = 20;  /* y solo existe dentro de este bloque */
        printf("Dentro del bloque: x=%d, y=%d\n", x, y);
    }  /* Fin de bloque: y se destruye aqui */

    /* printf("%d\n", y);  ERROR: y no existe fuera del bloque */

    printf("Fuera del bloque: x=%d\n", x);

    return 0;
}
```

**6.3 Variables globales**

Declaradas fuera de cualquier funcion, existen durante toda la
ejecucion del programa y son accesibles desde cualquier funcion.

```c
#include <stdio.h>

int contador_global = 0;  /* Variable global */

void incrementar(void) {
    contador_global++;
}

int main(void) {
    printf("Global: %d\n", contador_global);
    incrementar();
    printf("Global: %d\n", contador_global);
    incrementar();
    printf("Global: %d\n", contador_global);

    return 0;
}
```

7. CONSTANTES Y LITERALES
===============================================================================

**7.1 Literales enteros**

```c
#include <stdio.h>

int main(void) {
    int decimal   = 42;       /* Base 10 (decimal) */
    int octal     = 052;      /* Base 8 (octal) - comienza con 0 */
    int hex        = 0x2A;     /* Base 16 (hexadecimal) - comienza con 0x */
    int binario   = 0b101010; /* Base 2 (binario) - desde C23 */

    printf("decimal:   %d\n", decimal);
    printf("octal:     %d (052 en octal)\n", octal);
    printf("hex:       %d (0x2A en hex)\n", hex);

    return 0;
}
```

**7.2 Literales de punto flotante**

```c
#include <stdio.h>

int main(void) {
    double a = 3.14159;        /* Notacion decimal */
    double b = 3.14159e5;      /* Notacion cientifica: 3.14159 * 10^5 */
    double c = 0.0001;         /* Decimal pequeno */
    double d = 1.5e-4;         /* Cientifica: 0.00015 */
    float  f = 3.14f;          /* Sufijo f para float */

    printf("a = %f\n", a);
    printf("b = %f (3.14159e5)\n", b);
    printf("c = %f\n", c);
    printf("d = %f (1.5e-4)\n", d);
    printf("f = %f (float)\n", f);

    return 0;
}
```

**7.3 Literales de caracter**

```c
#include <stdio.h>

int main(void) {
    char a = 'A';           /* Caracter simple */
    char b = '\n';          /* Caracter de escape */
    char c = '\x41';        /* Hexadecimal: 0x41 = 'A' */
    char d = '\101';        /* Octal: 0101 = 'A' */

    printf("a = %c\n", a);
    printf("c (\\x41) = %c\n", c);
    printf("d (\\101) = %c\n", d);

    return 0;
}
```

8. BUENAS PRACTICAS CON VARIABLES
===============================================================================

**8.1 Siempre inicializa las variables**

```c
int contador = 0;     /* Bien */
float total = 0.0f;   /* Bien */
char nombre[50] = ""; /* Bien */
int x;                /* Mal: valor basura */
```

**8.2 Usa nombres descriptivos**

```c
int e;                /* Mal: no dice nada */
int edad_usuario;     /* Bien: describe el proposito */
int numero_estudiantes; /* Bien */
```

**8.3 Una declaracion por linea**

```c
int a, b, c;              /* Aceptable, pero menos legible */
int a = 0;                /* Mejor: una variable por linea */
int b = 0;
int c = 0;
```

**8.4 Declara cerca del uso**

```c
/* Mal: todas al inicio, lejos de donde se usan */
void funcion(void) {
    int a, b, c, d, e, f;
    /* ... muchas lineas ... */
    a = 10;
}

/* Bien: cerca de donde se necesitan */
void funcion(void) {
    int a = 10;
    /* usas a */
    int b = 20;
    /* usas b */
}
```

```c
#include <stdio.h>

int main(void) {
    printf("=== BUENAS PRACTICAS ===\n\n");
    printf("1. Inicializa siempre las variables\n");
    printf("2. Usa nombres descriptivos\n");
    printf("3. Una declaracion por linea\n");
    printf("4. Declara cerca del uso\n");
    printf("5. Usa constantes para valores magicos\n");

    return 0;
}
```

RESUMEN DEL CAPITULO
===============================================================================

- Una variable es un nombre que referencia una posicion de memoria.
- Los tipos basicos son: int, char, float, double, _Bool.
- int almacena enteros; modificadores: short, long, long long, unsigned.
- char almacena un caracter (1 byte), tambien se usa como entero pequeno.
- float (4 bytes) y double (8 bytes) para numeros con decimales.
- _Bool (C99) almacena 0 (falso) o 1 (verdadero); usar <stdbool.h>.
- sizeof devuelve el tamano en bytes de un tipo o variable.
- Cada tipo tiene especificadores de formato especificos.
- Inicializa siempre las variables para evitar valores basura.
- El ambito (scope) determina donde existe una variable.

PREGUNTAS FRECUENTES
===============================================================================

?** ?Que diferencia hay entre float y double?**
float tiene precision simple (4 bytes, ~7 digitos decimales), double
tiene precision doble (8 bytes, ~15 digitos decimales). En general,
usa double a menos que tengas restricciones estrictas de memoria.

?** ?Por que %d para int y %f para float?**
C usa especificadores de formato que indican al compilador como
interpretar los bytes en memoria. %d interpreta los bytes como
entero con signo, %f los interpreta como punto flotante. Usar el
especificador incorrecto produce resultados erroneos.

?** ?Que pasa si asigno un valor muy grande a un tipo pequeno?**
Se produce un desbordamiento (overflow). Por ejemplo, si asignas
300 a un char (maximo 255), el valor se trunca. Esto es un error
logico que debes evitar.

?** ?Puedo cambiar el tipo de una variable despues de declararla?**
No. En C, una vez que declaras una variable con un tipo, ese tipo
es fijo para toda la vida de la variable. C es un lenguaje de
tipado estatico.

DESAFIO DEL ARQUITECTO
===============================================================================

**Arquitecto de Memoria: El Mapa de Memoria Interactivo**

Disena un programa en C llamado `mapa_memoria.c` que declare
variables de diferentes tipos y muestre un "mapa de memoria" visual
que incluya para cada variable:

1. El nombre de la variable.
2. El tipo de dato.
3. La direccion de memoria (con %p).
4. El valor actual.
5. El tamano en bytes (con sizeof).
6. El rango teorico de valores que puede almacenar.

El programa debe presentar la informacion en una tabla formateada
con bordes ASCII. Ademas, debe mostrar:

- La direccion mas baja y mas alta de las variables.
- La distancia en bytes entre la primera y ultima variable.
- Una pequena "barra de memoria" visual que represente
  graficamente la posicion relativa de cada variable en la pila.

Ejemplo de salida:
```
+================== MAPA DE MEMORIA ==================+
| Variable     | Tipo   | Direccion  | Valor | Bytes |
+--------------+--------+------------+-------+-------+
| edad         | int    | 0x7ffc... |   25  |   4   |
| altura       | double | 0x7ffc... | 1.75  |   8   |
| ...          | ...    | ...        | ...   |  ...  |
+======================================================+
| Rango de direcciones: 0x7ffcXXXX a 0x7ffcYYYY        |
| Distancia total: 52 bytes                            |
+======================================================+
| BARRA DE MEMORIA:                                     |
| [edad][edad][altura   ][letra][...]                   |
+======================================================+
```

Este desafio te ayudara a visualizar como C organiza las variables
en memoria, un concepto fundamental para dominar el lenguaje.
