1. LA FUNCION printf
===============================================================================

printf (print formatted) es la funcion principal para mostrar
datos formateados en la salida estandar (generalmente la consola).

**1.1 Sintaxis basica**

```c
printf("cadena de formato", argumentos...);
```

La cadena de formato contiene texto literal y especificadores de
formato que comienzan con %.

```c
#include <stdio.h>

int main(void) {
    printf("Hola, %s! Tienes %d anios.
", "Luis", 25);
    return 0;
}
```

**1.2 Especificadores de formato**

| Especificador | Tipo de dato            | Ejemplo                |
|---------------|-------------------------|------------------------|
| %d o %i       | int (entero con signo)  | printf("%d", 42);      |
| %u            | unsigned int            | printf("%u", 42U);     |
| %f            | double (punto flotante) | printf("%f", 3.14);    |
| %c            | char (caracter)         | printf("%c", 'A');     |
| %s            | char* (cadena)          | printf("%s", "hola");  |
| %p            | void* (puntero)         | printf("%p", ptr);     |
| %x / %X       | int (hexadecimal)       | printf("%x", 255);     |
| %o            | int (octal)             | printf("%o", 255);     |
| %e / %E       | double (cientifica)     | printf("%e", 3.14);    |
| %g / %G       | double (la mas corta)   | printf("%g", 3.14);    |
| %%            | Literal %               | printf("100%%");       |
| %zu           | size_t                  | printf("%zu", sizeof(int)); |

```c
#include <stdio.h>

int main(void) {
    int entero = 42;
    unsigned int positivo = 3000000000U;
    double real = 3.1415926535;
    char letra = 'A';
    char* texto = "Hola, C";
    void* puntero = &entero;

    printf("Entero:        %d
", entero);
    printf("Unsigned:      %u
", positivo);
    printf("Real:          %f
", real);
    printf("Caracter:      %c
", letra);
    printf("Texto:         %s
", texto);
    printf("Puntero:       %p
", puntero);
    printf("Hexadecimal:   %x
", 255);
    printf("Octal:         %o
", 255);
    printf("Cientifica:    %e
", real);
    printf("Compacto:      %g
", real);
    printf("Porcentaje:    100%%
");

    return 0;
}
```

2. FORMATO AVANZADO DE printf
===============================================================================

**2.1 Sintaxis completa del especificador de formato**

```
%[banderas][ancho][.precision][longitud]especificador
```

**2.2 Banderas**

| Bande | Significado                         | Ejemplo        | Resultado |
|-------|-------------------------------------|----------------|-----------|
| -     | Alinea a la izquierda               | %-5d           | "42   "  |
| +     | Muestra el signo siempre            | %+d            | "+42"    |
| espacio| Deja un espacio para valores +     | % d            | " 42"    |
| 0     | Rellena con ceros                   | %05d           | "00042"  |
| #     | Formato alternativo (0x para hex)   | %#x            | "0x2a"   |

```c
#include <stdio.h>

int main(void) {
    int n = 42;
    int neg = -42;

    printf("=== BANDERAS DE FORMATO ===

");

    printf("Sin bandera:    |%d|%d|
", n, neg);
    printf("Alineacion izq: |%-5d|%-5d|
", n, neg);
    printf("Con signo:      |%+d|%+d|
", n, neg);
    printf("Con espacio:    |% d|% d|
", n, neg);
    printf("Relleno ceros:  |%05d|%05d|
", n, neg);
    printf("Hex alterno:    |%#x|%#X|
", n, n);

    return 0;
}
```

**2.3 Ancho y precision**

El **ancho** especifica el numero minimo de caracteres a imprimir.
La **precision** tiene significado diferente segun el tipo:

- Enteros (%d): numero minimo de digitos.
- Flotantes (%f): numero de decimales.
- Cadenas (%s): numero maximo de caracteres.

```c
#include <stdio.h>

int main(void) {
    printf("=== ANCHO ===

");

    printf("Sin ancho:    |%d|
", 42);
    printf("Ancho 5:      |%5d|
", 42);
    printf("Ancho 10:     |%10d|
", 42);
    printf("Ancho 5 izq:  |%-5d|
", 42);

    printf("
=== PRECISION ===

");

    printf("Flotantes:
");
    printf("Sin precision: |%f|
", 3.14159);
    printf("Con .2:        |%.2f|
", 3.14159);
    printf("Con .5:        |%.5f|
", 3.14159);
    printf("Con .10:       |%.10f|
", 3.14159);

    printf("
Enteros:
");
    printf("Sin precision: |%d|
", 42);
    printf("Con .5:        |%.5d|
", 42);

    printf("
Cadenas:
");
    printf("Sin precision: |%s|
", "Hola Mundo");
    printf("Con .4:        |%.4s|
", "Hola Mundo");

    printf("
Ancho + precision:
");
    printf("|%8.2f|
", 3.14159);
    printf("|%-8.2f|
", 3.14159);
    printf("|%08.2f|
", 3.14159);

    return 0;
}
```

**2.4 Especificadores de longitud**

| Prefijo | Tipo                 | Ejemplo               |
|---------|----------------------|-----------------------|
| h       | short                | %hd, %hu              |
| l       | long                 | %ld, %lu, %lf         |
| ll      | long long            | %lld, %llu            |
| L       | long double          | %Lf                   |
| z       | size_t               | %zu                   |
| t       | ptrdiff_t            | %td                   |

```c
#include <stdio.h>

int main(void) {
    short s = 100;
    long l = 100000L;
    long long ll = 10000000000LL;
    unsigned long ul = 100000UL;
    size_t sz = sizeof(int);
    long double ld = 3.141592653589793238L;

    printf("short:        %hd
", s);
    printf("long:         %ld
", l);
    printf("long long:    %lld
", ll);
    printf("unsigned long: %lu
", ul);
    printf("size_t:       %zu
", sz);
    printf("long double:  %.20Lf
", ld);

    return 0;
}
```

3. LA FUNCION scanf
===============================================================================

scanf (scan formatted) lee datos formateados desde la entrada
estandar (generalmente el teclado).

**3.1 Sintaxis basica**

```c
scanf("cadena de formato", &variables...);
```

Importante: las variables deben pasarse con el operador & (direccion),
excepto para cadenas (arrays de char).

```c
#include <stdio.h>

int main(void) {
    int edad;
    char nombre[50];

    printf("Ingresa tu edad: ");
    scanf("%d", &edad);  /* & es obligatorio para variables no array */

    printf("Ingresa tu nombre: ");
    scanf("%49s", nombre);  /* Los arrays NO llevan & */

    printf("Hola %s, tienes %d anios.
", nombre, edad);

    return 0;
}
```

**3.2 Especificadores de formato para scanf**

| Especificador | Tipo de dato leido       |
|---------------|--------------------------|
| %d            | int                      |
| %i            | int (auto: decimal, octal, hex) |
| %u            | unsigned int             |
| %f            | float*                   |
| %lf           | double*                  |
| %Lf           | long double*             |
| %c            | char (lee UN caracter)   |
| %s            | cadena (sin espacios)    |
| %x            | int hexadecimal          |
| %o            | int octal                |

*Los prefijos de longitud (h, l, ll, L) se aplican igual que en printf.

```c
#include <stdio.h>

int main(void) {
    int entero;
    unsigned int positivo;
    float real_f;
    double real_d;
    char letra;
    char palabra[100];

    printf("=== DEMO DE SCANF ===

");

    printf("Ingresa un entero: ");
    scanf("%d", &entero);

    printf("Ingresa un unsigned: ");
    scanf("%u", &positivo);

    printf("Ingresa un float: ");
    scanf("%f", &real_f);

    printf("Ingresa un double: ");
    scanf("%lf", &real_d);

    printf("Ingresa una letra: ");
    scanf(" %c", &letra);  /* Espacio antes de %c para ignorar whitespace */

    printf("Ingresa una palabra (sin espacios): ");
    scanf("%99s", palabra);

    printf("
=== RESULTADOS ===
");
    printf("Entero:     %d
", entero);
    printf("Unsigned:   %u
", positivo);
    printf("Float:      %f
", real_f);
    printf("Double:     %lf
", real_d);
    printf("Letra:      %c
", letra);
    printf("Palabra:    %s
", palabra);

    return 0;
}
```

**3.3 Problemas comunes con scanf**

**Problema: el buffer de entrada**

Cuando usas scanf con %d o %s, el caracter de nueva linea (
)
queda en el buffer. Esto puede causar que el siguiente %c lea
ese 
 en lugar del caracter esperado.

```c
#include <stdio.h>

int main(void) {
    int numero;
    char letra;

    printf("Ingresa un numero: ");
    scanf("%d", &numero);

    /* Solucion: limpia el buffer o pon un espacio antes de %c */
    printf("Ingresa una letra: ");
    scanf(" %c", &letra);  /* El espacio consume el \n residual */

    printf("Numero: %d, Letra: %c
", numero, letra);

    return 0;
}
```

**3.4 Valor de retorno de scanf**

scanf devuelve el numero de elementos que logro leer y asignar
correctamente. Esto es util para verificar errores de entrada.

```c
#include <stdio.h>

int main(void) {
    int a, b;
    int resultado;

    printf("Ingresa dos numeros: ");
    resultado = scanf("%d %d", &a, &b);

    if (resultado == 2) {
        printf("Leiste correctamente: %d y %d
", a, b);
    } else if (resultado == 1) {
        printf("Solo leiste un numero: %d
", a);
    } else {
        printf("No se pudo leer ningun numero.
");
    }

    printf("scanf devolvio: %d
", resultado);

    return 0;
}
```

4. getchar Y putchar
===============================================================================

getchar y putchar son funciones para leer y escribir un caracter
a la vez. Son mas simples que scanf/printf para caracteres.

**4.1 getchar()**

Lee un caracter de la entrada estandar. Devuelve el caracter como
un int (para poder devolver EOF, fin de archivo).

```c
#include <stdio.h>

int main(void) {
    char c;

    printf("Presiona una tecla: ");
    c = getchar();

    printf("Ingresaste: '%c' (codigo ASCII: %d)
", c, c);

    return 0;
}
```

**4.2 putchar()**

Escribe un caracter en la salida estandar.

```c
#include <stdio.h>

int main(void) {
    char letra = 'A';

    printf("Imprimiendo caracteres con putchar:
");

    putchar(letra);
    putchar('
');

    /* Imprime el abecedario */
    for (char c = 'A'; c <= 'Z'; c++) {
        putchar(c);
        putchar(' ');
    }
    putchar('
');

    return 0;
}
```

**4.3 Programa: eco de caracteres**

```c
#include <stdio.h>

int main(void) {
    char c;

    printf("Escribe texto (presiona Enter para salir):
");

    /* Lee caracteres hasta encontrar nueva linea */
    while ((c = getchar()) != '
') {
        putchar(c);  /* Eco del caracter */
    }

    printf("
Fin del programa.
");

    return 0;
}
```

6. ENTRADA DE CADENAS: gets, fgets y scanf
===============================================================================

**6.1 El problema con gets()**

gets() es peligroso porque no verifica el tamano del buffer. Por
eso fue eliminado del estandar C11. **NUNCA USES gets()**.

**6.2 fgets() - la alternativa segura**

```c
char* fgets(char* buffer, int tamano, FILE* stream);
```

Lee una linea completa (incluyendo espacios) y limita la lectura
al tamano del buffer.

```c
#include <stdio.h>
#include <string.h>

int main(void) {
    char nombre[50];
    char linea[100];

    printf("Ingresa tu nombre completo (con fgets): ");
    fgets(nombre, sizeof(nombre), stdin);

    /* fgets incluye el \n final; lo eliminamos si existe */
    size_t len = strlen(nombre);
    if (len > 0 && nombre[len - 1] == '
') {
        nombre[len - 1] = ' ';
    }

    printf("Hola, %s!
", nombre);

    printf("
Ingresa una linea de texto: ");
    fgets(linea, sizeof(linea), stdin);
    printf("Escribiste: %s", linea);

    return 0;
}
```

**6.3 scanf con %[^
] para leer lineas**

```c
#include <stdio.h>

int main(void) {
    char linea[100];

    printf("Ingresa una linea de texto: ");
    scanf(" %[^
]", linea);  /* Lee hasta nueva linea */

    printf("Leiste: %s
", linea);

    return 0;
}
```

**7.1 Calculadora simple con entrada del usuario**

```c
#include <stdio.h>

int main(void) {
    double a, b;
    char operacion;

    printf("=== CALCULADORA BASICA ===
");
    printf("Ingresa: numero operador numero
");
    printf("Ejemplo: 10 + 5
");
    printf("Operadores: + - * /

");

    printf("> ");
    scanf("%lf %c %lf", &a, &operacion, &b);

    switch (operacion) {
        case '+':
            printf("%.2f + %.2f = %.2f
", a, b, a + b);
            break;
        case '-':
            printf("%.2f - %.2f = %.2f
", a, b, a - b);
            break;
        case '*':
            printf("%.2f * %.2f = %.2f
", a, b, a * b);
            break;
        case '/':
            if (b != 0)
                printf("%.2f / %.2f = %.2f
", a, b, a / b);
            else
                printf("Error: division por cero
");
            break;
        default:
            printf("Operacion no valida
");
    }

    return 0;
}
```

**7.3 Contador de caracteres, palabras y lineas**

```c
#include <stdio.h>

int main(void) {
    char c;
    int caracteres = 0;
    int palabras = 0;
    int lineas = 0;
    int en_palabra = 0;

    printf("Ingresa texto (Ctrl+D para terminar):
");

    while ((c = getchar()) != EOF) {
        caracteres++;

        if (c == '
') {
            lineas++;
        }

        if (c == ' ' || c == '
' || c == '	') {
            en_palabra = 0;
        } else if (!en_palabra) {
            en_palabra = 1;
            palabras++;
        }
    }

    printf("
=== ESTADISTICAS ===
");
    printf("Caracteres: %d
", caracteres);
    printf("Palabras:   %d
", palabras);
    printf("Lineas:     %d
", lineas);

    return 0;
}
```

8. REDIRECCION DE ENTRADA/SALIDA
===============================================================================

Una caracteristica poderosa de Unix/Linux es la redireccion.

**8.1 Redireccion de salida (>)**

```bash
./programa > salida.txt   # La salida va al archivo
./programa >> salida.txt  # La salida se anade al archivo
```

**8.2 Redireccion de entrada (<)**

```bash
./programa < entrada.txt   # La entrada viene del archivo
```

**8.3 Tuberias (pipelines)**

```bash
echo "Hola" | ./programa   # La salida de echo es la entrada del programa
```

```c
#include <stdio.h>

int main(void) {
    char nombre[100];

    /* Si hay redireccion, lee del archivo/pipe */
    if (fgets(nombre, sizeof(nombre), stdin) != NULL) {
        printf("Leido desde la entrada: %s", nombre);
    } else {
        printf("No hay entrada disponible.
");
    }

    return 0;
}
```

===============================================================================
PREGUNTAS FRECUENTES
===============================================================================

?** ?Por que scanf no lee cadenas con espacios?**
Porque %s en scanf lee hasta el primer espacio en blanco. Para leer
lineas completas con espacios, usa fgets o scanf(" %[^\n]", buffer).

?** ?Que significa el & en scanf(&variable)?**
El operador & obtiene la direccion de memoria de la variable.
scanf necesita saber donde guardar el dato leido, por lo que requiere
la direccion, no el valor de la variable.

?** ?Que diferencia hay entre %f y %lf en printf?**
En printf, %f y %lf son equivalentes para double porque los argumentos
float se convierten automaticamente a double. En scanf, si son
diferentes: %f para float*, %lf para double*. Esto es una fuente
comun de errores.

?** ?Como limpio el buffer de entrada despues de scanf?**
Puedes usar: while (getchar() != '\n'); para consumir el resto de la
linea. O simplemente poner un espacio en el siguiente scanf: scanf(" %c", &c);

?** ?Que es EOF?**
EOF es "End of File", una constante definida en stdio.h (generalmente
-1). Se usa para indicar que no hay mas datos que leer, ya sea porque
se llego al final de un archivo o porque el usuario presiono Ctrl+D
(Linux) o Ctrl+Z (Windows).
