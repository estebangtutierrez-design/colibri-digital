Bienvenido al capitulo de Estructuras Anidadas.

Hasta ahora has visto estructuras que contienen tipos simples: int, float,
char[]. Pero las estructuras pueden contener otras estructuras como miembros.

Esto te permite modelar jerarquias de datos del mundo real de forma natural.
Un `Empleado` tiene una `Fecha` de nacimiento. Una `Empresa` tiene una
`Direccion`. Un `Pedido` tiene un `Cliente` y una lista de `Producto`.

Las estructuras anidadas son la clave para construir modelos de datos ricos
y expresivos en C.

--- INTRODUCCION ---

Una estructura anidada es simplemente una estructura que tiene como miembro
a otra estructura. En C, esto se logra de dos formas:

1. Declarando la estructura interna por separado y luego usandola como tipo
   de un miembro.
2. Declarando la estructura interna directamente dentro de la externa.

--- FORMA 1: ESTRUCTURA INTERNA POR SEPARADO ---

```c
typedef struct {
    int dia;
    int mes;
    int anio;
} Fecha;

typedef struct {
    char nombre[50];
    Fecha fechaNacimiento;
    float salario;
} Empleado;
```

--- FORMA 2: ESTRUCTURA INTERNA ANONIMA ---

```c
typedef struct {
    char calle[100];
    int numero;
    char ciudad[50];
} Direccion;

typedef struct {
    char nombre[50];
    struct {
        int dia;
        int mes;
        int anio;
    } fechaNacimiento;
    Direccion direccion;
    float salario;
} Empleado;
```

La primera forma es mas reutilizable. La segunda es mas privada.

--- INICIALIZACION DE ESTRUCTURAS ANIDADAS ---

Hay varias formas de inicializar estructuras anidadas:

```c
#include <stdio.h>
#include <string.h>

typedef struct {
    int dia;
    int mes;
    int anio;
} Fecha;

typedef struct {
    char nombre[50];
    Fecha fechaNacimiento;
    float salario;
} Empleado;

int main() {
    // Forma 1: inicializacion completa con llaves anidadas
    Empleado emp1 = {
        "Ana Garcia",
        {15, 3, 1992},
        45000.00
    };

    // Forma 2: inicializacion designada (C99)
    Empleado emp2 = {
        .nombre = "Carlos Lopez",
        .fechaNacimiento = {22, 7, 1988},
        .salario = 52000.00
    };

    // Forma 3: asignacion miembro por miembro
    Empleado emp3;
    strcpy(emp3.nombre, "Elena Martinez");
    emp3.fechaNacimiento.dia = 10;
    emp3.fechaNacimiento.mes = 11;
    emp3.fechaNacimiento.anio = 1995;
    emp3.salario = 38000.00;

    printf("Empleado 1: %s, Nac: %02d/%02d/%04d, $%.2f\n",
           emp1.nombre,
           emp1.fechaNacimiento.dia,
           emp1.fechaNacimiento.mes,
           emp1.fechaNacimiento.anio,
           emp1.salario);

    printf("Empleado 2: %s, Nac: %02d/%02d/%04d, $%.2f\n",
           emp2.nombre,
           emp2.fechaNacimiento.dia,
           emp2.fechaNacimiento.mes,
           emp2.fechaNacimiento.anio,
           emp2.salario);

    printf("Empleado 3: %s, Nac: %02d/%02d/%04d, $%.2f\n",
           emp3.nombre,
           emp3.fechaNacimiento.dia,
           emp3.fechaNacimiento.mes,
           emp3.fechaNacimiento.anio,
           emp3.salario);

    return 0;
}
```

--- ACCESO A MIEMBROS ANIDADOS ---

Para acceder a un miembro de una estructura anidada, simplemente encadena
operadores punto:

    variable_estructura.miembro_anidado.sub_miembro

Ejemplo:

    emp1.fechaNacimiento.dia   = 15;
    emp1.fechaNacimiento.mes   = 3;
    emp1.fechaNacimiento.anio  = 1992;

--- ESTRUCTURAS ANIDADAS CON PUNTEROS ---

Cuando trabajas con punteros a estructuras que tienen miembros anidados,
la sintaxis combina `->` y `.`:

```c
void mostrarEmpleado(const Empleado *emp) {
    printf("Nombre: %s\n", emp->nombre);
    // Con puntero: emp->fechaNacimiento.dia
    printf("Nacimiento: %02d/%02d/%04d\n",
           emp->fechaNacimiento.dia,
           emp->fechaNacimiento.mes,
           emp->fechaNacimiento.anio);
    printf("Salario: $%.2f\n", emp->salario);
}
```

--- ARRAYS DE ESTRUCTURAS CON MIEMBROS ANIDADOS ---

```c
#include <stdio.h>

typedef struct {
    float x;
    float y;
} Punto;

typedef struct {
    Punto superiorIzquierdo;
    Punto inferiorDerecho;
} Rectangulo;

int main() {
    Rectangulo rects[] = {
        {{0, 0}, {10, 10}},
        {{5, 5}, {15, 20}},
        {{-3, -1}, {2, 4}}
    };

    int n = sizeof(rects) / sizeof(rects[0]);

    for (int i = 0; i < n; i++) {
        float ancho = rects[i].inferiorDerecho.x - rects[i].superiorIzquierdo.x;
        float alto = rects[i].inferiorDerecho.y - rects[i].superiorIzquierdo.y;
        float area = ancho * alto;

        printf("Rectangulo %d: (%.1f,%.1f)-(%.1f,%.1f) -> area = %.1f\n",
               i + 1,
               rects[i].superiorIzquierdo.x,
               rects[i].superiorIzquierdo.y,
               rects[i].inferiorDerecho.x,
               rects[i].inferiorDerecho.y,
               area);
    }

    return 0;
}
```

--- ESTRUCTURAS ANIDADAS CON ARRAYS INTERNOS ---

Un caso muy comun es tener arrays de estructuras dentro de otra estructura:

```c
#include <stdio.h>
#include <string.h>

#define MAX_MATERIAS 10

typedef struct {
    char nombre[50];
    float calificacion;
} Materia;

typedef struct {
    char nombre[60];
    int numMaterias;
    Materia materias[MAX_MATERIAS];
    float promedio;
} Estudiante;

void calcularPromedio(Estudiante *est) {
    float suma = 0;
    for (int i = 0; i < est->numMaterias; i++) {
        suma += est->materias[i].calificacion;
    }
    est->promedio = (est->numMaterias > 0) ? suma / est->numMaterias : 0;
}

void mostrarEstudiante(const Estudiante *est) {
    printf("Estudiante: %s\n", est->nombre);
    printf("Materias:\n");
    for (int i = 0; i < est->numMaterias; i++) {
        printf("  %d. %s: %.1f\n",
               i + 1,
               est->materias[i].nombre,
               est->materias[i].calificacion);
    }
    printf("Promedio: %.2f\n", est->promedio);
    printf("\n");
}

int main() {
    Estudiante grupo[2];

    // Estudiante 1
    strcpy(grupo[0].nombre, "Luis Hernandez");
    grupo[0].numMaterias = 4;
    strcpy(grupo[0].materias[0].nombre, "Matematicas");
    grupo[0].materias[0].calificacion = 9.0;
    strcpy(grupo[0].materias[1].nombre, "Fisica");
    grupo[0].materias[1].calificacion = 8.5;
    strcpy(grupo[0].materias[2].nombre, "Programacion");
    grupo[0].materias[2].calificacion = 9.5;
    strcpy(grupo[0].materias[3].nombre, "Ingles");
    grupo[0].materias[3].calificacion = 8.0;
    calcularPromedio(&grupo[0]);

    // Estudiante 2
    strcpy(grupo[1].nombre, "Maria Torres");
    grupo[1].numMaterias = 3;
    strcpy(grupo[1].materias[0].nombre, "Historia");
    grupo[1].materias[0].calificacion = 7.5;
    strcpy(grupo[1].materias[1].nombre, "Literatura");
    grupo[1].materias[1].calificacion = 9.0;
    strcpy(grupo[1].materias[2].nombre, "Arte");
    grupo[1].materias[2].calificacion = 8.5;
    calcularPromedio(&grupo[1]);

    for (int i = 0; i < 2; i++) {
        mostrarEstudiante(&grupo[i]);
    }

    return 0;
}
```

--- AUTO-REFERENCIA: ESTRUCTURAS RECURSIVAS ---

Una estructura puede contener un puntero a su propio tipo. Esto es la base
de las listas enlazadas, arboles y grafos:

```c
typedef struct Nodo {
    int dato;
    struct Nodo *siguiente;  // Auto-referencia
} Nodo;
```

Nota: necesitas usar `struct Nodo` dentro de la definicion porque el
`typedef` aun no esta completo.

Ejemplo de lista enlazada simple:

```c
#include <stdio.h>
#include <stdlib.h>

typedef struct Nodo {
    int dato;
    struct Nodo *siguiente;
} Nodo;

void imprimirLista(const Nodo *cabeza) {
    const Nodo *actual = cabeza;
    while (actual != NULL) {
        printf("%d", actual->dato);
        if (actual->siguiente != NULL) printf(" -> ");
        actual = actual->siguiente;
    }
    printf("\n");
}

int main() {
    Nodo *cabeza = NULL;
    Nodo *segundo = NULL;
    Nodo *tercero = NULL;

    cabeza = malloc(sizeof(Nodo));
    segundo = malloc(sizeof(Nodo));
    tercero = malloc(sizeof(Nodo));

    cabeza->dato = 10;
    cabeza->siguiente = segundo;

    segundo->dato = 20;
    segundo->siguiente = tercero;

    tercero->dato = 30;
    tercero->siguiente = NULL;

    imprimirLista(cabeza);

    free(cabeza);
    free(segundo);
    free(tercero);

    return 0;
}
```

--- EJEMPLO COMPLETO: SISTEMA DE PEDIDOS ---

```c
#include <stdio.h>
#include <string.h>

typedef struct {
    int dia;
    int mes;
    int anio;
} Fecha;

typedef struct {
    char calle[80];
    int numero;
    char ciudad[40];
    char estado[30];
} Direccion;

typedef struct {
    char nombre[50];
    float precio;
    int stock;
} Producto;

typedef struct {
    int id;
    char cliente[60];
    Direccion direccionEnvio;
    Fecha fechaPedido;
    Fecha fechaEntrega;
    Producto productos[5];
    int numProductos;
    float total;
} Pedido;

float calcularTotal(Pedido *p) {
    float suma = 0;
    for (int i = 0; i < p->numProductos; i++) {
        suma += p->productos[i].precio;
    }
    return suma;
}

void mostrarPedido(const Pedido *p) {
    printf("=== PEDIDO #%d ===\n", p->id);
    printf("Cliente: %s\n", p->cliente);
    printf("Direccion: %s #%d, %s, %s\n",
           p->direccionEnvio.calle,
           p->direccionEnvio.numero,
           p->direccionEnvio.ciudad,
           p->direccionEnvio.estado);
    printf("Fecha pedido: %02d/%02d/%04d\n",
           p->fechaPedido.dia, p->fechaPedido.mes, p->fechaPedido.anio);
    printf("Fecha entrega: %02d/%02d/%04d\n",
           p->fechaEntrega.dia, p->fechaEntrega.mes, p->fechaEntrega.anio);
    printf("Productos (%d):\n", p->numProductos);
    for (int i = 0; i < p->numProductos; i++) {
        printf("  %d. %s - $%.2f\n",
               i + 1,
               p->productos[i].nombre,
               p->productos[i].precio);
    }
    printf("TOTAL: $%.2f\n\n", p->total);
}

int main() {
    Pedido pedidos[2] = {
        {
            1001,
            "Juan Perez",
            {"Av. Reforma", 123, "CDMX", "CDMX"},
            {10, 1, 2025},
            {15, 1, 2025},
            {
                {"Laptop", 15000.00, 0},
                {"Mouse", 850.50, 0},
                {"Teclado", 1200.00, 0}
            },
            3,
            0.0f
        },
        {
            1002,
            "Sofia Ramirez",
            {"Callejon del Beso", 45, "Queretaro", "QRO"},
            {12, 1, 2025},
            {18, 1, 2025},
            {
                {"Monitor", 8500.00, 0},
                {"Webcam", 1200.00, 0}
            },
            2,
            0.0f
        }
    };

    for (int i = 0; i < 2; i++) {
        pedidos[i].total = calcularTotal(&pedidos[i]);
        mostrarPedido(&pedidos[i]);
    }

    return 0;
}
```

--- CONSIDERACIONES DE MEMORIA ---

Cuando anidas estructuras, el tamaño total es la suma de los tamaños de
todos los miembros, incluyendo el padding de cada nivel:

```c
typedef struct { char c; int i; } Interna;    // 8 bytes (con padding)
typedef struct { Interna in; double d; } Externa; // 8 + 8 = 16 bytes
```

El compilador alinea cada estructura segun su miembro mas restrictivo.

═══════════════════════════════════════════════════════════════════════════════
                    PREGUNTAS FRECUENTES (FAQ)
═══════════════════════════════════════════════════════════════════════════════

P: ¿Puedo tener una estructura que se contenga a si misma?
R: No directamente (seria infinito). Pero puedes tener un puntero al mismo
   tipo de estructura (auto-referencia), como en listas enlazadas.

P: ¿Cuantos niveles de anidamiento puedo tener?
R: No hay limite teorico, pero practicamente se recomienda no exceder 3-4
   niveles para mantener la legibilidad.

P: ¿El padding se acumula en estructuras anidadas?
R: Si. Cada estructura tiene su propio padding interno, y la estructura
   contenedora puede agregar padding adicional para alinear a la interna.

P: ¿Puedo inicializar una estructura anidada con {0}?
R: Si. `= {0}` pone a cero todos los miembros, incluyendo las estructuras
   anidadas y sus sub-miembros.

P: ¿Es mejor usar punteros a structs anidadas o incluirlas directamente?
R: Incluirlas directamente es mas eficiente (contiguo en memoria). Usa
   punteros cuando necesites comparticion de memoria o estructuras opcionales.

P: ¿Puedo tener un array de estructuras dentro de una estructura?
R: Si, es muy comun. Por ejemplo, un `Pedido` con un array de `Producto`.
