BIENVENIDA DEL PROFESOR BÚHO
===============================================================================

¡Bienvenido, constructor de estructuras de datos! Has llegado al capítulo
donde la memoria dinámica cobra vida en su forma más elegante: las listas
enlazadas.

Una lista enlazada es una estructura de datos lineal compuesta por nodos,
donde cada nodo contiene un dato y un puntero al siguiente nodo. A
diferencia de los arrays, las listas enlazadas crecen y se encogen
dinámicamente sin necesidad de redimensionar ni copiar datos.

Dominarás todas las operaciones fundamentales: inserción al inicio,
al final y en posición arbitraria, eliminación, búsqueda, recorrido
y, por supuesto, liberación completa de la memoria.

Las listas enlazadas son la puerta de entrada a estructuras más complejas
como pilas, colas, árboles y grafos. ¡Prepárate para construir los
cimientos de tu futuro como científico de la computación!

===============================================================================
1. ESTRUCTURA DEL NODO
===============================================================================

El bloque fundamental de una lista enlazada es el nodo. Cada nodo
contiene:

  - dato: el valor que queremos almacenar (puede ser cualquier tipo)
  - siguiente: puntero al próximo nodo (NULL si es el último)

```c
#include <stdio.h>
#include <stdlib.h>

// Definición del nodo para lista de enteros
typedef struct Nodo {
    int dato;
    struct Nodo *siguiente;
} Nodo;

// La lista en sí es solo un puntero al primer nodo
typedef Nodo *Lista;

int main() {
    // Crear nodos individuales
    Nodo *n1 = (Nodo *)malloc(sizeof(Nodo));
    Nodo *n2 = (Nodo *)malloc(sizeof(Nodo));
    Nodo *n3 = (Nodo *)malloc(sizeof(Nodo));

    if (!n1 || !n2 || !n3) return 1;

    n1->dato = 10;
    n1->siguiente = n2;

    n2->dato = 20;
    n2->siguiente = n3;

    n3->dato = 30;
    n3->siguiente = NULL;  // Último nodo

    // Lista: 10 -> 20 -> 30 -> NULL
    Lista lista = n1;

    // Recorrer la lista
    printf("Lista: ");
    Nodo *actual = lista;
    while (actual != NULL) {
        printf("%d -> ", actual->dato);
        actual = actual->siguiente;
    }
    printf("NULL\n");

    // Liberar nodos individualmente
    free(n1);
    free(n2);
    free(n3);

    return 0;
}
```

El patrón general de cualquier lista enlazada se basa en:

  - Lista vacía: el puntero cabeza es NULL.
  - Lista con elementos: cabeza apunta al primer nodo.
  - Cada nodo apunta al siguiente, el último apunta a NULL.
  - Recorrido: empezar por cabeza y seguir siguientes hasta NULL.

===============================================================================
2. FUNCIONES BÁSICAS DE LISTA ENLAZADA
===============================================================================

Vamos a construir una biblioteca completa de lista enlazada simple.

```c
#include <stdio.h>
#include <stdlib.h>

typedef struct Nodo {
    int dato;
    struct Nodo *siguiente;
} Nodo;

typedef Nodo *Lista;

// 2.1. Crear un nuevo nodo
Nodo *nodo_crear(int valor) {
    Nodo *nuevo = (Nodo *)malloc(sizeof(Nodo));
    if (nuevo == NULL) return NULL;

    nuevo->dato = valor;
    nuevo->siguiente = NULL;
    return nuevo;
}

// 2.2. Insertar al inicio (push_front)
void lista_insertar_inicio(Lista *lista, int valor) {
    Nodo *nuevo = nodo_crear(valor);
    if (nuevo == NULL) return;

    nuevo->siguiente = *lista;  // El nuevo apunta al antiguo primer nodo
    *lista = nuevo;             // La cabeza ahora es el nuevo nodo
}

// 2.3. Insertar al final (push_back)
void lista_insertar_final(Lista *lista, int valor) {
    Nodo *nuevo = nodo_crear(valor);
    if (nuevo == NULL) return;

    if (*lista == NULL) {
        // Lista vacía: el nuevo es el primer nodo
        *lista = nuevo;
        return;
    }

    // Recorrer hasta el último nodo
    Nodo *actual = *lista;
    while (actual->siguiente != NULL) {
        actual = actual->siguiente;
    }
    actual->siguiente = nuevo;  // El último apunta al nuevo
}

// 2.4. Imprimir la lista
void lista_imprimir(Lista lista) {
    printf("Lista: ");
    Nodo *actual = lista;
    while (actual != NULL) {
        printf("%d -> ", actual->dato);
        actual = actual->siguiente;
    }
    printf("NULL\n");
}

// 2.5. Contar elementos
int lista_longitud(Lista lista) {
    int count = 0;
    Nodo *actual = lista;
    while (actual != NULL) {
        count++;
        actual = actual->siguiente;
    }
    return count;
}

// 2.6. Liberar toda la lista
void lista_liberar(Lista *lista) {
    Nodo *actual = *lista;
    while (actual != NULL) {
        Nodo *temp = actual;
        actual = actual->siguiente;
        free(temp);  // Liberar el nodo actual
    }
    *lista = NULL;  // La cabeza ahora es NULL
}

int main() {
    Lista lista = NULL;  // Lista vacía

    printf("Longitud inicial: %d\n", lista_longitud(lista));

    lista_insertar_inicio(&lista, 30);
    lista_insertar_inicio(&lista, 20);
    lista_insertar_inicio(&lista, 10);
    printf("Después de insertar 10, 20, 30 al inicio:\n");
    lista_imprimir(lista);
    printf("Longitud: %d\n\n", lista_longitud(lista));

    lista_insertar_final(&lista, 40);
    lista_insertar_final(&lista, 50);
    printf("Después de insertar 40, 50 al final:\n");
    lista_imprimir(lista);
    printf("Longitud: %d\n", lista_longitud(lista));

    lista_liberar(&lista);
    printf("Lista liberada. Cabeza: %p\n", (void*)lista);

    return 0;
}
```

Obsérvese que todas las funciones que modifican la lista reciben
Lista *lista (puntero a puntero). Esto es necesario para poder
modificar la cabeza de la lista desde la función.

3. INSERCIÓN EN POSICIÓN ARBITRARIA
===============================================================================

Podemos insertar un nodo en cualquier posición de la lista.

```c
#include <stdio.h>
#include <stdlib.h>

typedef struct Nodo {
    int dato;
    struct Nodo *siguiente;
} Nodo;

typedef Nodo *Lista;

Nodo *nodo_crear(int valor) {
    Nodo *nuevo = (Nodo *)malloc(sizeof(Nodo));
    if (nuevo) {
        nuevo->dato = valor;
        nuevo->siguiente = NULL;
    }
    return nuevo;
}

// Insertar en una posición específica (0 = inicio)
int lista_insertar_en(Lista *lista, int valor, int posicion) {
    if (posicion < 0) return -1;

    if (posicion == 0) {
        // Insertar al inicio
        Nodo *nuevo = nodo_crear(valor);
        if (nuevo == NULL) return -1;
        nuevo->siguiente = *lista;
        *lista = nuevo;
        return 0;
    }

    // Buscar la posición (necesitamos el nodo anterior)
    Nodo *anterior = *lista;
    int i = 0;
    while (anterior != NULL && i < posicion - 1) {
        anterior = anterior->siguiente;
        i++;
    }

    if (anterior == NULL) return -1;  // Posición fuera de rango

    Nodo *nuevo = nodo_crear(valor);
    if (nuevo == NULL) return -1;

    nuevo->siguiente = anterior->siguiente;
    anterior->siguiente = nuevo;
    return 0;
}

void lista_imprimir(Lista lista) {
    printf("[");
    Nodo *actual = lista;
    while (actual != NULL) {
        printf("%d", actual->dato);
        if (actual->siguiente != NULL) printf(" -> ");
        actual = actual->siguiente;
    }
    printf("]\n");
}

void lista_liberar(Lista *lista) {
    Nodo *actual = *lista;
    while (actual != NULL) {
        Nodo *temp = actual;
        actual = actual->siguiente;
        free(temp);
    }
    *lista = NULL;
}

int main() {
    Lista lista = NULL;

    // Construir lista: [10 -> 20 -> 30]
    lista_insertar_en(&lista, 10, 0);
    lista_insertar_en(&lista, 30, 1);
    lista_insertar_en(&lista, 20, 1);  // Insertar 20 en posición 1
    printf("Lista inicial: ");
    lista_imprimir(lista);

    // Insertar al inicio (posición 0)
    lista_insertar_en(&lista, 5, 0);
    printf("Después de insertar 5 en posición 0: ");
    lista_imprimir(lista);

    // Insertar al final
    lista_insertar_en(&lista, 99, lista_longitud(lista));
    printf("Después de insertar 99 al final: ");
    lista_imprimir(lista);

    lista_liberar(&lista);
    return 0;
}
```

Nota importante: el caso de insertar en posición 0 es especial porque
modifica la cabeza. El resto de posiciones solo modifican el puntero
siguiente del nodo anterior.

Si la posición es igual a la longitud de la lista, insertamos al final.
Si es mayor, la función devuelve error.

4. ELIMINACIÓN DE NODOS
===============================================================================

4.1. Eliminar el primer nodo

```c
#include <stdio.h>
#include <stdlib.h>

typedef struct Nodo {
    int dato;
    struct Nodo *siguiente;
} Nodo;

typedef Nodo *Lista;

// Resto de funciones básicas (nodo_crear, insertar_inicio, etc.)

int lista_eliminar_inicio(Lista *lista) {
    if (*lista == NULL) return -1;  // Lista vacía

    Nodo *temp = *lista;
    int valor = temp->dato;
    *lista = (*lista)->siguiente;  // La cabeza ahora es el segundo nodo
    free(temp);                    // Liberar el nodo eliminado

    return valor;  // Devolver el valor eliminado
}

int lista_eliminar_final(Lista *lista) {
    if (*lista == NULL) return -1;  // Lista vacía

    Nodo *actual = *lista;

    // Si solo hay un nodo
    if (actual->siguiente == NULL) {
        int valor = actual->dato;
        free(actual);
        *lista = NULL;
        return valor;
    }

    // Recorrer hasta el penúltimo nodo
    while (actual->siguiente->siguiente != NULL) {
        actual = actual->siguiente;
    }

    int valor = actual->siguiente->dato;
    free(actual->siguiente);
    actual->siguiente = NULL;

    return valor;
}

int lista_eliminar_por_valor(Lista *lista, int valor) {
    if (*lista == NULL) return -1;

    Nodo *actual = *lista;

    // Si el valor está en el primer nodo
    if (actual->dato == valor) {
        *lista = actual->siguiente;
        free(actual);
        return 0;
    }

    // Buscar el nodo anterior al que queremos eliminar
    while (actual->siguiente != NULL && actual->siguiente->dato != valor) {
        actual = actual->siguiente;
    }

    if (actual->siguiente == NULL) return -1;  // Valor no encontrado

    Nodo *temp = actual->siguiente;
    actual->siguiente = temp->siguiente;
    free(temp);
    return 0;
}

int lista_eliminar_en(Lista *lista, int posicion) {
    if (*lista == NULL || posicion < 0) return -1;

    if (posicion == 0) {
        Nodo *temp = *lista;
        *lista = (*lista)->siguiente;
        int valor = temp->dato;
        free(temp);
        return valor;
    }

    Nodo *anterior = *lista;
    int i = 0;
    while (anterior != NULL && i < posicion - 1) {
        anterior = anterior->siguiente;
        i++;
    }

    if (anterior == NULL || anterior->siguiente == NULL) return -1;

    Nodo *temp = anterior->siguiente;
    anterior->siguiente = temp->siguiente;
    int valor = temp->dato;
    free(temp);
    return valor;
}
```

La eliminación es donde los punteros dobles demuestran su valor:
al eliminar el primer nodo, necesitamos modificar la cabeza de la
lista, lo que requiere un puntero a puntero.

```c
void demo_eliminacion() {
    Lista lista = NULL;

    // Insertar algunos valores
    for (int i = 1; i <= 5; i++) {
        lista_insertar_final(&lista, i * 10);
    }
    printf("Lista original: ");
    lista_imprimir(lista);

    int eliminado = lista_eliminar_inicio(&lista);
    printf("Eliminado del inicio: %d -> ", eliminado);
    lista_imprimir(lista);

    eliminado = lista_eliminar_final(&lista);
    printf("Eliminado del final: %d -> ", eliminado);
    lista_imprimir(lista);

    lista_eliminar_por_valor(&lista, 30);
    printf("Eliminado 30 -> ");
    lista_imprimir(lista);

    eliminado = lista_eliminar_en(&lista, 1);
    printf("Eliminado en posición 1: %d -> ", eliminado);
    lista_imprimir(lista);

    lista_liberar(&lista);
}
```

5. BÚSQUEDA EN LISTA ENLAZADA
===============================================================================

```c
#include <stdio.h>
#include <stdlib.h>

typedef struct Nodo {
    int dato;
    struct Nodo *siguiente;
} Nodo;

typedef Nodo *Lista;

// Búsqueda lineal: devuelve el nodo o NULL
Nodo *lista_buscar(Lista lista, int valor) {
    Nodo *actual = lista;
    while (actual != NULL) {
        if (actual->dato == valor) {
            return actual;  // Encontrado
        }
        actual = actual->siguiente;
    }
    return NULL;  // No encontrado
}

// Búsqueda con índice: devuelve la posición o -1
int lista_buscar_indice(Lista lista, int valor) {
    Nodo *actual = lista;
    int indice = 0;
    while (actual != NULL) {
        if (actual->dato == valor) {
            return indice;
        }
        actual = actual->siguiente;
        indice++;
    }
    return -1;
}

// Contar ocurrencias de un valor
int lista_contar(Lista lista, int valor) {
    int count = 0;
    Nodo *actual = lista;
    while (actual != NULL) {
        if (actual->dato == valor) count++;
        actual = actual->siguiente;
    }
    return count;
}

// Obtener el valor en una posición (sin eliminar)
int lista_obtener(Lista lista, int posicion) {
    Nodo *actual = lista;
    int i = 0;
    while (actual != NULL) {
        if (i == posicion) return actual->dato;
        actual = actual->siguiente;
        i++;
    }
    // Error: posición fuera de rango
    fprintf(stderr, "Error: posición %d fuera de rango\n", posicion);
    exit(1);
}

// Modificar un valor en una posición
int lista_modificar(Lista lista, int posicion, int nuevo_valor) {
    Nodo *actual = lista;
    int i = 0;
    while (actual != NULL) {
        if (i == posicion) {
            int viejo = actual->dato;
            actual->dato = nuevo_valor;
            return viejo;
        }
        actual = actual->siguiente;
        i++;
    }
    return -1;  // No encontrado
}

int main() {
    Lista lista = NULL;

    for (int i = 0; i < 10; i++) {
        lista_insertar_final(&lista, i * 10);
    }
    // También añadir un duplicado
    lista_insertar_final(&lista, 30);

    lista_imprimir(lista);

    int buscar = 50;
    Nodo *encontrado = lista_buscar(lista, buscar);
    printf("¿%d está en la lista? %s\n",
           buscar, encontrado ? "Sí" : "No");

    int indice = lista_buscar_indice(lista, 30);
    printf("Primera ocurrencia de 30 en índice: %d\n", indice);

    printf("El 30 aparece %d veces\n", lista_contar(lista, 30));

    printf("Elemento en posición 3: %d\n", lista_obtener(lista, 3));

    lista_modificar(lista, 4, 999);
    printf("Después de modificar posición 4 a 999: ");
    lista_imprimir(lista);

    lista_liberar(&lista);
    return 0;
}
```

6. LISTA CON TIPO GENÉRICO (void*)
===============================================================================

Podemos almacenar cualquier tipo de dato usando void*.

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

typedef struct Nodo {
    void *dato;
    struct Nodo *siguiente;
} Nodo;

typedef struct {
    Nodo *cabeza;
    size_t tam_dato;  // Tamaño del tipo almacenado
} ListaGenerica;

void lista_gen_init(ListaGenerica *lista, size_t tam_dato) {
    lista->cabeza = NULL;
    lista->tam_dato = tam_dato;
}

int lista_gen_insertar_inicio(ListaGenerica *lista, void *valor) {
    Nodo *nuevo = (Nodo *)malloc(sizeof(Nodo));
    if (nuevo == NULL) return -1;

    nuevo->dato = malloc(lista->tam_dato);
    if (nuevo->dato == NULL) {
        free(nuevo);
        return -1;
    }

    memcpy(nuevo->dato, valor, lista->tam_dato);
    nuevo->siguiente = lista->cabeza;
    lista->cabeza = nuevo;
    return 0;
}

void lista_gen_imprimir_int(ListaGenerica *lista) {
    Nodo *actual = lista->cabeza;
    printf("[");
    while (actual != NULL) {
        printf("%d", *(int *)actual->dato);
        if (actual->siguiente != NULL) printf(" -> ");
        actual = actual->siguiente;
    }
    printf("]\n");
}

void lista_gen_imprimir_str(ListaGenerica *lista) {
    Nodo *actual = lista->cabeza;
    printf("[");
    while (actual != NULL) {
        printf("\"%s\"", (char *)actual->dato);
        if (actual->siguiente != NULL) printf(" -> ");
        actual = actual->siguiente;
    }
    printf("]\n");
}

void lista_gen_liberar(ListaGenerica *lista) {
    Nodo *actual = lista->cabeza;
    while (actual != NULL) {
        Nodo *temp = actual;
        actual = actual->siguiente;
        free(temp->dato);  // Liberar el dato copiado
        free(temp);        // Liberar el nodo
    }
    lista->cabeza = NULL;
}

int main() {
    // Lista de enteros
    ListaGenerica list_int;
    lista_gen_init(&list_int, sizeof(int));

    int vals[] = {10, 20, 30, 40, 50};
    for (int i = 0; i < 5; i++) {
        lista_gen_insertar_inicio(&list_int, &vals[i]);
    }

    printf("Lista genérica de enteros: ");
    lista_gen_imprimir_int(&list_int);
    lista_gen_liberar(&list_int);

    // Lista de strings
    ListaGenerica list_str;
    lista_gen_init(&list_str, 32);  // Strings de hasta 32 chars

    char str1[] = "Hola";
    char str2[] = "Mundo";
    char str3[] = "Lista";
    lista_gen_insertar_inicio(&list_str, str1);
    lista_gen_insertar_inicio(&list_str, str2);
    lista_gen_insertar_inicio(&list_str, str3);

    printf("Lista genérica de strings: ");
    lista_gen_imprimir_str(&list_str);
    lista_gen_liberar(&list_str);

    return 0;
}
```

7. PILA Y COLA CON LISTAS ENLAZADAS
===============================================================================

Las listas enlazadas son la base natural para implementar pilas y colas.

7.1. Pila (LIFO) con lista enlazada

```c
#include <stdio.h>
#include <stdlib.h>

typedef struct Nodo {
    int dato;
    struct Nodo *siguiente;
} Nodo;

typedef struct {
    Nodo *tope;
} Pila;

void pila_init(Pila *p) {
    p->tope = NULL;
}

void pila_apilar(Pila *p, int valor) {
    Nodo *nuevo = (Nodo *)malloc(sizeof(Nodo));
    if (nuevo == NULL) return;
    nuevo->dato = valor;
    nuevo->siguiente = p->tope;
    p->tope = nuevo;
}

int pila_desapilar(Pila *p) {
    if (p->tope == NULL) {
        fprintf(stderr, "Pila vacía\n");
        exit(1);
    }
    Nodo *temp = p->tope;
    int valor = temp->dato;
    p->tope = temp->siguiente;
    free(temp);
    return valor;
}

int pila_cima(Pila *p) {
    if (p->tope == NULL) {
        fprintf(stderr, "Pila vacía\n");
        exit(1);
    }
    return p->tope->dato;
}

int pila_vacia(Pila *p) {
    return p->tope == NULL;
}

void pila_liberar(Pila *p) {
    Nodo *actual = p->tope;
    while (actual != NULL) {
        Nodo *temp = actual;
        actual = actual->siguiente;
        free(temp);
    }
    p->tope = NULL;
}

int main() {
    Pila p;
    pila_init(&p);

    pila_apilar(&p, 10);
    pila_apilar(&p, 20);
    pila_apilar(&p, 30);

    printf("Cima: %d\n", pila_cima(&p));

    while (!pila_vacia(&p)) {
        printf("Desapilando: %d\n", pila_desapilar(&p));
    }

    pila_liberar(&p);
    return 0;
}
```

7.2. Cola (FIFO) con lista enlazada

```c
#include <stdio.h>
#include <stdlib.h>

typedef struct Nodo {
    int dato;
    struct Nodo *siguiente;
} Nodo;

typedef struct {
    Nodo *frente;
    Nodo *final;
} Cola;

void cola_init(Cola *q) {
    q->frente = NULL;
    q->final = NULL;
}

void cola_encolar(Cola *q, int valor) {
    Nodo *nuevo = (Nodo *)malloc(sizeof(Nodo));
    if (nuevo == NULL) return;
    nuevo->dato = valor;
    nuevo->siguiente = NULL;

    if (q->final == NULL) {
        q->frente = nuevo;
        q->final = nuevo;
    } else {
        q->final->siguiente = nuevo;
        q->final = nuevo;
    }
}

int cola_desencolar(Cola *q) {
    if (q->frente == NULL) {
        fprintf(stderr, "Cola vacía\n");
        exit(1);
    }
    Nodo *temp = q->frente;
    int valor = temp->dato;
    q->frente = temp->siguiente;
    if (q->frente == NULL) {
        q->final = NULL;
    }
    free(temp);
    return valor;
}

int cola_vacia(Cola *q) {
    return q->frente == NULL;
}

void cola_liberar(Cola *q) {
    Nodo *actual = q->frente;
    while (actual != NULL) {
        Nodo *temp = actual;
        actual = actual->siguiente;
        free(temp);
    }
    q->frente = NULL;
    q->final = NULL;
}

int main() {
    Cola q;
    cola_init(&q);

    cola_encolar(&q, 10);
    cola_encolar(&q, 20);
    cola_encolar(&q, 30);

    while (!cola_vacia(&q)) {
        printf("Desencolando: %d\n", cola_desencolar(&q));
    }

    cola_liberar(&q);
    return 0;
}
```

8. LISTA ORDENADA
===============================================================================

Una variante útil es mantener la lista siempre ordenada.

```c
#include <stdio.h>
#include <stdlib.h>

typedef struct Nodo {
    int dato;
    struct Nodo *siguiente;
} Nodo;

typedef Nodo *Lista;

void lista_insertar_ordenado(Lista *lista, int valor) {
    Nodo *nuevo = (Nodo *)malloc(sizeof(Nodo));
    if (nuevo == NULL) return;
    nuevo->dato = valor;

    // Insertar al inicio si la lista está vacía o el valor es menor
    if (*lista == NULL || valor < (*lista)->dato) {
        nuevo->siguiente = *lista;
        *lista = nuevo;
        return;
    }

    // Buscar la posición correcta
    Nodo *actual = *lista;
    while (actual->siguiente != NULL && actual->siguiente->dato < valor) {
        actual = actual->siguiente;
    }

    nuevo->siguiente = actual->siguiente;
    actual->siguiente = nuevo;
}

void lista_imprimir(Lista lista) {
    printf("Lista ordenada: ");
    Nodo *actual = lista;
    while (actual != NULL) {
        printf("%d -> ", actual->dato);
        actual = actual->siguiente;
    }
    printf("NULL\n");
}

void lista_liberar(Lista *lista) {
    Nodo *actual = *lista;
    while (actual != NULL) {
        Nodo *temp = actual;
        actual = actual->siguiente;
        free(temp);
    }
    *lista = NULL;
}

int main() {
    Lista lista = NULL;

    int valores[] = {30, 10, 50, 20, 40, 5, 35};
    int n = sizeof(valores) / sizeof(valores[0]);

    printf("Insertando: ");
    for (int i = 0; i < n; i++) {
        printf("%d ", valores[i]);
        lista_insertar_ordenado(&lista, valores[i]);
    }
    printf("\n");

    lista_imprimir(lista);
    lista_liberar(&lista);

    return 0;
}
```

9. INVERSIÓN DE LISTA ENLAZADA
===============================================================================

Invertir una lista enlazada es un problema clásico de entrevistas.

```c
#include <stdio.h>
#include <stdlib.h>

typedef struct Nodo {
    int dato;
    struct Nodo *siguiente;
} Nodo;

typedef Nodo *Lista;

void lista_invertir(Lista *lista) {
    Nodo *anterior = NULL;
    Nodo *actual = *lista;
    Nodo *siguiente = NULL;

    while (actual != NULL) {
        siguiente = actual->siguiente;  // Guardar el siguiente
        actual->siguiente = anterior;   // Invertir el puntero
        anterior = actual;              // Avanzar anterior
        actual = siguiente;             // Avanzar actual
    }

    *lista = anterior;  // La nueva cabeza es el último nodo
}

Lista lista_invertir_recursiva(Lista nodo) {
    if (nodo == NULL || nodo->siguiente == NULL) {
        return nodo;  // Caso base
    }

    Lista nueva_cabeza = lista_invertir_recursiva(nodo->siguiente);
    nodo->siguiente->siguiente = nodo;
    nodo->siguiente = NULL;

    return nueva_cabeza;
}

int main() {
    Lista lista = NULL;

    for (int i = 1; i <= 6; i++) {
        lista_insertar_final(&lista, i * 10);
    }

    printf("Original: ");
    lista_imprimir(lista);

    lista_invertir(&lista);
    printf("Invertida (iterativa): ");
    lista_imprimir(lista);

    lista = lista_invertir_recursiva(lista);
    printf("Invertida (recursiva): ");
    lista_imprimir(lista);

    lista_liberar(&lista);
    return 0;
}
```

10. LISTA CIRCULAR
===============================================================================

Una lista circular tiene el último nodo apuntando al primero.

```c
#include <stdio.h>
#include <stdlib.h>

typedef struct Nodo {
    int dato;
    struct Nodo *siguiente;
} Nodo;

typedef struct {
    Nodo *cabeza;
    int count;
} ListaCircular;

void lc_init(ListaCircular *lc) {
    lc->cabeza = NULL;
    lc->count = 0;
}

void lc_insertar(ListaCircular *lc, int valor) {
    Nodo *nuevo = (Nodo *)malloc(sizeof(Nodo));
    if (nuevo == NULL) return;
    nuevo->dato = valor;

    if (lc->cabeza == NULL) {
        lc->cabeza = nuevo;
        nuevo->siguiente = nuevo;  // Apunta a sí mismo
    } else {
        nuevo->siguiente = lc->cabeza->siguiente;
        lc->cabeza->siguiente = nuevo;
        int temp = lc->cabeza->dato;
        lc->cabeza->dato = nuevo->dato;
        nuevo->dato = temp;
    }
    lc->count++;
}

void lc_imprimir(ListaCircular *lc, int vueltas) {
    if (lc->cabeza == NULL) return;

    Nodo *actual = lc->cabeza;
    for (int i = 0; i < lc->count * vueltas; i++) {
        printf("%d -> ", actual->dato);
        actual = actual->siguiente;
    }
    printf("...\n");
}

void lc_liberar(ListaCircular *lc) {
    if (lc->cabeza == NULL) return;

    Nodo *actual = lc->cabeza;
    for (int i = 0; i < lc->count; i++) {
        Nodo *temp = actual;
        actual = actual->siguiente;
        free(temp);
    }
    lc->cabeza = NULL;
    lc->count = 0;
}

int main() {
    ListaCircular lc;
    lc_init(&lc);

    lc_insertar(&lc, 10);
    lc_insertar(&lc, 20);
    lc_insertar(&lc, 30);

    printf("Lista circular (2 vueltas): ");
    lc_imprimir(&lc, 2);

    lc_liberar(&lc);
    return 0;
}
```

RESUMEN DEL CAPÍTULO
===============================================================================

- Una lista enlazada simple está formada por nodos con dato y puntero
  al siguiente nodo; la lista se representa con un puntero a la cabeza.
- Las funciones que modifican la lista reciben puntero doble (Lista *)
  para poder cambiar la cabeza.
- Las operaciones fundamentales son: insertar (inicio, final, posición),
  eliminar, buscar, recorrer y liberar.
- La inserción y eliminación al inicio son O(1); al final son O(n),
  a menos que mantengamos un puntero al último nodo.
- La lista enlazada es la base para implementar pilas (LIFO) y colas
  (FIFO) dinámicas.
- Con void* podemos crear listas genéricas que almacenan cualquier tipo.
- La lista ordenada mantiene los elementos en secuencia durante la
  inserción.
- La inversión de lista es un problema clásico con solución iterativa
  y recursiva.

===============================================================================
PREGUNTAS FRECUENTES
===============================================================================

❓ **¿Por qué la inserción al final es O(n) en una lista simple?**
Porque necesitas recorrer toda la lista para encontrar el último nodo.
Se puede optimizar manteniendo un puntero adicional al último nodo.

❓ **¿Lista enlazada o array? ¿Cuándo usar cuál?**
Usa array para acceso aleatorio y listas para inserciones/eliminaciones
frecuentes. Las listas no tienen costo de redimensionamiento pero
no permiten acceso directo por índice.

❓ **¿Qué pasa si no libero la memoria de la lista?**
Cada nodo no liberado es un memory leak. Si la lista tiene 1000 nodos
y no los liberas, pierdes toda esa memoria hasta que el programa termina.

❓ **¿Puedo tener una lista enlazada de tipos mixtos?**
No directamente en C, porque cada nodo necesita saber el tipo de su dato.
Puedes usar uniones (union) o void* con un campo discriminante.
