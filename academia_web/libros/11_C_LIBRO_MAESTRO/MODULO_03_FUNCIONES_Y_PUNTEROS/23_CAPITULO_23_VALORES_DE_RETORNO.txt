Bienvenido al Capítulo 23. Las funciones no solo reciben datos,

"El valor de retorno es la promesa que una función hace al
llamador." — Anónimo

================================================================================
1. LA SENTENCIA RETURN
================================================================================

return termina la ejecución de una función y devuelve el control
al llamador. Opcionalmente envía un valor.

SINTAXIS:
  return;         // solo en funciones void
  return expresion;  // en funciones con tipo de retorno

Ejemplo básico:

--------------------------------------------------------------------------------
  #include <stdio.h>

  int cuadrado(int x) {
      return x * x;
  }

  int main(void) {
      printf("%d\n", cuadrado(5)); // 25
      return 0;
  }
--------------------------------------------------------------------------------

FLUJO DE EJECUCIÓN CON RETURN:

--------------------------------------------------------------------------------
  #include <stdio.h>

  int maximo(int a, int b) {
      if (a > b) {
          return a; // si a > b, termina aquí
      }
      return b;     // en caso contrario
  }

  int main(void) {
      int m = maximo(10, 20);
      printf("El maximo es: %d\n", m);
      return 0;
  }
--------------------------------------------------------------------------------

MÚLTIPLES RETURN EN UNA FUNCIÓN:

--------------------------------------------------------------------------------
  #include <stdio.h>

  int clasificar_nota(int puntaje) {
      if (puntaje < 0 || puntaje > 100) return -1;
      if (puntaje >= 90) return 4;
      if (puntaje >= 80) return 3;
      if (puntaje >= 70) return 2;
      if (puntaje >= 60) return 1;
      return 0;
  }

  int main(void) {
      int nota = clasificar_nota(85);
      if (nota == -1) {
          printf("Error: puntaje invalido\n");
      } else {
          printf("Clasificacion: %d\n", nota);
      }
      return 0;
  }
--------------------------------------------------------------------------------

================================================================================
2. TIPOS DE RETORNO
================================================================================

TIPO BÁSICO: int, float, char, etc.

--------------------------------------------------------------------------------
  int obtener_anio(void) { return 2026; }
  float pi(void) { return 3.14159f; }
  char inicial(const char* s) { return s[0]; }
  double raiz_cuadrada(double x) { return sqrt(x); }
--------------------------------------------------------------------------------

CONVERSIÓN DE TIPO EN EL RETORNO:

  El valor de return se convierte al tipo de retorno de la función.
  Esto puede causar pérdida de precisión:

--------------------------------------------------------------------------------
  #include <stdio.h>

  int dividir_int(int a, int b) {
      return a / b; // división entera, trunca
  }

  double dividir_double(int a, int b) {
      return (double)a / b; // conversión explícita
  }

  int main(void) {
      printf("%d\n", dividir_int(10, 3));     // 3
      printf("%.2f\n", dividir_double(10, 3)); // 3.33
      return 0;
  }
--------------------------------------------------------------------------------

RETORNO DE CHAR PROMOVIDO A INT:

--------------------------------------------------------------------------------
  #include <stdio.h>

  char get_char(void) {
      return 'A';
  }

  int main(void) {
      char c = get_char();
      printf("Caracter: %c, Codigo: %d\n", c, c);
      return 0;
  }
--------------------------------------------------------------------------------

================================================================================
3. FUNCIONES VOID
================================================================================

Una función void NO devuelve un valor. Puede usar return; para
terminar antes, pero no puede usar return con expresión.

--------------------------------------------------------------------------------
  #include <stdio.h>

  void imprimir_estado(int codigo) {
      if (codigo < 0) {
          printf("Error: codigo negativo\n");
          return; // termina la funcion aqui
      }
      if (codigo == 0) {
          printf("Estado: OK\n");
          return;
      }
      printf("Estado: desconocido (%d)\n", codigo);
  }

  int main(void) {
      imprimir_estado(0);
      imprimir_estado(-1);
      return 0;
  }
--------------------------------------------------------------------------------

USOS COMUNES DE FUNCIONES VOID:

  1. Procedimientos que solo ejecutan acciones:
     - imprimir_menu(), mostrar_ayuda(), inicializar_sistema()
  2. Funciones que modifican datos mediante punteros:

--------------------------------------------------------------------------------
  #include <stdio.h>

  void llenar_ceros(int arr[], int size) {
      for (int i = 0; i < size; i++) {
          arr[i] = 0;
      }
      // no necesita return
  }

  int main(void) {
      int datos[5] = {1, 2, 3, 4, 5};
      llenar_ceros(datos, 5);
      for (int i = 0; i < 5; i++) {
          printf("%d ", datos[i]);
      }
      return 0;
  }
--------------------------------------------------------------------------------

ERROR COMÚN: Usar return con valor en función void:

--------------------------------------------------------------------------------
  void funcion_void(void) {
      // return 42; // ERROR de compilacion
  }
--------------------------------------------------------------------------------

================================================================================
4. RETORNAR MÚLTIPLES VALORES CON PUNTEROS
================================================================================

Una función solo puede tener UN valor de retorno. Para devolver
múltiples valores, usamos parámetros de salida (punteros).

PATRÓN ESTÁNDAR:

--------------------------------------------------------------------------------
  #include <stdio.h>

  /**
   * Calcula estadisticas de un arreglo.
   * @param[in]  arr   Arreglo de entrada.
   * @param[in]  size  Tamano del arreglo.
   * @param[out] suma  Suma de elementos.
   * @param[out] prom  Promedio de elementos.
   * @param[out] max   Valor maximo.
   * @return 0 si ok, -1 si error.
   */
  int estadisticas(const int arr[], int size,
                   int* suma, double* prom, int* max) {
      if (arr == NULL || size <= 0) return -1;
      if (suma == NULL || prom == NULL || max == NULL) return -1;

      *suma = 0;
      *max = arr[0];
      for (int i = 0; i < size; i++) {
          *suma += arr[i];
          if (arr[i] > *max) *max = arr[i];
      }
      *prom = (double)*suma / size;
      return 0;
  }

  int main(void) {
      int datos[] = {10, 25, 8, 42, 15, 30};
      int n = sizeof(datos) / sizeof(datos[0]);
      int suma, max;
      double prom;

      if (estadisticas(datos, n, &suma, &prom, &max) == 0) {
          printf("Suma: %d\n", suma);
          printf("Promedio: %.2f\n", prom);
          printf("Maximo: %d\n", max);
      }
      return 0;
  }
--------------------------------------------------------------------------------

EJEMPLO: Devolver cociente y resto:

--------------------------------------------------------------------------------
  #include <stdio.h>
  #include <stdbool.h>

  bool division_segura(int dividendo, int divisor,
                       int* cociente, int* resto) {
      if (divisor == 0) return false;
      if (cociente == NULL || resto == NULL) return false;
      *cociente = dividendo / divisor;
      *resto = dividendo % divisor;
      return true;
  }

  int main(void) {
      int c, r;
      if (division_segura(17, 5, &c, &r)) {
          printf("17 / 5 = %d, resto %d\n", c, r);
      }
      return 0;
  }
--------------------------------------------------------------------------------

================================================================================
5. MANEJO DE ERRORES MEDIANTE RETORNO
================================================================================

Patrón común: el valor de retorno indica éxito/fracaso.

PATRÓN 1: Entero: 0 = éxito, -1 = error, positivo = código error.

--------------------------------------------------------------------------------
  #include <stdio.h>

  #define ERR_DIV_ZERO   -1
  #define ERR_NEGATIVO   -2
  #define ERR_NULL       -3

  int procesar_datos(const int* datos, int size) {
      if (datos == NULL) return ERR_NULL;
      if (size <= 0) return ERR_NEGATIVO;
      // ... procesamiento ...
      return 0; // exito
  }

  int main(void) {
      int r = procesar_datos(NULL, 5);
      if (r == ERR_NULL) {
          printf("Error: puntero nulo\n");
      }
      return 0;
  }
--------------------------------------------------------------------------------

PATRÓN 2: bool (true/false) para éxito/fracaso:

--------------------------------------------------------------------------------
  #include <stdio.h>
  #include <stdbool.h>

  bool leer_entero(const char* prompt, int* resultado) {
      if (prompt == NULL || resultado == NULL) return false;
      printf("%s", prompt);
      if (scanf("%d", resultado) != 1) return false;
      return true;
  }

  int main(void) {
      int numero;
      if (leer_entero("Ingrese un entero: ", &numero)) {
          printf("Leido: %d\n", numero);
      } else {
          printf("Error de lectura\n");
      }
      return 0;
  }
--------------------------------------------------------------------------------

PATRÓN 3: Puntero como retorno (NULL indica error):

  (Requiere malloc, ver Capítulo 26+)

--------------------------------------------------------------------------------
  #include <stdio.h>
  #include <stdlib.h>

  int* crear_arreglo(int size) {
      if (size <= 0) return NULL;
      int* arr = (int*)malloc(size * sizeof(int));
      return arr; // NULL si malloc falla
  }

  int main(void) {
      int* datos = crear_arreglo(10);
      if (datos == NULL) {
          printf("Error de memoria\n");
          return 1;
      }
      free(datos);
      return 0;
  }
--------------------------------------------------------------------------------

================================================================================
6. RETORNAR ARREGLOS
================================================================================

NO SE PUEDE retornar un arreglo directamente. El tipo de retorno
no puede ser un arreglo.

OPCIÓN 1: Retornar un puntero (a memoria estática o dinámica).

OPCIÓN 2: Modificar un arreglo pasado como parámetro.

OPCIÓN 3: Envolver en una estructura.

RETORNAR PUNTERO A ARREGLO ESTÁTICO (PELIGROSO):

--------------------------------------------------------------------------------
  #include <stdio.h>

  int* obtener_serie(void) {
      static int arr[5] = {1, 2, 3, 4, 5};
      return arr; // OK: static persiste
  }

  // PELIGRO: retornar puntero a variable local
  int* obtener_serie_mal(void) {
      int arr[5] = {1, 2, 3, 4, 5};
      return arr; // arr se destruye al retornar
  }
--------------------------------------------------------------------------------

RETORNAR CONSTRUCTOR DENTRO DE STRUCT:

--------------------------------------------------------------------------------
  #include <stdio.h>

  #define SIZE 3

  typedef struct {
      int data[SIZE];
  } ArregloFijo;

  ArregloFijo crear_arreglo(int a, int b, int c) {
      ArregloFijo arr;
      arr.data[0] = a;
      arr.data[1] = b;
      arr.data[2] = c;
      return arr; // copia la estructura completa
  }

  int main(void) {
      ArregloFijo mi_arr = crear_arreglo(10, 20, 30);
      for (int i = 0; i < SIZE; i++) {
          printf("%d ", mi_arr.data[i]);
      }
      return 0;
  }
--------------------------------------------------------------------------------

================================================================================
7. RETORNAR ESTRUCTURAS
================================================================================

Las estructuras pueden retornarse por valor (se copia todo).

--------------------------------------------------------------------------------
  #include <stdio.h>
  #include <math.h>

  typedef struct {
      double x;
      double y;
  } Punto;

  Punto crear_punto(double x, double y) {
      Punto p;
      p.x = x;
      p.y = y;
      return p;
  }

  double distancia(Punto a, Punto b) {
      double dx = a.x - b.x;
      double dy = a.y - b.y;
      return sqrt(dx * dx + dy * dy);
  }

  int main(void) {
      Punto p1 = crear_punto(0.0, 0.0);
      Punto p2 = crear_punto(3.0, 4.0);
      printf("Distancia: %.2f\n", distancia(p1, p2));
      return 0;
  }
--------------------------------------------------------------------------------

RETORNAR ESTRUCTURAS GRANDES:

  Para estructuras grandes, retornar por valor puede ser costoso.
  Mejor pasar un puntero como parámetro de salida:

--------------------------------------------------------------------------------
  #include <stdio.h>
  #include <string.h>

  typedef struct {
      char nombre[100];
      int edad;
      float salario;
      char direccion[200];
  } Empleado;

  // Eficiente: llena la estructura mediante puntero
  void empleado_init(Empleado* e, const char* nombre,
                     int edad, float salario) {
      strcpy(e->nombre, nombre);
      e->edad = edad;
      e->salario = salario;
      strcpy(e->direccion, "Pendiente");
  }

  int main(void) {
      Empleado emp;
      empleado_init(&emp, "Lucas", 30, 50000.0f);
      printf("%s: %d anos, $%.2f\n",
             emp.nombre, emp.edad, emp.salario);
      return 0;
  }
--------------------------------------------------------------------------------

================================================================================
8. RETORNAR PUNTEROS: PRECAUCIONES
================================================================================

Regla de oro: NUNCA retornes un puntero a una variable local.

--------------------------------------------------------------------------------
  #include <stdio.h>

  // PELIGROSO: retorna puntero a variable local
  int* get_numero_mal(void) {
      int x = 42;
      return &x; // x se destruye al salir
  }

  // CORRECTO: usa static o memoria dinamica
  int* get_numero_bien(void) {
      static int x = 42;
      return &x;
  }

  int main(void) {
      // int* p = get_numero_mal();
      // printf("%d\n", *p); // COMPORTAMIENTO INDEFINIDO

      int* q = get_numero_bien();
      printf("%d\n", *q); // OK
      return 0;
  }
--------------------------------------------------------------------------------

OTRO ERROR: Retornar puntero a string literal no debe modificarse:

--------------------------------------------------------------------------------
  #include <stdio.h>

  const char* obtener_mensaje(int codigo) {
      switch (codigo) {
          case 0:  return "Exito";
          case 1:  return "Advertencia";
          case -1: return "Error";
          default: return "Desconocido";
      }
  }

  int main(void) {
      const char* msg = obtener_mensaje(-1);
      printf("Mensaje: %s\n", msg);
      // msg[0] = 'X'; // COMPORTAMIENTO INDEFINIDO
      return 0;
  }
--------------------------------------------------------------------------------

================================================================================
9. LA FUNCIÓN MAIN Y SU VALOR DE RETORNO
================================================================================

main() siempre debe retornar int. El valor se usa como código de
salida del programa: 0 = éxito, != 0 = error.

--------------------------------------------------------------------------------
  #include <stdio.h>
  #include <stdlib.h>

  int main(void) {
      FILE* f = fopen("archivo.txt", "r");
      if (f == NULL) {
          printf("Error al abrir archivo\n");
          return EXIT_FAILURE; // generalmente 1
      }
      // ... procesar ...
      fclose(f);
      return EXIT_SUCCESS; // 0
  }
--------------------------------------------------------------------------------

Los códigos de salida los puede leer el sistema operativo:

  # En bash:
  ./programa && echo "éxito" || echo "fracaso"
  echo $?   # muestra el código de retorno de main()

Si main() no tiene return explícito, el compilador añade
implícitamente return 0; (desde C99).

main() con argumentos de línea de comandos:

--------------------------------------------------------------------------------
  #include <stdio.h>

  int main(int argc, char* argv[]) {
      printf("Argumentos: %d\n", argc);
      for (int i = 0; i < argc; i++) {
          printf("  argv[%d] = %s\n", i, argv[i]);
      }
      return 0;
  }
--------------------------------------------------------------------------------

================================================================================
10. RETORNAR Y EVALUAR EN EXPRESIONES
================================================================================

El valor de retorno se puede usar directamente en expresiones:

--------------------------------------------------------------------------------
  #include <stdio.h>

  int suma(int a, int b) { return a + b; }
  int resta(int a, int b) { return a - b; }

  int main(void) {
      int x = suma(10, 5);
      int y = resta(20, 8);
      int z = suma(resta(100, 30), suma(5, 3));
      //                   70         +    8     = 78
      printf("%d\n", z);

      // En condicionales:
      if (suma(2, 2) == 4) {
          printf("La matematica funciona!\n");
      }

      // En bucles:
      for (int i = 0; i < suma(3, 2); i++) {
          printf("%d ", i); // 0 1 2 3 4
      }
      return 0;
  }
--------------------------------------------------------------------------------

================================================================================
11. ELIMINACIÓN DE CÓDIGO MUERTO TRAS RETURN
================================================================================

Cualquier código después de return es inalcanzable y genera
advertencias del compilador.

--------------------------------------------------------------------------------
  #include <stdio.h>

  int ejemplo(void) {
      return 42;
      printf("Esto nunca se ejecuta\n"); // warning: dead code
  }

  int main(void) {
      printf("%d\n", ejemplo());
      return 0;
  }
--------------------------------------------------------------------------------

Para depuración, podemos usar condicionales de compilación:

--------------------------------------------------------------------------------
  #include <stdio.h>

  int calcular(int x) {
      #ifdef DEBUG
          printf("DEBUG: calcular(%d)\n", x);
      #endif
      return x * x;
  }

  int main(void) {
      printf("%d\n", calcular(5));
      return 0;
  }
--------------------------------------------------------------------------------

12. PATRONES DE RETORNO EN C MODERNO
================================================================================

PATRÓN TIPO (Result Type) con enum de error:

--------------------------------------------------------------------------------
  #include <stdio.h>
  #include <stdbool.h>

  typedef enum {
      OK = 0,
      ERR_DIVISION_CERO,
      ERR_OVERFLOW,
      ERR_NEGATIVO
  } ErrorCode;

  typedef struct {
      double valor;
      ErrorCode error;
  } Resultado;

  Resultado dividir_seguro(double a, double b) {
      Resultado res;
      if (b == 0.0) {
          res.error = ERR_DIVISION_CERO;
          res.valor = 0.0;
          return res;
      }
      res.valor = a / b;
      res.error = OK;
      return res;
  }

  int main(void) {
      Resultado r = dividir_seguro(10.0, 0.0);
      if (r.error == OK) {
          printf("Resultado: %.2f\n", r.valor);
      } else {
          printf("Error: %d\n", r.error);
      }
      return 0;
  }
--------------------------------------------------------------------------------

PATRÓN CALLBACK PARA ERROR:

--------------------------------------------------------------------------------
  #include <stdio.h>

  void manejar_error(const char* mensaje) {
      printf("ERROR: %s\n", mensaje);
  }

  int procesar(int valor, void (*on_error)(const char*)) {
      if (valor < 0) {
          on_error("Valor negativo no permitido");
          return -1;
      }
      return valor * 2;
  }

  int main(void) {
      procesar(-5, manejar_error);
      return 0;
  }
--------------------------------------------------------------------------------

RESUMEN DEL CAPÍTULO
================================================================================

  • return termina la función y opcionalmente devuelve un valor.
  • Una función solo puede tener UN valor de retorno directo.
  • Para múltiples valores, usar parámetros [out] con punteros.
  • void como retorno indica que no devuelve valor.
  • main() debe retornar int (0 = éxito, != 0 = error).
  • Nunca retornar puntero a variable local.
  • El código después de return es inalcanzable.
  • Los valores de retorno se pueden usar en expresiones.
  • Patrón común: retorno = código de error, parámetros = datos.
  • struct se puede retornar por valor (copia completa).

================================================================================
PREGUNTAS FRECUENTES (FAQ)
================================================================================

  P: ¿Puedo retornar más de un valor en C?
  R: Directamente no. Solo puedes retornar un valor. Para
     múltiples valores usa parámetros de salida (punteros) o
     una estructura que contenga varios campos.

  P: ¿Qué pasa si no pongo return en una función no-void?
  R: Comportamiento indefinido. La función devolverá basura
     (lo que haya en el registro de retorno). El compilador
     emitirá una advertencia.

  P: ¿Puedo retornar un arreglo?
  R: No directamente. Puedes retornar un puntero a un arreglo
     static o a memoria dinámica, o envolver el arreglo en
     una estructura.

  P: ¿Funciona return en funciones void?
  R: Sí, como return; (sin valor). Termina la función antes.
     return con expresión en función void es error.

  P: ¿Qué diferencia hay en retornar una estructura vs un puntero?
  R: Retornar la estructura copia todos los bytes. Puede ser
     lento para estructuras grandes. Retornar un puntero es
     más eficiente pero debe apuntar a memoria que persista.

  P: ¿Cuál es el propósito de EXIT_SUCCESS y EXIT_FAILURE?
  R: Son constantes definidas en <stdlib.h> para códigos de
     salida portables. EXIT_SUCCESS es 0, EXIT_FAILURE es 1
     (o el valor que el sistema defina).

  P: ¿El valor de retorno de main() siempre llega al SO?
  R: Sí, el sistema operativo recibe el código de salida.
     Puedes consultarlo en bash con $?, en Windows con
     %ERRORLEVEL%.
