Bienvenido al Capítulo 27. En capítulos anteriores vimos que C
pasa argumentos por valor, lo que impide modificar variables del
llamador. Los punteros son la solución: pasamos un puntero por
valor, y la función puede modificar el contenido apuntado.

"Un puntero es la única forma de que una función hable de vuelta
con quien la llamó." — Anónimo

================================================================================
OBJETIVOS DEL CAPÍTULO
================================================================================

  • Usar punteros como parámetros de funciones.
  • Implementar la función swap() correctamente.
  • Modificar variables del llamador desde la función.
  • Usar parámetros de salida [out] con punteros.
  • Pasar arreglos y estructuras por puntero.
  • Evitar copias innecesarias de datos grandes.
  • Combinar punteros y const para seguridad.
  • Implementar funciones con retorno múltiple.

================================================================================
1. REPASO: PASO POR VALOR
================================================================================

Sin punteros, las funciones no pueden modificar las variables
del llamador:

--------------------------------------------------------------------------------
  #include <stdio.h>

  void intentar_cambiar(int x) {
      x = 100; // solo cambia la copia local
  }

  int main(void) {
      int a = 5;
      intentar_cambiar(a);
      printf("a = %d\n", a); // 5, no cambió
      return 0;
  }
--------------------------------------------------------------------------------

La variable 'a' y el parámetro 'x' son ubicaciones de memoria
distintas. Modificar 'x' no afecta a 'a'.

================================================================================
2. PASO POR PUNTERO (PASO POR REFERENCIA SIMULADO)
--------------------------------------------------------------------------------

Pasamos la dirección de la variable. La función recibe un puntero
y puede modificar el contenido mediante *.

--------------------------------------------------------------------------------
  #include <stdio.h>

  void cambiar(int* p) {
      *p = 100; // modifica la variable apuntada
  }

  int main(void) {
      int a = 5;
      cambiar(&a);  // pasamos la dirección de a
      printf("a = %d\n", a); // 100
      return 0;
  }
--------------------------------------------------------------------------------

El puntero 'p' se pasa por valor (se copia la dirección), pero
esa copia apunta al mismo lugar que el original, por lo que
*p = 100 modifica la variable original.

ESQUEMA EN MEMORIA:

  main():
    a = 5         (dirección 0x1000)

  cambiar(&a):
    p = 0x1000    (copia de la dirección de a)
    *p = 100      → escribe 100 en 0x1000 → a ahora es 100

================================================================================
3. LA FUNCIÓN SWAP (INTERCAMBIAR)
================================================================================

El ejemplo clásico: intercambiar dos variables.

VERSIÓN INCORRECTA (sin punteros):

--------------------------------------------------------------------------------
  #include <stdio.h>

  void swap_mal(int a, int b) {
      int temp = a;
      a = b;
      b = temp;
      // solo intercambia las copias locales
  }

  int main(void) {
      int x = 10, y = 20;
      printf("Antes: x=%d, y=%d\n", x, y);
      swap_mal(x, y);
      printf("Despues: x=%d, y=%d\n", x, y); // sin cambios
      return 0;
  }
--------------------------------------------------------------------------------

VERSIÓN CORRECTA (con punteros):

--------------------------------------------------------------------------------
  #include <stdio.h>

  void swap(int* a, int* b) {
      int temp = *a;
      *a = *b;
      *b = temp;
  }

  int main(void) {
      int x = 10, y = 20;
      printf("Antes: x=%d, y=%d\n", x, y);
      swap(&x, &y);
      printf("Despues: x=%d, y=%d\n", x, y); // intercambiados
      return 0;
  }
--------------------------------------------------------------------------------

SALIDA:
  Antes: x=10, y=20
  Despues: x=20, y=10

VERSIÓN GENÉRICA (con void*):

--------------------------------------------------------------------------------
  #include <stdio.h>
  #include <string.h>

  void swap_generic(void* a, void* b, size_t size) {
      char temp[size]; // VLA (C99)
      memcpy(temp, a, size);
      memcpy(a, b, size);
      memcpy(b, temp, size);
  }

  int main(void) {
      int x = 10, y = 20;
      swap_generic(&x, &y, sizeof(int));
      printf("x=%d, y=%d\n", x, y); // 20 10

      double p = 3.14, q = 2.71;
      swap_generic(&p, &q, sizeof(double));
      printf("p=%.2f, q=%.2f\n", p, q); // 2.71 3.14
      return 0;
  }
--------------------------------------------------------------------------------

4. PARÁMETROS DE SALIDA [OUT]
--------------------------------------------------------------------------------

Usamos punteros para que una función "devuelva" múltiples valores.

EJEMPLO: CALCULAR ESTADÍSTICAS

--------------------------------------------------------------------------------
  #include <stdio.h>
  #include <stdbool.h>

  /**
   * Calcula estadisticas basicas de un arreglo.
   * @param[in]  arr   Arreglo de enteros.
   * @param[in]  n     Numero de elementos.
   * @param[out] suma  Suma total.
   * @param[out] prom  Promedio.
   * @param[out] max   Valor maximo.
   * @param[out] min   Valor minimo.
   * @return true si ok, false si error.
   */
  bool estadisticas(const int arr[], int n,
                    int* suma, double* prom,
                    int* max, int* min) {
      if (arr == NULL || n <= 0) return false;
      if (!suma || !prom || !max || !min) return false;

      *suma = 0;
      *max = arr[0];
      *min = arr[0];

      for (int i = 0; i < n; i++) {
          *suma += arr[i];
          if (arr[i] > *max) *max = arr[i];
          if (arr[i] < *min) *min = arr[i];
      }
      *prom = (double)*suma / n;
      return true;
  }

  int main(void) {
      int datos[] = {15, 8, 23, 42, 4, 16, 30};
      int n = sizeof(datos) / sizeof(datos[0]);
      int suma, max, min;
      double prom;

      if (estadisticas(datos, n, &suma, &prom, &max, &min)) {
          printf("Suma: %d\n", suma);
          printf("Prom: %.2f\n", prom);
          printf("Max:  %d\n", max);
          printf("Min:  %d\n", min);
      }
      return 0;
  }
--------------------------------------------------------------------------------

EJEMPLO: OPERACIÓN QUE MODIFICA DOS VALORES

--------------------------------------------------------------------------------
  #include <stdio.h>

  /**
   * Descompone un entero en cociente y resto de dividir por 10.
   * @param[in]  valor    Valor a descomponer.
   * @param[out] unidades Digito de unidades.
   * @param[out] resto    Valor sin el digito de unidades.
   */
  void descomponer(int valor, int* unidades, int* resto) {
      if (unidades) *unidades = valor % 10;
      if (resto) *resto = valor / 10;
  }

  int main(void) {
      int u, r;
      descomponer(1234, &u, &r);
      printf("Unidades: %d, Resto: %d\n", u, r);
      return 0;
  }
--------------------------------------------------------------------------------

5. PUNTEROS A ARREGLOS COMO PARÁMETROS
--------------------------------------------------------------------------------

Cuando pasamos un arreglo a una función, el parámetro es un
puntero. Las modificaciones afectan al original.

--------------------------------------------------------------------------------
  #include <stdio.h>

  // Estas dos declaraciones son equivalentes:
  void duplicar(int arr[], int n);
  void duplicar(int* arr, int n);

  void duplicar(int* arr, int n) {
      for (int i = 0; i < n; i++) {
          arr[i] *= 2; // modifica el original
      }
  }

  int main(void) {
      int nums[] = {1, 2, 3, 4, 5};
      int n = sizeof(nums) / sizeof(nums[0]);

      duplicar(nums, n);

      for (int i = 0; i < n; i++) {
          printf("%d ", nums[i]);
      }
      // 2 4 6 8 10
      return 0;
  }
--------------------------------------------------------------------------------

CONST PARA PROTEGER ARREGLOS DE SOLO LECTURA:

--------------------------------------------------------------------------------
  #include <stdio.h>

  // El arreglo NO se modifica (const)
  int suma_arr(const int* arr, int n) {
      int total = 0;
      for (int i = 0; i < n; i++) {
          total += arr[i];
          // arr[i] = 0; // ERROR: const
      }
      return total;
  }
--------------------------------------------------------------------------------

ARREGLOS MULTIDIMENSIONALES:

--------------------------------------------------------------------------------
  #include <stdio.h>

  // '[3]' es obligatorio para la segunda dimension (o mayor)
  void imprimir_matriz(int (*mat)[3], int filas) {
      for (int i = 0; i < filas; i++) {
          for (int j = 0; j < 3; j++) {
              printf("%d ", mat[i][j]);
          }
          printf("\n");
      }
  }

  int main(void) {
      int m[2][3] = {{1, 2, 3}, {4, 5, 6}};
      imprimir_matriz(m, 2);
      return 0;
  }
--------------------------------------------------------------------------------

================================================================================
6. PUNTEROS A ESTRUCTURAS COMO PARÁMETROS
--------------------------------------------------------------------------------

Pasar estructuras grandes por valor es costoso (se copia todo).
Pasarlas por puntero es eficiente.

--------------------------------------------------------------------------------
  #include <stdio.h>
  #include <string.h>

  typedef struct {
      char nombre[100];
      int edad;
      float salario;
  } Empleado;

  // Inicializar (modifica, puntero no const)
  void empleado_init(Empleado* e, const char* nombre,
                     int edad, float salario) {
      strcpy(e->nombre, nombre);
      e->edad = edad;
      e->salario = salario;
  }

  // Solo lectura (puntero const)
  void empleado_mostrar(const Empleado* e) {
      printf("Nombre: %s\n", e->nombre);
      printf("Edad: %d\n", e->edad);
      printf("Salario: %.2f\n", e->salario);
  }

  // Aumento de salario (modifica)
  void empleado_aumentar(Empleado* e, float porcentaje) {
      e->salario *= (1.0f + porcentaje / 100.0f);
  }

  int main(void) {
      Empleado emp;
      empleado_init(&emp, "Lucas Garcia", 30, 50000.0f);
      empleado_mostrar(&emp);
      empleado_aumentar(&emp, 10.0f); // 10% de aumento
      printf("Nuevo salario: %.2f\n", emp.salario);
      return 0;
  }
--------------------------------------------------------------------------------

COMPARACIÓN: Pase por valor vs por puntero

  Por valor:   void f(Empleado e) → copia 108 bytes
  Por puntero: void f(Empleado* e) → copia 8 bytes (64-bit)

Para estructuras grandes (> 16 bytes), usar puntero.

7. COMBINANDO PUNTEROS Y CONST
--------------------------------------------------------------------------------

Cuatro combinaciones de const con punteros:

  1. const int* p   → Puntero a entero constante.
     (El valor apuntado NO se puede modificar)
  2. int* const p   → Puntero constante a entero.
     (El puntero NO puede cambiar de dirección)
  3. const int* const p → Puntero constante a entero constante.
  4. int* p         → Puntero normal a entero normal.

--------------------------------------------------------------------------------
  #include <stdio.h>

  int main(void) {
      int a = 10, b = 20;

      // 1. Puntero a entero constante
      const int* p1 = &a;
      // *p1 = 30; // ERROR: valor constante
      p1 = &b;      // OK: puntero puede cambiar

      // 2. Puntero constante a entero
      int* const p2 = &a;
      *p2 = 30;    // OK: valor modificable
      // p2 = &b;  // ERROR: puntero constante

      // 3. Puntero constante a entero constante
      const int* const p3 = &a;
      // *p3 = 30; // ERROR
      // p3 = &b;  // ERROR

      return 0;
  }
--------------------------------------------------------------------------------

REGLAS PRÁCTICAS:
  • Parámetros de solo lectura → const int* p
  • Parámetros de salida       → int* p
  • Arreglos de solo lectura    → const int arr[]
  • Arreglos modificables       → int arr[]

8. FUNCIONES QUE RETORNAN PUNTEROS
--------------------------------------------------------------------------------

Las funciones pueden retornar punteros. Cuidado con la vigencia
de la memoria retornada.

--------------------------------------------------------------------------------
  #include <stdio.h>
  #include <stdlib.h>

  // Retorna puntero a memoria estatica (persistente)
  const char* obtener_mes(int numero) {
      static const char* meses[] = {
          "Invalido", "Enero", "Febrero", "Marzo",
          "Abril", "Mayo", "Junio", "Julio",
          "Agosto", "Septiembre", "Octubre",
          "Noviembre", "Diciembre"
      };
      if (numero < 1 || numero > 12) return meses[0];
      return meses[numero];
  }

  // Retorna puntero a memoria dinamica
  int* crear_arreglo(int n) {
      int* arr = malloc(n * sizeof(int));
      return arr; // NULL si fallo
  }

  int main(void) {
      printf("Mes 3: %s\n", obtener_mes(3)); // Marzo
      int* datos = crear_arreglo(10);
      if (datos != NULL) {
          datos[0] = 42;
          free(datos);
      }
      return 0;
  }
--------------------------------------------------------------------------------

NUNCA retornes puntero a variable local (automática):

--------------------------------------------------------------------------------
  int* error() {
      int local = 42;
      return &local; // local se destruye al retornar
  }
--------------------------------------------------------------------------------

9. PATRÓN: FUNCIÓN CON CALLBACK (AVANCE)
--------------------------------------------------------------------------------

Podemos pasar punteros a funciones como parámetros. Se ve en
detalle en el Capítulo 28, pero aquí un adelanto:

--------------------------------------------------------------------------------
  #include <stdio.h>

  // Función que aplica una operación a cada elemento
  void aplicar(int* arr, int n, int (*op)(int)) {
      for (int i = 0; i < n; i++) {
          arr[i] = op(arr[i]);
      }
  }

  int duplicar(int x) { return x * 2; }
  int cuadrado(int x) { return x * x; }
  int incrementar(int x) { return x + 1; }

  int main(void) {
      int datos[] = {1, 2, 3, 4, 5};
      int n = sizeof(datos) / sizeof(datos[0]);

      aplicar(datos, n, duplicar);
      for (int i = 0; i < n; i++)
          printf("%d ", datos[i]); // 2 4 6 8 10

      printf("\n");
      aplicar(datos, n, cuadrado);
      for (int i = 0; i < n; i++)
          printf("%d ", datos[i]); // 4 16 36 64 100

      printf("\n");
      return 0;
  }
--------------------------------------------------------------------------------

10. VALIDACIÓN DE PUNTEROS EN FUNCIONES
--------------------------------------------------------------------------------

Toda función que reciba punteros debe validarlos.

--------------------------------------------------------------------------------
  #include <stdio.h>
  #include <stdbool.h>

  int procesar(int* datos, int n, int* resultado) {
      // Validar todos los punteros
      if (datos == NULL || resultado == NULL) {
          return -1; // error
      }
      if (n <= 0) {
          return -2; // error de tamaño
      }
      *resultado = 0;
      for (int i = 0; i < n; i++) {
          *resultado += datos[i];
      }
      return 0; // éxito
  }

  int main(void) {
      int nums[] = {10, 20, 30};
      int res;
      if (procesar(nums, 3, &res) == 0) {
          printf("Resultado: %d\n", res);
      }
      // Prueba con NULL
      if (procesar(NULL, 3, &res) != 0) {
          printf("Error detectado correctamente\n");
      }
      return 0;
  }
--------------------------------------------------------------------------------

11. EJEMPLO COMPLETO: MANIPULACIÓN DE STRINGS CON PUNTEROS
--------------------------------------------------------------------------------

--------------------------------------------------------------------------------
  #include <stdio.h>
  #include <ctype.h>

  // Convertir a mayusculas usando punteros
  void a_mayusculas(char* str) {
      if (str == NULL) return;
      while (*str != '\0') {
          *str = toupper(*str);
          str++; // avanzar puntero
      }
  }

  // Contar caracteres (como strlen pero con punteros)
  int mi_strlen(const char* str) {
      const char* inicio = str;
      while (*str != '\0') {
          str++;
      }
      return str - inicio; // distancia entre punteros
  }

  // Copiar string (como strcpy)
  void mi_strcpy(char* dest, const char* src) {
      if (dest == NULL || src == NULL) return;
      while ((*dest++ = *src++) != '\0') {
          // el cuerpo está vacío intencionalmente
      }
  }

  // Invertir string in-place
  void invertir(char* str) {
      if (str == NULL) return;
      int len = mi_strlen(str);
      char* inicio = str;
      char* fin = str + len - 1;
      while (inicio < fin) {
          char temp = *inicio;
          *inicio = *fin;
          *fin = temp;
          inicio++;
          fin--;
      }
  }

  int main(void) {
      char texto[100] = "Hola Mundo";
      printf("Original: %s\n", texto);
      printf("Longitud: %d\n", mi_strlen(texto));

      a_mayusculas(texto);
      printf("Mayus:    %s\n", texto);

      invertir(texto);
      printf("Invertido: %s\n", texto);

      char copia[100];
      mi_strcpy(copia, texto);
      printf("Copia:    %s\n", copia);
      return 0;
  }
--------------------------------------------------------------------------------

12. ERRORES COMUNES CON PUNTEROS EN FUNCIONES
--------------------------------------------------------------------------------

ERROR 1: Olvidar el & al pasar argumentos.

--------------------------------------------------------------------------------
  void swap(int* a, int* b);

  int main(void) {
      int x = 10, y = 20;
      // swap(x, y);    // ERROR: pasa int, no int*
      swap(&x, &y);     // correcto
      return 0;
  }
--------------------------------------------------------------------------------

ERROR 2: Olvidar * dentro de la función.

--------------------------------------------------------------------------------
  void incrementar(int* p) {
      p++;   // ERROR: avanza el puntero, no incrementa el valor
      // (*p)++; // correcto: incrementa el valor apuntado
  }
--------------------------------------------------------------------------------

ERROR 3: No verificar NULL.

--------------------------------------------------------------------------------
  void imprimir(const char* str) {
      // printf("%s\n", str); // CRASH si str es NULL
      if (str != NULL) {
          printf("%s\n", str);
      }
  }
--------------------------------------------------------------------------------

ERROR 4: Dereferenciar puntero antes de validar.

--------------------------------------------------------------------------------
  void proceso(int* p) {
      *p = 10;  // CRASH si p es NULL
      // if (p != NULL) *p = 10; // orden correcto:
      // primero validar, luego dereferenciar
  }
--------------------------------------------------------------------------------

ERROR 5: Pasar estructura por valor cuando debería ser puntero.

--------------------------------------------------------------------------------
  void procesar_empleado(Empleado e) { // copia costosa
      // ...
  }
  // Mejor: void procesar_empleado(const Empleado* e)
--------------------------------------------------------------------------------

RESUMEN DEL CAPÍTULO
================================================================================

  • Los punteros permiten modificar variables del llamador.
  • swap() es el ejemplo clásico de modificación bidireccional.
  • Parámetros [out] con punteros para retornar múltiples valores.
  • Arreglos en parámetros siempre son punteros.
  • Estructuras grandes deben pasarse por puntero.
  • const protege la integridad de los datos apuntados.
  • Siempre validar punteros NULL al inicio de la función.
  • Nunca retornar puntero a variable local.
  • Usar & al pasar variables y * al acceder al valor.

================================================================================
PREGUNTAS FRECUENTES (FAQ)
================================================================================

  P: ¿Es obligatorio pasar siempre punteros a estructuras?
  R: No. Si la estructura es pequeña (≤ 16 bytes) y no
     necesitas modificarla, pasarla por valor está bien.
     La decisión es de eficiencia.

  P: ¿Qué diferencia hay entre int arr[] y int* arr como
     parámetro?
  R: Ninguna. El compilador trata int arr[] como int*.
     La notación arr[] es azúcar sintáctico que documenta
     la intención de "arreglo".

  P: ¿Puedo modificar el puntero mismo dentro de la función?
  R: Sí, pero el cambio no se refleja fuera a menos que
     uses un puntero a puntero (int**).

  P: ¿Cuándo usar const int* vs int* const?
  R: Usa const int* (puntero a entero constante) para
     parámetros de solo lectura (lo más común). Usa
     int* const (puntero constante) raramente, solo
     cuando quieras que el puntero no cambie.

  P: ¿Qué significa el error "segmentation fault"?
  R: Ocurre cuando intentas acceder a memoria que no te
     pertenece, típicamente por dereferenciar NULL o un
     puntero colgante.

  P: ¿Los punteros son más rápidos que pasar por valor?
  R: Para tipos grandes (> 16 bytes), sí. Para tipos
     pequeños (int, char, float), pasar por valor puede
     ser igual o más rápido.
