  1. ¿Qué es un Array?
  ────────────────────

  Un array en Go es una secuencia de elementos del MISMO TIPO con TAMAÑO FIJO
  determinado en tiempo de compilación. La sintaxis es:

      var nombre [tamaño]Tipo

  Ejemplo:

      var numeros [5]int

  Esto declara un array de 5 enteros, todos inicializados a cero (valor cero
  del tipo int).

  Características clave:

  • Tamaño fijo: [5]int y [10]int son tipos diferentes.
  • Almacenamiento contiguo en memoria: los elementos están uno al lado del otro.
  • Semántica de valor: al asignar o pasar un array, se COPIA todo el contenido.
  • El tamaño es parte del tipo: [3]int ≠ [4]int.

  2. Declaración e Inicialización
  ───────────────────────────────

  ================================================================================
  EJEMPLO 1: Declaración básica
  ================================================================================

  package main

  import "fmt"

  func main() {
      // Declaración con valor cero
      var a [5]int
      fmt.Println("Array zero value:", a) // [0 0 0 0 0]

      // Declaración con literal
      b := [5]int{10, 20, 30, 40, 50}
      fmt.Println("Literal:", b)

      // Índice específico
      c := [5]int{0: 100, 3: 200}
      fmt.Println("Índices específicos:", c) // [100 0 0 200 0]

      // Inferir tamaño con [...]
      d := [...]int{2, 4, 6, 8, 10, 12}
      fmt.Println("Tamaño inferido:", d, "len:", len(d))
  }

  ================================================================================
  EJEMPLO 2: Acceso por índice
  ================================================================================

  package main

  import "fmt"

  func main() {
      colores := [3]string{"rojo", "verde", "azul"}

      fmt.Println("Primero:", colores[0])
      fmt.Println("Segundo:", colores[1])
      fmt.Println("Tercero:", colores[2])

      // Modificar un elemento
      colores[1] = "amarillo"
      fmt.Println("Modificado:", colores)

      // longitud
      fmt.Println("Longitud:", len(colores))
  }

  ================================================================================
  EJEMPLO 3: Array de diferentes tipos
  ================================================================================

  package main

  import "fmt"

  func main() {
      // Array de strings
      var frutas [3]string
      frutas[0] = "manzana"
      frutas[1] = "pera"
      frutas[2] = "uva"
      fmt.Println("Frutas:", frutas)

      // Array de floats
      var precios [4]float64
      precios[0] = 12.5
      precios[1] = 24.99
      precios[2] = 9.95
      precios[3] = 3.50
      fmt.Println("Precios:", precios)

      // Array de booleanos
      var flags [3]bool
      flags[1] = true
      fmt.Println("Flags:", flags)
  }

  3. Arrays son Copias por Valor
  ──────────────────────────────

  En Go, los arrays son VALORES. Cuando asignas un array a otra variable o lo
  pasas a una función, se COPIA TODO el contenido. Esto es diferente de otros
  lenguajes donde los arrays son referencias.

  ================================================================================
  EJEMPLO 4: Copia de array
  ================================================================================

  package main

  import "fmt"

  func main() {
      original := [3]int{1, 2, 3}
      copia := original // Se copian TODOS los elementos

      copia[0] = 999

      fmt.Println("Original:", original) // [1 2 3] — intacto
      fmt.Println("Copia:", copia)        // [999 2 3]
  }

  ================================================================================
  EJEMPLO 5: Array como parámetro de función
  ================================================================================

  package main

  import "fmt"

  func modificar(arr [3]int) {
      arr[0] = 100
      fmt.Println("Dentro de función:", arr)
  }

  func main() {
      valores := [3]int{1, 2, 3}
      modificar(valores)
      fmt.Println("Fuera de función:", valores) // [1 2 3]
  }

  4. Comparación de Arrays
  ────────────────────────

  Los arrays en Go son COMPARABLES con == y != si el tipo de los elementos es
  comparable. Dos arrays son iguales si tienen el mismo tamaño y todos sus
  elementos son iguales.

  ================================================================================
  EJEMPLO 6: Comparación de arrays
  ================================================================================

  package main

  import "fmt"

  func main() {
      a := [3]int{1, 2, 3}
      b := [3]int{1, 2, 3}
      c := [3]int{3, 2, 1}

      fmt.Println("a == b:", a == b) // true
      fmt.Println("a == c:", a == c) // false

      // Arrays de diferentes tamaños NO se pueden comparar (error de compilación)
      // d := [4]int{1, 2, 3, 4}
      // fmt.Println(a == d) // ERROR: mismatched types [3]int and [4]int
  }

  ================================================================================
  EJEMPLO 7: Array de strings
  ================================================================================

  package main

  import "fmt"

  func main() {
      semana1 := [3]string{"lun", "mar", "mié"}
      semana2 := [3]string{"lun", "mar", "mié"}
      semana3 := [3]string{"jue", "vie", "sáb"}

      fmt.Println("¿Son iguales?", semana1 == semana2) // true
      fmt.Println("¿Son diferentes?", semana1 != semana3) // true

      // Los arrays solo son comparables si los elementos son comparables
      // [3]int sí, [3]map[int]int NO (los maps no son comparables)
  }

  5. Recorrer Arrays
  ──────────────────

  Formas de iterar sobre un array:

  ================================================================================
  EJEMPLO 8: Bucle for tradicional
  ================================================================================

  package main

  import "fmt"

  func main() {
      nums := [5]int{10, 20, 30, 40, 50}

      for i := 0; i < len(nums); i++ {
          fmt.Printf("Índice %d → Valor %d\n", i, nums[i])
      }
  }

  ================================================================================
  EJEMPLO 9: Bucle for-range
  ================================================================================

  package main

  import "fmt"

  func main() {
      nums := [5]int{10, 20, 30, 40, 50}

      for i, v := range nums {
          fmt.Printf("Índice %d → Valor %d\n", i, v)
      }
  }

  ================================================================================
  EJEMPLO 10: Solo valores con range
  ================================================================================

  package main

  import "fmt"

  func main() {
      letras := [4]string{"a", "b", "c", "d"}

      fmt.Println("Solo valores:")
      for _, v := range letras {
          fmt.Println(v)
      }

      fmt.Println("Solo índices:")
      for i := range letras {
          fmt.Println(i)
      }
  }

  6. Arrays Multidimensionales
  ────────────────────────────

  Go soporta arrays anidados (matrices). La sintaxis es [filas][columnas]Tipo.

  ================================================================================
  EJEMPLO 11: Matriz 2D
  ================================================================================

  package main

  import "fmt"

  func main() {
      // Matriz 3x3
      var matriz [3][3]int

      // Llenar con valores
      for i := 0; i < 3; i++ {
          for j := 0; j < 3; j++ {
              matriz[i][j] = i*3 + j + 1
          }
      }

      fmt.Println("Matriz 3x3:")
      for i := 0; i < 3; i++ {
          fmt.Println(matriz[i])
      }
  }

  ================================================================================
  EJEMPLO 12: Matriz con literal
  ================================================================================

  package main

  import "fmt"

  func main() {
      m := [2][3]int{
          {1, 2, 3},
          {4, 5, 6},
      }
      fmt.Println(m)

      fmt.Println("Elemento [1][2]:", m[1][2]) // 6
  }

  ================================================================================
  EJEMPLO 13: Matriz 3D
  ================================================================================

  package main

  import "fmt"

  func main() {
      cubo := [2][2][2]int{
          {
              {1, 2},
              {3, 4},
          },
          {
              {5, 6},
              {7, 8},
          },
      }
      fmt.Println("Cubo 2x2x2:", cubo)
      fmt.Println("cubo[1][0][1]:", cubo[1][0][1]) // 6
  }

  7. Array Literal con Índices
  ────────────────────────────

  Puedes especificar valores en posiciones concretas usando índices en el literal.

  ================================================================================
  EJEMPLO 14: Inicialización con claves
  ================================================================================

  package main

  import "fmt"

  func main() {
      // Array de 10 enteros, solo posiciones 0, 3, 5, 9 inicializadas
      arr := [10]int{
          0: 100,
          3: 200,
          5: 300,
          9: 400,
      }
      fmt.Println("Array con claves:", arr)
      // [100 0 0 200 0 300 0 0 0 400]
  }

  ================================================================================
  EJEMPLO 15: Constantes y arrays
  ================================================================================

  package main

  import "fmt"

  const (
      rojo   = 0
      verde  = 1
      azul   = 2
  )

  func main() {
      rgb := [3]string{rojo: "#FF0000", verde: "#00FF00", azul: "#0000FF"}

      }
  }

  8. Arrays y funciones len/cap
  ────────────────────────────

  Para arrays, len() y cap() siempre devuelven el mismo valor: el tamaño fijo.

  ================================================================================
  EJEMPLO 16: len y cap en arrays
  ================================================================================

  package main

  import "fmt"

  func main() {
      a := [8]int{1, 2, 3, 4, 5}

      fmt.Println("len:", len(a)) // 8
      fmt.Println("cap:", cap(a)) // 8 (siempre igual a len para arrays)
  }

  9. Arrays de Tipos Compuestos
  ─────────────────────────────

  ================================================================================
  EJEMPLO 17: Array de structs
  ================================================================================

  package main

  import "fmt"

  type Punto struct {
      X, Y int
  }

  func main() {
      puntos := [3]Punto{
          {1, 2},
          {3, 4},
          {5, 6},
      }
      fmt.Println(puntos)

      puntos[2] = Punto{100, 200}
      fmt.Println(puntos)
  }

  ================================================================================
  EJEMPLO 18: Array de punteros
  ================================================================================

  package main

  import "fmt"

  func main() {
      a, b, c := 10, 20, 30
      punteros := [3]*int{&a, &b, &c}

      fmt.Println("Valores:")
      for _, p := range punteros {
          fmt.Println(*p)
      }

      *punteros[1] = 999
      fmt.Println("b modificado:", b) // 999
  }

  10. Pasar Arrays por Puntero
  ────────────────────────────

  Para evitar copiar el array completo, pasa un puntero al array.

  ================================================================================
  EJEMPLO 19: Array por puntero
  ================================================================================

  package main

  import "fmt"

  func modificar(arr *[5]int) {
      arr[0] = 100
  }

  func main() {
      valores := [5]int{1, 2, 3, 4, 5}
      modificar(&valores)
      fmt.Println(valores) // [100 2 3 4 5]
  }

  ================================================================================
  EJEMPLO 20: Función que retorna un array
  ================================================================================

  package main

  import "fmt"

  func duplicar(arr [3]int) [3]int {
      for i := range arr {
          arr[i] *= 2
      }
      return arr
  }

  func main() {
      original := [3]int{1, 2, 3}
      resultado := duplicar(original)

      fmt.Println("Original:", original)
      fmt.Println("Resultado:", resultado)
  }

  ================================================================================
  EJEMPLO 23: Invertir un array
  ================================================================================

  package main

  import "fmt"

  func invertir(arr [5]int) [5]int {
      for i, j := 0, len(arr)-1; i < j; i, j = i+1, j-1 {
          arr[i], arr[j] = arr[j], arr[i]
      }
      return arr
  }

  func main() {
      original := [5]int{1, 2, 3, 4, 5}
      invertido := invertir(original)
      fmt.Println("Original:", original)
      fmt.Println("Invertido:", invertido)
  }

  EJEMPLO 25: Buscar elemento
  ================================================================================

  package main

  import "fmt"

  func buscar(arr [5]string, objetivo string) int {
      for i, v := range arr {
          if v == objetivo {
              return i
          }
      }
      return -1
  }

  func main() {
      frutas := [5]string{"manzana", "pera", "uva", "naranja", "kiwi"}
      fmt.Println("Posición de 'uva':", buscar(frutas, "uva"))     // 2
      fmt.Println("Posición de 'melon':", buscar(frutas, "melon")) // -1
  }

  RESUMEN DEL CAPÍTULO
  ================================================================================

  • Los arrays en Go tienen tamaño FIJO determinado en tiempo de compilación.
  • La sintaxis es [tamaño]Tipo, ejemplo: [5]int.
  • El tamaño es PARTE del tipo: [3]int y [5]int son tipos distintos.
  • Los arrays se inicializan con su valor cero si no se especifica otro.
  • Se puede usar [tamaño]Tipo{valores...} o [...]Tipo{valores...} para inferir.
  • Los índices van de 0 a len(array)-1.
  • Los arrays son VALORES: se copian al asignarse o pasarse a funciones.
  • Se comparan con == y != (si el tipo de elemento es comparable).
  • len() y cap() devuelven el tamaño fijo para arrays.
  • Se recorren con for i := 0; i < len(a); i++ o con for i, v := range a.
  • Los arrays multidimensionales se declaran como [f][c]Tipo.
  • Para evitar copias, usa punteros al array: func f(arr *[5]int).

  ================================================================================
  PREGUNTAS FRECUENTES (FAQ)
  ================================================================================

  ❓ ¿Puedo cambiar el tamaño de un array después de declararlo?
  → No. El tamaño es fijo en tiempo de compilación. Usa slices para dinamismo.

  ❓ ¿Qué significa que un array sea "valor" en Go?
  → Que al asignar o pasar un array, se copian todos los elementos. No es una
    referencia como en otros lenguajes.

  ❓ ¿Cómo evito copiar un array grande al pasarlo a una función?
  → Pasa un puntero al array: func procesar(arr *[1000]int).

  ❓ ¿Por qué [3]int y [4]int son tipos diferentes?
  → Porque el tamaño es parte del tipo en Go. Esto permite verificación estricta
    en compilación.

  ❓ ¿Los arrays en Go son comparables siempre?
  → Solo si el tipo de los elementos es comparable. Por ejemplo, [3]map[int]int
    NO es comparable.

  ❓ ¿Qué pasa si accedo a un índice fuera del rango?
  → El compilador puede detectar índices constantes fuera de rango. En tiempo de
    ejecución, causa un panic: "index out of range".

  ❓ ¿Para qué sirve [...] en la declaración?
  → Para que el compilador infiera el tamaño a partir de la cantidad de elementos
    en el literal.

  ❓ ¿Cuál es la diferencia entre var a [5]int y a := [5]int{}?
  → Ambas inicializan con valor cero. La segunda es un literal vacío. El resultado
    es idéntico.

  ❓ ¿Se pueden tener arrays de arrays?
  → Sí, se llaman arrays multidimensionales: [3][4]int.

  ❓ ¿len() y cap() son diferentes en arrays?
  → No, para arrays siempre devuelven el mismo valor.
