  1. La Función append()
  ──────────────────────

  append añade elementos al final de un slice. Si hay capacidad, modifica el
  array subyacente. Si no, crea uno nuevo. SIEMPRE debes asignar el resultado:

      s = append(s, elemento)

  Firma: func append(s []T, xs ...T) []T

  ================================================================================
  EJEMPLO 1: append básico
  ================================================================================

  package main

  import "fmt"

  func main() {
      var s []int

      s = append(s, 10)
      fmt.Println("s:", s, "len:", len(s), "cap:", cap(s))

      s = append(s, 20)
      fmt.Println("s:", s, "len:", len(s), "cap:", cap(s))

      s = append(s, 30)
      fmt.Println("s:", s, "len:", len(s), "cap:", cap(s))
  }

  ================================================================================
  EJEMPLO 2: append con slice nil
  ================================================================================

  package main

  import "fmt"

  func main() {
      var s []int
      fmt.Println("Antes de append:", s, "nil:", s == nil)

      s = append(s, 1)
      fmt.Println("Después de append:", s, "nil:", s == nil)

      // append funciona perfectamente con slices nil
      // Go crea el array subyacente automáticamente
  }

  ================================================================================
  EJEMPLO 3: append variádico (múltiples elementos)
  ================================================================================

  package main

  import "fmt"

  func main() {
      s := []int{1, 2, 3}

      // Añadir múltiples elementos
      s = append(s, 4, 5, 6)
      fmt.Println("s:", s) // [1, 2, 3, 4, 5, 6]

      // Añadir otro slice con ...
      extras := []int{7, 8, 9}
      s = append(s, extras...)
      fmt.Println("s:", s) // [1, 2, 3, 4, 5, 6, 7, 8, 9]
  }

  ================================================================================
  EJEMPLO 4: append de strings
  ================================================================================

  package main

  import "fmt"

  func main() {
      var frutas []string
      frutas = append(frutas, "manzana", "pera", "uva")
      frutas = append(frutas, "kiwi", "naranja")
      fmt.Println(frutas)
  }

  2. El Patrón de Asignación
  ──────────────────────────

  ================================================================================
  EJEMPLO 5: append siempre debe asignarse
  ================================================================================

  package main

  import "fmt"

  func main() {
      s := make([]int, 0, 2)
      s = append(s, 1) // OK: hay cap
      s = append(s, 2) // OK: hay cap
      s = append(s, 3) // Realocación: s apunta a NUEVO array

      fmt.Println("s:", s)

      // ERROR COMÚN: no asignar el resultado
      t := make([]int, 0, 2)
      append(t, 1)   // ¡Esto NO modifica t!
      fmt.Println("t:", t) // [] (vacío)
  }

  ================================================================================
  EJEMPLO 6: append dentro de función
  ================================================================================

  package main

  import "fmt"

  func agregar(nums []int, vals ...int) []int {
      return append(nums, vals...)
  }

  func main() {
      s := []int{1, 2, 3}
      s = agregar(s, 4, 5, 6)
      fmt.Println(s) // [1, 2, 3, 4, 5, 6]
  }

  3. Rendimiento de append
  ────────────────────────

  ================================================================================
  EJEMPLO 7: append con preasignación vs sin preasignación
  ================================================================================

  package main

  import (
      "fmt"
      "time"
  )

  func main() {
      n := 10000000

      start := time.Now()
      sinPre := make([]int, 0)
      for i := 0; i < n; i++ {
          sinPre = append(sinPre, i)
      }
      fmt.Println("Sin preasignación:", time.Since(start))

      start = time.Now()
      conPre := make([]int, 0, n)
      for i := 0; i < n; i++ {
          conPre = append(conPre, i)
      }
      fmt.Println("Con preasignación:", time.Since(start))
  }

  4. Append y el Share del Array Subyacente
  ─────────────────────────────────────────

  ================================================================================
  EJEMPLO 9: Peligro de append con slices compartidos
  ================================================================================

  package main

  import "fmt"

  func main() {
      base := []int{1, 2, 3, 4, 5}
      s1 := base[:3] // [1, 2, 3], cap=5
      s2 := base[3:] // [4, 5], cap=2

      s1 = append(s1, 100) // Hay capacidad: modifica base[3]
      fmt.Println("base:", base) // [1, 2, 3, 100, 5]
      fmt.Println("s1:", s1)     // [1, 2, 3, 100]
      fmt.Println("s2:", s2)     // [100, 5] — ¡Cambió!

      s1 = append(s1, 200) // Hay capacidad: modifica base[4]
      fmt.Println("base:", base) // [1, 2, 3, 100, 200]
      fmt.Println("s2:", s2)     // [200, 5] — extraño

      s1 = append(s1, 300) // Ya no hay cap: REALOCA
      fmt.Println("base:", base) // [1, 2, 3, 100, 200] (intacto)
      fmt.Println("s1:", s1)     // [1, 2, 3, 100, 200, 300] (nuevo array)
  }

  5. La Función copy()
  ────────────────────

  copy copia elementos entre slices. Firma:

      func copy(dst, src []T) int

  • Copia como máximo min(len(dst), len(src)) elementos.
  • Retorna el número de elementos copiados.
  • Soporta superposición (como memmove).

  ================================================================================
  EJEMPLO 10: copy básico
  ================================================================================

  package main

  import "fmt"

  func main() {
      src := []int{1, 2, 3, 4, 5}
      dst := make([]int, 3)

      n := copy(dst, src)
      fmt.Printf("Copiados %d: %v\n", n, dst) // [1, 2, 3]
  }

  ================================================================================
  EJEMPLO 11: copy con destino más grande
  ================================================================================

  package main

  import "fmt"

  func main() {
      src := []int{10, 20, 30}
      dst := make([]int, 5)

      n := copy(dst, src)
      fmt.Printf("Copiados %d: %v\n", n, dst) // [10, 20, 30, 0, 0]
  }

  ================================================================================
  EJEMPLO 12: copy completa (clonar)
  ================================================================================

  package main

  import "fmt"

  func clonar(s []int) []int {
      clone := make([]int, len(s))
      copy(clone, s)
      return clone
  }

  func main() {
      original := []int{1, 2, 3, 4, 5}
      copia := clonar(original)

      copia[0] = 999
      fmt.Println("Original:", original)
      fmt.Println("Copia:", copia)
  }

  6. copy con Superposición
  ─────────────────────────

  ================================================================================
  EJEMPLO 13: copy superpuesto (shift derecho)
  ================================================================================

  package main

  import "fmt"

  func main() {
      s := []int{10, 20, 30, 40, 50, 60, 70}

      // Desplazar elementos 2 posiciones a la derecha
      // copy(s[2:], s[:]) — PERO esto es superpuesto
      copy(s[2:], s[:])
      fmt.Println("Después de copy(s[2:], s[:]):", s)
      // Resultado: [10, 20, 10, 20, 30, 40, 50]
  }

  ================================================================================
  EJEMPLO 14: copy superpuesto (shift izquierdo)
  ================================================================================

  package main

  import "fmt"

  func main() {
      s := []int{10, 20, 30, 40, 50, 60, 70}

      // Eliminar los 2 primeros elementos
      copy(s, s[2:])
      fmt.Println("Después de copy(s, s[2:]):", s)
      // [30, 40, 50, 60, 70, 60, 70]
  }

  ================================================================================
  EJEMPLO 15: copy entre slices de string
  ================================================================================

  package main

  import "fmt"

  func main() {
      src := []string{"a", "b", "c", "d", "e"}
      dst := make([]string, 5)

      n := copy(dst, src)
      fmt.Printf("Copiados %d: %v\n", n, dst)

      dst[0] = "Z"
      fmt.Println("src intacto:", src)
  }

  8. Técnicas Avanzadas con append y copy
  ──────────────────────────────────────

  ================================================================================
  EJEMPLO 17: Insertar elemento en posición
  ================================================================================

  package main

  import "fmt"

  func insertar(s []int, idx, valor int) []int {
      s = append(s, 0)       // Hacer espacio
      copy(s[idx+1:], s[idx:]) // Desplazar
      s[idx] = valor
      return s
  }

  func main() {
      s := []int{1, 2, 3, 4, 5}
      fmt.Println("Original:", s)

      s = insertar(s, 2, 999)
      fmt.Println("Insertado 999 en índice 2:", s)
      // [1, 2, 999, 3, 4, 5]
  }

  ================================================================================
  EJEMPLO 18: Eliminar elemento por índice
  ================================================================================

  package main

  import "fmt"

  func eliminar(s []int, idx int) []int {
      return append(s[:idx], s[idx+1:]...)
  }

  func main() {
      s := []int{1, 2, 3, 4, 5}
      fmt.Println("Original:", s)

      s = eliminar(s, 2)
      fmt.Println("Eliminado índice 2:", s)
      // [1, 2, 4, 5]
  }

  ================================================================================
  EJEMPLO 19: Eliminar con preservación de orden (versión eficiente)
  ================================================================================

  package main

  import "fmt"

  func eliminarOrden(s []int, idx int) []int {
      copy(s[idx:], s[idx+1:])
      return s[:len(s)-1]
  }

  func main() {
      s := []int{1, 2, 3, 4, 5}
      s = eliminarOrden(s, 2)
      fmt.Println(s) // [1, 2, 4, 5]
  }

  ================================================================================
  EJEMPLO 20: Eliminar sin preservar orden (rápido)
  ================================================================================

  package main

  import "fmt"

  func eliminarRapido(s []int, idx int) []int {
      s[idx] = s[len(s)-1]
      return s[:len(s)-1]
  }

  func main() {
      s := []int{1, 2, 3, 4, 5}
      s = eliminarRapido(s, 1)
      fmt.Println(s) // [1, 5, 3, 4] — orden no preservado
  }

  ================================================================================
  EJEMPLO 21: Filtrar in-place con copy
  ================================================================================

  package main

  import "fmt"

  func filtrar(s []int, fn func(int) bool) []int {
      n := 0
      for _, v := range s {
          if fn(v) {
              s[n] = v
              n++
          }
      }
      return s[:n]
  }

  func main() {
      s := []int{1, 2, 3, 4, 5, 6, 7, 8}

      // Filtrar pares
      pares := filtrar(s, func(v int) bool { return v%2 == 0 })
      fmt.Println("Pares:", pares)
  }

  ================================================================================
  EJEMPLO 22: Append concatenando slices
  ================================================================================

  package main

  import "fmt"

  func concatener(slices ...[]int) []int {
      total := 0
      for _, s := range slices {
          total += len(s)
      }
      resultado := make([]int, 0, total)
      for _, s := range slices {
          resultado = append(resultado, s...)
      }
      return resultado
  }

  func main() {
      a := []int{1, 2, 3}
      b := []int{4, 5}
      c := []int{6, 7, 8, 9}

      r := concatener(a, b, c)
      fmt.Println(r) // [1, 2, 3, 4, 5, 6, 7, 8, 9]
  }

  ================================================================================
  EJEMPLO 26: append con crecimiento controlado
  ================================================================================

  package main

  import "fmt"

  func appendConCapacidad(s []int, elems ...int) []int {
      needed := len(s) + len(elems)
      if needed > cap(s) {
          newCap := cap(s) * 2
          for newCap < needed {
              newCap *= 2
          }
          newS := make([]int, len(s), newCap)
          copy(newS, s)
          s = newS
      }
      return append(s, elems...)
  }

  func main() {
      s := make([]int, 0, 2)
      for i := 0; i < 100; i++ {
          s = appendConCapacidad(s, i)
      }
      fmt.Println("len:", len(s), "cap:", cap(s))
  }

  EJEMPLO 29: append en búsqueda binaria

  package main

  import "fmt"

  func insertarOrdenado(s []int, v int) []int {
      idx := 0
      for idx < len(s) && s[idx] < v {

  EJEMPLO 30: Rebanar y copiar para aislar

  package main

  import "fmt"

  func aislar(s []int) []int {
      // Retorna un slice independiente (no comparte array)
      resultado := make([]int, len(s))
      copy(resultado, s)
      return resultado
  }

  func main() {
      datos := []int{1, 2, 3, 4, 5}
      sub := aislar(datos[1:4])

      sub[0] = 999
      fmt.Println("datos:", datos) // [1, 2, 3, 4, 5]
      fmt.Println("sub:", sub)     // [999, 3, 4]
  }

  RESUMEN DEL CAPÍTULO
  ================================================================================

  • append(s, x) añade x al final de s; SIEMPRE asigna el resultado.
  • append es variádico: append(s, a, b, c) o append(s, otro...).
  • append con slice nil funciona (Go crea el array automáticamente).
  • Si no hay capacidad, append realoca y retorna un nuevo slice header.
  • copy(dst, src) copia como máximo min(len(dst), len(src)).
  • copy retorna el número de elementos copiados.
  • copy maneja superposición correctamente (como memmove).
  • Para clonar: clone := make([]T, len(orig)); copy(clone, orig).
  • append y copy son eficientes y se usan en combinación para insertar,
    eliminar y filtrar slices.

  PREGUNTAS FRECUENTES (FAQ)
  ================================================================================

  ❓ ¿Por qué append devuelve un slice en vez de modificarlo in-place?
  → Porque puede realocar el array subyacente. El slice original apuntaría a
    memoria vieja. Es más seguro retornar el nuevo header.

  ❓ ¿Qué pasa si no asigno el resultado de append?
  → El slice original no se modifica (o se modifica parcialmente si había
    capacidad, pero no se actualiza len).

  ❓ ¿copy es más rápido que un bucle manual?
  → Generalmente sí. copy está optimizada en ensamblador y usa instrucciones
    SIMD cuando es posible.

  ❓ ¿copy funciona con tipos distintos?
  → No. Ambos slices deben ser del mismo tipo.

  ❓ ¿Qué sucede si dst y src se superponen?
  → copy maneja superposición correctamente (se comporta como memmove).

  ❓ ¿Puedo usar append con arrays?
  → No. append solo funciona con slices.

  ❓ ¿Cuándo debo usar copy vs asignación directa?
  → copy cuando necesitas copiar múltiples elementos. Asignación directa
    cuando es un solo valor.

  ❓ ¿append crea una nueva copia del slice header?
  → Sí, pero es barato (3 palabras). El contenido del array subyacente puede
    o no copiarse dependiendo de la realocación.

  ❓ ¿Cómo añado al principio de un slice?
  → No hay prepend nativo. Usa: s = append([]T{v}, s...).

  ❓ ¿Cuál es la complejidad de append?
  → Amortizada O(1). La mayoría de las llamadas son O(1) y ocasionalmente O(n)
    cuando hay realocación.
