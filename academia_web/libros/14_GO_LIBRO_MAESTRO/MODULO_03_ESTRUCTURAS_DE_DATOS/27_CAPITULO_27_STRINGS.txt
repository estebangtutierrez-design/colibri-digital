  1. El Paquete strings
  ─────────────────────

  El paquete strings proporciona funciones para manipular cadenas UTF-8.
  No modifica las cadenas originales (los strings son inmutables en Go),
  sino que retorna nuevas cadenas.

  Las funciones principales se agrupan en:
  • Búsqueda: Contains, HasPrefix, HasSuffix, Index, LastIndex, Count
  • División: Split, SplitN, SplitAfter, Fields, Join
  • Transformación: ToUpper, ToLower, Title, Trim, TrimSpace, Replace
  • Comparación: Compare, EqualFold

  2. Búsqueda de Subcadenas
  ─────────────────────────

  ================================================================================
  EJEMPLO 1: Contains y ContainsAny
  ================================================================================

  package main

  import (
      "fmt"
      "strings"
  )

  func main() {
      texto := "Go Master Book es un curso completo"

      fmt.Println("Contains 'Master':", strings.Contains(texto, "Master"))
      fmt.Println("Contains 'Python':", strings.Contains(texto, "Python"))

      // ContainsAny: contiene algún carácter del conjunto
      fmt.Println("ContainsAny 'aeiou':", strings.ContainsAny(texto, "aeiou"))
      fmt.Println("ContainsAny 'xyz':", strings.ContainsAny(texto, "xyz"))
  }

  ================================================================================
  EJEMPLO 2: HasPrefix y HasSuffix
  ================================================================================

  package main

  import (
      "fmt"
      "strings"
  )

  func main() {
      url := "https://golang.org"

      fmt.Println("HasPrefix 'https://':", strings.HasPrefix(url, "https://"))
      fmt.Println("HasSuffix '.org':", strings.HasSuffix(url, ".org"))
      fmt.Println("HasSuffix '.com':", strings.HasSuffix(url, ".com"))

      // Validación de URLs
      if strings.HasPrefix(url, "https://") {
          fmt.Println("URL segura")
      }
  }

  ================================================================================
  EJEMPLO 3: Index, LastIndex, IndexAny

  3. División y Unión
  ───────────────────

  ================================================================================
  EJEMPLO 5: Split y SplitN
  ================================================================================

  package main

  import (
      "fmt"
      "strings"
  )

  func main() {
      csv := "pera,manzana,uva,kiwi,naranja"

      partes := strings.Split(csv, ",")
      fmt.Println("Split:", partes)

      // SplitN con límite
      partes2 := strings.SplitN(csv, ",", 3)
      fmt.Println("SplitN (3):", partes2)

      // SplitAfter: incluye el separador
      partes3 := strings.SplitAfter(csv, ",")
      fmt.Println("SplitAfter:", partes3)
  }

  ================================================================================
  EJEMPLO 6: Fields (divide por espacios)
  ================================================================================

  package main

  import (
      "fmt"
      "strings"
  )

  func main() {
      texto := "   Go   es   un   lenguaje   moderno   "
      palabras := strings.Fields(texto)
      fmt.Printf("Fields: %q\n", palabras)
      // ["Go", "es", "un", "lenguaje", "moderno"]

      // Diferencia con Split: Fields maneja múltiples espacios
      texto2 := "uno   dos\tres\ncuatro"
      fmt.Printf("Fields: %q\n", strings.Fields(texto2))
  }

  ================================================================================
  EJEMPLO 7: Join
  ================================================================================

  package main

  import (
      "fmt"
      "strings"
  )

  func main() {
      palabras := []string{"Go", "es", "increíble"}
      frase := strings.Join(palabras, " ")
      fmt.Println("Join con espacio:", frase)

      csv := strings.Join(palabras, ",")
      fmt.Println("Join con coma:", csv)

      ruta := strings.Join([]string{"usr", "local", "go"}, "/")
      fmt.Println("Join como ruta:", ruta)
  }

  ================================================================================
  EJEMPLO 12: Replace y ReplaceAll
  ================================================================================

  package main

  import (
      "fmt"
      "strings"
  )

  func main() {
      texto := "gato perro gato gato pez gato"

      // Replace(old, new, n) — n reemplazos
      fmt.Println("Replace (2):", strings.Replace(texto, "gato", "perro", 2))

      // ReplaceAll — todos los reemplazos
      fmt.Println("ReplaceAll:", strings.ReplaceAll(texto, "gato", "hamster"))

      // Eliminar (reemplazar por cadena vacía)
      fmt.Println("Eliminar gato:", strings.ReplaceAll(texto, "gato ", ""))
  }

  5. Comparación de Cadenas
  ─────────────────────────

  ================================================================================
  EJEMPLO 13: Compare y EqualFold
  ================================================================================

  package main

  import (
      "fmt"
      "strings"
  )

  func main() {
      // Compare: retorna -1, 0, 1
      fmt.Println("Compare('a', 'b'):", strings.Compare("a", "b")) // -1
      fmt.Println("Compare('a', 'a'):", strings.Compare("a", "a")) // 0
      fmt.Println("Compare('b', 'a'):", strings.Compare("b", "a")) // 1

      // EqualFold: comparación sin distinción de mayúsculas
      fmt.Println("EqualFold 'Go', 'go':", strings.EqualFold("Go", "go"))
      fmt.Println("EqualFold 'Hola', 'hola':", strings.EqualFold("Hola", "hola"))
  }

  6. Construcción Eficiente con Builder
  ─────────────────────────────────────

  strings.Builder es una forma eficiente de construir cadenas sin crear
  múltiples strings intermedios (como +=).

  ================================================================================
  EJEMPLO 14: Builder básico
  ================================================================================

  package main

  import (
      "fmt"
      "strings"
  )

  func main() {
      var sb strings.Builder

      sb.WriteString("Hola, ")
      sb.WriteString("Mundo")
      sb.WriteByte('!')
      sb.WriteRune('🎯')

      resultado := sb.String()
      fmt.Println(resultado)
      fmt.Printf("Longitud: %d bytes\n", sb.Len())
  }

  ================================================================================
  EJEMPLO 15: Builder con formato
  ================================================================================

  package main

  import (
      "fmt"
      "strings"
  )

  func main() {
      var sb strings.Builder

      for i := 0; i < 100; i++ {
          sb.WriteString(fmt.Sprintf("línea %d\n", i))
      }

      fmt.Printf("Builder generó %d bytes\n", sb.Len())
  }

  ================================================================================
  EJEMPLO 16: Builder vs concatenación (rendimiento)
  ================================================================================

  package main

  import (
      "fmt"
      "strings"
      "time"
  )

  func main() {
      n := 100000
      palabras := []string{"hola", "mundo", "golang", "es", "genial"}

      start := time.Now()
      resultado := ""
      for i := 0; i < n; i++ {
          resultado += palabras[i%len(palabras)] + " "
      }
      fmt.Println("Concat:", time.Since(start))

      start = time.Now()
      var sb strings.Builder
      sb.Grow(n * 10)
      for i := 0; i < n; i++ {
          sb.WriteString(palabras[i%len(palabras)])
          sb.WriteByte(' ')
      }
      _ = sb.String()
      fmt.Println("Builder:", time.Since(start))
  }

  ================================================================================
  EJEMPLO 18: Pad (relleno manual)
  ================================================================================

  package main

  import (
      "fmt"
      "strings"
  )

  func padLeft(s string, n int, c byte) string {
      if len(s) >= n {
          return s
      }
      return strings.Repeat(string(c), n-len(s)) + s
  }

  func padRight(s string, n int, c byte) string {
      if len(s) >= n {
          return s
      }
      return s + strings.Repeat(string(c), n-len(s))
  }

  func main() {
      fmt.Printf("%q\n", padLeft("42", 5, '0'))     // "00042"
      fmt.Printf("%q\n", padRight("Go", 10, '-'))   // "Go--------"
  }

  package main

      "fmt"

  )

  func main() {
      texto := "Go Master Book 2024!"

      // Reemplazar dígitos por '#'

  9. Validación de Cadenas
  ────────────────────────

  ================================================================================
  EJEMPLO 20: Validaciones comunes
  ================================================================================

  package main

  import (
      "fmt"
      "strings"
      "unicode"
  )

  func esEmailValido(email string) bool {
      if !strings.Contains(email, "@") {
          return false
      }
      partes := strings.Split(email, "@")
      if len(partes) != 2 || partes[0] == "" || partes[1] == "" {
          return false
      }
      if !strings.Contains(partes[1], ".") {
          return false
      }
      return true
  }

  func main() {
      emails := []string{"user@example.com", "invalido", "@falta.com", "user@", ""}
      for _, e := range emails {
          fmt.Printf("%-20s → válido: %v\n", e, esEmailValido(e))
      }
  }

  10. strings.Cut (Go 1.18+)
  ──────────────────────────

  ================================================================================
  EJEMPLO 21: Cut
  ================================================================================

  package main

  import (
      "fmt"
      "strings"
  )

  func main() {
      // Cut divide una cadena alrededor del primer separador
      email := "usuario@dominio.com"
      antes, despues, encontrado := strings.Cut(email, "@")
      fmt.Printf("Antes: %s, Después: %s, Encontrado: %v\n",
          antes, despues, encontrado)

      // Cut es más simple y rápido que Split para este caso
      ruta := "/usr/local/go/bin"
      dir, archivo, _ := strings.Cut(ruta, "/local")
      fmt.Printf("Dir: %s, Archivo: %s\n", dir, archivo)
  }

  13. Limpieza de Texto
  ─────────────────────

  ================================================================================
  EJEMPLO 24: Limpiar y normalizar
  ================================================================================

  package main

  import (
      "fmt"
      "strings"
  )

  func limpiarTexto(texto string) string {
      texto = strings.TrimSpace(texto)
      texto = strings.ReplaceAll(texto, "\t", " ")
      for strings.Contains(texto, "  ") {
          texto = strings.ReplaceAll(texto, "  ", " ")
      }
      return texto
  }

  func main() {
      sucio := "   Hola\t\t  Mundo   Go   "
      limpio := limpiarTexto(sucio)
      fmt.Printf("Original: %q\n", sucio)
      fmt.Printf("Limpio: %q\n", limpio)
  }

  strings.Map(fn, s)            string     transformar runas

  RESUMEN DEL CAPÍTULO
  ================================================================================

  • El paquete strings ofrece decenas de funciones para manipular cadenas.
  • Las cadenas en Go son INMUTABLES; las funciones retornan nuevas cadenas.
  • Contains, HasPrefix, HasSuffix, Index, Count para búsqueda.
  • Split, Fields, Join para dividir y unir cadenas.
  • ToUpper, ToLower, Trim, Replace, Repeat para transformación.
  • Compare y EqualFold para comparar cadenas.
  • strings.Builder para construir cadenas eficientemente sin múltiples
    asignaciones de memoria.
  • Cut (Go 1.18+) simplifica dividir cadenas en dos partes.
  • Las funciones son seguras para UTF-8.

  ================================================================================
  PREGUNTAS FRECUENTES (FAQ)
  ================================================================================

  ❓ ¿Las funciones de strings modifican la cadena original?
  → No. Los strings son inmutables. Siempre retornan una nueva cadena.

  ❓ ¿Cuándo debo usar strings.Builder?
  → Cuando concatenes muchas cadenas (decenas o más). Para unas pocas, la
    concatenación con + es suficiente.

  ❓ ¿Split y Fields son equivalentes?
  → No. Fields divide por cualquier espacio (tab, newline, space) y colapsa
    múltiples. Split divide por un separador exacto.

  ❓ ¿Cuál es la diferencia entre Compare y los operadores <, >, ==?
  → Compare es más eficiente y da -1, 0, 1. Los operadores son equivalentes
    en resultado pero Compare es más explícito.

  ❓ ¿ContainsAny busca la subcadena completa o caracteres individuales?
  → Busca caracteres individuales del conjunto. Retorna true si encuentra
    alguno.

  ❓ ¿Cómo elimino espacios al inicio y final?
  → strings.TrimSpace(s).

  ❓ ¿Cómo convierto un string a mayúsculas?
  → strings.ToUpper(s).

  ❓ ¿Replace y ReplaceAll cuál es la diferencia?
  → Replace permite especificar el número de reemplazos; ReplaceAll reemplaza
    todos.

  ❓ ¿Cut está disponible en todas las versiones de Go?
  → Cut se introdujo en Go 1.18. Para versiones anteriores, usa SplitN con
    límite 2.

  ❓ ¿strings.Fields maneja UTF-8?
  → Sí. Todas las funciones de strings trabajan con UTF-8.
