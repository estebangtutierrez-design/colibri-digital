1. DECLARACIÓN DE VARIABLES
===============================================================================

Go ofrece varias formas de declarar variables:

1.1 Declaración con var (explícita)

```go
var nombre string = "Lucas"
var edad int = 30
var altura float64 = 1.75
var activo bool = true
```

1.2 Declaración con var (inferencia de tipo)

```go
var nombre = "Lucas"    // string
var edad = 30           // int
var altura = 1.75       // float64
var activo = true       // bool
```

1.3 Declaración corta (:=)

```go
nombre := "Lucas"       // string
edad := 30              // int
altura := 1.75          // float64
activo := true          // bool
```

La declaración corta `:=` solo se puede usar dentro de funciones.
Fuera de funciones, debes usar `var`.

1.4 Declaración múltiple

```go
var x, y int = 10, 20
var a, b, c = 1, "hola", true

nombre, edad := "Ana", 25
```

1.5 Declaración en bloque

```go
var (
    nombre   string = "Lucas"
    edad     int    = 30
    altura   float64
    activo   bool
)
```

===============================================================================
2. VALORES CERO
===============================================================================

En Go, todas las variables tienen un valor por defecto si no se
inicializan explícitamente. Esto elimina los bugs de variables
no inicializadas (comunes en C/C++).

```go
package main

import "fmt"

func main() {
    var numero int       // 0
    var decimal float64  // 0
    var texto string     // "" (vacío)
    var ok bool          // false

    fmt.Printf("int:     '%d'\n", numero)
    fmt.Printf("float64: '%f'\n", decimal)
    fmt.Printf("string:  '%s'\n", texto)
    fmt.Printf("bool:    '%t'\n", ok)
}
```

Tabla de valores cero:
| Tipo      | Valor cero |
|-----------|------------|
| int       | 0          |
| float64   | 0.0        |
| bool      | false      |
| string    | ""         |
| byte      | 0          |
| rune      | 0          |
| slice     | nil        |
| map       | nil        |
| puntero   | nil        |
| struct    | Campos en cero |
| interface | nil        |

```go
package main

import "fmt"

func main() {
    var (
        entero   int
        flotante float64
        cadena   string
        booleano bool
        slice    []int
        puntero  *int
    )

    fmt.Printf("int:       %v\n", entero)
    fmt.Printf("float64:   %v\n", flotante)
    fmt.Printf("string:    %q\n", cadena)
    fmt.Printf("bool:      %v\n", booleano)
    fmt.Printf("slice:     %v\n", slice)
    fmt.Printf("puntero:   %v\n", puntero)

    // nil no es lo mismo que cero
    fmt.Printf("slice == nil: %v\n", slice == nil)
    fmt.Printf("puntero == nil: %v\n", puntero == nil)
}
```

3. TIPOS NUMÉRICOS ENTEROS
===============================================================================

Go tiene una familia completa de tipos enteros:

3.1 Enteros con signo (int)

| Tipo  | Tamaño | Rango                      |
|-------|--------|----------------------------|
| int8  | 8 bits | -128 a 127                 |
| int16 | 16 bits| -32768 a 32767             |
| int32 | 32 bits| -2147483648 a 2147483647   |
| int64 | 64 bits| -9223372036854775808 a 9223372036854775807 |
| int   | 32/64  | Depende de la plataforma   |

```go
package main

import (
    "fmt"
    "math"
    "unsafe"
)

func main() {
    var a int8 = 127
    var b int16 = 32767
    var c int32 = 2147483647
    var d int64 = 9223372036854775807
    var e int = 42

    fmt.Printf("int8:   %d (%d bytes)\n", a, unsafe.Sizeof(a))
    fmt.Printf("int16:  %d (%d bytes)\n", b, unsafe.Sizeof(b))
    fmt.Printf("int32:  %d (%d bytes)\n", c, unsafe.Sizeof(c))
    fmt.Printf("int64:  %d (%d bytes)\n", d, unsafe.Sizeof(d))
    fmt.Printf("int:    %d (%d bytes)\n", e, unsafe.Sizeof(e))

    // Valores máximos y mínimos
    fmt.Println("Max int8:", math.MaxInt8)
    fmt.Println("Min int8:", math.MinInt8)
    fmt.Println("Max int16:", math.MaxInt16)
    fmt.Println("Max int32:", math.MaxInt32)
    fmt.Println("Max int64:", math.MaxInt64)
}
```

3.2 Enteros sin signo (uint)

| Tipo   | Tamaño | Rango          |
|--------|--------|----------------|
| uint8  | 8 bits | 0 a 255        |
| uint16 | 16 bits| 0 a 65535      |
| uint32 | 32 bits| 0 a 4294967295 |
| uint64 | 64 bits| 0 a 18446744073709551615 |
| uint   | 32/64  | Depende de la plataforma |

```go
package main

import (
    "fmt"
    "math"
)

func main() {
    var a uint8 = 255
    var b uint16 = 65535
    var c uint32 = 4294967295
    var d uint64 = 18446744073709551615

    fmt.Printf("uint8:  %d\n", a)
    fmt.Printf("uint16: %d\n", b)
    fmt.Printf("uint32: %d\n", c)
    fmt.Printf("uint64: %d\n", d)

    fmt.Println("Max uint8:", math.MaxUint8)
    fmt.Println("Max uint16:", math.MaxUint16)
    fmt.Println("Max uint32:", math.MaxUint32)
    fmt.Println("Max uint64:", math.MaxUint64)

    // Error: valor negativo en uint
    // var negativo uint8 = -1 // Error de compilación
}
```

3.3 byte y rune (alias)

```go
// byte es un alias de uint8
var b byte = 255

// rune es un alias de int32 (representa un código Unicode)
var r rune = 'A'
var emoji rune = '😀'

fmt.Printf("byte:  %d (es uint8)\n", b)
fmt.Printf("rune:  %c (código: %d)\n", r, r)
fmt.Printf("emoji: %c (código: %d)\n", emoji, emoji)
```

3.4 ¿Cuándo usar cada tipo?

- Usa `int` para la mayoría de casos (tamaño natural de la plataforma).
- Usa `int8/int16/int32/int64` cuando necesites un rango específico.
- Usa `uint` solo para cantidades que no pueden ser negativas.
- Usa `byte` para datos binarios o caracteres ASCII.
- Usa `rune` para caracteres Unicode.

```go
package main

import "fmt"

func main() {
    // Recomendado: int por defecto
    edad := 30
    contador := 0

    // Específico: cuando sabes el rango
    var temperatura int8 = -10  // Puede ser negativa, rango pequeño
    var año uint16 = 2026       // Solo positiva, rango suficiente
    var codigo byte = 65        // Un byte de datos

    fmt.Printf("edad: %d, contador: %d\n", edad, contador)
    fmt.Printf("temp: %d, año: %d, codigo: %c\n", temperatura, año, codigo)
}
```

4. TIPOS NUMÉRICOS FLOTANTES
===============================================================================

```go
package main

import (
    "fmt"
    "math"
)

func main() {
    var flotante32 float32 = 3.141592653589793
    var flotante64 float64 = 3.141592653589793

    fmt.Printf("float32: %.20f\n", flotante32) // Menos precisión
    fmt.Printf("float64: %.20f\n", flotante64) // Más precisión

    // Precisión
    fmt.Println("Max float32:", math.MaxFloat32)
    fmt.Println("Smallest float32:", math.SmallestNonzeroFloat32)
    fmt.Println("Max float64:", math.MaxFloat64)
    fmt.Println("Smallest float64:", math.SmallestNonzeroFloat64)

    // Valores especiales
    var infinito float64 = math.Inf(1)
    var menosInfinito float64 = math.Inf(-1)
    var nan float64 = math.NaN()

    fmt.Println("Infinito:", infinito)
    fmt.Println("-Infinito:", menosInfinito)
    fmt.Println("NaN:", nan)

    // Precaución con comparaciones de floats
    a := 0.1
    b := 0.2
    c := a + b
    fmt.Printf("0.1 + 0.2 = %.20f\n", c)
    fmt.Println("¿Es exactamente 0.3?", c == 0.3)
    fmt.Println("¿Cerca de 0.3?", math.Abs(c-0.3) < 1e-10)
}
```

Reglas:
- Usa `float64` por defecto (es el tipo inferido).
- Usa `float32` solo cuando necesites ahorrar memoria (gráficos, audio).
- Nunca compares floats con `==` directamente.
- Usa `math.Abs(diferencia) < epsilon` para comparar.

5. TIPO BOOLEANO
===============================================================================

```go
package main

import "fmt"

func main() {
    var activo bool = true
    var completado bool

    fmt.Println("activo:", activo)
    fmt.Println("completado:", completado) // false

    // Operadores lógicos
    a := true
    b := false

    fmt.Println("a && b:", a && b) // AND
    fmt.Println("a || b:", a || b) // OR
    fmt.Println("!a:", !a)         // NOT

    // Usos comunes
    edad := 18
    mayorDeEdad := edad >= 18
    fmt.Println("¿Es mayor de edad?", mayorDeEdad)

    // Bool desde comparación
    nombre := "Go"
    esGo := nombre == "Go"
    fmt.Println("¿Es Go?", esGo)

    // Zero value es false
    var decision bool
    if decision {
        fmt.Println("Esto no se imprime")
    } else {
        fmt.Println("decision es false por defecto")
    }
}
```

Go no permite convertir implícitamente bool a int (a diferencia de C).

```go
// Esto NO compila:
// var x int = 1
// if x { } // Error: x no es bool

// Correcto:
var x int = 1
if x > 0 {
    fmt.Println("x es positivo")
}
```

6. TIPO STRING
===============================================================================

Las cadenas en Go son secuencias de bytes (UTF-8) inmutables.

```go
package main

import (
    "fmt"
    "unicode/utf8"
)

func main() {
    // Declaración
    saludo := "Hola, Go!"
    var mensaje string = "Bienvenido"

    fmt.Println(saludo)
    fmt.Println(mensaje)

    // Longitud en bytes vs runas
    texto := "Hola, 世界"
    fmt.Println("Longitud en bytes:", len(texto))
    fmt.Println("Longitud en runas:", utf8.RuneCountInString(texto))

    // Indexación (da bytes, no runas)
    fmt.Println("Primer byte:", texto[0])

    // Recorrer por bytes
    for i := 0; i < len(texto); i++ {
        fmt.Printf("byte[%d]: %d (%c)\n", i, texto[i], texto[i])
    }

    // Recorrer por runas
    for i, r := range texto {
        fmt.Printf("runa[%d]: %c (código: %U)\n", i, r, r)
    }

    // Concatenación
    a := "Hola, "
    b := "mundo"
    c := a + b
    fmt.Println(c)

    // Inmutabilidad
    nombre := "Lucas"
    // nombre[0] = 'M' // Error: no se puede modificar
    nombre = "Marcos" // Esto crea una nueva cadena

    // Multilínea
    parrafo := `Esto es un
texto multilínea
con comillas invertidas.`

    fmt.Println(parrafo)

    // Caracteres escapados
    ruta := "C:\\Users\\Lucas\\go"
    fmt.Println(ruta)

    // Raw string (sin escapes)
    rutaRaw := `C:\Users\Lucas\go`
    fmt.Println(rutaRaw)
}
```

7. CONVERSIÓN DE TIPOS
===============================================================================

Go no tiene conversión implícita entre tipos. Todo debe ser explícito.

```go
package main

import (
    "fmt"
    "strconv"
)

func main() {
    // Conversión entre enteros
    var x int8 = 100
    var y int16 = int16(x) // Conversión explícita
    fmt.Println(y)

    // Conversión a float
    var entero int = 42
    var flotante float64 = float64(entero)
    fmt.Println(flotante)

    // Float a int (trunca)
    var pi float64 = 3.14159
    var enteroPi int = int(pi)
    fmt.Println(enteroPi) // 3

    // Int a string (con strconv)
    numero := 42
    texto := strconv.Itoa(numero)
    fmt.Printf("Tipo: %T, Valor: %s\n", texto, texto)

    // String a int
    cadena := "123"
    valor, err := strconv.Atoi(cadena)
    if err != nil {
        fmt.Println("Error:", err)
    } else {
        fmt.Println("Valor:", valor)
    }

    // String a float
    precio := "19.99"
    precioFloat, err := strconv.ParseFloat(precio, 64)
    if err != nil {
        fmt.Println("Error:", err)
    } else {
        fmt.Printf("Precio: %.2f\n", precioFloat)
    }

    // Formatear con Sprintf
    edad := 30
    mensaje := fmt.Sprintf("Tengo %d años", edad)
    fmt.Println(mensaje)
}
```

7.1 Reglas de conversión

```go
package main

import (
    "fmt"
    "math"
)

func main() {
    // De entero pequeño a grande (seguro)
    var a int8 = 100
    var b int16 = int16(a)
    var c int32 = int32(b)
    var d int64 = int64(c)
    fmt.Println(d)

    // De entero grande a pequeño (pierde datos)
    var grande int64 = math.MaxInt64
    var pequeño int32 = int32(grande) // Trunca
    fmt.Printf("Original: %d, Truncado: %d\n", grande, pequeño)

    // De signed a unsigned
    var negativo int8 = -1
    var positivo uint8 = uint8(negativo) // 255 (complemento a 2)
    fmt.Printf("negativo: %d, positivo: %d\n", negativo, positivo)

    // De float a int (trunca hacia cero)
    f := 3.99
    i := int(f)
    fmt.Printf("float: %.2f, int: %d\n", f, i)

    // Conversión entre tipos no compatibles requiere paquete strconv
    // var s string = string(65) // Esto da "A" (runa), no "65"
    s := string(65)
    fmt.Println(s) // "A", no "65"
}
```

8. INFERENCIA DE TIPOS
===============================================================================

Go infiere el tipo en las declaraciones cortas y en var sin tipo explícito.

```go
package main

import (
    "fmt"
    "reflect"
)

func main() {
    // Inferencia básica
    nombre := "Lucas"       // string
    edad := 30              // int
    altura := 1.75          // float64
    activo := true          // bool

    fmt.Printf("%T %T %T %T\n", nombre, edad, altura, activo)

    // Inferencia con expresiones
    suma := 10 + 20         // int
    promedio := float64(suma) / 3.0  // float64
    fmt.Printf("suma: %T, promedio: %T\n", suma, promedio)

    // Usando reflect para ver tipos
    valor := 42
    fmt.Println("Tipo:", reflect.TypeOf(valor))

    // El tipo depende del literal
    x := 42          // int
    y := 42.0        // float64
    z := 42 + 0i     // complex128
    fmt.Printf("%T %T %T\n", x, y, z)

    // unsigned inferido
    // u := 42     // int, no uint (por defecto int)
    // Si quieres uint, debes ser explícito
    var u uint = 42
    fmt.Println(u)
}
```

9. DECLARACIÓN VS ASIGNACIÓN
===============================================================================

```go
package main

import "fmt"

func main() {
    // Declaración con var
    var mensaje string
    mensaje = "Hola"
    fmt.Println(mensaje)

    // Declaración + asignación
    var saludo string = "Hola"
    fmt.Println(saludo)

    // Declaración corta (solo dentro de funciones)
    nombre := "Lucas"
    fmt.Println(nombre)

    // := declara y asigna (requiere al menos una variable nueva)
    x := 10
    x, y := 20, 30   // x ya existe, se reasigna; y es nueva
    fmt.Println(x, y)

    // z := "texto"   // Error si z ya existe en el mismo ámbito

    // Múltiples variables
    a, b, c := 1, 2, 3
    fmt.Println(a, b, c)

    // Intercambio de valores
    i, j := 10, 20
    i, j = j, i  // Ahora i=20, j=10
    fmt.Println(i, j)
}
```

10. CONSTANTES Y TIPOS ESPECIALES
===============================================================================

```go
package main

import "fmt"

func main() {
    // Constantes tipadas
    const Pi float64 = 3.14159
    const Nombre string = "Lucas"

    // Constantes no tipadas (toman el tipo según el contexto)
    const Edad = 30           // tipo implícito
    const Saludo = "Hola"     // tipo implícito

    fmt.Println(Pi, Nombre, Edad, Saludo)

    // Los tipos complejos no pueden ser constantes
    // const s []int = []int{1,2,3} // Error

    // Tipo puntero
    var x int = 42
    var p *int = &x
    fmt.Println("Valor:", x)
    fmt.Println("Dirección:", p)
    fmt.Println("Valor desde puntero:", *p)

    // Tipo runa (carácter Unicode)
    var letra rune = 'A'
    var emoji rune = '🚀'
    fmt.Printf("Letra: %c, Código: %d\n", letra, letra)
    fmt.Printf("Emoji: %c, Código: %d\n", emoji, emoji)

    // Tipo byte (uint8)
    var b byte = 'A'
    fmt.Printf("byte: %c = %d\n", b, b)
}
```

RESUMEN DEL CAPÍTULO
===============================================================================

- Las variables se declaran con `var` o con `:=` (declaración corta).
- Go tiene valores cero para todas las variables: 0, false, "".
- Tipos enteros: int8, int16, int32, int64, int, uint8-uint64, uint.
- byte = uint8, rune = int32 (caracteres Unicode).
- float32 y float64 para números decimales (usar float64 por defecto).
- bool para valores booleanos (true/false).
- string para cadenas UTF-8 inmutables.
- No hay conversión implícita entre tipos, todo es explícito.
- Usa `strconv` para convertir entre strings y números.
- La inferencia de tipos asigna int, float64, string y bool por defecto.

===============================================================================
PREGUNTAS FRECUENTES
===============================================================================

❓ **¿Por qué Go no tiene conversión implícita de tipos?**
Para evitar errores sutiles. Las conversiones implícitas son fuente
común de bugs en C/C++. Go prefiere la claridad: toda conversión
debe ser explícita y visible.

❓ **¿Debo usar int, int32 o int64?**
Usa `int` por defecto. Es el tipo natural de la plataforma (32 bits
en sistemas de 32 bits, 64 bits en sistemas de 64 bits). Usa tipos
específicos solo cuando necesites un rango concreto o estés
trabajando con datos binarios.

❓ **¿Cuál es la diferencia entre byte y uint8?**
Ninguna. `byte` es un alias de `uint8`. Se usa `byte` cuando el
valor representa un carácter ASCII o un byte de datos. Se usa
`uint8` cuando el valor es un número sin signo de 8 bits.

❓ **¿Qué es un rune?**
Un `rune` es un alias de `int32` que representa un punto de código
Unicode. Se usa para trabajar con caracteres que pueden ocupar
múltiples bytes en UTF-8.

❓ **¿Las cadenas en Go son mutables?**
No. Las cadenas son inmutables. No puedes modificar un carácter
individual. Para construir cadenas dinámicamente, usa
`strings.Builder` o slices de bytes.

❓ **¿Qué significa que un string sea UTF-8?**
Que internamente se almacena como una secuencia de bytes en
codificación UTF-8. Los caracteres ASCII (0-127) ocupan 1 byte.
Los caracteres acentuados ocupan 2 bytes. Los emojis ocupan 4 bytes.
La longitud en bytes no necesariamente coincide con la cantidad
de caracteres.
