  BIENVENIDA
  -------------------------------------------------------------------------------
  Bienvenido al Capítulo 55 de GO MASTER BOOK. El formato CSV (Comma-
  Separated Values) es el estándar de facto para el intercambio de
  datos tabulares. Go ofrece el paquete encoding/csv que maneja las
  complejidades del formato: comillas, escapes, delimitadores variables
  y diferentes estilos de fin de línea. Aprenderás a leer y escribir
  CSV de forma correcta y eficiente.

  FRASE CLAVE
  -------------------------------------------------------------------------------
  "CSV parece sencillo hasta que un campo contiene una coma o
   un salto de línea."

  OBJETIVOS DEL CAPÍTULO

  ✓ Trabajar con CSV que tiene cabeceras.

  1. INTRODUCCIÓN A encoding/csv
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  El paquete encoding/csv lee y escribe archivos CSV (RFC 4180).
  Sus dos tipos principales son Reader y Writer.

  Reglas RFC 4180:
  - Campos separados por coma.
  - Registros separados por nueva línea.
  - Campos con comas, comillas o saltos de línea van entre comillas dobles.
  - Las comillas dentro de un campo se escapan duplicándolas ("").

  2. LEYENDO CSV CON csv.NewReader
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  ```go
  package main

  import (
      "encoding/csv"
      "fmt"
      "os"
  )

  func main() {
      f, err := os.Open("datos.csv")
      if err != nil {
          fmt.Println("Error:", err)
          return
      }
      defer f.Close()

      reader := csv.NewReader(f)
      registros, err := reader.ReadAll()
      if err != nil {
          fmt.Println("Error al leer CSV:", err)
          return
      }

      for i, registro := range registros {
          fmt.Printf("Fila %d: %v\n", i, registro)
      }
  }
  ```

  Si el archivo datos.csv contiene:
  ```
  Nombre,Edad,Ciudad
  Ana García,30,Madrid
  Luis Pérez,25,"Bogotá, Colombia"
  María López,28,"Buenos Aires"
  ```

  La salida será:
  ```
  Fila 0: [Nombre Edad Ciudad]
  Fila 1: [Ana García 30 Madrid]
  Fila 2: [Luis Pérez 25 Bogotá, Colombia]
  Fila 3: [María López 28 Buenos Aires]
  ```

  3. LEYENDO FILA POR FILA CON Read
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  Para archivos grandes, es mejor leer fila por fila:

  ```go
  package main

  import (
      "encoding/csv"
      "fmt"
      "os"
  )

  func main() {
      f, err := os.Open("grande.csv")
      if err != nil {
          fmt.Println("Error:", err)
          return
      }
      defer f.Close()

      reader := csv.NewReader(f)
      lineas := 0

      for {
          registro, err := reader.Read()
          if err != nil {
              break
          }
          lineas++
          fmt.Printf("Línea %d: %s | %s\n", lineas, registro[0], registro[1])
      }

      if err != nil; err != nil {
          // err será io.EOF al final
          fmt.Printf("Total de líneas: %d\n", lineas)
      }
  }
  ```

  4. CONFIGURANDO EL READER
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  ```go
  type Reader struct {
      Comma            rune // delimitador de campos (por defecto ',')
      Comment          rune // carácter de comentario (por defecto 0, sin comentarios)
      FieldsPerRecord  int  // número esperado de campos (-1 = variable)
      LazyQuotes       bool // permite comillas sin cerrar
      TrimLeadingSpace bool // elimina espacios iniciales
      ReuseRecord      bool // reutiliza slice para rendimiento
      TrailingComma    bool // permite coma final (no estándar)
  }
  ```

  Ejemplo con opciones:

  ```go
  package main

  import (
      "encoding/csv"
      "fmt"
      "os"
  )

  func main() {
      f, _ := os.Open("datos.tsv")
      defer f.Close()

      reader := csv.NewReader(f)
      reader.Comma = '\t'       // archivo TSV (tab-separated)
      reader.Comment = '#'      // líneas que empiezan con # se ignoran
      reader.TrimLeadingSpace = true
      reader.FieldsPerRecord = -1 // permite número variable de campos
      reader.LazyQuotes = true

      registros, _ := reader.ReadAll()
      for _, r := range registros {
          fmt.Println(r)
      }
  }
  ```

  5. ESCRIBIENDO CSV CON csv.NewWriter
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  ```go
  package main

  import (
      "encoding/csv"
      "fmt"
      "os"
  )

  func main() {
      f, err := os.Create("salida.csv")
      if err != nil {
          fmt.Println("Error:", err)
          return
      }
      defer f.Close()

      writer := csv.NewWriter(f)
      defer writer.Flush() // IMPORTANTE: flush al final

      // Escribir cabecera
      writer.Write([]string{"Nombre", "Edad", "Ciudad"})

      // Escribir datos
      writer.Write([]string{"Ana García", "30", "Madrid"})
      writer.Write([]string{"Luis Pérez", "25", "Bogotá, Colombia"})
      writer.Write([]string{"María López", "28", "Buenos Aires"})

      // Verificar errores
      if err := writer.Error(); err != nil {
          fmt.Println("Error al escribir CSV:", err)
          return
      }
      fmt.Println("CSV escrito exitosamente")
  }
  ```

  Resultado (salida.csv):
  ```
  Nombre,Edad,Ciudad
  Ana García,30,Madrid
  Luis Pérez,25,"Bogotá, Colombia"
  María López,28,Buenos Aires
  ```

  6. ESCRIBIENDO TODAS LAS FILAS CON WriteAll
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  ```go
  package main

  import (
      "encoding/csv"
      "fmt"
      "os"
  )

  func main() {
      datos := [][]string{
          {"Producto", "Precio", "Cantidad"},
          {"Laptop", "1250.99", "5"},
          {"Mouse", "25.50", "20"},
          {"Teclado", "89.99", "15"},
      }

      f, err := os.Create("productos.csv")
      if err != nil {
          fmt.Println("Error:", err)
          return
      }
      defer f.Close()

      writer := csv.NewWriter(f)
      err = writer.WriteAll(datos)
      if err != nil {
          fmt.Println("Error:", err)
          return
      }
      fmt.Println("CSV escrito con WriteAll")
  }
  ```

  WriteAll internamente llama a Write para cada fila y luego a Flush.

  7. CONFIGURANDO EL WRITER
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  ```go
  type Writer struct {
      Comma   rune // delimitador de campos (por defecto ',')
      UseCRLF bool // usa \r\n en lugar de \n
  }
  ```

  Ejemplo:

  ```go
  writer := csv.NewWriter(f)
  writer.Comma = ';'    // punto y coma como delimitador
  writer.UseCRLF = true // saltos de línea Windows
  ```

  Para TSV:

  ```go
  writer := csv.NewWriter(f)
  writer.Comma = '\t'
  ```

  8. CSV CON CABECERAS — ACCESO POR NOMBRE
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  ```go
  package main

  import (
      "encoding/csv"
      "fmt"
      "os"
  )

  type Persona struct {
      Nombre string
      Edad   string
      Ciudad string
  }

  func leerCSVCabeceras(ruta string) ([]Persona, error) {
      f, err := os.Open(ruta)
      if err != nil {
          return nil, err
      }
      defer f.Close()

      reader := csv.NewReader(f)

      // Leer cabeceras
      cabeceras, err := reader.Read()
      if err != nil {
          return nil, err
      }

      // Mapa de índice por nombre de columna
      indices := make(map[string]int)
      for i, h := range cabeceras {
          indices[h] = i
      }

      // Leer datos
      var personas []Persona
      for {
          fila, err := reader.Read()
          if err != nil {
              break
          }
          personas = append(personas, Persona{
              Nombre: fila[indices["Nombre"]],
              Edad:   fila[indices["Edad"]],
              Ciudad: fila[indices["Ciudad"]],
          })
      }
      return personas, nil
  }

  func main() {
      personas, err := leerCSVCabeceras("personas.csv")
      if err != nil {
          fmt.Println("Error:", err)
          return
      }
      for _, p := range personas {
          fmt.Printf("%s, %s años, %s\n", p.Nombre, p.Edad, p.Ciudad)
      }
  }
  ```

  9. ESCRIBIENDO CSV DESDE STRUCTS
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  ```go
  package main

  import (
      "encoding/csv"
      "fmt"
      "os"
      "reflect"
  )

  type Producto struct {
      ID       int     `csv:"id"`
      Nombre   string  `csv:"nombre"`
      Precio   float64 `csv:"precio"`
      Stock    int     `csv:"stock"`
  }

  func EscribirCSVDesdeStructs[T any](ruta string, datos []T) error {
      f, err := os.Create(ruta)
      if err != nil {
          return err
      }
      defer f.Close()

      writer := csv.NewWriter(f)
      defer writer.Flush()

      // Obtener tags de los campos
      t := reflect.TypeOf((*T)(nil)).Elem()
      var cabeceras []string
      for i := 0; i < t.NumField(); i++ {
          tag := t.Field(i).Tag.Get("csv")
          if tag == "" {
              tag = t.Field(i).Name
          }
          cabeceras = append(cabeceras, tag)
      }

      // Escribir cabeceras
      writer.Write(cabeceras)

      // Escribir datos
      for _, item := range datos {
          v := reflect.ValueOf(item)
          var fila []string
          for i := 0; i < v.NumField(); i++ {
              fila = append(fila, fmt.Sprintf("%v", v.Field(i).Interface()))
          }
          writer.Write(fila)
      }

      return writer.Error()
  }

  func main() {
      productos := []Producto{
          {1, "Laptop", 1250.99, 10},
          {2, "Mouse", 25.50, 50},
          {3, "Teclado", 89.99, 30},
      }

      err := EscribirCSVDesdeStructs("productos.csv", productos)
      if err != nil {
          fmt.Println("Error:", err)
      } else {
          fmt.Println("Productos exportados a CSV")
      }
  }
  ```

  10. MANEJO DE ERRORES COMUNES EN CSV
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  ```go
  package main

  import (
      "encoding/csv"
      "fmt"
      "os"
      "strings"
  )

  func analizarCSV(ruta string) {
      f, err := os.Open(ruta)
      if err != nil {
          fmt.Println("Error al abrir:", err)
          return
      }
      defer f.Close()

      reader := csv.NewReader(f)
      reader.FieldsPerRecord = -1 // permite variable
      reader.LazyQuotes = true
      reader.TrimLeadingSpace = true

      numLinea := 0
      for {
          registro, err := reader.Read()
          if err != nil {
              if err.Error() == "EOF" {
                  break
              }
              // Error de campo, posiblemente
              if strings.Contains(err.Error(), "wrong number of fields") {
                  fmt.Printf("Línea %d: número incorrecto de campos: %v\n", numLinea+1, registro)
                  continue
              }
              if strings.Contains(err.Error(), "bare") {
                  fmt.Printf("Línea %d: comillas sin cerrar\n", numLinea+1)
                  continue
              }
              fmt.Printf("Línea %d: error %v\n", numLinea+1, err)
              continue
          }
          numLinea++
          fmt.Printf("OK línea %d: %v\n", numLinea, registro)
      }
  }

  func main() {
      // Crear CSV problemático de prueba
      csvProblematico := `nombre,edad
  Ana,30
  "Luis Pérez,25,Bogotá
  María,28,"Buenos Aires"
  `
      os.WriteFile("problemas.csv", []byte(csvProblematico), 0644)
      analizarCSV("problemas.csv")
  }
  ```

  11. CONVERSIÓN CSV ↔ JSON
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  ```go
  package main

  import (
      "encoding/csv"
      "encoding/json"
      "fmt"
      "os"
  )

  func CSVToJSON(rutaCSV, rutaJSON string) error {
      f, err := os.Open(rutaCSV)
      if err != nil {
          return err
      }
      defer f.Close()

      reader := csv.NewReader(f)
      registros, err := reader.ReadAll()
      if err != nil {
          return err
      }

      if len(registros) < 1 {
          return fmt.Errorf("CSV vacío")
      }

      cabeceras := registros[0]
      var datos []map[string]string

      for _, fila := range registros[1:] {
          entrada := make(map[string]string)
          for i, valor := range fila {
              if i < len(cabeceras) {
                  entrada[cabeceras[i]] = valor
              }
          }
          datos = append(datos, entrada)
      }

      jsonBytes, err := json.MarshalIndent(datos, "", "  ")
      if err != nil {
          return err
      }

      return os.WriteFile(rutaJSON, jsonBytes, 0644)
  }

  func main() {
      err := CSVToJSON("personas.csv", "personas.json")
      if err != nil {
          fmt.Println("Error:", err)
          return
      }
      fmt.Println("Conversión CSV a JSON completada")
  }
  ```

  12. PROCESANDO CSV GRANDE CON STREAMING
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  ```go
  package main

  import (
      "encoding/csv"
      "fmt"
      "os"
      "strconv"
  )

  type Venta struct {
      Fecha     string
      Producto  string
      Cantidad  int
      Precio    float64
  }

  func procesarVentas(ruta string) error {
      f, err := os.Open(ruta)
      if err != nil {
          return err
      }
      defer f.Close()

      reader := csv.NewReader(f)

      // Saltar cabecera
      reader.Read()

      totalIngresos := 0.0
      productoTop := ""
      maxCantidad := 0
      ventasPorDia := make(map[string]int)

      for {
          fila, err := reader.Read()
          if err != nil {
              break
          }

          if len(fila) < 4 {
              continue
          }

          cantidad, _ := strconv.Atoi(fila[2])
          precio, _ := strconv.ParseFloat(fila[3], 64)

          ingreso := float64(cantidad) * precio
          totalIngresos += ingreso
          ventasPorDia[fila[0]] += cantidad

          if cantidad > maxCantidad {
              maxCantidad = cantidad
              productoTop = fila[1]
          }
      }

      fmt.Printf("Total ingresos: $%.2f\n", totalIngresos)
      fmt.Printf("Producto más vendido: %s (%d unidades)\n", productoTop, maxCantidad)

      fmt.Println("\nVentas por día:")
      for dia, cant := range ventasPorDia {
          fmt.Printf("  %s: %d unidades\n", dia, cant)
      }

      return nil
  }

  func main() {
      // Crear datos de prueba
      ventasCSV := `Fecha,Producto,Cantidad,Precio
  2026-07-01,Laptop,5,1250.99
  2026-07-01,Mouse,20,25.50
  2026-07-02,Teclado,15,89.99
  2026-07-02,Monitor,8,450.00
  2026-07-03,Laptop,3,1250.99
  `
      os.WriteFile("ventas.csv", []byte(ventasCSV), 0644)
      procesarVentas("ventas.csv")
  }
  ```

  Salida esperada:
  ```
  Total ingresos: $8897.36
  Producto más vendido: Laptop (5 unidades)

  Ventas por día:
    2026-07-01: 25 unidades
    2026-07-02: 23 unidades
    2026-07-03: 3 unidades
  ```

  13. CSV CON DELIMITADOR PERSONALIZADO
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  ```go
  package main

  import (
      "encoding/csv"
      "fmt"
      "os"
      "strings"
  )

  func leerDelimitado(datos, delim string) ([][]string, error) {
      reader := csv.NewReader(strings.NewReader(datos))
      reader.Comma = []rune(delim)[0]
      reader.TrimLeadingSpace = true
      return reader.ReadAll()
  }

  func main() {
      // TSV
      tsv := "Nombre\tEdad\tCiudad\nAna\t30\tMadrid\nLuis\t25\tBogotá"
      registros, _ := leerDelimitado(tsv, "\t")
      fmt.Println("TSV:", registros)

      // Punto y coma
      scsv := "Nombre;Edad;Ciudad\nAna;30;Madrid\nLuis;25;Bogotá"
      registros, _ = leerDelimitado(scsv, ";")
      fmt.Println("SCSV:", registros)

      // Pipe
      psv := "Nombre|Edad|Ciudad\nAna|30|Madrid\nLuis|25|Bogotá"
      registros, _ = leerDelimitado(psv, "|")
      fmt.Println("PSV:", registros)
  }
  ```

  14. PROGRAMA COMPLETO — ANALIZADOR DE CSV
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  ```go
  package main

  import (
      "encoding/csv"
      "flag"
      "fmt"
      "os"
      "strconv"
      "strings"
  )

  type Estadisticas struct {
      Archivo     string
      Filas       int
      Columnas    int
      Cabeceras   []string
      ValoresNulos int
      Tipos       map[string]string
  }

  func analizarCSV(ruta string, delimitador rune, tieneCabecera bool) (*Estadisticas, error) {
      f, err := os.Open(ruta)
      if err != nil {
          return nil, err
      }
      defer f.Close()

      reader := csv.NewReader(f)
      reader.Comma = delimitador
      reader.LazyQuotes = true
      reader.TrimLeadingSpace = true

      datos, err := reader.ReadAll()
      if err != nil {
          return nil, err
      }

      if len(datos) == 0 {
          return &Estadisticas{Archivo: ruta}, nil
      }

      stats := &Estadisticas{
          Archivo: ruta,
          Filas:   len(datos),
          Columnas: len(datos[0]),
          Tipos:   make(map[string]string),
      }

      inicio := 0
      if tieneCabecera {
          stats.Cabeceras = datos[0]
          inicio = 1
          stats.Filas-- // no contar cabecera como fila
      } else {
          stats.Cabeceras = make([]string, stats.Columnas)
          for i := 0; i < stats.Columnas; i++ {
              stats.Cabeceras[i] = fmt.Sprintf("Columna_%d", i)
          }
      }

      // Analizar tipos por columna
      for c := 0; c < stats.Columnas; c++ {
          esNumero := true
          esEntero := true
          nulos := 0

          for f := inicio; f < len(datos); f++ {
              if c >= len(datos[f]) {
                  continue
              }
              valor := strings.TrimSpace(datos[f][c])
              if valor == "" || valor == "NULL" || valor == "null" || valor == "N/A" {
                  nulos++
                  continue
              }
              if _, err := strconv.Atoi(valor); err != nil {
                  esEntero = false
              }
              if _, err := strconv.ParseFloat(valor, 64); err != nil {
                  esNumero = false
              }
          }

          switch {
          case esEntero:
              stats.Tipos[stats.Cabeceras[c]] = "entero"
          case esNumero:
              stats.Tipos[stats.Cabeceras[c]] = "flotante"
          default:
              stats.Tipos[stats.Cabeceras[c]] = "texto"
          }

          stats.ValoresNulos += nulos
      }

      return stats, nil
  }

  func main() {
      ruta := flag.String("archivo", "", "Ruta al archivo CSV")
      delim := flag.String("delimitador", ",", "Delimitador de campos")
      cabecera := flag.Bool("cabecera", true, "¿La primera línea es cabecera?")
      flag.Parse()

      if *ruta == "" {
          fmt.Println("Uso: go run main.go -archivo datos.csv [-delimitador ','] [-cabecera true]")
          os.Exit(1)
      }

      delRune := []rune(*delim)[0]
      stats, err := analizarCSV(*ruta, delRune, *cabecera)
      if err != nil {
          fmt.Println("Error:", err)
          os.Exit(1)
      }

      fmt.Println("=== ANÁLISIS DE CSV ===")
      fmt.Printf("Archivo: %s\n", stats.Archivo)
      fmt.Printf("Filas: %d\n", stats.Filas)
      fmt.Printf("Columnas: %d\n", stats.Columnas)
      fmt.Println("\nCabeceras:")
      for i, h := range stats.Cabeceras {
          fmt.Printf("  %d: %s (%s)\n", i+1, h, stats.Tipos[h])
      }
      fmt.Printf("\nValores nulos: %d\n", stats.ValoresNulos)
  }
  ```

  Uso:
  ```
  go run main.go -archivo datos.csv -delimitador "," -cabecera true
  ```

  15. MANIPULACIÓN DE CSV (FILTRAR, ORDENAR, AGRUPAR)
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  ```go
  package main

  import (
      "encoding/csv"
      "fmt"
      "os"
      "sort"
      "strconv"
  )

  func filtrarCSV(rutaEntrada, rutaSalida string, columna string, condicion func(string) bool) error {
      f, err := os.Open(rutaEntrada)
      if err != nil {
          return err
      }
      defer f.Close()

      reader := csv.NewReader(f)
      datos, _ := reader.ReadAll()

      if len(datos) < 1 {
          return fmt.Errorf("CSV vacío")
      }

      cabeceras := datos[0]
      idxCol := -1
      for i, h := range cabeceras {
          if h == columna {
              idxCol = i
              break
          }
      }
      if idxCol < 0 {
          return fmt.Errorf("columna '%s' no encontrada", columna)
      }

      out, err := os.Create(rutaSalida)
      if err != nil {
          return err
      }
      defer out.Close()

      writer := csv.NewWriter(out)
      defer writer.Flush()

      // Escribir cabeceras
      writer.Write(cabeceras)

      // Filtrar filas
      for _, fila := range datos[1:] {
          if idxCol < len(fila) && condicion(fila[idxCol]) {
              writer.Write(fila)
          }
      }

      return writer.Error()
  }

  func ordenarCSV(rutaEntrada, rutaSalida string, columna string, ascendente bool) error {
      f, err := os.Open(rutaEntrada)
      if err != nil {
          return err
      }
      defer f.Close()

      reader := csv.NewReader(f)
      datos, _ := reader.ReadAll()

      if len(datos) < 2 {
          return fmt.Errorf("CSV sin datos")
      }

      cabeceras := datos[0]
      idxCol := -1
      for i, h := range cabeceras {
          if h == columna {
              idxCol = i
              break
          }
      }
      if idxCol < 0 {
          return fmt.Errorf("columna '%s' no encontrada", columna)
      }

      filas := datos[1:]
      sort.Slice(filas, func(i, j int) bool {
          if idxCol >= len(filas[i]) || idxCol >= len(filas[j]) {
              return false
          }
          if ascendente {
              return filas[i][idxCol] < filas[j][idxCol]
          }
          return filas[i][idxCol] > filas[j][idxCol]
      })

      out, err := os.Create(rutaSalida)
      if err != nil {
          return err
      }
      defer out.Close()

      writer := csv.NewWriter(out)
      defer writer.Flush()

      writer.Write(cabeceras)
      writer.WriteAll(filas)
      return writer.Error()
  }

  func agruparCSV(ruta string, columna string) (map[string]int, error) {
      f, err := os.Open(ruta)
      if err != nil {
          return nil, err
      }
      defer f.Close()

      reader := csv.NewReader(f)
      datos, _ := reader.ReadAll()

      if len(datos) < 2 {
          return nil, fmt.Errorf("CSV sin datos")
      }

      cabeceras := datos[0]
      idxCol := -1
      for i, h := range cabeceras {
          if h == columna {
              idxCol = i
              break
          }
      }
      if idxCol < 0 {
          return nil, fmt.Errorf("columna '%s' no encontrada", columna)
      }

      grupos := make(map[string]int)
      for _, fila := range datos[1:] {
          if idxCol < len(fila) {
              grupos[fila[idxCol]]++
          }
      }
      return grupos, nil
  }

  func main() {
      // Crear CSV de ejemplo
      ejemplo := `Producto,Categoría,Precio
  Laptop,Electrónica,1250.99
  Mouse,Electrónica,25.50
  Teclado,Electrónica,89.99
  Mesa,Hogar,299.99
  Silla,Hogar,149.99
  `
      os.WriteFile("inventario.csv", []byte(ejemplo), 0644)

      // Filtrar
      filtrarCSV("inventario.csv", "electronica.csv", "Categoría",
          func(v string) bool { return v == "Electrónica" })

      // Ordenar por precio descendente
      ordenarCSV("inventario.csv", "ordenado.csv", "Precio", false)

      // Agrupar por categoría
      grupos, _ := agruparCSV("inventario.csv", "Categoría")
      fmt.Println("Productos por categoría:")
      for cat, count := range grupos {
          fmt.Printf("  %s: %d\n", cat, count)
      }

      fmt.Println("\nArchivos generados: electronica.csv, ordenado.csv")
  }
  ```

  16. BUENAS PRÁCTICAS CON CSV
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  1. Siempre verifica el error de Flush en el Writer.
  2. Usa FieldsPerRecord = -1 si el archivo puede tener filas
     con diferente número de campos.
  3. Activa LazyQuotes para tolerar comillas mal formadas en
     archivos de terceros.
  4. TrimLeadingSpace para ignorar espacios después de comas.
  5. Para archivos grandes, usa Read() en lugar de ReadAll().
  6. Siempre escapa campos que puedan contener comas o comillas.
  7. Usa punto y coma como delimitador si los datos usan comas
     (ej. números con coma decimal en Europa).
  8. Prefiere TSV (tabulador) para datos que contengan comas.
  9. Lee la cabecera por separado para acceso por nombre de columna.
  10. Valida los datos después de parsear: los CSV pueden tener
      cualquier cosa.

  17. LIMITACIONES DE encoding/csv
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

  - No soporta RFC 4180 completo (no maneja saltos de línea
    incrustados en campos correctamente en todos los casos).
  - No detecta automáticamente el delimitador.
  - No soporta codificaciones diferentes a UTF-8.
  - No hay soporte nativo para compresión.
  - No tiene validación de tipos incorporada.

  Para CSV más complejos, considera:
  - github.com/jszwec/csvutil (mapeo a structs con tags)
  - github.com/gocarina/gocsv (mapeo automático)

  PREGUNTAS FRECUENTES (FAQ)
  -------------------------------------------------------------------------------

  ❓ ¿Qué pasa si el CSV tiene campos con comas?
  R: El paquete csv maneja automáticamente campos entre comillas
  dobles. "Bogotá, Colombia" se lee como un solo campo.

  ❓ ¿Cómo manejo CSV con BOM (Byte Order Mark)?
  R: La BOM (UTF-8) al inicio del archivo puede causar problemas.
  Puedes omitirla con `strings.TrimPrefix(datos, "\ufeff")`.

  ❓ ¿Cómo detecto automáticamente el delimitador?
  R: encoding/csv no lo hace. Puedes implementarlo contando
  ocurrencias de posibles delimitadores (,, ;, \t) en la cabecera.

  ❓ ¿Cuánta memoria usa ReadAll?
  R: Toda la información del archivo en memoria (como slice de slices).
  Para archivos grandes (> 100 MB), usa Read() en bucle.

  ❓ ¿Puedo escribir CSV directamente a HTTP Response?
  R: Sí, csv.NewWriter(responseWriter) escribe directamente al
  HTTP Response. No olvides el header Content-Type: text/csv.

  ❓ ¿Cómo manejo encoding/csv con archivos comprimidos?
  R: Envuelve el archivo con gzip.NewReader/NewWriter antes de
  pasarlo a csv.NewReader/NewWriter.

  ❓ ¿Qué es LazyQuotes?
  R: Permite que el Reader acepte comillas sin cerrar y otros
  usos no estándar de comillas. Útil para archivos generados
  por sistemas antiguos.

  DESAFÍO DEL ARQUITECTO
  -------------------------------------------------------------------------------

  🏆 MOTOR DE ETL PARA DATOS CSV

  Diseña e implementa un motor ETL (Extract, Transform, Load)
  para archivos CSV con las siguientes características:

  1. Extracción: lee múltiples archivos CSV desde un directorio,
     cada uno con estructura potencialmente diferente.
  2. Transformación: aplica reglas configurables:
     - Mapeo de columnas (renombrar, combinar, dividir).
     - Conversión de tipos (string a número, formato de fecha).
     - Limpieza (eliminar nulos, normalizar mayúsculas).
     - Filtrado (WHERE) y agregación (SUM, AVG, COUNT).
  3. Carga: escribe el resultado en un archivo CSV de salida.
  4. Configuración: archivo YAML/JSON con el pipeline de
     transformaciones.
  5. Reporte: genera un archivo de log con estadísticas del
     proceso (filas leídas, transformadas, errores).

  Ejemplo de archivo de configuración:
  ```json
  {
    "entrada": {
      "directorio": "./datos",
      "patron": "ventas_*.csv"
    },
    "transformaciones": [
      {"tipo": "renombrar", "de": "precio", "a": "total"},
      {"tipo": "convertir", "columna": "fecha", "formato": "2006-01-02"},
      {"tipo": "filtrar", "columna": "total", "min": 0}
    ],
    "salida": "resultado.csv"
  }
  ```

  Requisitos técnicos:
  - Usa encoding/csv para lectura/escritura.
  - Implementa un pipeline con canales para el flujo de datos.
  - Usa reflect para conversión dinámica de tipos.
  - Implementa manejo de errores por fila (fail-fast o tolerante).
  - Usa worker pool para procesar múltiples archivos en paralelo.

  Este desafío integra:
  - CSV avanzado (Reader/Writer, configuración)
  - Procesamiento concurrente con canales
  - Pipeline de transformaciones
  - Configuración y metaprogramación
