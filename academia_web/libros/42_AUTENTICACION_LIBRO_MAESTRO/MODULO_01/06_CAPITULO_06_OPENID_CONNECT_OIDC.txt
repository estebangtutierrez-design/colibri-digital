CONTENIDO DEL CAPÍTULO
===============================================================================

OIDC: capa de identidad sobre OAuth 2.0.

ID Token (JWT). UserInfo endpoint.

Discovery: /.well-known/openid-configuration.

EJEMPLO DE CÓDIGO
----------------------------------------

// ID Token
{
  "iss": "https://auth.example.com",
  "sub": "user123",
  "aud": "myapp",
  "exp": 1699000000,
  "iat": 1698913600,
  "email": "user@example.com",
  "email_verified": true,
  "name": "Alice"
}

// Verificar ID Token
const decoded = jwt.verify(idToken, jwks, {
  issuer: "https://auth.example.com",
  audience: "myapp"
})
