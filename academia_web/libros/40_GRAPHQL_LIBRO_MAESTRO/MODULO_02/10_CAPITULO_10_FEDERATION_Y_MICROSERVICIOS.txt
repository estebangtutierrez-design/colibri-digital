Apollo Federation: composición de subgraphs.

@key, @external, @requires, @provides.

Gateway: supergraph. Federation spec.

EJEMPLO DE CÓDIGO
----------------------------------------

// Subgraph A: Users
interface User @key(fields: "id") {
  id: ID!
  name: String!
}

// Subgraph B: Reviews
type Review @key(fields: "id") {
  id: ID!
  body: String!
  author: User! @provides(fields: "name")
}

extend type User @key(fields: "id") {
  id: ID! @external
  reviews: [Review]
}
