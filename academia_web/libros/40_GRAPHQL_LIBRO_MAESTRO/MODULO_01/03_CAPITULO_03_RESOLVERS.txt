CONTENIDO DEL CAPÍTULO
===============================================================================

Resolvers: funciones que resuelven cada campo.

Parent, args, context, info. DataLoader.

Resolver chain: user -> posts.

EJEMPLO DE CÓDIGO
----------------------------------------

const resolvers = {
  Query: {
    users: async (_, __, { db }) => db.user.findMany(),
    user: async (_, { id }, { db }) => db.user.findUnique({ where: { id } })
  },
  User: {
    posts: async (parent, _, { db }) =>
      db.post.findMany({ where: { userId: parent.id } })
  },
  Mutation: {
    createUser: async (_, { input }, { db }) =>
      db.user.create({ data: input })
  }
}
