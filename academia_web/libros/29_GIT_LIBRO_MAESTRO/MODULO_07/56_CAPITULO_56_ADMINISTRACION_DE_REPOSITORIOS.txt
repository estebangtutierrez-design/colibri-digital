CONTENIDO DEL CAPÍTULO
===============================================================================

1. GIT GC Y MANTENIMIENTO

git gc: recolector de basura, empaqueta objetos. git fsck: verifica

integridad. git prune: elimina objetos inalcanzables. Mantenimiento

periódico: manual o auto. Repos grandes: gc cada semana.



2. REPOSITORIOS MASIVOS

Shallow clone: --depth para solo últimos commits. Partial clone:

--filter=blob:none. Sparse checkout: solo directorios necesarios.

Scalar: Microsoft tool para repos grandes. git bundle para backups.



3. MONITOREO Y ESTADÍSTICAS

git count-objects -vH: tamaño, objetos. git verify-pack: verificar

packfiles. GitStats: estadísticas visuales. GitInspector: análisis

de contribuciones. Herramientas de monitoreo de disco.

EJEMPLO DE CÓDIGO
----------------------------------------

git gc --aggressive --prune=now
git fsck --full
git count-objects -vH

# Repos grandes
git clone --depth 1 --single-branch <url>
git clone --filter=tree:0 <url>  # Blobless clone

# Estadísticas
git shortlog -sn --all
git diff --stat
git rev-list --count --all
