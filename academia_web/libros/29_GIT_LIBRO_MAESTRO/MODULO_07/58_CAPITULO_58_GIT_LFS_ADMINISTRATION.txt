CONTENIDO DEL CAPÍTULO
===============================================================================

1. SERVIDOR LFS

GitHub, GitLab, Bitbucket tienen LFS incluido. Self-hosted: Git LFS

standalone server. Configuración: endpoint, auth, storage backend.

Almacenamiento: local, S3, GCS, Azure Blob. Transfer encoding.



2. GESTIÓN DE ALMACENAMIENTO

git lfs migrate import/export. Verificar uso: git lfs ls-files --size.

Políticas: archivos mayores a X MB van a LFS. Revisar periódicamente.

LFS storage quota según plan. Limpiar archivos no referenciados.



3. MIGRACIÓN LFS

git lfs migrate import --everything. Convierte archivos existentes.

git lfs migrate export para revertir. Cambiar pattern con otro migrate.

Verificar: git lfs ls-files. Push después de migración.

EJEMPLO DE CÓDIGO
----------------------------------------

# Migrar a LFS
git lfs migrate import --include="*.psd,*.zip,*.bin" --everything

# Exportar de LFS
git lfs migrate export --include="*.pdf" --everything

# Estadísticas
git lfs ls-files --size

# .gitattributes actualizado
*.psd filter=lfs diff=lfs merge=lfs -text
*.zip filter=lfs diff=lfs merge=lfs -text

# Verificar uso
du -sh .git/lfs/objects
