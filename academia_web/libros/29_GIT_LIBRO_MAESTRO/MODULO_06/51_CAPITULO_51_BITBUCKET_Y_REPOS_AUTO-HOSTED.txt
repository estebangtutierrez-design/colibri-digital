CONTENIDO DEL CAPÍTULO
===============================================================================

1. BITBUCKET CLOUD

Integración Atlassian: Jira, Confluence. Bitbucket Pipelines inte-

grado. Pull Requests con approvals. Code Insights: SonarQube, Snyk.

IP whitelisting. 2FA. Workspace limits.



2. BITBUCKET DATACENTER (SELF-HOSTED)

Control total sobre datos. Detección de smart mirrors. Elastic search

integrado. Alta disponibilidad con clustering. Active Directory/LDAP.

Replicación geográfica. Auditoría y cumplimiento.



3. SERVIDORES GIT PROPIOS

Gitolite: control de acceso fino. GitLab CE (self-managed). Gitea:

ligero, Go. Gogs: minimalista. cgit: web UI minimal. Kiln: integrado

con FogBugz. Choisir según necesidad: escala, features, recursos.
