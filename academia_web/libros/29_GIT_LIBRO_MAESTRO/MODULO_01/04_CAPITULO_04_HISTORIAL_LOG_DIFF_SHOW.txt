CONTENIDO DEL CAPÍTULO
===============================================================================

1. GIT LOG

git log --oneline, --graph, --all, --author, --since, --until, --grep.

git log -p muestra el diff de cada commit. %h hash corto, %an autor,

%s mensaje. Formato personalizado con --pretty=format.



2. GIT DIFF

git diff cambios sin staging. git diff --staged cambios en staging.

git diff HEAD~1 HEAD diff entre dos commits. git diff --stat resumen.



3. GIT SHOW

git show <commit> muestra el commit completo. git show HEAD~2:archivo.txt

muestra archivo en commit anterior. Útil para inspección forense.

EJEMPLO DE CÓDIGO
----------------------------------------

git log --oneline --graph --all --decorate
# Visualización completa del historial

git diff --word-diff
# Muestra cambios palabra por palabra

git show HEAD --stat
# Último commit con estadísticas

git log --since="2 weeks ago" --author="Juan" --oneline
# Filtro por fecha y autor
