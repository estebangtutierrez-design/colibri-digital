CONTENIDO DEL CAPÍTULO
===============================================================================

1. CONFIGURACIÓN GLOBAL

git config --global user.name, user.email, core.editor. Las configuraciones

se almacenan en ~/.gitconfig. Nivel system para todos los usuarios del sistema,

nivel global para el usuario, nivel local para el repositorio.



2. ALIAS ÚTILES

git config --global alias.lg 'log --oneline --graph --all'. Alias para

comandos frecuentes: git st, git ci, git br, git co, git df.



3. BUENAS PRÁCTICAS DE CONFIGURACIÓN

core.autocrlf=input (Linux/Mac) o true (Windows). core.safecrlf warn.

init.defaultBranch main. pull.rebase true. fetch.prune true.

EJEMPLO DE CÓDIGO
----------------------------------------

git config --global user.name "Tu Nombre"
git config --global user.email "tu@email.com"
git config --global init.defaultBranch main
git config --global alias.lg "log --oneline --graph --all"
git config --global alias.st "status -sb"
git config --global pull.rebase true
git config --global fetch.prune true
