¿Qué es Python?
---------------
Python es un lenguaje de programación de alto nivel, interpretado, multiparadigma
(orientado a objetos, imperativo y funcional). Fue creado por Guido van Rossum
y lanzado por primera vez en 1991. Su filosofía enfatiza la legibilidad del
código con una sintaxis clara y sencilla.

Características principales:
- Interpretado (no necesita compilación).
- Tipado dinámico (no declarar tipos).
- Orientado a objetos (todo es un objeto).
- Grandes bibliotecas estándar y de terceros.
- Multiplataforma (Windows, Linux, macOS).
- Ideal para: IA, ciencia de datos, automatización, web (Django/Flask), scripts.

Filosofía: "El Zen de Python" (import this).
- "Bello es mejor que feo."
- "Explícito es mejor que implícito."
- "Simple es mejor que complejo."

Versiones: Python 3.x es el estándar (Python 2 fue descontinuado en 2020).
La versión más reciente es Python 3.12 / 3.13 (2024).

Clases y objetos:

class Persona:
    # Constructor (__init__)
    def __init__(self, nombre, edad):
        self.nombre = nombre   # atributo público
        self.edad = edad
        self._id = 123         # convención: privado (_)
        self.__password = "secreto"  # name mangling (más privado)

    # Métodos
    def saludar(self):
        print(f"Hola, soy {self.nombre}")

    def __str__(self):   # toString
        return f"Persona({self.nombre}, {self.edad})"

    # Getter y Setter con @property
    @property
    def edad(self):
        return self._edad

    @edad.setter
    def edad(self, valor):
        if valor > 0:
            self._edad = valor
        else:
            raise ValueError("Edad debe ser positiva")

# Crear objeto:
p = Persona("Esteban", 30)
p.saludar()
print(p)
p.edad = 31
print(p.edad)

Herencia:
---------
class Empleado(Persona):
    def __init__(self, nombre, edad, salario):
        super().__init__(nombre, edad)   # llama al constructor padre
        self.salario = salario

    def saludar(self):   # Sobrescritura
        super().saludar()
        print(f"Salario: {self.salario}")

    def __str__(self):
        return f"Empleado({self.nombre}, {self.edad}, {self.salario})"

Try-except:

    x = int(input("Número: "))
    resultado = 10 / x

CAPITULO 16: PROYECTO FINAL - ANÁLISIS DE DATOS DE VENTAS
================================================================================

Objetivo: Analizar un archivo CSV de ventas y generar un informe con gráficos.

Datos (ventas.csv):
-------------------
fecha,producto,categoria,cantidad,precio_unitario,cliente
2024-01-01,Teclado,Electrónica,5,25.00,Juan
2024-01-02,Mouse,Electrónica,10,15.00,Maria
2024-01-03,Cuaderno,Papelería,20,2.50,Carlos
...

Pasos:
1. Cargar datos con pandas.
2. Limpiar datos (valores nulos, duplicados).
3. Agregar columna "total" = cantidad * precio_unitario.
4. Calcular:
   - Ventas totales por mes.
   - Producto más vendido.
   - Ingresos por categoría.
   - Cliente que más compra.
5. Visualizar resultados con matplotlib.
6. Generar un informe en HTML o PDF (con reportlab).

Código base:
------------
import pandas as pd
import matplotlib.pyplot as plt

# Cargar
df = pd.read_csv("ventas.csv")

# Limpiar
df.dropna(inplace=True)
df["fecha"] = pd.to_datetime(df["fecha"])

# Calcular total
df["total"] = df["cantidad"] * df["precio_unitario"]

# Ventas por mes
df["mes"] = df["fecha"].dt.month
ventas_mes = df.groupby("mes")["total"].sum()
ventas_mes.plot(kind="bar")
plt.title("Ventas por Mes")
plt.show()

# Producto más vendido
producto_mas_vendido = df.groupby("producto")["cantidad"].sum().idxmax()
print(f"Producto más vendido: {producto_mas_vendido}")

# Ingresos por categoría
ingresos_categoria = df.groupby("categoria")["total"].sum()
print(ingresos_categoria)

# Cliente que más compra
cliente_top = df.groupby("cliente")["total"].sum().idxmax()
print(f"Cliente que más compra: {cliente_top}")

# Guardar reporte
with open("reporte.txt", "w") as f:
    f.write("=== REPORTE DE VENTAS ===\n")
    f.write(f"Ventas totales: ${df['total'].sum():.2f}\n")
    f.write(f"Producto más vendido: {producto_mas_vendido}\n")
    f.write(f"Cliente top: {cliente_top}\n")
