  BIENVENIDO

  Las enumeraciones en Java son mucho más que simples listas de constantes.
  Desde Java 5, los enums son tipos de datos completos que pueden tener
  campos, métodos, constructores y comportamientos específicos por
  constante.

  Un enum en Java es una clase especial que extiende java.lang.Enum
  implícitamente. Esto significa que cada constante enum es una instancia
  del tipo enum, con toda la riqueza de la POO.

  En este capítulo explorarás desde la sintaxis básica hasta los patrones
  más avanzados con enums: métodos abstractos por constante, interfaces,
  estrategias (Strategy Pattern), y uso con switch y streams.

  --- 1. SINTÁXIS BÁSICA ---

  La forma más simple de un enum es una lista de constantes:

  ```java
  public enum DiaSemana {
      LUNES, MARTES, MIERCOLES, JUEVES, VIERNES, SABADO, DOMINGO
  }

  public class UsoBasico {
      public static void main(String[] args) {
          // Asignación
          DiaSemana hoy = DiaSemana.MIERCOLES;
          System.out.println("Hoy es: " + hoy);

          // Comparación con ==
          if (hoy == DiaSemana.MIERCOLES) {
              System.out.println("¡Mitad de semana!");
          }

          // switch
          switch (hoy) {
              case LUNES:
                  System.out.println("Inicio de semana");
                  break;
              case VIERNES:
                  System.out.println("Último día laboral");
                  break;
              case SABADO:
              case DOMINGO:
                  System.out.println("Fin de semana");
                  break;
              default:
                  System.out.println("Día laboral normal");
          }
      }
  }
  ```

  Los enums son tipos seguros: no puedes asignar un valor que no esté
  definido. Esto elimina los errores comunes con constantes enteras o
  strings.

  ```java
  // Esto NO compila:
  // DiaSemana d = 1;  // Error: tipos incompatibles
  // DiaSemana d = "LUNES";  // Error
  ```

  --- 2. CAMPOS Y CONSTRUCTORES ---

  Los enums pueden tener campos y constructores privados. Los constructores
  se invocan para cada constante en el momento de la declaración.

  ```java
  public enum TipoCambio {
      // Cada constante llama al constructor
      COMPRA("Compra de divisas", 'C', true),
      VENTA("Venta de divisas", 'V', true),
      TRANSFERENCIA("Transferencia bancaria", 'T', false),
      RETIRO_EFECTIVO("Retiro en efectivo", 'R', true),
      CONSULTA("Consulta de saldo", 'Q', false);

      // Campos
      private final String descripcion;
      private final char codigo;
      private final boolean requiereFirma;

      // Constructor (SIEMPRE privado, implícitamente)
      TipoCambio(String descripcion, char codigo, boolean requiereFirma) {
          this.descripcion = descripcion;
          this.codigo = codigo;
          this.requiereFirma = requiereFirma;
      }

      // Métodos
      public String getDescripcion() { return descripcion; }
      public char getCodigo() { return codigo; }
      public boolean isRequiereFirma() { return requiereFirma; }

      // Método que usa los campos
      public void mostrarInfo() {
          System.out.printf("%s (%c) - %s %s%n",
                  name(), codigo, descripcion,
                  requiereFirma ? "[Requiere firma]" : "");
      }
  }

  public class PruebaCampos {
      public static void main(String[] args) {
          System.out.println("=== TIPOS DE CAMBIO DISPONIBLES ===\n");
          for (TipoCambio tc : TipoCambio.values()) {
              tc.mostrarInfo();
          }

          System.out.println("\n=== CONSULTAS ===");
          TipoCambio op = TipoCambio.COMPRA;
          System.out.println("Operación: " + op);
          System.out.println("Código: " + op.getCodigo());
          System.out.println("Requiere firma: " + op.isRequiereFirma());
      }
  }
  ```

  --- 3. MÉTODOS ESPECÍFICOS POR CONSTANTE ---

  Puedes definir métodos abstractos en el enum y cada constante
  proporciona su propia implementación.

  ```java
  public enum OperacionMatematica {
      SUMA("+") {
          @Override
          public double aplicar(double a, double b) {
              return a + b;
          }
      },
      RESTA("-") {
          @Override
          public double aplicar(double a, double b) {
              return a - b;
          }
      },
      MULTIPLICACION("*") {
          @Override
          public double aplicar(double a, double b) {
              return a * b;
          }
      },
      DIVISION("/") {
          @Override
          public double aplicar(double a, double b) {
              if (b == 0) {
                  throw new ArithmeticException("División por cero");
              }
              return a / b;
          }
      },
      POTENCIA("^") {
          @Override
          public double aplicar(double a, double b) {
              return Math.pow(a, b);
          }
      };

      private final String simbolo;

      OperacionMatematica(String simbolo) {
          this.simbolo = simbolo;
      }

      public String getSimbolo() { return simbolo; }

      // Método abstracto que cada constante implementa
      public abstract double aplicar(double a, double b);

      @Override
      public String toString() {
          return simbolo;
      }
  }

  public class CalculadoraEnum {
      public static void main(String[] args) {
          double a = 10, b = 3;

          System.out.println("=== CALCULADORA CON ENUMS ===\n");
          for (OperacionMatematica op : OperacionMatematica.values()) {
              try {
                  double resultado = op.aplicar(a, b);
                  System.out.printf("%.2f %s %.2f = %.2f%n",
                          a, op.getSimbolo(), b, resultado);
              } catch (ArithmeticException e) {
                  System.out.printf("%.2f %s %.2f = ERROR: %s%n",
                          a, op.getSimbolo(), b, e.getMessage());
              }
          }

          // Uso dinámico
          System.out.println("\n=== USO DINÁMICO ===");
          OperacionMatematica op = OperacionMatematica.valueOf("SUMA");
          System.out.println(a + " " + op.getSimbolo() + " " + b
                  + " = " + op.aplicar(a, b));
      }
  }
  ```

  --- 4. MÉTODOS HEREDADOS DE ENUM ---

  Todos los enums heredan de java.lang.Enum, que proporciona métodos
  útiles:

  - name(): el nombre exacto de la constante como string.
  - ordinal(): la posición (0-based) en la declaración.
  - values(): devuelve un array con todas las constantes.
  - valueOf(String): devuelve la constante con el nombre dado.
  - compareTo(Enum): compara por ordinal.
  - toString(): igual que name(), pero puede sobrescribirse.

  ```java
  public enum NivelAcceso {
      INVITADO,    // ordinal 0
      USUARIO,     // ordinal 1
      MODERADOR,   // ordinal 2
      ADMINISTRADOR;  // ordinal 3

      @Override
      public String toString() {
          // Personalizar la representación
          return name().charAt(0) + name().substring(1).toLowerCase();
      }
  }

  public class MetodosEnum {
      public static void main(String[] args) {
          System.out.println("=== MÉTODOS DE ENUM ===\n");

          // values() y ordinal()
          System.out.println("Valores del enum:");
          for (NivelAcceso nivel : NivelAcceso.values()) {
              System.out.printf("  %s: ordinal=%d, name=%s, toString=%s%n",
                      nivel, nivel.ordinal(), nivel.name(), nivel);
          }

          // valueOf (conversión desde string)
          System.out.println("\n=== CONVERSIONES ===");
          NivelAcceso admin = NivelAcceso.valueOf("ADMINISTRADOR");
          System.out.println("valueOf('ADMINISTRADOR'): " + admin);

          // compareTo (compara por ordinal)
          System.out.println("\n=== COMPARACIONES ===");
          System.out.println("INVITADO.compareTo(USUARIO): "
                  + NivelAcceso.INVITADO.compareTo(NivelAcceso.USUARIO));
          System.out.println("ADMINISTRADOR.compareTo(MODERADOR): "
                  + NivelAcceso.ADMINISTRADOR.compareTo(NivelAcceso.MODERADOR));

          // equals (seguro, usar ==)
          System.out.println("\n=== IGUALDAD ===");
          System.out.println("ADMINISTRADOR == ADMINISTRADOR: "
                  + (NivelAcceso.ADMINISTRADOR == NivelAcceso.ADMINISTRADOR));
          System.out.println("ADMINISTRADOR.equals(ADMINISTRADOR): "
                  + NivelAcceso.ADMINISTRADOR.equals(NivelAcceso.ADMINISTRADOR));

          // Verificar si tiene permisos suficientes
          NivelAcceso usuario = NivelAcceso.USUARIO;
          System.out.println("\n¿USUARIO puede acceder a función de admin? "
                  + usuario.compareTo(NivelAcceso.ADMINISTRADOR) >= 0);
      }
  }
  ```

  --- 5. ENUMS CON SWITCH Y SWITCH EXPRESSION ---

  Los enums funcionan perfectamente con switch. Desde Java 14, también
  con switch expressions.

  ```java
  public enum Estacion {
      PRIMAVERA, VERANO, OTOÑO, INVIERNO
  }

  public class SwitchConEnum {
      public static void main(String[] args) {
          Estacion actual = Estacion.VERANO;

          // Switch tradicional
          System.out.print("Tradicional: ");
          switch (actual) {
              case PRIMAVERA:
                  System.out.println("Flores y clima templado");
                  break;
              case VERANO:
                  System.out.println("Calor y vacaciones");
                  break;
              case OTOÑO:
                  System.out.println("Hojas caídas y fresco");
                  break;
              case INVIERNO:
                  System.out.println("Frío y nieve");
                  break;
          }

          // Switch expression (Java 14+)
          String descripcion = switch (actual) {
              case PRIMAVERA -> "Renacimiento de la naturaleza";
              case VERANO   -> "Días largos y calurosos";
              case OTOÑO    -> "Tiempo de cosecha";
              case INVIERNO -> "Noches largas y frías";
          };
          System.out.println("Expression: " + descripcion);

          // Switch expression con yield (para bloques)
          int dias = switch (actual) {
              case PRIMAVERA -> 92;
              case VERANO -> 93;
              case OTOÑO -> 90;
              case INVIERNO -> {
                  int bisiesto = 2024 % 4 == 0 ? 1 : 0;
                  yield 90 + bisiesto;
              }
          };
          System.out.println("Días en " + actual + ": " + dias);

          // For-each con switch
          System.out.println("\n=== TODAS LAS ESTACIONES ===");
          for (Estacion e : Estacion.values()) {
              String msg = switch (e) {
                  case PRIMAVERA -> "🌼";
                  case VERANO -> "☀️";
                  case OTOÑO -> "🍂";
                  case INVIERNO -> "❄️";
              };
              System.out.println(e + ": " + msg);
          }
      }
  }
  ```

  --- 6. ENUMS E INTERFACES ---

  Los enums pueden implementar interfaces, permitiendo que diferentes
  constantes se comporten de manera polimórfica.

  ```java
  // Interfaz que implementarán los enums
  interface Notificable {
      void enviarNotificacion(String mensaje);
      String getTipo();
  }

  public enum CanalNotificacion implements Notificable {
      EMAIL {
          @Override
          public void enviarNotificacion(String mensaje) {
              System.out.println("[EMAIL] Enviando a correo: " + mensaje);
              System.out.println("[EMAIL] Mensaje formateado con HTML");
          }

          @Override
          public String getTipo() {
              return "Correo Electrónico";
          }
      },
      SMS {
          @Override
          public void enviarNotificacion(String mensaje) {
              System.out.println("[SMS] Enviando a teléfono: " + mensaje);
              System.out.println("[SMS] Mensaje truncado a 160 caracteres");
          }

          @Override
          public String getTipo() {
              return "Mensaje de Texto";
          }
      },
      PUSH {
          @Override
          public void enviarNotificacion(String mensaje) {
              System.out.println("[PUSH] Notificación en app: " + mensaje);
              System.out.println("[PUSH] Incluye sonido y vibración");
          }

          @Override
          public String getTipo() {
              return "Notificación Push";
          }
      },
      SLACK {
          @Override
          public void enviarNotificacion(String mensaje) {
              System.out.println("[SLACK] Mensaje en canal: " + mensaje);
              System.out.println("[SLACK] Menciona a @usuario");
          }

          @Override
          public String getTipo() {
              return "Mensaje Slack";
          }
      };

      // Método por defecto (opcional)
      public boolean esDisponible() {
          return true;
      }
  }

  public class EnumsConInterfaces {
      public static void main(String[] args) {
          System.out.println("=== CANALES DE NOTIFICACIÓN ===\n");

          // Uso polimórfico a través de la interfaz
          Notificable[] canales = CanalNotificacion.values();

          for (Notificable canal : canales) {
              System.out.println("--- " + canal.getTipo() + " ---");
              canal.enviarNotificacion("¡Oferta exclusiva! 50% descuento");
              System.out.println();
          }

          // Uso directo del enum
          CanalNotificacion slack = CanalNotificacion.SLACK;
          System.out.println("Canal específico: " + slack);
          slack.enviarNotificacion("Reunión en 5 minutos");

          // Método propio del enum
          System.out.println("Disponible: " + slack.esDisponible());
      }
  }
  ```

  --- 7. PATRÓN STRATEGY CON ENUMS ---

  Los enums son excelentes para implementar el patrón Strategy, donde
  cada constante representa una estrategia diferente.

  ```java
  import java.util.function.*;

  public enum EstrategiaDescuento {
      SIN_DESCUENTO("Ninguno") {
          @Override
          public double aplicar(double precio) {
              return precio;
          }
      },
      DESCUENTO_PORCENTAJE("10% de descuento") {
          @Override
          public double aplicar(double precio) {
              return precio * 0.9;
          }
      },
      DESCUENTO_CANTIDAD_FIJA("$50 de descuento") {
          @Override
          public double aplicar(double precio) {
              return Math.max(0, precio - 50);
          }
      },
      DESCUENTO_POR_MAYOR("15% por compras > $200") {
          @Override
          public double aplicar(double precio) {
              return precio > 200 ? precio * 0.85 : precio;
          }
      },
      DESCUENTO_CODIGO("Código PROMO2024: 20%") {
          @Override
          public double aplicar(double precio) {
              return precio * 0.8;
          }
      },
      DESCUENTO_COMPUESTO("Acumulativo: 10%+$30") {
          @Override
          public double aplicar(double precio) {
              return precio * 0.9 - 30;
          }
      };

      private final String descripcion;

      EstrategiaDescuento(String descripcion) {
          this.descripcion = descripcion;
      }

      public String getDescripcion() { return descripcion; }

      // Método abstracto: cada constante define su estrategia
      public abstract double aplicar(double precio);

      // Método para calcular con límite inferior
      public double aplicarConMinimo(double precio, double minimo) {
          return Math.max(minimo, aplicar(precio));
      }

      // Búsqueda por descripción (similar a valueOf pero con otro criterio)
      public static EstrategiaDescuento porDescripcion(String desc) {
          for (EstrategiaDescuento e : values()) {
              if (e.descripcion.equalsIgnoreCase(desc)) {
                  return e;
              }
          }
          throw new IllegalArgumentException(
                  "No se encontró estrategia: " + desc);
      }
  }

  public class CarritoCompra {
      private double total;
      private EstrategiaDescuento estrategia;

      public CarritoCompra(double total, EstrategiaDescuento estrategia) {
          this.total = total;
          this.estrategia = estrategia;
      }

      public double calcularTotal() {
          return estrategia.aplicarConMinimo(total, 0);
      }

      public void mostrarResumen() {
          System.out.println("=== RESUMEN DE COMPRA ===");
          System.out.println("Precio original: $" + total);
          System.out.println("Estrategia: " + estrategia.getDescripcion());
          System.out.println("Precio final: $" + calcularTotal());
          System.out.println("Ahorro: $" + (total - calcularTotal()));
      }

      public static void main(String[] args) {
          double precio = 250.0;

          System.out.println("=== ESTRATEGIAS DE DESCUENTO ===\n");
          for (EstrategiaDescuento e : EstrategiaDescuento.values()) {
              CarritoCompra carrito = new CarritoCompra(precio, e);
              carrito.mostrarResumen();
              System.out.println();
          }

          // Uso dinámico
          System.out.println("=== BÚSQUEDA POR DESCRIPCIÓN ===");
          CarritoCompra promo = new CarritoCompra(precio,
                  EstrategiaDescuento.porDescripcion("Código PROMO2024: 20%"));
          promo.mostrarResumen();
      }
  }
  ```

  --- 8. ENUMS CON STREAMS Y LAMBDAS ---

  Los enums se integran perfectamente con la Streams API y lambdas.

  ```java
  import java.util.*;
  import java.util.stream.*;

  public enum CategoriaProducto {
      ELECTRONICA("Electrónica", 0.15),
      ROPA("Ropa y Accesorios", 0.08),
      ALIMENTOS("Alimentos", 0.04),
      LIBROS("Libros", 0.05),
      MUEBLES("Muebles", 0.12),
      JUGUETES("Juguetes", 0.10),
      DEPORTES("Artículos Deportivos", 0.08),
      SALUD("Salud y Belleza", 0.06);

      private final String nombre;
      private final double impuesto;

      CategoriaProducto(String nombre, double impuesto) {
          this.nombre = nombre;
          this.impuesto = impuesto;
      }

      public String getNombre() { return nombre; }
      public double getImpuesto() { return impuesto; }

      public double calcularPrecioConImpuesto(double precioBase) {
          return precioBase * (1 + impuesto);
      }

      // Filtrar categorías con impuesto > 10%
      public static List<CategoriaProducto> impuestoMayorA(double umbral) {
          return Arrays.stream(values())
                  .filter(c -> c.impuesto > umbral)
                  .collect(Collectors.toList());
      }

      // Categoría con mayor impuesto
      public static CategoriaProducto mayorImpuesto() {
          return Arrays.stream(values())
                  .max(Comparator.comparingDouble(CategoriaProducto::getImpuesto))
                  .orElseThrow();
      }

      // Mapa de categorías
      public static Map<CategoriaProducto, String> mapaNombres() {
          return Arrays.stream(values())
                  .collect(Collectors.toMap(
                          c -> c, CategoriaProducto::getNombre));
      }
  }

  public class EnumsYStreams {
      public static void main(String[] args) {
          System.out.println("=== CATEGORÍAS DE PRODUCTOS ===\n");

          // forEach con method reference
          Arrays.stream(CategoriaProducto.values())
                  .forEach(System.out::println);

          System.out.println("\n=== CATEGORÍAS CON IMPUESTO > 10% ===");
          CategoriaProducto.impuestoMayorA(0.10)
                  .forEach(c -> System.out.println(
                          c.getNombre() + " - impuesto: "
                          + (c.getImpuesto() * 100) + "%"));

          System.out.println("\n=== MAYOR IMPUESTO ===");
          CategoriaProducto mayor = CategoriaProducto.mayorImpuesto();
          System.out.println(mayor.getNombre() + ": "
                  + (mayor.getImpuesto() * 100) + "%");

          System.out.println("\n=== PRECIO CON IMPUESTO ===");
          double precioBase = 100.0;
          Arrays.stream(CategoriaProducto.values())
                  .forEach(c -> System.out.printf(
                          "%s: $%.2f -> $%.2f%n",
                          c.getNombre(), precioBase,
                          c.calcularPrecioConImpuesto(precioBase)));

          // Agrupar por rango de impuesto
          System.out.println("\n=== AGRUPADOS POR RANGO DE IMPUESTO ===");
          Map<String, List<CategoriaProducto>> porRango = Arrays
                  .stream(CategoriaProducto.values())
                  .collect(Collectors.groupingBy(c -> {
                      if (c.getImpuesto() <= 0.05) return "Bajo (0-5%)";
                      else if (c.getImpuesto() <= 0.10) return "Medio (5-10%)";
                      else return "Alto (>10%)";
                  }));

          porRango.forEach((rango, cats) -> {
              System.out.println(rango + ":");
              cats.forEach(c -> System.out.println("  - " + c.getNombre()));
          });
      }
  }
  ```

  --- 9. ENUMS CON BODY ESPECÍFICO Y MÉTODOS COMPARTIDOS ---

  Puedes combinar métodos concretos (compartidos por todas las constantes)
  con métodos abstractos (específicos por constante). También puedes
  tener campos que se inicializan distinto según la constante.

  ```java
  import java.util.*;

  public enum PlanSuscripcion {
      FREE("Gratuito", 0.0, 1, false) {
          @Override
          public List<String> obtenerCaracteristicas() {
              return Arrays.asList("1 proyecto", "Almacenamiento 10MB",
                      "Soporte comunitario");
          }
      },
      BASICO("Básico", 9.99, 5, false) {
          @Override
          public List<String> obtenerCaracteristicas() {
              return Arrays.asList("5 proyectos", "Almacenamiento 1GB",
                      "Soporte por email", "API básica");
          }
      },
      PRO("Profesional", 29.99, 50, true) {
          @Override
          public List<String> obtenerCaracteristicas() {
              return Arrays.asList("50 proyectos", "Almacenamiento 50GB",
                      "Soporte prioritario", "API completa",
                      "Estadísticas avanzadas", "Exportación a PDF");
          }
      },
      EMPRESA("Empresarial", 99.99, Integer.MAX_VALUE, true) {
          @Override
          public List<String> obtenerCaracteristicas() {
              return Arrays.asList("Proyectos ilimitados",
                      "Almacenamiento 1TB", "Soporte 24/7 dedicado",
                      "API personalizada", "On-premise disponible",
                      "SSO y LDAP", "Auditoría completa",
                      "SLA garantizado del 99.9%");
          }
      };

      private final String nombre;
      private final double precioMensual;
      private final int maxProyectos;
      private final boolean incluyeAPI;

      PlanSuscripcion(String nombre, double precio, int proyectos,
              boolean api) {
          this.nombre = nombre;
          this.precioMensual = precio;
          this.maxProyectos = proyectos;
          this.incluyeAPI = api;
      }

      public String getNombre() { return nombre; }
      public double getPrecioMensual() { return precioMensual; }
      public int getMaxProyectos() { return maxProyectos; }
      public boolean isIncluyeAPI() { return incluyeAPI; }

      // Método abstracto: cada plan define sus características
      public abstract List<String> obtenerCaracteristicas();

      // Método concreto compartido
      public double getPrecioAnual() {
          return precioMensual * 12 * 0.9;  // 10% descuento anual
      }

      // Sugerir upgrade
      public PlanSuscripcion sugerirUpgrade() {
          PlanSuscripcion[] planes = values();
          int idx = ordinal();
          return idx < planes.length - 1 ? planes[idx + 1] : this;
      }

      // Verificar si el plan soporta cierta característica
      public boolean soportaProyectos(int cantidad) {
          return maxProyectos >= cantidad;
      }

      @Override
      public String toString() {
          return String.format("%s ($%.2f/mes)", nombre, precioMensual);
      }
  }

  public class PlanesSuscripcion {
      public static void main(String[] args) {
          System.out.println("=== PLANES DE SUSCRIPCIÓN ===\n");

          for (PlanSuscripcion plan : PlanSuscripcion.values()) {
              System.out.println(plan);
              System.out.println("  Precio anual: $" + plan.getPrecioAnual());
              System.out.println("  Proyectos: "
                      + (plan.getMaxProyectos() == Integer.MAX_VALUE
                              ? "Ilimitados"
                              : plan.getMaxProyectos()));
              System.out.println("  API: "
                      + (plan.isIncluyeAPI() ? "Sí" : "No"));
              System.out.println("  Características:");
              plan.obtenerCaracteristicas()
                      .forEach(c -> System.out.println("    ✓ " + c));
              System.out.println();
          }

          // Upgrade
          PlanSuscripcion actual = PlanSuscripcion.BASICO;
  ```

  --- 10. ENUMS Y SERIALIZACIÓN ---

  Los enums tienen serialización garantizada por la JVM. El mecanismo
  de serialización de enums es especial: solo se guarda el nombre, no
  los campos. Al deserializar, se devuelve la instancia única existente.

  ```java
  import java.io.*;

  public enum TipoDocumento implements Serializable {
      DNI("Documento Nacional de Identidad", 8),
      RUC("Registro Único de Contribuyente", 11),
      PASAPORTE("Pasaporte", 12),
      CARNET_EXTRANJERIA("Carnet de Extranjería", 15);

      private final String descripcion;
      private final int longitud;

      TipoDocumento(String descripcion, int longitud) {
          this.descripcion = descripcion;
          this.longitud = longitud;
      }

      public String getDescripcion() { return descripcion; }
      public int getLongitud() { return longitud; }

      public boolean validarFormato(String numero) {
          if (numero == null || numero.length() != longitud) return false;
          return numero.matches("\\d{" + longitud + "}");
      }
  }

  public class SerializacionEnum {
      public static void main(String[] args) throws Exception {
          // Serializar
          ByteArrayOutputStream baos = new ByteArrayOutputStream();
          ObjectOutputStream oos = new ObjectOutputStream(baos);
          oos.writeObject(TipoDocumento.DNI);
          oos.close();

          // Deserializar
          ByteArrayInputStream bais = new ByteArrayInputStream(
                  baos.toByteArray());
          ObjectInputStream ois = new ObjectInputStream(bais);
          TipoDocumento deserializado = (TipoDocumento) ois.readObject();
          ois.close();

          // Verificar que es la misma instancia (singleton garantizado)
          System.out.println("Original: " + TipoDocumento.DNI);
          System.out.println("Deserializado: " + deserializado);
          System.out.println("Misma instancia: "
                  + (TipoDocumento.DNI == deserializado));

          // Los campos se conservan
          System.out.println("Descripción: " + deserializado.getDescripcion());
          System.out.println("Longitud: " + deserializado.getLongitud());
          System.out.println("Validar '12345678': "
                  + deserializado.validarFormato("12345678"));
      }
  }
  ```

  --- 11. LIMITACIONES DE LOS ENUMS ---

  1. No pueden heredar de otra clase (ya extienden Enum).
  2. No pueden ser abstractos (aunque pueden tener métodos abstractos).
  3. No pueden tener constructores públicos (solo privados).
  4. No pueden ser genéricos (no puedes tener Enum<T>).
  5. No puedes crear instancias adicionales en runtime.
  6. Los campos deben ser inicializados en el constructor.

  ```java
  // Estas operaciones NO son posibles:

  // Error: no puede extender otra clase
  // public enum MiEnum extends OtraClase { }

  // Error: no puede ser genérico
  // public enum MiEnum<T> { }

  // Error: constructor no puede ser public
  // public enum MiEnum {
  //     A;
  //     public MiEnum() { }  // Error
  // }
  ```

  --- 12. ENUMS CON MAPAS Y BÚSQUEDA AVANZADA ---

  Puedes construir mapas estáticos para búsqueda eficiente por diferentes
  criterios.

  ```java
  import java.util.*;
  import java.util.stream.*;

  public enum Moneda {
      PEN("Sol Peruano", "S/", 1.0),
      USD("Dólar Americano", "$", 3.75),
      EUR("Euro", "€", 4.05),
      MXN("Peso Mexicano", "MX$", 0.22),
      COP("Peso Colombiano", "COL$", 0.00095),
      CLP("Peso Chileno", "CL$", 0.0045),
      ARS("Peso Argentino", "AR$", 0.010),
      BRL("Real Brasileño", "R$", 0.75);

      private final String nombre;
      private final String simbolo;
      private final double tasaAPEN;  // Tasa de cambio a Sol Peruano

      private static final Map<String, Moneda> POR_CODIGO;
      private static final Map<String, Moneda> POR_SIMBOLO;

      static {
          POR_CODIGO = Arrays.stream(values())
                  .collect(Collectors.toMap(
                          Enum::name, m -> m));
          POR_SIMBOLO = Arrays.stream(values())
                  .collect(Collectors.toMap(
                          Moneda::getSimbolo, m -> m));
      }

      Moneda(String nombre, String simbolo, double tasaAPEN) {
          this.nombre = nombre;
          this.simbolo = simbolo;
          this.tasaAPEN = tasaAPEN;
      }

      public String getNombre() { return nombre; }
      public String getSimbolo() { return simbolo; }
      public double getTasaAPEN() { return tasaAPEN; }

      // Conversión entre monedas
      public double convertirA(double monto, Moneda destino) {
          double enPEN = monto * this.tasaAPEN;
          return enPEN / destino.tasaAPEN;
      }

      // Búsqueda por código (más eficiente que valueOf)
      public static Moneda porCodigo(String codigo) {
          Moneda m = POR_CODIGO.get(codigo.toUpperCase());
          if (m == null) {
              throw new IllegalArgumentException(
                      "Moneda no encontrada: " + codigo);
          }
          return m;
      }

      // Búsqueda por símbolo
      public static Moneda porSimbolo(String simbolo) {
          Moneda m = POR_SIMBOLO.get(simbolo);
          if (m == null) {
              throw new IllegalArgumentException(
                      "Símbolo no encontrado: " + simbolo);
          }
          return m;
      }
  }

  public class ConversorMonedas {
      public static void main(String[] args) {
          System.out.println("=== CONVERSOR DE MONEDAS ===\n");

          double monto = 100;
          Moneda origen = Moneda.USD;
          Moneda destino = Moneda.PEN;

          double convertido = origen.convertirA(monto, destino);
          System.out.printf("%.2f %s (%s) = %.2f %s (%s)%n%n",
                  monto, origen, origen.getSimbolo(),
                  convertido, destino, destino.getSimbolo());

          // Tabla de conversiones
          System.out.println("=== TABLA DE CONVERSIONES (100 USD) ===");
          for (Moneda m : Moneda.values()) {
              if (m != Moneda.USD) {
                  double conv = Moneda.USD.convertirA(100, m);
                  System.out.printf("  100 USD = %.2f %s (%s)%n",
                          conv, m, m.getSimbolo());
              }
          }

          // Búsqueda rápida
          System.out.println("\n=== BÚSQUEDAS ===");
          System.out.println("Por código 'EUR': " + Moneda.porCodigo("EUR"));
          System.out.println("Por símbolo '$': " + Moneda.porSimbolo("$"));
          System.out.println("Por símbolo '€': " + Moneda.porSimbolo("€"));
      }
  }
  ```

  --- 13. EJEMPLO INTEGRADOR: MÁQUINA EXPENDEDORA ---

  ```java
  import java.util.*;
  import java.util.stream.*;

  public enum ProductoMaquina {
      REFRESCO("Refresco", 2.50, 10),
      PAPAS("Papas Fritas", 1.50, 15),
      CHOCOLATE("Barra de Chocolate", 2.00, 8),
      GALLETAS("Galletas", 1.00, 20),
      AGUA("Agua Mineral", 1.50, 12),
      CAFE("Café", 3.00, 7),
      CHICLES("Chicles", 0.50, 30),
      SANDWICH("Sandwich", 4.50, 5);

      private final String nombre;
      private final double precio;
      private int stock;

      ProductoMaquina(String nombre, double precio, int stockInicial) {
          this.nombre = nombre;
          this.precio = precio;
          this.stock = stockInicial;
      }

      public String getNombre() { return nombre; }
      public double getPrecio() { return precio; }
      public int getStock() { return stock; }
      public boolean estaDisponible() { return stock > 0; }

      public void reducirStock() {
          if (stock > 0) stock--;
      }

      public void reponer(int cantidad) {
          stock += cantidad;
      }
  }

  class MaquinaExpendedora {
      private double saldo;
      private List<String> historial;

      public MaquinaExpendedora() {
          this.saldo = 0;
          this.historial = new ArrayList<>();
      }

      public void insertarDinero(double monto) {
          if (monto <= 0) {
              throw new IllegalArgumentException(
                      "Monto debe ser positivo: " + monto);
          }
          saldo += monto;
          System.out.printf("Saldo insertado: $%.2f (Total: $%.2f)%n",
                  monto, saldo);
      }

      public void comprar(ProductoMaquina producto) {
          if (!producto.estaDisponible()) {
              System.out.println("Producto agotado: " + producto.getNombre());
              return;
          }
          if (saldo < producto.getPrecio()) {
              System.out.printf("Saldo insuficiente. Necesitas $%.2f más%n",
                      producto.getPrecio() - saldo);
              return;
          }

          saldo -= producto.getPrecio();
          producto.reducirStock();
          historial.add(producto.getNombre());
          System.out.println("¡Disfruta tu " + producto.getNombre() + "!");
      }

      public void mostrarCatalogo() {
          System.out.println("\n=== CATÁLOGO DE PRODUCTOS ===");
          System.out.printf("%-5s %-20s %8s %8s%n",
                  "CÓD.", "PRODUCTO", "PRECIO", "STOCK");
          ProductoMaquina[] productos = ProductoMaquina.values();
          IntStream.range(0, productos.length).forEach(i -> {
              ProductoMaquina p = productos[i];
              System.out.printf("%-5d %-20s $%-6.2f %3d %s%n",
                      i + 1, p.getNombre(), p.getPrecio(), p.getStock(),
                      p.estaDisponible() ? "" : "[AGOTADO]");
          });
          System.out.printf("%nSaldo actual: $%.2f%n", saldo);
      }

      public double devolverCambio() {
          double cambio = saldo;
          if (cambio > 0) {
              System.out.printf("Devolviendo cambio: $%.2f%n", cambio);
              saldo = 0;
          } else {
              System.out.println("No hay cambio que devolver.");
          }
          return cambio;
      }

      public void mostrarEstadisticas() {
          System.out.println("\n=== ESTADÍSTICAS DE VENTAS ===");
          System.out.println("Total ventas: " + historial.size());

          Map<String, Long> ventasPorProducto = historial.stream()
                  .collect(Collectors.groupingBy(
                          p -> p, Collectors.counting()));

          ventasPorProducto.forEach((prod, count) ->
                  System.out.println("  " + prod + ": " + count + " vendidos"));

          double totalVentas = Arrays.stream(ProductoMaquina.values())
                  .filter(p -> historial.contains(p.getNombre()))
                  .mapToDouble(p -> p.getPrecio())
                  .sum();
          System.out.printf("Ingresos totales: $%.2f%n", totalVentas);

          System.out.println("\n=== PRODUCTOS MÁS VENDIDOS ===");
          ventasPorProducto.entrySet().stream()
                  .sorted(Map.Entry.<String, Long>comparingByValue()
                          .reversed())
                  .limit(3)
                  .forEach(e -> System.out.println(
                          "  " + e.getKey() + ": " + e.getValue()));
      }
  }

  public class SimuladorMaquina {
      public static void main(String[] args) {
          MaquinaExpendedora maquina = new MaquinaExpendedora();
          Scanner scanner = new Scanner(System.in);
          int opcion;

          System.out.println("=== MÁQUINA EXPENDEDORA ===\n");

          do {
              maquina.mostrarCatalogo();
              System.out.println("\n1. Insertar dinero");
              System.out.println("2. Comprar producto");
              System.out.println("3. Devolver cambio");
              System.out.println("4. Mostrar estadísticas");
              System.out.println("5. Reponer productos");
              System.out.println("0. Salir");
              System.out.print("Seleccione: ");

              try {
                  opcion = Integer.parseInt(scanner.nextLine());
                  switch (opcion) {
                      case 1 -> {
                          System.out.print("Monto a insertar: ");
                          double monto = Double.parseDouble(
                                  scanner.nextLine());
                          maquina.insertarDinero(monto);
                      }
                      case 2 -> {
                          System.out.print(
                                  "Número de producto (1-"
                                  + ProductoMaquina.values().length + "): ");
                          int idx = Integer.parseInt(scanner.nextLine()) - 1;
                          if (idx >= 0 && idx
                                  < ProductoMaquina.values().length) {
                              maquina.comprar(
                                      ProductoMaquina.values()[idx]);
                          } else {
                              System.out.println("Opción inválida");
                          }
                      }
                      case 3 -> maquina.devolverCambio();
                      case 4 -> maquina.mostrarEstadisticas();
                      case 5 -> {
                          for (ProductoMaquina p : ProductoMaquina.values()) {
                              p.reponer(5);
                          }
                          System.out.println("Productos repuestos.");
                      }
                      case 0 -> System.out.println("Gracias por su compra");
                      default -> System.out.println("Opción inválida");
                  }
              } catch (NumberFormatException e) {
                  System.out.println("Error: ingrese un número válido.");
                  opcion = -1;
              }

              if (opcion != 0) {
                  System.out.println("\nPresione Enter para continuar...");
                  scanner.nextLine();
              }
          } while (opcion != 0);

          scanner.close();
      }
  }
  ```

  --- 14. BUENAS PRÁCTICAS ---

  1. Usa enums en lugar de constantes enteras o strings para conjuntos
     fijos de valores.
  2. Prefiere switch con enums sobre if-else encadenados.
  3. Usa métodos abstractos para comportamientos específicos por constante.
  4. Mantén los enums cohesivos: no agregues métodos no relacionados.
  5. Usa EnumSet y EnumMap para mejor rendimiento con colecciones.
  6. Implementa interfaces cuando los enums necesiten comportamientos
     polimórficos.
  7. Documenta cada constante con JavaDoc si es necesario.
  8. No abuses de los enums: para conjuntos dinámicos, usa colecciones.
  9. Usa Mapas estáticos para búsqueda eficiente por campos.
  10. Considera enums para implementar singletons (garantizado por la JVM).

  ```java
  // Uso de EnumSet y EnumMap
  import java.util.*;

  public class ColeccionesEnum {
      enum Dia { LUN, MAR, MIE, JUE, VIE, SAB, DOM }

      public static void main(String[] args) {
          // EnumSet: más eficiente que HashSet para enums
          Set<Dia> laborables = EnumSet.range(Dia.LUN, Dia.VIE);
          Set<Dia> finde = EnumSet.of(Dia.SAB, Dia.DOM);

          System.out.println("Laborables: " + laborables);
          System.out.println("Fin de semana: " + finde);

          // Operaciones con conjuntos
          Set<Dia> todos = EnumSet.allOf(Dia.class);
          System.out.println("Todos: " + todos);

          // EnumMap: más eficiente que HashMap para enums
          EnumMap<Dia, String> actividades = new EnumMap<>(Dia.class);
          actividades.put(Dia.LUN, "Reunión de equipo");
          actividades.put(Dia.MIE, "Presentación");
          actividades.put(Dia.VIE, "Review semanal");
          System.out.println("\nActividades: " + actividades);
      }
  }
  ```

  --- 15. COMPARATIVA: ENUM VS CLASE VS CONSTANTES ---

  +========================+========================================+========================================+========================================+
  | Característica         | Enum                                   | Clase con constantes static final      | Clase normal                           |
  +========================+========================================+========================================+========================================+
  | Type safety            | Sí (compilación)                       | No (cualquier int/string)              | Sí                                     |
  +------------------------+----------------------------------------+----------------------------------------+----------------------------------------+
  | Número fijo de valores | Sí                                     | No (cualquiera puede crear más)        | Sí (singleton)                         |
  +------------------------+----------------------------------------+----------------------------------------+----------------------------------------+
  | Métodos y campos       | Sí                                     | Sí (pero duplicado)                    | Sí                                     |
  +------------------------+----------------------------------------+----------------------------------------+----------------------------------------+
  | Switch                  | case directo                           | No (if-else)                           | No (if-else o instanceof)             |
  +------------------------+----------------------------------------+----------------------------------------+----------------------------------------+
  | Serialización           | Garantizada (singleton JVM)            | Manual                                 | Manual                                 |
  +------------------------+----------------------------------------+----------------------------------------+----------------------------------------+
  | Iteración (values())   | Sí                                     | Manual (array propio)                  | Manual                                 |
  +------------------------+----------------------------------------+----------------------------------------+----------------------------------------+
  | Herencia               | No (ya extiende Enum)                  | Sí                                     | Sí                                     |
  +------------------------+----------------------------------------+----------------------------------------+----------------------------------------+
  | Performance            | Muy buena                              | Buena                                  | Buena                                  |
  +------------------------+----------------------------------------+----------------------------------------+----------------------------------------+

  PREGUNTAS FRECUENTES (FAQ)

  1. ¿Los enums pueden tener estados mutables?
     Sí, pero no es recomendable porque los enums son singleton globales.
     Si necesitas estado mutable, considera una clase normal.

  2. ¿Puedo tener enums genéricos?
     No. Los enums no pueden tener parámetros de tipo.

  3. ¿Los enums son thread-safe?
     La creación de enums es thread-safe (garantizado por la JVM). Pero
     los campos mutables no son inherentemente thread-safe.

  4. ¿Puedo extender un enum?
     No. Los enums son finales implícitamente. No puedes heredar de un
     enum ni un enum puede heredar de otra clase.

  5. ¿Puedo tener métodos estáticos en un enum?
     Sí, como en cualquier clase.

  6. ¿Cuál es la diferencia entre EnumSet y HashSet?
     EnumSet es mucho más eficiente porque usa bit-vectors internamente.
     Siempre prefiere EnumSet sobre HashSet para enums.

  7. ¿Puedo usar enums como clave en un HashMap?
     Sí, pero prefiere EnumMap para mejor rendimiento.

  8. ¿Los enums pueden tener inicialización estática?
     Sí, los bloques static funcionan en enums.

  9. ¿Puedo comparar enums con ==?
     Sí, es seguro y recomendado. equals() también funciona (usa ==
     internamente).

  10. ¿Puedo clonar un enum?
        No. Clone() lanza CloneNotSupportedException. Los enums son
        singleton.
