1. EL PROBLEMA DE LA API ANTIGUA
===============================================================================

Antes de Java 8, trabajar con fechas era complicado:

```java
import java.util.Date;
import java.util.Calendar;

public class APIAntigua {
    public static void main(String[] args) {
        // Problema 1: Mutabilidad
        Date fecha = new Date();
        System.out.println("Fecha: " + fecha);
        // Los métodos deprecados muestran la fragilidad del diseño
        int dia = fecha.getDate(); // DEPRECATED!
        System.out.println("Día (deprecado): " + dia);

        // Problema 2: Calendar es verboso y confuso
        Calendar cal = Calendar.getInstance();
        cal.set(2024, Calendar.JANUARY, 15);
        int mes = cal.get(Calendar.MONTH); // ¡Enero es 0!
        System.out.println("Mes (0-indexado): " + mes);

        // Problema 3: No es thread-safe
        // DateFormat no es seguro para hilos

        // Problema 4: Diseño inconsistente
        // Date y Calendar tienen métodos similares pero incompatibles

        System.out.println("La API antigua tenía muchos problemas.");
    }
}
```

Limitaciones clave de la API antigua:
1. Mutabilidad: Date y Calendar se modifican, causando bugs en hilos.
2. Diseño confuso: Meses 0-indexados (Enero = 0), años desde 1900.
3. Falta de claridad: Date representa fecha Y hora, sin separación.
4. Formateo no thread-safe: SimpleDateFormat no es seguro para hilos.
5. Operaciones limitadas: Sumar días requiere cálculos manuales incómodos.

La nueva API java.time resuelve todos estos problemas.

2. LOCALDATE: FECHA SIN HORA
===============================================================================

LocalDate representa una fecha (año, mes, día) sin hora ni zona horaria.

```java
import java.time.LocalDate;
import java.time.Month;

public class LocalDateDemo {
    public static void main(String[] args) {
        // Crear fechas
        LocalDate hoy = LocalDate.now();
        LocalDate fechaEspecifica = LocalDate.of(2024, 12, 25);
        LocalDate desdeMes = LocalDate.of(2024, Month.DECEMBER, 25);
        LocalDate desdeString = LocalDate.parse("2024-12-25");

        System.out.println("Hoy: " + hoy);
        System.out.println("Navidad: " + fechaEspecifica);
        System.out.println("Parseada: " + desdeString);

        // Obtener componentes
        System.out.println("\nComponentes de hoy:");
        System.out.println("  Año: " + hoy.getYear());
        System.out.println("  Mes: " + hoy.getMonth()); // Month enum
        System.out.println("  Mes (num): " + hoy.getMonthValue()); // 1-12
        System.out.println("  Día: " + hoy.getDayOfMonth());
        System.out.println("  Día semana: " + hoy.getDayOfWeek());
        System.out.println("  Día año: " + hoy.getDayOfYear());

        // Verificar propiedades
        System.out.println("\nPropiedades:");
        System.out.println("  ¿Bisiesto? " + hoy.isLeapYear());
        System.out.println("  Días en mes: " + hoy.lengthOfMonth());
        System.out.println("  Días en año: " + hoy.lengthOfYear());
    }
}
```

Operaciones con fechas:

```java
public class OperacionesLocalDate {
    public static void main(String[] args) {
        LocalDate hoy = LocalDate.now();

        // Sumar y restar (devuelven NUEVO LocalDate, inmutable)
        LocalDate manana = hoy.plusDays(1);
        LocalDate mesPasado = hoy.minusMonths(1);
        LocalDate añoQueViene = hoy.plusYears(1);
        LocalDate siguienteJueves = hoy.plusDays(
            (DayOfWeek.THURSDAY.getValue() - hoy.getDayOfWeek().getValue() + 7) % 7);

        System.out.println("Hoy: " + hoy);
        System.out.println("Mañana: " + manana);
        System.out.println("Mes pasado: " + mesPasado);
        System.out.println("Año que viene: " + añoQueViene);

        // with(): cambiar componentes específicos
        LocalDate mismoMesOtroDia = hoy.withDayOfMonth(1);
        LocalDate mismoDiaOtroMes = hoy.withMonth(6);
        LocalDate principioAño = hoy.withDayOfYear(1);

        System.out.println("\nCon with():");
        System.out.println("  Primer día del mes: " + mismoMesOtroDia);
        System.out.println("  Mismo día en junio: " + mismoDiaOtroMes);
        System.out.println("  Primer día del año: " + principioAño);

        // Comparaciones
        LocalDate navidad = LocalDate.of(2024, 12, 25);
        System.out.println("\nComparaciones:");
        System.out.println("  ¿hoy es antes que navidad? " + hoy.isBefore(navidad));
        System.out.println("  ¿hoy es después? " + hoy.isAfter(navidad));
        System.out.println("  ¿igual? " + hoy.equals(navidad));
        System.out.println("  Comparación: " + hoy.compareTo(navidad));
    }
}
```

4. LOCALDATETIME: FECHA Y HORA
===============================================================================

LocalDateTime combina LocalDate y LocalTime.

```java
import java.time.LocalDateTime;
import java.time.LocalDate;
import java.time.LocalTime;

public class LocalDateTimeDemo {
    public static void main(String[] args) {
        // Crear
        LocalDateTime ahora = LocalDateTime.now();
        LocalDateTime desdeComponentes = LocalDateTime.of(
            2024, 12, 25, 10, 30, 0);
        LocalDateTime desdeDateYTime = LocalDateTime.of(
            LocalDate.now(), LocalTime.now());
        LocalDateTime desdeString = LocalDateTime.parse("2024-12-25T10:30:00");

        System.out.println("Ahora: " + ahora);
        System.out.println("Navidad 10:30: " + desdeComponentes);

        // Extraer componentes
        LocalDate fecha = ahora.toLocalDate();
        LocalTime hora = ahora.toLocalTime();

        System.out.println("\nComponentes extraídos:");
        System.out.println("  Fecha: " + fecha);
        System.out.println("  Hora: " + hora);

        // Combinar
        LocalDateTime combinado = LocalDateTime.of(fecha, hora);
        System.out.println("  Recombinado: " + combinado);

        // Operaciones
        LocalDateTime dentroDe3Dias = ahora.plusDays(3).plusHours(2);
        LocalDateTime haceUnaSemana = ahora.minusWeeks(1);

        System.out.println("\nOperaciones:");
        System.out.println("  Dentro de 3 días y 2h: " + dentroDe3Dias);
        System.out.println("  Hace una semana: " + haceUnaSemana);

        // Modificar componentes
        LocalDateTime aMediaNoche = ahora.withHour(0).withMinute(0).withSecond(0);
        LocalDateTime ultimoDiaMes = ahora.withDayOfMonth(
            ahora.toLocalDate().lengthOfMonth());

        System.out.println("  A medianoche: " + aMediaNoche);
        System.out.println("  Último día del mes: " + ultimoDiaMes);
    }
}
```

5. DATETIMEFORMATTER: FORMATEO Y PARSEO
===============================================================================

DateTimeFormatter permite convertir fechas a strings y viceversa.

```java
import java.time.LocalDateTime;
import java.time.format.DateTimeFormatter;
import java.time.format.FormatStyle;
import java.util.Locale;

public class DateTimeFormatterDemo {
    public static void main(String[] args) {
        LocalDateTime ahora = LocalDateTime.now();

        // Formateo predefinido
        System.out.println("Formateo predefinido:");
        System.out.println("  ISO_LOCAL_DATE: " +
            ahora.format(DateTimeFormatter.ISO_LOCAL_DATE));
        System.out.println("  ISO_LOCAL_TIME: " +
            ahora.format(DateTimeFormatter.ISO_LOCAL_TIME));
        System.out.println("  ISO_LOCAL_DATE_TIME: " +
            ahora.format(DateTimeFormatter.ISO_LOCAL_DATE_TIME));

        // Formatos localizados
        System.out.println("\nFormatos localizados (US):");
        System.out.println("  SHORT: " +
            ahora.format(DateTimeFormatter.ofLocalizedDateTime(FormatStyle.SHORT)));
        System.out.println("  MEDIUM: " +
            ahora.format(DateTimeFormatter.ofLocalizedDateTime(FormatStyle.MEDIUM)));
        System.out.println("  LONG: " +
            ahora.format(DateTimeFormatter.ofLocalizedDateTime(FormatStyle.LONG)));

        // Patrones personalizados
        DateTimeFormatter personalizado =
            DateTimeFormatter.ofPattern("dd/MM/yyyy HH:mm:ss");
        System.out.println("\nPatrón dd/MM/yyyy HH:mm:ss:");
        System.out.println("  " + ahora.format(personalizado));

        DateTimeFormatter otroFormato =
            DateTimeFormatter.ofPattern("EEEE, d 'de' MMMM 'de' yyyy");
        System.out.println("\nPatrón EEEE, d 'de' MMMM 'de' yyyy:");
        System.out.println("  " + ahora.format(otroFormato));

        // Con locale específico
        DateTimeFormatter formatterES =
            DateTimeFormatter.ofPattern("EEEE, d 'de' MMMM 'de' yyyy", new Locale("es"));
        System.out.println("\nEn español:");
        System.out.println("  " + ahora.format(formatterES));

        // Parseo (string -> fecha)
        String fechaStr = "25/12/2024 10:30:00";
        DateTimeFormatter parser =
            DateTimeFormatter.ofPattern("dd/MM/yyyy HH:mm:ss");
        LocalDateTime fechaParseada =
            LocalDateTime.parse(fechaStr, parser);
        System.out.println("\nParseado: " + fechaStr + " -> " + fechaParseada);

        // Tabla de patrones comunes
        System.out.println("\nPatrones comunes:");
        System.out.println("  yyyy-MM-dd -> " +
            ahora.format(DateTimeFormatter.ofPattern("yyyy-MM-dd")));
        System.out.println("  dd/MM/yyyy -> " +
            ahora.format(DateTimeFormatter.ofPattern("dd/MM/yyyy")));
        System.out.println("  HH:mm:ss -> " +
            ahora.format(DateTimeFormatter.ofPattern("HH:mm:ss")));
        System.out.println("  dd-MM-yyyy HH:mm -> " +
            ahora.format(DateTimeFormatter.ofPattern("dd-MM-yyyy HH:mm")));
    }
}
```

Letras de patrón más comunes:
- y: año (yyyy = 2024)
- M: mes (MM = 01-12, MMM = Ene, MMMM = Enero)
- d: día (dd = 01-31)
- E: día semana (E = jue, EEEE = jueves)
- H: hora (0-23)
- m: minuto
- s: segundo
- a: AM/PM

6. PERIOD: DIFERENCIA ENTRE FECHAS
===============================================================================

Period mide diferencias en años, meses y días.

```java
import java.time.LocalDate;
import java.time.Period;

public class PeriodDemo {
    public static void main(String[] args) {
        LocalDate nacimiento = LocalDate.of(1990, 5, 15);
        LocalDate hoy = LocalDate.now();

        // Calcular período entre dos fechas
        Period edad = Period.between(nacimiento, hoy);

        System.out.println("Nacimiento: " + nacimiento);
        System.out.println("Hoy: " + hoy);
        System.out.println("Edad: " + edad.getYears() + " años, " +
            edad.getMonths() + " meses, " + edad.getDays() + " días");

        // Crear períodos
        Period unMes = Period.ofMonths(1);
        Period dosSemanas = Period.ofWeeks(2);
        Period personalizado = Period.of(1, 6, 15); // 1 año, 6 meses, 15 días

        System.out.println("\nPeríodos:");
        System.out.println("  Un mes: " + unMes);
        System.out.println("  Dos semanas: " + dosSemanas);
        System.out.println("  Personalizado: " + personalizado);

        // Sumar/restar períodos
        LocalDate fechaLimite = hoy.plus(Period.ofMonths(3));
        System.out.println("\nFecha límite (+3 meses): " + fechaLimite);

        // Verificar propiedades
        Period periodo = Period.between(hoy, LocalDate.of(2025, 1, 1));
        System.out.println("\nHasta Año Nuevo:");
        System.out.println("  Meses: " + periodo.toTotalMonths());
        System.out.println("  Días: " + periodo.getDays());
        System.out.println("  Negativo: " + periodo.isNegative());

        // Período con negación
        Period negativo = periodo.negated();
        System.out.println("  Período negado: " + negativo);
    }
}
```

7. DURATION: DIFERENCIA ENTRE TIEMPOS
===============================================================================

Duration mide diferencias en nanosegundos (horas, minutos, segundos).

```java
import java.time.Duration;
import java.time.LocalTime;
import java.time.LocalDateTime;

public class DurationDemo {
    public static void main(String[] args) {
        // Entre horas
        LocalTime inicio = LocalTime.of(8, 0);
        LocalTime fin = LocalTime.of(17, 30);
        Duration jornada = Duration.between(inicio, fin);

        System.out.println("Jornada laboral:");
        System.out.println("  Inicio: " + inicio);
        System.out.println("  Fin: " + fin);
        System.out.println("  Duración: " + jornada.toHours() + " horas y " +
            jornada.toMinutesPart() + " minutos");
        System.out.println("  Total minutos: " + jornada.toMinutes());

        // Crear duraciones
        Duration dosHoras = Duration.ofHours(2);
        Duration treintaMin = Duration.ofMinutes(30);
        Duration cincoSeg = Duration.ofSeconds(5);
        Duration personalizado = Duration.ofMinutes(90); // 1h 30m

        System.out.println("\nDuraciones:");
        System.out.println("  2 horas en seg: " + dosHoras.getSeconds());
        System.out.println("  90 min: " + personalizado);

        // Operaciones con LocalDateTime
        LocalDateTime evento = LocalDateTime.of(2024, 12, 31, 23, 59);
        LocalDateTime mediaNoche = evento.plusMinutes(1);
        Duration hastaFinAño = Duration.between(LocalDateTime.now(), evento);

        System.out.println("\nHasta Año Nuevo:");
        System.out.println("  Horas: " + hastaFinAño.toHours());
        System.out.println("  Minutos: " + hastaFinAño.toMinutes());

        // Métodos de conversión
        Duration d = Duration.ofHours(25);
        System.out.println("\nConversiones de duración:");
        System.out.println("  25 horas en días: " + d.toDays());
        System.out.println("  Horas: " + d.toHours());
        System.out.println("  Minutos: " + d.toMinutes());
        System.out.println("  Segundos: " + d.toSeconds());
        System.out.println("  Millis: " + d.toMillis());
        System.out.println("  Nanos: " + d.toNanos());

        // Sumar duraciones
        Duration total = Duration.ofHours(8).plus(Duration.ofMinutes(45));
        System.out.println("\nTotal: " + total.toHours() + "h " +
            total.toMinutesPart() + "min");
    }
}
```

```java
        System.out.println("  Tokyo: " + ahoraTokyo);
        System.out.println("  NY: " + ahoraNY);

        ZonedDateTime enTokyo = ahoraMadrid.withZoneSameInstant(zonaTokyo);

        ZoneOffset offsetMadrid = ahoraMadrid.getOffset();

        // Listar zonas disponibles
        Set<String> zonas = ZoneId.getAvailableZoneIds();
        System.out.println("\nZonas disponibles: " + zonas.size());
        // Mostrar algunas europeas
        zonas.stream()
            .filter(z -> z.startsWith("Europe"))
            .limit(10)
            .forEach(System.out::println);
```

9. INSTANT: MOMENTO EN LA LÍNEA DE TIEMPO
===============================================================================

Instant representa un momento específico en la línea de tiempo UTC.

```java
import java.time.Instant;
import java.time.Duration;
import java.time.LocalDateTime;
import java.time.ZoneId;

public class InstantDemo {
    public static void main(String[] args) {
        // Instant.now() devuelve el momento actual en UTC
        Instant ahora = Instant.now();
        System.out.println("Instant ahora: " + ahora);

        // Instant no tiene fecha legible, pero es útil para:
        // - Timestamps precisos
        // - Medir tiempo entre eventos
        // - Trabajar con máquina (no humana)

        // Medir tiempo de ejecución
        Instant inicio = Instant.now();

        // Simular trabajo
        for (long i = 0; i < 100_000_000; i++) {
            Math.sqrt(i);
        }

        Instant fin = Instant.now();
        Duration duracion = Duration.between(inicio, fin);
        System.out.println("Tiempo de ejecución: " +
            duracion.toMillis() + " ms");

        // Convertir a LocalDateTime
        LocalDateTime fechaHora = LocalDateTime.ofInstant(
            ahora, ZoneId.systemDefault());
        System.out.println("Como LocalDateTime: " + fechaHora);

        // Época (epoch)
        System.out.println("Segundos desde epoch: " + ahora.getEpochSecond());
        System.out.println("Nanosegundos: " + ahora.getNano());

        // Crear desde epoch
        Instant epoch = Instant.ofEpochSecond(1_000_000_000L);
        System.out.println("1 billion seconds after epoch: " + epoch);
    }
}
```

```java
// EJEMPLO PRÁCTICO: Calculadora de Edad y Fechas
class CalculadoraFechas {
    public static void main(String[] args) {
        // Calcular edad exacta
        LocalDate nacimiento = LocalDate.of(1990, 5, 15);
        LocalDate hoy = LocalDate.now();
        Period edad = Period.between(nacimiento, hoy);

        System.out.println("=== CALCULADORA DE EDAD ===");
        System.out.println("Naciste el: " + nacimiento);
        System.out.println("Hoy es: " + hoy);
        System.out.println("Tienes: " + edad.getYears() + " años, " +
            edad.getMonths() + " meses y " + edad.getDays() + " días");

        // Próximo cumpleaños
        LocalDate proxCumple = nacimiento.withYear(hoy.getYear());
        if (proxCumple.isBefore(hoy) || proxCumple.equals(hoy)) {
            proxCumple = proxCumple.plusYears(1);
        }
        Period falta = Period.between(hoy, proxCumple);
        System.out.println("Próximo cumpleaños: " + proxCumple);
        System.out.println("Faltan: " + falta.getMonths() + " meses y " +
            falta.getDays() + " días");

        // Días laborables hasta fin de mes
        LocalDate finMes = hoy.withDayOfMonth(hoy.lengthOfMonth());
        int laborables = 0;
        for (LocalDate d = hoy.plusDays(1);
             !d.isAfter(finMes);
             d = d.plusDays(1)) {
            DayOfWeek dw = d.getDayOfWeek();
            if (dw != DayOfWeek.SATURDAY && dw != DayOfWeek.SUNDAY) {
                laborables++;
            }
        }
        System.out.println("Días laborables hasta fin de mes: " + laborables);

        // Formato personalizado
        DateTimeFormatter formatter =
            DateTimeFormatter.ofPattern("EEEE, d 'de' MMMM 'de' yyyy",
                new Locale("es"));
        System.out.println("\nHoy formateado: " + hoy.format(formatter));
    }
}
```

RESUMEN DEL CAPÍTULO

En este capítulo hemos aprendido:

• La API java.time (Java 8+) reemplaza a Date y Calendar con un diseño
  inmutable, fluido e intuitivo.

• LocalDate representa fecha (año, mes, día), LocalTime representa hora,
  y LocalDateTime combina ambos.

• DateTimeFormatter permite formatear y parsear fechas con patrones
  predefinidos, localizados o personalizados.

• Period mide diferencias entre fechas en años, meses y días.

• Duration mide diferencias entre tiempos en horas, minutos, segundos
  y nanosegundos.

• ZonedDateTime maneja zonas horarias con ZoneId.

• Instant representa un momento exacto en la línea de tiempo UTC,
  ideal para timestamps y mediciones.

• Clases especializadas: YearMonth, MonthDay, Year, DayOfWeek, Month.

PREGUNTAS FRECUENTES
===============================================================================

❓ **¿Debo usar java.util.Date alguna vez?**
No. Si estás usando Java 8 o superior, usa siempre java.time. Si trabajas
con APIs antiguas que usan Date, conviértelos con Date.from(instant) y
date.toInstant().

❓ **¿LocalDate vs LocalDateTime?**
Usa LocalDate si solo necesitas fecha (sin hora). Usa LocalDateTime si
necesitas fecha y hora. Si necesitas zona horaria, usa ZonedDateTime.

❓ **¿Cómo convierto Date antiguo a LocalDate?**
new Date().toInstant().atZone(ZoneId.systemDefault()).toLocalDate()

❓ **¿Qué significa que las clases sean inmutables?**
Una vez creado un objeto, no puede modificarse. Los métodos como plusDays()
devuelven un NUEVO objeto. Esto es seguro para hilos y evita bugs.

❓ **¿Cómo manejo fechas con JDBC?**
Usa java.sql.Date, pero conviértelas: Date.valueOf(localDate) y
resultSet.getDate().toLocalDate().

❓ **¿Qué es el problema de los años bisiestos?**
LocalDate lo maneja automáticamente. Por ejemplo, LocalDate.of(2024, 2, 29)
es válido (2024 es bisiesto), pero 2023-02-29 lanza DateTimeException.
