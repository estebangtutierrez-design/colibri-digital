1. ¿QUÉ SON LOS FLUJOS DE ENTRADA/SALIDA?
===============================================================================

Java trata la entrada y salida de datos como "flujos" (streams). Un stream
es una secuencia de datos que fluye desde un origen hacia un destino.

**System.in:** Flujo de entrada estándar (normalmente el teclado).
**System.out:** Flujo de salida estándar (normalmente la consola).
**System.err:** Flujo de error estándar (normalmente la consola).

```java
// Los tres flujos estándar de Java

class FlujosEstandar {
    public static void main(String[] args) {
        System.out.println("=== FLUJOS ESTÁNDAR DE JAVA ===");
        
        // System.out: salida normal (PrintStream)
        System.out.println("System.out.println - Salida normal");
        System.out.print("System.out.print - Sin salto de línea ");
        System.out.print("(continúa aquí)\n");
        System.out.printf("System.out.printf - Con formato: %.2f%n", 3.14159);
        
        // System.err: salida de error (PrintStream)
        System.err.println("System.err.println - Mensaje de error");
        System.err.println("(Se muestra en la consola, a menudo en rojo)");
        
        // System.in: entrada estándar (InputStream)
        System.out.println("\nSystem.in - Lee datos del teclado");
        System.out.println("(Lo usaremos con Scanner para leer fácilmente)");
        
        // Propiedades de los flujos
        System.out.println("\n=== PROPIEDADES ===");
        System.out.println("System.out es un: " + System.out.getClass().getName());
        System.out.println("System.in es un: " + System.in.getClass().getName());
        System.out.println("System.err es un: " + System.err.getClass().getName());
    }
}
```

Para leer datos de System.in de forma sencilla, Java nos ofrece la clase
Scanner, que veremos a continuación.

2. LA CLASE SCANNER: LECTURA BÁSICA
===============================================================================

Scanner es una clase de java.util que permite leer datos de diferentes
fuentes. Es la forma más fácil de leer entrada del teclado.

```java
// Uso básico de Scanner

import java.util.Scanner;

class ScannerBasico {
    public static void main(String[] args) {
        // Crear el Scanner (se pasa System.in como fuente de entrada)
        Scanner scanner = new Scanner(System.in);
        
        System.out.println("=== LECTURA BÁSICA CON SCANNER ===");
        System.out.print("¿Cómo te llamas? ");
        
        // nextLine(): lee una línea completa (hasta Enter)
        String nombre = scanner.nextLine();
        
        System.out.print("¿Cuántos años tienes? ");
        
        // nextInt(): lee un número entero
        int edad = scanner.nextInt();
        
        // IMPORTANTE: nextInt() NO consume el salto de línea
        // Si luego llamas a nextLine(), capturará ese salto
        // Solución: agregar scanner.nextLine() después de nextInt()
        scanner.nextLine();  // Consumir el salto de línea
        
        System.out.print("¿Cuál es tu ciudad? ");
        String ciudad = scanner.nextLine();
        
        // Mostrar la información recopilada
        System.out.println("\n=== TUS DATOS ===");
        System.out.println("Nombre: " + nombre);
        System.out.println("Edad: " + edad);
        System.out.println("Ciudad: " + ciudad);
        System.out.println("¡Gracias por compartir tus datos!");
        
        // Cerrar el Scanner (libera recursos)
        scanner.close();
    }
}
```

**Problema común: nextLine después de nextInt**

Cuando usas nextInt(), nextDouble(), etc., estos métodos NO consumen el
salto de línea (\n) que se genera al presionar Enter. Si luego llamas a
nextLine(), este método captura ese salto de línea pendiente y no te
permite escribir.

```java
// Demostración del problema con nextLine

import java.util.Scanner;

class ProblemaNextLine {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        System.out.println("=== DEMOSTRACIÓN DEL PROBLEMA ===");
        System.out.print("Edad: ");
        int edad = sc.nextInt();  // El usuario escribe 25 y presiona Enter
        // En el buffer queda: \n (el salto de línea)
        
        System.out.print("Nombre: ");
        String nombre = sc.nextLine();  // Lee el \n pendiente -> devuelve ""
        
        System.out.println("\nEdad: " + edad);
        System.out.println("Nombre: '" + nombre + "' (vacío! El problema!)");
        
        sc.close();
        
        System.out.println("\nSOLUCIÓN: Agregar sc.nextLine() después de nextInt()");
        System.out.println("para consumir el salto de línea pendiente.");
    }
}
```

```java
// Solución correcta

import java.util.Scanner;

class SolucionNextLine {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        System.out.println("=== SOLUCIÓN DEL PROBLEMA ===");
        System.out.print("Edad: ");
        int edad = sc.nextInt();
        sc.nextLine();  // 👈 Consumir el salto de línea pendiente
        
        System.out.print("Nombre: ");
        String nombre = sc.nextLine();  // Ahora funciona correctamente
        
        System.out.print("Altura (m): ");
        double altura = sc.nextDouble();
        sc.nextLine();  // 👈 Consumir el salto de línea
        
        System.out.println("\nEdad: " + edad);
        System.out.println("Nombre: " + nombre);
        System.out.println("Altura: " + altura);
        
        sc.close();
    }
}
```

3. MÉTODOS DE SCANNER PARA DIFERENTES TIPOS
===============================================================================

Scanner ofrece métodos para leer diferentes tipos de datos:

```java
// Todos los métodos de lectura de Scanner

import java.util.Scanner;

class MetodosScanner {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        System.out.println("=== MÉTODOS DE LECTURA DE SCANNER ===");
        
        // nextLine() - Lee una línea completa
        System.out.print("1. nextLine() - Frase: ");
        String frase = sc.nextLine();
        System.out.println("   Leído: '" + frase + "'");
        
        // next() - Lee una palabra (hasta espacio)
        System.out.print("2. next() - Palabra: ");
        String palabra = sc.next();
        System.out.println("   Leído: '" + palabra + "'");
        sc.nextLine();  // Limpiar buffer
        
        // nextInt() - Lee un entero
        System.out.print("3. nextInt() - Entero: ");
        int entero = sc.nextInt();
        System.out.println("   Leído: " + entero);
        sc.nextLine();
        
        // nextDouble() - Lee un decimal
        System.out.print("4. nextDouble() - Decimal: ");
        double decimal = sc.nextDouble();
        System.out.println("   Leído: " + decimal);
        sc.nextLine();
        
        // nextLong() - Lee un long
        System.out.print("5. nextLong() - Long: ");
        long largo = sc.nextLong();
        System.out.println("   Leído: " + largo);
        sc.nextLine();
        
        // nextFloat() - Lee un float
        System.out.print("6. nextFloat() - Float: ");
        float flotante = sc.nextFloat();
        System.out.println("   Leído: " + flotante);
        sc.nextLine();
        
        // nextBoolean() - Lee true/false
        System.out.print("7. nextBoolean() - Boolean (true/false): ");
        boolean bool = sc.nextBoolean();
        System.out.println("   Leído: " + bool);
        sc.nextLine();
        
        // nextByte() - Lee un byte
        System.out.print("8. nextByte() - Byte: ");
        byte b = sc.nextByte();
        System.out.println("   Leído: " + b);
        sc.nextLine();
        
        // nextShort() - Lee un short
        System.out.print("9. nextShort() - Short: ");
        short s = sc.nextShort();
        System.out.println("   Leído: " + s);
        sc.nextLine();
        
        sc.close();
    }
}
```

```java
// Métodos de verificación (hasNext*)

import java.util.Scanner;

class MetodosVerificacion {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        System.out.println("=== MÉTODOS DE VERIFICACIÓN (hasNext*) ===");
        System.out.println("Estos métodos verifican si hay un valor del");
        System.out.println("tipo especificado disponible.");
        System.out.println("Son útiles para evitar errores de entrada.\n");
        
        System.out.print("Ingresa un número entero: ");

            int numero = sc.nextInt();
            System.out.println("¡Número válido! " + numero);
        } else {
            System.out.println("❌ Eso no es un entero válido.");
            String entradaInvalida = sc.nextLine();  // Limpiar entrada incorrecta
            System.out.println("   (Se recibió: '" + entradaInvalida + "')");
        }
        
        sc.close();
        
        System.out.println("\n=== MÉTODOS DE VERIFICACIÓN DISPONIBLES ===");
        System.out.println("hasNext()      - Hay algún token?");
        System.out.println("hasNextLine()  - Hay otra línea?");
        System.out.println("hasNextInt()   - Hay un entero?");
        System.out.println("hasNextDouble()- Hay un decimal?");
        System.out.println("hasNextBoolean()- Hay un booleano?");
        System.out.println("hasNextLong()  - Hay un long?");
        System.out.println("hasNextFloat() - Hay un float?");
    }
}
```

4. VALIDACIÓN DE ENTRADA CON SCANNER
===============================================================================

Uno de los usos más importantes de Scanner es validar la entrada del
usuario. Los métodos hasNext* son perfectos para esto.

```java
// Validación de entrada con hasNext* y bucles

import java.util.Scanner;

class ValidacionEntrada {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        System.out.println("=== VALIDACIÓN DE ENTRADA ===");
        
        // Validar entero
        System.out.print("Ingresa un número entero: ");
        while (!sc.hasNextInt()) {
            System.out.print("❌ No es válido. Ingresa un entero: ");
            sc.nextLine();  // Descartar entrada inválida
        }
        int numero = sc.nextInt();
        sc.nextLine();
        System.out.println("✅ Entero válido: " + numero);
        
        // Validar double positivo
        System.out.print("\nIngresa un precio positivo: ");
        while (!sc.hasNextDouble()) {
            System.out.print("❌ No es válido. Ingresa un número: ");
            sc.nextLine();
        }
        double precio = sc.nextDouble();
        sc.nextLine();
        
        if (precio < 0) {
            System.out.println("❌ El precio no puede ser negativo. Se usará 0.0");
            precio = 0.0;
        }
        System.out.println("✅ Precio: $" + precio);
        
        // Validar nombre no vacío
        System.out.print("\nIngresa tu nombre (obligatorio): ");
        String nombre = sc.nextLine().trim();
        while (nombre.isEmpty()) {
            System.out.print("❌ El nombre no puede estar vacío. Ingresa: ");
            nombre = sc.nextLine().trim();
        }
        System.out.println("✅ Nombre: " + nombre);
        
        // Validar rango de edad
        System.out.print("\nIngresa tu edad (1-120): ");
        int edad;
        while (true) {
            if (sc.hasNextInt()) {
                edad = sc.nextInt();
                sc.nextLine();
                if (edad >= 1 && edad <= 120) {
                    break;  // Edad válida
                }
            } else {
                sc.nextLine();  // Descartar
            }
            System.out.print("❌ Edad inválida (1-120). Ingresa de nuevo: ");
        }
        System.out.println("✅ Edad: " + edad);
        
        System.out.println("\n¡Datos validados correctamente!");
        sc.close();
    }
}
```

```java
// Menú interactivo con validación

import java.util.Scanner;

class MenuInteractivo {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int opcion;
        
        System.out.println("=== MENÚ INTERACTIVO ===");
        
        do {
            // Mostrar menú
            System.out.println("\n=== CALCULADORA BÁSICA ===");
            System.out.println("1. Sumar");
            System.out.println("2. Restar");
            System.out.println("3. Multiplicar");
            System.out.println("4. Dividir");
            System.out.println("0. Salir");
            System.out.print("Elige una opción: ");
            
            // Validar opción
            while (!sc.hasNextInt()) {
                System.out.print("❌ Opción inválida. Elige 0-4: ");
                sc.nextLine();
            }
            opcion = sc.nextInt();
            sc.nextLine();
            
            if (opcion >= 1 && opcion <= 4) {
                System.out.print("Ingresa el primer número: ");
                while (!sc.hasNextDouble()) {
                    System.out.print("❌ Número inválido: ");
                    sc.nextLine();
                }
                double a = sc.nextDouble();
                sc.nextLine();
                
                System.out.print("Ingresa el segundo número: ");
                while (!sc.hasNextDouble()) {
                    System.out.print("❌ Número inválido: ");
                    sc.nextLine();
                }
                double b = sc.nextDouble();
                sc.nextLine();
                
                double resultado = 0;
                String operacion = "";
                
                switch (opcion) {
                    case 1 -> { resultado = a + b; operacion = "Suma"; }
                    case 2 -> { resultado = a - b; operacion = "Resta"; }
                    case 3 -> { resultado = a * b; operacion = "Multiplicación"; }
                    case 4 -> {
                        if (b == 0) {
                            System.out.println("❌ Error: División por cero");
                            continue;
                        }
                        resultado = a / b;
                        operacion = "División";
                    }
                }
                
                System.out.printf("%s: %.2f %s %.2f = %.2f%n", operacion, a, 
                    opcion == 1 ? "+" : opcion == 2 ? "-" : opcion == 3 ? "*" : "/", b, resultado);
            } else if (opcion != 0) {
                System.out.println("❌ Opción no válida. Intenta de nuevo.");
            }
            
        } while (opcion != 0);
        
        System.out.println("¡Hasta luego!");
        sc.close();
    }
}
```

5. SCANNER CON OTRAS FUENTES DE DATOS
===============================================================================

Scanner no solo lee del teclado. Puede leer de archivos, Strings y otras
fuentes. Esto lo hace muy versátil.

```java
// Scanner con diferentes fuentes

import java.util.Scanner;

class ScannerFuentes {
    public static void main(String[] args) {
        System.out.println("=== SCANNER CON DIFERENTES FUENTES ===");
        
        // 1. Scanner con String (analizar texto)
        System.out.println("\n1. Scanner con String:");
        String datos = "Juan 25 Madrid\nAna 30 Barcelona";
        Scanner scString = new Scanner(datos);
        
        while (scString.hasNextLine()) {
            String linea = scString.nextLine();
            System.out.println("  Línea: " + linea);
        }
        scString.close();
        
        // 2. Scanner con String y delimitador
        System.out.println("\n2. Scanner con delimitador personalizado:");
        String csv = "Java,Python,C++,JavaScript";
        Scanner scCSV = new Scanner(csv);
        scCSV.useDelimiter(",");  // Usar coma como separador
        
        while (scCSV.hasNext()) {
            System.out.println("  Lenguaje: " + scCSV.next());
        }
        scCSV.close();
        
        // 3. Scanner con números en un String
        System.out.println("\n3. Scanner para extraer números:");
        String mezclado = "El precio es 25.99 euros y el descuento 5.50";
        Scanner scNumeros = new Scanner(mezclado);
        
        double total = 0;
        int contador = 0;
        while (scNumeros.hasNext()) {
            if (scNumeros.hasNextDouble()) {
                double valor = scNumeros.nextDouble();
                total += valor;
                contador++;
                System.out.println("  Número encontrado: " + valor);
            } else {
                scNumeros.next();  // Saltar token no numérico
            }
        }
        System.out.println("  Suma total: " + total);
        scNumeros.close();
        
        // 4. Delimitador con expresión regular
        System.out.println("\n4. Delimitador con regex:");
        String texto = "Hola. Mundo! Java; es: genial.";
        Scanner scRegex = new Scanner(texto);
        scRegex.useDelimiter("[.?!;:]+");  // Separar por puntuación
        
        while (scRegex.hasNext()) {
            String token = scRegex.next().trim();
            if (!token.isEmpty()) {
                System.out.println("  Token: " + token);
            }
        }
        scRegex.close();
    }
}
```

6. SALIDA FORMATEADA CON PRINTF
===============================================================================

System.out.printf te permite controlar exactamente cómo se muestran los
datos. Es especialmente útil para tablas, alineación y precisión numérica.

```java
// Salida formateada con printf

import java.util.Scanner;

class SalidaFormateada {
    public static void main(String[] args) {
        System.out.println("=== SALIDA FORMATEADA CON PRINTF ===\n");
        
        // Conceptos básicos
        System.out.println("SINTAXIS: %[flag][ancho][.precisión]tipo\n");
        
        // Enteros (%d)
        System.out.println("--- ENTEROS (%d) ---");
        System.out.printf("Por defecto: '%d'%n", 123);
        System.out.printf("Con ancho 10: '%10d'%n", 123);
        System.out.printf("Relleno ceros: '%010d'%n", 123);
        System.out.printf("Alineado izq: '%-10d'%n", 123);
        System.out.printf("Con signo: '%+d'%n", 123);
        System.out.printf("Negativo: '%d'%n", -123);
        System.out.printf("Con coma: '%,d'%n", 1234567);
        
        // Decimales (%f)
        System.out.println("\n--- DECIMALES (%f) ---");
        System.out.printf("Por defecto: '%f'%n", 3.14159);
        System.out.printf("2 decimales: '%.2f'%n", 3.14159);
        System.out.printf("4 decimales: '%.4f'%n", 3.14159);
        System.out.printf("Ancho 10, 2 dec: '%10.2f'%n", 3.14159);
        System.out.printf("Relleno ceros: '%010.2f'%n", 3.14159);
        System.out.printf("Con coma: '%,.2f'%n", 1234567.89);
        
        // Strings (%s)
        System.out.println("\n--- STRINGS (%s) ---");
        System.out.printf("Por defecto: '%s'%n", "Hola");
        System.out.printf("Ancho 10: '%10s'%n", "Hola");
        System.out.printf("Alineado izq: '%-10s'%n", "Hola");
        System.out.printf("Mayúsculas: '%S'%n", "hola");
        System.out.printf("Truncado 3: '%.3s'%n", "Hola Mundo");
        
        // Caracteres (%c)
        System.out.println("\n--- CARACTERES (%c) ---");
        System.out.printf("Carácter: '%c'%n", 'A');
        System.out.printf("Código 65: '%c'%n", 65);
        
        // Booleanos (%b)
        System.out.println("\n--- BOOLEANOS (%b) ---");
        System.out.printf("true: '%b'%n", true);
        System.out.printf("false: '%b'%n", false);
        System.out.printf("Mayúsculas: '%B'%n", true);
        
        // Fechas (%t)
        System.out.println("\n--- FECHAS (%t) ---");
        System.out.printf("Fecha completa: '%tc'%n", new java.util.Date());
        System.out.printf("Fecha ISO: '%tF'%n", new java.util.Date());
        System.out.printf("Hora 24h: '%tT'%n", new java.util.Date());
        System.out.printf("Día/Mes/Año: '%td/%tm/%tY'%n", 
            new java.util.Date(), new java.util.Date(), new java.util.Date());
        
        // Ejemplo de tabla profesional
        System.out.println("\n--- TABLA PROFESIONAL ---");
        String formatoTabla = "| %-15s | %-5s | %-10s | %-8s |%n";
        System.out.println("+-----------------+-------+------------+----------+");
        System.out.printf(formatoTabla, "NOMBRE", "EDAD", "CIUDAD", "SALARIO");
        System.out.println("+-----------------+-------+------------+----------+");
        System.out.printf(formatoTabla, "Ana López", "28", "Madrid", "3500.50");
        System.out.printf(formatoTabla, "Luis Pérez", "35", "Barcelona", "4200.00");
        System.out.printf(formatoTabla, "María Ruiz", "31", "Valencia", "2800.75");
        System.out.println("+-----------------+-------+------------+----------+");
    }
}
```

Vamos a construir una aplicación completa que integra todo lo aprendido:

```java
// Sistema de registro de usuarios con validación

import java.util.Scanner;

class SistemaRegistro {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        System.out.println("=================================");
        System.out.println("   SISTEMA DE REGISTRO DE USUARIOS");
        System.out.println("=================================\n");
        
        // Variables para almacenar datos
        String nombre = "";
        String email = "";
        int edad = 0;
        double altura = 0.0;
        double peso = 0.0;
        boolean estudiante = false;
        
        // 1. Nombre
        System.out.print("Nombre completo: ");
        nombre = sc.nextLine().trim();
        while (nombre.isEmpty() || nombre.length() < 3) {
            System.out.print("❌ Mínimo 3 caracteres. Ingresa: ");
            nombre = sc.nextLine().trim();
        }
        
        // 2. Email
        System.out.print("Email: ");
        email = sc.nextLine().trim().toLowerCase();
        while (!email.contains("@") || !email.contains(".") || email.length() < 5) {
            System.out.print("❌ Email inválido. Ingresa: ");
            email = sc.nextLine().trim().toLowerCase();
        }
        
        // 3. Edad
        System.out.print("Edad: ");
        while (!sc.hasNextInt()) {
            System.out.print("❌ Ingresa un número: ");
            sc.nextLine();
        }
        edad = sc.nextInt();
        sc.nextLine();
        while (edad < 1 || edad > 120) {
            System.out.print("❌ Edad inválida (1-120): ");
            while (!sc.hasNextInt()) {
                System.out.print("❌ Ingresa un número: ");
                sc.nextLine();
            }
            edad = sc.nextInt();
            sc.nextLine();
        }
        
        // 4. Altura
        System.out.print("Altura (m, ej: 1.75): ");
        while (!sc.hasNextDouble()) {
            System.out.print("❌ Ingresa un número decimal: ");
            sc.nextLine();
        }
        altura = sc.nextDouble();
        sc.nextLine();
        while (altura < 0.5 || altura > 2.5) {
            System.out.print("❌ Altura inválida (0.5-2.5): ");
            while (!sc.hasNextDouble()) {
                System.out.print("❌ Ingresa un número decimal: ");
                sc.nextLine();
            }
            altura = sc.nextDouble();
            sc.nextLine();
        }
        
        // 5. Peso
        System.out.print("Peso (kg, ej: 70.5): ");
        while (!sc.hasNextDouble()) {
            System.out.print("❌ Ingresa un número decimal: ");
            sc.nextLine();
        }
        peso = sc.nextDouble();
        sc.nextLine();
        while (peso < 10 || peso > 300) {
            System.out.print("❌ Peso inválido (10-300): ");
            while (!sc.hasNextDouble()) {
                System.out.print("❌ Ingresa un número decimal: ");
                sc.nextLine();
            }
            peso = sc.nextDouble();
            sc.nextLine();
        }
        
        // 6. Estudiante
        System.out.print("¿Eres estudiante? (sí/no): ");
        String resp = sc.nextLine().trim().toLowerCase();
        estudiante = resp.equals("sí") || resp.equals("si") || resp.equals("s") || resp.equals("yes");
        
        // Calcular IMC
        double imc = peso / (altura * altura);
        String clasificacionIMC;
        if (imc < 18.5) {
            clasificacionIMC = "Bajo peso";
        } else if (imc < 25) {
            clasificacionIMC = "Normal";
        } else if (imc < 30) {
            clasificacionIMC = "Sobrepeso";
        } else {
            clasificacionIMC = "Obesidad";
        }
        
        // Mostrar resumen formateado
        System.out.println("\n=================================");
        System.out.println("   RESUMEN DEL REGISTRO");
        System.out.println("=================================");
        System.out.printf("  %-15s : %s%n", "Nombre", nombre);
        System.out.printf("  %-15s : %s%n", "Email", email);
        System.out.printf("  %-15s : %d años%n", "Edad", edad);
        System.out.printf("  %-15s : %.2f m%n", "Altura", altura);
        System.out.printf("  %-15s : %.1f kg%n", "Peso", peso);
        System.out.printf("  %-15s : %s%n", "Estudiante", estudiante ? "Sí" : "No");
        System.out.printf("  %-15s : %.2f%n", "IMC", imc);
        System.out.printf("  %-15s : %s%n", "Clasificación", clasificacionIMC);
        System.out.println("=================================");
        
        System.out.println("\n¡Registro completado con éxito!");
        sc.close();
    }
}
```

RESUMEN DEL CAPÍTULO
===============================================================================

- System.in es la entrada estándar, System.out y System.err son salidas.
- Scanner es la clase más fácil para leer datos del teclado (java.util.Scanner).
- nextLine() lee una línea; nextInt(), nextDouble() leen tipos específicos.
- Después de nextInt()/nextDouble() debes llamar a nextLine() para limpiar el buffer.
- Los métodos hasNext*() verifican si hay datos del tipo esperado disponibles.
- Scanner puede leer de Strings, archivos y otras fuentes cambiando el origen.
- useDelimiter() permite cambiar el separador de tokens.
- printf() permite formatear la salida con especificadores como %s, %d, %f.
- La validación de entrada es crucial para hacer programas robustos.
- Siempre cierra el Scanner con close() para liberar recursos.

===============================================================================
PREGUNTAS FRECUENTES
===============================================================================

❓ **¿Qué pasa si no cierro el Scanner?**
Si no cierras el Scanner, los recursos del sistema no se liberan. Aunque
en programas pequeños no causa problemas, en aplicaciones grandes puede
causar fugas de recursos. Siempre usa close().

❓ **¿Puedo tener varios Scanners abiertos al mismo tiempo?**
Técnicamente sí, pero no es recomendable tener más de un Scanner sobre
System.in. Pueden interferir entre sí. Crea uno solo y úsalo en todo el
programa.

❓ **¿Por qué Scanner no me deja leer después de nextInt()?**
Es el problema clásico: nextInt() no consume el salto de línea. Después
de nextInt(), llama a scanner.nextLine() para limpiar el buffer.

❓ **¿Cómo leo una contraseña sin que se vea en pantalla?**
Scanner no soporta lectura oculta. Puedes usar Console.readPassword()
(System.console().readPassword()), pero no funciona en todos los IDEs.

❓ **¿Scanner funciona con caracteres acentuados y ñ?**
Sí, Scanner maneja Unicode correctamente. Lee y muestra caracteres con
tildes y ñ sin problemas.
