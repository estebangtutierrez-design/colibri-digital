1. ¿QUÉ ES UN STRING?
===============================================================================

Un String en Java es una secuencia de caracteres (char). A diferencia de
los tipos primitivos, String es una CLASE (una de las más importantes de
Java). Esto significa que los Strings tienen métodos que puedes usar para
manipularlos.

**String Pool:** Java optimiza la memoria con un "pool" de Strings. Cuando
creas un String con comillas dobles, Java verifica si ya existe en el pool.
Si existe, reutiliza la referencia. Esto ahorra memoria.

**Inmutabilidad:** Los Strings son inmutables. Una vez creado un String,
su valor no puede cambiar. Cualquier "modificación" crea un nuevo String
en memoria.

```java
// Creación de Strings

class CreacionStrings {
    public static void main(String[] args) {
        System.out.println("=== CREACIÓN DE STRINGS ===");
        
        // Forma 1: Literal (recomendada)
        String saludo1 = "Hola";
        String saludo2 = "Hola";
        
        // Forma 2: Con new (NO recomendada)
        String saludo3 = new String("Hola");
        
        // Forma 3: A partir de un array de caracteres
        char[] caracteres = {'J', 'a', 'v', 'a'};
        String desdeArray = new String(caracteres);
        System.out.println("Desde array: " + desdeArray);
        
        // Forma 4: Usando el operador +
        String combinado = "Hola" + " " + "Mundo";
        System.out.println("Combinado: " + combinado);
        
        // Demostración del String Pool
        System.out.println("\n=== STRING POOL ===");
        System.out.println("saludo1 == saludo2: " + (saludo1 == saludo2));  // true
        System.out.println("saludo1 == saludo3: " + (saludo1 == saludo3));  // false
        
        // saludo1 y saludo2 apuntan al mismo objeto en el pool
        // saludo3 es un objeto diferente (forzado con new)
        
        System.out.println("\n=== INMUTABILIDAD ===");
        String texto = "Java";
        System.out.println("Original: " + texto);
        System.out.println("HashCode original: " + System.identityHashCode(texto));
        
        texto = texto + " es genial";
        System.out.println("Modificado: " + texto);
        System.out.println("HashCode modificado: " + System.identityHashCode(texto));
        
        System.out.println("¡Son objetos diferentes! El String original 'Java'");
        System.out.println("sigue en memoria, no se modificó.");
    }
}
```

**¿Por qué los Strings son inmutables?**

1. **Seguridad:** Las clases del sistema confían en que los Strings no cambian.
2. **Sincronización:** Son thread-safe (seguros para hilos).
3. **String Pool:** Pueden reutilizarse sin riesgo de modificaciones.
4. **HashCode:** Se puede cachear el hashCode porque nunca cambia.
5. **Rendimiento:** La inmutabilidad permite optimizaciones en la JVM.

```java
// Métodos principales de String - Parte 1

class MetodosString1 {
    public static void main(String[] args) {
        String texto = "  Programación en Java  ";
        
        System.out.println("=== MÉTODOS DE STRING (PARTE 1) ===");
        System.out.println("Texto original: '" + texto + "'");

        // length(): longitud de la cadena
        System.out.println("length(): " + texto.length());

        // trim(): elimina espacios al inicio y final
        String sinEspacios = texto.trim();
        System.out.println("trim(): '" + sinEspacios + "'");
        
        // toUpperCase() / toLowerCase(): cambiar mayúsculas/minúsculas
        System.out.println("toUpperCase(): " + texto.toUpperCase());
        System.out.println("toLowerCase(): " + texto.toLowerCase());
        
        // isEmpty() / isBlank(): verificar si está vacío o en blanco
        System.out.println("isEmpty(): " + texto.isEmpty());
        System.out.println("isBlank(): " + texto.isBlank());  // Java 11+
        
        String vacio = "";
        System.out.println("'': isEmpty=" + vacio.isEmpty() + ", isBlank=" + vacio.isBlank());
        
        String espacios = "   ";
        System.out.println("'   ': isEmpty=" + espacios.isEmpty() + ", isBlank=" + espacios.isBlank());
        
        // substring(inicio, fin): extraer subcadena
        // (inicio es inclusivo, fin es exclusivo)
        String java = "Programación en Java";
        System.out.println("\nsubstring(0, 11): '" + java.substring(0, 11) + "'");
        System.out.println("substring(15): '" + java.substring(15) + "'");
        System.out.println("substring(0, 1): '" + java.substring(0, 1) + "'");
        
        // replace(buscar, reemplazar)
        System.out.println("\nreplace('a', 'o'): " + texto.replace('a', 'o'));
        System.out.println("replace(\"Java\", \"Python\"): " + texto.replace("Java", "Python"));
        
        // repeat(n): repetir la cadena n veces (Java 11+)
        String risa = "Ja";
        System.out.println("repeat(3): " + risa.repeat(3));
    }
}
```

4. CONCATENACIÓN DE STRINGS
===============================================================================

La concatenación es una de las operaciones más comunes con Strings. Java
ofrece varias formas, cada una con sus características de rendimiento.

```java
// Formas de concatenar Strings

class Concatenacion {
    public static void main(String[] args) {
        System.out.println("=== CONCATENACIÓN DE STRINGS ===");
        
        // Forma 1: Operador +
        String saludo = "Hola, " + "mundo!";
        System.out.println("Con +: " + saludo);
        
        // Forma 2: Método concat()
        String nombre = "Ana".concat(" María").concat(" López");
        System.out.println("Con concat(): " + nombre);
        
        // Forma 3: StringBuilder (eficiente para muchas concatenaciones)
        StringBuilder sb = new StringBuilder();
        sb.append("Java");
        sb.append(" es ");
        sb.append("genial");
        sb.append("!");
        System.out.println("Con StringBuilder: " + sb.toString());
        
        // Forma 4: String.join()
        String unido = String.join(" ", "Java", "es", "poderoso");
        System.out.println("Con join(): " + unido);
        
        // Forma 5: String.format()
        String formateado = String.format("%s %s %s", "Aprender", "Java", "es divertido");
        System.out.println("Con format(): " + formateado);
        
        // Rendimiento: StringBuilder vs +
        System.out.println("\n=== RENDIMIENTO (concepto) ===");
        System.out.println("Para pocas concatenaciones: + es suficiente");
        System.out.println("Para muchas (bucles): StringBuilder es mucho más rápido");
        System.out.println("Cada + crea un nuevo String (inmutable)");
        
        // Demostración de inmutabilidad
        String s = "Hola";
        String sMayusculas = s.toUpperCase();
        System.out.println("\nString original: " + s);  // Sigue siendo "Hola"
        System.out.println("String modificado: " + sMayusculas);  // "HOLA"
        System.out.println("El original no se modificó (inmutable)");
    }
}
```

**¿Por qué StringBuilder es más rápido?**

Cada vez que concatenas con +, Java crea un nuevo objeto String (porque
los Strings son inmutables). Si concatenas 1000 veces en un bucle, creas
999 objetos temporales que el Garbage Collector debe limpiar.

StringBuilder mantiene un array interno que puede crecer según necesites,
sin crear objetos intermedios.

```java
// Comparación de rendimiento: + vs StringBuilder

class RendimientoConcatenacion {
    public static void main(String[] args) {
        System.out.println("=== COMPARACIÓN DE RENDIMIENTO ===");
        System.out.println("(Ejemplo conceptual - no ejecutes con valores grandes)");
        
        int cantidad = 10000;
        
        // Con + (lento)
        long inicio1 = System.currentTimeMillis();
        String resultado1 = "";
        for (int i = 0; i < cantidad; i++) {
            resultado1 = resultado1 + "a";
        }
        long fin1 = System.currentTimeMillis();
        System.out.println("\nCon +: " + (fin1 - inicio1) + " ms (longitud: " + resultado1.length() + ")");
        
        // Con StringBuilder (rápido)
        long inicio2 = System.currentTimeMillis();
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < cantidad; i++) {
            sb.append("a");
        }
        String resultado2 = sb.toString();
        long fin2 = System.currentTimeMillis();
        System.out.println("Con StringBuilder: " + (fin2 - inicio2) + " ms (longitud: " + resultado2.length() + ")");
        
        System.out.println("\n¡StringBuilder es drásticamente más rápido!");
        System.out.println("Usa '+' para pocas concatenaciones.");
        System.out.println("Usa StringBuilder para bucles o muchas concatenaciones.");
    }
}
```

5. STRINGBUILDER Y STRINGBUFFER
===============================================================================

StringBuilder y StringBuffer son clases que representan cadenas mutables.
La diferencia principal es que StringBuffer es thread-safe (sincronizado)
y StringBuilder no. En la mayoría de los casos, StringBuilder es suficiente.

```java
// StringBuilder en detalle

class StringBuilderDetalle {
    public static void main(String[] args) {
        System.out.println("=== STRINGBUILDER EN DETALLE ===");
        
        // Creación
        StringBuilder sb = new StringBuilder();          // capacidad por defecto 16
        StringBuilder sb2 = new StringBuilder("Java");   // con contenido inicial
        StringBuilder sb3 = new StringBuilder(100);      // capacidad inicial 100
        
        // append(): añadir al final
        sb.append("Hola ");
        sb.append("Mundo");
        sb.append(2024);
        sb.append(true);
        sb.append(3.14);
        System.out.println("append: " + sb.toString());
        
        // insert(): insertar en una posición
        sb2.insert(4, " es genial");  // "Java" + " es genial"
        System.out.println("insert: " + sb2.toString());
        
        // delete(): eliminar un rango
        StringBuilder sbBorrar = new StringBuilder("Programación en Java");
        sbBorrar.delete(0, 13);  // Elimina "Programación "
        System.out.println("delete: " + sbBorrar.toString());
        
        // deleteCharAt(): eliminar un carácter
        StringBuilder sbBorrarChar = new StringBuilder("Hola");
        sbBorrarChar.deleteCharAt(2);  // Elimina 'l'
        System.out.println("deleteCharAt: " + sbBorrarChar.toString());
        
        // replace(): reemplazar un rango
        StringBuilder sbReemplazar = new StringBuilder("Me gusta Python");
        sbReemplazar.replace(9, 15, "Java");
        System.out.println("replace: " + sbReemplazar.toString());
        
        // reverse(): invertir la cadena
        StringBuilder sbReverse = new StringBuilder("Java");
        System.out.println("reverse: " + sbReverse.reverse().toString());
        
        // capacity(): capacidad actual
        System.out.println("\ncapacity: " + sb.capacity());
        
        // ensureCapacity(): asegurar capacidad mínima
        sb.ensureCapacity(200);
        System.out.println("ensureCapacity(200): " + sb.capacity());
        
        // length(): longitud actual
        System.out.println("length: " + sb.length());
        
        // setLength(): establecer longitud
        StringBuilder sbLongitud = new StringBuilder("Hola Mundo");
        sbLongitud.setLength(4);
        System.out.println("setLength(4): '" + sbLongitud.toString() + "'");
        
        // charAt() y setCharAt()
        StringBuilder sbChar = new StringBuilder("Java");
        sbChar.setCharAt(0, 'D');  // "Java" -> "Dava"
        System.out.println("setCharAt(0, 'D'): " + sbChar.toString());
        System.out.println("charAt(1): " + sbChar.charAt(1));
        
        // substring(): extraer subcadena (no modifica el StringBuilder)
        StringBuilder sbSub = new StringBuilder("Aprendiendo Java");
        String sub = sbSub.substring(0, 11);
        System.out.println("substring(0,11): " + sub);
        
        // toString(): convertir a String
        String resultado = sb.toString();
        System.out.println("toString: " + resultado);
    }
}
```

```java
// StringBuilder: patrón de uso común

class StringBuilderUso {
    public static void main(String[] args) {
        System.out.println("=== PATRÓN COMÚN CON STRINGBUILDER ===");
        
        // Construir un mensaje complejo
        StringBuilder mensaje = new StringBuilder();
        
        mensaje.append("=== INFORMACIÓN DEL USUARIO ===\n");
        mensaje.append("Nombre: ").append("Ana López").append('\n');
        mensaje.append("Edad: ").append(25).append('\n');
        mensaje.append("Ciudad: ").append("Madrid").append('\n');
        mensaje.append("Profesión: ").append("Ingeniera de Software").append('\n');
        mensaje.append("Activo: ").append(true).append('\n');
        mensaje.append("===================================");
        
        System.out.println(mensaje.toString());
        
        // Construir un CSV
        StringBuilder csv = new StringBuilder();
        String[] nombres = {"Ana", "Luis", "Carlos", "María"};
        for (int i = 0; i < nombres.length; i++) {
            if (i > 0) {
                csv.append(",");
            }
            csv.append(nombres[i]);
        }
        System.out.println("\nCSV: " + csv.toString());
        
        // Construir una consulta SQL
        StringBuilder sql = new StringBuilder();
        sql.append("SELECT * FROM usuarios");
        sql.append(" WHERE activo = true");
        sql.append(" ORDER BY nombre ASC");
        sql.append(" LIMIT 10");
        System.out.println("\nSQL: " + sql.toString());
    }
}
```

**¿StringBuilder o StringBuffer?**

| Característica | StringBuilder | StringBuffer |
|----------------|--------------|--------------|
| Sincronizado | No | Sí |
| Seguro para hilos | No | Sí |
| Rendimiento | Más rápido | Más lento |
| Uso recomendado | Un solo hilo | Múltiples hilos |

En la práctica, usa StringBuilder a menos que estés trabajando con hilos
y necesites sincronización.

```java
// Formateo avanzado de Strings

class FormateoStrings {
    public static void main(String[] args) {
        System.out.println("=== FORMATEO DE STRINGS ===");
        
        // String.format() y printf()
        String nombre = "Ana";
        int edad = 25;
        double altura = 1.75;
        
        // %s = String, %d = entero, %f = decimal, %n = salto de línea
        String mensaje = String.format("Nombre: %s, Edad: %d, Altura: %.2f", nombre, edad, altura);
        System.out.println("String.format: " + mensaje);
        
        // printf imprime directamente
        System.out.printf("printf: Nombre: %s, Edad: %d, Altura: %.2f%n", nombre, edad, altura);
        
        // Especificadores de formato comunes
        System.out.println("\n=== ESPECIFICADORES DE FORMATO ===");
        System.out.printf("%%s:  '%s'%n", "texto");
        System.out.printf("%%S:  '%S'%n", "texto");  // Mayúsculas
        System.out.printf("%%d:  '%d'%n", 12345);
        System.out.printf("%%08d: '%08d'%n", 12345);  // Relleno con ceros
        System.out.printf("%%f:  '%f'%n", 3.14159);
        System.out.printf("%%.2f: '%.2f'%n", 3.14159);  // 2 decimales
        System.out.printf("%%10s: '%10s'%n", "hola");    // Alineado derecha
        System.out.printf("%%-10s: '%-10s'%n", "hola");  // Alineado izquierda
        System.out.printf("%%x:  '%x'%n", 255);          // Hexadecimal
        System.out.printf("%%o:  '%o'%n", 255);          // Octal
        System.out.printf("%%tF: '%tF'%n", new java.util.Date());  // Fecha ISO
        System.out.printf("%%tT: '%tT'%n", new java.util.Date());  // Hora
        
        // Tabla formateada
        System.out.println("\n=== TABLA FORMATEADA ===");
        System.out.printf("%-15s %-5s %-10s %-10s%n", "Nombre", "Edad", "Ciudad", "Salario");
        System.out.println("------------------------------------------------");
        System.out.printf("%-15s %-5d %-10s %-10.2f%n", "Ana López", 25, "Madrid", 3500.50);
        System.out.printf("%-15s %-5d %-10s %-10.2f%n", "Luis Pérez", 30, "Barcelona", 4200.75);
        System.out.printf("%-15s %-5d %-10s %-10.2f%n", "María Ruiz", 28, "Valencia", 2800.00);
        
        // formatted() (Java 15+)
        String plantilla = """
                Usuario: %s
                Edad:    %d
                Email:  %s
                """.formatted("Carlos", 35, "carlos@email.com");
        System.out.println("\n=== CON TEXT BLOCKS ===");
        System.out.println(plantilla);
    }
}
```

===============================================================================
RESUMEN DEL CAPÍTULO
===============================================================================

- String es una clase (no primitivo) y los Strings son inmutables.
- El String Pool optimiza memoria reutilizando literales iguales.
- Usa .equals() para comparar contenido de Strings (no ==).
- Los principales métodos son: length, charAt, substring, indexOf, replace, split, join.
- split() divide una cadena en array; join() une elementos con separador.
- StringBuilder es la opción eficiente para cadenas mutables y muchas concatenaciones.
- StringBuffer es como StringBuilder pero thread-safe.
- Los Text Blocks (Java 13+) permiten cadenas multilínea sin escapes.
- String.format() y printf() formatean cadenas con especificadores como %s, %d, %f.
- Para poca concatenación usa +; para muchas usa StringBuilder.

===============================================================================
PREGUNTAS FRECUENTES
===============================================================================

❓ **¿Por qué no debo usar new String("texto")?**
Creas un objeto innecesario. El literal "texto" ya crea un String en el
pool. Con new String("texto") creas otro objeto fuera del pool, duplicando
memoria.

❓ **¿String es un tipo primitivo o una clase?**
Es una clase. Pero Java le da un trato especial (operador +, literales con
comillas dobles). Es la única clase con literales propios en Java.

❓ **¿Cuántos objetos se crean con: String s = "Hola" + " " + "Mundo"?**
Solo uno. Java optimiza la concatenación de literales en tiempo de
compilación. El compilador genera directamente "Hola Mundo".

❓ **¿Cuándo debo usar StringBuilder en lugar de String?**
Cuando hagas muchas operaciones de concatenación/modificación en un bucle
o cuando construyas cadenas largas paso a paso. Para la mayoría de los
casos simples, String con + es suficiente.

❓ **¿StringBuilder no tiene métodos como toUpperCase o substring?**
No directamente en StringBuilder. Puedes convertir a String con toString(),
aplicar los métodos, y volver a StringBuilder si es necesario. Pero si
necesitas muchas operaciones así, considera usar String directamente.

🏗️ **Analizador de Texto Completo**

Diseña un programa Java que analice un texto (puede ser un párrafo
predefinido o ingresado por el usuario) y muestre un informe completo:

1. **Estadísticas básicas:**
   - Número total de caracteres (con y sin espacios).
   - Número total de palabras.
   - Número total de oraciones.
   - Número de líneas.

2. **Análisis de palabras:**
   - Palabra más larga.
   - Palabra más corta.
   - Frecuencia de cada palabra (usando split y un mapa conceptual).
   - Porcentaje de palabras que comienzan con vocal.

3. **Análisis de caracteres:**
   - Cantidad de vocales y consonantes.
   - Cantidad de dígitos, signos de puntuación y espacios.
   - Carácter más repetido (sin contar espacios).

4. **Transformaciones:**
   - Texto en mayúsculas.
   - Texto en minúsculas.
   - Cada palabra capitalizada (primera letra mayúscula).
   - Texto sin espacios duplicados.

5. **Generación de reporte:**
   - Usa StringBuilder para construir el reporte completo.
   - Usa String.format() para tabular los datos.
   - Usa Text Blocks si tu versión de Java lo soporta.

Ejemplo de salida:
```
========================================
     ANALIZADOR DE TEXTO
========================================

Texto: "Java es un lenguaje de programación
orientado a objetos y multiplataforma."

ESTADÍSTICAS BÁSICAS:
  Caracteres totales:    60
  Caracteres (sin espacios): 48
  Palabras:              9
  Oraciones:             1
  Líneas:                2

ANÁLISIS DE PALABRAS:
  Palabra más larga:     "programación" (12 letras)
  Palabra más corta:     "y" (1 letra)
  Palabras con vocal:    55.6%

ANÁLISIS DE CARACTERES:
  Vocales:               25
  Consonantes:           23
  Dígitos:              0
  Puntuación:           1
  Carácter más repetido: 'a' (6 veces)

TRANSFORMACIONES:
  MAYÚSCULAS: JAVA ES UN LENGUAJE...
  minúsculas: java es un lenguaje...
  Capitalizado: Java Es Un Lenguaje...
========================================
```

Este desafío integra todos los conceptos de Strings: métodos de búsqueda,
extracción, transformación, StringBuilder, formatos y análisis de texto.
