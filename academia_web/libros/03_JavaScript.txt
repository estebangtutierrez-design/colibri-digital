CAPITULO 1: INTRODUCCIÓN A JAVASCRIPT
================================================================================

¿Qué es JavaScript?
-------------------
JavaScript (JS) es un lenguaje de programación interpretado, multiparadigma
(orientado a objetos, imperativo y funcional). Es el lenguaje nativo de los
navegadores web, permitiendo interactividad y dinamismo en las páginas.
También se usa en el servidor con Node.js, en móviles con React Native, y en
escritorio con Electron.

Historia: Creado por Brendan Eich en 1995 en solo 10 días para Netscape.
Hoy es el lenguaje más popular del mundo.

Versiones: ES5 (2009), ES6 (2015) y versiones anuales posteriores.

CAPITULO 2: CONFIGURACIÓN DEL ENTORNO
================================================================================

Para probar JS tienes dos opciones:

1. Navegador (Frontend):
   - Abre la consola (F12 > Console) y escribe código directamente.
   - O crea un archivo .html con una etiqueta <script>.

2. Node.js (Backend):

   - Crea un archivo .js y ejecuta: node archivo.js

Tu primer código (en consola del navegador o Node.js):
console.log("¡Hola, Mundo!");

CAPITULO 3: SINTAXIS BÁSICA
================================================================================

Comentarios:
------------
// Comentario de una línea
/* Comentario de varias líneas */

Variables (var, let, const):
----------------------------
var nombre = "Antiguo"; // No recomendado (ámbito de función)
let edad = 30;          // Recomendado (ámbito de bloque)
const PI = 3.1416;      // Constante (no se puede reasignar)

Tipos de datos primitivos:
---------------------------
string: "Hola", 'Mundo', `Template`
number: 10, 3.14, NaN, Infinity
boolean: true, false
null: valor nulo (intencional)
undefined: valor no definido
symbol: único e inmutable
bigint: números enteros muy grandes (100n)

Operadores:
-----------
Aritméticos: + - * / % ** (potencia)
Asignación: = += -= *= /=
Comparación: == (solo valor), === (valor y tipo), !=, !==, >, <, >=, <=
Lógicos: && (AND), || (OR), ! (NOT)

Template Strings (literales de plantilla):
------------------------------------------
let nombre = "Ana";
console.log(`Hola ${nombre}, tienes ${30 + 5} años`);

CAPITULO 4: ESTRUCTURAS DE CONTROL
================================================================================

Condicional if/else:
--------------------
if (edad >= 18) {
    console.log("Mayor de edad");
} else {
    console.log("Menor de edad");
}

Operador ternario:
------------------
let mensaje = edad >= 18 ? "Mayor" : "Menor";

Switch:
-------
switch (dia) {
    case 1: console.log("Lunes"); break;
    case 2: console.log("Martes"); break;
    default: console.log("Otro día");
}

Bucles:
-------
for (let i = 0; i < 10; i++) {
    console.log(i);
}

let i = 0;
while (i < 10) {
    console.log(i);
    i++;
}

do {
    console.log(i);
    i++;
} while (i < 10);

for...of (itera sobre valores):
-------------------------------
for (let valor of [1,2,3]) { console.log(valor); }

for...in (itera sobre propiedades de objetos):
-----------------------------------------------
for (let clave in objeto) { console.log(clave, objeto[clave]); }

CAPITULO 5: FUNCIONES
================================================================================

Declaración de función:
-----------------------
function sumar(a, b) {
    return a + b;
}

Expresión de función (anónima):
-------------------------------
const restar = function(a, b) {
    return a - b;
};

Arrow functions (funciones flecha) - Moderno:
---------------------------------------------
const multiplicar = (a, b) => a * b;
const saludar = nombre => `Hola ${nombre}`;

Parámetros por defecto:
-----------------------
function saludar(nombre = "Invitado") {
    console.log(`Hola ${nombre}`);
}

Parámetros Rest (resto de argumentos):
--------------------------------------
function sumarTodos(...numeros) {
    return numeros.reduce((acc, n) => acc + n, 0);
}

CAPITULO 6: ARRAYS Y OBJETOS
================================================================================

Arrays (arreglos):
------------------
let frutas = ["manzana", "plátano", "naranja"];
frutas.push("pera");     // Agregar al final
frutas.pop();            // Eliminar último
frutas.unshift("kiwi");  // Agregar al inicio
frutas.shift();          // Eliminar primero
frutas.length;           // Tamaño
frutas[0];               // Acceso por índice

Métodos de array (modernos):
----------------------------
frutas.map(fruta => fruta.toUpperCase());
frutas.filter(fruta => fruta.length > 5);
frutas.find(fruta => fruta === "manzana");
frutas.includes("manzana");
frutas.forEach(fruta => console.log(fruta));

Objetos (pares clave-valor):
----------------------------
let persona = {
    nombre: "Esteban",
    edad: 30,
    saludar() {
        console.log(`Hola, soy ${this.nombre}`);
    }
};
persona.saludar();
persona.ciudad = "Houston"; // agregar propiedad
delete persona.edad;        // eliminar propiedad

Desestructuración:
------------------
const { nombre, edad } = persona;
console.log(nombre, edad);

Spread operator (...):
----------------------
let numeros = [1,2,3];
let copia = [...numeros, 4, 5]; // [1,2,3,4,5]
let persona2 = { ...persona, edad: 31 };

CAPITULO 7: MANIPULACIÓN DEL DOM
================================================================================

El DOM (Document Object Model) permite interactuar con la página HTML.

Seleccionar elementos:
----------------------
document.getElementById("miId");
document.querySelector(".miClase");
document.querySelectorAll("p");
document.getElementsByClassName("clase");

Modificar contenido y estilo:
-----------------------------
elemento.textContent = "Nuevo texto";
elemento.innerHTML = "<strong>HTML</strong>";
elemento.style.backgroundColor = "blue";
elemento.classList.add("activo");
elemento.classList.remove("oculto");

Crear y agregar elementos:
---------------------------
let nuevoDiv = document.createElement("div");
nuevoDiv.textContent = "Hola";
document.body.appendChild(nuevoDiv);

CAPITULO 8: EVENTOS EN EL NAVEGADOR
================================================================================

Escuchar eventos:
-----------------
boton.addEventListener("click", function() {
    alert("¡Clic!");
});

Evento con arrow:
-----------------
boton.addEventListener("mouseover", () => {
    console.log("Mouse encima");
});

Eventos comunes: click, dblclick, mouseover, mouseout, keydown, keyup, submit, load, DOMContentLoaded.

Prevenir comportamiento por defecto:
------------------------------------
formulario.addEventListener("submit", (e) => {
    e.preventDefault();
    console.log("Formulario no se envió");
});

Delegación de eventos:
----------------------
document.querySelector("ul").addEventListener("click", (e) => {
    if (e.target.tagName === "LI") {
        console.log("Clic en item:", e.target.textContent);
    }
});

CAPITULO 9: PROGRAMACIÓN ORIENTADA A OBJETOS
================================================================================

Prototipos (forma antigua):
---------------------------
function Persona(nombre, edad) {
    this.nombre = nombre;
    this.edad = edad;
}
Persona.prototype.saludar = function() {
    console.log(`Hola ${this.nombre}`);
};
let p = new Persona("Juan", 25);
p.saludar();

Clases (ES6 - Moderno):
-----------------------
class Persona {
    constructor(nombre, edad) {
        this.nombre = nombre;
        this.edad = edad;
    }
    saludar() {
        console.log(`Hola ${this.nombre}`);
    }
    static metodoEstatico() {
        console.log("Estático");
    }
    get getEdad() { return this.edad; }
    set setEdad(valor) { this.edad = valor; }
}

Herencia:
---------
class Empleado extends Persona {
    constructor(nombre, edad, salario) {
        super(nombre, edad);
        this.salario = salario;
    }
    saludar() {
        super.saludar();
        console.log(`Mi salario es ${this.salario}`);
    }
}

CAPITULO 10: MÓDULOS (ES6)
================================================================================

Exportar desde un archivo (miModulo.js):
----------------------------------------
export const PI = 3.1416;
export function sumar(a, b) { return a + b; }
export default class Calculadora { ... }

Importar en otro archivo:
-------------------------
import { PI, sumar } from './miModulo.js';
import Calculadora from './miModulo.js';

En el navegador usar <script type="module">.
En Node.js configurar "type": "module" en package.json.

CAPITULO 11: MANEJO DE ERRORES
================================================================================

try...catch...finally:
----------------------
try {
    let resultado = 10 / x; // x no definida
} catch (error) {
    console.log("Error:", error.message);
} finally {
    console.log("Siempre se ejecuta");
}

Lanzar errores:
---------------
function validarEdad(edad) {
    if (edad < 0) throw new Error("Edad negativa no permitida");
    return edad;
}

Errores personalizados:
-----------------------
class MiError extends Error {
    constructor(mensaje) {
        super(mensaje);
        this.name = "MiError";
    }
}

CAPITULO 12: PROMESAS Y ASYNC/AWAIT
================================================================================

Promesas (manejo asíncrono):
----------------------------
const miPromesa = new Promise((resolve, reject) => {
    setTimeout(() => {
        if (true) resolve("Exitoso");
        else reject("Fallo");
    }, 1000);
});

miPromesa
    .then(resultado => console.log(resultado))
    .catch(error => console.error(error))
    .finally(() => console.log("Terminado"));

Async/Await (sintaxis más limpia):
----------------------------------
async function obtenerDatos() {
    try {
        const resultado = await miPromesa;
        console.log(resultado);
    } catch (error) {
        console.error(error);
    }
}

Promise.all (ejecutar varias promesas en paralelo):
---------------------------------------------------
const [res1, res2] = await Promise.all([promesa1(), promesa2()]);

CAPITULO 13: FETCH API Y PETICIONES HTTP
================================================================================

Petición GET:
-------------
fetch('https://api.ejemplo.com/data')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error(error));

Con async/await:
----------------
async function getData() {
    try {
        const res = await fetch('https://api.ejemplo.com/data');
        const data = await res.json();
        console.log(data);
    } catch (error) {
        console.error(error);
    }
}

Petición POST:
--------------
fetch('https://api.ejemplo.com/data', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ nombre: "Esteban", edad: 30 })
})
.then(res => res.json())
.then(data => console.log(data));

CAPITULO 14: BUENAS PRÁCTICAS Y PATRONES
================================================================================

1. Usa const por defecto, let solo si vas a reasignar.
2. Evita var.
3. Usa === y !== en lugar de == y !=.
4. Nombra funciones y variables con camelCase.
5. Clases con PascalCase.
6. Comenta código complejo, no lo obvio.
7. Usa desestructuración y spread.
8. Evita mutar objetos y arrays directamente (prefiere copias).
9. Usa async/await para código asíncrono.
10. Maneja errores siempre (try/catch).

Patrón Módulo (Module Pattern):
-------------------------------
const modulo = (function() {
    let privado = 0;
    return {
        incrementar() { privado++; },
        obtener() { return privado; }
    };
})();
