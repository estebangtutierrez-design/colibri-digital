===============================================================================
TEORÍA
===============================================================================

TIPOS DE UNIDADES PARA COLUMNAS:

px → tamaño fijo.
% → porcentaje del contenedor.
fr → fracción del espacio restante.
min-content → el mínimo necesario para el contenido.
max-content → el máximo sin forzar saltos de línea.
auto → el navegador decide (similar a 1fr pero no se estira).

LÍNEAS NOMBRADAS:

Puedes nombrar las líneas del grid usando corchetes:

grid-template-columns: [inicio] 1fr [medio] 1fr [final];

Luego posicionas items con esos nombres:

.item { grid-column: inicio / medio; }

COLUMNAS RESPONSIVAS:

repeat(auto-fill, minmax(200px, 1fr))
→ Crea tantas columnas de 200px como quepan.
→ En pantalla grande: 6 columnas.
→ En pantalla mediana: 3 columnas.
→ En pantalla chica: 1 columna.

COLUMNAS IMPLÍCITAS:

Si un item se posiciona más allá de las columnas definidas,
se crean columnas implícitas (grid-auto-columns).

===============================================================================
ANALOGÍA
===============================================================================

Las columnas son los carriles de una autopista.

- px: carril de ancho fijo (como los de un puente).
- fr: carril que se estira según el tráfico.
- min-content: carril angosto, solo lo que necesita el vehículo.
- max-content: carril ancho para vehículos grandes.
- minmax(): carril con ancho mínimo y máximo.
- Líneas nombradas: nombres en los kilómetros (KM 10, KM 20).
- auto-fill: la autopista crea carriles según el espacio disponible.

===============================================================================
SINTAXIS
===============================================================================

/* Unidades mixtas */
.grid {
    grid-template-columns: 200px 1fr 1fr;
    /* primera columna fija, las otras se reparten el resto */
}

/* Líneas nombradas */
.grid {
    grid-template-columns: [sidebar] 250px [main] 1fr [final];
}
.item-sidebar { grid-column: sidebar / main; }
.item-main    { grid-column: main / final; }

/* min-content y max-content */
.grid {
    grid-template-columns: min-content 1fr max-content;
    /* 1ra: solo lo necesario. 2da: lo que sobre. 3ra: todo su contenido */
}

/* Columnas responsivas */
.grid {
    grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
}

=======================================================================
AUTO-FILL VS AUTO-FIT
=======================================================================

auto-fill:
   repeat(auto-fill, minmax(200px, 1fr))
   → Crea columnas incluso si están vacías.
   → Mantiene el espacio de las columnas vacías.

auto-fit:
   repeat(auto-fit, minmax(200px, 1fr))
   → Colapsa las columnas vacías a 0.
   → Los items existentes ocupan más espacio.

Usa auto-fill cuando quieras mantener una estructura fija.
Usa auto-fit cuando quieras que los items se expandan.

=======================================================================
GLOSARIO
=======================================================================

min-content: Tamaño mínimo necesario para el contenido.
max-content: Tamaño completo sin forzar saltos.
líneas nombradas: Nombres entre corchetes en grid-template-columns.
auto-fill: Crea columnas para llenar el espacio disponible.
auto-fit: Similar pero colapsa columnas vacías.
minmax(): Define tamaño mínimo y máximo.
grid-auto-columns: Tamaño de columnas implícitas.
