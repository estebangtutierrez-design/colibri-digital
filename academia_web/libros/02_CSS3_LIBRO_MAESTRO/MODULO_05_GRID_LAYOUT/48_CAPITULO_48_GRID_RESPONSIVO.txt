"Un grid responsivo se adapta al tamaño de la pantalla sin
romperse. Con auto-fill, auto-fit, minmax y media queries,
tu grid se ve bien en cualquier dispositivo."
                         — Profesor Búho AI

===============================================================================
BIENVENIDA
===============================================================================

Un grid fijo se ve bien en una pantalla y mal en todas las demás.
Un grid responsivo se adapta.

CSS Grid tiene herramientas integradas para crear layouts que
cambian automáticamente según el espacio disponible: auto-fill,
auto-fit, minmax() y, cuando es necesario, media queries.

===============================================================================
OBJETIVOS
===============================================================================

✓ Crear grids responsivos sin media queries.
✓ Usar auto-fill y auto-fit con minmax.
✓ Cambiar el número de columnas según la pantalla.
✓ Reordenar áreas con media queries.
✓ Combinar grid responsivo con flexbox.

===============================================================================
TEORÍA
===============================================================================

TRES TÉCNICAS PARA GRID RESPONSIVO:

1. auto-fill / auto-fit + minmax (sin media queries).
   repeat(auto-fill, minmax(250px, 1fr))
   → Crea tantas columnas de 250px como quepan.
   → En móvil: 1 columna. En tablet: 2. En desktop: 4.

2. Media queries en grid-template-columns.
   @media (max-width: 768px) {
       .grid { grid-template-columns: 1fr; }
   }
   @media (min-width: 769px) {
       .grid { grid-template-columns: repeat(2, 1fr); }
   }

3. Media queries en grid-template-areas.
   @media (max-width: 768px) {
       .layout {
           grid-template-areas:
               "header"
               "nav"
               "main"
               "sidebar"
               "footer";
       }
   }

===============================================================================
ANALOGÍA
===============================================================================

Un estante modular:
- auto-fill: pones estantes de 30cm. Si la pared mide 100cm,
  entran 3. Si mide 200cm, entran 6.
- minmax(30cm, 1fr): los estantes miden mínimo 30cm, pero si
  sobra espacio, se estiran para ocuparlo.
- Media query: cambias el diseño cuando pasas de una pared
  pequeña a una grande.

===============================================================================
SINTAXIS
===============================================================================

/* Sin media queries (auto-adaptable) */
.grid-auto {
    display: grid;
    grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
    gap: 16px;
}

/* Con media queries (control manual) */
.grid {
    display: grid;
    grid-template-columns: repeat(4, 1fr);
    gap: 16px;
}

@media (max-width: 900px) {
    .grid { grid-template-columns: repeat(2, 1fr); }
}

@media (max-width: 500px) {
    .grid { grid-template-columns: 1fr; }
}

/* Áreas responsivas */
.layout {
    display: grid;
    grid-template-areas:
        "header header"
        "sidebar main"
        "footer footer";
}

@media (max-width: 768px) {
    .layout {
        grid-template-areas:
            "header"
            "main"
            "sidebar"
            "footer";
        grid-template-columns: 1fr;
    }
}

=======================================================================
AUTO-FILL VS AUTO-FIT (RECORDATORIO)
=======================================================================

repeat(auto-fill, minmax(200px, 1fr))
→ Crea columnas aunque estén vacías. El espacio se mantiene.

repeat(auto-fit, minmax(200px, 1fr))
→ Colapsa las columnas vacías a 0. Los items se expanden.

Ejemplo: contenedor de 600px con items de 200px.
- auto-fill: 3 columnas de 200px (no sobra espacio).
- auto-fit: 3 columnas de 200px (mismo resultado).

Si el contenedor mide 700px:
- auto-fill: 4 columnas (la última vacía, ocupa 175px cada una).
- auto-fit: 3 columnas de 233px (se reparten el espacio extra).

Usa auto-fit para layouts donde quieres que los items se estiren.
Usa auto-fill para grids donde el número de columnas debe ser fijo.

=======================================================================
GLOSARIO
===============================================================================

auto-fill: Crea columnas para llenar el espacio disponible.
auto-fit: Similar pero colapsa columnas vacías.
minmax(): Rango mínimo y máximo para el tamaño de columna/fila.
repeat(): Repite un patrón de columnas/filas.
media query: Regla CSS que aplica estilos según el dispositivo.
responsivo: Diseño que se adapta al tamaño de pantalla.

ERRORES FRECUENTES:
- Confundir auto-fill con auto-fit
- No definir un min adecuado en minmax
- Olvidar que las media queries deben ir al final del CSS
PREGUNTAS FRECUENTES:
- ¿Cuándo usar auto-fill vs auto-fit?
  auto-fill si quieres estructura fija; auto-fit si quieres
  que los items se expandan.
- ¿Necesito media queries si uso auto-fill?
  No, pero a veces quieres cambiar la estructura (ej: sidebar).
