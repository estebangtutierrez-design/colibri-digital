"Grid no es solo otra forma de maquetar. Es la primera vez que CSS
tiene un sistema de layout bidimensional real: filas y columnas
al mismo tiempo."
                         — Profesor Búho AI

===============================================================================
BIENVENIDA
===============================================================================

Hasta ahora hemos usado Flexbox para layouts unidimensionales (una
sola dirección: fila O columna). Pero ¿qué pasa cuando necesitas
controlar AMBAS direcciones a la vez?

Ahí entra CSS Grid.

Grid te permite dividir una página en filas y columnas, y colocar
elementos en celdas específicas. Es el sistema de layout más
poderoso que existe en CSS.

===============================================================================
OBJETIVOS
===============================================================================

✓ Entender qué es Grid y para qué sirve.
✓ Crear un contenedor grid.
✓ Definir columnas con grid-template-columns.
✓ Definir filas con grid-template-rows.
✓ Colocar elementos en celdas específicas.
✓ Diferencia entre Grid y Flexbox.

===============================================================================
TEORÍA
===============================================================================

GRID VS FLEXBOX:

Flexbox → unidimensional (una fila O una columna).
Grid    → bidimensional (filas Y columnas a la vez).

Piensa en una hoja de cálculo (Excel). Flexbox es una fila de celdas.
Grid es toda la hoja.

Un contenedor grid se crea con:
  display: grid;

Sus hijos directos se convierten en grid items y se colocan
automáticamente en celdas.

===============================================================================
ANALOGÍA
===============================================================================

Flexbox es un estante: los libros se colocan uno al lado del otro
en una fila. Si no caben, se envuelven a la siguiente fila.

Grid es un tablero de ajedrez: tienes 8 filas y 8 columnas exactas.
Cada pieza ocupa una casilla y sabes exactamente dónde está.

===============================================================================
SINTAXIS BÁSICA
===============================================================================

.contenedor {
    display: grid;
    grid-template-columns: 100px 200px 100px; /* 3 columnas */
    grid-template-rows: 100px 100px;           /* 2 filas */
    gap: 10px;                                 /* espacio entre celdas */
}

Los hijos se colocan automáticamente en orden:
- El primero va a la celda (fila 1, columna 1)
- El segundo a (fila 1, columna 2)
- El tercero a (fila 1, columna 3)
- El cuarto a (fila 2, columna 1)
- Y así sucesivamente.

===============================================================================
CÓDIGO PRÁCTICO
===============================================================================

<style>
* { margin: 0; padding: 0; box-sizing: border-box; }
body { font-family: Arial; background: #f0f2f5; padding: 40px; }
h1 { margin-bottom: 20px; }
h2 { margin: 25px 0 10px; font-size: 1.1rem; }

.grid {
    display: grid;
    background: white;
    border-radius: 8px;
    padding: 10px;
    border: 2px solid #ddd;
    margin-bottom: 25px;
}

.item {
    background: #3498db;
    color: white;
    padding: 20px;
    font-weight: bold;
    text-align: center;
    border-radius: 4px;
}

/* Grid de 3 columnas iguales */
.grid-1 {
    grid-template-columns: 1fr 1fr 1fr;
    gap: 10px;
}

/* Grid de columnas fijas */
.grid-2 {
    grid-template-columns: 100px 200px 100px;
    grid-template-rows: 80px 80px;
    gap: 8px;
}

/* Grid con fracciones mezcladas */
.grid-3 {
    grid-template-columns: 2fr 1fr 1fr;
    gap: 10px;
}

/* Grid con columnas automáticas */
.grid-4 {
    grid-template-columns: repeat(4, 1fr);
    gap: 10px;
}

/* Grid con posición específica */
.grid-5 {
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: 60px 60px 60px;
    gap: 10px;
}

.item-destacado {
    background: #e74c3c;
    grid-column: 2 / 4;
    grid-row: 1 / 3;
}
</style>

<h1>Introducción a Grid</h1>

<h2>3 columnas iguales con 1fr</h2>
<div class="grid grid-1">
    <div class="item">1</div>
    <div class="item">2</div>
    <div class="item">3</div>
</div>

<h2>Columnas fijas y filas definidas</h2>
<div class="grid grid-2">
    <div class="item">1</div>
    <div class="item">2</div>
    <div class="item">3</div>
    <div class="item">4</div>
</div>

<h2>Columnas con diferentes fracciones</h2>
<div class="grid grid-3">
    <div class="item">2fr (doble)</div>
    <div class="item">1fr</div>
    <div class="item">1fr</div>
</div>

<h2>repeat() — atajo para columnas iguales</h2>
<div class="grid grid-4">
    <div class="item">1</div>
    <div class="item">2</div>
    <div class="item">3</div>
    <div class="item">4</div>
</div>

<h2>Posicionar un elemento en celdas específicas</h2>
<div class="grid grid-5">
    <div class="item">1</div>
    <div class="item">2</div>
    <div class="item item-destacado">3 (ocupa 2 columnas, 2 filas)</div>
    <div class="item">4</div>
    <div class="item">5</div>
</div>

===============================================================================
EXPLICACIÓN DE 1fr
===============================================================================

fr = fracción del espacio disponible.

grid-template-columns: 1fr 1fr 1fr
→ 3 columnas que se reparten el espacio en partes iguales.

grid-template-columns: 2fr 1fr 1fr
→ La primera columna recibe el DOBLE de espacio que las otras.

grid-template-columns: repeat(4, 1fr)
→ 4 columnas iguales. repeat() es un atajo.

===============================================================================
COLOCACIÓN AUTOMÁTICA
===============================================================================

Los elementos se colocan automáticamente en el grid:
- Llenan cada fila de izquierda a derecha.
- Cuando una fila se completa, pasan a la siguiente.
- Esto se llama "auto-placement" y es el comportamiento por defecto.

Si quieres colocar un elemento en una posición específica:
   .item {
       grid-column: 2 / 4;  /* de línea 2 a línea 4 */
       grid-row: 1 / 3;     /* de línea 1 a línea 3 */
   }

===============================================================================
ERRORES COMUNES
===============================================================================

1. Usar grid cuando solo necesitas una fila → usa flexbox.
2. Olvidar que gap solo funciona en contenedores grid/flex.
3. Poner grid en el elemento hijo en lugar del padre.
4. Confundir el número de líneas con el número de columnas:
   - 3 columnas → 4 líneas (los bordes).

===============================================================================
EJERCICIOS
===============================================================================

1. Crea un grid de 2 columnas iguales con 4 items.
2. Crea un grid de 4 columnas: 1fr 2fr 1fr 2fr.
3. Usa repeat(3, 100px) para crear 3 columnas fijas.
4. Coloca un item que ocupe 2 columnas completas.

===============================================================================
GLOSARIO
===============================================================================

Grid: Sistema de layout bidimensional (filas y columnas).
grid-template-columns: Define el ancho de las columnas.
grid-template-rows: Define el alto de las filas.
gap: Espacio entre celdas.
fr: Fracción del espacio disponible.
repeat(): Función para repetir patrones de columnas/filas.
grid-column: Posición del elemento en las columnas.
grid-row: Posición del elemento en las filas.
