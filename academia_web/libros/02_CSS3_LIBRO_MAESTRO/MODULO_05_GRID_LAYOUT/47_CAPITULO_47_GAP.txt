################################################################################

"El espacio entre elementos es tan importante como los elementos
mismos. Gap es la forma correcta de separar celdas en Grid.
Sin margin, sin padding raros. Solo gap."
                         — Profesor Búho AI

===============================================================================
BIENVENIDA
===============================================================================

Antes de gap, separar elementos en CSS requería margin en los
hijos, o selectores complejos como :not(:last-child).

gap llegó para simplificar: una sola propiedad en el contenedor
que define el espacio entre todas las celdas del grid. Sin
desbordamientos, sin márgenes colapsados.

===============================================================================
OBJETIVOS
===============================================================================

✓ Usar gap, row-gap y column-gap en grid.
✓ Diferenciar gap de margin en items.
✓ Usar gap con diferentes unidades.
✓ Combinar gap con padding.
✓ Entender gap en Flexbox.

===============================================================================
TEORÍA
===============================================================================

gap: 10px → mismo espacio entre filas y columnas.
gap: 20px 10px → 20px filas, 10px columnas.
row-gap: 20px → solo entre filas.
column-gap: 15px → solo entre columnas.

DIFERENCIA CON MARGIN:
- gap va en el contenedor, margin en los items.
- gap no se suma al tamaño del item.
- gap no colapsa como los márgenes.

DIFERENCIA CON PADDING:
- padding: espacio entre el borde del contenedor y los items.
- gap: espacio entre items.

Usa ambos juntos: padding para separar del borde, gap para items.

===============================================================================
ANALOGÍA
===============================================================================

Asientos de cine:
- padding → espacio entre la pared y la primera fila.
- row-gap → espacio entre filas de asientos.
- column-gap → espacio entre asientos de la misma fila.

===============================================================================
SINTAXIS
===============================================================================

.grid {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 16px;
}

/* gap diferente */
.grid { gap: 20px 10px; } /* filas columnas */

/* gap individual */
.grid { row-gap: 20px; column-gap: 10px; }

/* gap + padding */
.grid { padding: 20px; gap: 16px; }

/* gap en Flexbox */
.flex { display: flex; flex-wrap: wrap; gap: 16px; }

=======================================================================
GAP EN FLEXBOX
=======================================================================

gap también funciona en Flexbox desde 2021:

.flex {
    display: flex;
    flex-wrap: wrap;
    gap: 16px;
}

Antes se usaba margin en los items:
.item { margin: 8px; }

Hoy gap es la forma estándar y funciona en todos los navegadores.

=======================================================================
ERRORES COMUNES
===============================================================================

1. gap sin display: grid o flex → no funciona.
2. Confundir gap con justify-content: space-between.
   gap separa celdas, space-between distribuye espacio sobrante.
3. Usar gap y margin en items sin necesidad.
   gap es suficiente en la mayoría de casos.

=======================================================================
EJERCICIOS
===============================================================================

1. Grid 3 columnas con gap: 15px uniforme.
2. Grid con row-gap: 30px y column-gap: 10px.
3. Grid con padding: 20px y gap: 12px.
4. Flexbox con 5 items y gap: 10px.

=======================================================================
GLOSARIO
===============================================================================

gap: Espacio entre filas y columnas en grid/flex.
row-gap: Espacio solo entre filas.
column-gap: Espacio solo entre columnas.
