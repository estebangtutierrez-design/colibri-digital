===============================================================================
BIENVENIDA
===============================================================================

Hoy construimos un dashboard completo usando CSS Grid. Este
proyecto práctico integra todo lo aprendido: columnas, filas,
áreas, gap, items con span y diseño responsivo.

El dashboard tendrá:
- Sidebar con navegación.
- Header con búsqueda y perfil.
- Cards de estadísticas.
- Tabla de datos.
- Gráfico (simulado).
- Lista de actividades recientes.

===============================================================================
OBJETIVOS
===============================================================================

✓ Construir un dashboard real con Grid.
✓ Usar grid-template-areas para el layout.
✓ Hacer el dashboard responsivo.
✓ Crear componentes internos con flexbox y grid.
✓ Integrar diseño responsivo.

=======================================================================
CÓMO SE CONSTRUYÓ
=======================================================================

LAYOUT PRINCIPAL (body):
   grid-template-columns: 220px 1fr;
   grid-template-rows: 60px 1fr;
   grid-template-areas:
       "sidebar header"
       "sidebar content";

   → Sidebar fija a la izquierda, header arriba, contenido al centro.

STATS CARDS:
   grid-template-columns: repeat(4, 1fr);
   → 4 columnas iguales para las tarjetas de estadísticas.

GRÁFICO + ACTIVIDAD:
   grid-template-columns: 2fr 1fr;
   → El gráfico ocupa el doble que la actividad.

RESPONSIVO (≤900px):
   - Sidebar se oculta (display: none).
   - Stats pasan a 2 columnas.
   - Chart + Activity pasan a 1 columna.

RESPONSIVO (≤500px):
   - Stats pasan a 1 columna.
   - Input de búsqueda se reduce.

=======================================================================
GLOSARIO
===============================================================================

Dashboard: Panel de control con resumen de datos.
grid-template-areas: Define zonas nombradas del layout.
sidebar: Barra lateral con navegación.
sticky: Posicionamiento que sigue al hacer scroll.
badge: Etiqueta pequeña para indicar estado.
chart: Gráfico de barras simulado con CSS.
