===============================================================================
CARACTERÍSTICAS DEL DISEÑO DESKTOP
===============================================================================

  • Mucho espacio horizontal (>1024px).
  • Interacción con mouse (hover, cursor pointer).
  • Pantallas grandes (1080p, 1440p, 4K).
  • Conexiones rápidas (generalmente).
  • Múltiples ventanas y pestañas.

===============================================================================
LAYOUTS DE ESCRITORIO
===============================================================================

  Columnas múltiples:

    @media (min-width: 1024px) {
      .grid { grid-template-columns: repeat(4, 1fr); }
    }

  Sidebar persistente:

    @media (min-width: 1024px) {
      .layout { display: grid; grid-template-columns: 250px 1fr; }
    }

  Hero de ancho completo:

    @media (min-width: 1024px) {
      .hero { height: 80vh; }
      .hero h1 { font-size: 3.5rem; }
    }

===============================================================================
TIPOGRAFÍA EN ESCRITORIO
===============================================================================

  En pantallas grandes, el texto puede (y debe) ser más grande.

  @media (min-width: 1024px) {
    body { font-size: 18px; }
    h1 { font-size: 3rem; }
    h2 { font-size: 2.25rem; }
    p { line-height: 1.8; }
  }

  Ancho de lectura: 65–75 caracteres por línea.

  @media (min-width: 1024px) {
    .articulo { max-width: 70ch; }
  }

===============================================================================
ESPACIADO Y JERARQUÍA
===============================================================================

  En desktop tienes espacio para respirar. Úsalo bien.

  @media (min-width: 1024px) {
    section { padding: 80px 0; }
    .card { padding: 40px; }
    .gap { gap: 32px; }
  }

  La jerarquía visual se refuerza con:
    • Tamaños de fuente más contrastados
    • Márgenes más generosos
    • Sombras y elevación (hover)

===============================================================================
INTERACCIONES CON MOUSE
===============================================================================

  En desktop tienes hover. Úsalo para feedback visual.

  @media (hover: hover) {
    .btn:hover { transform: translateY(-2px); }
    .card:hover { box-shadow: 0 12px 32px rgba(0,0,0,0.12); }
    a:hover { color: #3498db; }
  }

  También tooltips y menús desplegables:

  .dropdown:hover .dropdown-menu { display: block; }

===============================================================================
NO TE EXCEDAS
===============================================================================

  En desktop es tentador añadir de todo. Pero:

  • No más de 4–6 columnas (depende del contenido).
  • No uses todo el ancho en textos (ilegible).
  • No asumas que todos tienen monitor 4K.
  • No pongas interacciones solo-hover (inaccesible).

  El diseño desktop debe sentirse espacioso, no vacío.

===============================================================================
EJERCICIOS
===============================================================================

1. Transforma un layout de móvil a desktop usando min-width.
2. Aumenta la tipografía en desktop: body a 18px, h1 a 3rem.
3. Añade efectos hover que no existían en móvil.
4. Limita el ancho de un artículo a 70ch para legibilidad.

===============================================================================
GLOSARIO
===============================================================================

Desktop:      Pantalla grande (>1024px).
hover:        Estado al pasar el mouse.
espaciado:    Márgenes y paddings que dan respiro visual.
jerarquía:    Organización visual por importancia.
ch:           Unidad de ancho de carácter (70ch = óptimo lectura).
