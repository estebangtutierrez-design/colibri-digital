===============================================================================
OBJETIVO DEL PROYECTO
===============================================================================

Construir un sitio web responsivo completo desde cero, aplicando
todos los conceptos del módulo:

  ✓ Meta viewport y configuración inicial.
  ✓ Media queries con estrategia mobile-first.
  ✓ Diseño para móvil, tablet y desktop.
  ✓ Imágenes responsivas.
  ✓ Tipografía adaptativa.
  ✓ Código CSS optimizado y bien estructurado.

===============================================================================
ESPECIFICACIONES
===============================================================================

  Tipo de sitio:       Portafolio / Landing page personal.
  Breakpoints:         600px (tablet), 1024px (desktop).
  Estrategia:          Mobile-first (min-width).
  Fuente:              Sistema (Arial, sans-serif) o Google Fonts.
  Imágenes:            Con max-width: 100% y loading="lazy".
  Tipografía:          Escala modular con rem y clamp().
  Optimización:        CSS minificado y crítico inline.
