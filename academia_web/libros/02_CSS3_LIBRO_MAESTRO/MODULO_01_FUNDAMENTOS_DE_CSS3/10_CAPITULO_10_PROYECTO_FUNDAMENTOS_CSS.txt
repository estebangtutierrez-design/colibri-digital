PARTE 1: ¿QUÉ ES CSS?
===============================================================================

CSS significa Cascading Style Sheets (Hojas de Estilo en Cascada).

───────────────────────────────────────────────────────────
DEFINICIÓN
───────────────────────────────────────────────────────────

CSS es un lenguaje que describe la presentación de un documento HTML.
Controla:
- Colores.
- Fuentes.
- Tamaños.
- Espaciados.
- Posicionamiento.
- Diseño responsive.
- Animaciones.
- Y mucho más.

───────────────────────────────────────────────────────────
LA RELACIÓN HTML Y CSS
───────────────────────────────────────────────────────────

+----------------------------------------------------------+
|  HTML + CSS = Página Web Completa                        |
|                                                           |
|  HTML:                                                    |
|  - Estructura                                             |
|  - Contenido                                              |
|  - Semántica                                              |
|                                                           |
|  CSS:                                                     |
|  - Diseño                                                 |
|  - Colores                                                |
|  - Distribución                                           |
|  - Animación                                              |
+----------------------------------------------------------+

HTML da la estructura.
CSS da el estilo.
Juntos crean la web.

===============================================================================
PARTE 2: LA SINTAXIS BÁSICA DE CSS
===============================================================================

La sintaxis de CSS es muy simple y fácil de aprender.

───────────────────────────────────────────────────────────
ESTRUCTURA BÁSICA
───────────────────────────────────────────────────────────

```css
selector {
    propiedad: valor;
    propiedad: valor;
}
Desglose:

selector → El elemento que quieres estilizar.

propiedad → Lo que quieres cambiar (color, tamaño, etc.).

valor → El nuevo valor (blue, 16px, etc.).

───────────────────────────────────────────────────────────
EJEMPLO
───────────────────────────────────────────────────────────

css
h1 {
    color: blue;
    font-size: 24px;
}
Resultado: Todos los <h1> serán azules y tendrán un tamaño de 24px.

===============================================================================
PARTE 3: UN EJEMPLO PRÁCTICO
===============================================================================

Veamos cómo CSS transforma una página HTML.

───────────────────────────────────────────────────────────
HTML SIN CSS
───────────────────────────────────────────────────────────

html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Mi Página</title>
</head>
<body>
    <h1>Bienvenido a mi sitio</h1>
    <p>Este es un párrafo de ejemplo.</p>
</body>
</html>
Resultado:

Texto en negro.

Fondo blanco.

Sin diseño.

───────────────────────────────────────────────────────────
HTML CON CSS
───────────────────────────────────────────────────────────

html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Mi Página</title>
    <style>
        body {
            background: #0d1117;
            color: #c9d1d9;
            font-family: Arial, sans-serif;
        }
        h1 {
            color: #58a6ff;
            text-align: center;
        }
        p {
            font-size: 18px;
            text-align: center;
        }
    </style>
</head>
<body>
    <h1>Bienvenido a mi sitio</h1>
    <p>Este es un párrafo de ejemplo.</p>
</body>
</html>
Resultado:

Fondo oscuro (#0d1117).

Texto en color claro (#c9d1d9).

Título en azul (#58a6ff) y centrado.

Párrafo centrado y con tamaño de 18px.

La misma estructura HTML, pero con diseño.

PARTE 4: ¿POR QUÉ CSS3?
===============================================================================

CSS3 es la versión más moderna de CSS.
Incluye características avanzadas que no existían antes.

───────────────────────────────────────────────────────────
CSS VS. CSS3
───────────────────────────────────────────────────────────

CSS (versiones antiguas)	CSS3
Diseño básico	Diseño avanzado
Sin animaciones	Transiciones y animaciones
Sin Flexbox/Grid	Flexbox y Grid
Sin responsive	Media Queries
Fuentes limitadas	Fuentes web (@font-face)
Sin sombras	Sombras (box-shadow, text-shadow)
───────────────────────────────────────────────────────────
¿QUÉ PUEDES HACER CON CSS3?
───────────────────────────────────────────────────────────

Diseño responsivo: Páginas que se adaptan a cualquier dispositivo.

Animaciones: Efectos visuales y movimiento.

Flexbox y Grid: Distribución profesional de contenido.

Gradientes y sombras: Efectos visuales avanzados.

Fuentes web: Tipografías personalizadas.

Transformaciones: Rotar, escalar, mover elementos.

PARTE 6: LAS TRES FORMAS DE USAR CSS
===============================================================================

CSS se puede integrar en HTML de tres formas.

───────────────────────────────────────────────────────────

CSS EN LÍNEA (INLINE)
───────────────────────────────────────────────────────────

Se escribe directamente en el atributo style de un elemento.

html
<h1 style="color: blue;">Título</h1>
Ventajas: Rápido, directo.
Desventajas: Mezcla contenido con estilo, no es reutilizable.
Cuándo usar: Solo en casos muy específicos.

───────────────────────────────────────────────────────────

CSS INTERNO (INTERNAL)
───────────────────────────────────────────────────────────

Se escribe dentro de la etiqueta <style> en el <head>.

html
<head>
    <style>
        h1 {
            color: blue;
        }
    </style>
</head>
Ventajas: Separa contenido de estilo (aunque en la misma página).
Desventajas: No es reutilizable entre páginas.
Cuándo usar: Páginas individuales o prototipos.

───────────────────────────────────────────────────────────

CSS EXTERNO (EXTERNAL)
───────────────────────────────────────────────────────────

Se escribe en un archivo .css separado y se enlaza con <link>.

html
<head>
    <link rel="stylesheet" href="style.css">
</head>
css
/* style.css */
h1 {
    color: blue;
}
Ventajas: Separa completamente contenido de estilo, reutilizable, fácil de mantener.
Desventajas: Requiere un archivo adicional.
Cuándo usar: ¡SIEMPRE! Es la forma profesional.

GLOSARIO DE TÉRMINOS
===============================================================================

CSS: Cascading Style Sheets (Hojas de Estilo en Cascada).

CSS3: La versión más moderna de CSS.

Selector: Elemento, clase o ID que se va a estilizar.

Propiedad: Característica a modificar (color, tamaño, etc.).

Valor: Valor asignado a la propiedad.

Inline: CSS escrito directamente en el atributo style.

Interno: CSS escrito en la etiqueta <style>.

Externo: CSS escrito en un archivo .css separado.
