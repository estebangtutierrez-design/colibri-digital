===============================================================================
INTRODUCCIÓN: EL ARTE DE DAR ESTILO A LA WEB
===============================================================================

Imagina que estás construyendo una casa.

- **HTML** es la estructura: las paredes, el techo, las ventanas, las puertas.
- **CSS** es la decoración: la pintura de las paredes, los muebles, las cortinas, la iluminación.

HTML te da la estructura básica, pero CSS es lo que hace que una página web sea agradable a la vista y fácil de usar.

---

📌 **¿Qué significa CSS?**

**C**ascading **S**tyle **S**heets

- **Cascading** → en cascada (los estilos se aplican en un orden jerárquico).
- **Style** → estilo (colores, fuentes, tamaños, posiciones).
- **Sheets** → hojas (documentos que contienen los estilos).

---

📌 **¿Qué es CSS3?**

CSS3 es la **versión más reciente y moderna** de CSS.
No es una única versión, sino una colección de **módulos** que se actualizan de forma independiente.

CSS3 introdujo:
- Esquinas redondeadas (`border-radius`).
- Sombras (`box-shadow`, `text-shadow`).
- Gradientes (degradados de color).
- Animaciones y transiciones.
- Flexbox y Grid (sistemas de layout modernos).
- Media queries (diseño responsivo).
- Variables CSS (custom properties).

---

📌 **¿Por qué es importante CSS?**

1. **Mejora la experiencia del usuario**: un diseño atractivo hace que los usuarios quieran quedarse.
2. **Aumenta la credibilidad**: un sitio bien diseñado inspira confianza.
3. **Facilita la navegación**: un buen diseño guía al usuario.
4. **Refuerza la identidad de marca**: los colores y estilos comunican personalidad.
5. **Permite el diseño responsivo**: las páginas se adaptan a cualquier dispositivo.

HTML + CSS: LA SANTÍSIMA TRINIDAD DE LA WEB
===============================================================================

Para crear una página web moderna, necesitas tres tecnologías:

---

📌 **1. HTML (HyperText Markup Language)**

Define la **estructura** y el **contenido** de la página.
Es el "esqueleto".

```html

📌 2. CSS (Cascading Style Sheets)

Define el diseño y la apariencia de la página.
Es la "piel" y el "vestuario".

css

Define el comportamiento y la interactividad.
Es el "cerebro" y los "músculos".

javascript
document.querySelector('button').addEventListener('click', function() {
    alert('¡Hola desde JavaScript!');
});
📌 Ejemplo visual de la trinidad

===============================================================================
LA EVOLUCIÓN DE CSS: DESDE CSS1 HASTA CSS3
===============================================================================

📌 CSS1 (1996)

La primera versión de CSS.
Incluía:

Propiedades básicas de texto (color, font-family, font-size).

Propiedades básicas de diseño (margin, padding, border).

Selectores simples (elemento, clase, ID).

📌 CSS2 (1998)

Añadió características importantes:

Posicionamiento (relative, absolute, fixed).

Flotadores (float).

Z-index (control de superposición).

Selectores más avanzados (hijos, adyacentes).

Pseudo-clases (hover, focus, active).

📌 CSS2.1 (2004-2011)

Una revisión de CSS2 para corregir errores y aclarar el estándar.
Fue la versión más usada durante años.

📌 CSS3 (2009 - actualidad)

CSS3 no es una única versión, sino una colección de módulos.
Cada módulo se actualiza de forma independiente.

Módulos principales de CSS3:

Selectores → selectores avanzados.

Box Model → modelo de caja.

Backgrounds & Borders → fondos y bordes (border-radius, box-shadow).

Colors → colores (HSL, RGBA).

Text → efectos de texto (text-shadow).

Fonts → fuentes web (@font-face).

Flexbox → layout flexible.

Grid → layout de cuadrícula.

Animations → animaciones (@keyframes).

Transitions → transiciones suaves.

Media Queries → diseño responsivo.

Variables → custom properties (variables CSS).

📌 CSS hoy

CSS sigue evolucionando con nuevas funcionalidades:

Container Queries → diseño basado en el contenedor.

Subgrid → alineación de subgrids.

Nesting → anidamiento de selectores (como SASS).

Cascade Layers → control de capas en la cascada.

===============================================================================
FORMA DE APLICAR CSS
===============================================================================

Hay tres formas de aplicar CSS a un documento HTML:

📌 1. CSS Externo (recomendado)

Se escribe en un archivo .css separado y se vincula al HTML.

html
<!-- index.html -->
<head>
    <link rel="stylesheet" href="style.css">
</head>
css
/* style.css */
h1 {
    color: blue;
}
Ventajas:

Código más limpio y organizado.

Reutilizable en varias páginas.

Fácil de mantener.

📌 2. CSS Interno

Se escribe dentro de la etiqueta <style> en el <head>.

html
<head>
    <style>
        h1 {
            color: blue;
        }
    </style>
</head>
Ventajas:

Útil para páginas únicas.

No requiere archivos adicionales.

Desventajas:

No es reutilizable.

Puede hacer el HTML más pesado.

📌 3. CSS en Línea (inline)

Se escribe directamente en el atributo style de un elemento HTML.

html
<h1 style="color: blue;">Título</h1>
Ventajas:

Útil para estilos muy específicos.

Máxima prioridad en la cascada.

Desventajas:

Mezcla contenido y diseño (mala práctica).

Difícil de mantener.

No es reutilizable.

📌 Recomendación del Profesor Búho

Siempre usa CSS externo para proyectos serios.
Es más limpio, más organizado y más fácil de mantener.

===============================================================================
SINTAXIS BÁSICA DE CSS
===============================================================================

Toda regla CSS tiene la siguiente estructura:

text
selector {
    propiedad: valor;
}
📌 Partes de una regla CSS

Parte	Descripción	Ejemplo
Selector	Indica a qué elemento se aplica el estilo	h1, .titulo, #principal
Propiedad	La característica que se va a modificar	color, font-size
Valor	El valor que se asigna a la propiedad	blue, 16px
Declaración	La combinación de propiedad y valor	color: blue;
Bloque	El conjunto de declaraciones entre { }	{ color: blue; font-size: 16px; }
🔵 Ejemplo completo

css
h1 {
    color: blue;
    font-size: 36px;
    text-align: center;
}
Selector: h1

Declaraciones:

color: blue;

font-size: 36px;

text-align: center;

Bloque: todo lo que está entre { }.

📌 Ejemplos de selectores

Selector	Ejemplo	Descripción
Elemento	h1 { }	Selecciona todos los <h1>
Clase	.titulo { }	Selecciona elementos con class="titulo"
ID	#principal { }	Selecciona el elemento con id="principal"

TU PRIMER CÓDIGO CSS
===============================================================================

📌 Paso 1: Crea un archivo HTML

html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi primer CSS</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <h1>¡Bienvenido a CSS!</h1>
    <p>Este es un párrafo de ejemplo.</p>
</body>
</html>
📌 Paso 2: Crea un archivo CSS (style.css)

css
h1 {
    color: blue;
    font-size: 36px;
    text-align: center;
}

p {
    color: #333;
    font-size: 18px;
    line-height: 1.6;
}
📌 Paso 3: Abre el HTML en el navegador

Verás:

El título en azul, tamaño 36px y centrado.

El párrafo en gris oscuro, tamaño 18px y con espacio entre líneas.

¡Acabas de escribir tu primer CSS!

===============================================================================

Imagina que HTML es una persona.

===============================================================================
ERRORES COMUNES AL APRENDER CSS
===============================================================================

❌ Error 1: Olvidar el punto y coma (;)

css
h1 {
    color: blue    /* ❌ Falta el ; */
    font-size: 36px;
}
❌ Error 2: Olvidar cerrar las llaves ({ })

css
h1 {
    color: blue;
    font-size: 36px;
/* ❌ Falta } */
❌ Error 3: Confundir clase con ID

css
#titulo { }   /* Selecciona un ID */
.titulo { }   /* Selecciona una clase */
❌ Error 4: No vincular el CSS en el HTML

html
<!-- ❌ Falta la etiqueta <link> -->
<head>
    <title>Mi página</title>
</head>
❌ Error 5: Usar CSS en línea para todo

html
<h1 style="color: blue; font-size: 36px;">Título</h1>
<p style="color: #333; font-size: 18px;">Párrafo</p>
<!-- ❌ Mala práctica, mezcla contenido y diseño -->

===============================================================================
PREGUNTAS FRECUENTES (FAQ)
===============================================================================

❓ ¿Cuál es la diferencia entre HTML y CSS?

HTML define la estructura y el contenido. CSS define el diseño y la apariencia.

❓ ¿Qué significa "Cascading" en CSS?

Significa que los estilos se aplican en un orden jerárquico (en cascada).
Los estilos más específicos tienen prioridad sobre los más generales.

❓ ¿Cuál es la mejor forma de aplicar CSS?

La mejor forma es usar CSS externo (archivo .css separado).
Es más limpio, reutilizable y fácil de mantener.

❓ ¿Qué es CSS3?

CSS3 es la versión más reciente de CSS.
Es una colección de módulos que se actualizan de forma independiente.

❓ ¿Necesito saber HTML antes de CSS?

Sí, es recomendable tener conocimientos básicos de HTML
antes de aprender CSS, porque CSS se aplica sobre HTML.
