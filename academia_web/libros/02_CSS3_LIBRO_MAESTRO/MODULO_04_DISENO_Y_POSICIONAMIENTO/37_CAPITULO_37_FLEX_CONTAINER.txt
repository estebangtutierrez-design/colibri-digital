Ahora es momento de profundizar en el **contenedor flex**.

El contenedor flex es el elemento que tiene `display: flex`.
Es el "jefe" que organiza a todos los hijos (flex items).

Las propiedades que se aplican al contenedor flex son las que
definen cómo se comportan los hijos en su interior:

- `flex-direction` → dirección de los elementos.
- `flex-wrap` → si se envuelven o no.
- `flex-flow` → combinación de ambos.
- `justify-content` → alineación en el eje principal.
- `align-items` → alineación en el eje transversal.
- `align-content` → alineación de líneas múltiples.
- `gap` → espacio entre elementos.

Hoy vas a dominar todas las propiedades del contenedor flex.

===============================================================================
INTRODUCCIÓN: ¿QUÉ ES UN CONTENEDOR FLEX?
===============================================================================

Un **contenedor flex** es cualquier elemento HTML que tenga
`display: flex` o `display: inline-flex`.

---

📌 **Sintaxis**
display: flex; /* Contenedor flex de bloque /
display: inline-flex; / Contenedor flex en línea */

text

---

📌 **Propiedades del contenedor flex**

| Propiedad | Descripción |
|-----------|-------------|
| `flex-direction` | Dirección de los elementos (row, column, etc.) |
| `flex-wrap` | Envoltorio de los elementos (wrap, nowrap, etc.) |
| `flex-flow` | Shorthand para flex-direction y flex-wrap |
| `justify-content` | Alineación en el eje principal |
| `align-items` | Alineación en el eje transversal |
| `align-content` | Alineación de líneas múltiples |
| `gap` | Espacio entre elementos flex |

---

📌 **Estructura básica**
┌─────────────────────────────────────────────────────────┐
│ CONTENEDOR FLEX │
│ ┌───────────┐ ┌───────────┐ ┌───────────┐ │
│ │ │ │ │ │ │ │
│ │ Item 1 │ │ Item 2 │ │ Item 3 │ │
│ │ │ │ │ │ │ │
│ └───────────┘ └───────────┘ └───────────┘ │
│ │
└─────────────────────────────────────────────────────────┘

text

===============================================================================
1. FLEX-DIRECTION: DIRECCIÓN DE LOS ELEMENTOS
===============================================================================

`flex-direction` define la dirección en la que se colocan los elementos flex.

---

📌 **Valores de flex-direction**

| Valor | Comportamiento |
|-------|----------------|
| `row` | Horizontal (izquierda → derecha) - **valor por defecto**. |
| `row-reverse` | Horizontal inversa (derecha → izquierda). |
| `column` | Vertical (arriba → abajo). |
| `column-reverse` | Vertical inversa (abajo → arriba). |

---

🔵 **Ejemplo: flex-direction: row (por defecto)**

------------------------------------------------------------
CSS
------------------------------------------------------------
.contenedor {
    display: flex;
    flex-direction: row;   /* Horizontal */
}
------------------------------------------------------------
┌─────────────────────────────────────────────┐
│ [Item 1] [Item 2] [Item 3] [Item 4] │
└─────────────────────────────────────────────┘

text

---

🔵 **Ejemplo: flex-direction: column**

------------------------------------------------------------
CSS
------------------------------------------------------------
.contenedor {
    display: flex;
    flex-direction: column;   /* Vertical */
}
------------------------------------------------------------
┌─────────────────────────────────────────────┐
│ [Item 1] │
│ [Item 2] │
│ [Item 3] │
│ [Item 4] │
└─────────────────────────────────────────────┘

text

---

🔵 **Ejemplo: flex-direction: row-reverse**

------------------------------------------------------------
CSS
------------------------------------------------------------
.contenedor {
    display: flex;
    flex-direction: row-reverse;   /* Horizontal inversa */
}
------------------------------------------------------------
┌─────────────────────────────────────────────┐
│ [Item 4] [Item 3] [Item 2] [Item 1] │
└─────────────────────────────────────────────┘

text

---

📌 **Cuándo usar cada valor**

- `row`: layouts horizontales (menús, barras, tarjetas en fila).
- `column`: layouts verticales (listas, formularios).
- `row-reverse` o `column-reverse`: para invertir el orden visual.

===============================================================================
2. FLEX-WRAP: ENVOLTORIO DE ELEMENTOS
===============================================================================

Por defecto, los elementos flex intentan caber en una sola línea.
`flex-wrap` controla si los elementos deben envolverse a múltiples líneas.

---

📌 **Valores de flex-wrap**

| Valor | Comportamiento |
|-------|----------------|
| `nowrap` | Todos los elementos en una sola línea (por defecto). |
| `wrap` | Los elementos se envuelven a múltiples líneas. |
| `wrap-reverse` | Los elementos se envuelven en dirección inversa. |

---

🔵 **Ejemplo: flex-wrap: wrap**

------------------------------------------------------------
CSS
------------------------------------------------------------
.contenedor {
    display: flex;
    flex-wrap: wrap;
    width: 300px;
}
------------------------------------------------------------

**Resultado:** los elementos se colocan en varias líneas si no caben en una.
┌─────────────────────────────┐
│ [Item1] [Item2] [Item3] │
│ [Item4] [Item5] │
└─────────────────────────────┘

text

---

🔵 **Ejemplo: flex-wrap: nowrap (por defecto)**

------------------------------------------------------------
CSS
------------------------------------------------------------
.contenedor {
    display: flex;
    flex-wrap: nowrap;
    width: 300px;
}
------------------------------------------------------------

**Resultado:** los elementos se comprimen o salen del contenedor.
┌─────────────────────────────┐
│ [Item1][Item2][Item3]... │
└─────────────────────────────┘

text

---

📌 **Cuándo usar flex-wrap**

- **`nowrap`**: cuando quieres que todos los elementos estén en una línea.
- **`wrap`**: en diseños responsivos, para que los elementos se adapten al ancho.
- **`wrap-reverse`**: para invertir el orden de las líneas.

===============================================================================
3. FLEX-FLOW: SHORTHAND DE DIRECCIÓN Y ENVOLTORIO
===============================================================================

`flex-flow` combina `flex-direction` y `flex-wrap` en una sola propiedad.

---

📌 **Sintaxis**
flex-flow: flex-direction flex-wrap;

text

---

🔵 **Ejemplo**

------------------------------------------------------------
CSS
------------------------------------------------------------
.contenedor {
    display: flex;
    flex-flow: row wrap;   /* Dirección: row, Envoltorio: wrap */
}
------------------------------------------------------------

Es equivalente a:
flex-direction: row;
flex-wrap: wrap;

text

---

🔵 **Otros ejemplos**
flex-flow: column wrap; /* Vertical, con envoltorio /
flex-flow: row nowrap; / Horizontal, sin envoltorio /
flex-flow: row-reverse wrap; / Horizontal inversa, con envoltorio */

text

5. ALIGN-ITEMS: ALINEACIÓN EN EL EJE TRANSVERSAL
===============================================================================

`align-items` controla la alineación de los elementos en el **eje transversal**.

---

📌 **Valores de align-items**

| Valor | Comportamiento |
|-------|----------------|
| `stretch` | Los elementos se estiran para llenar el contenedor (por defecto). |
| `flex-start` | Alinea los elementos al inicio del eje transversal. |
| `flex-end` | Alinea los elementos al final del eje transversal. |
| `center` | Centra los elementos en el eje transversal. |
| `baseline` | Alinea los elementos por su línea base de texto. |

---

🔵 **Ejemplo: align-items: stretch (por defecto)**

------------------------------------------------------------
CSS
------------------------------------------------------------
.contenedor {
    display: flex;
    align-items: stretch;
    height: 150px;
    border: 2px solid #333;
}
------------------------------------------------------------

**Resultado:** los elementos se estiran verticalmente para llenar el contenedor.

===============================================================================
6. ALIGN-CONTENT: ALINEACIÓN DE LÍNEAS MÚLTIPLES
===============================================================================

`align-content` controla la alineación de **líneas múltiples** de elementos flex
(es decir, cuando `flex-wrap: wrap` está activo y hay más de una línea).

---

📌 **Valores de align-content**

| Valor | Comportamiento |
|-------|----------------|
| `stretch` | Las líneas se estiran para llenar el espacio (por defecto). |
| `flex-start` | Alinea las líneas al inicio. |
| `flex-end` | Alinea las líneas al final. |
| `center` | Centra las líneas. |
| `space-between` | Distribuye líneas con espacio entre ellas. |
| `space-around` | Distribuye líneas con espacio alrededor. |

---

🔵 **Ejemplo: align-content: center con wrap**

------------------------------------------------------------
CSS
------------------------------------------------------------
.contenedor {
    display: flex;
    flex-wrap: wrap;
    align-content: center;
    height: 300px;
    width: 200px;
}
------------------------------------------------------------

**Resultado:** las líneas de elementos se centran verticalmente en el contenedor.
┌─────────────────────────────┐
│ │
│ [Item1] [Item2] │
│ [Item3] [Item4] │
│ │
└─────────────────────────────┘

text

---

🔵 **Ejemplo: align-content: space-between**

------------------------------------------------------------
CSS
------------------------------------------------------------
.contenedor {
    display: flex;
    flex-wrap: wrap;
    align-content: space-between;
    height: 300px;
}
------------------------------------------------------------

**Resultado:** las líneas se distribuyen con espacio entre ellas.

---

📌 **Diferencia entre align-items y align-content**

| Propiedad | Cuándo usar |
|-----------|-------------|
| `align-items` | Cuando hay **una sola línea** de elementos. |
| `align-content` | Cuando hay **múltiples líneas** de elementos (con `flex-wrap: wrap`). |

---

⚠️ **Atención**

Si solo hay una línea de elementos, `align-content` no tiene efecto.

===============================================================================
7. GAP: ESPACIO ENTRE ELEMENTOS
===============================================================================

`gap` crea espacio entre los elementos flex, sin necesidad de márgenes.

---

📌 **Sintaxis**
gap: valor;
gap: horizontal vertical; /* Dos valores */

text

---

🔵 **Ejemplo: gap entre elementos**

------------------------------------------------------------
CSS
------------------------------------------------------------
.contenedor {
    display: flex;
    flex-wrap: wrap;
    gap: 20px;   /* Espacio de 20px entre todos los elementos */
}
------------------------------------------------------------

**Resultado:** hay 20px de espacio entre cada elemento.

---

🔵 **Ejemplo: gap con dos valores**

------------------------------------------------------------
CSS
------------------------------------------------------------
.contenedor {
    display: flex;
    flex-wrap: wrap;
    gap: 10px 20px;   /* 10px vertical, 20px horizontal */
}
------------------------------------------------------------

---

📌 **Ventajas de gap sobre margin**

1. **No colapsa** (a diferencia de los márgenes).
2. **Se aplica al contenedor**, no a cada elemento.
3. **Más simple** y limpio.
4. **Funciona con flex-wrap**.

---

📌 **Comparativa: gap vs margin**

| Método | Ventajas | Desventajas |
|--------|----------|-------------|
| `gap` | Simple, no colapsa | No soportado en navegadores antiguos |
| `margin` | Ampliamente soportado | Puede colapsar, más código |

---

🟢 **Recomendación**

Usa `gap` siempre que sea posible (navegadores modernos).
Si necesitas soportar navegadores antiguos, usa `margin`.

===============================================================================
LA ANALOGÍA DEL PROFESOR BÚHO: EL CONTENEDOR FLEX COMO UN DIRECTOR DE ORQUESTA
===============================================================================

Imagina que el contenedor flex es un director de orquesta.

- **`flex-direction`** es la disposición de los músicos (en fila o en columna).
- **`flex-wrap`** es si los músicos caben en una fila o necesitan varias.
- **`justify-content`** es cómo se distribuyen los músicos en el escenario (juntos, separados, centrados).
- **`align-items`** es cómo se alinean verticalmente (arriba, abajo, centrados).
- **`align-content`** es cómo se distribuyen las filas de músicos.
- **`gap`** es el espacio entre los músicos.

El director decide cómo se organiza la orquesta.
Tú eres el director.

===============================================================================
ERRORES COMUNES CON EL CONTENEDOR FLEX
===============================================================================

❌ **Error 1: Olvidar `display: flex`**

   Si no defines `display: flex`, el contenedor no será flexible.

---

❌ **Error 2: Usar `align-content` con una sola línea**

   `align-content` solo funciona cuando hay múltiples líneas (`flex-wrap: wrap`).

---

❌ **Error 3: Confundir `justify-content` con `align-items`**

   `justify-content` alinea en el **eje principal**.
   `align-items` alinea en el **eje transversal**.

---

❌ **Error 4: No usar `gap` cuando es posible**

   `gap` es más limpio que `margin` para separar elementos.

---

❌ **Error 5: Asumir que `flex-direction: column` cambia `justify-content` y `align-items`**

   Cuando usas `column`, el eje principal es vertical y el transversal es horizontal.
   Las propiedades se intercambian.

===============================================================================
GLOSARIO DEL CAPÍTULO
===============================================================================

- **Contenedor flex**: Elemento con `display: flex`.
- **Eje principal (main axis)**: Dirección en la que fluyen los elementos.
- **Eje transversal (cross axis)**: Dirección perpendicular al eje principal.
- **flex-direction**: Controla la dirección del eje principal.
- **flex-wrap**: Controla si los elementos se envuelven.
- **flex-flow**: Shorthand de flex-direction y flex-wrap.
- **justify-content**: Alineación en el eje principal.
- **align-items**: Alineación en el eje transversal.
- **align-content**: Alineación de líneas múltiples.
- **gap**: Espacio entre elementos.

===============================================================================
PREGUNTAS FRECUENTES (FAQ)
===============================================================================

❓ **¿Qué es un contenedor flex?**

   Es un elemento con `display: flex` que organiza a sus hijos en un layout flexible.

---

❓ **¿Cuál es la diferencia entre justify-content y align-items?**

   `justify-content` alinea en el **eje principal**.
   `align-items` alinea en el **eje transversal**.

---

❓ **¿Cómo centro un elemento con Flexbox?**

   Con `justify-content: center` y `align-items: center`.

---

❓ **¿Cómo hago que los elementos se envuelvan a varias líneas?**

   Con `flex-wrap: wrap`.

---

❓ **¿Qué hace gap en Flexbox?**

   `gap` crea espacio entre los elementos flex, sin necesidad de márgenes.
