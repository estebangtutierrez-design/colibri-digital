===============================================================================
INTRODUCCIÓN: ¿QUÉ SON LOS FLEX ITEMS?
===============================================================================

Los **flex items** son los hijos directos de un contenedor flex.

- Un contenedor flex se crea con `display: flex`.
- Todos los hijos directos de ese contenedor se convierten en flex items.
- Los flex items pueden ser controlados individualmente.

---

📌 **Propiedades de los flex items**

| Propiedad | Descripción |
|-----------|-------------|
| `flex-grow` | Controla cuánto crece el elemento en el espacio disponible. |
| `flex-shrink` | Controla cuánto se encoge el elemento si falta espacio. |
| `flex-basis` | Define el tamaño inicial del elemento antes de crecer o encogerse. |
| `flex` | Shorthand para `flex-grow`, `flex-shrink` y `flex-basis`. |
| `order` | Cambia el orden de aparición de los elementos. |
| `align-self` | Sobrescribe `align-items` para un elemento específico. |

---

📌 **Características de los flex items**

1. **Todos los hijos directos** del contenedor flex se convierten en flex items.
2. **Los flex items se alinean** en el eje principal.
3. **Pueden crecer o encogerse** para llenar el espacio disponible.
4. **Pueden cambiar de orden** sin modificar el HTML.

===============================================================================
1. FLEX-GROW: CÓMO CRECE UN ELEMENTO
===============================================================================

`flex-grow` controla cuánto espacio adicional ocupa un elemento
cuando hay espacio disponible en el contenedor.

---

📌 **Sintaxis**
flex-grow: número;

text

- **Valor por defecto**: `0` (no crece).
- **Valor positivo**: el elemento crece para ocupar el espacio disponible.
- **Valor mayor**: el elemento ocupa más espacio proporcionalmente.

---

🔵 **Ejemplo: todos los elementos crecen igual**

------------------------------------------------------------
HTML
------------------------------------------------------------
<div class="contenedor">
    <div class="item">1</div>
    <div class="item">2</div>
    <div class="item">3</div>
</div>
------------------------------------------------------------

CSS:
------------------------------------------------------------
.contenedor {
    display: flex;
    width: 600px;
    border: 2px solid #333;
}

.item {
    flex-grow: 1;   /* Todos crecen igual */
    background-color: #3498DB;
    color: white;
    padding: 20px;
    margin: 5px;
}
------------------------------------------------------------

**Resultado:** Los tres elementos ocupan el mismo espacio (1/3 del contenedor cada uno).

---

🔵 **Ejemplo: un elemento crece más que los otros**

------------------------------------------------------------
CSS
------------------------------------------------------------
.item1 {
    flex-grow: 1;   /* Ocupa 1 parte */
    background-color: #3498DB;
}

.item2 {
    flex-grow: 2;   /* Ocupa 2 partes (el doble) */
    background-color: #28A745;
}

.item3 {
    flex-grow: 1;   /* Ocupa 1 parte */
    background-color: #DC3545;
}
------------------------------------------------------------

**Resultado:** El elemento 2 ocupa el doble de espacio que los otros.
┌─────────────────────────────────────────────────────┐
│ [ 1/4 ] [ 2/4 ] [ 1/4 ] │
└─────────────────────────────────────────────────────┘

text

---

📌 **Cómo funciona flex-grow**

1. El contenedor reparte el espacio disponible entre los elementos.
2. Cada elemento recibe una parte proporcional a su `flex-grow`.
3. La suma de todos los `flex-grow` determina la proporción.
Espacio total = 600px
flex-grow total = 1 + 2 + 1 = 4
Parte = 600 / 4 = 150px

Item1: 150px * 1 = 150px
Item2: 150px * 2 = 300px
Item3: 150px * 1 = 150px

text

---

📌 **Cuándo usar flex-grow**

- Para crear columnas de diferentes tamaños.
- Para que un elemento ocupe más espacio que otros.
- Para layouts donde el espacio debe distribuirse proporcionalmente.

===============================================================================
2. FLEX-SHRINK: CÓMO SE ENCOGE UN ELEMENTO
===============================================================================

`flex-shrink` controla cuánto se encoge un elemento cuando
no hay suficiente espacio en el contenedor.

---

📌 **Sintaxis**
flex-shrink: número;

text

- **Valor por defecto**: `1` (se encoge si es necesario).
- **Valor `0`**: el elemento no se encoge (mantiene su tamaño).
- **Valor mayor**: el elemento se encoge más que los otros.

---

🔵 **Ejemplo: elementos que se encogen**

------------------------------------------------------------
HTML
------------------------------------------------------------
<div class="contenedor">
    <div class="item">Elemento largo 1</div>
    <div class="item">Elemento largo 2</div>
    <div class="item">Elemento largo 3</div>
</div>
------------------------------------------------------------

CSS:
------------------------------------------------------------
.contenedor {
    display: flex;
    width: 300px;   /* Poco espacio */
    border: 2px solid #333;
}

.item {
    flex-shrink: 1;   /* Se encogen igual (por defecto) */
    background-color: #3498DB;
    color: white;
    padding: 20px;
    margin: 5px;
    width: 150px;   /* Tamaño deseado */
}
------------------------------------------------------------

**Resultado:** Los elementos se encogen para caber en el contenedor.

---

🔵 **Ejemplo: un elemento no se encoge**

------------------------------------------------------------
CSS
------------------------------------------------------------
.item1 {
    flex-shrink: 0;   /* No se encoge */
    background-color: #3498DB;
}

.item2 {
    flex-shrink: 1;   /* Se encoge */
    background-color: #28A745;
}

.item3 {
    flex-shrink: 1;   /* Se encoge */
    background-color: #DC3545;
}
------------------------------------------------------------

**Resultado:** El elemento 1 mantiene su tamaño, los otros se encogen para compensar.

---

📌 **Cuándo usar flex-shrink**

- Para evitar que un elemento se encoga (ej: un logo que debe mantener su tamaño).
- Para controlar qué elementos se encogen primero.

===============================================================================
3. FLEX-BASIS: TAMAÑO INICIAL
===============================================================================

`flex-basis` define el tamaño inicial de un elemento antes de que
`flex-grow` y `flex-shrink` se apliquen.

---

📌 **Sintaxis**
flex-basis: valor;

text

- **Valor por defecto**: `auto` (usa el tamaño del contenido).
- **Valor en px, %, em, rem, etc.**: tamaño fijo o relativo.

---

🔵 **Ejemplo: flex-basis básico**

------------------------------------------------------------
CSS
------------------------------------------------------------
.item {
    flex-basis: 100px;   /* Tamaño inicial de 100px */
    background-color: #3498DB;
    color: white;
    padding: 20px;
}
------------------------------------------------------------

---

🔵 **Ejemplo: flex-basis con crecimiento**

------------------------------------------------------------
CSS
------------------------------------------------------------
.item {
    flex-grow: 1;
    flex-basis: 100px;   /* Empieza en 100px, luego crece */
}
------------------------------------------------------------

---

📌 **Diferencia entre width y flex-basis**

| Característica | `width` | `flex-basis` |
|----------------|---------|--------------|
| En flexbox | Se usa si `flex-basis` es `auto` | Tiene prioridad sobre `width` |
| En otros contextos | Funciona normalmente | No existe |
| Recomendación | Usa `flex-basis` para flex items | Prefiere `flex-basis` |

---

🔵 **Ejemplo: flex-basis vs width**

------------------------------------------------------------
CSS
------------------------------------------------------------
.item {
    width: 100px;
    flex-basis: 150px;   /* Esto gana */
}
------------------------------------------------------------

**Resultado:** `flex-basis` tiene prioridad sobre `width`.

===============================================================================
4. FLEX: EL SHORTHAND
===============================================================================

`flex` es el shorthand que combina `flex-grow`, `flex-shrink` y `flex-basis`.

---

📌 **Sintaxis**
flex: flex-grow flex-shrink flex-basis;

text

---

🔵 **Ejemplo: flex con tres valores**

------------------------------------------------------------
CSS
------------------------------------------------------------
.item {
    flex: 1 1 100px;   /* grow: 1, shrink: 1, basis: 100px */
}
------------------------------------------------------------

---

🔵 **Ejemplo: flex con dos valores**

------------------------------------------------------------
CSS
------------------------------------------------------------
.item {
    flex: 1 100px;   /* grow: 1, shrink: 1 (por defecto), basis: 100px */
}
------------------------------------------------------------

---

🔵 **Ejemplo: flex con un valor**

------------------------------------------------------------
CSS
------------------------------------------------------------
.item {
    flex: 1;   /* grow: 1, shrink: 1, basis: 0% */
}
------------------------------------------------------------

---

📌 **Valores comunes de flex**

| `flex` | Significado |
|--------|-------------|
| `flex: 0 0 auto` | El elemento no crece ni se encoge. Tamaño según contenido. |
| `flex: 0 0 100px` | El elemento no crece ni se encoge. Tamaño fijo de 100px. |
| `flex: 1 1 auto` | El elemento crece y se encoge. Tamaño según contenido. |
| `flex: 1 1 0%` | El elemento crece y se encoge. Tamaño inicial 0%. |
| `flex: 1` | El elemento crece. Tamaño inicial 0%. |
| `flex: none` | El elemento no crece ni se encoge (equivale a `0 0 auto`). |
| `flex: initial` | Comportamiento por defecto (`0 1 auto`). |
| `flex: auto` | Equivale a `1 1 auto` (crece y se encoge). |

---

🔵 **Ejemplo: flex: 1 para elementos iguales**

------------------------------------------------------------
CSS
------------------------------------------------------------
.item {
    flex: 1;   /* Todos los elementos ocupan el mismo espacio */
}
------------------------------------------------------------

---

🔵 **Ejemplo: flex: none para elementos fijos**

------------------------------------------------------------
CSS
------------------------------------------------------------
.item-fijo {
    flex: none;   /* No crece ni se encoge */
    width: 150px;
}
------------------------------------------------------------

===============================================================================
5. ORDER: CAMBIAR EL ORDEN
===============================================================================

`order` cambia el orden de aparición de los elementos flex.

---

📌 **Sintaxis**
order: número;

text

- **Valor por defecto**: `0`.
- **Valor positivo**: el elemento aparece después.
- **Valor negativo**: el elemento aparece antes.

---

🔵 **Ejemplo: cambiar el orden**

------------------------------------------------------------
HTML
------------------------------------------------------------
<div class="contenedor">
    <div class="item item1">1 (orden: 0)</div>
    <div class="item item2">2 (orden: 1)</div>
    <div class="item item3">3 (orden: 2)</div>
</div>
------------------------------------------------------------

CSS:
------------------------------------------------------------
.item2 {
    order: -1;   /* Aparece primero */
}

.item3 {
    order: 0;    /* Orden normal */
}

.item1 {
    order: 1;    /* Aparece después */
}
------------------------------------------------------------

**Resultado:** El orden visual es: 2, 3, 1.

---

📌 **Usos de order**

1. **Mover elementos importantes al principio** (sin cambiar HTML).
2. **Crear layouts alternativos** en responsive.
3. **Cambiar el flujo de lectura** (accesibilidad).

---

⚠️ **Importante**

`order` solo cambia el orden visual, no el orden en el DOM.
Para accesibilidad, el orden en el DOM es el que cuenta.

===============================================================================
6. ALIGN-SELF: SOBRESCRIBIR LA ALINEACIÓN
===============================================================================

`align-self` sobrescribe `align-items` para un elemento específico.

---

📌 **Valores de align-self**

| Valor | Comportamiento |
|-------|----------------|
| `auto` | Usa el valor de `align-items` del contenedor. |
| `stretch` | Se estira para llenar el contenedor (por defecto). |
| `flex-start` | Se alinea al inicio del eje transversal. |
| `flex-end` | Se alinea al final del eje transversal. |
| `center` | Se centra en el eje transversal. |
| `baseline` | Se alinea por la línea base del texto. |

---

🔵 **Ejemplo: align-self individual**

------------------------------------------------------------
HTML
------------------------------------------------------------
<div class="contenedor">
    <div class="item item1">Item 1</div>
    <div class="item item2">Item 2 (centrado)</div>
    <div class="item item3">Item 3</div>
</div>
------------------------------------------------------------

CSS:
------------------------------------------------------------
.contenedor {
    display: flex;
    align-items: flex-start;   /* Todos al inicio */
    height: 150px;
    border: 2px solid #333;
}

.item {
    padding: 20px;
    background-color: #3498DB;
    color: white;
    margin: 5px;
}

.item2 {
    align-self: center;   /* Este elemento se centra */
    background-color: #28A745;
}
------------------------------------------------------------

**Resultado:** El item2 se centra verticalmente, mientras que los otros están al inicio.

---

📌 **Cuándo usar align-self**

- Para destacar un elemento.
- Para crear diseños asimétricos.
- Para ajustar un elemento específico sin afectar a los demás.

LA ANALOGÍA DEL PROFESOR BÚHO: FLEX ITEMS COMO MIEMBROS DE UNA ORQUESTA
===============================================================================

Imagina que los flex items son músicos en una orquesta.

- **`flex-grow`** es cuánto toca cada músico cuando hay tiempo extra.
- **`flex-shrink`** es cuánto se reducen cuando hay poco tiempo.
- **`flex-basis`** es su tamaño natural.
- **`flex`** es la combinación de todo.
- **`order`** es el orden en que tocan.
- **`align-self`** es la posición individual del músico en el escenario.

Cada músico tiene su propio rol, y el director (`flex-container`) los coordina.

===============================================================================
ERRORES COMUNES CON FLEX ITEMS
===============================================================================

❌ **Error 1: Usar flex-grow sin flex-basis**

   Si no defines `flex-basis`, el elemento usa `auto` (tamaño del contenido).
   **Solución:** usa `flex: 1` o define `flex-basis: 0`.

---

❌ **Error 2: Asumir que order cambia el DOM**

   `order` solo cambia el orden visual, no el orden en el DOM.
   **Solución:** tenlo en cuenta para accesibilidad.

---

❌ **Error 3: Usar `flex: none` para elementos fijos**

   `flex: none` es correcto. Asegúrate de definir `width` o `flex-basis`.

---

❌ **Error 4: Confundir `align-self` con `align-items`**

   `align-items` se aplica a todos los elementos. `align-self` solo a uno.

---

❌ **Error 5: Olvidar que los hijos directos son los únicos flex items**

   Elementos que no son hijos directos no se ven afectados por `flex-grow`, etc.

===============================================================================
PREGUNTAS FRECUENTES (FAQ)
===============================================================================

❓ **¿Cuál es la diferencia entre flex-grow y flex-shrink?**

   `flex-grow` controla cómo crece un elemento cuando hay espacio disponible.
   `flex-shrink` controla cómo se encoge un elemento cuando falta espacio.

---

❓ **¿Qué hace `flex: 1`?**

   `flex: 1` es equivalente a `flex-grow: 1; flex-shrink: 1; flex-basis: 0%;`
   El elemento crece para ocupar el espacio disponible.

---

❓ **¿Cómo hago que un elemento no se encoga?**

   Con `flex-shrink: 0;`.

---

❓ **¿Cómo cambio el orden de un elemento sin modificar el HTML?**

   Con `order: número;`.

---

❓ **¿Cuál es la diferencia entre align-self y align-items?**

   `align-items` se aplica a todos los elementos del contenedor.
   `align-self` se aplica a un elemento específico, sobrescribiendo a `align-items`.
