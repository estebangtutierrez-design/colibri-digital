PASO 3: EXPLICACIÓN DE FLEXBOX EN EL PROYECTO
===============================================================================

Vamos a repasar cómo hemos utilizado **cada propiedad de Flexbox** en este proyecto.

---

📌 **Header**
.header {
display: flex;
justify-content: space-between;
align-items: center;
flex-wrap: wrap;
gap: 1rem;
}

text

- `display: flex` → activa Flexbox.
- `justify-content: space-between` → logo a la izquierda, navegación a la derecha.
- `align-items: center` → centra verticalmente.
- `flex-wrap: wrap` → permite envolver en pantallas pequeñas.
- `gap: 1rem` → espacio entre elementos.
.nav {
display: flex;
gap: 1.5rem;
align-items: center;
}

text

- `display: flex` → navegación horizontal.
- `gap: 1.5rem` → espacio entre enlaces.
- `align-items: center` → centra verticalmente los enlaces.

---

📌 **Hero**
.hero {
display: flex;
justify-content: center;
align-items: center;
flex-direction: column;
}

text

- `display: flex` → activa Flexbox.
- `justify-content: center` → centra horizontalmente.
- `align-items: center` → centra verticalmente.
- `flex-direction: column` → apila el contenido verticalmente.
.hero-content {
display: flex;
flex-direction: column;
align-items: center;
gap: 1.5rem;
}

text

- `flex-direction: column` → apila título, párrafo y botón.
- `align-items: center` → centra horizontalmente.
- `gap: 1.5rem` → espacio entre elementos.

---

📌 **Galería**
.galeria {
display: flex;
flex-wrap: wrap;
justify-content: center;

text

- `display: flex` → activa Flexbox.

- `gap: 1.5rem` → espacio entre tarjetas.
.tarjeta {
display: flex;
flex-direction: column;
align-items: center;
flex: 0 1 auto;
}

text

- `display: flex` → activa Flexbox en la tarjeta.
- `flex-direction: column` → apila el contenido.
- `align-items: center` → centra horizontalmente.
- `flex: 0 1 auto` → no crece, se encoge si es necesario.

PREGUNTAS FRECUENTES (FAQ)
===============================================================================

❓ ¿Por qué usar Flexbox en lugar de float?

Flexbox es más moderno, más flexible y más fácil de usar.
Permite centrar elementos, distribuir espacio y controlar el orden
de forma sencilla.

❓ ¿Cómo hago que las tarjetas tengan el mismo tamaño?

Usa flex: 1 1 200px o flex: 1 en las tarjetas.
También puedes usar flex-basis para definir un tamaño base.

❓ ¿Cómo centro un elemento con Flexbox?

Con justify-content: center y align-items: center.

❓ ¿Cómo hago un diseño responsivo con Flexbox?

Usa flex-wrap: wrap para que los elementos se envuelvan,
y media queries para ajustar los tamaños en diferentes pantallas.

❓ ¿Puedo usar Flexbox en todos los navegadores?

Sí, todos los navegadores modernos soportan Flexbox.
Para navegadores muy antiguos (IE10), hay soporte limitado.
