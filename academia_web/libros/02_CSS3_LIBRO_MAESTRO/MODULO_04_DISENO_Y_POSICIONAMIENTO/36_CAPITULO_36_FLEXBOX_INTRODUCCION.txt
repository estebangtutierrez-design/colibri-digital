===============================================================================
INTRODUCCIÓN: ¿QUÉ ES FLEXBOX?
===============================================================================

Flexbox es un **modelo de layout unidimensional** que te permite organizar
elementos en filas o columnas, distribuyendo el espacio de forma flexible.

---

📌 **¿Por qué "flexible"?**

Porque los elementos se pueden:
- **Estirar** o **encoger** para llenar el espacio disponible.
- **Reordenar** sin cambiar el HTML.
- **Alinear** en cualquier dirección.
- **Distribuir** uniformemente el espacio.

---

📌 **Conceptos clave**

| Concepto | Descripción |
|----------|-------------|
| **Contenedor flex (flex container)** | El elemento que tiene `display: flex`. Es el padre. |
| **Elementos flex (flex items)** | Los hijos directos del contenedor flex. |
| **Eje principal (main axis)** | La dirección principal en la que fluyen los elementos. |
| **Eje transversal (cross axis)** | La dirección perpendicular al eje principal. |

---

📌 **Los dos ejes de Flexbox**
Eje principal (main axis)
←─────────────────────────────→
┌─────────────────────────────────┐
│ [item1] [item2] [item3] │ ← Elementos flex
│ │
│ ↑ │
│ │ Eje transversal (cross axis)│
│ ↓ │
└─────────────────────────────────┘

text

- **Eje principal**: la dirección en la que se colocan los elementos.
  - Por defecto: horizontal (izquierda → derecha).
- **Eje transversal**: la dirección perpendicular al eje principal.
  - Por defecto: vertical (arriba → abajo).

===============================================================================
1. DISPLAY: FLEX (EL CONTENEDOR FLEX)
===============================================================================

Para usar Flexbox, el primer paso es crear un **contenedor flex**
con `display: flex`.

---

🔵 **Ejemplo básico: contenedor flex**

------------------------------------------------------------
HTML
------------------------------------------------------------
<div class="contenedor-flex">
    <div class="item">Item 1</div>
    <div class="item">Item 2</div>
    <div class="item">Item 3</div>
</div>
------------------------------------------------------------

CSS:
------------------------------------------------------------
.contenedor-flex {
    display: flex;   /* Activa Flexbox */
    border: 2px solid #333;
    padding: 10px;
}

.item {
    background-color: #3498DB;
    color: white;
    padding: 20px;
    margin: 5px;
}
------------------------------------------------------------

**Resultado:** Los tres items se colocan en una fila horizontal,
uno al lado del otro, ocupando solo el espacio de su contenido.

---

📌 **Características de un contenedor flex**

1. **Los hijos directos** se convierten en elementos flex.
2. **Se alinean en una fila** horizontal (por defecto).
3. **Los elementos no se apilan verticalmente** (a menos que se indique).
4. **Los elementos se estiran** para llenar la altura del contenedor (por defecto).

---

🔵 **Comparación: con y sin Flexbox**

**Sin Flexbox (block)**
┌─────────────────────────────┐
│ [Item 1] │
│ [Item 2] │
│ [Item 3] │
└─────────────────────────────┘

text

**Con Flexbox**
┌─────────────────────────────────────┐
│ [Item 1] [Item 2] [Item 3] │
└─────────────────────────────────────┘

text

===============================================================================
2. FLEX-DIRECTION: CONTROLAR LA DIRECCIÓN
===============================================================================

`flex-direction` define la dirección en la que se colocan los elementos flex.

---

📌 **Valores de flex-direction**

| Valor | Comportamiento |
|-------|----------------|
| `row` | Horizontal (izquierda → derecha) - **valor por defecto**. |
| `row-reverse` | Horizontal inversa (derecha → izquierda). |
| `column` | Vertical (arriba → abajo). |
| `column-reverse` | Vertical inversa (abajo → arriba). |

---

🔵 **Ejemplo: flex-direction: row (por defecto)**

------------------------------------------------------------
CSS
------------------------------------------------------------
.contenedor {
    display: flex;
    flex-direction: row;   /* Por defecto */
}
------------------------------------------------------------

**Resultado:** los elementos se colocan en una fila horizontal, de izquierda a derecha.

---

🔵 **Ejemplo: flex-direction: column**

------------------------------------------------------------
CSS
------------------------------------------------------------
.contenedor {
    display: flex;
    flex-direction: column;   /* Vertical */
}
------------------------------------------------------------

**Resultado:** los elementos se colocan en una columna vertical, de arriba a abajo.

---

🔵 **Ejemplo: flex-direction: row-reverse**

------------------------------------------------------------
CSS
------------------------------------------------------------
.contenedor {
    display: flex;
    flex-direction: row-reverse;   /* Horizontal inversa */
}
------------------------------------------------------------

**Resultado:** los elementos se colocan de derecha a izquierda.

---

📌 **Eje principal y transversal según flex-direction**

| flex-direction | Eje principal | Eje transversal |
|----------------|---------------|-----------------|
| `row` | Horizontal (→) | Vertical (↓) |
| `row-reverse` | Horizontal (←) | Vertical (↓) |
| `column` | Vertical (↓) | Horizontal (→) |
| `column-reverse` | Vertical (↑) | Horizontal (→) |

===============================================================================
3. FLEX-WRAP: CONTROLAR EL ENVOLTORIO
===============================================================================

Por defecto, los elementos flex intentan caber en una sola línea.
`flex-wrap` controla si los elementos deben envolverse a múltiples líneas.

---

📌 **Valores de flex-wrap**

| Valor | Comportamiento |
|-------|----------------|
| `nowrap` | Todos los elementos en una sola línea (por defecto). |
| `wrap` | Los elementos se envuelven a múltiples líneas si es necesario. |
| `wrap-reverse` | Los elementos se envuelven en dirección inversa. |

---

🔵 **Ejemplo: flex-wrap: wrap**

------------------------------------------------------------
HTML
------------------------------------------------------------
<div class="contenedor">
    <div class="item">Item 1</div>
    <div class="item">Item 2</div>
    <div class="item">Item 3</div>
    <div class="item">Item 4</div>
    <div class="item">Item 5</div>
</div>
------------------------------------------------------------

CSS:
------------------------------------------------------------
.contenedor {
    display: flex;
    flex-wrap: wrap;   /* Permite múltiples líneas */
    width: 300px;
    border: 2px solid #333;
    padding: 10px;
}

.item {
    background-color: #3498DB;
    color: white;
    padding: 20px;
    margin: 5px;
    width: 80px;
}
------------------------------------------------------------

**Resultado:** Los elementos se colocan en varias líneas si no caben en una.

---

🔵 **Ejemplo: flex-wrap: nowrap (por defecto)**

------------------------------------------------------------
CSS
------------------------------------------------------------
.contenedor {
    display: flex;
    flex-wrap: nowrap;   /* Sin envoltorio */
    width: 300px;
}
------------------------------------------------------------

**Resultado:** Los elementos se comprimen para caber en una línea,
aunque eso signifique que se salgan del contenedor.

---

📌 **Flex-flow: shorthand**

`flex-flow` combina `flex-direction` y `flex-wrap` en una sola propiedad.

------------------------------------------------------------
CSS
------------------------------------------------------------
.contenedor {
    flex-flow: row wrap;   /* Dirección: row, Envoltorio: wrap */
}
------------------------------------------------------------

Es equivalente a:
flex-direction: row;
flex-wrap: wrap;

5. ALIGN-ITEMS: ALINEACIÓN EN EL EJE TRANSVERSAL
===============================================================================

`align-items` controla la alineación de los elementos en el **eje transversal**.

---

📌 **Valores de align-items**

| Valor | Comportamiento |
|-------|----------------|
| `stretch` | Los elementos se estiran para llenar el contenedor (por defecto). |
| `flex-start` | Alinea los elementos al inicio del eje transversal. |
| `flex-end` | Alinea los elementos al final del eje transversal. |
| `center` | Centra los elementos en el eje transversal. |
| `baseline` | Alinea los elementos por su línea base de texto. |

---

🔵 **Ejemplo: align-items: stretch (por defecto)**

------------------------------------------------------------
CSS
------------------------------------------------------------
.contenedor {
    display: flex;
    align-items: stretch;
    height: 150px;
    border: 2px solid #333;
}
------------------------------------------------------------

**Resultado:** los elementos se estiran verticalmente para llenar el contenedor.

---

🔵 **Ejemplo: align-items: center**

------------------------------------------------------------
CSS
------------------------------------------------------------
.contenedor {
    display: flex;
    align-items: center;
    height: 150px;
}
------------------------------------------------------------

**Resultado:** los elementos se centran verticalmente en el contenedor.

📌 **¿Qué es baseline?**

Alinea los elementos por la línea base del texto (la línea donde "se sientan" las letras).
Es útil cuando tienes textos de diferentes tamaños en el mismo contenedor.

===============================================================================
6. ALIGN-CONTENT: ALINEACIÓN DE LÍNEAS MÚLTIPLES
===============================================================================

`align-content` controla la alineación de **líneas múltiples** de elementos flex
(es decir, cuando `flex-wrap: wrap` está activo).

---

📌 **Valores de align-content**

| Valor | Comportamiento |
|-------|----------------|
| `stretch` | Las líneas se estiran para llenar el espacio (por defecto). |
| `flex-start` | Alinea las líneas al inicio. |
| `flex-end` | Alinea las líneas al final. |
| `center` | Centra las líneas. |
| `space-between` | Distribuye líneas con espacio entre ellas. |
| `space-around` | Distribuye líneas con espacio alrededor. |

---

🔵 **Ejemplo: align-content: center con wrap**

------------------------------------------------------------
CSS
------------------------------------------------------------
.contenedor {
    display: flex;
    flex-wrap: wrap;
    align-content: center;
    height: 300px;
    width: 300px;
}
------------------------------------------------------------

**Resultado:** las líneas de elementos se centran verticalmente en el contenedor.

---

📌 **Diferencia entre align-items y align-content**

| Propiedad | Cuándo usar |
|-----------|-------------|
| `align-items` | Cuando hay **una sola línea** de elementos. |
| `align-content` | Cuando hay **múltiples líneas** de elementos (con `flex-wrap: wrap`). |

8. EJEMPLO PRÁCTICO: CENTRADO PERFECTO
===============================================================================

Una de las tareas más comunes que Flexbox hace fácil es el centrado.

---

🔵 **Centrar un elemento vertical y horizontalmente**

------------------------------------------------------------
HTML
------------------------------------------------------------
<div class="contenedor">
    <div class="centrado">Centrado</div>
</div>
------------------------------------------------------------

CSS:
------------------------------------------------------------
.contenedor {
    display: flex;
    justify-content: center;   /* Centra horizontalmente */
    align-items: center;       /* Centra verticalmente */
    height: 300px;
    border: 2px solid #333;
}

.centrado {
    background-color: #3498DB;
    color: white;
    padding: 20px;
    border-radius: 8px;
}
------------------------------------------------------------

**Resultado:** el elemento está perfectamente centrado en ambos ejes.

---

🔵 **Centrar con Flexbox vs otros métodos**

| Método | Complejidad |
|--------|-------------|
| **Flexbox** | Muy fácil (2 líneas) |
| `position: absolute` + `transform` | Medio (más código) |
| `margin: auto` con `position: absolute` | Medio |
| Tablas | Complicado |

---

📌 **Conclusión**

Flexbox es la forma más sencilla y moderna de centrar elementos.

===============================================================================
LA ANALOGÍA DEL PROFESOR BÚHO: FLEXBOX COMO UNA GOMITA ELÁSTICA
===============================================================================

Imagina que los elementos flex son cuentas en una gomita elástica.

- La **gomita** es el contenedor flex (`display: flex`).
- La **dirección** de la gomita es `flex-direction`.
- Si la gomita es horizontal (`row`), las cuentas se alinean en fila.
- Si la gomita es vertical (`column`), las cuentas se alinean en columna.
- **`justify-content`** es cómo distribuyes las cuentas a lo largo de la gomita
  (juntas, separadas, centradas, etc.).
- **`align-items`** es cómo alineas las cuentas en el grosor de la gomita
  (arriba, abajo, centradas).
- **`flex-wrap`** es si permites que la gomita se doble en varias líneas
  o si las cuentas deben caber en una sola.

Flexbox es elástico y se adapta a tus necesidades.

===============================================================================
ERRORES COMUNES CON FLEXBOX
===============================================================================

❌ **Error 1: No aplicar `display: flex` al contenedor**

   Si no defines `display: flex` en el contenedor, los hijos no se comportarán como flex items.

---

❌ **Error 2: Olvidar que solo los hijos directos son flex items**

   Los elementos que no son hijos directos no se ven afectados por Flexbox.

---

❌ **Error 3: Usar `align-content` con una sola línea**

   `align-content` solo funciona cuando hay múltiples líneas (`flex-wrap: wrap`).

---

❌ **Error 4: Confundir `justify-content` con `align-items`**

   `justify-content` alinea en el **eje principal**.
   `align-items` alinea en el **eje transversal**.

---

❌ **Error 5: No definir `width` o `flex-basis` en elementos flex**

   Si no defines tamaños, los elementos se ajustarán según el contenido.

GLOSARIO DEL CAPÍTULO
===============================================================================

- **Flexbox**: Modelo de layout unidimensional para organizar elementos.
- **Contenedor flex**: Elemento con `display: flex`.
- **Elementos flex**: Hijos directos del contenedor flex.
- **Eje principal (main axis)**: Dirección principal (horizontal o vertical).
- **Eje transversal (cross axis)**: Dirección perpendicular al eje principal.
- **flex-direction**: Controla la dirección del eje principal.
- **flex-wrap**: Controla si los elementos se envuelven a múltiples líneas.
- **justify-content**: Alineación en el eje principal.
- **align-items**: Alineación en el eje transversal.
- **align-content**: Alineación de líneas múltiples.

===============================================================================
PREGUNTAS FRECUENTES (FAQ)
===============================================================================

❓ **¿Qué es Flexbox?**

   Es un modelo de layout que permite organizar elementos de forma flexible
   en una dirección (fila o columna).

---

❓ **¿Cuál es la diferencia entre justify-content y align-items?**

   `justify-content` alinea en el **eje principal**.
   `align-items` alinea en el **eje transversal**.

---

❓ **¿Cómo centro un elemento con Flexbox?**

   Con `justify-content: center` y `align-items: center`.

---

❓ **¿Cómo hago que los elementos se envuelvan a varias líneas?**

   Con `flex-wrap: wrap`.

---

❓ **¿Flexbox funciona en todos los navegadores?**

   Sí, en todos los navegadores modernos.
   Para navegadores antiguos (IE10), hay soporte limitado.
