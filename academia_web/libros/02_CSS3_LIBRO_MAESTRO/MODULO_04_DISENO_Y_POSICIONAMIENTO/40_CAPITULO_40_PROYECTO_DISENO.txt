EL PROYECTO: PORTAFOLIO PROFESIONAL
===============================================================================

Vamos a construir un **portafolio profesional** que incluye:

- **Header con navegación fija** → `position: fixed` + `z-index`.
- **Hero con texto superpuesto** → `position: absolute` sobre imagen de fondo.
- **Sección "Sobre mí"** → `float` para imagen con texto envuelto.
- **Sección "Proyectos"** → Flexbox para la galería de tarjetas.
- **Sección "Testimonios"** → Flexbox para testimonios.
- **Footer** → Flexbox para distribución.
- **Menú móvil** → `display: none` y `display: flex` con media query.

PASO 3: EXPLICACIÓN DE LOS CONCEPTOS APLICADOS
===============================================================================

Vamos a repasar cómo hemos utilizado **cada concepto** del Módulo IV en este proyecto.

---

📌 **1. `position: fixed` + `z-index` (Header)**
.header {
position: fixed;
top: 0;
left: 0;
width: 100%;
z-index: 1000;
}

text

- **`position: fixed`** → el header siempre está visible al hacer scroll.
- **`z-index: 1000`** → asegura que el header esté por encima de todo el contenido.

---

📌 **2. `position: relative` + `position: absolute` + `z-index` (Hero)**
.hero {
position: relative;
}

.hero-overlay {
position: absolute;
z-index: 1;
}

.hero-content {
position: relative;
z-index: 2;
}

text

- **`position: relative`** en el hero → contenedor de referencia.
- **`position: absolute`** en el overlay → cubre toda la sección.
- **`z-index: 1`** en el overlay → sobre la imagen de fondo.
- **`z-index: 2`** en el contenido → sobre el overlay.

---

📌 **3. `float: left` + `clear` (Sobre mí)**
.sobre-mi__imagen {
float: left;
margin-right: 1.5rem;
}

.sobre-mi__contenido::after {
content: '';
display: table;
clear: both;
}

text

- **`float: left`** → la imagen flota a la izquierda.
- **El texto fluye a su alrededor**.
- **`clear: both`** en el contenedor → evita que el float afecte a otros elementos.

---

📌 **4. Flexbox (Proyectos, Testimonios, Footer, Header)**
.galeria {
display: flex;
flex-wrap: wrap;
justify-content: center;
gap: 2rem;
}

text

- **`display: flex`** → activa Flexbox.
- **`flex-wrap: wrap`** → permite envolver a múltiples líneas.
- **`justify-content: center`** → centra las tarjetas.
- **`gap: 2rem`** → espacio entre tarjetas.

---

📌 **5. `display: none` y `display: flex` (Menú móvil)**
.nav {
display: none; /* Oculto en móvil */
}

.nav.abierto {
display: flex; /* Visible al hacer click */
}

text

- **`display: none`** → oculta el menú en móvil.
- **`display: flex`** → lo muestra al hacer click.

===============================================================================
PREGUNTAS FRECUENTES (FAQ)
===============================================================================

❓ **¿Por qué usar `position: fixed` en el header?**

   Para que el header siempre esté visible al hacer scroll,
   mejorando la navegación.

---

❓ **¿Cómo funciona el efecto de superposición en el hero?**

   Usamos `position: relative` en el contenedor y `position: absolute`
   en el overlay y en la imagen de fondo, con `z-index` para controlar
   qué capa está encima.

---

❓ **¿Por qué usar `float` en lugar de Flexbox para la imagen?**

   `float` es perfecto para envolver texto alrededor de una imagen,
   que es su propósito original.

---

❓ **¿Cómo hago que el menú se abra en móvil?**

   Usamos `display: none` para ocultarlo y `display: flex` para mostrarlo
   al hacer click, con `classList.toggle()` en JavaScript.

---

❓ **¿Puedo usar este proyecto en mi portafolio real?**

   ¡Sí! Personaliza el contenido, los colores y las imágenes,
   y será un excelente proyecto para mostrar tus habilidades.
