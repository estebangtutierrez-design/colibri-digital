===============================================================================
INTRODUCCIÓN: ¿QUÉ ES FLOAT?
===============================================================================

`float` es una propiedad CSS que hace que un elemento "flote" a la izquierda
o a la derecha de su contenedor, permitiendo que el contenido (como texto)
fluya a su alrededor.

---

📌 **Valores de float**

| Valor | Comportamiento |
|-------|----------------|
| `left` | El elemento flota a la izquierda. El contenido fluye a su derecha. |
| `right` | El elemento flota a la derecha. El contenido fluye a su izquierda. |
| `none` | El elemento no flota (valor por defecto). |

---

📌 **Características de float**

1. **El elemento sale del flujo normal** (parcialmente).
2. **Los elementos en línea** (como texto) fluyen alrededor del elemento flotante.
3. **Los elementos de bloque** se colocan debajo del elemento flotante (no fluyen).
4. **El elemento flotante se convierte en un "bloque"** (acepta width y height).
5. **Los elementos flotantes se alinean horizontalmente** (uno al lado del otro).

---

📌 **Propiedades relacionadas**

- `clear` → limpia el flotado (evita que los elementos floten alrededor).
- `float: none` → valor por defecto.

---

🟢 **Regla de oro**

**`float` se usa para que el contenido fluya alrededor de un elemento.**

Originalmente se creó para envolver texto alrededor de imágenes en artículos
y periódicos. Ese sigue siendo su mejor uso hoy en día.

===============================================================================
1. FLOAT: LEFT (FLOTAR A LA IZQUIERDA)
===============================================================================

`float: left` hace que el elemento flote a la izquierda.
El contenido (texto, imágenes, etc.) fluye a su derecha.

---

🔵 **Ejemplo: imagen con texto envuelto a la derecha**

------------------------------------------------------------
HTML
------------------------------------------------------------
<div class="articulo">
    <img src="imagen.jpg" alt="Imagen de ejemplo" class="imagen-flotante">
    <p>Este es un texto que fluye alrededor de la imagen. La imagen flota a la izquierda,
    y el texto se coloca a su derecha, envolviéndola de forma natural.</p>
</div>
------------------------------------------------------------

CSS:
------------------------------------------------------------
.imagen-flotante {
    float: left;
    width: 200px;
    margin-right: 20px;
    margin-bottom: 10px;
    border-radius: 8px;
}
------------------------------------------------------------

**Resultado:** La imagen flota a la izquierda y el texto fluye a su alrededor.

---

📌 **Características de float: left**

1. El elemento se mueve a la izquierda de su contenedor.
2. El contenido que le sigue fluye a su derecha.
3. Si hay varios elementos flotantes, se alinean horizontalmente.

---

🔵 **Ejemplo: varios elementos flotantes en fila**

------------------------------------------------------------
HTML
------------------------------------------------------------
<div class="contenedor">
    <div class="caja">Caja 1</div>
    <div class="caja">Caja 2</div>
    <div class="caja">Caja 3</div>
</div>
------------------------------------------------------------

CSS:
------------------------------------------------------------
.caja {
    float: left;
    width: 30%;
    margin: 1%;
    padding: 20px;
    background-color: #3498DB;
    color: white;
    border-radius: 8px;
    box-sizing: border-box;
}
------------------------------------------------------------

**Resultado:** Tres cajas se alinean horizontalmente.

===============================================================================
2. FLOAT: RIGHT (FLOTAR A LA DERECHA)
===============================================================================

`float: right` hace que el elemento flote a la derecha.
El contenido fluye a su izquierda.

---

🔵 **Ejemplo: imagen flotando a la derecha**

------------------------------------------------------------
CSS
------------------------------------------------------------
.imagen-flotante {
    float: right;
    width: 200px;
    margin-left: 20px;
    margin-bottom: 10px;
    border-radius: 8px;
}
------------------------------------------------------------

**Resultado:** La imagen flota a la derecha y el texto fluye a su izquierda.

---

📌 **Cuándo usar left vs right**

| Situación | float |
|-----------|-------|
| Imagen a la izquierda del texto | `float: left` |
| Imagen a la derecha del texto | `float: right` |
| Múltiples elementos en fila | `float: left` (generalmente) |
| Elemento decorativo a la derecha | `float: right` |

===============================================================================
3. FLOAT: NONE (SIN FLOTADO)
===============================================================================

`float: none` es el valor por defecto. El elemento no flota.

---

🔵 **Ejemplo**

------------------------------------------------------------
CSS
------------------------------------------------------------
.imagen {
    float: none;   /* Comportamiento normal */
}
------------------------------------------------------------

Es útil para **resetear** un float en un elemento específico,
o para sobreescribir un float heredado.

===============================================================================
4. CLEAR: LIMPIAR FLOATS
===============================================================================

Cuando un elemento flota, los elementos de bloque que le siguen
se colocan debajo de él (pero el contenido en línea fluye alrededor).

`clear` evita que un elemento se coloque al lado de un elemento flotante.

---

📌 **Valores de clear**

| Valor | Comportamiento |
|-------|----------------|
| `left` | Evita que el elemento se coloque al lado de floats a la izquierda |
| `right` | Evita que el elemento se coloque al lado de floats a la derecha |
| `both` | Evita que el elemento se coloque al lado de floats en ambos lados |
| `none` | Permite que el elemento se coloque al lado de floats (por defecto) |

---

🔵 **Ejemplo: clear en un párrafo**

------------------------------------------------------------
HTML
------------------------------------------------------------
<div class="caja-flotante">Flotante</div>
<p class="texto-normal">Este párrafo no usa clear.</p>
<p class="texto-limpiado">Este párrafo usa clear: both.</p>
------------------------------------------------------------

CSS:
------------------------------------------------------------
.caja-flotante {
    float: left;
    width: 150px;
    height: 100px;
    background-color: #3498DB;
    color: white;
    padding: 10px;
    margin-right: 15px;
}

.texto-normal {
    /* Este párrafo fluye alrededor del flotante */
}

.texto-limpiado {
    clear: both;   /* Este párrafo se coloca debajo del flotante */
    background-color: #f0f0f0;
    padding: 10px;
}
------------------------------------------------------------

**Resultado:** El primer párrafo fluye alrededor del flotante.
El segundo párrafo se coloca debajo del flotante.

---

📌 **Usos comunes de clear**

1. **Evitar que un párrafo fluya alrededor de una imagen**.
2. **Limpiar floats en un contenedor** para que no colapse.
3. **Crear separación después de un elemento flotante**.

===============================================================================
5. EL PROBLEMA DEL CONTENEDOR QUE COLAPSA
===============================================================================

Uno de los problemas más comunes con `float` es que **el contenedor colapsa**.

---

📌 **¿Qué significa "colapsar"?**

Cuando un contenedor solo contiene elementos flotantes,
su altura se reduce a 0 porque los elementos flotantes salen del flujo.

---

🔵 **Ejemplo del problema**

------------------------------------------------------------
HTML
------------------------------------------------------------
<div class="contenedor">
    <div class="caja">Caja 1</div>
    <div class="caja">Caja 2</div>
    <div class="caja">Caja 3</div>
</div>
<p>Este párrafo está después del contenedor.</p>
------------------------------------------------------------

CSS:
------------------------------------------------------------
.contenedor {
    background-color: #f0f0f0;
    border: 2px solid #333;
}

.caja {
    float: left;
    width: 30%;
    padding: 20px;
    background-color: #3498DB;
    color: white;
    margin: 1%;
    box-sizing: border-box;
}
------------------------------------------------------------

**Resultado:** El contenedor tiene altura 0 (no se ve el fondo ni el borde).
El párrafo se coloca directamente debajo de las cajas flotantes.

---

📌 **Soluciones para el contenedor colapsado**

1. **Añadir un elemento con `clear: both`** después de los flotantes.
2. **Usar `overflow: hidden`** en el contenedor.
3. **Usar `overflow: auto`** en el contenedor.
4. **Usar la técnica del clearfix** (la más robusta).

---

🔵 **Solución 1: Elemento con clear**

------------------------------------------------------------
HTML
------------------------------------------------------------
<div class="contenedor">
    <div class="caja">Caja 1</div>
    <div class="caja">Caja 2</div>
    <div class="caja">Caja 3</div>
    <div style="clear: both;"></div>   <!-- Elemento de limpieza -->
</div>
------------------------------------------------------------

**Desventaja:** añade un elemento HTML extra.

---

🔵 **Solución 2: overflow: hidden**

------------------------------------------------------------
CSS
------------------------------------------------------------
.contenedor {
    overflow: hidden;   /* Limpia los floats */
    background-color: #f0f0f0;
    border: 2px solid #333;
}
------------------------------------------------------------

**Ventaja:** simple y efectivo.
**Desventaja:** puede recortar contenido si hay desbordamiento.

---

🔵 **Solución 3: overflow: auto**

------------------------------------------------------------
CSS
------------------------------------------------------------
.contenedor {
    overflow: auto;   /* Limpia los floats */
    background-color: #f0f0f0;
    border: 2px solid #333;
}
------------------------------------------------------------

**Ventaja:** similar a hidden, pero muestra scroll si es necesario.

---

🔵 **Solución 4: Técnica del clearfix (la más robusta)**

------------------------------------------------------------
CSS
------------------------------------------------------------
.clearfix::after {
    content: "";
    display: table;
    clear: both;
}
------------------------------------------------------------

HTML:
------------------------------------------------------------
<div class="contenedor clearfix">
    <div class="caja">Caja 1</div>
    <div class="caja">Caja 2</div>
    <div class="caja">Caja 3</div>
</div>
------------------------------------------------------------

**Ventaja:** no añade HTML extra, es robusta y funciona en todos los navegadores.
**Desventaja:** requiere añadir una clase.

===============================================================================
6. LA TÉCNICA DEL CLEARFIX (MICRO CLEARFIX)
===============================================================================

El **clearfix** es una técnica CSS que limpia los floats sin añadir HTML extra.

---

📌 **El micro clearfix moderno**

------------------------------------------------------------
CSS
------------------------------------------------------------
.clearfix::after {
    content: "";
    display: table;
    clear: both;
}

/* Versión para navegadores antiguos (opcional) */
.clearfix {
    zoom: 1;   /* Activa el hasLayout en IE */
}
------------------------------------------------------------

---

📌 **Cómo funciona**

1. `::after` crea un pseudo-elemento después del contenido.
2. `display: table` hace que el pseudo-elemento se comporte como un bloque.
3. `clear: both` limpia los floats.
4. El pseudo-elemento es invisible (`content: ""`).

---

🔵 **Ejemplo de uso del clearfix**

------------------------------------------------------------
HTML
------------------------------------------------------------
<div class="contenedor clearfix">
    <div class="caja">Caja 1</div>
    <div class="caja">Caja 2</div>
    <div class="caja">Caja 3</div>
</div>
------------------------------------------------------------

CSS:
------------------------------------------------------------
.contenedor {
    background-color: #f0f0f0;
    border: 2px solid #333;
}

.caja {
    float: left;
    width: 30%;
    padding: 20px;
    background-color: #3498DB;
    color: white;
    margin: 1%;
    box-sizing: border-box;
}

.clearfix::after {
    content: "";
    display: table;
    clear: both;
}
------------------------------------------------------------

**Resultado:** el contenedor tiene altura y el fondo y borde son visibles.

===============================================================================
7. FLOAT Y LAYOUTS MODERNOS
===============================================================================

Hoy en día, `float` ya no se recomienda para layouts completos.

---

📌 **Alternativas modernas a float**

| Para... | Usa... |
|---------|--------|
| Layouts complejos | **Flexbox** o **Grid** |
| Alinear elementos en fila | **Flexbox** |
| Columnas | **Flexbox** o **Grid** |
| Centrar elementos | **Flexbox** |
| Envolver texto alrededor de imágenes | **Float** (¡sigue siendo válido!) |

---

📌 **¿Cuándo usar float hoy?**

1. **Envolver texto alrededor de imágenes** (su propósito original).
2. **Diseños muy simples** (donde Flexbox sería excesivo).
3. **Compatibilidad con navegadores muy antiguos** (IE6-9).

---

📌 **¿Cuándo NO usar float?**

1. **Layouts completos** (usa Flexbox o Grid).
2. **Columnas** (usa Flexbox o Grid).
3. **Centrado** (usa Flexbox).
4. **Alineación vertical** (usa Flexbox).

---

🔵 **Ejemplo: float sigue siendo útil para imágenes en artículos**

------------------------------------------------------------
CSS
------------------------------------------------------------
.articulo img {
    float: left;
    margin: 0 20px 10px 0;
    border-radius: 8px;
    max-width: 300px;
}
------------------------------------------------------------

**Este es el uso más válido y recomendado de `float` hoy en día.**

===============================================================================
LA ANALOGÍA DEL PROFESOR BÚHO: FLOAT COMO UNA PIEDRA EN UN RÍO
===============================================================================

Imagina que el contenido de tu página es un río.

- El **float** es una piedra que colocas en el río.
- El agua (el texto) fluye a su alrededor.
- La piedra no se mueve, pero el agua se adapta.

Si pones una piedra a la izquierda, el agua fluye por la derecha.
Si pones una piedra a la derecha, el agua fluye por la izquierda.

Eso es exactamente lo que hace `float`.

===============================================================================
ERRORES COMUNES CON FLOAT
===============================================================================

❌ **Error 1: Olvidar limpiar floats**

   Los elementos que siguen a un float pueden comportarse de forma inesperada.
   **Solución:** usa `clear: both` o la técnica del clearfix.

---

❌ **Error 2: El contenedor colapsa**

   Si todos los hijos flotan, el contenedor no tiene altura.
   **Solución:** usa overflow: hidden, overflow: auto o clearfix.

---

❌ **Error 3: Usar float para layouts completos**

   Hoy en día, Float no es la mejor opción para layouts.
   **Solución:** usa Flexbox o Grid.

---

❌ **Error 4: No definir un ancho en elementos flotantes**

   Los elementos flotantes sin un ancho definido pueden comportarse de forma extraña.
   **Solución:** siempre define un `width` en elementos flotantes.

---

❌ **Error 5: Usar float en elementos que deberían estar en el flujo**

   `float` saca elementos del flujo parcialmente.
   **Solución:** si no quieres que salga del flujo, no uses float.

GLOSARIO DEL CAPÍTULO
===============================================================================

- **Float**: Propiedad CSS que hace que un elemento flote y el contenido fluya a su alrededor.
- **Float: left**: El elemento flota a la izquierda.
- **Float: right**: El elemento flota a la derecha.
- **Float: none**: El elemento no flota (por defecto).
- **Clear**: Propiedad que evita que un elemento se coloque al lado de floats.
- **Clear: both**: Evita que el elemento se coloque al lado de floats en ambos lados.
- **Clearfix**: Técnica para limpiar floats sin añadir HTML extra.
- **Contenedor colapsado**: Cuando un contenedor con floats no tiene altura.
- **Flujo normal**: Cómo los elementos se colocan naturalmente en la página.

===============================================================================
PREGUNTAS FRECUENTES (FAQ)
===============================================================================

❓ **¿Qué es float en CSS?**

   `float` es una propiedad que hace que un elemento flote a la izquierda o derecha,
   permitiendo que el contenido fluya a su alrededor.

---

❓ **¿Cuándo usar float?**

   Se usa para envolver texto alrededor de imágenes o para layouts simples.
   Para layouts complejos, usa Flexbox o Grid.

---

❓ **¿Qué es el contenedor colapsado?**

   Ocurre cuando un contenedor solo contiene elementos flotantes y su altura se reduce a 0.
   Se soluciona con clearfix o overflow.

---

❓ **¿Qué es el clearfix?**

   Es una técnica CSS que limpia los floats sin añadir HTML extra.
   Usa `::after` con `clear: both`.

---

❓ **¿Float está obsoleto?**

   No, pero su uso se ha reducido. Hoy en día se usa principalmente para envolver texto
   alrededor de imágenes. Para layouts, se prefiere Flexbox o Grid.
