"Transform te permite mover, rotar, escalar e inclinar elementos.
Sin cambiar el flujo del documento. Sin afectar a los demás."
                         — Profesor Búho AI

===============================================================================
BIENVENIDA
===============================================================================

transform modifica la apariencia de un elemento mediante funciones
matemáticas: translate (mover), scale (escalar), rotate (rotar),
skew (inclinar). No afecta al layout ni a los elementos vecinos.

===============================================================================
OBJETIVOS
===============================================================================

✓ Usar translate para mover elementos.
✓ Usar scale para cambiar tamaño.
✓ Usar rotate para rotar.
✓ Usar skew para inclinar.
✓ Combinar múltiples transformaciones.
✓ Entender que transform no afecta al flujo.

===============================================================================
SINTAXIS
===============================================================================

/* Trasladar */
transform: translateX(50px);
transform: translateY(30px);
transform: translate(50px, 30px);

/* Escalar */
transform: scale(1.5);
transform: scaleX(2);
transform: scaleY(0.5);

/* Rotar */
transform: rotate(45deg);
transform: rotate(-90deg);

/* Inclinar */
transform: skewX(10deg);
transform: skewY(10deg);
transform: skew(10deg, 5deg);

/* Combinadas */
transform: translateX(20px) rotate(45deg) scale(1.2);

=======================================================================
TRANSFORM 3D
===============================================================================

translateZ, rotateX, rotateY, rotateZ, scaleZ, perspective.

.caja { transform: perspective(500px) rotateX(30deg); }

perspective(n): define la profundidad 3D.
rotateX: rota sobre el eje X.
rotateY: rota sobre el eje Y.
rotateZ: igual que rotate (eje Z).

=======================================================================
GLOSARIO
===============================================================================

translate: Mueve en el eje X, Y o ambos.
scale: Cambia el tamaño.
rotate: Rota en grados.
skew: Inclina el elemento.
transform-origin: Punto desde donde se aplica la transformación.
