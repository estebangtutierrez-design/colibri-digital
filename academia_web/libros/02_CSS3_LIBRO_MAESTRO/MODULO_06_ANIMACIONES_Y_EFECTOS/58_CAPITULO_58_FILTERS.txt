"Los filtros CSS son como los de Instagram: blur, brillo, contraste,
escala de grises. Todo sin JavaScript, todo sin imágenes."
                         — Profesor Búho AI

===============================================================================
OBJETIVOS
===============================================================================

✓ Usar blur, brightness, contrast, grayscale, sepia.
✓ Usar saturate, hue-rotate, invert, drop-shadow.
✓ Combinar múltiples filtros.
✓ Animar filtros con transiciones.

===============================================================================
SINTAXIS
===============================================================================

filter: blur(5px);
filter: brightness(0.5);
filter: contrast(200%);
filter: grayscale(100%);
filter: sepia(100%);
filter: saturate(200%);
filter: hue-rotate(90deg);
filter: invert(100%);
filter: opacity(50%);
filter: drop-shadow(5px 5px 5px rgba(0,0,0,0.5));
filter: brightness(1.2) contrast(1.1) saturate(1.3);

CÓDIGO PRÁCTICO
===============================================================================

<style>

.f-blur:hover img { filter: blur(4px); }
.f-bright:hover img { filter: brightness(1.5); }
.f-gray:hover img { filter: grayscale(100%); }
.f-sepia:hover img { filter: sepia(100%); }

.f-invert:hover img { filter: invert(100%); }

/* Drop shadow vs box shadow */
.drop-demo {
    width: 80px; height: 80px; background: #3498db;
    border-radius: 50%;
    filter: drop-shadow(4px 4px 8px rgba(0,0,0,0.3));
}
.box-demo {
    width: 80px; height: 80px; background: #3498db;
    border-radius: 50%;
    box-shadow: 4px 4px 8px rgba(0,0,0,0.3);
}
</style>
