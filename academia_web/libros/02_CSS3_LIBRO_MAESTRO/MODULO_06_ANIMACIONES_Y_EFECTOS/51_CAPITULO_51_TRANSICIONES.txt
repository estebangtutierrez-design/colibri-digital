"Las transiciones son el puente entre un estado y otro. Hacen que
los cambios no sean instantáneos, sino suaves y agradables."
                         — Profesor Búho AI

===============================================================================
BIENVENIDA
===============================================================================

Una transición permite que un valor CSS cambie gradualmente de un
estado a otro. Sin transición, los cambios son instantáneos. Con
transición, se animan suavemente.

Propiedades: transition-property, transition-duration,
transition-timing-function, transition-delay.

Atajo: transition: all 0.3s ease;

===============================================================================
OBJETIVOS
===============================================================================

✓ Aplicar transiciones a cambios de estado (:hover, :focus, :active).
✓ Controlar duración, timing y delay.
✓ Transicionar propiedades específicas.
✓ Crear efectos suaves en botones, cards y enlaces.

===============================================================================
SINTAXIS
===============================================================================

/* Atajo */
.elemento { transition: all 0.3s ease; }

/* Propiedades individuales */
.elemento {
    transition-property: background-color, transform;
    transition-duration: 0.3s;
    transition-timing-function: ease;
    transition-delay: 0s;
}

/* Transiciones diferentes para distintas propiedades */
.elemento {
    transition: background 0.3s ease, transform 0.5s ease-out;
}

===============================================================================
TIMING FUNCTIONS
===============================================================================

ease → lento-rápido-lento (por defecto).
linear → velocidad constante.
ease-in → lento al inicio.
ease-out → lento al final.
ease-in-out → lento al inicio y final.
cubic-bezier(n,n,n,n) → curva personalizada.

/* Transición simple */
.t1 .box { transition: background 0.3s; }
.t1:hover .box { background: #e74c3c; }

/* Transición con transform */
.t2 .box { transition: transform 0.3s; }
.t2:hover .box { transform: scale(1.2); }

.t3 .box {
    transition: background 0.3s, transform 0.3s, border-radius 0.3s;

    background: #2ecc71; transform: rotate(45deg); border-radius: 50%;

.t4 .box { transition: all 0.3s ease 0.2s; }

    transition: transform 1s;

=======================================================================
PROPIEDADES TRANSICIONABLES
=======================================================================

No todas las propiedades se pueden transicionar. Solo aquellas
que tienen valores numéricos o de color.

Sí se pueden transicionar:
- color, background-color, border-color
- width, height, padding, margin
- transform, opacity, filter
- box-shadow, font-size, border-radius
- top, right, bottom, left

NO se pueden transicionar:
- display (block → none es instantáneo)
- visibility (pero opacity sí)
- font-family
- propiedades no numéricas

=======================================================================
EJERCICIOS
===============================================================================

1. Botón que cambie de color y se agrande al hacer hover.
2. Card que se eleve (translateY negativo) con sombra más grande.
3. Imagen que se oscurezca (opacity) al pasar el mouse.
4. Menú donde los links cambien de color con transición suave.

=======================================================================
GLOSARIO
===============================================================================

transition: Atajo para animar cambios de estado.
transition-property: Qué propiedad animar.
transition-duration: Duración de la animación.
transition-timing-function: Curva de velocidad.
transition-delay: Retraso antes de empezar.
ease: Timing por defecto (lento-rápido-lento).
cubic-bezier(): Curva de velocidad personalizada.
