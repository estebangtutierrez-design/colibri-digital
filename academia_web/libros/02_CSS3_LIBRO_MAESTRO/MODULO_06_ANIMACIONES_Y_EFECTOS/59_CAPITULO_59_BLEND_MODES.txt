"Fusionar capas es una técnica clásica del diseño gráfico.
Con CSS puedes mezclar fondos, imágenes y colores usando
modos de fusión como multiplicar, superponer o pantalla."
                         — Profesor Búho AI

===============================================================================
OBJETIVOS
===============================================================================

✓ Usar background-blend-mode para mezclar fondos.
✓ Usar mix-blend-mode para mezclar elementos con su fondo.
✓ Crear efectos de texto sobre imagen.
✓ Usar modos: multiply, screen, overlay, difference.

===============================================================================
SINTAXIS
===============================================================================

/* Mezclar fondos del mismo elemento */
.elemento {
    background: url(imagen.jpg), linear-gradient(red, blue);
    background-blend-mode: multiply;
}

/* Mezclar elemento con fondo detrás */
.elemento { mix-blend-mode: overlay; }

Modos: normal, multiply, screen, overlay, darken, lighten,
color-dodge, hard-light, soft-light, difference, exclusion.
