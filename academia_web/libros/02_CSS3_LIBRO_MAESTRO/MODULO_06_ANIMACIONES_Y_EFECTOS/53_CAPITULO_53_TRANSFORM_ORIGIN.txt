"El origen define desde dónde se transforma un elemento.
Un mismo scale o rotate se ve completamente diferente según
el origen que elijas."
                         — Profesor Búho AI

===============================================================================
OBJETIVOS
===============================================================================

✓ Usar transform-origin con palabras clave, porcentajes y px.
✓ Aplicar transformaciones desde diferentes puntos.
✓ Crear efectos de acordeón, péndulo y tarjeta giratoria.

===============================================================================
SINTAXIS
===============================================================================

transform-origin: center;          /* 50% 50% (por defecto) */
transform-origin: top left;
transform-origin: top right;
transform-origin: bottom left;
transform-origin: bottom right;
transform-origin: 0% 0%;
transform-origin: 100% 50%;
transform-origin: 20px 30px;

===============================================================================
CÓDIGO PRÁCTICO
===============================================================================

<style>
* { margin: 0; padding: 0; box-sizing: border-box; }
body { font-family: Arial; background: #f0f2f5; padding: 40px; }
h1 { margin-bottom: 20px; }
h2 { margin: 25px 0 10px; font-size: 1.1rem; }

.demo { display: flex; gap: 25px; flex-wrap: wrap; margin-bottom: 30px; }

.card { width: 130px; text-align: center; cursor: pointer; }

.box {
    width: 100px; height: 100px; background: #3498db;
    border-radius: 6px; margin: 0 auto 8px;
    transition: transform 0.4s;
}

span { font-size: 12px; color: #888; }

.center:hover .box { transform: scale(1.3); transform-origin: center; background: #e74c3c; }
.top-left:hover .box { transform: scale(1.3); transform-origin: top left; background: #2ecc71; }
.top-right:hover .box { transform: scale(1.3); transform-origin: top right; background: #f39c12; }
.bottom:hover .box { transform: scale(1.3); transform-origin: bottom center; background: #9b59b6; }

/* Efecto puerta 3D */
.puerta {
    width: 80px; height: 120px; background: #8B4513;
    border-radius: 4px; margin: 0 auto;
    transition: transform 1s;
    transform-origin: left center;
}
.puerta:hover { transform: rotateY(60deg); }
</style>

<h1>Transform Origin</h1>

    <div class="card top-right"><div class="box"></div><span>top right</span></div>
    <div class="card bottom"><div class="box"></div><span>bottom</span></div>
</div>

<h2>Efecto puerta (3D)</h2>
<div class="card"><div class="puerta"></div><span>hover — se abre</span></div>
