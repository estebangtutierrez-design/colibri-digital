SECCIÓN I: FUNDAMENTOS (25 PUNTOS)
===============================================================================

  1. ¿Qué significa CSS y cuál es su propósito?

  2. Escribe 3 formas de incluir CSS en un HTML.

  3. ¿Cuál es la diferencia entre class e id?

  4. ¿Qué es la especificidad? Ordena de menor a mayor:
     - inline styles, id, class, elemento

  5. ¿Qué significa box-sizing: border-box y por qué se usa?

  6. Escribe el selector para:
     - Todos los párrafos dentro de un div con clase "articulo"
     - El primer hijo de una lista
     - Un input de tipo texto con atributo disabled

  7. ¿Qué es la cascada en CSS?

===============================================================================
SECCIÓN II: MODELO DE CAJA Y LAYOUT (25 PUNTOS)
===============================================================================

  8. Dibuja el modelo de caja CSS con todas sus partes.

  9. ¿Cuál es la diferencia entre margin y padding?

  10. ¿Qué hace display: flex? Nombra 3 propiedades del contenedor
      flex y 3 de los items.

  11. ¿Qué hace display: grid? Escribe un grid de 3 columnas
      iguales con un gap de 20px.

  12. Explica la diferencia entre:
      - justify-content y align-items
      - grid-template-columns y grid-template-rows
      - position: relative y position: absolute

  13. Crea un layout de 3 columnas que se convierta en 1 columna
      en móvil (escribe el HTML y CSS).

  14. ¿Qué propiedad usarías para centrar un elemento horizontal
      y verticalmente con Flexbox? ¿Y con Grid?

===============================================================================
SECCIÓN III: RESPONSIVE Y AVANZADO (25 PUNTOS)
===============================================================================

  15. ¿Qué hace la meta etiqueta viewport? Escríbela.

  16. Escribe una media query que aplique estilos solo en
      pantallas de 768px o más.

  17. ¿Cuál es la diferencia entre max-width y min-width?

  18. Explica qué es mobile-first. ¿Por qué se recomienda?

  19. Escribe una animación CSS que haga que un elemento se
      desvanezca (fade in) en 1 segundo.

  20. ¿Qué es clamp()? Escribe un ejemplo.

  21. ¿Qué hace transform: translateY(-10px)?

  22. ¿Cuál es la diferencia entre px, rem, em, vw y vh?

  23. Explica cómo hacer una imagen responsiva con srcset.

  24. ¿Qué es un keyframe? Escribe una animación que haga
      rebotar un elemento.

SECCIÓN IV: PROYECTO PRÁCTICO (25 PUNTOS)
===============================================================================

  25. Escribe el HTML y CSS completo para esta página:

      ┌──────────────────────────────────────┐
      │  Header: logo + 3 links              │
      ├──────────────────────────────────────┤
      │  Hero: título, texto, botón          │
      ├──────────────────────────────────────┤
      │  Sección: 3 cards en grid            │
      │  ┌──────┐ ┌──────┐ ┌──────┐         │
      │  │ Card1 │ │ Card2 │ │ Card3 │        │
      │  └──────┘ └──────┘ └──────┘         │
      ├──────────────────────────────────────┤
      │  Footer: copyright                   │
      └──────────────────────────────────────┘

      Requisitos:
      • Responsivo (1 col móvil, 3 col desktop).
      • Mobile-first.
      • Tipografía con rem.
      • Imagen responsiva en hero.
      • Efecto hover en cards (solo desktop).
      • Variables CSS para colores.
