===============================================================================
¿QUÉ ES UN DASHBOARD?
===============================================================================

Un dashboard es una interfaz que muestra información clave de un
sistema de forma visual y resumida: gráficas, tablas, tarjetas
con métricas, alertas, etc.

Características:
  • Sidebar de navegación.
  • Cards con métricas (KPIs).
  • Tablas de datos.
  • Gráficas (barras, líneas, pastel).
  • Responsivo (de escritorio a móvil).
