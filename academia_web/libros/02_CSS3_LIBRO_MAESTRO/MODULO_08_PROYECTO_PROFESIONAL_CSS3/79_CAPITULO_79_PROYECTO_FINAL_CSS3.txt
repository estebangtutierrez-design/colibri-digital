ESPECIFICACIONES GENERALES
===============================================================================

  Optimizado:     Lighthouse > 90 en todas las categorías.
  Publicado:      En GitHub Pages, Netlify o Vercel.

===============================================================================
OPCIONES DE PROYECTO
===============================================================================

  1. SITIO CORPORATIVO
     • 5+ secciones (hero, servicios, equipo, contacto, blog).
     • Navegación completa con links internos.
     • Footer con 4 columnas.
     • Formulario de contacto.

  2. TIENDA ONLINE (CATÁLOGO)
     • Grid de productos con imágenes.
     • Filtros por categoría (con CSS puro o checkbox).
     • Página de detalle de producto.
     • Carrito de compras (maqueta visual).

  3. PORTAFOLIO PROFESIONAL
     • Galería de proyectos con capturas.
     • Filtros por tecnología o categoría.
     • Página "Sobre mí" con historia personal.
     • Blog integrado (mínimo 3 artículos).

  4. DASHBOARD ADMINISTRATIVO
     • Sidebar de navegación.
     • Cards con KPIs.
     • Tabla de datos con estados.
     • Gráficos con CSS puro.
     • Mínimo 3 vistas (Dashboard, Usuarios, Reportes).

  5. PLATAFORMA EDUCATIVA
     • Página de inicio con cursos.
     • Grid de tarjetas de curso con progreso visual.
     • Página de detalle de curso.
     • Panel de progreso del estudiante.

===============================================================================
CHECKLIST DE ENTREGA
===============================================================================

  ESTRUCTURA Y HTML:
  ✓ 3+ páginas HTML enlazadas entre sí.
  ✓ HTML semántico (header, nav, main, section, article, footer).
  ✓ Meta viewport en todas las páginas.
  ✓ Títulos y descripciones meta para SEO.
  ✓ Imágenes con atributos alt descriptivos.

  CSS:
  ✓ Variables CSS (:root) para colores y medidas.
  ✓ CSS organizado en secciones (reset, variables, base, layout, componentes).
  ✓ Mobile-first con min-width.
  ✓ Mínimo 3 breakpoints (móvil, tablet, desktop).
  ✓ Sin !important (excepto utilidades justificadas).
  ✓ Sin IDs en selectores CSS (usar clases).
  ✓ Componentes reutilizables (btn, card, form).

  RESPONSIVE:
  ✓ Diseño funcional en 375px, 768px y 1440px.
  ✓ Imágenes con max-width: 100%.
  ✓ Tipografía con unidades rem y clamp().
  ✓ Navegación adaptable (menú hamburguesa o similar).

  RENDIMIENTO:
  ✓ CSS minificado.
  ✓ Imágenes optimizadas.
  ✓ Loading="lazy" en imágenes secundarias.
  ✓ Puntaje Lighthouse > 90.
