===============================================================================
¿QUÉ ES UNA LANDING PAGE?
===============================================================================

Una landing page es una página web diseñada para un único objetivo:
convertir al visitante (registro, compra, descarga, contacto).

Características:
  • Un solo propósito (call to action claro).
  • Sin distracciones (sin menús complejos).
  • Mensaje directo y visual.
  • Diseño que guía la mirada hacia el CTA.

===============================================================================
ESTRUCTURA DE UNA LANDING PAGE
===============================================================================

  ┌──────────────────────────────────┐
  │  Navbar (minimal, solo logo +    │
  │   CTA)                           │
  ├──────────────────────────────────┤
  │  HERO                            │
  │  — Título impactante             │
  │  — Subtítulo convincente         │
  │  — CTA principal                 │
  │  — Imagen/ilustración            │
  ├──────────────────────────────────┤
  │  BENEFICIOS / CARACTERÍSTICAS    │
  │  — Grid de 3 columnas            │
  │  — Iconos + texto breve          │
  ├──────────────────────────────────┤
  │  TESTIMONIOS / SOCIAL PROOF      │
  │  — Citas de usuarios             │
  │  — Logos de clientes             │
  ├──────────────────────────────────┤
  │  CTA FINAL (segunda oportunidad) │
  ├──────────────────────────────────┤
  │  FOOTER                          │
  │  — Links legales, redes sociales │
  └──────────────────────────────────┘

===============================================================================
PRINCIPIOS DE DISEÑO
===============================================================================

  • Jerarquía visual: el ojo debe ir al CTA.
  • Contraste: el botón debe destacar del fondo.
  • Espacio negativo: no satures. El blanco vende.
  • Velocidad: una landing lenta mata conversiones.
  • Mobile-first: 70%+ del tráfico de landing es móvil.
