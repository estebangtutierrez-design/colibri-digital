===============================================================================
INTRODUCCIÓN: ¿QUÉ ES BOX-SHADOW?
===============================================================================

`box-shadow` es una propiedad CSS que añade una o más sombras a un elemento.

Las sombras pueden ser:
- **Exteriores**: la sombra se proyecta fuera del elemento (las más comunes).
- **Interiores (`inset`)**: la sombra se proyecta dentro del elemento.

---

📌 **Representación visual de una sombra exterior**
+---------------------------------------------+
| |
| +-----------------------------------+ |
| | ELEMENTO | |
| | | |
| +-----------------------------------+ |
| |
+---------------------------------------------+
^ ^
| |
Sombra proyectada hacia la derecha y abajo

text

---

📌 **Sintaxis básica**
box-shadow: offset-x offset-y blur-radius spread-radius color;

text

| Valor | Descripción | Obligatorio |
|-------|-------------|-------------|
| `offset-x` | Desplazamiento horizontal | Sí |
| `offset-y` | Desplazamiento vertical | Sí |
| `blur-radius` | Radio de desenfoque (suavidad) | No (por defecto 0) |
| `spread-radius` | Expandido (tamaño) | No (por defecto 0) |
| `color` | Color de la sombra | Sí |
| `inset` | Sombra interior | No |

---

📌 **Valores por defecto**

- `offset-x`: 0
- `offset-y`: 0
- `blur-radius`: 0 (sin desenfoque, sombra dura)
- `spread-radius`: 0 (mismo tamaño que el elemento)
- `color`: el color del texto del elemento (usualmente negro)
- `inset`: no aplicado (sombra exterior)

===============================================================================
1. BOX-SHADOW: SINTAXIS COMPLETA
===============================================================================

---

📌 **Valores individuales**

**offset-x** (desplazamiento horizontal)
- Valor positivo: la sombra se mueve hacia la derecha.
- Valor negativo: la sombra se mueve hacia la izquierda.

**offset-y** (desplazamiento vertical)
- Valor positivo: la sombra se mueve hacia abajo.
- Valor negativo: la sombra se mueve hacia arriba.

**blur-radius** (radio de desenfoque)
- Valor positivo: la sombra se difumina.
- 0: sombra dura (sin desenfoque).
- A mayor valor, más suave es la sombra.

**spread-radius** (expandido)
- Valor positivo: la sombra se expande (se hace más grande).
- Valor negativo: la sombra se contrae (se hace más pequeña).
- 0: la sombra tiene el mismo tamaño que el elemento.

**color**
- Cualquier formato de color (hex, rgb, hsl, rgba, etc.).
- Soporta transparencia (recomendado para sombras realistas).

---

🔵 **Ejemplo básico: sombra simple**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    width: 200px;
    height: 100px;
    background-color: #3498DB;
    border-radius: 10px;
    box-shadow: 5px 5px 10px 0px rgba(0,0,0,0.3);
}
------------------------------------------------------------

**Resultado:**
- Desplazamiento horizontal: 5px a la derecha.
- Desplazamiento vertical: 5px hacia abajo.
- Desenfoque: 10px (sombra suave).
- Expandido: 0px (tamaño normal).
- Color: negro con 30% de opacidad.

---

📌 **La importancia de la transparencia**

Las sombras más realistas usan colores con **transparencia (RGBA o HSLA)**.
Esto permite que la sombra se mezcle con el fondo y se vea natural.
box-shadow: 5px 5px 10px rgba(0,0,0,0.3);

text

---

🔵 **Ejemplo: sombra sin desenfoque (dura)**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    box-shadow: 5px 5px 0px 0px rgba(0,0,0,0.3);
}
------------------------------------------------------------

**Resultado:** una sombra dura, sin difuminar.
Ideal para efectos de "cartel" o "estilo plano".

---

🔵 **Ejemplo: sombra con expandido positivo**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    box-shadow: 5px 5px 10px 10px rgba(0,0,0,0.3);
}
------------------------------------------------------------

**Resultado:** la sombra es más grande que el elemento (10px más en cada lado).

---

🔵 **Ejemplo: sombra con expandido negativo**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    box-shadow: 5px 5px 10px -5px rgba(0,0,0,0.3);
}
------------------------------------------------------------

**Resultado:** la sombra es más pequeña que el elemento (5px menos en cada lado).

===============================================================================
2. SOMBRAS INTERIORES (INSET)
===============================================================================

`inset` cambia la sombra de exterior a interior.
La sombra se proyecta hacia dentro del elemento.

---

📌 **Sintaxis**
box-shadow: inset offset-x offset-y blur-radius spread-radius color;

text

---

🔵 **Ejemplo: sombra interior básica**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    width: 200px;
    height: 100px;
    background-color: #3498DB;
    border-radius: 10px;
    box-shadow: inset 0px 0px 10px rgba(0,0,0,0.5);
}
------------------------------------------------------------

**Resultado:** la sombra se proyecta hacia el interior, creando un efecto de profundidad.

---

🔵 **Ejemplo: borde interior con sombra**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    width: 200px;
    height: 100px;
    background-color: white;
    border-radius: 10px;
    box-shadow: inset 0px 0px 0px 2px rgba(0,0,0,0.1);
}
------------------------------------------------------------

**Resultado:** un borde interior sutil sin usar `border`.

---

🔵 **Ejemplo: efecto de "agujero" con inset**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    width: 200px;
    height: 100px;
    background-color: #f0f0f0;
    border-radius: 10px;
    box-shadow: inset 0px 10px 20px rgba(0,0,0,0.2);
}
------------------------------------------------------------

**Resultado:** la caja parece tener un agujero o hendidura en la parte superior.

===============================================================================
3. MÚLTIPLES SOMBRAS
===============================================================================

Puedes aplicar múltiples sombras a un mismo elemento
separándolas con comas.

---

📌 **Sintaxis**
box-shadow: sombra1, sombra2, sombra3;

text

---

🔵 **Ejemplo: dos sombras (exterior e interior)**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    width: 200px;
    height: 100px;
    background-color: #3498DB;
    border-radius: 10px;
    box-shadow:
        5px 5px 15px rgba(0,0,0,0.3),    /* Sombra exterior */
        inset 0px -5px 10px rgba(0,0,0,0.2);   /* Sombra interior */
}
------------------------------------------------------------

---

🔵 **Ejemplo: sombras superpuestas para efecto tridimensional**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    width: 200px;
    height: 100px;
    background-color: #3498DB;
    border-radius: 10px;
    box-shadow:
        0px 1px 3px rgba(0,0,0,0.1),
        0px 5px 15px rgba(0,0,0,0.2),
        0px 10px 30px rgba(0,0,0,0.1);
}
------------------------------------------------------------

**Resultado:** un efecto de elevación muy realista con múltiples capas de sombra.

---

🔵 **Ejemplo: sombra de color (efecto de "glow" o resplandor)**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    width: 200px;
    height: 100px;
    background-color: #3498DB;
    border-radius: 10px;
    box-shadow:
        0px 0px 30px rgba(52, 152, 219, 0.5),
        0px 0px 60px rgba(52, 152, 219, 0.3);
}
------------------------------------------------------------

**Resultado:** la caja parece emitir luz azul (efecto "glow").

===============================================================================
4. SOMBRAS MÁS COMUNES EN EL DISEÑO MODERNO
===============================================================================

---

📌 **Sombra suave (la más común)**
box-shadow: 0px 4px 15px rgba(0,0,0,0.1);

text

Usado en tarjetas, botones y elementos elevados.

---

📌 **Sombra profunda (para elementos destacados)**
box-shadow: 0px 10px 30px rgba(0,0,0,0.2);

text

Usado en modales, menús desplegables y elementos flotantes.

---

📌 **Sombra elevada (para elementos que flotan)**
box-shadow: 0px 20px 50px rgba(0,0,0,0.15);

text

Usado para dar sensación de que el elemento está muy elevado.

---

📌 **Sombra interior (efecto de hundimiento)**
box-shadow: inset 0px 3px 8px rgba(0,0,0,0.15);

text

Usado en campos de formulario o elementos que parecen presionados.

---

📌 **Sombra de "neumorfismo" (effecto de relieve suave)**
box-shadow:
6px 6px 12px rgba(0,0,0,0.1),
-6px -6px 12px rgba(255,255,255,0.8);

text

Usado en diseños neumórficos (una tendencia de diseño).

---

📌 **Efecto "glow" (resplandor)**
box-shadow: 0px 0px 30px rgba(52, 152, 219, 0.5);

text

Usado para destacar elementos con un resplandor de color.

===============================================================================
5. USOS PRÁCTICOS DE BOX-SHADOW
===============================================================================

---

📌 **1. Tarjetas con elevación**

------------------------------------------------------------
CSS
------------------------------------------------------------
.tarjeta {
    background-color: white;
    border-radius: 12px;
    padding: 20px;
    box-shadow: 0px 4px 15px rgba(0,0,0,0.1);
    transition: box-shadow 0.3s;
}

.tarjeta:hover {
    box-shadow: 0px 10px 30px rgba(0,0,0,0.2);
}
------------------------------------------------------------

**Resultado:** la tarjeta se eleva al pasar el mouse.

---

📌 **2. Botones con sombra**

------------------------------------------------------------
CSS
------------------------------------------------------------
.boton {
    background-color: #007BFF;
    color: white;
    border: none;
    padding: 12px 24px;
    border-radius: 8px;
    box-shadow: 0px 4px 10px rgba(0,123,255,0.3);
    transition: box-shadow 0.3s;
}

.boton:hover {
    box-shadow: 0px 6px 20px rgba(0,123,255,0.5);
}
------------------------------------------------------------

**Resultado:** el botón tiene una sombra que se intensifica al pasar el mouse.

---

📌 **3. Inputs con sombra interior (focus)**

------------------------------------------------------------
CSS
------------------------------------------------------------
input[type="text"] {
    padding: 12px 16px;
    border: 2px solid #ced4da;
    border-radius: 8px;
    transition: box-shadow 0.3s;
}

input[type="text"]:focus {
    border-color: #007BFF;
    box-shadow: inset 0px 2px 8px rgba(0,0,0,0.1);
}
------------------------------------------------------------

**Resultado:** el input parece hundirse ligeramente al enfocarse.

---

📌 **4. Efecto de "flotación" en imágenes**

------------------------------------------------------------
CSS
------------------------------------------------------------
.imagen {
    border-radius: 12px;
    box-shadow: 0px 10px 30px rgba(0,0,0,0.15);
    transition: transform 0.3s, box-shadow 0.3s;
}

.imagen:hover {
    transform: translateY(-5px);
    box-shadow: 0px 20px 40px rgba(0,0,0,0.2);
}
------------------------------------------------------------

---

📌 **5. Modal o popup con sombra profunda**

------------------------------------------------------------
CSS
------------------------------------------------------------
.modal {
    background-color: white;
    border-radius: 16px;
    padding: 30px;
    box-shadow: 0px 20px 60px rgba(0,0,0,0.3);
    z-index: 1000;
}
------------------------------------------------------------

===============================================================================
6. BOX-SHADOW Y ACCESIBILIDAD
===============================================================================

---

📌 **Contraste y sombras**

Las sombras no deben afectar la legibilidad del texto.
Asegúrate de que el contraste entre el texto y el fondo sea suficiente.

---

📌 **No exagerar con las sombras**

Sombras demasiado grandes o difusas pueden causar fatiga visual.
Usa sombras con moderación.

---

📌 **Preferencia de movimiento reducido**

Si usas transiciones de sombra, respeta `prefers-reduced-motion`.

------------------------------------------------------------
CSS
------------------------------------------------------------
@media (prefers-reduced-motion: reduce) {
    .tarjeta, .boton, .imagen {
        transition: none;
    }
}
------------------------------------------------------------

===============================================================================
LA ANALOGÍA DEL PROFESOR BÚHO: LA SOMBRA COMO LA LUZ DEL SOL
===============================================================================

Imagina que los elementos de tu página son objetos en una mesa.

- La luz del sol viene desde un ángulo (offset-x, offset-y).
- La sombra se proyecta en dirección opuesta a la luz.
- Si la luz está lejos, la sombra es suave (blur-radius grande).
- Si la luz está cerca, la sombra es dura (blur-radius pequeño).
- Si el objeto está más alto, la sombra es más grande (spread-radius positivo).
- Si el objeto está más bajo, la sombra es más pequeña (spread-radius negativo).

Las sombras en CSS simulan este comportamiento natural.

===============================================================================
ERRORES COMUNES CON BOX-SHADOW
===============================================================================

❌ **Error 1: Usar colores sin transparencia**

   Si usas `box-shadow: 5px 5px 10px black;`, la sombra será opaca y se verá artificial.
   **Solución:** usa rgba o hsla con transparencia.

---

❌ **Error 2: No usar suficiente desenfoque**

   `box-shadow: 5px 5px 0px black;` da sombras duras y poco realistas.
   **Solución:** añade desenfoque (blur-radius) para suavizar.

---

❌ **Error 3: Sombras demasiado grandes**

   Sombras muy grandes pueden verse exageradas y poco profesionales.
   **Solución:** usa valores moderados.

---

❌ **Error 4: Olvidar el inset**

   Si quieres una sombra interior, no olvides `inset`.
   **Solución:** recuerda que `inset` cambia la dirección de la sombra.

---

❌ **Error 5: No usar múltiples sombras para efectos complejos**

   Una sola sombra puede ser limitada.
   **Solución:** usa múltiples sombras separadas por comas.

---

❌ **Error 6: Usar box-shadow en elementos con overflow: hidden**

   La sombra exterior no se ve si el elemento tiene `overflow: hidden`.
   **Solución:** si necesitas ambas, considera usar otro elemento para la sombra.

GLOSARIO DEL CAPÍTULO
===============================================================================

- **Box-shadow**: Propiedad CSS que añade sombras a un elemento.
- **Offset-x**: Desplazamiento horizontal de la sombra.
- **Offset-y**: Desplazamiento vertical de la sombra.
- **Blur-radius**: Radio de desenfoque (suavidad de la sombra).
- **Spread-radius**: Expandido de la sombra (tamaño).
- **Inset**: Palabra clave para sombras interiores.
- **Transparencia**: Uso de rgba o hsla para sombras realistas.
- **Efecto glow**: Resplandor alrededor de un elemento.
- **Neumorfismo**: Estilo de diseño con sombras suaves y colores pastel.
- **Múltiples sombras**: Varias sombras aplicadas a un mismo elemento, separadas por comas.

===============================================================================
PREGUNTAS FRECUENTES (FAQ)
===============================================================================

❓ **¿Cuál es la diferencia entre box-shadow y text-shadow?**

   `box-shadow` proyecta sombras en el **elemento** (la caja).
   `text-shadow` proyecta sombras en el **texto**.

---

❓ **¿Cómo hago una sombra sin desenfoque?**

   Usa `blur-radius: 0`.
   Ejemplo: `box-shadow: 5px 5px 0px rgba(0,0,0,0.3);`.

---

❓ **¿Cómo hago una sombra interior?**

   Usa la palabra clave `inset`.
   Ejemplo: `box-shadow: inset 0px 5px 15px rgba(0,0,0,0.2);`.

---

❓ **¿Cómo hago un resplandor (glow)?**

   Usa una sombra sin desplazamiento y con mucho desenfoque.
   Ejemplo: `box-shadow: 0px 0px 40px rgba(52,152,219,0.5);`.

---

❓ **¿Cómo hago un efecto neumórfico?**

   Usa dos sombras: una clara (blanca) y una oscura (negra con transparencia).
   Ejemplo: `box-shadow: 6px 6px 12px rgba(0,0,0,0.1), -6px -6px 12px rgba(255,255,255,0.8);`.
