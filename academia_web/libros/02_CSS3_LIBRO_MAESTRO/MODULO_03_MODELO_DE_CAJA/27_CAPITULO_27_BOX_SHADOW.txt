el padding, el margin, el border y el border-radius.

Pero hay una pregunta que seguramente te has hecho:

**¿Qué pasa cuando el contenido es más grande que el contenedor?**

- ¿Qué ocurre si tienes un párrafo muy largo dentro de una caja pequeña?
- ¿Qué pasa si una imagen es más ancha que su contenedor?
- ¿Cómo creas un área de scroll para mostrar contenido extenso?

La respuesta es la propiedad **`overflow`**.

`overflow` controla qué sucede cuando el contenido de un elemento
excede el tamaño de su caja.

Puedes:
- Mostrar el contenido desbordado (`visible`).
- Ocultarlo (`hidden`).
- Mostrar barras de scroll (`scroll`, `auto`).
- Controlar el desbordamiento en cada eje (`overflow-x`, `overflow-y`).

El `overflow` es fundamental para crear layouts limpios,
contenedores con scroll, tarjetas con contenido recortado
y muchos otros efectos.

Hoy vas a aprender a dominarlo.

===============================================================================
INTRODUCCIÓN: ¿QUÉ ES OVERFLOW?
===============================================================================

`overflow` es una propiedad CSS que controla qué sucede cuando el contenido
de un elemento es demasiado grande para caber en su caja.

---

📌 **Representación visual del overflow**
+---------------------------+
| CONTENEDOR (width fijo) |
| +---------------------+ |
| | Contenido que | |
| | no cabe en el | |
| | contenedor. | |
| | Se desborda... | |
| +---------------------+ |
| |
+---------------------------+

text

El contenido que sobresale es el **overflow**.

---

📌 **Propiedades de overflow**

| Propiedad | Descripción |
|-----------|-------------|
| `overflow` | Controla ambos ejes (horizontal y vertical) |
| `overflow-x` | Controla solo el eje horizontal |
| `overflow-y` | Controla solo el eje vertical |
| `text-overflow` | Controla cómo se muestra el texto que no cabe |
| `white-space` | Controla cómo se manejan los espacios en blanco |

---

📌 **Valores de overflow**

| Valor | Descripción |
|-------|-------------|
| `visible` | El contenido se desborda fuera de la caja (por defecto) |
| `hidden` | El contenido desbordado se recorta (no se ve) |
| `scroll` | Siempre muestra barras de scroll (aunque no sea necesario) |
| `auto` | Muestra barras de scroll solo si es necesario |

===============================================================================
1. OVERFLOW: VISIBLE (VALOR POR DEFECTO)
===============================================================================

`overflow: visible` es el valor por defecto.
El contenido que no cabe en la caja se desborda y se muestra fuera de ella.

---

🔵 **Ejemplo**

------------------------------------------------------------
HTML
------------------------------------------------------------
<div class="caja">
    Este es un texto muy largo que no cabe dentro del contenedor
    porque el contenedor tiene un ancho fijo y el texto es extenso.
</div>
------------------------------------------------------------

CSS:
------------------------------------------------------------
.caja {
    width: 200px;
    height: 50px;
    border: 2px solid #333;
    padding: 10px;
    overflow: visible;   /* Por defecto */
    background-color: #f0f0f0;
}
------------------------------------------------------------

**Resultado:** El texto se desborda y se ve fuera del contenedor.

---

📌 **Usos de overflow: visible**

- Es el comportamiento natural de los elementos.
- Se usa cuando quieres que el contenido sea completamente visible,
  incluso si sobresale.
- Es el valor por defecto, así que no suele necesitar declararse.

---

⚠️ **Precaución**

`overflow: visible` puede romper el layout si el contenido desbordado
interfiere con otros elementos.

===============================================================================
2. OVERFLOW: HIDDEN (RECORTAR CONTENIDO)
===============================================================================

`overflow: hidden` recorta cualquier contenido que sobresalga
de la caja. El contenido desbordado no se muestra.

---

🔵 **Ejemplo**

------------------------------------------------------------
HTML
------------------------------------------------------------
<div class="caja">
    Este es un texto muy largo que no cabe dentro del contenedor
    porque el contenedor tiene un ancho fijo y el texto es extenso.
</div>
------------------------------------------------------------

CSS:
------------------------------------------------------------
.caja {
    width: 200px;
    height: 50px;
    border: 2px solid #333;
    padding: 10px;
    overflow: hidden;   /* Recorta el contenido */
    background-color: #f0f0f0;
}
------------------------------------------------------------

**Resultado:** El texto que sobresale no se ve.
El contenido se recorta en el límite del contenedor.

---

📌 **Usos de overflow: hidden**

1. **Recortar imágenes** que no caben en su contenedor.
2. **Crear efectos visuales** con `border-radius` (recorta el contenido en las esquinas).
3. **Limpiar floats** (técnica de clearfix).
4. **Ocultar contenido** que no debe ser visible hasta que se interactúe.

---

🔵 **Ejemplo: imagen recortada con overflow: hidden**

------------------------------------------------------------
CSS
------------------------------------------------------------
.contenedor-imagen {
    width: 200px;
    height: 200px;
    overflow: hidden;
    border-radius: 50%;   /* Recorta la imagen en un círculo */
}

.contenedor-imagen img {
    width: 100%;
    height: 100%;
    object-fit: cover;
}
------------------------------------------------------------

**Resultado:** La imagen se recorta dentro del círculo.

---

📌 **Overflow: hidden y border-radius**

Cuando usas `overflow: hidden` con `border-radius`,
el contenido se recorta siguiendo la forma redondeada.

------------------------------------------------------------
CSS
------------------------------------------------------------
.tarjeta {
    width: 200px;
    border-radius: 16px;
    overflow: hidden;   /* El contenido se recorta con las esquinas redondeadas */
}
------------------------------------------------------------

===============================================================================
3. OVERFLOW: SCROLL (SIEMPRE MUESTRA BARRAS)
===============================================================================

`overflow: scroll` siempre muestra barras de desplazamiento
(horizontal y vertical), incluso si no hay contenido desbordado.

---

🔵 **Ejemplo**

------------------------------------------------------------
HTML
------------------------------------------------------------
<div class="caja">
    Este es un texto muy largo que no cabe dentro del contenedor.
    Por eso se muestra una barra de scroll para poder leerlo todo.
</div>
------------------------------------------------------------

CSS:
------------------------------------------------------------
.caja {
    width: 200px;
    height: 80px;
    border: 2px solid #333;
    padding: 10px;
    overflow: scroll;   /* Siempre muestra barras */
    background-color: #f0f0f0;
}
------------------------------------------------------------

**Resultado:** Se muestran barras de scroll (vertical y horizontal)
aunque el contenido no se desborde en ambos ejes.

---

📌 **Usos de overflow: scroll**

- Cuando sabes que el contenido **siempre** será más grande que el contenedor.
- Cuando quieres un comportamiento consistente en todos los navegadores.
- Para áreas de contenido dinámico (como chats o feeds).

---

⚠️ **Precaución**

`overflow: scroll` ocupa espacio para las barras de scroll
(aproximadamente 15-20px), incluso cuando no son necesarias.
Esto puede afectar al layout.

===============================================================================
4. OVERFLOW: AUTO (SCROLL SOLO SI ES NECESARIO)
===============================================================================

`overflow: auto` es el valor más usado.
Muestra barras de scroll **solo si es necesario**.

---

🔵 **Ejemplo**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    width: 200px;
    height: 80px;
    border: 2px solid #333;
    padding: 10px;
    overflow: auto;   /* Muestra scroll solo si es necesario */
    background-color: #f0f0f0;
}
------------------------------------------------------------

**Resultado:**
- Si el contenido cabe, no hay barras de scroll.
- Si el contenido no cabe, aparecen las barras necesarias.

---

📌 **Usos de overflow: auto**

- **Es el valor recomendado** para la mayoría de los casos.
- Áreas de contenido donde el tamaño puede variar.
- Contenedores de texto largo.
- Listas o tablas con muchos datos.

---

🔵 **Ejemplo: área de comentarios con scroll**

------------------------------------------------------------
CSS
------------------------------------------------------------
.comentarios {
    width: 100%;
    max-height: 200px;
    overflow: auto;
    border: 1px solid #ddd;
    padding: 10px;
    background-color: white;
    border-radius: 8px;
}
------------------------------------------------------------

**Resultado:** Los comentarios se muestran con scroll
si hay más de los que caben en el contenedor.

===============================================================================
5. OVERFLOW-X Y OVERFLOW-Y (EJES INDIVIDUALES)
===============================================================================

Puedes controlar el desbordamiento en cada eje por separado.

---

📌 **overflow-x**: controla el eje horizontal.

📌 **overflow-y**: controla el eje vertical.

---

🔵 **Ejemplo: scroll solo vertical**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    width: 200px;
    height: 80px;
    border: 2px solid #333;
    padding: 10px;
    overflow-x: hidden;   /* Oculta el desbordamiento horizontal */
    overflow-y: auto;     /* Muestra scroll vertical si es necesario */
    background-color: #f0f0f0;
    white-space: nowrap;  /* Evita que el texto se envuelva */
}
------------------------------------------------------------

**Resultado:** El texto que no cabe horizontalmente se oculta.
El texto que no cabe verticalmente muestra scroll.

---

🔵 **Ejemplo: scroll solo horizontal**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    width: 200px;
    height: 80px;
    border: 2px solid #333;
    padding: 10px;
    overflow-x: auto;     /* Muestra scroll horizontal si es necesario */
    overflow-y: hidden;   /* Oculta el desbordamiento vertical */
    background-color: #f0f0f0;
    white-space: nowrap;  /* Evita que el texto se envuelva */
}
------------------------------------------------------------

---

📌 **Usos comunes**

- **Contenedores de imágenes en galerías** (scroll horizontal).
- **Tablas con muchas columnas** (scroll horizontal).
- **Áreas de texto** (scroll vertical).

===============================================================================
6. TEXT-OVERFLOW: MANEJAR TEXTO QUE NO CABE
===============================================================================

`text-overflow` controla cómo se muestra el texto que no cabe
en una línea de texto.

---

📌 **Propiedades relacionadas**

- `text-overflow: clip` → recorta el texto (por defecto).
- `text-overflow: ellipsis` → muestra puntos suspensivos (...).

---

🔵 **Ejemplo: puntos suspensivos**

------------------------------------------------------------
CSS
------------------------------------------------------------
.texto-recortado {
    width: 150px;
    white-space: nowrap;        /* Evita que el texto se envuelva */
    overflow: hidden;           /* Oculta el desbordamiento */
    text-overflow: ellipsis;    /* Muestra ... al final */
    border: 1px solid #ddd;
    padding: 5px;
}
------------------------------------------------------------

**Resultado:** Si el texto es más largo que 150px, se muestra
un "..." al final.

---

🔵 **Ejemplo: texto sin puntos suspensivos (clip)**

------------------------------------------------------------
CSS
------------------------------------------------------------
.texto-clip {
    width: 150px;
    white-space: nowrap;
    overflow: hidden;
    text-overflow: clip;   /* Recorta el texto sin ... */
    border: 1px solid #ddd;
    padding: 5px;
}
------------------------------------------------------------

**Resultado:** El texto se corta bruscamente al final del contenedor.

---

📌 **Condiciones para que funcione text-overflow**

1. El elemento debe tener `white-space: nowrap` (o un valor que evite el wrap).
2. El elemento debe tener `overflow: hidden` o `overflow: auto`.
3. El elemento debe tener un `width` definido.

---

📌 **Usos de text-overflow: ellipsis**

- **Títulos de productos** en tarjetas.
- **Nombres de archivos** en listas.
- **Textos largos** en tablas.
- **Notificaciones** en menús.

===============================================================================
7. WHITE-SPACE: CONTROL DE ESPACIOS Y SALTO DE LÍNEA
===============================================================================

`white-space` controla cómo se manejan los espacios en blanco
y los saltos de línea en el texto.

---

📌 **Valores de white-space**

| Valor | Descripción |
|-------|-------------|
| `normal` | Los espacios se colapsan, el texto se envuelve (por defecto) |
| `nowrap` | El texto no se envuelve, se muestra en una sola línea |
| `pre` | Los espacios y saltos de línea se respetan (como `<pre>`) |
| `pre-wrap` | Se respetan espacios y saltos, pero el texto se envuelve |
| `pre-line` | Los espacios se colapsan, los saltos se respetan |

---

🔵 **Ejemplo: nowrap para text-overflow**

------------------------------------------------------------
CSS
------------------------------------------------------------
.linea-unica {
    white-space: nowrap;
    overflow: hidden;
    text-overflow: ellipsis;
    width: 150px;
    border: 1px solid #ddd;
    padding: 5px;
}
------------------------------------------------------------

---

🔵 **Ejemplo: pre (respetar formato)**

------------------------------------------------------------
CSS
------------------------------------------------------------
.poema {
    white-space: pre;
    font-family: monospace;
    border: 1px solid #ddd;
    padding: 10px;
}
------------------------------------------------------------

===============================================================================
LA ANALOGÍA DEL PROFESOR BÚHO: EL OVERFLOW COMO UNA PRESA
===============================================================================

Imagina que el contenido de un elemento es un río y su contenedor es una presa.

- **overflow: visible** → No hay presa. El río fluye libremente y se desborda.
- **overflow: hidden** → La presa es alta. El agua se detiene y no se desborda.
- **overflow: scroll** → Hay compuertas que siempre están abiertas (scroll).
- **overflow: auto** → Las compuertas se abren solo cuando el nivel del agua sube.

Tú decides cómo controlar el flujo del contenido.

===============================================================================
ERRORES COMUNES CON OVERFLOW
===============================================================================

❌ **Error 1: Usar overflow: visible y que el contenido rompa el layout**

   El contenido desbordado puede superponerse a otros elementos.
   **Solución:** usa `overflow: hidden` o `overflow: auto`.

---

❌ **Error 2: Usar overflow: scroll innecesariamente**

   `overflow: scroll` siempre muestra barras, incluso si no son necesarias.
   **Solución:** usa `overflow: auto` para mostrar scroll solo si es necesario.

---

❌ **Error 3: Olvidar white-space para text-overflow**

   `text-overflow: ellipsis` no funciona sin `white-space: nowrap`.
   **Solución:** recuerda las tres condiciones: `nowrap`, `overflow: hidden`, `width`.

---

❌ **Error 4: No considerar el espacio de las barras de scroll**

   Las barras de scroll ocupan espacio (15-20px).
   **Solución:** tenlo en cuenta al calcular el ancho.

---

❌ **Error 5: Usar overflow: hidden en elementos que necesitan scroll**

   Si ocultas el contenido sin scroll, el usuario no podrá verlo.
   **Solución:** usa `overflow: auto` si el contenido puede ser extenso.

GLOSARIO DEL CAPÍTULO
===============================================================================

- **Overflow**: Contenido que sobresale del contenedor.
- **Overflow: visible**: El contenido se desborda y es visible (por defecto).
- **Overflow: hidden**: El contenido desbordado se recorta y no se ve.
- **Overflow: scroll**: Siempre muestra barras de scroll.
- **Overflow: auto**: Muestra barras de scroll solo si es necesario.
- **Overflow-x**: Controla el desbordamiento horizontal.
- **Overflow-y**: Controla el desbordamiento vertical.
- **Text-overflow**: Controla cómo se muestra el texto que no cabe.
- **Ellipsis**: Puntos suspensivos (...) que indican texto cortado.
- **White-space**: Controla cómo se manejan espacios y saltos de línea.

===============================================================================
PREGUNTAS FRECUENTES (FAQ)
===============================================================================

❓ **¿Cuál es la diferencia entre overflow: hidden y overflow: clip?**

   `hidden` oculta el contenido desbordado. `clip` también oculta el contenido,
   pero no permite que se desplace (incluso con `overflow: auto`).
   `clip` es más nuevo y menos usado.

---

❓ **¿Cómo hago para que un contenedor tenga scroll vertical pero no horizontal?**

   Usa `overflow-y: auto` y `overflow-x: hidden`.

---

❓ **¿Por qué text-overflow: ellipsis no funciona en mi elemento?**

   Asegúrate de tener:
   - `white-space: nowrap`
   - `overflow: hidden`
   - Un `width` definido

---

❓ **¿Cómo creo una galería con scroll horizontal?**

   Usa `overflow-x: auto` y `display: flex` en el contenedor.
   Asegúrate de que los elementos no se envuelvan.

---

❓ **¿Qué es `-webkit-line-clamp`?**

   Es una propiedad para limitar el texto a un número de líneas
   y mostrar puntos suspensivos. Ejemplo: `-webkit-line-clamp: 3;`.
