- **Margen** (margin): el espacio fuera del borde, entre elementos.

Entender cómo funcionan estas cuatro capas es esencial para crear
diseños precisos, predecibles y profesionales.

===============================================================================
INTRODUCCIÓN: ¿QUÉ ES EL MODELO DE CAJA?
===============================================================================

El Modelo de Caja es el concepto fundamental que describe cómo se representan
los elementos HTML en una página web.

Cada elemento en HTML es una **caja rectangular**.

Imagina que cada elemento (un párrafo, un div, una imagen, un botón) está
dentro de una caja invisible. Esa caja tiene varias capas.

---

📌 **Las cuatro capas del Modelo de Caja**
+---------------------------------------------------------+
| MARGEN (margin) |
| +---------------------------------------------------+ |
| | BORDE (border) | |
| | +---------------------------------------------+ | |
| | | PADDING (padding) | | |
| | | +---------------------------------------+ | | |
| | | | CONTENIDO (content) | | | |
| | | | | | | |
| | | +---------------------------------------+ | | |
| | +---------------------------------------------+ | |
| +---------------------------------------------------+ |
| |
+---------------------------------------------------------+

text

1. **Contenido (content)**: El área donde está el texto, la imagen o los elementos hijos.
2. **Relleno (padding)**: Espacio transparente entre el contenido y el borde.
3. **Borde (border)**: Línea que rodea el padding (puede ser visible o invisible).
4. **Margen (margin)**: Espacio transparente fuera del borde, que separa un elemento de otro.

---

📌 **Propiedades del Modelo de Caja**

| Propiedad | Descripción |
|-----------|-------------|
| `width` | Ancho del **contenido** (por defecto) |
| `height` | Alto del **contenido** (por defecto) |
| `padding` | Espacio entre el contenido y el borde |
| `border` | Línea alrededor del padding |
| `margin` | Espacio entre el borde y otros elementos |

---

🟢 **Regla de oro**

**El tamaño total de un elemento = width + padding + border + margin**

Pero ojo: esto depende del `box-sizing`. Lo veremos más adelante.

===============================================================================
1. CONTENIDO (CONTENT): WIDTH Y HEIGHT
===============================================================================

Las propiedades `width` y `height` definen el tamaño del **contenido** de la caja.

---

📌 **width**

Define el ancho del contenido.

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    width: 200px;   /* El contenido ocupará 200px de ancho */
}
------------------------------------------------------------

---

📌 **height**

Define el alto del contenido.

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    height: 150px;   /* El contenido ocupará 150px de alto */
}
------------------------------------------------------------

---

🔵 **Ejemplo básico**

HTML:
------------------------------------------------------------
<div class="caja">Contenido</div>
------------------------------------------------------------

CSS:
------------------------------------------------------------
.caja {
    width: 200px;
    height: 100px;
    background-color: #3498DB;
    color: white;
}
------------------------------------------------------------

**Resultado:** Un rectángulo azul de 200px de ancho y 100px de alto.

---

📌 **Valores de width y height**

- **px**: Píxeles (valor fijo).
- **%**: Porcentaje del contenedor.
- **em, rem**: Relativo al tamaño de fuente.
- **vw, vh**: Relativo al viewport.
- **auto**: El navegador calcula el tamaño automáticamente (por defecto).
- **min-content, max-content**: Ajusta al contenido mínimo o máximo.

---

📌 **Valores especiales**

- `width: auto` (por defecto): el elemento ocupa todo el espacio disponible.
- `height: auto` (por defecto): la altura se ajusta al contenido.
- `min-width`, `max-width`: limitan el ancho mínimo y máximo.
- `min-height`, `max-height`: limitan la altura mínima y máxima.

---

🔵 **Ejemplo con min-width y max-width**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    min-width: 200px;
    max-width: 500px;
    width: 50%;   /* Si el contenedor es de 1000px, será 500px (50%) */
}
------------------------------------------------------------

---

⚠️ **Importante**

Por defecto, `width` y `height` se aplican **solo al contenido**.
El padding y el borde se añaden **fuera** de este tamaño.

Esto significa que si tienes:
width: 200px;
padding: 20px;
border: 2px solid black;

text

El ancho total será: 200 + 20 + 20 + 2 + 2 = 244px.

Veremos cómo cambiar esto con `box-sizing` más adelante.

===============================================================================
2. PADDING: ESPACIO INTERNO
===============================================================================

El `padding` es el espacio entre el contenido y el borde de la caja.

---

📌 **Sintaxis**
padding: valor; /* Aplica a los cuatro lados */
padding-top: valor;
padding-right: valor;
padding-bottom: valor;
padding-left: valor;

text

---

🔵 **Ejemplo básico**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    width: 200px;
    padding: 20px;
    background-color: #3498DB;
    color: white;
}
------------------------------------------------------------

**Resultado:** El contenido tiene 20px de espacio interno en cada lado.
El ancho total ahora es: 200 (width) + 20 (padding-izquierdo) + 20 (padding-derecho) = 240px.

---

📌 **Shorthand (atajos) de padding**

| Sintaxis | Significado |
|----------|-------------|
| `padding: 20px;` | 20px en los cuatro lados |
| `padding: 10px 20px;` | 10px arriba/abajo, 20px izquierda/derecha |
| `padding: 10px 20px 30px;` | 10px arriba, 20px izquierda/derecha, 30px abajo |
| `padding: 10px 20px 30px 40px;` | 10px arriba, 20px derecha, 30px abajo, 40px izquierda |

---

🔵 **Ejemplo con diferentes paddings**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    width: 200px;
    padding-top: 10px;
    padding-right: 20px;
    padding-bottom: 30px;
    padding-left: 40px;
    background-color: #3498DB;
    color: white;
}
------------------------------------------------------------

---

📌 **Usos comunes del padding**

1. **Crear espacio interno** para que el texto no toque el borde.
2. **Mejorar la legibilidad** y la estética.
3. **Aumentar el área de clic** en botones y enlaces.
4. **Crear separación entre el contenido y el borde**.

---

🔵 **Ejemplo: botón con padding**

------------------------------------------------------------
CSS
------------------------------------------------------------
.boton {
    padding: 15px 30px;
    background-color: #007BFF;
    color: white;
    border: none;
    border-radius: 5px;
    font-size: 1rem;
}
------------------------------------------------------------

**Resultado:** Un botón con espacio interno que lo hace más fácil de clicar.

===============================================================================
3. BORDER: EL BORDE DE LA CAJA
===============================================================================

El `border` es la línea que rodea el padding y el contenido.

---

📌 **Sintaxis**
border: ancho estilo color;

text

- **ancho**: tamaño del borde (px, em, etc.)
- **estilo**: `solid`, `dashed`, `dotted`, `double`, `none`, etc.
- **color**: cualquier formato de color.

---

🔵 **Ejemplo básico**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    border: 2px solid #333;
    padding: 20px;
    background-color: #f0f0f0;
}
------------------------------------------------------------

---

📌 **Propiedades individuales del borde**

| Propiedad | Descripción |
|-----------|-------------|
| `border-width` | Grosor del borde |
| `border-style` | Estilo del borde (solid, dashed, dotted, etc.) |
| `border-color` | Color del borde |
| `border-radius` | Redondeo de las esquinas |
| `border-top`, `border-right`, `border-bottom`, `border-left` | Borde por lado |

---

🔵 **Ejemplo con diferentes bordes por lado**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    border-top: 3px solid #007BFF;
    border-right: 2px dashed #28A745;
    border-bottom: 4px dotted #DC3545;
    border-left: 2px solid #FFC107;
    padding: 20px;
}
------------------------------------------------------------

---

📌 **Estilos de borde**

| Estilo | Descripción |
|--------|-------------|
| `solid` | Línea sólida |
| `dashed` | Línea discontinua (guiones) |
| `dotted` | Línea de puntos |
| `double` | Línea doble |
| `groove` | Efecto de relieve (3D) |
| `ridge` | Efecto de relieve inverso |
| `inset` | Efecto de hundimiento |
| `outset` | Efecto de saliente |
| `none` | Sin borde |
| `hidden` | Oculto (similar a none) |

---

🔵 **Ejemplo con border-radius**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    border: 2px solid #333;
    border-radius: 10px;   /* Esquinas redondeadas */
    padding: 20px;
}
------------------------------------------------------------

---

📌 **Usos comunes del borde**

1. **Delimitar visualmente** elementos (tarjetas, secciones).
2. **Crear énfasis** en elementos importantes.
3. **Diferenciar** elementos en un formulario.
4. **Crear efectos decorativos** con `border-radius` y diferentes estilos.

---

⚠️ **Importante**

El borde **suma** al tamaño total del elemento.
Si tienes `width: 200px` y `border: 2px solid black`, el ancho total será 204px.

===============================================================================
4. MARGIN: ESPACIO EXTERNO
===============================================================================

El `margin` es el espacio fuera del borde, que separa un elemento de otro.

---

📌 **Sintaxis**
margin: valor; /* Aplica a los cuatro lados */
margin-top: valor;
margin-right: valor;
margin-bottom: valor;
margin-left: valor;

text

---

🔵 **Ejemplo básico**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    margin: 20px;
    padding: 20px;
    border: 2px solid #333;
    background-color: #f0f0f0;
}
------------------------------------------------------------

**Resultado:** El elemento tiene 20px de espacio exterior en cada lado.
Se separará 20px de los elementos adyacentes.

---

📌 **Shorthand (atajos) de margin**

| Sintaxis | Significado |
|----------|-------------|
| `margin: 20px;` | 20px en los cuatro lados |
| `margin: 10px 20px;` | 10px arriba/abajo, 20px izquierda/derecha |
| `margin: 10px 20px 30px;` | 10px arriba, 20px izquierda/derecha, 30px abajo |
| `margin: 10px 20px 30px 40px;` | 10px arriba, 20px derecha, 30px abajo, 40px izquierda |

---

📌 **Margen automático (auto)**

`margin: auto` centra el elemento horizontalmente si tiene un ancho definido.

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    width: 200px;
    margin: 0 auto;   /* Centra horizontalmente */
}
------------------------------------------------------------

---

📌 **Colapso de márgenes (margin collapse)**

Cuando dos márgenes verticales se encuentran, el mayor de los dos prevalece.
Elemento A: margin-bottom: 30px
Elemento B: margin-top: 20px

text

El espacio entre ellos será de **30px** (el mayor), no de 50px.

Esto se llama **colapso de márgenes** y solo ocurre en dirección vertical.

---

🔵 **Ejemplo de colapso de márgenes**

------------------------------------------------------------
HTML
------------------------------------------------------------
<div class="caja1">Caja 1</div>
<div class="caja2">Caja 2</div>
------------------------------------------------------------

CSS:
------------------------------------------------------------
.caja1 {
    margin-bottom: 30px;
    background: #3498DB;
    padding: 10px;
}

.caja2 {
    margin-top: 20px;
    background: #28A745;
    padding: 10px;
}
------------------------------------------------------------

**Resultado:** El espacio entre las cajas será de 30px, no de 50px.

---

📌 **Usos comunes del margin**

1. **Separar elementos** entre sí.
2. **Centrar elementos** con `margin: auto`.
3. **Crear espacio** entre secciones.
4. **Alinear elementos** en un layout.

---

⚠️ **Importante**

El margen **no** suma al tamaño total del elemento en el modelo de caja estándar.
Pero afecta al espacio que el elemento ocupa en la página.

===============================================================================
5. BOX-SIZING: EL MODELO DE CAJA MODERNO
===============================================================================

Por defecto, el navegador usa el **modelo de caja estándar** (content-box).

En este modelo:
- `width` y `height` se aplican **solo al contenido**.
- `padding` y `border` se suman al tamaño total.

Pero hay un modelo alternativo: **border-box**.

En este modelo:
- `width` y `height` incluyen **contenido + padding + borde**.
- El padding y el borde **no** suman al tamaño total.

---

📌 **box-sizing: content-box** (por defecto)
width = content
total = width + padding + border

text

---

📌 **box-sizing: border-box** (recomendado)
width = content + padding + border
total = width (ya incluye padding y border)

text

---

🔵 **Ejemplo comparativo**

------------------------------------------------------------
CSS
------------------------------------------------------------
/* Modelo content-box (por defecto) */
.caja1 {
    width: 200px;
    padding: 20px;
    border: 2px solid #333;
    box-sizing: content-box;   /* Valor por defecto */
    /* Ancho total: 200 + 20 + 20 + 2 + 2 = 244px */
}

/* Modelo border-box (recomendado) */
.caja2 {
    width: 200px;
    padding: 20px;
    border: 2px solid #333;
    box-sizing: border-box;
    /* Ancho total: 200px (padding y borde incluidos) */
}
------------------------------------------------------------

---

📌 **La regla de oro**

**Usa `box-sizing: border-box` en todos los elementos.**

Se aplica globalmente así:

------------------------------------------------------------
CSS
------------------------------------------------------------
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}
------------------------------------------------------------

Esto facilita enormemente el cálculo de tamaños y evita sorpresas.

---

🔵 **Ejemplo práctico con border-box**

------------------------------------------------------------
CSS
------------------------------------------------------------
/* Reset global */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

.contenedor {
    width: 300px;
    padding: 20px;
    border: 3px solid #333;
    background-color: #f0f0f0;
}
------------------------------------------------------------

**Resultado:** El contenedor tendrá 300px de ancho total,
con 20px de padding y 3px de borde incluidos en ese tamaño.

===============================================================================
6. CÁLCULO DEL TAMAÑO TOTAL
===============================================================================

---

📌 **Modelo content-box (por defecto)**
Ancho total = width + padding-left + padding-right + border-left + border-right
Alto total = height + padding-top + padding-bottom + border-top + border-bottom

text

---

📌 **Modelo border-box (recomendado)**
Ancho total = width (ya incluye padding y border)
Alto total = height (ya incluye padding y border)

text

---

🔵 **Ejemplo de cálculo**
width: 200px
padding: 20px
border: 5px solid black

text

**Con content-box:**
- Ancho total = 200 + 20 + 20 + 5 + 5 = 250px

**Con border-box:**
- Ancho total = 200px (incluye padding y border)
- Espacio para contenido = 200 - 40 - 10 = 150px

===============================================================================
LA ANALOGÍA DEL PROFESOR BÚHO: EL MODELO DE CAJA COMO UNA CASA
===============================================================================

Imagina que el Modelo de Caja es una casa.

- **Contenido**: Es la habitación donde vives. Es lo que importa.
- **Padding**: Es el aislante entre la habitación y las paredes.
- **Border**: Son las paredes de la casa.
- **Margin**: Es el jardín o el espacio entre tu casa y la del vecino.

- `width` y `height`: El tamaño de la habitación.
- `padding`: El grosor del aislante.
- `border`: El grosor de las paredes.
- `margin`: El tamaño del jardín.

Si usas `box-sizing: border-box`, el tamaño que defines incluye la habitación,
el aislante y las paredes. Es más fácil de manejar.

===============================================================================
ERRORES COMUNES CON EL MODELO DE CAJA
===============================================================================

❌ **Error 1: Olvidar que padding y border suman al tamaño**

   Si defines `width: 200px` y luego añades `padding: 20px`, el elemento
   ocupará más de 200px. Usa `box-sizing: border-box` para evitarlo.

---

❌ **Error 2: No usar `box-sizing: border-box` globalmente**

   Aplica `box-sizing: border-box` a todos los elementos con el selector universal.

---

❌ **Error 3: Confundir padding con margin**

   - `padding`: espacio **dentro** del borde.
   - `margin`: espacio **fuera** del borde.

---

❌ **Error 4: No entender el colapso de márgenes**

   Los márgenes verticales se colapsan (se suman tomando el mayor).
   Si no lo sabes, puedes pensar que hay más espacio del que realmente hay.

---

❌ **Error 5: Usar márgenes negativos sin entenderlos**

   `margin: -10px` puede mover elementos, pero es una técnica avanzada.
   Úsala con cuidado.

GLOSARIO DEL CAPÍTULO
===============================================================================

- **Modelo de Caja (Box Model)**: Concepto que describe cómo se representan los elementos como cajas rectangulares.
- **Contenido (content)**: El área donde está el texto o los elementos hijos.
- **Relleno (padding)**: Espacio interno entre el contenido y el borde.
- **Borde (border)**: Línea que rodea el padding.
- **Margen (margin)**: Espacio externo entre el borde y otros elementos.
- **box-sizing**: Propiedad que controla cómo se calcula el tamaño total.
- **content-box**: Modelo por defecto (width solo para contenido).
- **border-box**: Modelo recomendado (width incluye padding y border).
- **Colapso de márgenes**: Fenómeno donde los márgenes verticales se combinan.
- **Shorthand**: Sintaxis abreviada para escribir propiedades.

===============================================================================
PREGUNTAS FRECUENTES (FAQ)
===============================================================================

❓ **¿Qué es el Modelo de Caja?**

   Es la forma en que el navegador representa cada elemento HTML como una caja
   con contenido, padding, borde y margen.

---

❓ **¿Cuál es la diferencia entre padding y margin?**

   Padding es espacio **dentro** del elemento. Margin es espacio **fuera** del elemento.

---

❓ **¿Qué hace `box-sizing: border-box`?**

   Hace que `width` y `height` incluyan el padding y el borde.
   Así el tamaño total es el que defines, sin sorpresas.

---

❓ **¿Qué es el colapso de márgenes?**

   Cuando dos márgenes verticales se encuentran, el mayor prevalece.
   Solo ocurre en dirección vertical.

---

❓ **¿Debo usar `box-sizing: border-box` siempre?**

   Sí, es una buena práctica aplicarlo globalmente a todos los elementos.
