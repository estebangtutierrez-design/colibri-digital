Ahora es momento de hablar del **borde (border)**.

El borde es la línea que rodea el padding y el contenido de un elemento.
Es una de las propiedades más visibles y versátiles de CSS.

- Puede ser sólido o punteado.

- Puede tener esquinas redondeadas.

===============================================================================
INTRODUCCIÓN: ¿QUÉ ES EL BORDER?
===============================================================================

El `border` es la línea que rodea el padding y el contenido de un elemento.

Imagina que tienes una fotografía enmarcada.

- La fotografía es el **contenido**.
- El espacio entre la fotografía y el marco es el **padding**.
- El marco es el **borde**.
- El espacio entre el marco y otros marcos es el **margin**.

---

📌 **Representación visual del border**
+---------------------------------------------------+
| BORDE (border) |
| +---------------------------------------------+ |
| | PADDING (relleno) | |
| | +---------------------------------------+ | |
| | | CONTENIDO (content) | | |
| | +---------------------------------------+ | |
| +---------------------------------------------+ |
+---------------------------------------------------+

text

---

📌 **Propiedades del border**

| Propiedad | Descripción |
|-----------|-------------|
| `border` | Shorthand para ancho, estilo y color |
| `border-width` | Grosor del borde |
| `border-style` | Estilo del borde (solid, dashed, dotted, etc.) |
| `border-color` | Color del borde |
| `border-radius` | Redondeo de las esquinas |
| `border-top`, `border-right`, `border-bottom`, `border-left` | Borde por lado |

---

📌 **Características del border**

1. **El borde se dibuja entre el padding y el margin**.
2. **Afecta al tamaño total del elemento** (con content-box).
3. **Puede tener diferentes grosores, estilos y colores en cada lado**.
4. **Puede ser redondeado con `border-radius`**.
5. **Puede ser invisible (`border: none`)** pero ocupar espacio si tiene ancho.

===============================================================================
1. BORDER: SHORTHAND COMPLETO
===============================================================================

El shorthand `border` te permite definir el ancho, estilo y color
en una sola línea.

---

📌 **Sintaxis**
border: width style color;

text

- **width**: Grosor del borde (px, em, rem, etc.)
- **style**: Estilo del borde (solid, dashed, dotted, etc.)
- **color**: Color del borde (cualquier formato de color)

---

🔵 **Ejemplo básico**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    border: 2px solid #333;
    padding: 20px;
    background-color: #f0f0f0;
}
------------------------------------------------------------

**Resultado:** Un borde de 2px, sólido, de color gris oscuro.

---

📌 **Estilos de borde**

| Estilo | Descripción | Ejemplo visual |
|--------|-------------|----------------|
| `solid` | Línea sólida | ───────── |
| `dashed` | Línea discontinua (guiones) | ─ ─ ─ ─ |
| `dotted` | Línea de puntos | · · · · · |
| `double` | Línea doble | ═════════ |
| `groove` | Efecto de relieve (hundido) | |
| `ridge` | Efecto de relieve (saliente) | |
| `inset` | Efecto de hundimiento | |
| `outset` | Efecto de saliente | |
| `none` | Sin borde (no ocupa espacio) | |
| `hidden` | Oculto (similar a none) | |

---

🔵 **Ejemplo con diferentes estilos**

------------------------------------------------------------
CSS
------------------------------------------------------------
.solid {
    border: 2px solid #333;
}

.dashed {
    border: 2px dashed #333;
}

.dotted {
    border: 2px dotted #333;
}

.double {
    border: 4px double #333;
}

.groove {
    border: 4px groove #333;
}

.ridge {
    border: 4px ridge #333;
}

.inset {
    border: 4px inset #333;
}

.outset {
    border: 4px outset #333;
}
------------------------------------------------------------

---

📌 **Borde invisible (none)**
border: none;

text

No se dibuja ningún borde.
El elemento no ocupa espacio de borde.
border: hidden;

text

Similar a `none`, pero usado en contextos de tabla.

===============================================================================
2. PROPIEDADES INDIVIDUALES DEL BORDE
===============================================================================

Puedes controlar el ancho, el estilo y el color del borde
de forma independiente.

---

📌 **border-width**

Define el grosor del borde.
border-width: 2px;
border-width: 1px 2px 3px 4px; /* arriba, derecha, abajo, izquierda */

text

---

📌 **border-style**

Define el estilo del borde.
border-style: solid;
border-style: solid dashed dotted double;

text

---

📌 **border-color**

Define el color del borde.
border-color: red;
border-color: red blue green yellow;

text

---

🔵 **Ejemplo con propiedades separadas**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    border-width: 3px;
    border-style: dashed;
    border-color: #007BFF;
    padding: 20px;
}
------------------------------------------------------------

**Resultado:** Un borde de 3px, discontinuo, de color azul.

---

📌 **Abreviaturas para cada lado**

Puedes definir el borde de cada lado por separado.

| Propiedad | Descripción |
|-----------|-------------|
| `border-top` | Borde superior |
| `border-right` | Borde derecho |
| `border-bottom` | Borde inferior |
| `border-left` | Borde izquierdo |

---

🔵 **Ejemplo con bordes diferentes en cada lado**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    border-top: 3px solid #007BFF;
    border-right: 2px dashed #28A745;
    border-bottom: 4px dotted #DC3545;
    border-left: 2px solid #FFC107;
    padding: 20px;
}
------------------------------------------------------------

---

📌 **Propiedades individuales por lado**

| Propiedad | Descripción |
|-----------|-------------|
| `border-top-width` | Grosor del borde superior |
| `border-top-style` | Estilo del borde superior |
| `border-top-color` | Color del borde superior |
| `border-right-width` | Grosor del borde derecho |
| `border-right-style` | Estilo del borde derecho |
| `border-right-color` | Color del borde derecho |
| `border-bottom-width` | Grosor del borde inferior |
| `border-bottom-style` | Estilo del borde inferior |
| `border-bottom-color` | Color del borde inferior |
| `border-left-width` | Grosor del borde izquierdo |
| `border-left-style` | Estilo del borde izquierdo |
| `border-left-color` | Color del borde izquierdo |

---

🔵 **Ejemplo con propiedades individuales**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    border-top-width: 3px;
    border-top-style: solid;
    border-top-color: #007BFF;

    border-right-width: 2px;
    border-right-style: dashed;
    border-right-color: #28A745;

    padding: 20px;
}

===============================================================================
3. BORDER-RADIUS: ESQUINAS REDONDEADAS
===============================================================================

`border-radius` redondea las esquinas de un elemento.

---

📌 **Sintaxis básica**
border-radius: valor;

text

Aplica el mismo redondeo a las cuatro esquinas.

---

🔵 **Ejemplo básico**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    border: 2px solid #333;
    border-radius: 10px;   /* Esquinas redondeadas */
    padding: 20px;
}
------------------------------------------------------------

---

📌 **Sintaxis con dos valores**
border-radius: top-left-top-right bottom-left-bottom-right;

text

- Primer valor: esquinas superiores (izquierda y derecha).
- Segundo valor: esquinas inferiores (izquierda y derecha).

---

🔵 **Ejemplo con dos valores**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    border: 2px solid #333;
    border-radius: 10px 20px;   /* 10px arriba, 20px abajo */
    padding: 20px;
}
------------------------------------------------------------

---

📌 **Sintaxis con cuatro valores**
border-radius: top-left top-right bottom-right bottom-left;

text

- top-left: esquina superior izquierda
- top-right: esquina superior derecha
- bottom-right: esquina inferior derecha
- bottom-left: esquina inferior izquierda

---

🔵 **Ejemplo con cuatro valores**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    border: 2px solid #333;
    border-radius: 10px 20px 30px 40px;   /* Cada esquina diferente */
    padding: 20px;
}
------------------------------------------------------------

---

📌 **Esquinas individuales**

| Propiedad | Descripción |
|-----------|-------------|
| `border-top-left-radius` | Esquina superior izquierda |
| `border-top-right-radius` | Esquina superior derecha |
| `border-bottom-right-radius` | Esquina inferior derecha |
| `border-bottom-left-radius` | Esquina inferior izquierda |

---

🔵 **Ejemplo con esquinas individuales**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    border: 2px solid #333;
    border-top-left-radius: 10px;
    border-top-right-radius: 20px;
    border-bottom-right-radius: 30px;
    border-bottom-left-radius: 40px;
    padding: 20px;
}
------------------------------------------------------------

---

📌 **Crear círculos con border-radius**

Si usas `border-radius: 50%` en un elemento cuadrado, obtienes un círculo.

------------------------------------------------------------
CSS
------------------------------------------------------------
.circulo {
    width: 100px;
    height: 100px;
    border-radius: 50%;
    background-color: #3498DB;
}
------------------------------------------------------------

---

📌 **Crear óvalos con border-radius**

Si usas `border-radius: 50%` en un elemento rectangular, obtienes un óvalo.

------------------------------------------------------------
CSS
------------------------------------------------------------
.oval {
    width: 150px;
    height: 100px;
    border-radius: 50%;
    background-color: #3498DB;
}
------------------------------------------------------------

---

📌 **Valores especiales**

- `border-radius: 50%` → círculo o elipse.
- `border-radius: 9999px` → esquinas completamente redondeadas (efecto "pastilla").

---

🔵 **Ejemplo: botón estilo "pastilla"**

------------------------------------------------------------
CSS
------------------------------------------------------------
.boton {
    padding: 10px 30px;
    border: none;
    border-radius: 9999px;   /* Esquinas completamente redondeadas */
    background-color: #007BFF;
    color: white;
    cursor: pointer;
}
------------------------------------------------------------

===============================================================================
4. BORDER Y EL MODELO DE CAJA
===============================================================================

El borde afecta al tamaño total del elemento.

---

📌 **Modelo content-box (por defecto)**

En el modelo estándar, el borde se **suma** al ancho y alto.
Ancho total = width + padding-left + padding-right + border-left + border-right
Alto total = height + padding-top + padding-bottom + border-top + border-bottom

text

---

🔵 **Ejemplo con content-box**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    width: 200px;
    padding: 20px;
    border: 5px solid #333;
    box-sizing: content-box;   /* Por defecto */
}
------------------------------------------------------------

**Cálculo del ancho total:**
- width: 200px
- padding-left: 20px
- padding-right: 20px
- border-left: 5px
- border-right: 5px

**Total:** 200 + 20 + 20 + 5 + 5 = **250px**

---

📌 **Modelo border-box (recomendado)**

En el modelo alternativo, el borde se **incluye** dentro del ancho y alto.
Ancho total = width (ya incluye padding y border)
Alto total = height (ya incluye padding y border)

text

---

🔵 **Ejemplo con border-box**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    width: 200px;
    padding: 20px;
    border: 5px solid #333;
    box-sizing: border-box;
}
------------------------------------------------------------

**Cálculo del ancho total:**
- width: 200px (incluye padding y border)
- Espacio para contenido: 200 - 40 - 10 = **150px**

**Total:** **200px**

---

🔴 **Importante**

Usar `box-sizing: border-box` hace que el borde no "rompa" el diseño
porque no aumenta el tamaño total del elemento.

**Siempre usa `box-sizing: border-box` globalmente.**

------------------------------------------------------------
CSS
------------------------------------------------------------

    border: none;
    padding: 10px 20px;

    color: white;

===============================================================================
6. BORDER Y ACCESIBILIDAD
===============================================================================

El borde también es importante para la accesibilidad.

---

📌 **Enlaces con borde visible**

Los enlaces deben tener un indicador visible de que son interactivos.

------------------------------------------------------------
CSS
------------------------------------------------------------
a {
    border-bottom: 2px solid transparent;
    text-decoration: none;
    transition: border-color 0.3s;
}

a:hover {
    border-bottom-color: #007BFF;
}
------------------------------------------------------------

---

📌 **Focus visible en formularios**

Nunca elimines el `outline` sin proporcionar una alternativa visible.

------------------------------------------------------------
CSS
------------------------------------------------------------
/* ❌ MALA PRÁCTICA */
input:focus {
    outline: none;
}

/* ✅ BUENA PRÁCTICA */
input:focus {
    outline: 3px solid #007BFF;
    outline-offset: 2px;
}

/* ✅ BUENA PRÁCTICA ALTERNATIVA */
input:focus {
    outline: none;
    border-color: #007BFF;
    box-shadow: 0 0 0 3px rgba(0,123,255,0.25);
}
------------------------------------------------------------

===============================================================================
LA ANALOGÍA DEL PROFESOR BÚHO: EL BORDE COMO EL MARCO DE UN CUADRO
===============================================================================

Imagina que el contenido de un elemento es una obra de arte.

- La obra de arte es el **contenido**.
- El espacio entre la obra y el marco es el **padding**.
- El **borde** es el marco de la obra.
- El espacio entre el marco y la pared es el **margin**.

El marco define la obra, la protege y la hace destacar.

- Un marco grueso llama la atención.
- Un marco delgado es sutil y elegante.
- Un marco sin color (transparente) no se nota.
- Un marco con esquinas redondeadas es más suave.

El borde en CSS cumple la misma función.

===============================================================================
ERRORES COMUNES CON BORDER
===============================================================================

❌ **Error 1: Olvidar que border suma al tamaño total**

   Con `box-sizing: content-box`, el borde aumenta el tamaño del elemento.
   **Solución:** usa `box-sizing: border-box`.

---

❌ **Error 2: Usar border sin border-style**

   `border: 2px red;` no funcionará porque falta el estilo.
   **Solución:** siempre define los tres valores: `border: 2px solid red;`.

---

❌ **Error 3: No usar border-radius correctamente**

   Para crear un círculo, el elemento debe tener el mismo ancho y alto,
   y `border-radius: 50%`.

---

❌ **Error 4: Eliminar outline sin alternativa**

   Los usuarios de teclado dependen del `:focus` para navegar.
   **Solución:** siempre proporciona un indicador visual alternativo.

---

❌ **Error 5: Usar demasiados bordes diferentes**

   Un diseño con 10 tipos de borde diferentes se ve inconsistente.
   **Solución:** mantén una o dos variedades de borde en todo el proyecto.

GLOSARIO DEL CAPÍTULO
===============================================================================

- **Border**: Línea que rodea el padding y el contenido de un elemento.
- **Border-width**: Grosor del borde.
- **Border-style**: Estilo del borde (solid, dashed, dotted, etc.).
- **Border-color**: Color del borde.
- **Border-radius**: Redondeo de las esquinas.
- **Shorthand**: Sintaxis abreviada para escribir varias propiedades en una línea.
- **Box-sizing: content-box**: Modelo por defecto donde border suma al tamaño.
- **Box-sizing: border-box**: Modelo donde border se incluye en el tamaño.
- **Solid**: Borde sólido.
- **Dashed**: Borde discontinuo (guiones).
- **Dotted**: Borde de puntos.
- **Double**: Borde doble.
- **Groove**: Borde con efecto de relieve (hundido).
- **Ridge**: Borde con efecto de relieve (saliente).
- **Inset**: Borde de hundimiento.
- **Outset**: Borde de saliente.

===============================================================================
PREGUNTAS FRECUENTES (FAQ)
===============================================================================

❓ **¿Cuál es la diferencia entre border y outline?**

   El `border` forma parte del Modelo de Caja y afecta al tamaño del elemento.
   El `outline` se dibuja **fuera** del borde y no afecta al tamaño.
   El `outline` no puede tener diferentes lados (es un solo borde completo).

---

❓ **¿Cómo se crea un círculo en CSS?**

   Con un elemento cuadrado (width = height) y `border-radius: 50%`.

---

❓ **¿Qué es border-radius: 9999px?**

   Es una técnica para crear esquinas completamente redondeadas
   (efecto "pastilla" o "píldora").

---

❓ **¿Por qué mi borde no se ve aunque lo he definido?**

   Puede ser porque olvidaste definir `border-style`.
   El borde necesita ancho, estilo y color. El estilo es obligatorio.

---

❓ **¿Cómo evito que el borde aumente el tamaño de mi elemento?**

   Usa `box-sizing: border-box` globalmente.
