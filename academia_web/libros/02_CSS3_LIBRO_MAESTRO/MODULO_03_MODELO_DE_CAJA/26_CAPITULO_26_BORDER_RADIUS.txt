===============================================================================
INTRODUCCIÓN: ¿QUÉ ES BORDER-RADIUS?
===============================================================================

`border-radius` es una propiedad CSS que redondea las esquinas de un elemento.

Permite transformar rectángulos perfectos en formas con esquinas curvas,
suaves y modernas.

---

📌 **Sintaxis básica**
border-radius: valor;

text

Aplica el mismo redondeo a las cuatro esquinas.

---

🔵 **Ejemplo básico**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    border: 2px solid #333;
    border-radius: 10px;   /* Todas las esquinas redondeadas */
    padding: 20px;
    background-color: #f0f0f0;
}
------------------------------------------------------------

**Resultado:** Un rectángulo con las cuatro esquinas redondeadas.

---

📌 **Valores de border-radius**

| Valor | Descripción |
|-------|-------------|
| `px` | Píxeles (redondeo fijo) |
| `%` | Porcentaje (redondeo relativo al tamaño) |
| `em`, `rem` | Relativo al tamaño de fuente |

---

📌 **Características de border-radius**

1. **El valor máximo es el 50%** del lado más corto.
2. **Con píxeles**, el redondeo es siempre del mismo tamaño.
3. **Con porcentajes**, el redondeo se adapta al tamaño del elemento.
4. **Se puede aplicar a cualquier elemento** que tenga un borde o fondo visible.
5. **No afecta al tamaño total** del elemento.

===============================================================================
1. BORDER-RADIUS: SINTAXIS COMPLETA
===============================================================================

---

📌 **Un valor (todas las esquinas)**
border-radius: 10px;

text

Aplica el mismo redondeo a las cuatro esquinas.

---

🔵 **Ejemplo**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    border-radius: 10px;
    background-color: #3498DB;
    padding: 20px;
    color: white;
}
------------------------------------------------------------

---

📌 **Dos valores (superiores e inferiores)**
border-radius: 10px 20px;

text

- Primer valor: esquinas superiores (izquierda y derecha).
- Segundo valor: esquinas inferiores (izquierda y derecha).

---

🔵 **Ejemplo**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    border-radius: 10px 30px;
    background-color: #3498DB;
    padding: 20px;
    color: white;
}
------------------------------------------------------------

**Resultado:** Las esquinas superiores tienen 10px de radio.
Las esquinas inferiores tienen 30px de radio.

---

📌 **Tres valores (superior izquierda, superior derecha/inferior izquierda, inferior derecha)**
border-radius: 10px 20px 30px;

text

- Primer valor: esquina superior izquierda.
- Segundo valor: esquinas superior derecha e inferior izquierda.
- Tercer valor: esquina inferior derecha.

---

🔵 **Ejemplo**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    border-radius: 10px 20px 30px;
    background-color: #3498DB;
    padding: 20px;
    color: white;
}
------------------------------------------------------------

**Resultado:**
- Superior izquierda: 10px.
- Superior derecha e inferior izquierda: 20px.
- Inferior derecha: 30px.

---

📌 **Cuatro valores (cada esquina)**
border-radius: top-left top-right bottom-right bottom-left;

text

- Primer valor: esquina superior izquierda.
- Segundo valor: esquina superior derecha.
- Tercer valor: esquina inferior derecha.
- Cuarto valor: esquina inferior izquierda.

---

🔵 **Ejemplo**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    border-radius: 10px 20px 30px 40px;
    background-color: #3498DB;
    padding: 20px;
    color: white;
}
------------------------------------------------------------

**Resultado:**
- Superior izquierda: 10px.
- Superior derecha: 20px.
- Inferior derecha: 30px.
- Inferior izquierda: 40px.

---

📌 **Regla mnemotécnica (sentido horario)**

Los valores de `border-radius` se aplican en **sentido horario**:
superior izquierda → superior derecha → inferior derecha → inferior izquierda

text

Es como las agujas del reloj: empiezas arriba a la izquierda y giras a la derecha.

===============================================================================
2. PROPIEDADES INDIVIDUALES DE BORDER-RADIUS
===============================================================================

Puedes controlar cada esquina por separado.

---

📌 **Propiedades individuales**

| Propiedad | Descripción |
|-----------|-------------|
| `border-top-left-radius` | Esquina superior izquierda |
| `border-top-right-radius` | Esquina superior derecha |
| `border-bottom-right-radius` | Esquina inferior derecha |
| `border-bottom-left-radius` | Esquina inferior izquierda |

---

🔵 **Ejemplo con propiedades individuales**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    border-top-left-radius: 10px;
    border-top-right-radius: 20px;
    border-bottom-right-radius: 30px;
    border-bottom-left-radius: 40px;
    background-color: #3498DB;
    padding: 20px;
    color: white;
}
------------------------------------------------------------

---

📌 **Valores asimétricos (elipse)**

Puedes usar dos valores separados por una barra (`/`) para crear redondeos
elípticos (vertical y horizontal diferentes).
border-radius: horizontal / vertical;

text

---

🔵 **Ejemplo de redondeo elíptico**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    border-radius: 20px / 40px;   /* Horizontal 20px, Vertical 40px */
    background-color: #3498DB;
    padding: 20px;
    color: white;
    width: 200px;
    height: 100px;
}
------------------------------------------------------------

---

📌 **Valores asimétricos con cuatro valores**

Puedes tener diferentes valores para cada esquina en horizontal y vertical.
border-radius: 10px 20px 30px 40px / 50px 60px 70px 80px;

text

===============================================================================
3. BORDER-RADIUS CON PORCENTAJES
===============================================================================

Usar porcentajes en `border-radius` hace que el redondeo se adapte
al tamaño del elemento.

---

📌 **Porcentajes básicos**

| Porcentaje | Efecto |
|------------|--------|
| `border-radius: 0%;` | Esquinas rectas (sin redondeo) |
| `border-radius: 10%;` | Esquinas ligeramente redondeadas |
| `border-radius: 25%;` | Redondeo notable |
| `border-radius: 50%;` | Círculo o elipse perfecta |

---

🔵 **Ejemplo: esquinas redondeadas con %**

------------------------------------------------------------
CSS
------------------------------------------------------------
.caja {
    border-radius: 10%;
    background-color: #3498DB;
    padding: 20px;
    color: white;
    width: 200px;
    height: 100px;
}
------------------------------------------------------------

---

🔵 **Ejemplo: círculo perfecto con %**

------------------------------------------------------------
CSS
------------------------------------------------------------
.circulo {
    width: 100px;
    height: 100px;
    border-radius: 50%;
    background-color: #3498DB;
}
------------------------------------------------------------

**Condición:** el elemento debe tener el mismo ancho y alto.

---

🔵 **Ejemplo: elipse perfecta con %**

------------------------------------------------------------
CSS
------------------------------------------------------------
.elipse {
    width: 150px;
    height: 100px;
    border-radius: 50%;
    background-color: #3498DB;
}
------------------------------------------------------------

**Condición:** el elemento debe tener ancho y alto diferentes.

---

📌 **Diferencias entre px y %**

| Característica | Píxeles (px) | Porcentaje (%) |
|----------------|--------------|----------------|
| **Tamaño** | Fijo | Relativo al tamaño del elemento |
| **Adaptabilidad** | No se adapta | Se adapta al tamaño |
| **Círculo perfecto** | Necesitas calcular | `border-radius: 50%` siempre funciona |
| **Uso común** | Esquinas ligeramente redondeadas | Círculos, elipses, formas adaptables |

===============================================================================
4. BORDER-RADIUS: 9999PX (EFECTO PASTILLA)
===============================================================================

`border-radius: 9999px` es una técnica para crear elementos
con esquinas completamente redondeadas, típicamente botones
o etiquetas tipo "pastilla" o "píldora".

---

📌 **¿Por qué 9999px?**

Un valor muy alto (como 9999px) asegura que las esquinas sean
completamente redondeadas, independientemente del tamaño del elemento.

---

🔵 **Ejemplo: botón tipo pastilla**

------------------------------------------------------------
CSS
------------------------------------------------------------
.boton-pastilla {
    padding: 12px 30px;
    background-color: #007BFF;
    color: white;
    border: none;
    border-radius: 9999px;   /* Esquinas completamente redondeadas */
    cursor: pointer;
    font-size: 1rem;
}
------------------------------------------------------------

---

📌 **Alternativas a 9999px**

- `border-radius: 50px;` (si el elemento es pequeño y sabes el tamaño).
- `border-radius: 9999px;` (la más segura y universal).

---

🔵 **Ejemplo: etiqueta tipo pastilla**

------------------------------------------------------------
CSS
------------------------------------------------------------
.etiqueta {
    display: inline-block;
    padding: 4px 12px;
    background-color: #28A745;
    color: white;
    border-radius: 9999px;
    font-size: 0.75rem;
    font-weight: 600;
}
------------------------------------------------------------

===============================================================================
7. BORDER-RADIUS Y ACCESIBILIDAD
===============================================================================

El `border-radius` también puede mejorar la accesibilidad.

---

📌 **Botones con esquinas redondeadas**

Los botones con esquinas redondeadas son más fáciles de reconocer
como elementos interactivos.

---

📌 **No excederse con el redondeo**

Un redondeo excesivo (como botones completamente redondos)
puede hacer que el texto se vea cortado o dificultar la lectura.

---

📌 **Contraste en elementos con border-radius**

Asegúrate de que el contraste entre el texto y el fondo sea suficiente,
especialmente en elementos con esquinas redondeadas.

---

📌 **Preferencias del usuario**

Algunos usuarios con problemas de procesamiento visual prefieren
esquinas rectas. No hay una forma estándar de desactivar el `border-radius`,
pero los navegadores modernos permiten a los usuarios establecer
sus propias hojas de estilo.

===============================================================================
LA ANALOGÍA DEL PROFESOR BÚHO: EL BORDER-RADIUS COMO EL PULIDO DE UNA PIEDRA
===============================================================================

Imagina que el diseño CSS es como trabajar con piedras.

- Una piedra sin pulir tiene bordes afilados y ásperos.
- Una piedra pulida tiene bordes suaves y agradables al tacto.

El `border-radius` es el pulido de tu diseño.

- 0px: la piedra sin pulir (afilada, dura, cuadrada).
- 10px: la piedra ligeramente pulida (todavía cuadrada, pero más suave).
- 50%: la piedra completamente pulida (redonda, suave, agradable).
- 9999px: la piedra convertida en una cuenta de collar (completamente redondeada).

El pulido transforma una piedra áspera en una joya.
El `border-radius` transforma un diseño rígido en uno moderno y acogedor.

===============================================================================
ERRORES COMUNES CON BORDER-RADIUS
===============================================================================

❌ **Error 1: Olvidar que un círculo necesita width = height**

   `border-radius: 50%` solo crea un círculo si el elemento tiene
   el mismo ancho y alto. Si no, crea una elipse.

---

❌ **Error 2: Usar border-radius en elementos muy pequeños**

   Si el elemento es muy pequeño, el redondeo puede distorsionarse.

---

❌ **Error 3: Usar valores negativos**

   `border-radius` no acepta valores negativos.

---

❌ **Error 4: No probar en diferentes tamaños de pantalla**

   `border-radius: 50%` se adapta, pero `border-radius: 20px` es fijo.
   En pantallas muy pequeñas, puede verse demasiado grande.

---

❌ **Error 5: Usar border-radius excesivo en texto**

   Si el redondeo es demasiado grande, el texto puede quedar cortado
   o verse extraño.

---

❌ **Error 6: No usar border-radius en elementos con overflow**

   Si un elemento tiene `overflow: hidden`, el `border-radius` recortará
   el contenido extra.

GLOSARIO DEL CAPÍTULO
===============================================================================

- **Border-radius**: Propiedad CSS que redondea las esquinas de un elemento.
- **Redondeo**: Curva que sustituye las esquinas rectas de un elemento.
- **Radio**: Distancia desde el centro de la curva hasta la esquina original.
- **Círculo perfecto**: Elemento cuadrado con `border-radius: 50%`.
- **Elipse**: Elemento rectangular con `border-radius: 50%`.
- **Efecto pastilla**: Elemento con `border-radius: 9999px` (esquinas completamente redondeadas).
- **Shorthand**: Sintaxis abreviada para escribir varias propiedades en una línea.
- **Overflow: hidden**: Recorta el contenido que sobresale del elemento.

===============================================================================
PREGUNTAS FRECUENTES (FAQ)
===============================================================================

❓ **¿Cuál es la diferencia entre border-radius: 50% y border-radius: 9999px?**

   `border-radius: 50%` crea un círculo perfecto (si width = height).
   `border-radius: 9999px` crea un efecto "pastilla" (esquinas completamente redondeadas)
   que se adapta a cualquier tamaño.

---

❓ **¿Por qué mi círculo no es un círculo perfecto?**

   Porque el elemento no tiene el mismo ancho y alto.
   `border-radius: 50%` solo crea un círculo si `width = height`.

---

❓ **¿Cómo hago un botón redondo?**

   Con `width: 50px; height: 50px; border-radius: 50%;`.

---

❓ **¿Cómo hago un botón tipo pastilla?**

   Con `padding: 10px 30px; border-radius: 9999px;`.

---

❓ **¿Puedo usar border-radius en imágenes?**

   Sí, puedes usar `border-radius` en imágenes. Si la imagen tiene
   `object-fit: cover`, el redondeo se aplica correctamente.
