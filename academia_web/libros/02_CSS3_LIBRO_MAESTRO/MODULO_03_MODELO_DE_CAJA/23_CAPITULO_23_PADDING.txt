APLICACIÓN EN EL MUNDO REAL
===============================================================================

El padding se utiliza en proyectos reales para:

1. Botones: separar el texto del borde del botón.
2. Tarjetas: crear espacio interior en cards y contenedores.
3. Tablas: separar el contenido de las celdas.
4. Formularios: dar espacio a los campos de entrada.
5. Artículos: crear márgenes interiores en párrafos y títulos.

Sin padding, cualquier página web se vería como un texto apretado y
difícil de leer. El padding es lo que da "aire" al diseño.

===============================================================================
HISTORIA
===============================================================================

La propiedad padding existe desde CSS1 (1996). Desde el principio, su
propósito fue crear espacio interior entre el contenido de un elemento
y su borde.

Antes de CSS, los diseñadores usaban imágenes transparentes y tablas
para crear espacios. Era un trabajo tedioso y poco flexible.

Con la llegada del padding, los diseñadores pudieron controlar el
espacio interior de cualquier elemento con una simple línea de código.
Esto revolucionó el diseño web y permitió crear interfaces mucho más
limpias y profesionales.

Hoy, el padding es una de las propiedades más utilizadas en cualquier
archivo CSS. Es imposible encontrar un sitio web profesional que no
haga uso de ella.

TEORÍA PROFUNDA
===============================================================================

¿QUÉ ES EL PADDING?

El padding es el espacio interior de un elemento. Es la distancia entre
el contenido (texto, imágenes, etc.) y el borde del elemento.

El padding forma parte del Modelo de Caja:

┌─────────────────────────────────────────────────────┐
│                      MARGIN                          │
│  ┌───────────────────────────────────────────────┐  │
│  │                   BORDER                       │  │
│  │  ┌─────────────────────────────────────────┐  │  │
│  │  │                PADDING                   │  │  │
│  │  │  ┌───────────────────────────────────┐  │  │  │
│  │  │  │           CONTENIDO               │  │  │  │
│  │  │  │                                   │  │  │  │
│  │  │  └───────────────────────────────────┘  │  │  │
│  │  └─────────────────────────────────────────┘  │  │
│  └───────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────┘

El padding es transparente y no puede tener color propio. Solo afecta
al espacio interior del elemento.

===============================================================================
SINTAXIS COMPLETA DEL PADDING
===============================================================================

El padding tiene 4 sub-propiedades:

1. padding-top:     Espacio superior.
2. padding-right:   Espacio derecho.
3. padding-bottom:  Espacio inferior.
4. padding-left:    Espacio izquierdo.

También tiene una propiedad abreviada: padding.

===============================================================================
DIAGRAMA EN TEXTO
===============================================================================

┌─────────────────────────────────────────────────┐
│                   padding-top                    │
│                                                   │
│   padding-left   ╔═══════════════════════╗   padding-right
│                  ║                      ║         │
│                  ║      CONTENIDO       ║         │
│                  ║                      ║         │
│                  ╚═══════════════════════╝         │
│                                                   │
│                  padding-bottom                    │
└─────────────────────────────────────────────────┘

Cada lado puede tener un valor diferente.

===============================================================================
CÓDIGO COMPLETO: FORMAS DE ESCRIBIR PADDING
===============================================================================

CÓDIGO CSS
------------------------------------------------------------

/* FORMA 1: Valor único (los 4 lados iguales) */
.caja-1 {
    padding: 20px;
}
/* top: 20px, right: 20px, bottom: 20px, left: 20px */

/* FORMA 2: Dos valores (vertical | horizontal) */
.caja-2 {
    padding: 10px 30px;
}
/* top: 10px, right: 30px, bottom: 10px, left: 30px */

/* FORMA 3: Tres valores (top | horizontal | bottom) */
.caja-3 {
    padding: 10px 20px 30px;
}
/* top: 10px, right: 20px, bottom: 30px, left: 20px */

/* FORMA 4: Cuatro valores (top | right | bottom | left) */
.caja-4 {
    padding: 10px 20px 30px 40px;
}
/* top: 10px, right: 20px, bottom: 30px, left: 40px */

/* FORMA 5: Propiedades individuales */
.caja-5 {
    padding-top: 10px;
    padding-right: 20px;
    padding-bottom: 30px;
    padding-left: 40px;
}

/* FORMA 6: Combinando (útil para sobrescribir un lado) */
.caja-6 {
    padding: 20px;
    padding-top: 40px;  /* Solo la parte superior es diferente */
}

===============================================================================
EXPLICACIÓN LÍNEA POR LÍNEA
===============================================================================

Analicemos el caso de los botones:

   .btn-pequeno { padding: 6px 12px; }
   6px: padding superior e inferior.
   12px: padding izquierdo y derecho.

   .btn-mediano { padding: 10px 24px; }
   Más espacio interior. El botón se ve más grande.

   .btn-grande { padding: 16px 40px; font-size: 20px; }
   Padding grande + fuente grande. Botón de llamada a la acción.

La regla general:
- Menos padding = botón más pequeño y compacto.
- Más padding = botón más grande y llamativo.

===============================================================================
PADDING Y EL TAMAÑO TOTAL
===============================================================================

El padding AFECTA al tamaño total del elemento.

Con box-sizing: content-box (valor por defecto):

   div { width: 300px; padding: 20px; }
   Tamaño total = 300px + 20px + 20px = 340px

Con box-sizing: border-box:

   div { width: 300px; padding: 20px; box-sizing: border-box; }
   Tamaño total = 300px (el padding se resta del contenido)
   Espacio para contenido = 300px - 20px - 20px = 260px

===============================================================================
UNIDADES DE PADDING
===============================================================================

px    → Píxeles. La más común y precisa.
em    → Relativa al tamaño de fuente del elemento.
rem   → Relativa al tamaño de fuente del elemento raíz (html).
%     → Porcentaje del ANCHO del contenedor.
vh/vw → Relativa al viewport.

Recomendación: Usa px para tamaños fijos y em/rem para diseños que
deban escalar con el texto.

===============================================================================
ERRORES COMUNES
===============================================================================

ERROR 1: CONFUNDIR PADDING CON MARGIN

   .caja { margin: 20px; }  /* MAL: espacio EXTERIOR */
   .caja { padding: 20px; } /* BIEN: espacio INTERIOR */

ERROR 2: NO CONSIDERAR QUE EL PADDING AUMENTA EL TAMAÑO

   .caja { width: 300px; padding: 20px; }
   /* Tamaño REAL: 340px si no usas border-box */

ERROR 3: PADDING EN ELEMENTOS INLINE

   El padding horizontal (left, right) funciona en elementos inline.
   El padding vertical (top, bottom) NO empuja a los elementos vecinos.

ERROR 4: PADDING EN PORCENTAJE INESPERADO

   padding: 10%;  /* 10% del ANCHO, no del alto */

===============================================================================
GLOSARIO
===============================================================================

Padding: Espacio interior entre el contenido de un elemento y su borde.
Box Model: Modelo de caja (contenido, padding, border, margin).
Box-sizing: Define cómo se calcula el tamaño total del elemento.
Content-box: Width/height solo incluyen el contenido.
Border-box: Width/height incluyen contenido, padding y border.

===============================================================================
CONSEJO PROFESIONAL
===============================================================================

1. Usa box-sizing: border-box globalmente.
2. Usa una escala de padding (4, 8, 12, 16, 24, 32, 48, 64).
3. Para botones, padding horizontal > vertical (proporción 1:2 o 1:3).
4. En diseños responsivos, considera usar em o porcentaje.
5. No abuses del padding: demasiado puede sentirse vacío.

===============================================================================
¿SABÍAS QUE...?
===============================================================================

¿Sabías que el padding-top y padding-bottom en porcentaje se calculan
sobre el ANCHO del contenedor, no sobre el alto?

Esto se usa para crear elementos con relación de aspecto fija:

   .video-contenedor {
       width: 100%;
       padding-top: 56.25%;  /* 16:9 */
   }
