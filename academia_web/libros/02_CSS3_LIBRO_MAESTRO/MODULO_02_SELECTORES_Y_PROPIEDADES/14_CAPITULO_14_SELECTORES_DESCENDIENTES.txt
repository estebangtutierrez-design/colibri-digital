===============================================================================
INTRODUCCIÓN
===============================================================================

Hasta ahora hemos aprendido a seleccionar elementos mediante su nombre,
utilizando clases, identificadores y el selector universal.

Sin embargo, muchas veces necesitaremos aplicar estilos solamente a los
elementos que se encuentran dentro de una sección específica.

Para resolver este problema existen los Selectores Descendientes.

Este tipo de selector permite indicar que un elemento será seleccionado
únicamente cuando esté contenido dentro de otro elemento.

Es una de las herramientas más utilizadas en el desarrollo web profesional.

===============================================================================
¿QUÉ ES UN SELECTOR DESCENDIENTE?
===============================================================================

Un selector descendiente selecciona un elemento que está contenido dentro de
otro elemento.

Su sintaxis consiste en escribir dos o más selectores separados por un espacio.

Sintaxis

selectorPadre selectorHijo{

propiedades;

}

El espacio entre ambos selectores indica una relación de descendencia.

===============================================================================
PRIMER EJEMPLO
===============================================================================

HTML

<div>

<h1>Bienvenidos</h1>

<p>Primer párrafo.</p>

</div>

CSS

div p{

color:blue;

}

Resultado

Únicamente el párrafo que está dentro del elemento div aparecerá en color azul.

El título no será afectado.

===============================================================================
OTRO EJEMPLO
===============================================================================

HTML

<section>

<p>Este párrafo pertenece a la sección.</p>

</section>

<p>Este párrafo está fuera de la sección.</p>

CSS

section p{

color:red;

}

Resultado

Solo el párrafo que se encuentra dentro de la sección cambiará de color.

===============================================================================
EJEMPLO PROFESIONAL
===============================================================================

HTML

<nav>

<ul>

<li>Inicio</li>

<li>Cursos</li>

<li>Contacto</li>

</ul>

</nav>

CSS

nav li{

display:inline-block;

margin:15px;

font-weight:bold;

}

Todos los elementos de la lista que estén dentro del menú recibirán el estilo.

===============================================================================
OTRO EJEMPLO PROFESIONAL
===============================================================================

HTML

<div class="tarjeta">

<h2>Título</h2>

<p>Descripción del producto.</p>

</div>

CSS

.tarjeta h2{

color:#0055cc;

}

.tarjeta p{

color:#555;

}

Cada elemento recibe un estilo diferente únicamente dentro de la tarjeta.

===============================================================================
VENTAJAS
===============================================================================

✓ Permite aplicar estilos específicos.

✓ Reduce la cantidad de clases innecesarias.

✓ Facilita la organización del código.

✓ Hace el diseño más profesional.

✓ Es ampliamente utilizado en proyectos modernos.

===============================================================================
BUENAS PRÁCTICAS
===============================================================================

Utilizar selectores descendientes únicamente cuando exista una verdadera
relación entre los elementos.

Mantener la estructura HTML organizada.

Evitar escribir cadenas demasiado largas de selectores.

Ejemplo recomendado

main article p{

color:#444;

}

===============================================================================
ERRORES COMUNES
===============================================================================

Error 1

Olvidar el espacio entre los selectores.

Incorrecto

divp{

color:red;

}

Correcto

div p{

color:red;

}

------------------------------------------------------------

Error 2

Crear cadenas excesivamente largas.

Ejemplo poco recomendable

body main section article div p{

color:red;

}

Este tipo de código dificulta el mantenimiento del proyecto.

===============================================================================
DIFERENCIA ENTRE SELECTOR SIMPLE Y DESCENDIENTE
===============================================================================

Selector simple

p{

color:blue;

}

Selecciona todos los párrafos.

------------------------------------------------------------

Selector descendiente

section p{

color:blue;

}

Selecciona únicamente los párrafos que están dentro de la etiqueta section.

RESUMEN
===============================================================================

En este capítulo aprendimos que los Selectores Descendientes permiten aplicar
estilos únicamente a los elementos contenidos dentro de otros elementos.

Su sintaxis utiliza un espacio entre el selector padre y el selector hijo,
permitiendo crear reglas mucho más específicas.

Este tipo de selector ayuda a mantener proyectos organizados, reutilizables y
más fáciles de mantener.
