===============================================================================
INTRODUCCIÓN
===============================================================================

En el capítulo anterior aprendimos el Selector Universal, capaz de aplicar
estilos a todos los elementos de una página.

Ahora aprenderemos uno de los selectores más utilizados en el desarrollo web:
el Selector de Clase.

Las clases permiten aplicar un mismo estilo a diferentes elementos sin importar
si son títulos, párrafos, imágenes o botones.

Gracias a ellas podemos reutilizar código y mantener nuestros proyectos
organizados.

===============================================================================
¿QUÉ ES UNA CLASE?
===============================================================================

Una clase es un nombre que asignamos a uno o varios elementos HTML para poder
aplicarles un mismo conjunto de estilos mediante CSS.

En HTML se utiliza el atributo:

class

Ejemplo:

<p class="importante">
Este texto es importante.
</p>

===============================================================================
SINTAXIS DEL SELECTOR DE CLASE
===============================================================================

En CSS las clases se escriben utilizando un punto (.)

Ejemplo:

.importante{

color:red;

}

El punto indica que estamos seleccionando una clase.

===============================================================================
PRIMER EJEMPLO
===============================================================================

HTML

<h1 class="titulo">
Lucas EGA Academy
</h1>

CSS

.titulo{

color:blue;

font-size:40px;

}

Resultado

El título aparecerá en color azul con un tamaño de 40 píxeles.

===============================================================================
UTILIZANDO LA MISMA CLASE EN VARIOS ELEMENTOS
===============================================================================

HTML

<h1 class="azul">Título</h1>

<p class="azul">Primer párrafo.</p>

<h2 class="azul">Subtítulo</h2>

CSS

.azul{

color:blue;

}

Resultado

Todos los elementos que tengan la clase "azul" mostrarán el mismo color.

===============================================================================
EJEMPLO PROFESIONAL
===============================================================================

HTML

<button class="boton">
Guardar
</button>

<button class="boton">
Cancelar
</button>

<button class="boton">
Eliminar
</button>

CSS

.boton{

background:#0077ff;

color:white;

padding:12px 25px;

border:none;

border-radius:6px;

cursor:pointer;

}

Los tres botones compartirán exactamente el mismo diseño.

===============================================================================
OTRO EJEMPLO PROFESIONAL
===============================================================================

HTML

<div class="tarjeta">

Contenido de la tarjeta.

</div>

CSS

.tarjeta{

background:white;

padding:20px;

border-radius:12px;

box-shadow:0px 5px 15px rgba(0,0,0,.2);

}

Este tipo de diseño es muy utilizado en paneles administrativos,
tiendas virtuales y sitios profesionales.

===============================================================================
VENTAJAS
===============================================================================

✓ Reutiliza estilos.

✓ Reduce la cantidad de código.

✓ Facilita el mantenimiento.

✓ Hace los proyectos más organizados.

✓ Es uno de los selectores más utilizados en CSS.

===============================================================================
BUENAS PRÁCTICAS
===============================================================================

Utilizar nombres claros para las clases.

Ejemplos:

.menu

.boton

.tarjeta

.encabezado

.footer

Evitar nombres poco descriptivos como:

.caja1

.objeto2

.prueba

===============================================================================
ERRORES COMUNES
===============================================================================

Error 1

Olvidar el punto.

Incorrecto

titulo{

color:red;

}

Correcto

.titulo{

color:red;

}

------------------------------------------------------------

Error 2

Asignar nombres confusos.

Incorrecto

.a

.b

.c

Correcto

.boton

.menu

.encabezado

===============================================================================
DIFERENCIA ENTRE HTML Y CSS
===============================================================================

HTML

class="titulo"

CSS

.titulo{

color:blue;

}

En HTML escribimos únicamente el nombre.

En CSS siempre utilizamos el punto antes del nombre.
