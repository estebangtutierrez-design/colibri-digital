===============================================================================
INTRODUCCIÓN
===============================================================================

En HTML aprendimos a construir la estructura de una página web.

Ahora comienza una nueva etapa.

CSS3 será el encargado de darle vida, color, tamaño y presentación a todos los elementos creados con HTML.

El primer selector que aprenderemos es el Selector Universal.

Como su nombre lo indica, tiene la capacidad de seleccionar absolutamente todos los elementos existentes dentro de una página web.

Aunque parece un selector muy sencillo, en proyectos profesionales tiene aplicaciones muy importantes.

===============================================================================
¿QUÉ ES UN SELECTOR?
===============================================================================

Un selector es una instrucción que indica a CSS qué elemento debe recibir un determinado estilo.

En otras palabras:

Primero seleccionamos el elemento.

Después aplicamos las propiedades CSS.

Ejemplo:

h1{
    color: blue;
}

En este caso:

Selector:
h1

Propiedad:
color

Valor:
blue

===============================================================================
¿QUÉ ES EL SELECTOR UNIVERSAL?
===============================================================================

El selector universal utiliza un asterisco (*).

Su función consiste en seleccionar todos los elementos existentes dentro del documento HTML.

Sintaxis:

*{

}

El símbolo

*

significa

TODOS LOS ELEMENTOS.

===============================================================================
PRIMER EJEMPLO
===============================================================================

CSS

*{

color:blue;

}

HTML

<h1>Lucas EGA Academy</h1>

<p>Aprendiendo CSS3.</p>

<h2>Profesor Búho AI</h2>

Resultado

Todos los textos aparecerán en color azul.

===============================================================================
OTRO EJEMPLO
===============================================================================

*{

font-family:Arial;

}

Toda la página utilizará la fuente Arial.

===============================================================================
EJEMPLO PROFESIONAL
===============================================================================

Uno de los usos más comunes consiste en eliminar los espacios automáticos que agregan los navegadores.

*{

margin:0;
padding:0;

}

Este código aparece en prácticamente todos los proyectos profesionales.

¿Por qué?

Porque cada navegador aplica márgenes diferentes.

Al eliminarlos, el diseñador comienza desde una base completamente limpia.

===============================================================================
OTRO EJEMPLO PROFESIONAL
===============================================================================

*{

box-sizing:border-box;

}

Esta propiedad facilita el cálculo del tamaño de todos los elementos.

Actualmente es considerada una buena práctica en desarrollo web.

===============================================================================
VENTAJAS
===============================================================================

✓ Fácil de utilizar.

✓ Selecciona todos los elementos.

✓ Ideal para reiniciar estilos.

✓ Muy utilizado en proyectos profesionales.

✓ Facilita la consistencia visual.

===============================================================================
DESVENTAJAS
===============================================================================

Puede afectar elementos que no deseábamos modificar.

Si el proyecto es muy grande puede aumentar el trabajo del navegador.

Debe utilizarse con criterio.

===============================================================================
BUENAS PRÁCTICAS
===============================================================================

Utilizar el selector universal únicamente para estilos generales.

Ejemplo:

*{

margin:0;
padding:0;
box-sizing:border-box;

}

Después utilizar selectores específicos para personalizar cada elemento.

===============================================================================
ERRORES COMUNES
===============================================================================

Error 1

Aplicar demasiados estilos al selector universal.

Incorrecto

*{

color:red;
font-size:50px;
background:black;
border:10px solid red;

}

Esto obliga a modificar posteriormente todos los elementos.

------------------------------------------------------------

Error 2

Pensar que sustituye los demás selectores.

No.

El selector universal únicamente sirve para establecer reglas generales.
