===============================================================================
INTRODUCCIÓN
===============================================================================

En el capítulo anterior aprendimos a utilizar los Selectores de Clase para
aplicar el mismo estilo a varios elementos.

Ahora conoceremos otro selector muy importante: el Selector ID.

Mientras que una clase puede utilizarse muchas veces, un ID está diseñado para
identificar un único elemento dentro de la página.

Los identificadores son muy utilizados en CSS, JavaScript y en los enlaces
internos de una página web.

===============================================================================
¿QUÉ ES UN ID?
===============================================================================

Un ID (Identificador) es un nombre único asignado a un solo elemento HTML.

Cada ID debe existir únicamente una vez dentro del documento.

Ejemplo:

<h1 id="principal">

Lucas EGA Academy

</h1>

===============================================================================
SINTAXIS DEL SELECTOR ID
===============================================================================

En CSS los ID se escriben utilizando el símbolo:

#

Ejemplo

#principal{

color:blue;

}

El símbolo numeral (#) indica que estamos seleccionando un identificador.

===============================================================================
PRIMER EJEMPLO
===============================================================================

HTML

<h1 id="titulo">

Bienvenidos

</h1>

CSS

#titulo{

color:green;

font-size:42px;

}

Resultado

El título aparecerá en color verde con un tamaño de 42 píxeles.

===============================================================================
OTRO EJEMPLO
===============================================================================

HTML

<p id="mensaje">

Este es un mensaje importante.

</p>

CSS

#mensaje{

background:yellow;

padding:15px;

font-weight:bold;

}

Resultado

El párrafo tendrá un fondo amarillo y texto resaltado.

===============================================================================
EJEMPLO PROFESIONAL
===============================================================================

HTML

<header id="encabezado">

Lucas EGA Academy

</header>

CSS

#encabezado{

background:#222;

color:white;

padding:25px;

text-align:center;

}

Este tipo de identificador suele utilizarse para el encabezado principal.

===============================================================================
OTRO EJEMPLO PROFESIONAL
===============================================================================

Todos los derechos reservados.

</footer>

}

===============================================================================
DIFERENCIA ENTRE CLASE E ID
===============================================================================

CLASE

Puede repetirse muchas veces.

Se escribe con:

.

Ejemplo

.menu

--------------------------------------------

ID

Solo puede existir una vez.

Se escribe con:

#

Ejemplo

#menuPrincipal

===============================================================================
¿CUÁNDO UTILIZAR UN ID?
===============================================================================

Cuando un elemento sea único.

Ejemplos:

Encabezado principal.

Pie de página.

Menú principal.

Logotipo.

Formulario principal.

Banner principal.

===============================================================================
VENTAJAS
===============================================================================

✓ Identifica elementos únicos.

✓ Facilita el acceso desde JavaScript.

✓ Permite crear enlaces internos.

✓ Hace el código más organizado.

✓ Es ampliamente utilizado en proyectos profesionales.

===============================================================================
BUENAS PRÁCTICAS
===============================================================================

Utilizar nombres descriptivos.

Ejemplos

#encabezado

#menuPrincipal

#logo

#piePagina

Evitar nombres como

#a

#objeto1

#caja5

===============================================================================
ERRORES COMUNES
===============================================================================

Error 1

Repetir el mismo ID.

Incorrecto

<h1 id="titulo">

<h2 id="titulo">

Cada ID debe ser único.

------------------------------------------------------------

Error 2

Olvidar el símbolo #

Incorrecto

titulo{

color:red;

}

Correcto

#titulo{

color:red;

}

===============================================================================
RESUMEN
===============================================================================

En este capítulo aprendimos que un ID identifica de forma única un elemento
HTML.

Los selectores ID utilizan el símbolo (#) y permiten aplicar estilos
específicos a elementos únicos dentro de una página.

También descubrimos la diferencia entre una Clase y un ID, una distinción
fundamental para escribir código profesional y mantener proyectos organizados.
