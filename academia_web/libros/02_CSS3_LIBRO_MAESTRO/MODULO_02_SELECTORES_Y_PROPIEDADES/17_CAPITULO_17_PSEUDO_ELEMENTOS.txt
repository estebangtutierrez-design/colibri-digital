===============================================================================
BIENVENIDA DEL PROFESOR BÚHO
===============================================================================

Querido estudiante…

En el capítulo anterior aprendiste sobre las pseudo-clases, que seleccionan
elementos según su **estado** (hover, focus, checked, etc.).

Pero hay otro tipo de selector que hace algo aún más poderoso:

**Los pseudo-elementos.**

Mientras que las pseudo-clases seleccionan elementos completos en un estado específico,
los pseudo-elementos seleccionan **partes** de un elemento,
o incluso crean **elementos fantasma** que no existen en el HTML.

- `::before` → crea un elemento antes del contenido.
- `::after` → crea un elemento después del contenido.
- `::first-letter` → estiliza la primera letra.
- `::first-line` → estiliza la primera línea.
- `::selection` → estiliza el texto seleccionado.
- `::placeholder` → estiliza el texto de marcador en inputs.

Los pseudo-elementos son una de las características más creativas de CSS.
Te permiten añadir decoración, iconos, efectos tipográficos y mucho más
sin tener que modificar el HTML.

Hoy vas a aprender a usarlos todos.

===============================================================================
INTRODUCCIÓN: PSEUDO-CLASES VS PSEUDO-ELEMENTOS
===============================================================================

Antes de profundizar, es importante entender la diferencia.

| Característica | Pseudo-clase | Pseudo-elemento |
|----------------|--------------|-----------------|
| **Sintaxis** | `:` (un solo dos puntos) | `::` (dos dos puntos) |
| **Selecciona** | Un estado completo del elemento | Una parte del elemento o contenido generado |
| **Ejemplo** | `:hover`, `:focus`, `:nth-child` | `::before`, `::after`, `::first-letter` |
| **Existe en HTML** | Sí, el elemento existe | No, el pseudo-elemento es creado por CSS |

📌 **Nota histórica**: En CSS2, los pseudo-elementos se escribían con un solo
dos puntos (`:before`). En CSS3, se estandarizó el uso de dos dos puntos (`::before`)
para diferenciarlos de las pseudo-clases. Hoy en día, los navegadores soportan ambas
sintaxis, pero se recomienda usar `::`.

===============================================================================
1. ::BEFORE Y ::AFTER (CONTENIDO GENERADO)
===============================================================================

`::before` y `::after` son los pseudo-elementos más poderosos y versátiles.

- **`::before`**: Inserta contenido **antes** del contenido del elemento.
- **`::after`**: Inserta contenido **después** del contenido del elemento.

---

📌 **Sintaxis**
selector::before {
content: "texto";
/* otras propiedades */
}

selector::after {
content: "texto";
/* otras propiedades */
}

text

---

🔴 **La propiedad `content` es OBLIGATORIA**

Sin `content`, el pseudo-elemento no se mostrará.
Puede ser un texto, una cadena vacía (`""`) o una imagen (`url()`).

---

🔵 **Ejemplo básico**

HTML:
------------------------------------------------------------
<p class="cita">El conocimiento es poder.</p>
------------------------------------------------------------

CSS:
------------------------------------------------------------
.cita::before {
    content: "« ";
    color: #3498DB;
    font-size: 1.5rem;
}

.cita::after {
    content: " »";
    color: #3498DB;
    font-size: 1.5rem;
}
------------------------------------------------------------

**Resultado:** El texto se mostrará como:
« El conocimiento es poder. »

---

📌 **Usos comunes de ::before y ::after**

1. **Decoración**: Añadir comillas, iconos o símbolos.
2. **Información adicional**: Añadir etiquetas como "Nuevo" o "Oferta".
3. **Limpiar floats** (clearfix).
4. **Crear elementos visuales**: Círculos, triángulos, barras.
5. **Animaciones**: Crear efectos decorativos.

---

🔵 **Ejemplo: Etiqueta "Nuevo"**

HTML:
------------------------------------------------------------
<span class="producto">Producto A</span>
------------------------------------------------------------

CSS:
------------------------------------------------------------
.producto {
    position: relative;
    padding-right: 40px;
}

.producto::after {
    content: "NUEVO";
    background-color: #FFC107;
    color: #333;
    font-size: 0.6rem;
    font-weight: bold;
    padding: 2px 8px;
    border-radius: 4px;
    position: absolute;
    right: 0;
    top: 0;
}
------------------------------------------------------------

---

🔵 **Ejemplo: Iconos con Font Awesome**

Puedes usar pseudo-elementos para añadir iconos sin modificar el HTML.

HTML:
------------------------------------------------------------
<a href="#" class="boton-descarga">Descargar</a>
------------------------------------------------------------

CSS:
------------------------------------------------------------
.boton-descarga::before {
    content: "\f019";  /* Código Unicode de Font Awesome */
    font-family: "Font Awesome 6 Free";
    font-weight: 900;
    margin-right: 8px;
}
------------------------------------------------------------

---

🔵 **Ejemplo: Triángulo decorativo**

HTML:
------------------------------------------------------------
<div class="mensaje">Mensaje importante</div>
------------------------------------------------------------

CSS:
------------------------------------------------------------
.mensaje {
    background-color: #007BFF;
    color: white;
    padding: 15px 20px;
    border-radius: 8px;
    position: relative;
}

.mensaje::after {
    content: "";
    position: absolute;
    bottom: -15px;
    left: 30px;
    border-width: 15px 15px 0 0;
    border-style: solid;
    border-color: #007BFF transparent transparent transparent;
    width: 0;
    height: 0;
}
------------------------------------------------------------

**Resultado:** Un triángulo que sale de la parte inferior, simulando un globo de diálogo.

---

📌 **`content` con `url()`**

Puedes usar imágenes como contenido.
selector::before {
content: url("icono.png");
margin-right: 5px;
}

text

---

⚠️ **Importante sobre ::before y ::after**

- Son **elementos en línea** por defecto. Puedes cambiar su `display`.
- Si el elemento padre no tiene `position: relative`, los pseudo-elementos
  con `position: absolute` se posicionarán respecto al ancestro más cercano con posición.
- No funcionan en elementos **reemplazados** como `<img>`, `<input>`, `<iframe>`, etc.

===============================================================================
2. ::FIRST-LETTER (PRIMERA LETRA)
===============================================================================

`::first-letter` selecciona la **primera letra** de un bloque de texto.

---

🔵 **Ejemplo: Letra capitular (efecto de libro)**

HTML:
------------------------------------------------------------
<p class="capitular">Había una vez un estudiante que quería aprender CSS...</p>
------------------------------------------------------------

CSS:
------------------------------------------------------------
.capitular::first-letter {
    font-size: 3.5rem;
    font-weight: bold;
    color: #007BFF;
    float: left;
    margin-right: 8px;
    line-height: 1;
}
------------------------------------------------------------

**Resultado:** La primera letra será mucho más grande y flotará a la izquierda,
como en los libros antiguos.

---

📌 **Propiedades permitidas**

Solo algunas propiedades funcionan con `::first-letter`:
- `font-*` (tamaño, peso, familia, etc.)
- `color`
- `background-*`
- `text-decoration`
- `text-transform`
- `line-height`
- `float`
- `margin`, `padding`, `border`

---

⚠️ **Restricciones**

- `::first-letter` solo funciona en **elementos de bloque** (p, div, h1, etc.).
- No funciona en elementos en línea.

===============================================================================
3. ::FIRST-LINE (PRIMERA LÍNEA)
===============================================================================

`::first-line` selecciona la **primera línea** de un bloque de texto.
La longitud de la primera línea depende del ancho del contenedor.

---

🔵 **Ejemplo: Primera línea destacada**

HTML:
------------------------------------------------------------
<p class="destacado">
    Esta es una frase larga que ocupa varias líneas.
    La primera línea tendrá un estilo especial.
</p>
------------------------------------------------------------

CSS:
------------------------------------------------------------
.destacado::first-line {
    font-weight: bold;
    color: #007BFF;
    text-transform: uppercase;
}
------------------------------------------------------------

**Resultado:** La primera línea (según cómo se ajuste el texto) será negrita, azul y mayúscula.

---

📌 **Propiedades permitidas**

Similares a `::first-letter`:
- `font-*`
- `color`
- `background-*`
- `text-decoration`
- `text-transform`
- `line-height`
- `word-spacing`, `letter-spacing`

===============================================================================
4. ::SELECTION (TEXTO SELECCIONADO)
===============================================================================

`::selection` se aplica cuando el usuario **selecciona** texto con el mouse o el teclado.

---

🔵 **Ejemplo: Cambiar el color de selección**

------------------------------------------------------------
CSS
------------------------------------------------------------
::selection {
    background-color: #007BFF;
    color: white;
}

/* También puedes aplicar a elementos específicos */
p::selection {
    background-color: #FFC107;
    color: #333;
}
------------------------------------------------------------

---

📌 **Propiedades permitidas**

- `color`
- `background-color`
- `text-shadow`
- `cursor`
- `outline`

---

🔴 **Importante**

El navegador tiene un color de selección predeterminado (generalmente azul con texto blanco).
Con `::selection` puedes personalizarlo para que coincida con tu diseño.

===============================================================================
5. ::PLACEHOLDER (MARCADOR DE INPUTS)
===============================================================================

`::placeholder` estiliza el texto de marcador de los campos `<input>` y `<textarea>`.

---

🔵 **Ejemplo: Placeholder personalizado**

HTML:
------------------------------------------------------------
<input type="text" placeholder="Escribe tu nombre...">
------------------------------------------------------------

CSS:
------------------------------------------------------------
::placeholder {
    color: #6C757D;
    font-style: italic;
    font-size: 0.9rem;
}

/* Para compatibilidad con navegadores antiguos */
::-webkit-input-placeholder { color: #6C757D; }
:-moz-placeholder { color: #6C757D; }
::-moz-placeholder { color: #6C757D; }
:-ms-input-placeholder { color: #6C757D; }
------------------------------------------------------------

---

📌 **Compatibilidad**

Usa el prefijo `::-webkit-` para compatibilidad con Safari y Chrome antiguos.
Hoy en día, `::placeholder` funciona en todos los navegadores modernos.

===============================================================================
6. ::MARKER (VIÑETAS DE LISTAS)
===============================================================================

`::marker` estiliza los marcadores de las listas (viñetas en `<ul>` y números en `<ol>`).

---

🔵 **Ejemplo: Viñetas personalizadas**

HTML:
------------------------------------------------------------
<ul class="lista-personalizada">
    <li>Elemento 1</li>
    <li>Elemento 2</li>
    <li>Elemento 3</li>
</ul>
------------------------------------------------------------

CSS:
------------------------------------------------------------
.lista-personalizada ::marker {
    color: #007BFF;
    font-size: 1.5rem;
}

.lista-personalizada li {
    margin-bottom: 10px;
}
------------------------------------------------------------

---

📌 **Propiedades permitidas**

- `color`
- `font-*`
- `content` (puedes reemplazar la viñeta por texto o emojis)

---

🔵 **Ejemplo con emojis**

------------------------------------------------------------
CSS
------------------------------------------------------------
.lista-emoji ::marker {
    content: "🚀 ";
    font-size: 1.2rem;
}
------------------------------------------------------------

===============================================================================
7. COMBINACIÓN DE PSEUDO-ELEMENTOS
===============================================================================

Puedes combinar pseudo-elementos entre sí y con pseudo-clases.

---

🔵 **Ejemplo: ::before + :hover**

------------------------------------------------------------
CSS
------------------------------------------------------------
.boton {
    position: relative;
    padding-left: 30px;
}

.boton::before {
    content: "▶ ";
    transition: transform 0.3s;
}

.boton:hover::before {
    transform: rotate(90deg);
}
------------------------------------------------------------

---

🔵 **Ejemplo: ::first-letter + :hover**

------------------------------------------------------------
CSS
------------------------------------------------------------
.capitular:hover::first-letter {
    color: #DC3545;
}
------------------------------------------------------------

---

🔵 **Ejemplo: ::selection + ::before**

No es posible combinar directamente, pero puedes usar selectores separados.

------------------------------------------------------------
CSS
------------------------------------------------------------
.articulo::before {
    content: "📄 ";
}

.articulo::selection {
    background-color: #FFC107;
}
------------------------------------------------------------

===============================================================================
LA ANALOGÍA DEL PROFESOR BÚHO: LOS PSEUDO-ELEMENTOS COMO MAQUILLAJE
===============================================================================

Imagina que tu HTML es una persona.

- **::before** es como maquillaje que te pones antes de salir.
- **::after** es como un accesorio que añades después (como un sombrero).
- **::first-letter** es como resaltar tu primera letra en un cartel.
- **::first-line** es como la primera frase que dices y que quieres que suene especial.
- **::selection** es como la atención que recibe tu texto cuando alguien lo señala.

Los pseudo-elementos no cambian la estructura de la persona (el HTML),
pero la hacen más atractiva, clara y expresiva.

===============================================================================
ERRORES COMUNES CON PSEUDO-ELEMENTOS
===============================================================================

❌ **Error 1: Olvidar la propiedad `content`**

   `::before` y `::after` no se mostrarán sin `content`.
   Si no quieres texto, usa `content: "";`.

---

❌ **Error 2: Confundir `:` con `::`**

   Las pseudo-clases usan `:` (hover, focus).
   Los pseudo-elementos usan `::` (::before, ::after).
   Aunque navegadores antiguos soportaban `:before`, la convención moderna es `::before`.

---

❌ **Error 3: Usar ::before y ::after en elementos reemplazados**

   No funcionan en `<img>`, `<input>`, `<iframe>`, `<video>`, etc.

---

❌ **Error 4: No usar `position` correctamente**

   Si usas `position: absolute` en un pseudo-elemento, el padre debe tener `position: relative`.

---

❌ **Error 5: Crear contenido accesible con pseudo-elementos**

   El contenido generado con `::before` y `::after` NO es accesible para lectores de pantalla.
   No uses pseudo-elementos para contenido crítico (textos importantes, etiquetas, etc.).

GLOSARIO DEL CAPÍTULO
===============================================================================

- **Pseudo-elemento**: Selector que estiliza una parte de un elemento o crea contenido virtual.
- **::before**: Crea un elemento antes del contenido.
- **::after**: Crea un elemento después del contenido.
- **::first-letter**: Estiliza la primera letra de un bloque.
- **::first-line**: Estiliza la primera línea de un bloque.
- **::selection**: Estiliza el texto seleccionado.
- **::placeholder**: Estiliza el marcador de inputs.
- **::marker**: Estiliza las viñetas de listas.
- **content**: Propiedad obligatoria para `::before` y `::after`; define el contenido generado.
- **Contenido generado**: Contenido creado por CSS, no presente en el HTML.

===============================================================================
PREGUNTAS FRECUENTES (FAQ)
===============================================================================

❓ **¿Cuál es la diferencia entre `:before` y `::before`?**

   `:before` es la sintaxis antigua (CSS2). `::before` es la sintaxis moderna (CSS3).
   Ambos funcionan en navegadores modernos, pero se recomienda usar `::`.

---

❓ **¿Puedo tener `::before` y `::after` en el mismo elemento?**

   Sí, un elemento puede tener ambos. `::before` va antes del contenido y `::after` después.

---

❓ **¿`::before` y `::after` funcionan en todos los elementos?**

   No. No funcionan en elementos reemplazados como `<img>`, `<input>`, `<iframe>`, `<video>`, etc.

---

❓ **¿Es accesible el contenido generado?**

   No. Los lectores de pantalla no leen el contenido de `::before` y `::after`.
   Usa pseudo-elementos solo para contenido decorativo, no crítico.

---

❓ **¿Puedo usar `::before` para añadir imágenes?**

   Sí, usando `content: url("imagen.png");`.
