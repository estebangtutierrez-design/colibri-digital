CONTENIDO DEL CAPÍTULO
===============================================================================

Blazor Server vs WebAssembly. .NET runtime en browser.

Components: @page, @code, @bind. Razor syntax.

Blazor WebAssembly: PWA, offline. Blazor Server: real-time.

EJEMPLO DE CÓDIGO
----------------------------------------

@page "/counter"
@rendermode InteractiveServer

<PageTitle>Counter</PageTitle>

<p>Count: @count</p>
<button @onclick="Increment">+</button>

@code {
  private int count = 0;
  private void Increment() => count++;
}
