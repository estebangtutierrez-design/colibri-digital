PARTE 1: ¿POR QUÉ EXISTEN DIFERENCIAS ENTRE NAVEGADORES?
===============================================================================

───────────────────────────────────────────────────────────
RAZONES PRINCIPALES
───────────────────────────────────────────────────────────

1. **Motores de renderizado diferentes:**
   - **Blink:** Chrome, Edge, Opera.
   - **Gecko:** Firefox.
   - **WebKit:** Safari.

2. **Versiones diferentes:**
   - Algunos usuarios usan navegadores antiguos.
   - No todos se actualizan automáticamente.

3. **Implementación de estándares:**
   - Algunos navegadores implementan estándares más rápido que otros.
   - Algunos navegadores implementan estándares de forma diferente.

4. **Características propietarias:**
   - Algunos navegadores tienen características que otros no.

5. **Preferencias del usuario:**
   - Los usuarios pueden cambiar configuraciones (tamaño de fuente, zoom, etc.).

───────────────────────────────────────────────────────────
MOTORES DE RENDERIZADO
───────────────────────────────────────────────────────────

| Navegador | Motor | Soporte de estándares |
|-----------|-------|----------------------|
| Chrome | Blink | Excelente |
| Edge | Blink | Excelente |
| Opera | Blink | Excelente |
| Firefox | Gecko | Excelente |
| Safari | WebKit | Bueno (a veces detrás) |

PARTE 2: PRINCIPALES DIFERENCIAS ENTRE NAVEGADORES
===============================================================================

───────────────────────────────────────────────────────────
1. CSS
───────────────────────────────────────────────────────────

- **Grid y Flexbox:** Soportados en todos los navegadores modernos.
- **Propiedades CSS:** Algunas propiedades pueden requerir prefijos (`-webkit-`, `-moz-`, `-ms-`).
- **Unidades:** `rem`, `em`, `vh`, `vw` son ampliamente soportadas.

**Ejemplo de prefijos CSS:**

```css
/* Prefijos para compatibilidad */
.elemento {
    -webkit-box-shadow: 0 4px 6px rgba(0,0,0,0.1); /* Safari */
       -moz-box-shadow: 0 4px 6px rgba(0,0,0,0.1); /* Firefox */
            box-shadow: 0 4px 6px rgba(0,0,0,0.1); /* Estándar */
}
───────────────────────────────────────────────────────────

JAVASCRIPT
───────────────────────────────────────────────────────────

ES6+: Soportado en navegadores modernos.

Fetch API: Soportada en Chrome, Firefox, Edge, Safari (versiones recientes).

Arrow functions: Soportadas en navegadores modernos.

Detectar soporte de Fetch API:

javascript
if ('fetch' in window) {
    // Usar fetch
} else {
    // Usar alternativa (XMLHttpRequest)
}
───────────────────────────────────────────────────────────

HTML5
───────────────────────────────────────────────────────────

Elementos semánticos: Soportados en todos los navegadores modernos.

Formularios: type="date", type="color", etc. pueden no funcionar en navegadores antiguos.

Video y audio: Soportados en todos los navegadores modernos.

Fallback para input de fecha:

html
<input type="date" name="fecha">
<!-- Si no soporta, se muestra como text -->
───────────────────────────────────────────────────────────

APIs
───────────────────────────────────────────────────────────

LocalStorage: Soportado en todos los navegadores modernos.

Geolocalización: Soportada en todos los navegadores modernos (requiere HTTPS).

Service Workers: Soportados en Chrome, Firefox, Edge, Safari (versiones recientes).

PARTE 4: FALLBACKS (ALTERNATIVAS PARA NAVEGADORES ANTIGUOS)
===============================================================================

Los fallbacks son alternativas que se usan cuando una característica no está disponible.

───────────────────────────────────────────────────────────

FALLBACKS EN CSS
───────────────────────────────────────────────────────────

css
/* Fallback para Grid */
.cuadricula {
    display: block; /* Fallback para navegadores antiguos */
    display: grid; /* Para navegadores modernos */
    grid-template-columns: repeat(3, 1fr);
}

/* Fallback para Flexbox */
.contenedor {
    display: block; /* Fallback */
    display: flex; /* Para navegadores modernos */
}
───────────────────────────────────────────────────────────

FALLBACKS EN HTML
───────────────────────────────────────────────────────────

html
<!-- Fallback para video -->
<video controls>
    <source src="video.mp4" type="video/mp4">
    <source src="video.ogv" type="video/ogg">
    <!-- Fallback -->
    <p>Tu navegador no soporta video.
       <a href="video.mp4">Descarga el video</a>.</p>
</video>

<!-- Fallback para input de color -->
<input type="color" value="#2b6cb0">
<!-- Si no soporta, se muestra como text -->
───────────────────────────────────────────────────────────

FALLBACKS EN JAVASCRIPT
───────────────────────────────────────────────────────────

javascript
// Fallback para fetch
if ('fetch' in window) {
    // Usar fetch
    fetch('/api/datos')
        .then(response => response.json())
        .then(data => console.log(data));
} else {
    // Fallback: XMLHttpRequest
    const xhr = new XMLHttpRequest();
    xhr.open('GET', '/api/datos');
    xhr.onload = function() {
        console.log(JSON.parse(xhr.responseText));
    };
    xhr.send();
}

PARTE 5: POLYFILLS (AÑADIR FUNCIONALIDADES FALTANTES)
===============================================================================

Los polyfills son scripts que añaden funcionalidades a navegadores que no las soportan.

───────────────────────────────────────────────────────────
POLYFILLS COMUNES
───────────────────────────────────────────────────────────

Polyfill	Funcionalidad	Uso
core-js	Características de ES6+	import "core-js/stable";
whatwg-fetch	Fetch API	import "whatwg-fetch";
promise-polyfill	Promesas	import "promise-polyfill";
html5shiv	Elementos semánticos HTML5	<script src="html5shiv.min.js"></script>
respond.js	Media queries en CSS	<script src="respond.min.js"></script>
───────────────────────────────────────────────────────────
CÓMO USAR POLYFILLS
───────────────────────────────────────────────────────────

html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">

    <!-- HTML5 Shiv para navegadores antiguos -->
    <!--[if lt IE 9]>
        <script src="https://cdnjs.cloudflare.com/ajax/libs/html5shiv/3.7.3/html5shiv.min.js"></script>
    <![endif]-->

    <!-- Polyfill para Fetch API -->
    <script src="https://cdn.jsdelivr.net/npm/whatwg-fetch@3.6.2/dist/fetch.umd.min.js"></script>

</head>
<body>
    <!-- Contenido -->
</body>
</html>
───────────────────────────────────────────────────────────
POLYFILL CON DETECCIÓN DE CARACTERÍSTICAS
───────────────────────────────────────────────────────────

javascript
// Detectar y cargar polyfill solo si es necesario
if (!('fetch' in window)) {
    const script = document.createElement('script');
    script.src = 'https://cdn.jsdelivr.net/npm/whatwg-fetch@3.6.2/dist/fetch.umd.min.js';
    document.head.appendChild(script);
}

PARTE 6: DETECCIÓN DE CARACTERÍSTICAS (FEATURE DETECTION)
===============================================================================

La detección de características verifica si una funcionalidad está disponible antes de usarla.

───────────────────────────────────────────────────────────
DETECCIÓN DE CARACTERÍSTICAS EN JAVASCRIPT
───────────────────────────────────────────────────────────

javascript
// Detectar LocalStorage
if ('localStorage' in window) {
    // Usar LocalStorage
    localStorage.setItem('test', 'ok');
} else {
    // Usar alternativa (cookies)
}

// Detectar Geolocalización
if ('geolocation' in navigator) {
    // Usar geolocalización
    navigator.geolocation.getCurrentPosition(...);
} else {
    // Mostrar mensaje o usar alternativa
}

// Detectar Intersection Observer
if ('IntersectionObserver' in window) {
    // Usar Intersection Observer
} else {
    // Usar alternativa (scroll events)
}
───────────────────────────────────────────────────────────
DETECCIÓN DE CARACTERÍSTICAS EN CSS
───────────────────────────────────────────────────────────

css
/* @supports para detectar características CSS */
@supports (display: grid) {
    .contenedor {
        display: grid;
        grid-template-columns: repeat(3, 1fr);
    }
}

@supports not (display: grid) {
    .contenedor {
        display: flex;
        flex-wrap: wrap;
    }
}
───────────────────────────────────────────────────────────
DETECCIÓN DE CARACTERÍSTICAS EN HTML
───────────────────────────────────────────────────────────

html
<!-- Atributo para detectar soporte de picture -->
<picture>
    <source srcset="imagen.webp" type="image/webp">
    <source srcset="imagen.jpg" type="image/jpeg">
    <img src="imagen.jpg" alt="Descripción">
</picture>
