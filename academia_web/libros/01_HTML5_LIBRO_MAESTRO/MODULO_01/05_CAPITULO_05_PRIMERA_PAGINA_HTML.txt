PASO 1 – CREAR EL ARCHIVO
===============================================================================

Abre tu editor de texto.

Selecciona:
Archivo
  ↓
Nuevo

Ahora guarda el archivo con el siguiente nombre:

index.html

Muy importante.
Debe terminar exactamente así:
.html

Nunca como:
index.txt
Nunca como:
index.doc
Nunca como:
index.pdf

El navegador solamente reconocerá los archivos HTML.

===============================================================================
PASO 2 – ESCRIBIR EL CÓDIGO BÁSICO
===============================================================================

Copia exactamente este código dentro de tu archivo:

CÓDIGO HTML

<!DOCTYPE html>
<html>
<head>
    <title>Mi Primera Página</title>
</head>
<body>
    <h1>¡Hola Mundo!</h1>
    <p>Bienvenido a Lucas EGA Academy.</p>
</body>
</html>

FIN DEL CÓDIGO

===============================================================================
PASO 3 – EXPLICACIÓN LÍNEA POR LÍNEA (COMO UN CIRUJANO)
===============================================================================

Ahora vamos a diseccionar cada línea para que entiendas perfectamente qué hace cada una.

───────────────────────────────────────────────────────────
LÍNEA 1: <!DOCTYPE html>
───────────────────────────────────────────────────────────

Esta línea es como un saludo al navegador.
Le dice: "Oye, soy una página moderna. Usa la versión 5 de HTML para leerme."
Sin esta línea, el navegador podría entrar en "modo antiguo" y mostrar tu página mal.

───────────────────────────────────────────────────────────
LÍNEA 2: <html>
───────────────────────────────────────────────────────────

Esta etiqueta es la raíz de todo tu documento.
Todo el contenido de tu página debe estar dentro de esta etiqueta.
Es como la base de la casa.

───────────────────────────────────────────────────────────
LÍNEA 3: <head>
───────────────────────────────────────────────────────────

La cabeza de tu página.
Aquí va información que NO se ve en la pantalla, pero que es muy importante:
el título de la pestaña, el idioma, los estilos, etc.

───────────────────────────────────────────────────────────
LÍNEA 4: <title>Mi Primera Página</title>
───────────────────────────────────────────────────────────

Esto es el texto que aparece en la pestaña del navegador.
Si no lo pones, la pestaña dirá "Sin título".
Elige un título descriptivo para tu página.

───────────────────────────────────────────────────────────
LÍNEA 5: </head>
───────────────────────────────────────────────────────────

Cierra la cabeza de la página.
Todo lo que abres, debes cerrarlo.

───────────────────────────────────────────────────────────
LÍNEA 6: <body>
───────────────────────────────────────────────────────────

El cuerpo de la página.
TODO lo que pongas aquí será visible para el usuario.
Textos, imágenes, botones, videos... todo va aquí.

───────────────────────────────────────────────────────────
LÍNEA 7: <h1>¡Hola Mundo!</h1>
───────────────────────────────────────────────────────────

Esto es un encabezado de nivel 1.
Es el título más importante de la página.
Se ve grande y en negrita.
Solo debe haber un <h1> por página (por buenas prácticas de SEO).

───────────────────────────────────────────────────────────
LÍNEA 8: <p>Bienvenido a Lucas EGA Academy.</p>
───────────────────────────────────────────────────────────

Esto es un párrafo.
Sirve para escribir texto normal.
El navegador deja un espacio entre párrafos automáticamente.

───────────────────────────────────────────────────────────
LÍNEA 9: </body>
───────────────────────────────────────────────────────────

Cierra el cuerpo de la página.

───────────────────────────────────────────────────────────
LÍNEA 10: </html>
───────────────────────────────────────────────────────────

Cierra la raíz del documento.

===============================================================================
PASO 4 – ¿QUÉ PASA DENTRO DEL NAVEGADOR? (LA MAGIA INTERNA)
===============================================================================

Cuando guardas el archivo y haces doble clic para abrirlo en el navegador, ocurre esto:

1. El navegador lee el archivo.
2. Identifica que es HTML por la extensión .html.
3. Lee la línea <!DOCTYPE html> y se prepara para usar el modo estándar.
4. Construye el DOM (Document Object Model): un árbol de nodos donde cada etiqueta es un nodo.
5. Interpreta la etiqueta <head> y aplica metadatos (como el título).
6. Interpreta la etiqueta <body> y pinta el contenido en la pantalla.
7. Muestra el título en la pestaña.
8. Dibuja el <h1> con un tamaño grande y en negrita.
9. Dibuja el <p> con un tamaño normal y en fuente por defecto.

Todo esto ocurre en milisegundos.

PASO 5 – ERRORES COMUNES AL CREAR LA PRIMERA PÁGINA
===============================================================================

No te preocupes si cometes estos errores. Todos los cometemos al principio.

───────────────────────────────────────────────────────────
ERROR 1: Olvidar cerrar una etiqueta
───────────────────────────────────────────────────────────

Ejemplo:
<h1>Hola Mundo

El navegador no sabrá dónde termina el encabezado y puede mostrar todo el resto como parte del mismo.

Solución: Siempre revisa que cada etiqueta que abre tenga su cierre.

───────────────────────────────────────────────────────────
ERROR 2: Escribir mal el nombre de la etiqueta
───────────────────────────────────────────────────────────

Ejemplo:
<he>Mi Título</he>

El navegador no reconoce "he" y lo ignora.

Solución: Escribe exactamente <h1>, <p>, <title>, etc.

───────────────────────────────────────────────────────────
ERROR 3: Olvidar la extensión .html
───────────────────────────────────────────────────────────

Guardar el archivo como "index.txt" o "index" en lugar de "index.html".

El navegador no lo abrirá como página web.

Solución: Verifica que la extensión sea .html.

───────────────────────────────────────────────────────────
ERROR 4: No cerrar las comillas en atributos (avanzado)
───────────────────────────────────────────────────────────

Ejemplo:
<img src=foto.jpg alt=Descripción>

El navegador puede confundirse con los espacios.

Solución: Siempre usa comillas:
<img src="foto.jpg" alt="Descripción">

───────────────────────────────────────────────────────────
ERROR 5: No guardar el archivo antes de abrirlo
───────────────────────────────────────────────────────────

Escribes el código, pero no guardas. Luego abres el archivo y ves la versión anterior.

Solución: Siempre guarda (Ctrl+S) antes de actualizar el navegador.

===============================================================================
PASO 6 – CÓMO SOLUCIONAR LOS ERRORES
===============================================================================

1. Lee el mensaje de error que muestra el navegador (presiona F12 para abrir las herramientas de desarrollo y ve a la pestaña "Consola").
2. Revisa que todas las etiquetas estén correctamente escritas y cerradas.
3. Asegúrate de que la extensión del archivo sea .html.
4. Guarda el archivo y actualiza el navegador (F5 o Ctrl+R).
5. Si ves caracteres extraños (como ñ o tildes rotas), asegúrate de que el archivo esté guardado en UTF-8 (en Bluefish, ve a "Documento" > "Codificación" > "Unicode (UTF-8)").

GLOSARIO DE TÉRMINOS
===============================================================================

Aquí tienes los términos clave que has aprendido en este capítulo:

- HTML: HyperText Markup Language. Lenguaje de marcado para crear páginas web.
- Etiqueta: Elemento que se escribe entre < >, como <h1> o <p>.
- Atributo: Información adicional dentro de una etiqueta (ej: src, alt).
- Navegador: Programa que interpreta el código HTML y muestra la página.
- Editor de texto: Herramienta para escribir código (ej: Bluefish).
- Extensión .html: La terminación que identifica a un archivo HTML.
- DOM: Document Object Model. El árbol de nodos que construye el navegador.
