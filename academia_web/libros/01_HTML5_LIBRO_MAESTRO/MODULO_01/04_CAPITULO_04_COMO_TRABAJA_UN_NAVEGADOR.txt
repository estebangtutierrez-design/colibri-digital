¿QUÉ ES UN NAVEGADOR?
===============================================================================

Un navegador es un programa diseñado para interpretar páginas web.

Su trabajo consiste en leer archivos HTML, CSS y JavaScript para construir la página que el usuario verá en la pantalla.

Algunos navegadores conocidos son:

• Google Chrome

• Mozilla Firefox

• Microsoft Edge

• Safari

• Opera

• Brave

Todos realizan prácticamente el mismo trabajo.

¿CÓMO LEE HTML?
===============================================================================

Supongamos este código.

<!DOCTYPE html>

<html>

<head>

<title>Lucas EGA Academy</title>

</head>

<body>

<h1>Bienvenido</h1>

<p>Aprenderemos HTML.</p>

</body>

</html>

El navegador comienza desde arriba.

No salta líneas.

No adivina.

Lee todo en orden.

Primero:

DOCTYPE

Después:

html

Después:

head

Después:

title

Después:

body

Después:

h1

Después:

p

Y así hasta terminar.

===============================================================================
EL ÁRBOL DOM
===============================================================================

Mientras lee el documento construye un árbol llamado DOM.

DOM significa:

Document Object Model.

Es una representación organizada de toda la página.

Ejemplo.

html

│

├── head

│      └── title

│

└── body

       ├── h1

       └── p

JavaScript utiliza este árbol para modificar la página.

===============================================================================
¿QUÉ HACE CON CSS?
===============================================================================

Cuando encuentra:

<link rel="stylesheet" href="style.css">

El navegador busca ese archivo.

Después aplica:

colores

fuentes

bordes

sombras

espacios

alineaciones

animaciones

Todo eso ocurre automáticamente.

===============================================================================
¿QUÉ HACE CON JAVASCRIPT?
===============================================================================

Cuando encuentra:

<script src="app.js"></script>

Carga el archivo.

Después lo ejecuta.

JavaScript puede:

Modificar HTML.

Cambiar colores.

Crear elementos.

Eliminar elementos.

Mostrar ventanas.

Realizar cálculos.

Actualizar datos.

===============================================================================
¿QUÉ ES RENDERIZAR?
===============================================================================

Renderizar significa:

Construir visualmente la página.

Es decir.

Convertir texto y código en algo que el usuario pueda ver.

Sin renderizado únicamente existiría código.

===============================================================================
ERRORES COMUNES
===============================================================================

Pensar que HTML dibuja la página.

No.

El navegador la dibuja.

--------------------------------------------------

Pensar que CSS cambia HTML.

No.

Sólo cambia su apariencia.

--------------------------------------------------

Pensar que JavaScript crea la página.

No.

Primero existe HTML.

Después JavaScript puede modificarlo.

¿SABÍAS QUE...?
===============================================================================

Google Chrome utiliza un motor llamado Blink.

Firefox utiliza Gecko.

Safari utiliza WebKit.

Aunque internamente son diferentes, todos intentan seguir los mismos estándares web para mostrar las páginas correctamente.
