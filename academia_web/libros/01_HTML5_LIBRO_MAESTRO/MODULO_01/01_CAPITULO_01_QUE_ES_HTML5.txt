¿POR QUÉ APRENDER HTML?
===============================================================================

Antes de escribir una sola etiqueta debemos responder una pregunta mucho más importante.

¿Por qué aprender HTML?

Porque absolutamente toda página web comienza con HTML.

No importa si visitas Google.

No importa si entras a Amazon.

No importa si navegas por YouTube.

No importa si utilizas Facebook.

No importa si consultas Wikipedia.

Todo comenzó con HTML.

Después llegaron otros lenguajes.

Pero el primer ladrillo siempre fue HTML.

Podemos imaginar Internet como una enorme ciudad.

En esa ciudad existen hospitales.

Escuelas.

Bancos.

Restaurantes.

Museos.

Tiendas.

Universidades.

Cada uno de esos edificios representa un sitio web.

Ahora piensa en algo.

¿Qué tienen en común todos los edificios del mundo?

Todos necesitan una estructura.

Primero se construyen los cimientos.

Después las paredes.

Luego el techo.

Después las ventanas.

Finalmente la decoración.

Las páginas web funcionan exactamente igual.

HTML construye la estructura.

CSS la hace hermosa.

JavaScript le da vida.

Si eliminas HTML...

No existe página.

Por eso HTML es el primer lenguaje que todo desarrollador debe aprender.

No porque sea el más difícil.

Sino porque es el fundamento de todo lo demás.
