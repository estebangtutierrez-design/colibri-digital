===============================================================================
PARTE 2: REQUISITOS DEL PROYECTO
===============================================================================

Tu sitio debe cumplir con TODOS los siguientes requisitos:

───────────────────────────────────────────────────────────
ESTRUCTURA Y ORGANIZACIÓN
───────────────────────────────────────────────────────────

✅ Estructura de carpetas profesional (css/, js/, images/, fonts/, assets/, docs/).
✅ Rutas relativas para todos los recursos internos.
✅ Indentación consistente (2 o 4 espacios).
✅ Etiquetas siempre cerradas.
✅ Atributos con comillas dobles.
✅ Nombres de archivos en minúscula y con guiones.

───────────────────────────────────────────────────────────
SEMÁNTICA
───────────────────────────────────────────────────────────

✅ <header> en todas las páginas.
✅ <nav> en todas las páginas.
✅ <main> en todas las páginas (solo uno por página).
✅ <section> para secciones temáticas.
✅ <article> para contenido independiente (blog, cursos).
✅ <aside> para contenido complementario.
✅ <footer> en todas las páginas.
✅ <figure> y <figcaption> para imágenes con contexto.
✅ <time> para fechas.
✅ <address> para información de contacto.
✅ Jerarquía de encabezados correcta (h1 → h2 → h3).
✅ Solo un <h1> por página.

───────────────────────────────────────────────────────────
ACCESIBILIDAD
───────────────────────────────────────────────────────────

✅ alt en todas las imágenes.
✅ label asociado con input (for/id) en formularios.
✅ Estructura semántica correcta.
✅ scope en tablas (si las hay).
✅ aria-label donde sea necesario.

───────────────────────────────────────────────────────────
METAETIQUETAS Y SEO
───────────────────────────────────────────────────────────

✅ <meta charset="UTF-8">.
✅ <meta name="viewport" content="width=device-width, initial-scale=1.0">.
✅ <title> descriptivo (50-60 caracteres) en cada página.
✅ <meta name="description"> atractiva (150-160 caracteres) en cada página.
✅ <meta name="robots" content="index, follow">.
✅ Open Graph (og:title, og:description, og:image, og:url, og:type, og:site_name, og:locale).
✅ Twitter Cards (twitter:card, twitter:title, twitter:description, twitter:image).
✅ <link rel="icon">.

───────────────────────────────────────────────────────────
CSS
───────────────────────────────────────────────────────────

✅ CSS externo en archivo style.css.
✅ CSS enlazado desde el <head> de todas las páginas.
✅ Estilos para: body, header, nav, main, section, article, aside, footer.
✅ Estilos para: formularios, botones, imágenes, tablas.
✅ Diseño atractivo y profesional.
✅ No usar CSS en línea (excepto casos muy específicos).
✅ No usar CSS interno (excepto pruebas rápidas).

───────────────────────────────────────────────────────────
JAVASCRIPT
───────────────────────────────────────────────────────────

✅ JavaScript externo en archivo app.js.
✅ JavaScript enlazado con defer desde el <head> o al final del <body>.
✅ Al menos 3 interacciones:
   - Menú móvil (toggle).
   - Botón "Volver arriba".
   - Cambio de tema (oscuro/claro) o similar.
✅ No usar JavaScript en línea (excepto casos muy específicos).

───────────────────────────────────────────────────────────
CONTENIDO
───────────────────────────────────────────────────────────

✅ Página principal con: hero, cursos destacados, testimonios, blog.
✅ Página de cursos con: lista de cursos con descripción y duración.
✅ Página de blog con: artículos recientes con fecha.
✅ Página de contacto con: formulario y datos de contacto.
✅ Página "Acerca de" con: historia, misión, visión, valores.

───────────────────────────────────────────────────────────
COMENTARIOS Y DOCUMENTACIÓN
───────────────────────────────────────────────────────────

✅ Comentario de cabecera en cada archivo HTML (autor, fecha, descripción).
✅ Comentarios que separan secciones (HEADER, MAIN, FOOTER).
✅ Comentarios explicativos en CSS y JavaScript.
✅ Archivo README.md con: descripción del proyecto, estructura, cómo ejecutarlo.

───────────────────────────────────────────────────────────
BUENAS PRÁCTICAS
───────────────────────────────────────────────────────────

✅ Indentación consistente.
✅ Semántica correcta.
✅ Accesibilidad básica.
✅ Rendimiento (imágenes optimizadas, carga de recursos).
✅ SEO básico.
✅ Código limpio y legible.

// Espera a que el DOM esté completamente cargado
document.addEventListener('DOMContentLoaded', function() {

    // ================================================================ */
    // 1. MENÚ MÓVIL (TOGGLE)                                           */
    // ================================================================ */
    const menuToggle = document.querySelector('.menu-toggle');
    const navPrincipal = document.querySelector('.nav-principal');

    if (menuToggle && navPrincipal) {
        menuToggle.addEventListener('click', function() {
            // Alternar clase 'abierto' en el menú
            navPrincipal.classList.toggle('abierto');

            // Actualizar aria-expanded
            const isOpen = navPrincipal.classList.contains('abierto');
            menuToggle.setAttribute('aria-expanded', isOpen);

            // Animar botón hamburguesa (opcional)
            const spans = menuToggle.querySelectorAll('span');
            if (isOpen) {
                spans.forEach((span, index) => {
                    if (index === 0) span.style.transform = 'rotate(45deg) translate(5px, 5px)';
                    if (index === 1) span.style.opacity = '0';
                    if (index === 2) span.style.transform = 'rotate(-45deg) translate(5px, -5px)';
                });
            } else {
                spans.forEach(span => {
                    span.style.transform = 'none';
                    span.style.opacity = '1';
                });
            }
        });
    }

    // ================================================================ */
    // 2. BOTÓN "VOLVER ARRIBA"                                        */
    // ================================================================ */
    const btnVolverArriba = document.getElementById('btn-volver-arriba');

    if (btnVolverArriba) {
        // Mostrar/ocultar botón según el desplazamiento
        window.addEventListener('scroll', function() {
            if (window.scrollY > 300) {
                btnVolverArriba.classList.add('visible');
            } else {
                btnVolverArriba.classList.remove('visible');
            }
        });

        // Volver arriba al hacer clic
        btnVolverArriba.addEventListener('click', function() {

GLOSARIO DE TÉRMINOS DEL MÓDULO III
===============================================================================

- div: Contenedor de bloque sin significado semántico.
- span: Contenedor en línea sin significado semántico.
- Semántica: Uso de etiquetas con significado.
- Metaetiqueta: Etiqueta en el <head> que describe la página.
- SEO: Optimización para motores de búsqueda.
- Open Graph: Protocolo para redes sociales.
- Ruta relativa: Dirección de un archivo desde la ubicación actual.
- Ruta absoluta: Dirección completa de un archivo.
- defer: Atributo que retrasa la ejecución de JavaScript.
- Comentario: Texto ignorado por el navegador.
- Refactorización: Mejorar el código sin cambiar su comportamiento.
