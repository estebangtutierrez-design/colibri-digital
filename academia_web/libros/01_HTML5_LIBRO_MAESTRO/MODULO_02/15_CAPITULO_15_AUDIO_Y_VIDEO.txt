PARTE 1: LA ETIQUETA <audio> (SONIDO EN TU PÁGINA)
===============================================================================

La etiqueta <audio> permite insertar archivos de sonido en una página web.

───────────────────────────────────────────────────────────
SINTAXIS BÁSICA
───────────────────────────────────────────────────────────

<audio src="cancion.mp3" controls></audio>

- src → Ruta del archivo de audio.
- controls → Muestra los botones de reproducción (play, pausa, volumen).

───────────────────────────────────────────────────────────
EJEMPLO COMPLETO
───────────────────────────────────────────────────────────

<audio src="musica-fondo.mp3" controls>
    Tu navegador no soporta el elemento de audio.
</audio>

El texto entre las etiquetas se muestra si el navegador no soporta <audio>.

===============================================================================
PARTE 2: ATRIBUTOS DE <audio>
===============================================================================

───────────────────────────────────────────────────────────
controls → MUESTRA LOS CONTROLES
───────────────────────────────────────────────────────────

<audio src="cancion.mp3" controls></audio>

Muestra: ▶️ Play, ⏸️ Pausa, 🔊 Volumen, ⏱️ Progreso.

───────────────────────────────────────────────────────────
autoplay → REPRODUCCIÓN AUTOMÁTICA
───────────────────────────────────────────────────────────

<audio src="cancion.mp3" autoplay></audio>

El audio comienza a sonar automáticamente al cargar la página.

⚠️ ADVERTENCIA: Muchos navegadores bloquean el autoplay a menos que el audio esté silenciado.

───────────────────────────────────────────────────────────
loop → REPETICIÓN INFINITA
───────────────────────────────────────────────────────────

<audio src="cancion.mp3" controls loop></audio>

El audio se repite constantemente.

───────────────────────────────────────────────────────────
muted → SILENCIADO POR DEFECTO
───────────────────────────────────────────────────────────

<audio src="cancion.mp3" controls muted></audio>

El audio comienza silenciado. El usuario puede activar el sonido.

Útil para autoplay sin molestar al usuario.

───────────────────────────────────────────────────────────
preload → PRECARGA DEL AUDIO
───────────────────────────────────────────────────────────

<audio src="cancion.mp3" preload="auto"></audio>

Valores:
- auto → El navegador decide cuánto precargar.
- metadata → Solo carga metadatos (duración, etc.).
- none → No precarga nada.

───────────────────────────────────────────────────────────
EJEMPLO CON TODOS LOS ATRIBUTOS
───────────────────────────────────────────────────────────

<audio src="musica.mp3" controls autoplay loop muted preload="auto">
    Tu navegador no soporta el elemento de audio.
</audio>

===============================================================================
PARTE 3: MÚLTIPLES FUENTES DE AUDIO (COMPATIBILIDAD)
===============================================================================

No todos los navegadores soportan los mismos formatos de audio.

Para garantizar la compatibilidad, usa múltiples fuentes con la etiqueta <source>.

───────────────────────────────────────────────────────────
FORMATOS DE AUDIO COMPATIBLES
───────────────────────────────────────────────────────────

- MP3 → El más universal. Compatible con todos los navegadores.
- OGG → Formato libre. Compatible con Firefox, Chrome, Edge.
- WAV → Sin compresión. Tamaño grande. Calidad máxima.
- AAC → Formato de Apple. Compatible con Safari.

───────────────────────────────────────────────────────────
EJEMPLO CON MÚLTIPLES FUENTES
───────────────────────────────────────────────────────────

<audio controls>
    <source src="cancion.mp3" type="audio/mpeg">
    <source src="cancion.ogg" type="audio/ogg">
    <source src="cancion.wav" type="audio/wav">
    Tu navegador no soporta el elemento de audio.
</audio>

El navegador elige el primer formato que pueda reproducir.

===============================================================================
PARTE 4: LA ETIQUETA <video> (PELÍCULAS EN TU PÁGINA)
===============================================================================

La etiqueta <video> permite insertar videos en una página web.

───────────────────────────────────────────────────────────
SINTAXIS BÁSICA
───────────────────────────────────────────────────────────

<video src="pelicula.mp4" controls width="640" height="360"></video>

- src → Ruta del archivo de video.
- controls → Muestra los controles de reproducción.
- width y height → Tamaño del reproductor.

───────────────────────────────────────────────────────────
EJEMPLO COMPLETO
───────────────────────────────────────────────────────────

<video src="video-intro.mp4" controls width="640" height="360">
    Tu navegador no soporta el elemento de video.
</video>

===============================================================================
PARTE 5: ATRIBUTOS DE <video>
===============================================================================

───────────────────────────────────────────────────────────
controls → MUESTRA LOS CONTROLES
───────────────────────────────────────────────────────────

<video src="video.mp4" controls width="640" height="360"></video>

Muestra: ▶️ Play, ⏸️ Pausa, 🔊 Volumen, ⏱️ Progreso, ⛶ Pantalla completa.

───────────────────────────────────────────────────────────
autoplay → REPRODUCCIÓN AUTOMÁTICA
───────────────────────────────────────────────────────────

<video src="video.mp4" autoplay muted width="640" height="360"></video>

⚠️ ADVERTENCIA: La mayoría de los navegadores solo permiten autoplay si el video está silenciado (muted).

───────────────────────────────────────────────────────────
muted → SILENCIADO POR DEFECTO
───────────────────────────────────────────────────────────

<video src="video.mp4" controls muted width="640" height="360"></video>

El video comienza silenciado.

───────────────────────────────────────────────────────────
loop → REPETICIÓN INFINITA
───────────────────────────────────────────────────────────

<video src="video.mp4" controls loop width="640" height="360"></video>

El video se repite constantemente.

───────────────────────────────────────────────────────────
poster → IMAGEN DE PORTADA
───────────────────────────────────────────────────────────

<video src="video.mp4" controls poster="portada.jpg" width="640" height="360"></video>

Muestra una imagen antes de que el video comience a reproducirse.
Ideal para dar una vista previa atractiva.

───────────────────────────────────────────────────────────
preload → PRECARGA DEL VIDEO
───────────────────────────────────────────────────────────

<video src="video.mp4" preload="metadata"></video>

Valores: auto, metadata, none.

───────────────────────────────────────────────────────────
EJEMPLO CON TODOS LOS ATRIBUTOS
───────────────────────────────────────────────────────────

<video src="video-promo.mp4" controls autoplay muted loop poster="portada.jpg" width="800" height="450">
    Tu navegador no soporta el elemento de video.
</video>

===============================================================================
PARTE 6: MÚLTIPLES FUENTES DE VIDEO (COMPATIBILIDAD)
===============================================================================

No todos los navegadores soportan los mismos formatos de video.

───────────────────────────────────────────────────────────
FORMATOS DE VIDEO COMPATIBLES
───────────────────────────────────────────────────────────

- MP4 (H.264) → El más universal. Compatible con TODOS los navegadores.
- WebM → Formato libre. Compatible con Firefox, Chrome, Edge.
- OGG → Formato libre. Compatible con Firefox, Chrome.

───────────────────────────────────────────────────────────
EJEMPLO CON MÚLTIPLES FUENTES
───────────────────────────────────────────────────────────

<video controls width="640" height="360" poster="portada.jpg">
    <source src="video.mp4" type="video/mp4">
    <source src="video.webm" type="video/webm">
    <source src="video.ogv" type="video/ogg">
    Tu navegador no soporta el elemento de video.
</video>

El navegador elige el primer formato que pueda reproducir.

===============================================================================
PARTE 7: VIDEO RESPONSIVE (QUE SE AJUSTE A LA PANTALLA)
===============================================================================

Para que un video se vea bien en todos los dispositivos, debe ser responsive.

───────────────────────────────────────────────────────────
MÉTODO 1: CSS CON max-width: 100%
───────────────────────────────────────────────────────────

<video src="video.mp4" controls style="max-width: 100%; height: auto;"></video>

Esto hace que el video se ajuste al ancho de su contenedor.

───────────────────────────────────────────────────────────
MÉTODO 2: CONTENEDOR CON PROPORCIÓN 16:9
───────────────────────────────────────────────────────────

<div style="position: relative; padding-bottom: 56.25%; height: 0; overflow: hidden;">
    <video src="video.mp4" controls style="position: absolute; top: 0; left: 0; width: 100%; height: 100%;">
    </video>
</div>

El padding-bottom: 56.25% crea una proporción de 16:9 (ancho:alto).
El video se estira para llenar el contenedor.

Este método es el más profesional para videos responsivos.

CÓDIGO HTML

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Audio y Video HTML5</title>
</head>
<body>

    <h1>Contenido Multimedia</h1>

    <!-- AUDIO BÁSICO -->
    <h2>Reproductor de Audio</h2>
    <audio controls>
        <source src="musica.mp3" type="audio/mpeg">
        <source src="musica.ogg" type="audio/ogg">
        Tu navegador no soporta el elemento de audio.
    </audio>

    <!-- AUDIO CON AUTOPLAY Y MUTED -->
    <h2>Audio con autoplay silenciado</h2>
    <audio controls autoplay muted>
        <source src="musica-fondo.mp3" type="audio/mpeg">
        Tu navegador no soporta el elemento de audio.
    </audio>

    <!-- VIDEO BÁSICO -->
    <h2>Reproductor de Video</h2>
    <video controls width="640" height="360">
        <source src="video-intro.mp4" type="video/mp4">
        <source src="video-intro.webm" type="video/webm">
        Tu navegador no soporta el elemento de video.
    </video>

    <!-- VIDEO CON POSTER -->
    <h2>Video con imagen de portada</h2>
    <video controls width="640" height="360" poster="portada-video.jpg">
        <source src="video-promo.mp4" type="video/mp4">
        <source src="video-promo.webm" type="video/webm">
        Tu navegador no soporta el elemento de video.
    </video>

    <!-- VIDEO RESPONSIVE (CON CSS) -->
    <h2>Video Responsive</h2>
    <div style="position: relative; padding-bottom: 56.25%; height: 0; overflow: hidden; max-width: 100%;">
        <video controls style="position: absolute; top: 0; left: 0; width: 100%; height: 100%;">
            <source src="video-tutorial.mp4" type="video/mp4">
            <source src="video-tutorial.webm" type="video/webm">
            Tu navegador no soporta el elemento de video.
        </video>
    </div>

    <!-- VIDEO CON LOOP Y AUTOPLAY -->
    <h2>Video en bucle (loop)</h2>
    <video controls loop autoplay muted width="640" height="360">
        <source src="video-loop.mp4" type="video/mp4">
        <source src="video-loop.webm" type="video/webm">
        Tu navegador no soporta el elemento de video.
    </video>

</body>
</html>

PARTE 11: ERRORES COMUNES CON AUDIO Y VIDEO
===============================================================================

───────────────────────────────────────────────────────────
ERROR 1: NO OFRECER MÚLTIPLES FUENTES
───────────────────────────────────────────────────────────

Incorrecto:
<video src="video.mp4"></video>

Consecuencia: En Firefox puede no reproducirse (si no soporta MP4).

Solución: Ofrece múltiples fuentes (MP4 + WebM).

───────────────────────────────────────────────────────────
ERROR 2: OLVIDAR EL ATRIBUTO controls
───────────────────────────────────────────────────────────

Incorrecto:
<video src="video.mp4"></video>

Consecuencia: El usuario no puede reproducir el video.

Solución: Siempre añade controls (a menos que tengas controles personalizados con JavaScript).

───────────────────────────────────────────────────────────
ERROR 3: USAR AUTOPLAY SIN MUTED
───────────────────────────────────────────────────────────

Incorrecto:
<video src="video.mp4" autoplay></video>

Consecuencia: La mayoría de los navegadores bloquean el autoplay.

Solución: Añade muted si usas autoplay.

───────────────────────────────────────────────────────────
ERROR 4: NO HACER EL VIDEO RESPONSIVE
───────────────────────────────────────────────────────────

Incorrecto:
<video src="video.mp4" width="800" height="450"></video>

Consecuencia: El video se sale de la pantalla en dispositivos móviles.

Solución: Usa el método del contenedor con padding-bottom o CSS.

───────────────────────────────────────────────────────────
ERROR 5: ARCHIVOS DE AUDIO O VIDEO DEMASIADO PESADOS
───────────────────────────────────────────────────────────

Incorrecto:
Usar un video de 500 MB en una página web.

Consecuencia: Tarda mucho en cargar.

Solución: Comprime y optimiza el archivo antes de subirlo. Usa formatos eficientes como WebM.

===============================================================================
PARTE 12: CÓMO SOLUCIONAR LOS ERRORES
===============================================================================

1. Siempre ofrece múltiples fuentes: MP4 + WebM para video; MP3 + OGG para audio.
2. Siempre añade controls para que el usuario pueda reproducir.
3. Para autoplay, añade muted (a menos que sea un proyecto donde el usuario espera sonido).
4. Usa el contenedor con padding-bottom: 56.25% para hacer el video responsive.
5. Optimiza tus archivos multimedia: comprime el video y el audio antes de subirlos.
6. Prueba en diferentes navegadores (Chrome, Firefox, Safari, Edge).

GLOSARIO DE TÉRMINOS
===============================================================================

- Audio: Sonido insertado en la página con <audio>.
- Video: Imagen en movimiento insertada con <video>.
- controls: Atributo que muestra los controles de reproducción.
- autoplay: Reproducción automática al cargar.
- loop: Repetición continua.
- muted: Audio silenciado por defecto.
- poster: Imagen de portada para el video.
- source: Etiqueta para definir múltiples fuentes.
- MP4: Formato de video más universal.
- WebM: Formato de video libre y eficiente.
- MP3: Formato de audio más universal.
- OGG: Formato de audio libre.
- Responsive: Que se ajusta a diferentes tamaños de pantalla.

¿SABÍAS QUE...?
===============================================================================

Antes de HTML5, para insertar un video en una página web necesitabas usar Flash, un plugin externo que era lento, inseguro y no funcionaba en dispositivos móviles.

HTML5 cambió todo eso, permitiendo que el video sea nativo en el navegador.

Hoy, casi todos los sitios web usan HTML5 para su contenido multimedia.
