PARTE 1: ¿QUÉ ES CANVAS?
===============================================================================

`<canvas>` es una etiqueta HTML5 que permite dibujar gráficos dinámicamente usando JavaScript.

───────────────────────────────────────────────────────────
SINTAXIS BÁSICA
───────────────────────────────────────────────────────────

<canvas id="miCanvas" width="400" height="300"></canvas>

───────────────────────────────────────────────────────────
CARACTERÍSTICAS PRINCIPALES
───────────────────────────────────────────────────────────

1. Es un LIENZO EN BLANCO: No tiene contenido por defecto.
2. Se dibuja con JAVASCRIPT: Usa el contexto 2D para dibujar.
3. ES DINÁMICO: Puedes animar y cambiar el contenido en tiempo real.
4. ES PÍXEL POR PÍXEL: A diferencia de SVG (vectorial), Canvas es rasterizado.
5. ES MÁS RÁPIDO: Para animaciones y juegos, Canvas es más eficiente que SVG.

───────────────────────────────────────────────────────────
CANVAS VS. SVG
───────────────────────────────────────────────────────────

| Característica | Canvas | SVG |
|----------------|--------|-----|
| Tipo | Rasterizado | Vectorial |
| Dibujo | Con JavaScript | Con etiquetas XML |
| Escalabilidad | Se pixela al agrandar | Siempre perfecto |
| Rendimiento | Mejor para animaciones | Mejor para gráficos estáticos |
| Interactividad | Requiere JavaScript | Puede usar CSS y JS |
| Ideal para | Juegos, animaciones, gráficos en tiempo real | Logos, iconos, gráficos interactivos |

PARTE 3: OBTENER EL CONTEXTO DE DIBUJO
===============================================================================

Para dibujar en Canvas, necesitas obtener el "contexto de dibujo" con JavaScript.

───────────────────────────────────────────────────────────
PASOS
───────────────────────────────────────────────────────────

1. Obtener el elemento canvas por su ID.
2. Obtener el contexto 2D con `getContext('2d')`.

───────────────────────────────────────────────────────────
CÓDIGO
───────────────────────────────────────────────────────────

<canvas id="lienzo" width="400" height="300"></canvas>

<script>
    // 1. Obtener el canvas
    const canvas = document.getElementById('lienzo');

    // 2. Obtener el contexto 2D
    const ctx = canvas.getContext('2d');

    // Ahora ctx tiene todos los métodos de dibujo
</script>

───────────────────────────────────────────────────────────
EL OBJETO CTX
───────────────────────────────────────────────────────────

- `ctx.fillStyle` → Color de relleno.
- `ctx.strokeStyle` → Color del borde.
- `ctx.lineWidth` → Ancho de línea.
- `ctx.fillRect(x, y, w, h)` → Dibujar rectángulo relleno.
- `ctx.strokeRect(x, y, w, h)` → Dibujar rectángulo con borde.
- `ctx.clearRect(x, y, w, h)` → Limpiar área.
- `ctx.beginPath()` → Iniciar un trazado.
- `ctx.closePath()` → Cerrar un trazado.
- `ctx.moveTo(x, y)` → Mover a una posición.
- `ctx.lineTo(x, y)` → Dibujar línea a una posición.
- `ctx.arc(x, y, radius, startAngle, endAngle)` → Dibujar arco/círculo.
- `ctx.fill()` → Rellenar el trazado actual.
- `ctx.stroke()` → Dibujar el borde del trazado actual.

PARTE 4: EL SISTEMA DE COORDENADAS DE CANVAS
===============================================================================

Canvas usa un sistema de coordenadas cartesiano.

───────────────────────────────────────────────────────────
COORDENADAS
───────────────────────────────────────────────────────────

- (0, 0) → Esquina SUPERIOR IZQUIERDA.
- El eje X crece hacia la DERECHA.
- El eje Y crece hacia ABAJO.

───────────────────────────────────────────────────────────
REPRESENTACIÓN VISUAL
───────────────────────────────────────────────────────────

(0,0) +--------------------------------------------------+ (400,0)
      |                                                  |
      |                                                  |
      |                                                  |
      |                                                  |
      |                                                  |
      |                                                  |
      +--------------------------------------------------+ (400,300)

- Las coordenadas X van de 0 a width.
- Las coordenadas Y van de 0 a height.

───────────────────────────────────────────────────────────
EJEMPLO DE COORDENADAS
───────────────────────────────────────────────────────────

// Dibujar un rectángulo en (50, 50) de 100x80
ctx.fillRect(50, 50, 100, 80);

// El rectángulo está a 50px de la izquierda y 50px de arriba.
// Tiene 100px de ancho y 80px de alto.

PARTE 5: DIBUJAR RECTÁNGULOS
===============================================================================

Los rectángulos son las figuras más básicas en Canvas.

───────────────────────────────────────────────────────────
fillRect(x, y, width, height) – RECTÁNGULO RELLENO
───────────────────────────────────────────────────────────

<canvas id="lienzo" width="400" height="300"></canvas>

<script>
    const canvas = document.getElementById('lienzo');
    const ctx = canvas.getContext('2d');

    // Rectángulo rojo en (50, 50) de 100x80
    ctx.fillStyle = 'red';
    ctx.fillRect(50, 50, 100, 80);
</script>

───────────────────────────────────────────────────────────
strokeRect(x, y, width, height) – RECTÁNGULO CON BORDE
───────────────────────────────────────────────────────────

<script>
    // Rectángulo con borde azul de 3px
    ctx.strokeStyle = 'blue';
    ctx.lineWidth = 3;
    ctx.strokeRect(200, 50, 100, 80);
</script>

───────────────────────────────────────────────────────────
clearRect(x, y, width, height) – LIMPIAR ÁREA
───────────────────────────────────────────────────────────

<script>
    // Limpiar un área del canvas (hacerla transparente)
    ctx.clearRect(50, 50, 100, 80);
</script>

───────────────────────────────────────────────────────────
EJEMPLO COMPLETO DE RECTÁNGULOS
───────────────────────────────────────────────────────────

<canvas id="rectCanvas" width="400" height="200"></canvas>

<script>
    const canvas = document.getElementById('rectCanvas');

    ctx.fillStyle = 'rgba(0, 0, 255, 0.5)';
    ctx.fillRect(140, 100, 100, 80);
</script>

PARTE 6: DIBUJAR LÍNEAS Y TRAZADOS
===============================================================================

Para dibujar líneas y formas complejas, usamos trazados (paths).

───────────────────────────────────────────────────────────
beginPath() – INICIAR UN TRAZADO
───────────────────────────────────────────────────────────

Indica que vamos a empezar a dibujar un trazado nuevo.

───────────────────────────────────────────────────────────
moveTo(x, y) – MOVER A UNA POSICIÓN
───────────────────────────────────────────────────────────

Mueve el "lápiz" a una posición sin dibujar.

───────────────────────────────────────────────────────────
lineTo(x, y) – DIBUJAR LÍNEA A UNA POSICIÓN
───────────────────────────────────────────────────────────

Dibuja una línea desde la posición actual hasta la nueva posición.

───────────────────────────────────────────────────────────
closePath() – CERRAR EL TRAZADO
───────────────────────────────────────────────────────────

Dibuja una línea desde la posición actual hasta el punto inicial.

───────────────────────────────────────────────────────────
stroke() – DIBUJAR EL BORDE DEL TRAZADO
───────────────────────────────────────────────────────────

Dibuja el contorno del trazado con el color y grosor actuales.

───────────────────────────────────────────────────────────
fill() – RELLENAR EL TRAZADO
───────────────────────────────────────────────────────────

Rellena el trazado con el color actual.

───────────────────────────────────────────────────────────
EJEMPLO: TRIÁNGULO
───────────────────────────────────────────────────────────

<script>
    const canvas = document.getElementById('miCanvas');
    const ctx = canvas.getContext('2d');

    ctx.beginPath();
    ctx.moveTo(100, 20);      // Punto inicial (arriba)
    ctx.lineTo(180, 180);     // Línea a (abajo derecha)
    ctx.lineTo(20, 180);      // Línea a (abajo izquierda)
    ctx.closePath();          // Línea al punto inicial

    ctx.fillStyle = '#ecc94b';
    ctx.fill();

    ctx.strokeStyle = '#1a202c';
    ctx.lineWidth = 3;
    ctx.stroke();
</script>

───────────────────────────────────────────────────────────
EJEMPLO: LÍNEA SIMPLE
───────────────────────────────────────────────────────────

<script>
    ctx.beginPath();
    ctx.moveTo(20, 100);
    ctx.lineTo(380, 100);
    ctx.strokeStyle = '#2b6cb0';
    ctx.lineWidth = 4;
    ctx.stroke();
</script>

───────────────────────────────────────────────────────────
EJEMPLO: LÍNEA DISCONTINUA
───────────────────────────────────────────────────────────

<script>
    ctx.setLineDash([10, 5]);  // 10px de línea, 5px de espacio
    ctx.beginPath();
    ctx.moveTo(20, 150);
    ctx.lineTo(380, 150);
    ctx.strokeStyle = '#e53e3e';
    ctx.lineWidth = 2;
    ctx.stroke();
    ctx.setLineDash([]);  // Restablecer a línea continua
</script>

PARTE 7: DIBUJAR CÍRCULOS Y ARCOS
===============================================================================

Los círculos y arcos se dibujan con `arc()`.

───────────────────────────────────────────────────────────
arc(x, y, radius, startAngle, endAngle, counterclockwise)
───────────────────────────────────────────────────────────

- `x` → Centro X.
- `y` → Centro Y.
- `radius` → Radio.
- `startAngle` → Ángulo de inicio (en radianes).
- `endAngle` → Ángulo de fin (en radianes).
- `counterclockwise` → (opcional) false = horario, true = antihorario.

───────────────────────────────────────────────────────────
RADIANES VS. GRADOS
───────────────────────────────────────────────────────────

- 0 radianes = 0° (derecha).
- π/2 radianes = 90° (abajo).
- π radianes = 180° (izquierda).
- 3π/2 radianes = 270° (arriba).
- 2π radianes = 360° (círculo completo).

Conversión: radianes = grados * Math.PI / 180

───────────────────────────────────────────────────────────
EJEMPLO: CÍRCULO COMPLETO
───────────────────────────────────────────────────────────

<script>
    ctx.beginPath();
    ctx.arc(200, 150, 80, 0, 2 * Math.PI);
    ctx.fillStyle = '#e53e3e';
    ctx.fill();
    ctx.strokeStyle = '#1a202c';
    ctx.lineWidth = 3;
    ctx.stroke();
</script>

───────────────────────────────────────────────────────────
EJEMPLO: ARCO (MEDIO CÍRCULO)
───────────────────────────────────────────────────────────

<script>
    ctx.beginPath();
    ctx.arc(200, 150, 80, 0, Math.PI);
    ctx.closePath();
    ctx.fillStyle = '#2b6cb0';
    ctx.fill();
</script>

───────────────────────────────────────────────────────────
EJEMPLO: SECTOR (PIZZA)
───────────────────────────────────────────────────────────

<script>
    ctx.beginPath();
    ctx.moveTo(200, 150);
    ctx.arc(200, 150, 80, 0, Math.PI / 3);
    ctx.closePath();
    ctx.fillStyle = '#ecc94b';
    ctx.fill();
</script>

───────────────────────────────────────────────────────────
EJEMPLO: CARA SONRIENTE (COMPUESTO)
───────────────────────────────────────────────────────────

<script>
    // Cara
    ctx.beginPath();
    ctx.arc(200, 150, 100, 0, 2 * Math.PI);
    ctx.fillStyle = '#f6ad55';
    ctx.fill();
    ctx.strokeStyle = '#1a202c';
    ctx.lineWidth = 3;
    ctx.stroke();

    // Ojo izquierdo
    ctx.beginPath();
    ctx.arc(160, 120, 12, 0, 2 * Math.PI);
    ctx.fillStyle = '#1a202c';
    ctx.fill();

    // Ojo derecho
    ctx.beginPath();
    ctx.arc(240, 120, 12, 0, 2 * Math.PI);
    ctx.fill();

    // Sonrisa
    ctx.beginPath();
    ctx.arc(200, 160, 50, 0, Math.PI);
    ctx.strokeStyle = '#1a202c';
    ctx.lineWidth = 4;
    ctx.stroke();
</script>

───────────────────────────────────────────────────────────
lineWidth – ANCHO DEL BORDE
───────────────────────────────────────────────────────────

lineCap – ESTILO DE EXTREMOS DE LÍNEA
───────────────────────────────────────────────────────────

ctx.lineCap = 'butt';    // Recto (default)

lineJoin – ESTILO DE UNIÓN DE LÍNEAS
───────────────────────────────────────────────────────────

ctx.lineJoin = 'miter';  // Recto (default)
ctx.lineJoin = 'round';  // Redondeado
ctx.lineJoin = 'bevel';  // Biselado

setLineDash – LÍNEA DISCONTINUA
───────────────────────────────────────────────────────────

ctx.setLineDash([10, 5]);  // 10px línea, 5px espacio

        const canvas3 = document.getElementById('canvas3');
        const ctx3 = canvas3.getContext('2d');
        let dibujando = false;

        // Fondo del canvas3
        ctx3.fillStyle = '#1a202c';
        ctx3.fillRect(0, 0, 600, 300);

        // Estilos de dibujo
        ctx3.lineWidth = 4;
        ctx3.lineCap = 'round';
        ctx3.lineJoin = 'round';
        ctx3.strokeStyle = '#58a6ff';

        canvas3.addEventListener('mousedown', function(event) {
            dibujando = true;
            const rect = canvas3.getBoundingClientRect();
            const x = event.clientX - rect.left;
            const y = event.clientY - rect.top;

            ctx3.beginPath();
            ctx3.moveTo(x, y);
        });

        canvas3.addEventListener('mousemove', function(event) {
            if (!dibujando) return;

            const rect = canvas3.getBoundingClientRect();
            const x = event.clientX - rect.left;
            const y = event.clientY - rect.top;

            ctx3.lineTo(x, y);
            ctx3.stroke();
        });

        canvas3.addEventListener('mouseup', function() {
            dibujando = false;
        });

        canvas3.addEventListener('mouseleave', function() {
            dibujando = false;
        });

        function limpiarCanvas3() {
            ctx3.fillStyle = '#1a202c';
            ctx3.fillRect(0, 0, 600, 300);
        }
    </script>

</body>
</html>

PARTE 12: ERRORES COMUNES CON CANVAS
===============================================================================

───────────────────────────────────────────────────────────
ERROR 1: OLVIDAR EL ID DEL CANVAS
───────────────────────────────────────────────────────────

Incorrecto:
<canvas width="400" height="300"></canvas>
<script>
    const canvas = document.getElementById('lienzo');  // No existe
</script>

Consecuencia: canvas es null y los métodos fallan.

Solución: Añade id al canvas.

───────────────────────────────────────────────────────────
ERROR 2: NO OBTENER EL CONTEXTO CORRECTO
───────────────────────────────────────────────────────────

Incorrecto:
const ctx = canvas.getContext('3d');  // No existe

Correcto:
const ctx = canvas.getContext('2d');

───────────────────────────────────────────────────────────
ERROR 3: OLVIDAR beginPath()
───────────────────────────────────────────────────────────

Incorrecto:
ctx.moveTo(20, 20);
ctx.lineTo(180, 180);
ctx.stroke();

Consecuencia: Se superpone con trazados anteriores.

Solución: Siempre usa beginPath() antes de un nuevo trazado.

───────────────────────────────────────────────────────────
ERROR 4: CONFUNDIR RADIANES CON GRADOS
───────────────────────────────────────────────────────────

Incorrecto:
ctx.arc(100, 100, 50, 0, 180);  // 180° no es 180 radianes

Correcto:
ctx.arc(100, 100, 50, 0, Math.PI);  // 180° = π radianes

───────────────────────────────────────────────────────────
ERROR 5: USAR fillRect CON ctx.stroke
───────────────────────────────────────────────────────────

Incorrecto:
ctx.strokeRect(20, 20, 100, 80);
ctx.fillStyle = 'red';
ctx.fill();  // No funciona con strokeRect

Correcto:
ctx.fillStyle = 'red';
ctx.fillRect(20, 20, 100, 80);
ctx.strokeRect(20, 20, 100, 80);

===============================================================================
PARTE 13: CÓMO SOLUCIONAR LOS ERRORES
===============================================================================

1. Siempre asigna un ID al canvas y usa getElementById.
2. Usa getContext('2d') para el contexto 2D.
3. Siempre usa beginPath() antes de un nuevo trazado.
4. Usa Math.PI para ángulos en radianes.
5. Distingue entre fillRect y strokeRect (no se combinan con fill/stroke).
6. Usa la consola del navegador (F12 → Console) para depurar.
7. Si el canvas no muestra nada, verifica que el contexto no sea null.

GLOSARIO DE TÉRMINOS

- strokeStyle: Color del borde.
- lineWidth: Ancho de línea.
