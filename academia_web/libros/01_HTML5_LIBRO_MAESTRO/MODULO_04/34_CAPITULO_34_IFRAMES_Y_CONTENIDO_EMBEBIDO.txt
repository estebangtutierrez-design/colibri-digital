===============================================================================
PARTE 1: ¿QUÉ ES UN IFRAME?
===============================================================================

`<iframe>` (inline frame) es una etiqueta HTML que permite mostrar otra página web dentro de la página actual.

───────────────────────────────────────────────────────────
SINTAXIS BÁSICA
───────────────────────────────────────────────────────────

<iframe src="https://ejemplo.com" width="600" height="400"></iframe>

───────────────────────────────────────────────────────────
CARACTERÍSTICAS
───────────────────────────────────────────────────────────

1. Muestra contenido de OTRA URL dentro de tu página.
2. Puede mostrar contenido externo (otro sitio) o interno (tu propio sitio).
3. Tiene su propio contexto (CSS, JavaScript, cookies).
4. Puede ser controlado con atributos de seguridad.

───────────────────────────────────────────────────────────
USOS COMUNES DE IFRAMES
───────────────────────────────────────────────────────────

- Mapas de Google Maps.
- Videos de YouTube, Vimeo.
- Formularios de Google Forms.
- Documentos de Google Drive.
- Tweets de Twitter.
- Publicaciones de Instagram.
- Reproductores de audio (Spotify, SoundCloud).
- Contenido de otros sitios.

PARTE 2: ATRIBUTOS PRINCIPALES DE IFRAME
===============================================================================

───────────────────────────────────────────────────────────
src → URL DEL CONTENIDO
───────────────────────────────────────────────────────────

<iframe src="https://ejemplo.com"></iframe>

La URL del contenido que se mostrará dentro del iframe.

───────────────────────────────────────────────────────────
width Y height → TAMAÑO
───────────────────────────────────────────────────────────

<iframe src="https://ejemplo.com" width="800" height="600"></iframe>

Define el ancho y alto del iframe en píxeles.

───────────────────────────────────────────────────────────
title → TÍTULO ACCESIBLE
───────────────────────────────────────────────────────────

<iframe src="https://ejemplo.com" title="Mapa de la ubicación"></iframe>

Describe el contenido del iframe para lectores de pantalla.

───────────────────────────────────────────────────────────
name → NOMBRE DEL IFRAME
───────────────────────────────────────────────────────────

<iframe src="https://ejemplo.com" name="marco"></iframe>

Permite que los enlaces se abran dentro del iframe usando `target="marco"`.

───────────────────────────────────────────────────────────
allowfullscreen → PANTALLA COMPLETA
───────────────────────────────────────────────────────────

<iframe src="https://ejemplo.com" allowfullscreen></iframe>

Permite que el contenido del iframe se vea en pantalla completa.

───────────────────────────────────────────────────────────
loading → CARGA PEROZOSA
───────────────────────────────────────────────────────────

<iframe src="https://ejemplo.com" loading="lazy"></iframe>

Carga el iframe solo cuando está cerca de ser visible.

───────────────────────────────────────────────────────────
sandbox → SEGURIDAD (VER PARTE 5)
───────────────────────────────────────────────────────────

<iframe src="https://ejemplo.com" sandbox></iframe>

Restringe las capacidades del iframe para mayor seguridad.

───────────────────────────────────────────────────────────
allow → PERMISOS ESPECÍFICOS
───────────────────────────────────────────────────────────

<iframe src="https://ejemplo.com" allow="autoplay; fullscreen"></iframe>

Permite funcionalidades específicas como autoplay o pantalla completa.

PARTE 3: CÓMO OBTENER LA URL DE IFRAME DE SERVICIOS POPULARES
===============================================================================

───────────────────────────────────────────────────────────
GOOGLE MAPS
───────────────────────────────────────────────────────────

1. Ve a Google Maps (https://maps.google.com).
2. Busca una ubicación.
3. Haz clic en "Compartir" → "Insertar un mapa".
4. Copia el código HTML que aparece.

EJEMPLO:

<iframe
    src="https://www.google.com/maps/embed?pb=!1m18!1m12!1m3!1d..."
    width="600"
    height="450"
    style="border:0;"
    allowfullscreen=""
    loading="lazy"
    referrerpolicy="no-referrer-when-downgrade">
</iframe>

───────────────────────────────────────────────────────────
YOUTUBE
───────────────────────────────────────────────────────────

1. Ve a YouTube y abre un video.
2. Haz clic en "Compartir" → "Insertar".
3. Copia el código HTML que aparece.

EJEMPLO:

<iframe
    width="560"
    height="315"
    src="https://www.youtube.com/embed/VIDEO_ID"
    title="Título del video"
    frameborder="0"
    allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
    allowfullscreen>
</iframe>

───────────────────────────────────────────────────────────
GOOGLE FORMS
───────────────────────────────────────────────────────────

1. Crea un formulario en Google Forms.
2. Haz clic en "Enviar" → icono de ">" → "Insertar en HTML".
3. Copia el código HTML que aparece.

EJEMPLO:

<iframe src="https://docs.google.com/forms/d/e/ID_DE_LA_ENCUESTA/viewform?embedded=true"
        width="640"
        height="800"
        frameborder="0"
        marginheight="0"
        marginwidth="0">
    Cargando formulario...
</iframe>

───────────────────────────────────────────────────────────
TWITTER (X)
───────────────────────────────────────────────────────────

1. Ve al tweet que quieres incrustar.
2. Haz clic en los tres puntos → "Incrustar tweet".
3. Copia el código HTML que aparece.

EJEMPLO:

<blockquote class="twitter-tweet">
    <p lang="es" dir="ltr">Texto del tweet</p>
    &mdash; Usuario (@usuario) <a href="https://twitter.com/usuario/status/123456789">Fecha</a>
</blockquote>
<script async src="https://platform.twitter.com/widgets.js" charset="utf-8"></script>

(Nota: Twitter usa un bloque de cita, no un iframe, pero funciona de manera similar.)

PARTE 5: SEGURIDAD CON IFRAMES – SANDBOX Y ALLOW
===============================================================================

Los iframes pueden ser peligrosos si no se usan con cuidado.
El atributo `sandbox` restringe las capacidades del iframe.

───────────────────────────────────────────────────────────
SANDBOX BÁSICO (MÁXIMA RESTRICCIÓN)
───────────────────────────────────────────────────────────

<iframe src="https://ejemplo.com" sandbox></iframe>

RESTRICCIONES:
- No ejecuta JavaScript.
- No permite formularios.
- No permite enlaces a otras páginas.
- No permite popups.
- No permite plugins.

───────────────────────────────────────────────────────────
SANDBOX CON PERMISOS ESPECÍFICOS
───────────────────────────────────────────────────────────

<iframe src="https://ejemplo.com" sandbox="allow-scripts allow-forms"></iframe>

PERMISOS COMUNES:
- `allow-scripts` → Permite JavaScript.
- `allow-forms` → Permite formularios.
- `allow-same-origin` → Permite acceso al mismo origen.
- `allow-popups` → Permite ventanas emergentes.
- `allow-modals` → Permite alertas y modales.
- `allow-top-navigation` → Permite navegar en el top-level.

───────────────────────────────────────────────────────────
EJEMPLO DE SANDBOX
───────────────────────────────────────────────────────────

<!-- Máxima seguridad (no hace nada) -->
<iframe src="https://ejemplo.com" sandbox></iframe>

<!-- Permite JavaScript y formularios -->
<iframe src="https://ejemplo.com" sandbox="allow-scripts allow-forms"></iframe>

<!-- Permite JavaScript y el mismo origen -->
<iframe src="https://ejemplo.com" sandbox="allow-scripts allow-same-origin"></iframe>

───────────────────────────────────────────────────────────
ALLOW – PERMISOS MODERNOS
───────────────────────────────────────────────────────────

<iframe src="https://youtube.com/embed/..."
        allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
        allowfullscreen>
</iframe>

PERMISOS COMUNES:
- `accelerometer` → Acelerómetro.
- `autoplay` → Reproducción automática.
- `camera` → Cámara.
- `encrypted-media` → Medios encriptados.
- `fullscreen` → Pantalla completa (usar `allowfullscreen` también).
- `geolocation` → Geolocalización.
- `gyroscope` → Giroscopio.
- `microphone` → Micrófono.
- `payment` → Pagos.

PARTE 9: EXPLICACIÓN LÍNEA POR LÍNEA
===============================================================================

───────────────────────────────────────────────────────────
GOOGLE MAPS
───────────────────────────────────────────────────────────

<iframe
    src="https://www.google.com/maps/embed?pb=!..."
    width="100%"
    height="450"
    style="border:0; max-width: 800px;"
    allowfullscreen=""
    loading="lazy"
    referrerpolicy="no-referrer-when-downgrade"
    title="Mapa de ubicación de Lucas EGA Academy">
</iframe>

- `src` → URL del mapa (generada por Google Maps).
- `width="100%"` → Ocupa todo el ancho del contenedor.
- `height="450"` → Altura fija.
- `style="border:0;"` → Elimina el borde por defecto.
- `allowfullscreen=""` → Permite pantalla completa.
- `loading="lazy"` → Carga perezosa.
- `title` → Texto accesible.

───────────────────────────────────────────────────────────
YOUTUBE
───────────────────────────────────────────────────────────

<iframe
    width="100%"
    height="450"
    src="https://www.youtube.com/embed/dQw4w9WgXcQ"
    title="Video tutorial de HTML5"
    frameborder="0"
    allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
    allowfullscreen
    loading="lazy"
    style="max-width: 800px;">
</iframe>

- `src` → URL del video de YouTube (usando /embed/).
- `allow` → Permisos específicos (autoplay, acelerómetro, etc.).
- `frameborder="0"` → Sin borde.

───────────────────────────────────────────────────────────
srcdoc – CONTENIDO DIRECTO
───────────────────────────────────────────────────────────

<iframe
    srcdoc="
        <html>
            <head>
                <style>body { font-family: Arial; padding: 20px; background: #1a202c; color: white; }</style>
            </head>
            <body>
                <h1>¡Contenido embebido!</h1>
                <p>Este contenido está escrito directamente en el iframe.</p>
            </body>
        </html>
    "
    width="100%"
    height="300"
    title="Contenido embebido directo"
    loading="lazy"
    style="max-width: 800px; border: none; border-radius: 8px;">
</iframe>

- `srcdoc` → Contiene el HTML completo que se mostrará en el iframe.
- Útil para contenido simple sin necesidad de un archivo externo.

───────────────────────────────────────────────────────────
SANDBOX
───────────────────────────────────────────────────────────

<iframe
    srcdoc="..."
    sandbox="allow-scripts"
    title="Iframe con sandbox"
    loading="lazy">
</iframe>

- `sandbox` → Restringe las capacidades del iframe.
- `allow-scripts` → Permite ejecutar JavaScript.

PARTE 10: ERRORES COMUNES CON IFRAMES
===============================================================================

───────────────────────────────────────────────────────────
ERROR 1: OLVIDAR EL ATRIBUTO title
───────────────────────────────────────────────────────────

Incorrecto:
<iframe src="https://maps.google.com"></iframe>

Consecuencia: Problemas de accesibilidad.

Solución: Añade `title="Mapa de ubicación"`.

───────────────────────────────────────────────────────────
ERROR 2: CONTENIDO EXTERNO QUE NO PERMITE IFRAMES
───────────────────────────────────────────────────────────

Algunos sitios bloquean el uso de iframes (por seguridad).

Consecuencia: El iframe muestra un error o queda en blanco.

Solución: Usa sitios que permitan iframes (Google Maps, YouTube, etc.).

───────────────────────────────────────────────────────────
ERROR 3: NO USAR loading="lazy"
───────────────────────────────────────────────────────────

Incorrecto:
<iframe src="https://ejemplo.com"></iframe>

Consecuencia: El iframe se carga aunque no sea visible.

Solución: Usa `loading="lazy"`.

───────────────────────────────────────────────────────────
ERROR 4: NO REDIMENSIONAR EL IFRAME
───────────────────────────────────────────────────────────

Incorrecto:
<iframe src="https://ejemplo.com" width="800" height="600"></iframe>

Consecuencia: En dispositivos móviles, el iframe se sale de la pantalla.

Solución: Usa `width="100%"` y `max-width`.

───────────────────────────────────────────────────────────
ERROR 5: SANDBOX DEMASIADO RESTRICTIVO
───────────────────────────────────────────────────────────

Incorrecto:
<iframe src="https://ejemplo.com" sandbox></iframe>

Consecuencia: El contenido no funciona (sin scripts, sin formularios, etc.).

Solución: Añade los permisos necesarios: `sandbox="allow-scripts allow-forms"`.

GLOSARIO DE TÉRMINOS
===============================================================================

- iframe: Marco que muestra otra página web dentro de la actual.
- src: URL del contenido del iframe.
- srcdoc: HTML directo dentro del iframe.
- sandbox: Atributo de seguridad que restringe capacidades.
- allowfullscreen: Permite pantalla completa.
- loading: Controla la carga (lazy/eager).
- title: Texto accesible para lectores de pantalla.
- frameborder: Borde del iframe (obsoleto, usar CSS).
- allow: Permisos específicos (autoplay, etc.).
