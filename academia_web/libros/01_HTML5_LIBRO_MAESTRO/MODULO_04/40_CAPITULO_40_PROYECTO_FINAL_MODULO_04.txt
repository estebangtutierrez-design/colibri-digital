===============================================================================
PARTE 2: REQUISITOS DEL PROYECTO
===============================================================================

Tu sitio debe cumplir con TODOS los siguientes requisitos:

───────────────────────────────────────────────────────────
ESTRUCTURA Y ORGANIZACIÓN
───────────────────────────────────────────────────────────

✅ Estructura de carpetas profesional (css/, js/, images/, media/, fonts/, assets/, docs/).
✅ Rutas relativas para todos los recursos internos.
✅ Indentación consistente.
✅ Etiquetas siempre cerradas.
✅ Atributos con comillas dobles.
✅ Nombres de archivos en minúscula con guiones.

───────────────────────────────────────────────────────────
SEMÁNTICA
───────────────────────────────────────────────────────────

✅ <header> en todas las páginas.
✅ <nav> en todas las páginas.
✅ <main> en todas las páginas (solo uno).
✅ <section> para secciones temáticas.
✅ <article> para contenido independiente.
✅ <aside> para contenido complementario.
✅ <footer> en todas las páginas.
✅ <figure> y <figcaption> para imágenes con contexto.
✅ <time> para fechas.
✅ <address> para información de contacto.
✅ Jerarquía de encabezados correcta (h1 → h2 → h3).
✅ Solo un <h1> por página.

───────────────────────────────────────────────────────────
FORMULARIOS AVANZADOS (Capítulo 31)
───────────────────────────────────────────────────────────

✅ Campos con required.
✅ Campos con minlength y maxlength.
✅ Campos con pattern (expresiones regulares).
✅ Campos con placeholder.
✅ Campos con autocomplete.
✅ Campos con autofocus.
✅ Mensajes de error personalizados (con oninvalid y oninput).
✅ Checkbox de aceptación de términos.

───────────────────────────────────────────────────────────
CONTROLES AVANZADOS (Capítulo 32)
───────────────────────────────────────────────────────────

✅ <datalist> para autocompletado.
✅ <output> para mostrar resultados.
✅ <progress> para barra de progreso.
✅ <meter> para medidores.
✅ <details> y <summary> para contenido desplegable.

───────────────────────────────────────────────────────────
MULTIMEDIA PROFESIONAL (Capítulo 33)
───────────────────────────────────────────────────────────

✅ Video con controls, poster, preload="metadata".
✅ Video con múltiples formatos (MP4 + WebM).
✅ Video con subtítulos (<track>).
✅ Audio con múltiples formatos (MP3 + OGG).
✅ Video con resolución adaptativa (media queries).

───────────────────────────────────────────────────────────
IFRAMES (Capítulo 34)
───────────────────────────────────────────────────────────

✅ Iframe de Google Maps (responsive).
✅ Iframe de YouTube (video incrustado).
✅ Atributos de seguridad (allowfullscreen, loading="lazy").
✅ Atributo title para accesibilidad.

───────────────────────────────────────────────────────────
SVG (Capítulo 35)
───────────────────────────────────────────────────────────

✅ Logotipo en SVG (texto y formas).
✅ Al menos una figura SVG adicional (icono, ilustración).
✅ viewBox para escalabilidad.

───────────────────────────────────────────────────────────
CANVAS (Capítulo 36)
───────────────────────────────────────────────────────────

✅ Canvas con dibujo interactivo.
✅ Al menos una figura dibujada (círculo, rectángulo, etc.).
✅ Eventos del mouse (click o mousemove).

───────────────────────────────────────────────────────────
ACCESIBILIDAD (Capítulo 37)
───────────────────────────────────────────────────────────

✅ Roles de landmarks (banner, navigation, main, complementary, contentinfo).
✅ aria-label en elementos sin texto visible.
✅ aria-hidden en elementos decorativos.
✅ aria-expanded en menús desplegables.
✅ aria-controls en controles.
✅ aria-live para notificaciones.
✅ alt en todas las imágenes.
✅ label asociado con input en formularios.

───────────────────────────────────────────────────────────
MICRODATOS Y SEO (Capítulo 38)
───────────────────────────────────────────────────────────

✅ Microdatos con Schema.org (al menos 2 tipos).
✅ Open Graph (og:title, og:description, og:image, og:url, og:type).
✅ Twitter Cards.
✅ Meta description en todas las páginas.
✅ Title optimizado en todas las páginas.

───────────────────────────────────────────────────────────
INTEGRACIÓN TOTAL (Capítulo 39)
───────────────────────────────────────────────────────────

✅ CSS externo enlazado desde el <head>.
✅ JavaScript externo con defer desde el <head>.
✅ Menú móvil (hamburguesa) con toggle.
✅ Botón "Volver arriba".
✅ Diseño responsive (media queries).
✅ Puntuación de Lighthouse ≥ 90 en Accesibilidad y SEO.
✅ Código limpio y documentado.

// Espera a que el DOM esté completamente cargado
document.addEventListener('DOMContentLoaded', function() {

    // ============================================================ */
    // 1. MENÚ MÓVIL (TOGGLE)                                        */
    // ============================================================ */
    const menuToggle = document.querySelector('.menu-toggle');
    const navPrincipal = document.querySelector('.nav-principal');

    if (menuToggle && navPrincipal) {
        menuToggle.addEventListener('click', function() {
            // Alternar clase 'abierto' en el menú
            navPrincipal.classList.toggle('abierto');

            // Alternar clase 'activo' en el botón (animación hamburguesa)
            menuToggle.classList.toggle('activo');

            // Actualizar aria-expanded
            const isOpen = navPrincipal.classList.contains('abierto');
            menuToggle.setAttribute('aria-expanded', isOpen);
        });
    }

    // ============================================================ */
    // 2. BOTÓN "VOLVER ARRIBA"                                     */
    // ============================================================ */
    const btnVolverArriba = document.getElementById('btn-volver-arriba');

    if (btnVolverArriba) {
        // Mostrar/ocultar botón según el desplazamiento
        window.addEventListener('scroll', function() {
            if (window.scrollY > 300) {
                btnVolverArriba.classList.add('visible');
            } else {
                btnVolverArriba.classList.remove('visible');
            }
        });

        // Volver arriba al hacer clic
        btnVolverArriba.addEventListener('click', function() {
            window.scrollTo({ top: 0, behavior: 'smooth' });
        });
    }

    // ============================================================ */
    // 3. CANVAS – DIBUJO INTERACTIVO                                */
    // ============================================================ */
    const canvas = document.getElementById('miCanvas');
    if (canvas) {
        const ctx = canvas.getContext('2d');
        let colorActual = '#2b6cb0';

        // Función para dibujar el círculo con el color actual
        function dibujarCirculo(color) {
            ctx.clearRect(0, 0, canvas.width, canvas.height);

            // Fondo
            ctx.fillStyle = '#1a202c';
            ctx.fillRect(0, 0, canvas.width, canvas.height);

            // Círculo
            ctx.beginPath();
            ctx.arc(150, 100, 70, 0, 2 * Math.PI);
            ctx.fillStyle = color || colorActual;
            ctx.fill();

            // Borde
            ctx.strokeStyle = '#58a6ff';
            ctx.lineWidth = 3;
            ctx.stroke();

            // Texto
            ctx.fillStyle = 'white';
            ctx.font = 'bold 36px Arial';
            ctx.textAlign = 'center';
            ctx.textBaseline = 'middle';
            ctx.fillText('L', 150, 100);
        }

        // Dibujar el círculo inicial
        dibujarCirculo(colorActual);

        // Cambiar color al hacer clic en el canvas
        const colores = ['#e53e3e', '#2b6cb0', '#38a169', '#ecc94b', '#f6ad55', '#805ad5', '#ed64a6'];

        canvas.addEventListener('click', function(event) {
            const rect = canvas.getBoundingClientRect();
            const x = event.clientX - rect.left;
            const y = event.clientY - rect.top;

            // Solo cambiar si el clic está dentro del área del canvas
            if (x >= 0 && x <= canvas.width && y >= 0 && y <= canvas.height) {
                const colorAleatorio = colores[Math.floor(Math.random() * colores.length)];
                colorActual = colorAleatorio;
                dibujarCirculo(colorAleatorio);
            }
        });
    }

    // ============================================================ */
    // 4. CONSOLA DE BIENVENIDA                                     */
    // ============================================================ */
    console.log('🦉 Lucas EGA Academy - Proyecto Final Módulo IV');
    console.log('📚 Todas las técnicas de HTML5 integradas.');
    console.log('🚀 ¡Sitio web completo y profesional!');
    console.log('🎨 Próximo paso: CSS3');

    // ============================================================ */
    // 5. NOTIFICACIÓN DE BIENVENIDA (aria-live)                    */
    // ============================================================ */
    // Mensaje de bienvenida que se anuncia con aria-live
    setTimeout(function() {
        const mensaje = document.getElementById('mensaje-notificacion');
        if (mensaje) {
            mensaje.textContent = '🦉 Bienvenido a Lucas EGA Academy. ¡Comienza tu camino hoy!';
        }
    }, 1000);
});
