PARTE 5: JAVASCRIPT – INTERACTIVIDAD
===============================================================================

El JavaScript debe añadir interactividad sin bloquear la carga.

───────────────────────────────────────────────────────────
CÓDIGO JAVASCRIPT (js/app.js)
───────────────────────────────────────────────────────────

// ================================================================ */
// ESPERA A QUE EL DOM ESTÉ LISTO                                   */
// ================================================================ */
document.addEventListener('DOMContentLoaded', function() {

    // ============================================================ */
    // 1. MENÚ MÓVIL (TOGGLE)                                       */
    // ============================================================ */
    const menuToggle = document.querySelector('.menu-toggle');
    const navPrincipal = document.querySelector('.nav-principal');

    if (menuToggle && navPrincipal) {
        menuToggle.addEventListener('click', function() {
            navPrincipal.classList.toggle('abierto');
            const isOpen = navPrincipal.classList.contains('abierto');
            this.setAttribute('aria-expanded', isOpen);
        });
    }

    // ============================================================ */
    // 2. BOTÓN "VOLVER ARRIBA"                                     */
    // ============================================================ */
    const btnVolver = document.createElement('button');
    btnVolver.textContent = '⬆';
    btnVolver.style.cssText = `
        position: fixed; bottom: 30px; right: 30px;
        background: #2b6cb0; color: white; border: none;
        border-radius: 50%; width: 50px; height: 50px;
        font-size: 24px; cursor: pointer; opacity: 0;
        transition: opacity 0.3s; z-index: 1000;
        box-shadow: 0 4px 10px rgba(0,0,0,0.2);
    `;
    document.body.appendChild(btnVolver);

    window.addEventListener('scroll', function() {
        btnVolver.style.opacity = window.scrollY > 300 ? '1' : '0';
    });

    btnVolver.addEventListener('click', function() {
        window.scrollTo({ top: 0, behavior: 'smooth' });
    });

    // ============================================================ */
    // 3. CANVAS – DIBUJO SIMPLE                                     */
    // ============================================================ */
    const canvas = document.getElementById('miCanvas');
    if (canvas) {
        const ctx = canvas.getContext('2d');
        // Fondo
        ctx.fillStyle = '#1a202c';
        ctx.fillRect(0, 0, canvas.width, canvas.height);
        // Círculo
        ctx.beginPath();
        ctx.arc(100, 100, 60, 0, 2 * Math.PI);
        ctx.fillStyle = '#2b6cb0';
        ctx.fill();
        ctx.strokeStyle = '#58a6ff';
        ctx.lineWidth = 3;
        ctx.stroke();
        // Texto
        ctx.fillStyle = 'white';
        ctx.font = 'bold 40px Arial';
        ctx.textAlign = 'center';
        ctx.textBaseline = 'middle';
        ctx.fillText('L', 100, 100);
    }

    // ============================================================ */
    // 4. MENSAJE DE BIENVENIDA EN CONSOLA                          */
    // ============================================================ */
    console.log('🦉 Lucas EGA Academy - Integración Total HTML5');
    console.log('📚 Todas las técnicas trabajando juntas.');

});

PARTE 7: ERRORES COMUNES EN LA INTEGRACIÓN
===============================================================================

───────────────────────────────────────────────────────────
ERROR 1: RUTAS INCORRECTAS DE ARCHIVOS
───────────────────────────────────────────────────────────

Incorrecto:
<link rel="stylesheet" href="style.css">  <!-- style.css no está en la raíz -->

Correcto:
<link rel="stylesheet" href="css/style.css">

Solución: Verifica que las rutas relativas sean correctas.

───────────────────────────────────────────────────────────
ERROR 2: SCRIPTS SIN defer O EN EL LUGAR INCORRECTO
───────────────────────────────────────────────────────────

Incorrecto:
<script src="js/app.js"></script>  <!-- Bloquea el renderizado -->

Correcto:
<script src="js/app.js" defer></script>

Solución: Siempre usa defer en el <head> o coloca los scripts al final del <body>.

───────────────────────────────────────────────────────────
ERROR 3: MICRODATOS INCOMPLETOS O MAL ANIDADOS
───────────────────────────────────────────────────────────

Solución: Usa el validador de Google para verificar los microdatos.

───────────────────────────────────────────────────────────
ERROR 4: FORMULARIO SIN VALIDACIÓN O SIN SERVER-SIDE
───────────────────────────────────────────────────────────

Solución: Añade validaciones HTML5 y siempre valida en el servidor.

───────────────────────────────────────────────────────────
ERROR 5: VIDEO SIN SUBTÍTULOS O SIN TEXTO ALTERNATIVO
───────────────────────────────────────────────────────────

Solución: Añade <track> para subtítulos y texto alternativo entre las etiquetas.

===============================================================================

1. Verifica todas las rutas relativas (css/, js/, images/, media/).
2. Usa `defer` en los scripts para no bloquear el renderizado.
3. Valida microdatos con el Rich Results Test de Google.
4. Añade validaciones HTML5 (required, pattern, etc.) y valida en el servidor.
5. Añade subtítulos a los videos y texto alternativo.
