PARTE 1: ¿QUÉ ES LA API DE GEOLOCALIZACIÓN?
===============================================================================

La API de Geolocalización de HTML5 permite obtener la ubicación geográfica del usuario.

───────────────────────────────────────────────────────────
CARACTERÍSTICAS
───────────────────────────────────────────────────────────

1. REQUIERE PERMISO: El usuario debe aceptar compartir su ubicación.
2. PRECISIÓN: Puede ser precisa (GPS) o aproximada (red WiFi, IP).
3. ASÍNCRONA: Las operaciones no bloquean la ejecución.
4. PRIVACIDAD: Solo funciona con el consentimiento del usuario.
5. DISPOSITIVOS: Funciona en móviles y computadoras.

───────────────────────────────────────────────────────────
FUENTES DE UBICACIÓN
───────────────────────────────────────────────────────────

1. GPS: La más precisa (si el dispositivo tiene GPS).
2. WiFi: Basada en redes WiFi cercanas.
3. Red celular: Basada en torres de telefonía.
4. IP: La menos precisa (basada en la dirección IP).

───────────────────────────────────────────────────────────
ACCESO A LA API
───────────────────────────────────────────────────────────

La API está disponible a través del objeto `navigator.geolocation` en JavaScript.

```javascript
// Verificar si el navegador soporta geolocalización
if ("geolocation" in navigator) {
    // El navegador soporta geolocalización
    console.log("✅ Geolocalización disponible");
} else {
    // El navegador NO soporta geolocalización
    console.log("❌ Geolocalización no disponible");
}

PARTE 2: getCurrentPosition() – OBTENER LA UBICACIÓN UNA VEZ
===============================================================================

navigator.geolocation.getCurrentPosition() obtiene la ubicación actual del usuario una sola vez.

───────────────────────────────────────────────────────────
SINTAXIS
───────────────────────────────────────────────────────────

javascript
navigator.geolocation.getCurrentPosition(
    function(posicion) {
        // Éxito: manejar la posición
    },
    function(error) {
        // Error: manejar el error
    },
    {
        // Opciones
        enableHighAccuracy: true,
        timeout: 5000,
        maximumAge: 0
    }
);
───────────────────────────────────────────────────────────
PARÁMETROS
───────────────────────────────────────────────────────────

callback de éxito: Función que se ejecuta cuando se obtiene la ubicación.

callback de error: Función que se ejecuta si ocurre un error.

opciones: Configuración de la geolocalización (opcional).

───────────────────────────────────────────────────────────
EJEMPLO BÁSICO
───────────────────────────────────────────────────────────

html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Geolocalización - Básico</title>
</head>
<body>
    <h1>📍 Mi ubicación</h1>

    <button onclick="obtenerUbicacion()">Obtener ubicación</button>

    <div id="resultado">
        <p>Presiona el botón para obtener tu ubicación.</p>
    </div>

    <script>
        function obtenerUbicacion() {
            const resultado = document.getElementById('resultado');

            // Verificar soporte
            if (!("geolocation" in navigator)) {
                resultado.innerHTML = '❌ Tu navegador no soporta geolocalización.';
                return;
            }

            resultado.innerHTML = '🔄 Obteniendo ubicación...';

            // Obtener ubicación
            navigator.geolocation.getCurrentPosition(
                function(posicion) {
                    // Éxito
                    const lat = posicion.coords.latitude;
                    const lon = posicion.coords.longitude;
                    const precision = posicion.coords.accuracy;

                    resultado.innerHTML = `
                        ✅ Ubicación obtenida:<br>
                        <strong>Latitud:</strong> ${lat}<br>
                        <strong>Longitud:</strong> ${lon}<br>
                        <strong>Precisión:</strong> ${precision} metros
                    `;
                },
                function(error) {
                    // Error
                    resultado.innerHTML = '❌ Error: ' + error.message;
                }
            );
        }
    </script>
</body>
</html>
───────────────────────────────────────────────────────────
EL OBJETO POSITION
───────────────────────────────────────────────────────────

Cuando la geolocalización tiene éxito, recibe un objeto Position con:

coords.latitude → Latitud en grados decimales.

coords.longitude → Longitud en grados decimales.

coords.accuracy → Precisión en metros.

coords.altitude → Altitud en metros (si está disponible).

coords.altitudeAccuracy → Precisión de la altitud.

coords.heading → Dirección (grados, 0-360).

coords.speed → Velocidad (metros/segundo).

timestamp → Fecha y hora de la medición.

PARTE 3: MANEJAR ERRORES DE GEOLOCALIZACIÓN
===============================================================================

Cuando la geolocalización falla, recibe un objeto PositionError.

───────────────────────────────────────────────────────────
TIPOS DE ERRORES
───────────────────────────────────────────────────────────

PERMISSION_DENIED → El usuario denegó el permiso.

POSITION_UNAVAILABLE → No se pudo obtener la ubicación.

TIMEOUT → La solicitud expiró.

───────────────────────────────────────────────────────────
CÓDIGOS DE ERROR
───────────────────────────────────────────────────────────

Código	Constante	Descripción
1	PERMISSION_DENIED	El usuario denegó el permiso.
2	POSITION_UNAVAILABLE	La ubicación no está disponible.
3	TIMEOUT	La solicitud expiró.
───────────────────────────────────────────────────────────
EJEMPLO CON MANEJO DE ERRORES
───────────────────────────────────────────────────────────

javascript
function obtenerUbicacion() {
    if (!("geolocation" in navigator)) {
        mostrarError('Geolocalización no soportada');
        return;
    }

    navigator.geolocation.getCurrentPosition(
        function(posicion) {
            // Éxito
            mostrarUbicacion(posicion);
        },
        function(error) {
            // Error
            let mensaje = '';

            switch (error.code) {
                case error.PERMISSION_DENIED:
                    mensaje = 'El usuario denegó el permiso de ubicación.';
                    break;
                case error.POSITION_UNAVAILABLE:
                    mensaje = 'No se pudo obtener la ubicación. Intenta de nuevo.';
                    break;
                case error.TIMEOUT:
                    mensaje = 'La solicitud de ubicación expiró.';
                    break;
                default:
                    mensaje = 'Error desconocido: ' + error.message;
            }

            mostrarError(mensaje);
        },
        {
            enableHighAccuracy: true,
            timeout: 10000,
            maximumAge: 0
        }
    );
}

PARTE 4: OPCIONES DE GEOLOCALIZACIÓN
===============================================================================

Las opciones permiten controlar cómo se obtiene la ubicación.

───────────────────────────────────────────────────────────
enableHighAccuracy – PRECISIÓN ALTA
───────────────────────────────────────────────────────────

javascript
{ enableHighAccuracy: true }  // Usa GPS (más preciso, más batería)
{ enableHighAccuracy: false } // Usa WiFi/IP (menos preciso, menos batería)
true → Usa el GPS del dispositivo (más preciso, consume más batería).

false → Usa WiFi, IP o torres celulares (menos preciso, más rápido).

───────────────────────────────────────────────────────────
timeout – TIEMPO DE ESPERA
───────────────────────────────────────────────────────────

javascript
{ timeout: 5000 } // 5 segundos
Tiempo máximo (en milisegundos) para esperar una respuesta.

───────────────────────────────────────────────────────────
maximumAge – EDAD MÁXIMA DE LA UBICACIÓN
───────────────────────────────────────────────────────────

javascript
{ maximumAge: 60000 } // 60 segundos
Edad máxima (en milisegundos) de una ubicación en caché.

0 → Siempre obtener una ubicación nueva.

60000 → Usar una ubicación de hasta 60 segundos de antigüedad.

───────────────────────────────────────────────────────────
EJEMPLO CON OPCIONES
───────────────────────────────────────────────────────────

javascript
const opciones = {
    enableHighAccuracy: true,
    timeout: 10000,
    maximumAge: 0
};

navigator.geolocation.getCurrentPosition(
    exito,
    error,
    opciones
);

PARTE 5: watchPosition() – SEGUIR LA UBICACIÓN EN TIEMPO REAL
===============================================================================

navigator.geolocation.watchPosition() permite seguir la ubicación del usuario en tiempo real.

───────────────────────────────────────────────────────────
SINTAXIS
───────────────────────────────────────────────────────────

javascript
const watchId = navigator.geolocation.watchPosition(
    function(posicion) {
        // Se ejecuta cada vez que la ubicación cambia
    },
    function(error) {
        // Manejar error
    },
    {
        // Opciones
        enableHighAccuracy: true,
        timeout: 5000,
        maximumAge: 0
    }
);
───────────────────────────────────────────────────────────
EJEMPLO: SEGUIR UBICACIÓN EN TIEMPO REAL
───────────────────────────────────────────────────────────

html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Geolocalización - En tiempo real</title>
</head>
<body>
    <h1>📍 Seguimiento en tiempo real</h1>

    <button onclick="iniciarSeguimiento()">▶ Iniciar seguimiento</button>
    <button onclick="detenerSeguimiento()">⏹ Detener seguimiento</button>

    <div id="resultado">
        <p>Inicia el seguimiento para ver tu ubicación en tiempo real.</p>
    </div>

    <script>
        let watchId = null;

        function iniciarSeguimiento() {
            const resultado = document.getElementById('resultado');

            if (!("geolocation" in navigator)) {
                resultado.innerHTML = '❌ Geolocalización no soportada.';
                return;
            }

            resultado.innerHTML = '🔄 Iniciando seguimiento...';

            watchId = navigator.geolocation.watchPosition(
                function(posicion) {
                    // Éxito: actualizar la ubicación
                    const lat = posicion.coords.latitude.toFixed(6);
                    const lon = posicion.coords.longitude.toFixed(6);
                    const precision = posicion.coords.accuracy;

                    resultado.innerHTML = `
                        ✅ Ubicación actualizada:<br>
                        <strong>Latitud:</strong> ${lat}<br>
                        <strong>Longitud:</strong> ${lon}<br>
                        <strong>Precisión:</strong> ${precision} metros<br>
                        <small>Actualizado: ${new Date().toLocaleTimeString()}</small>
                    `;
                },
                function(error) {
                    resultado.innerHTML = '❌ Error: ' + error.message;
                },
                {
                    enableHighAccuracy: true,
                    timeout: 5000,
                    maximumAge: 0
                }
            );
        }

        function detenerSeguimiento() {
            if (watchId !== null) {
                navigator.geolocation.clearWatch(watchId);
                watchId = null;
                document.getElementById('resultado').innerHTML = '⏹ Seguimiento detenido.';
            }
        }
    </script>
</body>
</html>
───────────────────────────────────────────────────────────
clearWatch() – DETENER EL SEGUIMIENTO
───────────────────────────────────────────────────────────

javascript
navigator.geolocation.clearWatch(watchId);

PARTE 6: PRIVACIDAD Y PERMISOS
===============================================================================

La geolocalización es una herramienta poderosa que debe usarse con responsabilidad.

───────────────────────────────────────────────────────────
SOLICITAR PERMISO
───────────────────────────────────────────────────────────

Cuando llamas a getCurrentPosition() o watchPosition(), el navegador solicita permiso al usuario.

El usuario puede:

Permitir: El sitio puede acceder a la ubicación.

Bloquear: El sitio no puede acceder a la ubicación.

Preguntar siempre: El navegador pregunta cada vez.

───────────────────────────────────────────────────────────
VERIFICAR EL ESTADO DEL PERMISO (PERMISSIONS API)
───────────────────────────────────────────────────────────

javascript
if ("permissions" in navigator) {
    navigator.permissions.query({ name: "geolocation" }).then(function(result) {
        if (result.state === "granted") {
            console.log("Permiso concedido.");
        } else if (result.state === "prompt") {
            console.log("Preguntará al usuario.");
        } else if (result.state === "denied") {
            console.log("Permiso denegado.");
        }
    });
}
───────────────────────────────────────────────────────────
BUENAS PRÁCTICAS DE PRIVACIDAD
───────────────────────────────────────────────────────────

Siempre pide permiso → No asumas que el usuario quiere compartir su ubicación.

Explica por qué necesitas la ubicación → Di algo como: "Necesitamos tu ubicación para mostrarte el clima de tu ciudad."

Ofrece una alternativa → Si el usuario no quiere compartir su ubicación, ofrece la opción de ingresarla manualmente.

No almacenes la ubicación innecesariamente → Si no la necesitas, no la guardes.

Usa HTTPS → La geolocalización solo funciona en sitios seguros (HTTPS).

        function obtenerUbicacion() {

                        <p><strong>Precisión:</strong> ${precision} metros</p>
                        <p><strong>Hora:</strong> ${new Date().toLocaleString()}</p>
                        <p><a href="https://www.google.com/maps?q=${lat},${lon}" target="_blank">Abrir en Google Maps</a></p>
                    `;

                    info.innerHTML = '✅ Ubicación obtenida correctamente.';
                },
                function(error) {
                    let mensaje = '';

                    switch (error.code) {
                        case error.PERMISSION_DENIED:
                            mensaje = 'El usuario denegó el permiso de ubicación.';
                            break;
                        case error.POSITION_UNAVAILABLE:
                            mensaje = 'No se pudo obtener la ubicación. Intenta de nuevo.';
                            break;

                            mensaje = 'La solicitud de ubicación expiró.';

                    info.innerHTML = '❌ ' + mensaje;
                },
                {
                    enableHighAccuracy: true,
                    timeout: 10000,
                    maximumAge: 0

        }

PARTE 9: ERRORES COMUNES CON GEOLOCALIZACIÓN
===============================================================================

───────────────────────────────────────────────────────────
ERROR 1: NO VERIFICAR SOPORTE DEL NAVEGADOR
───────────────────────────────────────────────────────────

❌ Incorrecto:

javascript
navigator.geolocation.getCurrentPosition(...);
✅ Correcto:

javascript
if ("geolocation" in navigator) {
    navigator.geolocation.getCurrentPosition(...);
} else {
    // Geolocalización no soportada
}
───────────────────────────────────────────────────────────
ERROR 2: NO MANEJAR EL PERMISO DENEGADO
───────────────────────────────────────────────────────────

❌ Incorrecto:

javascript
navigator.geolocation.getCurrentPosition(function(pos) {
    // Usar ubicación
});
✅ Correcto:

javascript
navigator.geolocation.getCurrentPosition(
    function(pos) { /* éxito */ },
    function(error) {
        if (error.code === error.PERMISSION_DENIED) {
            // Mostrar mensaje amigable al usuario
        }
    }
);
───────────────────────────────────────────────────────────
ERROR 3: NO USAR HTTPS
───────────────────────────────────────────────────────────

La geolocalización SOLO funciona en sitios seguros (HTTPS).

Si estás desarrollando localmente, usa localhost.
Si estás en producción, usa HTTPS.

───────────────────────────────────────────────────────────
ERROR 4: TIMEOUT DEMASIADO CORTO
───────────────────────────────────────────────────────────

❌ Incorrecto:

javascript
{ timeout: 1000 } // 1 segundo
✅ Correcto:

javascript
{ timeout: 10000 } // 10 segundos
La geolocalización puede tardar varios segundos, especialmente en interiores o sin GPS.

───────────────────────────────────────────────────────────
ERROR 5: NO MANEJAR EL CASO DE USO SIN GPS
───────────────────────────────────────────────────────────

En interiores o sin GPS, la precisión puede ser baja (cientos de metros).
Siempre verifica la precisión antes de usar la ubicación.

javascript
if (posicion.coords.accuracy > 100) {
    console.log("La precisión es baja, puede no ser exacta.");
}

PARTE 10: CÓMO SOLUCIONAR LOS ERRORES
===============================================================================

Siempre verifica si "geolocation" in navigator antes de usar la API.

Siempre maneja el error de permiso denegado de forma amigable.

Usa HTTPS para la geolocalización (o localhost para desarrollo).

Usa un timeout razonable (10 segundos o más).

Verifica la precisión de la ubicación antes de usarla.

Siempre explica al usuario por qué necesitas su ubicación.

GLOSARIO DE TÉRMINOS
===============================================================================

Geolocalización: Obtención de la ubicación geográfica de un dispositivo.

API: Application Programming Interface.

getCurrentPosition(): Obtiene la ubicación una vez.

watchPosition(): Sigue la ubicación en tiempo real.

clearWatch(): Detiene el seguimiento.

Latitud: Coordenada norte-sur (grados decimales).

Longitud: Coordenada este-oeste (grados decimales).

Precisión: Margen de error en metros.

Permiso: Consentimiento del usuario para acceder a la ubicación.

HTTPS: Protocolo seguro (requerido para geolocalización).
