PARTE 1: ¿QUÉ ES LOCALSTORAGE?
===============================================================================

LocalStorage es una API de HTML5 que permite almacenar datos de forma persistente en el navegador del usuario.

───────────────────────────────────────────────────────────
CARACTERÍSTICAS
───────────────────────────────────────────────────────────

1. PERSISTENTE: Los datos no se borran al cerrar el navegador.
2. SIN CADUCIDAD: Los datos permanecen hasta que se eliminan manualmente.
3. POR DOMINIO: Cada sitio web tiene su propio espacio de almacenamiento.
4. TAMAÑO: Aproximadamente 5-10 MB (dependiendo del navegador).
5. SOLO TEXTO: Solo se pueden guardar cadenas de texto.
6. SÍNCRONO: Las operaciones son síncronas (bloquean la ejecución).

───────────────────────────────────────────────────────────
ACCESO A LOCALSTORAGE
───────────────────────────────────────────────────────────

LocalStorage está disponible a través del objeto `localStorage` en JavaScript.

```javascript
// Guardar datos
localStorage.setItem('clave', 'valor');

// Leer datos
const dato = localStorage.getItem('clave');

// Eliminar un dato
localStorage.removeItem('clave');

// Eliminar todos los datos
localStorage.clear();
───────────────────────────────────────────────────────────
LOCALSTORAGE VS. COOKIES
───────────────────────────────────────────────────────────

Característica	LocalStorage	Cookies
Capacidad	~5-10 MB	~4 KB
Expiración	Permanente	Fecha específica
Envío al servidor	No	Sí (en cada petición)
Acceso	Solo JavaScript	JavaScript y servidor
Uso	Datos del cliente	Autenticación, sesiones

PARTE 2: GUARDAR DATOS – setItem()
===============================================================================

localStorage.setItem(clave, valor) guarda un dato en el almacenamiento local.

───────────────────────────────────────────────────────────
SINTAXIS
───────────────────────────────────────────────────────────

javascript
localStorage.setItem('nombre', 'Ana');
localStorage.setItem('edad', '25');
localStorage.setItem('tema', 'oscuro');
───────────────────────────────────────────────────────────
EJEMPLO COMPLETO
───────────────────────────────────────────────────────────

html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>LocalStorage - Guardar</title>
</head>
<body>
    <h1>Guardar en LocalStorage</h1>

    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" placeholder="Escribe tu nombre">
    <button onclick="guardarNombre()">Guardar</button>

    <p id="mensaje"></p>

    <script>
        function guardarNombre() {
            const nombre = document.getElementById('nombre').value;

            if (nombre === '') {
                document.getElementById('mensaje').textContent = '❌ Escribe un nombre';
                return;
            }

            // Guardar en LocalStorage
            localStorage.setItem('nombre_usuario', nombre);

            document.getElementById('mensaje').textContent = '✅ Nombre guardado: ' + nombre;
        }
    </script>
</body>
</html>
───────────────────────────────────────────────────────────
REGLAS IMPORTANTES
───────────────────────────────────────────────────────────

La CLAVE debe ser un texto (string).

El VALOR debe ser un texto (string).

Si la clave ya existe, se SOBRESCRIBE el valor.

===============================================================================

localStorage.getItem(clave) lee un dato del almacenamiento local.

SINTAXIS

javascript
const nombre = localStorage.getItem('nombre');
───────────────────────────────────────────────────────────
EJEMPLO COMPLETO
───────────────────────────────────────────────────────────

html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>LocalStorage - Leer</title>
</head>
<body>
    <h1>Leer de LocalStorage</h1>

    <button onclick="cargarNombre()">Cargar nombre guardado</button>
    <p id="mostrar"></p>

    <script>
        function cargarNombre() {
            const nombre = localStorage.getItem('nombre_usuario');

            if (nombre === null) {
                document.getElementById('mostrar').textContent = '❌ No hay nombre guardado';
            } else {
                document.getElementById('mostrar').textContent = '👤 Nombre guardado: ' + nombre;
            }
        }

        // Cargar automáticamente al abrir la página
        window.onload = cargarNombre;
    </script>

MANEJO DE DATOS NO EXISTENTES
───────────────────────────────────────────────────────────

Si la clave no existe, getItem() devuelve null.

javascript
const dato = localStorage.getItem('clave_inexistente');
console.log(dato); // null

PARTE 4: MODIFICAR DATOS
===============================================================================

Para modificar un dato, simplemente usa setItem() con la misma clave.

───────────────────────────────────────────────────────────
EJEMPLO
───────────────────────────────────────────────────────────

javascript
// Guardar inicialmente
localStorage.setItem('tema', 'claro');

// Modificar (sobrescribir)
localStorage.setItem('tema', 'oscuro');

// Ahora el valor es 'oscuro'
───────────────────────────────────────────────────────────
EJEMPLO COMPLETO: CAMBIAR TEMA
───────────────────────────────────────────────────────────

html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>LocalStorage - Modificar</title>
    <style>
        body { transition: background 0.3s, color 0.3s; }
        body.oscuro { background: #1a202c; color: #c9d1d9; }
        body.claro { background: #ffffff; color: #1a202c; }

    <h1>Selector de tema</h1>

    <button onclick="cambiarTema('claro')">☀️ Claro</button>

    <p id="tema-actual"></p>

    <script>
        function cambiarTema(tema) {
            // Guardar en LocalStorage
            localStorage.setItem('tema_preferido', tema);

            // Aplicar tema
            document.body.className = tema;

            document.getElementById('tema-actual').textContent = 'Tema actual: ' + tema;
        }

        // Cargar tema guardado al abrir la página
        window.onload = function() {
            const temaGuardado = localStorage.getItem('tema_preferido');
            if (temaGuardado) {
                document.body.className = temaGuardado;
                document.getElementById('tema-actual').textContent = 'Tema actual: ' + temaGuardado;
            }
        };
    </script>
</body>
</html>

PARTE 5: ELIMINAR DATOS – removeItem() Y clear()
===============================================================================

───────────────────────────────────────────────────────────

───────────────────────────────────────────────────────────

javascript
// Eliminar el nombre guardado

// Verificar que se eliminó
console.log(localStorage.getItem('nombre_usuario')); // null
───────────────────────────────────────────────────────────
clear() – ELIMINAR TODOS LOS DATOS
───────────────────────────────────────────────────────────

javascript
// Eliminar todo lo guardado en LocalStorage
localStorage.clear();
───────────────────────────────────────────────────────────
EJEMPLO COMPLETO
───────────────────────────────────────────────────────────

html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>LocalStorage - Eliminar</title>
</head>
<body>
    <h1>Gestión de LocalStorage</h1>

    <label for="dato">Dato a guardar:</label>
    <input type="text" id="dato" placeholder="Escribe algo">
    <button onclick="guardarDato()">Guardar</button>

    <button onclick="mostrarDatos()">Mostrar todos los datos</button>
    <button onclick="eliminarTodo()">Eliminar todo</button>

    <pre id="resultado"></pre>

    <script>
        function guardarDato() {
            const valor = document.getElementById('dato').value;
            if (valor === '') return;

            const clave = 'dato_' + Date.now();
            localStorage.setItem(clave, valor);
            document.getElementById('dato').value = '';
            mostrarDatos();
        }

        function mostrarDatos() {
            let texto = '=== DATOS GUARDADOS ===\n';
            let hayDatos = false;

            for (let i = 0; i < localStorage.length; i++) {
                const clave = localStorage.key(i);
                const valor = localStorage.getItem(clave);
                texto += clave + ': ' + valor + '\n';
                hayDatos = true;
            }

            if (!hayDatos) {
                texto += 'No hay datos guardados.';
            }

            document.getElementById('resultado').textContent = texto;
        }

        function eliminarTodo() {
            if (confirm('¿Seguro que quieres eliminar todos los datos?')) {
                localStorage.clear();
                mostrarDatos();
            }
        }

        // Mostrar datos al cargar
        window.onload = mostrarDatos;
    </script>
</body>
</html>

PARTE 6: GUARDAR OBJETOS Y ARRAYS – JSON
===============================================================================

LocalStorage solo guarda texto (strings).
Para guardar objetos o arrays, usa JSON.

───────────────────────────────────────────────────────────
JSON.stringify() – CONVERTIR OBJETO A TEXTO
───────────────────────────────────────────────────────────

javascript
const usuario = {
    nombre: 'Ana',
    edad: 25,
    email: 'ana@email.com'
};

// Convertir a texto JSON
const texto = JSON.stringify(usuario);
localStorage.setItem('usuario', texto);
───────────────────────────────────────────────────────────
JSON.parse() – CONVERTIR TEXTO A OBJETO
───────────────────────────────────────────────────────────

javascript
const texto = localStorage.getItem('usuario');
const usuario = JSON.parse(texto);

console.log(usuario.nombre); // 'Ana'
console.log(usuario.edad);   // 25
───────────────────────────────────────────────────────────
EJEMPLO: GUARDAR PREFERENCIAS DEL USUARIO
───────────────────────────────────────────────────────────

html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>LocalStorage - Objetos</title>
</head>
<body>
    <h1>Preferencias del usuario</h1>

    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre"><br>

    <label for="color">Color favorito:</label>
    <input type="color" id="color" value="#2b6cb0"><br>

    <button onclick="guardarPreferencias()">Guardar preferencias</button>
    <button onclick="cargarPreferencias()">Cargar preferencias</button>
    <button onclick="borrarPreferencias()">Borrar preferencias</button>

    <pre id="info"></pre>

    <script>
        function guardarPreferencias() {
            const preferencias = {
                nombre: document.getElementById('nombre').value,
                color: document.getElementById('color').value,
                fecha: new Date().toISOString()
            };

            localStorage.setItem('preferencias_usuario', JSON.stringify(preferencias));
            mostrarInfo('✅ Preferencias guardadas');
        }

        function cargarPreferencias() {
            const texto = localStorage.getItem('preferencias_usuario');

            if (texto === null) {
                mostrarInfo('❌ No hay preferencias guardadas');
                return;
            }

            const preferencias = JSON.parse(texto);

            document.getElementById('nombre').value = preferencias.nombre || '';
            document.getElementById('color').value = preferencias.color || '#2b6cb0';

            mostrarInfo('📋 Preferencias cargadas:\n' + JSON.stringify(preferencias, null, 2));
        }

        function borrarPreferencias() {
            localStorage.removeItem('preferencias_usuario');
            mostrarInfo('🗑️ Preferencias eliminadas');
        }

        function mostrarInfo(mensaje) {
            document.getElementById('info').textContent = mensaje;
        }

        // Cargar preferencias al abrir la página
        window.onload = cargarPreferencias;
    </script>
</body>
</html>

PARTE 8: BUENAS PRÁCTICAS
===============================================================================

───────────────────────────────────────────────────────────

NO GUARDES DATOS SENSIBLES
───────────────────────────────────────────────────────────

❌ Malo:

javascript
localStorage.setItem('password', '123456');
localStorage.setItem('token', 'abc123...');
LocalStorage no es seguro contra ataques XSS.
Nunca guardes contraseñas, tokens de autenticación o información personal sensible.

───────────────────────────────────────────────────────────

USA NOMBRES DE CLAVE DESCRIPTIVOS
───────────────────────────────────────────────────────────

✅ Bueno:

javascript
localStorage.setItem('tema_preferido', 'oscuro');
localStorage.setItem('ultimo_capitulo', '5');
localStorage.setItem('carrito_productos', JSON.stringify(productos));
❌ Malo:

javascript
localStorage.setItem('a', 'oscuro');
localStorage.setItem('b', '5');
localStorage.setItem('c', '[...]');
───────────────────────────────────────────────────────────

VERIFICA LA EXISTENCIA DE DATOS ANTES DE USARLOS
───────────────────────────────────────────────────────────

javascript
const dato = localStorage.getItem('clave');
if (dato !== null) {
    // Usar el dato
} else {
    // Usar valor por defecto
}
───────────────────────────────────────────────────────────

USA JSON PARA DATOS COMPLEJOS
───────────────────────────────────────────────────────────

javascript
// Guardar
localStorage.setItem('usuario', JSON.stringify(objeto));

// Leer
const objeto = JSON.parse(localStorage.getItem('usuario'));
───────────────────────────────────────────────────────────

NO ABUSES DE LOCALSTORAGE
───────────────────────────────────────────────────────────

LocalStorage es síncrono (bloquea la ejecución).

No guardes grandes cantidades de datos.

Para datos grandes (>5 MB), usa IndexedDB.

───────────────────────────────────────────────────────────

USA try-catch PARA MANEJAR ERRORES
───────────────────────────────────────────────────────────

javascript
try {
    localStorage.setItem('clave', 'valor');
} catch (error) {
    console.error('Error al guardar en LocalStorage:', error);
    // El usuario podría tener el almacenamiento lleno o deshabilitado
}

PARTE 9: EJEMPLO PRÁCTICO COMPLETO – PREFERENCIAS DE USUARIO
===============================================================================

html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">

<body>

    <h1>⚙️ Configuración del usuario</h1>

    <!-- Preferencias -->
    <section>
        <h2>Preferencias</h2>

        <div>
            <label for="tema">Tema:</label>
            <select id="tema">
                <option value="claro">☀️ Claro</option>
                <option value="oscuro">🌙 Oscuro</option>
            </select>
        </div>

        <div>
            <label for="tamano">Tamaño de fuente:</label>
            <input type="range" id="tamano" min="12" max="24" value="16">
            <span id="tamano-label">16px</span>
        </div>

        <div>
            <label for="nombre">Nombre de usuario:</label>
            <input type="text" id="nombre" placeholder="Escribe tu nombre">
        </div>

        <button onclick="guardarPreferencias()">💾 Guardar preferencias</button>
        <button onclick="cargarPreferencias()">📂 Cargar preferencias</button>
        <button onclick="resetearPreferencias()">🗑️ Resetear</button>
    </section>

    <!-- Estado -->
    <section>
        <h2>Estado actual</h2>
        <pre id="estado"></pre>
    </section>

    <script>
        // ============================================================ */
        // FUNCIONES PRINCIPALES                                        */
        // ============================================================ */

        function guardarPreferencias() {
            const preferencias = {
                tema: document.getElementById('tema').value,
                tamano: parseInt(document.getElementById('tamano').value),
                nombre: document.getElementById('nombre').value,
                fecha_actualizacion: new Date().toISOString()
            };

            try {
                localStorage.setItem('preferencias_usuario', JSON.stringify(preferencias));
                mostrarEstado('✅ Preferencias guardadas correctamente');
                aplicarPreferencias(preferencias);
            } catch (error) {
                mostrarEstado('❌ Error al guardar: ' + error.message);
            }
        }

        function cargarPreferencias() {
            const texto = localStorage.getItem('preferencias_usuario');

            if (texto === null) {
                mostrarEstado('❌ No hay preferencias guardadas');
                return;
            }

            try {
                const preferencias = JSON.parse(texto);

                document.getElementById('tema').value = preferencias.tema || 'claro';
                document.getElementById('tamano').value = preferencias.tamano || 16;
                document.getElementById('nombre').value = preferencias.nombre || '';

                document.getElementById('tamano-label').textContent = (preferencias.tamano || 16) + 'px';

                aplicarPreferencias(preferencias);
                mostrarEstado('📋 Preferencias cargadas\n' + JSON.stringify(preferencias, null, 2));

            } catch (error) {
                mostrarEstado('❌ Error al cargar: ' + error.message);
            }
        }

        function resetearPreferencias() {
            if (confirm('¿Seguro que quieres eliminar todas las preferencias?')) {
                localStorage.removeItem('preferencias_usuario');
                mostrarEstado('🗑️ Preferencias eliminadas');
                // Restablecer valores por defecto
                document.getElementById('tema').value = 'claro';
                document.getElementById('tamano').value = 16;
                document.getElementById('nombre').value = '';
                document.getElementById('tamano-label').textContent = '16px';
                aplicarPreferencias({ tema: 'claro', tamano: 16, nombre: '' });
            }
        }

        // ============================================================ */
        // APLICAR PREFERENCIAS                                         */
        // ============================================================ */

        function aplicarPreferencias(pref) {
            // Tema
            if (pref.tema === 'oscuro') {
                document.body.style.background = '#1a202c';
                document.body.style.color = '#c9d1d9';
            } else {
                document.body.style.background = '#ffffff';
                document.body.style.color = '#1a202c';
            }

            // Tamaño de fuente
            document.body.style.fontSize = (pref.tamano || 16) + 'px';
        }

        // ============================================================ */
        // MOSTRAR ESTADO                                                */
        // ============================================================ */

        function mostrarEstado(mensaje) {
            const estado = document.getElementById('estado');
            estado.textContent = mensaje;
        }

        // ============================================================ */
        // EVENTOS                                                      */
        // ============================================================ */

        // Mostrar tamaño actual al mover el slider
        document.getElementById('tamano').addEventListener('input', function() {
            document.getElementById('tamano-label').textContent = this.value + 'px';
        });

        // Cargar preferencias al abrir la página
        window.onload = cargarPreferencias;
    </script>

</body>
</html>

PARTE 10: ERRORES COMUNES CON LOCALSTORAGE
===============================================================================

───────────────────────────────────────────────────────────
ERROR 1: OLVIDAR CONVERTIR OBJETOS A JSON
───────────────────────────────────────────────────────────

❌ Incorrecto:

javascript
localStorage.setItem('usuario', { nombre: 'Ana' });
// Guarda: "[object Object]"
✅ Correcto:

javascript
localStorage.setItem('usuario', JSON.stringify({ nombre: 'Ana' }));
───────────────────────────────────────────────────────────
ERROR 2: NO VERIFICAR SI EL DATO EXISTE
───────────────────────────────────────────────────────────

❌ Incorrecto:

javascript
const nombre = localStorage.getItem('nombre');
console.log(nombre.toUpperCase()); // Error si nombre es null
✅ Correcto:

javascript
const nombre = localStorage.getItem('nombre');
if (nombre !== null) {
    console.log(nombre.toUpperCase());
}
───────────────────────────────────────────────────────────
ERROR 3: GUARDAR DATOS SENSIBLES
───────────────────────────────────────────────────────────

❌ Incorrecto:

javascript
localStorage.setItem('password', '123456');
✅ Correcto:

javascript
// Nunca guardes contraseñas, tokens o datos sensibles en LocalStorage.
───────────────────────────────────────────────────────────
ERROR 4: NO MANEJAR EXCEPCIONES
───────────────────────────────────────────────────────────

❌ Incorrecto:

javascript
localStorage.setItem('clave', 'valor');
✅ Correcto:

Desafío del mundo real: Preferencias de usuario en aplicaciones web

En aplicaciones como Twitter, YouTube o Netflix, tus preferencias (tema oscuro, idioma, etc.) se guardan en el servidor, pero también se almacenan localmente para que la aplicación recuerde tu configuración incluso sin conexión.

LocalStorage se usa para guardar preferencias del usuario, estado de la aplicación, y datos que no requieren autenticación.

Pregunta del Profesor Búho

¿Qué crees que pasaría si guardaras la contraseña de un usuario en LocalStorage? ¿Es seguro? ¿Por qué?

Consejo del desarrollador

Nunca guardes información sensible en LocalStorage. No es seguro contra ataques XSS (Cross-Site Scripting). Si un atacante puede ejecutar JavaScript en tu página, puede leer todo lo que hay en LocalStorage.

Para datos sensibles, usa cookies seguras (HttpOnly, Secure) o almacenamiento en el servidor.

GLOSARIO DE TÉRMINOS

LocalStorage: Almacenamiento persistente en el navegador.

setItem(): Método para guardar datos.

getItem(): Método para leer datos.

removeItem(): Método para eliminar un dato.

clear(): Método para eliminar todos los datos.

JSON.stringify(): Convierte un objeto a texto JSON.

JSON.parse(): Convierte texto JSON a objeto.

Persistente: Los datos no se borran al cerrar el navegador.

Síncrono: Las operaciones bloquean la ejecución.

XSS: Cross-Site Scripting (ataque de seguridad).

¿SABÍAS QUE...?
===============================================================================

LocalStorage fue introducido en HTML5 en 2008.
Antes de LocalStorage, los desarrolladores usaban cookies para guardar datos en el cliente, pero las cookies tienen un límite de solo 4 KB.

LocalStorage amplió ese límite a 5-10 MB, permitiendo aplicaciones web más ricas y complejas.
