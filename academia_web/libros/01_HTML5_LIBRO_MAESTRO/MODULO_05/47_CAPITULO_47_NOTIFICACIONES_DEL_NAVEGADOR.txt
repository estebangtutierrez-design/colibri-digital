PARTE 1: ¿QUÉ ES LA API DE NOTIFICACIONES?
===============================================================================

La API de Notificaciones permite que las páginas web envíen notificaciones al sistema operativo del usuario.

───────────────────────────────────────────────────────────
CARACTERÍSTICAS
───────────────────────────────────────────────────────────

1. REQUIERE PERMISO: El usuario debe aceptar recibir notificaciones.
2. VISIBLES FUERA DE LA PÁGINA: Aparecen incluso si la página no está visible.
3. INTERACTIVAS: Pueden tener acciones (botones).
4. PERSONALIZABLES: Título, cuerpo, icono, sonido, etc.
5. COMPATIBLES: Funcionan en navegadores modernos (Chrome, Firefox, Edge, Safari).
6. SOLO HTTPS: Funcionan solo en sitios seguros (HTTPS) o localhost.

───────────────────────────────────────────────────────────
ACCESO A LA API
───────────────────────────────────────────────────────────

La API está disponible a través del objeto `Notification` en JavaScript.

```javascript
// Verificar soporte
if ("Notification" in window) {
    console.log("✅ Notificaciones soportadas");
} else {
    console.log("❌ Notificaciones no soportadas");
}
───────────────────────────────────────────────────────────
ESTADOS DEL PERMISO
───────────────────────────────────────────────────────────

"default" → El usuario no ha sido preguntado aún (se mostrará el diálogo).

"granted" → El usuario ha concedido el permiso.

"denied" → El usuario ha denegado el permiso.

PARTE 2: SOLICITAR PERMISO
===============================================================================

Antes de enviar notificaciones, debes solicitar permiso al usuario.

───────────────────────────────────────────────────────────
SINTAXIS
───────────────────────────────────────────────────────────

javascript
Notification.requestPermission().then(function(permiso) {
    if (permiso === 'granted') {
        console.log('Permiso concedido');
        // Enviar notificación
    } else if (permiso === 'denied') {
        console.log('Permiso denegado');
    }
});
───────────────────────────────────────────────────────────
EJEMPLO COMPLETO
───────────────────────────────────────────────────────────

html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Notificaciones - Solicitar permiso</title>
</head>
<body>
    <h1>🔔 Notificaciones del navegador</h1>

    <button onclick="solicitarPermiso()">🔔 Solicitar permiso</button>
    <button onclick="enviarNotificacion()">📨 Enviar notificación</button>

    <div id="estado"></div>

    <script>
        function actualizarEstado() {
            const estado = document.getElementById('estado');
            if (!("Notification" in window)) {
                estado.textContent = '❌ Notificaciones no soportadas';
                return;
            }

            switch (Notification.permission) {
                case 'granted':
                    estado.textContent = '✅ Permiso concedido';
                    break;
                case 'denied':
                    estado.textContent = '❌ Permiso denegado';
                    break;
                default:
                    estado.textContent = '⏳ Permiso no solicitado aún';
                    break;
            }
        }

        function solicitarPermiso() {
            if (!("Notification" in window)) {
                alert('Notificaciones no soportadas');
                return;
            }

            Notification.requestPermission().then(function(permiso) {
                actualizarEstado();
                if (permiso === 'granted') {
                    alert('✅ Permiso concedido. Ahora puedes recibir notificaciones.');
                } else {
                    alert('❌ Permiso denegado. No podrás recibir notificaciones.');
                }
            });
        }

        function enviarNotificacion() {
            if (Notification.permission !== 'granted') {
                alert('❌ No tienes permiso para enviar notificaciones');
                return;
            }

            const notificacion = new Notification('¡Hola desde tu página web!', {
                body: 'Esta es una notificación de prueba.',
                icon: 'https://via.placeholder.com/64'
            });
        }

        // Mostrar estado al cargar
        window.onload = actualizarEstado;
    </script>
</body>
</html>

PARTE 3: CREAR NOTIFICACIONES
===============================================================================

new Notification(titulo, opciones) crea una nueva notificación.

───────────────────────────────────────────────────────────
SINTAXIS BÁSICA
───────────────────────────────────────────────────────────

javascript
const notificacion = new Notification('Título de la notificación', {
    body: 'Este es el contenido de la notificación.',
    icon: 'https://ejemplo.com/icono.png'
});
───────────────────────────────────────────────────────────
OPCIONES DE NOTIFICACIÓN
───────────────────────────────────────────────────────────

Opción	Descripción	Ejemplo
body	Texto adicional	'Mensaje de prueba'
icon	URL del icono	'https://ejemplo.com/icon.png'
silent	Sin sonido	true o false
tag	Agrupar notificaciones	'mensaje-123'
data	Datos adicionales	{ id: 123 }
actions	Botones de acción	[{action: 'ok', title: 'OK'}]
requireInteraction	Permanece hasta que se interactúa	true o false
───────────────────────────────────────────────────────────
EJEMPLO CON OPCIONES
───────────────────────────────────────────────────────────

javascript
const notificacion = new Notification('Nuevo mensaje de Ana', {
    body: 'Hola, ¿cómo estás?',
    icon: 'https://ejemplo.com/avatar-ana.png',
    tag: 'mensaje-ana-123',
    data: {
        remitente: 'Ana',
        id: 123
    },
    actions: [
        { action: 'responder', title: 'Responder' },
        { action: 'cerrar', title: 'Cerrar' }
    ]
});

// Manejar clic en la notificación
notificacion.onclick = function() {
    window.focus();
    console.log('Notificación clickeada');
};

// Manejar acciones
notificacion.onaction = function(event) {
    console.log('Acción:', event.action);
    if (event.action === 'responder') {
        // Abrir ventana de respuesta
    }
};

PARTE 4: MANEJAR EVENTOS DE NOTIFICACIONES
===============================================================================

Las notificaciones pueden tener eventos para interactuar con ellas.

───────────────────────────────────────────────────────────
onclick – CUANDO EL USUARIO HACE CLIC
───────────────────────────────────────────────────────────

javascript
notificacion.onclick = function() {
    // Enfocar la ventana de la página
    window.focus();
    console.log('El usuario hizo clic en la notificación');
};
───────────────────────────────────────────────────────────
onclose – CUANDO SE CIERRA LA NOTIFICACIÓN
───────────────────────────────────────────────────────────

javascript
notificacion.onclose = function() {
    console.log('La notificación se cerró');
};
───────────────────────────────────────────────────────────
onerror – CUANDO OCURRE UN ERROR
───────────────────────────────────────────────────────────

javascript
notificacion.onerror = function(event) {
    console.error('Error en la notificación:', event);
};
───────────────────────────────────────────────────────────
onaction – CUANDO EL USUARIO HACE CLIC EN UNA ACCIÓN
───────────────────────────────────────────────────────────

javascript
notificacion.onaction = function(event) {
    console.log('Acción seleccionada:', event.action);
    // event.action es el valor de la acción seleccionada
};

PARTE 5: NOTIFICACIONES CON ACCIONES (BOTONES)
===============================================================================

Las acciones permiten que el usuario interactúe con la notificación sin abrir la página.

───────────────────────────────────────────────────────────
SINTAXIS
───────────────────────────────────────────────────────────

javascript
const notificacion = new Notification('Título', {
    body: 'Contenido',
    actions: [
        { action: 'aceptar', title: '✅ Aceptar' },
        { action: 'rechazar', title: '❌ Rechazar' },
        { action: 'ver', title: '👀 Ver' }
    ]
});

notificacion.onaction = function(event) {
    switch (event.action) {
        case 'aceptar':
            console.log('Usuario aceptó');
            break;
        case 'rechazar':
            console.log('Usuario rechazó');
            break;
        case 'ver':
            console.log('Usuario quiere ver');
            window.focus();
            break;
    }
};
───────────────────────────────────────────────────────────
EJEMPLO COMPLETO
───────────────────────────────────────────────────────────

html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Notificaciones - Con acciones</title>
</head>
<body>
    <h1>🔔 Notificaciones con acciones</h1>

    <button onclick="enviarNotificacionAcciones()">📨 Enviar notificación con acciones</button>

    <div id="log"></div>

    <script>
        function enviarNotificacionAcciones() {
            if (Notification.permission !== 'granted') {
                alert('❌ No tienes permiso');
                return;
            }

            const notificacion = new Notification('📋 Nueva tarea asignada', {
                body: 'Tienes una nueva tarea: "Revisar el informe mensual".',
                icon: 'https://via.placeholder.com/64/2b6cb0/white?text=📋',
                tag: 'tarea-123',
                data: { tareaId: 123, proyecto: 'Informes' },
                actions: [
                    { action: 'ver', title: '👀 Ver tarea' },
                    { action: 'completar', title: '✅ Completar' },
                    { action: 'posponer', title: '⏰ Posponer' }
                ]
            });

            notificacion.onclick = function() {
                window.focus();
                agregarLog('📌 Clic en la notificación');
            };

            notificacion.onaction = function(event) {
                switch (event.action) {
                    case 'ver':
                        agregarLog('👀 Acción: Ver tarea');
                        window.focus();
                        break;
                    case 'completar':
                        agregarLog('✅ Acción: Completar tarea');
                        break;
                    case 'posponer':
                        agregarLog('⏰ Acción: Posponer tarea');
                        break;
                }
            };

            notificacion.onclose = function() {
                agregarLog('🚫 Notificación cerrada');
            };
        }

        function agregarLog(mensaje) {
            const log = document.getElementById('log');
            const div = document.createElement('div');
            div.textContent = new Date().toLocaleTimeString() + ' - ' + mensaje;
            log.appendChild(div);
        }

        // Solicitar permiso al cargar
        if ("Notification" in window && Notification.permission === 'default') {
            Notification.requestPermission();
        }
    </script>
</body>
</html>

html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">

    <h1>⏰ Recordatorios programados</h1>

    <div id="estado"></div>

    <script>
        function programarRecordatorio() {
            if (Notification.permission !== 'granted') {
                alert('❌ Necesitas permiso para enviar notificaciones');
                return;
            }

            const segundos = parseInt(document.getElementById('segundos').value) || 5;

            document.getElementById('estado').textContent =
                `⏳ Recordatorio programado en ${segundos} segundos...`;

            setTimeout(function() {
                const notificacion = new Notification('⏰ Recordatorio', {
                    body: `Han pasado ${segundos} segundos desde que programaste el recordatorio.`,
                    icon: 'https://via.placeholder.com/64/f6ad55/white?text=⏰',
                    requireInteraction: true
                });

                notificacion.onclick = function() {
                    window.focus();
                };

                document.getElementById('estado').textContent =
                    `✅ Recordatorio enviado a las ${new Date().toLocaleTimeString()}`;

            }, segundos * 1000);
        }

        // Solicitar permiso al cargar
        if ("Notification" in window && Notification.permission === 'default') {
            Notification.requestPermission();
        }
    </script>
</body>
</html>

// Generar un icono con un emoji en un canvas (opcional)
function generarIcono(emoji, color) {
    const canvas = document.createElement('canvas');
    canvas.width = 64;
    canvas.height = 64;
    const ctx = canvas.getContext('2d');
    ctx.fillStyle = color || '#2b6cb0';
    ctx.fillRect(0, 0, 64, 64);
    ctx.fillStyle = 'white';
    ctx.font = '40px Arial';
    ctx.textAlign = 'center';
    ctx.textBaseline = 'middle';
    ctx.fillText(emoji || '📌', 32, 34);
    return canvas.toDataURL();
}

// Usar en una notificación
const icono = generarIcono('🔔', '#2b6cb0');
const notificacion = new Notification('Título', {
    body: 'Contenido',
    icon: icono
});

PARTE 8: BUENAS PRÁCTICAS CON NOTIFICACIONES
===============================================================================

───────────────────────────────────────────────────────────

SOLICITA PERMISO EN EL MOMENTO ADECUADO
───────────────────────────────────────────────────────────

Solicita permiso cuando el usuario realiza una acción que justifica la notificación.

No solicites permiso al cargar la página (molesta al usuario).

✅ Bueno:

javascript
// Cuando el usuario hace clic en "Activar notificaciones"
button.onclick = function() {
    Notification.requestPermission();
};
❌ Malo:

javascript
// Al cargar la página
window.onload = function() {
    Notification.requestPermission(); // Molesto
};
───────────────────────────────────────────────────────────

NO ENVÍES NOTIFICACIONES SIN PERMISO
───────────────────────────────────────────────────────────

javascript
function enviarNotificacion() {
    if (Notification.permission === 'granted') {
        new Notification('...');
    } else {
        alert('No tienes permiso para enviar notificaciones');
    }
}
───────────────────────────────────────────────────────────

USA TAG PARA AGRUPAR NOTIFICACIONES
───────────────────────────────────────────────────────────

El atributo tag agrupa notificaciones similares.

javascript
// Estas dos notificaciones se agrupan (la segunda reemplaza a la primera)
new Notification('Mensaje', { tag: 'mensaje-123' });
new Notification('Mensaje actualizado', { tag: 'mensaje-123' });
───────────────────────────────────────────────────────────

RESPETA LA PRIVACIDAD DEL USUARIO
───────────────────────────────────────────────────────────

No envíes notificaciones con contenido sensible.

Siempre permite al usuario desactivar las notificaciones.

No envíes demasiadas notificaciones (spam).

───────────────────────────────────────────────────────────

USA HTTPS O LOCALHOST
───────────────────────────────────────────────────────────

Las notificaciones solo funcionan en sitios seguros (HTTPS) o localhost.

    <script>
        // ============================================================ */
        // ESTADO DEL PERMISO                                          */
        // ============================================================ */

        function actualizarEstado() {
            const estado = document.getElementById('estado');

            if (!("Notification" in window)) {
                estado.textContent = '❌ Notificaciones no soportadas';
                estado.className = 'estado denegado';
                return;
            }

            switch (Notification.permission) {
                case 'granted':
                    estado.textContent = '✅ Permiso concedido. Puedes recibir notificaciones.';
                    estado.className = 'estado concedido';
                    break;
                case 'denied':
                    estado.textContent = '❌ Permiso denegado. No recibirás notificaciones.';
                    estado.className = 'estado denegado';
                    break;
                default:
                    estado.textContent = '⏳ Permiso no solicitado aún. Haz clic en "Solicitar permiso".';
                    estado.className = 'estado default';
                    break;
            }
        }

        // ============================================================ */
        // SOLICITAR PERMISO                                            */
        // ============================================================ */

        function solicitarPermiso() {
            if (!("Notification" in window)) {
                alert('Notificaciones no soportadas');
                return;
            }

            if (Notification.permission === 'granted') {
                alert('✅ Ya tienes permiso');
                return;
            }

            if (Notification.permission === 'denied') {
                alert('❌ El permiso fue denegado. Debes habilitarlo desde la configuración del navegador.');
                return;
            }

            Notification.requestPermission().then(function(permiso) {
                actualizarEstado();
                if (permiso === 'granted') {
                    const notificacion = new Notification('✅ Permiso concedido', {
                        body: 'Ahora recibirás avisos de este sitio.',
                        icon: 'https://via.placeholder.com/64/238636/white?text=✅'
                    });
                    setTimeout(() => notificacion.close(), 3000);
                }
            });
        }

        // ============================================================ */
        // ENVIAR AVISO                                                */
        // ============================================================ */

        function enviarAviso(tipo, titulo, mensaje) {
            // Mostrar en la página
            const avisos = document.getElementById('avisos');
            const div = document.createElement('div');
            div.className = 'aviso';
            div.innerHTML = `
                <strong>${titulo}</strong><br>
                ${mensaje}
                <div class="fecha">${new Date().toLocaleString()}</div>
            `;
            avisos.appendChild(div);
            avisos.scrollTop = avisos.scrollHeight;

            // Enviar notificación (si hay permiso)
            if (Notification.permission === 'granted') {
                let icono = 'https://via.placeholder.com/64/2b6cb0/white?text=📌';

                if (tipo === 'alerta') {
                    icono = 'https://via.placeholder.com/64/e53e3e/white?text=⚠️';
                } else if (tipo === 'exito') {
                    icono = 'https://via.placeholder.com/64/238636/white?text=✅';
                }

                const notificacion = new Notification(titulo, {
                    body: mensaje,
                    icon: icono,
                    tag: 'aviso-' + Date.now()
                });

                notificacion.onclick = function() {
                    window.focus();
                };

                // Cerrar después de 10 segundos
                setTimeout(() => {
                    notificacion.close();
                }, 10000);
            }
        }

        // ============================================================ */
        // INICIALIZACIÓN                                               */
        // ============================================================ */

        actualizarEstado();

        // Si el permiso está en default, mostrar un mensaje amigable
        if ("Notification" in window && Notification.permission === 'default') {
            document.getElementById('estado').innerHTML = '💡 Haz clic en "Solicitar permiso" para activar las notificaciones.';
        }
    </script>

</body>

PARTE 11: ERRORES COMUNES CON NOTIFICACIONES
===============================================================================

───────────────────────────────────────────────────────────
ERROR 1: NO VERIFICAR SOPORTE DEL NAVEGADOR
───────────────────────────────────────────────────────────

❌ Incorrecto:

javascript
Notification.requestPermission();
✅ Correcto:

javascript
if ("Notification" in window) {
    Notification.requestPermission();
}
───────────────────────────────────────────────────────────
ERROR 2: ENVIAR NOTIFICACIONES SIN PERMISO
───────────────────────────────────────────────────────────

❌ Incorrecto:

javascript
new Notification('Título', { body: 'Contenido' });
✅ Correcto:

javascript
if (Notification.permission === 'granted') {
    new Notification('Título', { body: 'Contenido' });
}
───────────────────────────────────────────────────────────
ERROR 3: USAR NOTIFICACIONES EN HTTP
───────────────────────────────────────────────────────────

Las notificaciones solo funcionan en HTTPS (o localhost).

Consecuencia: Notification puede no estar disponible o el permiso se deniega automáticamente.

───────────────────────────────────────────────────────────
ERROR 4: SOLICITAR PERMISO AL CARGAR LA PÁGINA
───────────────────────────────────────────────────────────

❌ Incorrecto:

javascript
window.onload = function() {
    Notification.requestPermission(); // Molesto
};
✅ Correcto:

javascript
// Solicitar permiso solo cuando el usuario realiza una acción
button.onclick = function() {
    Notification.requestPermission();
};
───────────────────────────────────────────────────────────
ERROR 5: NO MANEJAR EL CASO DE PERMISO DENEGADO
───────────────────────────────────────────────────────────

javascript
function enviarNotificacion() {
    if (Notification.permission === 'granted') {
        // Enviar notificación
    } else {
        // Informar al usuario que no tiene permiso
        alert('No tienes permiso para recibir notificaciones');
    }
}

Consejo del desarrollador

Las notificaciones son una herramienta poderosa, pero fácilmente abusable. Usa las notificaciones para ayudar al usuario, no para molestarlo. La clave es la relevancia: envía notificaciones solo cuando realmente añadan valor.
