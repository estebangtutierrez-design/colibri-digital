───────────────────────────────────────────────────────────
PARTE 1: PREGUNTAS TEÓRICAS (RESPUESTAS)
───────────────────────────────────────────────────────────

HTML significa HyperText Markup Language. Es el lenguaje de marcado que da estructura a las páginas web.

HTML da la estructura, CSS da el estilo, JavaScript da la interactividad.

Una etiqueta semántica describe el significado del contenido (ej: <header>, <nav>, <main>). Es importante por SEO y accesibilidad.

La accesibilidad web es la práctica de hacer páginas usables por personas con discapacidad. Herramientas: Lighthouse, WAVE, Axe.

SEO es Search Engine Optimization. Elementos HTML que lo afectan: title, meta description, encabezados (h1-h6), alt en imágenes, URLs amigables.

Core Web Vitals son métricas de Google: LCP (carga visual), FID (interactividad), CLS (estabilidad visual).

LocalStorage es permanente (no se borra al cerrar el navegador). SessionStorage es temporal (se borra al cerrar la pestaña).

PWA es Progressive Web App. Tecnologías: manifest.json, Service Worker, HTTPS.

Control de versiones es un sistema que registra cambios en el código. Es importante para colaborar y tener historial.

Validación HTML es el proceso de verificar que el código cumple con los estándares. Herramientas: validador del W3C.

PARTE 5: CORREGIR ERRORES (RESPUESTAS)
───────────────────────────────────────────────────────────

HTML con errores:

Falta lang="es" en <html>

meta charset debe tener comillas: <meta charset="UTF-8">

<tilte> debe ser <title>

<BODY> debe ser <body> (minúsculas)
<h3> debe ser <h2> (jerarquía)
<img> debe tener alt

<div> dentro de <p> no es válido
Formulario con errores:

method debe ser POST (no GET)

Falta label para cada input

Falta required

Falta id para asociar label

button debe tener type="submit"

CSS con errores:

colorr debe ser color

Falta punto y coma después de color

Falta punto y coma después de background
