PARTE 1: ¿QUÉ ES CSS?
===============================================================================

CSS significa Cascading Style Sheets (Hojas de Estilo en Cascada).

───────────────────────────────────────────────────────────
DEFINICIÓN
───────────────────────────────────────────────────────────

CSS es un lenguaje que describe la presentación de un documento HTML.
Controla:
- Colores.
- Fuentes.
- Tamaños.
- Espaciados.
- Posicionamiento.
- Diseño responsive.
- Animaciones.
- Y mucho más.

───────────────────────────────────────────────────────────
SINTAXIS BÁSICA DE CSS
───────────────────────────────────────────────────────────

```css
selector {
    propiedad: valor;
    propiedad: valor;
}
Ejemplo:

css
h1 {
    color: blue;
    font-size: 24px;
}
selector → El elemento al que se aplica el estilo (h1).

propiedad → Lo que quieres cambiar (color, font-size).

valor → El nuevo valor (blue, 24px).

PARTE 2: HTML Y CSS – UNA RELACIÓN INSEPARABLE
===============================================================================

HTML y CSS trabajan juntos.
HTML da la estructura.
CSS da el estilo.

───────────────────────────────────────────────────────────
EJEMPLO: ANTES Y DESPUÉS
───────────────────────────────────────────────────────────

HTML sin CSS (solo estructura):

html
<h1>Título</h1>
<p>Este es un párrafo.</p>
Resultado:

Título en negro, tamaño por defecto.

Párrafo en negro, tamaño por defecto.

Sin colores, sin diseño.

HTML con CSS:

html
<!-- HTML -->
<h1 class="titulo-principal">Título</h1>
<p class="parrafo-destacado">Este es un párrafo.</p>
css
/* CSS */
.titulo-principal {
    color: #2b6cb0;
    font-size: 2.5em;
    text-align: center;
}

.parrafo-destacado {
    color: #c9d1d9;
    font-size: 1.2em;
    background: #161b22;
    padding: 20px;
    border-radius: 8px;
}
Resultado:

Título en azul (#2b6cb0), grande y centrado.

Párrafo en color claro, con fondo oscuro, padding y bordes redondeados.

PARTE 3: TRES FORMAS DE INTEGRAR CSS EN HTML
===============================================================================

Existen tres formas de conectar CSS con HTML.

───────────────────────────────────────────────────────────

CSS EN LÍNEA (INLINE)
───────────────────────────────────────────────────────────

Se escribe directamente en el atributo style de un elemento.

html
<h1 style="color: blue; font-size: 24px;">Título</h1>
Ventajas: Rápido, directo.
Desventajas: Mezcla contenido con estilo, no es reutilizable.

Cuándo usar: Solo en casos muy específicos (ej: estilos únicos).

───────────────────────────────────────────────────────────

CSS INTERNO (INTERNAL)
───────────────────────────────────────────────────────────

Se escribe dentro de la etiqueta <style> en el <head>.

html
<head>
    <style>
        h1 {
            color: blue;
            font-size: 24px;
        }
    </style>
</head>
Ventajas: Separa contenido de estilo (aunque en la misma página).
Desventajas: No es reutilizable entre páginas.

Cuándo usar: Páginas individuales o prototipos.

───────────────────────────────────────────────────────────

CSS EXTERNO (EXTERNAL)
───────────────────────────────────────────────────────────

Se escribe en un archivo .css separado y se enlaza con <link>.

html
<head>
    <link rel="stylesheet" href="style.css">
</head>
css
/* style.css */
h1 {
    color: blue;
    font-size: 24px;
}
Ventajas: Separa completamente contenido de estilo, reutilizable, fácil de mantener.
Desventajas: Requiere un archivo adicional.

Cuándo usar: ¡SIEMPRE! Es la forma profesional.

PARTE 4: CÓMO CSS TRANSFORMA TU PROYECTO HTML
===============================================================================

Veamos cómo tu proyecto HTML se transformará con CSS.

───────────────────────────────────────────────────────────
ANTES (SOLO HTML)
───────────────────────────────────────────────────────────

html
<header>
    <h1>Lucas EGA Academy</h1>
    <nav>
        <a href="index.html">Inicio</a>
        <a href="cursos.html">Cursos</a>
        <a href="contacto.html">Contacto</a>
    </nav>
</header>
Apariencia:

Texto negro sobre fondo blanco.

Enlaces azules subrayados.

Sin diseño visual.

───────────────────────────────────────────────────────────
DESPUÉS (CON CSS)
───────────────────────────────────────────────────────────

html
<!-- HTML (sin cambios en la estructura) -->
<header>
    <h1>Lucas EGA Academy</h1>
    <nav>
        <a href="index.html">Inicio</a>
        <a href="cursos.html">Cursos</a>
        <a href="contacto.html">Contacto</a>
    </nav>
</header>
css
/* CSS */
header {
    background: #1a202c;
    padding: 20px;
    display: flex;
    justify-content: space-between;
    align-items: center;
}

header h1 {
    color: #58a6ff;
    font-size: 1.5em;
}

header nav a {
    color: #c9d1d9;
    text-decoration: none;
    margin-left: 20px;
    font-weight: bold;
}

header nav a:hover {
    color: #58a6ff;
}
Apariencia:

Fondo oscuro (#1a202c).

Título en azul (#58a6ff).

Enlaces en color claro.

Enlaces sin subrayado.

Hover cambia el color a azul.

Distribución horizontal (flex).

La estructura HTML sigue siendo la misma.
Solo cambia la apariencia visual.

PARTE 5: CONCEPTOS BÁSICOS DE CSS
===============================================================================

───────────────────────────────────────────────────────────

SELECTORES
───────────────────────────────────────────────────────────

Los selectores eligen qué elementos se van a estilizar.

css
/* Selector de tipo (elemento) */
h1 { color: blue; }

/* Selector de clase */
.titulo { color: blue; }

/* Selector de ID */
#hero { background: #1a202c; }

/* Selector de descendiente */
header nav a { color: white; }

/* Selector de pseudoclase */
a:hover { color: red; }
───────────────────────────────────────────────────────────

PROPIEDADES COMUNES
───────────────────────────────────────────────────────────

Propiedad	Descripción	Ejemplo
color	Color del texto	color: blue;
background-color	Color de fondo	background-color: #1a202c;
font-size	Tamaño de fuente	font-size: 16px;
font-family	Fuente	font-family: Arial, sans-serif;
margin	Margen exterior	margin: 10px;
padding	Margen interior	padding: 20px;
border	Borde	border: 1px solid black;
width	Ancho	width: 100%;
height	Alto	height: 200px;
display	Tipo de visualización	display: flex;
flex	Distribución flexible	flex: 1;
grid	Distribución en cuadrícula	display: grid;
───────────────────────────────────────────────────────────

EL MODELO DE CAJA (BOX MODEL)
───────────────────────────────────────────────────────────

Todo elemento HTML es una caja.

text
+----------------------------------------------------------+
|  MARGEN (exterior)                                        |
|  +--------------------------------------------------+   |
|  |  BORDE                                            |   |
|  |  +------------------------------------------+   |   |
|  |  |  PADDING (interior)                      |   |   |
|  |  |  +----------------------------------+   |   |   |
|  |  |  |  CONTENIDO                        |   |   |   |
|  |  |  +----------------------------------+   |   |   |
|  |  +------------------------------------------+   |   |
|  +--------------------------------------------------+   |
+----------------------------------------------------------+
Contenido: El texto o elementos dentro.

Padding: Espacio entre el contenido y el borde.

Borde: Línea alrededor del padding.

Margen: Espacio entre el borde y otros elementos.

PARTE 6: EJEMPLO PRÁCTICO – TRANSFORMANDO TU PROYECTO
===============================================================================

Aquí tienes un ejemplo de cómo CSS transformará tu proyecto HTML.

───────────────────────────────────────────────────────────
HTML (SIN CAMBIOS EN LA ESTRUCTURA)
───────────────────────────────────────────────────────────

html
<header>
    <h1>Lucas EGA Academy</h1>
    <nav>
        <ul>
            <li><a href="index.html">Inicio</a></li>
            <li><a href="cursos.html">Cursos</a></li>
            <li><a href="contacto.html">Contacto</a></li>
        </ul>
    </nav>
</header>
───────────────────────────────────────────────────────────
CSS (EL DISEÑO)
───────────────────────────────────────────────────────────

css
/* ===== HEADER ===== */
header {
    background: #161b22;
    padding: 20px 40px;
    display: flex;
    justify-content: space-between;
    align-items: center;
    border-bottom: 1px solid #30363d;
}

/* ===== LOGO ===== */
header h1 {
    color: #58a6ff;
    font-size: 1.5em;
    font-weight: bold;
}

/* ===== NAVEGACIÓN ===== */
header nav ul {
    list-style: none;
    display: flex;
    gap: 20px;
    margin: 0;
    padding: 0;
}

header nav ul li a {
    color: #c9d1d9;
    text-decoration: none;
    font-weight: bold;
    padding: 5px 10px;
    border-radius: 6px;
    transition: background 0.3s, color 0.3s;
}

header nav ul li a:hover {
    color: #58a6ff;
    background: #0d1117;
}

header nav ul li a[aria-current="page"] {
    color: #58a6ff;
    background: #0d1117;
}
───────────────────────────────────────────────────────────
RESULTADO VISUAL
───────────────────────────────────────────────────────────

Fondo oscuro (#161b22) con borde inferior.

Título en azul (#58a6ff) a la izquierda.

Navegación a la derecha con enlaces en color claro.

Efecto hover: fondo oscuro (#0d1117) y texto azul.

Página activa: fondo oscuro y texto azul.

Transiciones suaves (0.3 segundos).

PARTE 10: ERRORES COMUNES AL EMPEZAR CON CSS
===============================================================================

───────────────────────────────────────────────────────────
ERROR 1: NO ENLAZAR EL CSS CORRECTAMENTE
───────────────────────────────────────────────────────────

❌ Incorrecto:

html
<link rel="stylesheet" href="style.css">
<!-- style.css no está en la misma carpeta -->
✅ Correcto:

html
<link rel="stylesheet" href="css/style.css">
───────────────────────────────────────────────────────────
ERROR 2: SELECTORES INCORRECTOS
───────────────────────────────────────────────────────────

❌ Incorrecto:

css
.heroe { color: blue; } /* La clase es "hero" */
✅ Correcto:

css
.hero { color: blue; }
───────────────────────────────────────────────────────────
ERROR 3: PROPIEDADES MAL ESCRITAS
───────────────────────────────────────────────────────────

❌ Incorrecto:

css
colorr: blue;
✅ Correcto:

css
color: blue;
───────────────────────────────────────────────────────────
ERROR 4: PUNTOS Y COMAS FALTANTES
───────────────────────────────────────────────────────────

❌ Incorrecto:

css
color: blue
background: red;
✅ Correcto:

css
color: blue;
background: red;
───────────────────────────────────────────────────────────
ERROR 5: NO USAR CLASES
───────────────────────────────────────────────────────────

❌ Incorrecto:

css
div { color: blue; }
/* Afecta a todos los divs */
✅ Correcto:

css
.destacado { color: blue; }
/* Solo afecta a los elementos con clase "destacado" */

PARTE 11: CÓMO SOLUCIONAR LOS ERRORES
===============================================================================

Verifica la ruta del archivo CSS.

Revisa que los selectores coincidan con las clases e IDs del HTML.

Escribe las propiedades correctamente.

Asegúrate de que cada propiedad termine con punto y coma.

Usa clases en lugar de selectores de tipo para mayor control.

GLOSARIO DE TÉRMINOS
===============================================================================

CSS: Cascading Style Sheets (Hojas de Estilo en Cascada).

Selector: Elemento, clase o ID que se va a estilizar.

Propiedad: Característica a modificar (color, tamaño, etc.).

Valor: Valor asignado a la propiedad.

Modelo de caja: Concepto de que todo elemento es una caja (contenido, padding, borde, margen).

Flexbox: Sistema de distribución flexible.

Grid: Sistema de distribución en cuadrícula.

Responsive: Adaptable a diferentes tamaños de pantalla.
