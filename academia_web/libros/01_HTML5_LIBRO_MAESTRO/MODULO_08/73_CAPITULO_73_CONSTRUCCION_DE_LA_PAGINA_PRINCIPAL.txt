                </svg>
                <span>Lucas EGA Academy</span>
            </a>
        </div>

        <!-- Botón menú móvil -->
        <button class="menu-toggle" aria-label="Abrir menú de navegación" aria-expanded="false">
            <span></span>
            <span></span>
            <span></span>
        </button>

        <!-- Navegación principal -->
        <nav role="navigation" aria-label="Navegación principal" class="nav-principal">
            <ul>
                <li><a href="index.html" aria-current="page">Inicio</a></li>
                <li><a href="cursos.html">Cursos</a></li>
                <li><a href="sobre.html">Sobre nosotros</a></li>
                <li><a href="contacto.html">Contacto</a></li>
            </ul>
        </nav>
    </div>
</header>
───────────────────────────────────────────────────────────
EXPLICACIÓN DEL HEADER
───────────────────────────────────────────────────────────

Logo: SVG con un círculo y la letra "L". Es vectorial y escalable.

Menú móvil: Botón hamburguesa (☰) que se muestra en dispositivos pequeños.

Navegación: Lista de enlaces a las páginas del sitio.

aria-current="page": Indica al lector de pantalla que esta es la página actual.

role="banner": Define el header como un landmark para accesibilidad.

===============================================================================

El hero es la sección principal de bienvenida.

CÓDIGO HTML

html
<!-- ===== SECCIÓN HERO ===== -->
<section id="hero" class="hero" aria-labelledby="hero-titulo">
    <div class="hero-contenedor">
        <div class="hero-texto">
            <h1 id="hero-titulo">Aprende programación con el Profesor Búho AI</h1>
            <p>Una academia inteligente creada para guiarte paso a paso, guardar tu progreso y ayudarte a construir proyectos reales.</p>
            <a href="cursos.html" class="btn-primario" aria-label="Ver cursos disponibles">Ver cursos</a>
        </div>
        <div class="hero-imagen">
            <img src="images/hero/hero-bg.webp" alt="Profesor Búho AI - Tu mentor personal de programación" width="500" height="400" loading="eager" decoding="async">
        </div>
    </div>
</section>
───────────────────────────────────────────────────────────
EXPLICACIÓN DEL HERO
───────────────────────────────────────────────────────────

h1: El título principal de la página (solo uno).

p: Descripción atractiva del servicio.

CTA: Botón "Ver cursos" que lleva a la página de cursos.

Imagen: El Profesor Búho AI con loading="eager" porque es la primera imagen.

decoding="async": La imagen se decodifica en segundo plano para mejorar el rendimiento.

aria-labelledby: Asocia la sección con su título para accesibilidad.

PARTE 7: CONSTRUCCIÓN DE LA SECCIÓN CONTACTO
===============================================================================

La sección de contacto incluye un formulario para que los usuarios se comuniquen.

CÓDIGO HTML

html
<!-- ===== SECCIÓN CONTACTO ===== -->
<section id="contacto" aria-labelledby="contacto-titulo">
    <h2 id="contacto-titulo">Contáctanos</h2>
    <p>¿Tienes preguntas? ¿Quieres colaborar? Escríbenos.</p>

    <form action="/enviar" method="POST" novalidate>

        <fieldset>
            <legend>Datos personales</legend>

            <div>
                <label for="nombre">Nombre completo *</label>
                <input type="text" id="nombre" name="nombre"
                       required minlength="3" maxlength="50"
                       pattern="[A-Za-záéíóúñ ]+"
                       placeholder="Ej: Ana Pérez"
                       autocomplete="name"
                       aria-describedby="nombre-ayuda"
                       aria-required="true">
                <small id="nombre-ayuda">Mínimo 3 caracteres, solo letras.</small>
            </div>

            <div>
                <label for="email">Correo electrónico *</label>
                <input type="email" id="email" name="email"
                       required
                       placeholder="tu@email.com"
                       autocomplete="email"
                       aria-describedby="email-ayuda"
                       aria-required="true">
                <small id="email-ayuda">Introduce un correo válido.</small>
            </div>

            <div>
                <label for="telefono">Teléfono</label>
                <input type="tel" id="telefono" name="telefono"
                       pattern="[0-9]{10}"
                       placeholder="55 1234 5678"
                       aria-describedby="telefono-ayuda">
                <small id="telefono-ayuda">10 dígitos, sin espacios.</small>
            </div>
        </fieldset>

        <fieldset>
            <legend>Tu mensaje</legend>

            <div>
                <label for="asunto">Asunto *</label>
                <select id="asunto" name="asunto" required aria-required="true">
                    <option value="">Selecciona un asunto</option>
                    <option value="consulta">Consulta general</option>
                    <option value="presupuesto">Solicitud de presupuesto</option>
                    <option value="soporte">Soporte técnico</option>
                    <option value="colaboracion">Propuesta de colaboración</option>
                </select>
            </div>

            <div>
                <label for="mensaje">Mensaje *</label>
                <textarea id="mensaje" name="mensaje" rows="5"
                          required minlength="10"
                          placeholder="Escribe tu mensaje aquí..."
                          aria-describedby="mensaje-ayuda"
                          aria-required="true"></textarea>
                <small id="mensaje-ayuda">Mínimo 10 caracteres.</small>
            </div>
        </fieldset>

        <div>
            <label>
                <input type="checkbox" name="terminos" required aria-required="true">
                Acepto los <a href="#" target="_blank" rel="noopener noreferrer">términos y condiciones</a> *
            </label>
        </div>

        <button type="submit" aria-label="Enviar mensaje">📨 Enviar mensaje</button>
    </form>
</section>
───────────────────────────────────────────────────────────
EXPLICACIÓN DEL FORMULARIO
───────────────────────────────────────────────────────────

fieldset y legend: Agrupan campos relacionados.

label y for: Asocia la etiqueta con el campo.

required: Campos obligatorios.

pattern: Valida el formato (nombre, teléfono).

aria-describedby: Proporciona ayuda adicional para lectores de pantalla.

aria-required: Indica que el campo es obligatorio.

novalidate: Desactiva la validación nativa (para manejar con JavaScript).

rel="noopener noreferrer": Seguridad en enlaces externos.

───────────────────────────────────────────────────────────
ERROR 1: MÚLTIPLES <h1> EN LA PÁGINA
───────────────────────────────────────────────────────────

❌ Incorrecto:

html
<h1>Título principal</h1>
<section>
    <h1>Otro título</h1>
</section>
✅ Correcto:

html
<h1>Título principal</h1>
<section>
    <h2>Subtítulo</h2>
</section>
───────────────────────────────────────────────────────────
ERROR 2: IMÁGENES SIN alt
───────────────────────────────────────────────────────────

❌ Incorrecto:

html
<img src="hero.jpg">
✅ Correcto:

html

❌ Incorrecto:

html
<label for="nombre">Nombre:</label>
<input type="text" id="nombre" placeholder="Nombre">

html

html
