===============================================================================
PARTE 1: ¿QUÉ ES UN WIREFRAME?
===============================================================================

Un wireframe es un esquema visual de una página web que muestra la ubicación de los elementos.

───────────────────────────────────────────────────────────
CARACTERÍSTICAS DE UN WIREFRAME
───────────────────────────────────────────────────────────

1. **BLANCO Y NEGRO:** Sin colores, sin tipografías, sin imágenes.
2. **ESTRUCTURAL:** Muestra la disposición de los elementos.
3. **FUNCIONAL:** Muestra la jerarquía y la navegación.
4. **SIMPLIFICADO:** Solo lo esencial, sin distracciones.

───────────────────────────────────────────────────────────
TIPOS DE WIREFRAMES
───────────────────────────────────────────────────────────

| Tipo | Descripción | Cuándo usar |
|------|-------------|-------------|
| **Baja fidelidad** | Dibujos a mano alzada, muy simples | Para ideas iniciales, lluvia de ideas |
| **Media fidelidad** | Esquemas más detallados, con estructura clara | Para definir la estructura |
| **Alta fidelidad** | Muy detallados, casi como la página real | Para presentar a clientes |

En este capítulo usaremos wireframes de **media fidelidad**.

───────────────────────────────────────────────────────────
HERRAMIENTAS PARA WIREFRAMES
───────────────────────────────────────────────────────────

**Digitales:**
- Figma (gratuito, profesional)
- Balsamiq (simple, para wireframes rápidos)
- Adobe XD (gratuito)
- Miro (pizarra colaborativa)

**Manuales:**
- Papel y lápiz (siempre funciona)

===============================================================================
PARTE 2: ELEMENTOS DE UN WIREFRAME
===============================================================================

Un wireframe muestra la ubicación de los elementos principales.

───────────────────────────────────────────────────────────
ELEMENTOS QUE DEBE INCLUIR UN WIREFRAME
───────────────────────────────────────────────────────────

1. **HEADER:** Logo, navegación, título.
2. **HERO:** Título principal, subtítulo, llamado a la acción.
3. **SECCIONES:** Bloques de contenido.
4. **IMÁGENES:** Espacios reservados para imágenes.
5. **TEXTOS:** Jerarquía de títulos y párrafos.
6. **BOTONES:** Ubicación de los llamados a la acción.
7. **FOOTER:** Información de contacto, enlaces legales.

───────────────────────────────────────────────────────────
SÍMBOLOS COMUNES EN WIREFRAMES
───────────────────────────────────────────────────────────

| Símbolo | Significado |
|---------|-------------|
| [ ] o ▢ | Espacio para imagen o video |
| X | Imagen (cuando no se usa placeholder) |
| ===== | Línea horizontal (separador) |
| T | Texto (título o párrafo) |
| → | Enlace o botón |
| ☰ | Menú hamburguesa |

GLOSARIO DE TÉRMINOS
===============================================================================

- Wireframe: Esquema visual de una página web.
- Baja fidelidad: Wireframe simple, dibujado a mano.
- Media fidelidad: Wireframe más detallado con estructura clara.
- Alta fidelidad: Wireframe muy detallado, casi como la página real.
- Navegación: Sistema de enlaces que permite moverse por el sitio.
- Estructura de carpetas: Organización de archivos en carpetas.
