PARTE 2: REQUISITOS DEL PROYECTO
===============================================================================

Tu sitio debe cumplir con TODOS los siguientes requisitos:

───────────────────────────────────────────────────────────
ORGANIZACIÓN (Capítulo 61)
───────────────────────────────────────────────────────────

✅ Estructura de carpetas profesional.
✅ Rutas relativas para todos los recursos internos.
✅ Indentación consistente.
✅ Etiquetas siempre cerradas.
✅ Atributos con comillas dobles.
✅ Nombres de archivos en minúscula con guiones.

───────────────────────────────────────────────────────────
CONVENCIONES (Capítulo 62)
───────────────────────────────────────────────────────────

✅ Archivos en minúscula con guiones.
✅ Clases e IDs en minúscula con guiones.
✅ Variables en camelCase.
✅ Comentarios útiles.
✅ Cabeceras en todos los archivos.

───────────────────────────────────────────────────────────
CONTROL DE VERSIONES (Capítulo 63)
───────────────────────────────────────────────────────────

✅ Repositorio Git.
✅ Al menos 5 commits con mensajes descriptivos.
✅ Rama `main` con código estable.
✅ README en el repositorio.

───────────────────────────────────────────────────────────
PREPARACIÓN PARA PUBLICACIÓN (Capítulo 64)
───────────────────────────────────────────────────────────

✅ HTML validado (sin errores).
✅ CSS validado (sin errores).
✅ Todos los enlaces funcionando.
✅ Todas las imágenes optimizadas.

───────────────────────────────────────────────────────────
HOSTING Y DOMINIOS (Capítulo 65)
───────────────────────────────────────────────────────────

✅ Sitio publicado en Netlify o GitHub Pages.
✅ URL pública funcionando.
✅ HTTPS activado.

───────────────────────────────────────────────────────────
OPTIMIZACIÓN FINAL (Capítulo 66)
───────────────────────────────────────────────────────────

✅ Imágenes en WebP.
✅ Imágenes comprimidas.
✅ `loading="lazy"` en imágenes fuera de pantalla.
✅ `width` y `height` en imágenes.
✅ HTML minimizado.
✅ CSS minimizado.
✅ JavaScript minimizado.
✅ Gzip/Brotli activado (en el hosting).

───────────────────────────────────────────────────────────
DOCUMENTACIÓN (Capítulo 67)
───────────────────────────────────────────────────────────

✅ README.md (descripción, estructura, instalación, uso).
✅ Manual de usuario.
✅ Manual técnico.

MANTENIMIENTO (Capítulo 68)

    // MENÚ MÓVIL (TOGGLE)                                           */
    // ============================================================ */
    const menuToggle = document.querySelector('.menu-toggle');
    const navPrincipal = document.querySelector('.nav-principal');

    if (menuToggle && navPrincipal) {
        menuToggle.addEventListener('click', function() {
            navPrincipal.classList.toggle('abierto');
            menuToggle.classList.toggle('activo');
            const isOpen = navPrincipal.classList.contains('abierto');
            menuToggle.setAttribute('aria-expanded', isOpen);
        });
    }

    // ============================================================ */
    // BOTÓN "VOLVER ARRIBA"                                       */
    // ============================================================ */
    const btnVolverArriba = document.getElementById('btn-volver-arriba');

    if (btnVolverArriba) {
        window.addEventListener('scroll', function() {
            if (window.scrollY > 300) {
                btnVolverArriba.classList.add('visible');
            } else {
                btnVolverArriba.classList.remove('visible');
            }
        });
        btnVolverArriba.addEventListener('click', function() {
            window.scrollTo({ top: 0, behavior: 'smooth' });
        });
    }
