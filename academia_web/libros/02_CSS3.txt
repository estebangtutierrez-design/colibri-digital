CAPITULO 1: INTRODUCCIÓN A CSS3
================================================================================

¿Qué es CSS?
------------
CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo que controla la
apariencia visual de las páginas web. Mientras que HTML estructura el contenido,
CSS lo hace bonito y atractivo.

CSS3 es la versión más moderna y trae características revolucionarias como:
- Animaciones y transiciones sin JavaScript
- Flexbox y Grid para layouts modernos y flexibles
- Gradientes, sombras y bordes redondeados
- Variables CSS (Custom Properties)
- Media queries para diseño responsive
- Filtros y efectos visuales
- Transformaciones 2D y 3D

¿Cómo se aplica CSS?
--------------------
Hay tres formas de aplicar CSS a un documento HTML:

1. CSS Inline (en la etiqueta HTML):
   <p style="color: red;">Texto rojo</p>
   (No recomendado para proyectos grandes)

2. CSS Interno (en el <head> del HTML):
   <style>
       p { color: red; }
   </style>

3. CSS Externo (archivo .css separado) - RECOMENDADO:
   <link rel="stylesheet" href="estilos.css">

CAPITULO 2: SINTAXIS Y SELECTORES BÁSICOS
================================================================================

Sintaxis básica:
----------------
selector {
    propiedad: valor;
}

Ejemplo:
p {
    color: blue;
    font-size: 16px;
}

Selectores básicos:
-------------------
p { }              -> Todos los párrafos
.nombre-clase { }  -> Elementos con class="nombre-clase"
#mi-id { }         -> Elemento con id="mi-id" (único)
* { }              -> Todos los elementos (universal)

Selectores combinados:
----------------------
nav a { }          -> Todos los <a> dentro de <nav>
ul > li { }        -> Solo <li> hijos directos de <ul>
h2 + p { }         -> Primer <p> después de <h2>
h2 ~ p { }         -> Todos los <p> después de <h2>

Selectores de atributo:
-----------------------
input[type="text"] { }    -> Inputs de tipo text
a[href^="https"] { }      -> Enlaces que empiezan con https
a[href$=".pdf"] { }       -> Enlaces que terminan en .pdf
input[type="checkbox"]:checked { } -> Checkbox marcados

Pseudo-clases (estados del elemento):
--------------
:hover      -> Al pasar el mouse
:focus      -> Al hacer clic o tab
:active     -> Al hacer clic (presionado)
:visited    -> Enlace ya visitado
:first-child    -> Primer hijo
:last-child     -> Último hijo
:nth-child(n)   -> El enésimo hijo (ej: 2n+1 para impares)
:not(.clase)    -> Elementos que NO tienen esa clase
:empty          -> Elementos sin hijos

Pseudo-elementos (partes del elemento):
-----------------
::before    -> Inserta contenido antes del elemento
::after     -> Inserta contenido después del elemento
::first-letter  -> Primera letra
::first-line    -> Primera línea
::selection     -> Texto seleccionado

Ejemplo con pseudo-elementos:
-----------------------------
p::first-letter {
    font-size: 2em;
    color: red;
}

p::before {
    content: "► ";
    color: blue;
}

CAPITULO 3: COLORES, FONDOS Y BORDES
================================================================================

Colores:
--------
color: red;                    /* Por nombre */
color: #ff0000;                /* Hexadecimal */
color: #f00;                   /* Hex abreviado */
color: rgb(255, 0, 0);         /* RGB */
color: rgba(255, 0, 0, 0.5);   /* RGBA con transparencia */
color: hsl(120, 100%, 50%);    /* HSL */
color: hsla(120, 100%, 50%, 0.5); /* HSLA */

Fondos (background):
-------
background-color: #f0f0f0;
background-image: url('fondo.jpg');
background-size: cover;           /* Cubre todo el área */
background-size: contain;         /* Sin cortar la imagen */
background-position: center;
background-repeat: no-repeat;
background-attachment: fixed;     /* Fondo fijo al hacer scroll */

/* Atajo (shorthand) */
background: #f0f0f0 url('fondo.jpg') no-repeat center/cover;

Gradientes:
-----------
background: linear-gradient(to right, red, blue);
background: linear-gradient(45deg, #ff6b6b, #4ecdc4);
background: linear-gradient(to bottom, #ff6b6b 0%, #4ecdc4 100%);
background: radial-gradient(circle, yellow, orange);
background: conic-gradient(red, yellow, green, blue, red);

Bordes:
-------
border: 2px solid #333;
border-width: 2px;
border-style: solid;               /* solid, dashed, dotted, double, none */
border-color: #333;

/* Bordes individuales */
border-top: 2px solid red;
border-right: 2px solid green;
border-bottom: 2px solid blue;
border-left: 2px solid yellow;

border-radius: 10px;               /* Bordes redondeados */
border-radius: 50%;                /* Círculo perfecto */
border-radius: 10px 20px 30px 40px; /* Esquinas individuales */

Sombras (box-shadow):
-------------------
box-shadow: 5px 5px 10px rgba(0,0,0,0.3);
/*           x    y    blur   color */
box-shadow: 0 4px 6px rgba(0,0,0,0.1);
box-shadow: inset 0 0 10px rgba(0,0,0,0.5);  /* Sombra interna */
box-shadow: 0 10px 20px rgba(0,0,0,0.2), 0 6px 6px rgba(0,0,0,0.1); /* Múltiples */

CAPITULO 4: MODELO DE CAJA (BOX MODEL)
================================================================================

Todo elemento HTML es una caja con 4 capas:

    +---------------------------+
    |         MARGIN            |  <- Espacio exterior (transparente)
    |   +-------------------+   |
    |   |      BORDER       |   |  <- Borde visible
    |   |   +-----------+   |   |
    |   |   |  PADDING  |   |   |  <- Espacio interior (relleno)
    |   |   |  CONTENT  |   |   |  <- Contenido real (texto, imágenes)
    |   |   +-----------+   |   |
    |   +-------------------+   |
    +---------------------------+

Propiedades:
------------
width: 300px;           /* Ancho del contenido */
height: 200px;          /* Alto del contenido */
padding: 20px;          /* Espacio interior (todos los lados) */
padding: 10px 20px;     /* arriba/abajo, izquierda/derecha */
padding: 10px 20px 30px 40px; /* arriba, derecha, abajo, izquierda */
padding-top: 10px;
padding-right: 20px;
padding-bottom: 30px;
padding-left: 40px;

margin: 20px;           /* Espacio exterior */
margin: 0 auto;         /* Centrar horizontalmente */
margin-top: 10px;
margin-right: 20px;
margin-bottom: 30px;
margin-left: 40px;

border: 2px solid #333;

box-sizing (Muy importante):
-----------
box-sizing: content-box;   /* Default: width solo contenido */
box-sizing: border-box;    /* width incluye padding + border */

/* SIEMPRE usa border-box para no volverte loco */
*, *::before, *::after {
    box-sizing: border-box;
}

CAPITULO 6: POSICIONAMIENTO
================================================================================

Display (tipo de caja):
-------
display: block;         /* Ocupa todo el ancho disponible */
display: inline;        /* Solo ocupa su contenido */
display: inline-block;  /* Inline pero acepta width/height */
display: none;          /* Oculta el elemento (desaparece del flujo) */
display: flex;          /* Flexbox */
display: grid;          /* CSS Grid */
display: table;         /* Comportamiento de tabla */

Visibility (visibilidad):
----------
visibility: visible | hidden; /* Oculta pero ocupa espacio */

Position:
--------
position: static;       /* Default, flujo normal */
position: relative;     /* Relativo a su posición normal */
position: absolute;     /* Relativo al ancestro posicionado más cercano */
position: fixed;        /* Relativo a la ventana (no se mueve al scroll) */
position: sticky;       /* Se pega al hacer scroll */

Ejemplos:
---------
/* Relative: se desplaza desde su posición original */
.caja {
    position: relative;
    top: 20px;
    left: 10px;
}

/* Absolute: posicionado dentro de .contenedor */
.contenedor { 
    position: relative; 
}
.hijo {
    position: absolute;
    top: 0;
    right: 0;
}

/* Fixed: barra fija en la parte superior */
.navbar {
    position: fixed;
    top: 0;
    left: 0;
    width: 100%;
    z-index: 1000;
}

/* Sticky: se pega al llegar al borde */
.header {
    position: sticky;
    top: 0;
}

z-index (orden de apilamiento):
-------
.capa1 { z-index: 1; }
.capa2 { z-index: 2; }  /* Esta encima de capa1 */
.capa3 { z-index: 999; } /* Encima de todos */

CAPITULO 7: FLEXBOX
================================================================================

Flexbox es un sistema unidimensional (una fila o columna).
Es ideal para: navegación, centrar elementos, distribuir espacio.

Contenedor flex:
----------------
.contenedor {
    display: flex;
}

Dirección (flex-direction):
---------
flex-direction: row;            /* Default: horizontal izquierda->derecha */
flex-direction: row-reverse;    /* Horizontal derecha->izquierda */
flex-direction: column;         /* Vertical arriba->abajo */
flex-direction: column-reverse; /* Vertical abajo->arriba */

Alineación horizontal (justify-content):
------------------------
justify-content: flex-start;    /* Alineados al inicio (default) */
justify-content: flex-end;      /* Alineados al final */
justify-content: center;        /* Centrados horizontalmente */
justify-content: space-between; /* Espacio entre elementos */
justify-content: space-around;  /* Espacio alrededor */
justify-content: space-evenly;  /* Espacio igual */

Alineación vertical (align-items):
--------------------
align-items: stretch;          /* Estiran al alto del contenedor (default) */
align-items: flex-start;       /* Alineados arriba */
align-items: flex-end;         /* Alineados abajo */
align-items: center;           /* Centrados verticalmente */
align-items: baseline;         /* Alineados por la línea base */

Wrap (salto de línea):
-----
flex-wrap: nowrap;             /* No saltan (default) */
flex-wrap: wrap;               /* Saltan a nueva línea */
flex-wrap: wrap-reverse;       /* Saltan hacia arriba */

Propiedades de los items (hijos):
--------------------------------
.item {
    flex-grow: 1;               /* Cuánto crece (proporción) */
    flex-shrink: 0;             /* No se encoge (0 = no) */
    flex-basis: 200px;          /* Tamaño base */
    flex: 1;                    /* Atajo: grow shrink basis */
    align-self: center;         /* Alineación individual */
    order: 2;                   /* Orden visual */
}

Ejemplo - Centrado perfecto (muy común):
----------------------------------------
.centrar {
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100vh;  /* 100% del alto de la ventana */
}

Ejemplo - Navegación horizontal:
--------------------------------
.nav {
    display: flex;
    justify-content: space-between;
    align-items: center;
    padding: 20px;
}
.nav-links {
    display: flex;
    gap: 20px;  /* Espacio entre enlaces */
}

CAPITULO 8: CSS GRID
================================================================================

CSS Grid es un sistema bidimensional (filas Y columnas).
Ideal para: layouts completos de página, galerías, paneles.

Contenedor grid:
----------------
.contenedor {
    display: grid;
}

Columnas (grid-template-columns):
---------
grid-template-columns: 200px 200px 200px;  /* 3 columnas de 200px */
grid-template-columns: 1fr 1fr 1fr;         /* 3 columnas iguales */
grid-template-columns: repeat(3, 1fr);      /* 3 columnas iguales */
grid-template-columns: 2fr 1fr 1fr;         /* Primera el doble */
grid-template-columns: minmax(200px, 1fr) 1fr; /* Mínimo 200px */

Filas (grid-template-rows):
-----
grid-template-rows: 100px 200px;           /* 2 filas */
grid-template-rows: repeat(4, 1fr);        /* 4 filas iguales */

Espacio entre celdas (gap):
-----------
gap: 20px;                    /* Mismo para filas y columnas */
gap: 10px 20px;              /* fila columna */

Áreas de grid (grid-template-areas):
---------
.contenedor {
    display: grid;
    grid-template-areas:
        "header header header"
        "sidebar main main"
        "footer footer footer";
    grid-template-columns: 200px 1fr 1fr;
    grid-template-rows: 80px 1fr 60px;
    gap: 20px;
    height: 100vh;
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.footer { grid-area: footer; }

Colocar items individualmente:
------------------------------
.item {
    grid-column: 1 / 3;        /* Ocupa columna 1 a 3 (excluyendo 3) */
    grid-row: 1 / 2;           /* Ocupa fila 1 a 2 */
    grid-area: 1 / 1 / 2 / 3;  /* fila-inicio / col-inicio / fila-fin / col-fin */
}

Auto-fill y auto-fit (responsive automático):
-----------
grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));
/* Crea tantas columnas como quepan, mínimo 250px */

grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
/* Similar pero colapsa columnas vacías */

Ejemplo de galería responsive:
------------------------------
.galeria {
    display: grid;
    grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
    gap: 20px;
}

CAPITULO 9: ANIMACIONES Y TRANSICIONES
================================================================================

Transiciones (cambios suaves entre estados):
-----------
.boton {
    background: blue;
    transition: background 0.3s ease, transform 0.2s;
}
.boton:hover {
    background: red;
    transform: scale(1.1);
}

Propiedades de transición:
--------------------------
transition-property: all;              /* Qué propiedad animar */
transition-duration: 0.3s;             /* Duración */
transition-timing-function: ease;      /* linear, ease-in, ease-out, ease-in-out */
transition-delay: 0.5s;                /* Retraso antes de empezar */

/* Atajo */
transition: all 0.3s ease 0.2s;

Animaciones con @keyframes (más complejas):
-----------------------------
@keyframes rebote {
    0% { transform: translateY(0); }
    50% { transform: translateY(-30px); }
    100% { transform: translateY(0); }
}

.pelota {
    animation: rebote 1s ease infinite;
}

Propiedades de animación:
-------------------------
animation-name: rebote;
animation-duration: 1s;
animation-timing-function: ease;
animation-delay: 0s;
animation-iteration-count: infinite;     /* Número de repeticiones */
animation-direction: alternate;          /* normal, reverse, alternate */
animation-fill-mode: forwards;           /* Mantiene el estado final */
animation-play-state: running;           /* paused | running */

/* Atajo */
animation: rebote 1s ease infinite alternate;

Keyframes comunes:
------------------
@keyframes fadeIn {
    from { opacity: 0; }
    to { opacity: 1; }
}

@keyframes slideIn {
    from { transform: translateX(-100%); opacity: 0; }
    to { transform: translateX(0); opacity: 1; }
}

@keyframes pulse {
    0% { transform: scale(1); }
    50% { transform: scale(1.1); }
    100% { transform: scale(1); }
}

@keyframes spin {
    from { transform: rotate(0deg); }
    to { transform: rotate(360deg); }
}

CAPITULO 10: MEDIA QUERIES Y DISEÑO RESPONSIVE
================================================================================

Media queries permiten aplicar estilos según las características del dispositivo
(tamaño de pantalla, orientación, etc.).

Sintaxis:
---------
@media (condicion) {
    /* Estilos que se aplican solo si se cumple la condición */
}

Breakpoints comunes (puntos de quiebre):
-------------------
@media (max-width: 480px) { }      /* Móvil pequeño (< 480px) */
@media (max-width: 768px) { }      /* Tablets y móviles grandes (< 768px) */
@media (min-width: 769px) and (max-width: 1024px) { }  /* Tablets */
@media (min-width: 1025px) { }     /* Desktop */
@media (min-width: 1440px) { }     /* Pantallas grandes */

Ejemplo práctico:
-----------------
.grid {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 20px;
}

@media (max-width: 1024px) {
    .grid { 
        grid-template-columns: repeat(2, 1fr); 
    }
}

@media (max-width: 768px) {
    .grid {
        grid-template-columns: 1fr;
        gap: 10px;
    }
    .header h1 { 
        font-size: 1.5rem; 
    }
}

@media (max-width: 480px) {
    .container {
        padding: 0 10px;
    }
    .btn {
        width: 100%;
    }
}

Otras condiciones (orientation, aspect-ratio, etc.):
---------------------------------------------------
@media (orientation: landscape) { /* Pantalla horizontal */ }
@media (orientation: portrait) { /* Pantalla vertical */ }
@media (prefers-color-scheme: dark) { /* Modo oscuro del sistema */ }
@media (prefers-color-scheme: light) { /* Modo claro */ }

Unidades relativas (importantes para responsive):
-------------------
vw  -> 1% del ancho de la ventana
vh  -> 1% del alto de la ventana
rem -> Relativo al tamaño de fuente del <html>
em  -> Relativo al tamaño de fuente del padre
%   -> Porcentaje del padre

Breakpoints basados en contenido (no en dispositivos):
-----------------------------------------------------
Haz que el diseño se rompa cuando el contenido lo necesite, no en tamaños fijos.

CAPITULO 11: VARIABLES CSS (CUSTOM PROPERTIES)
================================================================================

Las variables CSS (o Custom Properties) permiten reutilizar valores y cambiar
temas fácilmente.

Definir variables (--nombre):
-----------------------------
:root {  /* :root = html, ámbito global */
    --color-primario: #6366f1;
    --color-secundario: #ff6b6b;
    --color-fondo: #f8fafc;
    --color-texto: #1e293b;
    --fuente-principal: 'Roboto', sans-serif;
    --radio-borde: 10px;
    --sombra: 0 4px 6px rgba(0,0,0,0.1);
    --espaciado: 20px;
}

Usar variables (var()):
-----------------------
.boton {
    background: var(--color-primario);
    color: white;
    border-radius: var(--radio-borde);
    font-family: var(--fuente-principal);
    box-shadow: var(--sombra);
    padding: var(--espaciado);
}

.boton:hover {
    background: var(--color-secundario);
}

Modo oscuro con variables:
--------------------------
@media (prefers-color-scheme: dark) {
    :root {
        --color-fondo: #1e293b;
        --color-texto: #f8fafc;
        --sombra: 0 4px 6px rgba(0,0,0,0.5);
    }
}

Cambiar variables desde JavaScript:
-----------------------------------
document.documentElement.style.setProperty('--color-primario', '#ff0000');

CAPITULO 12: TRANSFORMACIONES 2D Y 3D
================================================================================

Transformaciones 2D:
--------------------
transform: translateX(50px);      /* Mover horizontal */
transform: translateY(30px);      /* Mover vertical */
transform: translate(50px, 30px); /* Mover ambos */
transform: scale(1.5);            /* Escalar (1.5 = 150%) */
transform: scaleX(2);             /* Escalar horizontal */
transform: scaleY(0.5);           /* Escalar vertical */
transform: rotate(45deg);         /* Rotar (grados) */
transform: rotate(0.5turn);       /* Rotar en vueltas */
transform: skewX(10deg);          /* Sesgar horizontal */
transform: skewY(10deg);          /* Sesgar vertical */
transform: translateX(50px) rotate(45deg) scale(1.2); /* Combinar */

Transformaciones 3D (requieren perspectiva):
---------------
.contenedor-3d {
    perspective: 1000px;          /* Define la profundidad */
}

.tarjeta-3d {
    transform-style: preserve-3d; /* Mantiene los hijos en 3D */
    transition: transform 0.6s;
}

.tarjeta-3d:hover {
    transform: rotateY(180deg);   /* Gira en Y */
}

/* Otras propiedades 3D */
transform: rotateX(45deg);        /* Gira en X */
transform: rotateY(45deg);        /* Gira en Y */
transform: rotateZ(45deg);        /* Gira en Z */
transform: translateZ(50px);      /* Mover en Z (profundidad) */
transform: rotate3d(1, 1, 0, 45deg); /* Eje personalizado */

backface-visibility: hidden;      /* Oculta la cara trasera */

CAPITULO 13: FILTROS Y EFECTOS VISUALES
================================================================================

Filtros CSS (se aplican a imágenes y elementos):
-----------
.imagen {
    filter: grayscale(100%);        /* Escala de grises */
    filter: sepia(100%);            /* Efecto sepia */
    filter: blur(5px);              /* Desenfoque */
    filter: brightness(150%);       /* Brillo */
    filter: contrast(200%);         /* Contraste */
    filter: hue-rotate(90deg);      /* Rotación de tono */
    filter: invert(100%);           /* Invertir colores */
    filter: saturate(200%);         /* Saturación */
    filter: drop-shadow(0 4px 8px rgba(0,0,0,0.5)); /* Sombra */
    filter: grayscale(50%) blur(2px) brightness(120%); /* Combinar */
}

.imagen:hover {
    filter: none;                   /* Quitar filtros */
}

Backdrop filter (efecto vidrio/glassmorphism):
----------------------------------------------
.glass {
    background: rgba(255,255,255,0.1);
    backdrop-filter: blur(10px);
    -webkit-backdrop-filter: blur(10px); /* Safari */
    border: 1px solid rgba(255,255,255,0.2);
    border-radius: 10px;
}

Clip-path (recortar elementos):
-----------
.circulo { 
    clip-path: circle(50%); 
}
.triangulo { 
    clip-path: polygon(50% 0%, 0% 100%, 100% 100%); 
}
.hexagono {
    clip-path: polygon(25% 0%, 75% 0%, 100% 50%, 75% 100%, 25% 100%, 0% 50%);
}

Mix-blend-mode (mezcla de colores):
----------------
.elemento {
    mix-blend-mode: multiply;   /* multiply, screen, overlay, darken, lighten */
}

CAPITULO 14: BUENAS PRÁCTICAS Y ORGANIZACIÓN
================================================================================

1. Nomenclatura BEM (Block__Element--Modifier):
   .bloque { }
   .bloque__elemento { }
   .bloque--modificador { }

   Ejemplo:
   .card { }
   .card__title { }
   .card__button { }
   .card--highlight { }

2. Organización de archivos CSS (estructura recomendada):
   /css/
       base.css          /* Reset y variables */
       layout.css        /* Grid, flexbox, contenedores */
       components.css    /* Botones, cards, formularios */
       utilities.css     /* Clases utilitarias (margen, padding, etc.) */
       responsive.css    /* Media queries */

3. Reset CSS (normalizar estilos por defecto):
   *, *::before, *::after {
       margin: 0;
       padding: 0;
       box-sizing: border-box;
   }
   body {
       line-height: 1.5;
       font-family: sans-serif;
   }
   img, picture, video, canvas, svg {
       display: block;
       max-width: 100%;
   }
   input, button, textarea, select {
       font: inherit;
   }

4. Usar rem para tamaños:
   html { font-size: 16px; }
   h1 { font-size: 2rem; }    /* 32px */
   h2 { font-size: 1.5rem; }  /* 24px */
   p { font-size: 1rem; }     /* 16px */

5. Usar variables CSS para colores y temas.

6. Evitar !important (a menos que sea absolutamente necesario).

7. Usar nombres de clases descriptivos y consistentes.

8. Comentar secciones complejas.
