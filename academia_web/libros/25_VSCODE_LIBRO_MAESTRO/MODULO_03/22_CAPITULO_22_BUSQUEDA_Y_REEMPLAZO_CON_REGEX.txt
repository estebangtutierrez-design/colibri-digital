CONTENIDO DEL CAPÍTULO
===============================================================================

1. BÚSQUEDA (Ctrl+F / Ctrl+Shift+F)

Ctrl+F busca en archivo actual. Ctrl+Shift+F busca en todo el proyecto.

El botón .* activa expresiones regulares.



2. REGEX ÚTILES

\d+ para dígitos, \w+ para palabras, [A-Z] para mayúsculas. Grupos de

captura con () referenciados como $1, $2 en reemplazo.



3. BÚSQUEDA GLOBAL

Incluir/excluir patrones de archivos. Resultados agrupados por archivo.

Preserve Case (Aa) mantiene mayúsculas/minúsculas en reemplazo.



4. REEMPLAZO AVANZADO

Convertir snake_case a camelCase con regex. Cambiar nombres de funciones.

Transformar estructuras de datos entre formatos.

EJEMPLO DE CÓDIGO
----------------------------------------

# console.log(.*) -> $1
# Buscar: console\.log\((.*)\)
# Reemplazar: logger.info($1)
# Buscar _([a-z]) -> \U$1 (camelCase)
