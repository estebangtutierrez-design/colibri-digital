CONTENIDO DEL CAPÍTULO
===============================================================================

1. SNIPPETS POR PROYECTO

.vscode/nombre.code-snippets para snippets compartidos en el equipo.

Placeholders con opciones: ${1|opcion1,opcion2|} para elegir entre valores.



2. TRANSFORMACIONES

${TM_FILENAME/(.*)\..*/$1/} nombre sin extensión. ${CURRENT_YEAR} año actual.

${TM_SELECTED_TEXT} texto seleccionado. Transformaciones regex en variables.



3. EMMET PERSONALIZADO

emmet.snippets en settings.json para snippets propios. Más rápidos que

snippets normales porque solo requieren Tab.



4. SNIPPETS MULTI-LENGUAJE

Archivo .code-snippets global con scope específico en cada entrada.

Un mismo snippet puede funcionar en JS, TS, JSX, TSX.

EJEMPLO DE CÓDIGO
----------------------------------------

{
  "Python Class with Docstring": {
    "scope": "python",
    "prefix": "classdoc",
    "body": [
      "class ${1:ClassName}:",
      "    """${2:Description}.",
      "    Args:",
      "        ${3}",
      "    """",
      "    def __init__(self, ${4}):",
      "        ${5:pass}"
    ]
  }
}
