CONTENIDO DEL CAPÍTULO
===============================================================================

1. MULTI-CURSOR

Alt+Click para cursor adicional. Ctrl+Alt+Up/Down para cursor arriba/abajo.

Ctrl+D selecciona siguiente ocurrencia y añade cursor.



2. SELECCIÓN DE OCURRENCIAS

Ctrl+Shift+L selecciona TODAS las ocurrencias simultáneamente. Útil para

renombrar variables locales o cambiar valores repetidos.



3. BOX SELECTION

Shift+Alt+arrastrar selecciona un bloque rectangular. Editar columnas de

datos, alinear código o modificar prefijos en múltiples líneas.



4. CASOS DE USO

Renombrar propiedades en múltiples líneas. Agregar/quitar prefijos. Editar

estructuras repetitivas como tests o datos mock. Convertir formatos.
