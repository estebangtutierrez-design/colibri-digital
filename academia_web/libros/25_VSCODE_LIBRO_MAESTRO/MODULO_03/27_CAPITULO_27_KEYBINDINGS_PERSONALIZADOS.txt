CONTENIDO DEL CAPÍTULO
===============================================================================

1. POR QUÉ PERSONALIZAR

Atajos de teclado son la herramienta de productividad más importante.

Personalizar tareas frecuentes ahorra horas cada semana.



2. CÓMO PERSONALIZAR

File > Preferences > Keyboard Shortcuts (Ctrl+K Ctrl+S). Busca comando,

doble clic y presiona combinación. Edita keybindings.json para avanzado.



3. CONCEPTOS AVANZADOS

When clauses: condiciones para que el atajo funcione (editorTextFocus, etc).

Chords: secuencias de teclas (Ctrl+K seguido de Ctrl+C).



4. EXTENSIONES DE PRODUCTIVIDAD

Vim (vscodevim.vim): navegación estilo Vim. Bookmarks: navegación rápida.

vs-code-tasks: gestión de tareas desde keybindings.

EJEMPLO DE CÓDIGO
----------------------------------------

[
  { "key": "ctrl+tab", "command": "workbench.action.nextEditor" },
  { "key": "ctrl+shift+tab", "command": "workbench.action.previousEditor" },
  { "key": "ctrl+alt+t", "command": "workbench.action.terminal.new" },
  { "key": "alt+1", "command": "workbench.action.openEditorAtIndex1" },
  { "key": "alt+2", "command": "workbench.action.openEditorAtIndex2" }
