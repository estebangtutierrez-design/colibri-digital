CONTENIDO DEL CAPÍTULO
===============================================================================

1. GIT INTEGRADO (Ctrl+Shift+G)

Archivos modificados, staging, commits, push, pull, ramas, resolución de

conflictos. Todo sin salir del editor.



2. DIFF EDITOR

Clic en archivo modificado abre el editor de diferencias. Verde: adiciones.

Rojo: eliminaciones. Navegar cambios con F7/Shift+F7.



3. RAMAS

Barra de estado muestra rama actual. Clic para cambiar, crear, publicar,

y sincronizar ramas.



4. RESOLUCIÓN DE CONFLICTOS

Merge Editor de tres paneles: Current, Incoming, Result. Botones para

aceptar uno, ambos, o editar manualmente.
