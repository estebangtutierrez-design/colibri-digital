CONTENIDO DEL CAPÍTULO
===============================================================================

1. INSTALACIÓN EN WINDOWS

Descarga el instalador desde code.visualstudio.com. Marca: agregar al menú

contextual, registrar como editor para archivos compatibles, y agregar al PATH.



2. INSTALACIÓN EN LINUX

sudo snap install code --classic (recomendado). O usa el repositorio oficial

de Microsoft para actualizaciones automáticas via apt.



3. INSTALACIÓN EN macOS

Descarga el .zip, arrastra VS Code.app a Applications. Abre VS Code, presiona

Cmd+Shift+P y ejecuta 'Shell Command: Install code command in PATH'.



4. VERIFICACIÓN

code --version muestra la versión. code --list-extensions muestra extensiones.

EJEMPLO DE CÓDIGO
----------------------------------------

code --version
code --list-extensions
