1. ¿QUÉ ES VS CODE?

Visual Studio Code (VS Code) es un editor de código desarrollado por Microsoft

para Windows, Linux y macOS. Es gratuito, de código abierto y se ha convertido

en el editor más utilizado por desarrolladores en todo el mundo desde su

lanzamiento en 2015. Está construido sobre Electron (Chromium + Node.js).



2. HISTORIA

VS Code nació como proyecto interno liderado por Erich Gamma (autor de Design

Patterns). La primera versión pública se lanzó el 29 de abril de 2015. Desde

entonces ha alcanzado más del 70% de cuota de mercado entre editores de código.



3. ARQUITECTURA INTERNA

Se compone de: interfaz de usuario (Electron), núcleo del editor (Monaco Editor),

sistema de extensiones, y Language Server Protocol (LSP) para inteligencia de

código multi-lenguaje.



4. ECOSISTEMA

Cuenta con más de 30,000 extensiones en el marketplace. La comunidad contribuye

activamente con temas, iconos, snippets y herramientas. Empresas como Google,

Amazon y Red Hat mantienen extensiones oficiales.
