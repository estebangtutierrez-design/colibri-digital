CONTENIDO DEL CAPÍTULO
===============================================================================

1. INTELLISENSE

Sistema de autocompletado contextual. Combina análisis de código, definiciones

de tipos, documentación y Language Server Protocol (LSP).



2. TIPOS DE SUGERENCIAS

Autocompletado (Ctrl+Space): palabras del documento. Snippets: fragmentos

predefinidos. ParamHints (Ctrl+Shift+Space): parámetros de funciones.

Quick Info (Ctrl+K Ctrl+I): información de la entidad bajo el cursor.



3. CONFIGURACIÓN

editor.quickSuggestions, editor.suggest.insertMode, editor.snippetSuggestions,

editor.suggestSelection. Ajusta comportamiento según preferencias.



4. LSP

Language Server Protocol proporciona inteligencia de código para cualquier

lenguaje. VS Code incluye servidores para JS/TS, CSS, HTML, JSON.

EJEMPLO DE CÓDIGO
----------------------------------------

{
  "editor.quickSuggestions": { "other": true, "comments": false, "strings": true },
  "editor.suggest.insertMode": "insert",
  "editor.snippetSuggestions": "inline",
  "editor.suggestSelection": "first",
  "editor.acceptSuggestionOnEnter": "off"
}
