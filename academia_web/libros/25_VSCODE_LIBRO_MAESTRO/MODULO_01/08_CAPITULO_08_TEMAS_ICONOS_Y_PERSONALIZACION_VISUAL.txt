CONTENIDO DEL CAPÍTULO
===============================================================================

1. TEMAS DE COLOR

File > Preferences > Color Theme o Ctrl+K Ctrl+T. One Dark Pro, Dracula,

Monokai Pro, GitHub Theme, Night Owl, Nord, Tokyo Night, Catppuccin.



2. TEMAS DE ICONOS

Cambian iconos de archivos y carpetas en el explorador. Material Icon Theme

es el más popular. Configurar en File > File Icon Theme.



3. PERSONALIZACIÓN AVANZADA

workbench.colorCustomizations en settings.json para colores específicos.

Modifica fondo del editor, color de pestañas, status bar, scrollbar, etc.



4. PRODUCT ICON THEME

Temas de iconos de producto: icons-carbon, icons-hood, etc. Cambia iconos

de la interfaz (nuevo archivo, guardar, etc.).

EJEMPLO DE CÓDIGO
----------------------------------------

{
  "workbench.colorTheme": "One Dark Pro",
  "workbench.iconTheme": "material-icon-theme",
  "workbench.colorCustomizations": {
    "editor.background": "#1e1e2e",
    "editor.foreground": "#cdd6f4",
    "tab.activeBorderTop": "#cba6f7",
    "statusBar.background": "#11111b"
  }
}
