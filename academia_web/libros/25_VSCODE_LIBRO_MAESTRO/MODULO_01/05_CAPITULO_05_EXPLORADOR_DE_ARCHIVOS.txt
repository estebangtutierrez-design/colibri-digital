CONTENIDO DEL CAPÍTULO
===============================================================================

1. EXPLORADOR (Ctrl+Shift+E)

Muestra la estructura del proyecto. Crear, renombrar, mover y eliminar archivos

desde clic derecho. Soporta drag and drop.



2. WORKSPACES

Colección de una o más carpetas en una misma ventana. Guardar como .code-workspace

para compartir configuración entre proyectos relacionados.



3. FILTROS

Campo de búsqueda en explorador para filtrar por nombre. Configura

files.exclude para ocultar node_modules, dist, .git, etc.



4. OPEN RECENT

File > Open Recent muestra últimos proyectos. Fija proyectos importantes.

window.restoreWindows controla qué ventanas restaurar al inicio.

EJEMPLO DE CÓDIGO
----------------------------------------

{
  "files.exclude": {
    "**/node_modules": true,
    "**/dist": true,
    "**/.git": true
  },
  "search.exclude": {
    "**/node_modules": true,
    "**/dist": true
  }
}
