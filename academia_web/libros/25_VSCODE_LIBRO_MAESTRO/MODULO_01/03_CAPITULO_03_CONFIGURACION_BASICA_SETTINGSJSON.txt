CONTENIDO DEL CAPÍTULO
===============================================================================

1. SETTINGS.JSON

Tres niveles de configuración: User (global), Workspace (.vscode/settings.json),

y por lenguaje. Se accede con Ctrl+, y el icono de archivo JSON en esquina superior.



2. CONFIGURACIONES ESENCIALES

editor.fontSize, editor.fontFamily, editor.tabSize, editor.wordWrap,

editor.minimap.enabled, files.autoSave, editor.formatOnSave.



3. CONFIGURACIÓN POR LENGUAJE

Usa corchetes con el nombre del lenguaje para configuraciones específicas.

Ej: [python] { editor.defaultFormatter, editor.formatOnSave }



4. SINCRONIZACIÓN

Settings Sync sincroniza configuración, extensiones y keybindings entre

dispositivos usando cuenta de Microsoft o GitHub.

EJEMPLO DE CÓDIGO
----------------------------------------

{
  "editor.fontSize": 14,
  "editor.fontFamily": "'Fira Code', monospace",
  "editor.fontLigatures": true,
  "editor.tabSize": 2,
  "editor.wordWrap": "on",
  "editor.minimap.enabled": true,
  "files.autoSave": "afterDelay",
  "files.autoSaveDelay": 1000,
  "git.enableSmartCommit": true
}
