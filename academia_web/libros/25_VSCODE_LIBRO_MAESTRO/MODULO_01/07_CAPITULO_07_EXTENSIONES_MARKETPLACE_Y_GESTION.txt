CONTENIDO DEL CAPÍTULO
===============================================================================

1. MARKETPLACE (Ctrl+Shift+X)

Más de 30,000 extensiones. Busca, instala y gestiona desde el editor. Cada

extensión muestra descripción, versión, rating, descargas y autor.



2. EXTENSIONES ESENCIALES

ESLint, Prettier, GitLens, Live Share, Material Icon Theme, Bracket Pair

Colorizer, Code Spell Checker, Path Intellisense, GitHub Copilot.



3. GESTIÓN

Habilitar/deshabilitar por workspace. Extensiones recomendadas en

.vscode/extensions.json. Extensiones no deseadas en unwantedRecommendations.



4. POR LENGUAJE

Python: ms-python.python + Pylance + Black. JS/TS: ESLint + Prettier.

Java: Extension Pack de Red Hat. Go: golang.go. Rust: rust-analyzer.
