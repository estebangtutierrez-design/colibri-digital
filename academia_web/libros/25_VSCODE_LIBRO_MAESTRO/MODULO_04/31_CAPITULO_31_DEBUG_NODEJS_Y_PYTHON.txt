CONTENIDO DEL CAPÍTULO
===============================================================================

1. NODE.JS

Soporte nativo para Node.js. Depura Express, scripts CLI, código asíncrono.

Breakpoints funcionan en async/await, promesas y callbacks.



2. PYTHON

Con ms-python.python: scripts, Flask/Django, Jupyter notebooks. Soporta

entornos virtuales y conda. Breakpoints en .py y .ipynb.



3. CÓDIGO ASÍNCRONO

Breakpoints en async/await, promesas y callbacks. Call stack muestra

contexto asíncrono y promesas encadenadas correctamente.



4. CONFIGURACIONES AVANZADAS

skipFiles para saltar node_internals. resolveSourceMapLocations para source

maps. runtimeExecutable para usar nodemon con --inspect.

EJEMPLO DE CÓDIGO
----------------------------------------

{
  "name": "Node: API",
  "type": "node",
  "request": "launch",
  "program": "${workspaceFolder}/src/server.js",
  "skipFiles": ["<node_internals>/**", "node_modules/**"],
  "restart": true,
  "runtimeExecutable": "${workspaceFolder}/node_modules/.bin/nodemon"
}
