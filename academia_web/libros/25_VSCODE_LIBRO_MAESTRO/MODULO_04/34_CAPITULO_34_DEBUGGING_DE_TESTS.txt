CONTENIDO DEL CAPÍTULO
===============================================================================

1. TEST EXPLORER

Icono de matraz. Descubre y ejecuta tests de Jest, Mocha, pytest, JUnit.

Ejecuta, depura y ve resultados desde interfaz unificada.



2. DEBUG TESTS

Clic derecho > Debug Test. Icono de debug junto al test. Breakpoints

funcionan durante ejecución de tests.



3. CONFIGURACIÓN

Cada framework necesita configuración específica. Jest: --inspect-brk.

pytest: module pytest con args. Vitest: vitest.mjs con --run.



4. COBERTURA

Coverage Gutters (ryanluker.vscode-coverage-gutters) muestra cobertura en

el editor. Verde: cubierto. Rojo: no cubierto.
