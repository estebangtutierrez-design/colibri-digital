CONTENIDO DEL CAPÍTULO
===============================================================================

1. DOCKER

ms-azuretools.vscode-docker. Gestiona contenedores, imágenes, volúmenes,

redes. Dockerizar proyectos con un clic. Logs de contenedores.



2. KUBERNETES

ms-kubernetes-tools.vscode-kubernetes-tools. Clusters, pods, servicios,

deployments. Autocompletado YAML para recursos K8s.



3. AWS / AZURE / GCP

AWS Toolkit, Azure Tools, Google Cloud Code. Ver recursos, desplegar

serverless, gestionar bases de datos desde el editor.



4. TERRAFORM

HashiCorp Terraform: autocompletado, formateo, validación de .tf. YAML

de Red Hat con schema validation para archivos de configuración.
