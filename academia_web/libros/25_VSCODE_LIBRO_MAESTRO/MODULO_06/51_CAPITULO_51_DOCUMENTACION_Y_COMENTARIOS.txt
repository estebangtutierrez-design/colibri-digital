CONTENIDO DEL CAPÍTULO
===============================================================================

1. DOCUMENTACIÓN AUTOMÁTICA

Document This (JSDoc), Python Docstring Generator (Google/NumPy/Sphinx).

Auto-generación de documentación en Enter después de función/clase.



2. MARKDOWN

Markdown All in One: TOC, listas, formato, atajos. Preview Enhanced para

vista previa avanzada con diagramas y matemáticas.



3. REST CLIENT

Ejecuta peticiones HTTP desde archivos .http. Documentar y probar APIs

sin Postman. Variables de entorno, headers, body JSON.



4. CODE SPELL CHECKER

Revisa ortografía en código, comentarios y strings. Múltiples idiomas.

Diccionarios personalizados. Ignorar patrones regex.
