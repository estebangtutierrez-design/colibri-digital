CONTENIDO DEL CAPÍTULO
===============================================================================

1. SQL TOOLS

SQLTools (mtxr.sqltools) con drivers para PostgreSQL, MySQL, SQLite.

Ejecución de queries, visualización de resultados, conexión a BD.



2. MONGODB

MongoDB for VS Code: conectarse a instancias, navegar colecciones,

ejecutar queries, documentos formateados.



3. REDIS

Redis client: exploración visual de keys, ejecución de comandos,

monitoreo en tiempo real.



4. DATABASE CLIENT

Clientes universales que soportan múltiples motores. Interfaz unificada.

Autocompletado SQL, snippets, exportación de resultados.
