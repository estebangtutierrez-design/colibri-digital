CONTENIDO DEL CAPÍTULO
===============================================================================

1. TEST EXPLORER

Vitest, Jest, pytest integrados. Run, debug, cobertura desde interfaz unificada.



2. COBERTURA

Coverage Gutters muestra verde/rojo en editor. SonarLint detecta code smells.

Error Lens muestra errores inline. Code Spell Checker corrige ortografía.



3. LINTING

ESLint + Prettier para JS/TS. Ruff para Python. EditorConfig para formato

consistente. Linting en tiempo real con Error Lens.



4. STATIC ANALYSIS

SonarLint, Snyk para seguridad. TypeScript strict mode. Pyright/Pylance.
