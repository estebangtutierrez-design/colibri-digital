CONTENIDO DEL CAPÍTULO
===============================================================================

1. TIMELINE

Vista Timeline en explorador (pestaña inferior). Historico de cambios local

y Git para el archivo activo. Restaurar versiones anteriores.



2. NAVEGACIÓN DE CAMBIOS

Alt+F7/F8 navega entre cambios. Ctrl+Shift+G + clic abre diff. Codificación

de colores y líneas de conexión en el diff.



3. COMPARACIÓN ENTRE RAMAS

Ctrl+Shift+P > Git: Compare References. O GitLens para comparación visual.

Muestra archivos diferentes y diffs detallados.



4. GIT BLAME

Git: Blame (Ctrl+Shift+P) muestra quién modificó cada línea. Útil para

saber a quién preguntar sobre código específico.

EJEMPLO DE CÓDIGO
----------------------------------------

# Alt+F7 - siguiente cambio
# Alt+Shift+F7 - cambio anterior
# Ctrl+K Ctrl+G - blame de linea actual
# Git: Compare References - comparar ramas
# Git: Open File on Remote - abrir en GitHub/GitLab
