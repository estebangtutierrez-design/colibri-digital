CONTENIDO DEL CAPÍTULO
===============================================================================

1. PANEL SOURCE CONTROL (Ctrl+Shift+G)

Centro de operaciones Git. Muestra archivos modificados (M), nuevos (U),

eliminados (D), conflictos (C). Cada cambio con estado y diff inline.



2. COMMITS

Mensaje en campo superior, Ctrl+Enter para commit. Commits con firma GPG.

Stage selectivo de líneas, no solo archivos completos.



3. RAMAS

Crear, cambiar, fusionar y eliminar ramas visualmente desde la interfaz.

Barra de estado muestra rama actual. Publicar rama con un clic.



4. STAGING SELECTIVO

Abre diff, selecciona líneas, clic derecho > Stage Selected Ranges.

Commits atómicos con cambios específicos, no archivos enteros.

# Flujo diario:

# 2. Stage archivos/lineas
# 3. Mensaje de commit (Conventional Commits)
# 4. Ctrl+Enter para commit
# 5. Sincronizar con boton sync
