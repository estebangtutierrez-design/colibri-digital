CONTENIDO DEL CAPÍTULO
===============================================================================

1. BLAME ANNOTATIONS

Anotaciones inline en cada línea: quién, cuándo, qué commit. Invaluable

para entender historial del código. Heatmap opcional.



2. EXPLORADOR DE REPOSITORIO

Commits por archivo, historial de ramas, búsqueda de commits, comparación

entre ramas, grafo completo de Git.



3. CODELENS

Información sobre commits encima de funciones y clases. Número de cambios,

autores, fecha del último cambio. Identifica código antiguo.



4. COMPARACIÓN ENTRE RAMAS

Compara archivos entre ramas. Ve commits que una rama tiene y otra no.

Fusiona selectivamente cambios.

EJEMPLO DE CÓDIGO
----------------------------------------

{
  "gitlens.blame.heatmap.enabled": true,
  "gitlens.currentLine.enabled": true,
  "gitlens.codeLens.enabled": true,
  "gitlens.hovers.enabled": true,
  "gitlens.statusBar.format": "${author}, ${date} - ${message}"
}
