CONTENIDO DEL CAPÍTULO
===============================================================================

1. GITHUB ACTIONS EXTENSION

github.vscode-github-actions. Monitorea y gestiona workflows. Estado, logs,

resultados de ejecuciones en tiempo real.



2. EDICIÓN DE WORKFLOWS

Autocompletado y validación YAML para .github/workflows/. Estructura,

eventos disponibles, acciones del marketplace.



3. MONITOREO

Ejecuciones en curso, logs en vivo, notificaciones de fallos. Sin cambiar

de ventana. Historial de ejecuciones anteriores.



4. EVENTOS

push, pull_request, schedule (cron), workflow_dispatch, repository_dispatch.

Workflows reutilizables con actions/checkout, setup-node, etc.

EJEMPLO DE CÓDIGO
----------------------------------------

name: CI
on: [push, pull_request]
jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with: { node-version: '20' }
      - run: npm ci
      - run: npm test
