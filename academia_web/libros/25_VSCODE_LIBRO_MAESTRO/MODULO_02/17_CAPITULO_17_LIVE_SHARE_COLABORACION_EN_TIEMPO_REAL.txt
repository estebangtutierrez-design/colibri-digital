CONTENIDO DEL CAPÍTULO
===============================================================================

1. LIVE SHARE

Extensión ms-vsliveshare.vsliveshare. Colabora en tiempo real compartiendo

tu editor con otros desarrolladores. Como Google Docs, pero para código.



2. FUNCIONALIDADES

Edición colaborativa, audio y texto integrados, compartir servidores locales,

compartir terminal, modo solo lectura o edición completa.



3. CASOS DE USO

Pair programming a distancia, code review en vivo, debugging colaborativo,

mentoría y enseñanza, resolución de bugs en equipo.



4. SEGURIDAD

Control total sobre qué compartir y con quién. Expulsar participantes en

cualquier momento. Sesiones con contraseña y expiración.

EJEMPLO DE CÓDIGO
----------------------------------------

# Iniciar: Ctrl+Shift+P > Live Share: Start Collaboration Session
# Compartir enlace con colaboradores
# Unirse desde VS Code o navegador
# Compartir servidor: Live Share: Share Server
# Compartir terminal: Live Share: Share Terminal
