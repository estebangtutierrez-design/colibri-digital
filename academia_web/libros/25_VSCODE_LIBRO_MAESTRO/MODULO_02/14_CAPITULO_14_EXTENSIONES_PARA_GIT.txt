CONTENIDO DEL CAPÍTULO
===============================================================================

1. GITLENS

Añade blame annotations en línea, exploración de repositorios, comparación

visual de archivos, code lens con información de commits, y más.



2. GIT HISTORY

Visualiza historial de commits de archivos individuales, ramas y repositorio.



3. GIT GRAPH

Representación visual del grafo de commits, ramas y merges. Operaciones

comunes de Git directamente desde el gráfico.



4. GITHUB PULL REQUESTS

Gestiona PRs desde VS Code: crear, revisar, comentar y fusionar sin salir.

EJEMPLO DE CÓDIGO
----------------------------------------

{
  "gitlens.blame.heatmap.enabled": true,
  "gitlens.currentLine.enabled": true,
  "gitlens.codeLens.enabled": true,
  "gitlens.statusBar.format": "${author}, ${date} - ${message}",
  "git.enableSmartCommit": true,
  "git.autofetch": true
}
