CONTENIDO DEL CAPÍTULO
===============================================================================

1. ¿QUÉ ES EMMET?

Plugin de productividad para escribir HTML y CSS usando abreviaciones que se

expanden a código completo. VS Code lo incluye por defecto.



2. SINTAXIS HTML

div.container>ul.list>li.item*5>a[href=#]{Link $} genera una estructura

completa de navegación con 5 items.



3. EMMET CSS

m10 -> margin: 10px, p20-30 -> padding: 20px 30px, d:f -> display: flex,

jc:c -> justify-content: center, ai:c -> align-items: center.



4. PERSONALIZACIÓN

Snippets de Emmet personalizados en settings.json. Configurar en qué tipos

de archivo está activo Emmet.

EJEMPLO DE CÓDIGO
----------------------------------------

!  -> HTML5 completo
div.container -> <div class="container"></div>
ul>li*3 -> lista de 3 items
table>tr*3>td*2 -> tabla 3x2
input:email -> <input type="email">
m10 -> margin: 10px
d:f -> display: flex
jc:c -> justify-content: center
