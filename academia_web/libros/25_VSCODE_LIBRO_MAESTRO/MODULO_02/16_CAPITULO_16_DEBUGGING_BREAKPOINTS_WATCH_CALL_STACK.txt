CONTENIDO DEL CAPÍTULO
===============================================================================

1. BREAKPOINTS

Clic en margen izquierdo del editor. Tipos: simple, condicional, hit count,

logpoint (registra mensaje sin detener).



2. PANEL DE DEBUG (Ctrl+Shift+D)

Variables: locales, globales y closures. Watch: expresiones personalizadas.

Call Stack: pila de llamadas actual. Breakpoints: lista completa.



3. COMANDOS

F5: iniciar/continuar. F10: saltar (step over). F11: entrar (step into).

Shift+F11: salir (step out). Ctrl+Shift+F5: reiniciar. Shift+F5: detener.



4. LAUNCH.JSON

Configuraciones en .vscode/launch.json. Tipos: launch (iniciar programa),

attach (conectar a proceso existente). Múltiples configuraciones.

EJEMPLO DE CÓDIGO
----------------------------------------

{
  "version": "0.2.0",
  "configurations": [
    {
      "name": "Node.js",
      "type": "node",
      "request": "launch",
      "program": "${workspaceFolder}/src/index.js",
      "skipFiles": ["<node_internals>/**"]
    },
    {
      "name": "Python",
      "type": "python",
      "request": "launch",
      "program": "${workspaceFolder}/app.py",
      "console": "integratedTerminal"
    }
  ]
}
