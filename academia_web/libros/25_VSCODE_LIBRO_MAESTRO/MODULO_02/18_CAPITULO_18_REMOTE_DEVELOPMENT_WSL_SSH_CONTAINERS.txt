CONTENIDO DEL CAPÍTULO
===============================================================================

1. REMOTE EXTENSION PACK

Tres extensiones de Microsoft: Remote - WSL, Remote - SSH, Remote - Containers.

Desarrolla en cualquier entorno desde VS Code.



2. REMOTE - SSH

Conéctate a servidores remotos vía SSH. Extensiones, configuraciones y

terminal se ejecutan en la máquina remota.



3. REMOTE - CONTAINERS

Desarrolla dentro de contenedores Docker. Define el entorno en devcontainer.json.

Ideal para equipos que necesitan entornos idénticos.



4. REMOTE - WSL

Para Windows con WSL2. VS Code se conecta a la distribución Linux. Todo

el desarrollo corre en Linux con acceso completo al sistema de archivos.
