===============================================================================
1. PARÁMETROS BÁSICOS
===============================================================================

Los parámetros son variables que la función recibe al ser llamada.

```javascript
function saludar(nombre) {
    console.log("Hola, " + nombre);
}

saludar("Ana");    // Hola, Ana
saludar("Luis");   // Hola, Luis

Parámetros múltiples se separan por coma:

```javascript
function presentar(nombre, edad, ciudad) {
    console.log(nombre + " tiene " + edad + " años y vive en " + ciudad);
}

presentar("Ana", 25, "Madrid");
// Ana tiene 25 años y vive en Madrid

===============================================================================
VALORES POR DEFECTO (DEFAULT PARAMETERS)
===============================================================================

Si no pasas un argumento, el parámetro es undefined.
Puedes asignar un valor por defecto.

```javascript
function saludar(nombre = "Invitado") {
    console.log("Hola, " + nombre);
}

saludar("Ana");     // Hola, Ana
saludar();          // Hola, Invitado

Los valores por defecto pueden ser cualquier expresión:

```javascript
function calcularPrecio(precio, impuesto = 0.21, envio = precio > 100 ? 0 : 5) {
    return precio + (precio * impuesto) + envio;
}

console.log(calcularPrecio(50));     // 50 + 10.5 + 5 = 65.5
console.log(calcularPrecio(200));    // 200 + 42 + 0 = 242

===============================================================================
EL OBJETO ARGUMENTS
===============================================================================

Dentro de una función, puedes acceder a arguments:
un objeto array-like con TODOS los argumentos pasados.

```javascript
function sumarTodo() {
    let total = 0;
    for (let i = 0; i < arguments.length; i++) {
        total += arguments[i];
    }
    return total;
}

console.log(sumarTodo(1, 2, 3));       // 6
console.log(sumarTodo(10, 20, 30, 40)); // 100

arguments no es un array real (no tiene .push, .map, etc.)

===============================================================================
PARÁMETRO REST (...)
===============================================================================

Rest permite capturar "el resto" de argumentos en un array real.

```javascript
function listarAmigos(nombre, ...amigos) {
    console.log(nombre + " tiene " + amigos.length + " amigos:");
    amigos.forEach(function(amigo) {
        console.log("  - " + amigo);
    });
}

listarAmigos("Ana", "Luis", "María", "Pedro");
// Ana tiene 3 amigos:
//   - Luis
//   - María
//   - Pedro

Rest siempre va al final de los parámetros:
```javascript
function producto(nombre, precio, ...caracteristicas) {
    console.log(nombre + ": $" + precio);
    console.log("Características: " + caracteristicas.join(", "));
}

producto("Laptop", 999, "16GB RAM", "512GB SSD", "i7");
```

===============================================================================
FUNCIONES COMO ARGUMENTOS (CALLBACKS)
===============================================================================

En JavaScript, las funciones son valores.
Puedes pasarlas como argumentos a otras funciones.

```javascript
function procesarNumeros(a, b, operacion) {
    return operacion(a, b);
}

function sumar(x, y) { return x + y; }
function restar(x, y) { return x - y; }

console.log(procesarNumeros(10, 5, sumar));   // 15
console.log(procesarNumeros(10, 5, restar));  // 5

También puedes pasar funciones anónimas:

```javascript
procesarNumeros(10, 5, function(x, y) {
    return x * y;
});  // 50
```

===============================================================================
LA ANALOGÍA DEL PROFESOR BÚHO: EL RESTAURANTE
===============================================================================

Un restaurante tiene pedidos (argumentos) que cambian cada vez.

El menú (función) recibe ingredientes (parámetros).
Si no especificas un ingrediente, tiene un valor por defecto.

```javascript
function hacerPizza(tamaño = "mediana", ...ingredientes) {
    console.log("🍕 Pizza " + tamaño + " con:");
    if (ingredientes.length === 0) {
        console.log("  - Queso (por defecto)");
    } else {
        ingredientes.forEach(function(ing) {
            console.log("  - " + ing);
        });
    }
}

hacerPizza("grande", "pepperoni", "champiñones", "aceitunas");
hacerPizza();  // Pizza mediana con queso
```

===============================================================================
ERRORES COMUNES
===============================================================================

❌ Error 1: Olvidar pasar argumentos
```javascript
function multiplicar(a, b) {
    return a * b;
}
console.log(multiplicar(5));  // NaN (undefined * 5)
```

❌ Error 2: Invertir el orden de los parámetros
```javascript
function registro(nombre, edad, email) { ... }
registro(25, "Ana", "ana@email.com");  // ❌ Orden incorrecto
```

❌ Error 3: Usar arguments en arrow functions
```javascript
const sumar = () => {
    console.log(arguments);  // ❌ No funciona en arrow functions
};
```

📝 Ejercicio 1: Función con valor por defecto
```javascript
function calcularDescuento(precio, descuento = 10) {
    return precio - (precio * descuento / 100);
}
console.log(calcularDescuento(100));      // 90
console.log(calcularDescuento(100, 50));  // 50
```

📝 Ejercicio 2: Función con REST
```javascript
function concatenar(separador, ...palabras) {
    return palabras.join(separador);
}
console.log(concatenar("-", "a", "b", "c"));  // a-b-c
```

📝 Ejercicio 3: Función que recibe callback
```javascript
function aplicarOperacion(array, callback) {
    let resultado = [];
    for (let i = 0; i < array.length; i++) {
        resultado.push(callback(array[i]));
    }
    return resultado;
}

let numeros = [1, 2, 3, 4, 5];
let cuadrados = aplicarOperacion(numeros, function(n) {
    return n * n;
});
console.log(cuadrados);  // [1, 4, 9, 16, 25]
```

===============================================================================
MINI PROYECTO: CALCULADORA CON CALLBACKS
===============================================================================

```javascript
function calculadora(a, b, callback) {
    return callback(a, b);
}

let suma = calculadora(10, 5, function(x, y) { return x + y; });
let resta = calculadora(10, 5, function(x, y) { return x - y; });
let multi = calculadora(10, 5, function(x, y) { return x * y; });
let div = calculadora(10, 5, function(x, y) {
    return y === 0 ? "Error" : x / y;
});

console.log("Suma: " + suma + ", Resta: " + resta);
console.log("Multi: " + multi + ", Div: " + div);
```
