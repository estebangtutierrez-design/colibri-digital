===============================================================================
BIENVENIDA DEL PROFESOR BÚHO
===============================================================================

Querido estudiante…

Una función puede hacer algo (acción) o devolver algo (valor).

Las que solo hacen algo se llaman PROCEDIMIENTOS.
Las que devuelven algo se llaman FUNCIONES (en sentido estricto).

La diferencia es return.

Hoy aprenderás todo sobre esta palabra clave fundamental.

===============================================================================
1. ¿QUÉ ES RETURN?
===============================================================================

return hace dos cosas:
1. Devuelve un valor.
2. Termina la función inmediatamente.

```javascript
function sumar(a, b) {
    return a + b;  // Devuelve el resultado y termina
}

let resultado = sumar(3, 4);
console.log(resultado);  // 7

===============================================================================
RETURN CON DIFERENTES TIPOS
===============================================================================

Puedes devolver cualquier tipo de dato:

```javascript
function devolverNumero() { return 42; }
function devolverTexto() { return "Hola"; }
function devolverBooleano() { return true; }
function devolverArray() { return [1, 2, 3]; }
function devolverObjeto() { return { nombre: "Ana", edad: 25 }; }
function devolverNull() { return null; }
function devolverFuncion() {
    return function() { console.log("Soy una función"); };
}

===============================================================================
RETURN ANTICIPADO
===============================================================================

Puedes tener múltiples return en una función.
El primero que se ejecuta termina la función.

```javascript
function clasificarEdad(edad) {
    if (edad < 0) return "Edad inválida";
    if (edad < 13) return "Niño";
    if (edad < 18) return "Adolescente";
    if (edad < 65) return "Adulto";
    return "Adulto mayor";
}

console.log(clasificarEdad(10));  // Niño
console.log(clasificarEdad(25));  // Adulto
console.log(clasificarEdad(-5));  // Edad inválida

===============================================================================
FUNCIÓN SIN RETURN
===============================================================================

Si una función no tiene return, devuelve undefined.

```javascript
function mostrarMensaje(msg) {
    console.log(msg);
    // No hay return
}

let valor = mostrarMensaje("Hola");
console.log(valor);  // undefined

===============================================================================
RETURN DE OBJETOS
===============================================================================

Muy usado para devolver múltiples valores:

```javascript
function calcularEstadisticas(numeros) {
    let suma = 0;
    let min = numeros[0];
    let max = numeros[0];

    for (let i = 0; i < numeros.length; i++) {
        suma += numeros[i];
        if (numeros[i] < min) min = numeros[i];
        if (numeros[i] > max) max = numeros[i];
    }

    return {
        suma: suma,
        promedio: suma / numeros.length,
        minimo: min,
        maximo: max,
        total: numeros.length
    };
}

let datos = calcularEstadisticas([10, 20, 30, 40, 50]);
console.log(datos.promedio);  // 30
console.log(datos.maximo);    // 50
```

===============================================================================
ERRORES COMUNES
===============================================================================

❌ Error 1: Olvidar return
```javascript
function duplicar(n) {
    let resultado = n * 2;
    // Falta: return resultado;
}
console.log(duplicar(5));  // undefined
```

❌ Error 2: Código después de return
```javascript
function ejemplo() {
    return "Fin";
    console.log("Esto nunca se ejecuta");  // ❌
}
```

❌ Error 3: Salto de línea después de return
```javascript
function ejemplo() {
    return     // JavaScript añade ; aquí
        "Hola";
}
console.log(ejemplo());  // undefined
```

function mayorDeTres(a, b, c) {

        edad: edad,
        ciudad: ciudad,
        saludar: function() {

        }
    };

```

📝 Ejercicio 3: Validación con return anticipado
```javascript
function validarFormulario(nombre, email, password) {
    if (!nombre) return "El nombre es obligatorio";
    if (!email.includes("@")) return "Email inválido";
    if (password.length < 6) return "Password muy corto";
    return "Válido";
}
```

function analizarTexto(texto) {
    if (!texto) return { error: "Texto vacío" };

    let palabras = texto.split(" ");

    return {
        original: texto,
        longitud: texto.length,
        palabras: palabras.length,

        invertido: texto.split("").reverse().join(""),
        primeraPalabra: palabras[0],
        ultimaPalabra: palabras[palabras.length - 1]
    };
}
