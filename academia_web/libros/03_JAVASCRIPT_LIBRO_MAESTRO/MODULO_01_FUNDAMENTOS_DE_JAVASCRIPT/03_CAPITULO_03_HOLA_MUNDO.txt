INTRODUCCIÓN: EL TRADICIONAL "HOLA MUNDO"
===============================================================================

El "Hola Mundo" es el programa más simple que existe en cualquier lenguaje de programación.

Su único propósito es mostrar un mensaje que diga "Hola Mundo" o "Hello World".

Parece trivial, pero es un ritual que todos los programadores han hecho.
Es la forma de probar que todo funciona: el editor, el navegador, la conexión entre archivos.

Es el "primer ladrillo" de tu carrera como programador.

---

📌 **¿Por qué "Hola Mundo"?**

La tradición comenzó con el libro "The C Programming Language" (1978)
de Brian Kernighan y Dennis Ritchie.
Desde entonces, casi todos los libros de programación comienzan con este ejemplo.

===============================================================================
MÉTODOS PARA MOSTRAR MENSAJES EN JAVASCRIPT
===============================================================================

Hay tres formas principales de mostrar mensajes en JavaScript.

---

📌 **1. console.log() → Muestra en la consola del navegador**

Es la forma más común de mostrar información mientras desarrollas.

```javascript
console.log("Hola Mundo desde la consola");
📌 2. alert() → Muestra en una ventana emergente

Es útil para interacciones simples con el usuario.

javascript
alert("Hola Mundo desde una alerta");
📌 3. document.write() → Escribe directamente en la página HTML

Escribe contenido en el documento HTML.

javascript
document.write("<h1>Hola Mundo desde la página</h1>");
🔵 Comparativa de métodos

Método	Dónde se muestra	Uso
console.log()	Consola del navegador	Depuración, desarrollo
alert()	Ventana emergente	Interacción simple
document.write()	En la página HTML	Escribir contenido en la página

===============================================================================
PASO 3: EXPLICACIÓN LÍNEA POR LÍNEA
===============================================================================

📌 Línea 1-4: Comentarios

javascript
// ============================================================
// MI PRIMER PROGRAMA EN JAVASCRIPT
// ============================================================
Los comentarios son texto que el navegador ignora.
Sirven para documentar el código.
// es el comentario de una línea.

📌 Línea 6: console.log()

javascript
console.log("¡Hola Mundo desde la consola!");
console → el objeto que representa la consola del navegador.

.log() → el método que muestra mensajes en la consola.

"¡Hola Mundo!" → el mensaje que se muestra (entre comillas).

📌 Línea 9: alert()

javascript
alert("¡Hola Mundo desde una alerta!");
alert() → muestra una ventana emergente.

El mensaje entre comillas es el que se muestra.

📌 Línea 12: document.write()

javascript
document.write("<p>¡Hola Mundo desde la página!</p>");
document → representa la página HTML.

.write() → escribe contenido en la página.

El contenido puede ser texto o HTML.

===============================================================================
ERRORES COMUNES AL ESCRIBIR "HOLA MUNDO"
===============================================================================

❌ Error 1: Olvidar las comillas

javascript
console.log(Hola Mundo);   // ❌ Error
console.log("Hola Mundo"); // ✅ Correcto
❌ Error 2: Olvidar el punto y coma

javascript
console.log("Hola Mundo")   // Funciona pero no es buena práctica
console.log("Hola Mundo");  // ✅ Correcto
❌ Error 3: Mayúsculas y minúsculas

JavaScript es sensible a mayúsculas.

javascript
Console.log("Hola Mundo");   // ❌ Error
console.log("Hola Mundo");   // ✅ Correcto
❌ Error 4: No vincular el script en el HTML

Si no incluyes la etiqueta <script src="script.js"></script>,
el código JavaScript no se ejecutará.

===============================================================================
PREGUNTAS FRECUENTES (FAQ)
===============================================================================

❓ ¿Cuál es la diferencia entre console.log() y alert()?

console.log() muestra mensajes en la consola (solo visible para el desarrollador).
alert() muestra una ventana emergente visible para todos los usuarios.

❓ ¿Por qué no veo mi mensaje en la consola?

Asegúrate de:

Haber guardado el archivo script.js.

Haber vinculado correctamente el script en el HTML.

Abrir la consola del navegador (F12 → Consola).

❓ ¿Cuál es la diferencia entre comillas simples y dobles?

No hay diferencia en JavaScript. Puedes usar ambas.
La única regla es: si usas comillas simples al inicio, debes cerrar con comillas simples.
Si usas comillas dobles, cierras con comillas dobles.

❓ ¿Qué pasa si olvido el punto y coma?

JavaScript puede funcionar sin punto y coma porque tiene "inserción automática de punto y coma".
Pero es mejor ponerlos para evitar errores.
