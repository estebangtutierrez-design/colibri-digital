BIENVENIDA DEL PROFESOR BÚHO

Querido estudiante…

Ahora que sabes qué tipos de datos existen, necesitas aprender a manipularlos.

Los **operadores** son símbolos que realizan operaciones sobre valores.

- Quieres sumar dos números → operador `+`.
- Quieres comparar dos valores → operador `===`.
- Quieres comprobar si algo es verdadero → operadores lógicos (`&&`, `||`, `!`).

Hay varios tipos de operadores en JavaScript:
- **Aritméticos** → `+`, `-`, `*`, `/`, `%`, `**`.
- **Asignación** → `=`, `+=`, `-=`, `*=`, `/=`.
- **Comparación** → `===`, `!==`, `>`, `<`, `>=`, `<=`.
- **Lógicos** → `&&` (Y), `||` (O), `!` (NO).
- **Tipo** → `typeof`.
- **Concatenación** → `+` (para strings).

Hoy vas a aprenderlos todos.

===============================================================================
1. OPERADORES ARITMÉTICOS
===============================================================================

Los operadores aritméticos realizan operaciones matemáticas.

---

📌 **Tabla de operadores aritméticos**

| Operador | Descripción | Ejemplo | Resultado |
|----------|-------------|---------|-----------|
| `+` | Suma | `5 + 3` | `8` |
| `-` | Resta | `5 - 3` | `2` |
| `*` | Multiplicación | `5 * 3` | `15` |
| `/` | División | `15 / 3` | `5` |
| `%` | Módulo (resto) | `10 % 3` | `1` |
| `**` | Potencia | `2 ** 3` | `8` |

---

🔵 **Ejemplos**

```javascript
let a = 10;
let b = 3;

console.log(a + b);   // 13
console.log(a - b);   // 7
console.log(a * b);   // 30
console.log(a / b);   // 3.333...
console.log(a % b);   // 1
console.log(a ** b);  // 1000
📌 Módulo (%): El resto de una división.

javascript
console.log(10 % 3);   // 1 (porque 10 ÷ 3 = 3, sobra 1)
console.log(10 % 2);   // 0 (porque 10 ÷ 2 = 5, sobra 0)
console.log(7 % 3);    // 1
📌 Potencia ()**: Eleva un número a la potencia de otro.

javascript
console.log(2 ** 3);   // 8 (2 * 2 * 2)
console.log(3 ** 2);   // 9 (3 * 3)
console.log(10 ** 2);  // 100
===============================================================================

OPERADORES DE ASIGNACIÓN
===============================================================================

Los operadores de asignación guardan valores en variables.

📌 Tabla de operadores de asignación

Operador	Descripción	Ejemplo	Equivalente
=	Asignación simple	x = 5	x = 5
+=	Suma y asigna	x += 3	x = x + 3
-=	Resta y asigna	x -= 3	x = x - 3
*=	Multiplica y asigna	x *= 3	x = x * 3
/=	Divide y asigna	x /= 3	x = x / 3
%=	Módulo y asigna	x %= 3	x = x % 3
**=	Potencia y asigna	x **= 3	x = x ** 3
🔵 Ejemplos

javascript
let x = 10;

x += 5;    // x = 15
x -= 3;    // x = 12
x *= 2;    // x = 24
x /= 4;    // x = 6
x %= 4;    // x = 2
x **= 3;   // x = 8
📌 Incremento y decremento

Operador	Descripción	Ejemplo
++	Incremento (suma 1)	x++ o ++x
--	Decremento (resta 1)	x-- o --x
javascript
let contador = 0;
contador++;    // contador = 1
contador++;    // contador = 2
contador--;    // contador = 1
===============================================================================

OPERADORES DE COMPARACIÓN
===============================================================================

Los operadores de comparación comparan dos valores y devuelven true o false.

📌 Tabla de operadores de comparación

Operador	Descripción	Ejemplo	Resultado
===	Igualdad estricta	5 === 5	true
!==	Desigualdad estricta	5 !== 3	true
>	Mayor que	5 > 3	true
<	Menor que	5 < 3	false
>=	Mayor o igual que	5 >= 5	true
<=	Menor o igual que	5 <= 3	false
⚠️ Importante: == vs ===

== compara valor con conversión automática.

=== compara valor Y tipo (más seguro).

javascript
console.log(10 == "10");   // true (conversión automática)
console.log(10 === "10");  // false (tipo diferente)
Regla: SIEMPRE usa === y !==.

🔵 Ejemplos

javascript
let edad = 25;

console.log(edad === 25);   // true
console.log(edad !== 25);   // false
console.log(edad > 18);     // true
console.log(edad < 18);     // false
console.log(edad >= 25);    // true
console.log(edad <= 20);    // false
===============================================================================

OPERADORES LÓGICOS
===============================================================================

Los operadores lógicos combinan valores booleanos.

📌 Tabla de operadores lógicos

Operador	Descripción	Ejemplo	Resultado
&&	Y (AND)	true && true	true
||	O (OR)	true && false	false
!	NO (NOT)	!true	false
🔵 Ejemplos

javascript
let esMayor = true;
let tienePermiso = false;

console.log(esMayor && tienePermiso);   // false
console.log(esMayor || tienePermiso);   // true
console.log(!esMayor);                   // false
📌 Tabla de verdad de AND (&&)

A	B	A && B
true	true	true
true	false	false
false	true	false
false	false	false
📌 Tabla de verdad de OR (||)

A	B	A || B
true	true	true
true	false	true
false	true	true
false	false	false
📌 Tabla de verdad de NOT (!)

A	!A
true	false
false	true
🔵 Ejemplos prácticos

javascript
let edad = 25;
let tieneLicencia = true;

// Si es mayor de 18 Y tiene licencia
if (edad >= 18 && tieneLicencia) {
    console.log("Puede conducir");
}

// Si es mayor de 18 O tiene permiso de los padres
if (edad >= 18 || tienePermiso) {
    console.log("Puede entrar");
}
===============================================================================

OPERADOR DE CONCATENACIÓN
===============================================================================

El operador + también se usa para unir strings (concatenación).

🔵 Ejemplo

javascript
let nombre = "Ana";
let apellido = "García";
let nombreCompleto = nombre + " " + apellido;
console.log(nombreCompleto);   // "Ana García"
📌 Concatenación con números

javascript
let edad = 25;
let mensaje = "Tengo " + edad + " años";
console.log(mensaje);   // "Tengo 25 años"
📌 Template strings (mejor que concatenación)

javascript
let edad = 25;
let mensaje = `Tengo ${edad} años`;
console.log(mensaje);   // "Tengo 25 años"
===============================================================================

PRECEDENCIA DE OPERADORES
===============================================================================

La precedencia determina el orden en que se evalúan los operadores.

📌 Orden de precedencia (de mayor a menor)

() → paréntesis

** → potencia

*, /, % → multiplicación, división, módulo

+, - → suma, resta

===, !==, >, <, >=, <= → comparación

&& → AND lógico

|| → OR lógico

🔵 Ejemplo

javascript
let resultado = 10 + 5 * 2;
console.log(resultado);   // 20 (primero multiplica: 5*2=10, luego suma 10+10=20)

let resultado2 = (10 + 5) * 2;
console.log(resultado2);  // 30 (primero el paréntesis: 10+5=15, luego *2=30)
===============================================================================
ERRORES COMUNES CON OPERADORES
===============================================================================

❌ Error 1: Usar = en lugar de ===

javascript
if (x = 5) {   // ❌ Asignación, no comparación
    // ...
}
Solución: usa === para comparar.

❌ Error 2: Confundir + (suma) con + (concatenación)

javascript
console.log("10" + 5);   // "105" (concatenación)
console.log("10" - 5);   // 5 (resta convierte a número)
❌ Error 3: No usar paréntesis cuando es necesario

javascript
let resultado = 10 + 5 * 2;   // 20 (multiplicación primero)
Solución: usa paréntesis para aclarar.

❌ Error 4: Confundir && con ||

javascript
if (edad >= 18 && tieneLicencia)   // Ambas condiciones deben ser true
if (edad >= 18 || tienePermiso)    // Una condición debe ser true

===============================================================================
PREGUNTAS FRECUENTES (FAQ)
===============================================================================

❓ ¿Cuál es la diferencia entre == y ===?

== compara valor con conversión automática.
=== compara valor Y tipo.

❓ ¿Qué hace el operador %?

Devuelve el resto de una división.

❓ ¿Qué es la precedencia de operadores?

El orden en que se evalúan los operadores en una expresión.

❓ ¿Cuándo usar && y cuándo ||?

&& (AND) requiere que ambas condiciones sean true.
|| (OR) requiere que al menos una condición sea true.

❓ ¿Qué es la concatenación?

Unir dos o más strings con el operador +.
