===============================================================================
1. ¿QUÉ ES LA POO?
===============================================================================

POO = Programación Orientada a Objetos.

Conceptos clave:

CLASE: Plantilla o molde.
OBJETO: Instancia concreta de una clase.
PROPIEDADES: Datos del objeto.
MÉTODOS: Comportamiento del objeto.

```javascript
// Clase = plantilla
class Perro {
    constructor(nombre, raza) {
        this.nombre = nombre;
        this.raza = raza;
    }

    ladrar() {
        console.log(this.nombre + " dice: ¡Guau!");
    }
}

// Objeto = instancia concreta
let firulais = new Perro("Firulais", "Labrador");
let luna = new Perro("Luna", "Pastor Alemán");

firulais.ladrar();  // Firulais dice: ¡Guau!
luna.ladrar();      // Luna dice: ¡Guau!
```

===============================================================================
2. LOS 4 PILARES DE LA POO
===============================================================================

1. ENCAPSULACIÓN: Ocultar datos internos.
2. ABSTRACCIÓN: Mostrar solo lo necesario.
3. HERENCIA: Clases que heredan de otras.
4. POLIMORFISMO: Mismo método, diferente comportamiento.

Los verás uno por uno en los próximos capítulos.

===============================================================================
3. EVOLUCIÓN DE JS HACIA LA POO
===============================================================================

```javascript
// Antes de ES6 (2015): funciones constructoras
function Persona(nombre) {
    this.nombre = nombre;
}
Persona.prototype.saludar = function() {
    console.log("Hola, soy " + this.nombre);
};

let ana = new Persona("Ana");
ana.saludar();

// Después de ES6: class (azúcar sintáctico)
class PersonaModern {
    constructor(nombre) {
        this.nombre = nombre;
    }
    saludar() {
        console.log("Hola, soy " + this.nombre);
    }
}

let luis = new PersonaModern("Luis");
luis.saludar();
```

MINI PROYECTO: COMPARATIVA OBJETO LITERAL VS CLASE
===============================================================================

```javascript
// Objeto literal (único)
let coche1 = {
    marca: "Toyota",
    arrancar() {
        console.log("🚗 " + this.marca + " arrancó");
    }
};

// Clase (múltiples instancias)
class Coche {
    constructor(marca, modelo) {
        this.marca = marca;
        this.modelo = modelo;
        this.encendido = false;
    }
    arrancar() {
        this.encendido = true;
        console.log("🚗 " + this.marca + " " + this.modelo + " arrancó");
    }
    apagar() {
        this.encendido = false;
        console.log("🛑 " + this.marca + " apagado");
    }
}

let toyota = new Coche("Toyota", "Corolla");
let honda = new Coche("Honda", "Civic");

toyota.arrancar();  // 🚗 Toyota Corolla arrancó
honda.arrancar();   // 🚗 Honda Civic arrancó
console.log(typeof Coche);  // function (las clases son funciones)
```
