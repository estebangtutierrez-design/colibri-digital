===============================================================================
BIENVENIDA DEL PROFESOR BÚHO
===============================================================================

Querido estudiante…

Los arrays guardan listas ordenadas.
Los objetos guardan información con nombre.

Piensa en un objeto como una ficha de persona:
- nombre: "Ana"
- edad: 25
- ciudad: "Madrid"

Los objetos son la forma más natural de representar
cosas del mundo real en JavaScript.

===============================================================================
OBJETIVOS DEL CAPÍTULO
===============================================================================

✓ Entender qué es un objeto.
✓ Crear objetos con notación literal.
✓ Acceder y modificar propiedades.
✓ Agregar y eliminar propiedades.
✓ Recorrer objetos con for...in.

===============================================================================
1. CREAR OBJETOS
===============================================================================

```javascript
// Literal (recomendado)
let persona = {
    nombre: "Ana",
    edad: 25,
    ciudad: "Madrid"
};

console.log(persona);
// { nombre: "Ana", edad: 25, ciudad: "Madrid" }
```

===============================================================================
2. ACCEDER A PROPIEDADES
===============================================================================

```javascript
let usuario = {
    nombre: "Carlos",
    email: "carlos@email.com",
    "fecha registro": "2024-01-15"
};

// Notación de punto
console.log(usuario.nombre);   // Carlos
console.log(usuario.email);    // carlos@email.com

// Notación de corchetes (obligatoria para nombres con espacios)
console.log(usuario["fecha registro"]);  // 2024-01-15

// Corchetes con variables
let clave = "nombre";
console.log(usuario[clave]);  // Carlos
```

===============================================================================
3. MODIFICAR Y AGREGAR
===============================================================================

```javascript
let libro = { titulo: "JS", paginas: 200 };

// Modificar
libro.paginas = 250;
console.log(libro.paginas);  // 250

// Agregar nuevas propiedades
libro.autor = "Ana García";
libro["año publicación"] = 2024;

console.log(libro);
// { titulo: "JS", paginas: 250, autor: "Ana García", "año publicación": 2024 }
```

===============================================================================
4. ELIMINAR PROPIEDADES
===============================================================================

```javascript
let coche = {
    marca: "Toyota",
    modelo: "Corolla",
    año: 2020,
    color: "rojo"
};

delete coche.color;
console.log(coche);
// { marca: "Toyota", modelo: "Corolla", año: 2020 }
```

===============================================================================
5. RECORRER OBJETOS
===============================================================================

```javascript
let estudiante = {
    nombre: "Sofía",
    edad: 22,
    curso: "JavaScript"
};

// for...in (recorre las CLAVES)
for (let clave in estudiante) {
    console.log(clave + ": " + estudiante[clave]);
}
// nombre: Sofía
// edad: 22
// curso: JavaScript

// Object.keys() → array de claves
console.log(Object.keys(estudiante));
// ["nombre", "edad", "curso"]

// Object.values() → array de valores
console.log(Object.values(estudiante));
// ["Sofía", 22, "JavaScript"]
```

MINI PROYECTO: FICHA DE ESTUDIANTE
===============================================================================

```javascript
function crearFichaEstudiante(nombre, edad, curso, notas) {
    return {
        nombre: nombre,
        edad: edad,
        curso: curso,
        notas: notas || [],
        activo: true,
        fechaIngreso: new Date().toLocaleDateString()
    };
}

function mostrarFicha(estudiante) {
    console.log("=== FICHA DEL ESTUDIANTE ===");
    for (let clave in estudiante) {
        console.log(clave + ": " + estudiante[clave]);
    }
}

function actualizarNota(estudiante, materia, nota) {
    estudiante.notas.push({ materia: materia, nota: nota });
    console.log("✅ Nota agregada: " + materia + " = " + nota);
}

// Uso
let ana = crearFichaEstudiante("Ana", 25, "JavaScript Master");

console.log("Nombre: " + ana.nombre);   // Ana
console.log("Curso: " + ana.curso);     // JavaScript Master

actualizarNota(ana, "Arrays", 95);
actualizarNota(ana, "Objetos", 88);
mostrarFicha(ana);
```
