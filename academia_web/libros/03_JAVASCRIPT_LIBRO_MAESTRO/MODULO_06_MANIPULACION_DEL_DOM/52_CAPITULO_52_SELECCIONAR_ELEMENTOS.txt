BIENVENIDA DEL PROFESOR BÚHO
===============================================================================

Querido estudiante…

Antes de modificar un elemento, necesitas SELECCIONARLO.

Es como ir a una biblioteca: primero buscas el libro,
luego lo tomas y lo modificas (subrayas, doblas páginas...).

JavaScript ofrece varios métodos de búsqueda:
- Por ID
- Por clase
- Por etiqueta
- Por selector CSS (el más poderoso)

===============================================================================
OBJETIVOS DEL CAPÍTULO
===============================================================================

✓ Usar getElementById.
✓ Usar getElementsByClassName y getElementsByTagName.
✓ Usar querySelector y querySelectorAll.
✓ Entender la diferencia entre Live y Static collections.

===============================================================================
1. SELECCIONAR POR ID
===============================================================================

```html
<p id="mensaje">Hola Mundo</p>
<button id="btn">Clic</button>
```

```javascript
let mensaje = document.getElementById("mensaje");
console.log(mensaje.textContent);  // "Hola Mundo"

// Devuelve un solo elemento (o null si no existe)
let noExiste = document.getElementById("no-existe");
console.log(noExiste);  // null
```

===============================================================================
2. SELECCIONAR POR CLASE
===============================================================================

```html
<div class="caja">Caja 1</div>
<div class="caja">Caja 2</div>
<div class="caja">Caja 3</div>
```

```javascript
let cajas = document.getElementsByClassName("caja");
console.log(cajas.length);  // 3
console.log(cajas[0].textContent);  // "Caja 1"

// ⚠️ Live collection: se actualiza si el DOM cambia
// No es un array real, no puedes usar forEach directamente
```

===============================================================================
3. SELECCIONAR POR ETIQUETA
===============================================================================

```html
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
```

```javascript
let items = document.getElementsByTagName("li");
console.log(items.length);  // 3

for (let i = 0; i < items.length; i++) {
    console.log(items[i].textContent);
}
```

===============================================================================
4. QUERYSELECTOR Y QUERYSELECTORALL
===============================================================================

Los métodos más flexibles: aceptan cualquier selector CSS.

```javascript
// querySelector: devuelve el PRIMERO que coincida
let primero = document.querySelector(".caja");
let porId = document.querySelector("#mensaje");
let porEtiqueta = document.querySelector("li");

// querySelectorAll: devuelve TODOS (NodeList estática)
let todasCajas = document.querySelectorAll(".caja");
let todosItems = document.querySelectorAll("li");

// Selectores CSS complejos
let botonForm = document.querySelector("form button.primary");
let primerItemLista = document.querySelector("ul.lista li:first-child");
let parImpar = document.querySelectorAll("tr:nth-child(even)");

// NodeList SÍ tiene forEach
todasCajas.forEach(function(caja, i) {
    console.log("Caja " + (i + 1) + ": " + caja.textContent);
});
```

===============================================================================
5. DIFERENCIAS IMPORTANTES
===============================================================================

```javascript
// getElementsByClassName → LIVE (se actualiza automáticamente)
let liveItems = document.getElementsByClassName("item");
console.log("Live antes: " + liveItems.length);  // ej: 3

let nuevo = document.createElement("div");
nuevo.className = "item";
document.body.appendChild(nuevo);

console.log("Live después: " + liveItems.length);  // 4 (se actualizó solo)

// querySelectorAll → STATIC (no se actualiza)
let staticItems = document.querySelectorAll(".item");
console.log("Static antes: " + staticItems.length);  // ej: 3

document.body.appendChild(document.createElement("div").className = "item");

console.log("Static después: " + staticItems.length);  // 3 (no cambió)
```

MINI PROYECTO: INVENTARIO DEL DOM
===============================================================================

```javascript
function explorarDOM() {
    console.log("=== EXPLORADOR DEL DOM ===");
    console.log("ID's encontrados: " + document.querySelectorAll("[id]").length);
    console.log("Clases distintas: " + new Set(
        Array.from(document.querySelectorAll("[class]")).map(
            el => el.className
        )
    ).size);
    console.log("Imágenes: " + document.querySelectorAll("img").length);
    console.log("Enlaces: " + document.querySelectorAll("a").length);
    console.log("Botones: " + document.querySelectorAll("button").length);
    console.log("Inputs: " + document.querySelectorAll("input").length);

    let primerTitulo = document.querySelector("h1");
    if (primerTitulo) {
        console.log("Primer título: " + primerTitulo.textContent);
    }
}
```

RESUMEN

✔ Live: se actualiza solo. Static: no cambia.
