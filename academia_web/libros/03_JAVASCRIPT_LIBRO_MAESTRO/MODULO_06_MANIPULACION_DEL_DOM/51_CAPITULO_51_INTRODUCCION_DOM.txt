1. ¿QUÉ ES EL DOM?
===============================================================================

Cuando el navegador carga una página HTML:

1. Lee el HTML.
2. Construye un árbol de objetos (DOM).
3. Cada etiqueta HTML se convierte en un nodo.
4. JavaScript puede modificar este árbol.

```html
<html>
  <body>
    <h1>Título</h1>
    <p>Texto</p>
  </body>
</html>
```

Se convierte en:
```
document
 └─ html
     └─ body
         ├─ h1  → "Título"
         └─ p   → "Texto"
```

===============================================================================
2. EL OBJETO DOCUMENT
===============================================================================

```javascript
// document es la puerta de entrada al DOM
console.log(document);        // Todo el documento
console.log(document.title);  // Título de la pestaña
console.log(document.URL);    // URL actual
console.log(document.head);   // Etiqueta <head>
console.log(document.body);   // Etiqueta <body>
```

===============================================================================
3. DIFERENCIA: HTML vs DOM
===============================================================================

HTML es estático (el archivo original).
DOM es dinámico (lo que ves después de cambios de JS).

```html
<!-- HTML original -->
<ul id="lista">
  <li>Item 1</li>
</ul>
```

```javascript
// JavaScript modifica el DOM
let ul = document.getElementById("lista");

let nuevoLi = document.createElement("li");
nuevoLi.textContent = "Item 2 (dinámico)";
ul.appendChild(nuevoLi);

// La página ahora muestra 2 items, pero el HTML original no cambió
```

===============================================================================
4. TIPOS DE NODOS
===============================================================================

```javascript
// Nodo de elemento
console.log(document.body.nodeType);  // 1 (ELEMENT_NODE)

// Nodo de texto
console.log(document.body.firstChild?.nodeType);  // 3 (TEXT_NODE)

// Los tipos principales:
// 1 - ELEMENT_NODE: etiquetas HTML (div, p, h1)
// 3 - TEXT_NODE: texto dentro de etiquetas
// 8 - COMMENT_NODE: comentarios HTML <!-- -->

// Navegar por el DOM
console.log(document.body.children);     // Solo elementos
console.log(document.body.childNodes);   // Todos los nodos (incluye texto)
console.log(document.body.firstChild);   // Primer hijo
console.log(document.body.lastChild);    // Último hijo
console.log(document.body.parentNode);   // Padre
```
