Para manejar estructuras bidimensionales (o más),

Se llaman BUCLES ANIDADOS.

Un bucle anidado es exactamente lo que suena:
un bucle dentro de otro bucle.
El bucle exterior da una vuelta,
y por cada vuelta, el bucle interior da TODAS sus vueltas.

1. ¿CÓMO FUNCIONAN LOS BUCLES ANIDADOS?
===============================================================================

Un bucle anidado es un bucle dentro de otro bucle.

Por cada iteración del bucle exterior,
el bucle interior se ejecuta COMPLETAMENTE.

📌 **Analogía del calendario**

Imagina un calendario.

El bucle exterior son los meses (12 vueltas).
El bucle interior son los días (28-31 vueltas por cada mes).

Por cada mes, recorres todos los días de ese mes.

---

🔵 **Ejemplo 1: El reloj**

```javascript
// Horas (bucle exterior): 0 a 23
// Minutos (bucle interior): 0 a 59

for (let h = 0; h < 24; h++) {
    for (let m = 0; m < 60; m++) {
        console.log(h + ":" + m);
    }
}

// Esto imprime:
// 0:00, 0:01, ... 0:59,
// 1:00, 1:01, ... 1:59,
// ...
// 23:00, 23:01, ... 23:59

// Total: 24 x 60 = 1440 iteraciones

---

🔵 **Ejemplo 2: Tabla de multiplicar (visual)**

```javascript
console.log("=== TABLA DE MULTIPLICAR ===");

for (let i = 1; i <= 5; i++) {
    console.log("--- Tabla del " + i + " ---");

    for (let j = 1; j <= 5; j++) {
        console.log(i + " x " + j + " = " + (i * j));
    }

    console.log("");  // Línea en blanco entre tablas
}

// Resultado:
// --- Tabla del 1 ---
// 1 x 1 = 1
// 1 x 2 = 2
// ...
// --- Tabla del 2 ---
// 2 x 1 = 2
// ...

MATRICES (ARRAYS BIDIMENSIONALES)
===============================================================================

Un array bidimensional es un array de arrays.
Como una hoja de cálculo: filas y columnas.

📌 **Declaración de una matriz**

```javascript
let matriz = [
    [1, 2, 3],  // Fila 0
    [4, 5, 6],  // Fila 1
    [7, 8, 9]   // Fila 2
];

---

🔵 **Ejemplo 3: Recorrer una matriz**

```javascript
let matriz = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
];

console.log("=== MATRIZ 3x3 ===");

// Bucle exterior: recorre las FILAS
for (let fila = 0; fila < matriz.length; fila++) {
    let filaStr = "";

    // Bucle interior: recorre las COLUMNAS de cada fila
    for (let col = 0; col < matriz[fila].length; col++) {
        filaStr += matriz[fila][col] + " ";
    }

    console.log("Fila " + fila + ": " + filaStr);
}

// Resultado:
// Fila 0: 1 2 3
// Fila 1: 4 5 6
// Fila 2: 7 8 9

---

🔵 **Ejemplo 4: Sumar todos los elementos de una matriz**

```javascript
let matriz = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
];

let suma = 0;

for (let i = 0; i < matriz.length; i++) {
    for (let j = 0; j < matriz[i].length; j++) {
        suma += matriz[i][j];
    }
}

console.log("Suma total: " + suma);

// Resultado:
// Suma total: 45

PATRONES CON BUCLES ANIDADOS
===============================================================================

Los bucles anidados son excelentes para dibujar patrones.

📌 **Patrón: Cuadrado de asteriscos**

```javascript
let tamaño = 5;

for (let i = 0; i < tamaño; i++) {
    let fila = "";
    for (let j = 0; j < tamaño; j++) {
        fila += "* ";
    }
    console.log(fila);
}

// Resultado:
// * * * * *
// * * * * *
// * * * * *
// * * * * *
// * * * * *

---

📌 **Patrón: Triángulo rectángulo**

```javascript
let altura = 5;

for (let i = 1; i <= altura; i++) {
    let fila = "";
    for (let j = 1; j <= i; j++) {
        fila += "* ";
    }
    console.log(fila);
}

// Resultado:
// *
// * *
// * * *
// * * * *
// * * * * *

Nota: j <= i hace que cada fila tenga tantos asteriscos
como el número de fila.

---

📌 **Patrón: Pirámide**

```javascript
let altura = 5;

for (let i = 1; i <= altura; i++) {
    let fila = "";

    // Espacios
    for (let j = 1; j <= altura - i; j++) {
        fila += " ";
    }

    // Asteriscos
    for (let k = 1; k <= 2 * i - 1; k++) {
        fila += "*";
    }

    console.log(fila);
}

// Resultado:
//     *
//    ***
//   *****
//  *******
// *********

LA ANALOGÍA DEL PROFESOR BÚHO: EL ESTACIONAMIENTO
===============================================================================

Imagina un estacionamiento de varios pisos.

- El bucle exterior = los pisos (piso 1, piso 2, piso 3).
- El bucle interior = los espacios en cada piso.

Para contar todos los carros:
1. Vas al piso 1 → recorres todos los espacios → anotas los carros.
2. Vas al piso 2 → recorres todos los espacios → anotas los carros.
3. Vas al piso 3 → recorres todos los espacios → anotas los carros.

```javascript
let estacionamiento = [
    // Piso 0: 3 espacios
    ["🚗", "🚙", ""],
    // Piso 1: 3 espacios
    ["", "🚗", "🚕"],
    // Piso 2: 3 espacios
    ["🚙", "🚗", "🚙"]
];

console.log("=== INVENTARIO DEL ESTACIONAMIENTO ===");
let totalCarros = 0;

for (let piso = 0; piso < estacionamiento.length; piso++) {
    let carrosEnPiso = 0;

    for (let espacio = 0; espacio < estacionamiento[piso].length; espacio++) {
        if (estacionamiento[piso][espacio] !== "") {
            carrosEnPiso++;
            totalCarros++;
            console.log("Piso " + piso + ", Espacio " + espacio +
                        ": " + estacionamiento[piso][espacio]);
        } else {
            console.log("Piso " + piso + ", Espacio " + espacio + ": vacío");
        }
    }

    console.log("→ Carros en piso " + piso + ": " + carrosEnPiso);
}

console.log("Total de carros: " + totalCarros);

ERRORES COMUNES CON BUCLES ANIDADOS
===============================================================================

❌ **Error 1: Usar la misma variable en ambos bucles**

```javascript
// ❌ MAL: i se usa en ambos bucles
for (let i = 0; i < 5; i++) {
    for (let i = 0; i < 5; i++) {  // ❌ i ya está declarada
        console.log(i);
    }
}

// ✅ BIEN: variables diferentes
for (let i = 0; i < 5; i++) {
    for (let j = 0; j < 5; j++) {
        console.log(i + ", " + j);
    }
}

Solución: Usa i para el exterior, j para el interior, k para el tercero.

❌ **Error 2: Rendimiento (demasiadas iteraciones)**

```javascript
// ❌ MAL: 1000 x 1000 = 1,000,000 de iteraciones
for (let i = 0; i < 1000; i++) {
    for (let j = 0; j < 1000; j++) {
        console.log(i, j);  // Lento
    }
}

Solución: Piensa si realmente necesitas tantas iteraciones.
Para grandes volúmenes, considera otras estructuras.

❌ **Error 3: Confundir filas y columnas**

```javascript
let matriz = [
    [1, 2, 3],
    [4, 5, 6]
];

// ❌ MAL: intercambiar i y j
for (let i = 0; i < matriz[0].length; i++) {  // Columnas
    for (let j = 0; j < matriz.length; j++) {  // Filas
        console.log(matriz[j][i]);  // Cambiado
    }
}

Solución: i usualmente = filas, j = columnas.
matriz[i][j] → i es fila, j es columna.

❌ **Error 4: No entender el flujo**

```javascript
// ¿Cuántas veces se imprime "Hola"?
for (let i = 1; i <= 3; i++) {
    for (let j = 1; j <= 3; j++) {
        console.log("Hola");
    }
}
// Respuesta: 9 veces (3 x 3)

Solución: total = exterior x interior.

📝 **Ejercicio 4: Matriz identidad**

Crea una matriz identidad de tamaño N (unos en la diagonal, ceros en el resto).

Solución:
```javascript
let n = Number(prompt("Tamaño de la matriz:"));
let identidad = [];

for (let i = 0; i < n; i++) {
    identidad[i] = [];
    for (let j = 0; j < n; j++) {
        if (i === j) {
            identidad[i][j] = 1;
        } else {
            identidad[i][j] = 0;
        }
    }
}

console.log("Matriz identidad " + n + "x" + n + ":");
for (let i = 0; i < n; i++) {
    console.log(identidad[i].join(" "));
}

```javascript
// ============================================================
// TABLERO DE AJEDREZ
// Mini proyecto del Capítulo 16
// ============================================================

console.log("=== TABLERO DE AJEDREZ ===");
console.log("");

// Creamos el tablero 8x8
let tablero = [];
let tamaño = 8;
let piezasBlancas = ["♖", "♘", "♗", "♕", "♔", "♗", "♘", "♖"];
let piezasNegras = ["♜", "♞", "♝", "♛", "♚", "♝", "♞", "♜"];

// Inicializamos el tablero vacío
for (let i = 0; i < tamaño; i++) {
    tablero[i] = [];
    for (let j = 0; j < tamaño; j++) {
        tablero[i][j] = "·";  // Casilla vacía
    }
}

// Colocamos piezas blancas (fila 0 y fila 1)
for (let j = 0; j < tamaño; j++) {
    tablero[0][j] = piezasBlancas[j];  // Piezas mayores
    tablero[1][j] = "♙";              // Peones
}

// Colocamos piezas negras (fila 6 y fila 7)
for (let j = 0; j < tamaño; j++) {
    tablero[6][j] = "♟";              // Peones
    tablero[7][j] = piezasNegras[j];  // Piezas mayores
}

// Mostramos el tablero con letras y números
console.log("  a b c d e f g h");

for (let i = 0; i < tamaño; i++) {
    let filaStr = (8 - i) + " ";

    for (let j = 0; j < tamaño; j++) {
        filaStr += tablero[i][j] + " ";
    }

    filaStr += (8 - i);
    console.log(filaStr);
}

console.log("  a b c d e f g h");

console.log("");
console.log("=== TABLERO INICIAL LISTO ===");

// Contamos las piezas
let totalPiezas = 0;

for (let i = 0; i < tamaño; i++) {
    for (let j = 0; j < tamaño; j++) {
        if (tablero[i][j] !== "·") {
            totalPiezas++;
        }
    }
}

console.log("Total de piezas en el tablero: " + totalPiezas);

===============================================================================
PREGUNTAS FRECUENTES (FAQ)
===============================================================================

❓ **¿Cuántos niveles de anidamiento puedo tener?**

Tantos como necesites, pero más de 3 niveles
dificulta la lectura del código.

❓ **¿Cómo nombro las variables?**

Convención universal: i, j, k.
i para el exterior, j para el interior, k para el tercero.

❓ **¿Qué pasa si el bucle interior depende del exterior?**

Funciona perfectamente. Es un patrón común:
```javascript
for (let i = 1; i <= 5; i++) {
    for (let j = 1; j <= i; j++) {
        // j depende de i
    }
}
```

❓ **¿Los bucles anidados son lentos?**

Depende del tamaño. 10×10 = 100 iteraciones (rápido).
1000×1000 = 1,000,000 (lento pero manejable).
10000×10000 = 100,000,000 (muy lento, evita).
