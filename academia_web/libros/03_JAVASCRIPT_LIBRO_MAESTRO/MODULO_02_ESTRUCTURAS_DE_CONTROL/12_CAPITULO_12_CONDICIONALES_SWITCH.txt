===============================================================================
1. ¿QUÉ ES SWITCH?
===============================================================================

Switch es una estructura condicional que evalúa una expresión
y ejecuta el código correspondiente al valor que coincide.

📌 **Sintaxis básica**

```javascript
switch (expresion) {
    case valor1:
        // Código si expresion === valor1
        break;
    case valor2:
        // Código si expresion === valor2
        break;
    case valor3:
        // Código si expresion === valor3
        break;
    default:
        // Código si ningún caso coincide
}

---

🔵 **Ejemplo 1: Día de la semana**

```javascript
let dia = 3;
let nombreDia;

switch (dia) {
    case 1:
        nombreDia = "Lunes";
        break;
    case 2:
        nombreDia = "Martes";
        break;
    case 3:
        nombreDia = "Miércoles";
        break;
    case 4:
        nombreDia = "Jueves";
        break;
    case 5:
        nombreDia = "Viernes";
        break;
    case 6:
        nombreDia = "Sábado";
        break;
    case 7:
        nombreDia = "Domingo";
        break;
    default:
        nombreDia = "Día inválido";
}

console.log("Día:", nombreDia);

// Resultado:
// Día: Miércoles

===============================================================================
PARTES DEL SWITCH
===============================================================================

📌 **La expresión**

Va entre paréntesis después de la palabra switch.
Puede ser una variable, una operación o cualquier valor.

```javascript
switch (expresion) { ... }

📌 **Los casos (case)**

Cada case representa un valor posible.
Usa === (comparación estricta) para evaluar.

```javascript
case 1:       // Compara: expresion === 1
case "rojo":  // Compara: expresion === "rojo"

📌 **El break**

break termina el switch.
Sin break, el código "cae" al siguiente caso (fall through).

```javascript
case 1:
    console.log("Uno");
    break;  // Sin esto, seguiría al case 2

📌 **El default**

default es el "else" del switch.
Se ejecuta si ningún case coincide.

```javascript
default:
    console.log("Opción no válida");

===============================================================================
FALL THROUGH (CAÍDA ENTRE CASOS)
===============================================================================

El fall through es cuando un case no tiene break
y el código continúa ejecutando el siguiente case.

A veces es un error. A veces es intencional.

🔵 **Ejemplo 2: Error por olvidar break**

```javascript
let fruta = "manzana";

switch (fruta) {
    case "manzana":
        console.log("Manzana: $1.00");
        // ❌ Olvidé el break
    case "pera":
        console.log("Pera: $1.50");
        break;
    case "naranja":
        console.log("Naranja: $0.75");
        break;
}

// Resultado:
// Manzana: $1.00
// Pera: $1.50  <-- También se ejecutó la pera. ¡Error!

¿Por qué pasó?
Porque al no tener break después de "manzana",
el programa continuó ejecutando el código de "pera".

---

🔵 **Ejemplo 3: Fall through INTENCIONAL**

A veces QUEREMOS que varios casos compartan el mismo código.

```javascript
let nota = "B";

switch (nota) {
    case "A":
    case "B":
    case "C":
        console.log("Aprobado");
        break;
    case "D":
    case "F":
        console.log("Reprobado");
        break;
    default:
        console.log("Nota inválida");
}

// Resultado:
// Aprobado

Las notas A, B y C comparten el mismo resultado "Aprobado".
En lugar de repetir el código, agrupamos los casos.

===============================================================================
SWITCH CON STRINGS
===============================================================================

Switch no solo funciona con números.
Puedes usarlo con strings, booleanos, incluso con expresiones.

🔵 **Ejemplo 4: Semáforo con strings**

```javascript
let color = prompt("¿De qué color está el semáforo?").toLowerCase();

switch (color) {
    case "verde":
        console.log("✅ Puedes cruzar");
        break;
    case "amarillo":
        console.log("⚠️ Precaución, prepárate para detenerte");
        break;
    case "rojo":
        console.log("🛑 Detente, no cruces");
        break;
    default:
        console.log("❌ Color de semáforo inválido");
}

🦉 **Profesor Búho dice:**
"Nota cómo usamos .toLowerCase() en el prompt.
Esto convierte lo que sea que el usuario escriba a minúsculas.
Así, si escribe 'Verde', 'VERDE' o 'verde', siempre funciona.
Es una técnica de defensa contra errores del usuario."

---

🔵 **Ejemplo 5: Menú de opciones con switch**

Este es el uso más común de switch en aplicaciones reales.

```javascript
let opcion = prompt(
    "=== MENÚ PRINCIPAL ===\n" +
    "1. Ver perfil\n" +
    "2. Editar configuración\n" +
    "3. Ver historial\n" +
    "4. Cerrar sesión\n" +
    "Elige una opción:"
);

switch (opcion) {
    case "1":
        console.log("🔍 Mostrando perfil del usuario...");
        break;
    case "2":
        console.log("⚙️ Abriendo configuración...");
        break;
    case "3":
        console.log("📜 Mostrando historial...");
        break;
    case "4":
        console.log("👋 Cerrando sesión...");
        break;
    default:
        console.log("❌ Opción inválida. Intenta de nuevo.");
}

===============================================================================
SWITCH CON TRUE (RANGOS)
===============================================================================

¿Qué pasa si necesitas rangos en switch?
Switch no soporta rangos directamente.
Pero puedes usar switch(true).

🔵 **Ejemplo 6: Rangos con switch(true)**

```javascript
let edad = 25;

switch (true) {
    case (edad >= 0 && edad <= 2):
        console.log("Bebé");
        break;
    case (edad >= 3 && edad <= 12):
        console.log("Niño");
        break;
    case (edad >= 13 && edad <= 17):
        console.log("Adolescente");
        break;
    case (edad >= 18 && edad <= 64):
        console.log("Adulto");
        break;
    case (edad >= 65):
        console.log("Adulto mayor");
        break;
    default:
        console.log("Edad inválida");
}

// Resultado:
// Adulto

¿Cómo funciona?
switch(true) evalúa cada case.
Si la condición del case es true, se ejecuta ese bloque.

🦉 **Profesor Búho dice:**
"Esta técnica es avanzada. No la uses siempre.
Pero cuando la necesites, saber que existe te salvará."

LA ANALOGÍA DEL PROFESOR BÚHO: EL CONTROL REMOTO
===============================================================================

Imagina un control remoto de TV.

- Presionas el botón 1 → Canal 1 (Noticias).
- Presionas el botón 2 → Canal 2 (Deportes).
- Presionas el botón 3 → Canal 3 (Películas).
- Cualquier otro botón → "Canal no disponible".

Eso es switch.
Una expresión (el botón presionado) y múltiples casos posibles.

```javascript
let canal = 2;

switch (canal) {
    case 1:
        console.log("Viendo Noticias");
        break;
    case 2:
        console.log("Viendo Deportes");
        break;
    case 3:
        console.log("Viendo Películas");
        break;
    default:
        console.log("Canal no disponible");
}

Ahora imagina un control remoto más inteligente.

Si presionas 1, 2 o 3, ves televisión normal.
Si presionas 4, 5 o 6, ves canales de paga.
Si presionas 7, ves solo contenido infantil.

Con switch puedes agrupar:

```javascript
switch (canal) {
    case 1:
    case 2:
    case 3:
        console.log("TV abierta");
        break;
    case 4:
    case 5:
    case 6:
        console.log("TV de paga");
        break;
    case 7:
        console.log("TV infantil");
        break;
    default:
        console.log("Canal no disponible");
}

Eso es el fall through: cuando no pones break, el caso "cae"
al siguiente, como si el control remoto saltara al siguiente botón.

===============================================================================
ERRORES COMUNES CON SWITCH
===============================================================================

❌ **Error 1: Olvidar el break**

```javascript
let codigo = 1;

switch (codigo) {
    case 1:
        console.log("Código 1");
        // ❌ Sin break
    case 2:
        console.log("Código 2");
        break;
}

// Resultado:
// Código 1
// Código 2  <-- Se ejecutó sin querer

Solución: Siempre verifica que cada case tenga su break.

❌ **Error 2: Olvidar el default**

```javascript
let mes = 13;

switch (mes) {
    case 1: console.log("Enero"); break;
    case 2: console.log("Febrero"); break;
    // ... más casos ...
    case 12: console.log("Diciembre"); break;
    // ❌ No hay default
}

// Si mes es 13, no pasa nada. El usuario se queda sin respuesta.

Solución: Siempre incluye default, aunque sea solo para mostrar un error.

❌ **Error 3: Confundir switch con if en rangos**

JavaScript
```javascript
let nota = 85;

// ❌ Esto NO funciona como esperas
switch (nota) {
    case (nota >= 90):  // Esto evalúa (85 >= 90) = false
        console.log("A");
        break;
    case (nota >= 80):  // Esto evalúa (85 >= 80) = true
        console.log("B");
        break;
}

Solución: Para rangos, usa if-else if. O usa switch(true) como vimos antes.

❌ **Error 4: Tipos mixtos**

Switch usa === (comparación estricta).

```javascript
let valor = "1";

switch (valor) {
    case 1:  // ❌ Esto es ===, "1" === 1 es false
        console.log("Número uno");
        break;
    case "1":  // ✅ Esto funciona
        console.log("String uno");
        break;
}

// Resultado:
// String uno

Solución: Asegúrate de que los tipos coincidan.
Si prompt() devuelve string, usa case "1", no case 1.

MINI PROYECTO: CONVERTIDOR DE DIVISAS
===============================================================================

Vamos a construir un conversor de divisas simple.

El programa:
1. Muestra un menú con 4 divisas.
2. Pide al usuario que elija una opción.
3. Pide la cantidad en dólares.
4. Muestra la conversión.

```javascript
// ============================================================
// CONVERTIDOR DE DIVISAS
// Mini proyecto del Capítulo 12
// ============================================================

console.log("=== CONVERTIDOR DE DIVISAS ===");

let mensaje = `
=== DIVISAS DISPONIBLES ===
1. Euro (EUR)     - 1 USD = 0.92 EUR
2. Libra (GBP)    - 1 USD = 0.79 GBP
3. Yen (JPY)      - 1 USD = 149.50 JPY
4. Peso MX (MXN)  - 1 USD = 17.25 MXN
5. Salir
`;

let opcion = prompt(mensaje + "\nElige una opción (1-5):");

if (opcion === null) {
    console.log("Programa cancelado por el usuario.");
    alert("Operación cancelada.");
} else {
    opcion = Number(opcion);

    switch (opcion) {
        case 1:
        case 2:
        case 3:
        case 4:
            let dolares = prompt("Ingresa la cantidad en dólares (USD):");

            if (dolares === null) {
                console.log("Programa cancelado.");
                alert("Operación cancelada.");
            } else {
                dolares = Number(dolares);

                if (isNaN(dolares) || dolares < 0) {
                    console.log("❌ Cantidad inválida.");
                    alert("❌ Debes ingresar un número válido y positivo.");
                } else {
                    let resultado;
                    let divisa;
                    let tasa;

                    switch (opcion) {
                        case 1:
                            tasa = 0.92;
                            divisa = "EUR (Euros)";
                            break;
                        case 2:
                            tasa = 0.79;
                            divisa = "GBP (Libras)";
                            break;
                        case 3:
                            tasa = 149.50;
                            divisa = "JPY (Yenes)";
                            break;
                        case 4:
                            tasa = 17.25;
                            divisa = "MXN (Pesos Mexicanos)";
                            break;
                    }

                    resultado = dolares * tasa;

                    console.log("💵 $" + dolares + " USD = " + resultado.toFixed(2) + " " + divisa);
                    alert("💵 $" + dolares + " USD = " + resultado.toFixed(2) + " " + divisa);
                }
            }
            break;
        case 5:
            console.log("👋 Saliendo del conversor...");
            alert("¡Gracias por usar el conversor!");
            break;
        default:
            console.log("❌ Opción inválida.");
            alert("❌ Opción inválida. Intenta de nuevo.");
    }
}

console.log("=== FIN DEL PROGRAMA ===");

===============================================================================
PREGUNTAS FRECUENTES (FAQ)
===============================================================================

❓ **¿Switch o if? ¿Cuál es mejor?**

Ninguno es mejor. Depende del contexto.
Switch es más legible para muchas opciones fijas.
If es más flexible para rangos y condiciones complejas.

❓ **¿Qué pasa si olvido todos los breaks?**

Todo el switch se ejecuta completo, desde el primer case que coincida
hasta el final. Es el error más común con switch.

❓ **¿Puedo usar switch con booleanos?**

Sí. switch(true) es una técnica para usar rangos con switch.

❓ **¿El default es obligatorio?**

No, pero es una buena práctica incluirlo.

❓ **¿Puedo poner un switch dentro de otro switch?**

Sí. Se llaman switch anidados. Pero úsalos con cuidado.

Conceptos relacionados:
if, else if, operadores de comparación, operadores lógicos.

Errores frecuentes:
1. Olvidar break (fall through accidental).
2. No incluir default.
3. Confundir switch con if para rangos.
4. Problemas de tipo (string vs número).

Analogías utilizadas:
1. Control remoto de TV (cada botón es un case).
2. Guardia de seguridad (agrupación de permisos).

- ¿Qué hace default?
