EL PROYECTO: ADIVINA EL NÚMERO
===============================================================================

Vamos a construir un juego donde:
1. La computadora piensa un número del 1 al 100.
2. El usuario intenta adivinarlo.
3. El programa da pistas ("mayor" o "menor").
4. El usuario tiene un límite de intentos.
5. Al final, muestra un resumen del juego.

---

📌 **Flujo del programa**

┌─────────────────────────────────────────────────────┐
│ Inicio                                               │
│ ↓                                                    │
│ Generar número aleatorio (1-100)                     │
│ ↓                                                    │
│ Mostrar reglas y límite de intentos                  │
│ ↓                                                    │
│ ┌─── Bucle de juego ---                              │
│ │ Pedir número al usuario                            │
│ │ Validar entrada (try...catch)                      │
│ │ Si es inválido → mostrar error → continuar         │
│ │ Si es correcto → ¡Felicidades! → salir del bucle  │
│ │ Si es mayor → pista "menor"                        │
│ │ Si es menor → pista "mayor"                        │
│ │ Intentos++                                         │
│ │ Si sin intentos → "Game Over" → salir del bucle   │
│ └────────────────────────────────────────────────────┘
│ ↓                                                    │
│ Mostrar resumen del juego                            │
│ ↓                                                    │
│ Fin                                                  │
└─────────────────────────────────────────────────────┘

PASO 2: CÓDIGO COMPLETO

```javascript
// ============================================================
// ADIVINA EL NÚMERO
// Proyecto del Módulo II - JavaScript Master Book
// ============================================================

// ============================================================
// FUNCIONES AUXILIARES
// ============================================================

function generarNumero(min, max) {
    // Genera un número entero aleatorio entre min y max (inclusive)
    return Math.floor(Math.random() * (max - min + 1)) + min;
}

function darPista(numeroSecreto, suposicion) {
    // Compara y devuelve la pista correspondiente
    if (suposicion < numeroSecreto) {
        return "mayor";
    } else if (suposicion > numeroSecreto) {
        return "menor";
    } else {
        return "correcto";
    }
}

function validarEntrada(entrada) {
    // Valida que la entrada sea un número válido entre 1 y 100
    let numero = Number(entrada);

    if (isNaN(numero)) {
        throw new Error("Debes ingresar un número válido.");
    }
    if (!Number.isInteger(numero)) {
        throw new Error("Debes ingresar un número entero.");
    }
    if (numero < 1 || numero > 100) {
        throw new Error("El número debe estar entre 1 y 100.");
    }

    return numero;
}

function mostrarResumen(nombre, historial, adivinado, secreto) {
    // Muestra el resumen final del juego
    console.log("");
    console.log("========== RESUMEN DEL JUEGO ==========");
    console.log("👤 Jugador: " + nombre);
    console.log("🎯 Número secreto: " + secreto);
    console.log("📊 Intentos realizados: " + historial.length);

    if (adivinado) {
        console.log("🏆 ¡Ganaste!");
    } else {
        console.log("😢 Game Over");
    }

    console.log("");
    console.log("📝 Historial de intentos:");

    for (let i = 0; i < historial.length; i++) {
        let intento = historial[i];
        console.log("   " + (i + 1) + ". " + intento.numero +
                    " → " + intento.resultado);
    }

    console.log("========================================");
}

// ============================================================
// FUNCIÓN PRINCIPAL
// ============================================================

function iniciarJuego() {
    console.log("========================================");
    console.log("       ADIVINA EL NÚMERO");
    console.log("========================================");
    console.log("");

    // Configuración del juego
    let min = 1;
    let max = 100;
    let maxIntentos = 7;
    let numeroSecreto = generarNumero(min, max);
    let intentos = 0;
    let adivinado = false;
    let historial = [];

    // Pedir nombre del jugador
    let nombreUsuario = prompt("¿Cuál es tu nombre?") || "Jugador";

    console.log("🎯 " + nombreUsuario + ", he pensado un número entre " +
                min + " y " + max + ".");
    console.log("💡 Tienes " + maxIntentos + " intentos para adivinarlo.");
    console.log("");

    // Bucle principal del juego
    while (intentos < maxIntentos && !adivinado) {
        intentos++;
        console.log("--- Intento " + intentos + " de " + maxIntentos + " ---");

        try {
            // Pedir número
            let entrada = prompt("¿Qué número crees que es?");
            if (entrada === null) {
                console.log("⚠️ Juego cancelado.");
                return;  // Sale de la función
            }

            // Validar entrada
            let suposicion = validarEntrada(entrada);

            // Obtener pista
            let pista = darPista(numeroSecreto, suposicion);

            // Registrar en historial
            historial.push({
                numero: suposicion,
                resultado: pista
            });

            // Evaluar resultado
            if (pista === "correcto") {
                console.log("🎉 ¡CORRECTO! " + nombreUsuario +
                            ", adivinaste el número " + numeroSecreto +
                            " en " + intentos + " intentos.");
                adivinado = true;
            } else if (pista === "mayor") {
                console.log("⬆️ El número es MAYOR que " + suposicion +
                            ". Llevas " + intentos + " intento(s).");
            } else {
                console.log("⬇️ El número es MENOR que " + suposicion +
                            ". Llevas " + intentos + " intento(s).");
            }

        } catch (error) {
            // Manejo de errores de validación
            console.log("❌ " + error.message);
            console.log("💡 Pista: " + error.message);
            intentos--;  // No contamos intentos inválidos
        }

        // Mostrar intentos restantes
        if (!adivinado) {
            let restantes = maxIntentos - intentos;
            if (restantes > 0) {
                console.log("   Intentos restantes: " + restantes);
            } else {
                console.log("   ❌ ¡Sin intentos restantes!");
            }
        }
        console.log("");
    }

    // Si no adivinó después de agotar intentos
    if (!adivinado) {
        console.log("😢 Game Over, " + nombreUsuario + ".");
        console.log("El número secreto era: " + numeroSecreto);
    }

    // Mostrar resumen
    mostrarResumen(nombreUsuario, historial, adivinado, numeroSecreto);

    // Preguntar si quiere jugar de nuevo
    let jugarDeNuevo = confirm("¿Quieres jugar de nuevo?");

    if (jugarDeNuevo) {
        console.log("🔄 Reiniciando juego...");
        console.log("");
        iniciarJuego();  // Recursión: se llama a sí misma
    } else {
        console.log("👻 ¡Gracias por jugar, " + nombreUsuario + "!");
    }
}

// ============================================================
// INICIAR EL JUEGO
// ============================================================

iniciarJuego();
```

PASO 3: EXPLICACIÓN LÍNEA POR LÍNEA
===============================================================================

📌 **1. Función generarNumero(min, max)**

```javascript
return Math.floor(Math.random() * (max - min + 1)) + min;
```

Math.random() genera un decimal entre 0 y 0.999...
Multiplicamos por (max - min + 1) para escalarlo.
Math.floor redondea hacia abajo.
Sumamos min para ajustar el rango.

📌 **2. Función darPista(secreto, suposicion)**

Compara la suposición con el secreto.
Devuelve "mayor", "menor" o "correcto".
Simple y clara: una responsabilidad única.

📌 **3. Función validarEntrada(entrada)**

Recibe un string del prompt.
Lo convierte a número con Number().
Lanza errores específicos con throw new Error().
Cada error tiene un mensaje claro para el usuario.

📌 **4. Función mostrarResumen()**

Usa un bucle for para recorrer el historial.
Muestra cada intento con su resultado.
Usa template strings para formatear la salida.

📌 **5. El bucle principal**

```javascript
while (intentos < maxIntentos && !adivinado) {
```

Dos condiciones: que queden intentos Y que no haya adivinado.
Si alguna falla, el bucle termina.

📌 **6. Try...catch en el bucle**

```javascript
try {
    let suposicion = validarEntrada(entrada);
    // ...
} catch (error) {
    intentos--;  // No contar intentos inválidos
}
```

Si el usuario ingresa "abc", validarEntrada lanza un error.
El catch lo atrapa, muestra el mensaje y resta el intento.

📌 **7. Historial de intentos**

```javascript
historial.push({
    numero: suposicion,
    resultado: pista
});
```

Cada intento se guarda como un objeto con dos propiedades.
Al final, se recorre el array para mostrar el resumen.

📌 **8. Reinicio del juego**

```javascript
if (jugarDeNuevo) {
    iniciarJuego();  // Recursión
}
```

La función se llama a sí misma para reiniciar el juego.
Es una forma elegante de reiniciar sin recargar la página.

PREGUNTAS FRECUENTES (FAQ)
===============================================================================

❓ **¿Por qué es importante validar la entrada?**

Porque los usuarios cometen errores.
Un programa robusto debe manejar entradas inválidas
sin romperse ni dar resultados incorrectos.

❓ **¿Qué hace Math.floor(Math.random() * 100) + 1?**

Genera un número entero aleatorio entre 1 y 100.
Math.random() da un decimal (0-0.999...).
Math.floor redondea hacia abajo.
+1 ajusta el rango para que empiece en 1.

❓ **¿Por qué usamos funciones?**

Para organizar el código en bloques reutilizables.
Cada función tiene una responsabilidad única.

❓ **¿Qué es la recursión?**

Cuando una función se llama a sí misma.
En el proyecto, iniciarJuego() se llama para reiniciar.
