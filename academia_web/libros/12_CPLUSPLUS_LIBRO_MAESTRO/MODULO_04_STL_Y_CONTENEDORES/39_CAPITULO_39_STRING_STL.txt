Bienvenido al capítulo sobre std::string, la clase de cadenas de
la STL. std::string es mucho mas que un reemplazo de char*: ofrece
un conjunto completo de operaciones para manipulacion de texto,
busqueda, extraccion, conversion y mas. Es uno de los tipos mas
usados en C++ moderno.

std::string es una clase que gestiona secuencias de caracteres.
Internamente almacena los caracteres en memoria contigua (como
vector<char>) y proporciona una rica interfaz de manipulacion.

La cabecera necesaria es <string>.

Declaracion e inicializacion:
  #include <string>

  std::string s1;                     // cadena vacia
  std::string s2 = "Hola";           // desde literal
  std::string s3("Mundo");           // desde constructor
  std::string s4 = s2 + " " + s3;   // concatenacion
  std::string s5(10, 'x');           // "xxxxxxxxxx"
  std::string s6(s2, 1, 3);          // "ola" (desde pos 1, 3 chars)
  std::string s7(s2.begin(), s2.end()); // desde iteradores

  // Inicializacion con lista
  std::string s8 {'H', 'o', 'l', 'a'};

  // C++17: string_view (vista sin copia)
  std::string_view sv = s2;  // vista de s2

Tamano y capacidad:
  s.size();       // numero de caracteres
  s.length();     // igual que size
  s.capacity();   // capacidad actual
  s.reserve(n);   // reserva espacio
  s.resize(n);    // cambia tamano
  s.resize(n, c); // cambia tamano con caracter de relleno
  s.shrink_to_fit();  // reduce capacidad a size
  s.empty();      // true si esta vacia
  s.max_size();   // maximo teorico

  s.clear();      // elimina todos los caracteres

Acceso a caracteres:
  s[0] = 'H';          // sin verificacion
  s.at(0) = 'h';       // con verificacion
  char& front = s.front();  // primer caracter
  char& back = s.back();    // ultimo caracter
  const char* cstr = s.c_str();  // puntero a arreglo estilo C
  const char* data = s.data();   // puntero a datos (sin null en C++17)

  s[100];  // UB si el indice es mayor que size
  s.at(100);  // lanza std::out_of_range

Asignacion:
  s = "nuevo texto";
  s.assign("otro texto");
  s.assign("texto", 3);  // "tex"
  s.assign(5, 'z');      // "zzzzz"
  s.assign({"a", "b", "c"});  // desde initializer list

Concatenacion:
  std::string a = "Hola";
  std::string b = " Mundo";

  std::string c = a + b;        // "Hola Mundo"
  std::string d = a + " C++";   // con literal

  a += b;   // append

  a.push_back('!');  // agrega un caracter
  a.pop_back();      // elimina el ultimo caracter

  // append con subcadena

Insercion:
  std::string s = "Hola";

  s.insert(1, "xxx");     // "Hxxxola"
  s.insert(s.begin(), '!'); // "!Hxxxola"
  s.insert(s.end(), '!');   // "!Hxxxola!"
  s.insert(s.begin() + 2, 3, 'y');  // "!Hyyyxxxola!"

  // insert con iteradores
  std::string otra = "Mundo";
  s.insert(s.begin(), otra.begin(), otra.end());

Extraccion (substr):
  std::string s = "Hola Mundo C++";

  std::string sub = s.substr(5, 5);   // "Mundo"
  std::string sub2 = s.substr(5);     // "Mundo C++" (hasta final)
  std::string sub3 = s.substr(0, 4);  // "Hola"

  // Si la posicion es mayor que size, lanza out_of_range

Eliminacion (erase):
  std::string s = "Hola Mundo";

  s.erase(4, 1);          // elimina 1 char en pos 4: "HolaMundo"
  s.erase(s.begin() + 4); // elimina el caracter en posicion 4
  s.erase(s.begin() + 4, s.begin() + 9); // rango
  s.erase();  // equivale a clear

REEMPLAZO (replace):

  s.replace(0, 4, "Adios");    // "Adios Mundo"
  s.replace(6, 5, "Planeta");  // "Adios Planeta"

  // replace con iteradores
  s.replace(s.begin(), s.begin() + 5, "Hola");

  // replace con subcadena de otra

  // replace con repeticion

BUSQUEDA (find y variantes):

  std::string s = "Hola Mundo C++ Mundo";

  // find: primera ocurrencia
  size_t pos = s.find("Mundo");  // 5
  pos = s.find("Mundo", 7);      // buscar desde posicion 7: 17
  pos = s.find('C');             // 11

  // rfind: ultima ocurrencia
  pos = s.rfind("Mundo");   // 17
  pos = s.rfind('o');       // 18

  // find_first_of: primer caracter de un conjunto
  pos = s.find_first_of("aeiou");  // primera vocal: 1 ('o')
  pos = s.find_first_of("aeiou", 3);  // desde posicion 3

  // find_last_of: ultimo caracter de un conjunto
  pos = s.find_last_of("aeiou");  // 18 ('o')

  // find_first_not_of: primer caracter NO en conjunto
  pos = s.find_first_not_of("Hola ");  // 5 ('M')

  // find_last_not_of: ultimo caracter NO en conjunto
  pos = s.find_last_not_of("Mundo ");  // 10 ('+')

  // Si no encuentra, devuelve std::string::npos
  if (pos == std::string::npos)
      std::cout << "No encontrado\n";

Join de vector a string:

  std::string join(const std::vector<std::string>& v,
                   const std::string& delim) {
      if (v.empty()) return "";
      std::string result = v[0];
      for (size_t i = 1; i < v.size(); ++i)
          result += delim + v[i];
      return result;
  }

  // Uso:

Trim (eliminar espacios al inicio/final):

  std::string trim(const std::string& s) {
      auto start = s.find_first_not_of(" \t\n\r");
      if (start == std::string::npos) return "";
      auto end = s.find_last_not_of(" \t\n\r");
      return s.substr(start, end - start + 1);
  }

  // Uso:

Rendimiento de std::string:
- s = s + x: O(n) crea nueva cadena, copia ambas.
- s += x: O(n) pero puede reutilizar capacidad.
- s.append(x): similar a +=.
- Usar reserve() y += para multiples concatenaciones.

  // Ineficiente (crea muchas temporales)
  std::string s = "";
  for (int i = 0; i < 1000; ++i)
      s = s + std::to_string(i);  // O(n^2)

  // Eficiente
  std::string s;
  s.reserve(4000);  // estimacion
  for (int i = 0; i < 1000; ++i)
      s += std::to_string(i);  // O(n)

Ejemplo completo: Procesador de texto.

  #include <iostream>
  #include <string>
  #include <vector>
  #include <sstream>
  #include <algorithm>
  #include <cctype>
  #include <fstream>

  class ProcesadorTexto {
  public:
      static std::string mayusculas(const std::string& s) {
          std::string r = s;
          std::transform(r.begin(), r.end(), r.begin(), ::toupper);
          return r;
      }

      static std::string minusculas(const std::string& s) {
          std::string r = s;
          std::transform(r.begin(), r.end(), r.begin(), ::tolower);
          return r;
      }

      static std::string capitalizar(const std::string& s) {
          std::string r = s;
          if (!r.empty()) r[0] = std::toupper(r[0]);
          for (size_t i = 1; i < r.size(); ++i) {
              if (r[i-1] == ' ')
                  r[i] = std::toupper(r[i]);
              else
                  r[i] = std::tolower(r[i]);
          }
          return r;
      }

      static int contarPalabras(const std::string& s) {
          int count = 0;
          bool inWord = false;
          for (char c : s) {
              if (std::isspace(c)) {
                  inWord = false;
              } else if (!inWord) {
                  inWord = true;
                  ++count;
              }
          }
          return count;
      }

      static std::string invertir(const std::string& s) {
          std::string r = s;
          std::reverse(r.begin(), r.end());
          return r;
      }

      static bool esPalindromo(const std::string& s) {
          std::string limpia;
          std::copy_if(s.begin(), s.end(),
                      std::back_inserter(limpia),
                      [](char c) { return std::isalnum(c); });
          std::transform(limpia.begin(), limpia.end(),
                        limpia.begin(), ::tolower);
          std::string rev = limpia;
          std::reverse(rev.begin(), rev.end());
          return limpia == rev;
      }

      static std::string reemplazarPalabra(
          const std::string& s,
          const std::string& de,
          const std::string& a) {
          std::string r = s;
          size_t pos = 0;
          while ((pos = r.find(de, pos)) != std::string::npos) {
              r.replace(pos, de.length(), a);
              pos += a.length();
          }
          return r;
      }

      static std::vector<std::string> split(
          const std::string& s, char delim) {
          std::vector<std::string> tokens;
          std::string token;
          std::istringstream iss(s);
          while (std::getline(iss, token, delim))
              if (!token.empty())
                  tokens.push_back(token);
          return tokens;
      }
  };

```cpp
//==============================================================
// EJEMPLO COMPLETO: SISTEMA DE ANALISIS DE TEXTO
//==============================================================
#include <iostream>
#include <string>
#include <vector>
#include <sstream>
#include <algorithm>
#include <cctype>
#include <map>
#include <iomanip>

class AnalizadorTexto {
private:
    std::string texto;
    std::vector<std::string> lineas;
    std::vector<std::string> palabras;

    void tokenizar() {
        lineas.clear();
        palabras.clear();

        std::istringstream iss(texto);
        std::string linea;
        while (std::getline(iss, linea))
            lineas.push_back(linea);

        std::istringstream iss2(texto);
        std::string palabra;
        while (iss2 >> palabra)
            palabras.push_back(palabra);
    }

public:
    void cargarTexto(const std::string& t) {
        texto = t;
        tokenizar();
    }

    void cargarArchivo(const std::string& nombre) {
        std::ifstream archivo(nombre);
        if (!archivo) {
            std::cerr << "Error al abrir archivo\n";
            return;
        }
        std::stringstream buffer;
        buffer << archivo.rdbuf();
        texto = buffer.str();
        tokenizar();
    }

    // Estadisticas basicas
    size_t numCaracteres() const { return texto.size(); }
    size_t numLineas() const { return lineas.size(); }
    size_t numPalabras() const { return palabras.size(); }

    size_t numCaracteresSinEspacios() const {
        return std::count_if(texto.begin(), texto.end(),
            [](char c) { return !std::isspace(c); });
    }

    std::string palabraMasLarga() const {
        if (palabras.empty()) return "";
        auto it = std::max_element(palabras.begin(), palabras.end(),
            [](const std::string& a, const std::string& b) {
                return a.size() < b.size();
            });
        return *it;
    }

    double longitudPromedio() const {
        if (palabras.empty()) return 0.0;
        size_t total = 0;
        for (const auto& p : palabras)
            total += p.size();
        return static_cast<double>(total) / palabras.size();
    }

    // Frecuencia de palabras
    std::map<std::string, int> frecuenciaPalabras() const {
        std::map<std::string, int> freq;
        for (const auto& p : palabras) {
            std::string lower = p;
            std::transform(lower.begin(), lower.end(),
                          lower.begin(), ::tolower);
            // Limpiar puntuacion
            lower.erase(
                std::remove_if(lower.begin(), lower.end(),
                    [](char c) { return std::ispunct(c); }),
                lower.end());
            if (!lower.empty())
                freq[lower]++;
        }
        return freq;
    }

    std::vector<std::pair<std::string, int>> topPalabras(int n) const {
        auto freq = frecuenciaPalabras();
        std::vector<std::pair<std::string, int>> ranking(
            freq.begin(), freq.end());
        std::partial_sort(ranking.begin(),
            ranking.begin() + std::min(n, (int)ranking.size()),
            ranking.end(),
            [](const auto& a, const auto& b) {
                return a.second > b.second;
            });
        if (ranking.size() > n)
            ranking.resize(n);
        return ranking;
    }

    // Busqueda
    std::vector<int> buscarLineas(const std::string& patron) const {
        std::vector<int> resultado;
        for (size_t i = 0; i < lineas.size(); ++i) {
            if (lineas[i].find(patron) != std::string::npos)
                resultado.push_back(i);
        }
        return resultado;
    }

    int contarOcurrencias(const std::string& patron) const {
        int count = 0;
        size_t pos = 0;
        while ((pos = texto.find(patron, pos)) != std::string::npos) {
            ++count;
            pos += patron.size();
        }
        return count;
    }

    // Transformaciones
    std::string aMayusculas() const {
        std::string r = texto;
        std::transform(r.begin(), r.end(), r.begin(), ::toupper);
        return r;
    }

    std::string aMinusculas() const {
        std::string r = texto;
        std::transform(r.begin(), r.end(), r.begin(), ::tolower);
        return r;
    }

    std::string reemplazar(const std::string& de,
                           const std::string& a) {
        std::string r = texto;
        size_t pos = 0;
        while ((pos = r.find(de, pos)) != std::string::npos) {
            r.replace(pos, de.length(), a);
            pos += a.length();
        }
        return r;
    }

    std::string invertirPalabras() const {
        std::vector<std::string> words = palabras;
        for (auto& w : words)
            std::reverse(w.begin(), w.end());
        std::string resultado;
        for (const auto& w : words)
            resultado += w + " ";
        if (!resultado.empty())
            resultado.pop_back();
        return resultado;
    }

    // Reporte completo
    void reporteCompleto() const {
        std::cout << "\n========== REPORTE DE TEXTO ==========\n";
        std::cout << "Caracteres: " << numCaracteres() << '\n';
        std::cout << "Caracteres (sin espacios): "
                  << numCaracteresSinEspacios() << '\n';
        std::cout << "Lineas: " << numLineas() << '\n';
        std::cout << "Palabras: " << numPalabras() << '\n';
        std::cout << "Palabra mas larga: '"
                  << palabraMasLarga() << "' ("
                  << palabraMasLarga().size() << " letras)\n";
        std::cout << "Longitud promedio: "
                  << std::fixed << std::setprecision(2)
                  << longitudPromedio() << '\n';
        std::cout << "\nTop 10 palabras mas frecuentes:\n";
        auto top = topPalabras(10);
        for (size_t i = 0; i < top.size(); ++i)
            std::cout << (i+1) << ". " << top[i].first
                      << " (" << top[i].second << ")\n";
        std::cout << "=======================================\n";
    }
};

int main() {
    AnalizadorTexto at;

    at.cargarTexto(
        "C++ es un lenguaje de programacion diseniado por Bjarne "
        "Stroustrup. C++ es conocido por su eficiencia y flexibilidad. "
        "La STL de C++ proporciona contenedores, iteradores y algoritmos "
        "que hacen el codigo mas seguro y expresivo. "
        "C++ se usa en sistemas operativos, juegos y aplicaciones "
        "de alto rendimiento. Aprender C++ es una inversion que "
        "vale la pena para cualquier programador."
    );

    at.reporteCompleto();

    std::cout << "\nLineas que contienen 'C++':\n";
    auto lineas = at.buscarLineas("C++");
    for (int l : lineas)
        std::cout << "  Linea " << l << "\n";

    std::cout << "\nOcurrencias de 'C++': "
              << at.contarOcurrencias("C++") << '\n';

    std::cout << "\nTexto en mayusculas (primeros 100 chars):\n";
    std::cout << at.aMayusculas().substr(0, 100) << "...\n";

    std::cout << "\nReemplazando 'C++' por 'CPP':\n";
    std::cout << at.reemplazar("C++", "CPP").substr(0, 100) << "...\n";

    return 0;
}
```

Puntos clave sobre std::string:

1. Es un contenedor de caracteres con interfaz rica.
2. size() y length() para tamano; capacity() para capacidad.
3. find, rfind, find_first_of, find_last_of para busqueda.
4. substr(pos, len) para extraer subcadenas.
5. append, +=, insert para construccion.
6. replace para reemplazar partes.
7. erase y clear para eliminacion.
8. c_str() para interop con APIs C.
9. stoi, stol, stod, to_string para conversion.
10. getline para leer lineas completas.
11. Operadores <, >, == para comparacion lexicografica.

================================================================================
  PREGUNTAS FRECUENTES (FAQ)
================================================================================

P: \?Que devuelve find si no encuentra el patron?
R: std::string::npos, que es un valor grande (generalmente -1
   convertido a size_t). Siempre comparar con npos.

P: \?Es std::string null-terminated?
R: SI, c_str() garantiza terminacion nula. En C++11, data()
   tambien es null-terminated. El string puede contener nulls
   internos, pero c_str() siempre termina en null.

P: \?Cual es la diferencia entre size() y length()?
R: Ninguna. Son sinonimos. length() existe por compatibilidad
   historica con la tradicion de cadenas. size() porque string
   es un contenedor.

P: \?Como convierto int a string con formato?
R: Usa std::to_string() para conversion basica. Para formato
   controlado, usa std::ostringstream con std::setprecision,
   std::fixed, etc.

P: \?Por que s = s + x es mas lento que s += x?
R: s = s + x crea un temporal (copia s), concatena x, y asigna.
   s += x agrega directamente a la capacidad existente.

P: \?Puedo usar string con algoritmos STL?
R: SI. string tiene begin()/end() que devuelven iteradores de
   acceso aleatorio, compatible con sort, find, transform, etc.

P: \?Cual es la diferencia entre s.find('x') y s.find("x")?
R: La primera busca un solo caracter, la segunda busca una
   cadena de longitud 1. El resultado es el mismo.

Disena e implementa un "Motor de Busqueda de Texto" usando
std::string y algoritmos STL. El sistema debe:

1. Indexar documentos (vector de strings):
   - Cada documento tiene titulo y contenido.
   - Almacenar en struct { string titulo; string contenido; }.
   - Usar vector<Documento>.

2. Indexacion:
   - Tokenizar cada documento en palabras.
   - Construir un map<string, set<int>> indiceInvertido:
     palabra -> conjunto de IDs de documentos.
   - Normalizar palabras (minusculas, sin puntuacion).

3. Busqueda:
   - Busqueda por palabra exacta (find en indice).
   - Busqueda por frase exacta (find en contenido).
   - Busqueda parcial (find_first_of).
   - Busqueda AND (interseccion de sets).
   - Busqueda OR (union de sets).
   - Busqueda con exclusion (diferencia de sets).

4. Ranking de resultados:
   - Por frecuencia de la palabra en el documento.
   - Por cantidad de palabras coincidentes.
   - Por posicion de la primera coincidencia.
   - Usar partial_sort para top N.

5. Sugerencias:
   - Si una palabra no se encuentra, sugerir palabras
     similares (distancia de Levenshtein simple).
   - Usar find_first_of para palabras con errores
     ortograficos comunes.

6. Interfaz:
   - Menu interactivo (REPL).
   - Comandos: buscar, buscar_frase, sugerir, top, indexar, salir.

- Sin bibliotecas externas (solo STL).
