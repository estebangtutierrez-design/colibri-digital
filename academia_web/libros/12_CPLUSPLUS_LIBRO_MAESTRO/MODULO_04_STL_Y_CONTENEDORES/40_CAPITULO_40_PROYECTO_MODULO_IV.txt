ESPECIFICACIONES DEL SISTEMA

Vamos a construir un "Sistema de Gestion de Contactos Empresarial"
con las siguientes capacidades:

1. Gestion de contactos (nombre, email, telefono, empresa, cargo).
2. Agenda personal con grupos (familia, trabajo, amigos, etc.).
3. Historial de interacciones con cada contacto.
4. Busqueda avanzada con multiples criterios.
5. Importacion/exportacion de contactos.
6. Estadisticas y reportes.
7. Deshacer/Rehacer operaciones.

ARQUITECTURA DEL SISTEMA

Contenedores utilizados:
- std::vector<Contacto>: almacenamiento principal de contactos.
- std::map<std::string, int>: indice por email -> ID.
- std::multimap<std::string, int>: indice por empresa -> IDs.
- std::set<std::string>: conjunto de grupos.
- std::map<int, std::set<std::string>>: grupos por contacto.
- std::stack<Snapshot>: deshacer operaciones.

  class GestorContactos {
  private:
      std::vector<Contacto> contactos;
      int nextId;

  public:
      GestorContactos() : nextId(1) {}

      int agregarContacto(const std::string& nombre,
                          const std::string& email,
                          const std::string& telefono,
                          const std::string& empresa,
                          const std::string& cargo,
                          int prioridad = 3) {
          contactos.push_back({
              nextId++, nombre, email, telefono,
              empresa, cargo, false, prioridad
          });
          return nextId - 1;
      }

      bool eliminarContacto(int id) {
          auto it = std::find_if(contactos.begin(), contactos.end(),
              [id](const Contacto& c) { return c.id == id; });
          if (it != contactos.end()) {
              contactos.erase(it);
              return true;
          }
          return false;
      }

      bool actualizarContacto(int id, const Contacto& nuevos) {
          auto it = std::find_if(contactos.begin(), contactos.end(),
              [id](const Contacto& c) { return c.id == id; });
          if (it != contactos.end()) {
              nuevos.id = id;  // preservar ID
              *it = nuevos;
              return true;
          }
          return false;
      }

      Contacto* buscarPorId(int id) {
          auto it = std::find_if(contactos.begin(), contactos.end(),
              [id](const Contacto& c) { return c.id == id; });
          if (it != contactos.end())
              return &(*it);
          return nullptr;
      }

      std::vector<Contacto> buscarPorNombre(
          const std::string& nombre) {
          std::vector<Contacto> resultado;
          std::copy_if(contactos.begin(), contactos.end(),
              std::back_inserter(resultado),
              [&nombre](const Contacto& c) {
                  return c.nombre.find(nombre) != std::string::npos;
              });
          return resultado;
      }

      void ordenarPorNombre() {
          std::sort(contactos.begin(), contactos.end(),
              [](const Contacto& a, const Contacto& b) {
                  return a.nombre < b.nombre;
              });
      }

      void ordenarPorPrioridad() {
          std::sort(contactos.begin(), contactos.end(),
              [](const Contacto& a, const Contacto& b) {
                  if (a.prioridad != b.prioridad)
                      return a.prioridad > b.prioridad;
                  return a.nombre < b.nombre;
              });
      }

      void marcarFavorito(int id, bool fav) {
          auto* c = buscarPorId(id);
          if (c) c->favorito = fav;
      }

      std::vector<Contacto> favoritos() const {
          std::vector<Contacto> resultado;
          std::copy_if(contactos.begin(), contactos.end(),
              std::back_inserter(resultado),
              [](const Contacto& c) { return c.favorito; });
          return resultado;
      }

      const std::vector<Contacto>& todos() const {
          return contactos;
      }

      size_t total() const { return contactos.size(); }

      Snapshot crearSnapshot() const {
          return { contactos, "" };
      }

      void restaurarSnapshot(const Snapshot& snap) {
          contactos = snap.contactos;
      }
  };

  class IndicesBusqueda {
  private:
      std::map<std::string, int> porEmail;     // email -> id
      std::multimap<std::string, int> porEmpresa; // empresa -> ids
      std::map<int, std::string> porId;         // id -> email

  public:
      void indexar(const Contacto& c) {
          porEmail[c.email] = c.id;
          porEmpresa.insert({c.empresa, c.id});
          porId[c.id] = c.email;
      }

      void reindexar(const std::vector<Contacto>& contactos) {
          porEmail.clear();
          porEmpresa.clear();
          porId.clear();
          for (const auto& c : contactos)
              indexar(c);
      }

      void eliminar(int id) {
          auto it = porId.find(id);
          if (it != porId.end()) {
              porEmail.erase(it->second);
              // Eliminar de multimap por empresa (buscar manual)
              // Es mas simple reindexar
              porId.erase(it);
          }
      }

      int buscarPorEmail(const std::string& email) const {
          auto it = porEmail.find(email);
          if (it != porEmail.end())
              return it->second;
          return -1;
      }

      std::vector<int> buscarPorEmpresa(
          const std::string& empresa) const {
          std::vector<int> resultado;
          auto [lo, hi] = porEmpresa.equal_range(empresa);
          for (auto it = lo; it != hi; ++it)
              resultado.push_back(it->second);
          return resultado;
      }

      std::vector<int> buscarPorRangoEmail(
          const std::string& inicio,
          const std::string& fin) const {
          std::vector<int> resultado;
          auto lo = porEmail.lower_bound(inicio);
          auto hi = porEmail.upper_bound(fin);
          for (auto it = lo; it != hi; ++it)
              resultado.push_back(it->second);
          return resultado;
      }

      void mostrarEstadisticas() const {
          std::cout << "\n=== INDICES ===\n";
          std::cout << "Emails indexados: " << porEmail.size() << '\n';
          std::cout << "Empresas indexadas: "
                    << porEmpresa.size() << '\n';
          // Empresas unicas
          std::set<std::string> unicas;
          for (const auto& [emp, id] : porEmpresa)
              unicas.insert(emp);
          std::cout << "Empresas unicas: " << unicas.size() << '\n';
      }
  };

  class GestorGrupos {
  private:
      std::set<std::string> gruposDisponibles;
      std::map<int, std::set<std::string>> gruposContacto;

  public:
      void crearGrupo(const std::string& nombre) {
          auto [it, inserted] = gruposDisponibles.insert(nombre);
          if (inserted)
              std::cout << "Grupo '" << nombre << "' creado.\n";
          else
              std::cout << "El grupo ya existe.\n";
      }

      void eliminarGrupo(const std::string& nombre) {
          if (gruposDisponibles.erase(nombre)) {
              // Eliminar de todos los contactos
              for (auto& [id, grupos] : gruposContacto)
                  grupos.erase(nombre);
              std::cout << "Grupo eliminado.\n";
          }
      }

      bool asignarGrupo(int contactoId, const std::string& grupo) {
          if (!gruposDisponibles.contains(grupo)) {
              std::cout << "Grupo no existe.\n";
              return false;
          }
          gruposContacto[contactoId].insert(grupo);
          return true;
      }

      bool quitarGrupo(int contactoId, const std::string& grupo) {
          auto it = gruposContacto.find(contactoId);
          if (it != gruposContacto.end()) {
              if (it->second.erase(grupo)) {
                  if (it->second.empty())
                      gruposContacto.erase(it);
                  return true;
              }
          }
          return false;
      }

      std::set<std::string> gruposDe(int contactoId) const {
          auto it = gruposContacto.find(contactoId);
          if (it != gruposContacto.end())
              return it->second;
          return {};
      }

      std::vector<int> contactosEnGrupo(
          const std::string& grupo) const {
          std::vector<int> resultado;
          for (const auto& [id, grupos] : gruposContacto) {
              if (grupos.contains(grupo))
                  resultado.push_back(id);
          }
          return resultado;
      }

      // Contactos que estan en varios grupos
      std::vector<int> contactosMultiGrupo(int minGrupos) const {
          std::vector<int> resultado;
          for (const auto& [id, grupos] : gruposContacto) {
              if (grupos.size() >= minGrupos)
                  resultado.push_back(id);
          }
          return resultado;
      }

      // Union de grupos
      std::set<std::string> gruposComunes(
          int id1, int id2) const {
          std::set<std::string> resultado;
          auto g1 = gruposDe(id1);
          auto g2 = gruposDe(id2);
          std::set_intersection(g1.begin(), g1.end(),
                               g2.begin(), g2.end(),
                               std::inserter(resultado,
                                            resultado.end()));
          return resultado;
      }

      void listarGrupos() const {
          std::cout << "\n=== GRUPOS DISPONIBLES ===\n";
          for (const auto& g : gruposDisponibles) {
              auto miembros = contactosEnGrupo(g);
              std::cout << "  - " << g
                        << " (" << miembros.size() << " contactos)\n";
          }
      }
  };

  class HistorialInteracciones {
  private:
      std::map<int, std::deque<Interaccion>> historial;
      static const size_t MAX_INTERACCIONES = 100;

  public:
      void agregarInteraccion(int contactoId,
                              const std::string& tipo,
                              const std::string& desc) {
          auto& interacciones = historial[contactoId];
          interacciones.push_back({
              "2024-01-15", tipo, desc
          });
          if (interacciones.size() > MAX_INTERACCIONES)
              interacciones.pop_front();
      }

      std::deque<Interaccion> interaccionesDe(
          int contactoId) const {
          auto it = historial.find(contactoId);
          if (it != historial.end())
              return it->second;
          return {};
      }

      void mostrarInteracciones(int contactoId) const {
          auto inter = interaccionesDe(contactoId);
          if (inter.empty()) {
              std::cout << "Sin interacciones.\n";
              return;
          }
          std::cout << "\nHistorial de interacciones:\n";
          for (const auto& i : inter) {
              std::cout << "  [" << i.fecha << "] "
                        << i.tipo << ": " << i.descripcion << '\n';
          }
      }

      // Ultimas N interacciones globales
      std::deque<std::pair<int, Interaccion>>
      ultimasGlobales(int n) const {
          std::deque<std::pair<int, Interaccion>> todas;
          for (const auto& [id, inter] : historial) {
              for (const auto& i : inter)
                  todas.push_back({id, i});
          }
          // Las mas recientes al final (deque)
          if (todas.size() > n) {
              // pop_front las mas viejas
              while (todas.size() > n)
                  todas.pop_front();
          }
          return todas;
      }
  };

MODULO 5: COLA DE PRIORIDAD (priority_queue)

  class ColaPrioridadContactos {
  private:
      struct ComparePrioridad {
          bool operator()(const Contacto& a,
                         const Contacto& b) const {
              if (a.prioridad != b.prioridad)
                  return a.prioridad < b.prioridad;
              return a.nombre > b.nombre;
          }
      };

      std::priority_queue<Contacto,
                          std::vector<Contacto>,
                          ComparePrioridad> cola;

  public:
      void encolar(const Contacto& c) {
          cola.push(c);
      }

      void encolarMultiples(
          const std::vector<Contacto>& contactos) {
          for (const auto& c : contactos)
              cola.push(c);
      }

      std::vector<Contacto> procesarTopN(int n) {
          std::vector<Contacto> resultado;
          for (int i = 0; i < n && !cola.empty(); ++i) {
              resultado.push_back(cola.top());
              cola.pop();
          }
          return resultado;
      }

      bool vacia() const { return cola.empty(); }
      size_t tamano() const { return cola.size(); }
  };

  class SistemaDeshacer {
  private:
      std::stack<std::vector<Contacto>> historial;
      std::stack<std::vector<Contacto>> redoStack;
      static const size_t MAX_DESHACER = 50;

  public:
      void guardarEstado(const std::vector<Contacto>& estado) {
          historial.push(estado);
          if (historial.size() > MAX_DESHACER) {
              // No podemos pop del fondo del stack facilmente
              // Solucion alternativa: almacenar en deque
          }
          // Limpiar redo
          while (!redoStack.empty())
              redoStack.pop();
      }

      bool deshacer(std::vector<Contacto>& actual) {
          if (historial.empty()) return false;
          redoStack.push(actual);
          actual = historial.top();
          historial.pop();
          return true;
      }

      bool rehacer(std::vector<Contacto>& actual) {
          if (redoStack.empty()) return false;
          historial.push(actual);
          actual = redoStack.top();
          redoStack.pop();
          return true;
      }

      void limpiar() {
          while (!historial.empty()) historial.pop();
          while (!redoStack.empty()) redoStack.pop();
      }
  };

  class SistemaNotificaciones {
  private:
      struct Notificacion {
          int id;
          std::string mensaje;
          std::string tipo;  // info, warning, error
          std::time_t timestamp;
      };

      std::queue<Notificacion> cola;
      int nextId;

  public:
      SistemaNotificaciones() : nextId(1) {}

      void notificar(const std::string& msg,
                     const std::string& tipo = "info") {
          cola.push({nextId++, msg, tipo, std::time(nullptr)});
          if (cola.size() > 100) cola.pop();
      }

      Notificacion siguiente() {
          if (cola.empty())
              return {-1, "", "", 0};
          Notificacion n = cola.front();
          cola.pop();
          return n;
      }

      void mostrarTodas() {
          std::cout << "\n=== NOTIFICACIONES ===\n";
          auto copia = cola;  // copia para no modificar original
          while (!copia.empty()) {
              auto n = copia.front(); copia.pop();
              std::cout << "[" << n.tipo << "] " << n.mensaje << '\n';
          }
      }

      bool vacia() const { return cola.empty(); }
      size_t pendientes() const { return cola.size(); }
  };

          for (const auto& c : contactos)

          std::cout << "\nContactos por empresa:\n";

          if (!contactos.empty()) {
              double suma = std::accumulate(
                  contactos.begin(), contactos.end(), 0.0,
                  [](double acc, const Contacto& c) {
                      return acc + c.prioridad;
                  });
              std::cout << "\nPrioridad promedio: "
                        << suma / contactos.size() << '\n';
          }

          // Contacto mas reciente (max_element por ID)
          auto masReciente = std::max_element(
              contactos.begin(), contactos.end(),
              [](const Contacto& a, const Contacto& b) {
                  return a.id < b.id;
              });
          if (masReciente != contactos.end())
              std::cout << "Ultimo contacto: "
                        << masReciente->nombre << '\n';

          // Favoritos (count_if)
          int favs = std::count_if(contactos.begin(),
              contactos.end(),
              [](const Contacto& c) { return c.favorito; });
          std::cout << "Favoritos: " << favs << '\n';

          // Top prioridades (partial_sort)
          if (contactos.size() >= 3) {
              auto copia = contactos;
              std::partial_sort(copia.begin(),
                  copia.begin() + 3, copia.end(),
                  [](const Contacto& a, const Contacto& b) {
                      if (a.prioridad != b.prioridad)
                          return a.prioridad > b.prioridad;
                      return a.nombre < b.nombre;
                  });

              std::cout << "\nTop 3 prioridades:\n";
              for (int i = 0; i < 3; ++i)
                  std::cout << "  " << (i+1) << ". "
                            << copia[i].nombre
                            << " (p: " << copia[i].prioridad << ")\n";
          }

          // Distribucion alfabetica
          std::map<char, int> distribucion;
          for (const auto& c : contactos) {
              if (!c.nombre.empty())
                  distribucion[std::toupper(c.nombre[0])]++;
          }
          std::cout << "\nDistribucion alfabetica:\n";
          for (const auto& [letra, cnt] : distribucion)
              std::cout << "  " << letra << ": " << cnt << '\n';

          std::cout << "==========================================\n";
      }
  };

class GestorContactos {
private:
    std::vector<Contacto> contactos;
    int nextId;

public:
    GestorContactos() : nextId(1) {}

    int agregar(const std::string& nombre, const std::string& email,
                const std::string& tel, const std::string& empresa,
                const std::string& cargo, int prioridad = 3) {
        contactos.push_back({nextId++, nombre, email, tel,
                            empresa, cargo, false, prioridad});
        return nextId - 1;
    }

    bool eliminar(int id) {
        auto it = std::find_if(contactos.begin(), contactos.end(),
            [id](const Contacto& c) { return c.id == id; });
        if (it != contactos.end()) {
            contactos.erase(it);
            return true;
        }
        return false;
    }

    bool actualizar(int id, const Contacto& nuevos) {
        auto it = std::find_if(contactos.begin(), contactos.end(),
            [id](const Contacto& c) { return c.id == id; });
        if (it != contactos.end()) {
            *it = nuevos;
            it->id = id;
            return true;
        }
        return false;
    }

    Contacto* buscarPorId(int id) {
        auto it = std::find_if(contactos.begin(), contactos.end(),
            [id](const Contacto& c) { return c.id == id; });
        return (it != contactos.end()) ? &(*it) : nullptr;
    }

    std::vector<Contacto> buscarPorNombre(const std::string& q) const {
        std::vector<Contacto> res;
        std::copy_if(contactos.begin(), contactos.end(),
            std::back_inserter(res),
            [&q](const Contacto& c) {
                std::string lower = c.nombre;
                std::transform(lower.begin(), lower.end(),
                              lower.begin(), ::tolower);
                std::string query = q;
                std::transform(query.begin(), query.end(),
                              query.begin(), ::tolower);
                return lower.find(query) != std::string::npos;
            });
        return res;
    }

    std::vector<Contacto> buscarPorPrioridad(int minP) const {
        std::vector<Contacto> res;
        std::copy_if(contactos.begin(), contactos.end(),
            std::back_inserter(res),
            [minP](const Contacto& c) { return c.prioridad >= minP; });
        return res;
    }

    void ordenarPorNombre() {
        std::sort(contactos.begin(), contactos.end(),
            [](const Contacto& a, const Contacto& b) {
                return a.nombre < b.nombre;
            });
    }

    void ordenarPorPrioridad() {
        std::sort(contactos.begin(), contactos.end(),
            [](const Contacto& a, const Contacto& b) {
                if (a.prioridad != b.prioridad)
                    return a.prioridad > b.prioridad;
                return a.nombre < b.nombre;
            });
    }

    void marcarFavorito(int id, bool fav) {
        if (auto* c = buscarPorId(id)) c->favorito = fav;
    }

    std::vector<Contacto> favoritos() const {
        std::vector<Contacto> res;
        std::copy_if(contactos.begin(), contactos.end(),
            std::back_inserter(res),
            [](const Contacto& c) { return c.favorito; });
        return res;
    }

    std::vector<Contacto> todos() const { return contactos; }
    size_t total() const { return contactos.size(); }
    bool vacio() const { return contactos.empty(); }

    Snapshot crearSnapshot() const {
        return {contactos, ""};
    }

    void restaurar(const Snapshot& snap) {
        contactos = snap.contactos;
        nextId = contactos.empty() ? 1 :
            std::max_element(contactos.begin(), contactos.end(),
                [](const Contacto& a, const Contacto& b) {
                    return a.id < b.id;
                })->id + 1;
    }

    void mostrarTodos() const {
        if (contactos.empty()) {
            std::cout << "No hay contactos.\n";
            return;
        }
        std::cout << "\n" << std::string(90, '=') << "\n";
        std::cout << std::left << std::setw(3) << "ID" << " "
                  << "  " << std::setw(20) << "NOMBRE"
                  << std::setw(25) << "EMAIL"
                  << std::setw(15) << "TELEFONO"
                  << std::setw(15) << "EMPRESA"
                  << "PRIO\n";
        std::cout << std::string(90, '-') << "\n";
        for (const auto& c : contactos)
            std::cout << c << '\n';
        std::cout << std::string(90, '=') << "\n";
        std::cout << "Total: " << contactos.size() << " contactos\n";
    }
};

// --- MODULO 2: INDICES ---

class IndicesBusqueda {
private:
    std::map<std::string, int> porEmail;
    std::multimap<std::string, int> porEmpresa;

public:
    void indexar(const Contacto& c) {
        porEmail[c.email] = c.id;
        porEmpresa.insert({c.empresa, c.id});
    }

    void reindexar(const std::vector<Contacto>& contactos) {
        porEmail.clear();
        porEmpresa.clear();
        for (const auto& c : contactos)
            indexar(c);
    }

    int buscarEmail(const std::string& email) const {
        auto it = porEmail.find(email);
        return (it != porEmail.end()) ? it->second : -1;
    }

    std::vector<int> buscarEmpresa(const std::string& emp) const {
        std::vector<int> res;
        auto [lo, hi] = porEmpresa.equal_range(emp);
        for (auto it = lo; it != hi; ++it)
            res.push_back(it->second);
        return res;
    }

    std::vector<std::string> empresasUnicas() const {
        std::set<std::string> unicas;
        for (const auto& [emp, id] : porEmpresa)
            unicas.insert(emp);
        return {unicas.begin(), unicas.end()};
    }

    void mostrarEstadisticas() const {
        std::cout << "\nIndices: " << porEmail.size()
                  << " emails, " << porEmpresa.size()
                  << " empresas ("
                  << empresasUnicas().size() << " unicas)\n";
    }
};

class GestorGrupos {
private:
    std::set<std::string> grupos;
    std::map<int, std::set<std::string>> asignaciones;

public:
    void crear(const std::string& g) {
        if (grupos.insert(g).second)
            std::cout << "Grupo '" << g << "' creado.\n";
    }

    void eliminarGrupo(const std::string& g) {
        if (grupos.erase(g)) {
            for (auto& [id, gs] : asignaciones) gs.erase(g);
        }
    }

    bool asignar(int id, const std::string& g) {
        if (!grupos.contains(g)) return false;
        asignaciones[id].insert(g);
        return true;
    }

    bool quitar(int id, const std::string& g) {
        auto it = asignaciones.find(id);
        if (it != asignaciones.end() && it->second.erase(g)) {
            if (it->second.empty()) asignaciones.erase(it);
            return true;
        }
        return false;
    }

    std::set<std::string> gruposDe(int id) const {
        auto it = asignaciones.find(id);
        return (it != asignaciones.end()) ? it->second : std::set<std::string>{};
    }

    std::vector<int> contactosEn(const std::string& g) const {
        std::vector<int> res;
        for (const auto& [id, gs] : asignaciones)
            if (gs.contains(g)) res.push_back(id);
        return res;
    }

    void listar() const {
        std::cout << "\nGrupos:\n";
        for (const auto& g : grupos) {
            auto ids = contactosEn(g);
            std::cout << "  - " << g << " (" << ids.size() << ")\n";
        }
    }
};

// --- MODULO 5: PRIORITY QUEUE ---

class ColaPrioridad {
private:
    struct Cmp {
        bool operator()(const Contacto& a, const Contacto& b) const {
            if (a.prioridad != b.prioridad)
                return a.prioridad < b.prioridad;
            if (a.favorito != b.favorito)
                return a.favorito > b.favorito;
            return a.nombre > b.nombre;
        }
    };
    std::priority_queue<Contacto, std::vector<Contacto>, Cmp> pq;

public:
    void push(const Contacto& c) { pq.push(c); }
    void pushMultiples(const std::vector<Contacto>& vv) {
        for (const auto& c : vv) pq.push(c);
    }
    std::vector<Contacto> topN(int n) {
        std::vector<Contacto> res;
        while (!pq.empty() && n--) {
            res.push_back(pq.top());
            pq.pop();
        }
        return res;
    }
    bool empty() const { return pq.empty(); }
    size_t size() const { return pq.size(); }
};

// --- MODULO 6: DESHACER ---

class SistemaDeshacer {
private:
    std::stack<std::vector<Contacto>> undoStack;
    std::stack<std::vector<Contacto>> redoStack;

public:
    void guardar(const std::vector<Contacto>& estado) {
        undoStack.push(estado);
        while (!redoStack.empty()) redoStack.pop();
    }

    bool deshacer(std::vector<Contacto>& actual) {
        if (undoStack.empty()) return false;
        redoStack.push(actual);
        actual = undoStack.top();
        undoStack.pop();
        return true;
    }

    bool rehacer(std::vector<Contacto>& actual) {
        if (redoStack.empty()) return false;
        undoStack.push(actual);
        actual = redoStack.top();
        redoStack.pop();
        return true;
    }
};

================================================================================
  PREGUNTAS FRECUENTES (FAQ)
================================================================================

P: \?Por que usar tantos contenedores diferentes?
R: Cada contenedor tiene fortalezas especificas. vector para
   almacenamiento secuencial, map para busqueda por clave,
   set para conjuntos unicos, deque para historial FIFO/LIFO,
   stack para deshacer, queue para notificaciones.

P: \?Como manejar la persistencia de datos?
R: Serializar los contenedores a archivos (binarios o texto).
   Para vector, escribir secuencialmente. Para map, escribir
   pares clave-valor. Para set, escribir elementos uno por uno.

P: \?Como escalar este sistema a miles de contactos?
R: Usar unordered_map en lugar de map para O(1) promedio.
   Usar indices secundarios con map. Considerar una base de
   datos real (SQLite) para volumenes mayores.

P: \?Que pasa con la memoria con tantos contenedores?
R: Los contenedores STL gestionan memoria automaticamente.
   Los snapshots de deshacer pueden consumir mucha memoria;
   limitar a N snapshots con deque.

P: \?Como probar el sistema?
R: Escribir tests unitarios para cada modulo. Usar assert
   para invariantes. Probar casos borde: contactos vacios,
   busquedas sin resultados, IDs inexistentes.

P: \?Puedo usar este codigo en produccion?
R: Es un proyecto educativo. Para produccion, anadir manejo
   de errores mas robusto, logging, seguridad y persistencia.
