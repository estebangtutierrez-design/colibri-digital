La STL proporciona más de 100 algoritmos en la cabecera <algorithm>
(y algunos en <numeric>). Operan sobre rangos definidos por
iteradores [first, last).

Clasificación de algoritmos:

1. Algoritmos de solo lectura (no modifican el rango):
   find, count, for_each, all_of, any_of, none_of, search, equal,
   mismatch, adjacent_find.

2. Algoritmos de modificación (modifican elementos):
   copy, move, transform, fill, generate, replace, swap, iter_swap,
   reverse, rotate, shuffle, random_shuffle.

3. Algoritmos de ordenación y búsqueda:
   sort, stable_sort, partial_sort, nth_element, binary_search,
   lower_bound, upper_bound, equal_range, merge, inplace_merge.

4. Algoritmos de conjunto (requieren rangos ordenados):
   set_union, set_intersection, set_difference,
   set_symmetric_difference, includes.

5. Algoritmos de montículo (heap):
   push_heap, pop_heap, make_heap, sort_heap.

6. Algoritmos numéricos (<numeric>):
   accumulate, inner_product, partial_sum, adjacent_difference,
   iota.

ALGORITMOS FUNDAMENTALES

sort: ordena elementos en orden ascendente.

  #include <algorithm>
  std::vector<int> v {5, 2, 8, 1, 9, 3};
  std::sort(v.begin(), v.end());
  // v: {1, 2, 3, 5, 8, 9}

  // Orden descendente
  std::sort(v.begin(), v.end(), std::greater<int>());

  // Con lambda
  std::sort(v.begin(), v.end(),
            [](int a, int b) { return a > b; });

  // Ordenar strings por longitud
  std::sort(v.begin(), v.end(),
            [](const std::string& a, const std::string& b) {
                return a.size() < b.size();
            });

  // stable_sort: preserva orden relativo de elementos iguales
  std::stable_sort(v.begin(), v.end());

  // partial_sort: ordena solo los primeros n elementos
  std::partial_sort(v.begin(), v.begin() + 3, v.end());
  // Los primeros 3 están ordenados, el resto puede no estarlo

  // nth_element: coloca el n-ésimo elemento en su posición correcta
  std::nth_element(v.begin(), v.begin() + 4, v.end());
  // El elemento en posición 4 es el que estaría ahí si estuviera
  // totalmente ordenado. Los de antes son menores, después mayores.

find: busca la primera ocurrencia de un valor.

  std::vector<int> v {10, 20, 30, 40, 50};
  auto it = std::find(v.begin(), v.end(), 30);
  if (it != v.end())
      std::cout << "Encontrado en posición "
                << std::distance(v.begin(), it) << '\n';

  // find_if: búsqueda por condición
  auto it = std::find_if(v.begin(), v.end(),
                         [](int x) { return x > 25; });
  // Devuelve iterador al primer elemento > 25

  // find_if_not: primer elemento que NO cumple
  auto it = std::find_if_not(v.begin(), v.end(),
                             [](int x) { return x < 30; });

binary_search: búsqueda binaria O(log n) en rango ordenado.

  std::vector<int> v {1, 3, 5, 7, 9, 11};
  bool existe = std::binary_search(v.begin(), v.end(), 5);  // true
  bool existe2 = std::binary_search(v.begin(), v.end(), 6); // false

  // lower_bound: primer elemento >= valor
  auto lo = std::lower_bound(v.begin(), v.end(), 6);
  // lo apunta a 7 (primer elemento >= 6)

  // upper_bound: primer elemento > valor
  auto hi = std::upper_bound(v.begin(), v.end(), 7);
  // hi apunta a 9 (primer elemento > 7)

  // equal_range: lower_bound + upper_bound
  auto [low, high] = std::equal_range(v.begin(), v.end(), 7);

count: cuenta ocurrencias de un valor.

  std::vector<int> v {1, 2, 2, 3, 2, 4, 2, 5};
  int n = std::count(v.begin(), v.end(), 2);  // 4

  // count_if: cuenta elementos que cumplen condición
  int pares = std::count_if(v.begin(), v.end(),
                            [](int x) { return x % 2 == 0; });  // 5

copy: copia elementos de un rango a otro.

  std::vector<int> src {1, 2, 3, 4, 5};
  std::vector<int> dst(src.size());

  std::copy(src.begin(), src.end(), dst.begin());
  // Requiere que dst tenga al menos src.size() elementos

  // copy_if: copia solo los que cumplen condición
  std::vector<int> pares;
  std::copy_if(src.begin(), src.end(),
               std::back_inserter(pares),
               [](int x) { return x % 2 == 0; });

  // copy_n: copia n elementos
  std::copy_n(src.begin(), 3, dst.begin());

  // copy_backward: copia desde el final hacia atrás
  std::copy_backward(src.begin(), src.end(), dst.end());

transform: aplica una función a cada elemento.

  std::vector<int> v {1, 2, 3, 4, 5};
  std::vector<int> resultado;

  // Unaria: transforma cada elemento
  std::transform(v.begin(), v.end(),
                 std::back_inserter(resultado),
                 [](int x) { return x * x; });
  // resultado: {1, 4, 9, 16, 25}

  // Binaria: combina elementos de dos rangos
  std::vector<int> a {1, 2, 3};
  std::vector<int> b {4, 5, 6};
  std::vector<int> suma;
  std::transform(a.begin(), a.end(), b.begin(),
                 std::back_inserter(suma),
                 [](int x, int y) { return x + y; });
  // suma: {5, 7, 9}

  // Transform in-place
  std::transform(v.begin(), v.end(), v.begin(),
                 [](int x) { return x * 2; });

for_each: aplica una función a cada elemento (no modifica por
defecto, pero puede hacerlo si recibe referencia).

  std::vector<int> v {1, 2, 3, 4, 5};

  // Modificar (recibir por referencia)
  std::for_each(v.begin(), v.end(),
                [](int& x) { x *= 2; });

  // Solo lectura
  std::for_each(v.begin(), v.end(),
                [](int x) { std::cout << x << ' '; });

  // Con estado (acumulador mutable)
  int suma = 0;
  std::for_each(v.begin(), v.end(),
                [&suma](int x) { suma += x; });

EXPRESIONES LAMBDA (recordatorio)

  [captura](parámetros) -> tipo_retorno { cuerpo }

  Captura por valor:  [x]
  Captura por ref:    [&x]
  Captura todo valor: [=]
  Captura todo ref:   [&]
  Mixto:              [&, x]  (todo por ref excepto x por valor)
  Mutable:            [x]() mutable { x++; }  (modifica copia interna)

  auto lambda = [](int a, int b) { return a + b; };
  int r = lambda(3, 4);  // 7

  // Lambda genérica (C++14)
  auto gen = [](auto a, auto b) { return a + b; };

ALGORITMOS ADICIONALES

replace: reemplaza ocurrencias de un valor.

  std::vector<int> v {1, 2, 3, 2, 4, 2, 5};
  std::replace(v.begin(), v.end(), 2, 99);
  // v: {1, 99, 3, 99, 4, 99, 5}

  // replace_if: reemplaza si cumple condición
  std::replace_if(v.begin(), v.end(),
                  [](int x) { return x % 2 == 0; }, 0);

fill: llena un rango con un valor.

  std::vector<int> v(10);
  std::fill(v.begin(), v.end(), 42);
  // v: {42, 42, 42, ...}

  // fill_n: llena n posiciones
  std::fill_n(v.begin(), 5, 99);

generate: llena un rango con valores generados por una función.

  std::vector<int> v(10);
  int n = 0;
  std::generate(v.begin(), v.end(), [&n]() { return n++; });
  // v: {0, 1, 2, 3, ...}

  // generate_n: genera n valores
  std::generate_n(v.begin(), 5, []() { return rand() % 100; });

reverse: invierte el orden de los elementos.

  std::vector<int> v {1, 2, 3, 4, 5};
  std::reverse(v.begin(), v.end());
  // v: {5, 4, 3, 2, 1}

rotate: rota los elementos (mueve [middle, last) al inicio).

  std::vector<int> v {1, 2, 3, 4, 5};
  std::rotate(v.begin(), v.begin() + 2, v.end());
  // v: {3, 4, 5, 1, 2}

shuffle: mezcla aleatoriamente.

  std::random_device rd;
  std::mt19937 g(rd());
  std::shuffle(v.begin(), v.end(), g);

unique: elimina duplicados consecutivos (usar con sort antes).

  std::vector<int> v {1, 1, 2, 3, 3, 3, 4, 5, 5};
  auto last = std::unique(v.begin(), v.end());
  v.erase(last, v.end());
  // v: {1, 2, 3, 4, 5}

  // unique_copy: copia elementos únicos a otro contenedor
  std::vector<int> unicos;
  std::unique_copy(v.begin(), v.end(),
                   std::back_inserter(unicos));

merge: fusiona dos rangos ordenados en uno ordenado.

  std::vector<int> a {1, 3, 5, 7};
  std::vector<int> b {2, 4, 6, 8};
  std::vector<int> resultado;
  std::merge(a.begin(), a.end(), b.begin(), b.end(),
             std::back_inserter(resultado));
  // resultado: {1, 2, 3, 4, 5, 6, 7, 8}

min_element / max_element: encuentra mínimo/máximo.

  auto min = std::min_element(v.begin(), v.end());
  auto max = std::max_element(v.begin(), v.end());
  // Devuelven iterador

  auto [min, max] = std::minmax_element(v.begin(), v.end());

Algoritmos de no modificación adicionales:

  // all_of: todos los elementos cumplen?
  bool todos = std::all_of(v.begin(), v.end(),
                           [](int x) { return x > 0; });

  // any_of: algún elemento cumple?
  bool alguno = std::any_of(v.begin(), v.end(),
                            [](int x) { return x > 10; });

  // none_of: ningún elemento cumple?
  bool ninguno = std::none_of(v.begin(), v.end(),
                              [](int x) { return x < 0; });

  // equal: dos rangos son iguales?
  bool igual = std::equal(v.begin(), v.end(), w.begin());

  // mismatch: primera posición donde difieren
  auto [it1, it2] = std::mismatch(v.begin(), v.end(), w.begin());

  // search: busca subrango
  std::vector<int> patron {3, 4, 5};
  auto it = std::search(v.begin(), v.end(),
                        patron.begin(), patron.end());

  // adjacent_find: primer par de elementos iguales adyacentes
  auto it = std::adjacent_find(v.begin(), v.end());

Algoritmos numéricos (<numeric>):

  #include <numeric>

  // accumulate: suma (u otra operación binaria)
  int suma = std::accumulate(v.begin(), v.end(), 0);
  int prod = std::accumulate(v.begin(), v.end(), 1,
                             [](int a, int b) { return a * b; });

  // inner_product: producto punto
  int dot = std::inner_product(a.begin(), a.end(), b.begin(), 0);

  // partial_sum: sumas parciales
  std::vector<int> parciales;
  std::partial_sum(v.begin(), v.end(),
                   std::back_inserter(parciales));
  // v: {1,2,3,4} -> parciales: {1,3,6,10}

  // iota: llenar con valores incrementales
  std::iota(v.begin(), v.end(), 0);  // {0,1,2,3,4}

//==============================================================
// EJEMPLO COMPLETO: ANÁLISIS DE DATOS CON ALGORITMOS STL
//==============================================================
#include <iostream>
#include <algorithm>
#include <numeric>
#include <vector>
#include <string>
#include <random>
#include <iomanip>
#include <map>
#include <sstream>

struct Venta {
    int id;
    std::string producto;
    double precio;
    int cantidad;
    std::string fecha;
    bool pagado;

    double total() const { return precio * cantidad; }
};

std::ostream& operator<<(std::ostream& os, const Venta& v) {
    os << "[" << v.id << "] " << v.producto << " x" << v.cantidad
       << " = $" << std::fixed << std::setprecision(2) << v.total()
       << " (" << v.fecha << ") "
       << (v.pagado ? "PAGADO" : "PENDIENTE");
    return os;
}

class AnalisisVentas {
private:
    std::vector<Venta> ventas;

public:
    void agregar(const Venta& v) { ventas.push_back(v); }

    void agregarMultiples(const std::vector<Venta>& nuevas) {
        ventas.insert(ventas.end(), nuevas.begin(), nuevas.end());
    }

    // ORDENACIÓN
    void ordenarPorFecha() {
        std::sort(ventas.begin(), ventas.end(),
            [](const Venta& a, const Venta& b) {
                return a.fecha < b.fecha;
            });
    }

    void ordenarPorTotal() {
        std::sort(ventas.begin(), ventas.end(),
            [](const Venta& a, const Venta& b) {
                return a.total() > b.total();
            });
    }

    void ordenarPorProducto() {
        std::sort(ventas.begin(), ventas.end(),
            [](const Venta& a, const Venta& b) {
                return a.producto < b.producto;
            });
    }

    // BÚSQUEDA
    std::vector<Venta> buscarPorProducto(const std::string& prod) {
        std::vector<Venta> resultado;
        std::copy_if(ventas.begin(), ventas.end(),
                     std::back_inserter(resultado),
            [&prod](const Venta& v) {
                return v.producto.find(prod) != std::string::npos;
            });
        return resultado;
    }

    Venta* buscarPorId(int id) {
        auto it = std::find_if(ventas.begin(), ventas.end(),
            [id](const Venta& v) { return v.id == id; });
        if (it != ventas.end())
            return &(*it);
        return nullptr;
    }

    // FILTRADO
    std::vector<Venta> ventasPendientes() const {
        std::vector<Venta> pendientes;
        std::copy_if(ventas.begin(), ventas.end(),
                     std::back_inserter(pendientes),
            [](const Venta& v) { return !v.pagado; });
        return pendientes;
    }

    std::vector<Venta> ventasMayoresA(double monto) const {
        std::vector<Venta> filtradas;
        std::copy_if(ventas.begin(), ventas.end(),
                     std::back_inserter(filtradas),
            [monto](const Venta& v) {
                return v.total() >= monto;
            });
        return filtradas;
    }

    // ESTADÍSTICAS
    double totalVentas() const {
        return std::accumulate(ventas.begin(), ventas.end(), 0.0,
            [](double acc, const Venta& v) {
                return acc + v.total();
            });
    }

    double promedioVentas() const {
        if (ventas.empty()) return 0.0;
        return totalVentas() / ventas.size();
    }

    std::vector<std::pair<std::string, int>> topProductos(int n) const {
        std::map<std::string, int> conteo;
        for (const auto& v : ventas)
            conteo[v.producto] += v.cantidad;

        std::vector<std::pair<std::string, int>> ranking(
            conteo.begin(), conteo.end());

        std::partial_sort(ranking.begin(),
                         ranking.begin() + std::min(n, (int)ranking.size()),
                         ranking.end(),
            [](const auto& a, const auto& b) {
                return a.second > b.second;
            });

        if (ranking.size() > n)
            ranking.resize(n);
        return ranking;
    }

    Venta ventaMasCara() const {
        auto it = std::max_element(ventas.begin(), ventas.end(),
            [](const Venta& a, const Venta& b) {
                return a.precio < b.precio;
            });
        if (it != ventas.end()) return *it;
        return Venta{};
    }

    double medianaVentas() const {
        if (ventas.empty()) return 0.0;

        std::vector<double> totales;
        totales.reserve(ventas.size());
        std::transform(ventas.begin(), ventas.end(),
                      std::back_inserter(totales),
                      [](const Venta& v) { return v.total(); });

        std::nth_element(totales.begin(),
                        totales.begin() + totales.size() / 2,
                        totales.end());

        if (totales.size() % 2 == 0) {
            auto m1 = totales[totales.size() / 2];
            std::nth_element(totales.begin(),
                            totales.begin() + totales.size() / 2 - 1,
                            totales.end());
            auto m2 = totales[totales.size() / 2 - 1];
            return (m1 + m2) / 2.0;
        }
        return totales[totales.size() / 2];
    }

    // TRANSFORMACIÓN
    void aplicarDescuento(double porcentaje) {
        std::for_each(ventas.begin(), ventas.end(),
            [porcentaje](Venta& v) {
                v.precio *= (1.0 - porcentaje / 100.0);
            });
    }

    void marcarTodoPagado() {
        std::for_each(ventas.begin(), ventas.end(),
            [](Venta& v) { v.pagado = true; });
    }

    // ELIMINACIÓN
    void eliminarVenta(int id) {
        auto it = std::remove_if(ventas.begin(), ventas.end(),
            [id](const Venta& v) { return v.id == id; });
        if (it != ventas.end())
            ventas.erase(it, ventas.end());
    }

    void eliminarPendientes() {
        auto it = std::remove_if(ventas.begin(), ventas.end(),
            [](const Venta& v) { return !v.pagado; });
        ventas.erase(it, ventas.end());
    }

    // REPORTES
    void resumen() const {
        if (ventas.empty()) {
            std::cout << "No hay ventas.\n";
            return;
        }

        std::cout << "\n=== RESUMEN DE VENTAS ===\n";
        std::cout << "Total ventas: $" << std::fixed
                  << std::setprecision(2) << totalVentas() << '\n';
        std::cout << "Promedio: $" << promedioVentas() << '\n';
        std::cout << "Mediana: $" << medianaVentas() << '\n';
        std::cout << "Cantidad: " << ventas.size() << '\n';

        auto masCara = ventaMasCara();
        std::cout << "Venta más cara: " << masCara.producto
                  << " ($" << masCara.precio << ")\n";

        auto pendientes = ventasPendientes();
        std::cout << "Pendientes de pago: " << pendientes.size() << '\n';
    }

    void mostrarTodo() const {
        std::cout << "\n=== TODAS LAS VENTAS ===\n";
        for (const auto& v : ventas)
            std::cout << v << '\n';
    }

    void mostrarTop(int n) {
        auto top = topProductos(n);
        std::cout << "\nTop " << n << " productos:\n";
        int i = 1;
        for (const auto& [prod, cant] : top)
            std::cout << i++ << ". " << prod
                      << " (vendidos: " << cant << ")\n";
    }
};

int main() {
    AnalisisVentas analisis;

    analisis.agregar({1, "Laptop", 899.99, 2, "2024-01-15", true});
    analisis.agregar({2, "Mouse", 29.99, 10, "2024-01-16", true});
    analisis.agregar({3, "Teclado", 79.99, 5, "2024-01-16", false});
    analisis.agregar({4, "Monitor", 349.99, 3, "2024-01-17", true});
    analisis.agregar({5, "Webcam", 59.99, 8, "2024-01-17", false});
    analisis.agregar({6, "Laptop", 899.99, 1, "2024-01-18", true});
    analisis.agregar({7, "Audifonos", 89.99, 4, "2024-01-18", true});
    analisis.agregar({8, "Mouse", 29.99, 15, "2024-01-19", false});

    analisis.mostrarTodo();
    analisis.resumen();

    analisis.ordenarPorTotal();
    analisis.mostrarTodo();

    std::cout << "\nBuscar 'Laptop':\n";
    auto laptops = analisis.buscarPorProducto("Laptop");
    for (const auto& v : laptops)
        std::cout << "  " << v << '\n';

    std::cout << "\nVentas pendientes:\n";
    auto pendientes = analisis.ventasPendientes();
    for (const auto& v : pendientes)
        std::cout << "  " << v << '\n';

    analisis.mostrarTop(3);

    std::cout << "\nAplicando descuento de 10%...\n";
    analisis.aplicarDescuento(10.0);

    std::cout << "\nEliminando venta 8...\n";
    analisis.eliminarVenta(8);

    analisis.resumen();

    std::cout << "\nMarcando todo como pagado...\n";
    analisis.marcarTodoPagado();
    analisis.mostrarTodo();

    return 0;
}

Puntos clave sobre algoritmos STL:

1. La cabecera <algorithm> contiene más de 100 algoritmos genéricos.
2. Operan sobre rangos [first, last) definidos por iteradores.
3. Clasificación: solo lectura, modificación, ordenación, conjunto.
4. sort: O(n log n), estable con stable_sort.
5. find: O(n), find_if con predicado.
6. binary_search: O(log n), requiere rango ordenado.
7. count: cuenta ocurrencias, count_if con condición.
8. copy: copia rangos, copy_if para filtrado.
9. transform: aplica función a cada elemento.
10. for_each: aplica función con posible efecto lateral.
11. remove: no elimina realmente (erase-remove idiom).
12. accumulate: suma/reduce (en <numeric>).

================================================================================
  PREGUNTAS FRECUENTES (FAQ)
================================================================================

P: ¿Por qué remove no elimina realmente los elementos?
R: remove reorganiza los elementos, moviendo los que no deben
   eliminarse al inicio, y devuelve el nuevo end. erase libera
   realmente la memoria. Se usan juntos.

P: ¿Qué algoritmo debo usar para ordenar?
R: sort para la mayoría de casos. stable_sort si necesitas
   preservar el orden relativo de elementos iguales.
   partial_sort si solo necesitas los primeros n ordenados.

P: ¿Cómo busco en un contenedor no ordenado?
R: Usa find (O(n)) para contenedores sin orden.
   binary_search solo funciona con rangos ordenados.

P: ¿Qué es más eficiente: for_each o range-based for?
R: Son equivalentes en rendimiento. El range-based for es más
   legible para casos simples. for_each puede ser mejor para
   pipelines de algoritmos.

P: ¿Los lambdas tienen costo de rendimiento?
R: No. Las lambdas son objetos función que el compilador puede
   inline-optimizar, resultando en código tan eficiente como
   funciones escritas a mano.

P: ¿Puedo usar algoritmos con arrays de C?
R: Sí, usando punteros como iteradores:
   int arr[] = {1,2,3,4,5};
   std::sort(arr, arr + 5);

P: ¿Cuándo debo usar transform vs for_each?
R: transform cuando quieres un resultado transformado en otro
   rango (sin efectos laterales). for_each cuando el propósito
   principal es el efecto lateral.

Diseña e implementa un "Sistema de Procesamiento de Texto"
usando algoritmos STL. El sistema debe:

1. Cargar un texto completo (string grande) y dividirlo en
   palabras (usando istream_iterator con stringstream).

2. Estadísticas del texto:
   - Número total de palabras (distance).
   - Palabras únicas (sort + unique + distance).
   - Frecuencia de cada palabra (count_if o map).
   - Palabra más frecuente (max_element con pair).
   - Longitud promedio (accumulate / size).
   - Palabra más larga y más corta (minmax_element).

3. Transformaciones:
   - Convertir todo a minúsculas (transform con tolower).
   - Reemplazar palabras ofensivas (replace).
   - Invertir cada palabra (for_each con reverse).
   - Eliminar palabras cortas (< 3 letras) (remove_if + erase).

4. Búsqueda:
   - Buscar una palabra específica (find).
   - Buscar palabras que contengan una subcadena (find_if).
   - Buscar anagramas (sort de letras + igualdad).

5. Ordenación:
   - Palabras por orden alfabético (sort).
   - Palabras por longitud (stable_sort).
   - Palabras por frecuencia descendente (partial_sort).
   - Ranking de palabras más usadas (nth_element para top K).

6. Generación de reporte:
   - Usar accumulate para construir un string con el resumen.
   - Usar copy con ostream_iterator para escribir a consola.
   - Usar for_each para mostrar estadísticas.

7. El sistema debe estar implementado con funciones que usen
   iteradores y algoritmos STL (sin bucles for manuales).

Requisitos:
- Usar al menos 15 algoritmos diferentes de <algorithm> y <numeric>.
- Sin bucles for/while explícitos (usar algoritmos + lambdas).
- Todas las funciones deben operar sobre iteradores.

Este desafío integra sort, find, count, copy, transform, for_each,
accumulate, remove_if, replace, minmax_element, partial_sort,
nth_element, lower_bound, upper_bound, equal_range, unique,
reverse, y otros algoritmos.
