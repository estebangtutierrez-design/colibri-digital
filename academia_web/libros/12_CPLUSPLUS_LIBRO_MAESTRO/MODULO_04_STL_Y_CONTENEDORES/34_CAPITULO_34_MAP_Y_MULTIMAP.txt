std::map es un contenedor asociativo ordenado que almacena pares
clave-valor. Las claves son únicas y se mantienen ordenadas según
un criterio de comparación (por defecto std::less, que usa operator<).
Internamente, map se implementa como un árbol rojo-negro (red-black
tree), garantizando O(log n) en inserción, búsqueda y eliminación.

std::multimap es similar pero permite claves duplicadas. No tiene
operator[] porque no hay una clave única para indexar.

La cabecera necesaria es <map>.

Declaración básica:
  #include <map>
  #include <string>

  std::map<int, std::string> m;     // mapa vacío: int→string
  std::map<std::string, double> precios;  // string→double

  // Inicialización con lista (C++11)
  std::map<std::string, int> edades {
      {"Ana", 25},
      {"Luis", 30},
      {"Carlos", 22}
  };

std::pair es la estructura que almacena cada elemento del mapa:
  using Par = std::pair<const std::string, int>;
  // first = clave (const), second = valor

Inserción básica:
  std::map<std::string, int> m;

  // insert con pair
  m.insert(std::pair<std::string, int>("Ana", 25));
  m.insert(std::make_pair("Luis", 30));

  // insert con initializer_list
  m.insert({"Carlos", 22});

  // operator[] (solo map, no multimap)
  m["David"] = 28;   // inserta si no existe
  m["Ana"] = 26;     // actualiza si existe

  // emplace (C++11)
  m.emplace("Eva", 35);

operator[] es especial:
- Si la clave existe: devuelve referencia al valor.
- Si la clave NO existe: inserta la clave con valor por defecto
  y devuelve referencia al nuevo valor.

  std::map<std::string, int> m;
  m["clave"] = 10;      // inserta "clave" con valor 10
  int x = m["otra"];    // inserta "otra" con valor 0, devuelve 0

  // Para verificar existencia sin insertar:
  if (m.find("clave") != m.end()) { ... }

find: búsqueda O(log n). Devuelve iterador al elemento o end().
  auto it = m.find("Ana");
  if (it != m.end())
      std::cout << it->first << ": " << it->second;
  else
      std::cout << "No encontrado";

erase: elimina por clave o iterador.
  m.erase("Ana");               // por clave (devuelve 1 si existía)
  m.erase(it);                  // por iterador
  m.erase(m.begin(), m.end());  // por rango (equivale a clear)

size, empty, clear:
  m.size();    // número de elementos
  m.empty();   // true si está vacío
  m.clear();   // elimina todos los elementos

contains (C++20): verifica existencia.
  if (m.contains("Ana"))
      std::cout << "Ana está en el mapa\n";

Iteración:
  for (const auto& par : m) {
      std::cout << par.first << " → " << par.second << '\n';
  }

  // Con iterador explícito
  for (auto it = m.begin(); it != m.end(); ++it) {
      std::cout << it->first << " → " << it->second << '\n';
  }

  // Iteración inversa
  for (auto it = m.rbegin(); it != m.rend(); ++it) {
      std::cout << it->first << " → " << it->second << '\n';
  }

  // Rangos (C++17)
  for (const auto& [clave, valor] : m) {
      std::cout << clave << " → " << valor << '\n';
  }

Operaciones de búsqueda adicionales:
  m.count("Ana");     // 1 si existe, 0 si no (en map)
  m.lower_bound("C"); // iterador al primer elemento >= "C"
  m.upper_bound("M"); // iterador al primer elemento > "M"
  m.equal_range("L"); // par de lower_bound y upper_bound

  auto [lo, hi] = m.equal_range("L");
  for (auto it = lo; it != hi; ++it)
      std::cout << it->first << ": " << it->second << '\n';

Comparador personalizado:
  std::map<int, std::string, std::greater<int>> m;  // orden descendente

  // Comparador custom
  struct PorLongitud {
      bool operator()(const std::string& a, const std::string& b) const {
          if (a.size() != b.size())
              return a.size() < b.size();
          return a < b;
      }
  };
  std::map<std::string, int, PorLongitud> m;

MULTIMAP: permite claves duplicadas.

  std::multimap<std::string, int> mm;

  mm.insert({"Ana", 25});
  mm.insert({"Ana", 30});  // permitido
  mm.insert({"Luis", 22});
  mm.insert({"Ana", 28});  // permitido

  // Búsqueda en multimap
  auto rango = mm.equal_range("Ana");
  for (auto it = rango.first; it != rango.second; ++it) {
      std::cout << it->first << ": " << it->second << '\n';
  }

  // count devuelve el número de ocurrencias
  int n = mm.count("Ana");  // 3

  // Eliminar todas las ocurrencias de una clave
  mm.erase("Ana");

  // Eliminar una ocurrencia específica
  auto it = mm.find("Ana");
  if (it != mm.end())
      mm.erase(it);

Operaciones de inserción con pista (hint):
  auto it = m.lower_bound("M");
  m.insert(it, {"Nuevo", 100});  // sugerencia de posición

  // Si la pista es correcta, la inserción es O(1) amortizado.

Mapas con tipos complejos como clave:
  struct Persona {
      std::string nombre;
      int edad;
      bool operator<(const Persona& otra) const {
          if (nombre != otra.nombre)
              return nombre < otra.nombre;
          return edad < otra.edad;
      }
  };

  std::map<Persona, std::string> personas;
  personas[{"Ana", 25}] = "Ingeniera";

O bien con comparador externo:
  struct CompararPersona {
      bool operator()(const Persona& a, const Persona& b) const {
          return a.edad < b.edad;
      }
  };
  std::map<Persona, std::string, CompararPersona> personas;

try_emplace (C++17): inserta si no existe, sin construir si ya existe.
  auto [it, inserted] = m.try_emplace("clave", 42);
  if (inserted)
      std::cout << "Insertado\n";
  else
      std::cout << "Ya existía con valor " << it->second << '\n';

insert_or_assign (C++17): inserta o actualiza.
  auto [it, inserted] = m.insert_or_assign("clave", 50);
  if (inserted)
      std::cout << "Insertado\n";
  else
      std::cout << "Actualizado a " << it->second << '\n';

merge (C++17): fusiona mapas.
  std::map<int, std::string> a {{1,"a"}, {2,"b"}, {3,"c"}};
  std::map<int, std::string> b {{3,"x"}, {4,"d"}, {5,"e"}};
  a.merge(b);
  // a: {1,a}, {2,b}, {3,c}, {4,d}, {5,e}
  // b: {3,x} (no se movió porque 3 ya existía en a)

Extract (C++17): extraer un nodo sin copiar.
  auto nodo = m.extract("clave");
  if (!nodo.empty()) {
      nodo.key() = "nueva_clave";  // modificar clave
      nodo.mapped() = 999;         // modificar valor
      m.insert(std::move(nodo));   // reinsertar
  }

Rendimiento: todas las operaciones principales son O(log n).

Ejemplo completo: Diccionario con map.

  #include <iostream>
  #include <map>
  #include <string>
  #include <fstream>
  #include <algorithm>
  #include <cctype>

  std::string toLower(const std::string& s) {
      std::string res;
      res.reserve(s.size());
      for (char c : s)
          res.push_back(std::tolower(c));
      return res;
  }

  class Diccionario {
  private:
      std::map<std::string, std::string> definiciones;

  public:
      void definir(const std::string& palabra,
                   const std::string& significado) {
          auto clave = toLower(palabra);
          definiciones[clave] = significado;
          std::cout << "Definición agregada: " << palabra << '\n';
      }

      std::string buscar(const std::string& palabra) const {
          auto clave = toLower(palabra);
          auto it = definiciones.find(clave);
          if (it != definiciones.end())
              return it->second;
          return "Palabra no encontrada";
      }

      bool existe(const std::string& palabra) const {
          return definiciones.contains(toLower(palabra));
      }

      void eliminar(const std::string& palabra) {
          if (definiciones.erase(toLower(palabra)))
              std::cout << "Palabra eliminada.\n";
          else
              std::cout << "Palabra no encontrada.\n";
      }

      void listar() const {
          for (const auto& [palabra, def] : definiciones) {
              std::cout << palabra << ": " << def << '\n';
          }
      }

      void listarPorRango(char inicio, char fin) const {
          auto lo = definiciones.lower_bound(std::string(1, inicio));
          auto hi = definiciones.upper_bound(std::string(1, fin));
          for (auto it = lo; it != hi; ++it)
              std::cout << it->first << ": " << it->second << '\n';
      }

      size_t contar() const { return definiciones.size(); }

      void cargar(const std::string& archivo) {
          // Simulación: cargar desde archivo
          std::cout << "Cargando desde " << archivo << "...\n";
      }

      void guardar(const std::string& archivo) const {
          std::cout << "Guardando en " << archivo << "...\n";
      }
  };

Uso de map para contar frecuencias:

  std::map<std::string, int> frecuencias;
  std::vector<std::string> palabras {"hola", "mundo", "hola", "c++"};

  for (const auto& p : palabras) {
      frecuencias[p]++;  // inserta con 0 si no existe, luego incrementa
  }

  for (const auto& [palabra, count] : frecuencias) {
      std::cout << palabra << ": " << count << '\n';
  }

Uso de multimap para agrupar:

  std::multimap<std::string, std::string> agenda;
  agenda.insert({"Ana", "111-1111"});
  agenda.insert({"Ana", "222-2222"});
  agenda.insert({"Luis", "333-3333"});
  agenda.insert({"Ana", "444-4444"});

  auto [ini, fin] = agenda.equal_range("Ana");
  std::cout << "Teléfonos de Ana:\n";
  for (auto it = ini; it != fin; ++it)
      std::cout << "  " << it->second << '\n';

Diferencias entre map y multimap:

  map                    multimap
  ─────────────────────────────────────────
  Claves únicas          Claves duplicadas
  operator[]             No operator[]
  Sin duplicate keys     Varias claves iguales
  insert devuelve         insert devuelve
  pair<iterator,bool>     iterator
  Ideal para diccionario  Ideal para agrupación

```cpp
//==============================================================
// EJEMPLO COMPLETO: SISTEMA DE GESTIÓN DE EMPLEADOS CON MAP
//==============================================================
#include <iostream>
#include <map>
#include <string>
#include <vector>
#include <algorithm>
#include <iomanip>

struct Empleado {
    int id;
    std::string nombre;
    std::string departamento;
    double salario;
    int aniosExperiencia;

    Empleado(int i, std::string n, std::string d, double s, int a)
        : id(i), nombre(std::move(n)), departamento(std::move(d)),
          salario(s), aniosExperiencia(a) {}
};

class GestionEmpleados {
private:
    // Mapa ID → Empleado
    std::map<int, Empleado> empleados;

    // Mapas de índice para búsqueda rápida
    std::map<std::string, std::vector<int>> porDepartamento;
    std::multimap<double, int> porSalario;
    std::multimap<int, int> porExperiencia;

public:
    void agregarEmpleado(const Empleado& emp) {
        auto [it, inserted] = empleados.try_emplace(emp.id, emp);
        if (!inserted) {
            std::cout << "Error: ID " << emp.id << " ya existe.\n";
            return;
        }
        porDepartamento[emp.departamento].push_back(emp.id);
        porSalario.insert({emp.salario, emp.id});
        porExperiencia.insert({emp.aniosExperiencia, emp.id});
        std::cout << "Empleado " << emp.nombre << " agregado.\n";
    }

    void eliminarEmpleado(int id) {
        auto it = empleados.find(id);
        if (it == empleados.end()) {
            std::cout << "Empleado no encontrado.\n";
            return;
        }

        const auto& emp = it->second;

        // Eliminar de índices
        auto& vec = porDepartamento[emp.departamento];
        vec.erase(std::remove(vec.begin(), vec.end(), id), vec.end());
        if (vec.empty())
            porDepartamento.erase(emp.departamento);

        // Eliminar de mapas multimap
        auto rango = porSalario.equal_range(emp.salario);
        for (auto rit = rango.first; rit != rango.second; ++rit) {
            if (rit->second == id) {
                porSalario.erase(rit);
                break;
            }
        }

        rango = porExperiencia.equal_range(emp.aniosExperiencia);
        for (auto rit = rango.first; rit != rango.second; ++rit) {
            if (rit->second == id) {
                porExperiencia.erase(rit);
                break;
            }
        }

        empleados.erase(it);
        std::cout << "Empleado " << id << " eliminado.\n";
    }

    void actualizarSalario(int id, double nuevoSalario) {
        auto it = empleados.find(id);
        if (it == empleados.end()) {
            std::cout << "Empleado no encontrado.\n";
            return;
        }

        auto& emp = it->second;

        // Actualizar índice de salarios
        auto rango = porSalario.equal_range(emp.salario);
        for (auto rit = rango.first; rit != rango.second; ++rit) {
            if (rit->second == id) {
                porSalario.erase(rit);
                break;
            }
        }
        emp.salario = nuevoSalario;
        porSalario.insert({nuevoSalario, id});
        std::cout << "Salario actualizado.\n";
    }

    Empleado* buscarPorId(int id) {
        auto it = empleados.find(id);
        if (it != empleados.end())
            return &(it->second);
        return nullptr;
    }

    std::vector<Empleado> buscarPorDepartamento(
        const std::string& dept) const {
        std::vector<Empleado> resultado;
        auto it = porDepartamento.find(dept);
        if (it != porDepartamento.end()) {
            for (int id : it->second) {
                auto emp = empleados.find(id);
                if (emp != empleados.end())
                    resultado.push_back(emp->second);
            }
        }
        return resultado;
    }

    std::vector<Empleado> buscarPorRangoSalarial(
        double min, double max) const {
        std::vector<Empleado> resultado;
        auto lo = porSalario.lower_bound(min);
        auto hi = porSalario.upper_bound(max);
        for (auto it = lo; it != hi; ++it) {
            auto emp = empleados.find(it->second);
            if (emp != empleados.end())
                resultado.push_back(emp->second);
        }
        return resultado;
    }

    std::vector<Empleado> topNMejoresSalarios(int n) const {
        std::vector<Empleado> resultado;
        auto it = porSalario.rbegin();
        for (int i = 0; i < n && it != porSalario.rend(); ++i, ++it) {
            auto emp = empleados.find(it->second);
            if (emp != empleados.end())
                resultado.push_back(emp->second);
        }
        return resultado;
    }

    void mostrarTodos() const {
        std::cout << "\n=== LISTA DE EMPLEADOS ===\n";
        std::cout << std::left << std::setw(5) << "ID"
                  << std::setw(20) << "Nombre"
                  << std::setw(15) << "Departamento"
                  << std::setw(10) << "Salario"
                  << "Experiencia\n";
        std::cout << std::string(65, '-') << '\n';
        for (const auto& [id, emp] : empleados) {
            std::cout << std::left << std::setw(5) << emp.id
                      << std::setw(20) << emp.nombre
                      << std::setw(15) << emp.departamento
                      << std::right << std::setw(8)
                      << std::fixed << std::setprecision(2)
                      << emp.salario
                      << std::setw(10) << emp.aniosExperiencia
                      << " años\n";
        }
    }

    void mostrarEstadisticas() const {
        if (empleados.empty()) {
            std::cout << "No hay empleados.\n";
            return;
        }

        double sumaSalarios = 0;
        int maxExp = 0;
        std::string deptMasGrande;
        size_t maxDeptSize = 0;

        for (const auto& [id, emp] : empleados) {
            sumaSalarios += emp.salario;
            if (emp.aniosExperiencia > maxExp)
                maxExp = emp.aniosExperiencia;
        }

        for (const auto& [dept, ids] : porDepartamento) {
            if (ids.size() > maxDeptSize) {
                maxDeptSize = ids.size();
                deptMasGrande = dept;
            }
        }

        std::cout << "\n=== ESTADÍSTICAS ===\n";
        std::cout << "Total empleados: " << empleados.size() << '\n';
        std::cout << "Salario promedio: $"
                  << sumaSalarios / empleados.size() << '\n';
        std::cout << "Departamentos: " << porDepartamento.size() << '\n';
        std::cout << "Depto más grande: " << deptMasGrande
                  << " (" << maxDeptSize << ")\n";
        std::cout << "Mayor experiencia: " << maxExp << " años\n";
    }
};

int main() {
    GestionEmpleados ge;

    ge.agregarEmpleado(Empleado(101, "Ana Pérez",
                        "Ventas", 45000, 5));
    ge.agregarEmpleado(Empleado(102, "Luis Gómez",
                        "IT", 65000, 8));
    ge.agregarEmpleado(Empleado(103, "Carlos Ruiz",
                        "Ventas", 42000, 3));
    ge.agregarEmpleado(Empleado(104, "María Díaz",
                        "RH", 48000, 6));
    ge.agregarEmpleado(Empleado(105, "Eva Torres",
                        "IT", 72000, 10));
    ge.agregarEmpleado(Empleado(106, "Juan Mora",
                        "IT", 55000, 4));

    ge.mostrarTodos();

    std::cout << "\nBuscar empleado 103:\n";
    auto* emp = ge.buscarPorId(103);
    if (emp)
        std::cout << emp->nombre << " - " << emp->departamento << '\n';

    std::cout << "\nEmpleados en IT:\n";
    auto itStaff = ge.buscarPorDepartamento("IT");
    for (const auto& e : itStaff)
        std::cout << "  " << e.nombre << '\n';

    std::cout << "\nEmpleados con salario entre 40000 y 55000:\n";
    auto rangoSal = ge.buscarPorRangoSalarial(40000, 55000);
    for (const auto& e : rangoSal)
        std::cout << "  " << e.nombre << " - $" << e.salario << '\n';

    std::cout << "\nTop 3 mejores salarios:\n";
    auto top = ge.topNMejoresSalarios(3);
    for (const auto& e : top)
        std::cout << "  " << e.nombre << " - $" << e.salario << '\n';

    std::cout << "\nActualizar salario de 102 a 70000:\n";
    ge.actualizarSalario(102, 70000);
    ge.mostrarTodos();

    std::cout << "\nEliminar empleado 106:\n";
    ge.eliminarEmpleado(106);
    ge.mostrarTodos();

    ge.mostrarEstadisticas();

    return 0;
}
```

================================================================================
  RESUMEN DEL CAPÍTULO
================================================================================

Puntos clave de map:
- Almacena pares clave-valor (std::pair<const Key, Value>).
- Claves únicas y orden automático (por defecto operator<).
- Búsqueda, inserción y eliminación O(log n).
- operator[] inserta con valor por defecto si la clave no existe.
- find devuelve iterador al elemento o end().
- erase puede eliminar por clave (devuelve count) o por iterador.

Puntos clave de multimap:
- Permite claves duplicadas.
- No tiene operator[] (no hay clave única).
- equal_range devuelve rango de iteradores para una clave.
- count devuelve el número de ocurrencias.

Cuándo usar map:
- Diccionarios (palabra → definición).
- Conteo de frecuencias.
- Índices y tablas de búsqueda.
- Cache simple (clave → valor).

Cuándo usar multimap:
- Agrupaciones (un autor → varios libros).
- Categorías con múltiples elementos.
- Índices secundarios.

================================================================================
  PREGUNTAS FRECUENTES (FAQ)
================================================================================

P: ¿Por qué map tiene búsqueda O(log n) en lugar de O(1)?
R: map se implementa con árbol rojo-negro, que mantiene los
   elementos ordenados. Para O(1) promedio, usa std::unordered_map
   (tabla hash, Capítulo extra). La ventaja de map es el orden.

P: ¿operator[] inserta un valor aunque solo acceda?
R: Sí. Si la clave no existe, operator[] la crea con valor por
   defecto. Para solo consultar, usa find() o contains().

P: ¿Puedo usar un tipo personalizado como clave?
R: Sí, siempre que tenga operator< (o proporciones un comparador).

P: ¿Qué devuelve insert en map?
R: Un std::pair<iterator, bool>. iterator apunta al elemento
   (existente o recién insertado). bool es true si se insertó.

P: ¿Cómo itero sobre multimap con una clave específica?
R: Usa equal_range(clave) que devuelve un par de iteradores
   [first, second) con todos los elementos de esa clave.

P: ¿Cuándo usar map vs unordered_map?
R: map cuando necesitas ordenación. unordered_map para mejor
   rendimiento O(1) promedio sin necesidad de orden.

P: ¿Qué pasa si modifico la clave de un elemento?
R: No se puede modificar directamente (es const). Para "modificar"
   la clave, extrae el nodo (extract), modifícalo, y reinserta.

Diseña e implementa un "Sistema de Gestión de Biblioteca" usando
std::map y std::multimap. El sistema debe:

1. Gestionar libros (ISBN único como clave):
   - Título, autor, año, género, copias totales, copias disponibles.
   - Usar map<string, Libro> donde string es el ISBN.

2. Gestionar usuarios:
   - ID único, nombre, email.
   - map<int, Usuario> para usuarios.

3. Préstamos: multimap<string, int> (ISBN → IDs de usuarios que
   tienen prestado ese libro). Un libro puede prestarse a varios.

4. Operaciones:
   - agregar_libro(): insert en map.
   - eliminar_libro(ISBN): find y erase.
   - registrar_usuario(): insert en map.
   - prestar(ISBN, usuarioID): verificar disponibilidad, insert
     en multimap de préstamos, decrementar copias disponibles.
   - devolver(ISBN, usuarioID): eliminar del multimap, incrementar
     copias.
   - buscar_por_titulo(): recorrer map.
   - buscar_por_autor(): usar multimap<string, string> (autor→ISBN).
   - libros_prestados_por(usuarioID): recorrer multimap de préstamos.
   - reporte_completo(): todos los libros con estado.

5. Índices secundarios con multimap:
   - autor → ISBN (multimap).
   - género → ISBN.
   - año → ISBN.

6. Persistencia opcional: cargar/guardar estado.

Este desafío integra map, multimap, pair, find, insert, erase,

secundarios.
