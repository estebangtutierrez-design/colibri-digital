Bienvenido al capítulo sobre std::set y std::multiset. Estos
contenedores asociativos almacenan elementos ordenados y únicos
(set) o con posibles duplicados (multiset). Son ideales para
verificar pertenencia, eliminar duplicados, y mantener colecciones
ordenadas con búsqueda eficiente O(log n).

std::set es un contenedor asociativo que almacena elementos únicos
y ordenados según un criterio de comparación. Internamente se
implementa como un árbol rojo-negro, ofreciendo O(log n) en
inserción, búsqueda y eliminación.

std::multiset permite elementos duplicados, manteniendo el orden.

La cabecera necesaria es <set>.

Declaración básica:
  #include <set>
  #include <string>

  std::set<int> s;                    // conjunto vacío de enteros
  std::set<int> s {3, 1, 4, 1, 5};   // {1, 3, 4, 5} (únicos y ordenados)
  std::set<std::string> nombres {"Ana", "Luis", "Carlos"};
  std::set<int> copia = s;

  std::multiset<int> ms;             // multiset vacío
  std::multiset<int> ms {1, 2, 2, 3, 3, 3};  // {1, 2, 2, 3, 3, 3}

Inserción:
  std::set<int> s;
  auto [it, inserted] = s.insert(10);  // inserted = true
  auto [it2, inserted2] = s.insert(10); // inserted2 = false

  // En multiset, insert siempre tiene éxito
  std::multiset<int> ms;
  auto it = ms.insert(10);  // siempre inserta, devuelve iterador
  ms.insert(10);  // ahora hay dos 10's

  // emplace (C++11)
  s.emplace(42);

  // Con pista (hint)
  s.insert(s.end(), 99);  // O(1) amortizado si la pista es correcta

Búsqueda:
  auto it = s.find(10);
  if (it != s.end())
      std::cout << "Encontrado: " << *it << '\n';

  // contains (C++20)
  if (s.contains(10))
      std::cout << "10 está en el conjunto\n";

  // count: en set es 0 o 1; en multiset puede ser > 1
  int n = s.count(10);    // 0 o 1
  int n2 = ms.count(10);  // número de ocurrencias

Eliminación:
  s.erase(10);              // por valor (devuelve 0 o 1)
  s.erase(it);              // por iterador
  s.erase(s.begin(), s.end());  // rango (equivale a clear)

  // En multiset, erase(valor) elimina TODAS las ocurrencias
  ms.erase(10);  // elimina todos los 10's

  // Para eliminar solo una ocurrencia en multiset:
  auto it_ms = ms.find(10);
  if (it_ms != ms.end())
      ms.erase(it_ms);

Iteración:
  for (const auto& elem : s)
      std::cout << elem << ' ';

  for (auto it = s.rbegin(); it != s.rend(); ++it)
      std::cout << *it << ' ';

  for (int x : s)
      std::cout << x << ' ';

Operaciones de rango (lower_bound, upper_bound, equal_range):

  std::set<int> s {1, 3, 5, 7, 9, 11};

  auto lo = s.lower_bound(5);   // primer elemento >= 5 -> 5
  auto hi = s.upper_bound(5);   // primer elemento > 5 -> 7

  for (auto it = s.lower_bound(3); it != s.upper_bound(8); ++it)
      std::cout << *it << ' ';  // 3, 5, 7

  auto [inicio, fin] = s.equal_range(5);
  // inicio apunta a 5, fin apunta a 7

Comparador personalizado:
  std::set<int, std::greater<int>> s;  // orden descendente

  struct PorLongitud {
      bool operator()(const std::string& a,
                      const std::string& b) const {
          if (a.size() != b.size())
              return a.size() < b.size();
          return a < b;
      }
  };

  std::set<std::string, PorLongitud> palabras;
  palabras.insert("casa");
  palabras.insert("sol");
  palabras.insert("arbol");
  // Orden: "sol"(3), "casa"(4), "arbol"(5)

Operaciones de conjunto (con <algorithm>):

  std::set<int> a {1, 2, 3, 4, 5};
  std::set<int> b {3, 4, 5, 6, 7};

  // Unión
  std::set<int> union_set;
  std::set_union(a.begin(), a.end(), b.begin(), b.end(),
                 std::inserter(union_set, union_set.end()));
  // union_set: {1,2,3,4,5,6,7}

  // Intersección
  std::set<int> interseccion;
  std::set_intersection(a.begin(), a.end(), b.begin(), b.end(),
                        std::inserter(interseccion, interseccion.end()));
  // interseccion: {3,4,5}

  // Diferencia
  std::set<int> diferencia;
  std::set_difference(a.begin(), a.end(), b.begin(), b.end(),
                      std::inserter(diferencia, diferencia.end()));
  // diferencia: {1,2}

  // Diferencia simétrica
  std::set<int> diff_simetrica;
  std::set_symmetric_difference(a.begin(), a.end(), b.begin(), b.end(),
                               std::inserter(diff_simetrica,
                                             diff_simetrica.end()));
  // diff_simetrica: {1,2,6,7}

  // Inclusión
  std::set<int> sub {3, 4};
  bool esSubconjunto = std::includes(a.begin(), a.end(),
                                     sub.begin(), sub.end());  // true

  bool iguales = (a == b);  // false

Otras funciones de set:
  s.size();      // número de elementos
  s.empty();     // true si está vacío
  s.clear();     // elimina todos
  s.max_size();  // máximo teórico
  s.swap(s2);    // intercambiar

  auto comp = s.key_comp();
  bool antes = comp(1, 2);  // true (1 < 2)

Extract (C++17): extraer nodo para reinsertar en otro set.
  std::set<int> s1 {1, 2, 3};
  std::set<int> s2 {4, 5, 6};

  auto nodo = s1.extract(2);
  if (!nodo.empty()) {
      nodo.value() = 99;
      s2.insert(std::move(nodo));
  }
  // s1: {1, 3}, s2: {4, 5, 6, 99}

Merge (C++17): fusiona dos sets.
  std::set<int> a1 {1, 2, 3};
  std::set<int> b1 {3, 4, 5};
  a1.merge(b1);
  // a1: {1,2,3,4,5}, b1: vacío (3 ya estaba en a1)

Uso de set para eliminar duplicados:
  std::vector<int> v {3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5};
  std::set<int> unicos(v.begin(), v.end());
  // unicos: {1, 2, 3, 4, 5, 6, 9}

  // Convertir de vuelta a vector
  v.assign(unicos.begin(), unicos.end());

Uso de set para verificar pertenencia:
  std::set<std::string> diccionario {"hola", "mundo", "casa"};
  if (diccionario.contains("hola"))
      std::cout << "Palabra válida\n";

Uso de multiset para estadísticas:
  std::multiset<int> notas {85, 90, 78, 85, 92, 85, 88};

  auto [min, max] = std::minmax_element(notas.begin(), notas.end());
  std::cout << "Min: " << *min << ", Max: " << *max << '\n';

  auto it = notas.lower_bound(80);
  std::cout << "Notas >= 80: " << std::distance(it, notas.end()) << '\n';

Ejemplo completo: Sistema de gestión de estudiantes con set.

  #include <iostream>
  #include <set>
  #include <string>
  #include <vector>
  #include <algorithm>

  class Curso {
  private:
      std::string nombreCurso;
      std::set<std::string> estudiantes;
      std::multiset<int> calificaciones;

  public:
      Curso(std::string nombre) : nombreCurso(std::move(nombre)) {}

      void inscribir(const std::string& estudiante) {
          auto [it, inserted] = estudiantes.insert(estudiante);
          if (inserted)
              std::cout << estudiante << " inscrito en "
                        << nombreCurso << '\n';
          else
              std::cout << estudiante << " ya estaba inscrito\n";
      }

      void desinscribir(const std::string& estudiante) {
          if (estudiantes.erase(estudiante))
              std::cout << estudiante << " eliminado del curso\n";
          else
              std::cout << estudiante << " no encontrado\n";
      }

      void agregarCalificacion(int nota) {
          calificaciones.insert(nota);
      }

      void agregarCalificacion(const std::string& estudiante, int nota) {
          if (estudiantes.contains(estudiante)) {
              calificaciones.insert(nota);
              std::cout << "Calificación " << nota
                        << " agregada para " << estudiante << '\n';
          } else {
              std::cout << "Estudiante no inscrito\n";
          }
      }

      bool estaInscrito(const std::string& estudiante) const {
          return estudiantes.contains(estudiante);
      }

      void listarEstudiantes() const {
          std::cout << "\nEstudiantes de " << nombreCurso << ":\n";
          for (const auto& e : estudiantes)
              std::cout << "  - " << e << '\n';
          std::cout << "Total: " << estudiantes.size() << '\n';
      }

      void unirCon(const Curso& otro) {
          std::set<std::string> combinado;
          std::set_union(estudiantes.begin(), estudiantes.end(),
                        otro.estudiantes.begin(), otro.estudiantes.end(),
                        std::inserter(combinado, combinado.end()));
          estudiantes = combinado;
      }

      void soloEnAmbos(const Curso& otro) {
          std::set<std::string> comunes;
          std::set_intersection(estudiantes.begin(), estudiantes.end(),
                              otro.estudiantes.begin(), otro.estudiantes.end(),
                              std::inserter(comunes, comunes.end()));
          estudiantes = comunes;
      }

      void mostrarEstadisticas() const {
          if (calificaciones.empty()) {
              std::cout << "No hay calificaciones\n";
              return;
          }

          double suma = 0;
          for (int n : calificaciones)
              suma += n;

          auto [min, max] = std::minmax_element(
              calificaciones.begin(), calificaciones.end());

          std::cout << "\nEstadísticas de calificaciones:\n";
          std::cout << "Cantidad: " << calificaciones.size() << '\n';
          std::cout << "Promedio: " << suma / calificaciones.size() << '\n';
          std::cout << "Mínima: " << *min << '\n';
          std::cout << "Máxima: " << *max << '\n';
      }

      size_t totalEstudiantes() const { return estudiantes.size(); }
  };

Rendimiento y consideraciones:
- Inserción: O(log n)
- Búsqueda: O(log n)
- Eliminación: O(log n)
- Iteración: O(n) (ordenada)

Para conjuntos pequeños (n < 100), la diferencia entre set y
vector es despreciable. set brilla cuando el conjunto es grande
y hay muchas operaciones de búsqueda.

Alternativas:
- std::unordered_set: O(1) promedio, sin orden (tabla hash).
- std::vector + sort/unique: mejor para conjuntos estáticos.
- std::set: para conjuntos dinámicos que requieren orden.

class Estudiante {
public:
    std::string nombre;
    int id;
    int edad;

    Estudiante(std::string n, int i, int e)
        : nombre(std::move(n)), id(i), edad(e) {}

    bool operator<(const Estudiante& otro) const {
        if (nombre != otro.nombre)
            return nombre < otro.nombre;
        if (id != otro.id)
            return id < otro.id;
        return edad < otro.edad;

        return id == otro.id;

================================================================================
  RESUMEN DEL CAPÍTULO
================================================================================

Puntos clave de std::set:
- Almacena elementos únicos ordenados.
- Búsqueda, inserción y eliminación O(log n).
- Los elementos son const (no modificables in-situ).
- Se requiere operator< o un comparador personalizado.
- Soporta operaciones de conjunto: unión, intersección, diferencia.

Puntos clave de std::multiset:
- Permite elementos duplicados.
- erase(valor) elimina todas las ocurrencias.
- Para eliminar una sola ocurrencia, usar find + erase(iterador).
- equal_range devuelve el rango de una clave.

Cuándo usar set:
- Verificar pertenencia a un conjunto.
- Eliminar duplicados de una colección.
- Mantener elementos ordenados dinámicamente.
- Operaciones de conjunto (unión, intersección).

Cuándo usar multiset:
- Contar frecuencias con orden.
- Almacenar múltiples valores iguales ordenados.
- Estadísticas ordenadas (percentiles, medianas).

================================================================================
  PREGUNTAS FRECUENTES (FAQ)
================================================================================

P: ¿Por qué set tiene búsqueda O(log n) en lugar de O(1)?
R: set usa un árbol balanceado (rojo-negro) que mantiene orden.
   Para O(1) promedio, usa std::unordered_set (tabla hash).

P: ¿Puedo modificar un elemento dentro de un set?
R: No directamente (los elementos son const). Debes extraer el
   nodo con extract() (C++17), modificarlo y reinsertarlo.

P: ¿Cuándo usar set vs unordered_set?
R: set cuando necesitas ordenación o rangos. unordered_set cuando
   solo necesitas pertenencia O(1) promedio.

P: ¿Qué operación es más rápida: find en set o en vector?
R: En set es O(log n). En vector no ordenado es O(n). En vector
   ordenado con binary_search es O(log n) pero no permite
   inserciones eficientes.

P: ¿Cómo itero solo una parte del set?
R: Usa lower_bound y upper_bound para obtener rangos:
   for (auto it = s.lower_bound(5); it != s.upper_bound(10); ++it)

P: ¿set acepta tipos personalizados?
R: Sí, siempre que definas operator< o proveas un comparador.

P: ¿Qué diferencia hay entre set y multiset en erase?
R: set.erase(key) elimina 0 o 1 elementos. multiset.erase(key)
   elimina TODAS las ocurrencias.

Diseña e implementa un "Sistema de Recomendación de Contenido"
usando std::set y std::multiset. El sistema debe:

1. Gestionar usuarios (set de IDs).
2. Gestionar contenido (películas, series, libros) con:
   - ID único, título, género, año, calificación promedio.
   - Almacenar en std::map<int, Contenido>.

3. Preferencias de usuarios: set<int> por cada usuario (gustos).
   - "A" le gusta X, Y, Z.
   - Almacenar en std::map<int, std::set<int>> gustos.

4. Sistema de calificaciones: multiset<pair<int,int>> (usuario,contenido)
   para calificaciones. O multiset del contenido ID.

5. Funcionalidades:
   - agregar_usuario(), eliminar_usuario()
   - agregar_contenido(), eliminar_contenido()
   - gustar(usuarioID, contenidoID) -> insert en set del usuario
   - no_gustar(usuarioID, contenidoID) -> erase del set
   - recomendar_para(usuarioID):
       * Encontrar usuarios con gustos similares (intersección de sets)
       * Recomendar contenido que les guste a ellos y no al usuario
       * Ordenar por popularidad
   - contenido_popular(): top 10 contenido más gustado
   - contenido_similar(contenidoID): contenido que gusta a los
     mismos usuarios

6. Criterio de similitud: dos usuarios son similares si la
   intersección de sus sets de gustos tiene al menos 3 elementos.

7. El algoritmo de recomendación debe:
   - Encontrar usuarios similares (intersección de sets)
   - Unir sus gustos (unión de sets)
   - Restar los gustos del usuario (diferencia de sets)
   - Ordenar por frecuencia en el multiset de calificaciones

Este desafío integra set, multiset, map, insert, find, erase,
contains, lower_bound, unión, intersección, diferencia de
conjuntos, y comparadores personalizados.
