===============================================================================
1. EL FLUJO DE SALIDA: cout
===============================================================================

cout (character output stream) es el flujo de salida estándar.
Por defecto, se envía a la consola.

1.1. SINTAXIS BÁSICA

```cpp
#include <iostream>

int main() {
    // Insercion multiple
    std::cout << "Hola" << " " << "Mundo" << std::endl;

    // Diferentes tipos
    std::cout << "Entero: " << 42 << std::endl;
    std::cout << "Flotante: " << 3.14159 << std::endl;
    std::cout << "Booleano: " << true << std::endl;
    std::cout << "Caracter: " << 'A' << std::endl;
    std::cout << "Puntero: " << &std::cout << std::endl;

    return 0;
}
```

1.2. cerr Y clog

```cpp
#include <iostream>

int main() {
    // cerr: salida de error (no bufferizado)
    std::cerr << "Error critico!" << std::endl;

    // clog: salida de log (bufferizado)
    std::clog << "Informacion de log" << std::endl;

    // Diferencia: cerr se muestra inmediatamente
    // clog puede tener retraso por buffer

    return 0;
}
```

1.3. endl VS \n

```cpp
#include <iostream>

int main() {
    // endl: salto de linea + flush (vacia el buffer)
    std::cout << "Linea 1" << std::endl;

    // \n: solo salto de linea (mas rapido)
    std::cout << "Linea 2\n";

    // flush manual
    std::cout << "Sin salto de linea" << std::flush;

    // Cuando usar cada uno:
    // - endl: cuando necesitas mostrar el texto inmediatamente
    // - \n: en la mayoria de los casos (mas eficiente)
    // - flush: para barras de progreso, indicadores

    return 0;
}
```

2. MANIPULADORES DE FORMATO

C++ proporciona manipuladores para controlar el formato de salida.

2.1. boolalpha Y noboolalpha

```cpp
#include <iostream>

int main() {
    std::cout << "Por defecto: " << true << " " << false << std::endl;

    std::cout << std::boolalpha;
    std::cout << "boolalpha: " << true << " " << false << std::endl;

    std::cout << std::noboolalpha;
    std::cout << "noboolalpha: " << true << " " << false << std::endl;

    return 0;
}
```

2.2. Bases numéricas: hex, oct, dec

```cpp
#include <iostream>

int main() {
    int valor = 255;

    std::cout << "Decimal:  " << std::dec << valor << std::endl;
    std::cout << "Hexadecimal: " << std::hex << valor << std::endl;
    std::cout << "Octal:    " << std::oct << valor << std::endl;

    // showbase: muestra el prefijo (0x, 0)
    std::cout << std::showbase;
    std::cout << "Hex con base: " << std::hex << valor << std::endl;
    std::cout << "Oct con base: " << std::oct << valor << std::endl;

    // uppercase para hex
    std::cout << std::uppercase;
    std::cout << "Hex mayus: " << std::hex << valor << std::endl;

    // Restaurar a decimal
    std::cout << std::dec << std::noshowbase << std::nouppercase;

    return 0;
}
```

2.3. Formato de punto flotante: fixed, scientific, setprecision

```cpp
#include <iostream>
#include <iomanip>

int main() {
    double pi = 3.14159265358979;
    double grande = 1234567.89;
    double chico = 0.000123456;

    std::cout << "=== Por defecto ===" << std::endl;
    std::cout << "pi: " << pi << std::endl;
    std::cout << "grande: " << grande << std::endl;
    std::cout << "chico: " << chico << std::endl;

    std::cout << "\n=== fixed ===" << std::endl;
    std::cout << std::fixed;
    std::cout << "pi: " << pi << std::endl;
    std::cout << "grande: " << grande << std::endl;

    std::cout << "\n=== scientific ===" << std::endl;
    std::cout << std::scientific;
    std::cout << "pi: " << pi << std::endl;
    std::cout << "chico: " << chico << std::endl;

    std::cout << "\n=== setprecision ===" << std::endl;
    std::cout << std::fixed << std::setprecision(2);
    std::cout << "pi (2 decimales): " << pi << std::endl;
    std::cout << "pi (4 decimales): " << std::setprecision(4) << pi << std::endl;
    std::cout << "pi (10 decimales): " << std::setprecision(10) << pi << std::endl;

    return 0;
}
```

2.4. setw, setfill Y alineación

```cpp
#include <iostream>
#include <iomanip>

int main() {
    // setw: ancho minimo del campo
    std::cout << "=== setw ===" << std::endl;
    std::cout << "|" << std::setw(10) << 42 << "|" << std::endl;
    std::cout << "|" << std::setw(10) << "Hola" << "|" << std::endl;

    // setfill: caracter de relleno
    std::cout << "\n=== setfill ===" << std::endl;
    std::cout << std::setfill('*');
    std::cout << "|" << std::setw(10) << 42 << "|" << std::endl;

    // Alineacion: left, right, internal
    std::cout << "\n=== Alineacion ===" << std::endl;
    std::cout << std::setfill(' ');

    std::cout << std::left;
    std::cout << "|" << std::setw(10) << "Izquierda" << "|" << std::endl;

    std::cout << std::right;
    std::cout << "|" << std::setw(10) << "Derecha" << "|" << std::endl;

    std::cout << std::internal;
    std::cout << std::showpos;
    std::cout << "|" << std::setw(10) << 42 << "|" << std::endl;

    return 0;
}
```

3.2. cin.get() Y cin.ignore()

```cpp
#include <iostream>

int main() {
    char c1, c2;

    std::cout << "Ingresa dos caracteres: ";

    // cin.get() lee un solo caracter (incluyendo espacios)
    c1 = std::cin.get();
    c2 = std::cin.get();

    std::cout << "caracter 1: '" << c1 << "'" << std::endl;
    std::cout << "caracter 2: '" << c2 << "'" << std::endl;

    // cin.ignore() descarta caracteres en el buffer
    std::cout << "\n--- cin.ignore ---" << std::endl;

    int numero;
    std::string texto;

    std::cout << "Numero: ";
    std::cin >> numero;

    // Ignorar hasta el proximo salto de linea
    std::cin.ignore(10000, '\n');

    std::cout << "Texto: ";
    std::getline(std::cin, texto);

    std::cout << "Numero: " << numero << ", Texto: " << texto << std::endl;

    return 0;
}
```

3.3. LECTURA DE LÍNEAS COMPLETAS: getline

```cpp
#include <iostream>
#include <string>
#include <limits>

int main() {
    std::string linea;

    std::cout << "Escribe una linea (con espacios): ";
    std::getline(std::cin, linea);
    std::cout << "Leiste: " << linea << std::endl;

    // Problema tipico: mezclar cin >> con getline
    int edad;
    std::string nombre;

    std::cout << "\nEdad: ";
    std::cin >> edad;

    // Limpiar el buffer de entrada
    std::cin.ignore(std::numeric_limits<std::streamsize>::max(), '\n');

    std::cout << "Nombre completo: ";
    std::getline(std::cin, nombre);

    std::cout << "\n" << edad << " - " << nombre << std::endl;

    return 0;
}
```

3.4. ESTADOS DEL FLUJO

```cpp
#include <iostream>
#include <limits>

int main() {
    int numero;

    std::cout << "Ingresa un numero entero: ";
    std::cin >> numero;

    // Verificar estados del flujo
    if (std::cin.good()) {
        std::cout << "Entrada correcta: " << numero << std::endl;
    } else if (std::cin.fail()) {
        std::cout << "Error: entrada invalida!" << std::endl;
        std::cin.clear();  // Limpiar estado de error
        std::cin.ignore(std::numeric_limits<std::streamsize>::max(), '\n');
    } else if (std::cin.bad()) {
        std::cout << "Error grave del flujo!" << std::endl;
    } else if (std::cin.eof()) {
        std::cout << "Fin de archivo alcanzado" << std::endl;
    }

    return 0;
}
```

4.1. ESCRITURA EN ARCHIVO

```cpp
#include <iostream>
#include <fstream>
#include <string>

int main() {
    // ofstream: output file stream

```

4.2. LECTURA DE ARCHIVO

```cpp
#include <iostream>
#include <fstream>
#include <string>

int main() {
    std::ifstream archivoEntrada("datos.txt");

    if (!archivoEntrada.is_open()) {
        std::cerr << "Error al abrir el archivo!" << std::endl;
        return 1;
    }

    // Lectura linea por linea
    std::string linea;
    int numLinea = 0;

    while (std::getline(archivoEntrada, linea)) {
        numLinea++;
        std::cout << "Linea " << numLinea << ": " << linea << std::endl;
    }

    archivoEntrada.close();

    return 0;
}
```

4.3. LECTURA/ESCRITURA (fstream)

```cpp
#include <iostream>
#include <fstream>
#include <string>

struct Producto {
    std::string nombre;
    double precio;
    int stock;
};

int main() {
    // Escribir productos
    std::ofstream out("productos.txt");
    Producto productos[] = {
        {"Laptop", 999.99, 15},
        {"Mouse", 25.50, 100},
        {"Teclado", 89.99, 42}
    };

    for (const auto& p : productos) {
        out << p.nombre << "," << p.precio
            << "," << p.stock << std::endl;
    }
    out.close();

    // Leer productos
    std::ifstream in("productos.txt");
    std::string linea;

    std::cout << "=== Productos leidos ===" << std::endl;
    while (std::getline(in, linea)) {
        // Parsear CSV simple
        size_t pos1 = linea.find(',');
        size_t pos2 = linea.find(',', pos1 + 1);

        std::string nombre = linea.substr(0, pos1);
        std::string precioStr = linea.substr(pos1 + 1, pos2 - pos1 - 1);
        std::string stockStr = linea.substr(pos2 + 1);

        double precio = std::stod(precioStr);
        int stock = std::stoi(stockStr);

        std::cout << nombre << " - $" << precio
                  << " - " << stock << " unds" << std::endl;
    }
    in.close();

    return 0;
}
```

4.4. MODOS DE APERTURA

| BANDERA          | DESCRIPCIÓN                           |
|------------------|---------------------------------------|
| ios::in          | Abrir para lectura (por defecto ifstream) |
| ios::out         | Abrir para escritura (por defecto ofstream) |
| ios::app         | Añadir al final (append mode)         |
| ios::ate         | Ir al final del archivo tras abrir    |
| ios::trunc       | Truncar archivo si existe             |
| ios::binary      | Modo binario (no texto)               |

```cpp
#include <iostream>
#include <fstream>

int main() {
    // Append mode: anade al final
    std::ofstream log("log.txt", std::ios::app);
    log << "Nueva entrada de log" << std::endl;
    log.close();

    // Binary mode
    std::ofstream bin("datos.bin", std::ios::binary);
    int numeros[] = {1, 2, 3, 4, 5};
    bin.write(reinterpret_cast<char*>(numeros), sizeof(numeros));
    bin.close();

    // Lectura binaria
    std::ifstream binIn("datos.bin", std::ios::binary);
    int leidos[5];
    binIn.read(reinterpret_cast<char*>(leidos), sizeof(leidos));
    for (int n : leidos) {
        std::cout << n << " ";
    }
    std::cout << std::endl;
    binIn.close();

    return 0;
}
```

```cpp
#include <iostream>
#include <sstream>
#include <string>

int main() {
    // Convertir string a numero
    std::string numeroStr = "42";
    std::stringstream ss(numeroStr);
    int numero;
    ss >> numero;
    std::cout << "Numero: " << numero + 10 << std::endl;

    // Convertir numero a string formateado
    std::stringstream ss2;
    ss2 << "El valor es: " << 3.14159;
    std::string resultado = ss2.str();
    std::cout << resultado << std::endl;

    // Parsear datos estructurados
    std::string datos = "Juan,25,1.75";
    std::stringstream ss3(datos);
    std::string nombre;
    int edad;
    double altura;

    std::getline(ss3, nombre, ',');
    ss3 >> edad;
    ss3.ignore();  // ignorar la coma
    ss3 >> altura;

    std::cout << nombre << " - " << edad << " - " << altura << std::endl;

    return 0;
}
```

8. ENTRADA SINCRONIZADA: cin vs getline
===============================================================================

```cpp
#include <iostream>
#include <string>
#include <limits>

int main() {
    // Escenario comun: menu numerico con opciones de texto
    int opcion;
    std::string nombre;

    while (true) {
        std::cout << "\n=== MENU ===" << std::endl;
        std::cout << "1. Ingresar nombre" << std::endl;
        std::cout << "2. Mostrar nombre" << std::endl;
        std::cout << "3. Salir" << std::endl;
        std::cout << "Opcion: ";
        std::cin >> opcion;

        if (std::cin.fail()) {
            std::cin.clear();
            std::cin.ignore(
                std::numeric_limits<std::streamsize>::max(), '\n');
            std::cout << "Opcion invalida!" << std::endl;
            continue;
        }

        std::cin.ignore(
            std::numeric_limits<std::streamsize>::max(), '\n');

        switch (opcion) {
            case 1:
                std::cout << "Nombre: ";
                std::getline(std::cin, nombre);
                break;
            case 2:
                if (nombre.empty()) {
                    std::cout << "No hay nombre ingresado" << std::endl;
                } else {
                    std::cout << "Nombre: " << nombre << std::endl;
                }
                break;
            case 3:
                std::cout << "Adios!" << std::endl;
                return 0;
            default:
                std::cout << "Opcion no valida" << std::endl;
        }
    }

    return 0;
}
```

```cpp
#include <iostream>
#include <fstream>
#include <iomanip>
#include <vector>

struct Alumno {
    std::string nombre;
    int edad;
    double promedio;
};

int main() {
    // Escribir archivo CSV
    std::ofstream out("alumnos.csv");
    out << "Ana Garcia,22,9.5" << std::endl;
    out << "Luis Perez,25,8.2" << std::endl;
    out << "Maria Lopez,21,9.8" << std::endl;
    out.close();

    // Leer archivo CSV con formato
    std::ifstream in("alumnos.csv");
    std::vector<Alumno> alumnos;
    std::string linea;

    while (std::getline(in, linea)) {
        std::stringstream ss(linea);
        Alumno a;

        std::getline(ss, a.nombre, ',');
        ss >> a.edad;
        ss.ignore();
        ss >> a.promedio;

        alumnos.push_back(a);
    }
    in.close();

    // Mostrar tabla formateada
    std::cout << std::left;
    std::cout << std::setw(20) << "Nombre"
              << std::setw(8) << "Edad"
              << std::setw(10) << "Promedio" << std::endl;
    std::cout << std::string(38, '-') << std::endl;

    for (const auto& a : alumnos) {
        std::cout << std::setw(20) << a.nombre
                  << std::setw(8) << a.edad
                  << std::fixed << std::setprecision(2)
                  << std::setw(10) << a.promedio
                  << std::endl;
    }

    return 0;
}
```

10.1. VALIDAR SIEMPRE LA APERTURA DE ARCHIVOS

```cpp
std::ifstream archivo("datos.txt");
```

10.2. USAR getline PARA TEXTO CON ESPACIOS

```cpp
// Mal: solo lee una palabra
std::cin >> nombre;

// Bien: lee toda la linea
std::getline(std::cin, nombre);
```

10.3. LIMPIAR EL BUFFER TRAS cin >>

```cpp
std::cin >> numero;
std::cin.ignore(std::numeric_limits<std::streamsize>::max(), '\n');
```

10.4. CERRAR SIEMPRE LOS ARCHIVOS

Aunque el destructor lo hace automáticamente, cerrar explícitamente
libera recursos inmediatamente.

```cpp
archivo.close();
```

10.5. USAR cerr PARA ERRORES, clog PARA LOGS

```cpp
std::cerr << "Error critico!" << std::endl;  // Inmediato
std::clog << "Info de depuracion" << std::endl;  // Bufferizado
```

===============================================================================
RESUMEN DEL CAPITULO
===============================================================================

- cout es el flujo de salida estandar; cerr para errores, clog para logs.
- endl inserta newline + flush; \n solo newline (mas eficiente).
- Manipuladores: setw, setprecision, fixed, scientific, boolalpha, etc.
- cin >> ignora espacios; getline lee toda la linea.
- getline necesita limpiar el buffer tras usar cin >>.
- Los flujos tienen estados: good, fail, bad, eof.
- fstream permite leer/escribir archivos (ifstream, ofstream).
- stringstream trata strings como flujos de E/S.
- Modos: in, out, app, binary, trunc, ate.
- Valida siempre que la entrada sea correcta.

===============================================================================
PREGUNTAS FRECUENTES
===============================================================================

P: Cual es la diferencia entre endl y '\n'?
R: endl inserta un salto de linea Y limpia el buffer (flush).
'\n' solo inserta el salto de linea. endl es util cuando
necesitas mostrar el texto inmediatamente; \n es mas eficiente
en general.

P: Por que getline no funciona despues de cin >>?
R: cin >> deja un salto de linea en el buffer. getline lee
hasta el siguiente salto de linea, por lo que obtiene una
linea vacia. Solucion: usar cin.ignore() antes de getline.

P: Como leo un archivo completo en un string?
R: Puedes usar stringstream:
    std::ifstream in("archivo.txt");
    std::stringstream buffer;
    buffer << in.rdbuf();
    std::string contenido = buffer.str();

P: Que significa std::ios::app?
R: Append. Abre el archivo para escritura pero los datos se
anexan al final en lugar de sobrescribir el archivo existente.

P: Como detecto el fin de archivo?
R: La funcion getline devuelve false cuando llega al fin.
Tambien puedes verificar in.eof() o usar while (in >> variable).
