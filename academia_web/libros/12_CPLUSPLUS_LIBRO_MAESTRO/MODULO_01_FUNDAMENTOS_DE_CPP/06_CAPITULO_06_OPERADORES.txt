1. OPERADORES ARITMÉTICOS
===============================================================================

Los operadores aritméticos realizan operaciones matemáticas
básicas sobre tipos numéricos.

| OPERADOR | NOMBRE             | EJEMPLO  | RESULTADO |
|----------|--------------------|----------|-----------|
| +        | Suma               | a + b    | Suma      |
| -        | Resta              | a - b    | Diferencia|
| *        | Multiplicación     | a * b    | Producto  |
| /        | División           | a / b    | Cociente  |
| %        | Módulo (resto)     | a % b    | Resto     |

```cpp
#include <iostream>

int main() {
    int a = 10, b = 3;

    std::cout << "a + b = " << (a + b) << std::endl;
    std::cout << "a - b = " << (a - b) << std::endl;
    std::cout << "a * b = " << (a * b) << std::endl;
    std::cout << "a / b = " << (a / b) << std::endl;
    std::cout << "a % b = " << (a % b) << std::endl;

    return 0;
}
```

1.1. DIVISIÓN DE ENTEROS VS FLOTANTES

La división se comporta diferente según los tipos:

```cpp
#include <iostream>

int main() {
    int a = 10, b = 3;
    double c = 10.0, d = 3.0;

    std::cout << "int / int = " << (a / b) << std::endl;      // 3
    std::cout << "double / double = " << (c / d) << std::endl; // 3.33333
    std::cout << "int / double = " << (a / d) << std::endl;    // 3.33333
    std::cout << "double / int = " << (c / b) << std::endl;    // 3.33333

    // Division entera trunca hacia cero
    std::cout << "-10 / 3 = " << (-10 / 3) << std::endl;  // -3

    return 0;
}
```

1.2. MÓDULO (%)
El módulo solo funciona con tipos enteros. Devuelve el resto.

```cpp
#include <iostream>

int main() {
    std::cout << "10 % 3 = " << (10 % 3) << std::endl;    // 1
    std::cout << "15 % 4 = " << (15 % 4) << std::endl;    // 3
    std::cout << "7 % 10 = " << (7 % 10) << std::endl;    // 7
    std::cout << "0 % 5 = " << (0 % 5) << std::endl;      // 0

    // Verificar si un numero es par
    int num = 42;
    if (num % 2 == 0) {
        std::cout << num << " es par" << std::endl;
    }

    // Resto con negativos (resultado tiene el signo del dividendo)
    std::cout << "-10 % 3 = " << (-10 % 3) << std::endl;  // -1
    std::cout << "10 % -3 = " << (10 % -3) << std::endl;   // 1

    return 0;
}
```

3. OPERADORES DE INCREMENTO Y DECREMENTO
===============================================================================

| OPERADOR | NOMBRE             | DESCRIPCIÓN                        |
|----------|--------------------|-------------------------------------|
| ++var    | Pre-incremento     | Incrementa, luego evalúa           |
| var++    | Post-incremento    | Evalúa, luego incrementa           |
| --var    | Pre-decremento     | Decrementa, luego evalúa           |
| var--    | Post-decremento    | Evalúa, luego decrementa           |

```cpp
#include <iostream>

int main() {
    int x = 5;
    int y = 5;

    // Pre-incremento
    std::cout << "Pre-incremento:" << std::endl;
    std::cout << "x = " << x << std::endl;
    std::cout << "++x = " << (++x) << std::endl;  // 6
    std::cout << "x despues = " << x << std::endl;  // 6

    // Post-incremento
    std::cout << "\nPost-incremento:" << std::endl;
    std::cout << "y = " << y << std::endl;
    std::cout << "y++ = " << (y++) << std::endl;  // 5 (evalua antes)
    std::cout << "y despues = " << y << std::endl;  // 6

    return 0;
}
```

3.1. DIFERENCIA PRÁCTICA

```cpp
#include <iostream>

int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int i = 0;
    int j = 0;

    // Post-incremento: usa el indice y luego avanza
    std::cout << "arr[i++] = " << arr[i++] << std::endl;  // 10, i = 1

    // Pre-incremento: avanza y luego usa
    ++j;
    std::cout << "arr[++j] = " << arr[++j] << std::endl;  // 30 (j=2, luego accede)

    std::cout << "i = " << i << ", j = " << j << std::endl;

    return 0;
}
```

3.2. TRAMPAS COMUNES

```cpp
#include <iostream>

int main() {
    int x = 5;

    // Comportamiento indefinido: modificar y usar la misma variable
    // int y = x++ + ++x;  // UB! No hacer!

    // Correcto: una sola operacion por expresion
    int y = x + 1;
    x++;

    std::cout << "y = " << y << ", x = " << x << std::endl;

    return 0;
}
```

4. OPERADORES RELACIONALES
===============================================================================

Comparan dos valores y devuelven un bool (true/false).

| OPERADOR | NOMBRE              | EJEMPLO   | RESULTADO            |
|----------|---------------------|-----------|----------------------|
| ==       | Igual a             | a == b    | true si iguales      |
| !=       | Diferente de        | a != b    | true si diferentes   |
| <        | Menor que           | a < b     | true si a < b        |
| >        | Mayor que           | a > b     | true si a > b        |
| <=       | Menor o igual que   | a <= b    | true si a <= b       |
| >=       | Mayor o igual que   | a >= b    | true si a >= b       |

```cpp
#include <iostream>

int main() {
    int a = 5, b = 10;

    std::cout << std::boolalpha;
    std::cout << "a == b: " << (a == b) << std::endl;
    std::cout << "a != b: " << (a != b) << std::endl;
    std::cout << "a < b:  " << (a < b)  << std::endl;
    std::cout << "a > b:  " << (a > b)  << std::endl;
    std::cout << "a <= b: " << (a <= b) << std::endl;
    std::cout << "a >= b: " << (a >= b) << std::endl;

    return 0;
}
```

4.1. COMPARACIÓN DE FLOTANTES

Nunca compares flotantes con == debido a errores de precisión.

```cpp
#include <iostream>
#include <cmath>

int main() {
    double a = 0.1 + 0.2;  // 0.30000000000000004
    double b = 0.3;

    // Mal
    if (a == b) {
        std::cout << "Iguales exactamente" << std::endl;
    } else {
        std::cout << "No son exactamente iguales" << std::endl;
        std::cout.precision(20);
        std::cout << "a = " << a << ", b = " << b << std::endl;
    }

    // Bien: comparar con epsilon
    const double EPSILON = 1e-9;
    if (std::fabs(a - b) < EPSILON) {
        std::cout << "Suficientemente iguales" << std::endl;
    }

    return 0;
}
```

5. OPERADORES LÓGICOS
===============================================================================

Operan sobre valores booleanos y producen un bool.

| OPERADOR | NOMBRE | EJEMPLO        | RESULTADO                     |
|----------|--------|----------------|-------------------------------|
| &&       | AND    | a && b         | true si ambos son true        |
| ||       | OR     | a || b         | true si al menos uno es true  |
| !        | NOT    | !a             | true si a es false            |

```cpp
#include <iostream>

int main() {
    bool verdad = true;
    bool falso = false;

    std::cout << std::boolalpha;
    std::cout << "true && true:  " << (verdad && verdad) << std::endl;
    std::cout << "true && false: " << (verdad && falso) << std::endl;
    std::cout << "false && true: " << (falso && verdad) << std::endl;
    std::cout << "false && false: " << (falso && falso) << std::endl;

    std::cout << "\ntrue || true:  " << (verdad || verdad) << std::endl;
    std::cout << "true || false: " << (verdad || falso) << std::endl;
    std::cout << "false || false: " << (falso || falso) << std::endl;

    std::cout << "\n!true:  " << (!verdad) << std::endl;
    std::cout << "!false: " << (!falso) << std::endl;

    return 0;
}
```

5.1. EVALUACIÓN EN CORTOCIRCUITO

```cpp
#include <iostream>

bool funcionCostosa() {
    std::cout << "  funcionCostosa ejecutada" << std::endl;
    return true;
}

bool funcionRapida() {
    std::cout << "  funcionRapida ejecutada" << std::endl;
    return false;
}

int main() {
    // Cortocircuito con AND: si la primera es false, no evalua la segunda
    std::cout << "AND cortocircuito:" << std::endl;
    bool r1 = funcionRapida() && funcionCostosa();
    std::cout << "Resultado: " << r1 << "\n" << std::endl;

    // Cortocircuito con OR: si la primera es true, no evalua la segunda
    std::cout << "OR cortocircuito:" << std::endl;
    bool r2 = funcionRapida() || funcionCostosa();
    std::cout << "Resultado: " << r2 << std::endl;

    return 0;
}
```

5.2. USO PRÁCTICO DE CORTOCIRCUITO

```cpp
#include <iostream>

int main() {
    int divisor = 0;
    int dividendo = 10;

    // Cortocircuito evita la division por cero
    if (divisor != 0 && dividendo / divisor > 5) {
        std::cout << "Resultado > 5" << std::endl;
    } else {
        std::cout << "Division por cero evitada" << std::endl;
    }

    // Puntero nulo y acceso seguro
    int* ptr = nullptr;
    if (ptr != nullptr && *ptr == 42) {
        std::cout << "Valor apuntado es 42" << std::endl;
    } else {
        std::cout << "Puntero nulo, no se accede" << std::endl;
    }

    return 0;
}
```

6. OPERADORES BIT A BIT (BITWISE)
===============================================================================

Operan directamente sobre los bits de los valores enteros.

| OPERADOR | NOMBRE     | DESCRIPCIÓN                     |
|----------|------------|----------------------------------|
| &        | AND        | 1 si ambos bits son 1            |
| |        | OR         | 1 si al menos un bit es 1        |
| ^        | XOR        | 1 si los bits son diferentes     |
| ~        | NOT        | Invierte todos los bits          |
| <<       | Shift izq  | Desplaza bits a la izquierda     |
| >>       | Shift der  | Desplaza bits a la derecha       |

```cpp
#include <iostream>
#include <bitset>

int main() {
    unsigned char a = 0b1100;  // 12
    unsigned char b = 0b1010;  // 10

    std::cout << "a = " << std::bitset<4>(a) << " (" << (int)a << ")" << std::endl;
    std::cout << "b = " << std::bitset<4>(b) << " (" << (int)b << ")" << std::endl;

    std::cout << "a & b = " << std::bitset<4>(a & b)
              << " (" << (int)(a & b) << ")" << std::endl;  // 1000 (8)
    std::cout << "a | b = " << std::bitset<4>(a | b)
              << " (" << (int)(a | b) << ")" << std::endl;  // 1110 (14)
    std::cout << "a ^ b = " << std::bitset<4>(a ^ b)
              << " (" << (int)(a ^ b) << ")" << std::endl;  // 0110 (6)
    std::cout << "~a    = " << std::bitset<4>(~a & 0xF)
              << " (" << (int)(~a & 0xF) << ")" << std::endl; // 0011 (3)

    unsigned char c = 0b0001;
    std::cout << "\nc << 2 = " << std::bitset<4>(c << 2)
              << " (" << (int)(c << 2) << ")" << std::endl;  // 0100 (4)
    std::cout << "c >> 1 = " << std::bitset<4>(c >> 1)
              << " (" << (int)(c >> 1) << ")" << std::endl;  // 0000 (0)

    return 0;
}
```

6.1. USOS PRÁCTICOS DE BITWISE

```cpp
#include <iostream>

int main() {
    // Multiplicar/dividir por potencias de 2
    int x = 5;
    std::cout << "5 << 1 = " << (x << 1) << std::endl;  // 10 (5 * 2)
    std::cout << "5 << 2 = " << (x << 2) << std::endl;  // 20 (5 * 4)
    std::cout << "5 >> 1 = " << (x >> 1) << std::endl;  // 2  (5 / 2)

    // Verificar si un bit esta activo
    unsigned char flags = 0b1010;
    if (flags & 0b0010) {
        std::cout << "Bit 1 activo" << std::endl;
    }

    // Activar un bit
    flags |= 0b0100;
    std::cout << "flags = " << std::bitset<4>(flags) << std::endl;

    // Desactivar un bit
    flags &= ~0b0010;
    std::cout << "flags = " << std::bitset<4>(flags) << std::endl;

    // Alternar un bit (XOR)
    flags ^= 0b1000;
    std::cout << "flags = " << std::bitset<4>(flags) << std::endl;

    return 0;
}
```

7. OPERADOR TERNARIO (?:)
===============================================================================

El único operador ternario en C++. Es una forma compacta de if-else.

```cpp
sintaxis: condicion ? expresion_si_true : expresion_si_false

#include <iostream>

int main() {
    int edad = 18;

    // Uso basico
    std::string mensaje = (edad >= 18) ? "Mayor de edad" : "Menor de edad";
    std::cout << mensaje << std::endl;

    // Anidado (no recomendado para mas de 2 niveles)
    int nota = 85;
    char calificacion = (nota >= 90) ? 'A'
                      : (nota >= 80) ? 'B'
                      : (nota >= 70) ? 'C'
                      : (nota >= 60) ? 'D'
                      : 'F';
    std::cout << "Calificacion: " << calificacion << std::endl;

    // Con valores
    int a = 10, b = 20;
    int maximo = (a > b) ? a : b;
    std::cout << "Maximo: " << maximo << std::endl;

    return 0;
}
```

7.1. TERNARIO VS IF-ELSE

```cpp
// if-else (5 lineas)
int max;
if (a > b) {
    max = a;
} else {
    max = b;
}

// ternario (1 linea)
int max = (a > b) ? a : b;

// Cuando usar cada uno:
// Usa ternario para asignaciones simples
// Usa if-else para logica compleja o efectos secundarios
```

9. OPERADORES DE PUNTERO Y REFERENCIA
===============================================================================

| OPERADOR | NOMBRE          | DESCRIPCIÓN                        |
|----------|-----------------|-------------------------------------|
| &        | Dirección de    | Obtiene la dirección de memoria     |
| *        | Desreferencia   | Accede al valor apuntado            |
| ->       | Acceso miembro  | Accede a miembro de struct/clase ptr|
| .        | Acceso miembro  | Accede a miembro de struct/clase    |
| []       | Subíndice       | Accede a elemento de arreglo        |

```cpp
#include <iostream>

struct Punto {
    int x, y;
};

int main() {
    int valor = 42;
    int* ptr = &valor;       // & obtiene direccion

    std::cout << "valor: " << valor << std::endl;
    std::cout << "&valor: " << &valor << std::endl;
    std::cout << "ptr: " << ptr << std::endl;
    std::cout << "*ptr: " << *ptr << std::endl;  // desreferencia

    // Acceso a miembros
    Punto p = {10, 20};
    Punto* pPtr = &p;

    std::cout << "p.x: " << p.x << std::endl;         // operador .
    std::cout << "pPtr->x: " << pPtr->x << std::endl;  // operador ->

    // Subindice
    int arr[] = {1, 2, 3, 4, 5};
    std::cout << "arr[2]: " << arr[2] << std::endl;    // operador []

    return 0;
}
```

10. OPERADOR DE RESOLUCIÓN DE ÁMBITO (::)
===============================================================================

Accede a nombres en namespaces, clases y enumeraciones.

```cpp
#include <iostream>

int valor = 100;  // Variable global

namespace Matematicas {
    int suma(int a, int b) { return a + b; }
    const double PI = 3.14159;
}

class Calculadora {
public:
    static int version() { return 2; }
};

int main() {
    int valor = 50;  // Variable local

    // :: para acceder a ambito global
    std::cout << "Local: " << valor << std::endl;
    std::cout << "Global: " << ::valor << std::endl;

    // :: para namespace
    std::cout << "PI: " << Matematicas::PI << std::endl;
    std::cout << "Suma: " << Matematicas::suma(5, 3) << std::endl;

    // :: para miembros estaticos de clase
    std::cout << "Version: " << Calculadora::version() << std::endl;

    return 0;
}
```

12.1. TABLA DE PRECEDENCIA (DE MAYOR A MENOR)

| NIVEL | OPERADORES                          | ASOCIATIVIDAD |
|-------|-------------------------------------|---------------|
| 1     | :: (resolucion ambito)              | Izq -> Der    |
| 2     | () [] -> . ++ -- (post) typeid      | Izq -> Der    |
| 3     | ++ -- (pre) + - ! ~ & * new delete  | Der -> Izq    |
| 4     | .* ->* (puntero a miembro)          | Izq -> Der    |
| 5     | * / %                               | Izq -> Der    |
| 6     | + -                                 | Izq -> Der    |
| 7     | << >>                               | Izq -> Der    |
| 8     | < <= > >=                           | Izq -> Der    |
| 9     | == !=                               | Izq -> Der    |
| 10    | & (bitwise AND)                     | Izq -> Der    |
| 11    | ^ (bitwise XOR)                     | Izq -> Der    |
| 12    | | (bitwise OR)                      | Izq -> Der    |
| 13    | && (logical AND)                    | Izq -> Der    |
| 14    | || (logical OR)                     | Izq -> Der    |
| 15    | ?: (ternario)                       | Der -> Izq    |
| 16    | = += -= *= /= etc (asignacion)      | Der -> Izq    |
| 17    | , (coma)                            | Izq -> Der    |

12.2. EJEMPLOS DE PRECEDENCIA

```cpp
#include <iostream>

int main() {
    // Ejemplo 1: aritmetica vs relacional
    int a = 5, b = 10, c = 15;

    // Sin parentesis: dificil de leer
    if (a + b > c && b - c < a) {
        std::cout << "Expresion compleja" << std::endl;
    }

    // Con parentesis: claro
    if ((a + b) > c && (b - c) < a) {
        std::cout << "Expresion clara" << std::endl;
    }

    // Ejemplo 2: mezcla de operadores
    int resultado = a + b * c;     // b * c primero
    int esperado = a + (b * c);    // explicito
    std::cout << "resultado: " << resultado << std::endl;

    // Ejemplo 3: shift antes que comparacion
    int mascara = 1 << 3;
    if (mascara > 0) {
        std::cout << "mascara: " << mascara << std::endl;
    }

    return 0;
}
```

12.3. REGLA DE ORO

Cuando tengas dudas sobre la precedencia, USA PARENTESIS.
El código es para humanos primero, para el compilador después.

```cpp
// Confuso
int x = a + b * c << 2 & 0xFF;

// Claro
int x = ((a + (b * c)) << 2) & 0xFF;
```

13. OPERADOR SPACESHIP (<=>) (C++20)
===============================================================================

El operador de tres vías (spaceship) simplifica las
comparaciones. Devuelve un tipo de ordenación.

```cpp
#include <iostream>
#include <compare>

struct Punto {
    int x, y;
    auto operator<=>(const Punto&) const = default;
};

int main() {
    Punto a{1, 2}, b{1, 3};

    if (a < b) {
        std::cout << "a es menor que b" << std::endl;
    }
    if (a != b) {
        std::cout << "a es diferente de b" << std::endl;
    }

    // Comparacion manual con spaceship
    std::strong_ordering orden = a.x <=> b.x;
    if (orden == std::strong_ordering::less) {
        std::cout << "a.x < b.x" << std::endl;
    }

    return 0;
}
```

===============================================================================
14. SOBRECARGA DE OPERADORES (INTRODUCCIÓN)
===============================================================================

C++ permite dar significado personalizado a los operadores
para tipos definidos por el usuario.

```cpp
#include <iostream>

struct Vector2 {
    double x, y;

    // Sobrecarga del operador +
    Vector2 operator+(const Vector2& otro) const {
        return {x + otro.x, y + otro.y};
    }

    // Sobrecarga del operador *
    Vector2 operator*(double escalar) const {
        return {x * escalar, y * escalar};
    }
};

int main() {
    Vector2 v1{1.0, 2.0};
    Vector2 v2{3.0, 4.0};

    Vector2 suma = v1 + v2;
    Vector2 escalado = v1 * 2.5;

    std::cout << "Suma: (" << suma.x << ", " << suma.y << ")" << std::endl;
    std::cout << "Escalado: (" << escalado.x << ", " << escalado.y << ")" << std::endl;

    return 0;
}
```

15. ERRORES COMUNES CON OPERADORES
===============================================================================

```cpp
#include <iostream>

int main() {
    int a = 5, b = 10;

    // Error 1: = en lugar de ==
    if (a = b) {  // Asignacion! No comparacion!
        std::cout << "Siempre se ejecuta (a = 10)" << std::endl;
    }
    std::cout << "a ahora es: " << a << std::endl;  // 10

    // Error 2: Punto y coma extra
    for (int i = 0; i < 5; i++);  // Bucle vacio!
    {
        std::cout << "Esto solo se ejecuta una vez" << std::endl;
    }

    // Error 3: Division entera inesperada
    double resultado = 1 / 2;  // 0.0, no 0.5
    std::cout << "1/2 = " << resultado << std::endl;

    // Correcto:
    double correcto = 1.0 / 2.0;
    std::cout << "1.0/2.0 = " << correcto << std::endl;

    // Error 4: Precedencia con operadores logicos
    bool cond1 = true, cond2 = true, cond3 = false;
    // if (cond1 && cond2 || cond3)  // Confuso
    if (cond1 && (cond2 || cond3)) {  // Claro
        std::cout << "Expresion con parentesis" << std::endl;
    }

    // Error 5: Modificar la misma variable dos veces en una expresion
    int x = 5;
    // int y = x++ + ++x;  // Comportamiento indefinido!

    return 0;
}
```

RESUMEN DEL CAPITULO
===============================================================================

- Operadores aritmeticos: + - * / % (modulo solo enteros).
- Asignacion compuesta: += -= *= /= etc.
- Incremento/decremento: ++x (pre) y x++ (post) son diferentes.
- Relacionales: == != < > <= >= devuelven bool.
- Logicos: && || ! con cortocircuito.
- Bitwise: & | ^ ~ << >> para manipulacion de bits.
- Ternario: condicion ? expr_true : expr_false.
- Precedencia: usa parentesis cuando tengas dudas.
- Spaceship (C++20): <=> simplifica comparaciones.
- Sobrecarga: los operadores pueden personalizarse.
- Errores comunes: = vs ==, division entera, UB por doble modificacion.

===============================================================================
PREGUNTAS FRECUENTES
===============================================================================

P: Cual es la diferencia entre ++x y x++?
R: ++x incrementa y luego devuelve el valor (pre-incremento).
x++ devuelve el valor original y luego incrementa
(post-incremento). En bucles for, ambos son equivalentes.

P: Por que 1/2 da 0 y no 0.5?
R: Porque es division entera. Ambos operandos son int, asi
que el resultado se trunca a int. Para obtener 0.5, al menos
un operando debe ser double (1.0/2 o 1/2.0).

P: Que es el cortocircuito?
R: En &&, si la primera expresion es false, la segunda no se
evalua. En ||, si la primera es true, la segunda no se evalua.
Esto es util para evitar errores (ej: verificar puntero nulo
antes de acceder).

P: Que operadores no se pueden sobrecargar?
R: :: (resolucion ambito), . (acceso miembro), .* (puntero a
miembro), ?: (ternario), sizeof, typeid. Los demas se pueden
sobrecargar.

P: Por que se llama "spaceship" al operador <=>?
R: Porque su forma <=> recuerda a una nave espacial (spaceship)
de los videojuegos clasicos. Se introdujo en C++20.
