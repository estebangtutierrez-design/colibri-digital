--- 1. SINTAXIS BÁSICA DE VARIADIC TEMPLATES ---

Un template variádico acepta cero o más argumentos de tipo.

```cpp
#include <iostream>
#include <string>

// Función variádica básica
template<typename... Args>
void imprimir(Args... args) {
    std::cout << "Numero de argumentos: " << sizeof...(Args) << "\n";
    std::cout << "Numero de argumentos: " << sizeof...(args) << "\n";
}

// El pack Args... es un template parameter pack
// args... es un function parameter pack

int main() {
    std::cout << "=== VARIADIC TEMPLATES BASICO ===\n";

    imprimir();                    // 0 argumentos
    imprimir(1);                   // 1 argumento
    imprimir(1, 2.5, "hola");     // 3 argumentos
    imprimir(1, 2, 3, 4, 5);      // 5 argumentos

    // Los tipos se deducen automáticamente
    // Args = {} (vacio)
    // Args = {int}
    // Args = {int, double, const char*}
    // Args = {int, int, int, int, int}

    return 0;
}
```

--- 2. RECURSIÓN EN VARIADIC TEMPLATES ---

Para procesar los argumentos, tradicionalmente se usa recursión.

```cpp
#include <iostream>
#include <string>

// Caso base: sin argumentos
void printAll() {
    std::cout << "\n";
}

// Caso recursivo: procesar primer argumento y delegar el resto
template<typename T, typename... Rest>
void printAll(T first, Rest... rest) {
    std::cout << first << " ";
    printAll(rest...);  // Llamada recursiva con el resto
}

// Versión con separador
template<typename Separator>
void printWithSep(Separator) {
    std::cout << "\n";
}

template<typename Separator, typename T, typename... Rest>
void printWithSep(Separator sep, T first, Rest... rest) {
    std::cout << first;
    if constexpr (sizeof...(Rest) > 0) {
        std::cout << sep;
    }
    printWithSep(sep, rest...);
}

int main() {
    std::cout << "=== RECURSION VARIADICA ===\n";

    printAll(1, 2, 3, 4, 5);
    printAll("Hola", 3.14, 42, 'A');
    printAll();  // Vacío

    printWithSep(", ", 1, 2, 3, 4, 5);
    printWithSep(" | ", "rojo", "verde", "azul");

    // printWithSep(", ");  // Error si no hay argumentos (caso base mal?)

    return 0;
}
```

--- 3. FOLD EXPRESSIONS (C++17) ---

C++17 introdujo fold expressions para operar sobre packs sin recursión.

```cpp
#include <iostream>
#include <string>

// Fold expression: unario derecho
template<typename... Args>
auto sumaUnarioDerecho(Args... args) {
    return (args + ...);  // (arg1 + (arg2 + (arg3 + ...)))
}

// Fold expression: unario izquierdo
template<typename... Args>
auto sumaUnarioIzquierdo(Args... args) {
    return (... + args);  // (((arg1 + arg2) + arg3) + ...)
}

// Fold expression: binario derecho
template<typename... Args>
auto sumaBinarioDerecho(Args... args) {
    return (args + ... + 0);  // (arg1 + (arg2 + (... + 0)))
}

// Fold expression: binario izquierdo
template<typename... Args>
auto sumaBinarioIzquierdo(Args... args) {
    return (0 + ... + args);  // (((0 + arg1) + arg2) + ...)
}

// Operadores comunes en fold expressions
template<typename... Args>
bool todosVerdaderos(Args... args) {
    return (args && ...);  // AND lógico de todos
}

template<typename... Args>
bool algunVerdadero(Args... args) {
    return (args || ...);  // OR lógico de todos
}

template<typename... Args>
void printFold(Args... args) {
    (std::cout << ... << args) << "\n";  // Fold con <<
}

int main() {
    std::cout << "=== FOLD EXPRESSIONS ===\n";

    std::cout << "Suma unario derecho: " << sumaUnarioDerecho(1, 2, 3, 4, 5) << "\n";
    std::cout << "Suma unario izquierdo: " << sumaUnarioIzquierdo(1, 2, 3, 4, 5) << "\n";
    std::cout << "Suma binario derecho: " << sumaBinarioDerecho(1, 2, 3, 4, 5) << "\n";
    std::cout << "Suma binario izquierdo: " << sumaBinarioIzquierdo(1, 2, 3, 4, 5) << "\n";

    // Fold vacío: necesita valor inicial
    std::cout << "Suma vacia: " << sumaBinarioDerecho() << "\n";

    // Operadores lógicos
    std::cout << "todosVerdaderos: " << todosVerdaderos(true, true, true) << "\n";
    std::cout << "todosVerdaderos: " << todosVerdaderos(true, false, true) << "\n";
    std::cout << "algunVerdadero: " << algunVerdadero(false, false, true) << "\n";
    std::cout << "algunVerdadero: " << algunVerdadero(false, false, false) << "\n";

    // Fold con stream
    printFold("Hola ", "mundo ", "C++17 ", "fold ", "expressions");

    return 0;
}
```

--- 4. PERFECT FORWARDING CON PACKS VARIÁDICOS ---

La combinación de variadic templates y forwarding es extremadamente
poderosa.

```cpp
#include <iostream>
#include <string>
#include <memory>
#include <vector>

// Función que forwardea todos sus argumentos
template<typename... Args>
auto crear(Args&&... args) {
    // Expansión del pack con forwarding
    return std::make_unique<std::string>(std::forward<Args>(args)...);
}

// Emulación de emplace_back
template<typename T, typename... Args>
void emplace_back(std::vector<T>& vec, Args&&... args) {
    vec.emplace_back(std::forward<Args>(args)...);
}

// Forwarding variádico con múltiples usos
template<typename... Args>
void logYProcesar(Args&&... args) {
    std::cout << "Procesando " << sizeof...(Args) << " argumentos\n";
    // Expandir con forwarding para múltiples destinos
    // (std::cout << ... << std::forward<Args>(args)) << "\n";
}

// Fábrica genérica con forwarding
template<typename T, typename... Args>
auto fabricar(Args&&... args) -> decltype(std::make_unique<T>(
    std::forward<Args>(args)...)) {
    return std::make_unique<T>(std::forward<Args>(args)...);
}

struct Punto {
    double x, y;
    Punto(double x, double y) : x(x), y(y) {
        std::cout << "Punto(" << x << ", " << y << ")\n";
    }
};

int main() {
    std::cout << "=== FORWARDING VARIADICO ===\n";

    // Crear objeto con forwarding
    auto str = crear("Hola Mundo");
    std::cout << "Creado: " << *str << "\n";

    // Crear con múltiples argumentos
    auto str2 = crear(10, 'A');
    std::cout << "Creado: " << *str2 << "\n";

    // Emplace_back variádico
    std::vector<std::string> vec;
    emplace_back<std::string>(vec, 5, 'X');
    emplace_back<std::string>(vec, "Hola");
    std::cout << "Vector: ";
    for (const auto& s : vec) std::cout << s << " ";
    std::cout << "\n";

    // Fábrica genérica
    auto p = fabricar<Punto>(3.0, 4.0);
    std::cout << "Punto fabricado: (" << p->x << ", " << p->y << ")\n";

    // Forwarding con lambdas (C++20)
    auto lambda = []<typename... Args>(Args&&... args) {
        return std::make_tuple(std::forward<Args>(args)...);
    };
    auto t = lambda(1, "hola", 3.14);
    std::cout << "Tuple: " << std::get<0>(t) << "\n";

    return 0;
}
```

--- 5. CLASES VARIÁDICAS: TUPLE SIMPLE ---

Implementación simplificada de std::tuple usando variadic templates.

```cpp
#include <iostream>
#include <string>
#include <utility>

// Caso base: tuple vacío
template<typename...>
struct Tuple {};

// Caso recursivo: head + tail
template<typename T, typename... Rest>
struct Tuple<T, Rest...> : Tuple<Rest...> {
    T value;

    Tuple() = default;

    template<typename U, typename... Args>
    Tuple(U&& v, Args&&... args)
        : Tuple<Rest...>(std::forward<Args>(args)...)
        , value(std::forward<U>(v)) {}

    Tuple(const Tuple&) = default;
    Tuple(Tuple&&) = default;
};

// Obtener el tipo en posición I
template<size_t I, typename T>
struct TupleElementType;

template<size_t I, typename T, typename... Rest>
struct TupleElementType<I, Tuple<T, Rest...>>
    : TupleElementType<I - 1, Tuple<Rest...>> {};

template<typename T, typename... Rest>
struct TupleElementType<0, Tuple<T, Rest...>> {
    using type = T;
};

// Obtener valor en posición I
template<size_t I>
struct GetHelper {
    template<typename T, typename... Rest>
    static auto& get(Tuple<T, Rest...>& t) {
        return GetHelper<I - 1>::get(
            static_cast<Tuple<Rest...>&>(t));
    }
};

template<>
struct GetHelper<0> {
    template<typename T, typename... Rest>
    static auto& get(Tuple<T, Rest...>& t) {
        return t.value;
    }
};

template<size_t I, typename... Args>
auto& get(Tuple<Args...>& t) {
    return GetHelper<I>::get(t);
}

// Sobrecarga de <<
template<typename T, typename... Rest>
std::ostream& operator<<(std::ostream& os, const Tuple<T, Rest...>& t) {
    os << t.value << " ";
    operator<<(os, static_cast<const Tuple<Rest...>&>(t));
    return os;
}

std::ostream& operator<<(std::ostream& os, const Tuple<>&) {
    return os;
}

int main() {
    std::cout << "=== TUPLE VARIADICA ===\n";

    Tuple<int, double, std::string> t(42, 3.14, "Hola");
    std::cout << "Tuple: " << t << "\n";

    std::cout << "get<0>: " << get<0>(t) << "\n";
    std::cout << "get<1>: " << get<1>(t) << "\n";
    std::cout << "get<2>: " << get<2>(t) << "\n";

    get<0>(t) = 100;
    std::cout << "Modificado: " << t << "\n";

    return 0;
}
```

--- 6. EXPANSIÓN DE PACKS EN DIFERENTES CONTEXTOS ---

Los packs pueden expandirse en varios contextos.

```cpp
#include <iostream>
#include <vector>
#include <string>
#include <tuple>
#include <memory>
#include <functional>

// 1. Expansión en lista de inicialización
template<typename... Args>
std::vector<int> crearVector(Args... args) {
    return {static_cast<int>(args)...};
}

// 2. Expansión en argumentos de función
template<typename... Args>
auto crearTupla(Args... args) {
    return std::make_tuple(args...);
}

// 3. Expansión en inicialización de array
template<typename... Args>
void procesarArray(Args... args) {
    int arr[] = {args...};
    for (int x : arr) std::cout << x << " ";
    std::cout << "\n";
}

// 4. Expansión en template argumentos
template<typename... Bases>
class MultiDerivada : public Bases... {
public:
    MultiDerivada(const Bases&... bases) : Bases(bases)... {}
    
    void mostrar() const {
        ((std::cout << typeid(Bases).name() << " "), ...);
    }
};

struct A { int a = 1; };
struct B { int b = 2; };
struct C { int c = 3; };

// 5. Expansión en lambda captures (C++14)
template<typename... Args>
auto crearLambda(Args... args) {
    return [args...]() {
        ((std::cout << args << " "), ...);
        std::cout << "\n";
    };
}

// 6. Expansión en packs de base classes
template<typename... Ts>
struct Overloader : Ts... {
    using Ts::operator()...;
};

template<typename... Ts>
Overloader(Ts...) -> Overloader<Ts...>;

int main() {
    std::cout << "=== EXPANSION EN CONTEXTOS ===\n";

    // Expansión en lista de inicialización
    auto vec = crearVector(1, 2, 3, 4, 5);
    for (int x : vec) std::cout << x << " ";
    std::cout << "\n";

    // Expansión en array
    procesarArray(10, 20, 30, 40, 50);

    // Expansión en herencia múltiple
    A a{10}; B b{20}; C c{30};
    MultiDerivada<A, B, C> md(a, b, c);
    std::cout << "MultiDerivada: " << md.a << ", " << md.b << ", " << md.c << "\n";

    // Expansión en lambda
    auto lam = crearLambda(1, 2, 3, "hola");
    lam();

    // Expansión con Overloader pattern (C++17)
    auto visitor = Overloader{
        [](int i) { std::cout << "int: " << i << "\n"; },
        [](double d) { std::cout << "double: " << d << "\n"; },
        [](const std::string& s) { std::cout << "string: " << s << "\n"; }
    };

    visitor(42);
    visitor(3.14);
    visitor(std::string("test"));

    return 0;
}
```

--- 7. PATRONES AVANZADOS CON VARIADIC TEMPLATES ---

Técnicas más avanzadas de metaprogramación variádica.

```cpp
#include <iostream>
#include <string>
#include <tuple>
#include <type_traits>
#include <vector>

// 1. Aplicar función a cada elemento del pack
template<typename Func, typename... Args>
void forEach(Func f, Args... args) {
    (f(args), ...);  // Fold sobre operador coma
}

// 2. Aplicar función y recolectar resultados
template<typename Func, typename... Args>
auto map(Func f, Args... args) {
    return std::make_tuple(f(args)...);
}

// 3. Filtrar tipos en tiempo de compilación
template<typename T>
struct TypeWrapper {
    using type = T;
};

template<typename... Ts>
struct TypeList {
    static constexpr size_t size = sizeof...(Ts);

    template<typename T>
    static constexpr bool contains = (std::is_same_v<T, Ts> || ...);

    template<template<typename> typename Pred>
    using filter = ...;  // Requiere recursión
};

// 4. Zip de packs
template<typename... Args>
void zipPrint(Args... args) {
    // Simplificación: imprimir pares
    ((std::cout << args << " "), ...);
    std::cout << "\n";
}

// 5. Index sequence
template<size_t... Is>
void printIndices(std::index_sequence<Is...>) {
    ((std::cout << Is << " "), ...);
    std::cout << "\n";
}

// 6. Apply: desempaquetar tuple en argumentos de función
template<typename Func, typename Tuple, size_t... Is>
auto applyImpl(Func f, Tuple&& t, std::index_sequence<Is...>) {
    return f(std::get<Is>(std::forward<Tuple>(t))...);
}

template<typename Func, typename... Args>
auto apply(Func f, std::tuple<Args...>& t) {
    return applyImpl(f, t, std::index_sequence_for<Args...>{});
}

// 7. Factorial de tipos (seleccionar Nth type)
template<size_t I, typename T, typename... Rest>
struct NthType : NthType<I - 1, Rest...> {};

template<typename T, typename... Rest>
struct NthType<0, T, Rest...> {
    using type = T;
};

template<size_t I, typename... Args>
using NthType_t = typename NthType<I, Args...>::type;

int main() {
    std::cout << "=== PATRONES AVANZADOS ===\n";

    // forEach
    forEach([](auto x) { std::cout << x << " "; }, 1, 2.5, "hola");
    std::cout << "\n";

    // map
    auto t = map([](auto x) { return x * x; }, 1, 2, 3, 4, 5);
    std::cout << "Mapped: ";
    std::apply([](auto... args) { ((std::cout << args << " "), ...); }, t);
    std::cout << "\n";

    // Index sequence
    printIndices(std::index_sequence<0, 1, 2, 3, 4>{});

    // Apply
    auto sum = [](int a, int b, int c) { return a + b + c; };
    auto tuple = std::make_tuple(10, 20, 30);
    std::cout << "Apply result: " << apply(sum, tuple) << "\n";

    // NthType
    using MyList = std::tuple<int, double, std::string>;
    NthType_t<0, int, double, std::string> x = 42;
    NthType_t<1, int, double, std::string> y = 3.14;
    NthType_t<2, int, double, std::string> z = "Hola";
    std::cout << "NthType: " << x << ", " << y << ", " << z << "\n";

    return 0;
}
```

--- 8. VARIADIC TEMPLATES CON CRTP ---

Combinando CRTP con variadic templates.

```cpp
#include <iostream>
#include <string>
#include <utility>

// Mixin con variadic templates
template<template<typename> typename... Mixins>
class Composed : public Mixins<Composed<Mixins...>>... {
public:
    Composed() = default;

    template<typename... Args>
    explicit Composed(Args&&... args) {
        // Inicialización de mixins con argumentos variádicos
    }

    void show() const {
        std::cout << "Composed<" << sizeof...(Mixins) << " mixins>\n";
    }
};

// Mixins
template<typename Derived>
struct Named {
    std::string name = "default";
    void setName(const std::string& n) { name = n; }
    const std::string& getName() const { return name; }
};

template<typename Derived>
struct Counted {
    static int count;
    Counted() { ++count; }
    ~Counted() { --count; }
    static int getCount() { return count; }
};

template<typename Derived>
struct Printable {
    void print() const {
        std::cout << "Printable mixin\n";
    }
};

template<typename Derived>
int Counted<Derived>::count = 0;

using MyComposed = Composed<Named, Counted, Printable>;

// Serializable con variadic
template<typename... Fields>
class Serializable {
    std::tuple<Fields...> data;

public:
    template<typename... Args>
    Serializable(Args&&... args)
        : data(std::forward<Args>(args)...) {}

    void serialize(std::ostream& os) const {
        std::apply([&os](const auto&... fields) {
            ((os << fields << " "), ...);
        }, data);
    }
};

int main() {
    std::cout << "=== CRTP + VARIADIC ===\n";

    MyComposed obj;
    obj.setName("ObjetoCompuesto");
    std::cout << "Name: " << obj.getName() << "\n";
    obj.print();
    std::cout << "Count: " << obj.getCount() << "\n";

    MyComposed obj2;
    std::cout << "Count (2 objetos): " << obj.getCount() << "\n";

    // Serializable
    Serializable<int, double, std::string> s(42, 3.14, "test");
    s.serialize(std::cout);
    std::cout << "\n";

    return 0;
}
```

--- 9. STD::VISIT Y VARIANT CON VARIADIC ---

std::variant y std::visit usan variadic templates intensivamente.

```cpp
#include <iostream>
#include <variant>
#include <string>
#include <vector>
#include <type_traits>

// Implementación simplificada de visit
template<typename Visitor, typename... Variants>
decltype(auto) visit(Visitor&& vis, Variants&&... vars) {
    // En la implementación real, se genera una tabla de dispatch
    // con expansión de packs variádicos
    return std::visit(std::forward<Visitor>(vis),
                      std::forward<Variants>(vars)...);
}

// Type list utilities
template<typename... Ts>
struct TypeList {
    static constexpr size_t size = sizeof...(Ts);

    template<template<typename> typename Pred>
    struct filter {
        // Implementación con recursión de plantillas
    };
};

// Overload pattern (C++17)
template<typename... Ts>
struct Overload : Ts... {
    using Ts::operator()...;
};

template<typename... Ts>
Overload(Ts...) -> Overload<Ts...>;

int main() {
    std::cout << "=== VARIANT + VISIT ===\n";

    // variant es un tipo variádico
    std::variant<int, double, std::string> v;

    v = 42;
    std::visit([](auto&& arg) {
        std::cout << "Valor: " << arg << "\n";
    }, v);

    v = 3.14;
    std::visit([](auto&& arg) {
        std::cout << "Valor: " << arg << "\n";
    }, v);

    v = "Hola variadico";
    std::visit([](auto&& arg) {
        std::cout << "Valor: " << arg << "\n";
    }, v);

    // Overload con variadic
    auto visitor = Overload{
        [](int i) { std::cout << "int " << i << "\n"; },
        [](double d) { std::cout << "double " << d << "\n"; },
        [](const std::string& s) { std::cout << "string " << s << "\n"; }
    };

    v = 100;
    std::visit(visitor, v);

    // Múltiples variantes
    using VarA = std::variant<int, double>;
    using VarB = std::variant<std::string, bool>;

    VarA a = 42;
    VarB b = "test";

    // visit con dos variantes
    auto visitor2 = Overload{
        [](int i, const std::string& s) {
            std::cout << "int(" << i << ") + string(" << s << ")\n";
        },
        [](int i, bool b) {
            std::cout << "int(" << i << ") + bool(" << b << ")\n";
        },
        [](double d, const std::string& s) {
            std::cout << "double(" << d << ") + string(" << s << ")\n";
        },
        [](double d, bool b) {
            std::cout << "double(" << d << ") + bool(" << b << ")\n";
        }
    };

    std::visit(visitor2, a, b);

    return 0;
}
```

--- 10. PARÁMETROS DE PLANTILLA NO-TIPO VARIÁDICOS ---

Los variadic templates también funcionan con parámetros no-tipo.

```cpp
#include <iostream>
#include <array>
#include <utility>

// Parámetros no-tipo variádicos
template<size_t... Dims>
class NDimensional {
    std::array<int, sizeof...(Dims)> dimensions;

public:
    NDimensional() : dimensions{Dims...} {
        std::cout << "NDimensional<";
        ((std::cout << Dims << (sizeof...(Dims) > 0 ? "x" : "")), ...);
        std::cout << "\b>\n";
    }

    size_t totalSize() const {
        size_t total = 1;
        for (auto d : dimensions) total *= d;
        return total;
    }

    size_t rank() const { return sizeof...(Dims); }
};

// Algo más complejo: valores como parámetros
template<int... Factors>
struct Multiplicador {
    static constexpr int factores[] = {Factors...};
    static constexpr int producto = (... * Factors);
};

// Secuencias de índices
template<size_t... Is>
void mostrarIndexSequence(std::index_sequence<Is...>) {
    ((std::cout << Is << " "), ...);
    std::cout << "\n";
}

int main() {
    std::cout << "=== PARAMETROS NO-TIPO VARIADICOS ===\n";

    NDimensional<3, 4, 5> nd;
    std::cout << "Rank: " << nd.rank() << "\n";
    std::cout << "Total size: " << nd.totalSize() << "\n";

    // Array 2D
    NDimensional<10, 20> nd2;
    std::cout << "Rank: " << nd2.rank() << ", Total: " << nd2.totalSize() << "\n";

    // Multiplicador
    std::cout << "Multiplicador<2,3,4>::producto = "
              << Multiplicador<2, 3, 4>::producto << "\n";

    // Index sequence
    mostrarIndexSequence(std::index_sequence<0, 1, 2, 3, 4>{});

    // std::make_index_sequence
    mostrarIndexSequence(std::make_index_sequence<10>{});

    // Aplicación: generar array de índices
    constexpr auto genArray = []<size_t... Is>(std::index_sequence<Is...>) {
        return std::array<size_t, sizeof...(Is)>{Is...};
    };

    auto arr = genArray(std::make_index_sequence<10>{});
    for (auto x : arr) std::cout << x << " ";
    std::cout << "\n";

    return 0;
}
```

```cpp
#include <iostream>
#include <string>
#include <vector>
#include <memory>
#include <tuple>
#include <functional>
#include <sstream>

// Sistema de eventos variádico
class EventBus {
    std::vector<std::function<void()>> handlers;

public:
    template<typename... Args>
    void publish(Args... args) {
        // Almacenar en una cola los argumentos...
        for (auto& handler : handlers) {
            handler();
        }
    }

    template<typename Func>
    void subscribe(Func&& f) {
        handlers.push_back(std::forward<Func>(f));
    }
};

// Logger variádico con forwarding
class Logger {
    std::string prefix;
public:
    explicit Logger(std::string p) : prefix(std::move(p)) {}

    template<typename... Args>
    void log(Args&&... args) const {
        std::ostringstream oss;
        oss << "[" << prefix << "] ";
        ((oss << std::forward<Args>(args) << " "), ...);
        std::cout << oss.str() << "\n";
    }
};

// Command pattern variádico
template<typename... Params>
class Command {
    std::function<void(Params...)> func;

public:
    explicit Command(std::function<void(Params...)> f) : func(std::move(f)) {}

    void execute(Params... params) {
        func(std::forward<Params>(params)...);
    }
};

// Builder variádico
class QueryBuilder {
    std::string query;
    std::vector<std::string> conditions;

public:
    QueryBuilder& select(const std::string& table) {
        query = "SELECT * FROM " + table;
        return *this;
    }

    template<typename... Args>
    QueryBuilder& where(Args... args) {
        std::ostringstream oss;
        oss << " WHERE ";
        ((oss << args << " "), ...);
        conditions.push_back(oss.str());
        return *this;
    }

    std::string build() const {
        std::string result = query;
        for (const auto& c : conditions) result += c;
        return result;
    }
};

// Variadic factory
template<typename T, typename... Args>
auto make_vector(Args&&... args) {
    return std::vector<T>{std::forward<Args>(args)...};
}

int main() {
    std::cout << "=== EJEMPLO INTEGRADOR ===\n";

    // Logger
    Logger log("APP");
    log.log("Iniciando", "sistema", "con", 3, "componentes");
    log.log("Error:", "archivo", "no", "encontrado");

    // Logger con forwarding
    std::string file = "config.ini";
    log.log("Leyendo", std::move(file));

    // QueryBuilder
    QueryBuilder qb;
    std::string sql = qb.select("usuarios")
                        .where("edad >", 18)
                        .where("activo =", true)
                        .build();
    std::cout << "SQL: " << sql << "\n";

    // Command
    Command<int, std::string> cmd(
        [](int id, const std::string& name) {
            std::cout << "Command: id=" << id << ", name=" << name << "\n";
        }
    );
    cmd.execute(42, "test");

    // make_vector variádico
    auto vec = make_vector<int>(1, 2, 3, 4, 5);
    for (int x : vec) std::cout << x << " ";
    std::cout << "\n";

    // Logger con fold
    auto logAll = []<typename... Args>(Args&&... args) {
        ((std::cout << std::forward<Args>(args) << " "), ...);
        std::cout << "\n";
    };

    logAll("Fold", "expression", "con", "variadic", 42);

    // Tuple apply con fold
    auto t = std::make_tuple(10, 20, 30);
    std::apply([](auto... args) {
        ((std::cout << args << " "), ...);
    }, t);
    std::cout << "\n";

    return 0;
}
```

--- 12. COMPILACIÓN CONDICIONAL CON IF CONSTEXPR ---

if constexpr + variadic = potente metaprogramación.

```cpp
#include <iostream>
#include <string>
#include <type_traits>

// Procesar tipos heterogéneos
template<typename T>
void procesarUno(T&& val) {
    if constexpr (std::is_integral_v<std::decay_t<T>>) {
        std::cout << "Entero: " << val << "\n";
    } else if constexpr (std::is_floating_point_v<std::decay_t<T>>) {
        std::cout << "Flotante: " << val << "\n";
    } else if constexpr (std::is_same_v<std::decay_t<T>, std::string>) {
        std::cout << "String: " << val << "\n";
    } else {
        std::cout << "Otro tipo\n";
    }
}

template<typename... Args>
void procesarTodos(Args&&... args) {
    (procesarUno(std::forward<Args>(args)), ...);
}

// Filter en tiempo de compilación
template<typename... Args>
void soloEnteros(Args... args) {
    auto printInt = [](auto x) {
        if constexpr (std::is_integral_v<decltype(x)>) {
            std::cout << x << " ";
        }
    };
    (printInt(args), ...);
    std::cout << "\n";
}

// Contar tipos específicos
template<typename Target, typename... Args>
constexpr size_t countType() {
    return (std::is_same_v<Target, Args> + ...);
}

int main() {
    std::cout << "=== IF CONSTEXPR + VARIADIC ===\n";

    procesarTodos(42, 3.14, std::string("hola"), 'A');

    soloEnteros(1, 2.5, 3, "texto", 4, 5.0, 6);

    std::cout << "Count int: " << countType<int, int, double, int, float>() << "\n";
    std::cout << "Count double: " << countType<double, int, double, int, double>() << "\n";

    // Comprobación de tipos
    static_assert(countType<int, int, int, double>() == 2);
    static_assert(countType<double, int, int, double>() == 1);

    return 0;
}
```

================================================================================
  RESUMEN DEL CAPÍTULO
================================================================================

Las plantillas variádicas permiten trabajar con un número arbitrario
de argumentos:

• template<typename... Args> declara un template parameter pack.
• args... expande el pack en diferentes contextos.
• sizeof...(Args) cuenta los argumentos.
• Recursión: caso base + caso variádico para procesar elementos.
• Fold expressions (C++17): operar sobre packs con operadores binarios.
• Perfect forwarding con packs: std::forward<Args>(args)...
• Clases variádicas: herencia múltiple, tuplas, variant.
• Parámetros no-tipo variádicos: template<size_t... Dims>.
• if constexpr + variadic: metaprogramación en tiempo de compilación.
• std::apply desempaqueta tuples en argumentos de función.

================================================================================
  PREGUNTAS FRECUENTES (FAQ)
================================================================================

P: ¿Cuántos argumentos puede tener un template variádico?
R: No hay límite teórico, pero los compiladores tienen límites
prácticos (generalmente 1024 o más).

P: ¿Cómo proceso cada argumento individualmente?
R: Con recursión (caso base + paso recursivo) o con fold expressions
en C++17.

P: ¿Qué operadores soportan fold expressions?
R: +, -, *, /, %, ^, &, |, <<, >>, &&, ||, , (coma), ->*, ==, !=,
<, >, <=, >=.

P: ¿Puedo usar parámetros no-tipo variádicos?
R: Sí: template<auto... Values> o template<size_t... Indices>.

P: ¿Cómo combino variadic templates con forwarding?
R: template<typename... Args> void f(Args&&... args) {
    g(std::forward<Args>(args)...);
}

P: ¿Qué es std::index_sequence?
R: Es una secuencia de índices en tiempo de compilación, útil para
desempaquetar tuples: std::index_sequence<0, 1, 2, ...>.
