BIENVENIDA
----------
Bienvenido al capítulo de stringstream. Esta clase hermana de fstream nos
permite tratar strings como si fueran flujos de E/S. Con istringstream y
ostringstream podemos parsear cadenas con formato, construir strings
complejos con manipulación de flujo, y convertir entre tipos de forma
eficiente y segura.

FRASE
-----
"Un string es un archivo en miniatura; un stringstream es la llave maestra
para abrirlo." — Anónimo

OBJETIVOS
---------
• Comprender la jerarquía istringstream, ostringstream, stringstream.
• Parsear strings con formato usando istringstream.
• Construir strings formateados con ostringstream.
• Convertir entre tipos (número  string, string  número).
• Combinar stringstream con archivos para procesamiento híbrido.
• Implementar serialización/deserialización en memoria.

CONTENIDO
---------

1. LAS TRES CLASES

  ┌─────────────────────────────────────────────────────┐
  │  istringstream  → lectura desde string (como ifstream)│
  │  ostringstream  → escritura hacia string (como ofstream)│
  │  stringstream   → lectura + escritura (como fstream)   │
  └─────────────────────────────────────────────────────┘

2. ISTRINGSTREAM

istringstream permite leer datos formateados desde un string.
Es ideal para:
  - Parsear líneas de archivos CSV.
  - Convertir strings a números.
  - Extraer campos de cadenas delimitadas.
  - Validar formato de entrada.

3. OSTRINGSTREAM

ostringstream permite escribir datos formateados en un string.
Es ideal para:
  - Construir mensajes complejos con formato.
  - Serializar datos a texto.
  - Generar reportes en memoria.
  - Concatenar eficientemente muchos fragmentos.

4. MÉTODO STR()

str() es el método clave: en ostringstream devuelve el string construido;
en istringstream establece el string a parsear.

5. CONVERSIÓN DE TIPOS

Antes de C++11, stringstream era la forma estándar de convertir entre
tipos (to_string y stoi llegaron con C++11). Sigue siendo útil para
conversiones con formato específico.

// EJEMPLO 1: ISTRINGSTREAM BÁSICO
//==============================================================================
void ejemplo_istringstream_basico()
{
    cout << "\n--- EJEMPLO 1: istringstream básico ---\n";

    string datos = "Juan 25 1.75 Programador";
    istringstream ss(datos);

    string nombre, profesion;
    int edad;
    double altura;

    ss >> nombre >> edad >> altura >> profesion;

    cout << "  Nombre: " << nombre << "\n";
    cout << "  Edad: " << edad << "\n";
    cout << "  Altura: " << altura << "\n";
    cout << "  Profesión: " << profesion << "\n";
}

// EJEMPLO 2: OSTRINGSTREAM BÁSICO
//==============================================================================
void ejemplo_ostringstream_basico()
{
    cout << "\n--- EJEMPLO 2: ostringstream básico ---\n";

    ostringstream ss;

    ss << "Producto: " << "Laptop Gamer" << "\n";
    ss << "Precio: $" << 1200.50 << "\n";
    ss << "Stock: " << 15 << " unidades\n";
    ss << "Disponible: " << boolalpha << true << "\n";

    string resultado = ss.str();
    cout << "  String construido:\n" << resultado;
}

// EJEMPLO 3: PARSEAR LÍNEA CSV
//==============================================================================
void ejemplo_parse_csv()
{
    cout << "\n--- EJEMPLO 3: Parsear CSV ---\n";

    string linea = "1,Ana Pérez,45000.50,Ingeniera";
    istringstream ss(linea);
    string campo;

    cout << "  Campos:\n";
    while (getline(ss, campo, ','))
        cout << "    " << campo << "\n";
}

// EJEMPLO 4: CONVERSIÓN STRING  INT
//==============================================================================
void ejemplo_string_a_int()
{
    cout << "\n--- EJEMPLO 4: String a entero ---\n";

    string numeros[] = {"42", "0xFF", "071", " 100 ", "abc", "123xyz"};

    for (const auto& s : numeros)
    {
        istringstream ss(s);
        int valor;

        if (ss >> valor)
            cout << "  '" << s << "'  " << valor << "\n";
        else
            cout << "  '" << s << "'  ERROR (no es entero válido)\n";
    }
}

        ss << v;

// EJEMPLO 6: FORMATEO AVANZADO CON OSTRINGSTREAM

void ejemplo_formateo_avanzado()
{
    cout << "\n--- EJEMPLO 6: Formateo avanzado ---\n";

    ostringstream ss;

    double pi = 3.141592653589793;
    double grande = 1234567.89;
    int numero = 42;

    ss << "Pi con 2 decimales: "
       << fixed << setprecision(2) << pi << "\n";
    ss << "Pi con 6 decimales: "
       << fixed << setprecision(6) << pi << "\n";
    ss << "Científico: "
       << scientific << setprecision(3) << grande << "\n";

    ss << "Ancho fijo: |"
       << setw(10) << setfill('.') << numero << "|\n";
    ss << "Izquierda: |"
       << left << setw(10) << setfill('-') << numero << "|\n";
    ss << "Derecha: |"
       << right << setw(10) << setfill('_') << numero << "|\n";

    ss << "Hexadecimal: 0x" << hex << uppercase << 255 << "\n";
    ss << "Octal: 0" << oct << 255 << dec << "\n";

    ss << "Booleano: " << boolalpha << true << " " << false << "\n";

    cout << ss.str();
}

// EJEMPLO 7: VALIDACIÓN DE ENTRADA CON ISTRINGSTREAM
//==============================================================================
bool validar_entero(const string& s, int& valor)
{
    istringstream ss(s);
    return (ss >> valor) && (ss.eof());
}

bool validar_double(const string& s, double& valor)
{
    istringstream ss(s);
    return (ss >> valor) && (ss.eof());
}

void ejemplo_validacion()
{
    cout << "\n--- EJEMPLO 7: Validación de entrada ---\n";

    vector<string> pruebas = {"42", "3.14", "abc", " 123 ", "12.5.5", "1e3"};

    for (const auto& p : pruebas)
    {
        int entero;
        double real;

        cout << "  '" << p << "'";
        if (validar_entero(p, entero))
            cout << "  entero: " << entero;
        else if (validar_double(p, real))
            cout << "  double: " << real;
        else
            cout << "  INVÁLIDO";
        cout << "\n";
    }
}

// EJEMPLO 8: STRINGSTREAM COMO BUFFER DE ARCHIVO
//==============================================================================
void ejemplo_buffer_archivo()
{
    cout << "\n--- EJEMPLO 8: Buffer en memoria para archivo ---\n";

    ostringstream buffer;

    // Escribir en buffer como si fuera archivo
    buffer << "REGISTRO_1;Ana;25;1.75\n";
    buffer << "REGISTRO_2;Luis;30;1.80\n";
    buffer << "REGISTRO_3;María;28;1.65\n";

    // Simular escritura a archivo
    ofstream archivo("buffer_ejemplo.txt");
    archivo << buffer.str();
    archivo.close();
    cout << "  Buffer volcado a archivo.\n";

    // Leer el archivo simulando con istringstream
    ifstream archivo_in("buffer_ejemplo.txt");
    ostringstream buffer_out;

    string linea;
    while (getline(archivo_in, linea))
    {
        // Procesar: reemplazar ; por |
        for (char& c : linea)
            if (c == ';') c = '|';
        buffer_out << linea << "\n";
    }
    archivo_in.close();

    cout << "  Procesado en memoria:\n" << buffer_out.str();
}

// EJEMPLO 9: SERIALIZACIÓN A STRING
//==============================================================================
struct Persona
{
    string nombre;
    int edad;
    double altura;
    vector<string> hobbies;

    string serializar() const
    {
        ostringstream ss;
        ss << nombre << "|" << edad << "|" << altura;
        for (const auto& h : hobbies)
            ss << "|" << h;
        return ss.str();
    }

    bool deserializar(const string& datos)
    {
        istringstream ss(datos);
        string campo;

        getline(ss, nombre, '|');
        if (!getline(ss, campo, '|')) return false;
        edad = stoi(campo);

        if (!getline(ss, campo, '|')) return false;
        altura = stod(campo);

        hobbies.clear();
        while (getline(ss, campo, '|'))
            hobbies.push_back(campo);

        return true;
    }
};

void ejemplo_serializacion()
{
    cout << "\n--- EJEMPLO 9: Serialización a string ---\n";

    Persona p;
    p.nombre = "Carlos Ruiz";
    p.edad = 35;
    p.altura = 1.82;
    p.hobbies = {"Leer", "Correr", "Programar", "Viajar"};

    string datos = p.serializar();
    cout << "  Serializado: " << datos << "\n";

    Persona q;
    if (q.deserializar(datos))
    {
        cout << "  Deserializado:\n";
        cout << "    Nombre: " << q.nombre << "\n";
        cout << "    Edad: " << q.edad << "\n";
        cout << "    Altura: " << q.altura << "\n";
        cout << "    Hobbies:";
        for (const auto& h : q.hobbies)
            cout << " " << h;
        cout << "\n";
    }
}

// EJEMPLO 11: DIVIDIR STRING EN TOKENS
//==============================================================================
vector<string> split(const string& s, char delim)
{
    istringstream ss(s);
    string token;
    vector<string> tokens;

    while (getline(ss, token, delim))
        tokens.push_back(token);

    return tokens;
}

void ejemplo_split()
{
    cout << "\n--- EJEMPLO 11: split() ---\n";

    string entrada = "uno,dos,tres,cuatro,cinco";
    auto tokens = split(entrada, ',');

    cout << "  Tokens de '" << entrada << "':\n";
    for (size_t i = 0; i < tokens.size(); ++i)
        cout << "    [" << i << "] " << tokens[i] << "\n";
}

// EJEMPLO 12: LEER ARCHIVO, PROCESAR CON STRINGSTREAM, ESCRIBIR
//==============================================================================
void ejemplo_flujo_completo()
{
    cout << "\n--- EJEMPLO 12: Archivo  stringstream  archivo ---\n";

    ofstream("notas.csv") << "Alumno,Mat,Fis,Qui\n"
                          << "Ana,85,90,78\n"
                          << "Luis,92,88,95\n"
                          << "Carlos,76,82,80\n";

    ifstream archivo("notas.csv");
    ostringstream salida;

    string linea;
    getline(archivo, linea); // cabecera
    salida << linea << ",Promedio\n";

    while (getline(archivo, linea))
    {
        istringstream ss(linea);
        string nombre;
        string campo;

        getline(ss, nombre, ',');
        salida << nombre;

        double suma = 0;
        int cont = 0;
        while (getline(ss, campo, ','))
        {
            double nota = stod(campo);
            suma += nota;
            cont++;
            salida << "," << nota;
        }
        salida << "," << fixed << setprecision(2) << (suma / cont) << "\n";
    }
    archivo.close();

    ofstream("notas_con_promedio.csv") << salida.str();
    cout << "  Archivo con promedios generado.\n";
}

// EJEMPLO 13: MEASURE TIEMPO DE CONCATENACIÓN
//==============================================================================
void ejemplo_eficiencia()
{
    cout << "\n--- EJEMPLO 13: Eficiencia ostringstream vs +=\n";

    const int N = 10000;

    // Método lento: +=
    string s1 = "";
    for (int i = 0; i < N; ++i)
        s1 += to_string(i) + " ";

    // Método rápido: ostringstream
    ostringstream ss;
    for (int i = 0; i < N; ++i)
        ss << i << " ";
    string s2 = ss.str();

    cout << "  s1 size: " << s1.size() << "\n";
    cout << "  s2 size: " << s2.size() << "\n";
    cout << "  ostringstream más eficiente para múltiples concatenaciones.\n";
}

//==============================================================================
// EJEMPLO 14: LECTURA DE DATOS CON FORMATO ESTRICTO
//==============================================================================
void ejemplo_formato_estricto()
{
    cout << "\n--- EJEMPLO 14: Formato estricto ---\n";

    string fecha_str = "2026-07-02";
    istringstream ss(fecha_str);

    int anio, mes, dia;
    char sep1, sep2;

    if (ss >> anio >> sep1 >> mes >> sep2 >> dia &&
        sep1 == '-' && sep2 == '-')
    {
        cout << "  Fecha válida: " << anio << "/" << mes << "/" << dia << "\n";
    }
    else
    {
        cout << "  Formato de fecha inválido.\n";
    }

    // Parsear hora
    string hora_str = "15:30:45";
    istringstream ss2(hora_str);
    int h, m, s;
    char dp1, dp2;

    if (ss2 >> h >> dp1 >> m >> dp2 >> s &&
        dp1 == ':' && dp2 == ':')
    {
        cout << "  Hora válida: " << h << ":" << m << ":" << s << "\n";
    }
}

// EJEMPLO 15: CONSTRUCTOR DE CONSULTAS SQL
//==============================================================================
void ejemplo_sql()
{
    cout << "\n--- EJEMPLO 15: Generar SQL ---\n";

    string tabla = "usuarios";
    vector<string> columnas = {"id", "nombre", "email", "edad"};
    string condicion = "edad > 18";

    ostringstream sql;

    sql << "SELECT ";
    for (size_t i = 0; i < columnas.size(); ++i)
    {
        if (i > 0) sql << ", ";
        sql << columnas[i];
    }
    sql << " FROM " << tabla;
    sql << " WHERE " << condicion;
    sql << " ORDER BY nombre ASC;";

    cout << "  SQL generado:\n  " << sql.str() << "\n";
}

RESUMEN
-------
Stringstream es una herramienta versátil que une el mundo de los flujos con
el de los strings. Con istringstream aprendimos a parsear cadenas con formato,
validar entradas y extraer campos. Con ostringstream aprendimos a construir
strings complejos con formato, serializar datos y generar contenido (HTML, SQL,
CSV) de forma eficiente. La combinación de stringstream con archivos permite
procesar datos en memoria antes de escribirlos a disco, mejorando el rendimiento
y la claridad del código.

PREGUNTAS FRECUENTES (FAQ)
--------------------------
P: Cuál es la diferencia entre stringstream y sprintf?
R: stringstream es type-safe, no requiere especificar formatos tipo %d,
   y funciona con tipos definidos por el usuario. sprintf es más rápido
   pero propenso a errores de búfer y tipo.

P: Cómo convierto un número a string con formato específico?
R: Usa ostringstream con los manipuladores (fixed, setprecision, setw)
   y luego llama a str().

P: Por qué mi istringstream no parsea correctamente?
R: Verifica que no haya espacios extra, que el delimitador sea el correcto,
   y que el estado del flujo se verifique después de cada extracción.

P: Es eficiente ostringstream para muchas concatenaciones?
R: Sí. ostringstream es mucho más eficiente que operator+= en strings
   porque gestiona el búfer internamente sin realocar en cada operación.

P: Puedo usar stringstream para lectura/escritura simultánea?
R: Sí, con stringstream (no istringstream/ostringstream). Es útil para
   modificar partes de un string.

P: Cómo saber si la conversión de string a número fue exitosa?
R: Verifica el estado del flujo después de la extracción y que se haya
   consumido todo el input (ss.eof()).
