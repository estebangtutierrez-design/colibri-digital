BIENVENIDA
----------

Bienvenido al capítulo de clases template, el corazón de la biblioteca
estándar de C++. Las clases como std::vector<T>, std::array<T,N>,
std::map<K,V>, y std::shared_ptr<T> son todas clases template que han
revolucionado la forma en que escribimos C++.

Mientras que las funciones template son ideales para algoritmos genéricos,
las clases template nos permiten crear tipos genéricos: contenedores que
almacenan cualquier tipo, punteros inteligentes que gestionan cualquier
recurso, y utilidades que funcionan con cualquier interfaz.

En este capítulo aprenderemos a definir clases template, a implementar
sus miembros dentro y fuera de la clase, a usar parámetros por defecto,
y a entender cómo std::vector<T> se construye paso a paso.

FRASE DEL CAPÍTULO
------------------

  "std::vector<T> no es solo un contenedor; es la demostración de que
   las clases template pueden ser eficientes, seguras, y elegantes."

1. SINTAXIS DE CLASES TEMPLATE
-------------------------------

La sintaxis general es:

  template <typename T>
  class MiClase {
      T miembro;
  public:
      MiClase(const T& valor) : miembro(valor) {}
      T getValor() const { return miembro; }
  };

Los parámetros pueden ser:
  - De tipo: typename T, class U
  - No-type: int N, size_t S
  - Template: template <typename> class Contenedor
  - Mixtos: typename T, int N

2. MIEMBROS DEFINIDOS DENTRO VS FUERA DE LA CLASE
--------------------------------------------------

  DENTRO (inline automáticamente):
      template <typename T>
      class Caja {
          T valor;
      public:
          void setValor(const T& v) { valor = v; }
          T getValor() const { return valor; }
      };

  FUERA (requiere template <typename T> en cada definición):
      template <typename T>
      class Caja {
          T valor;
      public:
          void setValor(const T& v);
          T getValor() const;
      };

      template <typename T>
      void Caja<T>::setValor(const T& v) { valor = v; }

      template <typename T>
      T Caja<T>::getValor() const { return valor; }

3. INSTANCIACIÓN DE CLASES TEMPLATE
-------------------------------------

Cuando escribes:

  Caja<int> cajaInt;
  Caja<double> cajaDouble;
  Caja<std::string> cajaString;

El compilador genera tres clases distintas:
  - Caja<int> con todos los T reemplazados por int.
  - Caja<double> con todos los T reemplazados por double.
  - Caja<std::string> con todos los T reemplazados por std::string.

A diferencia de las funciones, la deducción de tipos en clases (CTAD)
solo está disponible desde C++17.

4. CLASES TEMPLATE CON MÚLTIPLES PARÁMETROS
---------------------------------------------

  template <typename T, typename U = T>
  class Par {
      T primero;
      U segundo;
  public:
      Par() : primero(T()), segundo(U()) {}
      Par(const T& p, const U& s) : primero(p), segundo(s) {}
      // ...
  };

  Par<int, double> p1(1, 3.14);
  Par<int> p2(1, 2);  // U = int por defecto

5. CLASES TEMPLATE CON PARÁMETROS NO-TYPE
-------------------------------------------

  template <typename T, size_t N>
  class ArrayFijo {
      T datos[N];
  public:
      T& operator[](size_t i) { return datos[i]; }
      const T& operator[](size_t i) const { return datos[i]; }
      size_t size() const { return N; }
  };

  ArrayFijo<int, 10> arr;  // Array de 10 enteros
  ArrayFijo<double, 5> arr2;  // Array de 5 doubles

6. HERENCIA CON CLASES TEMPLATE
---------------------------------

  template <typename T>
  class Base {
  protected:
      T valor;
  public:
      Base(const T& v) : valor(v) {}
      virtual void imprimir() const {
          std::cout << "Base: " << valor << "\n";
      }
  };

  template <typename T>
  class Derivada : public Base<T> {
  public:
      Derivada(const T& v) : Base<T>(v) {}
      void imprimir() const override {
          std::cout << "Derivada: " << this->valor << "\n";
      }
  };

Nota: Necesitas usar this-> o Base<T>:: para acceder a miembros de la
clase base, porque son nombres dependientes.

7. FRIEND FUNCTIONS EN CLASES TEMPLATE
----------------------------------------

  template <typename T>
  class Caja {
      T valor;
  public:
      Caja(const T& v) : valor(v) {}

      // Friend function: operator<<
      friend std::ostream& operator<<(std::ostream& os, const Caja<T>& c) {
          os << "Caja(" << c.valor << ")";
          return os;
      }
  };

8. ALIAS DE TEMPLATES (C++11)
-------------------------------

  template <typename T>
  class MiVector { /* ... */ };

  template <typename T>
  using Vec = MiVector<T>;

  Vec<int> v;  // MiVector<int>

  // Alias para tipos parcialmente ligados
  template <typename T>
  using VecString = MiVector<std::pair<T, std::string>>;

9. CTAD - CLASS TEMPLATE ARGUMENT DEDUCTION (C++17)
-----------------------------------------------------

  template <typename T>
  class Caja {
      T valor;
  public:
      Caja(const T& v) : valor(v) {}
  };

  // C++17: Deducción de tipos del constructor
  Caja c1(42);         // Caja<int>
  Caja c2("hola");     // Caja<const char*>
  Caja c3(3.14);       // Caja<double>

  // También funciona con std::pair, std::tuple, etc.
  std::pair p(1, "hola");  // pair<int, const char*>

  // Puedes proporcionar deduction guides
  template <typename T>
  Caja(const T&) -> Caja<T>;

10. MIEMBROS ESTÁTICOS EN CLASES TEMPLATE
------------------------------------------

  template <typename T>
  class Contador {
  public:
      static int totalInstancias;
      Contador() { ++totalInstancias; }
      ~Contador() { --totalInstancias; }
  };

  template <typename T>
  int Contador<T>::totalInstancias = 0;

  // Cada instanciación tiene su propio static:
  // Contador<int>::totalInstancias es diferente de Contador<double>::totalInstancias

//==============================================================================
// EJEMPLO 1: Clase template básica - Caja<T>
//==============================================================================
#include <iostream>
#include <string>

template <typename T>
class Caja {
private:
    T valor;

public:
    // Constructor
    explicit Caja(const T& v) : valor(v) {}

    // Constructor por defecto
    Caja() : valor(T{}) {}

    // Setters y getters
    void setValor(const T& v) { valor = v; }
    T& getValor() { return valor; }
    const T& getValor() const { return valor; }

    // Operadores
    bool operator==(const Caja& otra) const {
        return valor == otra.valor;
    }

    bool operator<(const Caja& otra) const {
        return valor < otra.valor;
    }

    // Conversión
    operator T() const { return valor; }
};

// Friend operator<< definido fuera
template <typename T>
std::ostream& operator<<(std::ostream& os, const Caja<T>& c) {
    os << "Caja(" << c.getValor() << ")";
    return os;
}

int main() {
    Caja<int> ci(42);
    Caja<double> cd(3.14);
    Caja<std::string> cs("Hello");

    std::cout << ci << "\n";
    std::cout << cd << "\n";
    std::cout << cs << "\n";

    Caja<int> ci2(42);
    std::cout << "ci == ci2: " << (ci == ci2) << "\n";

    // Conversión implícita
    int v = ci;
    std::cout << "Valor convertido: " << v << "\n";

    return 0;
}

// EJEMPLO 2: Miembros definidos fuera de la clase
//==============================================================================
#include <iostream>
#include <cmath>

template <typename T>
class Punto2D {
private:
    T x, y;

public:
    Punto2D();
    Punto2D(T x, T y);

    T getX() const;
    T getY() const;
    void setX(T x);
    void setY(T y);

    double distancia(const Punto2D& otro) const;
    Punto2D operator+(const Punto2D& otro) const;
    Punto2D operator-(const Punto2D& otro) const;
    T productoEscalar(const Punto2D& otro) const;
};

// Definiciones fuera de la clase
template <typename T>
Punto2D<T>::Punto2D() : x(T{}), y(T{}) {}

template <typename T>
Punto2D<T>::Punto2D(T x, T y) : x(x), y(y) {}

template <typename T>
T Punto2D<T>::getX() const { return x; }

template <typename T>
T Punto2D<T>::getY() const { return y; }

template <typename T>
void Punto2D<T>::setX(T x) { this->x = x; }

template <typename T>
void Punto2D<T>::setY(T y) { this->y = y; }

template <typename T>
double Punto2D<T>::distancia(const Punto2D& otro) const {
    T dx = x - otro.x;
    T dy = y - otro.y;
    return std::sqrt(dx * dx + dy * dy);
}

template <typename T>
Punto2D<T> Punto2D<T>::operator+(const Punto2D& otro) const {
    return Punto2D(x + otro.x, y + otro.y);
}

template <typename T>
Punto2D<T> Punto2D<T>::operator-(const Punto2D& otro) const {
    return Punto2D(x - otro.x, y - otro.y);
}

template <typename T>
T Punto2D<T>::productoEscalar(const Punto2D& otro) const {
    return x * otro.x + y * otro.y;
}

template <typename T>
std::ostream& operator<<(std::ostream& os, const Punto2D<T>& p) {
    os << "(" << p.getX() << ", " << p.getY() << ")";
    return os;
}

int main() {
    Punto2D<int> p1(1, 2);
    Punto2D<int> p2(4, 6);

    Punto2D<double> p3(0.0, 0.0);
    Punto2D<double> p4(3.0, 4.0);

    std::cout << "p1: " << p1 << ", p2: " << p2 << "\n";
    std::cout << "p1 + p2: " << (p1 + p2) << "\n";
    std::cout << "Distancia p1-p2: " << p1.distancia(p2) << "\n";
    std::cout << "Producto escalar: " << p1.productoEscalar(p2) << "\n";
    std::cout << "Distancia p3-p4: " << p3.distancia(p4) << "\n";

    return 0;
}

//==============================================================================
// EJEMPLO 3: Implementación simplificada de Vector<T>
//==============================================================================
#include <iostream>
#include <stdexcept>
#include <algorithm>

template <typename T>
class Vector {
private:
    T* datos;
    size_t capacidad;
    size_t longitud;

    void reservar(size_t nuevaCapacidad) {
        T* nuevosDatos = new T[nuevaCapacidad];
        for (size_t i = 0; i < longitud; ++i) {
            nuevosDatos[i] = std::move(datos[i]);
        }
        delete[] datos;
        datos = nuevosDatos;
        capacidad = nuevaCapacidad;
    }

public:
    Vector() : datos(nullptr), capacidad(0), longitud(0) {}

    explicit Vector(size_t n, const T& valor = T{})
        : datos(new T[n]), capacidad(n), longitud(n) {
        for (size_t i = 0; i < n; ++i) {
            datos[i] = valor;
        }
    }

    Vector(const Vector& otro)
        : datos(new T[otro.capacidad]),
          capacidad(otro.capacidad),
          longitud(otro.longitud) {
        for (size_t i = 0; i < longitud; ++i) {
            datos[i] = otro.datos[i];
        }
    }

    Vector(Vector&& otro) noexcept
        : datos(otro.datos),
          capacidad(otro.capacidad),
          longitud(otro.longitud) {
        otro.datos = nullptr;
        otro.capacidad = 0;
        otro.longitud = 0;
    }

    ~Vector() { delete[] datos; }

    Vector& operator=(const Vector& otro) {
        if (this != &otro) {
            delete[] datos;
            capacidad = otro.capacidad;
            longitud = otro.longitud;
            datos = new T[capacidad];
            for (size_t i = 0; i < longitud; ++i) {
                datos[i] = otro.datos[i];
            }
        }
        return *this;
    }

    Vector& operator=(Vector&& otro) noexcept {
        if (this != &otro) {
            delete[] datos;
            datos = otro.datos;
            capacidad = otro.capacidad;
            longitud = otro.longitud;
            otro.datos = nullptr;
            otro.capacidad = 0;
            otro.longitud = 0;
        }
        return *this;
    }

    void push_back(const T& valor) {
        if (longitud >= capacidad) {
            size_t nuevaCap = (capacidad == 0) ? 1 : capacidad * 2;
            reservar(nuevaCap);
        }
        datos[longitud++] = valor;
    }

    void pop_back() {
        if (longitud > 0) {
            --longitud;
            datos[longitud].~T();
        }
    }

    T& operator[](size_t index) {
        return datos[index];
    }

    const T& operator[](size_t index) const {
        return datos[index];
    }

    T& at(size_t index) {
        if (index >= longitud) {
            throw std::out_of_range("Indice fuera de rango");
        }
        return datos[index];
    }

    size_t size() const { return longitud; }
    size_t capacity() const { return capacidad; }
    bool empty() const { return longitud == 0; }

    T* begin() { return datos; }
    T* end() { return datos + longitud; }
    const T* begin() const { return datos; }
    const T* end() const { return datos + longitud; }
};

template <typename T>
std::ostream& operator<<(std::ostream& os, const Vector<T>& v) {
    os << "[";
    for (size_t i = 0; i < v.size(); ++i) {
        if (i > 0) os << ", ";
        os << v[i];
    }
    os << "]";
    return os;
}

int main() {
    Vector<int> v;
    v.push_back(10);
    v.push_back(20);
    v.push_back(30);
    v.push_back(40);
    v.push_back(50);

    std::cout << "Vector: " << v << "\n";
    std::cout << "Size: " << v.size() << ", Capacity: " << v.capacity() << "\n";

    v.pop_back();
    std::cout << "Despues de pop_back: " << v << "\n";

    v[2] = 99;
    std::cout << "v[2] = 99: " << v << "\n";

    Vector<int> v2 = v;
    std::cout << "Copia: " << v2 << "\n";

    Vector<int> v3 = std::move(v);
    std::cout << "Movido: " << v3 << "\n";
    std::cout << "Original vacio: " << v << "\n";

    // CTAD (C++17)
    Vector v4 = {1, 2, 3, 4, 5};
    std::cout << "v4: " << v4 << "\n";

    return 0;
}

// EJEMPLO 4: Clase template con múltiples parámetros y default
//==============================================================================
#include <iostream>
#include <string>

template <typename K, typename V = std::string>
class Par {
private:
    K clave;
    V valor;

public:
    Par() : clave(K{}), valor(V{}) {}
    Par(const K& c, const V& v) : clave(c), valor(v) {}

    K& getClave() { return clave; }
    const K& getClave() const { return clave; }

    V& getValor() { return valor; }
    const V& getValor() const { return valor; }

    void setClave(const K& c) { clave = c; }
    void setValor(const V& v) { valor = v; }

    bool operator==(const Par& otro) const {
        return clave == otro.clave;
    }

    bool operator<(const Par& otro) const {
        return clave < otro.clave;
    }
};

// Especialización para cuando V es const char*
template <typename K>
class Par<K, const char*> {
private:
    K clave;
    std::string valor;

public:
    Par() : clave(K{}), valor("") {}
    Par(const K& c, const char* v) : clave(c), valor(v) {}

    K& getClave() { return clave; }
    const K& getClave() const { return clave; }

    std::string& getValor() { return valor; }
    const std::string& getValor() const { return valor; }

    void imprimir() const {
        std::cout << "Par(" << clave << ", " << valor << ")\n";
    }
};

template <typename K, typename V>
std::ostream& operator<<(std::ostream& os, const Par<K, V>& p) {
    os << "(" << p.getClave() << ": " << p.getValor() << ")";
    return os;
}

int main() {
    // Usando tipo por defecto V = std::string
    Par<int> p1(1, "uno");
    Par<int> p2(2, "dos");
    Par<int, double> p3(3, 3.14);

    std::cout << p1 << "\n";
    std::cout << p2 << "\n";
    std::cout << p3 << "\n";

    std::cout << "p1 == p2: " << (p1 == p2) << "\n";

    // Usando especialización parcial
    Par<int, const char*> p4(42, "respuesta");
    p4.imprimir();

    return 0;
}

//==============================================================================
// EJEMPLO 5: Parámetros no-type en clases template
//==============================================================================
#include <iostream>
#include <stdexcept>

template <typename T, size_t N>
class ArrayFijo {
private:
    T datos[N];

public:
    ArrayFijo() {
        for (size_t i = 0; i < N; ++i)
            datos[i] = T{};
    }

    ArrayFijo(std::initializer_list<T> lista) {
        size_t i = 0;
        for (const auto& elem : lista) {
            if (i < N) datos[i++] = elem;
        }
        while (i < N) datos[i++] = T{};
    }

    T& operator[](size_t i) {
        return datos[i];
    }

    const T& operator[](size_t i) const {
        return datos[i];
    }

    T& at(size_t i) {
        if (i >= N) throw std::out_of_range("Indice fuera de rango");
        return datos[i];
    }

    size_t size() const { return N; }

    T* begin() { return datos; }
    T* end() { return datos + N; }
    const T* begin() const { return datos; }
    const T* end() const { return datos + N; }

    void llenar(const T& valor) {
        for (size_t i = 0; i < N; ++i)
            datos[i] = valor;
    }
};

// Deduction guide para CTAD
template <typename T, typename... Args>
ArrayFijo(T, Args...) -> ArrayFijo<T, sizeof...(Args) + 1>;

template <typename T, size_t N>
std::ostream& operator<<(std::ostream& os, const ArrayFijo<T, N>& arr) {
    os << "[";
    for (size_t i = 0; i < N; ++i) {
        if (i > 0) os << ", ";
        os << arr[i];
    }
    os << "]";
    return os;
}

int main() {
    ArrayFijo<int, 5> arr1;
    arr1.llenar(42);
    std::cout << "arr1: " << arr1 << "\n";

    ArrayFijo<double, 3> arr2 = {1.1, 2.2, 3.3};
    std::cout << "arr2: " << arr2 << "\n";

    arr2[1] = 9.99;
    std::cout << "arr2 modificado: " << arr2 << "\n";

    // CTAD (C++17)
    ArrayFijo arr3 = {1, 2, 3, 4};
    std::cout << "arr3: " << arr3 << " (size=" << arr3.size() << ")\n";

    try {
        arr1.at(10);
    } catch (const std::out_of_range& e) {
        std::cout << "Excepcion: " << e.what() << "\n";
    }

    // Uso con range-based for
    std::cout << "Elementos arr2: ";
    for (const auto& v : arr2) {
        std::cout << v << " ";
    }
    std::cout << "\n";

    return 0;
}

//==============================================================================
// EJEMPLO 6: Herencia con clases template
//==============================================================================
#include <iostream>
#include <string>

// Clase base template
template <typename T>
class Animal {
protected:
    T nombre;
    int edad;

public:
    Animal(const T& n, int e) : nombre(n), edad(e) {}

    virtual void hacerSonido() const {
        std::cout << nombre << " hace un sonido generico\n";
    }

    virtual void imprimir() const {
        std::cout << "Animal(" << nombre << ", " << edad << ")\n";
    }

    virtual ~Animal() = default;
};

// Clase derivada template
template <typename T>
class Perro : public Animal<T> {
private:
    std::string raza;

public:
    Perro(const T& n, int e, const std::string& r)
        : Animal<T>(n, e), raza(r) {}

    void hacerSonido() const override {
        std::cout << this->nombre << " dice: Guau guau!\n";
    }

    void imprimir() const override {
        std::cout << "Perro(" << this->nombre << ", "
                  << this->edad << ", " << raza << ")\n";
    }
};

// Clase derivada template (segunda)
template <typename T>
class Gato : public Animal<T> {
private:
    bool esDomestico;

public:
    Gato(const T& n, int e, bool domestico = true)
        : Animal<T>(n, e), esDomestico(domestico) {}

    void hacerSonido() const override {
        std::cout << this->nombre << " dice: Miau!\n";
    }

    void imprimir() const override {
        std::cout << "Gato(" << this->nombre << ", "
                  << this->edad << ", domestico="
                  << (esDomestico ? "si" : "no") << ")\n";
    }
};

// Función que usa polimorfismo con templates
template <typename T>
void presentarAnimal(Animal<T>& animal) {
    animal.imprimir();
    animal.hacerSonido();
}

int main() {
    Perro<std::string> perro("Max", 3, "Labrador");
    Gato<std::string> gato("Luna", 2, true);

    // Polimorfismo dinámico con templates
    Animal<std::string>* animales[] = {&perro, &gato};

    for (auto* animal : animales) {
        animal->imprimir();
        animal->hacerSonido();
        std::cout << "---\n";
    }

    // Función template que acepta cualquier Animal<T>
    presentarAnimal(perro);
    presentarAnimal(gato);

    return 0;
}

// EJEMPLO 7: Miembros estáticos en clases template
//==============================================================================
#include <iostream>
#include <string>

template <typename T>
class Contador {
private:
    T dato;

public:
    static int totalCreados;
    static int totalVivos;

    Contador(const T& d = T{}) : dato(d) {
        ++totalCreados;
        ++totalVivos;
        std::cout << "Constructor. Creados: " << totalCreados
                  << ", Vivos: " << totalVivos << "\n";
    }

    Contador(const Contador& otro) : dato(otro.dato) {
        ++totalCreados;
        ++totalVivos;
        std::cout << "Copia. Creados: " << totalCreados
                  << ", Vivos: " << totalVivos << "\n";
    }

    Contador(Contador&& otro) noexcept : dato(std::move(otro.dato)) {
        ++totalCreados;
        ++totalVivos;
        std::cout << "Movimiento. Creados: " << totalCreados
                  << ", Vivos: " << totalVivos << "\n";
    }

    ~Contador() {
        --totalVivos;
        std::cout << "Destructor. Vivos: " << totalVivos << "\n";
    }

    T getDato() const { return dato; }
};

// Inicialización de miembros estáticos
template <typename T>
int Contador<T>::totalCreados = 0;

template <typename T>
int Contador<T>::totalVivos = 0;

void demoContadores() {
    Contador<int> c1(10);
    Contador<int> c2 = c1;
    Contador<int> c3 = std::move(c1);

    std::cout << "Estaticos int: creados="
              << Contador<int>::totalCreados
              << ", vivos=" << Contador<int>::totalVivos << "\n";

    Contador<double> d1(3.14);
    std::cout << "Estaticos double: creados="
              << Contador<double>::totalCreados
              << ", vivos=" << Contador<double>::totalVivos << "\n";
}

int main() {
    demoContadores();

    std::cout << "Despues de demoContadores:\n";
    std::cout << "Contador<int>::totalVivos = "
              << Contador<int>::totalVivos << "\n";

    return 0;
}

// Contenedor que usa otro contenedor como política de almacenamiento
template <typename T, template <typename, typename...> class Contenedor = std::vector>
class Bolsa {
private:
    Contenedor<T> items;

public:
    void agregar(const T& item) {
        items.push_back(item);
    }

    void agregar(T&& item) {
        items.push_back(std::move(item));
    }

    size_t size() const { return items.size(); }
    bool empty() const { return items.empty(); }

    auto begin() const { return items.begin(); }
    auto end() const { return items.end(); }
    auto begin() { return items.begin(); }
    auto end() { return items.end(); }

    T& operator[](size_t i) { return items[i]; }
    const T& operator[](size_t i) const { return items[i]; }
};

using BolsaList = Bolsa<T, std::list>;

    os << "[";
    bool primero = true;
    for (const auto& item : bolsa) {
        if (!primero) os << ", ";
        os << item;
        primero = false;
    }
    os << "]";
    return os;
}

    bolsaVec.agregar(30);

#include <iostream>
#include <string>
#include <utility>

template <typename T, typename U>
class Mapa {
private:
    std::vector<std::pair<T, U>> datos;

public:
    void insertar(const T& clave, const U& valor) {
        datos.push_back({clave, valor});
    }

    void imprimir() const {
        for (const auto& [c, v] : datos) {
            std::cout << c << " -> " << v << "\n";
        }
    }
};

// Deduction guide para Mapa desde initializer_list de pares
template <typename T, typename U>
Mapa(std::initializer_list<std::pair<T, U>>) -> Mapa<T, U>;

// Deduction guide desde pares individuales
template <typename T, typename U>
Mapa(const T&, const U&) -> Mapa<T, U>;

// Clase con deduction guide explícito
template <typename T>
class Wrapper {
    T valor;
public:
    Wrapper(const T& v) : valor(v) {}
    T get() const { return valor; }
};

// Deduction guide explícito (innecesario aquí, pero ilustrativo)
template <typename T>
Wrapper(const T&) -> Wrapper<T>;

int main() {
    // CTAD básico
    std::pair p(1, "hola");
    std::cout << "Pair: (" << p.first << ", " << p.second << ")\n";

    // CTAD con nuestro Mapa
    Mapa m = {
        std::pair{1, "uno"},
        std::pair{2, "dos"},
        std::pair{3, "tres"}
    };
    m.imprimir();

    // CTAD con Wrapper
    Wrapper w(42);
    std::cout << "Wrapper: " << w.get() << "\n";

    Wrapper ws(std::string("texto"));
    std::cout << "Wrapper string: " << ws.get() << "\n";

    return 0;
}

// EJEMPLO 11: Friend functions en clases template
//==============================================================================
#include <iostream>
#include <string>

template <typename T>
class Matriz2x2 {
private:
    T a, b, c, d;

public:
    Matriz2x2(T a, T b, T c, T d) : a(a), b(b), c(c), d(d) {}

    // Friend function como template (declaración)
    template <typename U>
    friend Matriz2x2<U> operator+(const Matriz2x2<U>& m1, const Matriz2x2<U>& m2);

    // Friend function inline (definición)
    friend Matriz2x2 operator*(const Matriz2x2& m1, const Matriz2x2& m2) {
        return Matriz2x2(
            m1.a * m2.a + m1.b * m2.c,
            m1.a * m2.b + m1.b * m2.d,
            m1.c * m2.a + m1.d * m2.c,
            m1.c * m2.b + m1.d * m2.d
        );
    }

    T determinante() const {
        return a * d - b * c;
    }

    // Friend para operator<<
    friend std::ostream& operator<<(std::ostream& os, const Matriz2x2& m) {
        os << "|" << m.a << " " << m.b << "|\n";
        os << "|" << m.c << " " << m.d << "|\n";
        return os;
    }
};

// Definición de friend function fuera de la clase
template <typename T>
Matriz2x2<T> operator+(const Matriz2x2<T>& m1, const Matriz2x2<T>& m2) {
    return Matriz2x2<T>(
        m1.a + m2.a, m1.b + m2.b,
        m1.c + m2.c, m1.d + m2.d
    );
}

int main() {
    Matriz2x2<int> m1(1, 2, 3, 4);
    Matriz2x2<int> m2(5, 6, 7, 8);

    std::cout << "Matriz 1:\n" << m1;
    std::cout << "Matriz 2:\n" << m2;

    auto suma = m1 + m2;
    std::cout << "Suma:\n" << suma;

    auto prod = m1 * m2;
    std::cout << "Producto:\n" << prod;

    std::cout << "Det(m1) = " << m1.determinante() << "\n";
    std::cout << "Det(m2) = " << m2.determinante() << "\n";

    return 0;
}

// EJEMPLO 12: Template de clase con iterador personalizado
//==============================================================================
#include <iostream>
#include <iterator>
#include <stdexcept>

template <typename T, size_t N>
class ListaCircular {
private:
    T datos[N];
    size_t head;
    size_t count;

public:
    ListaCircular() : head(0), count(0) {
        for (size_t i = 0; i < N; ++i) datos[i] = T{};
    }

    void push(const T& valor) {
        datos[head] = valor;
        head = (head + 1) % N;
        if (count < N) ++count;
    }

    T pop() {
        if (count == 0) throw std::runtime_error("Lista vacia");
        size_t index = (head + N - count) % N;
        --count;
        return datos[index];
    }

    size_t size() const { return count; }
    bool empty() const { return count == 0; }
    bool full() const { return count == N; }

    // Iterador personalizado
    class Iterador {
    private:
        T* ptr;
        size_t pos;
        size_t max;
    public:
        using iterator_category = std::forward_iterator_tag;
        using value_type = T;
        using difference_type = std::ptrdiff_t;
        using pointer = T*;
        using reference = T&;

        Iterador(T* p, size_t pos, size_t max)
            : ptr(p), pos(pos), max(max) {}

        reference operator*() { return *ptr; }
        pointer operator->() { return ptr; }

        Iterador& operator++() {
            ++pos;
            if (pos < max) ++ptr;
            return *this;
        }

        bool operator!=(const Iterador& otro) const {
            return pos != otro.pos;
        }
    };

    Iterador begin() { return Iterador(datos, 0, count); }
    Iterador end() { return Iterador(datos + count, count, count); }
};

template <typename T, size_t N>
std::ostream& operator<<(std::ostream& os, const ListaCircular<T, N>& lista) {
    // Necesitaríamos acceso a datos internos o hacer friend
    os << "ListaCircular(size=" << lista.size() << ")";
    return os;
}

int main() {
    ListaCircular<int, 5> buffer;
    buffer.push(10);
    buffer.push(20);
    buffer.push(30);

    std::cout << "Pop: " << buffer.pop() << "\n";
    std::cout << "Pop: " << buffer.pop() << "\n";
    std::cout << "Size: " << buffer.size() << "\n";

    buffer.push(40);
    buffer.push(50);
    buffer.push(60);
    buffer.push(70);

    std::cout << "Elementos: ";
    while (!buffer.empty()) {
        std::cout << buffer.pop() << " ";
    }
    std::cout << "\n";

    return 0;
}

FAQ - PREGUNTAS FRECUENTES
---------------------------

  P: ¿Puedo tener miembros virtuales en clases template?
  R: Sí, las clases template pueden tener funciones virtuales. Pero
  ten cuidado: cada instanciación genera una tabla V diferente.

  P: ¿Cómo se hereda de una clase template?
  R: Normalmente: class Derivada : public Base<T> { ... }. Recuerda
  que los nombres de la clase base son dependientes y necesitan this->.

  P: ¿Qué son los "nombres dependientes" en herencia con templates?
  R: Son nombres que dependen del parámetro T. Por ejemplo, Base<T>::x
  es dependiente. El compilador no los busca en la clase base hasta
  la instanciación. Usa this-> o Base<T>:: para acceder a ellos.

  P: ¿Cada instanciación de una clase template tiene sus propios
     miembros estáticos?
  R: Sí. Contador<int>::total es diferente de Contador<double>::total.
  Cada instanciación es una clase completamente independiente.

  P: ¿Qué es CTAD?
  R: Class Template Argument Deduction, disponible desde C++17. Permite
  que el compilador deduzca los parámetros del template de clase a
  partir de los argumentos del constructor.

  P: ¿Puedo tener destructores virtuales en clases template?
  R: Sí, y es recomendable si planeas usarlas como clases base para
  polimorfismo, igual que con clases no-template.

  P: ¿Qué es un deduction guide?
  R: Es una declaración que ayuda al compilador a deducir los parámetros
  del template cuando CTAB no puede hacerlo automáticamente.

DESAFÍO DEL ARQUITECTO
----------------------

  Implementa un "contenedor genérico expresivo" (ExpressiveContainer)
  que soporte operaciones funcionales al estilo de lenguajes como
  JavaScript o Python, pero usando templates.

  Requisitos:
    - template <typename T, typename Contenedor = std::vector<T>>
    - Método map(Funcion) que transforme cada elemento y devuelva un
      nuevo contenedor del tipo resultante.
    - Método filter(Predicado) que filtre elementos y devuelva un
      nuevo contenedor.
    - Método reduce(ValorInicial, FuncionBinaria) que reduzca a un
      solo valor.
    - Método forEach(Funcion) que aplique una función a cada elemento
      (sin devolver nada).
    - Método take(size_t n) que tome los primeros n elementos.
    - Método skip(size_t n) que salte los primeros n elementos.
    - Método unique() que elimine elementos duplicados.
    - Método sorted() que devuelva una copia ordenada.
    - Los métodos deben encadenarse (fluent interface).

  Ejemplo:
    auto resultado = contenedor
        .filter([](int x) { return x > 0; })
        .map([](int x) { return x * 2; })
        .sorted()
        .take(3);

  Pistas:
    - Usa SFINAE o if constexpr para manejar diferentes tipos de
      contenedores.
    - El tipo de retorno de map debe ser deducido de la función.
    - Usa std::back_inserter o push_back para construir resultados.
    - Para sorted, usa std::sort.
