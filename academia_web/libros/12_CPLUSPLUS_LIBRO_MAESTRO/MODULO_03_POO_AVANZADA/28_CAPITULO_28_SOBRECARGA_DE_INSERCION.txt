  ------------------------------------------------------------------
  1. FUNDAMENTOS DE operator<<
  ------------------------------------------------------------------

  El operador << (inserción) se sobrecarga como función NO miembro:

    std::ostream& operator<<(std::ostream& os, const MiClase& obj) {
        os << ... ;  // Escribir representación de obj
        return os;
    }

  Características:
  • Retorna std::ostream& para permitir encadenamiento (a << b << c).
  • Recibe el objeto por const reference (no lo modifica).
  • Normalmente necesita ser friend si accede a miembros privados.

  Forma general:
    os << "Componente1: " << obj.comp1 << ", Componente2: " << obj.comp2;

  ------------------------------------------------------------------
  2. FUNDAMENTOS DE operator>>
  ------------------------------------------------------------------

  El operador >> (extracción) también se sobrecarga como función
  no miembro:

    std::istream& operator>>(std::istream& is, MiClase& obj) {
        is >> obj.miembro1 >> obj.miembro2;
        return is;
    }

  Características:
  • Retorna std::istream& para encadenamiento.
  • Recibe el objeto por referencia (lo modifica).
  • Debe validar la entrada y gestionar errores.

  ------------------------------------------------------------------
  3. MANEJO DE ERRORES EN >>
  ------------------------------------------------------------------

  Si la entrada falla, el stream entra en estado de error (failbit).
  El operador debe verificar y restaurar el estado si es necesario:

    std::istream& operator>>(std::istream& is, MiClase& obj) {
        is >> obj.miembro1;
        if (!is) return is;  // Propagar error
        is >> obj.miembro2;
        return is;
    }

  Estados del stream: good(), eof(), fail(), bad().

  ------------------------------------------------------------------
  4. FORMATEO AVANZADO
  ------------------------------------------------------------------

  Los streams permiten controlar el formato:

    os << std::hex << 255;          // "ff"
    os << std::setprecision(3) << 3.14159;  // "3.14"
    os << std::setw(10) << "Hola";  // "      Hola"
    os << std::boolalpha << true;   // "true"
    os << std::fixed << 3.14;       // "3.140000"

  Puedes usar estos manipuladores en operator<< para formatear.

  ------------------------------------------------------------------
  5. SERIALIZACIÓN
  ------------------------------------------------------------------

  Para serialización simple, << y >> son suficientes. Para formatos
  estructurados (JSON, XML, etc.), se recomienda usar bibliotecas
  especializadas, pero la sobrecarga de << y >> es el punto de partida.

  ------------------------------------------------------------------
  6. MANIPULADORES PERSONALIZADOS
  ------------------------------------------------------------------

  Puedes crear tus propios manipuladores para cambiar el formato:

    struct FormatoBonito {};
    std::ostream& operator<<(std::ostream& os, FormatoBonito) {
        os << std::boolalpha << std::fixed;
        return os;
    }

    // Uso: os << FormatoBonito() << obj;

  //====================================================================
  //  EJEMPLO 1: IMPRESIÓN BÁSICA DE CLASES
  //====================================================================
  #include <iostream>
  #include <string>
  #include <iomanip>

  class Producto {
  private:
      std::string nombre_;
      double precio_;
      int stock_;
  public:
      Producto(const std::string& n, double p, int s)
          : nombre_(n), precio_(p), stock_(s) {}

      friend std::ostream& operator<<(std::ostream& os, const Producto& p);
      friend std::istream& operator>>(std::istream& is, Producto& p);
  };

  std::ostream& operator<<(std::ostream& os, const Producto& p) {
      os << "Producto{ "
         << "nombre: \"" << p.nombre_ << "\", "
         << "precio: $" << std::fixed << std::setprecision(2) << p.precio_ << ", "
         << "stock: " << p.stock_
         << " }";
      return os;
  }

  std::istream& operator>>(std::istream& is, Producto& p) {
      std::cout << "Ingrese nombre, precio y stock: ";
      is >> std::ws;  // Saltar whitespace
      std::getline(is, p.nombre_, ',');
      is >> p.precio_ >> p.stock_;
      return is;
  }

  int main() {
      Producto prod("Laptop Gamer", 1250.99, 10);

      std::cout << "=== OPERATOR<< BÁSICO ===" << std::endl;
      std::cout << prod << std::endl;

      std::cout << "\nEncadenamiento:" << std::endl;
      std::cout << "Producto 1: " << prod << "\nProducto 2: "
                << Producto("Mouse", 25.50, 50) << std::endl;

      return 0;
  }

  //====================================================================
  //  EJEMPLO 2: FORMATO AVANZADO Y MANIPULADORES
  //====================================================================
  #include <iostream>
  #include <string>
  #include <iomanip>
  #include <vector>

  class Reporte {
      std::string titulo_;
      std::vector<std::pair<std::string, double>> datos_;
  public:
      Reporte(const std::string& t) : titulo_(t) {}

      void agregar(const std::string& item, double valor) {
          datos_.emplace_back(item, valor);
      }

      friend std::ostream& operator<<(std::ostream& os, const Reporte& r);
  };

  std::ostream& operator<<(std::ostream& os, const Reporte& r) {
      // Guardar estado anterior
      auto fmt = os.flags();
      auto prec = os.precision();

      os << "\n" << std::string(50, '=') << "\n";
      os << "  " << r.titulo_ << "\n";
      os << std::string(50, '=') << "\n";

      os << std::left << std::setw(30) << "Concepto"
         << std::right << std::setw(15) << "Valor" << "\n";
      os << std::string(45, '-') << "\n";

      double total = 0;
      for (const auto& [item, valor] : r.datos_) {
          os << std::left << std::setw(30) << item
             << std::right << "$" << std::setw(13)
             << std::fixed << std::setprecision(2) << valor << "\n";
          total += valor;
      }

      os << std::string(45, '-') << "\n";
      os << std::left << std::setw(30) << "TOTAL"
         << std::right << "$" << std::setw(13)
         << std::fixed << std::setprecision(2) << total << "\n";
      os << std::string(50, '=') << "\n";

      // Restaurar estado
      os.flags(fmt);
      os.precision(prec);

      return os;
  }

  int main() {
      std::cout << "=== FORMATO AVANZADO ===" << std::endl;

      Reporte rpt("Reporte de Ventas - Julio 2024");
      rpt.agregar("Laptops", 15250.00);
      rpt.agregar("Monitores", 8730.50);
      rpt.agregar("Teclados", 2340.75);
      rpt.agregar("Mouses", 1890.25);
      rpt.agregar("Impresoras", 3450.00);

      std::cout << rpt << std::endl;

      // Demostrar que el estado se restaura
      std::cout << "Después: " << 3.14159 << " (precisión normal)" << std::endl;

      return 0;
  }

  //====================================================================
  //  EJEMPLO 3: operator>> CON VALIDACIÓN
  //====================================================================
  #include <iostream>
  #include <string>
  #include <sstream>
  #include <limits>

  class Fecha {
      int dia_, mes_, anyo_;
  public:
      Fecha(int d = 1, int m = 1, int a = 2000) : dia_(d), mes_(m), anyo_(a) {}

      bool esValida() const {
          if (mes_ < 1 || mes_ > 12 || dia_ < 1) return false;
          static const int diasMes[] = {31,28,31,30,31,30,31,31,30,31,30,31};
          int maxDias = diasMes[mes_ - 1];
          if (mes_ == 2 && ((anyo_ % 4 == 0 && anyo_ % 100 != 0) || anyo_ % 400 == 0))
              maxDias = 29;
          return dia_ <= maxDias;
      }

      friend std::ostream& operator<<(std::ostream& os, const Fecha& f);
      friend std::istream& operator>>(std::istream& is, Fecha& f);
  };

  std::ostream& operator<<(std::ostream& os, const Fecha& f) {
      os << f.dia_ << "/" << f.mes_ << "/" << f.anyo_;
      return os;
  }

  std::istream& operator>>(std::istream& is, Fecha& f) {
      char sep1, sep2;
      int d, m, a;

      // Intentar leer en formato dd/mm/aaaa
      is >> d >> sep1 >> m >> sep2 >> a;

      if (!is || sep1 != '/' || sep2 != '/') {
          // Error de formato
          is.setstate(std::ios::failbit);
          return is;
      }

      Fecha temp(d, m, a);
      if (!temp.esValida()) {
          is.setstate(std::ios::failbit);
          return is;
      }

      f = temp;
      return is;
  }

  template<typename T>
  T leerConValidacion(const std::string& mensaje) {
      T valor;
      while (true) {
          std::cout << mensaje;
          std::cin >> valor;
          if (std::cin.good()) return valor;
          std::cout << "Entrada inválida. Intente de nuevo.\n";
          std::cin.clear();
          std::cin.ignore(std::numeric_limits<std::streamsize>::max(), '\n');
      }
  }

  int main() {
      std::cout << "=== OPERATOR>> CON VALIDACIÓN ===" << std::endl;

      Fecha f;
      std::cout << "Ingrese fecha (dd/mm/aaaa): ";
      std::cin >> f;

      if (std::cin) {
          std::cout << "Fecha ingresada: " << f << std::endl;
      } else {
          std::cout << "Error: formato de fecha inválido." << std::endl;
          std::cin.clear();
          std::cin.ignore(std::numeric_limits<std::streamsize>::max(), '\n');
      }

      // Usando stringstream
      std::cout << "\nProbando con stringstream:" << std::endl;
      std::istringstream iss("31/12/2024");
      Fecha f2;
      if (iss >> f2) {
          std::cout << "Leído correctamente: " << f2 << std::endl;
      }

      std::istringstream iss2("99/99/9999");
      Fecha f3;
      if (!(iss2 >> f3)) {
          std::cout << "Fecha inválida detectada correctamente." << std::endl;
      }

      return 0;
  }

  //====================================================================
  //  EJEMPLO 4: SERIALIZACIÓN COMPLETA
  //====================================================================
  #include <iostream>
  #include <string>
  #include <sstream>
  #include <vector>
  #include <iomanip>

  class Persona {
      std::string nombre_;
      int edad_;
      double altura_;
      bool activo_;
  public:
      Persona() = default;
      Persona(const std::string& n, int e, double a, bool act)
          : nombre_(n), edad_(e), altura_(a), activo_(act) {}

      std::string serializar() const {
          std::ostringstream oss;
          oss << *this;
          return oss.str();
      }

      static Persona deserializar(const std::string& datos) {
          Persona p;
          std::istringstream iss(datos);
          iss >> p;
          return p;
      }

      friend std::ostream& operator<<(std::ostream& os, const Persona& p) {
          os << p.nombre_ << "|"
             << p.edad_ << "|"
             << std::fixed << std::setprecision(2) << p.altura_ << "|"
             << (p.activo_ ? "1" : "0");
          return os;
      }

      friend std::istream& operator>>(std::istream& is, Persona& p) {
          std::string linea;
          if (!std::getline(is, linea, '\n')) return is;

          std::istringstream ss(linea);
          std::string token;

          std::getline(ss, p.nombre_, '|');
          std::getline(ss, token, '|'); p.edad_ = std::stoi(token);
          std::getline(ss, token, '|'); p.altura_ = std::stod(token);
          std::getline(ss, token, '|'); p.activo_ = (token == "1");

          return is;
      }

      void mostrar() const {
          std::cout << "  Nombre: " << nombre_
                    << ", Edad: " << edad_
                    << ", Altura: " << altura_
                    << ", Activo: " << (activo_ ? "Sí" : "No")
                    << std::endl;
      }
  };

  int main() {
      std::cout << "=== SERIALIZACIÓN ===" << std::endl;

      Persona p1("Ana García", 30, 1.68, true);
      Persona p2("Carlos López", 25, 1.75, false);

      std::cout << "Personas originales:" << std::endl;
      p1.mostrar();
      p2.mostrar();

      // Serializar a string
      std::string datos1 = p1.serializar();
      std::string datos2 = p2.serializar();

      std::cout << "\nDatos serializados:" << std::endl;
      std::cout << "  p1: \"" << datos1 << "\"" << std::endl;
      std::cout << "  p2: \"" << datos2 << "\"" << std::endl;

      // Serializar múltiples a un stream
      std::ostringstream archivo;
      archivo << p1 << "\n" << p2 << "\n";

      // Deserializar del stream
      std::cout << "\nDeserializando del stream:" << std::endl;
      std::istringstream archivoLectura(archivo.str());
      Persona p3, p4;
      archivoLectura >> p3 >> p4;

      p3.mostrar();
      p4.mostrar();

      return 0;
  }

  //  EJEMPLO 5: FORMATO JSON CON <<
  //====================================================================
  #include <iostream>
  #include <string>
  #include <vector>
  #include <sstream>

  class JSONObject {
      std::string nombre_;
      std::string valor_;
      std::vector<JSONObject> hijos_;
      bool esArray_;
  public:
      JSONObject(const std::string& n = "", const std::string& v = "")
          : nombre_(n), valor_(v), esArray_(false) {}

      void agregar(const JSONObject& hijo) { hijos_.push_back(hijo); }
      void marcarArray() { esArray_ = true; }

      friend std::ostream& operator<<(std::ostream& os, const JSONObject& obj) {
          os << "\"" << obj.nombre_ << "\": ";

          if (!obj.hijos_.empty()) {
              if (obj.esArray_) os << "[\n";
              else os << "{\n";

              for (size_t i = 0; i < obj.hijos_.size(); ++i) {
                  os << "    " << obj.hijos_[i];
                  if (i < obj.hijos_.size() - 1) os << ",";
                  os << "\n";
              }

              if (obj.esArray_) os << "  ]";
              else os << "  }";
          } else {
              os << "\"" << obj.valor_ << "\"";
          }

          return os;
      }
  };

  int main() {
      std::cout << "=== JSON CON << ===" << std::endl;

      JSONObject persona("persona");
      JSONObject nombre("nombre", "Ana López");
      JSONObject edad("edad", "30");
      JSONObject activo("activo", "true");

      JSONObject direccion("direccion");
      JSONObject calle("calle", "Av. Central 123");
      JSONObject ciudad("ciudad", "CDMX");
      JSONObject pais("pais", "México");
      direccion.agregar(calle);
      direccion.agregar(ciudad);
      direccion.agregar(pais);

      JSONObject telefonos("telefonos");
      telefonos.marcarArray();
      JSONObject tel1("", "555-1234");
      JSONObject tel2("", "555-5678");
      telefonos.agregar(tel1);
      telefonos.agregar(tel2);

      persona.agregar(nombre);
      persona.agregar(edad);
      persona.agregar(activo);
      persona.agregar(direccion);
      persona.agregar(telefonos);

      std::cout << "{\n";
      std::cout << "  " << persona << "\n";
      std::cout << "}" << std::endl;

      return 0;
  }

================================================================================
  RESUMEN
================================================================================

  • operator<< se sobrecarga como función no miembro: retorna ostream&.
  • operator>> se sobrecarga como función no miembro: retorna istream&.
  • La forma canónica usa friend para acceso a miembros privados.
  • operator>> debe validar la entrada y gestionar errores del stream.
  • Los manipuladores (setw, setprecision, fixed, etc.) controlan formato.
  • Guardar y restaurar el estado del stream con flags() y precision().
  • Se pueden crear manipuladores personalizados.
  • Para serialización, << y >> son el punto de partida ideal.

================================================================================
  PREGUNTAS FRECUENTES (FAQ)
================================================================================

  P: ¿Por qué << y >> deben ser no-miembros?
  R: Porque el operando izquierdo es un stream (ostream/istream), no
    nuestra clase. No podemos modificar la clase stream.

  P: ¿Siempre necesito friend para <<?
  R: Si tu clase tiene getters públicos que exponen todo lo necesario,
    no. Pero típicamente es más limpio usar friend.

  P: ¿Cómo manejo errores en >>?
  R: Verifica el estado del stream después de cada extracción. Si algo
    falla, establece failbit con is.setstate(std::ios::failbit).

  P: ¿Puedo formatear en << sin afectar el stream global?
  R: Sí. Guarda el estado con auto fmt = os.flags() y restaura con
    os.flags(fmt) al final.

  P: ¿Cómo serializo objetos con punteros?
  R: Puedes serializar el valor apuntado o un identificador. Para
    grafos, necesitas manejar referencias circulares.
