  ------------------------------------------------------------------
  1. OPERADORES UNARIOS EN C++
  ------------------------------------------------------------------

  Los operadores unarios actúan sobre un solo operando. C++ permite
  sobrecargar los siguientes:

    +  -  ++  --  !  ~  *  &  sizeof  (no mutable)

  Sobrecargables: +, -, ++, --, !, ~, *, &

  NO sobrecargables: sizeof, typeid, alignof

  ------------------------------------------------------------------
  2. PREFIJO VS POSTFIJO (++ Y --)
  ------------------------------------------------------------------

  La diferencia sintáctica es sutil pero crucial:

    // Prefijo: incrementa y devuelve el nuevo valor
    MiClase& operator++();      // ++x

    // Postfijo: incrementa pero devuelve el valor anterior
    MiClase operator++(int);    // x++  (int ficticio para distinguir)

  El parámetro 'int' en postfijo es un marcador sintáctico; no recibe
  ningún valor real. Sirve únicamente para distinguir las dos versiones.

  Convenciones de retorno:
  • Prefijo: retorna referencia al objeto modificado (*this).
  • Postfijo: retorna una copia del valor anterior (por valor).

  Rendimiento: prefijo es más eficiente porque no crea copia temporal.
  Preferir ++i sobre i++ en la mayoría de contextos.

  ------------------------------------------------------------------
  3. OPERADOR UNARIO - Y +
  ------------------------------------------------------------------

  El operador unario - (negación) y + (identidad):

    MiClase operator-() const;   // -obj
    MiClase operator+() const;   // +obj

  - debe devolver el valor negado (sin modificar el original).
  + normalmente devuelve el mismo valor (copia de *this).

  Estos operadores NO modifican el objeto, por lo que se marcan const.

  ------------------------------------------------------------------
  4. OPERADOR ! (NEGACIÓN LÓGICA)
  ------------------------------------------------------------------

  bool operator!() const;  // !obj

  Se usa para negar lógicamente un objeto. Común en:
  • Smart pointers (verificar si apuntan a algo).
  • Validadores (verificar si algo es inválido).
  • Iteradores (verificar si llegó al final).

  Retorna bool normalmente, pero puede retornar cualquier tipo
  convertible a bool.

  ------------------------------------------------------------------
  5. OPERADOR ~ (COMPLEMENTO BITS)
  ------------------------------------------------------------------

  MiClase operator~() const;  // ~obj

  Aplica complemento bitwise. Útil para:
  • Máscaras de bits.
  • Conjuntos (set complement).
  • Tipos de banderas (flags).

  ------------------------------------------------------------------
  6. OPERADOR * (DESREFERENCIA) Y -> (ACCESO)
  ------------------------------------------------------------------

  T& operator*();        // *obj (desreferencia)
  const T& operator*() const;
  T* operator->();       // obj->miembro (acceso)

  Estos son esenciales para crear smart pointers e iteradores.

  operator-> tiene la particularidad de que su retorno se usa
  automáticamente para acceder al miembro. Si retorna un puntero,
  el compilador aplica -> recursivamente.

  ------------------------------------------------------------------
  7. OPERADOR & (DIRECCIÓN)
  ------------------------------------------------------------------

  T* operator&();  // &obj

  Sobrecargar este operador es RARO y peligroso. Cambia el
  comportamiento de tomar dirección. Generalmente NO se recomienda
  a menos que tengas una razón muy específica.

  Desde C++11, std::addressof() obtiene la dirección real incluso si
  operator& está sobrecargado.

  ```cpp
  //====================================================================
  //  EJEMPLO 1: PREFIJO Y POSTFIJO COMPLETO
  //====================================================================
  #include <iostream>
  #include <string>

  class Contador {
  private:
      int valor_;
      std::string nombre_;
  public:
      Contador(int v = 0, const std::string& n = "Contador")
          : valor_(v), nombre_(n) {}

      // Prefijo: ++c
      Contador& operator++() {
          ++valor_;
          std::cout << nombre_ << " ++prefijo: " << valor_ << std::endl;
          return *this;
      }

      // Postfijo: c++
      Contador operator++(int) {
          Contador temporal(*this);  // Guardar estado anterior
          ++valor_;                   // Incrementar
          std::cout << nombre_ << " postfijo++: " << valor_
                    << " (anterior: " << temporal.valor_ << ")" << std::endl;
          return temporal;            // Retornar estado anterior
      }

      // Prefijo: --c
      Contador& operator--() {
          --valor_;
          std::cout << nombre_ << " --prefijo: " << valor_ << std::endl;
          return *this;
      }

      // Postfijo: c--
      Contador operator--(int) {
          Contador temporal(*this);
          --valor_;
          std::cout << nombre_ << " postfijo--: " << valor_
                    << " (anterior: " << temporal.valor_ << ")" << std::endl;
          return temporal;
      }

      int valor() const { return valor_; }
  };

  int main() {
      std::cout << "=== PREFIJO VS POSTFIJO ===\n";

      Contador c(5);

      std::cout << "\n--- Postfijo ---\n";
      Contador resultado1 = c++;  // c++ retorna el valor anterior
      std::cout << "  resultado1 = " << resultado1.valor() << std::endl;
      std::cout << "  c ahora = " << c.valor() << std::endl;

      std::cout << "\n--- Prefijo ---\n";
      Contador resultado2 = ++c;  // ++c retorna el nuevo valor
      std::cout << "  resultado2 = " << resultado2.valor() << std::endl;
      std::cout << "  c ahora = " << c.valor() << std::endl;

      std::cout << "\n--- Diferencia práctica ---\n";
      int arr[] = {10, 20, 30};
      int i = 0;
      std::cout << "  arr[i++] = " << arr[i++] << " (i ahora es " << i << ")\n";
      i = 0;
      std::cout << "  arr[++i] = " << arr[++i] << " (i ahora es " << i << ")\n";

      return 0;
  }
  ```

  ```cpp
  //====================================================================
  //  EJEMPLO 2: PUNTO 2D CON OPERADORES UNARIOS COMPLETOS
  //====================================================================
  #include <iostream>
  #include <cmath>

  class Punto {
  private:
      double x_, y_;
  public:
      Punto(double x = 0, double y = 0) : x_(x), y_(y) {}

      // Negación unaria: -p
      Punto operator-() const {
          return Punto(-x_, -y_);
      }

      // Identidad unaria: +p
      Punto operator+() const {
          return *this;
      }

      // Negación lógica: !p (devuelve true si está en origen)
      bool operator!() const {
          return x_ == 0 && y_ == 0;
      }

      // Prefijo ++
      Punto& operator++() {
          ++x_;
          ++y_;
          return *this;
      }

      // Postfijo ++
      Punto operator++(int) {
          Punto temp(*this);
          ++(*this);
          return temp;
      }

      // Prefijo --
      Punto& operator--() {
          --x_;
          --y_;
          return *this;
      }

      // Postfijo --
      Punto operator--(int) {
          Punto temp(*this);
          --(*this);
          return temp;
      }

      // Complemento bitwise (~): intercambia x e y
      Punto operator~() const {
          return Punto(y_, x_);
      }

      double x() const { return x_; }
      double y() const { return y_; }
  };

  std::ostream& operator<<(std::ostream& os, const Punto& p) {
      os << "(" << p.x() << ", " << p.y() << ")";
      return os;
  }

  int main() {
      std::cout << "=== OPERADORES UNARIOS EN PUNTO ===\n";

      Punto p1(3, 4);
      Punto origen(0, 0);

      std::cout << "p1 = " << p1 << std::endl;
      std::cout << "-p1 = " << (-p1) << std::endl;
      std::cout << "+p1 = " << (+p1) << std::endl;
      std::cout << "!p1 = " << (!p1) << " (!origen = " << (!origen) << ")" << std::endl;
      std::cout << "~p1 = " << (~p1) << std::endl;

      ++p1;
      std::cout << "++p1 = " << p1 << std::endl;

      std::cout << "p1++ = " << (p1++) << " (luego p1 = " << p1 << ")" << std::endl;

      return 0;
  }
  ```

  ```cpp
  //====================================================================
  //  EJEMPLO 3: SMART POINTER SIMPLE CON OPERADORES UNARIOS
  //====================================================================
  #include <iostream>
  #include <string>

  template<typename T>
  class UniquePtr {
  private:
      T* ptr_;
  public:
      explicit UniquePtr(T* p = nullptr) : ptr_(p) {}

      ~UniquePtr() { delete ptr_; }

      // Desreferencia: *ptr
      T& operator*() {
          return *ptr_;
      }

      const T& operator*() const {
          return *ptr_;
      }

      // Acceso a miembro: ptr->metodo()
      T* operator->() {
          return ptr_;
      }

      const T* operator->() const {
          return ptr_;
      }

      // Negación lógica: !ptr (¿está vacío?)
      bool operator!() const {
          return ptr_ == nullptr;
      }

      // Conversión a bool (para if (ptr))
      explicit operator bool() const {
          return ptr_ != nullptr;
      }

      // Prohibir copia
      UniquePtr(const UniquePtr&) = delete;
      UniquePtr& operator=(const UniquePtr&) = delete;

      // Move
      UniquePtr(UniquePtr&& otro) noexcept : ptr_(otro.ptr_) {
          otro.ptr_ = nullptr;
      }

      UniquePtr& operator=(UniquePtr&& otro) noexcept {
          if (this != &otro) {
              delete ptr_;
              ptr_ = otro.ptr_;
              otro.ptr_ = nullptr;
          }
          return *this;
      }
  };

  class Objeto {
      std::string nombre_;
  public:
      Objeto(const std::string& n) : nombre_(n) {
          std::cout << "  Objeto "" << nombre_ << "" creado\n";
      }
      ~Objeto() {
          std::cout << "  Objeto "" << nombre_ << "" destruido\n";
      }
      void saludar() const {
          std::cout << "  Hola desde " << nombre_ << "\n";
      }
  };

  int main() {
      std::cout << "=== SMART POINTER UNARIO ===\n";

      UniquePtr<Objeto> ptr(new Objeto("MiObjeto"));

      // operator*
      (*ptr).saludar();

      // operator->
      ptr->saludar();

      // operator!
      std::cout << "!ptr = " << !ptr << " (debería ser 0)" << std::endl;

      // Conversión a bool
      if (ptr) {
          std::cout << "ptr es válido\n";
      }

      // Move
      UniquePtr<Objeto> ptr2 = std::move(ptr);
      std::cout << "Después del move: !ptr = " << !ptr << std::endl;

      // Al salir del ámbito, ptr2 se destruye y libera memoria
      std::cout << "Saliendo del ámbito...\n";
      return 0;
  }
  ```

  ```cpp
  //====================================================================
  //  EJEMPLO 4: ITERADOR CON OPERADORES UNARIOS
  //====================================================================
  #include <iostream>
  #include <string>
  #include <stdexcept>

  class Arreglo {
      int* datos_;
      size_t tam_;
  public:
      Arreglo(size_t n) : tam_(n) {
          datos_ = new int[n];
          for (size_t i = 0; i < n; ++i) datos_[i] = i * 10;
      }

      ~Arreglo() { delete[] datos_; }

      class Iterador {
      private:
          int* ptr_;
      public:
          Iterador(int* p) : ptr_(p) {}

          // Prefijo ++
          Iterador& operator++() {
              ++ptr_;
              return *this;
          }

          // Postfijo ++
          Iterador operator++(int) {
              Iterador temp(*this);
              ++(*this);
              return temp;
          }

          // Prefijo --
          Iterador& operator--() {
              --ptr_;
              return *this;
          }

          // Postfijo --
          Iterador operator--(int) {
              Iterador temp(*this);
              --(*this);
              return temp;
          }

          // Desreferencia
          int& operator*() { return *ptr_; }
          const int& operator*() const { return *ptr_; }

          // Acceso a miembro (para structs)
          int* operator->() { return ptr_; }
          const int* operator->() const { return ptr_; }

          // Comparación
          bool operator==(const Iterador& otro) const { return ptr_ == otro.ptr_; }
          bool operator!=(const Iterador& otro) const { return ptr_ != otro.ptr_; }
      };

      Iterador begin() { return Iterador(datos_); }
      Iterador end() { return Iterador(datos_ + tam_); }
  };

  int main() {
      std::cout << "=== ITERADOR CON ++ Y -- ===\n";

      Arreglo arr(5);

      std::cout << "Recorrido hacia adelante:\n";
      for (auto it = arr.begin(); it != arr.end(); ++it) {
          std::cout << "  " << *it;
      }
      std::cout << std::endl;

      std::cout << "Recorrido hacia atrás:\n";
      auto it = arr.end();
      --it;  // Retroceder al último
      while (true) {
          std::cout << "  " << *it;
          if (it == arr.begin()) break;
          --it;
      }
      std::cout << std::endl;

      std::cout << "Usando postfijo (menos eficiente pero funcional):\n";
      for (auto it = arr.begin(); it != arr.end(); it++) {
          std::cout << "  " << *it;
      }
      std::cout << std::endl;

      return 0;
  }
  ```

  ```cpp
  //====================================================================
  //  EJEMPLO 5: CONJUNTO DE BITS CON OPERADORES ~ Y !
  //====================================================================
  #include <iostream>
  #include <string>
  #include <bitset>

  template<size_t N>
  class ConjuntoBits {
      std::bitset<N> bits_;
  public:
      ConjuntoBits() = default;

      void agregar(size_t i) { if (i < N) bits_.set(i); }
      void quitar(size_t i) { if (i < N) bits_.reset(i); }
      bool contiene(size_t i) const { return i < N && bits_.test(i); }

      // Complemento: ~conjunto
      ConjuntoBits operator~() const {
          ConjuntoBits resultado;
          resultado.bits_ = ~bits_;
          return resultado;
      }

      // Negación lógica: !conjunto (¿está vacío?)
      bool operator!() const {
          return bits_.none();
      }

      // Intersección (usando &)
      ConjuntoBits operator&(const ConjuntoBits& otro) const {
          ConjuntoBits r;
          r.bits_ = bits_ & otro.bits_;
          return r;
      }

      // Unión (usando |)
      ConjuntoBits operator|(const ConjuntoBits& otro) const {
          ConjuntoBits r;
          r.bits_ = bits_ | otro.bits_;
          return r;
      }

      size_t contar() const { return bits_.count(); }

      std::string str() const {
          std::string s;
          for (size_t i = 0; i < N; ++i) {
              if (bits_.test(i)) s += std::to_string(i) + " ";
          }
          return s.empty() ? "{}" : "{ " + s + "}";
      }
  };

  int main() {
      std::cout << "=== CONJUNTO DE BITS ===\n";

      ConjuntoBits<10> A, B;

      A.agregar(1); A.agregar(3); A.agregar(5); A.agregar(7);
      B.agregar(3); B.agregar(4); B.agregar(5); B.agregar(6);

      std::cout << "A = " << A.str() << std::endl;
      std::cout << "B = " << B.str() << std::endl;
      std::cout << "~A = " << (~A).str() << std::endl;
      std::cout << "!A = " << !A << " (no-vacío: " << A.contar() << " elementos)" << std::endl;

      ConjuntoBits<10> vacio;
      std::cout << "!vacio = " << !vacio << std::endl;

      std::cout << "A & B = " << (A & B).str() << std::endl;
      std::cout << "A | B = " << (A | B).str() << std::endl;

      return 0;
  }
  ```

  ```cpp
  //====================================================================
  //  EJEMPLO 6: COMPARACIÓN RENDIMIENTO PREFIJO VS POSTFIJO
  //====================================================================
  #include <iostream>
  #include <chrono>
  #include <vector>

  class IntWrapper {
      int v_;
  public:
      IntWrapper(int v = 0) : v_(v) {}

      IntWrapper& operator++() {  // Prefijo
          ++v_;
          return *this;
      }

      IntWrapper operator++(int) {  // Postfijo
          IntWrapper temp(*this);
          ++v_;
          return temp;
      }

      int valor() const { return v_; }
  };

  int main() {
      const int N = 10000000;
      IntWrapper w(0);

      // Prefijo
      auto start = std::chrono::steady_clock::now();
      for (int i = 0; i < N; ++i) {
          ++w;
      }
      auto end = std::chrono::steady_clock::now();
      auto prefijo_ms = std::chrono::duration_cast<std::chrono::milliseconds>(end - start).count();
      std::cout << "Prefijo ++: " << prefijo_ms << " ms (resultado: " << w.valor() << ")\n";

      // Postfijo
      w = IntWrapper(0);
      start = std::chrono::steady_clock::now();
      for (int i = 0; i < N; ++i) {
          w++;
      }
      end = std::chrono::steady_clock::now();
      auto postfijo_ms = std::chrono::duration_cast<std::chrono::milliseconds>(end - start).count();
      std::cout << "Postfijo ++: " << postfijo_ms << " ms (resultado: " << w.valor() << ")\n";

      std::cout << "\nDiferencia: " << (postfijo_ms - prefijo_ms) << " ms ("
                << (100.0 * (postfijo_ms - prefijo_ms) / prefijo_ms) << "% más lento)\n";

      std::cout << "\nCon tipos primitivos, el compilador optimiza ambos igual.\n";
      std::cout << "Con clases pesadas, la copia temporal del postfijo es costosa.\n";

      return 0;
  }
  ```

================================================================================
  RESUMEN
================================================================================

  • ++ y --: prefijo retorna referencia (++x), postfijo retorna valor (x++).
  • El postfijo se implementa como: copia temporal → incrementa → retorna copia.
  • Prefijo es más eficiente que postfijo (evita copia temporal).
  • operator-() y operator+() son const y no modifican el objeto.
  • operator!() retorna bool indicando si el objeto es "falso/vacío/nulo".
  • operator*() (desreferencia) y operator->() (acceso) esenciales para
    smart pointers e iteradores.
  • operator~() aplica complemento bitwise, útil para conjuntos/máscaras.
  • operator&() sobrecargar es raro y peligroso; usar std::addressof.

================================================================================
  PREGUNTAS FRECUENTES (FAQ)
================================================================================

  P: ¿Por qué el postfijo tiene un parámetro 'int' que no se usa?
  R: Es un truco sintáctico del lenguaje para distinguir prefijo de
    postfijo. El compilador pasa 0 como argumento, pero no se usa.

  P: ¿Prefijo siempre es más rápido que postfijo?
  R: Con tipos primitivos, el compilador optimiza y son iguales. Con
    clases que requieren copia (como iteradores pesados), prefijo gana.

  P: ¿Puedo retornar void en operator++?
  R: Técnicamente sí, pero viola las convenciones y rompe la
    compatibilidad con tipos primitivos. No lo hagas.

  P: ¿operator-> debe retornar un puntero?
  R: Puede retornar cualquier objeto para el cual operator-> esté
    definido. El compilador aplica -> recursivamente hasta llegar a
    un puntero real.

  P: ¿Cuándo sobrecargar operator&?
  R: Casi nunca. Solo si estás implementando un smart pointer muy
    especializado. Prefiere std::addressof().
