  ------------------------------------------------------------------
  1. ¿QUÉ ES UNA INTERFAZ?
  ------------------------------------------------------------------

  En C++, una interfaz es una clase que contiene ÚNICAMENTE:
  • Funciones virtuales puras (métodos públicos).
  • Destructor virtual (puro o con implementación).
  • NO tiene datos miembro (o solo static const).
  • NO tiene constructores (o solo protected default).

  Convención de nomenclatura: prefijo I (ej: ISerializable, IImprimible).

    class IImprimible {
    public:
        virtual ~IImprimible() = default;
        virtual void imprimir() const = 0;
        virtual int obtenerPaginas() const = 0;
    };

  ------------------------------------------------------------------
  2. DIFERENCIA CON CLASE ABSTRACTA
  ------------------------------------------------------------------

  | Característica      | Interfaz           | Clase Abstracta    |
  |---------------------|--------------------|--------------------|
  | Datos miembro       | No (solo static)   | Sí                 |
  | Métodos concretos   | No                 | Sí                 |
  | Constructores       | No (o protected)   | Sí                 |
  | Propósito           | Contrato puro      | Base con estado    |
  | Herencia múltiple   | Recomendada        | Posible pero riesg.|

  ------------------------------------------------------------------
  3. HERENCIA MÚLTIPLE DE INTERFACES
  ------------------------------------------------------------------

  Una clase puede implementar múltiples interfaces:

    class Documento : public IImprimible, public IEscaneable,
                      public ISerializable {
        // Debe implementar TODOS los métodos de las 3 interfaces
    };

  Cada interfaz representa una capacidad diferente. Una clase adquiere
  todas las capacidades de las interfaces que implementa.

  ------------------------------------------------------------------
  4. CAPABILITY-BASED DESIGN (DISEÑO BASADO EN CAPACIDADES)
  ------------------------------------------------------------------

  En lugar de preguntar "¿qué es este objeto?", preguntamos
  "¿qué puede hacer este objeto?". Las interfaces definen capacidades.

  Ejemplo:
    • ISerializable: puede serializarse a JSON/XML.
    • IComparable: puede compararse con otros objetos.
    • IRenderizable: puede dibujarse en pantalla.
    • IAlmacenable: puede guardarse en base de datos.

  Una clase 'Producto' podría implementar ISerializable y IAlmacenable.
  Una clase 'VideoJuego' podría implementar IRenderizable e ISerializable.

  ------------------------------------------------------------------
  5. PRINCIPIO DE SEGREGACIÓN DE INTERFACES (ISP)
  ------------------------------------------------------------------

  ISP establece: "Ningún cliente debe ser forzado a depender de
  interfaces que no utiliza."

  MAL: Una interfaz grande IVehiculo con volar(), navegar(), conducir().
  BIEN: IVolable, INavegable, IConducible separadas.

  Beneficios:
  • Interfaces más pequeñas y cohesivas.
  • Las clases solo implementan lo que necesitan.
  • Menor acoplamiento.

  ------------------------------------------------------------------
  6. PATRÓN STRATEGY CON INTERFACES
  ------------------------------------------------------------------

  El patrón Strategy permite intercambiar algoritmos en tiempo de
  ejecución. Se implementa con una interfaz que define el algoritmo
  y clases concretas que lo implementan.

    class ICompresion {
    public:
        virtual ~ICompresion() = default;
        virtual void comprimir(const string& datos) = 0;
        virtual string descomprimir(const string& datos) = 0;
    };

    class CompresionZIP : public ICompresion { ... };
    class CompresionGZIP : public ICompresion { ... };

  ------------------------------------------------------------------
  7. NAVEGACIÓN ENTRE INTERFACES CON dynamic_cast
  ------------------------------------------------------------------

  dynamic_cast permite convertir entre interfaces en tiempo de
  ejecución, permitiendo descubrir qué capacidades tiene un objeto.

    IImprimible* doc = new Documento();
    if (auto* esc = dynamic_cast<IEscaneable*>(doc)) {
        esc->escanear();  // Capacidad descubierta en runtime
    }

  ```cpp
  //====================================================================
  //  EJEMPLO 1: INTERFACES PURAS EN C++
  //====================================================================
  #include <iostream>
  #include <string>
  #include <vector>

  class IImprimible {
  public:
      virtual ~IImprimible() = default;
      virtual void imprimir() const = 0;
      virtual int numPaginas() const = 0;
      virtual std::string titulo() const = 0;
  };

  class IExportable {
  public:
      virtual ~IExportable() = default;
      virtual void exportarPDF() const = 0;
      virtual void exportarHTML() const = 0;
  };

  class IAlmacenable {
  public:
      virtual ~IAlmacenable() = default;
      virtual void guardar(const std::string& ruta) const = 0;
      virtual void cargar(const std::string& ruta) = 0;
  };

  class Informe : public IImprimible, public IExportable,
                  public IAlmacenable {
      std::string titulo_;
      std::string contenido_;
      std::string autor_;
  public:
      Informe(const std::string& t, const std::string& c,
              const std::string& a)
          : titulo_(t), contenido_(c), autor_(a) {}

      void imprimir() const override {
          std::cout << "=== INFORME: " << titulo_ << " ===\n"
                    << "Autor: " << autor_ << "\n"
                    << contenido_ << "\n"
                    << "=== FIN INFORME ===" << std::endl;
      }

      int numPaginas() const override {
          return (contenido_.size() / 2000) + 1;
      }

      std::string titulo() const override { return titulo_; }

      void exportarPDF() const override {
          std::cout << "Exportando " << titulo_ << " a PDF... OK"
                    << std::endl;
      }

      void exportarHTML() const override {
          std::cout << "Exportando " << titulo_ << " a HTML... OK"
                    << std::endl;
      }

      void guardar(const std::string& ruta) const override {
          std::cout << "Guardando " << titulo_ << " en " << ruta
                    << std::endl;
      }

      void cargar(const std::string& ruta) override {
          std::cout << "Cargando informe desde " << ruta << std::endl;
      }
  };

  void procesarDocumento(IImprimible& doc) {
      std::cout << "Procesando: " << doc.titulo()
                << " (" << doc.numPaginas() << " pág)\n";
      doc.imprimir();
  }

  int main() {
      Informe inf("Reporte Anual", "Datos de ventas 2024...", "Ana García");

      std::cout << "=== COMO IImprimible ===\n";
      procesarDocumento(inf);

      std::cout << "\n=== COMO IExportable ===\n";
      IExportable& exp = inf;
      exp.exportarPDF();
      exp.exportarHTML();

      std::cout << "\n=== COMO IAlmacenable ===\n";
      IAlmacenable& alm = inf;
      alm.guardar("/docs/informe.dat");
      alm.cargar("/docs/informe.dat");

      return 0;
  }
  ```

  ```cpp
  //====================================================================
  //  EJEMPLO 2: CAPABILITY-BASED DESIGN
  //====================================================================
  #include <iostream>
  #include <string>
  #include <vector>
  #include <memory>

  class IVolable {
  public:
      virtual ~IVolable() = default;
      virtual void despegar() = 0;
      virtual void aterrizar() = 0;
      virtual double altitud() const = 0;
  };

  class IConducible {
  public:
      virtual ~IConducible() = default;
      virtual void acelerar() = 0;
      virtual void frenar() = 0;
      virtual void girar(int grados) = 0;
  };

  class INavegable {
  public:
      virtual ~INavegable() = default;
      virtual void zarpar() = 0;
      virtual void atracar() = 0;
      virtual double velocidad() const = 0;
  };

  class Avion : public IVolable {
      double altitud_ = 0;
  public:
      void despegar() override {
          std::cout << "Avión despegando...\n";
          altitud_ = 10000;
      }
      void aterrizar() override {
          std::cout << "Avión aterrizando...\n";
          altitud_ = 0;
      }
      double altitud() const override { return altitud_; }
  };

  class Auto : public IConducible {
      int velocidad_ = 0;
  public:
      void acelerar() override { velocidad_ += 10; std::cout << "Auto acelera a " << velocidad_ << " km/h\n"; }
      void frenar() override { velocidad_ = std::max(0, velocidad_ - 10); }
      void girar(int g) override { std::cout << "Auto gira " << g << " grados\n"; }
  };

  class Barco : public INavegable {
      double velocidad_ = 0;
  public:
      void zarpar() override { std::cout << "Barco zarpando...\n"; velocidad_ = 20; }
      void atracar() override { std::cout << "Barco atracando...\n"; velocidad_ = 0; }
      double velocidad() const override { return velocidad_; }
  };

  class Anfibio : public IConducible, public INavegable {
      int velocidad_ = 0;
      bool enAgua_ = false;
  public:
      void acelerar() override { velocidad_ += 5; }
      void frenar() override { velocidad_ = std::max(0, velocidad_ - 5); }
      void girar(int g) override { std::cout << "Anfibio gira " << g << " grados\n"; }
      void zarpar() override { enAgua_ = true; std::cout << "Anfibio entra al agua\n"; }
      void atracar() override { enAgua_ = false; std::cout << "Anfibio sale del agua\n"; }
      double velocidad() const override { return velocidad_; }
  };

  void probarConduccion(IConducible& v) {
      std::cout << "Prueba de conducción:\n";
      v.acelerar(); v.acelerar(); v.girar(45); v.frenar();
  }

  void probarNavegacion(INavegable& v) {
      std::cout << "Prueba de navegación:\n";
      v.zarpar(); std::cout << "Velocidad: " << v.velocidad() << " nudos\n"; v.atracar();
  }

  int main() {
      Auto a; Barco b; Anfibio anf;

      probarConduccion(a);
      std::cout << "\n";
      probarNavegacion(b);
      std::cout << "\n";
      probarConduccion(anf);
      probarNavegacion(anf);

      // Descubrimiento de capacidades
      IConducible* ptr = &anf;
      if (auto* nav = dynamic_cast<INavegable*>(ptr)) {
          std::cout << "\n¡El anfibio también navega!\n";
          nav->zarpar();
      }
      return 0;
  }
  ```

  ```cpp
  //====================================================================
  //  EJEMPLO 3: PATRÓN STRATEGY CON INTERFACES
  //====================================================================
  #include <iostream>
  #include <string>
  #include <memory>

  class ICompresion {
  public:
      virtual ~ICompresion() = default;
      virtual std::string comprimir(const std::string& datos) = 0;
      virtual std::string descomprimir(const std::string& datos) = 0;
      virtual std::string nombre() const = 0;
  };

  class CompresionZIP : public ICompresion {
  public:
      std::string comprimir(const std::string& d) override {
          return "[ZIP:" + d + "]";
      }
      std::string descomprimir(const std::string& d) override {
          if (d.size() > 5) return d.substr(5, d.size() - 6);
          return d;
      }
      std::string nombre() const override { return "ZIP"; }
  };

  class CompresionGZIP : public ICompresion {
  public:
      std::string comprimir(const std::string& d) override {
          return "{GZIP:" + d + "}";
      }
      std::string descomprimir(const std::string& d) override {
          if (d.size() > 6) return d.substr(6, d.size() - 7);
          return d;
      }
      std::string nombre() const override { return "GZIP"; }
  };

  class CompresionBrotli : public ICompresion {
  public:
      std::string comprimir(const std::string& d) override {
          return "<BROTLI:" + d + ">";
      }
      std::string descomprimir(const std::string& d) override {
          if (d.size() > 7) return d.substr(7, d.size() - 8);
          return d;
      }
      std::string nombre() const override { return "Brotli"; }
  };

  class GestorArchivos {
      std::unique_ptr<ICompresion> compresor_;
  public:
      GestorArchivos(std::unique_ptr<ICompresion> c) : compresor_(std::move(c)) {}

      void cambiarCompresion(std::unique_ptr<ICompresion> nuevo) {
          compresor_ = std::move(nuevo);
      }

      void guardar(const std::string& nombre, const std::string& datos) {
          auto comp = compresor_->comprimir(datos);
          std::cout << "[" << compresor_->nombre() << "] "
                    << nombre << ": " << comp << std::endl;
      }

      std::string cargar(const std::string& nombre, const std::string& comp) {
          auto orig = compresor_->descomprimir(comp);
          std::cout << "[" << compresor_->nombre() << "] "
                    << nombre << " descomprimido: " << orig << std::endl;
          return orig;
      }
  };

  int main() {
      GestorArchivos gestor(std::make_unique<CompresionZIP>());
      gestor.guardar("datos.txt", "Hola Mundo");
      gestor.cargar("datos.txt", "[ZIP:Hola Mundo]");

      gestor.cambiarCompresion(std::make_unique<CompresionGZIP>());
      gestor.guardar("datos.txt", "Hola Mundo con GZIP");

      gestor.cambiarCompresion(std::make_unique<CompresionBrotli>());
      gestor.guardar("datos.txt", "Mejor compresión");
      return 0;
  }
  ```

  ```cpp
  //====================================================================
  //  EJEMPLO 4: ISP — INTERFAZ SEGREGADA
  //====================================================================
  #include <iostream>
  #include <string>

  // MAL: Interfaz grande (viola ISP)
  class ITrabajador {
  public:
      virtual ~ITrabajador() = default;
      virtual void trabajar() = 0;
      virtual void comer() = 0;
      virtual void dormir() = 0;
      virtual void cobrar() = 0;
  };

  // BIEN: Interfaces segregadas
  interface ITrabajo {
  public:
      virtual ~ITrabajo() = default;
      virtual void trabajar() = 0;
  };

  class IAlimentacion {
  public:
      virtual ~IAlimentacion() = default;
      virtual void comer() = 0;
  };

  class IDescanso {
  public:
      virtual ~IDescanso() = default;
      virtual void dormir() = 0;
  };

  class IRemuneracion {
  public:
      virtual ~IRemuneracion() = default;
      virtual void cobrar() = 0;
  };

  // Robot solo necesita ITrabajo
  class Robot : public ITrabajo {
  public:
      void trabajar() override { std::cout << "Robot trabajando 24/7\n"; }
  };

  // Humano necesita todas
  class Humano : public ITrabajo, public IAlimentacion,
                 public IDescanso, public IRemuneracion {
  public:
      void trabajar() override { std::cout << "Humano trabajando 8h\n"; }
      void comer() override { std::cout << "Humano comiendo\n"; }
      void dormir() override { std::cout << "Humano durmiendo 8h\n"; }
      void cobrar() override { std::cout << "Humano cobrando salario\n"; }
  };

  // Trabajador temporal solo necesita ITrabajo e IRemuneracion
  class Freelancer : public ITrabajo, public IRemuneracion {
  public:
      void trabajar() override { std::cout << "Freelancer trabajando\n"; }
      void cobrar() override { std::cout << "Freelancer cobrando por proyecto\n"; }
  };

  int main() {
      std::cout << "=== ISP EN ACCIÓN ===\n";
      Robot r; r.trabajar();
      Humano h; h.trabajar(); h.comer(); h.dormir(); h.cobrar();
      Freelancer f; f.trabajar(); f.cobrar();
      std::cout << "Cada clase solo implementa lo que necesita.\n";
      return 0;
  }
  ```

  ```cpp
  #include <iostream>
  #include <string>
  #include <vector>
  #include <memory>

  class IReproducible {
  public:
      virtual ~IReproducible() = default;
      virtual void reproducir() = 0;
      virtual void pausar() = 0;
      virtual void detener() = 0;
      virtual int duracion() const = 0;
  };

  class IGrabable {
  public:
      virtual ~IGrabable() = default;
      virtual void grabar() = 0;
      virtual void detenerGrabacion() = 0;
  };

  class IEditable {
  public:
      virtual ~IEditable() = default;
      virtual void cortar(int inicio, int fin) = 0;
      virtual void copiar() = 0;
      virtual void pegar() = 0;
  };

  class Video : public IReproducible, public IEditable {
      int duracion_;
  public:
      Video(int d) : duracion_(d) {}

      void reproducir() override { std::cout << "Video reproduciéndose\n"; }
      void pausar() override { std::cout << "Video pausado\n"; }
      void detener() override { std::cout << "Video detenido\n"; }
      int duracion() const override { return duracion_; }

      void cortar(int i, int f) override { std::cout << "Video cortado de " << i << " a " << f << "\n"; }
      void copiar() override { std::cout << "Video copiado\n"; }
      void pegar() override { std::cout << "Video pegado\n"; }
  };

  class Audio : public IReproducible, public IGrabable {
      int duracion_;
  public:
      Audio(int d) : duracion_(d) {}

      void reproducir() override { std::cout << "Audio reproduciéndose\n"; }
      void pausar() override { std::cout << "Audio pausado\n"; }
      void detener() override { std::cout << "Audio detenido\n"; }
      int duracion() const override { return duracion_; }

      void grabar() override { std::cout << "Audio grabando...\n"; }
      void detenerGrabacion() override { std::cout << "Grabación detenida\n"; }
  };

  void procesarMedia(IReproducible& media) {
      std::cout << "Procesando medio (" << media.duracion() << "s):\n";
      media.reproducir();
      media.pausar();
      media.detener();

      if (auto* edit = dynamic_cast<IEditable*>(&media)) {
          std::cout << "  ¡También es editable!\n";
          edit->cortar(0, 30);
      }

      if (auto* grab = dynamic_cast<IGrabable*>(&media)) {
          std::cout << "  ¡También es grabable!\n";
          grab->grabar();
      }
  }

  int main() {
      Video v(120);
      Audio a(300);

      procesarMedia(v);
      std::cout << "---\n";
      procesarMedia(a);
      return 0;
  }
  ```

  ```cpp
  //====================================================================
  //  EJEMPLO 6: REPOSITORIO CON INTERFACES
  //====================================================================
  #include <iostream>
  #include <string>
  #include <vector>
  #include <memory>
  #include <map>

  template<typename T>
  class IRepositorio {
  public:
      virtual ~IRepositorio() = default;
      virtual void agregar(const T& item) = 0;
      virtual T obtener(const std::string& id) const = 0;
      virtual void eliminar(const std::string& id) = 0;
      virtual std::vector<T> listarTodos() const = 0;
      virtual int contar() const = 0;
  };

  class IBusqueda {
  public:
      virtual ~IBusqueda() = default;
      virtual std::vector<std::string> buscar(const std::string& termino) const = 0;
  };

  class ICacheable {
  public:
      virtual ~ICacheable() = default;
      virtual void limpiarCache() = 0;
      virtual bool estaEnCache(const std::string& clave) const = 0;
  };

  class Producto {
  public:
      std::string id, nombre;
      double precio;
  };

  class RepositorioProductos : public IRepositorio<Producto>,
                                public IBusqueda,
                                public ICacheable {
      std::map<std::string, Producto> datos_;
      std::map<std::string, Producto> cache_;
  public:
      void agregar(const Producto& p) override {
          datos_[p.id] = p;
          std::cout << "Producto " << p.id << " agregado.\n";
      }

      Producto obtener(const std::string& id) const override {
          return datos_.at(id);
      }

      void eliminar(const std::string& id) override {
          datos_.erase(id);
          std::cout << "Producto " << id << " eliminado.\n";
      }

      std::vector<Producto> listarTodos() const override {
          std::vector<Producto> v;
          for (const auto& [k, p] : datos_) v.push_back(p);
          return v;
      }

      int contar() const override { return datos_.size(); }

      std::vector<std::string> buscar(const std::string& termino) const override {
          std::vector<std::string> r;
          for (const auto& [k, p] : datos_) {
              if (p.nombre.find(termino) != std::string::npos)
                  r.push_back(k);
          }
          return r;
      }

      void limpiarCache() override { cache_.clear(); std::cout << "Cache limpiada.\n"; }
      bool estaEnCache(const std::string& clave) const override {
          return cache_.find(clave) != cache_.end();
      }
  };

  int main() {
      RepositorioProductos repo;

      repo.agregar({"A001", "Laptop", 1200});
      repo.agregar({"A002", "Mouse", 25});
      repo.agregar({"A003", "Teclado", 80});

      std::cout << "Total: " << repo.contar() << " productos\n";

      auto resultados = repo.buscar("Laptop");
      for (const auto& id : resultados)
          std::cout << "  Encontrado: " << id << "\n";

      repo.limpiarCache();
      std::cout << "En cache? " << repo.estaEnCache("A001") << "\n";

      return 0;
  }
  ```

================================================================================
  RESUMEN
================================================================================

  • Una interfaz en C++ es una clase con solo virtuales puras + destructor
    virtual, sin datos miembro.
  • Se usa el prefijo 'I' por convención (ej: ISerializable).
  • Diferencia con clase abstracta: interfaz no tiene estado ni
    implementación.
  • Herencia múltiple de interfaces permite diseño basado en capacidades.
  • ISP: interfaces pequeñas y cohesivas para no forzar dependencias.
  • dynamic_cast permite descubrir capacidades en tiempo de ejecución.
  • Patrón Strategy: intercambiar algoritmos mediante interfaces.
  • Capability-based design > Type-based design.

================================================================================
  PREGUNTAS FRECUENTES (FAQ)
================================================================================

  P: ¿Por qué C++ no tiene keyword 'interface'?
  R: Porque las interfaces se implementan naturalmente con clases que
    tienen solo virtuales puras. No se necesita sintaxis especial.

  P: ¿Puede una interfaz tener datos static?
  R: Sí, static const es aceptable (ej: constantes de versión).

  P: ¿Herencia múltiple de interfaces es segura?
  R: Sí. Como no hay estado ni implementación, no hay problema del
    diamante. Es la única forma recomendada de herencia múltiple.

  P: ¿Cómo verifico si un objeto implementa una interfaz?
  R: Con dynamic_cast. Si devuelve nullptr, no implementa la interfaz.
    Requiere RTTI habilitado y funciones virtuales.

  P: ¿Puedo tener una interfaz con métodos no públicos?
  R: En general no. Las interfaces definen el contrato público. Si
    necesitas métodos protegidos, probablemente necesitas clase abstracta.

  P: ¿Interfaz con método virtual con implementación?
  R: Entonces no es una interfaz pura, es una clase abstracta con
    implementación por defecto (válido en algunos casos).
