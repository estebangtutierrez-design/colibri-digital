  ------------------------------------------------------------------
  1. CONVERSIONES IMPLÍCITAS
  ------------------------------------------------------------------

  Las conversiones implícitas ocurren automáticamente cuando el
  compilador puede convertir un tipo a otro sin pérdida de información.

  Ejemplos comunes:
  • Promoción de tipos: short → int, float → double
  • Conversiones aritméticas: int → double en expresiones mixtas
  • Puntero a bool: cualquier puntero no-nulo → true
  • array a puntero: nombre del array → puntero al primer elemento
  • Derivada a base: puntero derivado → puntero base (upcasting)

  Las conversiones implícitas pueden ser peligrosas cuando hay
  pérdida de información (double → int, long → short).

  ------------------------------------------------------------------
  2. CONVERSIONES EXPLÍCITAS (CASTS)
  ------------------------------------------------------------------

  C++ proporciona cuatro tipos de cast explícito:

  • static_cast: Conversiones entre tipos relacionados (numéricos,
    punteros en misma jerarquía, void* → T*).
    Sintaxis: static_cast<Tipo>(expresión)

  • dynamic_cast: Downcasting seguro en jerarquías polimórficas.
    Requiere RTTI. Retorna nullptr (punteros) o lanza bad_cast
    (referencias) si falla.
    Sintaxis: dynamic_cast<Tipo*>(expresión)

  • const_cast: Añade o elimina cv-qualifiers (const, volatile).
    Sintaxis: const_cast<Tipo>(expresión)

  • reinterpret_cast: Conversión entre tipos no relacionados.
    Peligroso y no portable. Solo para casos muy específicos.
    Sintaxis: reinterpret_cast<Tipo>(expresión)

  ------------------------------------------------------------------
  3. OPERADORES DE CONVERSIÓN (operator tipo())
  ------------------------------------------------------------------

  Definen cómo convertir tu clase a otro tipo:

    class MiClase {
    public:
        operator TipoDestino() const {
            // Convertir *this a TipoDestino
            return valor;
        }
    };

  Ejemplos:
    operator bool() const;     // Conversión a bool
    operator int() const;      // Conversión a entero
    operator std::string() const;  // Conversión a string

  Desde C++11, se recomienda marcar operator bool() como explicit
  para evitar conversiones implícitas peligrosas.

  ------------------------------------------------------------------
  4. PALABRA CLAVE 'explicit'
  ------------------------------------------------------------------

  'explicit' impide que el compilador use el constructor o el operador
  de conversión en conversiones implícitas.

    class Racional {
    public:
        explicit Racional(int n);  // No puede: Racional r = 5;
        explicit operator double() const;  // No puede: double d = r;
    };

  Antes de C++11, explicit solo se aplicaba a constructores con un
  parámetro. Desde C++11, también se aplica a operadores de conversión.

  ------------------------------------------------------------------
  5. CONVERSIÓN DE CADENA A TIPO Y VICEVERSA
  ------------------------------------------------------------------

  Para convertir entre strings y otros tipos:
  • std::to_string(valor) → string
  • std::stoi, stol, stoll, stoul → enteros
  • std::stof, stod, stold → floats
  • std::stringstream para conversiones más complejas

  ------------------------------------------------------------------
  6. PELIGROS DE LAS CONVERSIONES
  ------------------------------------------------------------------

  • Pérdida de datos: double → int, long → short
  • Conversiones de punteros no relacionadas (reinterpret_cast)
  • const_cast en objetos originalmente const (undefined behavior)
  • dynamic_cast sin RTTI o en clases no polimórficas
  • Conversiones implícitas no esperadas (el problema del "most vexing
    parse" y otros)

  ```cpp
  //====================================================================
  //  EJEMPLO 1: OPERADORES DE CONVERSIÓN PERSONALIZADOS
  //====================================================================
  #include <iostream>
  #include <string>
  #include <sstream>
  #include <cmath>

  class NumeroComplejo {
      double real_, imag_;
  public:
      NumeroComplejo(double r = 0, double i = 0) : real_(r), imag_(i) {}

      // Conversión a double (magnitud)
      explicit operator double() const {
          return std::sqrt(real_ * real_ + imag_ * imag_);
      }

      // Conversión a bool (¿es diferente de cero?)
      explicit operator bool() const {
          return real_ != 0 || imag_ != 0;
      }

      // Conversión a string (formato "a + bi")
      operator std::string() const {
          std::ostringstream oss;
          oss << real_;
          if (imag_ >= 0) oss << " + " << imag_ << "i";
          else oss << " - " << -imag_ << "i";
          return oss.str();
      }

      double real() const { return real_; }
      double imag() const { return imag_; }
  };

  int main() {
      NumeroComplejo c1(3, 4);
      NumeroComplejo c2(0, 0);

      // Conversión explícita a double (magnitud)
      double magnitud = static_cast<double>(c1);
      std::cout << "Magnitud de (3+4i): " << magnitud << std::endl;

      // Conversión explícita a bool
      if (static_cast<bool>(c1)) {
          std::cout << "c1 es diferente de cero" << std::endl;
      }
      if (!static_cast<bool>(c2)) {
          std::cout << "c2 es cero" << std::endl;
      }

      // Conversión implícita a string (NO explicit)
      std::string str = c1;
      std::cout << "Como string: " << str << std::endl;

      return 0;
  }
  ```

  ```cpp
  //  EJEMPLO 2: EXPLICIT EN CONSTRUCTORES
  //====================================================================
  #include <iostream>
  #include <string>
  #include <vector>

  class Cadena {
      std::string datos_;
  public:
      // Constructor explícito: evita conversión implícita
      explicit Cadena(size_t n) : datos_(n, '?') {
          std::cout << "Cadena de " << n << " caracteres creada\n";
      }

      // Constructor no explícito
      Cadena(const char* s) : datos_(s) {
          std::cout << "Cadena desde literal: \"" << s << "\"\n";
      }

      const std::string& str() const { return datos_; }
  };

  void imprimirCadena(const Cadena& c) {
      std::cout << "  Imprimiendo: \"" << c.str() << "\"\n";
  }

  class Punto {
      int x_, y_;
  public:
      // explicit permite constructor con varios parámetros
      explicit Punto(int x, int y) : x_(x), y_(y) {}

      // explicit en conversión
      explicit operator std::string() const {
          return "(" + std::to_string(x_) + ", " + std::to_string(y_) + ")";
      }
  };

  int main() {
      std::cout << "=== EXPLICIT EN CONSTRUCTORES ===\n";

      // Cadena c1 = 5;      // ERROR: constructor explícito
      Cadena c2(5);           // OK: llamada directa
      Cadena c3 = "Hola";    // OK: constructor no explícito

      imprimirCadena("Mundo");  // OK: const char* → Cadena implícita
      // imprimirCadena(5);    // ERROR: no se puede convertir int a Cadena

      std::cout << "\n=== EXPLICIT EN OPERADORES ===\n";
      Punto p(3, 4);
      // std::string s = p;    // ERROR: operator string() es explicit
      std::string s = static_cast<std::string>(p);
      std::cout << "Punto como string: " << s << std::endl;

      return 0;
  }
  ```

  ```cpp
  //====================================================================
  //  EJEMPLO 3: STATIC_CAST Y SUS USOS
  //====================================================================
  #include <iostream>
  #include <string>
  #include <vector>

  void ejemploStaticCast() {
      std::cout << "=== STATIC_CAST ===\n";

      // 1. Conversiones numéricas seguras
      double pi = 3.14159;
      int entero = static_cast<int>(pi);  // Truncamiento
      std::cout << "double " << pi << " → int " << entero << std::endl;

      // 2. Promoción de tipos
      float f = 2.5f;
      double d = static_cast<double>(f);
      std::cout << "float " << f << " → double " << d << std::endl;

      // 3. Enum a entero
      enum Color { ROJO, VERDE, AZUL };
      Color c = AZUL;
      int colorInt = static_cast<int>(c);
      std::cout << "Color AZUL → int " << colorInt << std::endl;

      // 4. Entero a enum (cuidado)
      Color c2 = static_cast<Color>(1);
      std::cout << "int 1 → Color " << c2 << std::endl;

      // 5. void* a T* (cuando sabemos lo que hacemos)
      int valor = 42;
      void* ptrV = &valor;
      int* ptrInt = static_cast<int*>(ptrV);
      std::cout << "void* → int*: " << *ptrInt << std::endl;

      // 6. Upcasting (derivada a base)
      struct Base { virtual ~Base() = default; void quien() const { std::cout << "  Base\n"; } };
      struct Derivada : Base { void quien() const { std::cout << "  Derivada\n"; } };
      Derivada derv;
      Base& refBase = static_cast<Base&>(derv);  // Upcasting seguro
      refBase.quien();  // Base::quien() por el tipo estático
  }

  struct A {
      virtual ~A() = default;
      virtual void hablar() const { std::cout << "A::hablar\n"; }
  };

  struct B : A {
      void hablar() const override { std::cout << "B::hablar\n"; }
      void soloB() const { std::cout << "  Solo B\n"; }
  };

  int main() {
      ejemploStaticCast();

      // static_cast para downcasting (peligroso sin verificación)
      std::cout << "\n=== STATIC_CAST DOWNCASTING (peligroso) ===\n";
      A* a = new B;
      B* b = static_cast<B*>(a);  // OK porque sabemos que realmente es B
      b->soloB();

      A* a2 = new A;
      // B* b2 = static_cast<B*>(a2);  // UB: a2 no es B, pero compila
      // b2->soloB();  // Undefined behavior!

      delete a;
      delete a2;

      return 0;
  }
  ```

  ```cpp
  //====================================================================
  //  EJEMPLO 4: DYNAMIC_CAST (RTTI)
  //====================================================================
  #include <iostream>
  #include <string>
  #include <typeinfo>
  #include <vector>
  #include <memory>

  class Animal {
  public:
      virtual ~Animal() = default;
      virtual void sonido() const = 0;
  };

  class Perro : public Animal {
  public:
      void sonido() const override { std::cout << "!Guau!\n"; }
      void buscarHueso() const { std::cout << "  Buscando hueso...\n"; }
  };

  class Gato : public Animal {
  public:
      void sonido() const override { std::cout << "!Miau!\n"; }
      void ronronear() const { std::cout << "  Rrrrr...\n"; }
  };

  class Pajaro : public Animal {
  public:
      void sonido() const override { std::cout << "!Pío!\n"; }
      void volar() const { std::cout << "  Volando...\n"; }
  };

  int main() {
      std::cout << "=== DYNAMIC_CAST ===\n";

      std::vector<Animal*> animales;
      animales.push_back(new Perro());
      animales.push_back(new Gato());
      animales.push_back(new Pajaro());
      animales.push_back(new Perro());

      for (const auto* animal : animales) {
          animal->sonido();

          // Intentar downcasting seguro con dynamic_cast
          if (auto* perro = dynamic_cast<const Perro*>(animal)) {
              perro->buscarHueso();
          } else if (auto* gato = dynamic_cast<const Gato*>(animal)) {
              gato->ronronear();
          } else if (auto* pajaro = dynamic_cast<const Pajaro*>(animal)) {
              pajaro->volar();
          }

          // typeid para obtener información del tipo
          std::cout << "  (type: " << typeid(*animal).name() << ")\n";
      }

      // Limpieza
      for (auto* a : animales) delete a;

      // dynamic_cast con referencias (lanza std::bad_cast)
      std::cout << "\n=== Con referencias ===\n";
      try {
          Perro perro;
          Animal& ref = perro;
          auto& gatoRef = dynamic_cast<Gato&>(ref);  // Lanza bad_cast
          gatoRef.ronronear();
      } catch (const std::bad_cast& e) {
          std::cout << "  bad_cast: " << e.what() << std::endl;
      }

      return 0;
  }
  ```

  ```cpp
  //  EJEMPLO 5: CONST_CAST Y REINTERPRET_CAST
  //====================================================================
  #include <iostream>
  #include <string>

  // --- CONST_CAST ---
  void imprimirValor(const int* ptr) {
      // Queremos modificar temporalmente para depuración
      // (MAL EJEMPLO - solo demostración)
      int* p = const_cast<int*>(ptr);
      std::cout << "  Valor: " << *p << std::endl;
  }

  struct Config {
      mutable int cacheValid = 0;
      int datos = 42;

      int obtenerDatos() const {
          // const_cast para modificar mutable (innecesario aquí,
          // pero demostrativo)
          if (!cacheValid) {
              const_cast<Config*>(this)->cacheValid = 1;
          }
          return datos;
      }
  };

  // --- REINTERPRET_CAST ---
  struct Cabecera {
      uint32_t tipo;
      uint32_t tamanyo;
      char datos[1];
  };

  void ejemploReinterpret() {
      std::cout << "\n=== REINTERPRET_CAST ===\n";

      // 1. Bits de un float como entero
      float f = 3.14159f;
      uint32_t bits = *reinterpret_cast<uint32_t*>(&f);
      std::cout << "float " << f << " como entero: 0x" << std::hex << bits
                << std::dec << std::endl;

      // 2. Puntero a entero (size_t)
      int x = 42;
      uintptr_t addr = reinterpret_cast<uintptr_t>(&x);
      std::cout << "Dirección de x: " << addr << std::endl;

      // 3. Struct con campo flexible (protocolo de red)
      char buffer[sizeof(Cabecera) + 64];
      auto* cab = reinterpret_cast<Cabecera*>(buffer);
      cab->tipo = 1;
      cab->tamanyo = 64;
      std::cout << "Cabecera tipo=" << cab->tipo
                << ", tamaño=" << cab->tamanyo << std::endl;
  }

  int main() {
      std::cout << "=== CONST_CAST ===\n";

      int valor = 100;
      const int* cptr = &valor;
      imprimirValor(cptr);

      // Caso más peligroso - objeto originalmente const
      const int constante = 200;
      // int* ptr = const_cast<int*>(&constante);
      // *ptr = 300;  // UB: modificar objeto originalmente const

      Config cfg;
      std::cout << "Datos: " << cfg.obtenerDatos() << std::endl;

      ejemploReinterpret();

      std::cout << "\nADVERTENCIA: const_cast y reinterpret_cast\n";
      std::cout << "deben usarse SOLO cuando no hay alternativa.\n";

      return 0;
  }
  ```

  ```cpp
  //====================================================================
  //  EJEMPLO 6: CONVERSIONES ENTRE STRINGS Y TIPOS
  //====================================================================
  #include <iostream>
  #include <string>
  #include <sstream>
  #include <iomanip>
  #include <vector>

  class Conversor {
  public:
      // String a tipos numéricos
      static int aInt(const std::string& s) {
          return std::stoi(s);
      }
      static long aLong(const std::string& s) {
          return std::stol(s);
      }
      static double aDouble(const std::string& s) {
          return std::stod(s);
      }
      static float aFloat(const std::string& s) {
          return std::stof(s);
      }
      static bool aBool(const std::string& s) {
          return s == "true" || s == "1" || s == "yes";
      }

      // Tipos numéricos a string
      static std::string desdeInt(int v) { return std::to_string(v); }
      static std::string desdeDouble(double v, int precision = 2) {
          std::ostringstream oss;
          oss << std::fixed << std::setprecision(precision) << v;
          return oss.str();
      }
      static std::string desdeBool(bool v) { return v ? "true" : "false"; }

      // Conversión genérica con stringstream
      template<typename T>
      static std::string toString(const T& valor) {
          std::ostringstream oss;
          oss << valor;
          return oss.str();
      }

      template<typename T>
      static T fromString(const std::string& s) {
          T valor;
          std::istringstream iss(s);
          iss >> valor;
          return valor;
      }
  };

  struct Punto {
      int x, y;
  };

  std::ostream& operator<<(std::ostream& os, const Punto& p) {
      os << "(" << p.x << "," << p.y << ")";
      return os;
  }

  std::istream& operator>>(std::istream& is, Punto& p) {
      char ign;
      is >> ign >> p.x >> ign >> p.y >> ign;
      return is;
  }

  int main() {
      std::cout << "=== CONVERSIONES STRING <-> TIPO ===\n";

      // String a números
      std::string sNum = "42";
      std::string sPi = "3.14159";
      std::string sFlag = "true";

      int i = Conversor::aInt(sNum);
      double d = Conversor::aDouble(sPi);
      bool b = Conversor::aBool(sFlag);

      std::cout << "Int: " << i << " (dobled: " << (i * 2) << ")\n";
      std::cout << "Double: " << d << " (pi * 2 = " << (d * 2) << ")\n";
      std::cout << "Bool: " << b << std::endl;

      // Números a string
      std::string s1 = Conversor::desdeInt(255);
      std::string s2 = Conversor::desdeDouble(3.14159265, 4);
      std::string s3 = Conversor::desdeBool(true);

      std::cout << "\nDesde int: " << s1 << std::endl;
      std::cout << "Desde double (4 decimales): " << s2 << std::endl;
      std::cout << "Desde bool: " << s3 << std::endl;

      // Conversión genérica
      Punto p{10, 20};
      std::string sp = Conversor::toString(p);
      std::cout << "\nPunto a string: " << sp << std::endl;

      Punto p2 = Conversor::fromString<Punto>("(30,40)");
      std::cout << "String a Punto: (" << p2.x << "," << p2.y << ")\n";

      // Manejo de errores
      std::cout << "\n--- Manejo de errores ---\n";
      try {
          int malo = std::stoi("no es un número");
      } catch (const std::invalid_argument& e) {
          std::cout << "  Error (inválido): " << e.what() << std::endl;
      } catch (const std::out_of_range& e) {
          std::cout << "  Error (rango): " << e.what() << std::endl;
      }

      return 0;
  }
  ```

================================================================================
  RESUMEN
================================================================================

  • Las conversiones implícitas ocurren automáticamente; pueden ser
    peligrosas si hay pérdida de datos.
  • static_cast: conversiones entre tipos relacionados, upcasting seguro.
  • dynamic_cast: downcasting polimórfico con verificación en runtime.
  • const_cast: añadir/eliminar const; peligroso con objetos const reales.
  • reinterpret_cast: conversión de bits entre tipos no relacionados;
    muy peligroso y no portable.
  • 'explicit' previene conversiones implícitas no deseadas.
  • operator tipo() define conversiones personalizadas de clases.
  • Preferir siempre los casts de C++ sobre los C-style casts.

================================================================================
  PREGUNTAS FRECUENTES (FAQ)
================================================================================

  P: ¿Cuándo usar static_cast vs dynamic_cast?
  R: static_cast para upcasting y conversiones numéricas. dynamic_cast
    para downcasting cuando necesitas verificación en runtime (RTTI).

  P: ¿Es seguro reinterpret_cast?
  R: No. Solo debe usarse cuando se interactúa con hardware, protocolos
    de red, o buffers binarios. No es portable.

  P: ¿Debo marcar todos los constructores como explicit?
  R: Solo los de 1 parámetro que no deberían usarse como conversión
    implícita. Los constructores de copia/movimiento son excepción.

  P: ¿Qué operador de conversión debe ser explicit?
  R: operator bool() siempre debe ser explicit para evitar conversiones
    peligrosas (ej: if (obj) está bien, pero int x = obj no).

  P: ¿Cuándo usar const_cast?
  R: Cuando necesitas llamar a una API antigua que no es const-correct.
    Nunca para modificar un objeto que originalmente fue declarado const.
