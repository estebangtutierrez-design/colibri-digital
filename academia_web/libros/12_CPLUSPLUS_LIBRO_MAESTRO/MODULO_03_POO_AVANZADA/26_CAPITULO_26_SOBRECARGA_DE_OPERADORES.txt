  ------------------------------------------------------------------
  1. ¿QUÉ ES LA SOBRECARGA DE OPERADORES?
  ------------------------------------------------------------------

  La sobrecarga de operadores permite redefinir el significado de un
  operador cuando se aplica a objetos de una clase específica.

  Sintaxis general:
    Tipo operator@símbolo(lista de parámetros);

  Ejemplo:
    Vector operator+(const Vector& a, const Vector& b);

  C++ permite sobrecargar la mayoría de los operadores excepto:
  • :: (resolución de ámbito)
  • . (acceso a miembro)
  • .* (acceso a puntero a miembro)
  • ?: (ternario)
  • sizeof, typeid, alignof

  ------------------------------------------------------------------
  2. MIEMBRO VS NO-MIEMBRO
  ------------------------------------------------------------------

  Operadores que DEBEN ser miembros:
  • operator= (asignación)
  • operator[] (acceso)
  • operator() (llamada)
  • operator-> (acceso a miembro por puntero)
  • operator T (conversión)
  • operator++/-- (prefijo y postfijo - pueden ser ambos formas)

  Operadores que DEBEN ser no-miembros:
  • operator<< (inserción en ostream)
  • operator>> (extracción de istream)

  Operadores que PUEDEN ser ambos:
  • Binarios: +, -, *, /, %, ==, !=, <, >, etc.
  • Unarios: +, -, *, &, !, ~

  Regla general: si el operador modifica el objeto, debe ser miembro.
  Si no lo modifica, puede ser no-miembro (friend si necesita acceso).

  ------------------------------------------------------------------
  3. FORMA CANÓNICA
  ------------------------------------------------------------------

  Operador += (compuesto):
    MyClass& operator+=(const MyClass& otro) {
        // modificar this
        return *this;
    }

  Operador + (binario) a partir de +=:
    MyClass operator+(MyClass a, const MyClass& b) {
        a += b;
        return a;
    }
    // O como miembro:
    MyClass operator+(const MyClass& otro) const {
        MyClass resultado = *this;
        resultado += otro;
        return resultado;
    }

  Esta forma evita duplicación y es la práctica recomendada.

  ------------------------------------------------------------------
  4. OPERADORES DE COMPARACIÓN
  ------------------------------------------------------------------

  Desde C++20, con el operador <=> (spaceship), se pueden generar
  automáticamente todos los operadores de comparación:

    auto operator<=>(const MyClass&) const = default;

  Antes de C++20, se implementan manualmente:
    bool operator==(const MyClass&) const;
    bool operator!=(const MyClass&) const;
    bool operator<(const MyClass&) const;
    // etc.

  ------------------------------------------------------------------
  5. OPERADOR DE ASIGNACIÓN (operator=)
  ------------------------------------------------------------------

  Debe manejar:
  • Autoaignación (a = a)
  • Liberación de recursos antiguos
  • Copia de recursos nuevos
  • Retornar *this por referencia

  Forma canónica:
    MyClass& operator=(const MyClass& otro) {
        if (this != &otro) {
            // liberar recursos de this
            // copiar recursos de otro
        }
        return *this;
    }

  Si la clase gestiona recursos, se recomienda la Copy-and-Swap idiom.

  ------------------------------------------------------------------
  6. OPERADOR DE ACCESO (operator[])
  ------------------------------------------------------------------

  Generalmente se implementan dos versiones:
  - const: para acceso de solo lectura.
  - no-const: para lectura/escritura.

    T& operator[](size_t i) { return datos_[i]; }
    const T& operator[](size_t i) const { return datos_[i]; }

  ------------------------------------------------------------------
  7. OPERADOR DE LLAMADA (operator())
  ------------------------------------------------------------------

  Permite que un objeto se comporte como una función (functor):

    class Multiplicador {
        int factor_;
    public:
        Multiplicador(int f) : factor_(f) {}
        int operator()(int x) const { return x * factor_; }
    };

    Multiplicador por2(2);
    std::cout << por2(5);  // 10

  ```cpp
  //====================================================================
  //  EJEMPLO 1: SOBRECARGA COMPLETA DE CLASE NÚMERO RACIONAL
  //====================================================================
  #include <iostream>
  #include <numeric>  // for std::gcd

  class Racional {
  private:
      int num_, den_;

      void simplificar() {
          int g = std::gcd(num_, den_);
          num_ /= g;
          den_ /= g;
          if (den_ < 0) { num_ = -num_; den_ = -den_; }
      }

  public:
      Racional(int n = 0, int d = 1) : num_(n), den_(d) {
          if (den_ == 0) throw std::runtime_error("Denominador 0");
          simplificar();
      }

      // Operadores de asignación compuesta (modifican this)
      Racional& operator+=(const Racional& r) {
          num_ = num_ * r.den_ + r.num_ * den_;
          den_ *= r.den_;
          simplificar();
          return *this;
      }

      Racional& operator-=(const Racional& r) {
          num_ = num_ * r.den_ - r.num_ * den_;
          den_ *= r.den_;
          simplificar();
          return *this;
      }

      Racional& operator*=(const Racional& r) {
          num_ *= r.num_;
          den_ *= r.den_;
          simplificar();
          return *this;
      }

      Racional& operator/=(const Racional& r) {
          if (r.num_ == 0) throw std::runtime_error("División por 0");
          return *this *= Racional(r.den_, r.num_);
      }

      // Operador de incremento/decremento
      Racional& operator++() { return *this += Racional(1); }
      Racional operator++(int) { Racional tmp(*this); ++(*this); return tmp; }
      Racional& operator--() { return *this -= Racional(1); }
      Racional operator--(int) { Racional tmp(*this); --(*this); return tmp; }

      // Operadores unarios
      Racional operator+() const { return *this; }
      Racional operator-() const { return Racional(-num_, den_); }

      // Operador de acceso a partes
      int numerador() const { return num_; }
      int denominador() const { return den_; }
  };

  // Operadores binarios como no-miembros
  Racional operator+(Racional a, const Racional& b) { return a += b; }
  Racional operator-(Racional a, const Racional& b) { return a -= b; }
  Racional operator*(Racional a, const Racional& b) { return a *= b; }
  Racional operator/(Racional a, const Racional& b) { return a /= b; }

  bool operator==(const Racional& a, const Racional& b) {
      return a.numerador() == b.numerador() &&
             a.denominador() == b.denominador();
  }

  bool operator!=(const Racional& a, const Racional& b) { return !(a == b); }
  bool operator<(const Racional& a, const Racional& b) {
      return a.numerador() * b.denominador() < b.numerador() * a.denominador();
  }
  bool operator>(const Racional& a, const Racional& b) { return b < a; }
  bool operator<=(const Racional& a, const Racional& b) { return !(a > b); }
  bool operator>=(const Racional& a, const Racional& b) { return !(a < b); }

  std::ostream& operator<<(std::ostream& os, const Racional& r) {
      if (r.denominador() == 1) os << r.numerador();
      else os << r.numerador() << "/" << r.denominador();
      return os;
  }

  int main() {
      Racional r1(1, 2), r2(3, 4);

      std::cout << "r1 = " << r1 << std::endl;
      std::cout << "r2 = " << r2 << std::endl;
      std::cout << "r1 + r2 = " << (r1 + r2) << std::endl;
      std::cout << "r1 - r2 = " << (r1 - r2) << std::endl;
      std::cout << "r1 * r2 = " << (r1 * r2) << std::endl;
      std::cout << "r1 / r2 = " << (r1 / r2) << std::endl;
      std::cout << "r1 == r2? " << (r1 == r2) << std::endl;
      std::cout << "r1 < r2? " << (r1 < r2) << std::endl;
      std::cout << "++r1 = " << ++r1 << std::endl;
      std::cout << "-r2 = " << (-r2) << std::endl;

      return 0;
  }
  ```

  //  EJEMPLO 2: OPERATOR[] Y OPERATOR()
  //====================================================================
  #include <iostream>
  #include <string>
  #include <stdexcept>
  #include <map>

  template<typename T, size_t N>
  class ArrayFijo {
      T datos_[N];
  public:
      T& operator[](size_t i) {
          if (i >= N) throw std::out_of_range("Índice fuera de rango");
          return datos_[i];
      }

      const T& operator[](size_t i) const {
          if (i >= N) throw std::out_of_range("Índice fuera de rango");
          return datos_[i];
      }

      size_t size() const { return N; }

      T operator()(size_t i, T valorPorDefecto = T{}) const {
          if (i < N) return datos_[i];
          return valorPorDefecto;
      }
  };

  class ContadorPalabras {
      std::map<std::string, int> conteo_;
  public:
      int& operator[](const std::string& palabra) {
          return conteo_[palabra];
      }

      int operator()(const std::string& palabra) const {
          auto it = conteo_.find(palabra);
          return it != conteo_.end() ? it->second : 0;
      }

      void mostrar() const {
          for (const auto& [p, c] : conteo_)
              std::cout << "  " << p << ": " << c << std::endl;
      }
  };

  int main() {
      ArrayFijo<int, 5> arr;
      for (size_t i = 0; i < arr.size(); ++i)
          arr[i] = i * 10;

      std::cout << "arr[2] = " << arr[2] << std::endl;
      std::cout << "arr(10, -1) = " << arr(10, -1) << std::endl;

      ContadorPalabras cp;
      cp["hola"]++;
      cp["mundo"]++;
      cp["hola"]++;
      cp["c++"]++;

      std::cout << "
Conteo:" << std::endl;
      cp.mostrar();
      std::cout << "Frecuencia de 'hola': " << cp("hola") << std::endl;
      std::cout << "Frecuencia de 'adiós': " << cp("adiós") << std::endl;

      return 0;
  }

  //====================================================================
  //  EJEMPLO 3: OPERADOR DE ASIGNACIÓN Y COPY-AND-SWAP
  //====================================================================
  #include <iostream>
  #include <string>
  #include <cstring>
  #include <algorithm>

  class Cadena {
      char* datos_;
      size_t tam_;

  public:
      Cadena(const char* s = "") : tam_(std::strlen(s)) {
          datos_ = new char[tam_ + 1];
          std::strcpy(datos_, s);
          std::cout << "  Constructor: "" << s << ""\n";
      }

      Cadena(const Cadena& otra) : tam_(otra.tam_) {
          datos_ = new char[tam_ + 1];
          std::strcpy(datos_, otra.datos_);
          std::cout << "  Copia: "" << datos_ << ""\n";
      }

      Cadena(Cadena&& otra) noexcept : datos_(otra.datos_), tam_(otra.tam_) {
          otra.datos_ = nullptr;
          otra.tam_ = 0;
          std::cout << "  Move: "" << datos_ << ""\n";
      }

      // Copy-and-Swap idiom
      void swap(Cadena& otra) noexcept {
          std::swap(datos_, otra.datos_);
          std::swap(tam_, otra.tam_);
      }

      Cadena& operator=(Cadena otra) {  // Toma por valor (copia o move)
          swap(otra);                    // Intercambia con la copia
          return *this;                  // La copia se destruye al salir
      }

      ~Cadena() {
          if (datos_) {
              std::cout << "  Destructor: "" << datos_ << ""\n";
              delete[] datos_;
          }
      }

      const char* c_str() const { return datos_; }
      size_t size() const { return tam_; }

      char& operator[](size_t i) { return datos_[i]; }
      const char& operator[](size_t i) const { return datos_[i]; }
  };

  std::ostream& operator<<(std::ostream& os, const Cadena& c) {
      os << c.c_str();
      return os;
  }

  int main() {
      std::cout << "=== COPY-AND-SWAP ===\n";
      Cadena a("Hola");
      Cadena b("Mundo");
      Cadena c("C++");

      std::cout << "\nAntes: a=" << a << ", b=" << b << ", c=" << c << "\n";

      a = b;  // Asignación por copia
      std::cout << "Después de a = b: a=" << a << "\n";

      a = std::move(c);  // Asignación por move
      std::cout << "Después de a = move(c): a=" << a << ", c=" << c << "\n";

      a = a;  // Autoaignación (segura con copy-and-swap)

      std::cout << "\nSaliendo del ámbito...\n";
      return 0;
  }

  //====================================================================
  //  EJEMPLO 4: FUNCTOR — OPERATOR()
  //====================================================================
  #include <iostream>
  #include <string>
  #include <vector>
  #include <algorithm>
  #include <cmath>

  class Elevar {
      double exponente_;
  public:
      Elevar(double exp) : exponente_(exp) {}

      double operator()(double base) const {
          return std::pow(base, exponente_);
      }
  };

  class FiltroRango {
      double min_, max_;
  public:
      FiltroRango(double min, double max) : min_(min), max_(max) {}

      bool operator()(double valor) const {
          return valor >= min_ && valor <= max_;
      }
  };

  class Acumulador {
      double total_ = 0;
      int contador_ = 0;
  public:
      void operator()(double valor) {
          total_ += valor;
          ++contador_;
      }

      double promedio() const { return contador_ > 0 ? total_ / contador_ : 0; }
      double suma() const { return total_; }
      int count() const { return contador_; }
  };

  int main() {
      std::cout << "=== FUNCTORS ===\n";

      Elevar cuadrado(2);
      Elevar cubo(3);

      std::cout << "5^2 = " << cuadrado(5) << std::endl;
      std::cout << "5^3 = " << cubo(5) << std::endl;

      std::vector<double> datos = {1.5, 2.7, 3.2, 4.8, 5.1, 6.3, 7.0, 8.9};
      FiltroRango entre3y6(3.0, 6.0);

      std::cout << "\nValores entre 3 y 6:" << std::endl;
      for (double d : datos) {
          if (entre3y6(d))
              std::cout << "  " << d << std::endl;
      }

      // Usando for_each con functor con estado
      Acumulador acum;
      acum = std::for_each(datos.begin(), datos.end(), acum);
      std::cout << "\nSuma: " << acum.suma() << std::endl;
      std::cout << "Promedio: " << acum.promedio() << std::endl;

      // Lambda (C++11) vs Functor
      auto lambdaCuadrado = [](double x) { return x * x; };
      std::cout << "\nLambda: 5^2 = " << lambdaCuadrado(5) << std::endl;

      return 0;
  }

  //====================================================================
  //  EJEMPLO 5: OPERADORES DE CONVERSIÓN
  //====================================================================
  #include <iostream>
  #include <string>
  #include <sstream>

  class Dinero {
      long long centavos_;
      std::string moneda_;
  public:
      Dinero(double cantidad, const std::string& moneda = "USD")
          : centavos_(static_cast<long long>(cantidad * 100)), moneda_(moneda) {}

      // Conversión a double (pérdida de precisión posible)
      operator double() const {
          return static_cast<double>(centavos_) / 100.0;
      }

      // Conversión a string
      operator std::string() const {
          std::ostringstream oss;
          oss << moneda_ << " " << (centavos_ / 100) << "."
              << (centavos_ % 100 < 10 ? "0" : "") << (centavos_ % 100);
          return oss.str();
      }

      // Operador de salida
      friend std::ostream& operator<<(std::ostream& os, const Dinero& d) {
          os << static_cast<std::string>(d);
          return os;
      }

      // Operadores aritméticos
      Dinero& operator+=(const Dinero& d) {
          centavos_ += d.centavos_;
          return *this;
      }

      friend Dinero operator+(Dinero a, const Dinero& b) { return a += b; }
  };

  int main() {
      Dinero precio(29.99, "USD");
      Dinero impuesto(5.01, "USD");

      // Conversión implícita a double
      double totalDouble = static_cast<double>(precio) + static_cast<double>(impuesto);
      std::cout << "Total (double): " << totalDouble << std::endl;

      // Conversión implícita a string
      std::string precioStr = precio;
      std::cout << "Precio (string): " << precioStr << std::endl;

      // Uso con operador <<
      Dinero total = precio + impuesto;
      std::cout << "Total: " << total << std::endl;

      // Conversión en expresiones
      if (static_cast<double>(total) > 30.0) {
          std::cout << "El total excede $30" << std::endl;
      }

      return 0;
  }

  //====================================================================
  //  EJEMPLO 6: OPERADOR <=> (C++20 SPACESHIP)
  //====================================================================
  #include <iostream>
  #include <string>
  #include <compare>

  class Version {
      int major_, minor_, patch_;
  public:
      Version(int maj, int min, int pat)
          : major_(maj), minor_(min), patch_(pat) {}

      // C++20: genera ==, !=, <, <=, >, >= automáticamente
      auto operator<=>(const Version&) const = default;

      std::string str() const {
          return std::to_string(major_) + "."
               + std::to_string(minor_) + "."
               + std::to_string(patch_);
      }
  };

  class Punto {
      int x_, y_;
  public:
      Punto(int x, int y) : x_(x), y_(y) {}

      // Operador <=> personalizado
      auto operator<=>(const Punto& otro) const {
          if (auto cmp = x_ <=> otro.x_; cmp != 0) return cmp;
          return y_ <=> otro.y_;
      }

      bool operator==(const Punto& otro) const {
          return x_ == otro.x_ && y_ == otro.y_;
      }
  };

  int main() {
      std::cout << "=== OPERADOR <=> (C++20) ===\n";

      Version v1(2, 5, 0), v2(3, 0, 0), v3(2, 5, 0);

      std::cout << v1.str() << " < " << v2.str() << "? "
                << (v1 < v2) << std::endl;
      std::cout << v1.str() << " == " << v3.str() << "? "
                << (v1 == v3) << std::endl;
      std::cout << v2.str() << " > " << v1.str() << "? "
                << (v2 > v1) << std::endl;

      Punto p1(1, 2), p2(1, 5), p3(2, 0);

      std::cout << "\nP(1,2) < P(1,5)? " << (p1 < p2) << std::endl;
      std::cout << "P(1,2) < P(2,0)? " << (p1 < p3) << std::endl;

      return 0;
  }

================================================================================
  RESUMEN
================================================================================

  • La sobrecarga de operadores permite que tipos definidos por el usuario
    se comporten como tipos primitivos.
  • Casi todos los operadores se pueden sobrecargar excepto ::, ., .*, ?:.
  • Forma canónica: implementar operator+= y derivar operator+.
  • Miembro vs no-miembro: =, [], (), -> deben ser miembros; <<, >> no.
  • Copy-and-Swap idiom para operator= con seguridad excepcional.
  • Functors (operator()) permiten objetos con comportamiento de función.
  • C++20: operator<=> genera automáticamente todos los comparadores.

================================================================================
  PREGUNTAS FRECUENTES (FAQ)
================================================================================

  P: ¿Cuándo debo sobrecargar un operador?
  R: Cuando hace el código más natural y legible. Si el significado no
    es obvio, mejor usar una función con nombre.

  P: ¿Puedo cambiar la precedencia de un operador?
  R: No. La precedencia y asociatividad son fijas. Solo cambias el
    comportamiento cuando se aplica a tu tipo.

  P: ¿Puedo crear nuevos operadores?
  R: No. Solo puedes sobrecargar los operadores existentes. No puedes
    crear **, <-, o similares.

  P: ¿operator= debe ser miembro?
  R: Sí. Si no se define, el compilador genera uno por defecto (copia
    miembro a miembro).

  P: ¿Cuándo usar friend para operadores?
  R: Cuando el operador necesita acceso a miembros privados y no puede
    ser miembro (como operator<<).

  P: ¿Sobrecarga de operadores virtual?
  R: Los operadores no pueden ser virtuales directamente. Pero pueden
    llamar a funciones virtuales internamente.
