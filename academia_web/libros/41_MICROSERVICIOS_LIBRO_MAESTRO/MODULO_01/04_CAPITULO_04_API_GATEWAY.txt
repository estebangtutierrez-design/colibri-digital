CONTENIDO DEL CAPÍTULO
===============================================================================

Gateway: punto único de entrada. Enrutamiento, auth, rate limit.

Spring Cloud Gateway, Kong, NGINX, APISIX.

Route predicates, filters, load balancing.

EJEMPLO DE CÓDIGO
----------------------------------------

spring:
  cloud:
    gateway:
      routes:
        - id: user-service
          uri: lb://user-service
          predicates:
            - Path=/api/users/**
          filters:
            - StripPrefix=1
            - name: RequestRateLimiter
              args:
                redis-rate-limiter.replenishRate: 10
        - id: order-service
          uri: lb://order-service
          predicates:
            - Path=/api/orders/**
