1. ¿QUÉ ES VUE?

Vue.js es un framework progresivo para interfaces de usuario.

Creado por Evan You en 2014. Vue 3 lanzado en 2020.

Reactivo, basado en componentes, incrementalmente adoptable.

npm create vue@latest. Vite como bundler.

EJEMPLO DE CÓDIGO
----------------------------------------

npm create vue@latest
cd my-project
npm install
npm run dev
