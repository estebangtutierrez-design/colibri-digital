CONTENIDO DEL CAPÍTULO
===============================================================================

1. COMPOSITION API

setup() o <script setup>. ref, reactive, computed, watch.

Mejor organización lógica. Reutilización con composables.

Mejor soporte TypeScript.

2. OPTIONS API

data, methods, computed, watch, lifecycle hooks.

Familiar para desarrolladores Vue 2.

EJEMPLO DE CÓDIGO
----------------------------------------

<script setup lang="ts">
import { ref, computed } from "vue"
const count = ref(0)
const doubled = computed(() => count.value * 2)
function increment() { count.value++ }
</script>

<template>
  <button @click="increment">{{ count }} x2 = {{ doubled }}</button>
</template>
