                    BIENVENIDA A STRINGS AVANZADO
═══════════════════════════════════════════════════════════════════════════════════

Bienvenido, maestro del texto.

En los módulos anteriores trabajaste con strings de forma básica: concatenación,
interpolación, métodos simples. Pero los strings en Ruby tienen un arsenal
de métodos mucho más profundo.

Este capítulo está dedicado a los métodos avanzados de string que todo
Rubyista profesional debe conocer:

  • scan — extraer patrones de un string.
  • split — dividir en partes.
  • join — unir elementos en un string.
  • gsub/sub — reemplazar patrones.
  • match — encontrar coincidencias.
  • delete — eliminar caracteres específicos.
  • tr — traducir (reemplazar) caracteres.
  • squeeze — comprimir caracteres repetidos.
  • center/ljust/rjust — alinear texto.
  • chars — iterar sobre caracteres.

Estos métodos te permitirán procesar texto de forma elegante y eficiente.
Desde parsing de archivos hasta limpieza de datos, el manejo avanzado de
strings es una habilidad fundamental.

                    1. SCAN — EXTRAER PATRONES
═══════════════════════════════════════════════════════════════════════════════════

scan extrae todas las ocurrencias de un patrón (regex o string) del texto.
Devuelve un array con las coincidencias.

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 1 — scan con strings

```ruby
texto = "La casa es grande y la casa es bonita"

# Buscar todas las ocurrencias de "casa"
resultados = texto.scan("casa")
puts resultados.inspect  # => ["casa", "casa"]
puts resultados.length   # => 2
```

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 2 — scan con expresiones regulares

```ruby
texto = "Mi número es 123-4567 y el otro es 987-6543"

# Extraer todos los números de teléfono
telefonos = texto.scan(/\d{3}-\d{4}/)
puts telefonos.inspect  # => ["123-4567", "987-6543"]

# Extraer palabras de 5 letras
frase = "Ruby es un lenguaje hermoso y poderoso"
palabras_5 = frase.scan(/\b\w{5}\b/)
puts palabras_5.inspect  # => ["Ruby", "hermoso"]
```

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 3 — scan con grupos de captura

```ruby
texto = "Ana: 85, Bob: 73, Carlos: 95"

# Scan con grupos devuelve arrays anidados
resultados = texto.scan(/(\w+): (\d+)/)
puts resultados.inspect
# => [["Ana", "85"], ["Bob", "73"], ["Carlos", "95"]]

# Convertir a hash
hash = resultados.to_h
puts hash.inspect  # => {"Ana"=>"85", "Bob"=>"73", "Carlos"=>"95"}
```

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 4 — scan con bloque

```ruby
texto = "precio: 25, descuento: 5, total: 20"

texto.scan(/(\w+): (\d+)/) do |nombre, valor|
  puts "#{nombre.capitalize}: #{valor} euros"
end
# => Precio: 25 euros
# => Descuento: 5 euros
# => Total: 20 euros
```

═══════════════════════════════════════════════════════════════════════════════════
                    2. SPLIT — DIVIDIR STRINGS
═══════════════════════════════════════════════════════════════════════════════════

split divide un string en partes usando un separador (string o regex).

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 5 — split básico

```ruby
# Separar por espacios (por defecto)
frase = "Ruby es un lenguaje genial"
palabras = frase.split
puts palabras.inspect  # => ["Ruby", "es", "un", "lenguaje", "genial"]

# Separar por comas
csv = "Ana,25,Ingeniera"
datos = csv.split(",")
puts datos.inspect  # => ["Ana", "25", "Ingeniera"]
```

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 6 — split con regex

```ruby
# Separar por múltiples delimitadores
texto = "hola,mundo;ruby|python"
partes = texto.split(/[,;|]/)
puts partes.inspect  # => ["hola", "mundo", "ruby", "python"]

# Separar por espacios en blanco (incluyendo tabs y nuevas líneas)
texto2 = "uno\tdos\ntres  cuatro"
puts texto2.split(/\s+/).inspect  # => ["uno", "dos", "tres", "cuatro"]
```

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 7 — split con límite

```ruby
# limit: número máximo de partes
datos = "nombre:edad:ciudad:pais"

# Sin límite
puts datos.split(":").inspect  # => ["nombre", "edad", "ciudad", "pais"]

# Con límite 2
puts datos.split(":", 2).inspect  # => ["nombre", "edad:ciudad:pais"]

# Con límite 3
puts datos.split(":", 3).inspect  # => ["nombre", "edad", "ciudad:pais"]

# Con límite negativo (mantiene los separadores al final)
puts "a:b:c:".split(":", -1).inspect  # => ["a", "b", "c", ""]
```

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 8 — split para procesar archivos

```ruby
# Parsear una línea de log
log_line = "2026-07-02 10:30:45 ERROR Conexión fallida: timeout"
fecha, hora, nivel, *mensaje = log_line.split
puts "Fecha: #{fecha}"
puts "Hora: #{hora}"
puts "Nivel: #{nivel}"
puts "Mensaje: #{mensaje.join(" ")}"
# => Fecha: 2026-07-02
# => Hora: 10:30:45
# => Nivel: ERROR
# => Mensaje: Conexión fallida: timeout
```

═══════════════════════════════════════════════════════════════════════════════════
                    3. JOIN — UNIR ELEMENTOS
═══════════════════════════════════════════════════════════════════════════════════

join une los elementos de un array en un string, separándolos con el
separador especificado (por defecto, sin separador).

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 9 — join básico

```ruby
palabras = ["Ruby", "es", "genial"]

# Sin separador
puts palabras.join  # => "Rubyesgenial"

# Con separador
puts palabras.join(" ")  # => "Ruby es genial"
puts palabras.join("-")  # => "Ruby-es-genial"
puts palabras.join(", ")  # => "Ruby, es, genial"
```

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 10 — join para construir CSV

```ruby
def to_csv_line(datos)
  datos.map { |v| v.to_s.include?(",") ? "\"#{v}\"" : v }.join(",")
end

fila1 = to_csv_line(["Ana", 25, "Ingeniera"])
fila2 = to_csv_line(["Bob", 30, "Analista, Senior"])
puts fila1  # => Ana,25,Ingeniera
puts fila2  # => Bob,30,"Analista, Senior"
```

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 11 — join con arrays anidados

```ruby
matriz = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]

# Unir filas como strings
filas = matriz.map { |fila| fila.join("-") }
puts filas.inspect  # => ["1-2-3", "4-5-6", "7-8-9"]

# Unir todo
puts filas.join(" | ")  # => "1-2-3 | 4-5-6 | 7-8-9"
```

═══════════════════════════════════════════════════════════════════════════════════
                    4. GSUB / SUB — REEMPLAZAR
═══════════════════════════════════════════════════════════════════════════════════

sub reemplaza la PRIMERA ocurrencia. gsub reemplaza TODAS las ocurrencias
(global substitution).

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 12 — sub y gsub básicos

```ruby
texto = "gato perro gato pájaro gato"

# sub: solo la primera
puts texto.sub("gato", "perro")  # => "perro perro gato pájaro gato"

# gsub: todas
puts texto.gsub("gato", "perro")  # => "perro perro perro pájaro perro"
```

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 13 — gsub con regex

```ruby
texto = "Mi email es ana@email.com y el de bob es bob@mail.org"

# Ocultar emails
oculto = texto.gsub(/(\w+)@(\w+)\.(\w+)/, '\1@***.\3')
puts oculto
# => "Mi email es ana@***.com y el de bob es bob@***.org"

# Convertir a mayúsculas ciertos patrones
codigo = "user_123, product_456, order_789"
resultado = codigo.gsub(/\d+/) { |n| n.to_i * 2 }
puts resultado  # => "user_246, product_912, order_1578"
```

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 14 — gsub con bloque

```ruby
texto = "Los precios son: 25, 50, 75, 100"

# Aplicar IVA a los números
resultado = texto.gsub(/\d+/) do |num|
  (num.to_i * 1.18).round(2)
end
puts resultado  # => "Los precios son: 29.5, 59.0, 88.5, 118.0"
```

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 15 — gsub! (modifica el original)

```ruby
saludo = "Hola Mundo"
saludo.gsub!("Mundo", "Ruby")
puts saludo  # => "Hola Ruby"
```

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 16 — Backreferences en gsub

```ruby
texto = "2026-07-02"

# Reordenar fecha a formato DD/MM/YYYY
reordenado = texto.gsub(/(\d{4})-(\d{2})-(\d{2})/, '\3/\2/\1')
puts reordenado  # => "02/07/2026"

# También con \1, \2, etc.
# O con nombres (solo en Ruby 2.4+ con Regexp.new)
```

                    5. MATCH — ENCONTRAR COINCIDENCIAS
═══════════════════════════════════════════════════════════════════════════════════

match encuentra la PRIMERA coincidencia de un patrón. Devuelve un
MatchData object o nil.

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 17 — match básico

```ruby
texto = "Mi número de teléfono es 123-4567"

matched = texto.match(/(\d{3})-(\d{4})/)
if matched
  puts "Coincidencia completa: #{matched[0]}"  # => 123-4567
  puts "Grupo 1: #{matched[1]}"                # => 123
  puts "Grupo 2: #{matched[2]}"                # => 4567
  puts "Pre-match: #{matched.pre_match}"       # => "Mi número de teléfono es "
  puts "Post-match: #{matched.post_match}"     # => ""
end
```

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 18 — match con named captures

```ruby
texto = "Nombre: Ana, Edad: 30"

patron = /Nombre: (?<nombre>\w+), Edad: (?<edad>\d+)/
matched = texto.match(patron)

if matched
  puts matched[:nombre]  # => "Ana"
  puts matched[:edad]    # => "30"

  # También como métodos si hay match
  # (Ruby 2.4+)
end
```

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 19 — match? (Ruby 2.4+)

```ruby
texto = "Hola Mundo"

# match? devuelve true/false (más rápido, no crea MatchData)
puts texto.match?(/Mundo/)  # => true
puts texto.match?(/Ruby/)   # => false

# Generalmente más rápido que match si solo necesitas boolean
```

                    6. DELETE — ELIMINAR CARACTERES
═══════════════════════════════════════════════════════════════════════════════════

delete elimina todos los caracteres especificados del string.

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 20 — delete básico

```ruby
texto = "Hola Mundo 123"

# Eliminar vocales
sin_vocales = texto.delete("aeiou")
puts sin_vocales  # => "Hl Mnd 123"

# Eliminar dígitos
sin_digitos = texto.delete("0-9")
puts sin_digitos  # => "Hola Mundo "

# Eliminar espacios
sin_espacios = texto.delete(" ")
puts sin_espacios  # => "HolaMundo123"
```

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 21 — delete con rangos y complementos

```ruby
texto = "abc123def456"

# Eliminar letras de la a a la f
puts texto.delete("a-f")   # => "123456"

# Eliminar dígitos
puts texto.delete("0-9")   # => "abcdef"

# ^ al inicio niega (conserva solo los que coinciden)
puts texto.delete("^a-f")  # => "abcdef" (elimina lo que NO está en a-f)
puts texto.delete("^0-9")  # => "123456"
```

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 22 — delete! (modifica in-place)

```ruby
texto = "Hola, mundo!!!"
texto.delete!(",!?")
puts texto  # => "Hola mundo"
```

═══════════════════════════════════════════════════════════════════════════════════
                    7. TR — TRADUCIR CARACTERES
═══════════════════════════════════════════════════════════════════════════════════

tr traduce (reemplaza) caracteres según una correspondencia.
Similar al comando Unix tr.

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 23 — tr básico

```ruby
# Reemplazar caracteres individuales
puts "hola".tr("h", "m")      # => "mola"
puts "gato".tr("gt", "pr")    # => "paro" (g->p, t->r)

# Reemplazar rangos
puts "abc123".tr("a-z", "A-Z")  # => "ABC123"
puts "Hola".tr("A-Za-z", "a-zA-Z")  # => "hOLA" (invertir mayúsculas)
```

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 24 — tr avanzado

```ruby
# Cifrado César simple
def cifrar_cesar(texto, desplazamiento = 3)
  abc = "abcdefghijklmnopqrstuvwxyz"
  cifrado = abc.chars.rotate(desplazamiento).join
  texto.downcase.tr(abc, cifrado)
end

puts cifrar_cesar("hola mundo", 3)  # => "krod pxqgr"

# Reemplazar espacios por guiones
puts "Ruby es genial".tr(" ", "-")  # => "Ruby-es-genial"

# Comprimir caracteres con tr (usando "")
puts "Hello".tr("l", "")     # => "Heo" (elimina todas las l)
```

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 25 — tr_s (tr + squeeze)

```ruby
# tr_s reemplaza y comprime
puts "goolgle".tr_s("l", "o")  # => "googe" (l->o y comprime)

# Equivalente a:
puts "goolgle".tr("l", "o").squeeze("o")  # => "googe"
```

═══════════════════════════════════════════════════════════════════════════════════
                    8. SQUEEZE — COMPRIMIR REPETICIONES
═══════════════════════════════════════════════════════════════════════════════════

squeeze comprime caracteres repetidos consecutivos en uno solo.

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 26 — squeeze básico

```ruby
# Comprimir todos los caracteres repetidos
puts "Hoolaa Mundoo".squeeze   # => "Hola Mundo"

# Comprimir solo caracteres específicos
puts "Hoolaa Mundoo".squeeze("o")  # => "Hola Mundo" (solo comprime o)
puts "Hoolaa Mundoo".squeeze("a")  # => "Hoola Mundoo" (solo comprime a)

# squeeze! (modifica in-place)
texto = "AAA BBB CCC"
texto.squeeze!(" ")
puts texto  # => "AAA BBB CCC"
```

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 27 — squeeze para limpieza de datos

```ruby
# Limpiar espacios múltiples
texto_sucio = "Ruby    es     un    lenguaje     genial"
limpio = texto_sucio.squeeze(" ")
puts limpio  # => "Ruby es un lenguaje genial"

# Combinar con strip
puts limpio.strip  # => "Ruby es un lenguaje genial"
```

═══════════════════════════════════════════════════════════════════════════════════
                    9. CENTER / LJUST / RJUST — ALINEACIÓN
═══════════════════════════════════════════════════════════════════════════════════

Estos métodos alinean un string dentro de un ancho especificado.

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 28 — center

```ruby
texto = "Ruby"

# Centrar en 20 caracteres (rellena con espacios)
puts texto.center(20)  # => "        Ruby        "

# Centrar con relleno personalizado
puts texto.center(20, "-")  # => "--------Ruby--------"
puts texto.center(20, "*=")  # => "*=*=*=*=Ruby*=*=*=*="
```

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 29 — ljust y rjust

```ruby
texto = "Ruby"

# Alinear a la izquierda
puts texto.ljust(20, ".")  # => "Ruby................"

# Alinear a la derecha
puts texto.rjust(20, ".")  # => "................Ruby"

# Uso práctico: formatear números
puts 42.to_s.rjust(5, "0")   # => "00042"
puts 123.to_s.ljust(5)       # => "123  "
```

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 30 — Formateo de tablas

```ruby
productos = [
  ["Laptop", 1000, 5],
  ["Mouse", 25, 100],
  ["Teclado", 80, 50],
  ["Monitor", 300, 10]
]

puts "#{"Producto".ljust(15)} #{"Precio".rjust(10)} #{"Stock".rjust(10)}"
puts "-" * 35
productos.each do |nombre, precio, stock|
  puts "#{nombre.ljust(15)} #{precio.to_s.rjust(10)} #{stock.to_s.rjust(10)}"
end
# => Producto          Precio      Stock
# => -----------------------------------
# => Laptop              1000          5
# => Mouse                 25        100
# => Teclado               80         50
# => Monitor              300         10
```

                    10. CHARS — ITERAR CARACTERES
═══════════════════════════════════════════════════════════════════════════════════

chars devuelve un array de caracteres (strings de un solo carácter).

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 31 — chars básico

```ruby
palabra = "Ruby"
caracteres = palabra.chars
puts caracteres.inspect  # => ["R", "u", "b", "y"]

# Iterar sobre caracteres
palabra.chars.each_with_index { |c, i| puts "#{i}: #{c}" }
# => 0: R
# => 1: u
# => 2: b
# => 3: y
```

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 32 — chars con transformación

```ruby
# Contar vocales
texto = "Ruby es un lenguaje hermoso"
vocales = texto.downcase.chars.count { |c| "aeiou".include?(c) }
puts vocales  # => 11

# Invertir caracteres de una palabra
puts "Ruby".chars.reverse.join  # => "ybuR"
```

═══════════════════════════════════════════════════════════════════════════════════

Ejemplo 33 — each_char

```ruby
# each_chars es similar a chars.each
"Ruby".each_char { |c| print "#{c}." }
# => R.u.b.y.
```


Ejemplo 36 — partition y rpartition

```ruby
texto = "Ruby es genial"

# partition: divide en [antes, separador, después]

# rpartition: busca desde la derecha
texto2 = "a.b.c.d"
puts texto2.rpartition(".").inspect  # => ["a.b.c", ".", "d"]
```

Ejemplo 37 — start_with? y end_with?

```ruby
texto = "Ruby es genial"

puts texto.start_with?("Ruby")  # => true

```

Ejemplo 38 — hex, oct, to_i con base

```ruby
# Convertir string a número según base
puts "FF".hex       # => 255
puts "ff".hex       # => 255
puts "377".oct      # => 255
puts "1010".to_i(2) # => 10 (binario)
puts "1010".to_i(8) # => 520 (octal)
puts "FF".to_i(16)  # => 255 (hexadecimal)
```

```ruby
texto = "línea uno\nlínea dos\nlínea tres"
```

                     FAQ — PREGUNTAS FRECUENTES
═══════════════════════════════════════════════════════════════════════════════════

P: ¿Cuál es la diferencia entre sub y gsub?
R: sub reemplaza solo la primera ocurrencia. gsub reemplaza todas.

P: ¿Cómo uso grupos de captura en gsub?
R: Con \1, \2, etc. en el string de reemplazo. Ejemplo:
gsub(/(\w+)-(\w+)/, '\2-\1').

P: ¿Qué devuelve match si no encuentra nada?
R: nil. Siempre verifica: if matched = texto.match(/.../)

P: ¿Cuál es la diferencia entre tr y gsub?
R: tr reemplaza caracteres INDIVIDUALES según una correspondencia. Es mucho
más rápido que gsub para ese caso de uso. gsub trabaja con patrones regex.

P: ¿Cómo elimino todos los espacios de un string?
R: texto.delete(" ") o texto.gsub(/\s+/, "").

P: ¿Cómo cuento vocales en un string?
R: texto.downcase.count("aeiou").

P: ¿Qué hace chars?
R: Devuelve un array de strings de un solo carácter. "Ruby".chars => ["R","u","b","y"]

P: ¿Cómo divido un string en palabras?
R: texto.split (divide por espacios en blanco) o texto.scan(/\w+/).
