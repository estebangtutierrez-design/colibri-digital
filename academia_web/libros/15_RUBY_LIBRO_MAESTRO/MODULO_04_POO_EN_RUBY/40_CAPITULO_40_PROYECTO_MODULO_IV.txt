  1. VISIÓN GENERAL DEL PROYECTO

  El proyecto consiste en un sistema bancario con las siguientes
  capacidades:

  - Gestión de clientes (personas físicas y jurídicas)
  - Cuentas bancarias (corriente, ahorro, inversión)
  - Transacciones (depósitos, retiros, transferencias)
  - Cálculo de intereses y comisiones
  - Generación de reportes y estados de cuenta
  - Autenticación y control de acceso

  El sistema debe ser extensible: añadir nuevos tipos de cuentas o
  nuevas reglas de negocio no debe requerir modificar código existente.

  class Cliente
    attr_reader :id, :nombre, :email, :fecha_registro

    @@contador = 0

    def initialize(nombre:, email:, telefono: nil)
      @@contador += 1
      @id = "CLI-#{@@contador.to_s.rjust(5, '0')}"
      @nombre = nombre
      @email = email
      @telefono = telefono
      @fecha_registro = Time.now
      @activo = true
    end

    def activo?
      @activo
    end

    def desactivar
      @activo = false
    end

    def reactivar
      @activo = true
    end

    def to_s
      "#{@id}: #{@nombre} (#{@email})"
    end

    def tipo_cliente
      self.class.name
    end
  end

  class ClientePersonal < Cliente
    attr_reader :apellido, :fecha_nacimiento

    def initialize(nombre:, apellido:, email:, fecha_nacimiento: nil, telefono: nil)

      @apellido = apellido
      @fecha_nacimiento = fecha_nacimiento
    end

    def nombre_completo
      "#{@nombre} #{@apellido}"
    end

    def to_s
      "#{@id}: #{nombre_completo} (#{@email})"
    end
  end

  class Cuenta
    attr_reader :numero, :cliente, :fecha_apertura

    @@contador = 0

    def initialize(cliente:, saldo_inicial: 0)
      raise ArgumentError, "Cliente inválido" unless cliente.is_a?(Cliente)
      raise ArgumentError, "Saldo inicial no puede ser negativo" if saldo_inicial < 0

      @@contador += 1
      @numero = generar_numero
      @cliente = cliente
      @saldo = saldo_inicial
      @fecha_apertura = Time.now
      @activa = true
      @movimientos = []
      registrar_movimiento(:apertura, saldo_inicial, "Apertura de cuenta")
    end

    def depositar(monto, descripcion = "Depósito")
      validar_monto_positivo(monto)
      return "Cuenta inactiva" unless @activa

      @saldo += monto
      registrar_movimiento(:deposito, monto, descripcion)
      "Depósito exitoso. Saldo: $#{'%.2f' % @saldo}"
    end

    def retirar(monto, descripcion = "Retiro")
      validar_monto_positivo(monto)
      return "Cuenta inactiva" unless @activa
      return "Saldo insuficiente" if monto > @saldo

      comision = calcular_comision_retiro(monto)
      total = monto + comision

      return "Saldo insuficiente (incluyendo comisión)" if total > @saldo

      @saldo -= total
      registrar_movimiento(:retiro, monto, descripcion)
      registrar_movimiento(:comision, comision, "Comisión por retiro") if comision > 0
      "Retiro exitoso. Comisión: $#{'%.2f' % comision}. Saldo: $#{'%.2f' % @saldo}"
    end

    def saldo
      @saldo
    end

    def activa?
      @activa
    end

    def desactivar
      @activa = false
    end

    def activar
      @activa = true
    end

    def movimientos
      @movimientos.dup
    end

    def ultimos_movimientos(n = 5)
      @movimientos.last(n).reverse
    end

    def calcular_intereses
      raise NotImplementedError, "Cada tipo de cuenta debe implementar su cálculo de intereses"
    end

    def to_s
      "#{@numero}: #{@cliente.nombre} — $#{'%.2f' % @saldo}"
    end

    def estado_de_cuenta
      lines = []
      lines << "=" * 60
      lines << "ESTADO DE CUENTA #{@numero}"
      lines << "Titular: #{@cliente}"
      lines << "Tipo: #{self.class.name}"
      lines << "Saldo: $#{'%.2f' % @saldo}"
      lines << "Estado: #{@activa ? 'ACTIVA' : 'INACTIVA'}"
      lines << "-" * 60
      lines << "MOVIMIENTOS:"
      @movimientos.reverse.each_with_index do |m, i|
        lines << "  #{i + 1}. #{m[:fecha]} | #{m[:tipo].to_s.upcase.rjust(12)} | $#{'%.2f' % m[:monto]} | #{m[:descripcion]}"
      end
      lines << "=" * 60
      lines.join("\n")
    end

    protected

    def calcular_comision_retiro(monto)
      0
    end

    private

    def generar_numero
      tipo = self.class.name[0..2].upcase
      "#{tipo}-#{@@contador.to_s.rjust(8, '0')}"
    end

    def validar_monto_positivo(monto)
      raise ArgumentError, "Monto debe ser numérico" unless monto.is_a?(Numeric)
      raise ArgumentError, "Monto debe ser positivo" unless monto > 0
    end

    def registrar_movimiento(tipo, monto, descripcion = "")
      @movimientos << {
        fecha: Time.now.strftime("%Y-%m-%d %H:%M:%S"),
        tipo: tipo,
        monto: monto,
        descripcion: descripcion,
        saldo_despues: @saldo
      }
    end
  end

  class CuentaCorriente < Cuenta
    COMISION_POR_RETIRO = 5.0
    INTERES_ANUAL = 0.01
    LIMITE_DESCUBIERTO = 0

    def initialize(cliente:, saldo_inicial: 0, limite_descubierto: 0)
      super(cliente: cliente, saldo_inicial: saldo_inicial)
      @limite_descubierto = limite_descubierto
    end

    def retirar(monto, descripcion = "Retiro")
      validar_monto_positivo(monto)
      return "Cuenta inactiva" unless @activa

      if monto > @saldo + @limite_descubierto
        return "Límite de descubierto excedido. Disponible: $#{'%.2f' % (@saldo + @limite_descubierto)}"
      end

      comision = calcular_comision_retiro(monto)
      total = monto + comision

      @saldo -= total
      registrar_movimiento(:retiro, monto, descripcion)
      registrar_movimiento(:comision, comision, "Comisión por retiro") if comision > 0

      if @saldo < 0
        registrar_movimiento(:sobregiro, @saldo.abs, "Sobregiro utilizado")
        "Retiro exitoso (uso de sobregiro). Comisión: $#{'%.2f' % comision}. Saldo: $#{'%.2f' % @saldo}"
      else
        "Retiro exitoso. Comisión: $#{'%.2f' % comision}. Saldo: $#{'%.2f' % @saldo}"
      end
    end

    def calcular_intereses
      if @saldo > 0
        @saldo * INTERES_ANUAL / 12
      else
        0
      end
    end

    protected

    def calcular_comision_retiro(monto)
      COMISION_POR_RETIRO
    end
  end

  class CuentaAhorro < Cuenta
    TASA_INTERES = 0.04
    RETIROS_GRATIS_POR_MES = 3
    COMISION_RETIRO_EXTRA = 10.0

    def initialize(cliente:, saldo_inicial: 0)
      super(cliente: cliente, saldo_inicial: saldo_inicial)
      @retiros_este_mes = 0
      @mes_actual = Time.now.month
    end

    def retirar(monto, descripcion = "Retiro")
      actualizar_contador_mensual

      comision = calcular_comision_retiro(monto)
      total = monto + comision
      return "Saldo insuficiente" if total > @saldo

      @saldo -= total
      @retiros_este_mes += 1
      registrar_movimiento(:retiro, monto, descripcion)
      registrar_movimiento(:comision, comision, "Comisión por retiro extra") if comision > 0
      "Retiro exitoso. Comisión: $#{'%.2f' % comision}. Saldo: $#{'%.2f' % @saldo}"
    end

    def calcular_intereses
      (@saldo * TASA_INTERES / 12).round(2)
    end

    def aplicar_intereses
      interes = calcular_intereses
      if interes > 0
        @saldo += interes
        registrar_movimiento(:interes, interes, "Interés del mes")
        "Interés de $#{'%.2f' % interes} aplicado. Nuevo saldo: $#{'%.2f' % @saldo}"
      else
        "No hay intereses que aplicar"
      end
    end

    protected

    def calcular_comision_retiro(monto)
      @retiros_este_mes >= RETIROS_GRATIS_POR_MES ? COMISION_RETIRO_EXTRA : 0
    end

    private

    def actualizar_contador_mensual
      if Time.now.month != @mes_actual
        @retiros_este_mes = 0
        @mes_actual = Time.now.month
      end
    end
  end

  class CuentaInversion < Cuenta
    def initialize(cliente:, saldo_inicial: 0, tasa_riesgo: :moderada)
      super(cliente: cliente, saldo_inicial: saldo_inicial)

      @tasa_riesgo = tasa_riesgo
      @inversiones = []
      @tasas = {
        conservadora: 0.06,
        moderada: 0.10,
        agresiva: 0.16
      }
    end

    def invertir(monto, instrumento)
      validar_monto_positivo(monto)
      return "Cuenta inactiva" unless @activa
      return "Saldo insuficiente" if monto > @saldo

      @saldo -= monto
      @inversiones << {
        fecha: Time.now,
        monto: monto,
        instrumento: instrumento,
        rendimiento: 0
      }
      registrar_movimiento(:inversion, monto, "Inversión en #{instrumento}")
      "Inversión de $#{'%.2f' % monto} en #{instrumento} realizada"
    end

    def calcular_intereses
      @saldo * @tasas[@tasa_riesgo] / 12
    end

    def valor_total
      @saldo + @inversiones.sum { |i| i[:monto] }
    end

    def to_s
      "#{super} | Valor total: $#{'%.2f' % valor_total} | Riesgo: #{@tasa_riesgo}"
    end

    protected

    def calcular_comision_retiro(monto)
      monto * 0.01  # 1% de comisión
    end
  end

  class TransferenciaService
    def initialize
      @transferencias = []
    end

    def transferir(origen, destino, monto, descripcion = "Transferencia")
      raise ArgumentError, "Origen debe ser una Cuenta" unless origen.is_a?(Cuenta)
      raise ArgumentError, "Destino debe ser una Cuenta" unless destino.is_a?(Cuenta)
      validar_monto_positivo(monto)

      resultado_origen = origen.retirar(monto, "Transferencia a #{destino.numero}")
      return resultado_origen if resultado_origen.start_with?("Saldo insuficiente", "Límite", "Cuenta inactiva")

      destino.depositar(monto, "Transferencia de #{origen.numero}")
      @transferencias << {
        fecha: Time.now,
        origen: origen.numero,
        destino: destino.numero,
        monto: monto,
        descripcion: descripcion
      }
      {
        exito: true,
        mensaje: "Transferencia de $#{'%.2f' % monto} de #{origen.numero} a #{destino.numero} exitosa",
        comision: 0
      }
    end

    def historial
      @transferencias.dup
    end

    def transferencias_por_cuenta(numero_cuenta)
      @transferencias.select { |t| t[:origen] == numero_cuenta || t[:destino] == numero_cuenta }
    end

    private

    def validar_monto_positivo(monto)
      raise ArgumentError, "Monto debe ser positivo" unless monto.is_a?(Numeric) && monto > 0
    end
  end

  module Reportable
    def to_hash
      { class: self.class.name }.merge(instance_variables.each_with_object({}) do |var, hash|
        key = var.to_s.sub("@", "").to_sym
        value = instance_variable_get(var)
        hash[key] = value.is_a?(String) || value.is_a?(Numeric) || value.is_a?(TrueClass) || value.is_a?(FalseClass) ? value : value.to_s
      end)
    end
  end

  class ReporteService
    def generar_estado_cuenta(cuenta, formato = :texto)
      case formato
      when :texto
        cuenta.estado_de_cuenta
      when :resumido
        "#{cuenta.numero}: $#{'%.2f' % cuenta.saldo} (#{cuenta.ultimos_movimientos(3).size} movs recientes)"
      when :completo
        reporte = cuenta.estado_de_cuenta
        reporte += "\n\nINTERESES DEL MES:\n"
        reporte += "  Interés calculado: $#{'%.2f' % cuenta.calcular_intereses}\n"
        reporte
      else
        raise ArgumentError, "Formato desconocido: #{formato}"
      end
    end

    def generar_reporte_banco(cuentas)
      lines = []
      lines << "=" * 60
      lines << "REPORTE GENERAL DEL BANCO"
      lines << "Fecha: #{Time.now.strftime('%Y-%m-%d %H:%M:%S')}"
      lines << "-" * 60
      lines << "RESUMEN:"
      lines << "Total cuentas: #{cuentas.size}"
      lines << "Cuentas activas: #{cuentas.count(&:activa?)}"
      lines << "Saldo total: $#{'%.2f' % cuentas.sum(&:saldo)}"
      lines << "-" * 60
      lines << "DETALLE POR TIPO:"

      cuentas.group_by { |c| c.class.name }.each do |tipo, grupo|
        lines << "  #{tipo}: #{grupo.size} cuentas, $#{'%.2f' % grupo.sum(&:saldo)}"
      end

      lines << "=" * 60
      lines.join("\n")
    end

    def exportar_csv(cuentas)
      require 'csv'
      CSV.generate do |csv|
        csv << ["Número", "Tipo", "Cliente", "Saldo", "Activa", "Fecha Apertura"]
        cuentas.each do |c|
          csv << [c.numero, c.class.name, c.cliente.nombre, c.saldo, c.activa?, c.fecha_apertura.strftime("%Y-%m-%d")]
        end
      end
    end
  end

  module Autenticable
    def autenticar(password)
      @password_hash == hash_password(password)
    end

    def cambiar_password(password_actual, password_nuevo)
      if autenticar(password_actual)
        @password_hash = hash_password(password_nuevo)
        true
      else
        false
      end
    end

    private

    def hash_password(password)
      Digest::SHA256.hexdigest("#{password}#{@salt}")
    end

    def generar_salt
      SecureRandom.hex(16)
    end
  end

  class UsuarioSistema
    include Autenticable

    attr_reader :username, :rol, :empleado_desde

    ROLES = [:admin, :ejecutivo, :consultor, :auditor]

    def initialize(username:, password:, rol: :consultor)
      @username = username
      @rol = rol
      @salt = generar_salt
      @password_hash = hash_password(password)
      @empleado_desde = Time.now
      @activo = true
    end

    def puede?(accion)
      case @rol
      when :admin
        true
      when :ejecutivo
        %i[crear_cuenta depositar retirar transferir ver_reportes].include?(accion)
      when :consultor
        %i[ver_reportes ver_estado_cuenta].include?(accion)
      when :auditor
        %i[ver_reportes ver_historial ver_estado_cuenta].include?(accion)
      else
        false
      end
    end

    def to_s
      "#{@username} (#{@rol})"
    end
  end

  class AutenticacionService
    def initialize
      @usuarios = {}
    end

    def registrar_usuario(username, password, rol: :consultor)
      return "Usuario ya existe" if @usuarios.key?(username)
      @usuarios[username] = UsuarioSistema.new(username: username, password: password, rol: rol)
      "Usuario #{username} registrado como #{rol}"
    end

    def autenticar(username, password)
      usuario = @usuarios[username]
      return nil unless usuario
      return nil unless usuario.autenticar(password)
      usuario
    end

    def autorizar?(usuario, accion)
      usuario&.puede?(accion)
    end
  end

  class Banco
    include Reportable

    attr_reader :nombre, :cuentas, :clientes

    def initialize(nombre)
      @nombre = nombre
      @clientes = []
      @cuentas = []
      @transferencia_service = TransferenciaService.new
      @reporte_service = ReporteService.new
      @autenticacion_service = AutenticacionService.new
      @caja = 0.0
    end

    # Gestión de clientes
    def registrar_cliente_personal(nombre:, apellido:, email:, fecha_nacimiento: nil, telefono: nil)
      cliente = ClientePersonal.new(
        nombre: nombre,
        apellido: apellido,
        email: email,
        fecha_nacimiento: fecha_nacimiento,
        telefono: telefono
      )
      @clientes << cliente
      cliente
    end

    def registrar_cliente_empresarial(nombre:, razon_social:, rfc:, email:, representante_legal: nil, telefono: nil)
      cliente = ClienteEmpresarial.new(
        nombre: nombre,
        razon_social: razon_social,
        rfc: rfc,
        email: email,
        representante_legal: representante_legal,
        telefono: telefono
      )
      @clientes << cliente
      cliente
    end

    def buscar_cliente(id)
      @clientes.find { |c| c.id == id }
    end

    def clientes_activos
      @clientes.select(&:activo?)
    end

    # Gestión de cuentas
    def abrir_cuenta_corriente(cliente:, saldo_inicial: 0, limite_descubierto: 0)
      cuenta = CuentaCorriente.new(cliente: cliente, saldo_inicial: saldo_inicial, limite_descubierto: limite_descubierto)
      @cuentas << cuenta
      cuenta
    end

    def abrir_cuenta_ahorro(cliente:, saldo_inicial: 0)
      cuenta = CuentaAhorro.new(cliente: cliente, saldo_inicial: saldo_inicial)
      @cuentas << cuenta
      cuenta
    end

    def abrir_cuenta_inversion(cliente:, saldo_inicial: 0, tasa_riesgo: :moderada)
      cuenta = CuentaInversion.new(cliente: cliente, saldo_inicial: saldo_inicial, tasa_riesgo: tasa_riesgo)
      @cuentas << cuenta
      cuenta
    end

    def buscar_cuenta(numero)
      @cuentas.find { |c| c.numero == numero }
    end

    def cuentas_por_cliente(cliente)
      @cuentas.select { |c| c.cliente.id == cliente.id }
    end

    # Operaciones bancarias
    def depositar(numero_cuenta, monto, descripcion = "Depósito en ventanilla")
      cuenta = buscar_cuenta(numero_cuenta)
      return "Cuenta no encontrada" unless cuenta
      cuenta.depositar(monto, descripcion)
    end

    def retirar(numero_cuenta, monto, descripcion = "Retiro en ventanilla")
      cuenta = buscar_cuenta(numero_cuenta)
      return "Cuenta no encontrada" unless cuenta
      resultado = cuenta.retirar(monto, descripcion)
      resultado
    end

    def transferir(origen_num, destino_num, monto, descripcion = "Transferencia entre cuentas")
      origen = buscar_cuenta(origen_num)
      destino = buscar_cuenta(destino_num)
      return "Cuenta origen no encontrada" unless origen
      return "Cuenta destino no encontrada" unless destino

      @transferencia_service.transferir(origen, destino, monto, descripcion)
    end

    # Reportes
    def estado_cuenta(numero, formato = :texto)
      cuenta = buscar_cuenta(numero)
      return "Cuenta no encontrada" unless cuenta
      @reporte_service.generar_estado_cuenta(cuenta, formato)
    end

    def reporte_general
      @reporte_service.generar_reporte_banco(@cuentas)
    end

    def exportar_csv
      @reporte_service.exportar_csv(@cuentas)
    end

    # Intereses
    def aplicar_intereses_todas
      @cuentas.each do |cuenta|
        if cuenta.respond_to?(:aplicar_intereses)
          cuenta.aplicar_intereses
        elsif cuenta.respond_to?(:calcular_intereses)
          interes = cuenta.calcular_intereses
          if interes > 0
            cuenta.depositar(interes, "Interés automático")
          end
        end
      end
      "Intereses aplicados a todas las cuentas"
    end

    # Estadísticas
    def total_clientes
      @clientes.size
    end

    def total_cuentas
      @cuentas.size
    end

    def saldo_total
      @cuentas.sum(&:saldo)
    end

    def patrimonio_total
      @cuentas.sum do |c|
        c.respond_to?(:valor_total) ? c.valor_total : c.saldo
      end
    end

    def to_s
      "#{@nombre}: #{total_clientes} clientes, #{total_cuentas} cuentas, $#{'%.2f' % saldo_total}"
    end
  end

  class ServicioPrestamo
    def initialize(banco)
      @banco = banco
      @prestamos = []
    end

    def solicitar(cliente, monto, plazo_meses, tasa_anual = 0.15)
      # Validar capacidad de pago
      cuota = calcular_cuota(monto, tasa_anual, plazo_meses)

      prestamo = {
        id: @prestamos.size + 1,
        cliente: cliente,
        monto: monto,
        plazo_meses: plazo_meses,
        tasa_anual: tasa_anual,
        cuota_mensual: cuota.round(2),
        saldo_pendiente: monto,
        fecha_inicio: Time.now,
        pagos: []
      }
      @prestamos << prestamo

      # Abonar a la cuenta del cliente
      cuenta = @banco.cuentas_por_cliente(cliente).first
      cuenta.depositar(monto, "Préstamo aprobado ##{prestamo[:id]}") if cuenta

      prestamo
    end

    private

    def calcular_cuota(monto, tasa, meses)
      tasa_mensual = tasa / 12
      monto * (tasa_mensual * (1 + tasa_mensual) ** meses) / ((1 + tasa_mensual) ** meses - 1)
    end
  end

  10. GUÍA DE ESTILO Y BUENAS PRÁCTICAS

  Para mantener la calidad del código en este proyecto:

  1. Nomenclatura:
     - Clases: CamelCase (ClientePersonal, CuentaAhorro)
     - Métodos: snake_case (calcular_intereses, abrir_cuenta)
     - Variables de instancia: @snake_case (@saldo, @movimientos)
     - Constantes: MAYÚSCULAS (TASA_INTERES, COMISION_POR_RETIRO)

  2. Organización de archivos:
     - Un archivo por clase (o grupo pequeño de clases relacionadas)
     - Nombre del archivo en snake_case: cliente.rb, cuenta.rb
     - Separar en directorios: modelos/, servicios/, notificadores/

  3. Responsabilidad única:
     - Cuenta: solo gestión de cuenta
     - Cliente: solo datos del cliente
     - Servicios: operaciones que involucran múltiples objetos

  4. Encapsulación:
     - attr_reader por defecto, attr_accessor solo cuando necesario
     - Métodos privados para lógica interna
     - Proteger invariantes en setters y métodos de mutación

  5. Polimorfismo:
     - Duck typing para servicios intercambiables
     - Herencia para especialización de cuentas
     - Módulos para compartir comportamiento transversal

  6. Pruebas:
     - Una clase de prueba por clase del modelo
     - Probar casos felices y casos borde
     - Probar invariantes y validaciones

  PREGUNTAS FRECUENTES (FAQ)

  P: ¿Por qué usar herencia para los tipos de cuenta?
  R: Porque las cuentas comparten la mayoría del comportamiento
     (depositar, retirar, movimientos) y se diferencian en aspectos
     específicos (comisiones, intereses). La herencia permite reutilizar
     el código común y sobrescribir solo las diferencias.

  P: ¿Por qué TransferenciaService es una clase separada?
  R: Por el principio de responsabilidad única. Una transferencia
     involucra dos cuentas y posiblemente comisiones. Es una operación
     que no pertenece a ninguna cuenta individual. Al separarla, es
     más fácil de probar y mantener.

  P: ¿Cómo añado un nuevo tipo de cuenta sin modificar código existente?
  R: Crea una nueva clase que herede de Cuenta. Implementa los métodos
     específicos (calcular_intereses, calcular_comision_retiro). El
     Banco puede crear instancias de la nueva cuenta sin modificar
     ninguna clase existente (Principio Abierto/Cerrado).

  P: ¿Qué beneficios tiene usar módulos como Autenticable?
  R: Los módulos permiten compartir comportamiento entre clases que
     no están relacionadas por herencia. Autenticable podría usarse
     en UsuarioSistema, Cliente, o cualquier clase que necesite
     autenticación.

  P: ¿Cómo manejar errores en el sistema?
  R: El sistema usa ArgumentError para validaciones de parámetros y
     strings como mensajes de error para operaciones. En un sistema
     real, usarías excepciones personalizadas. Por ejemplo:
     class SaldoInsuficienteError < StandardError; end
