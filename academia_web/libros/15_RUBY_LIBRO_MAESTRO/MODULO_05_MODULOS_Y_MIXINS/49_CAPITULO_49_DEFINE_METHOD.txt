  BIENVENIDA
────────────────────────────────────────────────────────────────────────────────────

Bienvenido al Capítulo 49 del Ruby Master Book. En los capítulos anteriores
aprendiste a llamar métodos dinámicamente con `send`. Ahora darás el paso
siguiente: crear métodos DINÁMICAMENTE en tiempo de ejecución con
`define_method`, `class_eval`, e `instance_eval`.

La capacidad de definir métodos sobre la marcha es lo que hace que gemas como
ActiveRecord generen métodos como `find_by_nombre` o `where_edad` sin que
tengas que escribirlos uno por uno. Es también la base de los DSLs más
elegantes de Ruby, como RSpec, Sinatra y Rails routes.

En este capítulo aprenderás a usar `define_method` para crear métodos de
instancia, `define_singleton_method` para métodos singleton, `class_eval`
para abrir clases, e `instance_eval` para evaluar código en contexto de
objeto. También construiremos DSLs completos desde cero.

Ejemplo 1: `define_method` básico.

────────────────────────────────────────────────────────────────────────────────────

```ruby
# archivo: 49_01_define_method_basico.rb
# Creación básica de métodos con define_method

class Persona
  define_method :saludar do
    "¡Hola!"
  end

  define_method :despedirse do
    "¡Adiós!"
  end

  define_method :presentarse do |nombre|
    "Me llamo #{nombre}"
  end

  define_method :suma do |a, b|
    a + b
  end
end

p = Persona.new
puts p.saludar
puts p.despedirse
puts p.presentarse("Alice")
puts p.suma(5, 3)

# Los métodos definidos con define_method son métodos reales
puts "\nMétodos de Persona: #{Persona.instance_methods(false).inspect}"
```

Ejemplo 2: Generando métodos dinámicamente.

────────────────────────────────────────────────────────────────────────────────────

```ruby
# archivo: 49_02_generar_metodos.rb
# Generación masiva de métodos con define_method

class CalculadoraAvanzada
  OPERACIONES = {
    suma: :+,
    resta: :-,
    multiplicacion: :*,
    division: :/,
    potencia: :**,
    modulo: :%
  }

  OPERACIONES.each do |nombre, operador|
    define_method(nombre) do |a, b|
      a.send(operador, b)
    end
  end

  # También generar métodos con nombres compuestos
  [:suma, :resta, :multiplicacion].each do |op|
    define_method("#{op}_de_tres") do |a, b, c|
      send(op, send(op, a, b), c)
    end
  end
end

calc = CalculadoraAvanzada.new
puts "Suma: #{calc.suma(10, 5)}"
puts "Resta: #{calc.resta(10, 5)}"
puts "Multiplicación: #{calc.multiplicacion(10, 5)}"
puts "División: #{calc.division(10, 5)}"
puts "Potencia: #{calc.potencia(2, 8)}"
puts "Módulo: #{calc.modulo(10, 3)}"

puts "\nOperaciones de tres:"
puts "Suma de tres: #{calc.suma_de_tres(1, 2, 3)}"
puts "Resta de tres: #{calc.resta_de_tres(10, 2, 3)}"
puts "Multiplicación de tres: #{calc.multiplicacion_de_tres(2, 3, 4)}"
```

```ruby
# archivo: 49_03_define_method_closure.rb
# define_method captura el ámbito (closure)

class GeneradorDeMetodos
  def self.crear_metodos(*nombres)
    nombres.each do |nombre|
      define_method(nombre) do |*args|
        "Llamaste a #{nombre} con #{args.inspect}"
      end
    end
  end

  # Capturar variables del ámbito
  PREFIJOS = ["super", "mega", "ultra"]

  PREFIJOS.each do |prefijo|
    define_method("#{prefijo}_metodo") do |nombre|
      "#{prefijo.capitalize} #{nombre}"
    end
  end
end

g = GeneradorDeMetodos.new
GeneradorDeMetodos.crear_metodos(:foo, :bar, :baz)
puts g.foo(1, 2, 3)
puts g.bar("test")
puts g.baz

puts "\nMétodos con prefijo:"
puts g.super_metodo("Rápido")
puts g.mega_metodo("Poderoso")
puts g.ultra_metodo("Secreto")
```

```ruby
  end

  end
end

auto.define_singleton_method(:acelerar) do
  "Acelerando el automóvil..."
end

auto.define_singleton_method(:frenar) do |intensidad|
  "Frenando con intensidad #{intensidad}"
end

  "¡Haciendo wheelie!"
end

puts auto.acelerar
puts auto.frenar(5)

puts "\n#{moto.info}"
puts moto.hacer_wheelie

# La moto no tiene método acelerar
begin
  moto.acelerar
rescue => e
  puts "Error: #{e.message}"
end

# Singleton methods en objetos comunes
objeto = Object.new

  42
end
```

Ejemplo 5: `class_eval` para definir métodos.

────────────────────────────────────────────────────────────────────────────────────

```ruby
# archivo: 49_05_class_eval.rb
# Usando class_eval para abrir clases y definir métodos

class Producto
  def initialize(nombre, precio)
    @nombre = nombre
    @precio = precio
  end
end

# Abrir la clase y añadir métodos dinámicamente
Producto.class_eval do
  attr_reader :nombre, :precio

  define_method :to_s do
    "#{@nombre}: $#{@precio}"
  end

  define_method :descuento do |porcentaje|
    @precio * (1 - porcentaje)
  end

  define_method :== do |otro|
    @nombre == otro.nombre && @precio == otro.precio
  end
end

p1 = Producto.new("Laptop", 1200)
p2 = Producto.new("Mouse", 25)
p3 = Producto.new("Laptop", 1200)

puts p1.to_s
puts p1.descuento(0.1)
puts "#{p1} == #{p3}: #{p1 == p3}"
puts "#{p1} == #{p2}: #{p1 == p2}"
puts "Métodos de Producto: #{Producto.instance_methods(false).inspect}"
```

```ruby
# archivo: 49_06_instance_eval.rb
# instance_eval: evaluar código en el contexto de un objeto

class Configuracion
  def initialize
    @valores = {}
  end

  def set(clave, valor)
    @valores[clave] = valor
  end

  def get(clave)
    @valores[clave]
  end
end

config = Configuracion.new

# instance_eval permite acceder a variables privadas
config.instance_eval do
  @valores[:db_host] = "localhost"
  @valores[:db_port] = 5432
  @valores[:debug] = true
end

puts config.get(:db_host)
puts config.get(:db_port)

# Definir métodos singleton con instance_eval
config.instance_eval do
  def mostrar_config
    @valores.map { |k, v| "#{k}: #{v}" }.join("\n")
  end
end

puts "\nConfiguración:"
puts config.mostrar_config

# instance_eval con string (menos seguro)
config.instance_eval("@valores[:api_key] = 'secret123'")
puts "\nAPI Key: #{config.get(:api_key)}"
```

```ruby
# archivo: 49_07_atributos_dinamicos.rb
# Generando attr_accessor dinámicamente

module AtributosDinamicos
  def self.included(base)
    base.extend(ClassMethods)
  end

  module ClassMethods
    def atributo(*nombres)
      nombres.each do |nombre|
        define_method(nombre) do
          instance_variable_get("@#{nombre}")
        end

        define_method("#{nombre}=") do |valor|
          instance_variable_set("@#{nombre}", valor)
        end
      end
    end

    def atributo_con_default(nombre, default = nil)
      define_method(nombre) do
        if instance_variable_defined?("@#{nombre}")
          instance_variable_get("@#{nombre}")
        else
          default
        end
      end

      define_method("#{nombre}=") do |valor|
        instance_variable_set("@#{nombre}", valor)
      end
    end
  end
end

class Usuario
  include AtributosDinamicos

  atributo :nombre, :email, :edad, :activo
  atributo_con_default :rol, :usuario
  atributo_con_default :intentos, 0
end

u = Usuario.new
u.nombre = "Alice"
u.email = "alice@test.com"
u.edad = 30
u.activo = true

puts "Nombre: #{u.nombre}"
puts "Email: #{u.email}"
puts "Edad: #{u.edad}"
puts "Activo: #{u.activo}"
puts "Rol (default): #{u.rol}"
puts "Intentos (default): #{u.intentos}"

u.rol = :admin
puts "Rol (cambiado): #{u.rol}"
```

Ejemplo 8: Building a DSL con `define_method`.

────────────────────────────────────────────────────────────────────────────────────

```ruby
# archivo: 49_08_dsl_define_method.rb
# DSL para definir rutas web

module RutasDSL
  def self.extended(base)
    base.instance_variable_set(:@rutas, [])
  end

  def rutas
    @rutas || []
  end

  [:get, :post, :put, :delete, :patch].each do |metodo_http|
    define_method(metodo_http) do |path, **opciones, &bloque|
      @rutas ||= []
      @rutas << {
        metodo: metodo_http.upcase,
        path: path,
        opciones: opciones,
        bloque: bloque
      }

      # También definir método helper
      nombre_helper = "#{metodo_http}_#{path.gsub('/', '_').gsub(/[^a-zA-Z0-9_]/, '')}"
      define_method(nombre_helper) do
        "Ruta: #{metodo_http.upcase} #{path}"
      end
    end
  end

  def dibujar_rutas
    @rutas ||= []
    @rutas.each do |ruta|
      puts "  #{ruta[:metodo]} #{ruta[:path]}"
    end
  end
end

class MiApp
  extend RutasDSL

  get "/" do
    "Página principal"
  end

  get "/usuarios" do
    "Lista de usuarios"
  end

  post "/usuarios" do
    "Crear usuario"
  end

  get "/usuarios/:id" do
    "Mostrar usuario"
  end

  put "/usuarios/:id" do
    "Actualizar usuario"
  end

  delete "/usuarios/:id" do
    "Eliminar usuario"
  end

  get "/productos" do
    "Lista de productos"
  end

  post "/productos" do
    "Crear producto"
  end
end

puts "Rutas definidas:"
MiApp.dibujar_rutas

puts "\nHelpers:"
puts MiApp.new.get__usuarios
puts MiApp.new.delete__usuarios__id
```

Ejemplo 9: `define_method` con `method_missing`.

────────────────────────────────────────────────────────────────────────────────────

```ruby
# archivo: 49_09_method_missing_define_method.rb
# Combinando method_missing para descubrir y define_method para crear

module EncuentraMetodos
  def self.included(base)
    base.extend(ClassMethods)
  end

  module ClassMethods
    def encuentra_metodos!
      metodos_encontrados = []

      # Buscar en la base de datos simulada
      if respond_to?(:columnas)
        columnas.each do |col|
          metodos_encontrados << :"buscar_por_#{col}"
          metodos_encontrados << :"donde_#{col}"
        end
      end

      metodos_encontrados.each do |metodo|
        define_method(metodo) do |valor|
          nombre = metodo.to_s
          if nombre.start_with?('buscar_por_')
            campo = nombre.sub('buscar_por_', '')
            "#{self.class}.buscar_por_#{campo}(#{valor.inspect})"
          elsif nombre.start_with?('donde_')
            campo = nombre.sub('donde_', '')
            "#{self.class}.donde_#{campo}(#{valor.inspect})"
          end
        end
      end

      metodos_encontrados
    end

    def method_missing(name, *args, &block)
      nombre = name.to_s

      if nombre.start_with?('buscar_por_')
        campo = nombre.sub('buscar_por_', '')
        # Crear el método para futuras llamadas
        define_method(name) do |valor|
          "#{self.class}.buscar_por_#{campo}(#{valor.inspect}) (creado dinámicamente)"
        end
        # Ejecutar para esta llamada
        new.send(name, *args)
      elsif nombre.start_with?('donde_')
        campo = nombre.sub('donde_', '')
        define_method(name) do |valor|
          "#{self.class}.donde_#{campo}(#{valor.inspect}) (creado dinámicamente)"
        end
        new.send(name, *args)
      else
        super
      end
    end

    def respond_to_missing?(name, include_private = false)
      nombre = name.to_s
      nombre.start_with?('buscar_por_') || nombre.start_with?('donde_') || super
    end
  end
end

class UsuarioDinamico
  include EncuentraMetodos

  def self.columnas
    [:nombre, :email, :edad, :activo, :rol, :ciudad]
  end
end

puts "Métodos antes de encuentra_metodos!:"
puts "  #{UsuarioDinamico.instance_methods(false).length} métodos"

metodos = UsuarioDinamico.encuentra_metodos!
puts "Métodos encontrados: #{metodos.length}"
puts "  #{metodos.inspect}"

u = UsuarioDinamico.new
puts "\nUsando métodos generados:"
puts u.buscar_por_nombre("Alice")
puts u.donde_edad(25)
puts u.buscar_por_email("test@test.com")

puts "\nMétodo no pre-generado (se crea al vuelo):"
puts u.buscar_por_ciudad("Madrid")

puts "\nTotal métodos de instancia: #{UsuarioDinamico.instance_methods(false).length}"
```

Ejemplo 10: DSL de validación con `define_method`.

────────────────────────────────────────────────────────────────────────────────────

```ruby
# archivo: 49_10_dsl_validacion.rb
# DSL de validación con define_method

module Validaciones
  def self.included(base)
    base.extend(ClassMethods)
  end

  module ClassMethods
    def validaciones
      @validaciones || []
    end

    def validate(nombre, &bloque)
      @validaciones ||= []
      @validaciones << { nombre: nombre, bloque: bloque }

      define_method("validar_#{nombre}") do
        bloque.call(self)
      end
    end

    def validates_presence_of(*campos)
      campos.each do |campo|
        validate :"#{campo}_presente" do |obj|
          valor = obj.send(campo)
          if valor.nil? || (valor.respond_to?(:empty?) && valor.empty?)
            obj.errores << "#{campo} no puede estar vacío"
          end
        end
      end
    end

    def validates_length_of(campo, min: nil, max: nil)
      validate :"#{campo}_longitud" do |obj|
        valor = obj.send(campo)
        if valor
          if min && valor.length < min
            obj.errores << "#{campo} debe tener al menos #{min} caracteres"
          end
          if max && valor.length > max
            obj.errores << "#{campo} debe tener máximo #{max} caracteres"
          end
        end
      end
    end

    def validates_format_of(campo, con:)
      validate :"#{campo}_formato" do |obj|
        valor = obj.send(campo)
        if valor && !valor.match?(con)
          obj.errores << "#{campo} no tiene el formato correcto"
        end
      end
    end

    def validates_numericality_of(campo, **opciones)
      validate :"#{campo}_numerico" do |obj|
        valor = obj.send(campo)
        unless valor.is_a?(Numeric)
          obj.errores << "#{campo} debe ser numérico"
        end
        if opciones[:greater_than] && valor && valor <= opciones[:greater_than]
          obj.errores << "#{campo} debe ser mayor que #{opciones[:greater_than]}"
        end
        if opciones[:less_than] && valor && valor >= opciones[:less_than]
          obj.errores << "#{campo} debe ser menor que #{opciones[:less_than]}"
        end
      end
    end
  end

  def errores
    @errores || []
  end

  def valido?
    @errores = []
    self.class.validaciones.each do |v|
      send("validar_#{v[:nombre]}")
    end
    @errores.empty?
  end

  def reporte_errores
    valido? ? "Válido" : errores.map.with_index { |e, i| "#{i + 1}. #{e}" }.join("\n")
  end
end

class Formulario
  include Validaciones

  attr_accessor :nombre, :email, :edad, :password

  def initialize(atributos = {})
    @nombre = atributos[:nombre]
    @email = atributos[:email]
    @edad = atributos[:edad]
    @password = atributos[:password]
  end

  validates_presence_of :nombre, :email, :password
  validates_length_of :nombre, min: 2, max: 50
  validates_length_of :password, min: 8
  validates_format_of :email, con: /\A[\w+\-.]+@[a-z\d\-]+(\.[a-z\d\-]+)*\.[a-z]+\z/i
  validates_numericality_of :edad, greater_than: 0, less_than: 150

  validate :password_tiene_mayuscula do |obj|
    if obj.password && !obj.password.match?(/[A-Z]/)
      obj.errores << "password debe contener al menos una mayúscula"
    end
  end

  validate :password_tiene_numero do |obj|
    if obj.password && !obj.password.match?(/[0-9]/)
      obj.errores << "password debe contener al menos un número"
    end
  end
end

f1 = Formulario.new(
  nombre: "Ana",
  email: "ana@test.com",
  edad: 28,
  password: "Secure1A"
)
puts "Formulario 1: #{f1.reporte_errores}"

f2 = Formulario.new(
  nombre: "A",
  email: "invalido",
  edad: -5,
  password: "corto"
)
puts "\nFormulario 2:\n#{f2.reporte_errores}"
```

Ejemplo 11: Generación de métodos para consultas SQL.

────────────────────────────────────────────────────────────────────────────────────

```ruby
# archivo: 49_11_sql_define_method.rb
# Generación de métodos de consulta SQL

class ModeloBase
  def self.inherited(subclass)
    subclass.instance_variable_set(:@tabla, subclass.name.downcase + "s")
    subclass.instance_variable_set(:@columnas, [])
  end

  class << self
    attr_accessor :tabla, :columnas

    def columna(nombre, tipo = nil)
      @columnas << { nombre: nombre, tipo: tipo }
      attr_accessor nombre
    end

    def generar_metodos!
      @columnas.each do |col|
        nombre = col[:nombre]

        # buscar_por_nombre(valor)
        define_method("buscar_por_#{nombre}") do |valor|
          "SELECT * FROM #{@tabla} WHERE #{nombre} = '#{valor}'"
        end

        # donde_nombre(valor)
        define_method("donde_#{nombre}") do |valor|
          "SELECT * FROM #{@tabla} WHERE #{nombre} = '#{valor}'"
        end

        # contar_por_nombre
        define_method("contar_por_#{nombre}") do
          "SELECT #{nombre}, COUNT(*) FROM #{@tabla} GROUP BY #{nombre}"
        end

        # ordenar_por_nombre(direccion = :asc)
        define_method("ordenar_por_#{nombre}") do |direccion = :asc|
          "SELECT * FROM #{@tabla} ORDER BY #{nombre} #{direccion.upcase}"
        end

        # promedio_de_nombre
        if col[:tipo] == :numerico || col[:tipo] == :integer
          define_method("promedio_de_#{nombre}") do
            "SELECT AVG(#{nombre}) FROM #{@tabla}"
          end

          define_method("suma_de_#{nombre}") do
            "SELECT SUM(#{nombre}) FROM #{@tabla}"
          end

          define_method("maximo_de_#{nombre}") do
            "SELECT MAX(#{nombre}) FROM #{@tabla}"
          end

          define_method("minimo_de_#{nombre}") do
            "SELECT MIN(#{nombre}) FROM #{@tabla}"
          end
        end
      end

      # Método genérico buscar_por_cualquier_campo
      define_method(:buscar_por) do |campo, valor|
        "SELECT * FROM #{self.class.tabla} WHERE #{campo} = '#{valor}'"
      end
    end
  end

  def initialize(atributos = {})
    atributos.each { |k, v| send("#{k}=", v) if respond_to?("#{k}=") }
  end
end

class UsuarioModelo < ModeloBase
  columna :id, :integer
  columna :nombre, :string
  columna :email, :string
  columna :edad, :integer
  columna :activo, :boolean
  columna :salario, :numerico

  generar_metodos!
end

u = UsuarioModelo.new
puts "Buscar por nombre: #{u.buscar_por_nombre('Alice')}"
puts "Donde edad: #{u.donde_edad(25)}"
puts "Ordenar por nombre: #{u.ordenar_por_nombre(:desc)}"
```

```ruby
module SerializacionDinamica
  def self.included(base)
    base.extend(ClassMethods)
  end

  module ClassMethods
    def serializar(*campos)
      define_method(:to_h) do
        campos.each_with_object({}) do |campo, hash|
          hash[campo] = send(campo)
        end
      end

      campos.each do |campo|
        define_method("solo_#{campo}") do
          { campo => send(campo) }
        end
      end
    end

    def exportar_como(*formatos)
      formatos.each do |formato|
        define_method("exportar_como_#{formato}") do |**opciones|
          case formato
          when :json
            require 'json' unless defined?(JSON)
            JSON.generate(to_h, opciones)
          when :yaml
            require 'yaml' unless defined?(YAML)
            YAML.dump(to_h)
          when :xml
            to_h.map { |k, v| "  <#{k}>#{v}</#{k}>" }.join("\n")
          when :csv
            "#{to_h.keys.join(',')}\n#{to_h.values.join(',')}"
          else
            raise "Formato no soportado: #{formato}"
          end
        end
      end
    end
  end

  def to_s
    to_h.map { |k, v| "#{k}: #{v}" }.join(", ")
  end

  def inspect
    "#<#{self.class.name} #{to_h.map { |k, v| "#{k}=#{v.inspect}" }.join(', ')}>"
  end
end
```

```ruby
  def decorar_metodo(metodo, decorador)

  end
```

Ejemplo 16: `define_method` para callbacks.

────────────────────────────────────────────────────────────────────────────────────

```ruby
# archivo: 49_16_callbacks_define_method.rb
# Callbacks con define_method

module Callbacks
  def self.included(base)
    base.extend(ClassMethods)
  end

  module ClassMethods
    def callbacks
      @callbacks ||= { before: [], after: [], around: [] }
    end

    def before_save(*metodos, &bloque)
      callbacks[:before] << (metodos.first || bloque)
    end

    def after_save(*metodos, &bloque)
      callbacks[:after] << (metodos.first || bloque)
    end

    def around_save(*metodos, &bloque)
      callbacks[:around] << (metodos.first || bloque)
    end

    def define_callback_method(original)
      nombre_original = :"#{original}_without_callbacks"
      alias_method nombre_original, original

      define_method(original) do
        ejecutar_callbacks(:before, original)

        resultado = if self.class.callbacks[:around].any?
                      ejecutar_around_callbacks(original) {
                        send(nombre_original)
                      }
                    else
                      send(nombre_original)
                    end

        ejecutar_callbacks(:after, original)
        resultado
      end
    end
  end

  def ejecutar_callbacks(tipo, metodo)
    self.class.callbacks[tipo].each do |cb|
      case cb
      when Symbol then send(cb)
      when Proc then instance_exec(metodo, &cb)
      end
    end
  end

  def ejecutar_around_callbacks(metodo)
    cadena = self.class.callbacks[:around].reverse
    ejecutor = -> { yield }

    cadena.each do |cb|
      previo = ejecutor
      ejecutor = case cb
                 when Symbol then -> { send(cb, metodo) { previo.call } }
                 when Proc then -> { instance_exec(metodo, previo, &cb) }
                 end
    end

    ejecutor.call
  end
end

class Articulo
  include Callbacks

  attr_reader :titulo, :estado, :creado_en

  def initialize(titulo)
    @titulo = titulo

  end

  after_save :actualizar_estado
  around_save :medir_tiempo

  def save
    puts "  Guardando artículo '#{@titulo}'..."
    "Artículo guardado"
  end

  define_callback_method :save

  private

  def validar_titulo
    if @titulo.nil? || @titulo.empty?
      raise "El título no puede estar vacío"
    end
    puts "  Callback: Título validado"
  end

  end

    puts "  Callback: Estado actualizado a #{@estado}"
  end

  def medir_tiempo(metodo)
    inicio = Time.now
    puts "  Around callback: Inicio"
    resultado = metodo.call
    duracion = Time.now - inicio
    puts "  Around callback: Fin (#{format('%.4f', duracion)}s)"

  end
```

```ruby
# DSL para definir reglas de negocio

module ReglasNegocio
  def self.included(base)
    base.extend(ClassMethods)
  end

  module ClassMethods
    def reglas
      @reglas || []
    end

    def cuando(condicion, &bloque)
      @reglas ||= []
      @reglas << { condicion: condicion, accion: bloque }
    end

    def entonces(accion, &bloque)
      @reglas ||= []
      @reglas.last[:accion] = bloque || accion
    end

    def si(condicion)
      @reglas ||= []
      @reglas << { condicion: condicion }
      self
    end

    def y(condicion)
      @reglas ||= []
      if @reglas.last && @reglas.last[:condicion]
        cond_actual = @reglas.last[:condicion]
        @reglas.last[:condicion] = ->(obj) { cond_actual.call(obj) && condicion.call(obj) }
      end
      self
    end

    def o(condicion)
      @reglas ||= []
      if @reglas.last && @reglas.last[:condicion]
        cond_actual = @reglas.last[:condicion]
        @reglas.last[:condicion] = ->(obj) { cond_actual.call(obj) || condicion.call(obj) }
      end
      self
    end
  end

  def evaluar_reglas
    self.class.reglas.each do |regla|
      if regla[:condicion].call(self)
        resultado = instance_exec(&regla[:accion]) if regla[:accion]
        return resultado if resultado
      end
    end
    nil
  end
end
```

```ruby
# archivo: 49_18_busqueda_dinamica.rb
# Generación de métodos de búsqueda tipo ActiveRecord

class Repositorio
  def self.inherited(subclass)
    subclass.instance_variable_set(:@registros, [])
    subclass.instance_variable_set(:@indices, {})
  end

  class << self
    def registros
      @registros || []
    end

    def indice_para(campo)
      @indices ||= {}
      @indices[campo] ||= {}
    end

    def <<(registro)
      @registros ||= []
      @registros << registro
      indexar_registro(registro)
    end

    def indexar_registro(registro)
      @columnas.each do |col|
        idx = indice_para(col)
        valor = registro.send(col)
        idx[valor] ||= []
        idx[valor] << registro
      end
    end

    def generar_metodos_busqueda!
      @columnas.each do |col|
        nombre = col

        # buscar_por_nombre
        define_method("buscar_por_#{nombre}") do |valor|
          idx = self.class.indice_para(nombre)
          idx[valor] || []
        end

        # primero_por_nombre
        define_method("primero_por_#{nombre}") do |valor|
          buscar_por_nombre(valor).first
        end

        # todos_donde_nombre
        define_method("todos_donde_#{nombre}") do |valor|
          self.class.registros.select { |r| r.send(nombre) == valor }
        end

        # existe_por_nombre?
        define_method("existe_por_#{nombre}?") do |valor|
          !buscar_por_nombre(valor).empty?
        end

        # contar_por_nombre
        define_method("contar_por_#{nombre}") do |valor|
          buscar_por_nombre(valor).length
        end
      end

      # buscar_por (genérico)
      define_method(:buscar_por) do |campo, valor|
        idx = self.class.indice_para(campo)
        idx[valor] || []
      end
    end
  end

  def initialize(atributos = {})
    @atributos = atributos
    atributos.each { |k, v| instance_variable_set("@#{k}", v) }
    self.class << self
  end

  def method_missing(name, *args, &block)
    nombre = name.to_s
    if instance_variable_defined?("@#{nombre}")
      instance_variable_get("@#{nombre}")
    elsif @atributos.key?(nombre)
      @atributos[nombre]
    else
      super
    end
  end

  def respond_to_missing?(name, include_private = false)
    nombre = name.to_s
    instance_variable_defined?("@#{nombre}") || @atributos.key?(nombre) || super
  end

  def to_s
    atrs = @atributos.map { |k, v| "#{k}=#{v}" }.join(', ')
    "#{self.class.name}(#{atrs})"
  end
end

  @columnas = [:nombre, :categoria, :precio, :stock, :activo]
  generar_metodos_busqueda!
end
```

```ruby
    end
  end

  module ClassMethods
    def define_campo(nombre, tipo = nil)
      define_method(nombre) do
        instance_variable_get("@#{nombre}")
      end

      define_method("#{nombre}=") do |valor|
        if tipo && !valor.nil? && !valor.is_a?(tipo)
          raise TypeError, "#{nombre} debe ser #{tipo}, se recibió #{valor.class}"
        end
        instance_variable_set("@#{nombre}", valor)
      end
    end

    def define_campos(*campos)
      campos.each { |c| define_campo(c) }
    end

    def define_campos_con_tipos(tipos = {})
      tipos.each { |nombre, tipo| define_campo(nombre, tipo) }
    end
  end
end

  define_campos_con_tipos(

  )

  def initialize(atributos = {})
```

```ruby
# Mini ORM completo usando define_method

class MiniORM
  def self.inherited(subclass)
    subclass.instance_variable_set(:@tabla, subclass.name.downcase + "s")
    subclass.instance_variable_set(:@registros, [])
    subclass.instance_variable_set(:@columnas, [])
    subclass.instance_variable_set(:@relaciones, {})
  end

  class << self
    attr_accessor :tabla, :columnas, :relaciones

    def field(nombre, tipo = nil, **opciones)
      @columnas ||= []
      @columnas << { nombre: nombre, tipo: tipo, opciones: opciones }
      attr_accessor nombre

      if opciones[:default]
        define_method(nombre) do
          instance_variable_defined?("@#{nombre}") ? instance_variable_get("@#{nombre}") : opciones[:default]
        end
      end
    end

    def belongs_to(relacion, **opciones)
      @relaciones ||= {}
      @relaciones[relacion] = { tipo: :belongs_to, opciones: opciones }
      clase = opciones[:class_name] || relacion.to_s.capitalize

      define_method(relacion) do
        fk = opciones[:foreign_key] || :"#{relacion}_id"
        id_valor = send(fk)
        Object.const_get(clase).buscar_por_id(id_valor) if id_valor
      end
    end

    def has_many(relacion, **opciones)
      @relaciones ||= {}
      @relaciones[relacion] = { tipo: :has_many, opciones: opciones }
      clase = opciones[:class_name] || relacion.to_s.chop.capitalize
      fk = opciones[:foreign_key] || :"#{name.downcase}_id"

      define_method(relacion) do
        Object.const_get(clase).donde(fk => @id) rescue []
      end

      define_method("agregar_#{relacion.to_s.singularize}") do |objeto|
        objeto.send("#{fk}=", @id)
        objeto
      end
    end

    def crear(atributos)
      nuevo = new(atributos)
      @registros << nuevo
      nuevo
    end

    def todos
      @registros || []
    end

    def count
      todos.length
    end

    def primero
      todos.first
    end

    def ultimo
      todos.last
    end

    def buscar_por_id(id)
      todos.find { |r| r.id == id }
    end

    def donde(condiciones = {})
      todos.select do |r|
        condiciones.all? { |campo, valor| r.send(campo) == valor }
      end
    end

    def method_missing(name, *args, &block)
      nombre = name.to_s

      if nombre.start_with?('buscar_por_')
        campo = nombre.sub('buscar_por_', '')
        valor = args.first
        todos.find { |r| r.send(campo) == valor }
      elsif nombre.start_with?('donde_')
        campo = nombre.sub('donde_', '')
        valor = args.first
        todos.select { |r| r.send(campo) == valor }
      elsif nombre.start_with?('ordenar_por_')
        campo = nombre.sub('ordenar_por_', '')
        direccion = args.first || :asc
        todos.sort { |a, b| direccion == :asc ? a.send(campo) <=> b.send(campo) : b.send(campo) <=> a.send(campo) }
      else
        super
      end
    end

    def respond_to_missing?(name, include_private = false)
      nombre = name.to_s
      nombre.start_with?('buscar_por_') || nombre.start_with?('donde_') ||
        nombre.start_with?('ordenar_por_') || super
    end
  end

  def initialize(atributos = {})
    @id = SecureRandom.uuid
    atributos.each { |k, v| send("#{k}=", v) if respond_to?("#{k}=") }
  end

  def [](campo)
    send(campo) rescue instance_variable_get("@#{campo}")
  end

  def []=(campo, valor)
    send("#{campo}=", valor) rescue instance_variable_set("@#{campo}", valor)
  end

  def atributos
    self.class.columnas.each_with_object({}) do |col, h|
      h[col[:nombre]] = send(col[:nombre])
    end.merge(id: @id)
  end

  def to_s
    "#{self.class.name}(#{atributos.map { |k, v| "#{k}=#{v}" }.join(', ')})"
  end

  def guardar
    self.class.todos << self unless self.class.todos.include?(self)
    true
  end

  def eliminar
    self.class.todos.delete(self)
    true
  end
end
```

  RESUMEN DEL CAPÍTULO
────────────────────────────────────────────────────────────────────────────────────

`define_method` es la herramienta definitiva para generar métodos en Ruby:

  1. `define_method(:nombre, &bloque)` define métodos de instancia.
  2. `define_singleton_method` define métodos singleton.
  3. `class_eval` abre una clase para definir métodos.
  4. `instance_eval` evalúa código en contexto de objeto.
  5. Los closures capturan el ámbito donde se crean.
  6. Combinado con `method_missing`, puedes cachear métodos.
  7. Es la base de DSLs, ORMs, y generación de código.
  8. `def` es estático, `define_method` es dinámico.

────────────────────────────────────────────────────────────────────────────────────
  FAQ — PREGUNTAS FRECUENTES
────────────────────────────────────────────────────────────────────────────────────

P: ¿`define_method` vs `def` — cuál usar?
R: `def` es más rápido y legible. `define_method` es para cuando necesitas
   definir métodos dinámicamente basado en datos.

P: ¿Puedo usar `define_method` con bloques que reciben argumentos?
R: Sí. Los parámetros del bloque son los parámetros del método.

P: ¿`define_method` funciona con métodos privados?
R: Sí, usa `private` después o usa `module_function`.

P: ¿Puedo redefinir un método existente con `define_method`?
R: Sí. `define_method` sobrescribirá cualquier método existente.

P: ¿Cuál es la diferencia entre `class_eval` y `instance_eval`?
R: `class_eval` se ejecuta en el contexto de la clase (define métodos de
   instancia). `instance_eval` se ejecuta en el contexto de la instancia
   (define métodos singleton).

P: ¿`define_method` captura variables del ámbito?
R: Sí, como cualquier bloque, captura el ámbito donde fue creado (closure).
