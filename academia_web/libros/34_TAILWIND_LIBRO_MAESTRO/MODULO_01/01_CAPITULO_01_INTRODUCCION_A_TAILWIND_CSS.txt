CONTENIDO DEL CAPÍTULO
===============================================================================

Tailwind CSS: framework utility-first. Clases atómicas.

npm install tailwindcss. PostCSS plugin. JIT compiler.

Ventajas: no nombres, tamaño fijo, consistente.

EJEMPLO DE CÓDIGO
----------------------------------------

npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
