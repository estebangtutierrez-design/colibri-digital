CONTENIDO DEL CAPÍTULO
===============================================================================

1. VARIABLES

nombre=valor (sin espacios). echo $nombre muestra valor. readonly nombre

hace variable de solo lectura. unset nombre elimina variable.



2. TIPOS

Por defecto todo son strings. declare -i num=10 entero. declare -a arr=()

array. declare -A dict=() diccionario asociativo (bash 4+).



3. EXPANSIÓN

$variable o ${variable}. ${#variable} longitud. ${variable:0:3} substring.

${variable//a/b} reemplazar. ${variable:-default} valor por defecto.

$(comando) ejecuta comando. $(( expr )) aritmética.



4. VARIABLES ESPECIALES

$0 nombre del script. $1-$9 argumentos. $# número de argumentos.

$@ todos los argumentos. $? código de salida último comando. $$ PID del script.

EJEMPLO DE CÓDIGO
----------------------------------------

#!/bin/bash
nombre="Mundo"
echo "Hola, $nombre!"
echo "Script: $0"
echo "Args: $#"
echo "PID: $$"
echo "$(date): Inicio del script"
