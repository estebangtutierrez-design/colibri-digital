CONTENIDO DEL CAPÍTULO
===============================================================================

1. IF/ELIF/ELSE

if [ condicion ]; then comandos; fi. elif y else para más ramas.

fi cierra el bloque. [ y ] requieren espacios alrededor.



2. TEST ([ ])

[ -f archivo ] existe y es archivo. [ -d dir ] existe y es directorio.

[ -z string ] string vacío. [ -n string ] string no vacío.

[ $a -eq $b ] igualdad numérica. [ $a = $b ] igualdad string.



3. OPERADORES

Numéricos: -eq, -ne, -lt, -le, -gt, -ge. Strings: =, !=, <, >.

Archivos: -e existe, -f archivo, -d directorio, -r lectura, -w escritura.

A: &&, O: ||, Negación: !



4. CASE

case $var in patron1) comandos;; patron2) comandos;; *) default;; esac.

Patrones con | para múltiples. * para cualquier otro valor.

EJEMPLO DE CÓDIGO
----------------------------------------

#!/bin/bash
if [ -f "$1" ]; then
  echo "$1 existe y es archivo"
elif [ -d "$1" ]; then
  echo "$1 es un directorio"
else
  echo "$1 no existe"
fi

case $1 in
  *.txt) echo "Archivo texto" ;;
  *.sh) echo "Script bash" ;;
  *) echo "Otro tipo" ;;
esac
