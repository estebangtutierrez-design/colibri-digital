1. FOR

for var in lista; do comandos; done. for i in {1..5} rango.

for archivo in *.txt archivos por patrón. for ((i=0; i<5; i++)) estilo C.



2. WHILE

while [ condicion ]; do comandos; done. while read linea; do done < archivo

lee archivo línea por línea. while true bucle infinito (break para salir).



3. UNTIL

until [ condicion ]; do comandos; done. Ejecuta mientras condición sea

falsa. Útil para esperar a que algo ocurra (ej: puerto disponible).



4. CONTROL

break sale del bucle. continue salta a siguiente iteración.

exit sale del script con código (0 éxito, 1 error).

EJEMPLO DE CÓDIGO
----------------------------------------

#!/bin/bash
for i in {1..5}; do
  echo "Iteracion $i"
done

while IFS= read -r linea; do
  echo "Linea: $linea"
done < /etc/hostname

for archivo in *.log; do
  [ -f "$archivo" ] && echo "Procesando $archivo"
done
