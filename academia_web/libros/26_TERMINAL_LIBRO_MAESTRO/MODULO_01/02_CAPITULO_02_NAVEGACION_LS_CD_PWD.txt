CONTENIDO DEL CAPÍTULO
===============================================================================

1. PWD (Print Working Directory)

Muestra la ruta absoluta del directorio actual. Esencial para orientarse.



2. LS (List)

ls lista archivos. ls -l formato detallado. ls -a muestra ocultos.

ls -la combinación. ls -lh tamaños legibles. ls -R recursivo.



3. CD (Change Directory)

cd /ruta cambia a ruta absoluta. cd carpeta cambia a subcarpeta.

cd .. sube un nivel. cd ~ va al home. cd - va al directorio anterior.



4. RUTAS

Absoluta: desde raíz /. Relativa: desde posición actual. . es directorio

actual. .. es directorio padre. ~ es home del usuario.

EJEMPLO DE CÓDIGO
----------------------------------------

$ pwd
$ ls -la
$ cd /var/log
$ ls -lh
$ cd ~
