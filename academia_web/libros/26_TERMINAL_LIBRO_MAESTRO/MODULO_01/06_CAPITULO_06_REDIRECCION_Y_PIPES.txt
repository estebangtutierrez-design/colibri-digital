CONTENIDO DEL CAPÍTULO
===============================================================================

1. REDIRECCIÓN > y >>

comando > archivo escribe salida (sobrescribe). comando >> archivo agrega.

2> archivo redirige errores. &> archivo redirige todo.



2. PIPES |

comando1 | comando2 pasa salida de uno a entrada de otro. ls -la | less

navega lista larga. ps aux | grep nginx busca proceso.



3. TEES

comando | tee archivo muestra y guarda. tee -a archivo agrega en vez de

sobrescribir. Útil para logs de scripts.



4. EJEMPLOS PRÁCTICOS

cat archivo | wc -l cuenta líneas. history | grep git busca comandos Git.

ls -la | sort -k5 -n ordena por tamaño. dmesg | tail -20 últimas líneas.

EJEMPLO DE CÓDIGO
----------------------------------------

$ ls -la | wc -l
$ history | grep docker
$ dmesg | grep error | tail -5
$ ./script.sh 2>&1 | tee log.txt
