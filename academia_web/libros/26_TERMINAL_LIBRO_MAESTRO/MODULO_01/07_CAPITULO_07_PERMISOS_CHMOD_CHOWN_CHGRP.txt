CONTENIDO DEL CAPÍTULO
===============================================================================

1. PERMISOS EN LINUX

rwx: read (4), write (2), execute (1). Tres grupos: usuario (u), grupo (g),

otros (o). ls -l muestra: -rwxr-xr-x (primer carácter: - archivo, d directorio).



2. CHMOD

chmod 755 archivo: u=rwx, g=rx, o=rx. chmod u+x archivo agrega ejecución.

chmod -R 755 carpeta recursivo. chmod g-w archivo quita escritura al grupo.



3. CHOWN

chown usuario:grupo archivo cambia propietario y grupo. chown usuario archivo

solo propietario. chown -R recursivo.



4. CHGRP

chgrp grupo archivo cambia solo el grupo. Alternativa: chown :grupo archivo.

Necesitas permisos de superusuario para cambiar propietario.

EJEMPLO DE CÓDIGO
----------------------------------------

$ chmod 755 script.sh
$ chmod u+x script.sh
$ chown www-data:www-data index.html
$ chmod -R 644 public/
