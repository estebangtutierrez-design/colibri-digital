CONTENIDO DEL CAPÍTULO
===============================================================================

1. ¿QUÉ ES LA TERMINAL?

La terminal es una interfaz de texto para interactuar con el sistema operativo.

A diferencia de las interfaces gráficas (GUI), la terminal permite ejecutar

comandos directamente, automatizar tareas y acceder a funcionalidades avanzadas.



2. SHELL Y TERMINAL

La terminal es el programa que muestra la ventana. El shell es el intérprete

de comandos (bash, zsh, fish). El shell procesa los comandos y ejecuta programas.



3. PRIMEROS COMANDOS

echo 'Hola Mundo' imprime texto. whoami muestra usuario actual. pwd muestra

directorio actual. date muestra fecha y hora. clear limpia la pantalla.



4. PROMPT

El prompt ($ para usuario normal, # para root) indica que el shell está listo.

Muestra: usuario@host:directorio$. Personalizable con PS1 en .bashrc.

EJEMPLO DE CÓDIGO
----------------------------------------

$ echo 'Hola Mundo'
$ whoami
$ pwd
$ date
$ clear
