CONTENIDO DEL CAPÍTULO
===============================================================================

1. FIND

find /ruta -name '*.txt' busca por nombre. find . -type f solo archivos.

find . -type d solo directorios. find . -mtime -7 modificados en 7 días.

find . -size +10M archivos mayores a 10MB.



2. LOCATE

locate archivo busca en base de datos indexada (más rápido que find).

updatedb actualiza la base de datos. Menos flexible que find.



3. GREP BÁSICO

grep 'patron' archivo busca texto. grep -i ignora mayúsculas.

grep -r patrón búsqueda recursiva. grep -n muestra números de línea.

grep -c cuenta ocurrencias. grep -v invertido (líneas que NO coinciden).



4. GREP AVANZADO

grep -E 'patron1|patron2' múltiples patrones. grep '^inicio' líneas que

empiezan. grep 'fin$' líneas que terminan. grep '\d\{3\}' dígitos.

EJEMPLO DE CÓDIGO
----------------------------------------

$ find /home -name '*.py'
$ locate nginx.conf
$ grep -rn 'TODO' src/
$ grep -E 'ERROR|FATAL' app.log
