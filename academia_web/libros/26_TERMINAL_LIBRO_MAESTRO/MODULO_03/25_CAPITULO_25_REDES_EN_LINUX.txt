CONTENIDO DEL CAPÍTULO
===============================================================================

1. IP Y NETWORK CONFIG

ip addr show interfaces. ip link set eth0 up/down. ip route show tabla de

routing. ip neigh show tabla ARP. ifconfig obsoleto, usar ip.



2. SSH

ssh usuario@host conexión. scp archivo usuario@host:/ruta copia segura.

rsync -avz origen/ destino/ sincronización. ssh-keygen genera claves.

~/.ssh/config para configuración de hosts.



3. DNS

/etc/resolv.conf nameservers. /etc/hosts resolución local. host dominio

o nslookup dominio consultan DNS. dig dominio consulta detallada.



4. FIREWALL

ufw: ufw enable/disable, ufw allow 80/tcp, ufw status. iptables avanzado.

nftables reemplazo moderno de iptables. FirewallD en Fedora/RHEL.

EJEMPLO DE CÓDIGO
----------------------------------------

$ ip addr show
$ ip route show
$ ssh -i ~/.ssh/id_rsa user@server.com
$ rsync -avz --progress src/ user@server:/dest/
$ ufw allow 22/tcp
$ ufw status verbose
