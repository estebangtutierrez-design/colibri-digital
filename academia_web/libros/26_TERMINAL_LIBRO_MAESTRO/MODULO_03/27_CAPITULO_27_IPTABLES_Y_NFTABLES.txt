CONTENIDO DEL CAPÍTULO
===============================================================================

1. IPTABLES

Tablas: filter (default), nat, mangle. Cadenas: INPUT, OUTPUT, FORWARD.

Reglas: iptables -A INPUT -p tcp --dport 80 -j ACCEPT. -A agrega, -D elimina.



2. REGLAS BÁSICAS

iptables -P INPUT DROP política por defecto. -A INPUT -m state --state

ESTABLISHED,RELATED -j ACCEPT permite conexiones establecidas.

-A INPUT -p tcp --dport 22 -j ACCEPT permite SSH.



3. NFTABLES

Reemplazo moderno de iptables. Sintaxis más limpia. Rendimiento superior.

nft list ruleset muestra todas las reglas. Archivos .nftable para config.



4. PERSISTENCIA

iptables-save > /etc/iptables/rules.v4 guarda reglas. iptables-restore <

archivo restaura. netfilter-persistent en Debian/Ubuntu.

EJEMPLO DE CÓDIGO
----------------------------------------

# iptables
iptables -P INPUT DROP
iptables -A INPUT -m conntrack --ctstate ESTABLISHED,RELATED -j ACCEPT
iptables -A INPUT -p tcp --dport 22 -j ACCEPT
iptables -A INPUT -p tcp --dport 80 -j ACCEPT
iptables-save > /etc/iptables/rules.v4

# nftables - /etc/nftables.conf
table inet filter {
  chain input { type filter hook input priority 0; policy drop;
    ct state established,related accept
    tcp dport {22,80,443} accept
    iif lo accept
  }
}
