CONTENIDO DEL CAPÍTULO
===============================================================================

1. SYSLOG

Sistema de logging tradicional. Archivos en /var/log/. syslog, auth.log,

kern.log, dmesg. Configuración en /etc/rsyslog.conf.



2. JOURNALCTL

journalctl logs del sistema. journalctl -b arranque actual. journalctl -k

solo kernel. journalctl -p err solo errores. journalctl --no-pager sin paginación.



3. LOGROTATE

Rotación de logs. /etc/logrotate.conf y /etc/logrotate.d/. Compresión,

tamaño máximo, retención. test: logrotate -d /etc/logrotate.conf.



4. LOGGING CENTRALIZADO

rsyslog puede enviar logs a servidor remoto. Configurar action(type="omfwd")

con protocolo, host y puerto. Útil para auditoría y monitoreo.

EJEMPLO DE CÓDIGO
----------------------------------------

$ journalctl -u nginx --since '1 hour ago'
$ journalctl -f
$ tail -f /var/log/syslog
$ cat /etc/logrotate.d/nginx

/var/log/nginx/*.log {
  daily
  rotate 14
  compress
  delaycompress
  notifempty
  create 640 www-data adm
  postrotate
    [ ! -f /var/run/nginx.pid ] || kill -USR1 `cat /var/run/nginx.pid`
  endscript
}
