CONTENIDO DEL CAPÍTULO
===============================================================================

1. DISCOS

lsblk lista dispositivos de bloque. fdisk -l particiones. df -h espacio en

discos. du -sh carpeta tamaño de carpeta. blkid UUID de dispositivos.



2. PARTICIONES

fdisk /dev/sda particionar MBR. gdisk /dev/sda particionar GPT. parted.

mkfs.ext4 /dev/sda1 formatear. Crear particiones con fdisk (n, p, w).



3. FSTAB

/etc/fstab montajes automáticos. Formato: UUID dir tipo opciones dump pass.

opciones: defaults, noatime, errors=remount-ro. Usar UUID en vez de /dev/sdX.



4. MONTAJE

mount /dev/sda1 /mnt monta. umount /mnt desmonta. mount -a monta todo fstab.

findmnt ver árbol de montajes. swapoff/swapon para swap.

EJEMPLO DE CÓDIGO
----------------------------------------

$ lsblk
$ df -h
$ du -sh /home/user
$ blkid
# /etc/fstab
UUID=abc123 / ext4 defaults,errors=remount-ro 0 1
UUID=def456 /home ext4 defaults 0 2
UUID=ghi789 none swap sw 0 0
$ mount -a
