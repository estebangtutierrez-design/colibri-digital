CONTENIDO DEL CAPÍTULO
===============================================================================

1. TMUX

Terminal multiplexer: múltiples terminales en una ventana, persistencia,

sesiones desmontables. Instalar: apt install tmux.



2. COMANDOS BÁSICOS

tmux new -s nombre nueva sesión. tmux attach -t nombre reconecta.

tmux ls lista sesiones. Prefix (Ctrl+b) seguido de comando.



3. GESTIÓN DE PANELES

Ctrl+b % divide vertical. Ctrl+b " divide horizontal. Ctrl+b flecha navega.

Ctrl+b x cierra panel. Ctrl+b z maximiza/restaura. Ctrl+b espacio cambia layout.



4. GESTIÓN DE VENTANAS

Ctrl+b c nueva ventana. Ctrl+b , renombra. Ctrl+b n/p siguiente/anterior.

Ctrl+b w lista ventanas. Ctrl+b d desmonta sesión. Ctrl+b : prompt comandos.

EJEMPLO DE CÓDIGO
----------------------------------------

# Iniciar sesion
tmux new -s desarrollo
# Dentro de tmux (Prefix = Ctrl+b)
Ctrl+b %  # panel vertical
Ctrl+b "  # panel horizontal
Ctrl+b c  # nueva ventana
Ctrl+b ,  # renombrar ventana
Ctrl+b d  # desmontar (detach)
# Reconectar
tmux attach -t desarrollo
# Config ~/.tmux.conf
set -g mouse on
set -g default-terminal 'screen-256color'
bind | split-window -h
bind - split-window -v
