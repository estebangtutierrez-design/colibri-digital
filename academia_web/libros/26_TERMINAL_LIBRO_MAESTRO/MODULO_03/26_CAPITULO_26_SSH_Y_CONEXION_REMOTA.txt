CONTENIDO DEL CAPÍTULO
===============================================================================

1. CONFIGURACIÓN SSH

/etc/ssh/sshd_config: Port 22, PermitRootLogin no, PasswordAuthentication no.

PubkeyAuthentication yes. Siempre cambiar puerto por defecto.



2. SSH KEYS

ssh-keygen -t ed25519 genera clave moderna. -t rsa -b 4096 RSA seguro.

ssh-copy-id usuario@host copia clave al servidor. Claves en ~/.ssh/.



3. CONFIG CLIENTE (~/.ssh/config)

Host alias. HostName, User, Port, IdentityFile, LocalForward. Atajos para

conexiones frecuentes. ProxyJump para servidores intermedios.



4. TÚNELES Y AGENTE

ssh -L 8080:localhost:80 user@host tunel local. ssh -R remoto:local.

ssh -D 1080 socks proxy. ssh-agent para gestión de claves. ssh-add agrega.

EJEMPLO DE CÓDIGO
----------------------------------------

# ~/.ssh/config
Host servidor
  HostName 192.168.1.100
  User admin
  Port 2222
  IdentityFile ~/.ssh/servidor.key
  LocalForward 3306 localhost:3306

Host jump
  HostName gateway.example.com
  User user
  ProxyJump jump

$ ssh-keygen -t ed25519
$ ssh-copy-id user@server
$ ssh -D 1080 user@server # tunel SOCKS
