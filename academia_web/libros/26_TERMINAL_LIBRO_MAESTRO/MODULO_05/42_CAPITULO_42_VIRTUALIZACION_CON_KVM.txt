CONTENIDO DEL CAPÍTULO
===============================================================================

1. KVM/QEMU

Virtualización nativa en Linux. kvm-ok verifica soporte. virt-manager GUI.

qemu-system-x86_64 CLI. Paquetes: qemu-kvm, libvirt-daemon, virt-manager.



2. VIRSH

virsh list --all VMs. virsh start/stop/shutdown VM. virsh create xml.

virsh edit VM edita configuración. virsh snapshot-create-as snapshot.

virsh undefine elimina VM (sin discos).



3. CREAR VM

virt-install --name vm1 --ram 2048 --vcpus 2 --disk size=20 --cdrom iso.

--os-variant ubuntu22.04. --network bridge:br0. --graphics vnc.



4. REDES

Bridge: bridge network para VMs con IP propia. NAT: red privada.

Open vSwitch: SDN virtual. Macvtap: rendimiento directo.

EJEMPLO DE CÓDIGO
----------------------------------------

# Verificar soporte
kvm-ok

# Crear VM
virt-install \
  --name ubuntu-server \
  --ram 2048 \
  --vcpus 2 \
  --disk size=20 \
  --cdrom ubuntu-22.04.iso \
  --os-variant ubuntu22.04 \
  --network bridge=br0 \
  --graphics vnc

# Gestion con virsh
virsh list --all
virsh start ubuntu-server
virsh console ubuntu-server
