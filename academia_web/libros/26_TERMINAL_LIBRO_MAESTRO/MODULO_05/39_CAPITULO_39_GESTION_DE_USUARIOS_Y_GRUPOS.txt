CONTENIDO DEL CAPÍTULO
===============================================================================

1. USUARIOS

useradd -m -s /bin/bash usuario crea con home. passwd usuario asigna

contraseña. usermod -aG grupo usuario agrega a grupo. userdel -r usuario

elimina con home. who, w, users usuarios activos.



2. GRUPOS

groupadd grupo crea. groupmod -n nuevo viejo renombra. groupdel elimina.

groups usuario lista grupos. Archivos: /etc/passwd, /etc/shadow, /etc/group.



3. PERMISOS AVANZADOS

setuid (chmod u+s): ejecuta como propietario. setgid (chmod g+s): hereda grupo.

sticky bit (chmod +t): solo propietario elimina. ls -la muestra: s, S, t.



4. ACLs

getfacl archivo lista ACLs. setfacl -m u:user:rwx archivo asigna.

setfacl -m g:group:rx archivo. setfacl -x u:user archivo elimina.

setfacl -Rb archivo elimina todas.

EJEMPLO DE CÓDIGO
----------------------------------------

# Crear usuario desarrollador
useradd -m -s /bin/bash -G sudo,docker devuser
passwd devuser

# ACLs para proyecto compartido
setfacl -m u:devuser:rwx /var/www/project
setfacl -m g:www-data:rx /var/www/project
setfacl -d -m g:www-data:rx /var/www/project

# SUID ejemplo
chmod u+s /usr/bin/passwd
chmod g+s /shared/directory
chmod +t /tmp
