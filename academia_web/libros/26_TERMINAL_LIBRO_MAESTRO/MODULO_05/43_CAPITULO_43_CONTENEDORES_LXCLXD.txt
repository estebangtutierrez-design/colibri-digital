CONTENIDO DEL CAPÍTULO
===============================================================================

1. LXC/LXD

Contenedores a nivel de sistema operativo. Lighter que KVM, más aislado

que Docker. LXD es el hypervisor de contenedores moderno.



2. INSTALAR LXD

snap install lxd. lxd init configura: almacenamiento, red, clustering.

lxc list muestra contenedores. lxc image list images: imágenes.



3. GESTIÓN

lxc launch ubuntu:22.04 contenedor crea e inicia. lxc exec contenedor bash

ejecuta comando. lxc file push/pull archivos. lxc config set recursos.

lxc snapshot backup. lxc copy/move contenedores.



4. PERFILES Y RECURSOS

lxc profile create perfil. lxc profile edit perfil configura: CPU,

memoria, discos, redes. lxc profile assign container perfil aplica.

Límites: limits.cpu, limits.memory, limits.processes.

EJEMPLO DE CÓDIGO
----------------------------------------

# Instalar e iniciar
snap install lxd
lxd init --auto

# Contenedor basico
lxc launch ubuntu:22.04 web-server
lxc exec web-server -- apt update
lxc exec web-server -- apt install nginx -y

# Configurar recursos
lxc config set web-server limits.cpu 2
lxc config set web-server limits.memory 512MB

# Snapshot y backup
lxc snapshot web-server pre-update
lxc restore web-server pre-update

# Publicar imagen
lxc publish web-server --alias mi-ubuntu
