CONTENIDO DEL CAPÍTULO
===============================================================================

1. VARIABLES DE ENTORNO

export VAR=valor hace disponible a procesos hijo. env lista todas.

printenv VAR muestra valor. unset VAR elimina. $VAR o ${VAR}.



2. ARCHIVOS DE PERFIL

/etc/environment variables globales. ~/.bashrc bash interactivo no login.

~/.bash_profile login shell. ~/.profile genérico. /etc/profile global.



3. ORDEN DE CARGA

Login: /etc/profile → ~/.bash_profile → ~/.bashrc. No login: ~/.bashrc.

source archivo o . archivo recarga sin nuevo login.



4. PATHS

PATH=$PATH:/nuevo/directorio agrega al PATH. which comando encuentra.

type comando muestra cómo se interpreta. hash -r refresca caché.

EJEMPLO DE CÓDIGO
----------------------------------------

# ~/.bashrc
export EDITOR=vim
export BROWSER=firefox
export PATH=$PATH:$HOME/.local/bin:$HOME/go/bin

# Variables para desarrollo
export NODE_ENV=development
export DATABASE_URL="postgresql://localhost:5432/mydb"
export PYTHONPATH=$PYTHONPATH:$HOME/projects

# Alias
alias ll='ls -la'
alias gs='git status'
alias dc='docker compose'
alias ..='cd ..'
