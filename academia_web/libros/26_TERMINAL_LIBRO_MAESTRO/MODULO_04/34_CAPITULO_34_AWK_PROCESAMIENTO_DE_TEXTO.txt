CONTENIDO DEL CAPÍTULO
===============================================================================

1. AWK BÁSICO

awk '{print $1}' archivo imprime primera columna. awk -F: '{print $1}' /etc/passwd

usando : como separador. NF número de campos. NR número de línea.



2. AWK CON PATRONES

awk '/error/ {print}' imprime líneas con error. awk '$3 > 100 {print $1, $3}'

condiciones en columnas. awk 'NR > 1 {print}' salta primera línea (header).



3. AWK AVANZADO

awk 'BEGIN {sum=0} {sum+=$3} END {print sum}' suma columna. awk '{for(i=1;i<=NF;i++)

s+=i; print s}' procesa cada campo. Funciones: length, substr, index.



4. AWK SCRIPTS

awk -f script.awk archivo. Scripts con BEGIN, bloques por patrón, END.

Variables, arrays asociativos, printf para formato. Potente para análisis.

EJEMPLO DE CÓDIGO
----------------------------------------

# Promedio de una columna
awk '{sum+=$2; count++} END {print "Promedio:", sum/count}' data.txt

# Formato de tabla
awk '{printf "%-20s %5d\n", $1, $3}' datos.tsv

# Analizar access log
awk '{print $1}' access.log | sort | uniq -c | sort -rn | head -10

# Conversion de formato
awk -F',' '{print $2","$1}' data.csv > reordenado.csv
