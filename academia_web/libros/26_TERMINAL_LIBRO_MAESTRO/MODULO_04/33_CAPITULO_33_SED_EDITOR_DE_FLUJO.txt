1. SED BÁSICO

sed 's/old/new/' archivo reemplaza primera ocurrencia. sed 's/old/new/g'

global. sed 's/old/new/2' segunda ocurrencia. sed 's/old/new/g' archivo > nuevo.



2. SED CON DIRECCIONES

sed '3s/old/new/' línea 3. sed '3,10s/old/new/' líneas 3-10. sed '/patron/s/old/new/'

solo líneas con patrón. sed '/^$/d' elimina líneas vacías.



3. SED COMANDOS MÚLTIPLES

sed -e 's/old1/new1/g' -e 's/old2/new2/g' archivo. sed 's/old/new/g; s/foo/bar/g'

con ;. sed -f script.sed archivo desde archivo de comandos.



4. SED IN-PLACE

sed -i 's/old/new/g' archivo edita directamente. sed -i.bak 's/old/new/g'

archivo crea backup .bak. ¡Cuidado! No hay deshacer.

EJEMPLO DE CÓDIGO
----------------------------------------

# Reemplazar en multiples archivos
sed -i 's/localhost:3000/api.example.com/g' src/*.js

# Eliminar lineas de comentarios
sed '/^#/d' config.conf > config_clean.conf

# Extraer seccion especifica
sed -n '/^\[database\]/,/^\[/p' config.ini

# Formatear JSON mal formado
sed 's/,/,\n/g' data.json | sed 's/{/{\n/g' | sed 's/}/\n}/g'
