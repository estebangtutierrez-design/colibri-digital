CONTENIDO DEL CAPÍTULO
===============================================================================

1. PROMETHEUS + GRAFANA

Métricas de sistema (node_exporter), aplicación, BD. Dashboards.

Alertas con AlertManager. Notificaciones: email, Slack, PagerDuty.



2. LOGGING CENTRALIZADO

ELK Stack: Elasticsearch, Logstash, Kibana. Filebeat para enviar logs.

Grok patterns para parsear logs. Dashboards de errores y tendencias.



3. HEALTH CHECKS

Endpoint /health para load balancer. Checks: BD, Redis, disk, API externa.

Status page. Uptime monitoring (Pingdom, UptimeRobot).



4. ALERTAS

CPU > 80%, memoria > 90%, disco > 85%, certificado SSL próximo a expirar.

Respuesta 5xx > 5%, latencia > 2s. Backup fallido.

EJEMPLO DE CÓDIGO
----------------------------------------

# Prometheus config
scrape_configs:
  - job_name: 'node'
    static_configs:
      - targets: ['localhost:9100']
  - job_name: 'app'
    static_configs:
      - targets: ['localhost:3000']

# Alerta ejemplo (Prometheus rules)
groups:
  - name: alertas
    rules:
      - alert: HighCPU
        expr: 100 - (avg by(instance) (rate(node_cpu_seconds_total{mode="idle"}[5m])) * 100) > 80
        for: 5m
        annotations:
          summary: "CPU alta en {{ $labels.instance }}"
