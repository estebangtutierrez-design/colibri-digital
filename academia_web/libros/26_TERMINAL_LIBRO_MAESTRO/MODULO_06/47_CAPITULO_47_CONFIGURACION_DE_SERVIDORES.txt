CONTENIDO DEL CAPÍTULO
===============================================================================

1. SISTEMA BASE

Ubuntu Server 22.04 LTS. Usuario no-root con sudo. SSH con claves.

Firewall (ufw). Fail2ban. Automatic security updates. NTP sincronización.



2. SERVIDOR WEB

nginx: reverse proxy, SSL termination, static files, load balancing.

Configuración de virtual hosts. HTTP/2. gzip compresión. Caching.



3. BASE DE DATOS

PostgreSQL: configuración de rendimiento, autenticación, backups.

Connection pooling con PgBouncer. Replicación streaming. PITR recovery.



4. APLICACIÓN

Dockerizado con docker-compose. Variables de entorno. Health checks.

Graceful shutdown. Logging estructurado. Métricas con Prometheus.

EJEMPLO DE CÓDIGO
----------------------------------------

# Configuracion nginx
server {
  listen 443 ssl http2;
  server_name example.com;
  
  ssl_certificate /etc/letsencrypt/live/example.com/fullchain.pem;
  ssl_certificate_key /etc/letsencrypt/live/example.com/privkey.pem;
  
  location / {
    proxy_pass http://app:3000;
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
  }
  
  location /static {
    alias /var/www/static;
    expires 30d;
    add_header Cache-Control "public, immutable";
  }
}
